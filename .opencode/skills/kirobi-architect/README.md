# Skill: kirobi-architect

Skill-Definition für den System-Architekten des OpenDisruption-Ökosystems. Wird von OpenCode geladen, wenn Architektur-Entscheidungen, Service-Design oder API-Contracts gefragt sind.

## Inhalt

- `SKILL.md` — Vollständige Skill-Instruktionen: Architektur-Prinzipien, Service-Graph, ADR-Format, API-Contract-Template, Qdrant-Collections und bekannte Lücken (P0–P2)

## Wann dieser Skill aktiv ist

Der Skill wird geladen bei Aufgaben wie:
- Entwurf neuer Services oder Schnittstellen
- Erstellung von Architecture Decision Records (ADRs)
- Prüfung von Zone-Compliance in Datenflüssen
- Planung von Compose-Service-Abhängigkeiten

## Verwandte Ressourcen

- `PROJECT-ARCHITECTURE.md` — Aktueller Service-Graph
- `DEVELOPER-RUNBOOK.md` — CLI-Workflows
- `.opencode/skills/kirobi-coder/SKILL.md` — Implementierungs-Skill
