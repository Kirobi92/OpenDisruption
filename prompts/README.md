# prompts – Prompt-Bibliothek

**Zone:** WORKSPACE | **Verantwortlich:** kirobi-architect, kirobi-core

## Zweck
Zentrale Bibliothek aller Prompts für das gesamte Kirobi-Ökosystem. Strukturiert nach Anwendungsfall.

## Kategorien

| Verzeichnis | Inhalt |
|-------------|--------|
| `system/` | System-Prompts für Agenten |
| `tasks/` | Task-spezifische Prompts |
| `templates/` | Prompt-Templates |
| `chains/` | Multi-Step Prompt-Chains |
| `few-shot/` | Few-Shot Beispiele |
| `evaluation/` | Evaluation-Prompts |
| `personas/` | Persona-Definitionen |
| `installer/` | Installer-Agent-Prompts |
| `experiments/` | Experimentelle Prompts |

## Namenskonvention

```
[kategorie]-[aufgabe]-[version].md
Beispiel: system-kirobi-supervisor-v2.md
```
