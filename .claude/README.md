# .claude

Konfigurationsverzeichnis für Claude Code / Claude-basierte Agenten im OpenDisruption-Ökosystem.
Enthält Identitätsdefinitionen, ECC-Tool-Konfigurationen, Homunculus-Instinkte und Skill-Definitionen.

## Wichtige Dateien & Ordner

- `identity.json` — Agenten-Identität und Persönlichkeitsparameter für Claude
- `ecc-tools.json` — Tool-Konfiguration für ECC (Embedded Code Capabilities)
- `homunculus/` — Instinkte und Verhaltensregeln (inkl. vererbte OpenDisruption-Instinkte)
- `skills/` — Skill-Definitionen im Claude-Format
