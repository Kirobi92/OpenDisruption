# Kirobi Core

## Zweck
Das Kern-Verzeichnis des Kirobi-Agenten. Hier leben die Identität, Prompts, Routing-Logik, Policies und alle Konfigurationen, die Kirobi zu dem machen, was er ist.

| `core-routing.md` | Routing-Logik für Modelle und Agenten |
| `core-policies.md` | Operative Kern-Policies |
| `core-events.log` | Append-only Laufzeit-Event-Log (wird lokal erzeugt, Git-ignoriert) |

| `core-fallbacks.md` | Fallback-Strategien |
| `core-prompts/` | System-Prompts für alle Agenten |
| `schemas/` | Daten-Schemas |
