# archive/retired-ideas

Ideen, Konzepte und Entwürfe, die evaluiert wurden und bewusst nicht weiterverfolgt werden.

## Zweck

Nicht jede Idee wird umgesetzt — aber das Wissen, *warum* etwas verworfen wurde, ist wertvoll. Dieses Verzeichnis verhindert, dass dieselben Ideen mehrfach neu diskutiert werden.

## Konventionen

- Jede Datei sollte kurz erklären: **Was war die Idee?** und **Warum wurde sie zurückgestellt?**
- Format: Markdown mit Frontmatter (`zone`, `retired_at`, `reason`)
- Keine Löschung — Verzeichnis dient als Entscheidungsgedächtnis

## Beispiel-Frontmatter

```yaml
---
zone: WORKSPACE
retired_at: 2026-05-01
reason: Zu hoher Infrastrukturaufwand für den erwarteten Nutzen
---
```
