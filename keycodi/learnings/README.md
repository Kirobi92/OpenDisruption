# KeyCodi — Learnings (Append-only Fehler-Log)

**Zone:** WORKSPACE

Jede Reibung, jeder Fehler, jede überraschende Beobachtung wird hier als eigenes Dokument abgelegt. Vorhandene Einträge werden **nicht** gelöscht und **nicht** umsortiert.

## Konvention

- Dateiname: `NNNN-<kebab-slug>.md`, NNNN ist 4-stellig, fortlaufend, beginnend bei `0001`.
- Vorlage: `0000-template.md` (kopieren, NNNN hochzählen, Slug setzen).
- Pflichtfelder: Datum, Phase, Symptom, Hypothesen, Versuche, Ergebnis, Folgewirkung.
- Falls Issue/PR vorhanden: verlinken.
- Niemals SACRED-, FAMILY_PRIVATE- oder QUARANTINE-Inhalte aufnehmen — nur Metadaten und Zusammenfassungen.
