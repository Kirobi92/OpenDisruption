# KeyCodi — Decisions (ADRs)

**Zone:** WORKSPACE

Architecture Decision Records (ADRs) für KeyCodi-relevante Entscheidungen. Format: leichtgewichtige Markdown-Vorlage (`0000-adr-template.md`).

## Konvention

- Dateiname: `NNNN-<kebab-slug>.md`, NNNN 4-stellig, fortlaufend.
- Status: `proposed` → `accepted` | `rejected` | `superseded by NNNN`.
- ADRs werden **nicht** rückwirkend editiert (außer Status-Update + Verweis auf Nachfolger).
- Kontroverse Entscheidungen bekommen einen Sven-Sign-off-Vermerk.
