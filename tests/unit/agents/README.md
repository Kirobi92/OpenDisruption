# tests/unit/agents

Unit-Tests für alle spezialisierten Agenten des OpenDisruption-Ökosystems.
Jeder Agent hat ein eigenes Unterverzeichnis mit Smoke-Tests und Zonen-Sicherheitstests.

## Wichtige Unterordner

- `hermes/` — Tests für den Hermes-Extraktor (Ingestion, Zonen-Verweigerung)
- `openclaw/` — Tests für den OpenClaw-Agenten (Smoke, Zonen-Verweigerung)
- `obsidian/` — Tests für die Obsidian-Integration
- `opencode/` — Tests für den OpenCode-Agenten
