# Mietobjekt Registerkarte Garantie

Miete / Mietobjekt 

# Mietobjekt Registerkarte Garantie 
Mietobjekt Registerkarte Garantie 

Auf dieser Registerkarte werden Einstellungen für die Garantiezeit vorgenommen. Diese Einstellungen haben nur einen informativen Charakter. 
### Feld Garantiebeginn 
Der Beginn der Garantiezeit wird eingetragen. 

Tooltip: 

Beginn der Garantiezeit 

### Feld Garantieende 
Das Ende der Garantiezeit wird eingetragen. 

Tooltip: 

Ende der Garantiezeit 

### Feld Kulanzende 
Das Ende der Kulanzzeit wird eingetragen. 

Tooltip: 

Ende der Kulanzzeit 

### Feld Inbetriebnahme 
Das Datum der Inbetriebnahme des Mietobjekts wird eingetragen. 

Tooltip: 

Datum der Inbetriebnahme 

### Feld Ende 
Das Ende der Nutzung wird eingetragen. 

Tooltip: 

Angabe des Ende-Datums 

### Feld Letzte Wartung 
Das Datum der letzten Wartung wird eingetragen. 

### Feld Wartungsperiode 
Hier wählen Sie einen Zeitraum aus. 

### Feld Memo 
Hier kann ein individueller Text für interne Zwecke eingetragen werden. 

Tooltip: 

Enthält interne Textinformationen. 

### Feld Amtl. Prüf.termin 
Falls das Mietobjekt TÜV-geprüft ist, wird hier der nächste TÜV-Termin eingetragen. Wenn dieses Datum in einen Vermietungszeitraum fällt, erhalten Sie eine Hinweismeldung. 

Tooltip: 

Termin für TÜV/Wiederkehrende Prüfungen 

### Checkbox Betriebsstunden 
Wenn Sie diese Checkbox aktivieren, werden für das Mietobjekt Betriebsstunden gepflegt. Die nachfolgenden Felder werden zugänglich. 

Tooltip: 

Wenn gesetzt, können die nachfolgenden Felder gepflegt werden. 
Die Anzahl der zulässigen Betriebsstunden pro Tag werden bei der Rücknahme (Retoure) des Mietobjektes 
mit der Anzahl der tatsächlich geleisteten Betriebsstunden verglichen. 

### Feld Tag 
Hier geben Sie die Betriebsstunden für das Mietobjekt ein, die es regulär pro Tag in Betrieb sein darf. 

Tooltip: 

Betriebsstunden/Tag 
Anzahl der regulären Betriebsstunden für das Mietobjekt. 

### Feld Ist 
Hier tragen Sie bei Anlage des Mietobjektes die Betriebsstunden ein. Bei der Rückgabe des Mietobjektes wird der Ist-Zählerstand vom Gerät abgelesen und in das Mietobjekt rückaktualisiert. 

Tooltip: 

Betriebsstunden Ist 
Betriebsstunden laut Betriebsstundenanzeiger. 

### Feld Wartung nach 
Hier geben Sie einen Zählerstand ein. Wenn dieser erreicht ist, hat eine Wartung zu erfolgen. 

Tooltip: 

Wartung nach Betriebsstunden 
Das Mietobjekt muss gewartet werden, 
wenn sein Betriebsstundenzähler diesen Wert erreicht hat.
