# Sicherheiten 
Sicherheiten 

Dieser Dialog öffnet sich im Kunde >> Optionen >> Sicherheiten . Hier verwalten Sie die Sicherheiten des Kunden. Dafür geben Sie im Kopf des Dialogs zuerst die Kundendaten ein und anschließend die Informationen für die Sicherung der Wertsachen. 

Für aktive Sicherheiten, die das Gültig-bis-Datum, abzüglich Erinnerungsfrist, erreicht haben, kann eine Wiedervorlage an den hinterlegten Sachbearbeiter generiert werden. Diese Generierung kann per Job-Server automatisiert werden. 

Da manche Sicherheiten ein „Ablaufdatum“ haben, kann eine Sicherheit mit einer Laufzeit versehen werden (Spalten ‚Gültig ab‘ / ‚Gültig bis‘). 

Lautet der Sicherheitstyp ‚erhalten‘, ist das Feld Zinssatz unsichtbar und die Checkbox Limitwirksam sichtbar. Wird diese gesetzt, fließt der Bewertungsbetrag unter Berücksichtigung der Abschreibung und des Gültig-bis-Datums in das Kreditlimit des Kunden ein. 

Lautet der Sicherheitstyp ‚erteilt‘, ist die Checkbox Limitwirksam nicht sichtbar. 

Wird eine Sicherheit gelöscht, wird diese physikalisch gelöscht.
