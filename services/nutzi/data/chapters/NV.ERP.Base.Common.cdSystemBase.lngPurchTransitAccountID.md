# Kapitel NV.ERP.Base.Common.cdSystemBase.lngPurchTransitAccountID

Wenn die Checkbox Permanente Lagerbewertung aktiviert ist, ist hier zwingend ein (Transit-)Konto auszuwählen. Der Wareneingang wird über die Sachkontenübernahme an die Finanzbuchhaltung gebucht. Der Positionspreis wird auf das Transitkonto gebucht, mit dem Bestandsänderungskonto des Artikels als Gegenkonto. Nach erfolgter Rechnungsprüfung wird diese wieder entlastet. 

In diesem Zusammenhang beachten Sie bitte auch die Informationen zur Checkbox Alle Lager im Dialog Lagerorte . 

Tooltip: 

Wenn Sie hier ein (Transit-)Konto auswählen, wird die Wareneingangsbuchung über die Sachkontenübernahme an die Finanzbuchhaltung gemeldet.
