# Auswertungsstruktur Kore Registerkarte Kostenarten/Leistungen 
Auswertungsstruktur Registerkarte Kostenarten/Leistungen 

Auf dieser Registerkarte definieren Sie die Einstellungen für die Auswertungen über Kostenarten / Leistungen. 
### Combobox Auswertung 
Hier sind alle gespeicherten Auswertungsstrukturen der Kostenarten / Leistungen verfügbar. Wählen Sie eine Struktur aus, wird diese im linken Teil des Dialogs angezeigt und kann bearbeitet werden. 

### Feld Zeilentext 
In diesem Feld befindet sich die Beschriftung der markierten Stelle in der Auswertungsstruktur. Dieser Text kann frei editiert werden. 

Tooltip: 

Zeilentext 

### Feld Summentext 
Angabe des Zeilentextes bei Bezug auf eine Summenzeile. Das Wort „Summe“ wird dem Text automatisch vorangestellt. Dieser Text kann frei editiert werden. 

Wenn Sie den Button neben dem Feld Summentext drücken, wird der Zeilentext ins Feld Summentext übernommen. 

Tooltip: 

Der Summentext wird in der Auswertung in Spalte Bezeichnung auf der Summenzeile ausgewiesen. 

### Button Satz hinzufügen 

Tooltip: 

Automatisches Füllen des Feldes Summentext. 

### Feld 100% Basis 
In diesem Feld haben Sie die Möglichkeit bis zu 5 Stufen (1 bis 5) als Basen festzulegen. In der Kostenauswertung erzeugen Sie hiermit je Stufe 1-5 Vergleichsspalten. In jeder der Vergleichsspalten werden alle Auswertungsstufen auf die jeweilige 100%-Basis berechnet. 

### Feld Basistext 
Mit den Feldern 100% Basis und Text haben Sie die Möglichkeit bis zu 5 verschiedene 100%-Basis-Stufen festzulegen. 

Diese Felder werden zugänglich, wenn im Stukturbaum die Ebene Stufe markiert ist. 

Die Basis-Stufen werden mit den Ziffern 1-5 belegt. 

Im 1. Feld wird eine individuelle Nummer vergeben z.B. 1, im 2. Feld dazu die individuelle Bezeichnung z.B. „Basis DB I“. 

In einer Auswertung können maximal 5 verschiedene 100% Basis-Stufen zugewiesen werden. 

Im Dialog Kostenauswertungen , Registerkarte Base Reporting , Registerkarte Selektionen 2 , wird die Checkbox Vergleich mit 100% Basis aktiviert und eine 100% Basis-Stufe ausgewählt. => Die in der Auswertungsstruktur definierten Summen werden alle auf die gewählte 100% Basis prozentual berechnet und ausgewiesen. 

Wenn Sie den Button neben dem Feld 100% Basis drücken, wird immer die nächste Ebene mit einer Basis in der Auswertungs-Struktur gesucht. Ist die Suche unten angekommen, wird von oben gesucht.

### Combobox Anzeigemodus 
Diese Combobox wird eingeblendet, wenn im Baum die Hierarchieebene 'Stufe' und das Icon ausgewählt ist. 

Diese Combobox wird nur zugänglich, wenn im Stukturbaum die Ebene Stufe markiert wird. 

In der Combobox steht zur Verfügung: 
- Null : Der Zeilentext wird als Überschrift vor den Details ausgewiesen, danach folgt die Summenzeile. - Ohne Zeilentext : Der Zeilentext als Überschrift vor den Details wird unterdrückt. Nach den Details folgt die Summenzeile. - Nur Summenzeile : Der Zeilentext als Überschrift und die Details werden unterdrückt. Es wird nur die Summenzeile ausgewiesen. 
Die Einstellung wirkt sich in der Spalte 'Bezeichnung' im Dialog Kostenauswertungen und im Report aus. 

### Field Panel #fpnCELineCategory# 

Diese Radiobuttons bestimmen die weiteren Eigenschaften der einzelnen Ebenen der Auswertungsstruktur. Je nach gewählter Eigenschaft haben Sie unterschiedliche Möglichkeiten. 
### Radiobutton Stufe 
Innerhalb einer Stufe werden alle Basiswerte in der Auswertung zu einem Ergebnis zusammengerechnet. 

Sie können beliebig viele Stufen ineinander verschachteln. 

### Radiobutton Text 
Diese Positionen im Strukturaufbau sind reine Texte und dienen dem Layout der Auswertung. 

### Radiobutton Detail 
Nur einem Detail können Basiswerte zugeordnet werden! 

D.h. wenn Sie einer Stufe Basiswerte zuordnen wollen, müssen Sie zuvor innerhalb der Stufe ein Detail einfügen. 

### Radiobutton Basis 
Die Basis bilden die Kostenarten/Leitungen. Anhand ihrer Werte erfolgen die Berechnungen innerhalb einer Stufe. 

### Tabelle 

In der Tabelle befinden sich alle gespeicherten Kostenarten und Leistungen, welche einer Detail-Stufe zugeordnet werden können. Hierfür müssen Sie in der Struktur die Detailstufe und in der Tabelle eine oder mehrere Kostenarten/Leistungen markieren. 
### Tabellenspalte KA/L 
Hier wird eine Kosten- oder Leistungsartennummer für die Kostenrechnung angegeben. 

Tooltip: 

Kostenarten-/Leistungsnummer 

### Tabellenspalte Bezeichnung 
Der Name der Kostenart/Leistung wird angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte KA/L-Typ 
Hier wird der Kostenarten- / Leistungstyp angegeben. Zur Verfügung stehen: 
KA = Kostenart 
L = Leistung 

Tooltip: 

Kostenarten-/Leistungstyp 

### Checkbox Zugeordnete ausblenden 
Wenn Sie diese Checkbox nicht aktivieren, werden die bereits zugeordneten Elemente farbig hinterlegt. 

Diese Checkbox kann auch für eine bessere Übersicht genutzt werden. Wenn Sie diese Checkbox aktivieren, werden alle bereits in der Struktur zugeordneten Kostenarten/Leistungen aus der Tabelle ausgeblendet. In dem daneben liegenden Feld können Sie über die Kostenarten-/Leistungsnummer diese schnell in der Tabelle oder im Strukturbaum finden. 

### Dock Panel Operator 

Über die Radiobuttons Addieren bzw. Subtrahieren können Sie der markierten Summenstufe die Rechenfunktion für die Dialogtabelle bzw. den Report zu. Sie legen fest, ob eine Summenstufe im Report addiert oder subtrahiert werden soll.
