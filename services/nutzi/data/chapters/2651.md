# Teilabgang 
Teilabgang 

Dieses Menü öffnet sich über Anlage >> Buchen >> Teilabgang . Hier kann ein Teil einer Anlage (z.B. wenn ein Teil einer Anlage verkauft wurde) als Abgang gebucht werden. 

Der Teilabgang wird zwingend immer für alle Bereiche durchgeführt. Sie haben jedoch die Möglichkeit, ausgehend von jedem einzelnen Bereichskennzeichen, die prozentuale Verteilung anzustoßen. 

Dabei werden beim Anlagegut durch Buchungssätze die Werte (Buchwert) um den abgehenden Teil reduziert und für den abgehenden Anteil wird eine „Unteranlage“ generiert. Diese wird dann zu 100% „abgegangen“. 

Eine Anlage hat z.B. einen Buchwert von 1234,56 EUR. Tatsächlich sollen z.B. 300 EUR abgehen. Sie können die Höhe des Teilabgangs prozentual oder absolut angeben. Eine Eingabe in dem Feld Prozent bzw. Betrag aktualisiert automatisch das andere Feld. 

Somit wird der Teilabgang bei den Auswertungen transparent dargestellt. Auch eine rückwirkende Auswertung ist somit gewährleistet. 

Eine Annullierung des Teilabgangs kann generell nur über den Dialog Anlagenjournal erfolgen.
