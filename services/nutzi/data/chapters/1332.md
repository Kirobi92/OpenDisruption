# Wertestamm 
Wertestamm 

Dieser Dialog öffnet sich im Dialog Merkmal Klasse oder im Artikel , Registerkarte Merkmale über den Button Merkmale . Hier werden die ML-Werte von Werten (alternative Werte für den gleichen Satz in unterschiedlichen Sprachen) gepflegt. Diese werden z.B. auf dem Artikelstamm sichtbar, wenn die Sprache für Merkmale geändert wird und Sie für denselben Wert eine andere ML-Version sehen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Wertestamm 
Tooltip: 

In diesem Dialog werden Merkmalswerte definiert und gepflegt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Combobox Datentyp 
Aus dieser Combobox wählen Sie den Datentyp aus. Zur Verfügung stehen u.a. Zeichenfolge, Ganzzahl, Dezimalzahl. Wenn dieser Typ einmal gespeichert ist, darf er nicht mehr geändert werden. 

Tooltip: 

Typ des Merkmals. 
Der Typ darf nach dem Speichern nicht mehr geändert werden. 
Unterstützt werden: Ganzzahl, Kommazahl, Zeichenfolge, Text, Html Text, Datum 
- Zeichenfolge: maximal 250 Zeichen.
