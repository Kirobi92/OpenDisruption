# Kapitel NV.ERP.MM.CashDesk.cdCashDeskParameters.sCashDeskSerialID

Dies ist die eindeutige ID dieser Konfiguration (GUID) für den Dateinamen. Ist noch keine ID gesetzt, handelt es sich noch um einen neuen, ungespeicherten Datensatz. 

Tooltip: 

Seriennummer der Software-Kasse. 
Mit dieser Seriennummer muss die Registrierkasse beim Finanzamt angemeldet werden.
