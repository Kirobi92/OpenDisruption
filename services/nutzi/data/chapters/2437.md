# OP-Altersgliederung Registerkarte Selektionen 2 
OP-Altersgliederung Registerkarte Selektionen 2 

Auf dieser Registerkarte stehen weitere Selektionsmöglichkeiten zur Verfügung. 
### Feld Betrag von 
Geben Sie hier einen Betrag ein, ab dem selektiert werden soll. 

Tooltip: 

Angabe des Betrags 

### Feld Betrag bis 
Geben Sie hier einen Betrag ein, bis zu dem selektiert werden soll. 

Tooltip: 

Angabe des Betrags 

### Combobox OP-Kennz. 
Wählen Sie hier ein OP-Kennzeichen zur Selektion aus. 

Tooltip: 

OP-Kennzeichen 

### Checkbox Rest aus WaWi 
Wenn Sie diese Checkbox aktivieren, werden offene Posten in die Tabelle eingespielt, die in der OP-Anlage die Kennzeichnung Rest aus Wawi erhalten haben. 

Tooltip: 

Rest aus Warenwirtschaft 

### Checkbox Nicht versichert 
Wenn Sie diese Checkbox aktivieren, werden offene Posten in die Tabelle eingespielt, bei denen im Debitorenstamm auf der Registerkarte Versicherung die Beurteilungsart Nicht versichert ausgewählt ist. 

### Checkbox Pauschal vers. 
Wenn Sie diese Checkbox aktivieren, werden offene Posten in die Tabelle eingespielt, bei denen im Debitorenstamm auf der Registerkarte Versicherung die Beurteilungsart Pauschal versichert ausgewählt ist. 

Tooltip: 

Pauschal versichert 

### Checkbox Namentl. vers. 
Wenn Sie diese Checkbox aktivieren, werden offene Posten in die Tabelle eingespielt, bei denen im Debitorenstamm auf der Registerkarte Versicherung die Beurteilungsart Namentlich versichert ausgewählt ist. 

Tooltip: 

Namentlich versichert 

### Combobox Vers.art von 
Wählen Sie hier zur Selektion eine Versicherungsart aus. Es werden dann Kunden mit entsprechenden Einträgen ausgewählt. 

Tooltip: 

Versicherungsart von 

### Combobox bis 
Wählen Sie hier zur Selektion eine Versicherungsart aus. Es werden dann Kunden mit entsprechenden Einträgen ausgewählt. 

### Combobox Kreditversicherer 
Tooltip: 

Kredit-Versicherer 

### Combobox Kundenstatus 
Wählen Sie hier zur Selektion einen Kundenstatus aus. 

### Combobox Bonitätsstatus 
Zur Selektion kann hier der Bonitätsstatus ausgewählt werden. In diesem Zusammenhang gibt es auf der Registerkarte Altersgliederung eine gleichnamige Spalte. 

### Combobox D.Gruppierung 
Hier wählen Sie eine Gruppierung aus. Diese wird auf dem Report angedruckt. Zur Verfügung stehen keine, Vertreter und Kundenstatus . 

Tooltip: 

Gruppierung des Drucks 

### Combobox Niederlassung 
Hier wird die Niederlassungsnummer angegeben. 

Tooltip: 

Angabe der Niederlassung
