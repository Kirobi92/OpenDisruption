# Auftragsarten Registerkarte Druck 
Auftragsarten Registerkarte Druck 

Zu jeder Auftragsart können eigene Formularvarianten erstellt und auf der Registerkarte Druck hinterlegt werden. Verknüpfen Sie die Formulare mit einem Windows-Druckertreiber. Die Drucke werden dann automatisch auf die gewünschten Drucker verteilt. So können beispielsweise die Packliste und/oder der Lieferschein automatisch zu dem im Lager stehenden Drucker geschickt werden. Oder die Auftragsbestätigung wird mit einem Faxdruckertreiber verknüpft und somit automatisch per Fax versendet. 

Die Comboboxen werden wie folgt genutzt: 

Formular, Drucker und Anzahl AB: 
Hier definieren Sie das Formular, den Drucker, auf dem das Formular ausgedruckt werden soll und die Anzahl der Kopien für Auftragsbestätigungen. 

Formular, Drucker und Anzahl PL: 
Hier definieren Sie das Formular, den Drucker, auf dem das Formular ausgedruckt werden soll und die Anzahl der Kopien für Packlisten. 

Formular, Drucker und Anzahl LS: 
Hier definieren Sie das Formular, den Drucker, auf dem das Formular ausgedruckt werden soll und die Anzahl der Kopien für Lieferscheine. 

Formular, Drucker und Anzahl PR: 
Hier definieren Sie das Formular, den Drucker, auf dem das Formular ausgedruckt werden soll und die Anzahl der Kopien für Proforma-Rechnungen. 

Formular, Drucker und Anzahl RG: 
Hier definieren Sie das Formular, den Drucker, auf dem das Formular ausgedruckt werden soll und die Anzahl der Kopien für Rechnungen. 

Durch Kopieren und Umbenennen der mitgelieferten Standardformulare können verschiedene Varianten erzeugt und den einzelnen Auftragsarten zugeordnet werden. 

Bankverbindung: 
Aus dieser Combobox können Sie zur Auftragsart eine bestimmte Bankverbindung auswählen. Diese wird dann immer bei dieser Auftragsart gezogen. 

Was den Druck von Logos etc. auf den Reports betrifft, so ist folgendes zu beachten: 

Es können alle gängigen Bild-Formate (bmp, gif, jpeg, png, tiff, …) ausgewählt werden. 

JPEG, GIF und PNG werden 1:1 gespeichert und auch so an den Report übergeben. 

Alle anderen nicht internetfähigen Formate (bmp, tiff, …) werden verlustfrei in ein PNG umgewandelt. 

Empfehlung: 

Verwenden Sie PNG als Format. 

Vermeiden Sie nach Möglichkeit Bilder auf Subreports.

### Feld Anzahl LS

Geben Sie die Anzahl Kopien für Proforma-Rechnungen an. 

Tooltip: 

Geben Sie die Anzahl Kopien für Proforma-Rechnungen an. 

### Feld Anzahl RG 
Hier kann die Anzahl der Rechnungen für den Einzel- und Massendruck hinterlegt werden. Diese Anzahl gilt nur für Rechnungen, nicht für Gutschriften und Auftragsbestätigungen. Diese Einstellung hat beim Rechnungsdruck im Versenden Priorität vor anderen diesbezüglichen Einstellungen (z.B. Auftragsarten). 

Tooltip: 

Geben Sie die Anzahl Kopien für Rechnungen an. 

### Combobox Bankverbindung 
Hier wird die Standard-Bankverbindung angezeigt. Gibt es keine speziell gekennzeichnete Bankverbindung, wird die jüngste Bankverbindung zum Kunden bzw. Lieferanten angezeigt. Für diesen Vorgang kann die Bankverbindung auch geändert werden. 

Tooltip: 

Ordnen Sie der Auftragsart eine Bankverbindung zu. 

### Combobox IC/CC-Strecke 
Aus dieser Combobox können Sie Druckoptionen für IC/CC-Strecke auswählen: 

- beide Lieferscheine drucken - nur Co-LLieferschein drucken - nur UA-Lieferschein drucken 
Diese Optionen gelten beim Erst- und Wiederholungsdruck.
