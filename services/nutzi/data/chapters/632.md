# Html Editor Registerkarte Format 
HTML Editor Registerkarte Format 

Auf dieser Registerkarte können Sie das Aussehen/Format des Texts bearbeiten. 

Voreingestellt sind die gängigsten Schriftarten, d.h. Schriftarten, die auf den gängigsten Betriebssystemen installiert sind. Es können aber auch andere, speziell auf Ihrem Betriebssystem installierte Schriftarten manuell eingegeben werden. Bei manueller Eingabe einer Schriftart müssen Sie vor den Namen der Schriftart einen Punkt schreiben / setzen (z.B. .Arial Narrow). Diese Schriftart wird nicht in der Combobox gespeichert! 

Dabei ist allerdings zu beachten, dass bei einem Start von eNVenta ERP auf einem anderen Betriebssystem diese Schriftart evtl. nicht hinterlegt ist. 

Folgende Buttons stehen zur Bearbeitung von Text zur Verfügung: 

- Fettdruck (Bold) des Texts 

- Kursivdruck (Italic) des Texts 

- Unterstreichen des Texts 

- Durchstreichen des Texts 

- Hochgestellte Anzeige des Texts 

- Tiefgestellte Anzeige des Texts 

- Linksbündig 

- Rechtsbündig 

- Zentriert 

- Blocksatz 

- Anpassung der Schriftart 

- Anpassung der Schriftfarbe 

- Anpassung der Schriftgröße 

- Anpassung der Hintergrundfarbe 

Um einen geschriebenen Text entsprechend anzupassen, muss dieser markiert werden und über einen der oberen Buttons angepasst werden. Die Anzeige wird in diesem Fall sofort geändert. Bei Klick auf den Button wird die Bearbeitung beendet und der geänderte Text wird übernommen. 

Des Weiteren sind diesen Buttons Tooltips und Shortcuts hinterlegt: 

- Strg + Umschalt + B/F - Fettdruck - Strg + Umschalt + I/K - Kursiv - Strg + Umschalt + U - Unterstreichung - Strg + R - Rechtsbündig - Strg + L - Linksbündig - Strg + M - Zentriert - Strg + B - Blocksatz - Strg + E - Enter - Strg + T - Tabulator - Strg + N - Leerzeichen 
Schriftart, Zeichengröße, Farbe, Fettdruck etc. können auch vor Beginn des Schreibens aktiviert werden. Dazu setzen Sie den Cursor an die gewünschte Stelle, drücken den entsprechenden Button, z.B. für Fettdruck und geben den Text ein. Dieser wird solange in Fettdruck geschrieben, bis dieser Button erneut gedrückt wird. 

Haben Sie den Text wie gewünscht überarbeitet bzw. formatiert, kann der Text mit Klick auf den Button übernommen werden. Wenn Sie den Button klicken, werden Sie gefragt, ob Sie die Änderungen übernehmen möchten. 

Wenn Sie sich auf der Registerkarte Bearbeiten befinden, können Sie auch auf die Registerkarte Einfügen im oberen Bereich wechseln. 

Ein Wechsel zwischen den Registerkarten Format und Einfügen ist auch über folgende Shortcuts möglich: 

- Strg + 1 = Registerkarte Format - Strg + 2 = Registerkarte Einfügen
