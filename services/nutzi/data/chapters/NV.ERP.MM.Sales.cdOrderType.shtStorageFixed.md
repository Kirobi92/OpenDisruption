# Kapitel NV.ERP.MM.Sales.cdOrderType.shtStorageFixed

Eine Auftragsart kann einem Lager zugeordnet werden, d.h. beim Erfassen eines solchen Auftrags wird automatisch das zugeordnete Lager eingestellt. Soll gewährleistet werden, dass eine Auftragsart grundsätzlich nur für ein bestimmtes Lager erlaubt ist, wird dieser Schalter gesetzt. 

Beispiel: Abbuchung vom Konsignationslager. 

Tooltip: 

Wenn aktiviert, ist gewährleistet, dass die Auftragsart grundsätzlich nur für ein bestimmtes Lager erlaubt ist.
