# Artikel Registerkarte Memos 
Artikel Registerkarte Memos 

Auf der Registerkarte Memos können Informationen zum Artikel hinterlegt werden, die in den entsprechenden Dialogen in Info-Feldern angezeigt werden. 

Siehe Kapitel Memo . 
### Feld Memo 
Allgemeines Memo zu Informationszwecken oder zum Andruck über die Reportprogrammierung, sichtbar nur im Artikelstamm. 

Tooltip: 

Enthält interne Textinformationen. 

### Feld Artikelhinweis 
Hier steht Ihnen ein weiteres allgemeines Memo für Einträge, z.B. für die Gefahrgutverwaltung (Zusammensetzung der gefährlichen Inhaltsstoffe) zur Verfügung. Per Reportprogrammierung kann dieser Text auf die Gefahrgutetiketten gedruckt werden. 

Tooltip: 

Angabe eines Artikelhinweises 

### Feld Aufträge 
Dieses Memo wird angezeigt, wenn eine Auftragsposition mit diesem Artikel angelegt wird. 

Tooltip: 

Angabe des Auftrags-Memos 

### Feld Bestellmemo 
Der Text der Rahmenbestellung wird IMMER eingetragen, wenn es sich um eine Bestellung mit Bezug auf eine Rahmenbestellung handelt (primär, kann optional übernommen werden). 

Tooltip: 

Angabe des Bestell-Memos
