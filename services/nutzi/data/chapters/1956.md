# Sollpreis aktualisieren

Produktion (PPS) / Betriebsauftrag 

# Betriebsauftrag Registerkarte Positionen 
Betriebsauftrag Registerkarte Positionen 

Auf dieser Registerkarte werden automatisch alle Komponenten der Stückliste eingelesen. 

Nachträglich eingefügte, neue Materialpositionen in einem Betriebsauftrag können nicht nur am Ende der Tabelle, sondern unterhalb einer markierten Zeile eingefügt werden. D.h. Sie markieren eine Zeile, drücken den Button Neu und geben die neue Position ein. 

Damit dann noch dargestellt werden kann, was zusammengehört, steht die Spalte ‚BA-Pos‘ zur Verfügung. Hier kann eine Zwischennummer gefunden werden. eNVenta füllt diese Spalte mit einem Vielfachen von 10 (10, 20, 30, …). 

Die Position kann auch nur eingefügt werden, wenn dies möglich ist; sonst am Ende der Tabelle. 

Beispiel: 
Eine Position hat die Nummer 10, die nächste die Nummer 20. Sie möchten eine neue Position zwischen Position 10 und 20 einfügen. In dem Fall vergibt eNVenta die Nummer 15. 

Anmerkung: Nicht nur ‚10er‘-Positionsschritte sind möglich, sondern auch Positionen mit ‚Einer‘-Schritten können eingefügt werden – je nachdem wie die Produktionsstückliste ursprünglich definiert wurde. 

Beispiel: 
Eine Position hat die Nummer 1, die nächste die Nummer 5. Sie möchten eine neue Position zwischen Position 1 und 5 einfügen. In dem Fall vergibt eNVenta die Nummer 3. 
### Button Neu 

Dieser Button ermöglicht direkt die Eingabe einer Position. Dabei stehen dieselben Funktionen (SML-Recherche, …) zur Verfügung wie aus dem Auftrag etc. bekannt. 

Beim Hinzufügen eines neuen Artikels in einen Betriebsauftrag wird die Menge aus dem Artikelauswahl-Dialog nicht übernommen.

### Button Charge 

Dieser Button öffnet den Dialog Chargenauswahl . Hier können Sie der Position eine Chargennummer zuordnen. 

Wird im Betriebsauftrag eine Chargennummer vergeben, wird diese automatisch auf die zugrundeliegende Auftragsposition übernommen. Eine manuelle Anpassung ist nicht notwendig! 

Siehe Kapitel Chargenauswahl .

### Button Seriennummer 

Mit diesem Button können Sie Seriennummern vergeben. 

Seriennummern können auf den „Kopf“ einer Produktionsstückliste, wie auch auf dessen Endteilen vergeben werden. Bei Zuordnung einer Seriennummer zu einem Endteil erhalten diese Seriennummern den Status Verbaut . 

Siehe Kapitel Seriennummern-Zuordnung .

### Button Akt.Sollpreis 

Mit diesem Button können zu jedem Zeitpunkt vor Abschluss des Betriebsauftrags die Sollpreise der markierten Positionen aktualisiert, d.h. neu ermittelt werden. So kann der Zeitpunkt für das Festschreiben der Sollpreise manuell bestimmt werden. Ist das vor der Freigabe geschehen, so werden die Preise durch die Freigabe nicht mehr überschrieben. 

Tooltip: 

Sollpreise aktualisieren

Die Chargenverwaltung steht nur zur Verfügung, wenn sie lizenziert ist. Dies erkennen Sie, wenn im Dialog Konfiguration die Checkbox aktiv gesetzt ist.

### Tabellenspalte Bereits umgebucht 
Diese Spalte steht zur Verfügung, damit ein Sachbearbeiter schnell erkennen kann, ob bereits Umlagerungen für eine Position innerhalb eines Betriebsauftrags getätigt wurden. 

Diese Spalte beinhaltet die Menge, die bereits auf das Produktionslager umgebucht wurde. 

Ein Doppelklick in diese Spalte öffnet das Lagerjournal.
