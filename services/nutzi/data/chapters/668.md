# EK-Rabattgruppen-Zuordnung 
EK-Rabattgruppen-Zuordnung 

Über Lieferant >> Stamm >> Rabattgruppen ) bzw. den Button Rab. Gruppen im Dialog Rabatt Einkauf gelangen Sie zur EK-Rabattgruppen-Zuordnung . Der Dialog öffnet sich auch in der Erweiterten Konditionsfindung über das Menü Stamm . 

Einkaufs-Rabattgruppen können Zeitraum-gesteuert angelegt werden, damit auch zukünftige Konditionen sauber gepflegt werden können. Ebenso können mehrere Artikel, vollkommen unabhängig voneinander, zu einer EK-Rabattgruppe zusammengefasst werden. 

In diesem Dialog werden den Rabattgruppen eines Lieferanten Artikel, Artikel-Bereiche oder Artikelgruppen zugeordnet. (Dieser Lieferant ist im Artikelstamm als Hauptlieferant geführt.) Dafür steht jeweils eine Registerkarte zur Verfügung. Es ist grundsätzlich eine Mischung aller Optionen für eine EK-Rabattgruppe möglich. 

Eine Rabattgruppe kann auch mehreren Lieferanten zugeordnet werden. 

In den Rabattgruppen können Artikel, Artikelbereiche oder Warengruppen mit identischen Grundrabatten für die Rabattverwaltung im Einkauf zusammengefasst werden. Dies erleichtert die Pflege der Rabattverwaltung. 

Beim Einfügen oder Ändern der Rabattgruppen wird vom System kontrolliert, ob die Artikelnummer überschneidungsfrei festgelegt wurde. 

Die Suche über bereits erfasste Rabattgruppen können Sie pro Lieferant und Artikelnummer starten.

### Combobox Gültig von 
Rabattgruppen eines Lieferanten können über diese Combobox mit einem Gültigkeitsbeginn versehen werden. Wird keine Gültigkeit angegeben, wird die aktuelle Gültigkeit automatisch zur Selektion übernommen. Wird der zum Lieferanten hinterlegte Gültigkeitsbeginn angegeben, werden die passenden Datensätze selektiert und angezeigt. 

Wenn Sie nun zu einem späteren Zeitpunkt die Rabattgruppen ändern und ein neues Gültig-ab-Datum setzen, wird der „alte“ Rabattgruppensatz zu diesem Datum beendet, d.h. die Spalte ‚Gültig bis‘ erhält einen ‚Eintrag‘. Auf diese Weise ist sichergestellt, dass es immer nur eine gültige EK-Rabattgruppen-Kondition geben kann und es nicht zu Überschneidungen kommt.
