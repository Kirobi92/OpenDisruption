# Kapitel NV.ERP.Base.Common.cdParameterFI.shtPaymTransPerUser

Bei Aktivierung dieses Parameters kann der angemeldete Benutzer im Bankeinzug und im Zahlungsverkehr nur seine eigenen Vorschläge bearbeiten. 

Tooltip: 

Zahlungsverkehr / Bankeinzug benutzerspezifisch verarbeiten.
