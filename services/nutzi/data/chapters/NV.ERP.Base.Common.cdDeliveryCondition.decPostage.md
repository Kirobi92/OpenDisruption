# Kapitel NV.ERP.Base.Common.cdDeliveryCondition.decPostage

Hier kann eine Porto- bzw. Frachtpauschale angegeben werden, die automatisch für eine bestimmte Lieferbedingung in das Angebot bzw. den Auftrag übernommen wird. Eine Änderung ist jederzeit möglich. 

Tooltip: 

Geben Sie eine eine Porto- bzw. Frachtpauschale ein.
