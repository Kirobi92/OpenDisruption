# Katalogverwaltung Registerkarte Artikel 
Katalogverwaltung Registerkarte Artikel 

Auf dieser Registerkarte werden den Produkten die zugehörigen Artikel zugewiesen. 

Zur gleichzeitigen Bearbeitung mehrerer Artikel der Tabelle steht eine Mehrfachselektion zur Verfügung. 

Die Aufgaben der Registerkarte Artikel sind: 
- Anzeigen, hinzufügen und löschen von Artikeln in einem Produkt. - Anzeigen von Artikeln in einer Gruppe. Es werden die Artikel aller Produkte der Gruppe angezeigt. - Erzeugen neuer Produkte über die Artikelauswahl. - Sortieren der Artikel im Produkt. 
Ist ein Produkt ausgewählt, werden die ausgewählten Artikel dem Produkt zugeordnet. 

Ist eine Gruppe ausgewählt, wird für jeden ausgewählten Artikel ein neues Produkt in dieser Gruppe und diesem Produkt der jeweilige Artikel zugeordnet. Dabei werden Artikel-Bezeichnung und Artikel-Text in den verfügbaren Sprachen in das Produkt übernommen. 

Ist auf der Registerkarte Artikel ein Produkt ausgewählt, werden die Artikel in der Tabelle in der Reihenfolge angezeigt, in der die Artikel im Produkt hinterlegt sind. 

Um schnell in die Merkmalsverwaltung zu gelangen, stehen die Buttons Artikel und Merkmale bearb. zur Verfügung. 
### Button Artikel 

Mit diesem Button verzweigen Sie in den Artikelstamm. 

### Button Merkmale 

Mit diesem Button öffnet sich der Dialog Merkmale . Dort können Sie die Definition der einzelnen Merkmale anpassen, z.B. Auswahlwerte eintragen. 

Siehe Kapitel Merkmale . 

### Button Merkmale bearb. 

Mit diesem Button werden die Artikel unterhalb des gewählten Produkts an die Merkmalsverwaltung übergeben. Außerdem wird die Merkmalsklasse übergeben, damit die entsprechenden Werte voreingestellt werden können. 

Aus diesem Grund gibt es im Artikelstamm auf der Registerkarte Merkmale die Spalte ‚Wert (ML)‘, um direkt die ML-Werte zu pflegen. 

Die Merkmalsverwaltung lässt sich außerdem über das Tastenkürzel <Strg + S> öffnen. 

Tooltip: 

Merkmale bearbeiten

### Button Nach oben verschieben 

Mit diesem Button kann ein Artikel nach oben verschoben werden. Die Tabelle wird nicht sofort aktualisiert, dazu wählen Sie das Produkt neu aus. 

Wurde für die Artikel noch keine Reihenfolge festgelegt, stimmt die Anzeige der Tabelle nicht immer mit der Darstellung im Baum überein. 

Tooltip: 

Nach oben verschieben 

### Button Nach unten verschieben 

Mit diesem Button kann ein Artikel nach unten verschoben werden. Die Tabelle wird nicht sofort aktualisiert, dazu wählen Sie das Produkt neu aus. 

Wurde für die Artikel noch keine Reihenfolge festgelegt, stimmt die Anzeige der Tabelle nicht immer mit der Darstellung im Baum überein. 

Tooltip: 

Nach unten verschieben 

### Button Satz hinzufügen 

Mit diesem Button starten Sie die Suche nach dem Artikel, den Sie im gleichnamigen Feld eingetragen haben. 

Tooltip: 

Neuer Artikel 

### Button Löschen 

Mit diesem Button löschen Sie einen oder mehrere Artikel, die Sie in der Tabelle markiert haben. 

Der Button funktioniert nur, wenn ein Produkt ausgewählt ist und nicht, wenn eine Gruppe ausgewählt ist. 

Tooltip: 

Artikel löschen
