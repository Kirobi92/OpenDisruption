# Verbuchen der Inventurposition 
Mit diesem Dialog wird die Inventurposition verbucht. 

Wurde im Rahmen der Filterung der Platz bereits eingescannt, so gilt dieser als gescannt und muss hier nicht nochmal bestätigt werden. 

Ist das nicht der Fall, so ist die erste Eingabe, die erwartetet wird, der Lagerplatz der gezählt wird. 

Anschließend kann durch die Mengeneingabe die gezählte Menge eingetragen werden. 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Barcode 

Wird dieser Button gedrückt, so wird, wenn eine Kamera am Gerät vorhanden ist, die Barcodeerkennung aktiviert. 

Dabei werden folgende Barcodetypen erkannt: Code128, EAN8, EAN13. 

### Button Einstellungen 

Dieser Button öffnet das Einstellungsmenü für den Scanner. 

### Button Neuer Satz 

Über diesen Button können Sie neue Inventurpositionen erfassen. 
Der Button steht zur Verfügung, wenn die Inventurposition einen leeren Platz vorschlägt. 
Dies soll die Bearbeitung eines leeren Platzes beschleunigen. 
Nach erfolgter Anlage einer neuen Position wird der leereintrag gelöscht und in die Auswahl zurückgekehrt. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern!
