# BMECat - Import Registerkarte Einheiten 
BMECat Import Registerkarte Klass. System / Einheiten 

Hier werden die Einheiten im Klassifikationssystem angegeben. 
### Tabelle 

### Tabellenspalte ID 
Hier wird der eindeutige Identifikator der Maßeinheit innerhalb des Klassifikationssystem angegeben. 

Tooltip: 

Gibt den eindeutigen Bezeichner der Maßeinheit innerhalb des Klassifikationssystems an. 

### Tabellenspalte Name 
Hier wird der eindeutige Namen der Maßeinheit innerhalb des Klassifikationssystems angegeben. 

Tooltip: 

Gibt den eindeutigen Namen der Maßeinheit innerhalb des Klassifikationssystems an. 

### Tabellenspalte Beschreibung 
Hier wird eine Beschreibung angegeben.
