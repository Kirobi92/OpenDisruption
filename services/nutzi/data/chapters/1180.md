# Projekterfassung im Auftrag 
Projekterfassung 

Im Auftrag unter dem Menü Optionen >> Generieren gibt es das Menü Projekterfassung . Die Funktion <Strg> + <J> öffnet ebenfalls die Projekterfassung . So können Sie einfach und schnell aus dem Auftrag heraus ein Projekt erfassen. Geben Sie hierzu nur die wichtigsten Daten ein und klicken auf den Button , um das Projekt zu speichern. 

Falls bereits ein Projekt hinterlegt ist, werden die Informationen eingespielt und können geändert werden. 

Um den Dialog aufrufen zu können, muss mindestens die Kundennummer im Auftrag eingetragen sein. 

Wenn Sie die Projektverwaltung öffnen, können Sie auch hier sofort eine Projektnummer eingeben und bekommen die entsprechenden Daten angezeigt. 

Ist dem Auftrag bereits ein Projekt hinterlegt und Sie öffnen das Menü Projekterfassung , sind dessen Daten bereits eingelesen und können bearbeitet werden.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern!

### Feld GEO-Koordinaten 
Hier werden geografische Daten zur besseren Orientierung angegeben. Damit lassen sich Besuche o.ä. besser planen und Sie können die Karte zur besseren Orientierung und Festlegung einer Besuchs-Reihenfolge nutzen. 

Das Format der GEO-Koordinaten lautet: 
XX.yyyyyy,AA.bbbbbb 

Beispiel für Stockach, Nissen&Velten: 
47.848029,09.005334 

Tooltip:

### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen.
