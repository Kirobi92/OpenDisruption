# Übersicht Betriebsaufträge Registerkarte Allgemein 
Übersicht Betriebsaufträge Registerkarte Allgemein 

Die Tabelle auf dieser Registerkarte enthält Daten entsprechend der Selektionskriterien. 
### Tabelle Betriebsaufträge 

Die Tabelle listet die Suchergebnisse anhand der Suchkriterien auf. Die Werte innerhalb der Tabelle sind nicht änderbar. Mit einem Doppelklick auf einen Betriebsauftrag öffnet sich dieser. Eine Sortierung nach Spaltenköpfen ist möglich. Ebenso ein Export ins Excel. 
### Tabellenspalte Betriebsauftrag 
Hier wird die Nummer des Betriebsauftrags angezeigt. 

Tooltip: 

Angabe der Betriebsbauftragsnr. 

### Tabellenspalte Status 
Hier wird der Status des Betriebsauftrags angezeigt. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Lieferant 
Die Lieferantennummer aus dem Betriebsauftrag wird angezeigt. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Firma 1 
Die Firmenadresse wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Artikel 
Die Nummer des produzierten Artikels wird angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Der Name des Artikels wird angezeigt. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Beschreibung 
Die Artikelbeschreibung aus dem Artikelstamm wird angezeigt. 

Tooltip: 

Angabe einer Beschreibung 

### Tabellenspalte Menge bestellt 
Die bestellte Menge wird angezeigt. 

Tooltip: 

Angabe der Menge, welche bestellt wurde. 

### Tabellenspalte Startdatum 
Das Startdatum der Produktion wird angezeigt. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Tabellenspalte Datum bis 
Hier steht das Datum, an dem der Betriebsauftrag beendet wurde. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Tabellenspalte Lieferdatum 
Hier wird das Lieferdatum angezeigt. 

Tooltip: 

Angabe des Liefertermins 

### Tabellenspalte Gesperrt 
Wenn ein Betriebsauftrag gesperrt ist, erkennen Sie dies an der gesetzten Checkbox in dieser Spalte. 

Tooltip: 

Aktiviert: Setzt den Datensatz auf 'gesperrt'.
