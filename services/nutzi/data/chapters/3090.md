# Importkonfiguration 
Importkonfiguration 

Dieser Dialog öffnet sich im Artikel Import Manager über den Button Formate und dient dazu die verschiedenen Datenformate, welche importiert werden sollen, zu verwalten. Hierfür stehen je nach File-Adapter verschiedene Optionen zur Auswahl. 

Die Bezeichnung, der Datenbereich und die Beschreibung können frei eingegeben werden. Die Formatnummer wird von eNVenta vergeben. Wird kein Datenbereich vorgegeben, wird automatisch die Formatnummer als Datenbereich in die Datenbank geschrieben (Feld SB_Scope ). Dies wird im Dialog nicht visualisiert. 

In Bezug auf die komplexere Schnittstelle wird ein weiterer X-Mapper zur Verfügung gestellt, der nicht nur CSV-Dateien über den Format-Dialog umsetzt, sondern auch weitere Datei-Formate (XML, ASCII) unterstützt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Importkonfiguration 
Tooltip: 

In diesem Dialog können Sie die verschiedenen Datenformate verwalten, welche importiert werden sollen.

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Feld Import Format 
Hier wählen Sie den entsprechenden Adapter aus. Zur Verfügung stehen: Datenimport, Datanorm und Datenbank . 

Wenn Datanorm ausgewählt ist, werden die 3 Checkboxen Einfügen, Ändern und Löschen angezeigt. Über diese Option entscheiden Sie, ob neben dem Einfügen auch Änderungen und Löschungen über den Datenimport erfolgen sollen. 

Tooltip: 

Angabe des Importformats 

### Feld Bezeichnung 
Hier wird die Bezeichnung des Importformats angegeben. 

Tooltip: 

Bezeichnung des Importformats 

### Feld Datenbereich 
Hier kann ein Datenbereich von bis zu 20 Stellen definiert werden. Damit können verschiedene Importe noch besser voneinander abgegrenzt werden. 

Tooltip: 

Die importierten Daten gehören zu einem Datenbereich. Es werden nur Daten desselben Bereichs miteinander aktualisiert. 

Es wird empfohlen, mindestens einen Datenbereich je Datenlieferant zu nutzen. Wird kein Datenbereich definiert, setzt das System automatisch die Importformat-ID als Bereich. 

### Feld Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung
