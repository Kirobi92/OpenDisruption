# Kapitel NV.ERP.Base.Common.cdSystemMM.shtBacklogItemStatus2

Wenn Sie diesen Parameter aktivieren, wird die erstellte Restposition automatisch in den Status 2 (Lieferfreigabe) gesetzt (sonst Status 1). Hierbei erfolgt eine Verfügbarkeitsprüfung, damit auch nur wirklich verfügbare Ware in Status 2 gesetzt wird.
