# Chargen Registerkarte Tabelle 
Charge Registerkarte Tabelle 

Auf dieser Registerkarte werden alle Chargen angezeigt, die angelegt sind und zuvor über eine Suche eingelesen wurden. 
### Tabelle 

### Tabellenspalte Charge 
Hier wird die Chargennummer angezeigt. 

Tooltip: 

Angabe der Chargennummer 

### Tabellenspalte Ursprungscharge 
Hier wird die Ursprungscharge angezeigt. 

Tooltip: 

Angabe der Ursprungscharge 

### Tabellenspalte Pseudo 
Hier wird die Pseudo-Charge angezeigt. 

Mehr zum Thema Pseudo-Charge erfahren Sie im Kapitel Pseudo-Charge . 

Tooltip: 

Diese Chargennr. ist keine richtige Charge. Sie dient als Platzhalter bis man die korrekten Chargeninformationen vom Liefranten erhalten hat 

### Tabellenspalte Status 
Hier wird der Status angezeigt. 

### Tabellenspalte Verfallsdatum 
Hier wird das Verfallsdatum der Charge angezeigt. 

Tooltip: 

Angabe des Verfallsdatums 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung des Artikels angezeigt. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Firma 1 
Die Firmenbezeichnung wird automatisch vom System eingetragen, nachdem die Lieferantennummer eingetragen wurde. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Lieferant 
Die Lieferantennummer wird angezeigt. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Prüfzeugnis 
Hier wird die Prüfzeugnisnummer angegeben. 

Tooltip: 

Angabe der Prüfzeugnisnummer 

### Tabellenspalte Zeugnisversion 
Hier wird die Zeugnisversion angegeben. 

### Tabellenspalte Lager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
In diesem Feld wird die Anzahl der tatsächlich eingelesenen Datensätze angezeigt.
