# Importkonfiguration Registerkarte Allgemein 
Importkonfiguration Registerkarte Allgemein 

Diese Registerkarte gibt die Auswahl der verfügbaren Importtypen Datenimport (Standard), Datenbank oder Datanorm an. 

Die Option Datenbank kann nur mit einer entsprechenden Programmierung genutzt werden. 

Die Funktion Datanorm wird nicht empfohlen. Es ist besser, wenn Sie den Importtypen Datenimport nutzen und das entsprechende Format hinterlegen. 
### Combobox Import Typ 
Hier wählen Sie den entsprechenden Adapter aus. Zur Verfügung stehen: Datenimport und Datanorm. 

Wenn Datanorm ausgewählt ist, werden die 3 Checkboxen Einfügen, Ändern und Löschen angezeigt. Über diese Option entscheiden Sie, ob neben dem Einfügen auch Änderungen und Löschungen über den Datenimport erfolgen sollen. 

### Feld Letzter Lauf 
Hier wird das Datum des letzten Laufs eingetragen. 

### Feld Transferdatum 
Hier wird das Datum des letzten Transfers eingetragen. 

Tooltip: 

Angabe des Transferdatums 

### Feld Import-Version 
Hier wird die letzte Version des Imports eingetragen. Diese Nummer wird von eNVenta automatisch generiert. 

### Checkbox Einfügen 
Wenn Sie diese Checkbox aktivieren, werden Artikel importiert, die noch nicht im Schattenstamm enthalten sind. 

### Checkbox Ändern 
Wenn Sie diese Checkbox aktivieren, werden Artikel, die bereits im Schattenstamm existieren, aktualisiert. 

### Checkbox Löschen 
Wenn das Lösch-Kennzeichen im Format gesetzt ist, wird dieses ausgewertet und entsprechend markierte Artikel aus dem Schattenstamm gelöscht.
