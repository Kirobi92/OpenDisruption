# Servicevertrag Registerkarte Text 
Servicevertrag Registerkarte Text 

Auf dieser Registerkarte haben Sie die Möglichkeit verschiedene Memos zu diesem Vertrag zu hinterlegen. 
### Feld Vertragsmemo 
Hier wird ein individueller Text zum Vertrag zur internen Information angegeben. 

Tooltip: 

Enthält interne Textinformationen zum Vertrag. 

### Feld Servicememo 
Hier wird ein individueller Text zur internen Information zum Service angegeben. 

Tooltip: 

Enthält interne Textinformationen zum Service. 

### Feld Rechnungsmemo 
Hier wird ein individueller Text zur Rechnung zur internen Information angegeben. 

Tooltip: 

Enthält interne Textinformationen zur Rechnung.
