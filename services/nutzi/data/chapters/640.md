# Versandarten 
Versandart 

Unter dem Menü Stamm >> Versandarten finden Sie den Dialog zum Anlegen und Verwalten von Versandarten für Kunden. Die Verwaltung der Lieferbedingungen ist aus vielen Dialogen heraus aufrufbar. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Versandart 
Tooltip: 

In diesem Dialog verwalten Sie Grunddaten für den Versand der Waren an den Kunden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Tabelle Versandarten 

### Tabellenspalte Versandart 
Jeder Versandart wird eine Nummer zugeordnet, mit der sie eindeutig identifiziert werden kann. Bitte beachten Sie, dass jede Nummer nur einmal vorkommen darf. 

Aufgrund der verschiedenen Kommunikationspartner (z.B. nexMart, Nordwest, E/D/E) und abgestimmter Kennungen ist diese Spalte (für einen neu anzulegenden Datensatz) editierbar. Aufgrund dessen wird der Fokus auch auf diese Spalte gelegt und es ist dafür gesorgt, dass eine Versandart nicht doppelt belegt wird. 

Tooltip: 

Eindeutige Nummer der Versandart, die in der Datenbank gespeichert wird. 

### Tabellenspalte Versandartbezeichnung 
Hier geben Sie eine möglichst sprechende Beschreibung der Versandart ein, um diese schnell wieder erkennen zu können. Diese Versandarten werden in den Vorgängen zur Auswahl zur Verfügung gestellt. 

Tooltip: 

Geben Sie eine Beschreibung der Versandart ein. 

### Tabellenspalte Transportweg 
Hier können Sie einen Transportweg hinterlegen. Dies dient der Gegenprüfung, ob eine Versandart für eine Tour mit einem vorbelegten Transportweg gekoppelt werden kann. 

Tooltip: 

Transportweg - mögliche Werte 
leer = Automatisch 
0 = Fuhrpark 
1 = Paketdienst (Felder 'Versender' und 'Versand Code' müssen gefüllt sein) 
2 = Overnight Dienst (Feld 'Versender' und 'Versand Code' müssen gefüllt sein) 
3 = Spedition 

### Tabellenspalte EDI Kennung 
Wenn diese Spalte einen Wert enthält, wird dieser beim ORDERS Export im HD11 in das Feld Transportart geschrieben. 

Nur wenn die Gateway-Versandartkennung vorhanden ist, wird die Versandartnummer überschrieben. 

Tooltip: 

Gateway Versandkennung 

### Tabellenspalte Versender 
Hier kann der zuständige Versender eingestellt werden. Die Versender pflegen Sie im Dialog Picklisten . 

Beim Ändern des Versenders werden automatisch alle Versandcodes zur Versandart gelöscht. 

Tooltip: 

Geben Sie den zuständigen Versender ein. 

### Tabellenspalte Übergabe an V-LOG 
Mit der Checkbox in dieser Spalte geben Sie an, ob in der Kommissionierung im Versand eine automatische Übergabe des Lieferscheins an die Paketversender (MHP V-Log) durchgeführt werden soll. 

Tooltip: 

Wenn aktiviert, wird eine automatische Übergabe des Lieferscheins an die Paketversender (MHP V-Log) durchgeführt. 

### Tabellenspalte Übergabe an PTV 
Wenn Sie diese Checkbox aktivieren, wird die Versandart an die PTV-Schnittstelle angebunden. 

Tooltip: 

Wenn aktiviert, wird die Versandart an die PTV-Schnittstelle angebunden. 

### Tabellenspalte Versandlager 
Aus der Combobox in dieser Spalte wählen Sie ein Versandlager aus. 

Tooltip: 

Definiert das Versandlager 

### Tabellenspalte Keine Tourenkommissionierung 
Wenn Sie die Checkbox in dieser Spalte aktivieren, wird für die Tour nicht kommissioniert. 

Tooltip: 

Wenn aktiviert, wird für die Tour nicht kommissioniert. 

### Tabellenspalte Keine WBZ aktualisieren 
Wenn die Checkbox in dieser Spalte gesetzt ist, wird die Wiederbeschaffungszeit nicht aktualisiert. Die entsprechende Checkbox im Bestellkopf wird vorbelegt. Der Sachbearbeiter hat die Möglichkeit, die Einstellung bezüglich WBZ zu ändern. 

Siehe Kapitel Wiederbeschaffungszeit . 

Tooltip: 

Wenn gesetzt, wird die Wiederbeschaffungszeit nicht aktualisiert. 

### Tabellenspalte Verordnung 
Hier wird die zugehörige Gefahrgutverordnung eingetragen. 

Tooltip: 

Angabe der zugehörigen Gefahrgutverordnung 

### Tabellenspalte Direktentnahme 
Wenn Sie diese Checkbox aktivieren, kann bei dieser Versandart die Ware direkt vom Lager entnommen werden. 

Tooltip: 

Wenn aktiviert, wird die Ware direkt vom Lager entnommen. 

### Tabellenspalte Kein LS-Druck bei Scan 
Wenn die Checkbox in dieser Spalte aktiviert ist, wird bei dieser Versandart kein Lieferschein gedruckt, da die Daten gescannt werden. 

Tooltip: 

Wenn aktiviert, wird bei dieser Versandart kein Lieferschein gedruckt, da die Daten gescannt werden. 

### Tabellenspalte Lieferant 
Die Nummer des Lieferanten, durch den die Lieferung erfolgt, wird hinterlegt. 

Tooltip: 

Eingabe der Lieferantennummer 

### Tabellenspalte Name 
Nach Eingabe der Lieferantennummer wird Firma1 automatisch aus dem Lieferantenstamm gezogen. 

### Tabellenspalte Wiedervorlage nach Pick (Anz. Tage) 
In dieser Spalte geben Sie an, nach wie vielen Tagen (nach der Kommissionierung) eine Wiedervorlage an den Erfasser des Auftrags erstellt werden soll. 

Die Wiedervorlage wird nur dann erstellt, wenn in dieser Zeit kein LS-Druck für den Auftrag stattfindet. (Ziel: Abholungen besser kontrollieren zu können) 

Tooltip: 

Gibt an, nach wie vielen Tagen eine Wiedervorlage an den Erfasser des Auftrags erstellt werden soll. Die Wiedervorlage wird nur dann erstellt, wenn in dieser Zeit kein LS-Druck für den Auftrag stattfindet. (Ziel: Abholungen besser kontrollieren zu können) 

### Tabellenspalte Sperrgut 
Wenn Sie diese Checkbox aktivieren, dürfen mit dieser Versandart Sperrgut-Artikel versendet werden. 

Sollte in einem Auftrag ein Sperrgutartikel vorhanden sein und gleichzeitig ist eine Versandart ausgewählt in der diese Checkbox nicht gesetzt ist, können Sie diesen Auftrag nicht speichern / abschließen. 

Diese Kennung wird auch an die Schnittstellen übergeben. Diese Information zieht u.U. eine andere Handhabung des Versendens mit sich. 

Tooltip: 

Ware wird als Sperrgut gekennzeichnet. 

### Tabellenspalte Kein Bezugslager 
Wenn Sie die Checkbox in dieser Spalte aktivieren, findet keine Umlagerung vom Bezugs- auf das Versandlager statt. 

Tooltip: 

Es erfolgt kein automatischer Vorschlag für eine interne Umlagerung vom Bezugslager, 
falls das Versandlager nicht lieferfähig ist. 

### Tabellenspalte Keine Behälterzuordnung 
Wenn die Checkbox in dieser Spalte gesetzt wird, ist die Versandart nicht auf einen bestimmten Behältertyp festgelegt. 

Tooltip: 

Wenn aktiviert, ist die Versandart nicht auf einen bestimmten Behältertyp festgelegt. 

### Tabellenspalte Kommiss. Etikettendrucker 
Aus der Combobox in dieser Spalte wählen Sie einen Etikettendrucker aus. 

Tooltip: 

Wählen Sie den Etikettendrucker aus. 

### Tabellenspalte Ecomm 
Versandarten mit dieser Kennung stehen im eNVenta ERP-Webshop für den Kunden zur Auswahl zur Verfügung. 

Tooltip: 

Angabe, ob die Versandart im ECommerce zur Verfügung stehen darf. 

### Tabellenspalte Keine Tourendispo. 
Aufträge mit dieser Versandart bzw. dieser Kennung werden bei der Tourenplanung nicht berücksichtigt. 

Tooltip: 

Wenn aktiviert, werden Vorgänge mit dieser Versandart 
bzw. dieser Kennung bei der Tourenplanung nicht berücksichtigt. 
Die Wiederbeschaffungszeit wird nicht berücksichtigt. 

### Tabellenspalte Paketverfolgung URL 
Hier können Sie für die Nachverfolgung des Paketwegs eine URL bzw. eine Tracking-Nummer eintragen. 

Tooltip: 

Geben Sie die URL mit Platzhalter {0} für die TrackingID an. 

### Tabellenspalte Thekenkommissionierung 
Wenn die Checkbox in dieser Spalte gesetzt wird, kann diese Versandart auch an der Theke kommissioniert werden. 

Tooltip: 

Kennzeichen, ob diese Versandart für die LVS Theke verwendet werden kann. 

### Tabellenspalte LS Drucker 
Aus der Combobox in dieser Spalte wählen Sie einen Lieferscheindrucker aus. 

Tooltip: 

Lieferscheindrucker 

### Tabellenspalte Express 
Wenn Sie die Checkbox in der Spalte'Express' setzen, wird die Bestellung mit Expressversand gekennzeichnet. Anschließend weisen Sie im Dialog Bestellung auf der Registerkarte Kopfdaten die gewünschte Versandart aus. 

Es ist notwendig, im X-Mapping Editor die entsprechende Struktur für openTrans® 2.1 anzugeben. Die Checkbox Express steht im X-Mapping Editor zur Verfügung und ist über den Ausdruckseditor auswertbar.
