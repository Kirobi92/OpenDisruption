# Kapitel NV.ERP.Base.Common.cdSystemBase.shtPrintJobTimeoutMinutes

Mit einem Eintrag in dieses Feld wird festgelegt, nach wie vielen Minuten ein offener Druck-Job als überfällig betrachtet werden soll. Wenn der Timeout erreicht ist, wird dieser in der Liste der Druck-Jobs als Warnung kenntlich gemacht. Wenn in diesem Feld kein Wert definiert ist, wird ein Druck-Job nie mit einem Timeout versehen. 

Tooltip: 

Timeout für Druck-Aufträge in Minuten 
Nach Ablauf dieser Zeit werden sie als fehlerhaft angezeigt.
