# Parameter Projekt 
Parameter Projekt 

Im Navigator unter System >> Projekt steht Ihnen der Eintrag Parameter Projekt zur Verfügung. Zu einer Ressourcengruppe für Mitarbeiter (Combobox Ress. Gr. MA ) bzw. Fahrzeuge (Combobox Ress. Gr. Fahrz. ) können hier Spesen- und Kostenartikel definiert werden. 

Falls eine Ressourcengruppe für die Mitarbeiter definiert wurde, werden nur die Ressourcen dieser Gruppe als Leistungsressource in den Terminen angeboten. Falls hier keine Gruppe angegeben wurde, werden alle Ressourcen präsentiert. 

Die Ressourcengruppe für die Fahrzeuge bestimmt, welche Ressourcen als Fahrzeugressourcen im Termin zur Verfügung stehen. Wenn hier nichts angegeben wurde, können keine Fahrzeugressourcen gewählt werden. Die Spesen- und Kostenartikel in den Projektparametern wiederum bestimmen, welche Artikel im Termin gewählt werden können.
