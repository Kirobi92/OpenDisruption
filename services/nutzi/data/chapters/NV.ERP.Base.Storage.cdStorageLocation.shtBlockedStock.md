# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtBlockedStock

Dieser Parameter sperrt das Lager für den Verkauf und kann nur manuell bebucht werden. Das Sperrlager wird dort geprüft, wo die Ware das Unternehmen verlässt bzw. von extern beliefert wird. 

Sinnvoll ist, diese Option beispielsweise bei Qualitätssicherungslagern, Produktionslagern oder Ausschusslagern. Sperrlager sind in der Bestandsanzeige sichtbar, die Mengen werden jedoch nicht in die Verfügbarkeit eingerechnet. Manuelle Lagerbuchungen sind, wie bereits erwähnt, jederzeit möglich. 

Tooltip: 

Wenn aktiviert, ist das Lager für den Verkauf gesperrt.
