# Theken Zuweisung Scanner 
Zuweisung Scanner 

Dieser Dialog öffnet sich über den Button Z. Scanner im Dialog Thekenerfassung , wenn eine Zeile der Tabelle markiert ist. 

In der Zuteilungsübersicht werden alle Picks aufgelistet, welche entweder noch nicht freigegeben sind bzw. welche auf dem Pickmonitor sind. So hat der Sachbearbeiter die Möglichkeit Picks, welche zur Kommissionierung freigegeben aber nicht kommissioniert wurden, neu zu vergeben und direkt einem Scanner mit dem Button Scanner zu. zuzuordnen. 

Wird ein Pick direkt einem Scanner zugeordnet, so hat der Sachbearbeiter die Möglichkeit diesen abzulehnen oder anzunehmen. Lehnt er ihn ab, geht der Pick auf den Pickmonitor. 

Im Dialog Lagerorte kann auf der Registerkarte LVS Allgemein der Zeitraum Entbindung Express eingestellt werden. Reagiert ein Mitarbeiter innerhalb dieses Zeitraums nicht, wird der Pick auf den Pickmonitor gegeben.

### Checkbox Freestyle möglich 
Freestyle bedeutet: Ein Kunde kommt an den Tresen, um Ware zu kaufen. Der Sachbearbeiter am Tresen nimmt die wichtigsten Daten des Kunden auf (wie z.B. Kundennummer und Ansprechpartner). Anschließend geht es mit einem handgeschriebenen Zettel ins Lager und die gewünschten Waren werden "on the fly" gepickt (ohne Vorerfassung). Das System erstellt automatisch die Positionen im Auftrag und bucht die Kommissionierung. 

Wenn Sie diese Checkbox aktivieren, kann der Kommissionierer auch Artikel picken, die vorher nicht an der Thekenerfassung aufgenommen wurden. 

Hat der Sachbearbeiter beim Picken die Freestyle Berechtigung, so kann er vor dem Anscannen eines Lagerplatzes oder nach Beenden aller Picks einen Freestylepick durchführen. Dazu wird mit der F3 -Taste auf den Dialog Freestyle umgeschaltet. 

Folgender Ablauf wird dann abgearbeitet: 
- Platz anscannen - LHM anscannen (wenn von einem LHM was gepickt werden soll) - Artikel auswählen (wenn mehr als einer vorhanden) - Charge wählen (wenn mehr als eine vorhanden ist - Chargenartikel natürlich vorausgesetzt) - Menge auswählen - Kommissionierbehälter auswählen 
Anschließend wird direkt in die Ausgangssituation zurückgesprungen. 

Durch nochmaliges Anscannen eines Freestyle-Pick-Lagerplatzes kann die Entnahme zum Teil oder komplett zurückgelagert oder erhöht werden.
