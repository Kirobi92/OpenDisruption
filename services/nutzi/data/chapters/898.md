# Tourendisposition mit Gefahrgut 
Bei der Verwendung der Tourendisposition bzw. bei der Tourenplanung erfolgt pro Auftrag eine Prüfung gegen die 1000-Punkte-Regelung. Sobald eine Tour mehr als 1000 Punkte hat, kann der Tour kein weiteres Gefahrgut zugeladen werden. Die einzige Ausnahme ist, wenn die Tour bzw. der Fahrer eine ADR-Freigabe haben. 

Die automatische Tourenplanung berücksichtigt neben Aufträgen auch Bestellungen und Retouren, welche zur Abholung markiert sind. Diese werden mit der berechneten Punktezahl einbezogen. Dies erfolgt auch, wenn zwischenzeitlich während der Fahrt die Anzahl der Punkte unter 1000 fallen könnte. Davon wird prinzipiell aus Vorsicht ausgegangen, damit für die Tour nicht 1000 Punkte überschritten werden, falls z.B. ein Kunde eines früheren Stopps die Ware nicht entgegennehmen kann. 

Bei der automatischen Tourenplanung erfolgt die Einplanung einer Auftragsposition (analog Bestellung und Retoure) nur, wenn noch genügend „freie Punkte“ auf der Tour vorhanden sind. Sollten keine Punkte mehr frei sein, so wird automatisch die nächste verfügbare Tour gewählt. 

Achtung: eNVenta berücksichtigt keine Tunnelbeförderung!!! 

Bei der manuellen Tourenzuordnung erscheint eine Warnmeldung, falls der Fuhrparkleiter die Einplanung eines Vorgangs auf eine Tour manuell durchführt. 

Pro Tour werden die Gesamtpunkte der zugeordneten Belege angezeigt: 

ADR Informationen zur Tour 

Mittels des Buttons A neben der Tour bzw. dem Stopp können die ADR Info zum Vorgang angezeigt werden. 

Bei der Zuordnung zur Tour wird zusätzlich geprüft, ob das zulässige Höchstgewicht pro Tour gemäß ADR (8000 Kg) überschritten wird, ansonsten erfolgt keine automatische Einplanung. Ebenso erfolgt eine Prüfung gegen die maximale Anzahl der EQ-Packstücke (1000) bei der Tourenplanung. Beide Einstellungen sind im Dialog Parameter ADR hinterlegt und können ggf. angepasst werden.
