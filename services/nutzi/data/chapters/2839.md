# Empfehlungsagent 
Empfehlungsagent 

Dieser Dialog öffnet sich im Dialog Monitor Empfehlungen , wenn Sie einen Agent markiert haben und den Button neben der Tabelle Agent drücken. Hier legen Sie Empfehlungsagenten fest. 

Nach der erfolgreichen Generierung der Tupel können so diese Daten mittels des Empfehlungsagenten analysiert werden, damit der Sachbearbeiter diese Daten im täglichen Geschäft nutzen kann. 

Siehe Kapitel Monitor Empfehlungen .

### Combobox Artikelgrp-Modus 
Hier kann eingeschränkt werden, dass die Empfehlungen nur aus der gleichen Artikelgruppe, wie der Bezugsartikel kommen sollen, oder nur aus anderen Gruppen. 

Tooltip: 

Einschränkung der Vorschläge auf Artikel anderer oder der gleichen Artikelgruppe des Bezugsartikels 

### Combobox Präferenz 
Die historische Tupel-Häufigkeit ist das übliche Kriterium. Es kann aber alternativ auch die Marge mit einbezogen werden. Dann werden die Artikel mit den höchsten Margen gewählt, die die Mindesthäufigkeit erfüllen. 

Tooltip: 

Kriterium für die Auswahl der Vorschläge 

### Feld Mindesthäufigkeit 
Die Häufigkeit ist die gewichtete Häufigkeit der Tupel mit dem Bezugsprodukt. Je aktueller das Tupel, desto höher die Gewichtung. Mit diesem Parameter kann die Relevanz der Empfehlungen gesteuert werden. Zu hohe Werte führen zu wenigen Empfehlungen. Zu niedrige Werte führen zu vielen, aber weniger relevanten Empfehlungen. In Verbindung mit der Präferenz Marge sollte der Wert tendenziell etwas höher angegeben werden. 

Tooltip: 

Gewichtete statistische Mindesthäufigkeit des historischen gemeinsamen Bedarfs der Artikel. Dies hat direkten Einfluss auf Qualität und Anzahl möglicher Vorschläge. 

### Checkbox Rückfall erlaubt 
Beim Abruf von Vorschlägen wird das Vorschlagslevel (1 = Kunde, 2 = Cluster, 3 = Total) angegeben. Sollte es z.B. auf dem Level Cluster keine ausreichenden Vorschläge geben, kann per Rückfall auf das nächst niedrigere Level Total zurückgegriffen werden. 

Tooltip: 

Rückfall auf Total-Ebene, wenn nicht genügend spezifische Vorschläge vorhanden sind. 

### Checkbox Kein Alternativartikel 
Artikel, die als Alternative im Bezugsartikel oder in der gleichen Alternativgruppe eingetragen sind, werden von den Vorschlägen ausgeschlossen.
