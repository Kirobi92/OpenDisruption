# Generieren der Intrastat Monatsmeldung 
Intrastat generieren 

Im Dialog Intrastat erzeugt der Button Generieren die entsprechende Monatsmeldung. 

Nach Auswahl des Monats, Jahres und wenn nötig der entsprechenden Gesellschaft, wird die Selektion erzeugt. Mit dem Button Übernahme ( ) kann der Bezugsmonat in den Dialog Intrastat übernommen werden. 

Es kann nunmehr analysiert werden nach Einkauf und Verkauf (Radiobuttons Eingang , Versendung bzw. Alle ). Ebenso werden die vollständigen und unvollständigen Vorgänge (Radiobuttons Alle , Vollständige und Unvollständige ) gelistet. 

Bei der Ermittlung von Exportvorgängen werden auch Aufträge berücksichtigt, welche nur über eine gedruckte Rechnung verfügen und keinen Lieferschein besitzen. Im Normalfall erfolgt die Ermittlung auf Basis des Lieferscheins. Sollte aber keiner vorhanden sein, aber eine Rechnung, so wird anhand der Rechnung der Vorgang in die Intrastat-Statistik übernommen. 

Nach der Prüfung (Button Prüfen im Dialog Intrastat ) erscheint die entsprechende Meldung. 

Ist alles in Ordnung, wird über den Button Exportieren die XML-Datei in den vordefinierten Pfad abgelegt. 

Ist die Meldung nicht ok, ist es notwendig nachzuarbeiten. 

### Mehrländerfähigkeit 
Bei der Generierung einer Intrastat sind zwingend der Bezugsmonat- und jahr sowie das Land anzugeben. Sollte die Währung des angegebenen Landes abweichend sein, so wird das Feld Kurs zugänglich und der entsprechende Umrechnungskurs kann angegeben werden. Anhand des Kurses erfolgt eine Umrechnung der Vorgänge von der Systemwährung in die ausgewählte Währung. (Zumeist wird ein Durchschnittswert für die Periode vom Amt festgelegt und daher erfolgt die Ermittlung der Werte nicht pro Vorgang.)

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Mit diesem Button wird der Bezugsmonat in den Dialog Intrastat übernommen. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
