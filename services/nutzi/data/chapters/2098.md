# Besuchsleitstand 
Besuchsleitstand 

In diesem Menü werden die Kunden-Besuche der Vertreter aus der Übersicht in Form eines Gantt-Diagramms dargestellt und können hier ggf. auch angepasst werden. 

Die Buttons im Kopf des Dialogs dienen der Selektion z.B. auf den heutigen Tag, Morgen oder nächste Woche. 

Über die Felder Datum/Zeit von – bis kann die Anzeige auf einen bestimmten Zeitraum begrenzt werden. 

Alle Zuteilungen von Vertretern werden angezeigt, welche gleichzeitig als Ressource angelegt sind. 

Folgende Status finden Verwendung: 

Blau Gelb Grün 
| offener Besuch |
| --- |
| freigegebener Besuch (in Bearbeitung) |
| abgeschlossener Besuch |

Um einen Besuch zu verschieben, kann dieser einfach per Drag&Drop verschoben werden. Durch einen Klick auf den Termin werden die Details unterhalb des Gantt-Diagramms angezeigt und mit einem Doppelklick auf einen Termin wird direkt der Besuchsbericht geöffnet. 

Ein Einsatz lässt sich nur verschieben, wenn der Status des Einsatzes nicht auf ‚abgeschlossen‘ steht. Auch das Größer-Ziehen eines Balkens ist möglich. 

Bereits vorhandene Einsätze im gleichen Zeitraum werden nicht berücksichtigt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Besuchsleitstand 
Tooltip: 

In diesem Dialog werden die Besuche bei Kunden in einer Übersicht dargestellt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist.
