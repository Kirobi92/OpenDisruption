# Navigator 
Navigator 

Der Navigator stellt die modulübergreifende Programmzentrale dar. Sie erreichen diese Registerkarte, wenn Sie auf das Symbol klicken. Aus ihm erfolgt der Aufruf einzelner Module und Eingabedialoge. Zusätzlich steuert er die generellen Systemeinstellungen und Zugriffe. 

Über das Menü Datei können Sie das Layout zurücksetzen, sich abmelden oder eNVenta ERP beenden. 

Über das Menü System definieren Sie die Systemeinstellungen für Ihre Arbeit mit eNVenta ERP . Dazu gehören u.a. die Parametereinstellungen, Auftrags- und Versandarten. 

Die Systemeinstellungen der hier aufgeführten Module werden Ihnen in den jeweiligen Kapiteln erläutert. 

Über das Menü Favoriten können Sie häufig benötigte Dialoge als Favoriten anlegen. 

Favoriten im Navigator 

Folgende Einträge werden angeboten: 
- Hinzufügen (um einen Link hinzuzufügen) - Entfernen (um einen Link zu entfernen) - Autom. Start - Neuer Link 
Über das Menü Neuer Link öffnet sich der Dialog URL , um einen neuen Favoriteneintrag zu erzeugen. 

Siehe Kapitel Favoriten über Links erzeugen . 

Favoriten lassen sich auch per Drag & Drop in den Navigator ziehen. Streichen Sie mit der Maus über die Buttonleiste bis die Hand erscheint (siehe nachfolgende Abbildung). Jetzt halten Sie die linke Maustaste fest. Der Frame verdunkelt sich und sie können das Element in den Favoritenbereich hineinziehen. 

Das erspart Ihnen eine Auswahl über den Navigator. Diese werden in der Reihenfolge so angeordnet, wie sie im Navigator eingetragen sind. 

Eine individuelle Sortierung ist nicht möglich! 

Auf der Registerkarte Navigator sehen Sie, angeordnet wie in einem Dateimanager, alle lizenzierten Programm- und Zusatzmodule. Durch einen Doppelklick auf ein Programmmodul öffnet sich eine weitere Ebene, in der alle Dialoge des zugehörigen Moduls aufgelistet werden, die ebenfalls per Doppelklick geöffnet werden können. 

Einige Dialoge, wie z.B. der Kunde sind in mehr als einem Modul verfügbar. Es handelt sich dabei immer um denselben Dialog. Wenn er bereits in einem Modul aufgerufen wurde und nun aus einem anderen erneut gewählt wird, erkennt der Navigator den erneuten Zugriff und stellt den bereits geöffneten Dialog zur Verfügung. 

Das auf der Registerkarte enthält die Menüs Versions-Info , TeamViewer und E-Mail an Support . 

Der Aufruf von eNVenta ERP erfolgt über den Broker. 

Der Broker ist das Programm auf dem Server, welches später die Anwendung ausführt, um die XML-Requests des Clients verarbeiten zu können. Dazu benötigt der Broker neben den (als .dll kompilierten) Dateien der Anwendung einige Standard-Libraries. 

Der Login kann auch mit der Enter -Taste ausgeführt werden. 

Die Anwendung kann auch über die Tastenkombination <Strg+Umschalt+X> geschlossen werden.

## Menüeinträge 

### Menü Datei 
Unter diesem Menüpunkt finden Sie die Einstellungen: 

- Abwesenheit - CSC temp. deaktivieren: Damit lässt sich vorübergehend das Client Side Custimzing abschalten. - Historiendaten einsehen: Damit wird die Funktion Historiendaten einsehen aktiviert. 
Siehe Kapitel Historisierungsübersicht und Daten Historisieren . 

### Menü Datei / Abwesenheit 
Hier werden die Abwesenheiten von Mitarbeitern, wie z.B. Urlaub und Krankheit, verwaltet. Diese Pflege erfolgt im Regelfall zentral über eine Zeiterfassungs-Software und wird dann per CSV-Datei in eNVenta ERP importiert. 

Siehe Kapitel Sachbearbeiter Abwesenheit . 

### Menü Datei / CSC temp. deaktivieren 
Damit es beim Update von Packages nicht zu Problemen kommt, kann über die Funktion CSC temp. deaktivieren das Client Side Customizing kurzfristig abgeschaltet werden. In einer Meldung erhalten Sie den Hinweis, dass die Änderungen erst nach einem erneuten Öffnen des Forms zur Verfügung stehen. 

Diese Funktion sollte grundsätzlich nur dem Administrator zugänglich sein, da alle Einstellungen des Client Side-Customizing deaktiviert werden. 

### Menü Datei / Historiendaten Einsehen 
Wenn Sie im Navigator unter Datei den Eintrag Historiendaten einsehen anklicken, wird diese Funktion aktiviert. Oberhalb des Navigators wird in rot der Begriff Historie angezeigt. So ist schnell zu erkennen, dass die Historisierung aktiv ist. Ist der Modus inkl. Historiendaten aktiv, so wird immer in beiden Tabellen (Aktive Tabelle und Historien-Tabelle) gesucht, eine Unterscheidung ist für den User nicht möglich.

### Menü Favoriten 
Über diesen Menüpunkt können Sie z.B. Projektaufgaben als Favoriten im Navigator ablegen, um Sie sofort nach dem Öffnen von eNVenta zu Verfügung zu haben. 

Siehe Beispiel: 

Favoriten 

Es stehen folgende Einträge zur Verfügung: 

- Hinzufügen - Entfernen - Autom. Starten - Neuer Link 

### Menü Favoriten / Hinzufügen 
Über dieses Menü fügen Sie einen Favoriten in den Navigator ein. Dazu markieren Sie z.B. eine Projektaufgabe im Dialog Projekt auf der Registerkarte Phasen und ziehen sie per Drag & Drop in den Navigator unterhalb des Begriffs Favoriten . 

### Menü Favoriten / Entfernen 
Über dieses Menü entfernen Sie einen markierten Favoriten wieder aus dem Navigator. 

### Menü Favoriten / Autom. Start 
Mit dieser Funktion kann ein Favorit im Navigator so definiert werden, dass der Dialog beim Öffnen von eNVenta sofort gestartet wird. Dieser Favorit ist dann mit dem Kennzeichen (*) versehen. 

Diese Funktion bietet dem Anwender ein schnelles Herstellen der täglich benötigten Arbeitsumgebung. 

### Menü Favoriten / Neuer Link 
Mit diesem Menü können Sie über den sich öffnenden Dialog UrL einen neuen Favoriteneintrag erzeugen. Dazu geben Sie den URL-Namen und den Link-Namen ein. 

Siehe Kapitel Neuer Link . 

### Menü ?

Siehe Kapitel Versions-Info . - TeamViewer: Zum Öffnen einer TeamViewer Sitzung. - E-Mail an Support: So kann auf einfache Weise, direkt aus dem System heraus, eine E-Mail an den Support erfasst werden. Siehe Kapitel E-Mail an Support . - Customizing: So gelangen Sie ins Client Side Customizing.
