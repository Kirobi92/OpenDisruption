# Lieferavis im Wareneingang 
Auswahl Lieferavis 

Im Wareneingang >> Optionen >> Lieferavis öffnet sich der Dialog Auswahl Lieferavis . Sie finden hier alle über Business Gateway eingelesenen, noch nicht verarbeiteten Lieferscheine des Lieferanten. Mit einem Klick auf den Button werden die markierten Lieferscheine in den Wareneingang übernommen. 

Hier stehen alle Wareneingänge zur Verfügung, welche ins System eingepflegt (z.B. über EDI), aber noch nicht verarbeitet wurden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Auswahl Lieferavis 
Tooltip: 

In diesem Dialog stehen alle Wareneingänge zur Verfügung, welche ins System eingepflegt (z.B. über EDI), aber noch nicht verarbeitet wurden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt.

Ist ein Lager ausgewählt, werden alle Lieferavis-„Köpfe“ angezeigt, zu denen eine Position im entsprechenden Lager existiert.

### Tabelle Der Lieferavis 

Hier werden alle noch nicht verarbeiteten Lieferscheine angezeigt. Bei einer Fehlermeldung wird die Positionszeile rot markiert und in der Spalte 'Dateiname' wird der Name der Datei angezeigt, um die Datei schnell wieder finden zu können. 

Diese Fehlermeldung wird dann auch im Gateway angezeigt.
