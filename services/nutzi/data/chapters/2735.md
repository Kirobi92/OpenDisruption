# Kostenarten-/Leistungs-Blatt 
Kostenarten-/Leistungs-Blatt 

Dieser Dialog öffnet sich im Buchen Kore >> Auswertungen >> Kostenarten-/Leistungs-Blatt . 

Hauptunterschied gegenüber dem Kostenstellen- /Kostenarten- / Leistungsarten- / Kostenträgerjournal ist es, dass hier nicht die einzelnen Buchungen angezeigt werden, sondern die Periodensummen. 

Als Pendant zur Finanzbuchhaltung können das Kostenstellen- /Kostenarten- / Leistungs- / Kostenträgerjournal mit den Kontoblättern verglichen werden, das Kostenartenblatt, Kostenstellenblatt und Kostenträgerblatt mit der Saldenliste. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Kostenarten-/Leistungs-Blatt 
Tooltip: 

In diesem Dialog werden die Periodensummen, entsprechend Ihrer Selektionskriterien, angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
