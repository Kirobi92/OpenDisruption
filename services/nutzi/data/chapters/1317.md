# Angebotsübersicht 
Angebotsübersicht 

Mit diesem Programmteil können Sie sich entweder ganz allgemein oder nach Selektion auf einen Vertreter die bestehenden Angebote anzeigen lassen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Angebotsübersicht 
Tooltip: 

In diesem Dialog können Sie sich entweder ganz allgemein oder nach Selektion auf einen Vertreter die bestehenden Angebote anzeigen lassen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

Dieser Button startet den Druck. 

### Button Suche 

Dieser Button startet die Suche entsprechend Ihrer Selektionskriterien. 

### Button Neu 

Dieser Button leert die Felder.

Dieser Button übernimmt einen Vorgang in den Dialog Angebot .

Mit diesem Button können Sie Einstellungen für diesen Programmteil definieren. 

Siehe Kapitel Einstellungen . 

### Combobox Art 
Für die Selektion der Angebote über die Angebotsart, wählen Sie hier den entsprechenden Eintrag aus. 

Tooltip: 

Eingabe der Auftragsart 

### Combobox Klasse 
Zur Selektion von Angeboten über die Angebotsklasse wählen Sie hier die gewünschte Angebotsklasse aus. 

### Combobox Status 
Hier wird der Status eines Angebots angegeben. 

Zur Verfügung stehen folgende Status: 

- offen - teilw. erhalten - erhalten - verloren - überarbeitet 

Tooltip: 

Status 

### Combobox Unterstatus 
Aus dieser Combobox wählen Sie einen Unterstatus. Dieser definiert den Hauptstatus genauer und kann frei definiert werden. Die Definition der Unterstatus wird unter System >> Verkauf >> Angebots-Unterstatus oder über das Menü Stamm >> Unterstatus vorgenommen. Die Spalte ‚Unterstatus’ wird vom System gefüllt. 

Siehe Kapitel Systemeinstellungen im Verkauf / Angebots-Unterstatus . 

Tooltip: 

Unterstatus 

### Combobox Vertreter 1 
Zur Selektion der Angebote über einen Vertreter, wählen Sie hier die gewünschte Vertreternummer aus. 

Tooltip: 

Angabe der Vertreternummer 

### Feld Kunde 
Zur Selektion von Angeboten über eine Kundennummer, tragen Sie hier die gewünschte Kundennummer ein. 

Tooltip: 

Angabe der Kundennummer 

### Button Suche 

Mit diesem Button können Sie einen Kunden suchen. 

Siehe Kapitel Kundenauswahl . 

### Feld Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Projekt 
Zur Selektion der Daten über ein Projekt, geben Sie hier die gewünschte Projektnummer ein. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Mit diesem Button können Sie ein Projekt suchen. 

Siehe Kapitel Projektauswahl . 

### Combobox Verlustgrund 
Zur Selektion der Daten über einen Verlustgrund, wählen Sie hier den entsprechenden Eintrag aus. 

Tooltip: 

Angabe eines Verlustgrunds 

### Feld Wahrsch.% ab 
Dieses Feld dient der weiteren Eingrenzung der Angebote, die angezeigt werden sollen. Diese Wahrscheinlichkeit wird als Prozentsatz in den Angeboten definiert. Der Prozentsatz gibt die Wahrscheinlichkeit an, dass aus dem Angebot ein Auftrag wird. 

### Combobox Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Combobox Erfasser 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz erfasst hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der den Datensatz erfasst hat. 

### Feld Erfassung von 
Zur Selektion der Daten über ein Erfassungsdatum, geben Sie hier das entsprechende Datum ein. 

Tooltip: 

Datum und Uhrzeit der Erfassung 

### Feld Erfassung bis 
Zur Selektion der Daten über einen Datumsbereich, geben Sie hier ein weiteres Erfassungsdatum ein. 

Tooltip: 

Zeigt das Erfassungsdatum
