# Auftragsstückliste Registerkarte Arbeitsplan 
Auftragsstückliste Registerkarte Arbeitsplan 

Auf dieser Registerkarte haben Sie die Möglichkeit individuelle Arbeitspläne zu verwalten. Dies ist beispielsweise interessant, wenn bearbeitetes Material verkauft wird (Anarbeitung) oder bei kleineren Aufträgen schnell ein entsprechender Betriebsauftrag erstellt werden soll. Mit Hilfe der Auftragsstückliste können dann auch die Bearbeitungsschritte aus dem Kundenauftrag heraus für die Produktion konfiguriert werden. 

Wird für eine Produktionsstückliste im Auftrag eine Rückstandsposition gebildet, wird der auftragsspezifische Arbeitsplan übernommen. Der komplette Arbeitsplan inkl. aller Anpassungen wird übernommen. 

Im Gegensatz zu Materialpositionen der Stückliste wird aber nicht automatisch für jede Auftragsstückliste auch ein Auftrags-Arbeitsplan angelegt. 

Es wird davon ausgegangen, dass der Arbeitsplan in aller Regel im Standard bleibt. Eine Anpassung erfolgt daher nur im Bedarfsfall. 

Um einen auftragsindividuellen Arbeitsplan zu verwenden, muss daher für die betreffende Baugruppe der Stamm-Arbeitsplan übernommen werden oder manuell einzelne Arbeitsschritte aus dem Stamm ausgewählt werden. 

Sobald eine individuelle Arbeitsplanposition angelegt ist, wirkt der Arbeitsplan aus dem Stamm nicht mehr. 

Dies gilt dann aber nur für die betreffende Baugruppe. Für andere Baugruppen gilt weiterhin der Stamm. 

Dafür gibt es links im Stücklistenbaum folgende Positionen, die mit verschiedenen Symbolen gekennzeichnet sind: Materialpositionen, Baugruppen und Arbeitsgänge. Jede dieser Eintragsarten hat ein eigenes graphisches Symbol zur leichteren Unterscheidung. Dies sind: 

- Das -Symbol kennzeichnet eine Baugruppe. - Das -Symbol kennzeichnet eine Materialposition. - Das -Symbol kennzeichnet einen Arbeitsgang. 
Um einen individuellen Arbeitsplan anzulegen, muss die betreffende Baugruppe markiert werden. Dann nehmen Sie die Einstellungen auf dieser Registerkarte vor. 

Bereits durch einen neuen Arbeitsgang wird der gesamte Arbeitsplanstamm für diese Baugruppe deaktiviert. 

Die Positionsnummer kann vorgegeben werden, sonst erfolgt eine automatische Vergabe. 

Bei der Erstellung eines Betriebsauftrags prüft das System, ob ein individueller Arbeitsplan vorhanden ist oder nicht. Je nachdem werden die entsprechenden Daten verwendet. Dies betrifft auch die automatische Zuordnung von Ressourcen und Material. 

Herkunft der Arbeitsplanposition aus dem Stamm-Arbeitsplan 

Ist in die Auftragsstückliste eine Arbeitsplanposition durch „Übernahme“ aus dem Stamm eingefügt worden, so werden auch die im Stamm-Arbeitsplan zugeordneten Ressourcen und Materialien übernommen. 

Herkunft der Arbeitsplanposition aus dem Arbeitsgangstamm 

In diesem Fall werden nur die im Arbeitsgangstamm bereits zugeordneten Ressourcen übernommen. 
### Button Neu 

Dieser Button öffnet den Dialog Auswahl Arbeitsgang , um einen Arbeitsschritt aus dem Arbeitsgangstamm zu übernehmen. Nach Auswahl eines Arbeitsgangs werden alle Daten aus dem Arbeitsgangstamm übernommen und können an die Bedürfnisse angepasst werden. 

Siehe Kapitel Produktion / Arbeitsgang auswählen . 

### Button Übernahme 

Dieser Button öffnet den Dialog Variante Gesamt . Der Standard-Arbeitsplan der Baugruppe wird in die angegebene Variante komplett übernommen und kann angepasst werden. 

Siehe Kapitel Gesamtvariante . 

### Feld Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000).

### Feld Zeichnung

Tooltip: 

Nummer der Bauplanzeichnung 

### Feld Rüstzeit (mm,Is) 
Angabe der Rüstzeit für den Arbeitsgang in Minuten und Industriesekunden. Die Angabe wird pro Arbeitsgang einmalig berücksichtigt. 

Rüstzeit (mm=Echtminute, Is=Industriesekunde) 

Zur Information: 

1 Industriesekunde (Is) = 0,006 Echtminuten (mm) 
(z.B.: 8 Stunden 45 Minuten = 525 Echtminuten = 87500 Industriesekunden) 

Tooltip: 

Rüstzeit (mm=Echtminute,Is=Industriesekunde) 

Zur Information:

(z.B.: 8 Stunden 45 Minuten = 525 Echtminuten = 87500 Industriesekunden)

### Feld Pro Menge 
Geben Sie hier an, für wie viele Einheiten der Lohn gilt. Wenn keine Angabe erfolgt, wird „pro 1“ angenommen.

Angabe, für wie viele Einheiten der Lohn gilt.
