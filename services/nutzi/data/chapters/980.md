# Parameter Verkauf Registerkarte Artikelsuche 
Parameter Verkauf Registerkarte Artikelsuche 

Für die Suche von Artikeln im Auftrag können Sie auf dieser Registerkarte den Suchradius über die Checkboxen bestimmen: 
- Fremdartikelnummer - GTIN - EAN/Einheits Nr. - Einheiten EAN - Kunden Artikel - Liefer.Bestellnummer 
Diese Checkboxen sind standardmäßig aktiviert! 

Die Suchreihenfolge bestimmt sich wie folgt: 
- Passende Artikelnummer - Passende Fremdartikelnummer - Passende GTIN - Passende EAN oder Einheitsnummer - Passende EAN in Artikeleinheiten - Passende Kundenartikelnummer - Passende Bestellnummer oder EAN im Rabatt Einkauf - Ähnliche Artikelnummer (LIKE-Suche) - Ähnliche Einheitsnummer (LIKE-Suche) 
So ist es möglich auch nach Fremdartikelnummern zu suchen, wie z.B. Werksnummer und EAN-Nummer. 

Siehe Kapitel Fremdartikelnummern . 

Möchten Sie die Suche wieder nur auf eigene Artikel begrenzen, ist es erstens notwendig die Checkbox Fremdartikelnummern zu deaktivieren und zweitens den Suchindex neu zu indizieren. 
### Checkbox GTIN 
Wenn Sie diese Checkbox aktivieren, steuern Sie die Suche nach Artikeln im System gezielt über die GTIN. 

Tooltip: 

Mit dieser Checkbox können Sie die Suche nach Artikeln im System gezielt über die GTIN steuern. 

### Checkbox EAN / Einheits Nr. 
Wenn Sie diese Checkbox aktivieren, steuern Sie die Suche nach Artikeln im System gezielt über die Einheiten EAN. 

Tooltip: 

Mit dieser Checkbox können Sie die Suche nach Artikeln im System gezielt über die EAN steuern. 

### Checkbox Einheiten EAN 
Hier kennzeichnen Sie, ob bei der Artikelsuche die EAN aus den abweichenden Artikeleinheiten verwendet werden soll. 

Tooltip: 

Mit dieser Checkbox können Sie die Suche nach Artikeln im System gezielt über die Einheiten EAN steuern. 

### Checkbox GTIN in Artikeleinheiten 
Tooltip: 

Mit dieser Checkbox können Sie die Suche nach Artikeln im System gezielt über die Einheiten GTIN steuern. 

### Checkbox Fremdartikelnummern 
Hier geben Sie ein Kennzeichen an, ob bei der Artikelsuche die Fremdartikelnummer verwendet werden soll. 

Tooltip: 

Mit dieser Checkbox können Sie die Suche nach Artikeln im System gezielt über die Fremdartikelnummer steuern. 

### Checkbox Kundenartikel 
Wenn Sie diese Checkbox aktivieren, steuern Sie die Suche nach Artikeln im System gezielt über die Kundenartikelnummer. 

Tooltip: 

Mit dieser Checkbox können Sie die Suche nach Artikeln im System gezielt über die Kundenartikelnummer steuern. 

### Checkbox Liefer. Bestellnummer 
Wenn Sie diese Checkbox aktivieren, können Sie die Suche nach Artikeln im System gezielt über die Lieferanten-Bestellnummer des Rabatt-Einkaufs steuern. 

Tooltip: 

Mit dieser Checkbox können Sie die Suche nach Artikeln im System gezielt über die Lieferanten-Bestellnummer, sowie die EAN des Rabatt-Einkaufs steuern. 

### Feld Feldliste 
In diesem Feld werden die ausgewählten Felder der konfigurierbaren Artikelsuche ( Konfiguration der Artikelsuche ) angezeigt, da der Inhalt dieser Felder die Einträge im Artikel Index darstellt und somit auch bei der Suche berücksichtigt wird. 

Tooltip: 

Hier werden die Felder aus der Konfigurierbaren Artikelsuche angezeigt. 

### Checkbox Artikelsuche wie eNVenta 4.4 und früher 
Wenn Sie diese Checkbox aktivieren, läuft die Suche nach Artikeln wie in den eNVenta Versionen 4.4 und älter. 

Tooltip: 

Das Vorgehen für die Suche nach Artikeln wird wieder wie in den Versionen 4.4 und älter durchgeführt. 
Das Suchverhalten kann in der Online-Hilfe nachgelesen werden. 
Dies hat keinen Einfluss auf das Sucherverhalten im Artikelstamm! 
Hiervon ist die direkte Übernahme von Artikel in Dialogen sowie der Dialog „Artikelauswahl“ betroffen.
