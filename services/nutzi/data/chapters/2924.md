# Event Management Registerkarte Service Protokoll 
Event Management Registerkarte Service Protokoll 

Auf dieser Registerkarte werden die Protokolle angezeigt. Um die Datenmenge zu begrenzen, kann über das Datum und den Status (Information, Warnung, Fehler) selektiert werden. 
### Feld Datum von 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Checkbox Information 
Wenn Sie diese Checkbox aktivieren, werden die Daten über den Status Information selektiert. 

### Checkbox Warnung 
Wenn Sie diese Checkbox aktivieren, werden die Daten über den Status Warnung selektiert. 

### Checkbox Fehler 
Wenn Sie diese Checkbox aktivieren, werden die Daten über den Status Fehler selektiert. 

### Tabelle 

Die Tabelle listet die verarbeiteten Datensätze auf. 
### Tabellenspalte Anfangszeit 
Tooltip: 

Zeitpunkt zudem die Verarbeitung des Ereignisses gestartet wurde 

### Tabellenspalte Modulname 
Tooltip: 

Zeigt an welches Modul des Adapters ausgeführt wurde. 
Beim EDI werden die einzelnen Nachrichtentypen als Modul aufgeführt. 
Beim AdapterGeneralIO die Export-/Importkonfigurationen 

### Tabellenspalte Aktivitätsparameter 
Tooltip: 

Kommaseparierte Liste der Werte, die zum asynchonen Laden des zu verarbeitenden Datensatzes notwendig sind. 

### Feld Fehlermeldung 
In diesem Feld wird die Fehlermeldung angezeigt.
