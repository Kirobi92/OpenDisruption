# Gutscheinartikel Registerkarte Allgemein 
Gutschein - / Gratisartikel Registerkarte Allgemein 

Auf dieser Registerkarte werden die Gutscheinartikel gepflegt. 
### Tabelle 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Betrag von 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Betrag bis 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Button Satz hinzufügen 

Mit diesem Button kann ein neuer Datensatz in die Tabelle eingefügt werden. 

### Button Löschen 

Mit diesem Button entfernen Sie einen Datensatz in der Tabelle.
