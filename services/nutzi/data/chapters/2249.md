# Katalogverwaltung Registerkarte Sichtbeschränkung 
Katalogverwaltung Registerkarte Sichtbeschränkung 

Auf dieser Registerkarte definieren Sie, wer welche Katalogeinträge sehen darf und wer nicht. 
### Button Satz hinzufügen 

Mit diesem Button legen Sie eine neue Zeile an. In dieser Zeile versteckt sich eine Combobox, aus der Sie den entsprechenden Eintrag auswählen. 

### Button Löschen 

Mit diesem Button löschen Sie einen Eintrag.
