# GAP Position ändern Registerkarte Rahmen-Info 
Position Registerkarte Rahmen-Info 

Auf dieser Registerkarte erhalten Sie Informationen bezüglich einer Rahmenvereinbarung. 

Ist ein Rahmenvertragsbezug vorhanden, wird die Registerkarte mit einem Sternchen versehen. Dies ist das Kennzeichen, dass es auf der Registerkarte eine Information gibt. 

Beim Kopieren eines Auftrags, der aus einem Rahmenvertrag generiert wurde, werden die Rahmeninformationen übergeben. D.h. auf dieser Registerkarte werden die Rahmeninformationen angezeigt. 

WICHTIG: 

Der autogenerierte Text 'Abruf aus RahmenvertragXY' wird nicht mehr automatisch als Positionstext generiert. Hierzu steht ein alternatives Property zur Verfügung. 

Wenn Sie Ihre Reports selbst verwalten, müssen Sie in folgenden Reports das Property sOutlAgreeCallText hinzufügen: 

- rptPurchaseDocumentFree - rptPurchaseDocument - rptGoodsReceipt - rptSalesDocumentFree - rptSalesDocument - rptSalesDocBacklog - rptNV4_PickOrderLabel

### Feld VK pro 
Hier wird eine Preiseinheit angegeben. Dieser Preis kann sich auf 10, 100, 1000 oder eine andere Anzahl Artikeleinheiten beziehen. Standard ist der Wert 1. 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- 1 Einheit - je 10 Einheiten - je 100 Einheiten
