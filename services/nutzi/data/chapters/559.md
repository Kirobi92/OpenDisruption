Grundeinstellungen für eNVenta ERP / Einrichtung des Lucene Service 

# Lucene Service aktivieren 
Um den Lucene Service Host zu aktivieren, muss dieser analog dem Broker in FS installiert werden. Hierbei muss bei der Installation des Service die entsprechende Einstellung vorgenommen werden. 

Beim Publish einer Anwendung kann im Bereich der Service Hosts die entsprechende Einstellung für den Lucene Service Host vorgenommen werden. 

Service Host Lucene Service Host 

Im Feld Server wird der Server angegeben, auf dem der Lucene-Dienst läuft. 

Im Feld Port wird ein freier Port angegeben unter welchem der Lucene-Dienst erreichbar sein soll. 

Im Feld Hostname wird die Bezeichnung des Dienstes angegeben, unter welcher dieser in der Registry eingetragen wird. Mittels der Checkbox Custom Name kann das Feld aktiviert und die Bezeichnung überschrieben werden. 

Über die Felder Username und Password kann angegeben werden, unter welchem User-Kontext der Dienst gestartet werden soll. Dieser Benutzer benötigt im späteren Verlauf entsprechende Schreib- und Leserechte auf dem Speicherort des Lucene-Verzeichnisses. 

Im Feld Configuration kann dem Lucene Service optional eine XML-basierte Konfiguration mitgegeben werden, welche das Verhalten steuert. Wird keine Konfiguration hinterlegt, so gibt es einen Lucene-Service, welcher über alle Mandanten alle Daten indexiert.
