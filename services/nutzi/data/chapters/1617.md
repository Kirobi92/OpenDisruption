Lagerverwaltungssystem (LVS) 

# LVS Tagesstatistik 
LVS Tagesstatistik 

Dieser Dialog öffnet sich im Navigator >> Auswertungen >> Lager >> LVS Tagesstatistik . Hier kann eine tägliche Statistik über Picks erstellt werden. So erhalten Sie eine Statistik über die Pickleistung pro Tag. 

Das Tagesdatum ist in den Feldern Datum von/bis voreingestellt, kann aber angepasst werden. 

Ein Lager ist zwingend auszuwählen. Der Kommissioniertyp und der Sachbearbeiter können als Selektionskriterium genutzt werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld LVS Tagesstatistik 
Tooltip: 

In diesem Dialog können Sie eine tägliche Statistik über Picks erstellen. 
Für mehr Informationen drücken Sie F1.

### Combobox Lagernr

### Combobox Entnahmeart 
Hier kann zur Selektion der Kommissionstyp ausgewählt werden. Zur Selektion stehen: 

- Entnahme - Umlagerung - Einlagerung - Inventur - Fehlerkorrektur 

Tooltip:

- Entnahme 
- Umlagerung 
- Einlagerung 
- Inventur 
- Fehlerkorrektur 

### Liste Status 
In diesem Feld kann auf die verschiedenen Pick-Status der Lageraufträge eingegrenzt werden. Dabei ist es möglich, eine Mehrfachauswahl vorzunehmen. Wenn kein Status ausgewählt ist, wird dieser nicht eingeschränkt. 

Tooltip: 

Selektion über Pick-Status der Lageraufträge. 

### Feld Datum von 
Hier wird vom Programm automatisch das Datum der Erfassung eingetragen. Wenn das Von-/Bis-Datum angegeben ist, werden die Picks in diesem Zeitraum aufsummiert und entsprechend in der Tabelle angezeigt. 

Tooltip: 

Selektion über Datum. 

### Feld Datum bis 
Wenn das Von-/Bis-Datum angegeben ist, werden die Picks in diesem Zeitraum aufsummiert und entsprechend in der Tabelle angezeigt. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Combobox Pick-Sachbearbeiter 
Für die Pickstatistik kann hier ein Sachbearbeiter ausgewählt werden. Es stehen nur die Sachbearbeiter zur Auswahl, bei denen an diesem Lagerort ein Scanner Sachbearbeiter angelegt ist. 

Tooltip: 

Selektion über Sachbearbeiter.
