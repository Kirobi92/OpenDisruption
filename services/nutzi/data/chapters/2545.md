# Kundenbonitätsliste Registerkarte Selektionen 2

Finanzbuchhaltung / Sicherheiten-Verwaltung / Kundenbonitätsliste 

# Kundenbonitätsliste Registerkarte Selektionen 2 
Kundenbonitätsliste Registerkarte Selektionen 2 

Auf dieser Registerkarte können Sie weitere Selektionskriterien festlegen. 
### Combobox Kundengruppe 
Hier wird eine Kundengruppe angegeben. 

Tooltip: 

Auswahl der Kundengruppe 

### Combobox Kundenstatus 
Aus dieser Combobox können Sie einen Kundenstatus zur Selektion auswählen. 

Tooltip: 

0, leer = Normal 
1 = Gesamtsperre 
2 = Liefersperre 
3 = Fakturasperre 

### Combobox Sperrgrund 
Hier können Sie zur Datenselektion einen Sperrgrund auswählen. 

Tooltip: 

Sperrgrundnummer zum Kundenstatus 

### Combobox Rechtsform 
Hier wird die Rechtsform des Unternehmens angegeben, wie z.B. GmbH oder AG. Diese werden in einer Pickliste gepflegt. 

Tooltip: 

Angabe der Unternehmen-Rechtsform 

### Combobox Versandlager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager 

### Combobox Niederlassung 
Hier wird die Niederlassungsnummer angegeben. 

Tooltip: 

Angabe der Niederlassung 

### Tabelle 

In diesem Feld kann für die Anzeige in der Tabelle noch das Gebiet / Bundesland als Selektionskriterium gewählt werden. 
### Tabellenspalte Gebiet 
Hier kann das Gebiet bzw. das Bundesland angegeben werden. 

Tooltip: 

Angabe einer Beschreibung
