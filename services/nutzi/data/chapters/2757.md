# Hilfskostenstellen 
Hilfskostenstellen 

Eine Hilfskostenstelle (auch Vorkostenstelle, Sekundärkostenstelle oder Nebenkostenstelle genannt) ist eine Kostenstellenkategorie, die in der Kostenstellenrechnung zum Zwecke der Verrechnung innerbetrieblicher Leistungen gebildet wird. Hilfskostenstellen stehen im Gegensatz zu Hauptkostenstellen. Auf Hilfskostenstellen selbst werden keine Kostenträger gefertigt, sondern sie erbringen lediglich die für den Produktionsprozess notwendigen Infrastrukturleistungen, wie etwa Energieversorgung, innerbetrieblicher Transport und Verwaltung. In der innerbetrieblichen Leistungsverrechnung werden die Gemeinkosten der Hilfskostenstellen direkt oder indirekt auf die Hauptkostenstellen verrechnet, da die Kosten einer Hilfskostenstelle nicht direkt auf einen Kostenträger umgelegt werden können. Wählen Sie dabei möglichst verursachungsgerechte Bezugsgrößen für die Inanspruchnahme der Leistungen. 

Die Hilfs- und Vorkostenstellen werden gemeinsam im Hilfskostenstellenstamm verwaltet. Dieser ist im Navigator unter Hilfskostenstellen zu finden. 

### Ändern und Löschen einer Hilfskostenstelle 
Sie können die beschriebenen Einstellungen zu Hilfskostenstellen jederzeit ändern. 

Das Löschen einer Hilfskostenstelle mit dem Button ist nur möglich, solange in der Kostenrechnung noch nichts auf die Kostenstelle gebucht wurde. Wurden schon Buchungen ausgeführt, akzeptiert das Programm die Löschung nicht. 

Aus dem Programmmodul Kostenrechnung heraus kann nicht nachgeprüft werden, ob bereits in der Finanzbuchhaltung auf diese Kostenstelle gebucht wurde. Auch eine Überprüfung der im Sachkontenstamm in der Finanzbuchhaltung hinterlegten Kostenstellen erfolgt nicht. Solche Buchungen würden dann aber bei der Übernahme in die Kore ( Kore-Übernahme ) mit einem entsprechenden Fehlertext abgebrochen werden. 

Zu Ihrer Sorgfaltspflicht gehört es zu überprüfen, ob es keine Vorkostenstellen oder Kostenarten/Leistungen gibt, die auf die Hilfskostenstelle, die Sie löschen möchten, verteilt werden. Wird eine Hilfskostenstelle gelöscht, so werden sämtliche Verweise auf diese Hilfskostenstelle ebenfalls gelöscht. Wenn Sie nicht zuvor die Zuordnungen korrigiert haben, kann dies dazu führen, dass z.B. eine Kostenart nicht zu 100% auf Kostenstellen verteilt wird. 

### Konvertieren einer Hilfskostenstelle zur Hauptkostenstelle 
Möchten Sie eine bisherige Hilfskostenstelle zukünftig als Hauptkostenstelle einsetzen, so müssen Sie die Daten nicht jeweils löschen und wieder neu eingeben, sondern können dies bequem mit dem Button Konvertier . erledigen. 

Sie finden diese Funktion im Hilfskostenstellenstamm auf der Registerkarte Allgemein . Durch Betätigen des Buttons wird die Hilfskostenstelle gelöscht und eine entsprechende Hauptkostenstelle neu angelegt. 

Wenn Sie noch Veränderungen an den hinterlegten Angaben vornehmen wollen, so können Sie diese nun im Kostenstellenstamm eingeben. 

Siehe Kapitel Kostenstellen . 

Die Zuordnungen von Kostenarten/Leistungen zu dieser Kostenstelle werden durch die Konvertierung nicht verändert.
