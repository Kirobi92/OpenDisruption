# Verbandsabrechnung Debitor Registerkarte Abrechnungsvorschlag 
Verbandsabrechnung Deb. Registerkarte Abrechnungsvorschlag 

Die den Selektionskriterien entsprechenden offenen Posten werden Ihnen in der Tabelle auf der Registerkarte Abrechnungsvorschlag angezeigt. 

Ein bereits erzeugter aber noch nicht verarbeiteter Abrechnungsvorschlag wird dadurch nicht gelöscht. 

Mit einem Klick auf das graue Quadrat, links vom ersten Spaltennamen, markieren Sie alle Datensätze. 

Wenn sich der Abrechnungsvorschlag auf einen Verband bezieht, werden in der Tabelle dieser Registerkarte die Spalten ‚Verband‘ und ‚Verb.-Mitgliedsnr‘ sichtbar. Diese Nummern werden im Kundenstamm hinterlegt. 

Wenn Sie die Checkbox Alle Abrechn. konten oder Alle Zentralzahler aktivieren, erfolgt die Verbuchung immer über die Spalte ‚Verband‘ bzw. ‚Zentralzahler‘. Enthält ein selektierter OP keinen Verband oder Zentralzahler, kann dieser Satz im Vorschlag auch nicht verbucht werden. 

Einlesen der Daten 

Beim Einlesen verbandsspezifischer Daten werden die Daten in die Spalten übernommen. 

Unabhängig vom Vorzeichen, das in der Spalte ‚Skonto‘ der Importdatei steht bzw. in der Spalte ‚Betr.in HW‘, wird erst durch die erfolgreiche Zuordnung zu einem OP von diesem der reine Wert als Skonto abgesetzt. Dementsprechend wird der Betrag in dieser Spalte negativ oder positiv dargestellt. 
### Button Neue Zeile

### Button Löschen 

Um gezielt OPs aus einem Vorschlag zu löschen, markieren Sie diese OPs direkt in der Tabelle und betätigen danach diesen Button. 

### Button Beibehalten 

Wenn Sie auf diesen Button klicken, werden alle nicht markierten Sätze aus dem Vorschlag gelöscht. So lassen sich Datensätze einfacher gestalten.

### Tabellenspalte Skontierfähig 
Wird ein Wert ausgewiesen und die Spalte ‚Skonto %‘ ist leer, wird nicht mit dem Skonto des Kunden auf den skontierfähigen Betrag gerechnet. Die Spalte ‚Skonto‘ bleibt leer, da bei der Direktverbuchung gar kein Skonto berücksichtigt wird. 

Bitte beachten: Wird der Wert für die Spalte ‚Skonto‘ gedreht, wird auch der Wert für die Spalte ‚Skontierfähig‘ gedreht. 

Tooltip: 

Skontierfähiger Wert, 
inkl. Steuer, Landeswährung
