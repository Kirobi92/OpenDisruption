# Kapitel NV.ERP.Base.Common.cdPaymentCondCustomer.lngChargeOffAccount

Hier geben Sie eine Kontonummer für die Ausbuchung eines offenen Postens ein. Damit wird Ihnen eine manuelle Ausbuchung erspart. 

Beispiel: 
Ein Kunde zahlt normalerweise seine Ware auf Rechnung. Es kommt aber vor, dass er das eine oder andere Mal bar bezahlt. Dieser Betrag soll aber in der Finanzbuchhaltung auf das Konto des Kunden (z.B. für Umsatzbetrachtungen) gebucht werden. 

Tooltip: 

Geben Sie eine Kontonummer für die Ausbuchung eines offenen Postens ein. 
Damit wird Ihnen eine manuelle Ausbuchung erspart.
