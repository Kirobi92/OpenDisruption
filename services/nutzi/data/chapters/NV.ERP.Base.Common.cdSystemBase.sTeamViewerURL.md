# Kapitel NV.ERP.Base.Common.cdSystemBase.sTeamViewerURL

In dieses Feld geben Sie die URL ein für den Download der Fernwartungssoftware Teamviewer. Nach der Eingabe erfolgt eine Prüfung, ob der Pfad bzw. das Ziel erreichbar ist. Wenn das Ziel nicht erreichbar ist, erhalten Sie eine Fehlermeldung und die URL wird zurückgesetzt. 

Der TeamViewer kann im Navigator >> ? >> TeamViewer geöffnet werden. 

Tooltip: 

Hier geben Sie die URL für den Team-Viewer ein.
