# Kapitel NV.ERP.Base.Common.cdSystemMM.shtPostWithoutCosts

Wenn dieser Parameter gesetzt ist, werden Daten an die Kostenrechnungs-Schnittstelle übergeben, auch wenn diese nicht lizenziert ist. 

Diese Logik wirkt aber nur dann, wenn der Parameter Permanente Lagerbewertung im Dialog Parameter Allgemein nicht gesetzt ist und – falls es sich um eine Fakturierungsbuchung eines Auftrags handelt – die Checkbox Int. Warenabgang in der entsprechenden Auftragsart ebenfalls nicht gesetzt ist. 

Tooltip: 

Wenn diese Checkbox gesetzt ist, werden Daten an die Kostenrechnungs-Schnittstelle übergeben, auch wenn diese nicht lizenziert ist.
