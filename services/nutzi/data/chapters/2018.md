Produktion (PPS) / Arbeitsgänge 

# Arbeitsgänge Registerkarte Tabelle 
Arbeitsgangstamm Registerkarte Tabelle 

Auf dieser Registerkarte werden alle Arbeitsgänge aufgelistet. Eine Änderung innerhalb der Tabelle ist nicht möglich. 
### Tabelle 

### Tabellenspalte Arbeitsgang 
Hier wird eine Arbeitsgangnummer angegeben. 

Tooltip: 

Angabe der Arbeitsgangnummer 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Im Gegenzug ist im Kundenstamm zwingend die Lieferantennummer anzugeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
In diesem Feld wird die Anzahl der tatsächlich gefundenen Datensätze eingetragen. 

Tooltip: 

Anzahl der tatsächlich gefundenen Datensätze
