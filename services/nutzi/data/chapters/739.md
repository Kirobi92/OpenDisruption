# Variantenstamm 
Variantenstamm 

Dieser Dialog öffnet sich im Dialog Stückliste über den Button Varianten . 

Beim Öffnen des Dialogs wird die Stücklisten-Artikelnummer übergeben und die vorhandenen Varianten werden in den Dialog eingelesen. 

Die Standardvariante hat die Nr. 0; sie darf nicht gelöscht werden. 

Um eine Variante zu löschen, muss die Stückliste komplett aufgelöst werden. Es dürfen keine Komponenten oder Endteile mehr mit dieser Variante bestehen. 

Beim Löschen einer Stückliste werden auch die dazugehörigen Varianten gelöscht. 

In diesem Zusammenhang steht die Combobox Variante im Dialog Stückliste (auf der Registerkarte Komponente ) zur Verfügung. 

Varianten werden auch in den nachfolgend aufgelisteten Dialogen angezeigt, bzw. stehen zur Selektion zur Verfügung: 

- Kalkulation Stückliste - Prod. Auftrag erstellen - Stückliste verfügbar - Stücklistendruck - Variantenwahl - Variante Gesamt - Arbeitsplan Registerkarte Allgemein - Kuppelprodukte

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button.

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken.

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
