# Datenexport Registerkarte Format

Allgemeine Funktionalitäten in eNVenta ERP / Datentransfer (Import und Export von Daten) / Datenexport 

# Datenexport Registerkarte Format 
Datenexport Registerkarte Format 

Auf dieser Registerkarte legen Sie das Format für den Export fest. 
### Button Hinzufügen 

Dieser Button fügt einen markierten Eintrag aus dem linken Feld in die rechte Tabelle ein. Hier können dem Feldnamen weitere Eigenschaften zugeordnet werden. 

### Button Zuweisen 

Dieser Button ändert einen Eintrag der rechten Tabelle auf den Wert ab, den Sie links angeklickt haben. 

### Button Zuweisung entf. 

Mit diesem Button wird eine Zuordnung entfernt. 

Tooltip: 

Entfernt eine Zuordnung 

### Baumansicht Eigenschaften 

In diesem Feld befinden sich die Stammdaten. 

### Button Neu 

Dieser Button fügt eine neue Zeile in die Tabelle ein. 

### Button Löschen 

Dieser Button markiert eine Zeile der Tabelle als "zu löschen". 

### Button Umsetzung 

Dieser Button öffnet den Dialog Daten Umsetzung zur Umsetzung von externen in interne Daten. Eine entsprechende Kennzeichnung wird in der Spalte 'Ums.' angezeigt. 

Siehe Kapitel Daten Umsetzung . 

### Tabelle 

Diese Tabelle beinhaltet Daten, die exportiert werden. Die Reihenfolge der Daten kann über die Buttons oberhalb der Tabelle geändert werden. 
### Tabellenspalte Spalte 
Hier steht die Spaltenbezeichnung der Export-Datei. 

### Tabellenspalte Pos. 
Hier geben Sie eine Ziffer an, welche die Reihenfolge der Spalte festlegt. 

### Tabellenspalte Länge 
Hier wird die Feldlänge angegeben. 

### Tabellenspalte Eigenschaft 
Die Eigenschaft beschreibt das Feld in eNVenta . 

### Tabellenspalte Ums. 
Hier wird das Kennzeichen angezeigt, das Sie über den Button Umsetzung festgelegt haben. 

### Tabellenspalte Formatvorgabe 
Hier können Sie verschiedene, eindeutige Formate vorgeben. Z.B. für Datum "dd.MM.yyyy" -> 10.01.2018. 

Tooltip: 

Mögliche Formatvorgabe: z.B. Datum "dd.MM.yyyy" -> 10.01.2019 

### Tabellenspalte Standardwert 
Der hier angegebene Wert übersteuert den Wert aus eNVenta . 

### Button Nach oben 

Mit diesem Button verschieben Sie einen Eintrag der Tabelle nach oben. 

Tooltip: 

Nach oben verschieben 

### Button Nach unten verschieben 

Mit diesem Button verschieben Sie einen Eintrag der Tabelle nach unten. 

Tooltip: 

Nach unten verschieben
