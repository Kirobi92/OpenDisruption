# Kapitel NV.ERP.Base.Common.cdSystemMM.shtGrossPriceGroup2

Preise, die über eine hier hinterlegte Preisgruppe ermittelt werden, werden für das Endkundengeschäft benötigt und wenn die Positionspreise inkl. Steuer ausgewiesen werden müssen. 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird.
