# Artikel Information 
Artikel Information 

Dieser Dialog öffnet sich in der Bestellung (Registerkarte Positionen ) und im Bestellautomat (Registerkarte Vorschläge ) über den Button Historie bzw. in der Disposition über den Button neben dem Feld Artikel . 

Die Anzeige der historischen Daten für Verkauf und Einkauf kann auf die Anzahl Monate eingeschränkt werden. Dazu geben Sie in das Feld Anz. Monate im Kopf des Dialogs einen Wert ein. Dieser Wert ist automatisch mit 12 Monaten vorbelegt. Damit kann die Zeit für das Einlesen der Daten verringert werden. 

Lagerspezifische Umsätze 

Einkaufs- und Verkaufsumsätze können optional auch auf ein bestimmtes Lager oder auf eine bestimmte Lagergruppe eingegrenzt werden. Dazu stehen im Kopf des Dialogs die Comboboxen Lager und Lagergruppe zur Verfügung. 

Beim Aufruf aus folgenden Dialogen wird diese Selektion vorbelegt: 

Bestellautomat 
Wenn Sie auf den Button Historie klicken wird das Lager aus der selektierten Bestellvorschlagsposition übergeben. 

VK-Belege bzw. EK-Belege 
Wenn Sie auf den Button Historie und klicken, wird das Lager aus der selektierten Position übergeben. 

Rahmenauftrag / Rahmenbestellung 
Wenn Sie auf den Button klicken, wird das Lager aus dem Rahmenkopf übergeben. 

Disposition 
Wenn Sie auf den Button klicken, wird das Lager aus der Registerkarte Disposition übergeben.
