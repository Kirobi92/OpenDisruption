# Kunden Lager 
Kunden Lager 

Dieser Dialog öffnet sich im Kunde >> Optionen >> Kundenlager . Ein Kunde kann einem oder mehreren Lagern zugeordnet werden. Ist für das jeweilige Lager im Lagerstamm eine Kundenprüfung aktiviert, kann der Kunde nur aus den zugeordneten Lagern bedient werden. Die Zuordnung hinterlegen Sie in diesem Dialog. 

Es gelten folgende Regeln: 

- Wenn Kundenlager definiert sind, dann beschränken diese die Auswahl des Versandlagers auf Kundenrechnungsanschrift (Registerkarte Logistik ) und Kundenlieferanschrift ( Versandanschrift ). - Wird ZUERST ein Kunden-Versandlager im Kundenstamm oder ein Versandlager in der Versandanschrift hinterlegt und DANACH ein Kundenlager definiert, dann werden die zu diesem Zeitpunkt bereits zugeordneten Lager als gültig angesehen und werden mit dem Speichern der nun manuell erfassten Kundenlager automatisch miteingefügt. - Wird nachträglich wieder ein Lager aus der Tabelle Kundenlager entfernt und existiert zu diesem Zeitpunkt bereits eine Zuordnung dieses Lagers am Kunden und/oder Versandanschrift, dann kommt es beim Speichern zu einer entsprechenden Ausnahme. 
Lesen Sie dazu noch das Thema Standard-Versandweg und Standardlager im Auftrag .

### Button Suche 

Mit diesem Button starten Sie die Suche.

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie eine neue Zeile in die Tabelle ein. Die Kundennummer wird vom Programm automatisch eingetragen. Ergänzen Sie bitte die Zeile mit der Lagernummer. Wahlweise können Sie auch noch eine spezielle Kundennummer erfassen. Dieses Feld ist in erster Linie für ein spezielles Lieferantenlager (Konsignationslager) vorgesehen. Tragen Sie dann die Kundennummer beim Lieferanten ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
