# Intrastat Position 
Intrastat Position 

Ein Doppelklick in eine Zeile der Tabelle im Dialog Intrastat öffnet den Dialog Intrastat Position . Hier werden detaillierte Informationen zu einer gewählten Position angegeben und können bearbeitet werden. Alle bisher ermittelten Informationen werden hier angezeigt. Bei einer Bearbeitung werden die Daten entsprechend in den Vorgang zurückgeschrieben. 

Ein weiterer Doppelklick öffnet den Beleg. 

Eine Intrastat Position kann aus mehreren Auftrags- und / oder Bestellpositionen bestehen. Dem Benutzer wird bei Auswahl einer Intrastat Position eine Liste der kumulierten Auftrags- und / oder Bestellpositionen angezeigt. Alle Positionen, die in der Intrastat Position verrechnet wurden, werden hier angezeigt. 

Mit einem Doppelklick in die jeweilige Auftrags- und oder Bestellposition, gelangen Sie in den jeweiligen Vorgang. 

Abholaufträge sind nicht Intrastat-relevant. Dies vor dem Hintergrund, dass Abholung keine Bewegung der Ware über die Grenze ist, sondern ein Kauf der Ware im eigenen Land.

### Feld USt-IdNr. 
Hier wird die Umsatzsteuer-Identifikationsnummer des Kunden angegeben. 

Ist die Funktion System >> Parameter Allgemein >>UST-ID via WS prüfen aktiviert, so wird die Umsatzsteuer-Identifikationsnummer bei der Eingabe in den Dialogen Kunde, Lieferant, Angebot, Auftrag, Gutschrift, Bestellung etc. per Webservice geprüft. 

Achtung: Diese Prüfung ist rechtlich nicht bindend. Eine rechtlich bindende Prüfung der Umsatzsteuer-Identifikationsnummer kann nur schriftlich bei der zuständigen Behörde erfolgen. 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Falls die USt.-IdNr. nicht ermittelt werden konnte, gelten folgende Fallbacks: 
Für Privatpersonen wird die fiktive ID QN999999999999 gesetzt. 
Für juristische Personen wird im Fall eines Dreiecksgeschäfts die ID XX999999999999 gesetzt, wobei XX durch das Länderkürzel des Empfängers ersetzt wird. 
In allen anderen Fällen (Reihengeschäft, Direktverkauf) gilt QV999999999999.

### Combobox Verkehrszweig 
Im Zusammenhang mit Intrastat ist es erforderlich, den Verkehrszweig anzugeben. Diese Nummer weist auf das grenzüberschreitende Beförderungsmittel gemäß Intrastat-Leitfaden hin. 

Der Verkehrszweig gibt an, wie die Ware geliefert wird, z.B. mit der Bahn oder dem LKW. Dafür steht die gleichnamige Spalte zur Verfügung. 

Ist diese Verkehrszweignummer nicht bekannt, wird das Einkaufsland eingetragen. Hierfür wird die Versandadresse der Ware verwendet. Ist diese nicht bekannt, dann wird die Rechnungsadresse verwendet. 

Folgende Verkehrszweige stehen zur Verfügung: 

- Seeverkehr - Eisenbahnverkehr - Straßenverkehr - Luftverkehr - Postsendungen - Rohrleitungen - Binnenschifffahrt - Eigener Antrieb 

Tooltip: 

Nummer des zugehörigen Verkehrszweigs für Intrastat
