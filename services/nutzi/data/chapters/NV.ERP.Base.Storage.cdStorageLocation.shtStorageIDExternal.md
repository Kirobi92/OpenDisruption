# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtStorageIDExternal

In dieses Feld kann eine Lagernummer oder eine Lagergruppennummer des fremden Systems eingetragen werden, mit dem verchlüsselt wird. 

Wird die Checkbox Lagergruppe aktiviert, kann hier nur eine Lagergruppennummer eingetragen werden. 

Tooltip: 

Die Lager-/Lagergruppen-Nummer des fremden Systems, 
mit dem verschlüsselt wird. 
Ist die Checkbox 'Lagergruppe' gesetzt, so handelt es sich hierbei 
um die Nummer der Lagergruppe, andernfalls um die Lagernummer auf dem fremden System.
