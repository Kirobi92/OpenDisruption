# Merkmalsklasse 
Merkmalsklasse 

Dieser Dialog öffnet sich über den Button Klassen im Dialog Artikel Merkmalsverwaltung , den Button Klasse im Artikel auf der Registerkarte Merkmale oder mit einem Doppelklick auf einen Eintrag im "Baum". Eine Klasse ist eine Gruppierung von Merkmalen. 

Hier ordnen Sie den Klassen Merkmale zu. Beispiel: Für einen Bleistift können Sie Härte, Farbe etc. als Merkmal einer Klasse zuordnen. In der rechten Tabelle können Sie einem Merkmal noch einen eindeutigen Wert zuordnen. 

Die Klassen sind hierarchisch aufgebaut. Die Struktur kann in der Artikel Merkmalsverwaltung per Drag&Drop angepasst werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Merkmalsklasse 
Tooltip: 

In diesem Dialog ordnen Sie den Klassen Merkmale zu. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Merkmale 

Dieser Button öffnet den Dialog Merkmale . 

Siehe Kapitel Merkmale . 

### Feld ID 
Hierbei handelt es sich um die interne Merkmalsklassennummer. Diese muss manuell eingegeben werden und ist nicht relevant für externe Systeme. 

Tooltip: 

Klassen ID 

### Combobox Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Feld Bezeichnung 
Hier kann eine individuelle Bezeichnung der Klasse eingegeben werden. 

### Feld Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Combobox System 
Hier wählen aus, zu welchem Merkmalsklassensystem die System ID angezeigt wird. 

Tooltip: 

Klassifikationssystem 

### Feld System ID 
Wird hier eine System ID vergeben, wird die Klasse dem System "zugeordnet". Die Zuordnung erfolgt nicht durch die Auswahl eines Systems, sondern nur durch die Vergabe einer ID zum System. 

Tooltip: 

Klassenbezeichnung aus dem Klassifikationssystem. Kann von der eNVenta-internen Bezeichnung der Klasse abweichen. 

### Feld Bildpfad 
Hier geben Sie den Bildpfad ein. 

Tooltip: 

Server-Pfad zu der Bilddatei. 

### Button Suche 

Mit diesem Button können Sie die Bilddatei suchen. 

Tooltip: 

Bild laden 

### Bezeichnungsfeld Bild 
Tooltip: 

(Optional) Kann z.B. eine Grafik beinhalten, welche zu Erklärungszwecken angezeigt wird (z.B. welches Maß einem Buchstaben entspricht etc.). Das Bild wird über einen Server-Pfad ausgewählt. 

### Abbildung Bild 
Zur besseren Erklärung kann hier ein Bild/eine Grafik eingefügt werden (z.B. welches Maß einem Buchstaben entspricht etc.). Das Bild wird über einen Server-Pfad ausgewählt. 

### Feld URL 
Hier kann z.B. ein Pfad auf eine Website eingetragen werden, die beispielsweise weitere Erklärungen anbietet. 

Tooltip: 

(Optional) kann z.B. auf eine Webseite mit weiteren Erläuterungen zu den Merkmalen verweisen.
