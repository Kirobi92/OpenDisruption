# Kapitel NV.ERP.Base.Common.cdPaymentCondCustomer.dtSpecialCashDiscountDays

Hier wird eine zusätzliche Karenzzeit gewährt (Skonto fällig + Anzahl Tage). 

Das Programm ermittelt bei Zahlung innerhalb dieser Karenz den Skonto automatisch.
