# Kapitel NV.ERP.Base.Common.cdSystemMM.shtStorageIDVirtual

Hier geben Sie ein virtuelles Lager für Umlagerungen ein zwischen eigenen bzw. IC-Lagern. Dieses Lager muss ein statisches Lager sein und wird für Zwischenbuchungen im Intercompany-Prozess genutzt. Ist der gesamte Prozess Intercompany abgeschlossen, so ist das virtuelle Lager wieder entlastet. 

Mehr zum Thema Intercompany erfahren Sie im Kapitel Intercompany .
