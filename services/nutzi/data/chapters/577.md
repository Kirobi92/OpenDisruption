# Parameter Telefon 
Parameter Telefon 

Für die Anbindung der Telefonschnittstelle können unter System >> Allgemein >> Parameter Telefon Einstellungen für den Benutzer und das Telefon hinterlegt werden. 

Diese Einstellungen sind an jedem Arbeitsplatz, der an die Telefon-Einstellungen angebunden werden soll, vom Administrator vorzunehmen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Parameter Telefon 
Tooltip: 

In diesem Dialog können Einstellungen für den Benutzer und das Telefon hinterlegt werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Feld Arbeitsplatz 
Hier steht der Name des Rechners für die Telefonschnittstelle. 

### Feld Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Combobox Line 
Hier wählen Sie die für den Benutzer gültige Telefon-Line aus. Eingestellt wird diese in den Telefon-Parametern des Betriebssystems. 

Tooltip: 

Angabe wird zur Kontaktaufnahme mit Telefonanlage benötigt. 

### Feld Amtskz. raus 
In diesem Feld können Sie die von Ihrer Telefonanlage benötigten Amtskennzahlen für ausgehende Anrufe hinterlegen. 

Tooltip: 

Amtsvorwahl für Telefonate nach außen. 
Einstellung muss mit Telefonanlage kompatibel sein. 

### Feld Länderkennzeichen 
Geben Sie hier die für Ihr Land gültige Ländervorwahl an. 

Tooltip: 

Angabe der gültigen Länderversion 

### Feld Basisrufnr. 
Damit bei einem ankommenden Anruf auch die internen Rufnummern erkannt werden können, geben Sie hier eine sogenannte „Vorrufnummer“ ein. D.h. die externe Rufnummer, ohne Länder- oder Ortsvorwahl. Im Beispiel N+V wäre das die Vorrufnummer 879. 

Tooltip: 

Basisrufnummer (Teil der Rufnummer vor der Durchwahl ohne Landes- und Ortsvorwahl) 

### Feld Amtskz. rein 
In diesem Feld können Sie die von Ihrer Telefonanlage benötigten Amtskennzahlen für eingehende Anrufe hinterlegen. 

Tooltip: 

Amtsvorwahl für eingehende Telefonate. 
Einstellung muss mit Telefonanlage kompatibel sein. 

### Feld Vorwahl 
Die hier hinterlegte Vorwahl Ihres Ortsnetzes wird bei eingehenden Anrufen geprüft und für die Anruferkennung mit verwendet. Dies ist dann relevant, wenn in eNVenta ERP Telefonnummern ohne Vorwahl eingetragen wurden. 

Tooltip: 

Angabe der eigenen Ortsvorwahl. 

### Checkbox Fokus eingehende Anrufe 
Über diese Checkbox kann jeder Sachbearbeiter für sich einstellen, ob bei einem eingehenden Anruf der Fokus automatisch auf ‚Telefon‘ gesetzt werden soll oder nicht. 

Ist das Fokussieren aktiviert, wird der Fokus direkt auf den Dialog gesetzt und Sie können sofort die Informationen zum Anrufer sehen. 

Unabhängig davon, ob der Parameter zur Fokussierung aktiv ist, erscheint der Dialog Telefon in der Navigation, auch wenn diese per Autod-Hide-Funktion gerade nicht sichtbar ist. 

Durch die Fokussierung wird der Cursor aus dem aktuellen Vorgang entfernt. Hierdurch kann auch z.B. eine automatische Suche des Artikels geöffnet werden, wenn Sie gerade eine Artikelnummer im Auftrag erfassen wollten. 

Es wird empfohlen, den Parameter nicht zu setzen. 

Tooltip: 

Fokus bei eingehenden Anrufen setzen. 

### Checkbox Nutzung WebService 
Wenn dieser Parameter gesetzt ist, erfolgt die Ermittlung des Kontaks bei eingehenden Anrufen nicht direkt über den Client, sondern über den WebService des TapierviseHost. Dies hat den Vorteil, dass bereits begonnene Eingaben nicht sofort unterbrochen werden. (Siehe Checkbox Fokus eingehende Anrufe ). 
Es wird empfohlen, den parameter zu setzen, um die Eingabe bei eingehenden Anrufen nicht zu stören. 

Voraussetzung: 
Der Tapi Webservice hat zwingend installiert zu sein, wenn der Parameter gesetzt ist! 

Tooltip: 

Nutzung des Webservice für die automatische Ermittlung Kontakte 
bei ein- und ausgehenden Anrufen. 

### Feld Länder-Vorwahl 
Hier wird die für Ihr Land gültige Ländervorwahl angegeben. 

Tooltip: 

Angabe der Länder-Vorwahl 

### Feld Timer Intervall 
Hier können Sie ein Intervall (in Sekunden) eingeben, um neue Termine in der Wiedervorlage zu prüfen. Werte zwischen 1 und 60 sind möglich. Ist hier nichts eingetragen, wird standardmäßig 30 gesetzt.
