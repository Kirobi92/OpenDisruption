# Preispflege Registerkarte Allgemein 
Preispflege Registerkarte Allgemein 

Auf dieser Registerkarte geben Sie die detaillierten Preisdaten ein. 
### Combobox Gruppe 
Hier können Sie eine Brutto-Preisgruppe auswählen, die im zugrundeliegenden Vorgang herangezogen wird und bestimmt, auf welchen Verkaufspreis zugegriffen werden soll. 

Siehe Kapitel Preisgruppen . 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

### Combobox Abs. Operator 
Hier kann die gewählte Preisgruppe der Selektion (Artikel, Artikelgruppe) um einen absoluten Wert verändert werden. In der Combobox wählen Sie die entsprechende Rechenoperation aus. 

### Feld Betrag 
Hier wird der Preis angegeben. 

Tooltip: 

Preis 

### Combobox % Operator 
Hier kann die gewählte Preisgruppe in Prozent geändert werden. In der Combobox wählen Sie die entsprechende Rechenoperation aus. 

### Feld Prozent 
Hier wird ein prozentualer Rabatt oder Zuschlag angegeben. Ein positiver Wert bedeutet Rabatt, ein negativer Wert bedeutet Zuschlag. 

Tooltip: 

Angabe eines prozentualen Rabatts / Zuschlags 

### Feld Menge 
Wenn der Radiobutten Rabatt aktiviert ist, kann in diesem Feld eine Staffelmenge eingetragen werden. Mit der Staffelmenge ist hier der Wert aus der Spalte 'ab Menge' gemeint, wie er in der Rabattverwaltung Verkauf in der Tabelle auf der Registerkarte Staffel definiert ist. 

Ein im Feld Staffel Abs. oder Staffel % angegebener Wert wird nur auf Staffeleinträge mit entsprechender "ab Menge" angewandt. 

Tooltip: 

Angabe der Menge 

### Combobox Staffel Abs Operator 
Wenn der Radiobutten Rabatt aktiviert ist, wählen Sie hier einen Operator aus. Im Feld daneben geben Sie einen Preis ein. Dieser Preis ändert den Preis im Dialog Rabatt Verkauf auf der Registerkarte Staffel bei zur Staffelmenge passender 'ab Menge' und bereits vorhandenem Preiseintrag. 

### Feld Betrag 
Hier wird der Preis angegeben. 

Tooltip: 

Preis 

### Combobox Staffel % Operator 
Wenn der Radiobutton Rabatt aktiviert ist, kann in diesem Feld ein Prozentsatz für die Staffel eingetragen werden. Dieser Prozentsatz ändert den 'Rabatt %' im Dialog Rabatt Verkauf auf der Registerkarte Staffel bei zur Staffelmenge passender 'ab Menge' und bereits vorhandenem Rabatt %. 

### Feld Prozent 
Hier wird ein prozentualer Rabatt oder Zuschlag angegeben. Ein positiver Wert bedeutet Rabatt, ein negativer Wert bedeutet Zuschlag. 

Tooltip: 

Angabe eines prozentualen Rabatts / Zuschlags 

### Feld Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer

Suche
