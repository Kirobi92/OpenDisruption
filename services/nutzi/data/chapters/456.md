# ZUGFeRD Einstellungen Registerkarte Tabelle 
ZUGFeRD Registerkarte Tabelle 

Die Registerkarte Tabelle beinhaltet immer alle Datensätze, die den vorgegebenen Suchkriterien entsprechen. Ist der Parameter Automatisch Tabelle zeigen im Dialog Parameter Allgemein aktiviert, wird die Registerkarte automatisch geöffnet, wenn es mehr als einen Suchtreffer gibt. Per Doppelklick auf den gewünschten Datensatz werden die Daten in die entsprechenden Felder des Dialogs eingelesen und die Registerkarte Allgemein aktiviert. Z.B. werden im Kundenstamm alle Kunden und entsprechende Ansprechpartner eingelesen. 

Siehe Kapitel Registerkarte Tabelle . 
### Tabelle 

### Tabellenspalte Name 
Hier geben Sie den Namen der ZUGFeRD Einstellung an. 'Default' muss gesetzt werden, um die Funktionalität zu verwenden.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Dieses Feld zeigt die Anzahl der tatsächlich eingelesenen Datensätze.
