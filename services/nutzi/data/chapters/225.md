# Artikelauswahl Registerkarte Alternativen 
Registerkarte Alternativen 

Auf dieser Registerkarte werden Alternativartikel zum gewählten Artikel angezeigt, sofern diese definiert sind. Diese Daten werden ebenfalls nur eingelesen, wenn der Benutzer auf die Registerkarte klickt. 
### Tabelle 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Hier wird die Artikelbezeichnung angezeigt. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Artikelgruppe 
Wenn der Artikel zu einer Artikelgruppe gehört, wird diese hier angezeigt. 

Tooltip: 

Angabe der Artikelgruppe

Hier wird die Menge angegeben.
