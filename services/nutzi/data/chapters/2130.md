Vertrieb / Aktivitätenstamm 

# Aktivitätenstamm Registerkarte Allgemein 
Aktivitätenstamm Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die allgemeinen Einstellungen. 
### Feld Vorlage 
Hier können Sie eine Vorlagennummer angeben. Dazu ist es notwendig, im Kopf des Dialogs zuvor einen Aktivitätstypen auszuwählen. 

Tooltip: 

Angabe der Vorlagennummer 

### Button Suche 

Mit diesem Button können Sie eine Vorlage auswählen. Es öffnet sich der Dialog Vorlagenauswahl . 

Siehe Kapitel Dokument erstellen (Vorlage auswählen) . 

Tooltip: 

Vorlagenauswahl 

### Button Verzweigen 

Dieser Button öffnet den Dialog MS Word- / OO Writer-Vorlage , wenn eine entsprechende Vorlage ausgewählt ist. 

Siehe Kapitel Word- und Office-Vorlagen . 

Tooltip: 

Vorlage Bearbeiten 

### Button Löschen 

Mit diesem Button leeren Sie das Feld. 

Tooltip: 

Vorlage Entfernen 

### Button Öffnen 

Dieser Button öffnet den Dialog Dokument Vorlagen . Hier können neue Vorlagen erstellt werden, welche dann über den Button zur Verfügung gestellt werden. 

Siehe Kapitel Dokument Vorlagenstamm . 

Tooltip: 

Dokument Vorlagen
