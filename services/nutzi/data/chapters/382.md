# Artikel Cockpit Registerkarte Logistik 
Artikel Cockpit Registerkarte Logistik 

Auf dieser Registerkarte erhalten Sie eine Sicht auf 

- alle im Pickpool enthaltenen Positionen (Bereich Pickpool ) - alle Platzzuordnungen des Artikels (Bereich Platzzuordnung ) - alle Lageraufträge zum Artikel (Bereich Lagerauftrag ) - alle LHM-Zuordnungen von Artikel und Aufträgen (Bereich LHM Zuordnung ) 
### Tabelle Pick-Pool 

### Tabellenspalte Lager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Lagerplatz 
Hier wird angegeben, auf welchem Lagerplatz die Packstücknummer zu finden ist. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Menge 
Hier wird die Menge angegeben. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein .

Angabe der Kundennummer

### Tabellenspalte Typ 
Hier wird der Typ der Lagerbuchung angegeben. Folgende Werte sind möglich: 

A = Auftrag, Gutschrift 
G = Gutschrift (obsolet, jetzt A) 
AT = Umlagerungsauftrag (Ziellager) 
X = Auftrag mit Bestellbezug (LVS-spezif.) 
V = Umbuchung auf Versandlager 
VT = Umbuchung auf Versandlager (Ziellager) 
R = VE-Retoure 
W = Bestellung, Belastung 
B = Bestellung (obsolet, jetzt W) 
N = Neutral (z.B. manuell) 
I = Inventur 

Korrekturbuchungen: 
WK = Wertkorrektur nach WE (Bestellung, Belastung) 
BK = Wertkorrektur nach WE (Wertbelastung) 
WKIE = Wertkorrektur wg. nachträgl. NK-Zuschlagung 
IK = Wertkorrektur nach Inventur 
K = Wertkorrektur duch Korrekturprogramm 
KA = Bestandskorrektur durch Auto-Korrektur
