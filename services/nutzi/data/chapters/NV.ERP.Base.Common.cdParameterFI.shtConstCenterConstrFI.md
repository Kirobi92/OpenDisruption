# Kapitel NV.ERP.Base.Common.cdParameterFI.shtConstCenterConstrFI

Der Kostenstellenzwang kann für die Finanzbuchhaltung, unabhängig von der Warenwirtschaft, aktiviert oder deaktiviert werden. 

Die OP-Übernahme ist davon ausgeschlossen, damit der OP in Zahl-, Einzugs- und Mahnvorschlag etc. zur Verfügung steht. 

Tooltip: 

Mit diesem Parameter wird der Kostenstellenzwang für die Finanzbuchhaltung aktiviert. 
Die OP-Übernahme ist davon ausgeschlossen, damit der OP in Zahl-, Einzugs- und Mahnvorschlag etc. zur Verfügung steht!
