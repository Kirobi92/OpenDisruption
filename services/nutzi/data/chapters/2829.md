# Preisoptimierung Registerkarte Automatisch 
Preisoptimierung Registerkarte Automatisch 

Im oberen Bereich der Registerkarte werden die Agenten gelistet. Mit dem -Button kann ein neuer Agent angelegt werden. Ein Doppelklick erlaubt das Ändern und Löschen eines Agenten. 

Zu jedem Agenten wird auch der saldierte Erfolg angezeigt, d.h. die Veränderungen der Erträge seit der jeweils letzten Preisänderung. Dabei wird ein Zeithorizont von maximal 12 Monaten ausgewertet. 

Mit einem Klick auf einen Agenten wird der mittlere Bereich gefüllt. Es werden die Artikel gelistet, die durch den Agenten gebucht wurden. Die aktuellsten Buchungen erscheinen zuerst. Auch hier wird jeweils der Erfolg angezeigt, sodass eine Analyse des Agentenerfolgs bis auf Artikelebene möglich wird. 

Ein Klick auf einen Artikel zeigt im unteren Bereich die graphische Darstellung der Artikelhistorie und rechts eine Liste der Preisänderungen durch diesen Programmteil. 

In der Liste der Artikel können ein oder mehrere Einträge markiert und deren weitere Bearbeitung durch den Agenten über den Button Beenden gestoppt werden. Über den Button Preisanpassung können auf die markierten Artikel Preisänderungen gebucht werden. So kann auf einfache Art und Weise manuell eingegriffen werden.
