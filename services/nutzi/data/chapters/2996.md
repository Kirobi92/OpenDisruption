# Konfiguration der Intrastat 
Konfiguration Intrastat 

Der Dialog Konfiguration Intrastat öffnet sich über System >> Allgemein >> Schnittstellen >> Konfiguration Intrastat . Hier können Sie die Einstellungen zu Intrastat vornehmen, d.h. die Konfiguration festlegen. 

Wenn Sie die Gesellschaftsverwaltung in der Intrastat nutzen möchten, ist dies generell nur möglich, wenn Sie diese generell auf der Datenbank konfiguriert und parametrisiert haben. Dazu setzen Sie im Dialog Parameter Gesellschaft die Checkbox Aktiviert . 

Siehe Kapitel Parameter Gesellschaft . 

Im Feld Kennnummer wird die Kennnummer des Melders angegeben. Sie setzt sich zusammen aus: 

- Bu/Fa des Bundeslandes des Finanzamtes (2-stellig) - Steuernummer (11-stellig, bei 10-stelliger hat eine 0 angefügt zu werden) - Unterscheidungsnummer (3-stellig) 
Das Feld für die Kennnummer hat 16 Stellen und wird an die Export-Datei übertragen. 

Die Kennnummer kann manuell angepasst werden. 

Aus der Combobox Mapping wählen Sie die Datei aus. Mit dem Button gelangen Sie in den Dialog X-Mapping Editor . 

Siehe Kapitel X-Mapping Editor . 

Standardmäßig wird ein Mapping für die Intrastat-Meldung in Deutschland mitgeliefert. Dieses ist zwingend einmalig einzuspielen und kann dann entsprechend verwendet werden. Hierzu den Button klicken und im darauffolgenden Dialog X-Mapping Auswahl über die Checkbox Aus Vorlage alle Vorgaben einlesen und die Intrastat-Definition auswählen. 

Mehrländerfähigkeit 

Hier sind mehrere Einträge möglich, allerdings darf es pro Land nur einen Eintrag geben und das Land ist zwingend anzugeben. Es empfiehlt sich die Daten in der Intrastat Firma zu pflegen, damit die Daten der Materialnummer, Kennnummer usw. entsprechend übernommen werden. 

Es erfolgt dann eine Prüfung, ob das Land für Intrastat freigegeben ist. Wenn nicht, erfolgt eine Meldung und das Speichern ist nicht möglich. Bei Update werden die bisherigen Stammdaten Intrastat für Deutschland übernommen. 

Dadurch, dass es für jedes Land einen Eintrag gibt, ist sichergestellt, dass für jedes Land ein eigenes Dateiformat für das Mapping und ein eigener Ablageort existieren.

### Button Referenz-Link

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
