# Ressourcengruppen in der Katalogverwaltung 
Ressourcengruppen 

Ressourcengruppen können in der Katalogverwaltung über den Menüpunkt Stamm >> Ressourcengruppen definiert werden. Im Standard werden bereits fünf verschiedene Ressourcengruppen ausgeliefert. 

Bei der Anlage neuer, eigener Ressourcengruppen ist unbedingt eine Gruppennummer größer 10 zu verwenden. 

Ressourcen sind so zu organisieren, dass gleichartige Ressourcen (Produktbild Groß, Produktbild Klein, Produktbild Vorne, Produktbild Hinten, Produktbild Druck, Hersteller Link, Hersteller Info, …) derselben Gruppe zugewiesen werden. Damit ist die Klassifizierung einer Ressource möglich. Ebenfalls kann eingestellt werden, ob eine Gruppe / ein Produkt mehrfach über eine Ressource einer bestimmten Gruppe verfügen kann. 

Außerdem kann festgelegt werden, welche Ressourcentypen (Datei, Referenz, Text) für eine Ressourcengruppe angeboten werden sollen und ein Standard-Ressourcenschlüssel wird hinterlegt. 

Wenn Sie die Checkbox in der Spalte ‚Mehrfach‘ aktivieren, so kann die Ressource mehrfach zugeordnet werden. 

Wenn zu einer Ressource die Checkbox in der Spalte ‚Referenz Link‘ angeklickt ist, steht Ihnen in der Katalogverwaltung auf der Registerkarte Ressourcen der Button Verlinkung zur Verfügung. Damit kann eine Verbindung von der Ressource zu einem Hersteller-Datensatz hergestellt werden. 

Siehe Katalogverwaltung Registerkarte Ressourcen . 
### Standard-Ressourcen Schlüssel 
Ist ein Standard-Ressourcenschlüssel definiert, werden alle Ressourcen dieser Gruppe mit diesem Schlüssel verschlüsselt. Dieser Schlüssel kann im Webshop für den NVResourceProvider genutzt werden, um bspw. über den Schlüssel Supplier.sWebsite durch einen Referenz Link auf einen Lieferantenstammsatz zuzugreifen und von dort auf die Eigenschaft "Website" zurückzugreifen. Damit ist eine sehr flexible Möglichkeit geschaffen worden, um Informationen überall aus dem ERP-System nachlesen zu können.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

### Tabellenspalte BMEcat Export 
Wenn Sie die Checkbox in dieser Spalte aktivieren, steht die Ressource für den BMECat Export zur Verfügung. 

### Tabellenspalte BMEcat Verwendung 
In dieser Spalte kann ein Verwendungszweck für den BMEcat eingetragen werden. Zur Verfügung stehen: 

- data_sheet -> Datenblatt - detail -> vergrößertes Bild - icon -> kleines Pictogramm - logo -> Logo zu einem Produkt oder Lieferanten - safety_data_sheet -> Sicherheitsdatenblatt - thumbnail -> Vorschaubild - others -> Sonstiger Verwendungszweck
