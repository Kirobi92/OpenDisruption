# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtNV4_StorageToResolve

Dieses Lager dient dazu, die Fälle, bei welchen eine Korrekturbuchung durchzuführen ist, auf ein sogenanntes Klärungslager zu verbuchen. Hier können dann die Verantwortlichen prüfen, woher diese Differenzen kommen. Bei diesem Lager handelt es sich um ein statisches Lager. 

Tooltip: 

Definiert das Lager für Korrekturbuchungen.
