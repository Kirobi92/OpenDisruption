# Kasseneinstellungen Registerkarte Allgemein 
Kasseneinstellungen Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie grundlegende Einstellungen für eine Kasse. 
### Feld Kunde 
Um einen Kassenauftrag zu generieren, wird die Kundennummer hinterlegt. Es empfiehlt sich, einen extra Kassenkunden im Kundenstamm zu konfigurieren. 

Tooltip: 

Hier geben Sie eine Kundennummer ein, um einen Kassenauftrag zu generieren. 

### Button Suche 

Mit diesem Button können Sie einen Kunden suchen. 

Siehe Kapitel Kundenauswahl . 

### Combobox Standardwährung 
Die hier ausgewählte Währung wird als Standardwährung für die Kasse gesehen. Die Abrechnung wird in dieser Währung erstellt. Alle Preise werden während des Kassiervorgangs in der ausgewählten Währung ausgewiesen. Wird ein Kunde einer abweichenden Währung in der Kasse angegeben, werden alle Preise in der Währung des Kunden angezeigt. 

Das Feld wird gesperrt, sobald es einen Kassenvorgang bzw. Journaleintrag gibt. 

Tooltip: 

Standard-Währung der Kasse 

### Combobox Auftragsart 
Diese Auftragsart steuert den Ablauf der Preisberechnung und Erfassung der Preise. Es werden alle Auftragsarten mit den Parametern autom. Fakturafreigabe zur Auswahl angeboten. Dies ist notwendig damit alle Rechnungen sofort kassiert werden können und als Auftrag in der Historie ersichtlich sind. 

Achtung: Wenn dem Kunden eine Auftragsart hinterlegt ist, hat diese Priorität vor der Auftragsart der Kasse. 

Es empfiehlt sich daher nicht, für den Kassenkunden eine Auftragsart im Kundenstamm zu hinterlegen; insbesondere keine Auftragsart, die von der der Kasse abweicht. 

Tooltip: 

Hier wählen Sie die Auftragsart. 

### Button Öffnen 

Mit diesem Button gelangen Sie in die Auftragsarten . 

### Combobox Lager 
Mit Hilfe der Lagernummer lässt sich das Entnahme-Lager für die Aufträge der Kasse festlegen. Diese Entnahme wird dann mit Hilfe des Auftrags automatisch verbucht. 

Tooltip: 

Definiert das Lager 

### Combobox Zahlung 
Hier wählen Sie die Zahlungsbedingung aus. 

Tooltip: 

Hier wählen Sie die Zahlungsbedingung aus. 

### Combobox Versandart 
Aus dieser Combobox wählen Sie die Versandart aus. 

Für das Kassieren von Sperrgut wählen Sie hier den Eintrag Abholung aus. Dabei sollte für die Versandart Abholung eingestellt sein, dass Sperrgut zugelassen ist. 

Der Eintrag, den Sie hier auswählen, übersteuert die Versandart, die im Kundenstamm hinterlegt ist! 

Tooltip: 

Art, wie die Sendung vom Absender zum Empfänger gelangt. 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Dialog Kassengruppe , um eine Gruppe auszuwählen. 

Siehe Kapitel Kassengruppe . 

### Combobox Kassengruppe 
Hier wählen Sie eine Kassengruppe aus. Kassengruppen werden im gleichnamigen Dialog angelegt und verwaltet. 

Die Kassengruppe wird verwendet, falls mehrere Arbeitsplätze die gleiche Kasse bedienen sollen. 

Tooltip: 

Kassengruppe auswählen 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Dialog Kassengruppe , um eine Gruppe auszuwählen. 

Siehe Kapitel Kassengruppe . 

### Checkbox Konditionen 
Ist diese Checkbox aktiviert, wird die komplette Preisfindung durchgeführt. Die Preisfindung ist in diesem Fall analog zum Verkauf. Die einzige Ausnahme sind Preise auf Projektbasis. 

Tooltip: 

Führt die komplette Preisfindung durch. 

### Checkbox Anbruch Verpackungseinheiten 
Wenn Sie diese Checkbox aktivieren, wird der Anbruch von Verpackungen beim Kassiervorgang geprüft. So können Sie selbst entscheiden, ob bei der Eingabe des Artikels der Anbruch geprüft werden soll. 

Tooltip: 

Wenn für einen Artikel Verpackungseinheiten angelegt sind, öffnet sich ein Dialog. Hier erhalten Sie einen entsprechenden Hinweis und die Angabe der nächst größeren und kleineren Menge. 

### Checkbox Rückgeld in Fremdwährung 
Ist diese Checkbox gesetzt, kann das Rückgeld in Fremdwährung ausbezahlt werden. Diese Fremdwährung kann eine beliebige Währung sein, wird aber in der Regel die Währung des Kunden sein. 

Die Checkbox kann nicht gesetzt werden, wenn der Parameter Nur in Systemwährung kassieren gesetzt ist. In diesem Fall kann in beide Richtungen nur in der Systemwährung kassiert werden. 

### Checkbox Nur in Standardwährung kassieren 
Ist diese Checkbox gesetzt, kann nur in Kassenwährung kassiert werden. 

### Checkbox Dialog Bezahlen Autom. Schließen 
Wenn Sie diese Checkbox aktivieren, schließt sich der Dialog Bezahlen , nachdem ein Vorgang abgeschlossen ist. 

Tooltip: 

Dialog Bezahlen automatisch schließen. 

### Checkbox Kassierer freigeben 
Ist diese Option aktiviert, kann der Kassierer bei der Erfassung von Waren noch manuell geändert werden. Ist sie deaktiviert, ist es immer fix der aktuell angemeldete Benutzer. 

Tooltip: 

Ist diese Option aktiviert, kann der Kassierer bei der Erfassung von Waren noch manuell geändert werden. 

### Checkbox Kasse mit Ist-Wert neu eröffnen 
Ist diese Checkbox aktiviert, wird die Kasse beim Abschluss mit dem IST-Stand neu eröffnet. Diese Option kann allerdings im Abschlussformular für den Einzelfall geändert werden. Dabei wird der aktuelle Saldo einer Abrechnung automatisch als Einlage gebucht. 

### Checkbox Gutscheine an der Kasse erstellen 
Ist dieser Parameter gesetzt, kann ein Gutschein in der Kasse erstellt und auch verkauft werden. Ist dieser Parameter nicht gesetzt, so können nur bereits existierende Gutscheine als Zahlungsmittel in der Kasse verwendet werden. 

Siehe Kapitel Gutscheinauswahl . 

Tooltip: 

Gutschein an der Kasse erstellen 

### Checkbox Drucken beim Kassieren 
Wenn Sie diese Checkbox aktivieren, erfolgt automatisch der Druck sämtlicher Gutscheine auf dem Kassenbon. 

Tooltip: 

Ausdrucken der Gutscheine beim Kassieren. 

### Combobox Nr. generieren 
Um die Nummer für Gutscheine einheitlich und auch übersichtlich zu gestalten, können Sie dies über die Auswahl aus dieser Combobox steuern. 

Folgende Einträge stehen zur Verfügung: 
- Keine Generierung - alphanumerisch - numerisch 
Je nach Einstellung erfolgt die Generierung der Gutscheinnummern im Kassiervorgang bei der Anlage eines neuen Gutscheins. 

Tooltip: 

Automatische Gutscheingenerierung an der Kasse
