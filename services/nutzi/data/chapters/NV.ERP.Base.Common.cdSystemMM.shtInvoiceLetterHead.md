# Kapitel NV.ERP.Base.Common.cdSystemMM.shtInvoiceLetterHead

Diese Checkbox steht zur Verfügung, um aus der Kommissionierung heraus den Andruck des Briefkopfs auf der Rechnung zu steuern. Dafür ist allerdings Voraussetzung, dass die Checkbox k.Rg.-Druck VS im Kundenstamm bzw. Auftrag nicht gesetzt ist! 

Wenn der Parameter gesetzt ist, wird beim Rechnungsdruck aus dem Versand heraus immer der Briefkopf auf der Rechnung gedruckt (analoge Einstellung z.B. zum Massendruck Verkauf). 

Tooltip: 

Beim Druck der Rechnung wird der Rechnungsbriefkopf mitgedruckt.
