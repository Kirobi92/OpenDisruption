# Anfrage Registerkarte Artikelinfo 
Registerkarte Artikelinfo 

Auf dieser Registerkarte erhalten Sie weitere Informationen zum Artikel, den Sie in der Tabelle markiert haben. 
### Feld Auftrag 
Hier wird die Auftragsnummer angegeben. 

Tooltip: 

Angabe der Auftragsnummer 

### Button Verzweigen

Tooltip:

### Feld Artikel 
Hier wird die Artikelnummer der Position angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Button Information

Artikelinformation 

### Button Verzweigen

### Feld Bezeichnung 
Hier wird die Artikelbezeichnung der Position angegeben. 

Tooltip: 

Artikelbezeichnung 

### Feld Letzter EK 
Hier wird der letzte Einkaufspreis angegeben. Dieser kann zum Zeitpunkt der Rechnungsprüfung geändert werden. 

Tooltip: 

Angabe des letzten Einkaufspreises 

### Feld GEK 
Hier wird der gewichtete mittlere Einkaufspreis angegeben, der bei der Rechnungsprüfung neu errechnet werden kann. Dieser wird erst bei Abschluss einer Inventur berechnet anhand der neuen Menge 'Inventur IST' und dem (bei Positionen, falls gewünscht) neuen 'Bew. EK'. 

Dieser berechnete Wert wird dann in den 'GEK, Lager' zurückgeschrieben. 

Der GEK wird wie folgt berechnet: 

(((Buchbestand – noch nicht rechnungsgeprüfter WE) x alter GEK) + (Rechnungsprüfungsmenge x neuer EK))) / (Buchbestand – noch nicht rechnungsgeprüfter WE + Rechnungsprüfungsmenge) 

Hinweis: Als Alternative zum gewichteten Einkaufspreis gibt es auch den lagerspezifischen GEK. 

Die Errechnung des GEKs findet auf der Datenbank per SQL statt. 

In diesem Zusammenhang beachten Sie bitte auch die Informationen zur Checkbox Alle Lager im Dialog Lagerorte . 

Tooltip: 

Gewichteter mittlerer Einkaufspreis 

### Feld EK pro 
Hier wird der Einkaufspreis je Anzahl Lagereinheiten angeben. Der Einkaufspreis kann sich auf 10, 100, 1000 oder eine andere Anzahl Einheiten des Artikels beziehen. Standardmäßig wird hier eine 1 eingetragen. 

Tooltip: 

Einkaufspreis je Anzahl Lagereinheiten.
