# Intrastat - Warennummer kopieren 
Warennummer kopieren 

Wenn Sie den Button im Dialog Warennummer drücken, öffnet sich dieser. 

Hier können eine Sie neue Warennummer anlegen, um z.B. ein neues Gültigkeitsdatum anzugeben – oder eben eine alte Warennummer mit neuer Gültigkeit definieren. 

Die Checkbox Zuordnungen übernehmen steht zur Verfügung, um die Zuordnung der zu kopierenden Warennummer ebenfalls zu übernehmen. Zum Beispiel die Einheit oder die Gültigkeiten. Eine bestehende Warennummer-Artikel-Zuordnung wird ebenfalls übernommen.

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten.

### Combobox Bes. Maßeinheit 
Hier wird die besondere Maßeinheit für Intrastat angegeben. Diese kann an der Warennummer hinterlegt werden. 

Die Menge einer Intrastatposition wird immer in der Maßeinheit ihrer Warennummer angegeben. Wurde keine Maßeinheit hinterlegt, so wird stattdessen die Masse der Intrastatposition in kg angegeben. 

Der Umrechnungsfaktor zur Umrechnung der besonderen Maßeinheit einer Warennummer in die Lagereinheit eines Artikels muss für jeden Artikel der Warennummer gepflegt werden.
