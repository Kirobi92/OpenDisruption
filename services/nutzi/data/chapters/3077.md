# Importprofile Registerkarte Allgemein 
Importprofilkonfiguration Registerkarte Allgemein 

Auf der linken Seite stehen alle zuvor definierten Importprofilgruppen zur Auswahl. Eigenschaften, die in keiner Gruppierung hinterlegt sind, werden in der Default-Gruppe angezeigt. Aufgrund des Einsatzes von Gruppen wird die Abarbeitung der einzelnen Eigenschaften (alphabetisch nach Eigenschaftsnamen sortiert) hinfällig. 

Die Spalten in der rechten Tabelle beziehen sich auf die zugeordneten Eigenschaften der links markierten Importprofilgruppe. 

Mit den Checkboxen in den Spalten ‚Einfügen‘, ‚Aktualisieren‘ und ‚Leer aktualisieren‘ wird folgendes festgelegt: 

Einfügen: 
Fügt neue Werte aus dem Artikel Schattenstamm in den Artikelstamm ein. 

Aktualisieren: 
Es werden Updates aus dem Artikel Schattenstamm in den Artikelstamm übernommen. 

Leer aktualisieren: 
Leere Werte aus dem Artikel Schattenstamm leeren die entsprechenden Werte im Artikelstamm. 

Für die Referenz-Objekte (Merkmale, Profilgruppen, Ressourcen etc.) gibt es eine allgemein gültige Option je Referenz. Die Rangfolge ist: Artikel > Artikelgruppe > EK-Rabattgruppe > Lieferant > Datenlieferant 

So ist eine Änderung durch den Artikel Schattenstamm untersagt, wenn der Artikel ein Update verbietet.

### Checkbox Einfügen 
Wenn Sie diese Checkbox aktivieren, wird die Eigenschaft Einfügen auf die markierten Zeilen der Tabelle übernommen. Wenn Sie die Einstellung gewählt haben, klicken Sie auf den Button Übernahme um die Ei-genschaft auf die markierten Zeilen zu übernehmen. Sollte keine Zeile in der Tabelle markiert sein, so werden die Eigenschaften auf alle Zeilen übernommen. 

Tooltip: 

Das Einfügen von neuen Werten ist erlaubt, wenn das Ziel leer ist. 

### Checkbox Aktualisieren 
Wenn Sie diese Checkbox aktivieren, wird die Eigenschaft Aktualisieren auf die markierten Zeilen der Tabelle übernommen. Wenn Sie die Einstellung gewählt haben, klicken Sie auf den Button Übernahme um die Ei-genschaft auf die markierten Zeilen zu übernehmen. Sollte keine Zeile in der Tabelle markiert sein, so werden die Eigenschaften auf alle Zeilen übernommen. 

Tooltip: 

Erlaubt das Aktualisieren von Feldern mit neuen Werten. 

### Checkbox Leer aktualisieren 
Wenn Sie diese Checkbox aktivieren, wird die Eigenschaft Leer aktualisieren auf die markierten Zeilen der Tabelle übernommen. Wenn Sie die Einstellung gewählt haben, klicken Sie auf den Button Übernahme um die Eigenschaft auf die markierten Zeilen zu übernehmen. Sollte keine Zeile in der Tabelle markiert sein, so werden die Eigenschaften auf alle Zeilen übernommen. 

Tooltip: 

Erlaubt das Aktualisieren von Feldern mit leeren Werten. 

### Button Übernahme 

Nach markieren einer Eigenschaft, übernehmen Sie diese mit diesem Button auf die markierten Zeilen. Sollte keine Zeile in der Tabelle markiert sein, so werden die Eigenschaften auf alle Zeilen übernommen.

dem über der Tabelle gewählten Wert.
