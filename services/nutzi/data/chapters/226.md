# Artikelauswahl Registerkarte Zubehör 
Registerkarte Zubehör 

Diese Daten werden ebenfalls nur eingelesen, wenn der Benutzer auf die Registerkarte klickt. Gezeigt wird eine weitere Tabelle (analog der oberen) mit Zubehörartikeln. Hier kann ebenfalls die Menge eingegeben und ausgewählt werden. (Eine kombinierte Auswahl ist in allen Tabellen möglich!) 
### Tabelle 

### Tabellenspalte Zubehörartikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung

Hier wird der Zubehörtyp angegeben.
