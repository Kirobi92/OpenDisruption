# Artikelplatz 
Artikelplatz 

Dieser Dialog öffnet sich im Dialog Lagerplatz auf der Registerkarte Lagerplatz über den Button Artikel Z. In diesem Dialog wird hinterlegt, für welche Artikel dieser Fixplatz ein Artikelfixplatz ist. 

Im Kopf des Dialogs sind die Felder Lagernr , Zone und Lagerplatz (aus dem Dialog Lagerplatz ) voreingestellt.

### Tabellenspalte Min. Menge 
Hier wird die minimale Menge des Artikels auf diesem Lagerplatz angezeigt. Ist diese Mindestmenge erreicht, muss eine Umlagerung durchgeführt werden. Vorausgesetzt, das Nachschubsystem für diesen Platz ist entsprechend eingestellt. 

Tooltip: 

Zeigt den Mindest-Lagerbestand. 

### Tabellenspalte Nachschubsystem 
Das Nachschubsystem steuert den Zeitpunkt, wann ein Lagerplatz zur Nachbestückung vorgeschlagen wird. Der Lagerplatz wird beim Nachschubsystem komplett leer vorgeschlagen, wenn der Lagerplatz komplett leer ist. Der Lagerplatz wird beim Nachschubsystem 2 prophylaktisch vorgeschlagen, wenn der Buchbestand die Mindestmenge am Lagerplatz erreicht oder unterschritten hat. 

- komplett Leer: Umlagerung wird initialisiert, wenn der Lagerplatz komplett leer ist. 
- prophylaktisch: Umlagerung wird initialisiert, wenn der Mindestbestand erreicht wurde. 

Tooltip: 

Nachschubsystem für automatische Nachbestückung: 
- komplett leer: Umlagerung wird initialisiert, wenn der Lagerplatz komplett leer ist. 
- prophylaktisch: Umlagerung wird initialisiert, wenn der Mindestbestand erreicht wurde.

Tooltip:
