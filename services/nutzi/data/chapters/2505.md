# Auswertungsstruktur Transfer 
Auswertungsstruktur Transfer 

Dieser Dialog öffnet sich in den Dialogen Auswertungsstruktur und Auswertungsstruktur Kore über Optionen >> Import bzw. Export . Hier kann eine Auswertungsstruktur im XML-Format importiert werden. Beim Import von Basiswer-ten (Konten, Steuerschlüssel, Kostenstellen) kann zwischen „keinen“, „allen“ und „nur solchen, die auf der Zieldatenbank existieren“ gewählt werden. 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
