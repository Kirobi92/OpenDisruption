# Auswertungsstruktur Registerkarte Bilanz

# Auswertungsstruktur Registerkarte Bilanz 
Auswertungsstruktur Registerkarte Bilanz 

Auf dieser Registerkarte definieren Sie die Struktur für eine Bilanz. 
### Combobox Auswertung 
Als erstes wählen Sie hier die Bilanz aus, die Sie bearbeiten möchten. 

### Combobox Rechn.leg.kreis 
Hier kann der passende Rechnungslegungskreis ausgewählt werden. 

Tooltip: 

Angabe des Rechnungslegungskreises 

### Feld Zeilentext 
In diesem Feld legen Sie den Namen für eine Stufe, einen Text oder ein Detail fest. 

Tooltip: 

Zeilentext 

### Feld Summentext 
In diesem Feld können individuelle Summentexte eingegeben werden. Wird hier nichts eingetragen, steht in der Auswertung (im Dialog und im Report) kein Text beim Ergebnis. 

Tooltip: 

Der Summentext wird in der Auswertung in Spalte Bezeichnung auf der Summenzeile ausgewiesen. 

### Button Hinzufügen 

Tooltip: 

Automatisches Füllen des Feldes Summentext. 

### Feld 100% Basis 
In diesem Feld haben Sie die Möglichkeit bis zu 5 Stufen (1 bis 5) als Basen festzulegen. In der Saldenauswertung ( Buchen >> Auswertungen >> Sachkonten ) erzeugen Sie hiermit je Stufe 1-5 Vergleichsspalten. In jeder der Vergleichsspalten werden alle Auswertungsstufen auf die jeweilige 100%-Basis berechnet. 

### Feld 100% Basistext 
Hier wird der Text der gewählten Stufe angegeben. 

Tooltip: 

Basis festlegen 

### Button Suche 

Wenn Sie diesen Button drücken, wird immer die nächste Zeile mit einer Basis in der Auswertungs-Struktur gesucht. Ist die Suche unten angekommen, wird von oben gesucht. 

Tooltip: 

Zur nächsten 100% Basis springen 

### Combobox Anzeigemodus 
Aus dieser Combobox wählen Sie die Art der Anzeige aus. 

Bitte beachten: Die Auswahl Nur Summe wirkt sich erst auf dem Report Bilanz oder GuV aus. Das heißt, in der Dialog-Tabelle werden die Basis-Zuordnungen (Konten mit Salden) trotzdem ausgewiesen, damit die Summe nachvollziehbar ist. 

Beispiel 1: Anzeigemodus Auswahl null = Der Report wird mit Zeilentext (Überschrift), mit Basis-Zuordnungen und mit Summentext erstellt. 

Beispiel 2: Anzeigemodus Auswahl Ohne Zeilentext = Der Report wird ohne Zeilentext (Überschrift), mit Basis-Zuordnungen und mit Summentext erstellt. 

Beispiel 3: Anzeigemodus Auswahl Nur Summe = Der Report wird ohne Zeilentext (Überschrift), ohne Basis-Zuordnungen, nur mit Summentext erstellt. 

### Checkbox MIS Auswertung 
Wenn Sie diese Checkbox aktivieren, können Sie den einzelnen Auswertungsstufen Kennziffern für das MIS zuordnen. Dazu steht Ihnen die Combobox MIS Referenz zur Verfügung. 

Tooltip: 

Angabe der MIS-Auswertung 

### Button Öffnen 

Dieser Button öffnet die MIS Referenz Texte. 

Siehe Kapitel MIS Referenz Texte . 

### Dock Panel Zeilentyp 

Diese Radiobuttons bestimmen die weiteren Eigenschaften der einzelnen Ebenen der Auswertungsstruktur. Je nach gewählter Eigenschaft haben Sie unterschiedliche Möglichkeiten. 
### Radiobutton Stufe 
Innerhalb einer Stufe werden alle Werte der Konten in der Auswertung zu einem Ergebnis zusammengerechnet. 

Sie können beliebig viele Stufen ineinander verschachteln. 

### Radiobutton Text 
Diese Positionen im Strukturaufbau sind reine Texte und dienen dem Layout der Auswertung. 

### Radiobutton Detail 
Nur einem Detail können Konten zugeordnet werden! 

D.h. wenn Sie einer Stufe Konten zuordnen wollen, müssen Sie zuvor innerhalb der Stufe ein Detail einfügen. 

### Radiobutton Basis 
Die Basis bilden die Konten. Anhand ihrer Werte erfolgen die Berechnungen innerhalb einer Stufe. 

### Combobox MIS Referenz 
Hier können Sie einen MIS Referenz Text auswählen. 

Siehe Kapitel MIS Referenz Text . 

Tooltip: 

MIS Referenznummer 

### Tabelle 

In dieser Tabelle werden alle Konten angezeigt, die im Sachkontenstamm auf der Registerkarte Abschluss als Bilanz-Konto gekennzeichnet sind. Alle Konten, die noch in der Tabelle stehen, sind in der Auswertungsstruktur nicht zugeordnet. Die Tabelle ist numerisch sortiert, kann jedoch auch durch Klick auf den Spaltennamen nach den weiteren Spalten sortiert werden. 
### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Checkbox Check Zuordnungen 
Wenn Sie diese Checkbox aktivieren, wird beim Druck der Bilanz (Finanzbuchhaltung, Buchen, Abschluss, Auswertung) auf vollständige Zuordnung aller Konten geprüft. Sie werden mit einer entsprechenden Meldung darauf hingewiesen. 

Über falsch zugeordnete Konten erhalten Sie eine Auflistung. Ist das Sachkonto in einer Auswertung enthalten und Sie ändern die Zuordnung, erhalten Sie ebenfalls einen Hinweis. 

Beim Druck einer Auswertung, z.B. Bilanz, erscheint eine Meldung, welche auf die fehlenden Zuordnungen von Sachkonten hinweist. Sie können hier entscheiden, ob Sie die Fehler ignorieren möchten oder den gesamten Vorgang abbrechen möchten. 

Tooltip: 

Prüfe auf vollständige Zuordnung aller Konten 

### Feld Konto 
In diesem Feld können Sie eine Kontonummer eingeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Mit diesem Button sucht eNVenta ERP das eingegebene Konto im Strukturaufbau, in der Kontentabelle und markiert es. Ist die Platzierung außerhalb des Anzeigebereichs, muss manuell hin geblättert/gescrollt werden. 

Tooltip: 

Suche Konto 

### Dock Panel Zuordnung 

Im Bereich der Zuordnung werden Aktiva und Passiva festgelegt und somit die Zuweisung zur Bilanzseite Aktiva bzw. Passiva . 

Auf der Ebene Detail kann durch Aktivierung des Radiobuttons Umkehrung bei einem Detail eine Zuordnung auf der anderen Bilanzseite vorgenommen werden. Diese Ebene muss zuvor auf der anderen Bilanzseite angelegt werden, dann kann sie im grauen Kasten ausgewählt werden. Je nach Kontensaldo – Aktiva im Haben, Passiva im Soll – wird das Konto dann in der Bilanz auf der anderen Bilanzseite ausgewiesen: Aktivakonto auf der Bilanzseite Passiva , Passivakonto auf der Bilanzseite Aktiva . 

Wird zusätzlich die Checkbox gesamtes Detail aktiviert, wird bei negativem Saldo der Summe der Einzelkonten der ausgewählten Ebene Detail , das gesamte Detail der anderen Bilanzseite zugeordnet und nicht nur die Einzelkonten mit umgekehrten Salden. 

Über den Radiobutton Ergebnis deklarieren Sie hier den Jahresüberschuss aus GuV auf einem Detail. 
### Radiobutton Aktiva 
Wenn Sie diesen Radiobutton aktivieren, werden Aktiva festgelegt. 

### Radiobutton Passiva 
Wenn Sie diesen Radiobutton aktivieren, werden Passiva festgelegt. 

### Radiobutton Umkehrung 
Wenn Sie diesen Radiobutton aktivieren, wird bei einem Detail die Zuordnung auf der anderen Bilanzseite vorgenommen. 

### Radiobutton Ergebnis 
Wenn Sie diesen Radiobutton aktivieren, deklarieren Sie den Jahresüberschuss aus GuV auf einem Detail. 

### Checkbox Gesamtes Detail 
Wenn Sie diese Checkbox aktivieren, wird bei negativem Saldo der Summe der Einzelkonten der ausgewählten Ebene Detail das gesamte Detail der anderen Bilanzseite zugeordnet und nicht nur die Einzelkonten mit umgekehrten Salden.
