# Neuplanung 
Neuplanung 

Über Plantafel >> Optionen >> Neuberechnung bzw. Planung Betriebsaufträge >> Button Terminierung erreichen Sie den gleichnamigen Dialog. 

Hier können markierte Betriebsaufträge neu eingeplant werden. Offene und bereits eingeplante Betriebsaufträge können neu terminiert werden. 

Vorgehensweise bei der Neuplanung: 

- Alle betroffenen Betriebsaufträge ab einem vorgegebenen Starttermin bis zu einem optionalen Endetermin werden geparkt und in eine Liste eingetragen. - Jedem Listeneintrag wird ein Prioritätenschlüssel zugewiesen. - Die Liste wird sortiert. - Es erfolgt eine Neueinplanung in Reihenfolge der sortierten Liste, beginnend mit dem höchsten Prioritätenschlüssel ab dem vorgegebenen Starttermin. 
Prioritätenschlüssel bei der Neuplanungsmethode „Nach Termin“: 

- Fixtermin (wenn vorhanden, sonst. Max. Termin) - Priorität - Alter Starttermin - Artikelnummer Produkt 
Bei der Priorität „Nach Artikel“ werden alle gleichen Artikel zu einem Fixtermin eingeplant, anschließend der nächste Artikel, usw. 

Es wird grundsätzlich vorwärts terminiert, Simulationen werden nicht berücksichtigt. 

Bei eventuellen Fehlern werden Sie mit einer Meldung darauf hingewiesen.

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Neuplanen 

Mit diesem Button lösen Sie die Neuplanung der Betriebsaufträge aus. Dabei wird eine bestimmte Sortierreihenfolge eingehalten, die im Betriebsauftrag (Registerkarte Kopfdaten ) im Feld Priorität definiert werden kann. Außerdem spielt der Eintrag im Feld Fixer Termin eine Rolle. 

Ist im Betriebsauftrag keine Priorität definiert, zieht die Einstellung aus den Parametern der Produktion (Feld Priorität Standard ). 

Siehe Kapitel Parameter Produktion .

Dieses Feld steht zur individuellen Eingabe zur Verfügung. 

Hier können Daten erfasst werden, für die kein spezielles Feld vorgesehen ist. Dieses Feld kann ebenfalls zur Artikelauswahl (Suche) verwendet werden. Jedes Feld kann maximal 80 Zeichen enthalten.
