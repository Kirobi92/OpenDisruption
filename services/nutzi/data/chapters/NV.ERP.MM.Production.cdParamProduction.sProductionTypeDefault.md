# Kapitel NV.ERP.MM.Production.cdParamProduction.sProductionTypeDefault

Falls nichts anderes gewählt wird, können Sie bereits hier eine Standard-Produktionsart auswählen. 

Die angegebene Produktionsart wird verwendet, wenn keine sonstige Produktionsart, z.B. bei der Generierung aus der Stückliste heraus, angegeben wurde. Zur Verfügung stehen die Einträge, welche im Dialog Produktionsarten angelegt wurden.
