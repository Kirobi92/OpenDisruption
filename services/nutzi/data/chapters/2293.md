# Webshop Registerkarte Übersicht 
Webshop Registerkarte Übersicht 

Die Informationen dieser Registerkarte sind nochmals aufgeteilt auf die Registerkarten Aktive Sitzungen und Benutzerübersicht . Im oberen Bereich der Registerkarte ist es möglich, den Webshop anhand der ID auszuwählen. Für den ausgewählten Shop werden Ihnen anschließend in den verschiedenen Ansichten die Daten angezeigt. 

Außerdem ist ersichtlich, wie viele Benutzer im Moment auf den Shop zugreifen. Der Wert neben der Webshop ID gibt die aktiven Benutzer aus. Die Zahl gibt den Wert der User aus, die sich gegenwärtig im Shop befinden, unabhängig ob diese angemeldet sind oder nicht. 

Die Registerkarte steht u.a. für Auswertungen zur Verfügung; wann, welcher Benutzer im Shop online war und welches Produkt für ihn interessant war. 
### Combobox Webshop 
Zur Selektion der Übersicht können Sie hier die ID eines Webshops auswählen.
