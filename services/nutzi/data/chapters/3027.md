# Satzbeschreibung für den Lohnimport von Datev 10 und 12 

### Schnittstellen / Dateibeschreibung 
Datei mit 10 Spalten 
| Lohnprogramm | Datev |
| --- | --- |
| Version | |
| Stand | 17.07.2009 |

#### 1. Dateibeschreibung 
a. Typ: CSV-Datei, Semikolon-getrennt 
b. Feldbeschreibung 

Nr. Spalte Inhalt Beispiel 
| | | |
| --- | --- | --- |
| 1 | Betrag (Vorsicht mit 1000er Trennung!) | 2.500,00 |
| 2 | Soll-/Haben-Kennung | H |
| 3 | Sachkonto (Gegenkonto) | 6010 |
| 4 | Buchungsperiode (JJJJMM) | 200906 |
| 5 | Hilfstext | Null |
| 6 | Buchungsperiode 2 (ohne Bedeutung) | 2906 |
| 7 | Sachkonto (Hauptkonto; d.h. Bezug zu Soll- / Haben-Kennung) | 3790 |
| 8 | Kostenstelle | 300 |
| 9 | Kostenstelle / Kostenträger | |
| 10 | Buchungstext | Gehalt (oder auch Null) |

#### 2. Übernahmeanweisung / -matrix 
Spalte 'Ziel-Tabelle' Spalte 'Quelldatei' Anweisungen 
| | | |
| --- | --- | --- |
| Buchungsdatum | laut Vorgabe-Dialog | |
| Belegdatum | 4. Buchungsperiode | in Monats-Letzter umwandeln |
| Belegnummer | laut Vorgabe-Dialog | |
| Konto Haben | 3. Gegenkonto oder 7. Hauptkonto | Wenn SollHaben = S dann Gegenkonto, sonst Hauptkonto |
| Konto Soll | 3. Gegenkonto oder 7. Hauptkonto | Wenn SollHaben = S dann Hauptkonto, sonst Gegenkonto |
| Kostenstelle | 8. Kostenstelle | |
| Kostenträger | 9. Kostenstelle 2 | Wenn keine Kostenträger vorhanden, dann ignorieren. |
| Betrag | 1. Betrag | Betrag |
| Buchungstext | laut Vorgabe-Dialog oder 12. Buchungstext | in allgem. Funktion gesteuert |
| Steuerschlüssel | in allgem. Funktion gesteuert | |
| NLNR | In allgem. Funktion gesteuert | |

#### Sonstiges: 
Der Anwender muss die Datei zuvor soweit überarbeiten, dass keine leeren bzw. ungültige Zeilen vorhanden sind.

| Lohnprogramm | Datev |

| Version | |

#### 1. Dateibeschreibung 
a. Typ: CSV-Datei, Semikolon-getrennt 
b. Feldbeschreibung 

Nr. Spalte Inhalt Beispiel 
| | | |
| --- | --- | --- |
| 1 | Betrag ohne Komma und mit Soll-/Haben-Kennung | 43724 S |
| 2 | Leer | Null |
| 3 | Sachkonto (Gegenkonto) | 4120 |
| 4 | Buchungsperiode (JJJJMM) | 200906 |
| 5 | Hilfstext | Null |
| 6 | Buchungsperiode 2 (ohne Bedeutung) | 2906 |
| 7 | Sachkonto (Hauptkonto; d.h. Bezug zu Soll-/Haben-Kennung | 1755 |
| 8 | Kostenstelle | 300 |
| 9 | Kostenstelle 2 / Kostenträger | 99999999999 |
| 10 | Leer | Null |
| 11 | Leer | Null |
| 12 | Buchungstext | Gehalt (oder auch Null) |

#### 2. Übernahmeanweisung / -matrix 
Spalte 'Ziel-Tabelle' Spalte 'Quell-Datei' Anweisungen 
| | | |
| --- | --- | --- |
| Buchungsdatum | laut Vorgabe-Dialog | |
| Belegdatum | 4. Buchungsperiode | in Monats-Letzter umwandeln |
| Belegnummer | laut Vorgabe-Dialog | |
| Konto Haben | 3. Gegenkonto oder 7. Hauptkonto | Wenn SollHaben = S, dann Gegenkonto, sonst Hauptkonto |
| Konto Soll | 3. Gegenkonto oder 7. Hauptkonto | Wenn SollHaben = S, dann Hauptkonto, sonst Gegenkonto |
| Kostenstelle | 8. Kostenstelle | |
| Kostenträger | 9. Kostenstelle 2 | Wenn keine Kostenträger vorhanden sind, dann ignorieren. |
| Betrag | 1. Betrag | Betrag muss separiert werden. Diesen durch 100 teilen. |
| Buchungstext | laut Vorgabe im Dialog oder 12. Buchungstext | in allgem. Funktion gesteuert |
| Steuerschlüssel | in allgem. Funktion gesteuert | |
| NLNR | in allgem. Funktion gesteuert | |

#### Sonstiges: 
Die 1. Spalte muss entsprechend separiert werden.
