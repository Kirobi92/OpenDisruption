# Kundeninformation Registerkarte Kontakt 
Kundeninformation Registerkarte Kontakt 

Auf dieser Registerkarte werden alle wichtigen Kontaktdaten des Kunden angezeigt. 
### Feld Firma 1 
Die Zeile 1 des Firmennamens wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Firma 2 
Die Zeile 2 des Firmennamens wird angezeigt. 

Tooltip: 

Zeile 2 des Firmennamens. 

### Feld Firma 3 
Die Zeile 3 des Firmennamens wird angezeigt. 

Tooltip: 

Zeile 3 des Firmennamens. 

### Feld Straße 
Die Straße zur Firma wird angezeigt. 

Tooltip: 

Angabe des Straßennamens 

### Feld PLZ / Ort 
Die Postleitzahl des Orts wird angezeigt. 

Tooltip: 

Postleitzahl des Datensatzes 

### Feld Ort 
Der Ort wird angezeigt. 

Tooltip: 

Angabe des Orts 

### Feld PLZ / Ort / PF 
Die Postleitzahl zum Postfach der Adresse wird angezeigt. 

Tooltip: 

Postleitzahl, Ort und Nummer des Postfaches 

### Feld Ort (PF) 
Hier wird der Ort angegeben. 

Tooltip: 

Ort des Postfachs 

### Feld Postfach 
Die Adresse des Postfachs wird angezeigt. 

### Feld Land 
Das Land zum Firmensitz wird angezeigt. 

Tooltip: 

Hier wird das Land des Firmensitzes angegeben. 

### Feld GEO-Koordinaten 
Hier werden geografische Daten zur besseren Orientierung angegeben. Damit lassen sich Besuche o.ä. besser planen und Sie können die Karte zur besseren Orientierung und Festlegung einer Besuchs-Reihenfolge nutzen. 

Das Format der GEO-Koordinaten lautet: 
XX.yyyyyy,AA.bbbbbb 

Beispiel für Stockach, Nissen&Velten: 
47.848029,09.005334 

Tooltip: 

Format der GEO-Koordinaten lautet: 
XX.yyyyyy,AA.bbbbbb 

Beispiel für Stockach, Nissen&Velten: 
47.848029,09.005334 

### Feld Telefon 
Die hinterlegte Telefonnummer des Kunden wird angezeigt.

Angabe der Telefonnummer 

### Feld Fax

Angabe der Fax-Nummer 

### Feld E-Mail 
Hier wird die E-Mail-Adresse angezeigt. 

Tooltip: 

Angabe der E-Mail-Adresse.

### Feld Webseite 
Der Link zur Website wird angezeigt. 

Tooltip: 

Angabe des Links zur Website 

### Feld Code 1

### Feld Code 2

### Feld Code 3 
Der Text, den Sie im zugrundeliegenden Vorgang in das Code-Feld eingetragen haben, wird angezeigt. 

### Feld Registergericht

Tooltip: 

Angabe des Registergerichts 

### Feld Firmenregister 
Hier wird das Firmenregister angezeigt.

Angabe des Firmenregisters 

### Combobox Rechtsform 
Hier wird die Rechtsform des Unternehmens angezeigt, wie z.B. GmbH oder AG. 

Tooltip: 

Angabe der Unternehmen-Rechtsform 

### Feld ABC 
Hier wird die Kategorisierung aus der ABC-Analyse angezeigt.

### Feld Vertreter

Tooltip:

### Feld Zuständig

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Zahlung 
Hier wird die Zahlungsbedingung angezeigt. 

Tooltip: 

Text der Zahlungsbedingung 

### Feld Versandart 
Hier wird die Versandart angezeigt. 

Tooltip: 

Geben Sie eine Beschreibung der Versandart ein. 

### Feld Lieferung 
Hier wird die Lieferbedingung angezeigt. 

Tooltip: 

Angabe der Vereinbarung zur Lieferung. 
Steuert unter anderem die Angabe von Portokosten oder Franko-Lieferung. 

### Feld Auftragsart 
Hier wird die Auftragsart angezeigt. 

Tooltip: 

Zeigt den Namen der gewählten Auftragsart aus der Tabelle.
