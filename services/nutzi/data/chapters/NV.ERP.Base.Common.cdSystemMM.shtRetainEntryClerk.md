# Kapitel NV.ERP.Base.Common.cdSystemMM.shtRetainEntryClerk

Wenn Sie diese Checkbox aktivieren, wird der Erfasser aus dem Originalbeleg in die Kopie übernommen (z.B. wenn ein Angebot in einen Auftrag kopiert wird). 

Diese Einstellung ist wichtig für die korrekte Zuordnung der Kostenstellen, wenn beispielsweise aus dem Angebot ein Auftrag wird. 

Ist diese Checkbox bereits im Dialog Parameter Verkauf Registerkarte Allgemein gesetzt, ist sie im Dialog Beleg kopieren bereits vorbelegt. 

Siehe Kapitel Kopieren eines Belegs im Verkauf . 

Siehe Kapitel Parameter Verkauf . 

Tooltip: 

Falls gesetzt, wird der Erfasser aus dem Originalbeleg in die Kopie übernommen. 
Eine globale Version dieses Parameters lässt sich auf dem Dialog "Parameter Verkauf" finden.
