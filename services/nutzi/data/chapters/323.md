Allgemeine Funktionalitäten in eNVenta ERP / Datentransfer (Import und Export von Daten) 

# Server Datei Manager 
Server Datei Manager 

Unter System >> Allgemein >> Dateiverwaltung gibt es den Eintrag Server Datei Manager . Hier können verschiedene Dateien auf dem Broker verwaltet werden. 

Hier wird der aktuellste Dialog des Betriebssystems des Anwenders für die Dateiauswahl verwendet. Auf diese Weise werden die aktuellsten Funktionen des Betriebssystems genutzt und eine eigene Verwaltung der Funktion in eNVenta / FS ist nicht notwendig. 

Achtung: Einige Betriebssysteme erzeugen die Datei ohne Dateiendung, z.B. xml, wenn im Explorer die Option „Dateinameerweiterung anzeigen“ nicht aktiv ist. 

Der obere Bereich des Dialogs zeigt das Quellverzeichnis an und der untere den Zielbereich. Die Daten können zwischen den beiden Bereichen verschoben bzw. kopiert werden (es können auch mehrere Dateien gleichzeitig verschoben oder kopiert werden). 

Verzeichnisse mit Unterverzeichnissen können ebenfalls verschoben werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Server-Datei-Manager 
Tooltip: 

In diesem Dialog können verschiedene Dateien auf dem Broker verwaltet werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
