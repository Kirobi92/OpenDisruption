# Kundenauswahl 
Kundenauswahl 

Dieser Dialog öffnet sich beispielsweise im Auftrag über den -Button neben dem Feld Kunde . Er bietet die Möglichkeit, mit verschiedenen Kriterien einen Kunden zu suchen. Im Standard erfolgt die Suche auf die Rechnungsanschrift. 

Sie können eines oder mehrere der oberen Felder ausfüllen. Nach Groß- und Kleinschreibung wird nicht unterschieden. 

Um den Suchvorgang zu starten, betätigen Sie den Button . Haben Sie keines der Felder vorbelegt, wird die Suche über die kompletten Stammdaten ausgeführt. 

Eine generelle Suche kann zu langen Wartezeiten führen.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter .

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 
Im Fall von gelöschten Einmalkunden werden diese Kunden mit diesem Button wiederhergestellt (dazu sind zuvor dien entsprechenden Kunden in der Tabelle zu markieren).

Dieser Button schließt den Dialog.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
