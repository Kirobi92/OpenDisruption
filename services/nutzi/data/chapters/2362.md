# Speichern von Selektionen /(Einstellungen) 
Der Dialog Einstellungen steht auch in nachfolgenden Dialogen der Finanzbuchhaltung über den Button Einstellungen zur Verfügung: 
- Liquiditätsübersicht - Umsatzbetrachtung - Saldenlisten - Kontoblätter (die Wahl der Registerkarte stellt in diesem Fall bereits die Selektion dar!) - OP Debitor - OP Kreditor - OP Sachkonten - OP Altersgliederung Debitor - OP Altersgliederung Kreditor 
Damit können die Einstellungen des aufgerufenen Dialogs gespeichert und wieder zurückgegeben werden. 

Stichtage werden nicht in die Einstellungen gespeichert, z.B. in den Dialogen Liquiditätsübersicht und Altersgliederung OP . 

Mit Doppelklick in die obere Tabelle wählen Sie eine Einstellung aus. Die Parameter dazu werden in die untere Tabelle eingelesen. Sie können übernommene Daten des aufrufenden Dialogs übernehmen, indem Sie auf den Button klicken und im Feld Bezeichnung einen Namen erfassen. 

Siehe Kapitel Einstellungen .
