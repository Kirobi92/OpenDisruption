# Factoring OP Typen 
Factoring OP Typen 

Im Menü Stamm bzw. über den Button OP Typen in den Factoring Parametern öffnet sich der Dialog Factoring OP Typen . In diesem Dialog verwalten Sie zu den Factoring Banken die entsprechenden Factoring-Typen. Diese werden von den einzelnen Banken vorgegeben. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Factoring OP Typen 
Tooltip: 

In diesem Dialog verwalten Sie zu den Factoringgebern die entsprechenden Factoring-Typen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Tabelle 

### Tabellenspalte Factoringgeber 
Aus der Combobox in dieser Spalte wählen Sie den Namen der Factoring Bank aus. 

Tooltip: 

Angabe des Factoringgebers 

### Tabellenspalte Aktiv 
Wenn Sie diese Checkbox aktivieren, ist die Bank aktiv. 

Tooltip: 

Aktiviert das Factoring 

### Tabellenspalte Factoring-Typ 
Hier geben Sie eine Abkürzung für den Factoring-Typ ein, welcher von der gewählten Bank vorgegeben wird. 

### Tabellenspalte Bez. Factoring-Typ 
Hier geben Sie den Namen des Factoring-Typs ein. 

### Tabellenspalte Nicht übermitteln 
Wenn Sie die Checkbox in dieser Spalte aktivieren, werden diese Daten nicht übermittelt, d.h. sie werden im Dialog Factoring auch nicht eingelesen. 

### Tabellenspalte Typ 
In dieser Combobox wählen Sie den Typ aus, zur Verfügung stehen Rechnung und Gutschrift . 

Diese Typen spielen eine wichtige Rolle im Debitoren Factoring in Bezug auf die Registerkarte Neue OPs und werden beim Export zur Identifizierung benötigt (für den Fall, dass diese Typen getrennt exportiert werden sollen). 

Dieses Feld wird zum Pflichtfeld, wenn die Checkbox Nicht Übermitteln nicht gesetzt ist! 

### Tabellenspalte Textm.-code 
Wenn Sie die Checkbox in dieser Spalte aktivieren, handelt es um einen Textmarkierungscode oder, wenn diese Checkbox nicht aktiviert ist, um einen Finanzmarkierungscode. 

Tooltip: 

Commerz Factoring: OP-Typ ist Textmarkierungscode
