# CSC - Übersicht über aktuelle Einstellungen

Wissenswertes über eNVenta ERP / Client Side Customizing (CSC) 

# CSC - Übersicht über aktuelle Einstellungen 
Mit der Tastenkombination <Strg + Umschalt + F12> können Sie in jedem eNVenta -Dialog, der Client Side Customizing unterstützt, einen Dialog öffnen, der eine Übersicht anzeigt. In dieser Übersicht werden alle Einstellungen aufgelistet, die zum aktuellen Zeitpunkt durch das Client Side Customizing gesetzt wurden. 

Hier können Einträge ebenfalls gefiltert werden. 

In der letzten Zeile des Dialogs wird der Form-Status angezeigt. Folgende Status stehen zur Verfügung: 

- Assigning default values 
Standardwerte dürfen vorbelegt werden (z.B. bei neuen Objekten) - Ignoring default values 
Standardwerte werden nicht vorbelegt (z.B. bei bereits gespeicherten Objekten oder nach der Suche) - Not initialised, default values disabled 
Das CSC wurde nicht initialisiert oder das Form unterstützt kein CSC. Standardwerte werden nicht gesetzt.
