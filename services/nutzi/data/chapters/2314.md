# Benutzerprofil Registerkarte Versand 
Benutzerprofil Registerkarte Versand 

Diese Registerkarte zeigt Ihnen links die Adress-Informationen zur Firma, wie sie im Shop eingetragen wurden. Im Bereich Versandadresse können Sie abweichende Daten eingeben. 

Wenn Sie den Benutzer einem bestehenden Kunden zugeordnet haben, kann hier die Adresse aus den Lieferanschriften des Kunden übernommen werden. 

Wenn Sie die Adresse aus der Anmeldung des Webshops übernehmen, klicken Sie auf den Button . In diesem Fall wird eine neue Lieferanschrift für den Kunden angelegt. 

Bei einem neu angelegten Kunden funktioniert dies genau gleich, vorausgesetzt der Benutzer wurde bereits als Kunde registriert. 
### Feld Ref. Anschrift Nr. 
Hier wird die Nummer der Lieferadresse angegeben. 

Tooltip: 

Angabe der Lieferanschrift 

### Feld Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben.

### Feld GEO-Koordinaten 
Hier werden geografische Daten zur besseren Orientierung angegeben. Damit lassen sich Besuche o.ä. besser planen und Sie können die Karte zur besseren Orientierung und Festlegung einer Besuchs-Reihenfolge nutzen. 

Das Format der GEO-Koordinaten lautet: 
XX.yyyyyy,AA.bbbbbb 

Beispiel für Stockach, Nissen&Velten: 
47.848029,09.005334
