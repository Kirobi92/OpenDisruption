# Tabellen-Filter 
Tabellen-Filter 

Hier können zentral Filter für den Zugriff auf Tabellen definiert werden. Anforderungen wie z.B. „Der Vertreter darf nur seine Kunden, Aufträge usw. sehen“ können sehr elegant und effizient mit dieser Funktion unter System >> Administration >> Tabellen-Filter eingestellt werden. 

Zuerst legen Sie eine neue Zeile für den Tabellenfilter an. Hier legen Sie die Tabelle und den Filternamen fest. 

Hier die grobe Vorgehensweise, dargestellt anhand folgenden Beispiels: 

Der User SYSADM darf nur Kunden vom Vertreter 3 sehen. Dazu geben Sie in die Spalte 'Tabelle' eine Bezeichnung ein (z.B. den Begriff Kunden ), für den Sie einen Filter setzen möchten. Dann geben Sie in der Spalte 'Filter Name' eine Bezeichnung für den Filter ein. Anschließend wechseln Sie mit dem Button in den Dialog Tab.-Filter bearbeiten und definieren die Spalten (Registerkarte Spalten ), auf die der Filter gesetzt wird. Außerdem können Sie auf der Registerkarte Variablen noch die entsprechenden Variablen eintragen oder über den Button Variablen auswählen. 

Siehe Kapitel Tabellen-Filter bearbeiten . 

Siehe Kapitel Filter-Variablen .

Hier wählen Sie die Tabelle aus, für die ein Filter angelegt werden soll. 

Tooltip: 

Angabe der DB-Tabelle 

### Tabelle 

Über den Button wird in der Tabelle eine neue Zeile eingeblendet und aus der Combobox in der Spalte 'Tabelle' wählen Sie die gewünschte aus. Eine freie Eingabe ist aus Sicherheitsgründen nicht möglich, da dieser Tabellenname sofort in der Datenbank gespeichert wird und nicht mehr änderbar ist. 

Die Spalte 'Filter' ist frei definierbar, aber ein Pflichtfeld! 
### Tabellenspalte Tabelle 
Hier wird die Datenbanktabelle angegeben. 

Tooltip: 

Angabe der DB-Tabelle 

### Feld Filter 
Hier können Sie Bedingungen erstellen, diese sind an die allgemeinen SQL-Regeln angelehnt. 

### Button Öffnen 

Mit diesem Button öffnet sich eine Auswahl an Tabellen-Filtern. 

Tooltip: 

Öffnet den Dialog zur Bearbeitung des markierten Tabellen-Filter. 
Es können nur Tabellen-Filter editiert werden, welche zuvor manuell angelegt wurden.
