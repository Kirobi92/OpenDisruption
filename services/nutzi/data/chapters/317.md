# Datentransfer (Import und Export von Daten) 
Stammdaten (wie Artikel, Kunden- und Partnerdaten) können importiert und exportiert werden und stehen so zur Weiterverarbeitung für andere Programme zur Verfügung. 

Der Datentransfer wird in folgenden Kapiteln näher beschrieben: 

- Datenexport - Datenimport - Daten Umsetzung - CSV Format Konversion - Daten ins Excel übertragen
