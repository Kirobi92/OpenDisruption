# Benutzerfreigaben 
Benutzerfreigaben 

Dieser Dialog öffnet sich im Besuchskalender über das Menü Optionen . 

Jeder Benutzer kann anderen Benutzern Freigaben (Lese- und / oder Schreibrechte) für seinen Kalender einräumen. Beispiele: 

- Der Innendienstmitarbeiter soll Termine des Außendienstmitarbeiters einsehen können, für eine reibungslose Zusammenarbeit. - Ein Vertriebsleiter hat in alle Termine seiner (Außendienst-) Mitarbeiter Einblick. - Ein Vertriebsassistent braucht aufgrund seiner Position ebenfalls einen Einblick in die Kalender bzw. die Möglichkeit Änderungen im Kalender vorzunehmen. 
Dabei wird zwischen Lese- und Schreibrecht unterschieden. Administratoren können alle Berechtigungen einsehen und ändern. Alle anderen Benutzer sehen nur die Berechtigungen auf den eigenen Kalender. 

Eine Freigabe kann über die Checkbox in der Spalte 'Gesperrt' gesperrt werden. 

Hat ein Benutzer Freigaben bekommen, so werden seine Ressourcen (wenn im Dialog Sachbearbeiter zugeordnet) im Kalender als zusätzlich auswählbare Ressourcen angeboten. 

Diese Einstellungen sind notwendig, da 
- nicht jeder die Termine von jedem sehen darf - nicht jeder für jeden Termine eintragen bzw. ändern darf - aber z.B. der Vertriebsleiter alle Besuche der ADM sehen können soll.

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Neuer Satz 

Mit diesem Button können Sie der Tabelle einen neuen Eintrag hinzufügen. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
