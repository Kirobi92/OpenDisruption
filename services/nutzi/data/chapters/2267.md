Hier werden die Werte im Klassifikationssystem angegeben. 
### Tabelle 

### Tabellenspalte ID 
Hier wird ein eindeutiger Identifikator eines Vorgabewertes innerhalb des Klassifikationssystems angegeben. 

Tooltip: 

Eindeutiger Identifikator eines Vorgabewertes innerhalb des Klassifikationssystems. 

### Tabellenspalte Name 
Hier wird der eindeutige Name des Vorgabewertes innerhalb des Klassifikationssystems angegeben. 

Tooltip: 

Gibt den eindeutigen Namen des Vorgabewertes innerhalb des Klassifikationssystems an.

Hier wird eine Beschreibung angegeben.
