# Ressourcenauslastung 
Ressourcenauslastung 

Über den Button Auslastung im Projektleitstand öffnet sich das Menü Ressourcenauslastung . Hier werden die Kapazität und der Auslastungsgrad der gewählten Ressource angezeigt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Ressourcenauslastung 
Tooltip: 

In diesem Dialog werden die Kapazität und der Auslastungsgrad der gewählten Ressource angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Drucken 

Mit diesem Button starten Sie den Druck. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Baumansicht Ressourcengrp. 

In dieser Übersicht sehen Sie alle bereits angelegten Ressourcen. Durch Auswahl einer Ressourcengruppe werden alle Ressourcen dieser Gruppe in der Tabelle eingelesen. Bei Auswahl einer Hauptgruppe werden ebenfalls alle Untergruppen eingelesen. 

### Combobox Ressourcentyp 
Hier wählen Sie zur Selektion einen Ressourcentyp aus. 

Tooltip: 

Angabe des Ressourcentyps 

### Combobox Prozesstyp 
Aus dieser Combobox kann ein Prozesstyp (Aufgabe, Betriebsauftrag, Serviceeinsatz, manuell) ausgewählt werden. 

### Feld Datum von 
Hier geben Sie zur Selektion ein Start-Datum und eine Uhrzeit ein. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Datum bis 
Hier geben Sie zur Selektion ein Bis-Datum und eine Uhrzeit ein. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Grün < x % 
Hier geben Sie einen Prozentwert für eine Auslastung. Grün bedeutet wenig Auslastung. 

### Feld Rot > x % 
Geben Sie hier einen Prozentwert ein, für eine zu hohe Auslastung. 

Angaben zwischen Grün und Rot werden in Gelb angezeigt. 

### Dock Panel Angabe von Datum / Zeit festlegen. 

Über diese Radiobuttons steuern Sie die Einträge in den Feldern Datum / Zeit von / bis . 

### Tabelle Ressourcen 

### Tabellenspalte Gruppe 
Der Name der Ressourcengruppe wird eingetragen. (Ressourcen werden im Modul Werkzeuge verwaltet.) 

Siehe Kapitel Werkzeuge / Ressourcen . 

Tooltip: 

Angabe der Ressourcengruppen-Bezeichnung 

### Tabellenspalte Ressource 
Die Nummer der Ressource wird eingetragen. 

Tooltip: 

Angabe der Ressource 

### Tabellenspalte Bezeichnung 
Eine genaue Bezeichnung der Ressource wird eingetragen. 

Tooltip: 

Name der Ressource 

### Tabellenspalte Kapazität Zeitraum 
Angabe der maximal verfügbaren Zeit im gewählten Zeitraum. Die Angabe wird automatisch aus dem Betriebskalender errechnet, welcher der Ressourcengruppe zugrundeliegt. 

### Tabellenspalte Kapazität benötigt 
Angabe, welcher Zeitraum in der Selektion bereits verplant ist. 

### Tabellenspalte Kapazität verfügbar 
Differenz zwischen ‚Kapazität Zeitraum‘ und ‚Kapazität benötigt‘. 

### Tabellenspalte Auslastung % 
Hier wird Ihnen die Auslastung der Ressource in Prozent angezeigt. 

### Tabellenspalte Belastung % 
Angabe der Belastung aus dem Ressourcenstamm. Die Angabe wird bei der Farbdarstellung im Feld Auslastung % berücksichtigt. 

### Tabellenspalte Prozessanzahl 
Hier wird die Anzahl der Prozesse angezeigt, welche innerhalb des Zeitraums berücksichtigt werden.
