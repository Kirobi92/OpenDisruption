# Integration Services 
Integration Services 

Für die Installation der SSIS-Skripte für eNVenta MIS steht der Dialog Integration Services zur Verfügung. 

Dieser Dialog wird über den Button Erweitert im Dialog Parameter MIS auf der Registerkarte Integration Services geöffnet. Der Button ist nur aktiv, wenn unter SSIS / Pakete eine Server-Verbindung eingetragen ist. 

eNVenta adressiert hier den SSIS Package Storage. Die SSIS werden nach wie vor über den MIS Remote Service angesprochen. 

Der Benutzer sieht die Pakete wie im SQL-Server Management Studio und kann die Pakete hier auch einzeln ausführen. Zudem können hier auch Verzeichnisse in den SSIS angelegt / umbenannt / gelöscht werden. 

Die Paket-Variablen werden hierbei durch die Konfiguration in den Parametern übersteuert, so dass Sie die Pakete direkt mit der richtigen Konfiguration ausführen können. Das Ausführen kann über den Button Paket ausf. gestartet werden. Der Benutzer bekommt den aktuellen Fortschritt als Laufbalken angezeigt. Die Sprache der Meldungen ist allerdings von der Sprache der SQL-Server Installation abhängig. 

Die Pakete können per Button-Klick aus dem Broker-Verzeichnis in die SSIS installiert werden. eNVenta entscheidet aufgrund des Source-Connection-Typs, ob die Oracle- oder SqlServer Pakete installiert werden. 

Das Paketverzeichnis muss weiterhin in den Parametern gepflegt werden, damit anhängige Pakete korrekt geladen werden können. 

Bitte beachten Sie auch die separate Installations-Anleitung für MIS.
