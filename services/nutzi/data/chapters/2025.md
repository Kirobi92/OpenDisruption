# Kalkulationsbasis des Materials

Produktion (PPS) / Kalkulation in der Produktion 

# Kalkulation in der Produktion Registerkarte Kalkulation 
Kalkulation Registerkarte Kalkulation 

Um eine möglichst individuelle und genaue Kalkulation zu erzielen, wird auf die jeweilige Situation des zu kalkulierenden Elements Rücksicht genommen. 

Dabei ist von Bedeutung: 

- Der Status (vor, während oder nach dem Produktionsvorgang) - Das Vorhandensein von Unterprozessen: Betriebsauftrag oder Bestellung - Soll oder Ist 
Kalkulationsbasis Material 

Soll Ist 
Sonst Sollpreis + Nebenkosten. 
Sonst Istpreis + Nebenkosten. 
Falls nicht angegeben, dann Preis + NK aus Artikel. 
| | | |
| --- | --- | --- |
| Menge | Gesamtmenge (Soll) unter Berücksichtigung von Materialschwund | Istmenge, falls BDE erfolgt und Status > 3, sonst Gesamtmenge (Soll) unter Berücksichtigung von Materialschwund. |
| Preis | Wenn Unter-Betriebsauftrag vorhanden, dann Selbstkosten Soll aus Unter-Betriebsauftrag. | Wenn Unter-Betriebsauftrag vorhanden, dann Selbstkosten Ist aus Unter-Betriebsauftrag. |

Wurde das Material auftragsbezogen bestellt, so fließt der tatsächliche Preis durch die Rechnungsprüfung in die Ist-Werte des Materials. 

Konnten keine Soll-Werte ermittelt werden (weil nicht vorher festgehalten), so werden die Ist-Werte übernommen. 

Kalkulationsbasis ‚Fertigung‘: 

Soll Ist 
| | | |
| --- | --- | --- |
| Menge (Zeit) | Bestellte Menge x variable Zeit + Rüstzeit je zugeordnete Ressource (Anteil berücksichtigt) | Reservierte Kapazitäten. Falls nicht vorhanden, dann gleich Soll. |
| Preis | Stücklohn falls angegeben, sonst Sollzeiten und Kostensatz. | Stücklohn falls angegeben, sonst terminierte Zeiten (wenn vorhanden) und Kostensatz. (Terminierungen werden durch BDE aktualisiert). |

Die verwendeten Ampelfarben auf Zeilenebene einer Kalkulation oder im Strukturbaum des Produktionsauftrags sind von den eingestellten Toleranzen abhängig. 

Die Toleranzen werden in den Produktionsparametern hinterlegt.
