# Aufgabenstamm Registerkarte Allgemein

Projekt Management / Aufgabenstamm 

# Aufgabenstamm Registerkarte Allgemein 
Aufgabenstamm Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die allgemeinen Einstellungen zur Aufgabe. 
### Feld Memo 1 
Hier kann ein individueller Text als Memo eingetragen werden. 

### Feld Sollzeit (hh,mm) 
Dieses Feld steht für Auswertungen im Projekt-Controlling zur Verfügung. Hier geben Sie die Sollzeit zur Ausführung der Aufgabe ein. 

### Combobox Bereich 
Wenn Sie aus dieser Combobox einen Eintrag auswählen, wird die Struktur sofort entsprechend dem ausgewählten Bereich eingelesen: 

- Es werden nur Phasen angezeigt, mit gleicher oder ohne Bereichskennung. - Es werden nur Aufgaben angezeigt, mit gleicher oder ohne Bereichskennung. - Bei den Aufgaben werden nur solche angezeigt, bei denen die zugehörige Phase dem Bereich entspricht oder keine Kennung hat. 
Dadurch lassen sich Aufgaben filtern. Dies dient der übersichtlicheren Darstellung bei größeren Projekten. 

### Combobox Verantwortlich 
Hier geben Sie den Verantwortlichen für eine Projektaufgabe ein. 

Beim erneuten Öffnen dieser Projektaufgabe ist der Verantwortliche bereits voreingestellt. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabelle Checkliste 

In dieser Tabelle kann zu einer Aufgabe eine Checkliste mit beliebig vielen Einträgen erfasst werden. Wird eine Aufgabe aus dem Stamm in ein Projekt übernommen, so wird auch die Checkliste übernommen. Im Aufgabenstamm vordefinierte Checklisten erleichtern die lückenlose Abarbeitung der Aufgabe in den Projekten. 
### Tabellenspalte Pos 
Hier wird die Positionsnummer angegeben. Diese Nummer muss je Vorgang eindeutig sein! 

Tooltip: 

Positionsnummer 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Button OK 

Mit diesem Button speichern Sie die Eingaben. 

### Button Satz hinzufügen 

Mit diesem Button können Sie einen weiteren Eintrag in der Tabelle einfügen. 

### Button Löschen 

Mit diesem Button löschen Sie einen Eintrag der Tabelle.
