# Verbuchungsregeln im Auszug-/Kassenimport 
Verbuchungsregeln 

Der Button Verb.Regel im Dialog Auszug-/Kassenimport öffnet das Menü Verbuchungsregeln mit dem Sie Regeln festlegen können, welche wiederkehrende Kontobewegungen erkennen (z.B. Mieten). Somit ist es auch möglich, Buchungen vorzunehmen, die keine Geldeingänge / Geldausgänge in Verbindung mit Debitoren und Kreditoren sind. 

Wenn Sie das Kassenmodul im Einsatz haben und Sie Ihre übernommenen Kassenrechnungen automatisch ausgleichen wollen, können Sie dies über die Verbuchungsregeln automatisch steuern. 

Buchungsregeln, die auf ein OP-führendes Sachkonto zielen, werden ignoriert. Es kann nicht unterschieden werden, ob ein OP anzulegen oder auszugleichen ist. Diese Sätze bleiben in der Tabelle des Dialogs Auszug-/Kassenimport stehen und sind zwingend über den Button Fibu Soll oder Fibu Haben (auf der Registerkarte Import Daten ) auszubuchen! 

Bei Aktiv-Schaltung einer Verbuchungsregel wird geprüft, ob bereits eine aktive Verbuchungsregel mit gleichem Namen, Zweck, S/H, Buchungstext und Buchungsschlüssel existiert. Achten Sie deshalb bitte beim Neuanlegen einer Verbuchungsregel darauf, dass Name und Zweck immer verschieden sind! 

Im Folgenden werden die Bedeutungen der Spalten und die Zugehörigkeit zur Registerkarte Import Daten erklärt.

Die Bezeichnungen der Spalten, die in der obigen Auflistung grau hinterlegt sind, sind auch im Dialog grau hinterlegt, um zu verdeutlichen, dass diese Felder für die Suche relevant sind. 

Die Spalten in der Tabelle auf der Registerkarte Verbuchungsregeln , bei denen eine Zuordnung besteht, müssen den gleichen Inhalt haben, wie die entsprechenden Spalten auf der Registerkarte Import Daten . 

In den Verbuchungsregeln der Auszug-/Kassenimport -Verarbeitung kann der Steuerschlüssel unabhängig vom Sachkontenstamm für jede Verbuchungsregel gesteuert werden. 

Es besteht die Möglichkeit, eine Kasse gezielt einer Verbuchungsregel zuzuordnen. Dafür steht die Spalte ‚Zusatz‘ zur Verfügung. Ist in dieser Spalte eine Kassennummer eingetragen, so darf diese Verbuchungsregel bei der Automatik nur in Bezug auf diese Kassennummer verwendet werden. 

Ist die Spalte nicht gefüllt, ist die Verwendung wie aus dem Standard bekannt. 

Der Abgleich erfolgt auf die gleichnamige Spalte im Dialog Auszug-/Kassenimport auf der Registerkarte Kassenübernahme .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
