# X-Mapping Editor Registerkarte Namespaces 
Registerkarte Namespaces 

Auf dieser Registerkarte wird die URL für den Namespace für evtl. notwendige Validierungen angegeben. 

Der Standard von eNVenta basiert immer auf dem Namespace BMEcat sowie einem Verweis auf das OpenTrans-Format 2.1 für die Verwendung des X Mapping Editors bei der Verarbeitung in EDI-XML. 
### Combobox Namespace 
Über diese Combobox kann der konkrete Namespace einem Knoten zugewiesen werden. Zur Auswahl stehen Default und BMECat .
