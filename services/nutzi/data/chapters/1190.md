# Kreditlimit-Status 
Kreditlimit-Status 

Im Zusammenhang mit der „erweiterten Kreditlimitprüfung“ steht Ihnen unter Optionen das Menü Kreditlimit-Status zur Verfügung. So haben Sie die Möglichkeit, das Kreditlimit eines Kunden/Auftrags anzusehen und zu beantragen. Auf diese Weise kann schnell eine Entscheidung bzgl. des weiteren Vorgehens getroffen werden bzw. steht eine Argumentationshilfe zur Verfügung. 

Der Dialog kann auch über den Shortcut <Strg + Umschalt + K> geöffnet werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Kreditlimit-Status 
Tooltip: 

In diesem Dialog haben Sie die Möglichkeit, das Kreditlimit eines Kunden/Auftrags anzusehen und zu beantragen. 
Für mehr Informationen drücken Sie F1.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button ist es möglich, ein Sonder-Kreditlimit einzurichten, wenn Sie den Dialog über das Menü Optionen geöffnet haben. 

Wird hier ein Sonder-Kreditlimit angelegt, speichern Sie dies. Sie erhalten eine Fehlermeldung, wenn Sie kein Kreditlimit eingeben bzw. eine 0. 

Existiert zum Kunden ein Kreditlimit und beim Packlistendruck erfahren Sie, dass das Kreditlimit überschritten wurde, wird automatisch ein Eintrag in die Tabelle Kreditlimit Status geschrieben. In diesem Fall ist der Button nicht mehr aktiv! 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button KL Freigabe 

Mit diesem Button können Sie die Kreditsperre aufheben, ein entsprechender Text wird in die Spalte ‚Bemerkung‘ eingetragen. Haben Sie die Berechtigung ein Auftrags-Kreditlimit anzulegen, so wird dies bei der Speicherung direkt freigegeben, ansonsten müssen Sie den Auftrag per Wiedervorlage an einen entsprechenden Sachbearbeiter mit der notwendigen Berechtigung weiterleiten.

### Tabellenspalte Kreditlimit 
Hier kann ein Standard-Kreditlimit für den Kunden (in System-Währung) festgelegt werden. Falls der Betrag überschritten wird, erhalten Sie eine Meldung, wenn Sie einen Auftrag zu diesem Kunden erfassen. Sie erhalten die Möglichkeit, eine Begründung für die Überschreitung einzugeben. 

Die absoluten Kopfrabatte werden in der Kreditlimitprüfung nicht berücksichtigt. 

Das Kreditlimit ist immer in "Haus-Währung" anzugeben!
