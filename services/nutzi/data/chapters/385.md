# Protokoll Löschung 
Protokoll Löschung 

Diesen Dialog finden Sie unter System >> Administration >> Protokoll Löschung . Über diesen Dialog können die definierten Löschdokumentationen verfolgt werden: 

Bei einem SQL-Server wird dieser Menüeintrag mit grauem Hintergrund angezeigt und kann nicht ausgewählt werden! 

Wenn es eine Oracle Datenbank ist, können Sie sowohl die Überwachung für Anpassungen und Löschungen einzeln konfigurieren. 

Die Überwachung bzw. die Protokollierung kann für jede vorhandene Tabelle und für ausgewählte Spalten erfolgen. Dabei kann der Anwender (Administrator) frei definieren, welche Daten überwacht werden sollen. 

Sie können alle Löschungen, die Tabelle und die Spalten (nur solche, die zur Überwachung definiert sind) betreffend, anzeigen lassen. 

Neben den Comboboxen Tabelle und Feldname werden dynamisch die "Keyfelder" mit den entsprechenden Labels angezeigt. 

Wird ein Tabellenname eingegeben, so wird der entsprechende Spalteneintrag herausgenommen. Gleiches gilt für die Spalten ‚Feldname‘ und die ‚Keyfelder‘.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme).
