# Event Management Registerkarte Aktionen 
Event Management Registerkarte Aktionen 

Nach Auswahl der Selektionskriterien werden die Aktionen in die Tabelle eingelesen. 
### Feld Datum von 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Checkbox Information 
Wenn Sie diese Checkbox aktivieren, werden die Daten über den Status Information selektiert. 

### Checkbox Warnung 
Wenn Sie diese Checkbox aktivieren, werden die Daten über den Status Warnung selektiert. 

### Checkbox Fehler 
Wenn Sie diese Checkbox aktivieren, werden die Daten über den Status Fehler selektiert. 

### Tabelle 

Die Tabelle listet die verarbeiteten Datensätze auf. 
### Tabellenspalte Datum 
Hier wird das Datum angegeben. 

### Tabellenspalte Status 
Hier wird der Status der Position angegeben. 

1-offen 
2-offen, zur Lieferung freigegeben 
3-offen, zur Faktura freigegeben 
4-fakturiert 
5-geliefert (ohne Berechnung) 
6-storniert 

Tooltip: 

Zeigt Fehler-Level an (Information, Warnung oder Fehler). 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Referenz 
Hier wird eine Referenz angegeben. 

Tooltip: 

Angabe einer Referenz 

### Tabellenspalte Nachricht 
Hier wird ein Fehlertext angegeben. 

Tooltip: 

Angabe eines Fehlertexts
