# Artikelauswahl zur Merkmalsverwaltung 
Artikelauswahl 

Dieser Dialog öffnet sich mit dem Button Art. Auswahl im Dialog Merkmalsverwaltung . Zur Selektion der Daten stehen die Felder im Kopf des Dialogs zur Verfügung. 

Dieser Dialog entspricht nicht dem Dialog Artikelauswahl , den Sie z.B. aus der Auftragserfassung kennen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Artikelauswahl 
Tooltip: 

Dieser Dialog bietet eine Artikelauswahl für die Artikel Merkmalsverwaltung. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Tabellenspalte VK 1 
Hier wird ein VK (Verkaufspreis) angegeben, den Sie als Unternehmen definiert haben. 

eNVenta kann im Artikelstamm 3 Preisgruppen, also 3 verschiedene Verkaufspreise besitzen. Zusätzlich können weitere Preisgruppen im Dialog Preisgruppen Details angelegt werden. 

Für die Ermittlung des Verkaufspreises geht jedes Unternehmen anders vor; je nach Kalkulation von Gewinnmarge, Aktionen, Handlingspauschalen usw. 

Für die Ermittlung des VKs, basierend auf dem EK + %-Aufschläge, kann die Artikelkalkulation genutzt werden. 

Wir empfehlen für die Anlage von VKs, die Preisgruppen-Details zu nutzen. Dies bietet folgende Vorteile: 

- Der Preis kann mit einem Gültigkeitszeitram versehen werden. - Es können Mengenstaffeln angelegt werden. - Die Nachverfolgbarkeit von Preisänderungen ist gewährleistet. - etc. 
Siehe Kapitel Artikelkalkulation . 

Siehe Kapitel Preisgruppen und Preisgruppen Details .

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
