# Artikel Cockpit Registerkarte Logistik Stamm 
Artikel Cockpit Registerkarte Log. Stamm 

Auf dieser Registerkarte erhalten Sie eine Sicht auf 

- alle festen Lagerplätze (Bereich Artikelfestplatz ) - alle Platzzuordnungen (Bereich Platzbestand ) - alle LHM zu einem Artikel (Bereich Artikel LHM ) 
### Tabelle Artikelfestplatz 

### Tabellenspalte Lager 
Die Lagernummer wird angezeigt. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Bereich 
Geben Sie hier den Bereich ein, in dem sich der Lagerplatz befindet. 

Tooltip: 

Geben Sie hier den Bereich ein, in dem sich der Lagerplatz befindet. 

### Tabellenspalte Lagerplatz 
Hier wird angezeigt, auf welchem Lagerplatz der Artikel zu finden ist. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Lagereinheit 
Die Lagereinheit des Artikels wird angezeigt. 

Tooltip: 

Zeigt die Lagereinheit des Artikels. 

### Tabellenspalte Faktor VK 
Der Verkaufsfaktor des Artikels wird angezeigt. 

Tooltip: 

Zeigt den VK-Faktor 

### Tabellenspalte Min. Menge 
Hier wird die Menge angezeigt, die mindestens im Lager vorhanden sein muss. 

Tooltip: 

Zeigt den Mindest-Lagerbestand. 

### Tabellenspalte Max. Menge 
Hier wird die Menge angezeigt, die maximal im Lager vorhanden sein darf. 

Tooltip: 

Zeigt die max. Lagermenge des Artikels.
