# Multidimensional Expressions

MIS 

# MDX Abfrage 
MDX Abfrage 

Dieser Dialog öffnet sich in der Online Analyse >> Optionen >> MDX Abfrage und liefert die Abfrage, welche auf die Data-Warehouse-Datenbank abgesetzt wird. 

So wie relationale Datenbanken SQL als Abfragesprache unterstützen, hat sich für OLAP-Datenbanken MDX etabliert. MDX steht für Multidimensional Expressions und orientiert sich zumindest von der Syntax an SQL. Funktional hat sie mit SQL allerdings nichts gemein, sondern ist auf die spezifischen Anforderungen im Handling mit multidimensionalen Datenquellen zugeschnitten. Deshalb kann in dieser Einführung auch nicht auf die einzelnen Befehle und Möglichkeiten der Sprache eingegangen werden. Da die Sprache ursprünglich von Microsoft entwickelt wurde, finden Sie eine Sprachreferenz in der Online-Dokumentation zum SQL-Server. 

Begriffserläuterung Einführung 

Relationale und multidimensionale Datenbanken sind auf unterschiedliche Ziele hin entwickelt worden. Während relationale Datenbanken optimale Datenstrukturen besitzen (die Daten sind normalisiert, um Redundanzen zu verhindern), sind multidimensionale Datenbanken abfrageoptimal. 

Dabei hält die OLAP-Datenbank bereits vorberechnete Ergebnisse für mögliche Abfragen und Aggregationen bereit. 

Da viele potenzielle Wertkombinationen bereits vorgehalten werden, ist das Antwortverhalten der Datenbank genau anders herum als bei einer relationalen Datenbank: Je stärker die Werte verdichtet sind, desto einfacher ist die Beantwortung einer Anfrage, da direkt auf vorberechnete Wertkombinationen zugegriffen werden kann.
