# Anfrage Registerkarte Serienanfrage 
Anfrage Registerkarte Serienanfrage 

Gehört eine Anfrage zu einer Serie, so haben Sie die Möglichkeit, dem Benutzer die Information zu liefern, welche Anfragen zu dieser Serie existieren. Hierfür bietet der Dialog Anfrage die Registerkarte Serienanfrage an, in der die Kopfdaten aller Anfragen einer Serie tabellarisch gelistet werden. Diese Registerkarte ist nur dann zugänglich, wenn die geladene Anfrage zu einer Serie von Anfragen gehört. 

Ein Doppelklick in eine Zeile der Tabelle öffnet die entsprechende Anfrage. 

In diesem Zusammenhang stehen auf der Registerkarte Positionen die Spalten ‚Preis‘ und ‚Rabatt1%‘ zur Verfügung, damit das Ergebnis der Anfrage eingetragen werden kann. Die Pflege der Antwort ist auch im Dialog Positionen möglich. 

Die Rückantworten auf die Anfragepositionen werden in die Registerkarte Positionen eingetragen. 

Dazu stehen folgende Spalten zur Verfügung: 

Preisbindung bis: 
Hier wird ein potentielles Preisbindungsdatum gepflegt. Bei der Erzeugung einer Bestellung erfolgt eine Prüfung auf das Preisbindungsdatum. 

Lieferdatum: 
Der vom Lieferanten genannte Liefertermin wird eingetragen. 

Info: 
Freies Textfeld zur Erfassung sonstiger Hinweise. 

Mit einem Doppelklick in eine Zeile der Tabelle wird die entsprechende Anfrage geöffnet. 

Um Rückantworten auf Serienanfragen oder auch Einzelanfragen zu vergleichen und zu bearbeiten, gibt es den Dialog Anfragespiegel .
