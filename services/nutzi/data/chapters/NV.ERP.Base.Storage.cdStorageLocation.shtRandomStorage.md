# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtRandomStorage

Dieser Parameter setzt das Modul Chaotische Lagerverwaltung voraus. Eine chaotische Lagerplatzverwaltung kann lagerbezogen eingeschaltet werden. Bei chaotischen Lagern ist es grundsätzlich notwendig bei jeder Buchung im System die Lagerplätze mit anzugeben. Die Bestände werden lagerplatzgenau geführt. Ein Artikel kann sich auf beliebig viele Plätze innerhalb eines Lagers verteilen. 

Siehe Kapitel Chaotische Lagerverwaltung . 

Tooltip: 

Kennzeichnet das Lager als chaotisches Lager. 
Dazu ist erforderlich, dass das Modul Chaotische Lagerverwaltung im Einsatz ist.
