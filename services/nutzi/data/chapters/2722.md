# Kostenauswertungen Registerkarte Selektionen 2 
Kostenauswertungen Registerkarte Selektion 2 

Über diese Registerkarte können weitere Einstellungen zu den verschiedenen Auswertungsmöglichkeiten in den unteren Registerkarten gemacht werden. 
### Combobox Dim.-Filter 
Hier kann der Dimensionsfilter ausgewählt werden. 

### Feld Projekt 
Hier können Sie eine Projektnummer eintragen. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Über diesen Button kann ein Projekt gesucht werden. 

Siehe Kapitel Projektauswahl . 

### Checkbox Vergleich mit 100%-Basis 
Diese Checkbox korrespondiert mit der Auswertungsstruktur in der Fibu, Bereich 100% Basis . Dort haben Sie die Möglichkeit bis zu 5 Stufen (1 bis 5) als Basen festzulegen. 

Wird in der Saldenauswertung die Checkbox aktiviert, erzeugen Sie hiermit Vergleichsspalten. In jeder der Vergleichsspalten werden alle Auswertungsstufen auf diese jeweilige 100% Basis berechnet. 

### Checkbox Vorzeichen umdrehen 
Wenn Sie diese Checkbox aktivieren, wird das Vorzeichen umgedreht. 

Tooltip: 

Vorzeichen umdrehen 

### Combobox Verd. auf Grp. 
Aus dieser Combobox wählen Sie aus, auf welche Gruppe verdichtet wird. 

Tooltip: 

Verdichtung auf Gruppe 

### Checkbox Nur selektierte Kostenstellen 
Wenn Sie diese Checkbox aktivieren, werden nur selektierte Kostenstellen ausgewertet. 

Tooltip:
