# Artikelnummer ändern 
Artikelnr ändern 

Dieser Dialog öffnet sich im Navigator >> Werkzeuge >> Artikelnr ändern . Hier können Sie die Änderungen, die Sie hier an bestehenden Artikelnummern vornehmen, in das gesamte System übernehmen. Dabei können die geplanten Änderungen auf Wunsch sofort nach der Dateneingabe realisiert werden. 

Die Änderungen können auch rückgängig gemacht werden. 

Die Funktion Artikelnummer ändern ist bewusst von der Replikation ausgeschlossen! 

Der Funktionsumfang geht weit über die Tabelle Artikel und die Änderung der Artikelnummer hinaus; alle von diesem Artikel betroffenen Vorgänge werden berücksichtigt. Dies macht einen Automatismus im Mandantenumfeld unmöglich, da all diese Vorgänge auch repliziert werden müssen. 

Die Funktion Artikelnr ändern erzeugt einen Statuseintrag, der Auskunft über den Erfolg der Änderung gibt (Status: Ok, Offen, Error ). Sämtliche Änderungen werden automatisch in der o.g. Tabelle gespeichert. Wieder editierbar sind jedoch nur diejenigen Datensätze, die den Status Offen oder Error aufweisen. 

Bei Datensätzen, die mit dem Status Ok gespeichert werden, wird in zwei Zusatzfeldern die Information über den aktuell eingeloggten Benutzer sowie über die aktuelle Systemzeit festgehalten. Nachfolgend weitere Details: 

- Auf einer Oracle-Datenbank kann dieses Modul nur ausgeführt werden, wenn Sie sich als SYSADM angemeldet haben. - Wenn bei der Änderung einer Artikelnummer irgendein Fehler aufgetreten ist, werden alle bis zu diesem Zeitpunkt durchgeführten Änderungen zu dieser Artikelnummer zurückgesetzt. - Artikelnummern mit Null-Werten werden nicht geändert - Die Artikelnummer, die geändert werden soll, muss im Artikelstamm vorhanden sein. - Die neue Artikelnummer, auf die geändert werden soll, darf noch nicht im Artikelstamm vorkommen. - Eine neu eingegebene Artikelnummer ('ArtikelnrAlt' oder 'ArtikelnrNeu') mit dem Status Offen oder Error darf in keinem Datensatz in der Tabelle bereits vorkommen. 
Um eine Historie der bereits geänderten Artikelnummern zu erhalten, geben Sie im Kopf des Dialogs im Feld Status den Status OK ein und drücken den Button .

## Menüeinträge 

### Menü Optionen 

### Menü Optionen / Änderung in Spalten 
Der Dialog Änderung in Spalten gibt an, nach welchen Spaltenbezeichnungen die Datenbank gescannt wird und entsprechend in diesen die Artikelnummer ersetzt wird. Das Kennzeichen ‚Standard‘ sagt, dass diese Spalten immer ersetzt werden. 

Siehe Kapitel Änderung in Spalten . 

### Menü Optionen / Artikelnummer prüfen 
Diese Option liest alle Artikel in den Dialog Artikelnr ändern ein, welche nicht der neuen Artikelnummerndefinition (ab Version 3.5) entsprechen. Vergleichen Sie dazu die Beschreibung zum Feld Artikelnr Format im Dialog Parameter Verkauf , Registerkarte Allgemein . Damit können diese über den Dialog / die Funktion Artikelnr ändern abgeräumt werden. Damit ist gemeint, dass Sie diesen Artikelnummern eine neue Nummer in der Spalte ‚Artikelnr neu‘ zuordnen. Es ist nicht notwendig, die ganze Liste abzuarbeiten. Artikel, die nicht mehr gebraucht werden, können aus der Liste gelöscht (Button ) werden. Bei einem erneuten Öffnen des Dialogs werden diese Artikel dann nicht mehr angezeigt. 

Wenn keine Artikel gefunden werden, erhalten Sie die Meldung, dass der Datencheck erfolgreich war.
