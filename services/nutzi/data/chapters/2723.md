# Kostenauswertungen Registerkarte Übersicht 
Kostenausswertung Registerkarte Übersicht 

Wenn Sie den Radiobutton Kostenstelle aktivieren und die Registerkarte Übersicht auswählen, wird die Checkbox Bezugsgröße sichtbar. Hiermit können für die Kostenstelle Kennzahlen ermittelt werden (z.B. Stundensätze, Kosten / Einheit). Die Bezugsgrößen werden im Menü Stamm >> Umlagen angelegt und gepflegt. Die Zuordnung zur Kostenstelle erfolgt im Stammsatz. 

Der Aufbau der Strukturen wird im Menü Stamm >> Auswertungsstruktur Kore definiert. Die Handhabung ist analog dem Aufbau einer Bilanz bzw. G+V.
