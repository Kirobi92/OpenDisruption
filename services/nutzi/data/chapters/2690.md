# Annullierung in der Kore 
Annullierung Kore 

Dieser Dialog öffnet sich im Buchen Kore >> Buchen >> Annullieren . Hier können Buchungen rückgängig gemacht werden. 

Sie können nur Buchungen mit der Herkunft ‚KO‘ annullieren; d.h. nur Buchungen, die in Buchen Kore gebucht wurden. 

Ein Doppelklick in eine Zeile öffnet den Dialog Kore-Buchungen . Die entsprechende Belegnummer ist voreingestellt. 

Für das Löschen von Datensätzen gilt, dass nur die markierten Datensätze – nach einer entsprechenden Meldung – gelöscht werden. 
## Steuerelemente des Formulars

Tooltip:
