# Teilnehmeraktivität Registerkarte Allgemein 
Aktivität - Teilnehmer Registerkarte Allgemein 

Auf dieser Registerkarte können Sie im Feld Information Informationen zur Kundenaktivität hinterlegen. 

Des Weiteren stehen Ihnen hier die im Aktivitätsschritt definierten (mit Namen versehenen) Felder und Schalter zur Verfügung. 
### Feld Information 
Hier kann ein individueller Text zur internen Information eingetragen werden. 

Tooltip: 

Enthält interne Textinformationen.
