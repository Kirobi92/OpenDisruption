# Produktionsarten 
Produktionsart 

Unter System >> Produktion steht der Eintrag Produktionsarten zur Verfügung. In diesem Dialog können Produktionsarten gepflegt werden. 

Die Produktionsart funktioniert analog der Auftrags- oder Einkaufart und ermöglicht eine flexiblere Steuerung der unternehmensinternen Abläufe. So lassen sich Lager vorgeben, verschiedene Kalkulationsfaktoren hinterlegen etc. 

Die Produktionsart kann anschließend im Betriebsauftrag vorgegeben werden. Beim Auflösen der Stückliste wird sie in jeden Betriebsauftrag, wie auch in den Produktionsauftrag eingetragen. 

Diese Produktionsart wird auch im Dialog Material disponieren Registerkarte Produzieren berücksichtigt. In den Dialogen Übersicht Betriebsaufträge und Planung Betriebsaufträge kann nach Produktionsart selektiert werden. 

Die Produktionsauftragsart ist nicht Teil des Schlüssels von Produktionsauftrag oder Betriebsauftrag, sondern dient nur der Selektion bzw. der Steuerungsvorgabe für Reports etc. 

Zur Anlage einer neuen Produktionsart drücken Sie zuerst den Button , um den Dialog zu leeren. Dann machen Sie Ihre Eingaben sowohl im Kopf des Dialogs als auch auf den Registerkarten. 
### Hierarchien: 
Bei der Ermittlung der Vorgabewerte in den Betriebsaufträgen hat der Eintrag im Dialog Produktionsarten Vorrang vor einem Eintrag im Dialog Parameter Produktion . 

Hier eine Auflistung der Vorgabe-Werte, welche durch die Produktionsarten gesteuert werden. 

Hierarchie Materiallager: 
- Benutzerangabe - Produktionsart - Parameter Produktion 
Hierarchie Produktionslager: 
- Benutzerangabe - Produktionsart - Parameter Produktion 
Hierarchie Kostenstelle: 
- Produktionsart - Arbeitsplan 
Hierarchie Kostenträger: 
- Produktionsart - Artikelstamm 
Hierarchie Gemeinkostensätze: 
- Produktionsart (auch 0 zählt) - Arbeitsplan - Parameter Produktion 
Beispiel zur Kostenträgerhierarchie: 

Bleibt die Kostenträgerdefinition über Produktionsarten leer, dann zieht der Kostenträger aus dem Artikelstamm (über Stamm >> Umsatzsteuer Spezifisch ). --> In den Kopfdaten des Betriebsauftrags wird der Kostenträger aus dem Artikelstamm übernommen. 

Über System >> Produktion >> Produktionsarten wird ein Kostenträger zugeordnet. Auch im Artikelstamm wird ein (anderer) Kostenträger festgelegt. Im entsprechenden Betriebsauftrag wird in der Registerkarte Kopfdaten der Kostenträger aus der Produktionsart übernommen. 

Beispiel zur Kostenstellenhierarchie: 
Die Kostenstelle der Produktionsart "sticht" die Kostenstelle des Arbeitsplans! 

Der Button Steuerung öffnet den Dialog Formularsteuerung .

- Betriebsauftrag - Beleg 2 (wird nur angeboten, wenn ein Label im Dialog Parameter Produktion gesetzt ist) - Beleg 3 (wird nur angeboten, wenn ein Label im Dialog Parameter Produktion gesetzt ist)

### Button Referenz-Link 

Dieser Button öffnet den Dialog Referenz Links . 

Referenz Links bieten die Möglichkeit Dokumenten-Links bzw. URLs als Info zu hinterlegen, um 2 Vorgänge / Referenzen miteinander zu verknüpfen. Dies erspart Ihnen lange Archiv-Recherchen. 

Siehe Kapitel Referenz Links . 

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Steuerung 

Dieser Button öffnet die Formularsteuerung. 

Siehe Kapitel Formularsteuerung .
