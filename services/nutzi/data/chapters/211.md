# Suchbedingung / Suche mit SQL Registerkarte Bedingungen 
Suchbedingung Registerkarte Bedingungen 

Auf dieser Registerkarte definieren Sie die Suchkriterien. Dazu drücken Sie zuerst den Button Neue Zeile , um eine neue Zeile in die Tabelle einzufügen. 

Wenn Sie 2 verschiedene Felder in einer Suchabfrage eintragen, so bildet sich daraus automatisch eine UND-Verbindung (z.B. Artikel ist in der Artikelgruppe „Demo“ und ein Diversartikel). 

Wenn pro Zeile das gleiche Feld abgefragt wird, so handelt es sich automatisch um eine ODER-Abfrage (z.B. Artikel ist in der Artikelgruppe „Demo“ oder in der Artikelgruppe Update). 

Die Combobox Operator 1 enthält folgende Einträge: 
- leer - von - größer - gleich - ungleich - beginnt mit - enthält - endet mit - ist leer - hat Wert 
Die Combobox Operator 2 enthält folgende Einträge: 
- leer - bis - kleiner 
Nicht jeder Operator 2 lässt sich mit einem Operator 1 verknüpfen! Entsprechend wird die Spalte ‚Hinweis‘ gefüllt. 

Mit Klick auf den Button -> Expert wird die definierte Abfrage in die Registerkarte Expertenmodus übernommen und kann dort bearbeitet werden. 
### Button Neue Zeile 

Dieser Button fügt eine neue Zeile in die Tabelle ein. 

### Button Lösche Zeile 

Mit diesem Button wird eine Zeile der Tabelle gelöscht. 

### Button -> Expert 

Mit diesem Button können Kriterien der Tabelle in das Experten-Register verschoben werden, damit das SQL nachgearbeitet werden kann. 

Tooltip: 

In den Expertenmodus verschieben. 

### Feld Datentabelle 
In diesem Feld wird die zugrunde liegende Datentabelle angezeigt, z.B. Auftragskopf. 

### Tabelle 

### Tabellenspalte Feld 
Aus der Combobox in dieser Spalte wählen Sie ein Feld aus, z.B. Abrechnungsart, AKL-Typ. 

### Tabellenspalte Operator 1 
Aus der Combobox in dieser Spalte wählen Sie den 1. Operator aus, z.B. !=Ungleich, >=Bis. 

### Tabellenspalte Wert 1 
In diese Spalte geben Sie einen Wert ein. 

### Tabellenspalte Operator 2 
Aus der Combobox in dieser Spalte wählen Sie einen 2. Operator aus. 

### Tabellenspalte Wert 2 
Hier geben Sie einen weiteren Wert ein. 

### Tabellenspalte Hinweis 
Je nachdem, was Sie in den Spalten einstellen, erhalten Sie in dieser Spalte noch einen Hinweis, z.B. wenn die Spalte 'Feld' nicht gefüllt ist.
