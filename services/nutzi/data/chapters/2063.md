# CRM Übersicht 
CRM Übersicht 

CRM bedeutet Customer-Relationship-Management und bezeichnet die konsequente Ausrichtung eines Unternehmens auf seine Kunden, Lieferanten und Adressen und die systematische Gestaltung der Beziehungen. 

Dieser Dialog öffnet sich im Navigator >> Vertrieb >> CRM Übersicht . Hier erhalten Sie eine effiziente Übersicht über CRM-Aktivitäten, -Situationen und –Aufgaben. Sie können auf Kunden, Objekte, Gewerke, Vertreter etc. selektieren, so dass die Informationen verdichtet auf die gewünschten Selektionen dargestellt werden und die aktuelle Situation auf einen Blick ersichtlich ist. 

Bei einer TAPI-Anbindung kann die CRM Übersicht direkt aus der Registerkarte im Navigator über den gleichnamigen Button erreicht werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld CRM Übersicht 
Tooltip: 

In diesem Dialog erhalten Sie eine effiziente Übersicht über CRM-Aktivitäten, -Situationen und –Aufgaben. 
Für mehr Informationen drücken Sie F1.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
