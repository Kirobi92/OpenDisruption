# Kalkulationsmöglichkeiten im Auftrag 
Eine Auftragsposition bzw. ein gesamter Auftrag kann aus der Auftragsbearbeitung heraus kalkuliert werden. Dazu stehen 2 komfortable und übersichtliche Kalkulationsschemata auf Knopfdruck zur Verfügung. Dies sind 

- die Gesamtkalkulation über den gleichnamigen Dialog - die Kalkulation einer Position über den Dialog Position Registerkarte Kalkulation 
Der Dialog Gesamtkalkulation öffnet sich über den Button Kalkulation im Kopf des Dialogs Auftrags .
