# Vertreterauswahl 
Vertreterauswahl 

In diesem Dialog haben Sie die Möglichkeit, mit verschiedenen Kriterien einen Vertreter zu suchen. (Der Dialog öffnet sich im Auftrag , Angebot , Gutschriften und Auskunft , jeweils über die Registerkarte Kopfdaten mit dem Button neben dem Feld Vertreter .) 

Angelegt werden die Vertreter im Vertreterstamm. 

Siehe Kapitel Stammdaten / Vertreter . 

Sie können eines oder mehrere der oberen Felder ausfüllen. Um den Suchvorgang zu starten, betätigen Sie den Button . Haben Sie keines der Felder vorbelegt, wird die Suche über die kompletten Stammdaten ausgeführt. 

Eine generelle Suche kann zu langen Wartezeiten führen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Vertreterauswahl 
Tooltip: 

Mit diesem Dialog können Sie Vertreter suchen. 
Für mehr Informationen drücken Sie F1.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter .

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen
