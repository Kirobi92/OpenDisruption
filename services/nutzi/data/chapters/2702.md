# Kostenarten / Leistungen importieren 
Kostenarten / Leist. importieren 

Dieser Dialog öffnet sich im Buchen Kore >> Stamm >> Kostenarten / Leist. importieren . Hier haben Sie die Möglichkeit, Aufwandskonten aus der Finanzbuchhaltung als Kostenarten bzw. Leistungsarten zu definieren. Dazu müssen Sie nicht jede Kostenart einzeln anlegen, sondern können die Daten aus der Finanzbuchhaltung übernehmen. 

Dabei haben Sie die Möglichkeit, die Daten der Tabelle nach G+V-Konten bzw. Bilanzkonten zu selektieren. 

Alle nicht übernommenen G+V-Konten bzw. Bilanzkonten werden angezeigt und zur Übernahme als Kostenart angeboten. 

Dazu wird im Sachkontenstamm auf der Registerkarte Kostenrechnung die Checkbox Kostenrechnung aktiviert und das Feld Kostenart gefüllt. 

Duch Klick auf den Button starten Sie die Suche nach allen bisher nicht in die Kostenrechnung übernommenen Aufwands- und Erlöskonten. In der Spalte ‚Typ’ sind Aufwandskonten mit einem A gekennzeichnet, Erlöskonten mit einem E . 

Die Tabelle weist nach erfolgter Übernahme alle noch nicht übernommenen Konten aus. 

Als Kostenarten werden die übernommenen Aufwandskonten mit den folgenden Standardeinträgen belegt. Kontrollieren Sie diese Belegungen und ändern Sie diese ggf. ab. 

Kostenart: 
= Kontonummer aus der Finanzbuchhaltung (unabhängig vom angegebenen Nummernkreis in den Systemdaten). 

Bezeichnung: 
= Kontobezeichnung aus der Finanzbuchhaltung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Kostenarten / Leist. importieren 
Tooltip: 

In diesem Dialog können Sie Aufwandskonten aus der Finanzbuchhaltung als Kostenarten bzw. Leistungsarten definieren. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button erscheinen die Konten, die noch nicht übernommen wurden, also z.B. von der letzten Übernahme ausgeschlossen waren oder in der Zwischenzeit neu angelegte Konten. 

### Button Neu 

Mit diesem Button leeren Sie die Felder. 

### Button Übernahme 

Mit diesem Button starten Sie die Übernahme. 

### Button Löschen 

Mit diesem Button werden markierte Einträge der Tabelle gelöscht. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Dock Panel Auswahl 

Hier wählen Sie aus, ob Sie Konten aus der G+V oder Bilanz importieren möchten.
