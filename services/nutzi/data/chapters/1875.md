# Versandauflösung 
Versandauflösung 

Wenn Sie eine Versandauflösung vornehmen wollen, öffnen Sie im Wareneingang mit dem Button Eingang den Dialog Eingangsbuchung . Hier wählen Sie das Versandlager aus und klicken den Button Versand an. 

Es öffnet sich die Versandauflösung . Hier erscheinen die zum ausgewählten Wareneingang gehörenden Auftragspositionen. In der Tabelle sehen Sie, welcher Auftrag zu welchem Kunden gehört, mit weiteren Informationen zum Artikel. 

Bei der Versandauflösung werden alle Aufträge aus der Tabelle berücksichtigt. 

D.h. Aufträge, die bei der Versandauflösung nicht berücksichtigt werden sollen, müssen (über den Button ) mit einem Löschkennzeichen versehen und mit dem Button gelöscht werden, bevor Sie mit dem Button Buchen die Versandauflösung für die restlichen Positionen starten. 

Ist die Wareneingangsmenge höher als die Menge der aufgelisteten Auftragspositionen, wird die Restmenge auf das in der Bestellung ausgewählte Lager gebucht. 

Vergleichen Sie hierzu das Kapitel Buchen des Wareneingangs . 

Der Ablauf der Wareneingangsbuchung erfolgt in drei Schritten: 
- Qualitätssicherungsstufe - Versandauflösung (Zuteilung der offenen Auftragspositionen nach Vorschlag) - Wareneingangsbuchung mit Berücksichtigung der Umbuchungen auf das Versandlager 
Für den angegebenen Wareneingang werden Auftragspositionen zur Auflösung vorgeschlagen. Dabei gelten folgende Regeln: 

- Nur Positionen im Status 1. - Nur Positionen aus der gleichen Lagergruppe wie das Versandlager (das Versandlager wird dabei ausgeschlossen). - Das Versandlager darf kein chaotisches Lager sein. - Zuerst die Position mit dem ältesten Lieferdatum, dann volle Auflösung des Kunden soweit möglich. - Teilmengen werden nicht aufgelöst. - Divers-Artikel werden nicht aufgelöst (außer bei direkter Zuordnung zur Auftragsposition). - Die einer Bestellposition zugeordnete Auftragsposition wird in jedem Fall (Feld Sonderbestellung ) vorgeschlagen. - Die Qualitätskennziffer (QKZ) muss leer oder 1 sein. 
Es wird davon ausgegangen, dass es keine Freigaben ohne vorhandene Waren gibt! 

Mengenänderungen im Wareneingang können in den Auftrag übernommen werden. 

Der erzeugte Vorschlag wird in der Tabelle RueckstandWE mit Status 0 gespeichert. 

Der Benutzer hat nun die Möglichkeit, durch Löschen von Zeilen einzelne Positionen aus dem Vorschlag herauszunehmen. 

Es kann zudem ein Druck für die Einlagerungsliste erzeugt werden. 

Mit dem Button Buchen werden die aufzulösenden Auftragspositionen auf das Versandlager umgestellt. 

Folgende Buchungen sind notwendig: 
- Die Lagernummer in der Auftragsposition wird auf das Versandlager umgestellt. - Reservierungen auf das Versandlager umstellen. Etwaige Chargen- oder Lagerplatzzuordnungen rückgängig machen. - Die Datensätze im Vorschlag werden auf Status 1 gesetzt. 
Bei einem weiteren Aufruf der Versandauflösung werden die Datensätze wieder angezeigt. Es ist jedoch keine Buchung mehr möglich. Dies dient der Auskunft und einem späteren Druck.

Wenn Sie diesen Button drücken, wird direkt eine Packliste gedruckt, wenn die entsprechende Checkbox im Dialog Parameter VS / LO (Packlistendruck bei Versandauflösung) gesetzt ist.

### Tabellenspalte Wunschdatum 
Wünscht der Kunde eine frühere Lieferung, wird der entsprechende Wunschtermin in das Feld Wunschdatum eingetragen. Im Programmteil Liefervorschlag kann das Programm gegebenenfalls bei frühzeitiger Lieferung den Wunsch des Kunden berücksichtigen und eine frühere Lieferung vorschlagen. 

Dieses Wunschdatum wird in der Tourendisposition nicht berücksichtigt; dort wird der Liefertermin zugrundegelegt. 

Existieren 2 Positionen mit unterschiedlichem Lieferdatum, so werden diese Positionen zum „kleineren“ Datum berücksichtigt (um Kosten zu senken).
