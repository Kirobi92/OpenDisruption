# Kursbewertung Registerkarte Debitoren 
Kursbewertung Registerkarte Debitoren 

Auf dieser Registerkarte werden Ihnen die ausgewählten Daten für die Neubewertung der Debitoren angezeigt. 

Wenn in der Kursbewertung ein selektierter OP mit Teilzahlung zu tun hat, erhalten Sie beim Verbuchen der Kursbewertung eine Meldung OPs haben bereits Teilzahlungen verbucht. Wenn Sie fortfahren werden die Teilzahlungsverknüpfungen aufgehoben! und die betroffenen OPs werden in der Tabelle markiert. Sie können mit OK fortfahren oder abbrechen und dann die markierten OPs über den Button Löschen aus der Kursbewertung nehmen. 
### Button Löschen 

Mit diesem Button löschen Sie einen Eintrag der Tabelle.
