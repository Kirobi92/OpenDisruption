# Dokument Vorlagenstamm 
Dokument Vorlagen 

Dieser Dialog öffnet sich über Menü Druck >> Vorlagenstamm . Hier definieren Sie Dokumentvorlagen für MS Word / OO Writer, E-Mail, E-Mail Html, Fax und SMS. 

Wenn Sie den Dialog Dokument Vorlagen aus dem Dialog Drucken oder Einzeldruck heraus öffnen, werden E-Mail- oder Fax-Vorlagen zur Auswahl angeboten. 

Zum Anlegen neuer Dokumentvorlagen ist immer der Weg über das Menü Druck >> Vorlagenstamm zu nehmen. 

Haben Sie E-mail ausgewählt, öffnet sich der Dialog E-Mail-Vorlagen mit einem Doppelklick in die Zeile der gewünschten E-Mail-Vorlage oder über den Button Details . 

Siehe Kapitel Email-Vorlagen . 

Wenn Sie einen Doppelklick auf eine SMS-/Fax-Vorlage ausführen, öffnet sich der Dialog SMS Vorlage bzw. Fax Vorlage . 

Siehe Kapitel SMS Vorlage . 

Siehe Kapitel Fax Vorlage . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Dokument Vorlagen 
Tooltip: 

In diesem Dialog definieren Sie Dokumentvorlagen für MS Word / OO Writer, E-Mail, E-Mail Html, Fax und SMS. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report

Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Mit diesem Button kann eine Vorlage kopiert werden. Diese Kopie wird als Kopie der ursprünglichen Vorlage gekennzeichnet. Die Bezeichnung kann entsprechend geändert werden. 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Details 

Dieser Button öffnet das Menü Word-Vorlagen , wenn Sie die Vorlagenart Microsoft Word ausgewählt haben. Ist E-Mail ausgewählt, öffnet sich der Dialog E-Mail-Vorlagen mit einem Doppelklick in die Zeile der gewünschten E-Mail-Vorlage, oder über den Button Details . 

Siehe Kapitel Email-Vorlagen . 

Siehe Kapitel Word- und Office-Vorlagen . 

### Feld Objekt 
Der Eintrag in diesem Feld basiert auf dem Menü, aus dem heraus Sie die Dokument-Vorlagen geöffnet haben. 

### Feld Akt. Bereich 
Der Eintrag in diesem Feld zeigt Ihnen den aktuell ausgewählten Bereich an. 

### Checkbox Nur Bereichsbezogene anzeigen 
Wird diese Checkbox gesetzt, so wird das Objekt dem Dokument bzw. der Vorlage zugewiesen. Dieses Dokument ist dann immer im gleichen Dialog als bereichsbezogen sichtbar. 

### Tabelle Der Dokumentvorlagen 

In dieser Tabelle stehen alle bereits angelegten Vorlagen zur Auswahl zur Verfügung. 
### Tabellenspalte Vorlage 
Hier können Sie der Vorlage eine individuelle Nummber zuordnen. 

### Tabellenspalte Vorlagenart 
Hier wird der Vorlagentyp angegeben. Folgende Einträge stehen zur Auswahl: 

- MS Word / OO Writer - E-Mail - E-Mailt HTML - Fax - SMS 

Tooltip: 

Angabe des Typs 

### Tabellenspalte Beschreibung 
Hier wird die Beschreibung der Vorlage angegeben. 

### Tabellenspalte Bereichsbezogen 
Hier wird angezeigt, ob das Objekt bereichsbezogen ist. 

### Tabellenspalte Kontextabh. Sprache 
Wenn Sie diese Checkbox aktivieren, wird die Sprache des Kunden, des Lieferanten oder der Adresse ermittelt und für die Ermittlung der korrekten Vorlage verwendet. 

Tooltip:
