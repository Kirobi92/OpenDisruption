# Kapitel NV.ERP.Base.Common.cdSystemMM.shtDeliveryNoteAddScan

Mit dieser Funktion ist es möglich, weitere Belege einer Tour über die Funktion „Lieferscheine scannen“ hinzuzufügen. Ein Fahrer scannt z.B. einen weiteren Lieferschein ein, welcher vom Vortag übrig geblieben ist oder ein Kunde wollte spontan, statt die Ware zu holen, eine Lieferung. So kann hier schnell der Lieferschein der Tour zugeordnet werden (Achtung: Zuschläge usw. erfolgen hierdurch nicht, es geht nur um die schnelle Umplanung/Einplanung eines Beleges auf die Tour). 

Tooltip: 

Nicht geladene Belege beim Scannen von Lieferscheinen hinzufügen.
