# VK-Positionen bearbeiten Registerkarte Kalkulation Set 
Position Registerkarte Kalkulation Set 

Diese Registerkarte ist nur sichtbar, wenn es sich bei der Position um ein Artikel-Set bzw. eine Stückliste handelt. 

Hier können Sie Artikel-Sets kalkulieren. Dazu ist es notwendig zuvor die Handelsstückliste über Optionen >> Artikel-Set >> H-Stückliste in Set auflösen in ein Artikel-Set aufzulösen. 
### Button Kalkulieren 

Wenn Sie diesen Button drücken, werden die Preise der einzelnen Set-Artikel kalkuliert. 

Tooltip: 

Kalkuliert die Preise auf Komponentenebene des Sets. 
Dann werden diese Preise aufsummiert. 
Erst mit "Übernehmen" werden die Preise auf den Header des Artikelsets übernommen. 

Nicht durchführbar im Fall von aktivierter Checkbox "Preis auf Komp." im Artikelstamm. 

### Button Übernehmen 

Wenn Sie diesen Button drücken, wird der Wert in die Position des Set-Artikels im Auftrag übernommen. 

Tooltip: 

Übernehme Preise auf Header. 

### Bezeichnungsfeld Menge 
Tooltip: 

Angabe der Menge 

### Feld Menge 
Dieser Wert wird aus dem Auftrag voreingestellt. 

Tooltip: 

Angabe der Menge 

### Feld Alle Preise pro 
Hier wird eine Preiseinheit angegeben. Dieser Preis kann sich auf 10, 100, 1000 oder eine andere Anzahl Artikeleinheiten beziehen. Standard ist der Wert 1. 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- 1 Einheit - je 10 Einheiten - je 100 Einheiten 

Tooltip: 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- je 1 Einheit 
- je 10 Einheiten 
- je 100 Einheiten 

### Feld Naturalrabatt 
Hier wird ein evtl. Naturalrabatt angezeigt. 

Tooltip: 

Angabe der Menge des Naturalrabatts 

### Feld Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind! 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Feld Kurs 
Hier wird der Kurs angegeben. 

Der Wechselkurs wird durch das Programm aus der Währungstabelle vorbelegt.

### Tabellenspalte Einheit VK 
Hier wird die Beschreibung der Einheit aus der Pickliste angezeigt. Dies dient vor allem der Optimierung der Reports, da Abkürzungen leicht missverstanden werden können. Bei vorhandenem Customizing mit dem Feld Einheit ist dies ggf. nachzuziehen. 

Ist in der Pickliste keine Beschreibung gepflegt, so wird wie bisher die Bezeichnung aus eNVenta angezeigt. 

Tooltip: 

Einheit VK
