# Vermietungsposition Registerkarte Betriebsstunden 
Vermietungsposition Registerkarte Betriebsstunden 

Auf dieser Registerkarte wird der Ist-Wert zum Zeitpunkt der Erstvermietung eingetragen. Die Werte stammen alle aus dem Mietobjekt. 
### Checkbox Betriebsstunden 
Wenn Sie diese Checkbox aktivieren, wird die Position nach Betriebsstunden abgerechnet. Die nachfolgenden Felder werden zugänglich. 

Die Anzahl der zulässigen Betriebsstunden pro Tag werden bei der Rücknahme (Retoure) des Mietobjektes mit der Anzahl der tatsächlich geleisteten Betriebsstunden verglichen. 

Tooltip: 

Wenn gesetzt, können die nachfolgenden Felder gepflegt werden.

### Feld Tag 
Hier geben Sie die Anzahl der regulären Betriebsstunden pro Tag für das Mietobjekt ein. 

Tooltip: 

Betriebsstunden/Tag 
Anzahl der regulären Betriebsstunden für das Mietobjekt. 

### Feld Ist 
Hier geben Sie die Betriebsstunden ein, die das Objekt anzeigt, die also tatsächlich geleistet wurden. 

Tooltip: 

Betriebsstunden Ist 
Betriebsstunden laut Betriebsstundenanzeiger. 

### Feld Endstand 
Hier geben Sie den Endstand der Betriebsstundenanzeige.

Betriebsstunden Ende
