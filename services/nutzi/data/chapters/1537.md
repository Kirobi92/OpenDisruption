# Versandart ändern 
Versandart ändern 

Dieser Dialog öffnet sich in der Kommissionierung >> Optionen >> PTV Schnittstelle >> Versandart ändern . In diesem Menü können Sie eine Versandart auswählen und sie mit dem Button übergeben. Jede der hier angegebenen Versandarten ist für die Übergabe an die V-LOG-Schnittstelle geeignet. 

Der Vorgang wird wie eine normale V-LOG-Übertragung ausgeführt. 

Die Änderung der Versandart ist nur temporär, bis die V-LOG-Übertragung abgeschlossen ist. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Versandart ändern 
Tooltip: 

In diesem Dialog können Sie Versandarten für die Übergabe an V-LOG auswählen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Tabelle 

### Tabellenspalte Versandart 
Hier geben Sie eine eindeutige Nummer für die Versandart an, mit welcher diese in der Datenbank gespeichert wird. Diese kann frei vergeben werden und zwischen 1 und 999 liegen. 

Tooltip: 

Eindeutige Nummer der Versandart, die in der Datenbank gespeichert wird. 

### Tabellenspalte Versandartbezeichnung 
Hier geben Sie eine möglichst sprechende Beschreibung der Versandart ein, um diese schnell wieder erkennen zu können. Diese Versandarten werden in den Vorgängen zur Auswahl zur Verfügung gestellt. 

Tooltip: 

Geben Sie eine Beschreibung der Versandart ein. 

### Tabellenspalte Versender 
Hier kann der zuständige Versender eingestellt werden. Die Versender pflegen Sie im Dialog Picklisten . 

Beim Ändern des Versenders werden automatisch alle Versandcodes zur Versandart gelöscht. 

Siehe Kapitel Picklisten . 

Tooltip: 

Geben Sie den zuständigen Versender ein. 

### Tabellenspalte Überg. Versender 
Mit dieser Einstellung geben Sie an, ob in der Kommissionieren im Versand eine automatische Übergabe des Lieferscheins an die Paketversender (MHP V-Log) durchgeführt werden soll. 

Tooltip: 

Wenn aktiviert, wird eine automatische Übergabe des Lieferscheins an die Paketversender (MHP V-Log) durchgeführt. 

### Tabellenspalte Übergabe an PTV 
Wenn Sie diesen Parameter aktivieren, wird die Versandart an die PTV-Schnittstelle angebunden werden. 

Tooltip: 

Wenn aktiviert, wird die Versandart an die PTV-Schnittstelle angebunden. 

### Tabellenspalte Lieferant 
Hier kann die Lieferantennummer eingetragen werden. 

Tooltip: 

Eingabe der Lieferantennummer 

### Tabellenspalte Keine WBZ aktualisieren 
Wenn dieser Parameter gesetzt ist, wird die Wiederbeschaffungszeit nicht aktualisiert. 

Siehe Kapitel Wiederbeschaffungszeit . 

Tooltip: 

Wenn gesetzt, wird die Wiederbeschaffungszeit nicht aktualisiert. 

### Tabellenspalte Verordnung 
Hier wird die zugehörige Gefahrgutverordnung eingetragen. 

Tooltip: 

Angabe der zugehörigen Gefahrgutverordnung
