# Erfassung eines neuen Termins 
Wiedervorlage erfassen 

Um einen neuen Termin festzusetzen, öffnen Sie über den Button (Wiedervorlage) auf den einzelnen Dialogen das Menü Wiedervorlage erfassen . 

Ein leerer Wiedervorlage-Dialog kann über den Button Wiedervorl. auf der Registerkarte Termine geöffnet werden. 

In der Wiedervorlage können Termine zu verschiedenen Datensätzen oder Vorgängen, wie z.B. Artikel, Kunde, Aufträge, Angebote etc., festgehalten werden. 

Um Überschneidungen bei der Zuordnung von Wiedervorlagen zu vermeiden, ist der Wiedervorlage ein Nummernkreis hinterlegt. 

Alternativ bietet der Dialog eine Übersicht über alle Wiedervorlagen zum ausgewählten Vorgang. 

Als Termin wird standardmäßig das aktuelle Tagesdatum gesetzt. Der Eintrag kann geändert werden. 

Uhrzeit in der Wiedervorlage: 

Ergänzend kann zum Termin eine Uhrzeit für die Wiedervorlage angegeben werden. Die Wiedervorlage erscheint dann erst zu der angegebenen Uhrzeit beim Sachbearbeiter. Wird keine Uhrzeit angegeben, wird automatisch 00:00 in das Feld eingetragen und der Termin wird sofort angezeigt. 

In der Übersicht der Wiedervorlagen wird im Feld Termin auch die Uhrzeit angezeigt. Der Sachbearbeiter kann den Termin auch auf eine andere Uhrzeit verschieben und die Wiedervorlage erscheint erst dann wieder in der Tabelle. 

Standardmäßig wird nun das Feld Datum im Navigator Registerkarte Wiedervorlage neben dem Sachbearbeiter leer gelassen, d.h. alle offenen Wiedervorlagen einschließlich des aktuellen Datums/Uhrzeit werden angezeigt. Wird ein Datum eingegeben, werden nur Wiedervorlagen von diesem Tag angezeigt. 

Termine können nur für den Programmbereich erfasst werden, aus dem heraus die Terminerfassung aufgerufen wurde. Dies wird im Feld Vorgang angezeigt. 

Sie können außerdem festlegen, ob der Termin für eine ganze Abteilung gelten soll. Dies geschieht, indem Sie in der Combobox Abteilungsnr eine Abteilung auswählen. Dann erhält jeder Benutzer dieser Abteilung, wenn er sich in eNVenta ERP anmeldet, die Mitteilung, dass Termine in der Wiedervorlage enthalten sind. 

Siehe Kapitel Abteilungen . 

Oder aber Sie legen den Termin speziell für einen Benutzer fest, indem Sie einen Benutzer aus der Combobox Sachbearbeiter per Mausklick auswählen. Dann erhält nur dieser Benutzer die Meldung, dass ein Termin in seiner Wiedervorlage liegt, wenn er sich in eNVenta ERP anmeldet. 

Wenn Sie einen Termin an den Absender zurücksenden möchten, brauchen Sie ihn dazu nicht unbedingt aus der Combobox Sachbearbeiter aussuchen. Dazu machen Sie in der Tabelle einen Doppelklick auf eine Zelle der Spalten 'Von Sachbearb.', 'Sachbearbeiter' oder 'Abteilung', dann wird dieser Wert übernommen. 

So kann ein Call, oder aber auch ein anderer Vorgang, einfach "zurück" gesendet werden. 

In der Combobox Termin-Typ können Sie den Typ des Termins festlegen ( zur Bearbeitung, zur Information, zur Genehmigung ). 

Um dem Termin eine Priorität zuzuweisen, steht Ihnen die Combobox Priorität zur Verfügung. Hier können Sie wählen zwischen normal, wichtig, sehr wichtig . Diese Prioritäten werden farblich gekennzeichnet. 

Die Priorität kann beim Erzeugen einer Wiedervorlage und im Navigator geändert werden. Wird aus einem Call eine Wiedervorlage erzeugt, wird die Priorität voreingestellt. Wird im Call die Priorität geändert, wird diese bei allen offenen Wiedervorlagen (zu diesem Call) angeglichen. 

Bei der Erfassung eines neuen Termins mit Kennzeichnung zur Bearbeitung kann eine Zuständigkeit in Form eines Eigners hinterlegt werden. Der Eigner wird immer mit dem aktuellen Sachbearbeiter vorbelegt und kann während der Erfassung geändert werden. 

Um eine Terminüberwachung für den Eigner zu erreichen, hinterlegen Sie im Bereich Warnung einen Warntermin für die Überwachung der Bearbeitung. 

Beispiel 1: 
WV Termin = 14.12.2019 16:54, Warnung = 1 Tag -> "Eignerwarnung" = 15.12.2019, die "Warnung" wird das erste Mal am 16.12.2019 angezeigt (= NACH 15.12.2019). 

Beispiel 2: 
WV Termin = 10.03.2019, Warnung = 0 Tag -> "Eignerwarnung" = 10.03.2019, die "Warnung" wird das erste Mal am 11.03.2019 angezeigt (= NACH 10.03.2019). 

Der hinterlegte Eigner und die zur Erledigung des Termins vorgegebene Zeit können nachträglich nicht mehr geändert werden. 

Nur wenn bei der Erstellung des Termins oben genannte Felder gefüllt wurden, erscheinen nicht erledigte Termine mit Eigner und Warnung nach Überschreitung des Warntermins auf der Registerkarte Termine beim Eigner zur Warnung in rot . 

Mit dem Button werden die Änderungen gespeichert, mit wird der Dialog ohne zu speichern geschlossen.

### Button Favorit 

Mit diesem Button können Sie einen Favoriten generieren. D.h. Sie öffnen aus einem Vorgang heraus den Dialog Wiedervorlage erfassen , drücken den Button Favorit und damit steht der zugrundeliegende Vorgang sofort im Navigator im oberen Bereich zur Verfügung. Damit erübrigt sich eine mühsame Suche nach Vorgängen. 

Dabei wird zuerst der Favoritenbereich angezeigt (z.B. Call-Center), in dem der Vorgang sich befindet und darunter dann der eigentliche Favorit. Die Sortierung der Favoriten erfolgt nach der Sortierung im Navigator. 

Um einen Favoriten wieder zu löschen, markieren Sie diesen und löschen diesen über das Menü Favoriten >> Entfernen . 

Tooltip: 

Favorit generieren

### Combobox Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Combobox Abteilung 
Hier wird die Abteilung des Sachbearbeiters angegeben. 

Wenn Sie die Wiedervorlage an eine Abteilung senden, ist folgendes zu beachten! 

Sowie ein Sachbearbeiter der gewählten Abteilung die Wiedervorlage auf erledigt setzt, ist die Wiedervorlage auch bei allen anderen Mitarbeitern der Abteilung erledigt ! Die Wiedervorlage verschwindet aus der aktuellen Anzeige der offenen Termine. 

Eine mögliche Vorgehensweise könnte sein: 

Der Mitarbeiter aus der Abteilung öffnet die Wiedervorlage, entscheidet sich, diese zu bearbeiten und ändert entweder die Wiedervorlage (über die Combobox Sachbearbeiter ) auf sich persönlich oder leitet sich die Wiedervorlage selbst weiter. 

### Feld Termin 
Hier können Sie eingeben, wann die Wiedervorlage beim Empfänger ankommen soll. Das Tagesdatum ist voreingestellt. 

### Combobox Termin-Typ

- zur Bearbeitung - zur Info - zur Genehmigung 

### Combobox Priorität 
Hier geben Sie die Priorität des Termins ein. Zur Verfügung stehen: 
- normal - wichtig - sehr wichtig 

Tooltip: 

Priorität des Termins 

### Combobox Warnung 
Um die Erledigung einer Wiedervorlage zu gewährleisten können Sie hier auswählen, wann der Eigner des Termins eine Wiedervorlage erhalten soll. Zur Verfügung stehen 

- leer - Tage - Woche - Monat 

### Feld Anzahl Tage 
Wenn in der Combobox Warnung Tage ausgewählt wurde, können Sie hier eine Anzahl an Tagen eingeben. 

Tooltip: 

Wenn als Warnung "Tage" ausgewählt ist, können Sie hier die Anzahl der Tage angeben. 

### Combobox Eigner 
Bei der Erfassung eines neuen Termins mit der Kennzeichnung zur Bearbeitung kann eine Zuständigkeit in Form eines Eigners hinterlegt werden. Der Eigner wird immer mit dem aktuellen Sachbarbeiter unzugänglich vorbelegt. 

Wird eine Warnung mit Warnfrist erfasst, kann der Eigner geändert werden. Der Eigner erhält nach Ablauf der Warnfrist eine Wiedervorlage zur Terminüberwachung. 

Der hinterlegte Eigner und die zur Erledigung des Termins vorgegebene Zeit können nachträglich nicht mehr geändert werden. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Selektierte Nachr. 
Wenn Sie eine Zeile der Tabelle markieren, wird in diesem Feld wird angezeigt, welche Nachrichtentexte (Feld Nachricht ) frühere Absender von Wiedervorlagen gesendet haben.
