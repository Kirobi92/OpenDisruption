# Lagerorte Registerkarte LVS Allgemein

Allgemeine Funktionalitäten in eNVenta ERP / Lagerverwaltung / Lagerorte 

# Lagerorte Registerkarte LVS Allgemein 
Lagerorte Registerkarte LVS Allgemein 

Auf dieser Registerkarte nehmen Sie generelle Einstellungen für das Lager-Verwaltungs-System (LVS) vor. Diese Registerkarte steht für beide LVS-Varianten zur Verfügung. 
### Checkbox LVS-Lager 
Wenn Sie diese Checkbox aktivieren, werden Lageraufträge für dieses Lager erzeugt. Ansonsten läuft die Durchführung von Wareneinlagerung und –auslagerung wie im Standard ab. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, werden Lageraufträge für dieses Lager erzeugt. 
Ansonsten läuft die Durchführung von Wareneinlagerung und –auslagerung wie im Standard ab. 

### Feld Startzeit 
Hier geben Sie eine Uhrzeit ein, zu der die Vorkommissionierung auf diesem Lager startet. Alles was bis zu diesem Zeitpunkt nicht gepickt ist, wird erst in die nächste Tour eingeplant. Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Tooltip: 

Geben Sie eine Uhrzeit ein, zu der die Vorkommissionierung auf diesem Lager startet. 

### Feld Startdatum 
Hier geben Sie das Lieferdatum ein, das zum aktuellen Zeitpunkt des Versands auf diesem Lager gültig ist (in Abhängigkeit von der Vorkommissionierung). 

Tooltip: 

Geben Sie das Lieferdatum ein, das zum aktuellen Zeitpunkt des Versands 
auf diesem Lager gültig ist (in Abhängigkeit von der Vorkommissionierung). 

### Feld Crossd.-Diff. 
Hier kann angegeben werden, für wie viele Tage in die Zukunft zum Lieferdatum der Dialog Cross Docking bei der Einlagerung von auftragsbezogener Ware geöffnet werden soll. Ist kein Eintrag vorhanden, wird zum Lieferdatum + 2 Tage der Dialog Cross Docking geöffnet. 

Siehe Kapitel Cross Docking . 

Tooltip: 

Abstand Lieferdatum vom aktuellen Datum fürs Crossdocking. 

### Checkbox Permanente Inventur 
Wenn Sie diese Checkbox aktivieren, aktivieren Sie die permanente Inventur. 

Tooltip: 

Permanente Inventur 

### Feld Deaktiviert bis 
Hier kann die permanente Inventur per Eingabe des Datums bis zu einem bestimmten Zeitpunkt deaktiviert werden. 

Tooltip: 

Restmengenprüfung 

### Feld Deakt. Grund 
Hier kann ein Grund eingegeben werden, weshalb die permanente Inventur deaktiviert wurde. 

Tooltip: 

Geben Sie einen Grund für die Deaktivierung der permanenten Inventur ein. 

### Feld Füllstand Uml. 
Dieses Feld steht in Zusammenhang mit der Lageroptimierung auf Basis von LHM-Maximalbeständen. 

Damit nicht unnötige Umlagerungen stattfinden, sich eine Verdichtung also „lohnt“, geben Sie hier den Füllstand für die Umlagerung an. D.h. den Füllstand, ab dem das Ladehilfsmittel zur Umlagerung vorgeschlagen werden soll. Falls kein Füllstand angegeben ist, wird automatisch ‚75% Füllstand‘ als Grundlage übernommen. 

Siehe Kapitel Umlagerung intern . 

Tooltip: 

Füllstand, ab dem das Ladehilfsmittel zur Umlagerung vorgeschlagen wird. 

### Combobox Lagerleiter 
Hier wählen Sie einen Drucker aus. Dieser Drucker befindet sich in der Regel bei der Lagerleitung. 

Tooltip: 

Wählen Sie den Drucker für wichtige Dokumente des Lagerleiters. 

### Combobox Etik. Lagerleiter 
Hier wählen Sie einen Etikettendrucker aus. Dieser Drucker befindet sich in der Regel bei der Lagerleitung. 

Tooltip: 

Wählen Sie den Drucker für Etiketten. 

### Feld Max. Anz. Etik. 
Hier geben Sie die maximale Anzahl für den Druck der Etiketten ein. 

Damit kann vermieden werden, dass bei einer Fehleingabe zu viele Etiketten gedruckt werden. 

Tooltip: 

Maximale Anzahl für den Etikettendruck. 

### Combobox Leitung Logistik 
Hier wählen Sie den Logistik-Leiter für das gewählte Lager aus. 

Tooltip: 

Wählen Sie den Leiter für das gewählte Lager aus. 

### Feld Entb.-Express 
Für die Bearbeitung von Expressaufträgen am Scanner kann hier eingetragen werden, nach wie viel Sekunden der Sachbearbeiter den Expressauftrag annehmen muss, bevor sich der Pick vom Sachbearbeiter löst und der Pick auf den Pickmonitor freigegeben wird. 

Tooltip: 

Zeitraum für die automatische Entbindung von Expressaufträgen in Sekunden. 

### Feld Timeout Pickzeit 
Hier kann ein Timeout festgelegt werden, nach wie vielen Minuten sich der Pick vom Sachbearbeiter löst und wieder zurück auf den Pickmonitor geht. Dabei gehen alle noch nicht kommissionierten Picks, bei denen der Lagerplatz noch nicht angescannt wurde, zurück auf den Pickmonitor. Ist der Lagerplatz schon angescannt, hat der Pick zwingend vom Sachbearbeiter bearbeitet zu werden. 

Tooltip: 

Timeout Pickzeit in Minuten 

### Checkbox Zuladungsliste drucken 
Hier wird der Drucker ausgewählt, an dem die Zuladungslisten für Zuladungsartikel gedruckt werden sollen. 

Tooltip: 

Zuladungsliste drucken 

### Checkbox Zuladungsetikett drucken 
Hier wird der Etikettendrucker ausgewählt, an dem die Zuladungsetiketten für Zuladungsartikel gedruckt werden sollen. 

Tooltip: 

Zuladungsetikett drucken 

### Checkbox Keine negativen Lagerbestände 
Wenn Sie diese Checkbox aktivieren, wird verhindert, dass in einem LVS-Lager negative Lagerplatzbestände entstehen. 

Tooltip: 

Keine negativen Lagerbestände 

### Checkbox Nullmenge buchen 
Ist dieser Schalter gesetzt, wird eine Buchung der Menge 0 im Lagerauftrag ermöglicht. 

Ist dieser Schalter nicht gesetzt, so wird in diesem Moment keine Buchung vorgenommen. 

Wenn die Checkboxen 

- Prüfziffer am Lagerplatz buchen? - Lager LHM fähig - 0 buchen 
nicht gesetzt sind, läuft im Programm das 2-stufige LVS. 

Tooltip: 

Ermöglicht eine Buchung der Menge 0 im Lagerauftrag. 

### Checkbox WV bei Bestandskorrektur 
Wenn Sie diese Checkbox aktivieren, erhält der Logistikleiter bei jeder Bestandskorrektur (während des Pickvorgangs) eine automatische Wiedervorlage zur Info. 

Tooltip: 

Damit erhält der Logistikleiter bei jeder Bestandskorrektur (während des Pickvorgangs) eine automatische Wiedervorlage zur Info. 

### Checkbox Verkaufslager 
Wenn Sie diese Checkbox aktivieren, wird bei einer Umlagerung in dieses Lager automatisch ein Verkaufsetikett gedruckt. 

Tooltip: 

Aktiviert bei einer Umlagerung auf dieses Lager automatisch den Druck eines Verkaufsetiketts. 

### Checkbox Fixe Versandbehälter verwenden 
Wenn dieser Parameter aktiviert ist, können Fix-Behälter für den Versand zugeordnet werden. 

Tooltip: 

Ist dieser Parameter aktiviert, können die Fix-Behälter für den Versand zugeordnet werden. 

### Checkbox Rückstandserinnerung unterdrücken 
Wird durch das LVS automatisch eine Rückstandsposition erzeugt (Anzahl gepickt < benötigte Anzahl), wird normalerweise für den Erfasser des Auftrags eine Wiedervorlage zur Information erzeugt und gesendet. Wenn Sie dies verhindern wollen, aktivieren Sie diese Checkbox. 

Tooltip: 

Mit diesem Parameter wird der automatische Versand einer Wiedervorlage an den Erfasser des Auftrags verhindert. 

### Checkbox Lagerplatznummer autom. ermitteln 
Wenn Sie diese Checkbox aktivieren, wird beim Erstellen eines Lagerplatzes die Platznummer automatisch aus Ebene, Gang, Haus, x- und y-Koordinate ermittelt. 

Tooltip: 

Beim Erstellen eines Lagerplatzes wird automatisch die Platznummer aus Ebene, Gang, Haus, x- und y-Koordinate ermittelt. 

### Checkbox Prüfziffer am Lagerplatz benutzen 
Wird dieser Parameter gesetzt, erhalten alle Lagerplätze eine dreistellige Prüfziffer, welche per Random Zahlengenerator ermittelt wird. Diese Prüfziffer wird am Lagerplatz im Barcode mit angedruckt. 

Bei der Einlagerung müssen nun die komplette Platznummer und die Prüfziffer eingegeben werden. 

Bei der Auslagerung sind „Platznummer + Prüfziffer“ oder nur die Prüfziffer möglich. 

Tooltip: 

Alle Lagerplätze erhalten eine dreistellige Prüfziffer, welche per Random Zahlengenerator ermittelt wird. 

### Checkbox Lager LHM-fähig 
Wenn diese Checkbox gesetzt ist, hat das Lager die Voraussetzung erfüllt, dass hier mit Ladehilfsmitteln gearbeitet werden kann. 

Tooltip: 

Wenn aktiviert, kann mit Ladehilfsmitteln gearbeitet werden. 

### Checkbox Gewicht Beachten 
Wenn Sie diesen Parameter aktivieren, wird das Gewicht bei Platz- und Behälterberechnung beachtet. 

Tooltip: 

Beachte Gewicht bei Platz und Behälterberechnung.
