# IDEA Export Registerkarte Allgemein 
IDEA Export Registerkarte Allgemein 

Auf dieser Registerkarte können neue Export-Definitionen angelegt werden. 
### Combobox Export Definition 
Wenn Sie die Bezeichnung definiert haben, wird dieser als Export-Definition verwendet und hier angezeigt. 

Erst wenn Sie eine Export Definition gewählt haben, öffnet sich die Registerkarte Medien . 

### Checkbox Keine Leerspalten 
Wenn Sie diese Checkbox aktivieren, werden in allen zu exportierenden Datensätzen die Spalten, in denen kein Wert enthalten ist, nicht übertragen. 

Ein visueller Vergleich zwischen Exportdatei und der Registerkarte Spalten ist dann nicht empfehlenswert. 

Diese Checkbox wird nicht gespeichert, ist aber bei Dialogstart aktiviert. 

Tooltip: 

Leere Spalten unterdrücken 

### Feld Bezeichnung 
Um eine neue Exportdefinition anzulegen, geben Sie in dieses Feld eine Bezeichnung ein, die Sie frei wählen und speichern diesen Eintrag. 

Tooltip: 

Angabe der Bezeichnung 

### Feld Firma 
Hier geben Sie die Firmenanschrift des Datenlieferanten an. 

### Feld Standort 
Hier geben Sie den Standort des Datenlieferanten an. 

### Feld Bemerkung 
Dieses Feld steht für individuellen Text zur Verfügung. 

### Combobox Rechn.leg.kreis 
Hier wählen Sie einen Rechnungslegungskreis aus. 

Siehe Kapitel Rechnungslegungskreise in der Fibu . 

Tooltip: 

Angabe des Rechnungslegungskreises
