# Artikel Registerkarte Wettbewerber 
Artikel Registerkarte Wettbewerber 

Auf der Registerkarte Wettbewerber haben Sie die Möglichkeit, Informationen des Wettbewerbs über den selektierten Artikel zu sammeln. So sind Sie vorbereitet, um z.B. in einem Kunden- oder Lieferantengespräch fundiert argumentieren zu können. 

Voraussetzung: der Wettbewerber hat zwingend im Wettbewerberstamm (Modul Vertrieb ) angelegt zu sein. 

Um eine neue Wettbewerbsinformation hinzuzufügen, klicken Sie auf den Button und füllen die Spalten der Registerkarte. Zusätzlich zu den Informationen kann eine Bewertung der Wettbewerbsinformation vorgenommen werden. Dies kann in Form von +, ++, -, -- oder einfach als verloren/gewonnen gespeichert werden. 

Die Daten können direkt in der Tabelle bearbeitet werden. 
### Tabelle Wettbewerbsdaten 

### Tabellenspalte Wettbewerber 
Hier wird die Nummer des Wettbewerbers angegeben. 

Tooltip: 

Angabe der Wettbewerbernummer 

### Tabellenspalte AKL-Nummer 
Angabe einer AKL-Nummer. In diesem Fall ist das eine AKL-Nummer, zu welcher der Wettbewerber z.B. im Auftrag zur Information erfasst worden ist. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Tabellenspalte Firma 1 
Die Firmenbezeichnung wird automatisch zur entsprechenden AKL-Nummer eingelesen. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Artikelgruppe 
Mit Hilfe dieser Combobox ordnen Sie dem Artikel eine Artikelgruppe zu. Die in der Combobox wählbaren Gruppen werden im Menü Stamm >> Artikelgruppen angelegt. 

Siehe Kapitel Artikelgruppen . 

Tooltip: 

Angabe der Artikelgruppe 

### Tabellenspalte Artikel 
Die Artikelnummer wird manuell festgelegt und nicht automatisch vom System. 

Die Länge der Artikelnummer beträgt maximal 30 Zeichen und kann alphanumerische Zeichen sowie Sonderzeichen enthalten. Eine abweichende Länge kann im Dialog Parameter Verkauf festgelegt werden. 

Siehe Kapitel Parameter Verkauf Registerkarte Allgemein . 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Lieferanten-Artikel 
Die Lieferanten-Artikelnummer wird angezeigt. 

Tooltip: 

Angabe der Lieferanten-Artikelnummer 

### Tabellenspalte Text 
Hier kann ein individueller Text eingetragen werden. 

### Tabellenspalte Referenzwert 
Hier können Sie eine Bewertung für den Artikel beim Lieferanten auswählen. 

Tooltip: 

Referenzwert des Artikels. 

### Tabellenspalte Preis 
In dieser Spalte steht eine Preisinformation aus Kundensicht (Verkauf). 

Tooltip: 

Preis des Artikels beim Wettbewerber 

### Tabellenspalte EK 
Der Einkaufspreis des Artikels wird eingetragen. 

In diesem Fall wird hier der Preis eingetragen, zu dem der Wettbewerber den Artikel bezogen hat. 

Tooltip: 

Preis 

### Button Satz hinzufügen 

Mit diesem Button fügen Sie einen neuen Eintrag in die Tabelle ein. 

### Button Speichern 

Mit diesem Button speichern Sie die Einstellungen in der Tabelle. 

### Button Löschen 

Mit diesem Button löschen Sie einen Eintrag der Tabelle.
