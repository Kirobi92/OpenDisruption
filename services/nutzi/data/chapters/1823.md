# Rahmenbestellungen Registerkarte Kopfdaten 
Rahmenbestellungen Registerkarte Kopfdaten 

Auf dieser Registerkarte können weitere Einstellungen vorgenommen werden. 
### Feld Bestätigung 
Hier kann die Auftrags-/Bestätigungsnummer des Lieferanten eingetragen werden. 

Tooltip: 

Angabe der Bestätigungsnummer 

### Feld Best. Datum 
Für eine bessere Nachverfolgbarkeit eines Vorgangs, kann hier das Datum der Bestätigung eingetragen werden. 

Tooltip: 

Bestätigungsdatum 

### Combobox Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Tooltip: 

Angabe der Währung 

### Feld Kürzel RB 
Hier geben Sie ein Kürzel / eine Abkürzung für die Rahmenbestellung an. 

Tooltip: 

Kürzel Rahmenbestellung 

### Feld Abschlussdatum 
In dieses Feld geben Sie das Datum ein, an welchem Tag die Vereinbarung getroffen wurde. 

Tooltip: 

Abschlussdatum: 

In das Feld Abschlussdatum geben Sie das Datum 
über den Abschluss der Vereinbarung an. 

### Checkbox Periodische Abrufe 
Wenn diese Checkbox aktiviert wird, besteht die Möglichkeit, Rahmenbestellungen periodisch zu generieren. 

Siehe Absatz Menü Optionen / Periodisch generieren. 

Tooltip: 

Wenn aktiviert, können Rahmenbestellungen periodisch generiert werden. 

### Feld Startdatum 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld Menge 
Hier geben Sie die Menge für den Abruf ein. 

Tooltip: 

Menge des Abrufs 

### Feld Termin 
Dieser Termin entspricht dem Tag des Monats, an dem die Bestellung durchgeführt werden soll. Z. B. 1, 5, 15... 

Tooltip: 

Geben Sie an, an welchem Tag bestellt werden soll. 

### Dock Panel Periode 

In diesem Bereich legen Sie die Zeitabstände für die Bestellungen fest.
