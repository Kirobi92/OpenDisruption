# Ereignisdefinition Registerkarte Verknüpfung 
Registerkarte Verknüpfung 

Diese Registerkarte bietet die Möglichkeit, eine Verknüpfung auf die Komponente zu erstellen, die im Kopfbereich ausgewählt ist. 

Im Feld auf der rechten Seite wird dann das Statement angezeigt. 
### Button Hinzufügen 

Mit diesem Button wird das selektierte Feld hinzugefügt. 

Tooltip: 

Hinzufügen des selektierten Felds 

### Button AND 

Mit diesem Button wird die Add-Verknüpfung hinzugefügt. 

Tooltip: 

Add Verknüpfung hinzufügen 

### Button OR 

Mit diesem Button wird die OR-Verknüpfung hinzugefügt. 

Tooltip: 

OR Verknüpfung hinzufügen
