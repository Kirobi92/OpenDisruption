# Aufgabe ändern Registerkarte Auftrag 
Aufgabe ändern Registerkarte Auftrag 

Ist der Auftragsposition bereits eine Aufgabe zugeordnet, öffnet sich der Dialog Aufgabe ändern für die Bearbeitung dieser verknüpften Aufgabe. Auf dieser Registerkarte können Sie die aktuelle Projektaufgabe auch manuell über den Button Auftragspos . mit einer Auftragsposition verknüpfen. Über den Button Löschen wird die Referenz zu einer Auftragsposition wieder zurückgesetzt. 

Wenn die Aufgabe beendet wird, wird die zugeordnete Auftragsposition automatisch auf Fakturafreigabe (Status 3) gesetzt. Dies passiert auch dann, wenn die zu dieser Projektaufgabe erfassten Leistungen abgerechnet werden. Die erfassten Leistungen werden dann nicht in einen neuen Auftrag eingefügt, sondern erhalten direkt den Status "Fakturiert" und es wird lediglich die verknüpfte Position im vorhandenen Auftrag auf Fakturafreigabe gesetzt. So wird verhindert, dass Leistungen doppelt abgerechnet werden. 

Wenn auf dieser Registerkarte ein Eintrag vorhanden ist, wird die Registerkarte mit einem * gekennzeichnet. 
### Button Auftrag 

Über diesen Button öffnet sich der Dialog Auftragsauswahl . Hier kann der Aufgabe direkt ein Auftrag zugeordnet werden. Dies ist besonders für Dienstleistungen interessant, die sich aus dem Auftrag ergeben und welche ebenfalls abgerechnet werden sollen. Dabei stehen nur Aufträge zur Auswahl, die noch nicht fakturiert wurden. 

Bei der Leistungsabrechnung wird zunächst geprüft, ob es Leistungen gibt, die einem neu zu erzeugenden Auftrag zugeordnet werden müssen. Falls dem so ist, muss nach wie vor eine Auftragsart angegeben werden. Falls alle Leistungen einer Auftragsposition bzw. einem Auftrag zugeordnet sind, entfällt die Prüfung auf die Auftragsart. Nachdem die Leistungsabrechnung gelaufen ist, wird nun nicht mehr nur angezeigt, welche Aufträge erzeugt wurden, sondern auch welche Aufträge erweitert wurden. Es wird auch angezeigt, wenn ein Auftrag nicht erweitert werden konnte (wenn er z.B. schon fakturiert ist).

### Button Auftragspos. 

Über diesen Button kann die Aufgabe mit einer Auftragsposition verknüpft werden. 

Tooltip: 

Auftragsposition zuordnen 

### Button Löschen 

Über diesen Button wird die Referenz zu einer Auftragsposition wieder zurückgesetzt. 

### Feld Belegnummer 
Der entsprechende Auftragsbeleg kann eingetragen werden. 

Tooltip: 

Angabe der Belegnummer 

### Feld Auftragsart 
Wenn dieses Feld gefüllt ist, wird in der Auftragserfassung die hier hinterlegte Auftragsart voreingestellt. Sie kann jederzeit geändert werden. 

Dieses Feld steht in direktem Zusammenhang mit dem Parameter Auftragsart fix im Dialog Auftragsarten . 

Tooltip: 

Eingabe der Auftragsart 

### Feld Fixposnummer 
Falls die Aufgabe einer bestimmen Auftragspositionsnummer zugeordnet werden soll, tragen Sie diese hier ein. 

Tooltip: 

Eindeutige Positionsnummer des Datensatzes.
