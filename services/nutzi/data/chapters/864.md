# Sicherheiten Registerkarte Sicherheiten 
Sicherheiten Registerkarte Sicherheiten 

Auf dieser Registerkarte pflegen Sie die Detailinformationen zu den Sicherheiten. Dazu benutzen Sie die Comboboxen und Felder unterhalb der Tabelle. Die Tabelle selbst wird automatisch mit diesen Einstellungen gefüllt. 
### Tabelle 

### Tabellenspalte Typ 
Hier wird der Typ angegeben. 

Tooltip: 

Angabe des Typs 

### Tabellenspalte Gruppe 
Hier wird die Gruppennummer angegeben. 

Tooltip: 

Angabe der Gruppennummer 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Bewertung 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Tabellenspalte Löschdatum 
Hier wird das Löschdatum angegeben. 

Tooltip: 

Angabe eines Löschdatums 

### Combobox Gruppe 
In dieser Combobox wählen Sie die Gruppe der Sicherheit aus. Dies können z.B. Immobilien und Fahrzeuge sein. 

Wenn Sie die Gruppe Avale gewählt haben, werden unterhalb der Tabelle andere / weitere Felder eingeblendet. 

Wenn Sie die Gruppe Kraftfahrzeuge gewählt haben, werden weitere Felder zur Bewertung sichtbar und zugänglich. 

Tooltip: 

Angabe der Gruppennummer 

### Feld Bezeichnung 
Wenn Sie eine Gruppe ausgewählt haben, wird in diesem Feld automatisch der Name der Gruppe angezeigt. 

Tooltip: 

Angabe der Bezeichnung 

### Feld Aufbew.ort 
In dieses Feld tragen Sie ein, wo die Sicherheit aufbewahrt wird. Dies kann ein Bankdepot sein für Wertpapiere oder eine Garage für ein KFZ. 

### Combobox Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

In diesem Fall kann aus der Combobox auch ein anderer Sachbearbeiter gewählt werden. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Mit diesem Button können Sie ein Projekt suchen. 

Siehe Kapitel Projektauswahl . 

### Checkbox Limitwirksam 
Wenn der Sicherheitstyp 'erhalten' lautet, ist diese Checkbox sichtbar. Wird diese gesetzt, fließt der Bewertungsbetrag unter Berücksichtigung der Abschreibung und des Gültig-bis-Datums in das Kreditlimit des Kunden ein. 

### Feld Gültig von 
In diesem Feld geben Sie an, ab wann die Bewertung der Sicherheit gelten soll. 

### Feld Gültig bis 
In diesem Feld geben Sie an, bis wann die Bewertung der Sicherheit gelten soll. 

### Feld Bewertung 
In diesem Feld geben Sie den Betrag ein, mit dem Sie die Sicherheit finanziell bewerten. 

Tooltip: 

Angabe des Betrags 

### Feld Erinnerungsfrist 
Um rechtzeitig vor Ablauf des Gültigkeitszeitraums informiert zu werden, geben Sie hier die Anzahl Tage an. Sie werden dann entsprechend x-Tage vorher daran erinnert. 

Tooltip: 

Erinnerungsfrist (Angabe in Tagen) 

### Feld Zinssatz 
Dieses Feld wird nur sichtbar und zugänglich, wenn Sie die Gruppe Avale auswählen. 

Für erteilte Sicherheiten können Sie in diesem Feld einen Prozentwert für die Zinsberechnung eintragen. 

Tooltip: 

Angabe des Zinssatzes 

### Feld Bewertung
