# BMEcat - Katalogexport Registerkarte Exportoptionen 
BMEcat - Katalogexport Registerkarte Exportoptionen 

Diese Registerkarte bietet Funktionen für den Export von Keywords. 

Zur Pflege stehen die Checkbox Fremdartikelnummer , das Feld Fremdartikelnummer und die Checkbox Stichwörter exportieren zur Verfügung. Über diese Funktionen ist es möglich, Fremdnummern, die am Artikel gepflegt sind, im BMEcat als Keywords zu exportieren. Damit wird die Artikelsuche noch einfacher, da mehr Suchkriterien zur Verfügung stehen. 

Wenn Sie die Checkbox Stichwörter exportieren aktivieren, werden diese exportiert. 

Der Dialog Fremdartikelnummern-Typen öffnet sich über den Button . Hier können Sie die Fremdartikelnummern-Typen bestimmen. 

Hier stehen auch Funktionen zur Verfügung, um auch sogenannte Zubehörartikel bzw. Alternativartikel exportieren zu können. Hier legen Sie fest, welche Zubehörtypen exportiert werden sollen und mit welchem BMEcat-Referenztyp. 

Wenn Sie die Check box Alt.-Artikel exp. aktivieren, werden auch Artikel aus dem Katalog, welche untereinander als Alternativartikel verknüpft sind, exportiert. 

Es werden nur Referenzen auf Artikel exportiert, welche ebenfalls im Export enthalten sind. 

Über den Bereich Exportoptionen / Preise können Preise für den Export festgelegt werden. 
### Checkbox Klassen Systeme 
Wenn Sie diese Checkbox aktivieren, wird das Klassifikationssystem exportiert. 

Tooltip: 

Klassifikationssystem exportieren 

### Combobox Merkmal 
Hier wird die Merkmalsnummer angegeben. 

Tooltip: 

Angabe der Merkmal-Nr. 

### Checkbox Artikelmerkmale 
Wenn Sie diese Checkbox aktivieren, werden die Artikel mit Merkmalen exportiert; ob sie einer Klasse zugeordnet sind oder nicht. 

Tooltip: 

Artikel mit Merkmalen exportieren 

### Checkbox Klassenlose Merkmale 
Wenn Sie diese Checkbox aktivieren, werden auch klassenlose Merkmale exportiert. 

Dies kann die Dauer des Exports vervielfältigen! 

Tooltip: 

Auch Klassenlose Merkmale werden exportiert. 

Achtung! Dies kann die Dauer des Exports vervielfachen! 

### Checkbox Fremdartikelnummern exportieren 
Wenn Sie diese Checkbox aktivieren, werden Fremdartikelnummern exportiert. 

Tooltip: 

Ist dieser Parameter aktiv, werden die Fremdartikelnummern 
am Artikel je nach Konfiguration als Stichwörter exportiert. 

### Feld Liste 
Hier wird eine Liste von Fremdartikelnummern-Typen angegeben, die, wenn am Artikel gepflegt, als Stichwörter exportiert werden. 

Tooltip: 

Eine Liste von Fremdartikelnummern-Typen, 
die, wenn am Artikel gepflegt, als Stichwörter 
exportiert werden. 

### Button Öffnen 

Mit diesem Button öffnet sich der Dialog Fremdartikelnummern-Typen . 

Siehe Kapitel Parameter Auswahlliste . 

### Checkbox Stichwörter Exportieren 
Wenn Sie diese Checkbox aktivieren, werden Stichwörter exportiert. 

Tooltip: 

Wenn dieser Parameter aktiv ist, werden die 
am Artikel gepflegten Stichwörter mit exportiert. 

### Checkbox Nur E-Commerce Artikel 
Wenn diese Checkbox aktiviert ist, werden nur Artikel exportiert, die nicht als "k.E-Commerce" markiert sind. 

Tooltip: 

Ist dieser Parameter aktiv, werden nur Artikel exportiert, die 
nicht als "k. E-Commerce" markiert sind. 

### Checkbox Artikel-Referenzlinks exportieren 
Wenn diese Checkbox aktiviert ist, werden die Referenz Links der einzelnen Artikel mit exportiert. 

Tooltip: 

Wenn dieser Parameter aktiv ist, werden die 
Referenz Links der einzelnen Artikel mit exportiert. 

### Feld Referenz-Link-Typ 
Dieses Feld enthält eine Liste von Referenz-Link-Typen, die für den Artikel mit exportiert werden sollen. 

Tooltip: 

Enthält eine Liste von Referenzlink-Typen, die für den Artikel mit exportiert werden sollen. 

### Tabelle Link-Typ 

In diesem Bereich stehen Link-Typen wie z.B. Artikelbild zur Verfügung. Über den Button können weitere Link-Typen angelegt werden. 

Tooltip: 

Exportiert werden Referenzlinks von Artikeln. 
Die Angabe des Referenzlink-Typs wird, entsprechend des Mappings in dieser Tabelle, in MIME_PURPOSE umgesetzt. 
Wenn kein Link-Typ angegeben ist oder der Link-Typ eines Artikels keinem MIME_PURPOSE zugeordnet ist, wird der Referenzlink als MIME ohne PURPOSE exportiert. 
### Tabellenspalte Link-Typ 
Hier wird der Referenz-Link-Typ angegeben. 

Tooltip: 

Angabe des Referenz-Link-Typs 

### Tabellenspalte Mime_Purpose 
Tooltip: 

Erwünschter Verwendungszweck des MIME-Dokumentes

Mit diesem Button können Sie weitere Link-Typen hinzufügen. 

### Button Löschen

die alternativen Artikel als Referenzen exportiert.

Filtert die exportieren Artikel nach der Artikellistung des angegebenen Kunden.
