# Bestellautomat Registerkarte Umlagerungsaufträge 
Bestellautomat Registerkarte Umlagerungsaufträge 

Auf dieser Registerkarte besteht die Möglichkeit Umlagerungsaufträge zu generieren, um ein Zentrallager aufzufüllen. Dazu stehen Ihnen der Button Umlagern und diese Registerkarte zur Verfügung. 

Die möglichen Umlagerungspositionen müssen zuvor über den Bestellvorschlag gesucht werden, damit diese im Bestellautomaten verwendet werden können. 

Folgende Regeln müssen beachtet werden: 
- Vorschlag nur für ein einzelnes Lager (Keine Lagergruppe). - Dem Lager ist unbedingt eine Kundennummer zuzuordnen. - Dem Lager muss eine Auftragsart zugeordnet sein. Empfehlung: Diese Auftragsart beinhaltet das Lager als Ziellager. - Umlagerungsvorschläge werden stets mit dem deterministischen Verfahren berechnet, unabhängig vom Dispoverfahren aus dem Artikelstamm. - Der Bestellvorschlag wird mit KZAutomatik = 5 belegt. 
In dieser Registerkarte werden alle erstellten Umlagerungsaufträge – entspricht Verkaufsaufträgen – angezeigt, welche zuvor generiert wurden. Per Doppelklick auf einen Eintrag wird der entsprechende Verkaufsauftrag geöffnet. 

Über den Button Umlagern öffnet sich ein Dialog und Sie werden gefragt, ob Sie einen Umlagerungsauftrag generieren möchten. Nach dieser Abfrage wird eine Liste von Aufträgen nach Lagernummer und Belegart erzeugt. Diese werden in der Registerkarte Umlagerungsaufträge angezeigt; mit einem Doppelklick kann ein Auftrag aufgerufen werden. 

Das Lager, von dem die Ware für die Umlagerung entnommen wird, muss im Dialog Lager pro Artikel in der Spalte ‚Std. Lager Umlagerung‘ per Checkbox gepflegt werden. 

Ist zu einem Artikel die Checkbox Std. Lager Umlagerung nicht aktiviert, so wird das Zentrallager als Entnahmelager eingetragen.

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden.

### Tabellenspalte Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben.
