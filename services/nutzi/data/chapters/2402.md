# Stapelbearbeitung in der Finanzbuchhaltung 
Die eNVenta Finanzbuchhaltung bietet die Möglichkeit, Buchungen eine Stapelnummer mitzugeben. Über die Stapelnummer können alle dazugehörenden Buchungen selektiert und abgestimmt werden. 

Die Einstellung für die Stapelbearbeitung nehmen Sie in den Parametern der Finanzbuchhaltung (Registerkarte Allgemein , Checkbox Buchen mit Stapelnummer ) vor. Wenn hier Buchen mit Stapelnummer aktiviert ist, wird in einigen Dialogen, wie z.B. im 

- Buchen - Bankeinzug - Zahlungsverkehr - Übernahme - Auszug-/Kassenimport - Wiederkehrende Buchungen - Zinsrechnungen - OP-Verrechnungen - und Pauschalwertberechnung 
ein Feld für die Stapelnummer zur Verfügung gestellt. 

Die Stapelnummern werden im Dialog Stapelbearbeitung verwaltet. Dieser Dialog öffnet sich im Journal über den Button Stapelbearbeitung . Es werden alle gefundenen Stapel in die Tabelle eingelesen. Wenn Sie auf eine Zeile in der Tabelle der Stapelbearbeitung einen Doppelklick ausführen, öffnet sich der Dialog Journal und zeigt alle Buchungen des Stapels an. 

Über den Button Definitiv setzen wird die Stapelnummer entfernt. Erst jetzt können die Datensätze an die Kostenrechnung übergehen. 

Beispiel für das Buchen mit Stapelnummer: 

- Bank-Kontoauszüge: Alle Buchungen dieses Kontoauszugs werden mit der gleichen Stapelnummer eingebucht. Die Abstimmung des Saldos ergibt eine Abweichung, weil das falsche Bankkonto bebucht wurde. Im Dialog Stapelbearbeitung können Sie diesen Stapel markieren und löschen. Stimmt die Abstimmung, setzen Sie diesen Stapel auf definitiv. - Lieferantenrechnungen: Sie buchen einen Stapel Rechnungen mit der gleichen Stapelnummer ein. Die Rechnungsbeträge addieren Sie und stimmen diese mit der Buchen-Abstimmsumme ab. Ermitteln Sie eine Abweichung können Sie über die Stapelbearbeitung diesen Stapel mit allen Belegen ins Journal einlesen und gezielt nach der Abweichung suchen.
