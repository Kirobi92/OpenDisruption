# Kapitel NV.ERP.Base.Customer.cdCustomer.decReturnsPercentOrder

In diesem Feld kann für die Rücknahme von Bestellungen ein Prozentsatz eingetragen werden, um den Gutschriften bei Rücknahme der Ware gemindert werden.
