# Kapitel NV.ERP.Base.Common.cdParameterFI.lngFactoringTransitAccount

Tragen Sie hier ein Sachkonto (Transitkonto) ein. Dieses Konto dient als „Scharnier“ bzw. Zwischenkonto. 

Das Transitkonto wird z.B.in der Verbandsabrechnung für Debitoren bzw. Kreditoren und Kursbewertung benötigt. 

Wenn das Transitkonto für die notwendigen Buchungsvorgänge nicht hinterlegt ist, werden Sie mit einer Meldung darauf hingewiesen.
