# Kundengruppen 
Kundengruppe 

Dieser Dialog öffnet sich im Kunde >> Stamm >> Kundengruppe . In diesem Dialog können Kunden in Gruppen zusammengefasst werden. Die Anlage von Kundengruppen ist zwingend notwendig (d.h. es muss jeweils mindestens eine Gruppe angelegt sein). Insbesondere für analytische Zwecke und für eine bessere Strukturierung der Stammdaten ist es sehr hilfreich, wenn differenzierte Kundengruppen gebildet wurden. 

In der Auswahlliste sehen Sie alle Kundengruppen mit Kürzel, die bereits bestehen. Wenn Sie eine bestehende Kundengruppe markieren, sehen Sie im unteren Teil des Dialogs alle Rabatte und Zuschläge, die zu dieser Kundengruppe hinterlegt wurden. 

Eine neue Kundengruppe legen Sie an, indem Sie zuerst mit dem Button alle Datenfelder leeren. Tragen Sie nun im Datenfeld ID das Kürzel und im Datenfeld Bezeichnung die exakte Bezeichnung der neuen Gruppe ein. Rabatte zu einer Kundengruppe können Sie mit dem Button aus dem Menü Rabatte/Zuschläge auswählen und zuordnen. 

Siehe Kapitel VK Rabatte und Zuschläge . 

Um die Änderungen zu speichern, klicken Sie den Button . Nun steht die neue Gruppe im Dialog Kunde zur Verfügung. 

Wenn Sie eine Gruppe entfernen möchten, markieren Sie diese in der Liste und betätigen den Button . Sie können nur Gruppen löschen, die nicht verknüpft sind, d.h. denen keine Kunden zugeordnet sind.
