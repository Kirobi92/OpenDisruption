# Kapitel NV.ERP.Base.Common.cdSystemMM.sAdMatCodeSeparator

Hier wird ein frei definiertes Trennzeichen (z.B. einen Punkt oder ein Semikolon) für das Werbemittel angegeben. Optimaler Weise wird ein Zeichen verwendet, das in keiner Artikelnummer vorkommt. 

Tooltip: 

Optimaler Weise wird ein Zeichen gewählt, das in keiner Artikelnummer vorkommt. Dieses Zeichen dient als Trennzeichen in der späteren Auftragserfassung um die Artikelnummer von der WM-Nummer lösen zu können.
