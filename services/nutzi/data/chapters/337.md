# Daten Umsetzung Registerkarte Autom. Umsetzung 
Daten Umsetzung Registerkarte Automatische Umsetzung 

Auf dieser Registerkarte können Sie eine automatische Umsetzung definieren. Die automatische Umsetzung kann beispielsweise bei Lieferanten erfolgen. Hier wurden bereits im Vorfeld die Umsetzungen im Dialog Import Schlüssel Verwaltung eingegeben. 

Dazu wählen Sie aus der Combobox Umsetz. Funktion einen sogenannten Look-up-Eintrag aus. Im Feld Beschreibung wird angezeigt, was genau ermittelt wird. 

In der Tabelle werden die Parameter gelistet, welche für die gewählte Funktion registriert sind. 

Um importierten Positionstext im Auftrag übersichtlicher darstellen zu können, kann der Positionstext zeilenweise dargestellt werden. Dafür steht der Wert Zeilenumbruch zur Verfügung. Dazu ist in der CSV-Datei im Positionstext an der entsprechenden Stelle ein Zeichen bzw. eine Zeichenfolge vorhanden (Beispiel ¶, LF (LineFeed), /n). Nach jedem Zeichen / Zeichenfolge (z.B. LF) wird beim Import ein Zeilenumbruch durchgeführt, damit der Positionstext im Auftrag übersichtlich dargestellt wird. Diese Zeichen können im Gateway für den entsprechenden Import angegeben werden. 

Beispiel: 
Ausgangsdatei, (Spalte Postext) als CSV mit Semikolon als Trennzeichen: 

………;"Elementbreite / Elementhöhe in mm: /1822/1393 
LF Elementfarbe: Ton: ROMA 9016 (Matt glatt) 
LF Kasten: Ausführung Linksroller 
Rückseite: geschlossen 138er LF Kastengröße, Welle: Typ: 60er 
LF INTEGO Ausstattung: Träger: 8 mm 
Winkel: 15 mm, Schiene links: Typ: S-1";….. 

Zeilenumbruch innerhalb einer Spalte beim Import der Daten setzen 
Dazu wählen Sie hier die Funktion 'Ersatz' aus der Combobox Umsetz. Funktion aus. Als erster Parameter wird die Spalte ausgewählt, als 2. das Zeichen / die Zeichenfolge, die zum Zeilenumbruch ersetzt werden soll, z.B. "\n" und im 3. Parameter wählen Sie in der Combobox den Eintrag "3 : Zeilenumbruch" aus. Siehe nachfolgende Abbildung. 

Daten Umsetzung Zeilenumbruch 

### Combobox Umsetz. Funktion 
Hier wählen Sie einen Eintrag für die Umsetzung aus; einen sogenannten Look-up-Eintrag. Mit Hilfe der automatisierten Umsetzung kann anhand der importierten Daten der entsprechende Wert in eNVenta ermittelt und weiterverwendet werden, z.B. aus Lieferantennummer AB10 wird die Lieferantennummer 4712. 

Zur Verfügung stehen: 
- Ermittlung der Lieferantennr. - Ermittlung der Artikelnr. - Umrechnungsfaktor -> Prozentsatz - Trim - Dateiverweis - Ersetzung 
Im Feld Beschreibung wird angezeigt, was genau ermittelt wird. 

In der Tabelle werden die Parameter gelistet, welche für die gewählte Funktion registriert sind. 

Tooltip: 

Angabe des Komponententyps 

### Feld Beschreibung 
Hier wird zu jeder Umsetzungsfunktion angezeigt, was ermittelt wird. 

Folgende Beschreibungen stehen zur Verfügung: 

- Ermittlung der Lieferantennr.: Ermittelt die eNVenta Lieferantennummer aus einer externen Lieferantennummer. - Ermittlung der Artikelnr.: Ermittelt die eNVenta Artikelnummer aus einer externen Lieferantennummer und der Werknummer. - Umrechnungsfaktor -> Prozentsatz: Konvertiert einen Faktor in einen Prozentsatz (Beispiel: 0,7 -> 30%) - Trim: Entfernt alle Vorkommen zu spezifizierender Zeichen oder einer Zeichenkette am Anfang und/oder Ende der Import-Zeichenfolge. - Dateiverweis: Übernimmt bzw. kopiert den angegebenen Dateiverweis. - Ersetzung: Ersetzt ein beliebiges Zeichen mit einem anderen Zeichen. 

Tooltip: 

Angabe einer Beschreibung 

### Tabelle Parameter 

Wenn eine Umsetzungsfunktion ausgewählt wurde, werden in dieser Tabelle die entsprechenden Parameter zur Bearbeitung eingelesen. Dann können der importierte Wert und der fixe Wert zugeordnet werden. 

Siehe Kapitel Import-Schlüssel Verwaltung . 
### Tabellenspalte Nr. 
Hier wird eine fortlaufende Nummer angegeben. 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung des Parameters angegeben, der zur Umsetzungsfunktion passt. 

### Tabellenspalte Typ 
Hier wird der Typ angezeigt, z.B. String. 

### Tabellenspalte Importierter Wert 
Hier wird angegeben, welches Feld den Wert, z.B. die externe Lieferantennummer, enthält. 

### Tabellenspalte Fixer Wert 
Hier wird die Zuweisung zum Feld in eNVenta ausgewählt, z.B. Lieferant. 

Zur Umsetzungsfunktion Trim stehen hier folgende Werte zur Verfügung: 
- Trimme Zeichen vom Start - Trimme Zeichen vom Ende - Trimme Zeichen von Start+Ende - Trimme Text vom Start - Trimme Text vom Ende - Trimme Text von Start+Ende
