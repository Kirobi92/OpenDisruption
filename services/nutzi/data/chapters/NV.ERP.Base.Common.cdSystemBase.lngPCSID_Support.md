# Kapitel NV.ERP.Base.Common.cdSystemBase.lngPCSID_Support

Für den Versand einer E-Mail an den Support von Nissen & Velten kann hier eine Adress-, Kunden- oder Lieferantennummer eingetragen werden. Hierbei muss der Kontakt zum Support einmalig in den Stammdaten entsprechend erfasst werden, damit die E-Mail und weitere Kontaktdaten entsprechend ausgelesen werden können. 

Tooltip: 

Für den automatischen Versand einer E-Mail an den Support von N+V geben Sie hier die AKL-Nummer ein.
