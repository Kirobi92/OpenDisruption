# Adresse Registerkarte Memo 
Adresse Registerkarte Memo 

Auf dieser Registerkarte geben Sie individuelle interne Informationen ein. 

Wenn die Adresse in einen Kundenstammsatz umgewandelt wird, wird das Vertriebsmemo in das Auftragsmemo übertragen. 
### Feld Allgemein 
Hier geben Sie allgemeine Informationen zur selektierten Adresse ein. 

Tooltip: 

Allgemeine Informationen können eingetragen werden. 

### Button Auswahl 

Mit diesem Button gelangen Sie in den Dialog Textbausteine . Damit besteht die Möglichkeit Textbausteine für den Adressstamm zu verwenden. Mittels der Textbausteine können entsprechende Vorlagen verwaltet und im Anschluss mit Inhalt gefüllt werden. 

Siehe Kapitel Textbausteine . 

Tooltip: 

Auswahl der Textbausteine öffnen 

### Feld Vertrieb 
Informationen, die Sie hier eingeben, werden jedes Mal aufgerufen, wenn Sie die Adresse z.B. im Besuchsbericht, in der Call-Erfassung oder in der Serienaktivität aufrufen. 

Tooltip: 

Enthält interne Textinformationen.
