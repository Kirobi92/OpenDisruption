Grundeinstellungen für eNVenta ERP / Parameter Drag and Drop 

# Parameter Drag and Drop Registerkarte Artikelbild 
Parameter Drag Drop Registerkarte Artikelbild 

Über diese Registerkarte können Sie einstellen, ob die Sachbearbeiter ein Artikelbild per Drag & Drop in den Artikelstamm ziehen können. Standardmäßig ist diese Funktion deaktiviert und muss entsprechend vom Administrator freigeschaltet werden. 

Damit bei der Anlage eines Artikelbilds dies für alle verfügbar ist, muss der zentrale Speicherort (ein Server File Pfad) der Artikelbilder angegeben werden sowie ein Link-Typ für die automatische Anlage des Referenz-Links. 

Dies funktioniert nicht auf einem F5-Broker!!! 
### Checkbox Deaktiviere Artikelbild einfügen 
Wenn Sie diese Checkbox aktivieren, wird die Option deaktiviert, eine von extern gezogene Datei dem Ziel als Artikelbild hinzuzufügen. 

Tooltip: 

Deaktiviert die Option, eine von extern gezogene Datei dem Ziel als Artikelbild hinzuzufügen. 

### Feld Standardpfad 
Hier geben Sie den Standard-Pfad ein. 

Tooltip: 

Speicherort für neue Artikelbilder, welche per Drag&Drop eingefügt werden. 

### Combobox Link-Typ 
Aus dieser Combobox können Sie den Link-Typen auswählen. 

Tooltip: 

Referenzlink-Typ für neue Artikelbilder, welche per Drag&Drop eingefügt werden.
