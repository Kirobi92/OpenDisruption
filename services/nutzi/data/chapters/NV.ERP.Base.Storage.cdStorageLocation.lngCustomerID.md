# Kapitel NV.ERP.Base.Storage.cdStorageLocation.lngCustomerID

Hier geben Sie die Kundennummer für interne Beschaffungsaufträge an. Diese wird verwendet, wenn aus dem System heraus ein Umlagerungsauftrag erstellt wird. Der Umlagerungsauftrag benötigt diese Kundennummer. 

Tooltip: 

Angabe der Kundennr. für interne Beschaffungsaufträge.
