# Hilfskostenstellen Registerkarte Umlagen 
Hilfskostenstellen Registerkarte Umlagen 

Durch die Angaben in der Registerkarte Umlagen wird die Verteilung von Kostenarten/Leistungen bzw. Kostenstellen auf diese Kostenstelle durch die festgelegten Umlageschlüssel und Werte möglich. Die Werte können periodengerecht eingegeben werden, um eine periodengerechte Umlage zu gewährleisten. Einträge / Änderungen in der Tabelle erfolgen automatisch, sobald in der Tabelle Umlagen ein Eintrag bzw. eine Änderung getätigt wird. 

Eine prozentuale Verteilung von Kostenarten auf die Kostenstelle ist ohne Eingabe eines Umlageschlüssels auf dieser Registerkarte immer möglich. 

Eine Zuordnung, Änderung oder Entfernung einer Umlage ist nur über den Dialog Umlage-Detail möglich. 
### Button Umlage 

Dieser Button öffnet den Dialog Umlage-Detail . 

Siehe Kapitel Umlage-Detail .
