# Belastungsarten Registerkarte Rollenrechte 
Belastungsarten Registerkarte Rollenrechte 

Diese Registerkarte dient dazu den Belastungsarten und damit den Benutzern entsprechende Rollen/Rechte zuzuordnen. Auf diese Weise können Sie einschränken, dass die Benutzer nur bestimmte Einkaufarten betrachten und/oder einfügen können. 

Auch wenn für einen Sachbearbeiter die Rollensperrungen eingerichtet wurden, werden diese Belastungen in den Statistiken berücksichtigt. 

Hat ein Sachbearbeiter mehrere Rollen zugeordnet, so genügt es wenn bereits für eine Rolle der Zugriff gesperrt wurde. 
### Button Neu 

Über diesen Button legen Sie eine neue Zeile an und können anschließend die Rolle auswählen, für welche Sie die Berechtigung einschränken möchten. 

### Button Löschen 

Dieser Button löscht eine Zeile der Tabelle und damit eine Rollensperrung. 

### Tabelle 

### Tabellenspalte Rolle 
Hier wird dem Sachbearbeiter eine Rolle zugeordnet. 

Tooltip: 

Eine Rolle wird einem Sachbarbeiter zugeordnet. 

### Tabellenspalte Ansicht Sperren 
Beim Setzen der Checkbox in dieser Spalte kann der Sachbearbeiter keine Belastungen dieser Belastungsart betrachten. Daher wird auch automatisch die Checkbox in der Spalte 'Speichern sperren' gesetzt. 

Tooltip: 

Der Sachbearbeiter kann diesen Vorgang nicht sehen. 

### Tabellenspalte Speichern sperren 
Wenn die Checkbox in dieser Spalte gesetzt ist, kann der Sachbearbeiter die Belastungen dieser Belastungsart betrachten, allerdings keinerlei Änderungen vornehmen. 

Tooltip: 

Der Sachbearbeiter kann die Vorgänge ansehen, aber nicht ändern.
