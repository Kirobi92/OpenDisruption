# Kontext Im-/Exportieren 
Adapter Context Im/Exportieren 

Dieser Dialog öffnet sich im Gateway Adapter Kontext >> Optionen >> Kontext Im/Exportieren , wo Einstellungen zum Export bzw. Import gemacht werden können. 

Zum Einspielen einer kompletten Konfiguration inkl. Gateway-Events, Prozessregeln und X-Mapping-Definition kann die entsprechende XML-Datei im Importpfad ausgewählt werden. Mittels Klick auf den Button Import wird die Konfiguration als neuer Gateway Adapter Kontext eingespielt.Wenn Sie eine Einstellung exportieren möchten, können Sie dies über den Button Export entsprechend vornehmen. 

Es ist möglich Gateway Adapter inkl. weiteren notwendigen abhängigen Daten, bestehend aus Event Management und X-Mapping Einstellungen, zu importieren und zu exportieren. Auf diese Weise können Einstellungen zwischen Mandanten einer Datenbank einfach übernommen werden. 

Die Einstellungen für einen Gateway Kontext exportiert werden. 

Die notwendigen Daten werden als XML-Datei im ausgewählten Pfad gespeichert. 

Die Einstellungen für einen Gateway Kontext können importiert werden. 

Über die Auswahl des Pfades wählen Sie die zu importierende XML-Datei. 

Der Ablauf für den Import kann mehrstufig gestaltet werden: 

- Import und Analyse der Datei - Ermittlung notwendiger Konfigurationen - Einstellungen von Konfigurationen durch den Benutzer - Import nach erfolgter Einstellung. 
Die Datei wird eingelesen und der Adapter-Typ in der Importdatei wird ermittelt. 

Für die Konfigurationen werden die vorhandenen Prozess-Setting Interfaces genutzt. Dabei werden 2 Hauptkonfigurationsgruppen aufgebaut: Allgemeine Konfigurationen und Adapter-spezifische Konfigurationen. 

Zu den allgemeinen Konfigurationen gehören: 

- Eine Auflistung von potentiell überschreibbaren Adaptern, von denen einer ausgewählt werden kann, um durch den aktuellen Import überschrieben werden zu können. - Eine Auflistung der Event Management-Events welche durch diesen Import gehandelt werden würden. 
Die Adapter-spezifischen Konfigurationen werden durch den identifizierten Adapter in Kombination mit dem übergebenen XML ermittelt. 

Einstellungen von Konfigurationen durch den Benutzer: 

Der Button Konfigurieren öffnet den Dialog Prozessregeln . Dort finden die Konfigurationen statt. 

Siehe Kapitel Prozessregeln . 

Import nach erfolgter Einstellung 

Mit dem Button Import wird dann der tatsächliche Import aufgerufen. Die Import-Logiken steuern dann den Import Parameter-abhängig.
