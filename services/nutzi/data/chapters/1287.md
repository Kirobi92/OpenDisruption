# Rahmenauftrag Registerkarte Abrufe

Verkauf / Rahmenauftrag / Rahmenauftrag Registerkarte Positionen 

# Rahmenauftrag Registerkarte Abrufe 
Registerkarte Abrufe 

Auf dieser Registerkarte werden bereits getätigte Abrufe dieses Artikels angezeigt. 
### Tabelle 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Abrufmenge 
Hier wird die Abrufmenge angegeben. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Lieferdatum 
Hier wird der Liefertermin angegeben. 

Tooltip: 

Angabe des Liefertermins 

### Tabellenspalte Vertrag EK 
Hier wird die Vertragsnummer angegeben. 

Tooltip: 

Angabe der Vertragsnummer 

### Tabellenspalte Kürzel RB 
Hier geben Sie ein Kürzel / eine Abkürzung für die Rahmenbestellung an. 

Tooltip: 

Angabe des Suchnamens 

### Tabellenspalte Termin 
Hierbei handelt es sich um ein Textfeld, in welchem ein beliebiger Hinweistext als "Stichtag" angegeben werden kann. 

Tooltip: 

Textfeld zur Angabe eines Stichtags
