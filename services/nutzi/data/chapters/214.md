# Suchabfragen verwalten 
Abfragen verwalten 

Dieser Dialog öffnet sich im Dialog Suchbedingung über den Button Abfrage . Die Verwaltung bzw. Speicherung von Abfragen (zur Wiederverwendung) steht Ihnen überall zur Verfügung, wo Sie über SQL-Befehle suchen können, z.B. Artikelstamm. 

Hier kann eine Abfrage gespeichert werden, welche zuvor im Dialog Suchbedingung angelegt wurde. Für einen neuen Eintrag geben Sie im Feld Bezeichnung einen Namen für die Abfrage ein. Speichern Sie Ihre Abfrage. Pro Abfrage erscheint dann eine Zeile in den gespeicherten Abfragen (linke Tabelle). 

Vorhandene Abfragen können mit dem Button in den Dialog Suchbedingung übernommen werden und von dort aus kann die Suche dann endgültig gestartet werden. 

Wenn eine Abfrage bereits gespeichert wurde und danach eine Änderung im Select-Statement vorgenommen wurde, so kann danach diese Änderung nicht in der bereits bestehenden Abfrage neu gespeichert/geändert werden.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
