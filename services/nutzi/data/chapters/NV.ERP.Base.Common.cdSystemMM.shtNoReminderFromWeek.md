# Kapitel NV.ERP.Base.Common.cdSystemMM.shtNoReminderFromWeek

Wenn Sie diesen Parameter aktivieren und eine Kalenderwoche wird als Termin vorgegeben, wird daraus kein Lieferdatum ermittelt. 

Ist die Checkbox nicht aktiviert, wird bei Angabe einer Kalenderwoche immer der Freitag der Woche automatisch als Lieferdatum eingetragen.
