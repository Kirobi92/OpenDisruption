# Kapitel NV.ERP.MM.Production.cdParamProduction.decMaterialOverhead

Geben Sie den Zuschlag für die Materialgemeinkosten für die Kalkulation in % ein. Der Gemeinkostenzuschlag wird dann für jede Produktion übernommen. Der Gemeinkostenzuschlag kann für jeden Arbeitsplan und jeden Betriebsauftrag separat geändert werden.
