# Datenimport bearbeiten Registerkarte Preisgruppen 
Import Bearbeiten Registerkarte Preisgruppen 

Auf dieser Registerkarte werden die Preisgruppen eingelesen. Die eingelesenen Daten können nach unterschiedlichen Kriterien gefiltert werden. 
### Button Bearbeiten 

Mit diesem Button öffnet sich der Dialog Import-Editor Preisgruppen , um markierte Preisgruppen zu bearbeiten. 

Siehe Kapitel Import-Editor Preisgruppen . 

### Button PG Verarbeiten 

Mit diesem Button werden nur die markierten Daten für die Preisgruppen Details (keine Artikelstammdaten!) verarbeitet. 

Tooltip: 

Ausgewählte Preisgruppen in den Schattenstamm übernehmen 

### Feld SQL-Bedingung 
Tooltip: 

SQL Bedingung aktiv 

### Tabelle 

### Tabellenspalte Importschlüssel 
Hier wird der Import-Schlüssel angegeben. 

Tooltip: 

Der "Import-Schlüssel" ist ein generierter Wert, der als eindeutige ID (Referenzschlüssel) zur Referenzierung des Datensatzes dient. Er wird aus den Haupteigenschaften des Datensatzes zusammengesetzt. Jeder Datensatz benötigt einen eigenen, eindeutigen Import-Schlüssel. Es wird kein neuer Datensatz angelegt, wenn es bereits einen Datensatz mit demselben Import-Schlüssel gibt, sondern der bereits vorhandene überschrieben/aktualisiert. 

### Tabellenspalte Haupt-Importschlüssel 
Hier wird der Haupt-Importschlüssel angegeben. 

Tooltip: 

Angabe des Haupt-Importschlüssels 

### Tabellenspalte Preisgruppe 
Hier wird die Preisgruppe angegeben 

Tooltip: 

Angabe der Preisgruppe 

### Tabellenspalte Preis 
Hier wird der Preis angegeben. 

Tooltip: 

Preis 

### Tabellenspalte Rabatt 
Hier wird ein prozentualer Rabatt oder Zuschlag angegeben. Ein positiver Wert bedeutet Rabatt, ein negativer Wert bedeutet Zuschlag. 

Tooltip: 

Angabe eines prozentualen Rabatts / Zuschlags 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Tabellenspalte Auslaufdatum 
Hier kann für eine Preisgruppe ein Auslaufdatum eingetragen werden. 

Tooltip: 

Auslaufdatum der Gültigkeit des Datensatzes
