# E-Mail Vorlagen 
E-Mail-Vorlage 

Dieser Dialog öffnet sich mit einem Doppelklick auf einen E-Mail-Eintrag im Dialog Dokument Vorlagen . Hier geben Sie Text für E-Mails ein. 

Betreff, Text und Anhang können nur für ausgewählte Vorlagen erfasst werden. 

Tipp: 

Damit in der E-Mail auch das Rechnungsdatum bzw. die Rechnungsnummer ausgelesen werden kann, ist es notwendig dass vor der Erstellung der E-Mail das Property "oObj.oSalesDocument.dtDocumentDate" gefüllt wird. Dieses Property ist das richtige, wenn das Druckdatum ausgelesen werden soll. Standardmäßig wird in der E-Mail dann das Datum MIT Uhrzeit angezeigt. Wenn Sie das verhindern möchten, ist es notwendig in der E-Mail-Vorlage ein entsprechendes Format anzugeben. 

Z.B. könnten Sie statt 

</oObj.oSalesDocument.dtDocumentDate/> 

dann 

</oObj.oSalesDocument.dtDocumentDate; d. MMMM yyyy/> 

verwenden.
