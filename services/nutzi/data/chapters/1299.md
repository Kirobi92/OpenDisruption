# Rabatt Verkauf Registerkarte Tabelle 
Rabatt Verkauf Registerkarte Texte 

Hier werden sämtliche, bereits angelegte Rabatte angezeigt. 
### Tabelle 

### Tabellenspalte Rabatt 
Hier wird eine Rabattnummer angegeben. 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Artikel bis 
Hier wird die Bis-Artikelnummer angegeben. 

Tooltip: 

Angabe der Bis-Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte AG/RG 
Hier wird die Artikelgruppen- oder Rabattgruppennummer angegeben. 

### Tabellenspalte Kunde 
Hier geben Sie die Kundennummer an, entweder um einen neuen Rabatt für diesen Kunden anzulegen oder zur Selektion. 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Firma 1 
Hier wird die Zeile 1 des Firmennamens angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Projekt 
Wenn der Rabatt auf ein bestimmtes Projekt bezogen ist, wird hier die entsprechende Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Tabellenspalte Kundengruppe 
Hier wird eine Kundengruppe ausgewählt. 

Tooltip: 

Zeigt die Kundengruppe. 

### Tabellenspalte Änderung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der letzten Änderung eingetragen. 

Tooltip: 

Datum und Uhrzeit der letzten Änderung 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die tatsächlich gefundene Anzahl an Datensätze angezeigt.
