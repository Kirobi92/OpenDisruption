# Plantafel 
Plantafel 

Die Plantafel bietet einen Überblick über die Auslastung der Ressourcen. Dazu werden selektiv Ressourcen gelistet. Für jede Ressource werden auf der Zeile die Reservierungen für einen bestimmten Zeitraum graphisch angezeigt. 

Per Drag&Drop können Sie hier Produktions- und Betriebsaufträge sowie Arbeitsgänge ein- und umplanen. Mit der Zoom-Funktion kann die Darstellung der Aufträge und Kapazitäten, bezogen auf einen Tag bis hin zu ganzen Wochen oder Monaten, angezeigt werden. Durch die Auswahl eines Arbeitsgangs auf der Plantafel werden automatisch alle wichtigen Auftragsinformationen angezeigt. Per Mausklick gelangen Sie dann direkt in den einzelnen Betriebsauftrag. 

Mit einem Klick auf einen Betriebsauftrag werden dessen Daten unter dem Gantt-Chart angezeigt. Mit einem Doppelklick auf einen Balken öffnet sich ebenfalls der Betriebsauftrag. 

Wenn beim Neuterminieren von einem oder mehreren Betriebsaufträgen in den Plantafeln Ausnahmesituationen auftreten (z.B. Terminüberschreitungen des Liefertermins im zugehörigen Verkaufs-Auftrag), erhalten Sie eine Hinweismeldung mit einer Liste der entsprechenden Überschneidungen. Diese Meldung erhalten Sie ebenfalls bereits bei der Einplanung eines Betriebsauftrags, wenn es nicht möglich ist, den Liefertermin einzuhalten. 

### Modi zum Gantt-Diagramm 
Mit einem Klick der rechten Maustaste auf einen Balken öffnet sich ein Menü, das Ihnen verschiedene Möglichkeiten für das Gantt-Diagramm bietet: 

Auswahlmodus: 
Es können keine Änderungen innerhalb des Gantt-Diagramms vorgenommen werden. 

Anlegemodus: 
Es können diverse Änderungen an den Daten innerhalb des Gantt-Diagramms vorgenommen werden (das betrifft die Bearbeitung der Balken). Sie können einen Arbeitsgang verschieben, einer anderen Ressource zuordnen oder auch komplett löschen. Beim Verändern eines Arbeitsgangs erscheint immer der Dialog Terminierung Vorgang . 

Siehe Kapitel Terminierung eines Produktionsvorgangs . 

Gruppe kollabieren: 
Alle Arbeitsgänge werden auf eine Zeile des Gantt-Diagramms zusammengefasst. 

Gruppe expandieren: 
Alle Arbeitsgänge einer Gruppe werden separat zu den zugeordneten Ressourcen angezeigt. 

Löschen: 
Ein markierter Arbeitsgang wird gelöscht.

### Button BA markieren 

Wenn Sie einen Balken markieren und diesen Button drücken, werden alle zum gleichen Betriebsauftrag gehörenden Balken rot dargestellt. Der Detail-Level wird automatisch umgestellt. 

### Button BA-Farben 

In der Plantafel der Produktion findet eine Farbunterscheidung der verschiedenen Betriebsaufträge statt. Hierfür klicken Sie auf diesen Button, um max. 20 Betriebsaufträge in unterschiedlichen Farben darstellen zu können. Die Farben werden rein zufällig zugeordnet. Wenn eine Reservierung der Ressource aus einem anderen Programmbereich (z.B. Service) vorhanden ist, werden diese Balken immer in schwarz angezeigt. 

### Button Simul. markieren 

Wird auf diesen Button geklickt, so wird die eingelesene Simulation innerhalb der Plantafel rot markiert. Hierfür muss eine Simulationsnummer eingetragen sein, damit diese Simulation in der Plantafel angezeigt wird. Die simulierten Reservierungen der Ressourcen werden so berücksichtigt, als wäre der Betriebsauftrag komplett in den Echt-Betrieb eingeplant. 

Für die Simulation stehen die gleichen Funktionalitäten zur Verfügung wie für jeden anderen Betriebsauftrag, z.B. das Verschieben der Reservierung und eine entsprechende neue Einplanung. 

Für die Eingabe der Simulations-Nummer klicken Sie auf den Button , um den Dialog Auswahl Simulation zu öffnen. 

Es kann immer nur eine Simulation in der Plantafel angezeigt werden. 

Siehe Kapitel Auswahl Simulation .

### Radiobutton Aufträge 
Wenn Sie diesen Radiobutton anklicken, zeigt der Balken einen ganzen Betriebsauftrag von der Startzeit der ersten Reservierung für die Ressource bis zum Ende der letzten Reservierung der Ressource. Die Beschriftung zeigt die Betriebsauftragsnummer und die Artikelnummer des Produkts. 

Da Lücken nicht gezeigt werden, sind auch Überschneidungen mit anderen Vorgängen auf der Ressource sichtbar, ohne dass dies tatsächlich eine Parallel-Einlastung sein muss. 

Die Farben zeigen den Status des Arbeitsgangs: 

Blau = eingeplant 
Gelb = freigegeben 
Grün = fertig 

### Radiobutton Arbeitsgang 
Der Balken zeigt einen ganzen Arbeitsgang von der Startzeit der ersten Reservierung für die Ressource bis zum Ende der letzten Reservierung der Ressource. Die Beschriftung zeigt die Betriebsauftragsnummer und die Arbeitsgangbezeichnung. 

Da Lücken nicht gezeigt werden, sind auch Überschneidungen mit anderen Vorgängen auf der Ressource sichtbar, ohne dass dies tatsächlich eine Parallel-Einlastung sein muss. 

Die Farben zeigen den Status des Arbeitsgangs: 

Blau = offen / geparkt 
Gelb = in Arbeit 
Grün = fertig 

### Radiobutton Alle Details 
Es werden alle Detailreservierungen gezeigt. Lücken entstehen z.B. durch Pausen und Arbeitszeitregelungen. 

Die Beschriftung zeigt die Betriebsauftragsnummer und die Arbeitsgangbezeichnung.
