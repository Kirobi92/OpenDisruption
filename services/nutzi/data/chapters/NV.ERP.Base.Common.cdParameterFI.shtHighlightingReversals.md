# Kapitel NV.ERP.Base.Common.cdParameterFI.shtHighlightingReversals

Wenn Sie diese Checkbox aktivieren, werden Stornobuchungen in eNVenta farblich hervorgehoben und können so schneller erkannt werden. 

Die farbliche Kennzeichnung der Buchungen ist wie folgt: 

- Stornobuchungen: rot - Ursprungsbuchungen: grün - Neue Buchungen: schwarz 

Tooltip: 

Farbliche Hervorhebung von Stornobuchungen.
