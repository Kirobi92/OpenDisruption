# Rahmenbestellungen Registerkarte Texte 
Rahmenbestellung Registerkarte Texte 

Auf dieser Registerkarte können Sie Informationen zur Rahmenbestellung hinterlegen. 

Bei der Generierung einer Bestellung aus der Rahmenbestellung heraus (Button Generiere ), werden der Bestell-Starttext und der Bestell-Endtext automatisch übernommen. 

Die Texte werden NICHT übernommen, wenn zu einer bereits angelegten Bestellung eine Position aus einer Rahmenbestellung hinzugefügt wird. 
### Feld Bestellmemo 
Hier wird das Bestell-Memo angegeben und angezeigt, wenn der Lieferant oder Artikel in Anfrage, Bestellung oder Belastung ausgewählt wird. 

Tooltip: 

Angabe des Bestell-Memos 

### Feld Bestell-Starttext 
Hier kann ein individueller Text als Starttext im Vorgang eingetragen werden. Dieser kann auf den Formularen angedruckt werden. 

### Button Html-Editor 

Tooltip: 

Html-Editor zur Bearbeitung von Html-Texten. 

### Feld Bestell-Endetext 
Hier kann ein individueller Text als Endetext im Vorgang eingetragen werden. Dieser kann auf den Formularen angedruckt werden. 

### Button Suche 

Dieser Button öffnet den Dialog Textbausteine zur Selektion eines Textbausteins.
