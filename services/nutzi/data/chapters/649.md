# Text-Wizard Registerkarte Felder 
Text-Wizard Registerkarte Felder 

Bietet ein Dialog oder auch eine Funktion mehrere Positionslisten an, werden diese hier angeboten. 

Alle zur Verfügung stehenden Felder für den Aufbau einer Vorlage werden hier angeboten. Welche Felder dies sind, hängt davon ab, aus welchem Modul heraus der Text-Wizard geöffnet wird. 

In der Registerkarte Felder hat eine Collection angewählt zu werden. Siehe nachfolgende Beispiel-Screenshots. 

Text-Wizard 1 

Ein Doppelklick auf die Collection fügt jetzt auf der Registerkarte Text {{LIST ...}} ein. 

Text-Wizard 2 

Anzeigen einer Collection 

Zum Anzeigen einer Collection ist darauf zu achten, dass in der Collection (siehe Abbildung Text-Wizard 3 ) der gelb markierte Begriff Text... ersetzt wird durch das Setzen eines Hochkommas vor und nach dem eingegebenen Text und / oder Wert aus der Collection. Ohne Hochkommas (rot markiert in Abbildung Collection im Text-Wizard ) funktioniert die Syntax nicht.
