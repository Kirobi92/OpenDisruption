# X-Mapping Editor Registerkarte CSV-Feld 
Registerkarte CSV-Feld 

Hier werden Einstellungen für das CSV-Feld vorgenommen. 
### Combobox Wert in " " 
Hier geben Sie den Wert der Spalte in Anführungszeichen an, z.B. "Wert". 

Diese Einstellung definiert diese Option nur für diese Spalte. Sie können allerdings für alle Spalten diese Option auf der Zeile definieren. 

Tooltip: 

Der Wert der Spalte steht in Anführungszeichen, z.B.: "Wert". 

Diese Einstellung definiert diese Option nur für diese Spalte. 
Sie können allerdings für alle Spalten diese Option auf der Zeile definieren. 

### Feld Nr. CSV-Spalte 
Hier wird die Spaltennummer des Felds angegeben. 

Sie können hier definieren, welche Spalte, d.h. welche Nummer (beginnend mit 1) der CSV-Zeile diesem Feld entspricht. 

Wenn Sie das Feld leer lassen, ergibt sich die Spaltennummer für das Feld aus den vorherigen CSV-Feldern, d.h. in der Reihenfolge ihrer Definition ergibt sich auch die Spaltennummer.
