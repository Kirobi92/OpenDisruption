# Parameter Webshop Registerkarte Allgemein 
Registerkarte Allgemein 

Hier definieren Sie die allgemeinen Einstellungen für E-Mails. 

Alternativ zum Versand über den eigenen Mailservice des Webshops kann der SMTP-Dienst von eNVenta ERP übernommen werden. 

Wenn Sie die SMTP-Einstellungen aus den System-Parametern nutzen wollen, entfernen Sie alle Felder auf der Registerkarte, bis auf die Absender-Mail-Adresse. 
### Feld Absender E-Mail 
Angabe des Absenders der E-Mail. Diese Adresse erscheint immer als Absender der E-Mail. 

Wenn eine Antwort auf diese E-Mail vom System nicht abrufbar ist, hat dies unbedingt innerhalb des Mailtextes erwähnt zu werden. Es gibt von eNVenta ERP kein technisches Abfangen von Antwort-E-Mails. 

Um sicherzugehen, dass eine E-Mail an die richtige Adresse gesendet wird, prüft eNVenta beim Speichern oder Ändern einer Mailadresse, ob die Mail-Adresse der „Norm“ entspricht. Falls nicht wird ein Hinweis ausgegeben. Dabei wird das Speichern nicht grundsätzlich verhindert. Auf diese Weise werden z.B. Outlook-Adressbucheinträge wie z.B. „#EW_ERP“ bei Bedarf gespeichert und genutzt. 

Dabei läuft die Prüfung ausschließlich bei Anlage und Änderung einer Mail-Adresse. Werden andere Einträge, z.B. des Kundenstammsatzes geändert, so kommt diese Meldung nicht. 

Werden Stammdaten über das Gateway importiert, so werden u.U. fehlerhafte Mailadressen importiert (ggfs. mit Warnung). Bei einem Import unter Verwendung des Jobservers erfolgt dies ebenfalls in Form einer Warnung (im Jobprotokoll). 

### Feld Name 
Hier geben Sie die IP-Adresse bzw. den Namen des Servers ein. 

Wird der Servername angegeben, ist sicherzustellen, dass der Webshop den SMTP-Server über den DNS-Server finden kann. 

### Feld Port 
Hier geben Sie den Port an, auf welchen der SMTP-Dienst läuft. In der Regel wird der Port 25 genutzt. 

### Feld Benutzer 
Hier geben Sie den Benutzer für den SMTP-Dienst an. Wenn der SMTP-Dienst eine Authentifizierung benötigt, kann hier der Benutzername angegeben werden. 

### Feld Passwort 
Hier geben Sie das Passwort zum Benutzer für den SMTP-Dienst an. Das Passwort wird nicht in Klarschrift angezeigt.
