# Parameter Vertrieb Registerkarte Allgemein 
Parameter Vertrieb Registerkarte Allgemein 

Auf dieser Registerkarte werden die Einstellungen vorgenommen. 
### Combobox WM Trennz. 
Hier wählen Sie ein Trennzeichen für das Werbemittel aus. Optimaler Weise wird ein Zeichen gewählt, das in keiner Artikelnummer vorkommt. Dieses Zeichen dient als Trennzeichen in der späteren Auftragserfassung, um die Artikelnummer von der WM-Nummer lösen zu können. 

Tooltip: 

Optimaler Weise wird ein Zeichen gewählt, das in keiner Artikelnummer vorkommt. Dieses Zeichen dient als Trennzeichen in der späteren Auftragserfassung um die Artikelnummer von der WM-Nummer lösen zu können. 

### Combobox WMCode Länge 
Hier wählen Sie die Länge des Werbemittelcodes aus. 

Tooltip: 

Länge des Werbemittelcodes 

### Combobox Artikelnr Länge 
Hier wählen Sie die Länge einer Default-Artikelnummer aus, in Verbindung mit dem Werbemittelcode. 

Tooltip: 

Länge einer Default-Artikelnummer in Verbindung mit dem Werbemittelcode. 

### Feld Besuchsdauer 
Hier geben Sie die Besuchsdauer vor, für die Einplanung neuer Termine. Die Besuchsdauer wird pro Besuchsbericht angepasst. 

Tooltip: 

Dauer des Besuchs für die Einplanung neuer Termine. 

### Feld E-Mail-BBest. 
Für den Versand der Besuchsbestätigungen (aus Dialog Besuchskalender bzw. Dialog Besuchstermin-Verwaltung ) kann hier eine entsprechende E-Mail-Vorlage eingetragen werden. 

Tooltip: 

E-Mail-Vorlage für Besuchsbestätigungen 

### Button Suche 

Mit diesem Button öffnet sich der Dialog Vorlagenauswahl zur Auswahl einer E-Mail-Vorlage. 

Siehe Kapitel Dokument erstellen (Vorlage auswählen) . 

Tooltip: 

Dokumentvorlage Suchen 

### Button Öffnen 

Mit diesem Dialog gelangen Sie in die Dokumentenverwaltung, um eine neue Vorlage zu erstellen. 

Siehe Kapitel Dokument Vorlagenstamm . 

Tooltip: 

Dokumentvorlagen Verwalten 

### Combobox Ress. Grp. Vertreter 
Hier geben Sie die Ressourengruppe an, welche für die Terminplanung (meist Außendienstmitarbeiter) verwendet werden soll. 

Tooltip: 

Ressourcengruppe der Vertreter
