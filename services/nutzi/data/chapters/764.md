# Rücknahmetexte 
Rücknahmetexte 

Dieser Dialog öffnet sich im Artikel >> Stamm >> Rücknahmetexte . Hier können Sie für die Rücknahme eines Artikels einen Text, ein Gültigkeitsdatum und ein Auslaufdatum festlegen. 

Indem Sie den Rücknahmetexten Gültigkeiten zuordnen (Spalten ‚Von Datum‘ / ‚Bis Datum‘) sind diese Rücknahmetexte auch zeitlich steuerbar. D.h. wenn sich z.B. zum nächsten Jahreswechsel eine gesetzliche Regelung auswirkt, so kann dies hier komfortabel gesteuert werden. In diesem Zusammenhang ist auch die Spalte ‚Auslaufdatum‘ zu sehen. 

Sind am Entscheidungstag mehrere Texte gültig, so ist derjenige spezifischer, welcher früher abläuft. Bei Gleichheit der, welcher später beginnt. 

Beim Klick in die Spalte ‚Memo‘ öffnet sich ein Dialog für eine Texteingabe. 

Diese Rücknahmetexte stehen im Artikelstamm auf der Registerkarte Verkauf in der Combobox K. Rücknahme zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
