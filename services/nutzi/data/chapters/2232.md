# Parameter Webshop Registerkarte Allgemein

E-Commerce Katalog / Parameter Webshop 

# Parameter Webshop Registerkarte Allgemein 
Parameter Webshop Registerkarte Allgemein 

Hier nehmen Sie allgemeine Einstellungen zum Webshop vor. 
### Feld Webshop 
Bei der Neuanlage eines Webshops vergeben Sie hier manuell eine Nummer. 

Um einen bereits angelegten Shop auszuwählen, geben Sie in dieses Feld die Nummer des gewünschten Shops ein. 

Tooltip: 

Angabe der Webshop ID 

### Feld Startgruppe 
Hier können Sie eine Kataloggruppe als Startgruppe definieren. Dieses Feld kann nicht per Tastatureingabe gefüllt werden, sondern nur über den bereits vorhandenen Katalogauswahldialog. 

Wird ein Webshop im Dialog Parameter Webshop verändert oder neu angelegt und es wird vergessen, die Startgruppe festzulegen, so erscheint eine Fehlermeldung, dass die Startgruppe gesetzt werden muss. 

Siehe Kapitel Katalogauswahl . 

Tooltip: 

Legen Sie hier die Startgruppe Ihres Kataloges an. 

### Button Öffnen 

Mit diesem Button öffnen Sie eine Katalogauswahl. 

Siehe Kapitel Katalogauswahl . 

Tooltip: 

Klicken Sie hier, um eine Katalogruppe auszuwählen. 

### Feld Pfad 
Hier wird die URL zum Shop eingetragen. 

### Button Verzweigen 

Über diesen Button können Sie einen Shop direkt aufrufen. 

### Combobox Geschäftlich 
Hier können Sie eine Auftragsart für Geschäftskunden auswählen. 

Tooltip: 

Eingabe der Auftragsart 

### Combobox Privat 
Hier kann eine Auftragsart für Privat-Kunden ausgewählt werden. Diese Auftragsart zählt für alle Kunden mit dem Kennzeichen „Privat Kunde“ aus dem Kundenstamm. 

Tooltip: 

Eingabe der Auftragsart 

### Combobox Angebotsart 
Hier können Sie eine Angebotsart auswählen. Ist hier keine Angebotsart ausgewählt, wird geprüft, ob im Kundenstamm eine Angebotsart hinterlegt ist und wenn nicht, wird die Standardangebotsart (A) gezogen. Die Angebotsart ist mit einem ‚A‘ am Anfang gekennzeichnet. 

Tooltip: 

Angebotsart 

### Combobox Bestellart 
Hier wählen Sie eine Bestellart aus. Bei der Übernahme eines Warenkorbes in einen Auftrag wird die gewählte Bestellart in den Auftrag eingetragen. Damit stehen weitere Auswertungsmöglichkeiten zur Verfügung. 

Tooltip: 

Stellt die Bestellart dar und soll zur Vereinfachung der Auswertung der Webshopaufträge dienen. 

### Feld Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Button Verzweigen 

Mit diesem Button können Sie in den Kundenstamm verzweigen. 

### Button Suche 

Mit diesem Button können Sie Kunden suchen. 

Siehe Kapitel Kundenauswahl . 

### Feld Firma 1 
Der 1. Adresszeile der Firma wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Ang. Gültigkeit 
Hier geben Sie eine Ganzzahl für die Angebotsgültigkeit ein. Für den Fall, dass im Angebot keine Gültigkeitsdauer hinterlegt ist, wird dann die hier eingetragene Gültigkeit gezogen. 

### Combobox Bankverbindung 
Hier wird die Standard-Bankverbindung angezeigt. Gibt es keine speziell gekennzeichnete Bankverbindung, wird die jüngste Bankverbindung zum Kunden bzw. Lieferanten angezeigt. Für diesen Vorgang kann die Bankverbindung auch geändert werden. 

### Combobox Abholungsbed. 
Hier können Sie eine Bedingung für die Abholung festlegen. Damit sorgen Sie dafür, dass die korrekten Lieferbedingungen und Versandarten (hauptsächlich Abholung und Tour ) festgelegt und verwendet werden. Diese werden aus den Versandarten gezogen, wenn im Dialog Versandarten die Checkbox in der Spalte ‚Ecomm‘ gesetzt ist. 

Siehe Kapitel Versandarten . 

Tooltip: 

Hier können Sie eine Abholungsbedingung festlegen. ACHTUNG: dies hat nur Auswirkungen auf den WebAccess! 

### Combobox Anlieferungsbed. 
Hier können Sie eine Bedingung für die Anlieferung festlegen. Damit sorgen Sie dafür, dass die korrekten Lieferbedingungen und Versandarten (hauptsächlich Abholung und Tour ) festgelegt und verwendet werden. Diese werden aus den Versandarten gezogen, wenn im Dialog Versandarten die Checkbox in der Spalte ‚Ecomm‘ gesetzt ist. 

Siehe Kapitel Versandarten . 

Tooltip: 

Hier können Sie eine Anlieferbedingung festlegen. 
ACHTUNG: dies hat nur Auswirkungen auf den WebAccess! 

### Combobox Gesellschaft 
Hier wird die Gesellschaftsnummer für das Verfahren 'Multicompany' angegeben. 

Siehe Kapitel Gesellschaftsverwaltung für Multicompany . 

Tooltip: 

Gesellschaft für das Verfahren "Eine Warenwirtschaft". 

### Checkbox Privatkunde 
Wenn Sie diese Checkbox aktivieren, hat dies Auswirkungen auf das E-Commerce und die Zinsrechnung. 

- E-Commerce: Im Shop werden die Preise als Bruttopreise (inkl. MwSt.) ausgewiesen. - Zinsrechnung: Es gelten die Zinsparameter für Privatkunden. - Auftrag: Wenn konfiguriert und die Leistungserbringung findet im Ausland statt, dann gilt steuerlich das 'Bestimmungslandprinzip. Siehe Kapitel Unterstützung One-Stop-Shop . 

Tooltip: 

Diese Checkbox wirkt sich aus in 
1. E-Commerce: Im Shop werden die Preise als Bruttopreise (inkl. MwSt.) ausgewiesen. 
2. Zinsrechnung: Es gelten die Zinsparameter für Privatkunden. 
3. Auftrag: Wenn konfiguriert und Leistungserbringung im Ausland, dann gilt steuerlich das 'Bestimmungslandprinzip'
