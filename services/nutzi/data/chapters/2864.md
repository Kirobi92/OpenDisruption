# Archiv-Pool Eintrag 
Archiv-Pool Eintrag 

Der Button Details im Archiv-Pool auf der Registerkarte Offen bzw. Fehler öffnet den Dialog Archiv-Pool Eintrag . 

Im oberen Bereich sehen Sie die Referenz auf das Archiv bzw. den Archiveintrag. Daneben sehen Sie nochmals den Fehlertext und warum eine Übertragung ins Archiv gescheitert ist. Wenn Sie den Dialog Archiv-Pool-Eintrag aus der Registerkarte Fehler öffnen, ist das Feld Fehlertext zugänglich. 

Im unteren Bereich sehen Sie die Zuordnung der Daten in das Archiv bzw. die Referenzen, welche aus eNVenta ERP an das Archiv übergeben werden sollen, um den Beleg zu identifizieren. Wurde z.B. ein falscher Datentyp verwendet, können Sie dies hier ändern, damit der Beleg nachträglich noch in das Archiv übernommen werden kann. 

Bei Klick auf den Button wird die Änderung übernommen und der Eintrag erscheint nicht mehr in der Registerkarte Fehler . Über die Registerkarte Offen kann der Beleg nun ins Archiv übernommen werden. 

Die Spalte 'Gruppennr' steht im Zusammenhang mit der Archivierung von Sammelbelegen. D.h. bei Sammelbelegen gibt es mehrere Gruppen.

### Feld Druck-Status 
Hier wird der Druck-Status angegeben. Dieser gibt Auskunft über den Verarbeitungsstatus des Dokuments: 

| -1 | Druckauftrag nicht auffindbar |
| --- | --- |
| 0 | Transaktion nicht abgeschlossen |
| 1 | Druck wurde vom PrintService noch nicht begonnen |
| 2 | Beleg wird gerade gedruckt |
| 3 | Beleg ist erfolgreich gedruckt bzw. verarbeitet worden |
| -3 | Beleg erfolgreich in der Datenbank erstellt und korrekt verarbeitet |
| 4 | Beleg ist fehlerhaft verarbeitet |
