# Abschluss von Teilproduktionen/Betriebsauftrag Registerkarte Kuppelprodukte 
Abschluss Produktion Registerkarte Kuppelproduktion 

Auf der Registerkarte Kuppelprodukte können zusätzlich in einem Arbeitsgang weitere Kuppelprodukte zugebucht werden. Dazu muss lediglich in der Tabelle der Kuppelprodukte die Menge eingegeben werden. 

Chargenführende Produkte benötigen zusätzlich eine Chargennummer (Button Charge ). Diese kann frei eingegeben werden. Alternativ kann per Auswahl auf eine bestehende Charge zugebucht werden. Wurde bereits eine Buchung gemacht, kann die Charge nicht mehr geändert werden. 

Es ist möglich, eine Ausschussmenge anzugeben (nur für den Hauptartikel). Diese Menge wird auf das ebenfalls anzugebende Nachbearbeitungslager gebucht. 

Bei der Verbuchung werden folgende Hauptschritte durchgeführt: 

- Erstellen einer Liste von durchzuführenden Buchungen. - Bei fertiger Buchung: Abschluss der Arbeitsgänge soweit noch nicht erfolgt. - Abbuchung des Materials, wenn noch keine BDE-Buchung erfolgt ist. Wurde einmal für ein Material BDE gebucht, wird angenommen, dass das für dieses Material vollständig per BDE erfolgt. Liegt keine BDE-Buchung vor, so wird angenommen, dass eine Abbuchung nach Sollmenge erfolgen soll. - Bei Fertigbuchung: Der Materialstatus wird auf "abgeschlossen" gesetzt. - Buchung der Produkte. Liegen bereits BDE-Buchungen für das Produkt mit dem Status "gemeldet" (1) vor, so werden diese auf Status 2 gesetzt. 
Alle noch nicht per BDE gemeldeten Buchungen werden als BDE-Produkt-Buchungen erzeugt und sind im BDE-Journal zu finden. - Der Status des Betriebsauftrags wird auf „teilgeliefert“ bzw. „abgeschlossen“ gesetzt. - Optional: Mengenupdate im Auftrag, falls zugeordnet. - Optional: Selbstkosten in Auftrag übernehmen, falls zugeordnet. - Optional: Selbstkosten als EK in Artikel schreiben. - Bei Abschluss eines Betriebsauftrags erfolgt eine Buchung von Wareneingang und Wareneinsatz an die Kostenrechnung. Bei Direktbuchung werden die Daten ebenfalls an die Kostenrechnung übergeben. Bei Erstellung eines QS-Auftrags wird der Wareneinsatz direkt gebucht, die Daten des Wareneingangs bei Buchung des QS-Auftrags. 
Es ist möglich, Kuppelprodukte wieder aus der Tabelle zu entfernen. Dabei bleibt die Zeile mit dem Kuppelprodukt erhalten, aber die Menge ist geleert. 

Lesen Sie hierzu das Kapitel BDE .
