# Fokus Vereinbarungen 
Fokus Vereinbarungen 

Das Element Fokus Vereinbarungen bietet Zusatzinformationen zu dem aktuellen Kunden. Es kann ohne Aufruf von weiteren Dialogen erkannt werden, ob entsprechende Vereinbarungen vorhanden sind. Ein direktes Aufrufen per Doppelklick ist möglich.
