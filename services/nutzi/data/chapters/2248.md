# Katalogverwaltung Registerkarte Stichworte 
Katalogverwaltung Registerkarte Stichworte 

Auf dieser Registerkarte werden Stichworte zu einem Produkt eingegeben. Nach diesen Begriffen kann im Webshop und im eGate gesucht werden. 

Dazu markieren Sie links im Baum einen Artikel, geben auf dieser Registerkarte das Stichwort an (Button ) und speichern. Anschließend ist eGate neu zu synchronisieren. 

Ansonsten kann auch eine Volltextsuche gestartet werden. 

Die Stichwortsuche ist allerdings etwas schneller und wird deshalb empfohlen. 

Die Schreibweise (groß/klein, Umlaute, ß) ist unerheblich. 
### Button Satz hinzufügen 

Mit diesem Button kann ein neues Stichwort angelegt werden. 

Tooltip: 

Neues Stichwort 

### Button Löschen 

Mit diesem Button kann ein Stichwort gelöscht werden. 

Tooltip: 

Schlüsselwort löschen
