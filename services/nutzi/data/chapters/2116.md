# Merkmale zuweisen 
Merkmale zuweisen 

Dieser Dialog bietet die Möglichkeit, Kunden mit einem bestimmten Merkmal von der Cross-Over-Analyse auszuschließen. 

Um diesen Dialog zu öffnen, gehen Sie wie folgt vor: 

Nachdem Sie in der Cross-Over-Analyse einen Artikel ausgewählt haben, übernehmen Sie mit dem -Button (neben dem Feld Bezeichnung ) ein Merkmal aus dem Dialog Merkmalszuweisung . Alle Kunden mit diesem Merkmal werden in die Tabelle Kunden eingelesen. Anschließend drücken Sie den Button , um diesen Dialog zu öffnen, der ebenfalls Merkmale zuweisen heißt. 

In dieser Tabelle wählen Sie nun alle Kunden aus, die aus der Tabelle im Dialog Merkmale eintragen gelöscht werden sollen und drücken dazu den Button . 

Mit dem Button werden die Merkmale zu den noch vorhandenen Kunden generiert. 

Wenn die Checkbox Alte Merkmale löschen aktiviert ist, werden die vorhandenen Merkmale der angezeigten Kunden zuerst gelöscht.
