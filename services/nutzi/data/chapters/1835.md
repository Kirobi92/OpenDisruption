# Mahnparameter Einkauf Registerkarte Mahntext 2 
Mahnparameter Registerkarte Mahnungstext 2 

Auf dieser Registerkarte kann ein weiterer Mahntext eingetragen werden. 
### Feld Titel 
In diesem Feld bekommt der Mahntext einen Titel (Überschrift). 

Tooltip: 

Geben Sie dem Mahntext einen Titel. 

### Feld Mahntext 2 
In diesem Feld kann für die jeweilige Mahnstufe ein sprachabhängiger Text zur Mahnung hinterlegt werden. 

Tooltip: 

Mahntext zur Mahnung.
