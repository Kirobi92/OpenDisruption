# Server Datei Pfad Konverter 
Server Datei Pfad Konverter 

Dieser Dialog öffnet sich in der Katalogverwaltung >> Optionen >> Konvertierung >> Ressourcen -> Server Datei , in den Kasseneinstellungen , der Artikelbildverwaltung etc. Je nachdem aus welchem Dialog Sie diesen Dialog öffnen, kann der Titel 
- Ordner auswählen - Datei auswählen - Datei(en) auswählen - Datei speichern 
lauten. 

Dieser Dialog bietet eine Suchen- und Ersetzen-Funktionalität für Tabellen-Werte. 

Bei Auswahl einer Verbindung wird nach Einträgen gesucht, welche dem physikalischen Pfad aus der Verbindung entsprechen. Dieser Teil des Pfades kann dann durch den Wert im Feld durch ersetzt werden, welcher mit dem Archivnamen aus der Verbindung vorbelegt wird.
