# Fokus Verbindungen 
Fokus Verbindungen 

Das Element Fokus-Verbindungen öffnet sich im Fokus-Element Verbindungen über den Button und bietet Zusatzinformationen zu dem aktuellen Vorgang. Es kann ohne Aufruf von weiteren Dialogen erkannt werden, ob zu dem aktuellen Vorgang gedruckte Dokumente oder Referenz-Links vorhanden sind. Ein direktes Aufrufen verbundener Objekte per Doppelklick ist möglich. 

Das Element wird bei jeder Änderung im offenen Vorgang automatisch aktualisiert. 
## Steuerelemente des Formulars 

### Feld Kurztext 
Tooltip: 

Generierter Anzeigetext oder Referenztext.
