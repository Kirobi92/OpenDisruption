Allgemeine Funktionalitäten in eNVenta ERP / Drucken in eNVenta ERP / Formularsteuerung 

# Formularsteuerung Registerkarte Allgemein 
Formularsteuerung Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die Einstellungen für das Formular im Einzelnen. 
### Combobox Funktion 
In dieser Combobox stehen 2 Funktionen zur Auswahl zur Verfügung: 

Ersetzen: Ersetzt das oben angegebene Formular durch den Eintrag in „Formular Neu“. 

Zusätzlich: Druckt nach dem oben angegebenen Formular als nächstes den Eintrag in „Formular Neu“. 

### Combobox Formular Neu 
Hier geben Sie an, welches Formular das erste Formular ersetzt oder ergänzt. 

### Combobox Drucker 
Hier wählen Sie aus, auf welchem Drucker das Formular ausgegeben werden soll. 

Tooltip: 

Angabe des Druckernamens 

### Feld Kopien 
Hier geben Sie die Anzahl der Kopien an, die für das Formular erzeugt werden sollen. 

### Checkbox Immer Drucken 
Wenn diese Checkbox gesetzt ist, wird dafür gesorgt, dass auch beim E-Mail-Versand der Beleg zusätzlich als Ausdruck kommt. 

Werden mehrere zusätzliche Formulare auf den Drucker geschickt und ist dabei auch nur für ein Formular diese Checkbox gesetzt, wird auch das "normale" Formular im Massendruck per E-Mail versendet! 

### Checkbox Mehrfachdruck erzwingen 
Wenn eine Schleife der Formulare, z.B. durch mehrere Prioritäten, erkannt wird, so werden die Formulare trotzdem gedruckt.
