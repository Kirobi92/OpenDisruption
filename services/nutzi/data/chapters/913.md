# Projekt Registerkarte Tabelle 
Projekt Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Projekte angezeigt. 
### Tabelle 

### Tabellenspalte Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Tabellenspalte Projektname 
Hier wird der Projektname angegeben. 

Tooltip: 

Name des Projekts 

### Tabellenspalte Projektart 
Hier wählen Sie die Projektart aus. 

Tooltip: 

Auswahl der Projektart 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Ort 
Hier wird der Ort angegeben. 

Tooltip: 

Angabe des Orts 

### Tabellenspalte Erfassung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der Erfassung eingetragen. 

Tooltip: 

Datum und Uhrzeit der Erfassung 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die Anzahl der tatsächlich angezeigten Datensätze angezeigt.
