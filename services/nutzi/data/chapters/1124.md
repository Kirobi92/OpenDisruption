# Menge an einem Set-Artikel ändern 
Art-Set Menge ändern 

Dieser Dialog öffnet sich im Auftrag >> Optionen >> ArtikelSet >> Arikel-Set Menge ändern , wenn der Mitarbeiter die entsprechende Berechtigung aus der Berechtigungsgruppe hat. 

Siehe Kapitel Berechtigungsgruppe . 

Hier können Mengenänderungen an einem Set-Artikel vorgenommen werden. Dabei findet eine automatische Aktualisierung der Endteilpositionen des Sets nur dann statt, wenn alle Komponenten im o.g. Sinn änderbar sind. 

Wenn aber nur eine Komponente teilgeliefert ist, einer Packliste angehört oder einen Bezug zu einer Dispo- oder Bestellposition hat, findet keinerlei Mengenaktualisierung an den Komponenten statt. In diesen Fällen erscheint eine Messagebox, die auf die Situation hinweist und empfiehlt, die notwendigen Mengenanpassungen im Anschluss manuell vorzunehmen.
