# DATEV-Export Fibu Registerkarte Sachkontenstamm 
DATEV Export FiBu Registerkarte Sachkontenstamm 

Auf dieser Registerkarte werden Ihnen Sachkonten für eine Übergabe an DATEV angeboten. 

Bei einem Wechsel auf diese Registerkarte ändern sich im Kopf des Dialogs zum Teil die Felder und deren Beschriftung! 
### Tabelle 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Tabellenspalte DATEV Konto 
Hier wird die DATEV-Kontonummer angegeben, falls sie abweichend von der internen Kontonummer ist. 

Tooltip: 

DATEV-Kontonummer, falls abweichend von der internen Kontonummer.
