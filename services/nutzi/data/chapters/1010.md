# Retourenarten verwalten 
Retourenarten 

Dieser Dialog öffnet sich im Navigator >> System >> Verkauf >> Retourenarten . 

Hier können Sie Retourenarten anlegen. Dazu geben Sie in das Feld Retourenart eine Zahl ein und in das Feld Bezeichnung einen passenden Namen. Vergewissern Sie sich zuvor, ob diese Nummer nicht bereits existiert. Dazu sehen Sie sich die Registerkarte Tabelle an, hier erhalten Sie einen Überblick über alle Retourenarten. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Retourenarten 
Tooltip: 

In diesem Dialog legen Sie Retourenarten an. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Referenz-Link 

Dieser Button steht zur Verfügung, um Dokumenten-Links sowie URL‘s als Info in eNVenta ERP zu hinterlegen oder um 2 Vorgänge/Referenzen miteinander zu verknüpfen. 

Siehe Kapitel Referenz-Links . 

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Ein Klick auf diesen Button setzt den Wert zurück; die Felder werden geleert. 

### Button Kopieren 

Mit diesem Button wird der aktuelle Datensatz übernommen, allerdings ist das Feld Retourenart jetzt leer (das bedeutet Status Neu ) und muss neu vergeben werden. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

Nach dem Löschen wird der erste Datensatz wieder ausgewählt. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Steuerung 

Dieser Button öffnet die Formularsteuerung, um auf Kunden-, Lieferanten-, Sachbearbeiter- oder Abteilungsebene eine Steuerung der zu druckenden Formulare vorzunehmen. 

Wenn eine Firma für einen ihrer Kunden die komplette Logistik zum Endkunden übernimmt, kann es auch zu Retouren kommen, die verarbeitet werden müssen. Um dies ebenfalls erfüllen zu können, gelangen Sie in den Dialog Formularsteuerung . 

Im Dialog Formularsteuerung werden, wie gewohnt die Standardformulare für die Retoure angeboten, mit der Möglichkeit diese analog der bestehenden Logik für Auftragsreports etc. zu ersetzen oder zusätzlich zu drucken. D.h. bei allen Druck-Prozessen der Retouren muss zusätzlich geprüft werden, ob eine Formularsteuerung hinterlegt ist. 

Siehe Kapitel Formularsteuerung . 

Tooltip: 

Öffnet den Dialog 'Formularsteuerung'.

Hier tragen Sie den Namen der Retourenart ein bzw. kann er zur Suche der Retourenart genutzt werden. 

Tooltip:
