# BMEcat - Katalogexport Registerkarte Artikel 
BMECat - Katalogexport Registerkarte Artikel 

Auf dieser Registerkarte werden mit dem Button Laden die Artikel angezeigt, die exportiert werden. Wenn Sie einen Eintrag der Tabelle markieren, sehen Sie unten weitere Registerkarten mit Informationen zum Artikel. 
### Button Laden 

Mit diesem Button werden Artikeldaten in die Tabelle geladen. Da das Einlesen von Artikeldaten sehr viel Zeit benötigt, können Sie entscheiden, zu welchem Zeitpunkt Sie die Daten laden möchten.
