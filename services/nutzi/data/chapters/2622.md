# Auszug-/Kassenimport Registerkarte Kontoauszug 
Auszug-/Kassenimport Registerkarte Kontoauszug 

Auf dieser Registerkarte definieren Sie die Einstellungen für den einzulesenden Kontoauszug. 
### Combobox Bankverbindung 
Um die Daten für eine Bank einlesen zu können, wählen Sie zuvor die Bank über die Combobox Bankverbindung aus. Die Kontonummer wird mit angezeigt. 

Hier stehen Ihnen nur Banken zur Auswahl zur Verfügung, die in den Parametern über den Button Bankverbindung angelegt sind. 

Siehe Kapitel Bankverbindungen . 

### Button Öffnen 

Mit diesem Button können Sie in den Dialog Bankverbindung (aus Finanzbuchhaltung >> Parameter Fibu) verzweigt. Die zuvor ausgewählte Bankverbindung ist markiert und kann bearbeitet werden. 

### Checkbox Nur Zahlungen für dieses Konto einlesen 
Wenn Sie diese Checkbox aktivieren, werden nur Zahlungen für das selektierte Konto eingelesen. 

### Feld Konto 
In dieses Feld tragen Sie die Kontonummer der Bank aus dem Sachkontenstamm ein, für welche die Verarbeitung durchgeführt werden soll oder wählen sie über den Button aus. 

Dieses Feld ist ein Pflichtfeld!!! 

Damit Sie bei einer manuellen Verbuchung nicht nach jeder Buchung die Kontonummer neu eingeben müssen, empfiehlt es sich, diese in den Bankverbindungen zu hinterlegen. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Mit diesem Button können Sie ein Konto suchen. 

Siehe Kapitel Kontoauswahl . 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Sachkontenstamm. 

### Feld Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Feld Saldo 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Feld S/H 
Hier wird das Soll- / Haben-Kennzeichen angegeben. 

Tooltip: 

Soll / Haben-Kennzeichnung 

### Feld Währungskürzel 
Hier wird das ISO-Kürzel des international standardisierten Währungskürzels angegeben, da dieses für den Zahlungsverkehr verwendet wird. 

Tooltip: 

Angabe des ISO-Kürzels 

### Feld Belegnummer 
Hier wird die Belegnummer angegeben. 

Wenn das Feld beim Einlesen der Import-Datei leer ist, wird die Belegnummer aus der Import-Datei übernommen. Dabei werden führende Nullen entfernt! 

Bitte prüfen Sie, was Sie tatsächlich importieren: Konto auszug <> Konto umsatz . 

Der Job Server (automatisches Einlesen und Verbuchen des Kontoauszugs) macht nur Sinn, wenn die Import-Datei den Kontoauszug enthält. Der Kontoauszug enthält immer eine Auszugsnummer > 0. Diese kann in das Feld Belegnummer übernommen werden. 

Tooltip: 

Angabe der Belegnummer 

### Checkbox Aus Importdaten 
Wenn Sie diese Checkbox aktivieren, wird vor der Verbuchung geprüft, ob alle Sätze eine Belegnummer mitbringen. Falls nicht, werden Sie mit einer Meldung darauf hingewiesen. 

Wenn der Anwender eine Belegnummer erfasst hat und diese Checkbox gesetzt ist, wird die erfasste Belegnummer nur herangezogen, wenn die Spalte ‚Auszblatt‘ keinen Wert enthält. 

Tooltip: 

Spalte 'Auszblatt' als Belegnummer verwenden 

### Feld Buchungsdatum 
Über dieses Feld steuern Sie, in welche Periode die Buchungen in die Finanzbuchhaltung gebucht werden. 

Das Feld ist mit dem aktuellen Tagesdatum vorbelegt. 

Tooltip: 

Angabe des Buchungsdatums 

### Checkbox Aus Importdaten 
Wenn Sie diese Checkbox aktivieren, wird das Feld Buchungsdatum geleert, da die Daten direkt aus der Auszug-/Kassenimport-Datei übernommen werden. Dies ist vor allem dann interessant, wenn der Auszug-/Kassenimport mehrere Termine enthält, die evtl. auch aus verschiedenen Monaten stammen. 

Tooltip: 

Spalte 'BuDat' als Buchungsdatum verwenden 

### Feld Belegdatum 
Dieses Feld steuert, welches Belegdatum die Buchungen erhalten. 

Das Feld ist mit dem aktuellen Tagesdatum vorbelegt. 

Tooltip: 

Angabe des Belegdatums 

### Checkbox Aus Importdaten 
Wenn Sie diese Checkbox aktivieren, wird das Feld Belegdatum geleert, da die Daten direkt aus der Auszug-/Kassenimport-Datei übernommen werden. Dies ist vor allem dann interessant, wenn der Auszug-/Kassenimport mehrere Termine enthält, die evtl. auch aus verschiedenen Monaten stammen. 

Tooltip: 

Spalte 'Valdat' als Belegdatum verwenden

### Feld Karenzprozent 
Hier können Sie für die Karenz einen Prozentsatz eintragen. 

Der Prozentsatz wird auf den Auszug-/Kassenimport-Zahlbetrag angewandt. 

Der Wert kann in den Parametern der Finanzbuchhaltung hinterlegt werden.
