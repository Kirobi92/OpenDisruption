# Angebot Registerkarte Wettbewerber 
Registerkarte Wettbewerber 

Auf dieser Registerkarte erhält der Sachbearbeiter die Möglichkeit, die Daten des Wettbewerbers einzusehen, soweit diese eingegeben wurden. So kann der Sachbearbeiter den Preis zum Wettbewerb beurteilen oder auch nur eine allgemeine Beurteilung erhalten. Die Informationen können allgemein oder auch artikelbezogen angegeben werden. 

Um eine neue Wettbewerbsinformation hinzuzufügen, klicken Sie den Button und füllen die Felder unterhalb der Registerkarte. Zusätzlich zu den Informationen kann eine Bewertung der Wettbewerbsinformation vorgenommen werden. Dies kann in Form von +, ++, -, -- oder einfach als verloren / gewonnen gespeichert werden. 

Beim Öffnen der Registerkarte werden alle Daten zu dem Kunden eingelesen, welche vorhanden sind. Wurde in der Registerkarte Positionen ein Artikel markiert, so werden nur Einträge mit diesem Artikel bzw. der Artikelgruppe eingelesen. 

Im Marketing steht der Dialog Wettbewerber zur Verfügung. Dort können ebenfalls Wettbewerber verwaltet werden. 

Siehe Kapitel Wettbewerber . 
### Feld AKL-Nummer 
Hier wird eine Adress-/Kunden- oder Lieferantennummer angegeben. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Feld Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabelle Wettbewerber 

Diese Tabelle bietet eine Übersicht über detaillierte Wettbewerbsdaten. 
### Tabellenspalte Erfassung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der Erfassung eingetragen. 

Tooltip: 

Datum und Uhrzeit der Erfassung 

### Tabellenspalte Wettbewerber 
Hier wird die Nummer des Wettbewerbers angegeben. 

Tooltip: 

Angabe der Wettbewerbernummer 

### Tabellenspalte Artikel 
Hier geben Sie die entsprechende Artikelnummer zum Wettbewerber ein. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Lieferanten-Artikel 
Hier wird die Lieferanten-Artikelnummer angegeben, d.h. die Artikelnummer, welche der Lieferant für diesen Artikel verwendet. 

Tooltip: 

Angabe der Lieferanten-Artikelnummer 

### Tabellenspalte Artikelgruppe 
Mit Hilfe dieser Combobox ordnen Sie dem Artikel eine Artikelgruppe zu. Die in der Combobox wählbaren Gruppen werden im Menü Stamm >> Artikelgruppen angelegt. 

Siehe Kapitel Artikelgruppen . 

Tooltip: 

Angabe der Artikelgruppe 

### Tabellenspalte Referenzwert 
Hier wird der Referenzwert des Artikels angegeben. 

Tooltip: 

Referenzwert des Artikels. 

### Tabellenspalte Preis 
Hier wird der Preis des Artikels beim Wettbewerber angegeben. 

Tooltip: 

Preis des Artikels beim Wettbewerber 

### Tabellenspalte Text 
Hier wird ein beliebiger Text angegeben. 

### Button Satz hinzufügen 

Dieser Button fügt eine neue Zeile in die Tabelle ein. Diese steht dann zur Bearbeitung zur Verfügung. 

Tooltip: 

Neue Zeile einfügen 

### Button OK 

Mit diesem Button speichern Sie die Eingaben/Änderungen. 

Tooltip: 

Speichern 

### Button Löschen 

Dieser Button kennzeichnet eine Zeile der Tabelle mit dem -Kennzeichen. 

Tooltip: 

Zeile löschen
