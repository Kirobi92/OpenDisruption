# Rückmeldung von Fahraufträgen Registerkarte Stopps 
Rückmeldung Fahrauftrag Registerkarte Stopps 

Auf dieser Registerkarte erfassen/scannen Sie alle zusätzlichen Daten zu einer gefahrenen Tour. Dies sind: 

- Haltepunkt (Stopp) - Anzahl Hübe - Standzeit (min) - Wartezeit (min) - Abladezeit (min) 
Wenn Sie einen Haltepunkt in der Tabelle anklicken, kann der Eintrag bearbeitet werden. Die geänderten Haltepunkte werden in der Tabelle markiert. 

Die Daten der Tour werden nur gespeichert, wenn die Tour geladen ist. 

Am Ende der Rückmeldung wird die Tour abgeschlossen und damit für weitere Änderungen gesperrt. Nur ein Benutzer mit speziellen Rechten kann diesen Abschluss wieder aufheben und Änderungen an den Einträgen vornehmen. 

Siehe Kapitel Berechtigungsgruppen . 
### Feld Stopp 
Hier geben Sie den Haltepunkt ein. 

Tooltip: 

Angabe der Stopp-Nr. der Tour 

### Feld Anzahl Hübe 
Hier kann die Anzahl der Hübe pro Stopp angegeben werden. Der Stopp wird in der Tabelle daneben markiert. 

Die Daten werden gespeichert und können über die Datenbank ausgewertet werden. 

Tooltip: 

Anzahl der Hübe (Nutzung Kran, etc.) pro markiertem Stopp 

### Feld Standzeit (m.) 
Hier kann die Standzeit pro Stopp angegeben werden. Der Stopp wird in der Tabelle daneben markiert. 

Die Daten werden gespeichert und können über die Datenbank ausgewertet werden. 

Tooltip: 

Standzeit in Minuten 

### Feld Wartezeit (m.) 
Hier kann die Wartezeit pro Stopp angegeben werden. Der Stopp wird in der Tabelle daneben markiert. 

Die Daten werden gespeichert und können über die Datenbank ausgewertet werden. 

Tooltip: 

Wartezeit in Minuten 

### Feld Abladezeit (Min.) 
Hier kann die Abladezeit pro Stopp angegeben werden. Der Stopp wird in der Tabelle daneben markiert. 

Die Daten werden gespeichert und können über die Datenbank ausgewertet werden. 

Tooltip: 

Abladezeit in Minuten 

### Tabelle 

Nach Eingabe der Tournummer werden in dieser Tabelle die dazugehörigen Haltepunkte eingelesen. 

Mit Klick auf einen Haltepunkt können Sie in den Feldern links von der Tabelle die Daten im Einzelnen ansehen und ändern. 
### Tabellenspalte Stopp 
Hier wird der Haltepunkt angezeigt. 

Tooltip: 

Angabe der Stopp-Nr. der Tour 

### Tabellenspalte KL-Nr.

Tooltip:

### Tabellenspalte PLZ

Tooltip: 

Postleitzahl des Datensatzes 

### Tabellenspalte Straße 
Die Straße zur Firma wird angegeben.

Angabe des Straßennamens
