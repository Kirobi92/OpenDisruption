# Zahlvorschlag verarbeiten Registerkarte Einstellungen 
ZV verarbeiten Registerkarte Einstellungen 

Auf dieser Registerkarte definieren Sie die Einstellungen für die Verarbeitung des Zahlvorschlags. 
### Feld Verarb.tag 
Über dieses Feld kann der ausführenden Bank ein gewünschter Verarbeitungstag mitgeteilt werden. 

Tooltip: 

Angabe des Buchungsdatums 

### Combobox Rule Book 
Hier werden die Spezifikationen der Datenformate SEPA angeboten. 

Hier stehen 2 Auswahlmöglichkeiten zur Verfügung. 

pain.001.003.03 bzw. pain.001.002.03 

Je nach Auswahl wird die Information aus der Combobox in die XML-Datei geschrieben. 

Tooltip: 

Regelwerk SEPA-Version 

### Feld Hinweis 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen. 

### Radiobutton Diese Vorschläge ignorieren 
Sollen OPs ignoriert werden, aktivieren Sie diesen Radiobutton. 

### Radiobutton Für diese Vorschläge Schecks drucken! 
Wollen Sie für diese OPs einen Scheck drucken, aktivieren Sie den Radiobutton. Damit wird das Feld Start-Schecknummer zugänglich. Hier haben Sie die Möglichkeit, die Startnummer für die zu druckenden Schecks einzugeben. 

### Feld Buchungsdatum 
Hier wird das Buchungsdatum angegeben. 

Tooltip: 

Angabe des Buchungsdatums 

### Feld Belegdatum 
Hier wird das Belegdatum angegeben. 

Tooltip: 

Angabe des Belegdatums 

### Feld Stapelnummer 
Hier wird die Stapelnummer angegeben, wenn Sie mit Stapelnummern arbeiten. 

Tooltip: 

Angabe der Stapelnummer
