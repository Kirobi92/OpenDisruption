# Kapitel NV.ERP.MM.Production.cdParamProduction.decProductionOverhead

Geben Sie den Zuschlag der Fertigungsgemeinkosten für die Kalkulation in % ein. Der Gemeinkostenzuschlag wird dann für jede Produktion übernommen. Der Gemeinkostenzuschlag kann für jeden Arbeitsplan und jeden Betriebsauftrag separat geändert werden.
