# Einkaufsstückliste Registerkarte Komponente 
Einkaufsstückliste Registerkarte Komponente 

Für die GEK-Berechnung der Komponenten zur Lagerberechnung kann im Feld Kalkulationsanteil ein prozentualer Wert gepflegt werden. Der Kalkulationsanteil für den GEK kann bereits in der Stammstückliste im Artikelstamm, Menü Stamm >> Stückliste eingetragen werden. 

Die Verteilung des GEKs (der EK-Preise der Komponenten) erfolgt hierbei nur für die zerlegten Komponenten der Stückliste und verteilt die Ersparnisse entsprechend. Die Verteilung des GEKs erfolgt hierbei nach Wert oder Gewicht, entsprechend dem Eintrag in der Combobox Stücklisten Kalk des Artikelstamms auf der Registerkarte Einkauf . Ist dort nichts hinterlegt, so zieht entsprechend der Eintrag in den Einkaufs-Parametern. Ist eine manuelle Angabe in der Stückliste erfolgt, so muss die Summe aller Komponenten Anteil-GEK 100,00% ergeben, sonst erscheint eine entsprechende Hinweismeldung. Die Prüfung auf 100,00% berücksichtigt auch die verschiedenen Ebenen und Varianten der Stückliste. 

In diesem Zusammenhang beachten Sie bitte auch die Informationen zur Checkbox Alle Lager im Dialog Lagerorte und das Kapitel GEK-Ermittlung . 

### Button Bestand 

Mit diesem Button erhalten Sie einen Überblick über den tatsächlichen Bestand des Artikels. 

Siehe Kapitel Bestandsanzeige . 

### Button Auflösen 

Wenn Sie einer Produktionsstückliste eine weitere Komponente hinzufügen, welche ebenfalls eine Produktionsstückliste ist, werden mit Klick auf diesen Button alle Komponenten der neu hinzugefügten Stückliste eingefügt. Die Komponente hat nach dem Auflösen eine eigene Baumstruktur ('+') links vom Artikelnamen. 

### Feld Artikel 
Wenn Sie im TreeView einen Eintrag markieren, wird hier die Artikelnummer angezeigt. Wird der Stücklistenartikel markiert, kann in diesem Feld nichts geändert werden. 

Wird eine Komponente markiert, kann hier ein anderer Artikel als Komponente eingetragen werden. Die Daten zu diesem gewählten Artikel werden in den Feldern darunter angezeigt. 

Tooltip: 

Zeigt die Artikelnummer, die im Baum links markiert ist.
