# BMEcat - Katalogexport Registerkarte Allgemein 
BMECat - Katalogexport Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie allgemeine Einstellungen für den Export. 
### Feld Gültig von 
Hier geben Sie an, ab wann der Katalog gültig ist. 

Tooltip: 

Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Feld Gültig bis 
Hier geben Sie an, bis wann der Katalog gültig ist. 

Tooltip: 

Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Combobox WMCode 
Hier wählen Sie den Werbemittelcode aus. 

Tooltip: 

Angabe des Werbemittelkennzeichens 

### Feld Ress. Verz. 
Hier ist das Unterverzeichnis der Ressource eingetragen. 

Tooltip: 

Unterverzeichnis für Ressourcen 

### Feld Ressourcengruppen 
In diesem Feld legen Sie die zu exportierende Ressourcengruppe fest. Auf diese Weise werden Ressourcengruppen pro Katalog exportiert. 

Tooltip: 

Filtert die exportierten Ressourcengruppen. 

### Button Öffnen 

Wenn Sie diesen Button drücken, öffnet sich der bereits bekannte Dialog Ressourcengruppen . Hier wählen Sie eine Ressourcengruppe, die bereits als BMEcat Export definiert ist, für den Export aus. 

Siehe Kapitel Ressourcengruppen . 

### Checkbox Html-Beschr. 
Wenn Sie diese Checkbox aktivieren, wird das Klassifikationssystem exportiert. 

Tooltip: 

Die Beschreibung wird als HTML-Text exportiert. 

### Checkbox Leere Grp. filtern 
Wenn Sie diese Checkbox aktivieren, wird die Katalogstruktur gefiltert und leere Gruppen werden entfernt. 

Tooltip: 

Filtert die Katalogstruktur und entfernt leere Gruppen.
