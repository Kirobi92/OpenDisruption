# Prüfauftrag Registerkarte Allgemein

Produktion (PPS) / Prüfauftrag 

# Prüfauftrag Registerkarte Allgemein 
Prüfauftrag Registerkarte Allgemein 

Auf dieser Registerkarte geben Sie die Details zur Erzeugung eines Prüfauftrags ein. 

Neben der Artikelnummer müssen Sie auf dieser Registerkarte auch die zu prüfende Menge des Artikels angeben. Wird der Prüfauftrag bei der Abschlussbuchung des Betriebsauftrags generiert, sind alle Felder automatisch gefüllt. 
### Feld Menge 
Die Summe aus den Feldern Akzeptiert und Beanstandet wird angezeigt. 

Tooltip: 

Zeigt die Gesamtsumme der zu prüfenden Menge. 

### Feld Menge akzept. 
Geben Sie hier die akzeptierte Menge ein. 

Tooltip: 

Akzeptierte Menge des Artikels 

### Feld Beanstandet 
Geben Sie hier die beanstandete Menge ein. 

Die Summe der Felder Akzeptiert und Beanstandet muss mit der Summe aus dem Prüfauftrag übereinstimmen. 

Tooltip: 

Beanstandete Menge des Artikels 

### Feld QKZ 
Hier kann eine QKZ (1-4) erfasst werden. 

Tooltip: 

Angabe der QKZ 

### Feld Prozess 
Hier wird der geprüfte Betriebsauftrag eingetragen. 

Tooltip: 

Zeigt den geprüften Betriebsauftrag. 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Betriebsauftrag. 

### Combobox Ziellager 
Geben Sie hier das Lager ein, auf welches das fertige Produkt gebucht werden soll. Bei Angabe einer QKZ wird nur die Menge mit QKZ = 1 auf das Ziellager gebucht. Bei Generierung des Prüfauftrags aus dem Betriebsauftrag wird das Ziellager daraus übernommen. 

Tooltip: 

Definiert das Lager, auf das zugelagert werden soll. 

### Combobox NB-Lager 
Geben Sie hier das Lager für die Nachbearbeitung ein. Es wird die Menge mit QKZ > 1 auf das Nachbearbeitungslager gebucht. 

Tooltip: 

Definiert das Nachbearbeitungslager 

### Feld Prüfbericht 
Hier können Sie Textinformationen über das Prüfergebnis eingetragen werden. 

Tooltip: 

Information zum Prüfergebnis
