# Auszug-/Kassenimport Registerkarte Kassenübernahme 
Auszug-/Kassenimport Registerkarte Kassenübernahme 

Auf dieser Registerkarte werden alle Kassen (mit Kassennummer und –bezeichnung) und deren Saldo in der Tabelle aufgelistet. Mit Markieren der Kassen in der Tabelle legen Sie fest, von welchen Kassen die Daten übernommen werden sollen. Es werden nur Kassen eingelesen, für die ein Abschluss stattgefunden hat und die noch nicht übernommen worden sind. 

Wenn Sie die Checkbox Wiederholung aktivieren, können bereits übernommene Daten (Vorschlags-Selektion) ein weiteres Mal übernommen werden. In diesem Fall wird auch das Feld Datum von zur Selektion editierbar. 

Das Feld Datum bis ist mit dem Tagesdatum vorbelegt. Hier hat auf jeden Fall ein Datum zu stehen, sonst ist keine Übernahme möglich. 

Die Registerkarte Kassenübernahme bietet zusätzlich noch die Felder Karenzbetrag, Karenzprozent, Stapelnummer und Buchungsdatum an. So kann jeder Sachbearbeiter weitere Daten eingeben. 

Das Feld Buchungsdatum wird mit dem Tagesdatum vorbelegt. 

Wenn Sie die Checkbox aus Auszug-/Kassenimport aktivieren, wird das Buchungsdatum aus der Auszug-/Kassenimport-Datei eingelesen. 

Über die Felder Abr. nr. von/bis können Sie eine weitere Selektion innerhalb der Auszug-/Kassenimport-Datei auslösen, und zwar über die Abrechnungsnummern von-bis. 

Bei der Abrechnung über den gleichnamigen Dialog wird keine Auszug-/Kassenimport-Datei mehr erstellt. 

Für die automatische Kassenübernahme ist zwingend diese Registerkarte zu aktivieren und anschließend ein entsprechender Job zu definieren. 
### Combobox Übernahme 
Hier kann ein gespeicherter/angelegter Vorschlag ausgewählt werden. 

### Checkbox Wiederholung 
Wenn Sie diese Checkbox aktivieren, können bereits übernommene Daten (Vorschlags-Selektion) ein weiteres Mal übernommen werden. In diesem Fall wird auch das Feld Datum von zur Selektion editierbar. 

### Tabelle 

### Tabellenspalte Kasse 
Hier wird die Nummer der Kasse angegeben. 

Tooltip: 

Definiert die Kassennummer 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung der Kasse angegeben. 

Tooltip: 

Hier wird die Kassenbezeichnung angegeben. 

### Tabellenspalte Kassenkonto 
Hier geben Sie die Kontonummer ein, nach der Sie suchen möchten. Hier darf nur ein Konto ausgewählt werden, das in der Währung der Kasse geführt wird. 

Tooltip: 

Kassenkontonummer 

### Tabellenspalte Saldo FW 
Hier wird der Betrag in Fremdwährung angegeben. 

Tooltip: 

Angabe des Betrags in Fremdwährung 

### Tabellenspalte Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind! 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Feld Datum von 
Dieses Feld ist mit dem Tagesdatum vorbelegt. Hier hat auf jeden Fall ein Datum zu stehen, sonst ist keine Übernahme möglich. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld Datum bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Feld Abr.nr von 
Hier können Sie eine Abrechnungsnummer eintragen. So können Sie eine weitere Selektion innerhalb der Auszug-/Kassenimport-Datei auslösen, und zwar über die Abrechnungsnummer von-bis. 

### Feld Abr.nr bis 
Zur Selektion über eine Abrechnungsnummer oder einen Abrechnungsnummernbereich können Sie hier eine Abrechnungsnummer eintragen. 

### Feld Karenzbetrag 
Hier können Sie einen Karenzbetrag eintragen. 

Der Wert kann in den Parametern der Finanzbuchhaltung hinterlegt werden. 

Siehe Kapitel Parameter Fibu . 

Tooltip: 

Angabe des Betrags 

### Feld Karenzprozent 
Hier können Sie für die Karenz einen Prozentsatz eintragen. 

Der Prozentsatz wird auf den Auszug-/Kassenimport-Zahlbetrag angewandt. 

Der Wert kann in den Parametern der Finanzbuchhaltung hinterlegt werden. 

Siehe Kapitel Parameter Fibu . 

### Feld Stapelnummer 
Hier kann zur Selektion die Stapelnummer eingetragen werden. 

Tooltip: 

Angabe der Stapelnummer 

### Feld Buchungsdatum 
Über dieses Feld steuern Sie, in welche Periode die Buchungen in die Finanzbuchhaltung gebucht werden. 

Das Feld ist mit dem aktuellen Tagesdatum vorbelegt. 

Tooltip: 

Angabe des Buchungsdatums 

### Checkbox Aus Importdaten 
Wenn Sie diese Checkbox aktivieren, wird das Feld Buchungsdatum geleert, da die Daten direkt aus der Auszug-/Kassenimport-Datei übernommen werden. Dies ist vor allem dann interessant, wenn die Auszug-/Kassenimport-Datei mehrere Termine enthält, die evtl. auch aus verschiedenen Monaten stammen.
