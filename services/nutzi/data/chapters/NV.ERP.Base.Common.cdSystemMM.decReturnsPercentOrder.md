# Kapitel NV.ERP.Base.Common.cdSystemMM.decReturnsPercentOrder

(= Abschlag für Retouren mit auftragsbezogener Bestellung / Beschaffungsware) 

Hier kann standardmäßig ein Prozentsatz für die Rücknahme (einer Auftragsposition) von Beschaffungsware (= Einstellung aus dem Dispoverfahren) eingetragen werden. Um diesen Prozentsatz wird die Gutschrift bei Rücknahme der Ware gemindert.
