# Job-Übersicht 
Job Übersicht 

In der Job Übersicht werden alle verfügbaren Job-Server angezeigt. Es können beliebig viele Job-Server installiert werden. 

Bei Klick auf den Button Jobs im Dialog Job-Server Übersicht oder aus dem Menü System >> Job-Server >> Jobs öffnet sich die Job-Übersicht . 

Job Übersicht aus Menü System aufrufen 

Wenn Sie in der oberen Tabelle einen Job anklicken, erscheinen in der Tabelle darunter die Daten zu den einzelnen Läufen dieses Jobs. Hier sehen Sie dann für jeden Lauf, ob dieser erfolgreich abgeschlossen wurde und auch den Fortschritt des Laufs. 

Bei der Einrichtung eines Jobs kann direkt eine Job-Server Gruppe angegeben werden. Dazu steht in der Job Übersicht die Spalte ‚Job-Server Grp‘ zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Job Übersicht 
Tooltip: 

In diesem Dialog werden alle verfügbaren Job-Server angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Ausführen 

Wenn Sie diesen Button drücken, wird ein markierter Job sofort ausgeführt, unabhängig wann laut Zeitplan die nächste Ausführung geplant ist. 

Tooltip: 

Mit diesem Button wird der Job sofort ausgeführt. 

### Button Abbrechen 

Wenn Sie diesen Button drücken, wird ein markierter Job abgebrochen. 

Tooltip: 

"Abbrechen" an den Job-Server senden. 

### Button Bearbeiten 

Dieser Button öffnet den Dialog Job Zeitplan , um die Zeitplanung des Jobs zu überarbeiten. 

Siehe Kapitel Job Zeitplan . 

### Button Aufräumen 

Wenn Sie diesen Button drücken, können alte, erfolgreich beendete Jobs gelöscht werden. Es öffnet sich hierzu der Dialog Jobs aufräumen . 

Siehe Kapitel Jobs aufräumen . 

### Combobox Server 
Zur Selektion können Sie hier einen Server auswählen. 

### Checkbox Sofort 
Wenn Sie diese Checkbox aktivieren, wird der Sofort-Job unverzüglich gestartet, wenn ein Thread (z.B. Prozessor-Kern) frei ist und wartet nicht auf einen Termin laut Zeitplanung. 

Außerdem werden solche Jobs zusätzlich in die Tabelle eingelesen. 

### Tabelle Jobs 

Diese Tabelle listet alle Jobs auf. 

Wenn Sie in dieser Tabelle einen Job anklicken, erscheinen in der Tabelle darunter die Daten zu den einzelnen Läufen dieses Jobs. Hier sehen Sie dann für jeden Lauf, ob dieser erfolgreich abgeschlossen wurde und auch den Fortschritt des Laufs. 
### Tabellenspalte Bezeichnung 
Hier wird der Name des Jobs angezeigt. 

### Tabellenspalte Status 
Hier wird der Status des Jobs angezeigt. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Zeitplan 
Hier wird der definierte Zeitplan angezeigt. 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabelle Details 

Wenn Sie in der oberen Tabelle einen Eintrag markieren, werden in der unteren Tabelle die Details des Jobs angezeigt. 

Mit einem Doppelklick auf einen Eintrag in dieser Tabelle öffnet sich der Dialog Job-Instanz , in welchem alle Details zum einzelnen Lauf sichtbar sind. 

Siehe Kapitel Job-Instanz .
