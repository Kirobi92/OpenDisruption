# Mehrere Rechnungsnummernkreise 
In eNVenta gibt es besondere Nummernkreise für den Rechnungsdruck. 

Hintergrund für die Verwendung dieser Sondernummernkreise ist auf Auftragsebene, dass während eines Rechnungsmassendrucks der Standard-Rechnungsnummernkreis sehr „belastet“ ist. So kann es passieren, dass zum Beispiel für die Faktura von Barverkaufsaufträgen keine Rechnungsnummer durch eNVenta gezogen werden kann. Dies hat zur Folge, dass ein Barverkaufskunde nicht in adäquater Zeit bedient werden kann, weil keine Rechnung erzeugt wird. 

Auch nachts besteht dieses Problem weiter. Nachts erfolgen typischerweise ebenfalls Rechnungsmassendrucke. Weil es auf Basis der Tourenverwaltung Nachnahmeaufträge gibt, deren Rechnungen typischerweise mit dem Generieren der Lieferscheine für die Tour mit ausgedruckt werden und dies ebenfalls nachts geschieht, besteht auch hier das Problem des nicht „erreichbaren“ Standard-Rechnungsnummernkreises. 

Die Hinterlegung der Rechnungs-Nummernkreise erfolgt pro Auftragsart. Im Standard werden die Nummern für die Faktura von allen Rechnungen und Gutschriften aus demselben Nummernkreis gezogen. Als Sondernummernkreis kann in den Auftragsarten nun ein abweichender Rechnungsnummernkreis je Auftragsart, z.B. für Barverkäufe hinterlegt werden. Dies sollte die Ausnahme sein! 

Damit die Gutschriften weiterhin den gleichen Nummernkreis verwenden kann in der Gut-schriftsart auf eine entsprechende Auftragsart mit Sonder-Nummernkreis verwiesen werden, z.B. Gutschriftauszahlung an der Kasse. 

In eNVenta erfolgt keine Prüfung, ob eine Nummernkreisnummer zum Format eines Nummernkreises passt!
