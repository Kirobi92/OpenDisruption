# Aufgabenstamm Registerkarte Nachverfolgung 
Aufgabenstamm Registerkarte Nachverfolgung 

Auf dieser Registerkarte legen Sie die Einstellungen für die Nachverfolgung der Aufgabe fest. Z.B. wer wann eine Wiedervorlage erhalten soll und ob eine Synchronisation mit Outlook stattfinden soll. 

So können Sie sicherstellen, dass die Aufgabe nicht vergessen wird. 
### Checkbox Keine Outl.-Sync. 
Über diese Checkbox steuern Sie, dass die Projektaufgabe keine Termine in den Outlook-Kalender schreiben kann. 

Wird eine bereits synchronisierte Projektaufgabe im Nachhinein auf Keine Outl.-Syn. gestellt, wird der entsprechende Outlook-Termin beim Ressourcenabgleich wieder gelöscht. 

### Combobox Terminart 
Hier können Sie den Typ des Termins auswählen. 

### Combobox Terminanzeige 
Über diese Combobox steuern Sie die Terminart für Outlook. Folgende Einstellungen werden angeboten: 

- beschäftigt - frei - mit Vorbehalt - abwesend 
Eine Änderung in eNVenta ERP übersteuert die Änderung in Outlook. 

### Combobox WV Sachbearb. 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld WV Text 
Hier können Sie den Text für eine Wiedervorlage eintragen. 

### Feld Aufgabe 
Es kann sinnvoll sein, nach Abschluss einer Aufgabe eine neue Folgeaufgabe zu erzeugen. Dazu geben Sie im Bereich Automatische Folgeaufgabe bei Abschluss eine Aufgabennummer ein. Diese Folgeaufgabe kann dann auch automatisch einem Benutzer zugewiesen werden. Dazu wählen Sie aus der Combobox Sachbearbeiter einen zuständigen Sachbearbeiter aus. 

Beispiel: Nach Abschluss einer Produktinstallation erhält die Verwaltung automatisch eine Aufgabe zur Fakturierung des Projektabschnitts. 

Tooltip: 

Angabe der Aufgabennummer
