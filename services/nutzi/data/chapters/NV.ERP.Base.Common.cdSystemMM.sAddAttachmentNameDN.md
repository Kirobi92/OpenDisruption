# Kapitel NV.ERP.Base.Common.cdSystemMM.sAddAttachmentNameDN

Hier geben Sie die Formel zur Ermittlung des Dateinamens ein bei der Generierung einer E-Mail für den Lieferschein-Druck. Es wird immer fix der Wert "Lieferschein" verwendet, gefolgt von der Formel als Dateiname.
