# Webshop Registerkarte Benutzerübersicht 
Registerkarte Benutzerübersicht 

Diese Registerkarte zeigt an, wie viele Besucher den Shop angeschaut haben. 
### Button Benutzer 

Mit diesem Button gelangen Sie in den Dialog Benutzerkonto . 

Siehe Kapitel Konfiguration der Webshop Benutzer . 

### Tabelle 

### Tabellenspalte Reg. Datum 
Hier wird das Registrierungsdatum angegeben. 

Tooltip: 

Registrierdatum 

### Tabellenspalte Benutzer 
Hier wird der Webshop-Benutzer angegeben. 

Tooltip: 

Angabe des Webshop-Nutzers 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Firma 1 
Der 1. Adresszeile der Firma wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Partner 
Hier wird der Name des Ansprechpartners angegeben. 

Tooltip: 

Name des Ansprechpartners 

### Button Akt. 

Mit diesem Button können Sie den Stand der Daten aktualisieren. 

Tooltip: 

Aktualisieren
