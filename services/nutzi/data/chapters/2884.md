# Konfiguration Transformer 
Konfiguration Transformer 

Der Dialog Konfiguration Transformer öffnet sich im Dialog Generischer IO Adapter über den Button neben der Combobox Transformation . 

In diesem Dialog ist es möglich, ein Objekt und seine Daten in ein anderes Objekt zu überführen. Dabei besitzt das Zielobjekt eine Methode, mit deren Hilfe sie sich selbst aus dem Quellobjekt erstellen kann. 

Eine Transformationskonfiguration enthält folgende Daten: 

- Typ des Quellobjekts - Typ des Zielobjekts - die aufzurufende Zielmethode - das Parameter-Mapping für die Methode 
So können beispielsweise Daten von einem Objekt a in ein Objekt b überführt werden. 

Eine Transformationskonfiguration kann zur Laufzeit konfiguriert werden. Die verfügbaren Objekte und Methoden werden im Transformationseditor abgerufen und verwendet. Dies gilt auch für die Parameter der Zielmethode. Diese können aus dem Quellobjekt ausgewählt und den Methodenparametern frei übergeben werden. 

Quell- und Zielobjekt können jeweils über die Buttons gewählt werden. 

Für das Parameter-Mapping stehen zwei Auswahlbereiche zur Verfügung, mit denen Auswahl und Zuordnung der Felder bequem erledigt werden können. 

Links können die verfügbaren Felder ausgewählt werden, rechts werden die Parameter der Zielmethode angezeigt. Durch den dazwischen wird das ausgewählte Feld einem Parameter zugeordnet, je nachdem, welcher Parameter rechts gerade angewählt ist. 

In der Tabelle Parameterzuordnung für Ziel-Methode können nicht nur Properties angegeben werden, sondern auch ein Ausdruckseditor oder ein Fixwert. Dafür stehen die gleichnamigen Spalten zur Verfügung. 

Für die Komponenten können spezielle Transformer-Komponenten genutzt werden. (Diese müssen entsprechend ausprogrammiert sein.) Es können auch beliebige andere Komponenten genutzt werden, welche über eine entsprechende Load-Bezeichnung verfügen, als auch die speziellen EDI-Komponenten für Lagerbestandsbericht, Bestellexport und Verkaufsexport. 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Referenz-Link 

Dieser Button steht zur Verfügung, um Dokumenten-Links sowie URL‘s als Info in eNVenta ERP zu hinterlegen oder um 2 Vorgänge/Referenzen miteinander zu verknüpfen. 

Siehe Kapitel Referenz-Links . 

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
