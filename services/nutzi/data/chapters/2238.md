# Parameter Webshop Registerkarte Warenkorb

E-Commerce Katalog / Parameter Webshop / Parameter Webshop Registerkarte E-Mail 

# Parameter Webshop Registerkarte Warenkorb 
Registerkarte Warenkorb 

Hier definieren Sie Einstellungen zum Warenkorb. 
### Feld Vorlage 
Angabe der E-Mail-Vorlage für die Warenkorb-E-Mail. 

Diese Vorlage hat zwingend eine E-Mail-Vorlage zu sein! 

### Button Suche 

Über diesen Button können Sie eine Vorlage suchen. 

Siehe Kapitel Dokument Vorlagen . 

### Feld E-Mail BCC 
Geben Sie eine E-Mail-Adresse an, welche eine Blindkopie der E-Mail bekommt. 

Der Empfänger sieht hiervon nichts. 

Um sicherzugehen, dass eine E-Mail an die richtige Adresse gesendet wird, prüft eNVenta beim Speichern oder Ändern einer Mailadresse, ob die Mail-Adresse der „Norm“ entspricht. Falls nicht wird ein Hinweis ausgegeben. Dabei wird das Speichern nicht grundsätzlich verhindert. Auf diese Weise werden z.B. Outlook-Adressbucheinträge wie z.B. „#EW_ERP“ bei Bedarf gespeichert und genutzt. 

Dabei läuft die Prüfung ausschließlich bei Anlage und Änderung einer Mail-Adresse. Werden andere Einträge, z.B. des Kundenstammsatzes geändert, so kommt diese Meldung nicht. 

Werden Stammdaten über das Gateway importiert, so werden u.U. fehlerhafte Mailadressen importiert (ggfs. mit Warnung). Bei einem Import unter Verwendung des Jobservers erfolgt dies ebenfalls in Form einer Warnung (im Jobprotokoll). 

### Combobox Report Anh. 
Geben Sie an, welcher Report an die Mail angehängt wird. Dies ist im Standard der Report “CartConfirmation”. 

### Combobox Sprache 
Geben Sie an, in welcher Sprache die E-Mail versendet wird. 

### Feld Voransicht 
Hier wird der Inhalt der E-Mail (aus der Vorlage) angezeigt.
