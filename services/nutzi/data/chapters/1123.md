# Handelsstücklisten auflösen 
Es ist möglich in Auftrag, Gutschrift und Angebot eine Handelsstücklistenposition nach ihrer Erfassung wieder aufzulösen, indem für jedes ihrer Endteile eine entsprechende Position in den Auftrag eingefügt wird. Diese erweiterte Verwaltung von Handelsstücklisten wird auch Set-Verwaltung genannt. 

Diese Funktion ist hinsichtlich der Verwaltung einzelner Positionen von Vorteil. 

Beispiel: 
Ein Lichtband besteht aus Leuchtmittel, Leuchtkörper und einer Aufhängung, welche nur zusammen verkauft werden. Die Verwaltung hierfür läuft über eine Handelsstückliste. Zunächst muss sichergestellt sein, dass alle Bestandteile dafür auf Lager sind. Ist nun eine dieser Komponenten ein Beschaffungsartikel, so können die restlichen Positionen bereits geliefert werden. Hierfür muss die Handelsstückliste in seine Komponenten aufgelöst werden und entsprechend im Auftrag gekennzeichnet, eingetragen sein. 

Um diese Auflösung zu ermöglichen, muss im Artikelstamm der Parameter für Artikel-Sets gesetzt werden.
