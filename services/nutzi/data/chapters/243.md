# Kunde(n) und Lieferant(en) in Besuchsplan eintragen 
Historie Besuchspläne 

Der Menüpunkt Optionen >> Kunde/n in Besuchsplan eintragen (z.B. im Kundenstamm, Auftrag) öffnet diesen Dialog. Damit können alle Kunden/Adressen, die sich in der Registerkarte Tabelle bzw. im Haupt-Dialog befinden, in einen Besuchsplan übernommen werden. 

Im Besuchsplan öffnet sich dieser Dialog über das Menü Optionen und den Button neben dem Feld Besuchsplannr . 

In der Aktivität öffnet sich dieser Dialog über Optionen >> Aktionsteilnehmer in Besuchsplan eintragen . 

Im Feld Vertreter geben Sie den Vertreter an (oder suchen ihn über den Button ), dessen Besuchspläne zur Auswahl angezeigt werden sollen. Sie können die Auswahl über die Felder Erfassungsdatum von / bis sowie die Checkboxen offen und besucht eingrenzen. 

Mit Markieren einer Zeile und Klick auf den Button können Sie diesen Besuchsplan in den Dialog Besuchsplan übernehmen.
