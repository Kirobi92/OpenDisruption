# Neue RP Position 
RP Neue Position 

Dieser Dialog öffnet sich über den Button Neu in der Rechnungsprüfung auf der Registerkarte Allgemein . 

Hier können Sie Kostenartikel erfassen, wie z.B. Fracht und Nebenkosten, welche zuvor nicht in der Bestellung erfasst wurden. Dazu legen Sie zuvor im Artikelstamm Artikelnummern an, speziell für Fracht bzw. Nebenkosten. Diese geben Sie ein oder wählen sie über den Button aus und füllen die anderen Felder entsprechend aus. 

Es können nur Artikel ohne Bestandsführung eingegeben werden. 

Diese neue Position wird als weitere Bestellposition in die zugrundeliegende Bestellung eingetragen.

Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten.

### Dock Panel Verwendung VK 

Dieser Bereich bietet 3 Optionen. 

Die gewählte Option wird als zusätzliches Feld in die Bestellposition eingetragen. So kann auch nachträglich erkannt werden, was der Benutzer bestimmt hat. 

Einem bereits fakturierten Auftrag können keine Fracht- oder Kostenpositionen mehr zugeordnet werden. 

Die Generierung bzw. Verteilung erfolgt erst mit der Freigabe der Rechnungsprüfung! 
### Radiobutton Keine Übernahme 
Wenn Sie diesen Radiobutton aktivieren, wird die Position nicht in den zugehörigen Auftrag übernommen. Dieser Radiobutton ist standardmäßig gesetzt, wenn die zuvor markierte Rechnungsprüfungsposition keinen Auftragsbezug hat. 

Tooltip: 

Wenn aktiviert, wird die Position nicht in den zugehörigen Auftrag übernommen. 

### Radiobutton Frachtposition 
Wenn Sie diesen Radiobutton aktivieren, wird die Position als Frachtkostenposition in den Auftrag übernommen. Dabei wird unterschieden, ob der Auftrag „Franko“ ist oder nicht. 

- Auftrag Franko: Die Frachtkosten aus der Rechnungsprüfung müssen auf Positionen als zusätzliche Nebenkosten verteilt werden (und zwar nur auf Positionen, die auch in der Rechnungsprüfung zu diesem Auftrag vorhanden sind). - Auftrag nicht Franko: Eine zusätzliche Frachtkostenposition wird generiert (VK = EK). Falls eine Frachtposition mit gleicher Artikelnummer vorhanden ist, tragen Sie nur den EK ein. 

Tooltip: 

Wenn aktiviert, wird die Position als Frachtkostenposition in den Auftrag übernommen. 

### Radiobutton Kostenposition 
Wenn Sie diesen Radiobutton aktivieren, wird die Position 1:1 in den Auftrag übernommen, d.h. die Position wird mit gleichem Artikel und Wert als neue Position generiert. 

Tooltip: 

Wenn aktiviert, wird die Position 1:1 in den Auftrag übernommen.
