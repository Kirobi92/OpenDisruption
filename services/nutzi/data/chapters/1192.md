# Positionen verschieben 
Positionen verschieben 

Mit diesem Dialog können Sie Positionen eines Vorgangs (z.B. Auftrag , Angebot oder Gutschrift >> Optionen >> Position verschieben ) in einen anderen Vorgang verschieben. Auf der rechten Seite des Dialogs werden die Daten aus dem aktuell geöffneten Auftrag, dem Angebot oder der Gutschrift angezeigt. 

Über den Button können Sie den Beleg suchen, aus welchem Sie eine oder mehrere Positionen in den aktuellen Vorgang übernehmen möchten. 

Über die Pfeilbuttons in der Mitte können Positionen von rechts nach links - und umgekehrt – verschoben und damit dem entsprechenden Auftrag zugeordnet werden. Es können einzelne aber auch mehrere Positionen gleichzeitig kopiert/verschoben werden. Dazu markieren Sie die entsprechenden Positionen in der Tabelle. 

Diese Positionen werden unterhalb der bereits vorhandenen Positionen angefügt. 

Wenn Sie die Checkbox Neu kalkulieren vor der Übernahme der Position aktivieren, wird der Preis neu kalkuliert. 

Ansonsten wird der Preis 1:1 übernommen. 

Wenn Sie die Checkbox Verschieben aktivieren, wird die „verschobene Position“ im Ursprungsbeleg gelöscht! 

Es gelten folgende Regeln: 

- Neue Positionen können nicht in fakturierte Belege eingefügt werden. - Eine Mehrfachselektion ist möglich. - Beim Verschieben muss zusätzlich nach dem Kopieren die „alte" Position gelöscht werden. Das ist nur erlaubt, wenn Status < 3 und kein Eintrag für Packlistennummer und Lieferscheinnummer existiert. - Es ist zwingend notwendig, dass nach Verlassen des Dialogs der Vorgang im aufrufenden Dialog neu eingelesen wird.

### Checkbox Verschieben 
In der Positionsumwandlung werden auch Kopiervorgänge dokumentiert. Dafür nutzen Sie die Checkbox Verschieben . 

Wenn die Checkbox Verschieben gesetzt ist, markieren Sie einen Eintrag in der entsprechenden Tabelle und verschieben ihn mit dem entsprechenden Pfeil-Button. Damit wird ein Eintrag beim Verschieben ganz allgemein erzeugt. 

Die "kopierte" Position im Ursprungsbeleg wird gelöscht. 

Soll die kopierte Position im Ursprungsbeleg nicht gelöscht werden, verschieben Sie die Position, ohne dass die Checkbox Verschieben gesetzt ist! 

Das Verschieben einer abgeschlossenen Auftragsposition wird auch mit der Aktion Verschieben gekennzeichnet, auch wenn rein technisch ein Kopiervorgang und ein logisches Löschen erfolgt.
