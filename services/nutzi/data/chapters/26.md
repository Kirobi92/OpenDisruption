### Bezeichnungsfeld Schritt 5: Versions-Informationen aktualisieren 
Schritt 5: Versions-Informationen aktualisieren 

Setzen der Version. Mit Klick auf den Button Ausführen wird die Version als 'aktualisiert' in der Datenbank vermerkt. Beim nächsten Start von eNVenta erscheint der Update-Assistent nicht mehr. 

Wenn beim Start noch ein Update-Script auszuführen ist, der Benutzer aber keine Berechtigung hat, so erscheint eine entsprechende Hinweismeldung.

Mit diesem Button wird Schritt 5 ausgeführt.

### Button CSC-Bereinigung 

Diesen Button drücken Sie nur einmalig bei einem Update auf eNVenta 4.0. Es öffnet sich der Dialog CSC-Bereinigung für eine Analyse. Dieser Vorgang kann mehrere Minuten dauern. Anschließend werden alle Bereinigungsmaßnahmen aufgelistet. 

Lesen Sie dazu die Update-Anleitung für das Client Side Customizing . 

Siehe Kapitel CSC-Bereinigungen . 

### Checkbox Hilfe URL löschen 
Damit nach einem Update von eNVenta auch immer die aktuelle Online-Hilfe eingebunden wird, ist die URL für die Online-Hilfe eine fixe URL. Dies gilt auch, wenn von Standard auf WS upgedatet wird! Auf diese Weise steht immer die Standard-Online-Hilfe für eNVenta zur Verfügung. 

Wenn Sie mit Ihrer eigenen Online-Hilfe weiterarbeiten möchten, so deaktivieren Sie die Checkbox. 

Durch die leere URL werden Sie beim Aufruf der Online-Hilfe durch drücken von F1 auf die zentrale Online-Hilfe umgeleitet. Diese zentrale Online-Hilfe wird von Nissen & Velten gepflegt. 

Pro eNVenta -Version und in Abhängigkeit ob Standard oder Großhandelspaket wird die entsprechende Online-Hilfe aufgerufen. 

Die allgemeine URL lautet hierbei https://help.nissen-velten.de/{1}_DE{2}/?load={0} 

{1} steht für die Versionsnummer 
{2} steht für das optionale Package, z.B. _WS 
{0} steht für den Dialog 

Tooltip: 

Löscht die vorhandene Hilfe URL und aktiviert die Online Hilfe.
