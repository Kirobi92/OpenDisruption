# Preisänderung 
Preisänderung 

Dieser Dialog öffnet sich über den Button Preisänderung im Dialog Preisoptimierung . 

Es können die Daten für eine Preisänderung eingegeben werden. Dazu muss immer eine Preisgruppe angegeben werden und zusätzlich entweder ein Prozentsatz oder ein Betrag. 

Der Button löst die Buchung für die zuvor markierten Artikel aus. 

Achtung: Gibt es bei einem Artikel keinen Eintrag zu der angegebenen Preisgruppe, wird die Buchung für diesen Artikel nicht ausgeführt. Es entstehen also KEINE NEUEN Preisgruppeneinträge, sondern lediglich Änderungen bestehender Preise. 

Siehe Kapitel Preisoptimierung .

### Feld Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind!
