# Material disponieren Registerkarte Umlagern 
Material disponieren Registerkarte Umlagern 

Die Registerkarte Umlagern steht für die Umlagerung von Materialbeständen zur Verfügung. 

Für eine vereinfachte Umlagerung von Materialbeständen aus dem Materiallager auf das Produktionslager erzeugt eNVenta ERP mit dem Button Vorschlag die möglichen Einträge für die Tabelle. Dabei werden automatisch nur Positionen berücksichtigt, für die Bedarf besteht und Bestand auf dem Materiallager zur Verfügung steht. Die Menge wird automatisch der Situation angepasst. 

Alternativ kann auch mit dem Button eine Position übernommen werden. Eventuell muss die Menge durch den Sachbearbeiter angepasst werden. 

Vor der Umbuchung muss immer eine Chargenzuordnung erfolgt sein. Diese wird bei der Umbuchung nochmals geprüft, ggf. wird die Umbuchung abgelehnt. 

Siehe Kapitel eNVenta ERP Allgemeine Funktionalitäten / Chargenauswahl . 

Um vor der Umbuchung von Material vom Materiallager auf das Produktionslager eine manuelle Zuordnung von Plätzen zu ermöglichen, steht der Button Plätze auf der Registerkarte Umlagern zur Verfügung. 

Der Button Plätze ist nur aktiv, wenn das Materiallager chaotisch ist. Es erscheint der Zuordnungsdialog für die Auslagerung von Lagerplätzen. 

Ist der Artikel chargengeführt, muss zuerst die Charge festgelegt werden, bevor ein Platz zugeordnet werden kann. Es können dann auch nur Plätze ausgewählt werden mit Artikeln dieser Charge. 

Sobald eine manuelle Platzzuordnung vorgenommen wird, ist der Benutzer für die Vollständigkeit verantwortlich. Eine automatische Korrektur/Ergänzung der Plätze findet nicht statt. 

Ist kein Platz zugeordnet, erfolgt eine automatische Zuordnung. 

Sicherheitshalber werden beim Löschen oder Abschließen einer Materialposition alle evtl. noch vorhandenen Platzzuordnungen gelöscht. 

Kann die Umlagerung vom Kommissionierer nicht genau wie vorgegeben durchgeführt werden, muss dies durch manuelle Korrektur-Umbuchungen berichtigt werden. 
### Button Vorschlag 

Mit diesem Button werden für eine vereinfachte Umlagerung von Materialbeständen aus dem Materiallager auf das Produktionslager die möglichen Einträge für die Tabelle erzeugt. Dabei werden automatisch nur Positionen berücksichtigt, für die Bedarf besteht und Bestand auf dem Materiallager zur Verfügung steht. Die Menge wird automatisch der Situation angepasst.

Tooltip: 

Auslagern von Lagerplätzen 

### Button Umbuchen 

Der Button Umbuchen führt die Zu- und Abbuchungen in einer Transaktion durch. Dies beinhaltet auch die Chargenbuchung und die Lagerplatzbuchung im Falle eines chaotischen Lagers. Die Zuordnung der Lagerplätze erfolgt dabei voll automatisch auf Basis des in den Verkaufsparametern angegebenen Verfahrens. Ist kein Verfahren angegeben, wird Verfahren 1 (alphabetisch) ausgeführt. Daher ist es wichtig, dass bei chargenführenden Artikeln die Charge vorher (über den Button Charge ) ausgewählt wird. 

Es ist auch möglich, eine Rückbuchung von Material aus dem Produktionslager auf das Materiallager zu buchen. Dazu muss lediglich die Menge negativ angegeben werden. 

Bei erfolgreich durchgeführten Buchungen wird die Zeile grün markiert, sonst evtl. rot oder es erscheint eine Fehlermeldung. 

Die Umbuchung kann mehrfach ausgeführt werden. 

Tooltip: 

Führt Umbuchungen durch. 

### Button Drucken 

Mit diesem Button können Sie eine Vorschlagsliste drucken. 

Da die chaotischen Lagerplätze nur temporär während des Buchungsvorgangs zugeteilt und festgehalten werden, können diese auf der Liste nicht angedruckt werden. Statische Plätze werden angedruckt. 

Tooltip: 

Druckt eine Vorschlagsliste.

Diese Spalte beinhaltet die Menge, die bereits auf das Produktionslager umgebucht wurde. 

Ein Doppelklick in diese Spalte öffnet das Lagerjournal.
