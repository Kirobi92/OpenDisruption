# Serviceeinsatz Registerkarte Positionen 
Serviceeinsatz Registerkarte Positionen 

Auf dieser Registerkarte erhalten Sie eine Übersicht über die Positionen (Artikelnummern für Material oder bestimmte Prüfungen, z.B. den Test einer Anlage nach Wartung), die zu einem Einsatz definiert sind. 

Des Weiteren können auf dieser Registerkarte auch neue Positionen angelegt werden. 
### Button Neu 

Über diesen Button können Sie weitere Positionen zu einem Serviceeinsatz anlegen. Auch Stücklistenartikel können angelegt werden (allerdings nur Endteile). 

Tooltip: 

Generiert eine neue Position. 

### Button Löschen 

Dieser Button löscht eine Zeile der Tabelle. 

Tooltip: 

Löscht eine markierte Position. 

### Button Vorlagen 

Dieser Button fügt in die Tabelle Artikel ein, die im Menü Einsatzarten angelegt wurden. Alle nicht gesperrten Artikel werden übernommen, über die gesperrten Artikel wird eine Meldung angezeigt. 

Siehe Kapitel Einsatzarten . 

Tooltip: 

Liest Artikel aus den Einsatzarten ein. 

### Button Freigeben 

Dieser Button gibt den Artikel zur Bearbeitung frei. Der Status ändert sich dabei auf ‚freigegeben‘. Der Positionstext eines freigegebenen Artikels kann nicht mehr geändert werden! 

Tooltip: 

Freigabe einer Position zur Bearbeitung. 

### Combobox Lager 
Wählen Sie aus dieser Combobox ein Lager aus, so werden alle Artikel von diesem Lager entnommen. In der Tabelle kann ein anderes Lager zum Artikel gewählt werden. 

Tooltip: 

Definiert das Lager 

### Tabelle 

### Tabellenspalte Pos 
Die Nummer der Position wird automatisch vom System vorgegeben. 

Tooltip: 

Positionsnummer 

### Tabellenspalte Status 
Der Status des Artikels in diesem Einsatz wird eingetragen. Wenn Sie einen weiteren Artikel in die Tabelle eingeben, ist der Status dieses Artikels offen . 

Tooltip: 

Angabe des Status 

### Tabellenspalte Artikel 
Die Nummer des Artikels wird eingetragen. 

Gesperrte Artikel können nicht in einem Serviceeinsatz aufgelistet werden, unabhängig von der Eingabe - ob direkt eingegeben oder über Artikelauswahl, Einsatzart oder bei der Schnellerfassung des Serviceeinsatzes. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Der Name des Artikels wird automatisch eingetragen, wenn die Artikelnummer eingegeben wird. Der Name kann geändert werden. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Menge 
Tragen Sie die Menge des benötigten Artikels ein. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Einheit VK 
Die Verkaufseinheit des Artikels wird eingetragen. 

Tooltip: 

Verkaufseinheit 

### Tabellenspalte VK pro 
Der Verkaufspreis pro Artikeleinheit wird eingetragen. 

### Tabellenspalte Lager 
Wählen Sie aus dieser Combobox das Lager aus, von dem der Artikel entnommen werden soll. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Keine Fakturierung 
Wenn Sie diese Checkbox aktivieren, wird der Artikel in den Auftrag übernommen und in die Spalte ‚Preis‘ wird 0,00 eingetragen. 

Tooltip: 

Wenn aktiviert, wird keine Rechnung gestellt. 

### Tabellenspalte Beleg 
Hier wird die Nummer des Belegs aus den Belegarten Auftrag, Gutschrift, Bestellung und Belastung (Wawi) angegeben. 

Tooltip: 

Angabe der Belegnummer 

### Tabellenspalte Belegart 
Wenn dieses Feld gefüllt ist, wird in der Auftragserfassung die hier hinterlegte Auftragsart voreingestellt. Sie kann jederzeit geändert werden. 

Dieses Feld steht in direktem Zusammenhang mit dem Parameter Auftragsart fix im Dialog Auftragsarten . 

Tooltip: 

Angabe der Belegart 

### Tabellenspalte Bearbeitet 
Das Datum der letzten Wartung wird eingetragen, wenn im Kopf des Dialogs die Checkbox Letzte Wartung aktiviert wurde. 

Tooltip: 

Datum der letzten Änderung 

### Button Nach oben verschieben 

Mit diesem Button können Sie eine Position der Tabelle nach oben verschieben. 

Tooltip: 

Position nach oben verschieben. 

### Button Nach unten verschieben 

Mit diesem Button können Sie eine Position der Tabelle nach unten verschieben. 

Tooltip: 

Verschiebt eine Position nach unten. 

### Feld Postext 
Hier wird der Text zur Position angezeigt. 

Tooltip: 

Zusätzlicher Text zur Position 

### Feld Beleg 
Ist zu einer Position der Tabelle ein Auftrag generiert worden, so steht die Nummer des Auftrags in diesem Feld. 

Wird ein Auftrag, der aus einem Serviceeinsatz generiert wurde, gelöscht oder storniert, so werden der zugehörige Serviceeinsatz und alle seine Positionen wieder geöffnet. Der Status des Serviceeinsatzes wird wieder auf „offen“ gesetzt. Der Status der einzelnen Positionen wird ebenfalls auf „offen“ gesetzt. 

Wird eine einzelne Auftragsposition gelöscht, so erscheint eine entsprechende Warnmeldung, der Serviceeinsatz bleibt allerdings unverändert. 

Tooltip: 

Angabe der Belegnummer 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Auftrag.
