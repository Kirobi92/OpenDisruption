# Kapitel NV.ERP.Base.Common.cdSystemBase.binLogoBig

Es können alle möglichen Grafiken ausgewählt werden (bmp, gif, jpg, png, tiff, …). Der -Button ermöglicht die Suche über eine Dateiauswahl. 

Die Bilder werden intern im PNG-Format gespeichert (komprimiert ohne Verluste) und auch in diesem Format an den Report übergeben. 

Möchten Sie das Logo entsprechend austauschen, so beachten Sie bitte diese Größe im Original-Bild und füllen dies ggf. durch einen transparenten Bereich auf, falls Ihr Logo kleiner ausfallen sollte. 

Tipp zur Vorgehensweise: 

Es kann eine Word-Vorlage verwendet werden, in der Sie Ihr Logo an die entsprechende Stelle platzieren. 

Öffnen Sie die mitgelieferte Word-Datei. Diese finden Sie als Ressource im Broker. 

Per Rechtsklick -> "Bild ändern" öffnet sich der Explorer und Sie können das vorhandene Bild mit Ihrem Logo ersetzen. 

Per Rechtsklick -> "Größe und Position" können Sie die Größe und Position des Bildes definieren. 

Die definierte Word-Vorlage muss anschließend noch in eine Bild-Datei umgewandelt werden: 

Dafür öffnen Sie folgende Webseite: http://www.zamzar.com/ 

Schritt 1: Über „durchsuchen“ öffnet sich der Explorer, indem Sie ihre Word-Vorlage auswählen, die konvertiert werden soll. 

Schritt 2: Wählen Sie das Format aus, in das die Vorlage umgewandelt werden soll. 
Am besten sollte das PNG-Format verwendet werden. 

Schritt 3: Fügen Sie ihre E-Mailadresse ein. 

Schritt 4: Konvertieren Sie die Datei. 

Anschließend erhalten Sie eine E-Mail mit dem Downloadlink. Über den Link wird die Webseite von Zamar geöffnet, auf der Sie ihr Logo downloaden können. 

Die Standardmaße des LogoBig oder LogoBigAlternativ sind 21,00 x 4,40 cm. 

Tooltip: 

Das Seitenverhältnis dieses Logos entspricht 21:4,4.
