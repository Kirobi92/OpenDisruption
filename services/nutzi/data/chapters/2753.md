# Leistungen Registerkarte Kostenstellen 
Leistungen Registerkarte Kostenstellen 

Auf dieser Registerkarte werden die Kostenstellen hinterlegt, auf welche die Leistungen verteilt werden sollen. Bei Neuanlage einer Leistung ist die Tabelle noch leer. 
### Button Auswählen 

Mit diesem Button öffnen Sie die Kostenstellen-Wahl, um der Kostenart Kostenstellen zuzuordnen. Haben Sie die Umlageart % gewählt, müssen Sie einen Verteilungsschlüssel angeben; die Summe muss 100 % betragen. 

Ist die Umlageart „Schlüssel“ gewählt, werden nur Kostenstellen angeboten, welche den gleichen Umlageschlüssel besitzen. 

Um Kostenstellen aus der Zuordnung zu löschen, müssen Sie dies ebenfalls im Menü Kostenstellen-Wahl durchführen. 

Siehe Kapitel Kostenstellenauswahl .
