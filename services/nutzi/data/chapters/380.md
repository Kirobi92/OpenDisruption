# Artikel Cockpit Registerkarte Rahmen 
Artikel Cockpit Registerkarte Rahmen 

Auf dieser Registerkarte erhalten Sie eine Sicht auf 

- alle Rahmenauftragspositionen zum Artikel (Bereich Rahmenauftrag ) - alle Rahmenbestellungen zum Artikel (Bereich Rahmenbestellung ) - offene Abrufe zur markierten Rahmenauftragsposition (Bereich Verkauf Rahmenabrufe ) - offene Abrufe zur markierten Rahmenbestellposition (Bereich Einkauf Rahmenabrufe ) 
Ein Doppelklick in eine einzelne Zeile öffnet den zugrundeliegenden Vorgang. 
### Tabelle Rahmenauftrag 

### Tabellenspalte Vertrag 
Hier wird die Vertragsnummer angegeben. 

Tooltip: 

Angabe der Vertragsnummer 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer

6-storniert

Angabe des Positionsstatus

### Tabellenspalte VK pro 
Hier wird eine Preiseinheit angegeben. Dieser Preis kann sich auf 10, 100, 1000 oder eine andere Anzahl Artikeleinheiten beziehen. Standard ist der Wert 1. 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- 1 Einheit - je 10 Einheiten - je 100 Einheiten

### Tabellenspalte Beschreibung 
Zu einer Position kann ein bis zu 2000 Zeichen langer Zusatztext eingegeben und in den Formularen gedruckt werden. 

Bitte achten Sie darauf, einen doppelten Druck des Textes in den Formularen zu verhindern, indem Sie nicht den Artikeltext, sondern nur den Positionstext drucken. 

Beachten Sie außerdem, dass alle Änderungen im Dialog Position auch im zugrundeliegenden Vorgang mit dem Button gespeichert werden müssen.

### Tabellenspalte Ohne Spezifikation 
Wenn Sie diese Checkbox aktivieren, wird der Artikel als "Dummy-Artikel" verwendet. D.h. dieser Artikel kann für Rahmenabrufe verwendet werden und dient als Platzhalter für alle Artikel aus der Artikelgruppe, in der er sich befindet. Auf diese Art können Sie auf eine Artikelgruppe einen Rabatt für eine bestimmte Abrufmenge geben.
