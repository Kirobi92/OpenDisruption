# Kostenauswertungen Registerkarte Base Reporting 
Kostenauswertungen Registerkarte Base Reporting 

Diese Registerkarte steht nur für die Auswertungen nach Kostenart (Radiobutton Kostenart aktivieren) zur Verfügung. Hier stehen folgende festdefinierte Auswertungsblöcke zur Verfügung, die miteinander kombiniert werden können: 

- Budget - Monatl. Ergebnis - Diff. monatl. % - Endergebnis - Diff. Ergebnis % 
Die Basiszahl wird beim Aufbau der Auswertung (im Menü Auswertungen >> Kostenausw. Verwalten ) definiert. 

Eine Druckfunktion ist nur speziell für diese Registerkarte vorhanden. 
### Checkbox Planwerte 
Wenn Sie diese Checkbox aktivieren, wird die Spalte 'Plan Jahr' eingeblendet. Egal welche Periode von/bis gewählt ist, wird hier der Jahressaldo ausgewiesen. 

Tooltip: 

Blendet Spalte "Plan Jahr" ein. Egal welche Periode von/bis gewählt ist, hier wird der Jahressaldo ausgewiesen.
