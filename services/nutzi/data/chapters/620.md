# Massenänderung Registerkarte Vorschau 
Massenänderung Registerkarte Vorschau 

Auf dieser Registerkarte werden sämtliche Datensätze aufgelistet, für welche die Änderung übernommen werden soll. Also z.B. bei einer Änderung in einem Lieferanten-Stammsatz sämtliche Lieferanten. 

Sie haben jetzt die Möglichkeit Einträge zu ändern (dazu muss die Checkbox Manueller Modus aktiviert werden), bzw. Einträge zu löschen oder auf den ursprünglichen Wert zurück zu setzen. Dazu stehen hier ebenfalls die Buttons Löschen bzw. Reset zur Verfügung. 

Bei einer Massenänderung im Rabatt Einkauf stehen hier zusätzlich die Spalten 'Artikelnr' und 'Bezeichnung' zur Verfügung. So ist ersichtlich, ob die zu ändernde Rabattnummer an einem Artikel hängt. 

Sind auf dieser Registerkarte Fehler vorhanden und diese werden nicht gelöscht, werden beim Speichern überhaupt keine Daten geändert! 
### Button Löschen 

Mit diesem Button kann ein Eintrag der Tabelle gelöscht bzw. von der Änderung ausgeschlossen werden. 

Tooltip: 

Selektierte Zeilen löschen 

### Button Zurücksetzen 

Mit diesem Button kann ein Eintrag wieder in den Ursprungszustand zurückgesetzt werden. 

Tooltip: 

Auf Ursprungszustand zurücksetzen 

### Checkbox Manueller Modus 
Wenn Sie diese Checkbox aktivieren, können Sie die Einträge ändern.
