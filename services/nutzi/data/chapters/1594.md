# Lagerplätze in chaotischen Lagern 
Lagerplatz 

In diesem Dialog können sämtliche Lagerplätze in chaotischen Lagern angezeigt und bearbeitet werden. Nach der Auswahl des Lagers können die Bereiche ausgewählt werden, deren Plätze angezeigt werden sollen. 

In der Logistik ist ein Lagerplatz definiert als ein eindeutig identifizierbarer Ort in einem Regal, in dem Güter gelagert werden können. Jeder Lagerplatz besitzt eine Bezeichnung aufgrund seiner Koordinaten (System zur eindeutigen Identifikation von Lagerplätzen) im Lager. Diese Koordinate setzt sich aus folgenden Informationen zusammen: 

- Ebene / Etage - Gang - Haus - Y-Achse immer in dm - X-Achse immer in dm - Prüfziffer 
Siehe nachfolgende Abbildung: 

Lagerplatz Koordinaten 

Wenn der Parameter automatische Lagerplatznr. ( Lagerorte , Registerkarte LVS Allgemein ) nicht aktiviert ist, kann der Lagerplatz frei vergeben werden. 

Jeder Lagerplatz hat eine Prüfziffer, wenn der Parameter Prüfziffer am Lagerort benutzen? ( Lagerorte , Registerkarte LVS Allgemein ) aktiviert ist. Bei dieser Prüfziffer handelt es sich um eine dreistellige Nummer, welche als Identifikation dient, wenn der Lagerplatz nicht anhand eines Scans identifiziert werden kann. Die Prüfziffer wird automatisch vom System generiert. 

Siehe Kapitel Lagerorte . 

### Lagerplatzetiketten 
Der Lagerplatz wird nicht vom zugeordneten Artikel nachgelesen, sondern die zugeordneten Artikel vom Lagerplatz. 

Für die Labels gilt folgender Aufbau: 

Es gibt 3 Etiketten. Dies sind Label Small , Label Medium und Label Big . Diese unterscheiden sich in der Basis nur in der Größe. 

Auf dem Label (Etikett) Big wird der Barcode standardmäßig nicht angedruckt. Der entstehende Platz wird für die größer geschriebene Lagerplatzinformation genutzt. 

Die Nummern für Gang, Ebene etc. müssen 2-stellig sein.

### Field Panel Ebene 

Hier geben Sie die Informationen für die Ebene an. Dieser Wert muss 2-stellig sein. 

Mit 'Ebene' ist hier das Stockwerk im Lager bzw. auch die physische Lagerhallennummer gemeint. 
### Feld von 
Zur Selektion eines Lagerplatzes oder eines Lagerbereichs muss eine zweistellige Nummer für die Ebene eingetragen werden. Der Wert hier definiert den Anfang der Ebene. 

### Feld bis 
Zur Selektion eines Lagerbereichs muss eine zweistellige Nummer für die Ebene eingetragen werden. Der Wert hier definiert das Ende der Ebene. 

### Field Panel Gang 

Hier geben Sie die Informationen für den Gang an. Dieser Wert muss 2-stellig sein. 

Ein Gang ist sozusagen eine Seite des Weges, also eine Regalreihe. 
### Feld von 
Zur Selektion eines Lagerplatzes oder eines Lagerbereichs muss eine zweistellige Nummer für den Gang eingetragen werden. Mit Gang ist hier der Weg zwischen den Regalen gemeint. 

Hier in diesem Feld geben Sie den Anfang des Gangs an. 

### Feld bis 
Zur Selektion eines Lagerbereichs muss eine zweistellige Nummer für den Gang eingetragen werden. 

Hier geben Sie das Ende des Gangs an. 

### Field Panel Haus 

Hier geben Sie die Informationen für das Haus an. Dieser Wert muss 2-stellig sein. 

Ein Haus ist der Abschnitt zwischen 2 vertikalen Trägern eines Lagerregals, d.h. ein Abschnitt des Regals. 
### Feld von 
Zur Selektion eines Lagerplatzes oder eines Lagerbereichs muss eine zweistellige Nummer für das Haus eingetragen werden. 

Der Wert hier definiert den Anfang des Hauses. 

### Feld bis 
Zur Selektion eines Lagerbereichs muss eine zweistellige Nummer für das Haus eingetragen werden. 

Der Wert hier definiert das Ende des Hauses. 

### Field Panel Y-Koordinate 

Hier geben Sie die Informationen für die Y-Koordinate an. Dieser Wert muss 2-stellig sein. 

Die X- und Y-Koordinaten dienen der Positionierung innerhalb des Hauses. D.h. diese Werte definieren die genaue Position in horizontaler und vertikaler Ausrichtung. Die Nutzung der Koordinaten kann kundenindividuell interpretiert werden. 
### Feld von 
Zur Selektion eines Lagerplatzes oder eines Lagerbereichs muss eine zweistellige Nummer für die Y-Koordinate eingetragen werden. Der Wert definiert den Anfang auf der Y-Koordinate. 

### Feld bis 
Zur Selektion eines Lagerbereichs muss eine zweistellige Nummer für die Y-Koordinate eingetragen werden. Der Wert definiert das Ende auf der Y-Koordinate. 

### Field Panel X-Koordinate 

Hier geben Sie die Informationen für die X-Koordinate an. Dieser Wert muss 2-stellig sein. 

Die X- und Y-Koordinaten dienen der Positionierung innerhalb des Hauses. D.h. diese Werte definieren die genaue Position in horizontaler und vertikaler Ausrichtung. Die Nutzung der Koordinaten kann kundenindividuell interpretiert werden. 
### Feld von

### Feld bis
