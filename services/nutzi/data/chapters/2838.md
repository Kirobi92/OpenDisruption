# Monitor Empfehlungen 
Monitor Empfehlungen 

Dieser Dialog öffnet sich aus dem Navigator >> Monitor Empfehlungen . 

In diesem Dialog können die verschiedenen Empfehlungsagenten erstellt und konfiguriert werden. Der Dialog zeigt zudem die Nutzung und den Erfolg der Agenten mit Hilfe verschiedener Kennzahlen. 

Mit dem Button wird ein neuer Empfehlungsagent angelegt. Der Parameterdialog wird aufgerufen. 

Per Doppelklick kann ein bestehender Agent zur Bearbeitung oder zum Löschen aufgerufen werden. 

Mit einem einfachen Klick auf einen Agenten wird dessen Statistik angezeigt. Die Konversionsraten bedeuten den Erfolgsanteil der Empfehlungen. Dazu wird die Anzahl der Empfehlungen zur Anzahl der Käufe (derselben Kunden) ins Verhältnis gesetzt. Durch den Verlauf der Konversionsraten kann auf die Erfolgsentwicklung des Agenten geschlossen werden. Unabhängige Ereignisse, wie Artikelaktionen, Marktveränderungen etc. können hierbei ebenfalls einwirken. 

Die Häufigkeitsentwicklung der Empfehlungen dieses Agenten wird in der zweiten Graphik gezeigt. 

Nach der erfolgreichen Generierung der Tupel können diese Daten nun mittels des Empfehlungsagenten analysiert werden, damit der Sachbearbeiter diese Daten im täglichen Geschäft nutzen kann. Die Generierung erfolgt hierbei mittels eines Empfehlungsagenten. Hierzu müssen Sie im Dialog Monitor Empfehlungen im oberen Bereich einen neuen Agenten anlegen (Button neben der Tabelle). Bei der Markierung eines Agenten können Sie darunter über eine zeitliche Beobachtung den Erfolg der Empfehlungen betrachten. 

Die Generierung der Empfehlungen erfolgt über den Dialog Empfehlungsagent . 

Siehe Kapitel Tupel-Generierung .
