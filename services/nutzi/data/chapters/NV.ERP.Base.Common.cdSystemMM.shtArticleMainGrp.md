# Kapitel NV.ERP.Base.Common.cdSystemMM.shtArticleMainGrp

Hier kann die Anzahl der Stellen festgelegt werden, welche die Artikel-Hauptgruppe der Artikelgruppen bestimmen. 

Beispiel: 
Es existieren folgende Artikelgruppen: 

| EW | Eisenwaren (Hauptgruppe) |
| --- | --- |
| EWGS | Gewindeschrauben |
| EWFS | Flügelschrauben |

Dies ermöglicht eine Konditionierung auf Artikel-Hauptgruppenebene. In der Prioritätenreihenfolge steht die Artikel-Hauptgruppe unterhalb der Artikelgruppe. 

Siehe Kapitel Preisermittlung im Verkauf . 

Siehe Kapitel Preisermittlung im Einkauf . 

Wenn Sie nachträglich die Zeichenlänge einer Artikelhauptgruppe erhöhen, erfolgt eine Prüfung, wenn noch Artikelzuordnungen zu Artikelgruppen der gleichen Anzahl Stellen existieren. Dann wird eine Änderung abgelehnt. 

Die Artikel-Hauptgruppe muss in den Warengruppen erfasst werden, wenn für die Artikel-Hauptgruppe in der Rabattverwaltung spezielle Konditionen hinterlegt werden sollen. 

Ein Artikel kann nicht direkt einer Hauptgruppe zugeordnet werden.

Zeichen Artikel-Hauptgruppe 
Hier kann die Anzahl der Stellen festgelegt werden, welche die Hauptgruppe der Artikelgruppen bestimmen. 
Beispiel: 
Es existieren folgende Artikelgruppen: 
EW - Eisenwaren (Hauptgruppe) 
EWGS - Gewindeschrauben 
EWFS - Flügelschrauben 
Dies ermöglicht eine Rabattierung auf Artikel-Hauptgruppenebene. 
In der Prioritätenreihenfolge steht die Artikel-Hauptgruppe unterhalb der Artikelgruppe.
