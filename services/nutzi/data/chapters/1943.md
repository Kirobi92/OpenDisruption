# Produktionsarten Registerkarte Druck 
Produktionsart Registerkarte Druck 

Hier definieren Sie zu einer Produktionsart die Einstellungen für den Druck, wie z.B. das Formular und den Drucker. 
### Button Customize 

Über diesen Button können Sie den Report anpassen. 

Tooltip: 

Report Customizing 

### Combobox Formular BA 
Hier wird das Druck-Formular für den Betriebsauftrag ausgewählt. 

Tooltip: 

Formular für Betriebsauftrag 

### Combobox Drucker BA 
Hier wird der Name des Druckers angegeben. 

Tooltip: 

Name des Druckers 

### Feld Anzahl BA 
Hier geben Sie an, wie oft der Betriebsauftrag gedruckt werden soll. 

Tooltip: 

Anzahl an Ausdrucken
