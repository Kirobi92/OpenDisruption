# X Mapping Test Registerkarte Daten 
X Mapping Test Registerkarte Daten 

Auf dieser Registerkarte wird das Ergebnis eines Imports angezeigt, bzw. setzen Sie hier die Werte für einen Export. 
### Checkbox Nur Änderungen 
Wenn Sie diese Checkbox aktivieren, werden nur Änderungen eingelesen. 

### Button Laden (gem. Kontext) 

Tooltip: 

Wenn das Datenobjekt einen X-Mapping-Kontext unterstützt, 
werden die Daten gemäß den Kontext-Parametern geladen. 

### Button Neues Listen-Element 

Dieser Button fügt einer ausgewählten Liste (Collection) ein neues Element hinzu. 

Tooltip: 

Fügt einer ausgewählten Liste (Collection) ein neues Element hinzu. 

### Button Listen-Element löschen 

Dieser Button löscht ein ausgewähltes Listen-Element aus seiner Liste (Collection). 

Tooltip: 

Löscht ein ausgewähltes Listen-Element aus seiner Liste (Collection). 

### Feld SQL-Bedingung 
Hier geben Sie die SQL-Bedingung zum Laden von Test-Daten an. 

Tooltip: 

SQL-Bedingung zum Laden von Test-Daten. 

### Button SQL Bedingung erstellen 

Über diesen Button kann eine SQL-Bedingung bearbeitet werden. 

Tooltip: 

SQL-Bedingung editieren 

### Button Suchen 

Mit diesem Button werden Daten mittels der angegebenen SQL-Bedingung geladen. 

Tooltip: 

Daten werden mittels der angegebenen SQL-Bedingung geladen. 

### Feld Eigenschaft 
Hier wird die Eigenschaft eines markierten Elements der Tabelle angezeigt. 

Tooltip: 

Anzeige der Eigenschaft eines markierten Elements in der Tabelle. 

### Feld Typ 
Hier wird der Typ eines markierten Elements der Tabelle angezeigt. 

Tooltip: 

Typ der Objekteigenschaft 

### Feld Wert 
In diesem Feld geben Sie den Wert der Objekteigenschaft an. Beim Import werden hier die importierten Werte angezeigt und können kopiert bzw. manipuliert werden. 

Tooltip: 

Der Wert der Objekteigenschaft wird angegeben. 

### Feld Tooltip 
Hier geben Sie den Tooltip der Objekteigenschaft an. 

Tooltip: 

Der Tooltip der Objekteigenschaft.
