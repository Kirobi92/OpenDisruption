# Sachbearbeiter- Scanner Registerkarte Menü

Lagerverwaltungssystem (LVS) / Sachbearbeiter- Scanner 

# Sachbearbeiter- Scanner Registerkarte Menü 
Sachbearbeiter Scanner Registerkarte Menü 

Auf dieser Registerkarte können Sie festlegen, welche Menüpunkte und Aufgaben für den Sachbearbeiter zugänglich sein sollen. 

Nur was hier aufgelistet ist, steht dem Sachbearbeiter später zur Bearbeitung zur Verfügung. 
### Button Neu 

Mit diesem Button können Sie eine neue Funktion anlegen. 

### Button Löschen 

Mit diesem Button können Sie einen Eintrag löschen. 

### Tabelle 

### Tabellenspalte Nr. 
Hier wird die Nummer des Menü-Punkts angezeigt, wenn Sie einen bereits angelegten Menü-Punkt in die Tabelle einlesen. 

Bei der Neuanlage eines Menü-Punktes wird diese Nummer automatisch vom System beim Speichern vergeben. 

### Tabellenspalte Menüpunkt 
In dieser Combobox wählen Sie den jeweiligen Menü-Eintrag des Sachbearbeiters aus: 

- Behälterbuchung - Ein- Auslagerung - Einlagerung - Endkontrolle - Information - Inventur - Kommissionieren - LHM - LHM Registrieren - LHM Umlagern - Man. Umlagerung - Platz aktivieren - Platz LHM fähig - seq. Wareneingang - Tourenplatzbuchung - Umlagerung - Verk. Wareneingang - Wareneingang 

### Button Nach oben verschieben 

Mit diesem Button können Sie einen Eintrag nach oben verschieben. 

Tooltip: 

Nach oben verschieben 

### Button Nach unten verschieben 

Mit diesem Button können Sie einen Eintrag nach unten verschieben. 

Tooltip: 

Nach unten verschieben
