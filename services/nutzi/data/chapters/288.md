# Lagerorte 
Lagerorte 

eNVenta ERP erlaubt die Verwaltung von bis zu 999 verschiedenen Lagern. 

Reservierungen und Bestellungen können lagerbezogen durchgeführt werden. 

Ein Artikel kann pro Lagerort nur einmal in die Liste aufgenommen werden. 

Mit diesem Programmteil, der über das Menü Lager aufgerufen wird, können Sie Lagerorte definieren. Nur die hier angegebenen Lager stehen in anderen Programmteilen zur Bearbeitung zur Verfügung. 

Um ein Lager neu anzulegen, leeren Sie zunächst mit dem Button die Datenfelder für eine Neueingabe. Beginnen Sie mit Ihren Einstellungen auf der Registerkarte Allgemein und vergeben Sie zuerst eine neue, eindeutige Lagernummer und eine Bezeichnung des Lagers. 

Falls gewünscht, kann noch die Anschrift des Lagers angegeben werden. Speichern Sie das neue Lager mit dem Button .

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Link-Lager Sync 

Mit diesem Button können Sie Lagerplätze einem Link-Lager zuordnen. In diesem Zusammenhang stehen die Checkbox Link-Lager und die Comboboxen Link-Lager 1-3 zur Verfügung. 

Siehe Kapitel Link-Lager . 

Tooltip: 

Link-Lager Synchronisation 

### Button Lager de/aktivieren 

Mit diesem Button können einzelne Lager deaktiviert bzw. wieder aktiviert werden. Dadurch wird die Checkbox Inaktiv aktiviert (im Fall von Lagerdeaktivierung und umgekehrt). Damit brauchen Lager, die nicht mehr zur Verfügung stehen, nicht physisch gelöscht werden. Dies kann aus historischen Zwecken sinnvoll sein. 

Wenn Sie deaktiviert sind, können die Lager nicht mehr bebucht werden. 

Tooltip: 

Mit diesem Button wird ein Lager deaktiviert bzw. wieder aktiviert.
