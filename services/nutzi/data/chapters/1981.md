# Material splitten 
Material splitten 

Der Button Splitten auf der Registerkarte Positionen im Betriebsauftrag öffnet das Menü Material splitten . 

Jede Materialposition wird durch eine Positionsnummer, eine Partienummer und eine Fixposnummer identifiziert. 

Die Fixposnummer ist eindeutig und unveränderbar. Für eine Position kann es aber mehrere Partien geben. Diese haben die gleiche Positionsnummer, aber unterschiedliche Partienummern. Die Partienummer wird innerhalb einer Position automatisch hochgezählt. 

Das Splitten von Materialpositionen ist unbedingt notwendig, wenn Sie mehrere Chargen für eine Position verwenden. Hier muss vor der Chargenzuordnung der Splitt erfolgen. 

Beim Buchungsvorgang wird folgendes durchgeführt: 

- Die neue Position wird als Kopie der alten erzeugt und die neue Teilmenge eingesetzt. Es werden eine neue Fixposnummer und eine neue Partienummer erzeugt. - Die Menge der alten Position wird um die Menge der neuen Position reduziert. 
Dabei tritt folgendes Problem auf: Beide Positionen sind anschließend unabhängig, d.h. beide werden parallel bei der Produktionsbuchung abgebucht. 

Damit dies möglich ist, müssen die Teilmengen der beiden Positionen in Beziehung zur Gesamtmenge des Betriebsauftrags gesetzt werden. Dies geschieht mit Hilfe der Felder Menge und Menge pro . Dabei entstehen Dezimalwerte. 

Die Gesamtmenge wird jedoch nach der im Artikel festgesetzten Einheit gerundet.
