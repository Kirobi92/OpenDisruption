# Artikelkalkulation Registerkarte Kopfdaten 
Artikelkalkulation Registerkarte Kopfdaten 

Auf dieser Registerkarte können Sie zur Artikelkalkulation individuelle Informationen eingeben, wie z.B. einen Text zur internen Information, den Zeitraum der Preisgültigkeit. 

Wenn Sie zu einer einzelnen Artikelposition in die Felder PG Von Datum / PG Bis Datum ein Datum eingeben, welches nicht im Zeitraum des zugeordneten Preisgruppen-Datums (Dialog Preisgruppen Details ) liegt, kann über den Button Input Neu kein Update erfolgen. 
### Feld Memo 
Hier können Sie einen individuellen Text eingeben, z.B. eine Erklärung der Kalkulation. 

Tooltip: 

Enthält interne Textinformationen. 

### Feld PG-Datum von 
Um einen Zeitraum für eine Preis-Gültigkeit zukünftiger Preise zu definieren, geben Sie hier das Start-Datum ein. Dieses Datum wird für die Suche und auch beim Schreiben des Ergebnisses beim Buchen genutzt. 

Siehe Kapitel Preisgruppen-Details . 

Tooltip: 

Wenn Sie zu einer einzelnen Artikelposition in dieses Feld ein Datum eingeben, welches nicht im Zeitraum des zugeordneten Preisgruppen-Datums (Dialog 'Preisgruppen Details') liegt, kann über den Button 'Input Neu' kein Update erfolgen. 

### Feld PG-Datum bis 
Um das Ende eines Preis-Gültigkeitszeitraum zu definieren, geben Sie hier das entsprechende Ende-Datum ein. Dieses Datum hat zum Datum im Feld PG von Datum zu passen. 

Dieses Datum wird für die Suche und auch beim Schreiben des Ergebnisses beim Buchen genutzt. 

Tooltip: 

Wenn Sie zu einer einzelnen Artikelposition in dieses Feld ein Datum eingeben, welches nicht im Zeitraum des zugeordneten Preisgruppen-Datums (Dialog 'Preisgruppen Details') liegt, kann über den Button 'Input Neu' kein Update erfolgen. 

### Checkbox Auslaufdatum setzen 
Wenn diese Checkbox gesetzt ist, dann soll das Auslaufdatum beim alten Preis auf das Von Datum vom neuen Preis minus 1 Tag, mindestens aber auf das Von-Datum der "alten Kondition" gesetzt werden (beide könnten das gleiche Startdatum haben). 

Siehe Kapitel Preisgruppen-Details .
