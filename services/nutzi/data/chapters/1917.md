# Rabatt Einkauf Registerkarte Tabelle 
Rabatt Einkauf Registerkarte Tabelle 

Auf dieser Registerkarte werden Ihnen sämtliche Rabatte, die bereits angelegt wurden, anhand der Suchkriterien angezeigt. 
### Tabelle 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte EAN Art. 
Hier wird die EAN-Nummer des Artikels angegeben. EAN bedeutet European Article Number und bezeichnet eine internationale, eindeutige Kennzeichnung für Produkte. Der EAN Code wird meist als maschinenlesbarer Strichcode mit Zahlen auf der Verpackung einer Ware außen gut sichtbar angebracht, um von Barscannern / Scannerkassen schnell eingelesen werden zu können. 

Tooltip: 

Die EAN-Nummer wird angegeben. Diese EAN-Codenummer dient als Suchfeld in der Auftrags- und Bestellerfassung. 

### Tabellenspalte EAN Lief.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
