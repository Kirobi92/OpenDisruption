# Kapitel NV.ERP.Base.Common.cdParameterFI.shtDiffMessageAutoOpen

Ist dieser Parameter aktiviert, öffnet sich der Dialog Differenzmeldung Detail automatisch, wenn in einem Ausgleich/einer Rückzahlung (Buchungskreis DA, DZ, KA, KZ), z.B. über Button OP eine OP Anlage erfolgte. Ansonsten muss der Anwender darauf achten, vor dem Abschließen der Buchung im Dialog OP – Ausbuchung den Button Diff.meld. auszuwählen. 

Siehe Kapitel Differenzmeldung Detail . 

Siehe Kapitel OP - Ausbuchung . 

Tooltip:
