# EDI Parameter 
EDI Parameter 

Dieser Dialog öffnet sich im Gateway >> Optionen >> EDI >> EDI Parameter . Hier werden die allgemeinen Parameter für den Import und Export verwaltet. Zusätzlich wird hier angezeigt, wann der letzte Import (Feld Letzter Import ) erfolgte. 

Wird beim Import einer ORDRSP eine Menge größer der Bestellmenge bestätigt, so bekommt der entsprechende Sachbearbeiter eine Wiedervorlage. Diese Wiedervorlage ist mit der Bestellung (als Referenz) verknüpft, damit diese direkt angesehen werden kann. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld EDI Parameter 
Tooltip: 

In diesem Dialog werden die allgemeinen Parameter für den Import und Export verwaltet. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
