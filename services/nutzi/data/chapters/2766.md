# Kostenstellen Registerkarte Allgemein 
Kostenstellen Registerkarte Allgemein 

Auf dieser Registerkarte nehmen Sie allgemeine Einstellungen für die Kostenstelle vor. 
### Button Konvertier. 

Mit diesem Button konvertieren Sie eine Kostenstelle zur Hilfskostenstelle. 

Diese Funktion ist beschrieben im Absatz Konvertieren einer Hilfskostenstelle in eine Kostenstelle. 

Tooltip: 

Konvertieren zur Hilfskostenstelle

### Combobox Bezugsgröße 1 
Bezugsgrößen sind Teiler, mit denen der Saldo der Kostenstellen geteilt wird. So können z.B. Maschinenstunden, Prozesskosten, Kennzahlen wie Kosten/Mitarbeiter usw. berechnet werden. 

Die Bezugsgrößen werden im Menü Stamm >> Umlagen angelegt und gepflegt. 

Siehe Kapitel Umlagen .

### Combobox Abteilung 
Die Abteilung des Ansprechpartners beim Kunden wird angezeigt. 

Tooltip: 

Angabe der Abteilung des Ansprechpartners beim Kunden. 

### Feld Zuständig 
Hier kann der für die Kostenstelle zuständige Abteilungsleiter o.ä. eingetragen werden. 

### Checkbox Nicht manuell bebuchbar 
Wird diese Checkbox gesetzt, steht diese Hauptkostenstelle in der Finanzbuchhaltung für eine manuelle Buchung nicht zur Verfügung. 

In der Kostenrechnung kann weiterhin manuell auf diese Hauptkostenstelle gebucht werden. 

Wird durch Setzen dieser Checkbox eine Kore-Schablone verändert, so erscheint eine Hinweismeldung.
