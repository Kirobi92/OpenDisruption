# Mandatsverwaltung 
Mandatsverwaltung 

Der Button Mandate im Kopf des Kundenstamms öffnet die Mandatsverwaltung. Hier wird die eindeutige Mandatsreferenz für das SEPA-Verfahren eingestellt. 

Die SEPA-Nummer setzt sich zusammen aus: 
- dem Länderkennzeichen (2-stellig), für Deutschland DE - der Prüfziffer (2-stellig) - der BLZ (8-stellig) - der Kontonummer (10-stellig) - fehlende Stellen werden von vorn mit 0 aufgefüllt! 
Die Kundennummer wird aus dem Kundenstamm übernommen. Anschließend wählen Sie die nötige Bankverbindung in der gleichnamigen Combobox aus. 

Zum Anlegen neuer Mandate drücken Sie den Button und geben die Daten in die Felder unterhalb der Tabelle ein. 

Abgelaufene und gelöschte Mandate werden in einer Historie gespeichert. 

Die Mandate können unterschriftsreif ausgedruckt werden, auch wenn noch keine Bankverbindung erfasst wurde. 

Das Datum der letzten Nutzung und das automatische Ablaufdatum werden angezeigt. 

Ausgedruckte Mandate können archiviert werden. 

Wird ein Mandat gelöscht, wird zuerst das Löschdatum gesetzt. Nur diese logisch gelöschten Mandate können in einem weiteren Löschvorgang endgültig gelöscht werden! 

Wird ein Kunde gelöscht, ist zwingend auch das Mandat zu löschen.
