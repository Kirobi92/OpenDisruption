# Aufgabe erzeugen 
Aufgabe erzeugen 

Dieser Dialog öffnet sich im Auftrag >> Optionen >> Generieren >> Projektaufgabe . Wenn einer Auftragsposition noch keine Projektaufgabe zugeordnet ist, öffnet sich dieser Dialog zum Erzeugen einer solchen Aufgabe. In diesem Dialog werden die Projektnummer, die Phase und die Basisaufgabe angegeben, um mit diesen Informationen eine entsprechende Projektaufgabe anzulegen. 

Ist der Auftragsposition bereits eine Aufgabe zugeordnet, öffnet sich der Dialog für die Bearbeitung dieser verknüpften Aufgabe. In diesem Aufgaben-Dialog ( Aufgabe ändern ) gibt es eine 4. Registerkarte Auftrag . Auf dieser Registerkarte können Sie die aktuelle Projektaufgabe auch manuell über den Button Auftragspos . mit einer Auftragsposition verknüpfen. Über den Button Löschen wird die Referenz zu einer Auftragsposition wieder zurückgesetzt. 

Wenn die Aufgabe nun beendet wird, wird die zugeordnete Auftragsposition automatisch auf Fakturafreigabe (Status 3) gesetzt. Dies passiert auch dann, wenn die zu dieser Projektaufgabe erfassten Leistungen abgerechnet werden. Die erfassten Leistungen werden dann nicht in einen neuen Auftrag eingefügt, sondern erhalten direkt den Status "Fakturiert" und lediglich die verknüpfte Position wird im vorhandenen Auftrag auf Fakturafreigabe gesetzt. So wird verhindert, dass Leistungen doppelt abgerechnet werden.

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
