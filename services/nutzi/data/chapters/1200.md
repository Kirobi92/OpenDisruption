# EDI Auftrag / Auftragsänderung Registerkarte EDI Protokoll 
EDI Auftrag / Auftragsänderung Registerkarte EDI Protokoll 

Hier werden Information des EDI Protokolls angezeigt. 
### Feld Anwendungsname 
Hier wird der Name der Anwendung angezeigt, z.B. cAdapterEDI. 

Tooltip: 

Der Name des Adapter-Typs über den die Verarbeitung durchgeführt wurde 

### Feld Modulname 
Hier wird der Modulname angezeigt, z.B. EDI Auftragsimport. 

Tooltip: 

Zeigt an welches Modul des Adapters ausgeführt wurde. 
Beim EDI werden die einzelnen Nachrichtentypen als Modul aufgeführt. 
Beim AdapterGeneralIO die Export-/Importkonfigurationen 

### Feld Status 
Hier wird der Status der Transaktion angezeigt. 

### Feld Transaktionsnr 
Hier wird die Nummer der Transaktion angezeigt. 

### Feld Anfangszeit 
Hier werden das Datum und die Anfangszeit der Transaktion angegeben. 

Tooltip: 

Zeitpunkt zudem die Verarbeitung des Ereignisses gestartet wurde 

### Feld End-Zeit 
Hier werden das Datum und die End-Zeit der Transaktion angegeben. 

### Feld Fehlermeldung 
In diesem Feld werden sämtliche Fehler der Datei angezeigt.
