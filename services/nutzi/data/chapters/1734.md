# Dispo-Pool 
Dispo-Pool 

Dieser Dialog öffnet sich im Dialog Disposition >> Vorgänge >> Dispo-Pool . 

Artikelpositionen, die in der Disposition erfasst werden, können in den Dispo-Pool gestellt werden, statt sofort in einer Bestellung generiert zu werden. 

Der Dispo-Pool entspricht einem Sammelkorb. Sie sammeln hier über einen von Ihnen vorgegebenen Zeitraum Artikelpositionen. Auf diese Weise ist es nicht notwendig, wegen jedes Artikels oder zu jedem Lieferanten sofort einzelne Bestellungen zu generieren. Stattdessen können Sie zu einem festen Termin, z.B. jeden Abend oder 1 bis 2 Mal die Woche, eine Gesamtbestellung erfassen oder die einzelnen Dispositionen noch mal prüfen. 

Hinweis: Sie können eine einzelne Bestellung pro Kunde generieren (über den Button Bestellen ) oder eine Bestellung für mehrere Kunden beim gleichen Lieferanten (über den Button ). 

In die Spalte ‚Info‘ tragen Sie einen Buchstaben ein, der eine Information zum Artikel definiert. Die zur Verfügung stehenden Informationskennzeichen werden im Kapitel Artikel-Kurzinformationen erklärt. 

Siehe Kapitel Artikel-Kurzinformationen . 

Im Dispo-Pool können Sie sich Positionen anzeigen lassen, sortiert nach Lieferanten- oder Artikelnummer. Dies kann auch noch zeitlich eingegrenzt werden. 

Damit können Sie z.B. Ihre Lagerhaltung optimieren oder bei höherer Bestellmenge evtl. noch einen Rabatt bei einem Lieferanten erzielen. 

Bei der Erzeugung einer Bestellung aus dem Dispo-Pool heraus werden das Lieferdatum und die KW der Bestellpositionen auf das aktuelle Datum gesetzt. Das Lieferdatum im Bestellkopf wird ebenfalls auf das Tagesdatum aktualisiert. 

Wenn Sie in der Disposition einen Positionstext geändert haben, wird er in die Bestellung übergeben.
