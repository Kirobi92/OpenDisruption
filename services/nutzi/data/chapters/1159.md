# Gruppentitel Registerkarte Bild 
Gruppentitel Registerkarte Bild 

Diese Registerkarte bietet die Möglichkeit dem Gruppentitel eine Bilddatei hinzuzufügen. 
### Abbildung Bild 
Hier kann eine Bilddatei eingefügt werden. 

### Button Suche 

Mit diesem Button können Sie nach Bilddateien suchen. 

### Button Löschen 

Dieser Button löscht die Bilddatei.
