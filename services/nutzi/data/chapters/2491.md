# Jahresabschluss Registerkarte Vortrag

Finanzbuchhaltung / Jahresabschluss 

# Jahresabschluss Registerkarte Vortrag 
Jahresabschluss Registerkarte Vortrag 

Auf dieser Registerkarte geben Sie an, welche Konten abgeschlossen werden sollen. 
### Button Vortrag 

Mit diesem Button übertragen Sie die Konten in das neu angelegte Buchungsjahr. 

### Checkbox Sachkonten 
Wenn Sie diese Checkbox aktivieren, kann ein Jahresabschluss nur für diesen Bereich vorgenommen werden. 

### Checkbox Debitoren 
Wenn Sie diese Checkbox aktivieren, kann ein Jahresabschluss nur für diesen Bereich vorgenommen werden. 

### Checkbox Kreditoren 
Wenn Sie diese Checkbox aktivieren, kann ein Jahresabschluss nur für diesen Bereich vorgenommen werden.
