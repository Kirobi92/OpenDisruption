# Sachbearbeiterstamm Registerkarte Stellvertreter 
Sachbearbeiter Registerkarte Stellvertreter 

Diese Registerkarte bietet die Möglichkeit Stellvertreter anzulegen und zu verwalten. So werden Vorgänge eines Sachbearbeiters bei dessen Abwesenheit von einem zugeordneten Kollegen bearbeitet. 

Der berechtigte Stellvertreter erhält z.B. Wiedervorlagen, falls der zuständige Sachbearbeiter verhindert ist (z.B. Krankheit, Urlaub). Ob ein Sachbearbeiter zur Verfügung steht oder nicht wird aus der Zeiterfassung ermittelt. 

Für die Vertretung wird in der Spalte ‚Priorität‘ die Priorität der Vertretung festgelegt. D.h. der Mitarbeiter mit Priorität 1 bekommt die Wiedervorlagen zuerst, dann der Mitarbeiter mit Priorität 2 usw. 
### Button Neu 

Mit diesem Button können Sie der Tabelle einen weiteren Stellvertreter zuordnen. 

### Button Löschen 

Mit diesem Button können Sie einen Stellvertreter aus der Tabelle löschen.
