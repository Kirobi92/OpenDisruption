# Sachkontenstamm Registerkarte Kostenrechnung 
Sachkonto Registerkarte Kostenrechnung 

Über diese Registerkarte steuern Sie, ob das Konto für die Kostenrechnung relevant ist. 

Sie können einem kostenrechnungsrelevanten Sachkonto eine oder beliebig viele Kostenstellen und Kostenträger in der Tabelle Kostenstellen zuordnen. 

Diese Kostenstellen bzw. Kostenträger werden beim Buchen im Dialog Verteilung automatisch vorgeschlagen. 

Über den Button nehmen Sie die Zuordnung der Kostenstelle(n), Kostenträger sowie die Verteilung (in %) auf die Kostenstelle(n) / Kostenträger vor. 

Diese Kostenstellen werden beim Buchen im Dialog Verteilung automatisch vorgeschlagen. 

In der Spalte ‚Prozent’ wird die prozentuale Verteilung pro Kostenstelle(n) eingetragen. 

Die Summe daraus muss nicht 100 ergeben, darf jedoch den Wert 100 nicht überschreiten. 

Durch markieren einer Zeile und betätigen des Buttons Lösche Z. kann eine Zuordnung wieder aufgehoben werden.

### Checkbox Kostenrechnung 
Soll ein Sachkonto in der Kostenrechnung zugeordnet werden, aktivieren Sie die Checkbox Kostenrechnung . In diesem Fall ist auch zwingend eine Kostenart einzutragen. 

Tooltip: 

Kostenrechnung zum Konto 

### Checkbox KSt-Zwang 
Diese Checkbox lässt eine individuelle Einstellung pro Sachkonto zu. 

Wenn diese Checkbox oder die Checkbox KTr-Zwang aktiviert ist, so ist zwingend eine/mehrere Kostenstelle(n) bzw. Kostenträger oder beides bei der Buchung einzugeben, ansonsten kann diese nicht abgeschlossen werden. 

Tooltip: 

Kostenstellenzwang 

### Checkbox KTr-Zwang 
Diese Checkbox lässt eine individuelle Einstellung pro Sachkonto zu. 

Wenn diese Checkbox oder die Checkbox KSt-Zwang aktiviert ist, so ist zwingend eine/mehrere Kostenstelle(n) bzw. Kostenträger oder beides bei der Buchung einzugeben, ansonsten kann diese nicht abgeschlossen werden. 

Tooltip: 

Kostenträgerzwang 

### Feld Kostenart 
Aus dieser Combobox wählen Sie die gewünschte Kostenart für die Kostenrechnung aus. Vom Programm wird automatisch geprüft, ob die Kostenart im Kostenartenstamm der Kostenrechnung angelegt ist. Ist dies nicht der Fall, muss die gewünschte Kostenart zuerst im Kostenartenstamm der Kostenrechnung angelegt werden, bevor eine Zuordnung in der Finanzbuchhaltung möglich ist. 

Diese Kostenarten müssen nicht den Fibu-Konten entsprechen. 

Bei der automatischen Übernahme der Fibu-Konten als Kostenarten in die Kostenrechnung werden die entsprechenden Fibu-Konten automatisch mit der Checkbox Kostenrechnung und der entsprechenden Kostenart belegt.
