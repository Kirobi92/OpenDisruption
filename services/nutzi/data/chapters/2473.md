# OP-Aufstellung 
OP-Aufstellung 

Dieser Dialog öffnet sich im Buchen >> OP >> OP-Aufstellung . Hier haben Sie die Möglichkeit sogenannte Monatsaufsteller / Kontoauszüge über einen separaten Dialog zu erstellen. Vorschläge können erstellt und gespeichert werden. Die Verarbeitung erfolgt wie in den Dialogen Mahnwesen, Bankeinzug und Zahlungsverkehr . 

Die erzeugte Aufstellung kann per Mail, Fax oder Brief versendet werden. 

In der OP-Aufstellung werden generell nur offene Posten ohne Zahldatum selektiert. Sie können u.a. nach Debitoren oder Kreditoren selektieren und Sie können wählen, ob Sie einen Kontoauszug oder Monatsaufsteller generieren wollen. 

Monatsaufsteller gibt es nur für Debitoren. 

Wenn Sie Monatsaufsteller wählen, ist zwingend auch ein Monat aus der Combobox daneben zu wählen. Es werden nur OPs ausgewählt, deren Skontofälligkeit oder Nettofälligkeit im gewählten Monat liegen oder überfällig sind (Rechnungen, Gutschriften, Abschlagszahlung (AZ), Teilzahlung (TZ)). 

Bitte beachten: Wenn Die Checkbox Monatsaufsteller aktiviert ist und der Saldo eines Kunden zur gewählten Periode ein Guthaben ergibt, werden die Sätze nicht eingelesen. Ein Monatsaufsteller soll nur über offene Rechnungen informieren. 

Bedingungen für das Einlesen der OP’s für Monatsaufsteller/Kontoauszug Debitoren: 

Wird die Checkbox Monatsaufsteller vor der Suche aktiviert, werden nur OP’s von Debitoren eingelesen, bei denen im Debitorenstamm ebenfalls die Checkbox Monatsaufsteller aktiviert ist. 

Wird die Checkbox Kontoauszug vor der Suche aktiviert, werden nur OP’s von Debitoren eingelesen, bei denen im Debitorenstamm ebenfalls die Checkbox Kontoauszug aktiviert ist. 

Eine Suche ist auch ohne Auswahl Monatsaufsteller bzw. Kontoauszug möglich. So kann nach der Suche entschieden werden, ob ein Monatsaufsteller bzw. Kontoauszug für die ausgewählten Daten erzeugt werden soll. 

Bedingungen für das Einlesen der OP’s für Kontoauszug Kreditoren: 

Wird die Checkbox Kontoauszug vor der Suche aktiviert, werden nur OP’s von Kreditoren eingelesen, bei denen im Kreditorenstamm ebenfalls die Checkbox Kontoauszug aktiviert ist. 

Eine Suche ist auch ohne Auswahl Kontoauszug möglich. So kann nach der Suche entschieden werden, ob ein Kontoauszug für die ausgewählten Daten erzeugt werden soll. 

Der Button Löschen löscht die markierten Debitoren / Kreditoren OPs aus dem Vorschlag. Der Button Weitere OP ermöglicht die Aufnahme weiterer OP’s zu einem Kunden, die im Vorschlag nicht selektiert sind. 

Wenn im Kundenstamm (Registerkarte Rechnungswesen Erweitert ) beim Monatsaufsteller die Checkbox Gutschriften sof. fällig aktiviert ist, werden alle Gutschriften, AZ, TZ unabhängig ihrer Fälligkeit vor Fälligkeit eingelesen.
