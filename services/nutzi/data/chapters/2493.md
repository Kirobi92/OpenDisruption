# Buchungskreise 
Buchungskreise 

Dieser Dialog öffnet sich im Buchen >> Stamm >> Buchungskreise . 

Die vorhandenen Buchungskreise werden als Standard automatisch eingetragen. Dies sind: 

- DA = 1 - Debitoren-Ausgleich - DG = 2 - Debitoren-Gutschriften - DR = 3 - Debitoren-Rechnungen - DZ = 4 - Debitoren-Rückzahlungen - FB = 5 - Finanzbuchhaltung - KA = 6 - Kreditoren-Ausgleich - KG = 7 - Kreditoren-Gutschriften - KR = 8 - Kreditoren-Rechnungen - KZ = 9 - Kreditoren Rückzahlung - z1 = B - Bankauszug - ZB = Z - Zwischenabschluss-Buchungen 
Es können vom Benutzer weder neue Buchungskreise hinzugefügt, noch bestehende gelöscht werden, die Spalte 'Name' ist unzugänglich. 

Nur die Spalten 'Format' und 'Nummer' sind für Einträge zugänglich. Hier können für die einzelnen Buchungskreise Nummernkreise festgelegt werden. In der Spalte 'Format' wird das Zahlenformat (z.B. TTMMNN), in der Spalte 'Nummer' der Startwert des zugehörigen Buchungskreises eingetragen. Sie können die Werte eintragen bzw. ändern, indem Sie mit der Maus direkt in das entsprechende Feld klicken. Ordnen Sie einem Buchungskreis einen Nummernkreis zu, so wird dieser automatisch beim Aufruf des Buchungskreises im Buchen im Feld Belegnummer eingestellt und bei jeder Buchung wird eine fortlaufende Belegnummer generiert. 

Das Feld Belegnummer im Dialog Buchen kann nach Festlegen eines Nummernkreises nicht mehr manuell bearbeitet werden. 

Nummernkreise können nur einem Buchungskreis zugeordnet werden. Eine Zuordnung auf Benutzerebene ist nicht möglich.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
