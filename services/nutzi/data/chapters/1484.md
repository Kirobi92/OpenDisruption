# Gutscheinauswahl 
Gutscheinauswahl 

Dieser Dialog öffnet sich über den Button Gutschein im Kopf des Dialogs Kassieren . Sie können hier einen Gutschein auswählen, der beim Kassieren berücksichtigt werden soll oder einen neuen Gutschein erzeugen. 

In der Kasse kann ein Gutschein nur erzeugt werden, wenn in den Kasseneinstellungen die Checkbox Gutscheine an der Kasse erstellen gesetzt ist. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Gutscheinauswahl 
Tooltip: 

In diesem Dialog können Sie den Gutschein auswählen, der beim Bezahlen in Abzug gebracht werden soll oder erzeugen einen neuen Gutschein. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Auswahl 

Mit diesem Button übernehmen Sie einen Beleg in den zugrundeliegenden Vorgang. 

Tooltip: 

Speichern und übernehmen 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Feld Gutschein 
Wird eine existierende Gutscheinnummer eingegeben, werden alle Daten des Gutscheins automatisch eingelesen. 

Ist die Gutscheinerstellung in der Kasse möglich, kann die neue Nummer des Gutscheins eingetragen werden. 

Tooltip: 

Die eindeutige Bezeichnung des Gutscheins wird angegeben. 

### Combobox Gutscheinart 
Hier wählen Sie die Art des Gutscheins aus. Die Auswahl zeigt alle Zahlungsarten mit dem Parameter Gutschein an. Die Einstellung kann verwendet werden, um Gutscheine z.B. für bestimmte Aktionen zu erstellen. 

Diese Gutscheinart wird im Dialog Zahlungsart definiert. 

Siehe Kapitel Zahlungsart . 

Tooltip: 

Angabe der Gutscheinart 

### Feld Initialwert 
Geben Sie den Wert des Gutscheins ein. 

Bei Angabe eines existierenden Gutscheins wird der Initialwert des Gutscheins automatisch eingelesen. 

Tooltip: 

Wert des Gutscheins bei Erstellung 

### Combobox Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Die Systemwährung (Euro für die deutsche Version) MUSS in der Tabelle unter der Währungsnummer 1 und dem Kurs 1,0 angegeben werden! 

Tooltip: 

Geben Sie eine eindeutige, fortlaufende Nummer ein. 

### Checkbox Aktivieren 
Wenn Sie diese Checkbox anklicken, wird der Gutschein aktiviert, d.h. er kann genutzt werden. 

### Feld Status 
Anzeige des Status des Gutscheins. Folgende Status sind möglich: 

- Nicht aktiviert - Aktiviert - Benutzt - Geschlossen 

### Feld Restwert 
Anzeige des Restwerts des angegebenen Gutscheins. 

Tooltip: 

Wert des Gutscheins nach Verwendung
