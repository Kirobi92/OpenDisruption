# AG-Ressourcen 
AG - Ressourcen 

Über den Button Ressourcen im Arbeitsplan auf der Registerkarte Allgemein öffnet sich der Dialog AG – Ressourcen . Hier können einem Arbeitsgang eine oder mehrere Ressourcen zugeordnet werden. Diese Zuordnungen werden automatisch auf den Arbeitsplan für einen Betriebsauftrag als „Vorlage“ übernommen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Ressourcen 
Tooltip: 

In diesem Dialog können einem Arbeitsgang eine oder mehrere Ressourcen zugeordnet werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

Die Angaben in der Tabelle werden aus der Ressourcenauswahl übernommen und sind nicht änderbar.

Die Bezeichnung / der Name der Ressource wird hier angezeigt. 

Tooltip: 

Angabe der Ressourcengruppen-Bezeichnung 

### Tabellenspalte Anzahl 
Es werden x Ressourcen aus der gewählten Ressourcengruppe in den Arbeitsgang übernommen. Dies ist eine feste Zahl und zieht nur bei Auswahl einer Ressourcengruppe. 

### Tabellenspalte Splitt 
Der Arbeitsgang kann auf mehrere Zeitfenster aufgeteilt werden, um z.B. die Zeit bis zum Feierabend zu nutzen und nicht erst am nächsten Vormittag mit der Arbeit zu beginnen. 

### Tabellenspalte Ressource 
Die Nummer der Ressource wird eingetragen. 

Tooltip: 

Angabe der Ressource 

### Tabellenspalte Bezeichnung 
Der Name der Ressource wird eingetragen. 

Tooltip: 

Name der Ressource 

### Tabellenspalte Fix 
Bei Auswahl einer Ressourcengruppe UND einer Ressource wird diese als fixiert markierte Ressource immer in den Arbeitsgang übernommen und notfalls eine weitere Ressource automatisch hinzugefügt. Z.B. müssen immer 1 Meister/Vorarbeiter + ein beliebiger Hilfsarbeiter vorhanden sein. Die zusätzliche Ressource wird vom System gewählt. 

### Tabellenspalte Anteil% 
Wenn ein Arbeitsgang mehr als eine Ressource hat, kann über diese Spalte der Anteil dieser Ressourcen am Arbeitsgang definiert werden. So können die Kosten exakt verteilt werden.
