# Kapitel NV.ERP.MM.Rental.cdRentalParam.shtFODiffers

Wenn Sie diese Checkbox aktivieren, wird bei Folgeaufträgen der jeweils andere Berechnungszeitpunkt gewählt. Z.B. kann bei Erstmiete über den erwarteten Mietzeitraum eine Erstrechnung per Vorfakturierung erfolgen. Alle weiteren werden dann per Nachfakturierung berechnet. 

Dies dient der Vorbelegung bei der Neuanlage von Mietobjekten und kann je Mietobjekt geändert werden. 

Beispiel: 
Die Erstrechnung kann per Vorfakturierung erfolgen, die Folgerechnungen per Nachfakturierung.
