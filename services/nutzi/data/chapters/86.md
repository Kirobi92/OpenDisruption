# Firmennachrichten 
Firmennachrichten 

Dieser Dialog öffnet sich in eNVenta Zoom >> Optionen >> Unternehmensnachrichten verwalten . Dieser Dialog steht nur autorisierten Personen zur Verfügung und bietet die Möglichkeit Firmennachrichten zu verwalten. 

Über das Von Datum wird gesteuert, dass eine Nachricht nur angezeigt wird, wenn das Tagesdatum dieses Datum erreicht hat. Zusätzlich muss die Nachricht den Status „Aktiv“ haben. 

Mit dem Status deaktiviert können die Nachrichten wieder für die Benutzer entfernt werden. 

Mit dem Status in Vorbereitung werden Nachrichten gesteuert, welche noch nicht fertig sind. Diese werden auch nicht angezeigt, wenn das „von Datum“ erreicht ist.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Über diesen Button wird eine neue Nachricht angelegt, welche dann später allen Mitarbeitern angezeigt wird, welche das entsprechende Zoom-Element nutzen. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
