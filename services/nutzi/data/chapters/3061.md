# Artikel Schattenstamm Registerkarte Merkmalsklassen 
Registerkarte Merkmale 

Auf dieser Registerkarte werden Merkmale angezeigt. 

Im Import der Merkmale wird erst versucht, ein passendes, bereits existierendes Merkmal zu finden, bevor ein neues Merkmal geschaffen wird. Die Ausprägung wird dann mit dem alten bzw. neuen Merkmal verknüpft und später dem Artikel bereitgestellt. 
### Tabelle 

### Tabellenspalte Haupt-Importschlüssel 
Hier wird der Haupt-Importschlüssel angegeben.

Hier wird die Merkmalsklasse angegeben.
