# CRM Übersicht Registerkarte Besuchsberichte 
CRM Übersicht Registerkarte Besuchsbericht 

Auf dieser Registerkarte werden je nach Selektion die entsprechenden Besuchsberichte angezeigt. 
### Feld Von Besuchstag 
Hier wird das Von-Datum des Besuchstags zur Selektion angegeben. 

Tooltip: 

Angabe des Besuchstags 

### Feld Bis Besuchstag 
Hier kann das Bis-Datum des Besuchstags zur Selektion angegeben werden. 

Tooltip: 

Angabe des Besuchstags 

### Combobox Besuchsart 
Hier wird die Art des Besuchs ausgewählt. Z.B. Erstbesuch, Folgebesuch. 

Tooltip: 

Angabe der Besuchsart 

### Combobox Status 
Hier wird der Status der Position angegeben. 

1-offen 
2-offen, zur Lieferung freigegeben 
3-offen, zur Faktura freigegeben 
4-fakturiert 
5-geliefert (ohne Berechnung) 
6-storniert 

Tooltip: 

Angabe des Positionsstatus 

### Checkbox Drucken 
Wenn Sie diese Checkbox aktivieren, werden die Informationen gedruckt. 

### Tabelle 

### Tabellenspalte AKL-Nummer 
Hier wird eine Adress-/Kunden- oder Lieferantennummer angegeben. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Tabellenspalte AKL-Typ 
Hier wird der AKL-Typ angegeben. 

Zur Verfügung stehen: 
A = Adresse 
K = Kunde 
L = Lieferant 
V = Vertreter 
P = Kontakt 

Tooltip: 

AKL-Typ: 
A = Adresse 
K = Kunde 
L = Lieferant 
V = Vertreter 
P = Kontakt 

### Tabellenspalte Besuch 
Hier wird die Nummer des Besuchs angegeben. 

Tooltip: 

Angabe der Besuchsnummer 

### Tabellenspalte Ziel/Thema 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen. 

### Tabellenspalte Besuchsdatum 
Hier wird das Datum des Besuchstags angegeben. 

Tooltip: 

Angabe des Besuchstags 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Besuchsart 
Hier wird die Art des Besuchs angegeben. 

Tooltip: 

Angabe der Besuchsart 

### Tabellenspalte Vertreter 
Hier wird die Zeile 1 des Firmennamens angezeigt, nachdem ein Vertreter ausgewählt wurde. 

Tooltip: 

Zeile 1 des Firmennamens.
