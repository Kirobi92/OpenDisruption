# Kapitel NV.ERP.Base.Common.cdDeliveryCondition.shtFranco

Wenn dieser Parameter gesetzt ist, erfolgt die Lieferung franco. Eventuell im Auftrag angegebene Porto- und Verpackungskosten werden nicht auf den Warenwert aufgeschlagen, sondern sind im Warenwert enthalten. Das Franco-Kennzeichen wird in Angebot und Auftrag übernommen und kann dort geändert werden. 

Tooltip:
