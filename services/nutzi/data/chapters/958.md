# Projektübersicht 
Projektübersicht 

Die Projektübersicht bietet die Möglichkeit Projekte in einer Übersicht anzuzeigen. Wenn Sie im Kopf des Dialogs eine Selektion vornehmen, wird Ihnen das gesuchte Projekt in die Tabelle eingelesen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Projektübersicht 
Tooltip: 

In diesem Dialog werden Projekte in einer Übersicht angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Drucken 

Mit diesem Button starten Sie den Druck, um die wichtigsten Projektinformationen als Liste (in Querformat) zu drucken. Die Selektionskriterien werden übergeben. 

### Button Suche 

Wenn Sie bei leerem Dialog diesen Button drücken, werden in der Tabelle auf der Registerkarte Allgemein alle bereits angelegten Projekte aufgelistet. 

### Button Neu 

Mit diesem Button leeren Sie die Felder. 

### Button Auswahl 

Mit einem Doppelklick in eine Zeile oder mit diesem Button gelangen Sie in den Dialog Projekt . Die Daten zum ausgewählten Projekt werden automatisch vom System eingelesen. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
