# Planung Betriebsaufträge Registerkarte Materialbedarf 
Planung Betriebsaufträge Registerkarte Materialbedarf 

Auf dieser Registerkarte wird der Materialbedarf angezeigt. Zunächst markieren Sie auf der Registerkarte Allgemein einen oder mehrere Betriebsaufträge und dann drücken Sie den Button Berechnen . Der kumulierte Materialbedarf (Spalte ‚Bedarf‘) des markierten Betriebsauftrags wird eingetragen. Zusätzlich wird für jeden Artikel eine reguläre Bestellvorschlagsrechnung durchgeführt. Führt dies zu einem Beschaffungsvorschlag, wird dieser in die Spalte ‚Dispomenge‘ eingetragen. Sonst bleibt diese leer. 

So können Sie den Materialbedarf ermitteln und eine übergreifende Beschaffung auslösen. 
### Button Berechnen 

Wenn Sie diesen Button drücken, wird der kumulierte Materialbedarf (Spalte ‚Bedarf‘) des markierten Betriebsauftrags eingetragen. 

Tooltip: 

Hier wird der kumulierten Materialbedarf berechnet. 

### Button Bestand 

Mit diesem Button erhalten Sie einen Überblick über den Bestand des Artikels. 

Siehe Kapitel Bestandsanzeige . 

Tooltip: 

Öffnet die Bestandsübersicht. 

### Button Disponieren 

Mit diesem Button wird für jede Zeile (die einen Eintrag in der Spalte ‚Disponiert‘ hat) ein Eintrag in den Dispopool geschrieben und die Menge aus der Spalte ‚Disponiert‘ übergeben. 

Tooltip: 

Erzeugt einen Eintrag im Dispopool 

### Button Produktion 

Über diesen Button wird für den ersten markierten Artikel ein Dialog zur Generierung des Produktionsauftrags aufgerufen. Das ist nur möglich, wenn es sich um eine Produktionsstückliste handelt. Es werden ein neutraler Produktionsauftrag und ein Betriebsauftrag für die oberste Ebene generiert. Es erfolgt keine automatische Terminierung. 

Tooltip: 

Erstellt einen Produktionsauftrag

### Tabellenspalte Meldebestand 
Hier wird eine Menge für den Meldebestand angegeben. Dieser Meldebestand ist wichtig für die Lagerdisposition. Wird der Meldebestand unterschritten, entsteht je nach Dispositionsverfahren ein Bestellvorschlag. 

Wird der Meldebestand versehentlich kleiner angegeben als der Mindestbestand, so erscheint eine Meldung. 

Tooltip: 

Wird die hier angegebene Menge erreicht, wird eine Meldung ausgegeben.
