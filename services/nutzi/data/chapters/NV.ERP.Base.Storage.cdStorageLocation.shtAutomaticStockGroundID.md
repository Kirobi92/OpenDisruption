# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtAutomaticStockGroundID

Wenn Sie diesen Parameter aktivieren, wird beim Erstellen eines Lagerplatzes die Platznummer automatisch aus Ebene, Gang, Haus, x- und y-Koordinate ermittelt.
