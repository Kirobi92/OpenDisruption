# Mietobjekt Registerkarte Vermietung 
Mietobjekt Registerkarte Vermietung 

Diese Registerkarte zeigt in einer Übersicht, welche Vermietungsvorgänge zum selektierten Mietobjekt bestehen. Eine neu angelegte Vermietung wird umgehend in diese Tabelle eingelesen. 
### Button Suche 

Mit diesem Button suchen Sie Vermietungen zum Mietobjekt. 

### Tabelle 

### Tabellenspalte Vermietung 
Hier wird die Nummer der Vermietung angegeben. 

### Tabellenspalte Status 
Hier wird der Status der Position angegeben. 

1-offen 
2-offen, zur Lieferung freigegeben 
3-offen, zur Faktura freigegeben 
4-fakturiert 
5-geliefert (ohne Berechnung) 
6-storniert 

Tooltip: 

Angabe des Positionsstatus 

### Tabellenspalte Posstatus 
Hier wird der Status der Position angegeben. 

1-offen 
2-offen, zur Lieferung freigegeben 
3-offen, zur Faktura freigegeben 
4-fakturiert 
5-geliefert (ohne Berechnung) 
6-storniert 

Tooltip: 

Angabe des Positionsstatus 

### Tabellenspalte Pos 
Hier wird die Positionsnummer angegeben. Diese Nummer muss je Vorgang eindeutig sein! 

Tooltip: 

Positionsnummer 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Firma 1 
Hier wird die Zeile 1 des Firmennamens angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Tabellenspalte Abrechnung von 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Tabellenspalte Abrechnung bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Tabellenspalte Vermietet bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein.
