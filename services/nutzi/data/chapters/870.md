# Export der Kundenpreise Registerkarte Allgemein 
Export Kundenpreise Registerkarte Allgemein 

Auf dieser Registerkarte grenzen Sie die Informationen für den Export ein. Dazu stehen die Felder und Checkboxen zur Verfügung. Dabei können Sie auch festlegen, ob z.B. Bruttopreise oder Sonderpreise verwendet werden sollen. Die Tabelle enthält dann die gewählten Einstellungen zur Information. Dabei ist aber noch kein Export erfolgt! Den Export starten Sie über den gleichnamigen Button, wenn die Einstellungen zuvor gespeichert wurden. Nach dem Import erhalten Sie eine Meldung, ob der Export erfolgt ist.

### Combobox Preisgruppe 
Hier können Sie eine Brutto-Preisgruppe auswählen, die im zugrundeliegenden Vorgang herangezogen wird und bestimmt, auf welchen Verkaufspreis zugegriffen werden soll. Die Preisgruppe, die Sie hier auswählen, übersteuert die Standardpreisgruppe des Kunden. 

Ansonsten läuft die normale Preislogik durch. D.h. sobald es eine höher priorisierte Kondition gibt (Preispyramide) wird diese exportiert. 

Wenn bei aktivierter Bestpreisfindung auf eine Preisgruppe zugegriffen wird, wird logischerweise die beste Kondition exportiert, anstatt der beim Export ausgewählten Preisgruppe.

### Liste Artikelstatus 
Für einen differenzierten Export von Kundenpreisen kann hier über den Artikelstatus selektiert werden. So werden nur die Artikel exportiert, die dem gewählten Status entsprechen. Damit kann der Export effizienter und schneller erfolgen indem z.B. keine Preise für gesperrte oder ausgelaufene Artikel exportiert werden.

### Tabellenspalte VK 
Hier wird ein VK (Verkaufspreis) angegeben, den Sie als Unternehmen definiert haben. 

eNVenta kann im Artikelstamm 3 Preisgruppen, also 3 verschiedene Verkaufspreise besitzen. Zusätzlich können weitere Preisgruppen im Dialog Preisgruppen Details angelegt werden. 

Für die Ermittlung des Verkaufspreises geht jedes Unternehmen anders vor; je nach Kalkulation von Gewinnmarge, Aktionen, Handlingspauschalen usw. 

Für die Ermittlung des VKs, basierend auf dem EK + %-Aufschläge, kann die Artikelkalkulation genutzt werden. 

Wir empfehlen für die Anlage von VKs, die Preisgruppen-Details zu nutzen. Dies bietet folgende Vorteile: 

- Der Preis kann mit einem Gültigkeitszeitram versehen werden. - Es können Mengenstaffeln angelegt werden. - Die Nachverfolgbarkeit von Preisänderungen ist gewährleistet. - etc. 
Siehe Kapitel Artikelkalkulation . 

Siehe Kapitel Preisgruppen und Preisgruppen Details .

### Tabellenspalte VK pro 
Hier wird eine Preiseinheit angegeben. Dieser Preis kann sich auf 10, 100, 1000 oder eine andere Anzahl Artikeleinheiten beziehen. Standard ist der Wert 1. 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- 1 Einheit - je 10 Einheiten - je 100 Einheiten
