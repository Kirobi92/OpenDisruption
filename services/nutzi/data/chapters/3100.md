# Datenimport bearbeiten Registerkarte Verweise 
Import Bearbeiten Registerkarte Verweise 

Auf dieser Registerkarte werden die Verweise eingelesen. Die eingelesenen Daten können nach unterschiedlichen Kriterien gefiltert werden. 
### Button Bearbeiten 

Mit diesem Button können die Verweise bearbeitet werden. Es öffnet sich der Dialog Import Editor - Verweise . 

Tooltip: 

Artikelverweise bearbeiten 

### Button Verweise Verarbeiten 

Mit diesem Button werden ausgewählte Verweise in den Schattenstamm übernommen. 

Tooltip: 

Ausgewählte Verweise in den Schattenstamm übernehmen

### Tabellenspalte Importschlüssel 
Hier wird der Importschlüssel angegeben. 

Tooltip: 

Der "Import-Schlüssel" ist ein generierter Wert, der als eindeutige ID (Referenzschlüssel) zur Referenzierung des Datensatzes dient. Er wird aus den Haupteigenschaften des Datensatzes zusammengesetzt. Jeder Datensatz benötigt einen eigenen, eindeutigen Import-Schlüssel. Es wird kein neuer Datensatz angelegt, wenn es bereits einen Datensatz mit demselben Import-Schlüssel gibt, sondern der bereits vorhandene überschrieben/aktualisiert.
