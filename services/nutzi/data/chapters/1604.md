# Lagerzone Registerkarte Skills 
Lagerzone Registerkarte Skills 

Um in bestimmten Lagerbereichen arbeiten zu können, bzw. zu dürfen, benötigt ein Sachbearbeiter bestimmte Fähigkeiten und Berechtigungen (sogenannte Skills). Dazu gehört z.B. der Gabelstapler-Führerschein in einer Hochregalzone. Um diese Informationen zu pflegen, steht Ihnen diese Registerkarte zur Verfügung. 
### Button Neuer Skill 

Mit diesem Button können neue Skills angelegt werden. 

Tooltip: 

Notwendigen Skill zur Zone hinzufügen. 

### Button Löschen 

Dieser Button löscht einen Eintrag der Tabelle. 

Tooltip: 

Skill-Zuordnung entfernen
