# Inventur - Positionsauswahl 
Inventur - Positionswahl 

Dieser Dialog öffnet sich im Inventurscan , wenn der Scan keine eindeutige Zuordnung findet und Sie die Tab-Taste drücken. 
## Steuerelemente des Formulars 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. 

### Feld Position 
In diesem Feld wird angezeigt, an welcher Position der Artikel in der Inventur-Tabelle steht. 

Mit dem Button Zurück können Sie die Position eine Position zurücksetzen. 

Mit dem Button Vor können Sie die Position eine Position vorsetzen. 

### Feld Status 
Hier wird der Status der Inventur angezeigt. 

Tooltip: 

Angabe des Status-Texts 

### Feld Änderung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der letzten Änderung eingetragen. 

Tooltip: 

Datum und Uhrzeit der letzten Änderung 

### Feld Änderer 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Lager 
Hier wird die zugrundeliegende Lagernummer angezeigt. 

Tooltip: 

Definiert das Lager 

### Feld Lagerplatz 
Hier wird angegeben, auf welchem Lagerplatz die Packstücknummer zu finden ist. 

Tooltip: 

Angabe des Lagerplatzes 

### Feld Artikel 
Hier wird der zugrundeliegende Artikel angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Button Artikelbild 

Mit diesem Button verzweigen Sie in den Dialog Artikelbild . 

Siehe Kapitel Artikelbild . 

### Feld Artikelbezeichnung 
Hier wird die Artikelbezeichnung angezeigt. 

Tooltip: 

Artikelbezeichnung 

### Feld Ein. Lager 
Hier wird das Gewicht des Artikels angezeigt. 

Tooltip: 

Angabe der Lagereinheit. Dazugehörige Pickliste heißt „Mengeneinheit“. 

### Feld Charge 
Ist der Artikel, den Sie eingetragen haben ein Chargenartikel, wird die Chargennummer des Artikels angezeigt. 

Wenn eine Position neu erfasst wird, kann hier die Chargennnummer eingetragen werden. 

Tooltip: 

Angabe der Chargennummer 

### Feld Inventur Soll 
Hier wird die Sollmenge des Artikels eingetragen. 

Tooltip: 

Angabe der Menge 

### Feld Inventur Ist 
Hier wird die IST-Menge eingetragen. 

Tooltip: 

Angabe der Menge 

### Button Zurück 

Mit diesem Button können Sie die Position eine Position zurücksetzen. 

### Button Auswahl 

Mit diesem Button übernehmen Sie den Artikel in den zugrundeliegenden Dialog Inventurscan . 

### Button Weiter 

Mit diesem Button können Sie die Position eine Position vorsetzen.
