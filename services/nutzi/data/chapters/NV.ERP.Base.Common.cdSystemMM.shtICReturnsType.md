# Kapitel NV.ERP.Base.Common.cdSystemMM.shtICReturnsType

Tooltip: 

Die Einstellung „Abfragedialog in Retoure“ bewirkt beim Verbuchen der Kundenretoure in Stufe 2, 
dass dem Sachbearbeiter ein Dialog angezeigt wird, um situationsbedingt entscheiden zu können, ob eine IC-Rückführung sinnvoll ist oder nicht. 
Die Einstellung „Automatisch“ bewirkt beim Verbuchen der Retoure in Stufe 2, dass automatisch ein Umlagerungseintrag der IC-Position/(en) im anderen Mandanten angelegt wird, 
vorausgesetzt dieser akzeptiert eine IC-Retoure. Wenn nicht, erfolgt eine entsprechende Schlussmeldung nach Abschluss der WE2-Buchung. 
Die Einstellung „Deaktiviert“ deaktiviert die autom. IC-Retourenlogik.
