# Scannerprotokoll 
Scannerprotokoll 

Dieser Dialog öffnet sich im Lagerauftrag >> Scanner >> Scannerprotokoll . Hier kann die komplette Scanner-Kommunikation angezeigt und gelöscht werden. Dieses Kapitel steht für das 1-stufige LVS zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Scannerprotokoll 
Tooltip: 

In diesem Dialog kann die komplette Scanner-Kommunikation angezeigt und gelöscht werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Job-Server 

Dieser Button steht zur Verfügung, um bei der Scanner-Protokollierung eine bessere Übersicht zu erhalten und die Laufzeiten bei Abfragen, aufgrund des Datenwachstums, so kurz als möglich gestalten zu können. 

Dieser Job löscht Einträge der Tabelle unter der konfigurierbaren Bedingung „Löschen von Einträgen mit einem Bis Datum “. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Neuer Satz 

Mit diesem Button können Sie der Tabelle einen neuen Eintrag hinzufügen. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Combobox Hin / Rück 
Hier können Sie auswählen, ob Sie eingehende oder abgehende Nachrichten ansehen wollen. 

### Combobox Sachbearbeiter 
Hier kann zur Selektion ein Sachbearbeiter ausgewählt werden. 

Tooltip: 

Wählen Sie den Sachbearbeiter aus. 

### Combobox Menüpunkt 
Hier wählen Sie aus, welcher Bereich (z.B. Tourenplatzbuchung, Kommissionieren) angezeigt werden soll. 

### Feld Datum von 
Hier werden ein Datum und eine Uhrzeit angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Datum bis 
Um die Suche über einen Datumsbereich einzuschränken, wird hier ein Bis-Datum angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Komm. von 
Für die Suche über eine Kommissionsnummer kann hier eine Nummer eingetragen werden. 

Tooltip: 

Eingabe der Kommunikationsnummer 

### Feld Komm. bis 
Für die Suche über einen Kommissionsnummernbereich kann hier eine Nummer eingetragen werden. 

### Feld Anzeigewert 
Für die Suche über einen Anzeigewert kann hier z.B. eine Lagerauftragsnummer eingetragen werden. Dabei handelt es sich um einen Wert, der dem Scanner User angezeigt wurde. 

### Feld Eingabewert 
Für die Suche über einen Eingabewert kann hier z.B. ein LHM eingetragen werden. 

### Feld Merker Info 
Für die Suche eines bestimmten Merkers kann hier z.B. eine Zahl eingetragen werden. Es werden alle relevanten Merker des Lagerauftrags nach dem eingegebenen Wert durchsucht. Bei einem Merker handelt es sich zum Beispiel um eine Referenznummer (Auftrags-/Bestellnummer). 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Max. Anzahl Datensätze, die eingelesen werden. 

### Tabelle Kommunikation 

Fand innerhalb des selektierten Zeitraums eine Kommunikation am Scanner statt, wird sie in dieser Tabelle angezeigt. Eine Kommunikation ist definiert, wenn ein Scanner-Dialog an einen Scanner übertragen wurde, oder der Sachbearbeiter einen Wert eingetragen hat. 
### Tabellenspalte Kommunikation 
In dieser Spalte wird die Nummer der Kommunikation angezeigt. 

### Tabellenspalte Erfassung 
In dieser Spalte wird der Zeitpunkt der Kommunikation angezeigt. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Tabellenspalte Menüpunkt 
In dieser Spalte wird die Bezeichnung des Scanner-Menüpunkts der Kommunikation angezeigt. Also der Menüpunkt, an dem der Scanner-User zu diesem Zeitpunkt angemeldet war. 

### Tabellenspalte Sachbearbeiter 
In dieser Spalte wird der Scanner-User der Kommunikation angezeigt. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Kommunikation 
Hier wird die Nummer der Kommunikation angezeigt. 

### Feld Erfassung 
Hier wird der Zeitpunkt der Kommunikation angezeigt. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Feld Sachbearbeiter 
Hier wird der Scanner-User der Kommunikation angezeigt. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Combobox Menüpunkt 
Hier wird die Bezeichnung des Scanner Menüpunkts der Kommunikation angezeigt. Also der Menüpunkt an dem der Scanner-User zu diesem Zeitpunkt angemeldet war. 

### Feld Scanner-Dialog emulieren 
In diesem Feld wird der Scanner-Dialog zu der Kommunikationsnummer emuliert. Der Lagerleiter kann sich den tatsächlichen Scanner-Dialog zum dortigen Zeitpunkt anzeigen lassen. 

### Tabelle Merker 

Es werden alle relevanten Merker des Lagerauftrages nach dem eingegebenen Wert durchsucht. Bei einem Merker handelt es sich zum Beispiel um eine Referenz-Nummer(Auftrags-/Bestellnummer). 
### Tabellenspalte Merker Typ 
In dieser Spalte wird die Bezeichnung des Datenfeldes angezeigt. 

### Tabellenspalte Merker 
In dieser Spalte wird der Wert des Datenfeldes angezeigt. 

### Tabelle Eingabe 

Es werden Informationen wie Eingabewert und -typ angezeigt. 
### Tabellenspalte Eingabewert 
In dieser Spalte wird der eingegebene Wert des Scanner-Users angezeigt. 

### Tabellenspalte Eingabewert Typ 
In dieser Spalte wird die Bezeichnung des Datenfeldes vom Eingabewert angezeigt. 

### Tabellenspalte Eingabefeld Typ 
In dieser Spalte wird der Feldtyp des Datenfeldes angezeigt. 

### Tabellenspalte Länge 
In dieser Spalte wird die Länge des Datenfeldes angezeigt. 

### Tabellenspalte Eingabefeld Vorschlag 
In dieser Spalte wird der vorgeschlagene Wert angezeigt, der dem Scanner-User angezeigt wurde.
