# Stücklisten 
Stückliste 

Dieser Dialog öffnet sich im Artikel >> Stamm >> Stücklisten . Hier können Stücklisten für Produktions- und Handelsstücklisten erfasst und verwaltet werden. 

Stücklisten können kalkuliert und die Verfügbarkeit ihrer Bestandteile geprüft werden. 

Mit einem Doppelklick auf eine Artikelposition werden rechts auf der Registerkarte Komponente Informationen zum Artikel eingelesen. 

Siehe Kapitel Stücklistentypen . 

### Erfassen von Stücklisten 
Um eine neue Stückliste zu erfassen, gehen Sie bitte folgendermaßen vor: 

Geben Sie im Feld Stückliste die Artikelnummer der Stückliste ein. Bitte beachten Sie, dass der Artikel im Stamm als Handels- oder Produktionsstückliste (siehe Feld Teileart im Artikelstamm ) gekennzeichnet ist. Schließen Sie die Eingabe mit der Taste Enter oder TAB ab. Die Bezeichnung der Stückliste wird im Feld darunter angezeigt. Wahlweise können Sie auch den Button benutzen, um mit der bekannten Artikelauswahl den gewünschten Stücklistenartikel zu suchen und auszuwählen. 

Zur Erfassung einer neuen Position (Komponente) drücken Sie den Button . Alle Datenfelder werden geleert und Sie können mit der Eingabe der Positionsdaten (auf der Registerkarte Komponente ) beginnen. Bitte planen Sie zunächst die Reihenfolge der anzulegenden Positionen, damit die Positionsnummern richtig erfasst werden können. Für jede neue Position ist eine Positionsnummer festzulegen. 

Eine automatische Nummernvergabe erfolgt nicht. 

Die von Ihnen vergebenen Nummern legen auch die Reihenfolge der Positionen und damit auch die Druckreihenfolge fest. Die Positionsnummer hat maximal 4 Stellen. 

Damit später jederzeit ohne großen Aufwand neue Positionen an beliebiger Stelle in die Stückliste eingefügt werden können, empfehlen wir, die Positionsnummer in 10er oder 100er Schritten anzugeben, also z.B. 10, 20, 30, ... 

Anschließend geben Sie bitte die Artikelnummer der Komponente ein. Alternativ dazu können Sie auch den Button verwenden, um mit Hilfe der Artikelauswahl einen Artikel zu wählen. Wurde die Artikelnummer korrekt angegeben und das Feld mit Enter oder TAB verlassen, erscheint die zugehörige Artikelbezeichnung. 

Jetzt können die weiteren Positionsdaten erfasst werden. 

Um eine mehrstufige Stückliste zu erfassen, markieren Sie zuerst die Baugruppe, die eine neue Komponente erhalten soll. Dann verfahren Sie, wie oben beschrieben. 

Wenn für eine Komponente mehrere Varianten existieren, so müssen diese die gleiche Positionsnummer in der Stückliste besitzen, wenn die Varianten untereinander ausgetauscht werden sollen. 

Haben die Varianten unterschiedliche Positionsnummern innerhalb der Stückliste, so werden diese Variantenartikel entsprechend ergänzt, wenn die entsprechende Variante ausgewählt ist. 

Für jede Position bleibt grundsätzlich die Variante 0 in der Stückliste stehen, außer es gibt einen Ersatz mit der gewählten Variante auf der gleichen Positionsnummer. 

Der Austausch der Varianten kann in der Auftragsstückliste über den Button Var. Einzeln pro Komponente erfolgen (diese muss vor dem Klick auf den Button markiert werden) oder für die gesamte Stückliste mittels Button Var. Alle . 

Siehe Kapitel Auftragsstückliste . 

#### Bei mehrstufigen Stücklisten ist folgendes zu beachten: 
Der übergeordnete Artikel muss im Artikelstamm als Handels- oder Produktionsstückliste erfasst sein. Zu einem Endteil können keine Komponenten erfasst werden! 

Bitte beachten Sie bei der Anlage von Unterpositionen, dass diese nicht nur für die aktuelle Stückliste gelten, sondern auch eine Veränderung in allen Stücklisten bedeuten, die den übergeordneten Artikel ebenfalls als Komponente beinhalten! 

Sie können auch eine andere Handels- oder Produktionsstückliste als Komponente in die aktuelle Stückliste einfügen. Dies bedeutet, dass die neue Position mit all Ihren Komponenten in der Stückliste erscheint. 

Nachdem Sie alle Positionen der Stückliste erfasst haben, schließen Sie bitte Ihre Eingaben mit dem Button . 

### Suchen einer Stückliste (Anzeigeoptionen) 
Suchen und anzeigen können Sie eine Stückliste ganz einfach dadurch, dass Sie in das Feld Stückliste am oberen Dialog-Rand die entsprechende Stücklistennummer eingeben und dann das Feld mit der Taste Enter oder Tab verlassen. Die Stückliste wird gelesen und angezeigt. 

Alternativ dazu können Sie auch mit dem danebenstehenden Button den Dialog Artikelauswahl öffnen und eine Auswahl treffen, falls Ihnen die Artikelnummer nicht bekannt ist. 

Bei sehr umfangreichen Stücklisten oder Stücklisten mit (vielen) Varianten bietet Ihnen das Programm mehrere Möglichkeiten, die Anzeige der Stückliste auf die Komponenten zu beschränken, die Sie für Ihre Arbeit gerade benötigen. 

### Erfassen von Stücklisten-Varianten 
Die Stücklistenverwaltung von eNVenta ERP bietet die Möglichkeit, in einer Stückliste verschiedene Varianten zu definieren. 

Jede Stücklistenposition erhält außer der Positionsnummer auch eine Variantennummer. Standardmäßig wird bei der Neuerfassung einer Position immer die Variantennummer 0 vorgeschlagen. Dies ist die Standardvariante, d.h. die im Normalfall verwendete Variante. 

Auch bei der Übernahme einer Stückliste in einen Auftrag wird zunächst immer die Variante 0 verwendet. Diese kann nachträglich durch den Benutzer gegen andere Varianten ausgetauscht werden. Sind zu einer Stückliste keine Komponenten mit der Variante 0 vorhanden, wird die Auftragsstückliste ohne Positionen angelegt. 

Es sind 3 verschiedene Fälle denkbar: 

- Zu einer Stücklistenposition existiert nur die Variante 0. Diese Position wird in jedem Fall verwendet, auch dann, wenn z.B. eine Stückliste in einer anderen Variante produziert wird. Die Variante 0 ist sozusagen ein Joker, der immer gilt solange es zu der Position keine spezielle Variante gibt. - Zu einer Stücklistenposition existieren die Variante 0 und zusätzlich weitere Varianten. Wird eine Stückliste z.B. in der Variante 2 produziert oder kalkuliert, wird die Positionsvariante 2 verwendet, falls sie vorhanden ist. Existiert zu dieser Position keine Variante 2, findet die Standardvariante 0 Verwendung. - Zu einer Stückliste existiert keine Variante 0, sondern nur Varianten mit einer Variantennummer größer 0. Wird die Stückliste z.B. in der Variante 2 produziert oder kalkuliert, findet die Positionsvariante 2 Verwendung. Existiert diese nicht, wird keine Position mit dieser Positionsnummer verwendet. 
Die Variantennummer kann bei der Neuerfassung einer Position frei vergeben werden, indem einfach die automatisch eingetragene Variante 0 überschrieben wird. 

Soll zu einer bereits vorhandenen Stücklistenposition eine zusätzliche Variante erfasst werden, legen Sie bitte eine neue Position an. Vergeben Sie die gleiche Positionsnummer, aber eine andere, noch nicht vorhandene Variantennummer. 

Bei Handelsstücklisten dürfen Varianten nur Endteile sein, da beim Variantentausch in der Auftragsbearbeitung (mit eNVenta ERP ) nur die Position, nicht aber eine eventuell dahinterstehende Stücklistenstruktur übernommen wird!
