# Artikel Cockpit Registerkarte Bewegung 
Artikel Cockpit Registerkarte Bewegung 

Auf dieser Registerkarte werden Artikelbewegungen angezeigt: 

- alle Bestellvorschläge des Artikels (Bereich Bestellvorschläge ) - alle offenen Bestellungen mit einem Status < 3 ( Bereich offene Bestellungen ) - alle offenen Aufträge Status 1 oder 2 ohne Packliste und Lieferschein (Bereich offene Auftragspositionen ) - zu einem markierten Datensatz in den Bereichen offene Bestellungen bzw. offene Auftragspositionen wird für diesen Datensatz die Änderungshistorie des Auftragskopfes und der Position angezeigt. 
Mit einem Doppelklick in eine Zeile der Tabelle 

- Bestellvorschläge: der Bestellautomat wird geöffnet und Sachbearbeiter, Lager und Hauptlieferant vorbelegt, die Registerkarte Vorgänge geöffnet und gesucht. - Offene Bestellungen: die entsprechende Bestellung wird geöffnet. - Offene Auftragspositionen: der entsprechende Auftrag wird geöffnet.
