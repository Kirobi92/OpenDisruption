# Html Text-Wizard Registerkarte Vorlagen 
Text-Wizard Registerkarte Vorlagen 

Hier stehen die Funktionen ToHtml und ToPlain zur Verfügung. 

Die Funktion ToHtml formatiert jeden Text innerhalb dieser Funktion automatisch in Html um. 

Mit der Funktion ToPlain wird die Vorlage wieder „normaler“ Text. 
### Button Hinzufügen 

Mit diesem Button fügen Sie die gewählte Vorlage ins Dokument ein.
