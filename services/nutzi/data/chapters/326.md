# Verbindungsmanager 
Verbindungsmanager 

Falls Kataloge (Quelldaten) auf einem Server hinterlegt sind, richten Sie den Verbindungsmanager ein. Diesen Dialog finden Sie unter System >> Allgemein >> Dateiverwaltung >> Verbindungsmanager . Hier können für verschiedene Archivtypen (file, ftp, DB, arc) die Verbindungen gepflegt werden, welche dann später mit Daten gefüllt werden. Im oberen Bereich wird der Name angegeben. Dieser steht den Benutzern zur Ansicht zur Verfügung. Hier wird empfohlen, entsprechend „aussagekräftige“ Namen zu vergeben. 

Über die Standardberechtigungen wird der generelle Zugriff auf die angegebenen Verbindungen gesteuert. Im Standard erfolgt der Zugriff auf die Server über den Benutzer des Application Pools. 

Siehe Kapitel Vorbereitung des Datenimports in den Artikelschattenstamm .

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind.

### Combobox Verbindungsart 
Hier wählen Sie die Art der Verbindung aus. Zur Verfügung stehen Datei, Ftp und SFTP (SSH). 

Wenn SFTP (SSH) gewählt ist, werden weitere Felder auf der Registerkarte Allgemein sichtbar und zugänglich. Dort definieren Sie weitere Einstellungen zu SFTP (SSH). Diese Verbindungsart basiert auf dem SSH Netzwerkprotokoll, welches eine verschlüsselte Verbindung zwischen den Endgeräten herstellt. 

Der Public Key muss auf dem Server gespeichert sein. 

Tooltip: 

Selektieren Sie ihre Verbindungsart hier 

### Feld Name 
Hier wird der Name der Verbindung angezeigt, wenn Sie eine Verbindungsart ausgewählt haben. Dieses Feld steht auch zur Suche nach Verbindungen zur Verfügung. 

Tooltip: 

Legen Sie hier den Namen der Verbindung fest. 

### Feld Verb. beschr. 
Hier kann eine kurze Beschreibung der Verbindung eingetragen werden. 

Tooltip: 

Geben Sie hier eine Verbindungsbeschreibung ein. 

### Checkbox Anzeige 
Wenn Sie diese Checkbox aktivieren, darf sich der Sachbearbeiter die Verbindungsdaten anzeigen lassen. 

Tooltip: 

Gibt die Standardberechtigung an, ob die Verzeichnisse der Verbindung sichtbar sind. 

### Checkbox Lesen 
Wenn Sie diese Checkbox aktivieren, darf der Sachbearbeiter die Verbindungsdaten lesen. 

Tooltip: 

Setzen Sie den Haken hier, um standardmäßig lesende Berechtigung auf diese Verbindung zu erlauben. 

### Checkbox Schreiben 
Wenn Sie diese Checkbox aktivieren, darf der Sachbearbeiter Verbindungsdaten schreiben. 

Tooltip: 

Gibt die Standardberechtigung für das Beschreiben der Verzeichnisse an. 

### Checkbox Löschen 
Wenn Sie diese Checkbox aktivieren, darf der Sachbearbeiter die Verbindungsdaten löschen. 

Tooltip: 

Gibt an, ob die Verbindung standardmäßig gelöscht werden kann. 

### Checkbox Herunterladen 
Wenn Sie diese Checkbox aktivieren, darf der Sachbearbeiter die Verbindungsdaten herunterladen. 

Tooltip: 

Gibt die Standardberechtigung für das Herunterladen von Dateien aus der Verbindung an. 

### Checkbox Hochladen 
Wenn Sie diese Checkbox aktivieren, darf der Sachbearbeiter die Verbindungsdaten hochladen. 

Tooltip: 

Gibt die Standardberichtigung für das Hochladen von Dateien in das Verzeichnis an.
