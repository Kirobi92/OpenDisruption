# Tourenstamm Registerkarte Tabelle 
Tourenstamm Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Touren angezeigt. 
### Tabelle 

### Tabellenspalte Route 
Tooltip: 

Angabe der Tourengruppe 

### Tabellenspalte Tour 
Hier geben Sie an, mit welcher Tour (Route) (z.B. täglich, wöchentlich) der Kunde bzw. Lieferant immer (regelmäßig) beliefert werden soll. Die Touren, die zur Verfügung stehen, werden im Tourenstamm gepflegt. 

Siehe Kapitel Tourenstamm . 

Tooltip: 

Angabe der Stammtour 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Tabellenspalte Wochentag 
Hier wird der Wochentag der Tour angegeben. 

Tooltip: 

Wochentag der Tour 

### Tabellenspalte Startzeit 
Hier wird die Startzeit angegeben. 

Tooltip: 

Angabe der Startzeit (hh:mm) 

### Tabellenspalte Fahrzeug 
Hier wird der Name des Fahrzeugs angegeben. 

Tooltip: 

Angabe von Kennzeichen und Fahrzeugtyp 

### Tabellenspalte Tourenlager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager, von welchem aus die Tour startet. 

### Tabellenspalte Tourzyklus 
Hier wird der Tourzyklus angegeben, mit dem die Auslieferung erfolgt. 

Tooltip: 

Angabe, mit welchem Tourzyklus die Auslieferung erfolgt. 

### Tabellenspalte Versandart 
Hier wird die Nummer der Versandart angegeben. 

Tooltip: 

Art, wie die Sendung vom Absender zum Empfänger gelangt. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
In diesem Feld wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
