# Anlagenklassen 
Anlagenklassen 

Dieses Menü öffnet sich über Anlage >> Stamm >> Anlagenklassen . Hier erfassen Sie die Anlagenklassen. Dabei empfiehlt sich eine Anlehnung an die Anlagekonten, da z.B. im Anlagenspiegel eine Gruppierung nach Anlagenklassen erfolgt. Geben Sie für jede Anlagenklasse eine eindeutige Nummer der Anlagenklasse und Bezeichnung ein. 

Im Zusammenhang mit dem Dialog Anlagenklassengruppen steht die Spalte ‚Anlagenklassengruppe‘ zur Verfügung. Diese Spalte enthält eine Combobox aus der Sie eine Anlagenklassengruppe auswählen können. 

Eine Anlagenklasse kann nicht gelöscht werden, wenn Anlagengüter zu dieser vorhanden sind. Weitere Informationen entnehmen Sie bitte dem Anlagenstamm.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button.

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten.

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken.

Dieser Button schließt den Dialog.
