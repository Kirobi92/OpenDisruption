# Differenztexte 
Differenztext 

Dieser Dialog öffnet sich im Navigator >> System >> Finanzbuchhaltung >> Differenztexte und z.B. im Dialog Differenzmeldung Detail über den Button Diff. Texte . In diesem Menü ist es möglich, bis zu 10 Differenztexte, z.B. für den Fall einer unberechtigten Zahlungskürzung, vorzudefinieren. 

Sie können einen vorhandenen Text auch ändern, indem Sie den entsprechenden Text aus dem Feld Auswahl mit einem Mausklick markieren. Es ist nun möglich, alle Textfelder zu ändern. 

### Mehrsprachigkeit 
Eine Differenzmeldung kann in Abhängigkeit von der Sprache des Empfängers (Kunde/Lieferant) ausgedruckt werden. Dazu ist es erforderlich, dass die Differenztexte mehrsprachig erfasst werden. 

In diesem Dialog wird zuerst die Systemsprache angezeigt und darunter die Fremdsprache. Zu jedem Differenztext in Systemsprache können Differenztexte in verschiedenen Fremdsprachen angelegt werden. 

Im Dialog Differenzmeldung Detail wird die Combobox Sprache mit der Sprache des Empfängers (Kunde/Lieferant) vorbelegt. Dort erhalten Sie auch weitere Informationen zur Mehrsprachigkeit.
