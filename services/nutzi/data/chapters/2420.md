# OP Debitor Detail 
OP Debitor Detail 

Dieser Dialog öffnet sich im Dialog Offene Posten Debitor über den Button Details , den Button oder einen Doppelklick in eine Zeile. Hier wird dargestellt, wie sich ein offener Posten aus dem Dialog Offene Posten Debitor im Einzelnen zusammensetzt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld OP Debitor Detail 
Tooltip: 

In diesem Dialog wird dargestellt, wie sich ein offener Posten aus dem Dialog 'Offene Posten Debitor' im Einzelnen zusammensetzt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten.

### Tabellenspalte Valutadatum 
Wenn das Valutadatum eines OPs geändert wird, wird geprüft, ob das Valutadatum vor, gleich bzw. nach dem Belegdatum ist. 

Ist das Valutadatum vor dem Belegdatum, erscheint eine Meldung und gibt einen Hinweis darauf, dass jetzt die Fälligkeiten (Nettofälligkeit und Skontofällig 1-3) anhand des Belegdatums berechnet werden. Wenn das Valutadatum älter bzw. gleich ist, erhalten Sie eine Meldung mit der Frage, ob die Fälligkeiten anhand des Valutadatums neu berechnet werden sollen. 

Tooltip: 

Datum für die Rechnungsstellung und die Berechnung des Zahlungszieles. 

### Tabellenspalte Einzugssperre 
Wenn Sie den Haken in dieser Spalte aktivieren, wird für den OP eine Einzugssperre gültig. 

Der hier eingetragene Wert wird auch an den Dialog Offene Posten Debitor übergeben. 

Damit nicht für jeden neuen OP eine Einzugssperre gesetzt werden muss, gibt es im Kundenstamm auf der Registerkarte Rechnungswesen die Checkbox Einzugssperre . 

Auf bereits in einem Einzugsvorschlag enthaltene OPs wirkt sich das Setzen dieser Checkbox nicht aus. 

Siehe Kundenstamm Registerkarte Rechnungswesen . 

Tooltip: 

Diesen OP nicht im Bankeinzugsvorschlag berücksichtigen. 

### Tabellenspalte E-Sperre bis 
Wenn Sie in diese Spalte ein Datum setzen, wird der Haken in der Spalte 'Einzugssperre' automatisch mit gesetzt. So ersparen Sie sich die Mühe, regelmäßig gesperrte OPs zu überwachen, um sie wieder zu entsperren. 

Wenn Sie das Sperr-Datum löschen, wird immer auch die Checkbox in der Spalte 'Einzugssperre' entfernt. 

Das heißt, möchten Sie aus einer befristeten Einzugssperre eine unbefristete Einzugssperre machen, müssen Sie nach dem Entfernen des Datums in der Spalte ‚E-Sperre bis‘ die Checkbox in der Spalte 'Einzugssperre' aktivieren. 

Wird ein Sperr-Datum gesetzt, erfolgt im Feld OP-Memo automatisch ein Eintrag mit folgenden Informationen: 

- Tagesdatum - Sachbearbeiter - Name des Sperrkennzeichens - Datum der Sperre (entfällt, wenn nur das Sperr-Kennzeichen gesetzt wird) 
Bitte beachten: Wird beim Erfassen einer Sperre / eines Sperr-Datums zusätzlich ein individuelles OP-Memo auf einen Satz erfasst, muss ein "Zwischen"-Speichern erfolgen, ansonsten wird durch Aktivieren der Sperre oder Erfassen des Sperr-Datums das individuelle OP-Memo überschrieben. Zuvor vorhandene OP-Memos bleiben erhalten. 

Tooltip: 

Einzugssperre bis

### Tabellenspalte Factoring-Typ 
Wenn für den Kunden das Factoring aktiviert ist, wird diese Spalte sichtbar und kann teilweise geändert werden. Der Anwender kann im bereits gebuchten OP nachträglich den übermittlungsfähigen Factoring-Typ, z.B. Rechnung, erfassen. Dann wird dieser OP im Dialog Debitoren Factoring, Registerkarte Neue OP beim Suchen selektiert. 

Solange ein OP nicht gemeldet ist, kann der Factoring-Typ geändert werden. 

Zur Auswahl stehen die Einträge: 

- leer - Gutschr. GD - Gutschrift - Rech. AR - Rechnung - Veto
