# Projekt Registerkarte Planung 
Projekt Registerkarte Planung 

Auf dieser Registerkarte werden der terminliche Ablauf und Status des Projekts grafisch dargestellt. 

Um den Status einzelner Phasen schnell überprüfen zu können, stehen die Felder Von Datum , Bis Datum und die Combobox Phase zur Verfügung. 

Sind die Felder Von Datum und/oder Bis Datum gefüllt, werden sie als „Höchstwerte“ für das Gantt festgelegt und die Skalierung angepasst. 

Mit den Buttons am oberen Rand der Registerkarte können Sie die Ansicht des Dialogs / der Tabelle verändern, bzw. haben Sie die Möglichkeit zu drucken. 

Die Phasen erscheinen als dunkelgraue Zeilen, die Elemente einer Phase (Aufgaben, Einsätze und Betriebsaufträge) als hellgraue Zeilen unterhalb der Phase. 

Eine Phase kann mit allen Elementen auf- und zugeklappt werden. Auf der Ebene der Phasenzeile wird zudem ein dunkelgrauer Balken mit „Haken“ an den Enden dargestellt. Dieser Zeitraum zeigt den in der Phase vorgegebenen Planungszeitraum für die Phase. Auf diese Weise ist leicht erkennbar, ob die Elemente der Phase im geplanten Zeitraum ablaufen oder sich Verschiebungen ergeben haben. 

Damit zwischen Aufgaben, Einsätzen und Betriebsaufträgen leichter unterschieden werden kann, ist jedem Balken ein graphisches Zeichen vorangestellt: 

| Pfeil nach unten | Aufgabe |
| --- | --- |
| Dreieck nach unten | Betriebsauftrag |
| Pfeil nach links | Serviceeinsatz |

Per Drag & Drop können Balken terminlich verschoben werden. Außerdem ist es möglich, ein Element in eine andere Phase hineinzuziehen. 

Die Farben der Balken entsprechen dem Status des Elements. 

Nur terminlich festgelegte Elemente können hier visualisiert werden. 
### Button Aktualisieren 

Mit diesem Button aktualisieren Sie die Anzeige im Gantt-Diagramm. 

### Feld Datum von 
Um die Suche nach Phasen einzuschränken, geben Sie hier ein Von Datum ein. 

### Feld Datum bis 
Um die Suche nach Phasen einzuschränken, geben Sie hier ein Bis Datum ein. 

Zur Selektion über einen Datumsbereich füllen Sie die Felder Von Datum und Bis Datum . 

### Combobox Phase 
Hier wählen Sie die Phase aus, die Sie überprüfen möchten. 

Die Combobox ist gefüllt mit einer Liste von allen Phasen, die zu dem Projekt gehören. 

Wird in einem Projekt der Phasenname geändert und das Projekt nicht gespeichert, wird die Auswahl in der Combobox nicht aktualisiert und wird bei Auswahl der veränderten Phase nichts finden.
