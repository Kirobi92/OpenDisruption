# Werksverkehr in der Gesellschaftsverwaltung 
Die Gesellschaftsverwaltung für eine gemeinsame Warenwirtschaft steht nur zur Verfügung, wenn die entsprechende Lizenz freigeschaltet ist! 

Der interne Werksverkehr wird über entsprechende Aufträge und Bestellungen dargestellt. 

Wird nun über eine Auftragsposition ein interner Werksverkehr ausgelöst, wird das Lager, das die Ware liefert, im Feld Bezugslager festgehalten; das Lager das ausliefert, im Feld Versandlager . Wird nun Ware von einem Bezugslager der „einen Gesellschaft“ auf ein Versandlager der „anderen Gesellschaft“ umgelagert, so werden für diese Vorgänge entsprechende interne Verrechnungsbelege erstellt. Die Basis der Verrechnungsbelege wird aus den Gesellschaften der Lagerorte entnommen. In diesem Fall ist der Kunde, der entnimmt, die Gesellschaft des Versandlagers und der Lieferant die Gesellschaft des Bezugslagers. Die entsprechenden Kunden und Lieferanten können aus der Matrix entnommen werden.
