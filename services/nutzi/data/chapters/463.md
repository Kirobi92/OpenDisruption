# Import Items erzeugen 
Basis Dialog Dokumente 

Dieser Dialog öffnet sich im Navigator >> System >> Initialisierung >> Import Items erzeugen und dient dazu aus dem laufenden System Import-Schemata zu erzeugen, um diese z.B. auf einer Test-Datenbank importieren zu können. 

Die Tabelle auf der linken Seite des Dialogs beinhaltet die Spalten (Properties) der zu exportierenden Tabelle(Component). 

Der ausgewählte Eintrag der Tabelle wird in die Bezeichnung des Dialogs integriert. 

Über die beiden Radiobuttons kann definiert werden, ob Spaltenname oder Property-Bezeichnung angezeigt werden soll. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Basis Dialog Dokumente 
Tooltip: 

Dieser Dialog dient dazu aus dem laufenden System Import-Schemata zu erzeugen, um diese z.B. auf einer Test-Datenbank importieren zu können. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Speichern 

Mit diesem Button speichern Sie die Eingaben. 

Tooltip: 

Export starten 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Radiobutton Property 
Wenn Sie diesen Radiobutton aktivieren, werden in der Tabelle die Properties angezeigt. 

### Radiobutton Spaltenname 
Wenn Sie diesen Radiobutton aktivieren, werden in der Tabelle die Spaltennamen angezeigt. 

### Combobox Auswahl 
Je nachdem, ob Sie den Radiobutton Spaltenname oder Property aktiviert haben, stehen in diesem Feld die entsprechenden Einträge zur Auswahl zur Verfügung. 

### Tabelle 

Diese Tabelle beinhaltet die Spalten (Properties) der zu exportierenden Tabelle (Component). 

Der ausgewählte Eintrag der Tabelle wird in die Bezeichnung des Dialogs integriert. 

### Button Alles nach rechts übertragen 

Mit Klick auf diesen Button werden die gewählten Properties der Exportdefinition hinzugefügt und erscheinen auf der Registerkarte Spalten .
