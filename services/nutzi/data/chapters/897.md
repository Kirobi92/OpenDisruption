# ADR-Verpackung 
ADR-Verpackung 

Dieser Dialog öffnet sich im Artikel >> Stamm >> ADR >> ADR-Verpackung . Hier werden der Verpackungsschlüssel und andere wichtige Informationen zur ADR-konformen Verpackung eines Gefahrgutartikels angegeben. 

Pro LHM-Typ muss die ADR-Verpackung hinterlegt werden, wenn Gefahrgüter in diesem LHM verpackt werden sollen. 

Bei der Verpackung wird geprüft, ob der Gefahrgutartikel in die ADR-Verpackung (LHM-Typ) verpackt werden darf. Die Prüfung erfolgt anhand der Gefahrgutverpackung (X, Y, Z) und Verpackungsgruppe (I, II, III) gemäß der ADR Tabelle A . 

Die ADR gibt über die Verpackungsklasse vor, welche Gefahrgüter mit welchen Verpackungen transportiert werden dürfen. Die Verpackungsklasse unterteilt sich hier in I-III. Dies definiert die Gefährlichkeit des verpackten Stoffes. 

Die Transportfähigkeit der Verpackung wird in den Kategorien X, Y, Z festgelegt. Wobei hier folgendes gilt: 

Kategorie Beschreibung 
| | |
| --- | --- |
| X | Aufnahme von Gefahrgut der Gruppe I, II, III möglich |
| Y | Aufnahme von Gefahrgut der Gruppe II, III möglich |
| Z | Aufnahme von Gefahrgut der Gruppe III möglich |

Ein Artikel, der seinerseits keine eigene ADR-Verpackung mitbringt, darf nicht in ein LHM verpackt werden, welches kein Ladehilfsmittel im Sinne der ADR darstellt. 

Neben der Gefahrgutverpackung müssen die Verpackungsart, die Werkstoffart und der UN-Verpackungscode angegeben werde. Die Grunddaten hierzu werden aus der Pickliste ausgewählt. Die Pickliste wird automatisch angelegt. 

Aus der Angabe der Daten wird automatisch die UN-Verpackungsbezeichnung generiert, welche für das Reporting verwendet wird. 

### ADR-konform verpackte Artikel 
Es gibt die Möglichkeit, dass Artikel bereits im Lager ADR-konform verpackt liegen. Dies kann z.B. die Spraydose sein, wenn diese vom Lieferanten entsprechend deklariert wurde bzw. direkt darauf abgedruckt ist. In diesem Fall kann nach dem Einlesen des Artikels im Artikelstamm der Dialog ADR-Verpackung geöffnet werden und die Daten werden analog zu den LHM-Typen gepflegt. Solche hinterlegten Artikel müssen in der Verpackung (Dialog Packstücke ) nicht nochmals (um)verpackt werden.

### Feld Verpackungstyp 
Hier wird der Verpackungstyp angegeben. 

Gefahrgutartikel müssen immer einem Packstück (LHM) zugeordnet sein, außer sie sind bereits ADR konform verpackt. 

A - Artikel mit einer ADR-konformen Verpackung. 
LHM - Ladehilfsmittel das zum Verpacken verwendet werden soll.

### Combobox Gefahrgutverpackung 
Hier wird die Gefahrgutverpackung angegeben. 

Die Gefahrgutverpackungen werden in drei Gruppen unterteilt, die nur für Gefahrgüter bestimmter Verpackungsgruppen verwendet werden: 

X = hohe Gefahr, kann Gefahrgüter der Verpackungsgruppen I, II und III aufnehmen. 
Y = mittlere Gefahr, kann Gefahrgüter der Verpackungsgruppen II und III aufnehmen. 
Z = geringe Gefahr, kann Gefahrgüter der Verpackungsgruppe III aufnehmen. 

Tooltip:
