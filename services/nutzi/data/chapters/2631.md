# Kreditorenvorerfassung Registerkarte Vorerfassung 
Kreditorenvorerfassung Registerkarte Vorerfassung 

Auf dieser Registerkarte werden weitere Daten zur Erfassung der Rechnung/Gutschrift eingegeben bzw. automatisch vorbelegt. 
### Feld Betrag FW 
Dieses Feld wird zugänglich, wenn es sich beim Lieferanten um einen Fremdwährungs-Lieferanten handelt. 

Tooltip: 

Angabe des Betrags in Fremdwährung 

### Feld Währung 
Hier wird das Währungskürzel des Kunden angezeigt. 

Tooltip: 

Währungskürzel 

### Feld Betrag 
Hier geben Sie den Rechnungsbetrag ein. 

Tooltip: 

Angabe des Betrags 

### Combobox Buchungstext + 
In diesem Feld können Sie einen Standardtext über die Combobox auswählen oder manuell erfassen. Wird bei einer Buchung in diesem Feld als letztes Zeichen „+“ eingegeben, so wird die Buchung ausgeführt. Dabei wird das Zeichen „+“ wieder automatisch aus dem Buchungstext entfernt. 

Tooltip: 

Zusätzliche Information zur Buchung 

### Combobox Bankverbindung 
Hier wird die Standard-Bankverbindung angezeigt. Gibt es keine speziell gekennzeichnete Bankverbindung, wird die jüngste Bankverbindung zum Kunden bzw. Lieferanten angezeigt. Für diesen Vorgang kann die Bankverbindung auch geändert werden. 

### Combobox Zahlung 
Hier wird die Zahlungsbedingung angegeben. Es werden Nummer und Text angezeigt. 

Tooltip: 

Zahlungsbedingung 

### Feld Valutadatum 
Hier geben Sie das Datum ein, an dem die Rechnung im Unternehmen eingegangen ist. 

Tooltip: 

Datum für die Rechnungsstellung und die Berechnung des Zahlungszieles. 

### Feld Kurs 
Hier wird der hinterlegte Wechselkurs angezeigt. 

Tooltip: 

Währungskurs 

### Combobox Abteilung 
Über diese Combobox haben Sie die Möglichkeit, die Abteilung auszuwählen, welche für die Rechnung zuständig ist. Vorbelegt wird die Combobox mit der Abteilung des Sachbearbeiters, der die Rechnung bearbeitet. 

### Feld Zuständig 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Pufferzeit 
Hier tragen Sie ein, wie viele Tage vor Skonto-Fälligkeit die Rechnung/Gutschrift in der Meldeliste aufgeführt werden soll. Die Pufferzeit wird aus den Fibu-Parametern vorbelegt und kann in jedem Vorgang neu angepasst werden. 

### Feld Rückdatum soll 
In diesem Feld wird unter Berücksichtigung des Belegdatums, der Zahlungsbedingung und der Pufferzeit das Rückmeldedatum ermittelt. Die Abfrage des Rückmeldedatums erfolgt auf <=! (<= bedeutet hier „kleiner gleich“!). 

### Feld (+) 
Wenn Sie hier ein Plus setzen, haben Sie die Möglichkeit sich zu einem späteren Zeitpunkt (wenn alle vorangegangen Felder gefüllt wurden) für die oben beschriebene Vorgehensweise zu entscheiden. 

### Feld Memo 
In diesem Feld können Sie entsprechende Informationen zum erfassten Beleg eintragen. 

Tooltip: 

Enthält interne Textinformationen.
