# Parameter Gelangensbestätigung 
Parameter Gelangensbestätigung 

Dieser Dialog öffnet sich im System >> Verkauf >> Gelangensbestätigung oder über den Button Parameter im Dialog Gelangensbestätigung . Hier definieren Sie die Standardeinstellungen für die Gelangensbestätigung, wie die Email-Vorlage, die Email-Vorlage für die Mahnung mit Rechnungsandruck oder Artikelandruck. 

Der Versand von E-Mails ist nur über SMTP möglich! 

Siehe Kapitel Gelangensbestätigung . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Parameter Gelangensbestätigung 
Tooltip: 

In diesem Dialog definieren Sie die Standardeinstellungen für die Gelangensbestätigung, wie die E-Mail-Vorlage, die E-Mail-Vorlage für die Mahnung mit Rechnungsandruck, sowie Artikelandruck. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
