# Bankeinzug Registerkarte Selektionen 2

Finanzbuchhaltung / Bankeinzug 

# Bankeinzug Registerkarte Selektionen 2 
Registerkarte Selektionen 2 

Auf dieser Registerkarte stehen weitere Selektionskriterien zur Verfügung. 
### Combobox Kundengruppe 
Hier wird eine Kundengruppe angegeben. 

Tooltip: 

Zeigt die Kundengruppe. 

### Combobox Währung 
Hier wird das Währungskürzel angegeben. 

Tooltip: 

Währungskürzel 

### Combobox LKZ 
Hier wird das Länderkennzeichen zum Datensatz angegeben. 

Tooltip: 

Länderkennzeichen des Datensatzes 

### Checkbox Factoring aktiv 
Wenn der Kunde am Factoring teilnimmt, aktivieren Sie diese Checkbox. 

Tooltip: 

Aktiviert das Factoring 

### Combobox Niederlassung 
Hier wird die Niederlassungsnummer angegeben. 

Tooltip: 

Angabe der Niederlassung 

### Checkbox BEZ verzögert 
Wenn Sie diese Checkbox aktivieren, wird der Bankeinzug nicht sofort ausgeführt.
