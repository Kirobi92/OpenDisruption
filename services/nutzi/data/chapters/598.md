# Neuer Referenz-Link Registerkarte Link erstellen / bearbeiten 
Neuer Referenz-Link Registerkarte Link erstellen / bearbeiten 

Auf dieser Registerkarte können Sie neue Referenzen anlegen. 

Auf der linken Seite sehen Sie die Quelle des Referenz-Links, welche aus dem vorhergehenden Dialog übernommen wurde. Sie sehen hierbei das Programmmodul, die Referenz und ggf. die Referenzposition, wenn sich die Referenz z.B. auf eine einzelne Auftragsposition bezieht. 

Auf der rechten Seite bestimmen Sie das Ziel bzw. den 2. Teil des Referenz-Links. Dazu ist als erstes die Checkbox als nächstes setzen zu aktivieren. Jetzt können Sie beispielsweise einen Call oder jedes andere beliebige Modul bzw. Vorgang öffnen, zu dem Sie eine Referenz herstellen möchten; klicken Sie im geöffneten Call-Dialog den Button an und die Referenz wird in die Registerkarte Link erstellen/bearbeiten übernommen. 
### Button Abbrechen 

Mit diesem Button können Sie den gesamten Vorgang abbrechen. Dabei werden alle nicht gespeicherten Referenz-Links verworfen. 

Tooltip: 

Link-Erfassung abbrechen 

### Button URL 

Diesen Button klicken Sie an, wenn Sie eine URL als Referenz setzen möchten. Die Feldbezeichnungen im Bereich ‚Ziel‘ werden angepasst. Geben Sie hier nun den Pfad ein, um eine Referenz zu einer URL herzustellen. Dabei ist es wichtig, den Dateitypen anzugeben, um die weitere reibungslose Verarbeitung zu gewährleisten, z.B. ob es sich um eine doc-, pdf- oder xml-Datei handelt. 

### Button Server Datei 

Mit diesem Button ist es beim Hinzufügen oder Bearbeiten von Referenz-Links möglich, eine Datei aus dem Server-Datei-Archiv auszuwählen. Dabei wird der Link-Typ nicht vorbelegt, da es sich ja um verschiedene Dateien handeln kann (dies muss nicht zwangsläufig ein Bild sein). 

Generell sollte bei einem Referenz-Link auf eine Datei versucht werden immer eine Server Datei auszuwählen, damit sichergestellt ist, dass jeder Sachbearbeiter Zugriff auf diese Datei hat.

Hier wird die Version angegeben, wenn die Versionierung für den gewählten Link-Typen aktiv ist.

### Combobox Archiv 
Ist in diesem Dialog ein Link-Typ gewählt, dem ein Archiv hinterlegt ist, wird dieses automatisch in der Combobox voreingestellt. 

Ist das hinterlegte Archiv nicht mit dem Quelltyp verschlagwortet, erscheint eine entsprechende Fehlermeldung. 

Es ist jederzeit möglich ein Archiv, welches verschlagwortet ist, frei in der Combobox zu wählen, sowohl mit als auch ohne Link-Typ. 

Ist ein Archiv ausgewählt, so wird die Datei automatisch ohne folgende Benutzer-Interaktion in das Archiv eingefügt. 

Tooltip: 

Logisches Archiv 

### Feld Auslaufdatum 
Hier wird das Auslaufdatum der Gültigkeit des Datensatzes angegeben. 

Tooltip: 

Dieses Datum gibt an, wann ein Referenzlink ungültig wird.
