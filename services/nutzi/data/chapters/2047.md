# Simulation Registerkarte Terminierung 
Simulation Registerkarte Terminierung 

Auf dieser Registerkarte wird das Gantt-Chart für den kompletten mehrstufigen Produktionsablauf dargestellt. 

Siehe Kapitel Gantt-Diagramm . 

Die Farben der Balken zeigen den Status des jeweiligen Arbeitsgangs: 

Blau : noch auszuführende Arbeitsgänge 
Rot : in Bearbeitung befindliche Arbeitsgänge 
Gelb : unterbrochene Arbeitsgänge 
Grün : abgeschlossene Arbeitsgänge 

Zusammenhängende bzw. abhängige Balken desselben Betriebsauftrags werden mit einem Link verbunden angezeigt. 

Per Doppelklick auf einen Eintrag in der Spalte ‚Betriebsauftrag / Arbeitsgang‘ oder einen Balken (rechts im Dialog) kann direkt in den Betriebsauftrag verzweigt werden. 
### Button Einplanen > 

Dieser Button führt eine Vorwärtsterminierung durch und die benötigten Kapazitäten der Werkzeuge für den aktuellen Produktionsauftrag werden reserviert. 

Dabei gilt: Die „untersten“ Betriebsaufträge werden zuerst terminiert. Am Schluss folgt der „oberste“ Betriebsauftrag. Gibt es auf einer Ebene mehrere Betriebsaufträge, so werden diese parallel eingeplant. Mit dem „untersten“ Betriebsauftrag ist die Baugruppe gemeint, welche zuerst produziert werden muss, da diese bei der nächsthöheren Baugruppe als Komponente genutzt wird. 

Tooltip: 

Terminierung vorwärts 

### Button Parken 

Dieser Button setzt die komplette Terminierung des Produktionsauftrags zurück. Soweit noch nicht begonnen, wird der einzelne Betriebsauftrag auf Status offen zurückgesetzt. Begonnene Betriebsaufträge erhalten den Status geparkt . 

Tooltip: 

Setzt die Terminierung des Produktionsauftrags zurück. 

### Button < Einplanen 

Dieser Button führt eine Rückwärtsterminierung durch und die benötigten Kapazitäten der Werkzeuge für den aktuellen Produktionsauftrag werden reserviert. 

Tooltip: 

Terminierung rückwärts 

### Feld Datum/Zeit von 
Hier legen Sie das Startdatum für die Produktion fest.
