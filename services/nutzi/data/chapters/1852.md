# Druck der Inventuraufnahmeliste und Inventurauswertung 
Folgende Reports stehen Ihnen standardmäßig zur Verfügung: 

- InventoryList (Inventur-Aufnahme): 
Diese Liste enthält für jeden Artikel folgende Informationen: 
- Artikelnummer - Artikelbezeichnung - Lagerplatz - Chargennummer - Sollmenge und eine weitere Spalte (Menge), in welche Sie die gezählte Menge eintragen können. Anhand dieser Liste ist eine Schnellerfassung in der Tabelle möglich. Der Druck dieser Aufnahmeliste ist zu jedem Zeitpunkt möglich. - InventoryEvaluation (Inventur-Auswertung): 
Nachdem Sie die Erfassung einer Inventurauswertung abgeschlossen haben, ist ein Druck sinnvoll. 
Die Liste enthält für jeden Artikel folgende Informationen: 

Differenz zwischen Inventur-Ist-Bestand - Inventur-Soll-Bestand (Menge), Lagerwert (Basis LEK), Lagerwert (Basis GEK), Differenz (Basis LEK), Differenz (Basis GEK) 

### Berechnung der Differenzen 
Bei der Berechnung der Differenzwerte zwischen dem Sollwert (EK Preis x SollMenge) und dem Istwert (EK Preis x IstMenge) wird der Bewertungsprozentsatz berücksichtigt. Durch die Angabe des Bewertungsprozentsatzes kann der Istwert anteilig reduziert werden, entsprechend fällt der Differenzwert höher aus. 

Der Differenzwert wird für jede Kalk-Basis gerechnet und innerhalb der Inventur gespeichert. Wollen Sie eine Position komplett abschreiben, so muss ein Bewertungsprozentsatz von 0 eingegeben werden. In diesem Fall werden alle EK-Preise der Position auf den Erinnerungswert von 1,00 Euro abgeschrieben.
