# Parameter Kostenrechnung 
Parameter Kore 

Dieser Dialog öffnet sich im Navigator >> System >> Kore >> Parameter . 

Hier steht die Checkbox Kostenträger Gültig von/bis aktiv zur Verfügung. So können Kostenträger zeitlich begrenzt werden. Darüber können dann Projekte (z.B. Baustellen) überwacht werden. Des Weiteren können Kostenträger durchgängig mit Planwerten bearbeitet werden. 

Wenn Sie diese Checkbox aktivieren, werden im Dialog Kostenträger die Felder Gültig von und Gültig bis sichtbar und in diversen Auswertungsdialogen die Checkbox seit Beginn .

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Checkbox Kostenträger gültig von/bis aktiv 
Wenn Sie diese Checkbox aktivieren, 

- werden im Dialog Kostenträger die Felder Gültig ab und Gültig bis eingeblendet und sind änderbar. - wird in diversen Auswertungsdialogen (z.B. Kostenträgerjournal, Kostenauswertungen) die Checkbox seit Beginn eingeblendet und ist änderbar. - wird im Menü Auswertungen die Auswahl Kostenträgerblatt seit Beginn ausgewiesen. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, werden im Dialog 'Kostenträger' die Felder 'Gültig von' und 'Gültig bis' sichtbar und in div. Auswertungsdialogen die Checkbox 'seit Beginn'. 
So können Kostenträger zeitlich begrenzt werden. Darüber können dann Projekte (z.B. Baustellen) überwacht werden.
