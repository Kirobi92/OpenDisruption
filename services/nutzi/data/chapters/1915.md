# Rabatt Einkauf Registerkarte Gültigkeit 
Rabatt Einkauf Registerkarte Gültigkeit 

Für eine komfortable Handhabung der Rabattverwaltung steht die Registerkarte Gültigkeit zur Verfügung. 

Werden eine Staffelstufe oder eine Gültigkeit geändert, werden die Felder Sachbearbeiter und Änderungsdatum aktualisiert. Damit dies sofort angezeigt werden kann, muss die Rabattverwaltung komplett neu eingelesen werden. Dadurch springt die Anzeige automatisch auf die Registerkarte Allgemein . 

Es werden folgende Werte übergeben: 
- Gültig ab - Gültig bis - Basispreis - Rabatt1% - Rabatt2% - Preisgruppe - Preisgruppendatum - Standardstaffel 
Wenn Preisgruppe UND Basispreis angegeben sind, "gewinnt" immer der Basispreis. Die Preisgruppe wird nur dann verwendet, wenn kein Basispreis angegeben wird. Die Spalte 'PG-Datum' definiert dann (optional) das Datum, zu dem der Preis in der angegebenen Preisgruppe gesucht wird. 

Die Spalte 'PG-Datum bis' definiert (optional), wie lange mit dem PG-Datum gearbeitet werden soll. Ist dieses Datum abgelaufen, wird wieder mit dem Tagesdatum in die Preisfndung eingestiegen. 

Bei einem Zugriff wird immer der Datensatz gesucht, der dem Tagesdatum am nächsten ist. 

Damit nicht zwangsweise immer eine neue Staffel angelegt werden muss, nur weil z.B. ein neuer Basispreis wirksam werden soll, muss für die Tabelle die Checkbox in der Spalte 'Standardstaffel’ aktiviert werden. Ist diese Checkbox gesetzt, wird dieses Gültigkeitsdatum in der Combobox der Registerkarte Staffel nicht angeboten, sondern bei Zugriff die Standardstaffel verwendet. 

Wird eine Gültigkeit als „zu löschen“ markiert, erscheint eine Hinweismeldung, falls noch eine Rabattstaffel zur Gültigkeit vorhanden ist. Der Vorgang kann an dieser Stelle korrekt abgebrochen werden. Beim Löschen der Gültigkeit wird die entsprechende Staffel komplett mit gelöscht. 

Auf der Registerkarte Gültigkeit kann mit <Strg + Umschalt + F3 > gespeichert werden. 

Mit F3 wird wie bisher alles gespeichert. 
### Tabelle Gültigkeiten 

### Tabellenspalte Gültig von 
Das Datum, das Sie hier auswählen, bestimmt den Beginn der Gültigkeit. 

### Tabellenspalte Basispreis 
Hier wird der Basispreis angegeben. 

Tooltip: 

Angabe des Basispreises 

### Tabellenspalte Rabatt 1 % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Rabatt 2 % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Preisgruppe 
Hier können Sie eine Brutto-Preisgruppe auswählen, die im zugrundeliegenden Vorgang herangezogen wird und bestimmt, auf welchen Verkaufspreis zugegriffen werden soll. 

Siehe Kapitel Preisgruppen . 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

### Tabellenspalte PG-Datum 
Mit dem Preisgruppendatum kann ein abweichendes Datum angegeben werden, mit dem Preise aus Preisgruppen ermittelt werden. Dieses abweichende Datum ist beschränkt bis zum Datum "Gültig bis". 

Tooltip: 

Preisgruppendatum: 

Mit dem Preisgruppendatum kann ein abweichendes Datum angegeben werden, mit dem Preise aus Preisgruppen ermittelt werden. Dieses abweichende Datum ist beschränkt bis zum Datum "Gültig bis". 

### Tabellenspalte PG-Datum bis 
Hier wird das Bis-Datum angegeben für die Gültigkeit der Preisgruppe. 

Tooltip: 

Bis-Gültigkeitsdatum des Preisgruppendatums 

### Tabellenspalte Standardstaffel 
Hier wird die Standardstaffel angegeben. Dabei bedeutet 
1 - aktiviert 
0 - deaktiviert 

Tooltip: 

Angabe einer Standardstaffel 

### Button Löschen 

Mit diesem Button löschen Sie eine Zeile der Tabelle. 

### Button Satz hinzufügen 

Mit diesem Button fügen Sie eine neue Zeile in die Tabelle ein.
