# Kapitel NV.ERP.Base.Common.cdCountry.sForeignTaxNumber

Sind Sie (auch) in diesem Land steuerpflichtig, wird für die länderspezifische Umsatzsteuervoranmeldung diese Steuernummer verwendet.
