# Dokument erstellen (Vorlage auswählen) 
Vorlagenauswahl 

Dieser Dialog öffnet sich über das Menü Druck >> Dokument erstellen . Hier können Sie zum aktuellen Vorgang (Kunde, Auftrag, Lieferant, usw.) Word-Dokument-Vorlagen, SMS-Vorlagen, Fax-Vorlagen bzw. E-Mail-Vorlagen auswählen. Es öffnet sich die Auswahl mit den vorhandenen Dokumentvorlagen, aus denen Sie die gewünschte mit dem Button öffnen können. Angelegt werden diese Vorlagen im Dialog Dokument Vorlagen . 

Siehe Kapitel Dokument Vorlagen . 

Wenn Sie ein Dokument archivieren, so wird dieses beim Verlassen der Dokument-Bearbeitung in das Archiv gespeichert. Nach dem Schließen des Dokuments erscheint der Dialog Export nach MS-Word . 

Nach dem Schließen des Dialogs über den Button wird das Dokument archiviert. 

Über den Windows-Schließen-Button (der oberste / äußerste -Button) wird das Dokument nicht archiviert. 

Mit einem Klick auf eine SMS-Vorlage öffnet sich der Dialog SMS senden . 

Die entsprechende Mobilnummer hinterlegen Sie im Dialog Ansprechpartner . 

Siehe Kapitel SMS senden . 

Mit einem Klick auf eine Fax-Vorlage öffnet sich der Dialog Fax senden . 

Siehe Kapitel Fax senden .

### Combobox Sprache 
Hier wählen Sie die Sprache für die Vorlagenart aus. In dieser Combobox werden nur die Sprachen angezeigt, in denen die selektierte Vorlage gewählt ist. 

### Checkbox Dokument archivieren 
Wenn Sie diese Checkbox aktivieren, wird die Vorlage im Archiv gespeichert. Dies geht nur für MS Word- bzw. OO Writer-Vorlagen! 

Hierbei gilt: ist hier nicht das zu verwendende Archiv explizit hinterlegt, wird das Standard Mapping des Objekts "gezogen". 

Dabei ist folgendes zu beachten: 

Wird ein archivierbares Dokument (Word-Dokument) geöffnet und geändert, so wird dieses korrekt archiviert, vorausgesetzt der Dialog wird nicht zuvor aus eNVenta ERP heraus geschlossen. Parallel zum eNVenta ERP -Dialog wird das Dokument in Word von OpenOffice geöffnet, um es dort zu bearbeiten und zu drucken. Erst beim Schließen des eNVenta ERP -Dialogs erfolgt die Weitergabe an das Archiv. 

Ist Word doch vorher geschlossen worden, so kann der eNVenta ERP -Dialog mit mehrfachem Klick auf Schließen ohne Archivierung geschlossen werden. 

Tooltip: 

Dokument archivieren - NUR für MS Word-/OO Writer-Vorlagen 

### Combobox Archiv-Zuordnung 
Hier wählen Sie ein Archiv aus, wenn Sie das Dokument archivieren möchten. Die Archive werden im Menü Archiv-Zuordnung gepflegt. 

### Checkbox Bereichsbezogen 
Wenn Sie diese Checkbox aktivieren, sind die Vorlagen nur für das Modul gültig, aus dem die Vorlagen aufgerufen wurden.
