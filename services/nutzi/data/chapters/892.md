# ADR-Zuordnung 
ADR-Zuordnung 

Dieser Dialog öffnet sich über den Button ADR Zuordnungswahl im Dialog ADR Tabelle A oder im Artikel >> Stamm >> ADR-Zuordnung . 

Über diesen Dialog wird gesteuert, welche UN-Nummer der Tabelle A zu welchem Artikel gehört. 

Die UN-Nummer eines Artikels wird hierarchisch ermittelt: 

- Artikel - Artikelgruppe - Artikelhauptgruppe 
Dazu gibt es auf der Registerkarte Zuordnungen eine Tabelle, welche folgende Informationen beinhaltet: 

- UN-Nummer - Bezeichnung - Verpackungsklasse - Zuordnungstyp (Artikelgruppe oder Artikel) - Zuordnungsschlüssel (Artikelgruppennummer oder Artikelnummer) - Faktor Gefahrguteinheit zu Artikellagereinheit - Ausrichtungspfeil (links bzw. rechts) - Anteil der Beschreibung, welcher tatsächlich verwendet werden soll - ADR Mengeneinheit (Liter, wenn es sich um ein Gas / eine Flüssigkeit handelt, Kg, wenn es ein Feststoff ist.) 
Für die Zuordnung der Artikel müssen diese über den Button neben der Tabelle Artikel eingelesen werden. Es können hierbei mehrere Artikel auf einmal eingelesen werden. Im Anschluss erfolgt über den Button neben der Tabelle UN-Nummer die Suche der UN-Nummer. Wenn die entsprechenden Einträge in den beiden Tabellen markiert sind, kann mittels Button die Zuordnung erzeugt werden. 

Bei der Generierung der Zuordnung muss der Faktor Gefahrguteinheit angegeben werden. Dieser bezieht sich darauf, welche Menge Gefahrgut im Verhältnis zur Lagereinheit vorhanden ist. Details zur Berechnung der Punkte siehe im Kapitel Berechnung der Punkte . 

In der Spalte ‚Ausrichtungspfeil‘ geben Sie an, wie die Gefahrgut-Etiketten auf das Packstück geklebt werden müssen. Diese Information können Sie auf den Packlisten andrucken. 

Sollte es eine Pflichtangabe der ADR Einheit in Kg oder Litern (LIT) geben, so kann dies in der Zuordnungstabelle erfolgen. Die Angabe wird beim Druck herangezogen. Selbst bei Lagerung in Kg oder Litern muss die Angabe erfolgen. 

Wird ein Artikel oder eine Artikelgruppe gelöscht, so werden die ADR-Artikelzuordnungen zum Artikel gelöscht. 

Die Zuordnung von Diversartikeln zu UN-Nummern ist nicht möglich. Sollte Gefahrgut mittels Diversartikel beschafft werden, so muss dies in eine eindeutige Artikelnummer für diesen Vorgang umgewandelt werden. 

Die Zuordnung von Artikel zu UN-Nummer können Sie auch über den Artikel Schattenstamm importieren. Bitte stellen Sie hierbei sicher, dass zuvor die Tabelle A importiert wurde, wenn Sie die Zuordnung importieren.

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden.

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
