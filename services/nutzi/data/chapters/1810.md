# Belastung Registerkarte Meldungen 
Registerkarte Meldungen 

Wenn ein Vorgang zu einem Artikel erfasst wird und diesem Artikel ist kein EK hinterlegt, wird systemseitig der LEK verwendet. Sie werden mit einer Meldung auf der Registerkarte Meldungen darauf hingewiesen. 
### Feld Text 
Hier wird eine Meldung angezeigt, wenn der Preis aus dem Artikelstamm kommt.
