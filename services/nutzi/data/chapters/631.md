# Html Editor 
Html Editor 

Dieser Dialog öffnet sich über den Button und dient der Bearbeitung von Html-Texten in eNVenta ERP (z.B. in Textbausteinen, Text-Wizard, Auftrag, Call-Center, Email-Versand, Bestellung, Start- und Endetexte in Angebot, Auftrag, Gutschrift etc.). 

Hier kann Text bearbeitet werden: Es können Fettdruck, Farbe, Schriftgröße und Zeichengröße gesetzt werden sowie Bilder und Links eingefügt werden. Die einzelnen Funktionen werden nachfolgend beschrieben. Es werden die Befehle der Html-Standard Version 3.2 verarbeitet. 

Folgende Abfragen sind möglich: 

Syntax für Listen: 
{{LIST {{oObj.oSalesOrderDetailColl}} {{oObj.oSalesOrderDetailColl:oArticle.sCountryOfOrigin}} }} 

Syntax für IF-Conditions: 
{{IF {{oObj.sCode1}} {{=}} {{""}} {{oObj.sCode2}} {{oObj.sCode1}}}} 

Syntax für TOHTML: 
{{TOHTML {{oObj.sArticleName}}}} 

Syntax für TOPLAIN: 
{{TOPLAIN {{oObj.sStartTextHTML}}}} 

## Steuerelemente des Formulars 

### Bezeichnungsfeld Html-Editor 
Tooltip: 

In diesem Dialog können Sie Html-Texte bearbeiten. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
