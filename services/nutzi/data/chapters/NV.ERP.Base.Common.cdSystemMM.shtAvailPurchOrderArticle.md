# Kapitel NV.ERP.Base.Common.cdSystemMM.shtAvailPurchOrderArticle

Wenn Sie diese Checkbox aktivieren, erscheint immer die Verfügbarkeitsprüfung, wenn Sie im Auftrag einen Beschaffungsartikel eintragen und der Bestand nicht ausreichend ist. So ist es möglich im Auftrag den Liefertermin anzupassen, damit dem Kunden immer das korrekte Lieferdatum angegeben wird. 

Außerdem öffnet sich die Verfügbarkeitsprüfung, wenn bei dynamischer Verfügbarkeit der Liefertermin außerhalb der Wiederbeschaffungszeit liegt. 

Die Position wird in Status 1 gesetzt. 

Tooltip: 

Ist dieser Parameter aktiv, so wird, abweichend von der Standardlogik, der Verfügbarkeitsdialog 
für Beschaffungsartikel geöffnet. Dies aber nur, wenn der Bestand nicht ausreicht.
