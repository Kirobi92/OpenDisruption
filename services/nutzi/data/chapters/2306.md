# Versandprofile im Webshop Registerkarte Tabelle 
Versandprofile Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Versandarten für den Webshop angezeigt. 
### Tabelle 

### Tabellenspalte Bezeichnung 
Hier geben Sie einen Namen für das Profil an. 

Tooltip: 

Geben Sie hier eine Bezeichnung für das Profil an. 

### Tabellenspalte Webshop 
Hier wird die ID des Webshops angegeben, für welchen die Versandart gültig ist. 

Tooltip: 

Geben Sie hier den Webshop an, für welchen das Versandprofil gültig ist. 

### Tabellenspalte Min. Bestellwert 
Hier wird der Mindestbestellwert angegeben, der für die Versandart gilt. 

Tooltip: 

Geben Sie hier einen Mindestbestellwert an. 

### Tabellenspalte Kundenart 
Hier wird die Kundenart angegeben (z.B. Geschäftlich, Privat). 

Tooltip: 

Angabe der Kundenart 

### Tabellenspalte Land 
Hier wird das Land angegeben. 

Tooltip: 

Angabe des Lands 

### Tabellenspalte Versandbedingung 
Hier wird die Versandbedingung angegeben. 

Tooltip: 

Angabe der Versandbedingung 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Dieses Feld zeigt die Anzahl der tatsächlich eingelesenen Datensätze.
