# Kontakt-Historie 
Kontakt-Historie 

Der Button Kontakte (z.B. im Kundenstamm) öffnet eine Art Historie, in der alle Vorgänge zu einem ausgewählten Kunden, Lieferanten oder zu einer Adresse angezeigt werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Kontakt-Historie 
Tooltip: 

In diesem Dialog sehen Sie eine Art Historie, in der alle Vorgänge zu einem ausgewählten Kunden, Lieferanten oder zu einer Adresse angezeigt werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Registerkarte Historie 

Kontakt-Historie Registerkarte Historie 

Sie haben hier eine chronologische Übersicht der letzten Kontakte zum Kunden, geordnet nach Programmbereichen. In den einzelnen Programmbereichen wird grundsätzlich nach dem Kontaktdatum absteigend sortiert. 

Aufträge, welche aufgrund der Berechtigung nicht geöffnet werden können, werden in eckiger Klammer angezeigt. 

Beispiel: 

Kontakt-Historie Registerkarte Historie 

Wenn Sie einen Eintrag markieren, wird im rechten Bereich der Html-Report dazu angezeigt und unterhalb wird die Chronologie grafisch und damit sehr übersichtlich dargestellt. Durch entsprechende Anpassung des Html-Reports, am Vorgang zu konfigurieren, haben Sie die Möglichkeit Einfluss auf die angezeigten Daten zu nehmen. 
### Feld AKL-Nummer 
Hier wird eine Adress- / Kunden- oder Lieferantennummer angezeigt, je nach dem welcher Vorgang zugrunde liegt. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Feld Firma 1 
Die Firmenadresse wird automatisch angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Combobox Partner 
Die Nummer des Kontakts / Ansprechpartners wird angegeben. 

Wenn Sie hier einen Partner auswählen, werden nur Kontakte zu diesem Partner eingelesen. Damit kann im Fall einer großen Anzahl von Kontakten eine schnelle Suche gewährleistet werden. 

Tooltip: 

Nummer des Kontakts/Partners 

### Feld Projekt 
Wenn die Kontakthistorie aus einem Projekt geöffnet wird, wird die Projektnummer hier angezeigt. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden dem Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

### Baumansicht Vorgang 

Hier werden sämtliche Vorgänge aufgelistet, die den Selektionskriterien entsprechen. 

Wenn Sie hier einen Eintrag markieren, werden im rechten Feld die Details dazu angezeigt.

### Registerkarte Belegfolge 

Kontakt-Historie Registerkarte Belegfolge 

Wenn diese Registerkarte Informationen enthält, erkennen Sie das am * neben dem Namen der Registerkarte. Hier wird die Belegfolge zum Vorgang angezeigt, welchen Sie in der Registerkarte Historie zuvor markiert haben. 

Beispiel: 
Kontakt-Historie Registerkarte Belegfolge 

Die Farben der Boxen bedeuten folgendes: 

Dunkelblau: die Box ist ein Druckbeleg 
Orange: die Box ist das aktuelle Element 

Objekte können zur Anzeige der Belegfolge auf die Registerkarte gezogen werden (Drag&Drop). 

Mit Klick auf eine Box kann der Html-Report aufgerufen werden.
