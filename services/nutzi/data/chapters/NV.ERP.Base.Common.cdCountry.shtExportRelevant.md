# Kapitel NV.ERP.Base.Common.cdCountry.shtExportRelevant

Dieses Kennzeichen legt fest, ob das Land für zollrelevante Geschäftsvorfälle im Exportbereich wichtig ist. Dieses Kennzeichen hat nur informativen Charakter. 

Ein AES-Vorgang kann nur für Lieferadressen erfasst werden, wenn diese Checkbox gesetzt ist.
