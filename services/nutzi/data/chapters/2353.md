# Factoring Parameter 
Factoring Parameter 

Dieser Dialog öffnet sich im Navigator >> System >> Finanzbuchhaltung >> Factoring Parameter . Hier definieren Sie grundlegende Parameter für das Factoring. 

Voraussetzung ist allerdings, dass im Dialog Parameter Fibu auf der Registerkarte Debitoren + Kreditoren die Checkbox Factoring aktiv gesetzt ist. 

Siehe Kapitel Parameter Finanzbuchhaltung . 

Über F5 (Neu) können Sie eine Bank neu anlegen und auf der Registerkarte Allgemein die Einstellungen vornehmen. 

Hinweis: Die Dialoge Factoring Parameter und Factoring OP Typen stehen auch im Debitoren Factoring >> Stamm zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Factoring Parameter 
Tooltip: 

In diesem Dialog definieren Sie grundlegende Parameter für das Factoring. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button OP Typen 

Dieser Button öffnet den Dialog Factoring OP Typen , um die entsprechenden Factoring Typen der Factoring Banken zu verwalten. 

Siehe Kapitel Factoring OP Typen .
