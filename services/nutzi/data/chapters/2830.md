# Preisoptimierung Registerkarte Absatz-Trendanalyse 
Preisoptimierung Registerkarte Absatz-Trendanalyse 

Diese Registerkarte zeigt die nach dem Verfahren der Trendanalyse ermittelten Artikelvorschläge. Die Spalte 'Absatz' zeigt eine Balkengraphik der letzten Absatzmengen je Periode oder doppelter Periode (siehe Spalte 'Periodenmulti': 1 = Einzelperiode, 2 = Doppelperiode). 

Es werden auch Artikel angezeigt, für die bereits eine manuelle oder automatische Preisänderung stattgefunden hat. Erkennbar ist dies an dem Datum in der Spalte 'Preisänderung'. Ist die Preisänderung mindestens eine Periode her, kann auch ein Erfolg berechnet und in der Spalte 'Betrag Erfolg' ausgewiesen werden. An der Spalte 'Agent ID' kann erkannt werden, ob es sich um eine manuelle Preisänderung (0) oder einen Agenten (Nummer des Agenten) handelt. 

Markiert man eine Zeile, so wird die Graphik zur schnellen Erkennung der Artikelsituation unterhalb der Tabelle angezeigt. 

Um manuelle Preisänderungen auszuführen, muss eine oder mehrere Zeilen markiert und der Button Preisänderung geklickt werden. Dies ist immer nur für eine Preisgruppe möglich. Bei erneutem Aufruf kann zusätzlich auf eine weitere Preisgruppe gebucht werden usw.
