# Picklisten 
Picklisten 

Picklisten sind List- oder Comboboxen, aus denen Sie einen Eintrag auswählen können. Der Programmteil wird aus dem Menü System >> Allgemein >> Picklisten aufgerufen und ermöglicht die zentrale Verwaltung aller im Programm eNVenta ERP verwendeten Picklisten. Einträge in Picklisten können hier verändert, gelöscht oder neu eingegeben werden. Sie dienen lediglich der Information oder für Auswertungen. 

In der Tabelle auf der rechten Seite werden die bereits angelegten Einträge aufgelistet. Ein Mausklick in eine belegte Zeile im Bereich Einträge setzt den Cursor in die gewählte Zeile. Der bestehende Eintrag kann so verändert werden. Dazu kann ein individueller Text in die Spalte ‚Beschreibung‘ eingetragen werden. Damit haben Sie die Möglichkeit für eine Fremdsprache eine abweichende Bezeichnung einzugeben. Dieser Begriff ist die Grundlage für eine individuelle Übersetzungslogik und muss per Customizing aktiviert werden. 

Dieses Feld muss nicht gefüllt werden. Ist hier nichts eingetragen, wird der Begriff aus der Spalte ‚Bezeichnung‘ in den verschiedenen Dialogen in der Combobox der Pickliste verwendet. 

### Dublettenprüfung für Mengeneinheiten 
Dublettenprüfung in den Picklisten 

Beim Anlegen neuer Einträge für die Mengeneinheiten in den Picklisten gibt es eine Dublettenprüfung. Wird ein neuer Eintrag angelegt, bei dem sich entweder die Bezeichnung oder die Beschreibung überschneiden, erhalten Sie eine Fehlermeldung (siehe oben). Auch ist es nicht erlaubt, dass die Bezeichnung eines Eintrags der Beschreibung eines anderen Datensatzes entspricht. Groß- und Kleinschreibung werden dabei nicht berücksichtigt! 

Für die visuelle Darstellung gilt: hat der mit dem Einheitenname matchende Picklisteneintrag auch einen Wert im Feld Beschreibung , dann wird diese Beschreibung visualisiert, ansonsten wird der Einheitennamen angezeigt (Beispiel: Pickliste: Bezeichnung = „Kt“, Beschreibung = „Karton“). 

Für bereits bestehende ‚Beschreibungs-Dubletten‘ wird ein Updateskript bereitgestellt, welches die gefundenen Dubletten mit dem Suffix „Dub“ markiert. Der Sachbearbeiter hat dann diese Einträge manuell zu korrigieren. Für die ‚Bezeichnungs-Dubletten‘ ist dies nicht möglich, da diese einen Schlüsselwert haben und daher nicht einfach per Update geändert werden dürfen.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Sprachen 

Mit diesem Button können Sie Fremdsprachentexte auswählen.
