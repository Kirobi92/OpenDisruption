# Gutschriftsarten Registerkarte Rollenrechte 
Gutschriftsarten Registerkarte Rollenrechte 

Diese Registerkarte dient dazu, den Gutschriftsarten und damit den Benutzern entsprechende Rollen/Rechte zuzuordnen. Auf diese Weise können Sie einschränken, dass die Benutzer nur bestimmte Gutschriftsarten betrachten und/oder einfügen können. Da jeder Sachbearbeiter einer Rolle zugeteilt ist, wird darüber die Berechtigung der Gutschriftsart zugeteilt, z.B. ist so sichergestellt, dass kein Unbefugter Gutschriften erfassen kann. 

Werden keine Rollenrechte vergeben, sehen alle Sachbearbeiter alle Gutschriften, falls der Dialog zugänglich ist. 

Auch wenn für einen Sachbearbeiter die Rollensperrungen eingerichtet wurden, werden diese Aufträge in den Statistiken berücksichtigt. 

Hat ein Sachbearbeiter mehrere Rollen zugeordnet, so genügt es wenn bereits für eine Rolle der Zugriff gesperrt wurde. 
### Button Neu 

Über diesen Button legen Sie eine neue Zeile an und können anschließend die Rolle auswählen, für welche Sie die Berechtigung einschränken möchten. Dazu wählen Sie aus der Combobox in der Spalte ‚Rolle‘ eine Rolle aus und können dazu die Checkboxen in den Spalten ‚Ansicht sperren‘ und ‚Speichern sperren‘ aktivieren. Es stehen alle Rollen aus dem Authentication Service zur Verfügung. 

### Button Löschen 

Mit diesem Button können Sie eine Rollensperrung wieder löschen. 

### Tabelle Rollenrechte 

### Tabellenspalte Rolle 
Hier geben Sie die Rolle an. 

Tooltip: 

Eine Rolle wird einem Sachbarbeiter zugeordnet. 

### Tabellenspalte Ansicht Sperren 
Wenn die Checkbox in dieser Spalte gesetzt wird, kann der Sachbearbeiter keine Gutschriften dieser Gutschriftsart betrachten. Daher wird auch automatisch die Checkbox in der Spalte 'Speichern Sperren' gesetzt. 

Tooltip: 

Der Sachbearbeiter kann diesen Vorgang nicht sehen. 

### Tabellenspalte Speichern sperren 
Ist die Checkbox in der Spalte 'Speichern Sperren' gesetzt, kann der Sachbearbeiter die Gutschriften dieser Gutschriftsart betrachten, allerdings keinerlei Änderungen vornehmen. 

Tooltip: 

Der Sachbearbeiter kann die Vorgänge ansehen, aber nicht ändern.
