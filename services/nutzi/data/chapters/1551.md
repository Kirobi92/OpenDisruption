# Doppelbelieferungsschutz 
Doppelbelieferungsschutz 

Kommt es bei der Einplanung eines Belegs bzw. bei der Umtourung eines Belegs zu einer Doppellieferung, so öffnet sich automatisch der Dialog Doppelbelieferungsschutz (wenn der Parameter Doppelbelieferungsschutz im Dialog Parameter VS / LO auf der Registerkarte Tour aktiv). 

Hier werden oben auf der linken Seite der aktuelle Stopp und das Lieferdatum angezeigt. 

Die rechte Tabelle zeigt die Touren, welche in mindestens einem angezeigten Beleg verwendet wurden. 

Der Sachbearbeiter hat nun die Möglichkeit einzelne, mehrere oder alle Belege der jeweiligen Tabelle durch drücken des Buttons einer Tour aus der rechten Tabelle zuzuordnen. Dabei wird die Zuordnung auf potentielle Probleme/Fehler überprüft und die Information, analog der Tourendisposition, angezeigt. Das System merkt sich die Zuordnung und trägt sie in die jeweilige Spalte ‚Zieltour‘ der jeweiligen Zeile ein. Dieses Ergebnis wird sich erst mal nur gemerkt. 

Wird der Button gedrückt, wird dieser Merker in den Dialog Tourendisposition übernommen. Erst wenn alle Entscheidungen getroffen sind (akzeptieren von Fehlern bei der Tourenzuordnung, Doppelbelieferungsschutz für alle Stopps), werden die getroffenen Umtourungen auch tatsächlich durchgeführt. 

Mit dem Button kann der Vorgang abgebrochen werden. 

Siehe Kapitel Doppelte Anlieferung ausschließen .
