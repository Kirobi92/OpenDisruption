# Produktionsauftrag Registerkarte Allgemein 
Produktionsauftrag Registerkarte Allgemein 

Auf der Registerkarte Allgemein wird die hierarchische Struktur der Betriebsaufträge dargestellt. LED’s in den Ampelfarben signalisieren den Status des Betriebsauftrags. 

Rot : noch nicht freigegeben oder geparkt 
Gelb : derzeit in Produktion befindlich 
Grün : abgeschlossen 

Auf diese Weise kann der Bearbeitungsstand des gesamten Produktionsvorgangs auf einen Blick erfasst werden. 

Mit einem Klick auf das '+' vor dem Betriebsauftrag kann der komplette Baum des Produktionsauftrags geöffnet werden. 

Mit einem Doppelklick auf einen Betriebsauftrag wird dieser direkt geöffnet. 

Neben der Übersicht der Betriebsaufträge zum Produktionsauftrag sehen Sie alle wichtigen Daten zu den Betriebsaufträgen. Auf diese Weise können Sie schnell sehen, in welchen Betriebsaufträgen Sie eingreifen müssen. 
### Feld Artikel 
Hier steht die Nummer des Artikels aus dem Betriebsauftrag. 

Es ist möglich einen Betriebsauftrag für einen gesperrten Artikel anzulegen. Sie werden mit einer Meldung darauf hingewiesen, dass der Artikel gesperrt ist. 

Tooltip: 

Angabe der Artikelnummer 

### Button Verzweigen 

Dieser Button verzweigt in den Artikelstamm. 

### Button Information 

Mit diesem Button gelangen Sie in den Dialog Artikelinformation . Dort erhalten Sie weitere Informationen zum Artikel. 

Siehe Kapitel Artikelinformation . 

Tooltip: 

Artikelinformation 

### Feld Bezeichnung 
Die Bezeichnung dient der einfachen Identifikation des Artikels und kann auch zur Suche verwendet werden. 

Bei Eingabe der Artikelnummer wird das Feld Bezeichnung automatisch mit dem Namen des Artikels gefüllt. 

Tooltip: 

Artikelbezeichnung 

### Feld Menge bestellt 
Hier wird die bestellte Menge angezeigt. 

Tooltip: 

Angabe der Menge, welche bestellt wurde. 

### Feld Lagereinheit 
Hier wird die Lagereinheit angegeben, d.h. die Einheit in welcher der Artikel auf dem Lager geführt wird. 

Tooltip: 

Angabe der Lagereinheit. Dazugehörige Pickliste heißt „Mengeneinheit“. 

### Feld Menge geliefert 
Hier wird die gelieferte Menge angegeben. D.h. die tatsächlich durch den Lieferscheindruck als gelieferte, im Auftrag eingetragene, Menge. 

Bei teilproduzierten Betriebsaufträgen wird die gelieferte Menge automatisch aktualisiert. 

Tooltip: 

Menge geliefert: 

Die tatsächlich durch den Lieferscheindruck als gelieferte, im Auftrag eingetragene Menge. 

### Feld Status 
Der Status des Betriebsauftrags wird angezeigt. 

Tooltip: 

Angabe des Status 

### Feld Lieferdatum 
Hier wird der Liefertermin angegeben. 

Tooltip: 

Angabe des Liefertermins 

### Feld Startdatum 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld Datum bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein.
