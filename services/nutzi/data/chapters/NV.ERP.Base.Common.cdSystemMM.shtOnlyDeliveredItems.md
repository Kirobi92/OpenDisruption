# Kapitel NV.ERP.Base.Common.cdSystemMM.shtOnlyDeliveredItems

Wenn diese Checkbox gesetzt ist, werden Positionen nur in die Retoure übernommen, wenn sie auch geliefert wurden. Geprüft wird auf die Lieferscheinnummer in den Auftragspositionen.
