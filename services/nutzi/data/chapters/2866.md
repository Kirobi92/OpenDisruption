# Ressource Registerkarte Allgemein 
Ressource Registerkarte Allgemein 

Auf dieser Registerkarte sehen Sie zur ausgewählten Ressource die belegten Zeiten. 

Die Spalte ‚Hierarchie‘ gibt einen Überblick über anstehende Betriebsaufträge und die Ebenen, wie diese abgearbeitet werden. Beginn und Ende des Betriebsauftrags werden daneben angezeigt. 

Wenn Sie mit der rechten Maustaste im „Kalender“ auf der Registerkarte Allgemein einen Doppelklick ausführen, wird ein Dialog mit verschiedenen Auswahlmöglichkeiten angezeigt. 

Folgende Modi stehen zur Verfügung: 

Auswahlmodus : 
Es sind keine Änderungen innerhalb des Gantt-Diagramms möglich. 

Anlegemodus: 
Es ist möglich einzelne Balken zu verschieben, um z.B. eine Terminierung auf den nächsten Tag zu verschieben oder eine neue Reservierung der Ressource anzulegen. 

Gruppe kollabieren: 
Alle Ebenen der Gruppe werden zusammengefasst dargestellt. 

Gruppe expandieren: 
Alle Ebenen der Gruppe werden einzeln dargestellt. 

Tiefer stufen: 
Der markierte Balken bzw. die Reservierung wird eine Ebene nach unten gestuft. 

Höher stufen: 
Der markierte Balken bzw. die Reservierung wird eine Ebene nach oben gestuft. 

Löschen: 
Der markierte Balken bzw. die bestehende Reservierung der Ressource wird gelöscht. 
### Button Aktualisieren 

Mit diesem Button werden alle Daten zur Ressource, die Hierarchie und das Gantt-Diagramm neu eingelesen. 

### Button Heute 

Wenn Sie diesen Button drücken, wird die Anzeige in der Tabelle auf den heutigen Tag eingestellt. 

### Button Woche 

Wenn Sie diesen Button drücken, wird die Anzeige in der Tabelle auf die Woche eingestellt. 

### Button < Monat 

Dieser Button verschiebt den Monat des Startdatums nach hinten. 

Tooltip: 

Vorheriger Monat 

### Button Monat 

Wenn Sie diesen Button drücken, wird die Anzeige in der Tabelle auf den aktuellen Monat eingestellt. 

Tooltip: 

Mit diesem Button wird die Anzeige in der Tabelle auf den aktuellen Monat eingestellt. 

### Button Monat > 

Dieser Button verschiebt den Monat des Startdatums nach vorne. 

Tooltip: 

Nächster Monat 

### Feld Datum/Zeit von 
Für das Gantt-Diagramm geben Sie hier das Start-Datum für die Darstellung an. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Datum/Zeit bis 
Für das Gantt-Diagramm geben Sie hier das Ende-Datum für die Darstellung an. 

Tooltip: 

Eingabe von Datum und Uhrzeit
