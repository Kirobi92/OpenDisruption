# Kapitel NV.ERP.Base.Storage.cdStorageLocation.lngCustomerIDExternal

Hier geben Sie eine externe Kundennummer ein. D.h. eine Kundennummer mit der bei Ihrem IC-/CC-Partner bestellt wird. 

Bei dieser Kundennummer handelt es sich um die Kundennummer aus dem Kundenstamm Ihres IC-/CC-Partners. 

Der IC-/CC-Partner hat Ihnen zwingend mitzuteilen, unter welcher Kundennummer Sie bei ihm bestellen sollen. 

Mit dieser Kundennummer erfolgt die Bestellung beim IC-/CC-Partner. 

Tooltip: 

Mit dieser Kundennummer wird bei Ihrem IC-/CC-Partner bestellt.
