Finanzbuchhaltung / Sicherheiten-Verwaltung / Kundenbonitätsliste 

# Kundenbonitätsliste Registerkarte Selektionen 1 
Kundenbonitätsliste Registerkarte Selektionen 1 

Auf dieser Registerkarte legen Sie die Selektionskriterien für die Anzeige in der Tabelle fest. 
### Radiobutton Kunden 
Wenn Sie diesen Radiobutton aktivieren, erfolgt die Selektion auf Kundennummer. 

Tooltip: 

Selektion nach Kontonummer 

### Radiobutton Bezeichnung 
Wenn Sie diesen Radiobutton aktivieren, erfolgt die Selektion auf Kundenname. 

Tooltip: 

Selektion nach Bezeichnung 

### Radiobutton Suchname 
Wenn Sie diesen Radiobutton aktivieren, erfolgt die Selektion auf Suchname. 

Tooltip: 

Selektion nach Suchname 

### Feld Kunde von 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Konto von 

### Feld Kontobez. von 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Bezeichnung Von 

### Feld Suchname von 
Hier wird der Suchname angegeben. Außerdem kann hier eine Belegnummer zu einem Vorgang eingetragen werden. Wenn der Vorgang existiert, wird der Vorgang geladen. 

Tooltip: 

Suchname von
