# Betriebsauftrag Registerkarte Terminierung 
Betriebsauftrag Registerkarte Terminierung 

Diese Registerkarte ermöglicht Ihnen eine Übersicht über die Zeitplanung der Arbeitsgänge. Die Terminierung wird immer auf Basis der Ressourcenzuordnungen der Arbeitsgänge durchgeführt. Ist einem Arbeitsgang keine Ressource zugeordnet, wird die Standardressource der Produktions-Parameter verwendet. Dabei wird immer der Kalender der Ressource bzw. Ressourcengruppe verwendet. 

Bei der Berechnung der Kapazität wird die Summe der Zeiten aller Reservierungen angezeigt, d.h. auch bei gleichzeitiger Verwendung mehrerer Ressourcen wird die Summe der Ressourcen angezeigt. 

Besteht für einen Arbeitsgang bereits eine Terminierung (Reservierung), dann ist eine Änderung der Arbeitszeit nicht mehr möglich. Sie erhalten eine Meldung, dass noch Reservierungen existieren und deshalb keine Änderung möglich ist. Damit die Arbeitszeit geändert werden kann, ist die Aufhebung der Terminierung über den Button Aufheben vorzunehmen. 

Eine Materialverfügbarkeit wird in der Terminierung nie geprüft. Die Verfügbarkeit liegt in der Verantwortung des Disponenten. 
### Button Vorwärts 

Dieser Button löst eine Vorwärtsterminierung ohne Reservierung des Materials aus. 

Tooltip: 

Terminierung ohne Reservierung 

### Button Rückwärts 

Dieser Button löst eine Rückwärtsterminierung ohne Reservierung des Materials aus. 

Tooltip: 

Terminierung ohne Reservierung 

### Button Res. Vorw. 

Dieser Button löst eine Vorwärtsterminierung mit gleichzeitiger Reservierung des Materials aus. 

Tooltip: 

Terminierung und Reservierung 

### Button Res. Rückw. 

Dieser Button löst eine Rückwärtsterminierung mit gleichzeitiger Reservierung des Materials aus. 

Tooltip: 

Terminierung und Reservierung 

### Button Aufheben 

Dieser Button löscht Terminierung und Reservierung.
