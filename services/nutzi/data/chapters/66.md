# Fokus-Element Information 

## Verbindungen 

### Inhalt 
Die Daten des zuletzt aufgerufenen Vorgangs- oder Stammdatensatzes werden analysiert und die Verbindungen zu Stammdaten, Vorgängen und Dokumenten erkannt und aufgelistet. 
### Ziel / Nutzen 
Das Element bietet Zusatzinformationen zu dem aktuellen Vorgang. Es kann ohne Aufruf von weiteren Dialogen erkannt werden, ob zu dem aktuellen Vorgang gedruckte Dokumente oder Referenz-Links vorhanden sind. Ein direktes Aufrufen verbundener Objekte per Doppelklick ist möglich. 
### Darstellung 
Liste der verbundenen Objekte gruppiert in den Kategorien Stammdaten, Vorgänge, Dokumente, Referenz-Links, Herkunft (Quelle), Verwendung (Ziel) 
### Parameter 
Es kann festgelegt werden, welche Kategorien von Verbindungen dargestellt werden: 
- Verbindungen (Stammdaten, Vorgänge) - Dokumente (gedruckte Belege zu Auftrag/Angebot/Gutschrift, Anfrage/Bestellung/Belastung oder Betriebsauftrag) - Referenz-Links - Herkunft / Verwendung (durch Kopieren von Auftrag/Angebot/Gutschrift, Anfrage/Bestellung/Belastung) 
### Drill Down 

Aufruf des Vorgangs/Dokuments 
### Aktualisierung 
Automatisch beim Einlesen eines neuen Vorgangs oder beim Blättern in Vorgängen. Alternativ kann auch ein Vorgang/Dokument per Drag&Drop auf das Element gezogen werden.BerechtigungZugangsberechtigt sind alle Benutzer, die auch das Recht für den Dialog Fokus Verbindungen (frmDashItemObjectRelations) besitzen. 

## Wiedervorlagen 

### Inhalt 
Es werden die letzten Wiedervorlagen zu dem zuletzt aufgerufenen Vorgangs- oder Stammdatensatz aufgelistet. 
### Ziel / Nutzen 
Es kann auf einen Blick und ohne weitere Mausklicks erkannt werden, ob es (offene) Wiedervorlagen zu diesem Vorgang/Objekt gibt und wen diese betreffen. 
### Darstellung 
Liste der Wiedervorlagen chronologisch absteigend. 
### Parameter 
Die maximale Anzahl Wiedervorlagen kann vorgegeben werden. Bei fehlender Angabe wird 10 angenommen. 
### Drill Down 
keiner 
### Aktualisierung 
Automatisch beim Einlesen eines neuen Vorgangs oder beim Blättern in Vorgängen. Alternativ kann auch ein Vorgang/Dokument per Drag&Drop auf das Element gezogen werden. 
### Berechtigung 
Zugangsberechtigt sind alle Benutzer, die auch das Recht für den Dialog Fokus Wiedervorlagen (frmDashItemReminder) besitzen. 

## Letzte Kontakte 

### Inhalt 
Für den zuletzt aufgerufenen Vorgang/Objekt wird ermittelt, um welchen Kunden, Lieferanten oder Interessenten (Adresse) es sich handelt. Das Element zeigt die letzten Kontakte der Kontakthistorie. 
### Ziel / Nutzen 
Es kann auf einen Blick und ohne weitere Mausklicks erkannt werden, ob, wann und mit wem die letzten Kontakte zu diesem Kunden/Lieferanten/Adresse stattgefunden haben. Der direkte Aufruf per Doppelklick ist möglich. 
### Darstellung 
Liste der letzten Kontakte chronologisch absteigend. 
### Parameter 
Die maximale Anzahl letzter Kontakte kann vorgegeben werden. Bei fehlender Angabe wird 10 angenommen. 
### Drill Down 
Aufruf des Vorgangs/Dokuments 
### Aktualisierung 
Automatisch beim Einlesen eines neuen Vorgangs oder beim Blättern in Vorgängen. Alternativ kann auch ein Vorgang/Dokument per Drag&Drop auf das Element gezogen werden. 
### Berechtigung 
Zugangsberechtigt sind alle Benutzer, die auch das Recht für den Dialog Fokus Kontakte (frmDashItemContact) besitzen. 

## Änderungen 

### Inhalt 
Für den zuletzt aufgerufenen Vorgang/Objekt wird die Änderungshistorie ermittelt. Das Element zeigt eine chronologische Liste der letzten Änderungen. Voraussetzung ist, dass für die Objektart die Protokollierung eingeschaltet wurde. 
(Menü System >> Konfiguration-Protokoll ) 
### Ziel / Nutzen 
Es kann auf einen Blick und ohne weitere Mausklicks erkannt werden, ob, wann und von wem die letzten Änderungen gemacht worden sind. 
### Darstellung 
Liste der letzten Änderungen chronologisch absteigend. 
### Parameter 
Die maximale Anzahl letzter Änderungen kann vorgegeben werden. Bei fehlender Angabe wird 50 angenommen. 
### Drill Down 
keiner 
### Aktualisierung 
Automatisch beim Einlesen eines neuen Vorgangs oder beim Blättern in Vorgängen. Alternativ kann auch ein Vorgang/Dokument per Drag&Drop auf das Element gezogen werden. 
### Berechtigung 
Zugangsberechtigt sind alle Benutzer, die auch das Recht für den Dialog Änderungsprotokoll (frmActivityLog) besitzen. 

## Vereinbarungen 

### Inhalt 
Gibt es zu dem zuletzt aufgerufenen Vorgang oder Stammdatensatz einen Kunden, so werden zu diesem Kunden optional folgende Vereinbarungen aufgelistet 

- offene Projekte - laufende Rahmenaufträge - laufende Serviceverträge - 
### Ziel / Nutzen 
Das Element bietet Zusatzinformationen zu dem aktuellen Kunden. Es kann ohne Aufruf von weiteren Dialogen erkannt werden, ob entsprechende Vereinbarungen vorhanden sind. Ein direktes Aufrufen per Doppelklick ist möglich. 
### Darstellung 
Liste der Vereinbarungen gruppiert in den Kategorien Projekte, Rahmenaufträge, Serviceverträge, Vermietungen. 
### Parameter 
Es kann per Checkboxen festgelegt werden, welche Kategorien von Vereinbarungen dargestellt werden sollen. 
### Drill Down 
Aufruf des Vorgangs/Dokuments 
### Aktualisierung 
Automatisch beim Einlesen eines neuen Vorgangs oder beim Blättern in Vorgängen. Alternativ kann auch ein Vorgang/Dokument per Drag&Drop auf das Element gezogen werden. 
### Berechtigung 
Zugangsberechtigt sind alle Benutzer, die auch das Recht für den Dialog Fokus Vereinbarungen (frmDashItemCustomerRelations) besitzen. 

## Zusatzfelder 

### Inhalt 
In diesem Element werden die Zusatzfelder des aktuellen Objektes angezeigt. Via Drag & Drop lassen sich ebenfalls Objekte hineinziehen. 
### Ziel / Nutzen 
Es können auf einen Blick die Zusatzfelder eingesehen werden, ohne weitere Detail-Dialoge öffnen zu müssen. 
### Darstellung 
Tabelle der Zusatzfelder mit Eigenschaft und Wert 
### Parameter 
Mittels des Parameters lässt sich die Ansicht soweit manipulieren, dass entweder nur die mit Werten bestückten oder alle definierten Zusatzfelder des Objektes angezeigt werden. 
### Drill Down 
keiner 
### Aktualisierung 
Automatisch beim Einlesen eines neuen Vorgangs oder beim Blättern in Vorgängen. Alternativ kann auch ein Vorgang/Dokument per Drag&Drop auf das Element gezogen werden. 
### Berechtigung 
Zugangsberechtigt sind alle Benutzer, die auch das Recht für den Dialog Zusatzfelder Definieren (frmAddFieldDefDialog) besitzen.
