# Kapitel NV.ERP.MM.CashDesk.cdCashDeskParameters.shtBackInCustCurrency

Ist diese Checkbox gesetzt, kann das Rückgeld in Fremdwährung ausbezahlt werden. Diese Fremdwährung kann eine beliebige Währung sein, wird aber in der Regel die Währung des Kunden sein. Die Checkbox kann nicht gesetzt werden, wenn der Parameter Nur in Systemwährung kassieren gesetzt ist. In diesem Fall kann in beide Richtungen nur in der Systemwährung kassiert werden
