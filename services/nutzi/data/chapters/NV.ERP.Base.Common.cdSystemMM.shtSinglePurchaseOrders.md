# Kapitel NV.ERP.Base.Common.cdSystemMM.shtSinglePurchaseOrders

Ist dieser Parameter gesetzt, wird beim Verarbeiten des Dispo-Pools oder aus dem Bestellautomat, bei auftragsbezogenen Einträgen pro Auftrag eine Bestellung generiert. Ist der Parameter nicht gesetzt, wird pro Lieferant und Lager eine Bestellung generiert. 

Tooltip: 

Ist dieser Parameter gesetzt, wird beim Verarbeiten des Dispo-Pools bei auftragsbezogenen Einträgen pro Auftrag eine Bestellung generiert. 
Ist der Parameter nicht gesetzt, wird pro Lieferant eine Bestellung generiert.
