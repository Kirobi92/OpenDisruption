# Kapitel NV.ERP.Base.Common.cdShippingCondition.shtNV4_DirectPushPickOrder

Wenn Sie diesen Parameter aktivieren, kann bei dieser Versandart die Ware direkt vom Lager entnommen werden. 

Tooltip: 

Wenn aktiviert, wird die Ware direkt vom Lager entnommen.
