# Kapitel NV.ERP.Base.Common.cdSystemMM.shtBatchesPerGoodsReceiptItem

Dieser Schalter ermöglicht zwei verschiedene Verfahren der Chargenerfassung beim Wareneingang: 

aus: Es wird nur einmal je Wareneingang eine Chargennummer eingegeben, sie gilt dann für alle Positionen (optimal für eine individuelle Vergabe der Chargennummer). 

ein: Für jede Wareneingangsposition müssen eine oder mehrere Chargennummern erfasst werden. Dies ist vor allem für die Erfassung von Fremdchargennummern notwendig.
