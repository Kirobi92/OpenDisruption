# Spezifische Umsatzsteuer Registerkarte Allgemein 
Umsatzsteuer Spezifisch Registerkarte Allgemein 

Auf dieser Registerkarte können Sie die spezifische Umsatzsteuer im Einzelnen definieren. 
### Combobox StSchl. VK 
Hier wählen Sie den Steuerschlüssel für Verkauf aus. 

Tooltip: 

Steuerschlüssel Verkauf

Hier geben Sie das Erlöskonto für den Verkauf ein. 

Tooltip:
