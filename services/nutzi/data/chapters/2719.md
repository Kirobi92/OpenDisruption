# Dimensionsfilter 
Kore Dimensionsfilter 

Dieser Dialog öffnet sich im Buchen Kore >> Stamm >> Dimensionensfilter . 

Über den Dimensionsfilter können in den diversen Auswertungsdialogen zusätzliche Suchkriterien gesteuert werden. Hierbei wird aus den verschiedenen Dimensionen jeweils eine Suchbedingung erzeugt, welche dann in einer Combobox zur Verfügung gestellt wird. 

Über die Radiobuttons eigene, andere, alle steuern Sie, ob nur Ihre eigenen Dimensionsfilter, die anderer Benutzer oder alle angelegten Filter in der Tabelle angezeigt werden sollen. 

Für jede aktive Dimension können Sie nun einen Start- und Zielwert für die Auswertung eintragen. Ist kein “einfacher” Start-/Zielwert möglich, nutzen Sie den Button SQL um eine SQL-Bedingung einzustellen, diese wird dann in der Spalte “SQL-Statement” angezeigt. 

Siehe Kapitel eNVenta ERP Allgemeine Funktionalitäten / Suchen mit SQL . 

Über die Checkbox Aktiv in der letzten Spalte steuern Sie, ob der Dimensionsfilter in den Auswertungsdialogen zur Verfügung steht.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button SQL 

Über diesen Button starten Sie eine SQL-Abfrage.
