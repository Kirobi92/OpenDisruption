# Kapitel NV.ERP.Base.Customer.cdCustomer.lngIln

Die International Location Number (für EDI / Gateway) wird eingegeben. Diese Nummer dient der Identifikation zwischen Geschäftspartnern. 

EDI bedeutet: Electronic Data Interchange. Damit wird der elektronische Nachrichtenaustausch zwischen Unternehmen in allen Bereichen vereinfacht. Verfügt der Kunde über eine EDI-ILNr, hinterlegen Sie diese hier, um die EDI-Programmerweiterungen nutzen zu können. 

Die ILN muss eindeutig sein! 

Das Zusatzmodul EDI / GATEWAY gehört nicht zum Standard-Programmumfang.
