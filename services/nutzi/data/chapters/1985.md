# Materialumbuchung 
Materialumbuchung 

Um mehr Transparenz bei der Disposition im Betriebsauftrag zu erzielen, steht der Button Umbuchung auf der Registerkarte Positionen zur Verfügung. Mit diesem Button öffnet sich zu einer Zeile der Tabelle der Dialog Materialumbuchung . 

Wenn mit einem separaten Produktionslager (und mit Umlagerungen) gearbeitet wird und der Betriebsauftrag NICHT zu 100% abgeschlossen wird (Unter- bzw. Überproduktion), dann ist es mit dem Dialog Materialumbuchung bei einer Überproduktion einfach möglich, die zusätzlichen Mengen (Stücklistenkomponenten) auf das Produktionslager zu buchen. Bei einer Unterproduktion ebenso: die übrig gebliebenen Restmengen im Produktionslager werden ebenso einfach wieder vom Produktionslager zurück in das ursprüngliche Lager gebucht. 

Der Dialog kann auch nach Abschluss des Betriebsauftrags noch aufgerufen werden. 

Dieser Button ist nur aktiv, wenn das Produktionslager vom Materiallager abweicht. 

Der Dialog Materialumbuchung listet evtl. durchgeführte Umlagerungen auf. Diese Auflistung erfolgt pro Artikelposition. 

Dieser Dialog bietet ebenfalls die Möglichkeit, im Fall von Über- oder Unterproduktion das benötigte Material bereitzustellen bzw. überschüssiges Material wieder auf das Materiallager zurückzubuchen. 

Der Dialog kann auch noch nach Abschluss eines Betriebsauftrags aufgerufen werden. 

Unterhalb der Tabelle werden zur markieren Position der Bestand und die Verfügbarkeit auf dem Produktionslager bzw. dem Materiallager angezeigt. Die Positionen werden farblich markiert.

Tooltip:

Mit diesem Button löschen Sie einzelne Positionen.

### Button Charge 

Mit diesem Button können Sie eine Charge auswählen und zuordnen. Ein Splitt ist nicht möglich! 

### Button Plätze 

Mit diesem Button können Sie eine Platzzuordnung vom chaotischen Materiallager vornehmen. Bei Rückbuchung auf ein chaotisches Materiallager wird der Platz automatisch zugewiesen. 

### Button Umbuchen 

Dieser Button startet alle Umbuchungen, anschließend wird der Dialog geschlossen. 

Tooltip: 

Umbuchungen durchführen 

### Feld Betriebsauftrag 
Hier wird der zugrundeliegende Betriebsauftrag angezeigt. 

Tooltip: 

Angabe der Betriebsbauftragsnr. 

### Feld Menge 
Hier geben Sie die Produktionsmenge ein. Bei negativer Menge (Unterproduktion) vertauschen sich die Lager (Rückbuchung). Eine Eingabe/Änderung der Menge führt automatisch zu einer neuen Berechnung der Tabellenwerte. 

Um eine Fehlbedienung zu vermeiden, wird je nach positiver oder negativer Menge der Buchungstyp (Materialbereitstellung, Rückbuchung) angezeigt. (rechts daneben) 

Tooltip: 

Überproduktion: positive Menge 
Unterproduktion: negative Menge 

### Tabelle 

In der Tabelle wird die benötigte Menge des Materials je Position vorgeschlagen. Einzelne Positionen können gelöscht werden (Mehrfachselektion), Mengen sind änderbar. Dies gilt auch für Baugruppen.
