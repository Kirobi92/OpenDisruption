# Kapitel NV.ERP.MM.Sales.cdOrderType.shtInvoiceReleaseOnCustomerOrder

Wenn Sie diesen Parameter aktivieren, wird mit Freigabe der Rechnung einer auftragsbezogenen Bestellung die Position des Auftrags nicht nur automatisch auf Status 2 (Lieferfreigabe), sondern gleich auf Status 3 (Fakturafreigabe) gesetzt. 

Tooltip:
