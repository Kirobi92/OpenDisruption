# Kapitel NV.ERP.MM.Sales.cdOrderTypeExtended.shtValidityPeriod

Die Bindefrist, die Sie hier einstellen, hat Vorrang vor dem Standardwert aus den Parametern Verkauf. Diese Bindefrist kann manuell angepasst werden. 

Die Bindefrist wird in den einzelnen Beleg übernommen und kann dort ebenfalls wieder überschrieben werden, z.B. bei Sondervereinbarung mit dem Kunden. 

Beim Kopieren eines Angebots in ein anderes Angebot oder einen Auftrag bleibt die Bindefrist innerhalb derselben Angebotsart erhalten. Wird eine andere Angebotsart in der Zielkopie verwendet, gelten die dort hinterlegten Bindefristen. 

Tooltip: 

Hier können Sie eine belegspezifische Bindefrist hinterlegen. 
Diese überschreibt die in den Verkaufsparametern 
gesetzte "Standard-Bindefrist".
