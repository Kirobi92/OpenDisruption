# Kapitel NV.ERP.Base.Common.cdSystemMM.sAddAttachmentNameI

Hier geben Sie die Formel zur Ermittlung des Dateinamens ein bei der Generierung einer E-Mail für den Rechnungsdruck. Es wird immer fix der Wert "Rechnung" verwendet, gefolgt von der Formel als Dateiname.
