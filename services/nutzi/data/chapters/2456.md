# Verbandsabrechnung Kreditor Registerkarte Verbandsabrechnung 
Registerkarte Verbandsabrechnung 

Für eine Selektion auf Verband können Sie hier die entsprechenden Selektionskriterien eingeben. 
### Feld Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Dieser Button öffnet den Dialog Kontoauswahl , um komfortabel das gesuchte Konto zu wählen. 

Siehe Kapitel Kontoauswahl . 

### Feld Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Feld Verband 
Gehört der Lieferant zu einem Verband und soll die Zahlung der offenen Posten über diesen Verband erfolgen, bzw. die Verbandsanschrift auf der Rechnung gedruckt werden (wenn dies im Report entsprechend eingestellt ist), muss hier die Lieferantennummer des Verbands eingetragen werden. 

Im Lieferantenstamm wird im Feld Verband immer der zum Tagesdatum passende Verband gespeichert. Der Button neben dem Feld Verband öffnet den Dialog Verbandshistorie . Damit kann eine Historie auf Verbände zur Verfügung gestellt werden. 

Siehe Kapitel Verbandshistorie . 

Tooltip: 

Kontonummer des Verbands 

### Button Suche 

Dieser Button öffnet den Dialog Kontoauswahl , um komfortabel das gesuchte Konto zu wählen. 

Siehe Kapitel Kontoauswahl . 

### Checkbox Alle 
Wenn Sie diese Checkbox aktivieren, wird der Lauf für alle Verbände gestartet (Lieferanten, die einen Eintrag im Lieferantenstamm, Feld Verband haben). 

Tooltip: 

Alle Abrechn.konten 

### Feld Zentralzahler 
Hier wird eine Zentralzahler-Nummer angegeben. 

Tooltip: 

Kontonummer des Zentralzahlers 

### Button Suche 

Dieser Button öffnet den Dialog Kontoauswahl , um komfortabel das gesuchte Konto zu wählen. 

Siehe Kapitel Kontoauswahl . 

### Checkbox Alle 
Wenn Sie diese Checkbox aktivieren, wird der Lauf für alle Zentralzahler gestartet (Lieferanten, die einen Eintrag im Lieferantenstamm, Feld Zentralzahler haben). 

Eine automatische Erhöhung der Nettofälligkeit ist hier nicht möglich. 

Tooltip: 

Alle Zentralzahler 

### Combobox Niederlassung 
Hier wird die Niederlassungsnummer angegeben. 

Tooltip: 

Angabe der Niederlassung 

### Checkbox Lese CSV 
Diese Checkbox sowie die zugehörige Auswahlbox für die einzulesende Datei, ermöglichen Ihnen, einen Abrechnungsvorschlag über eine vom Verband/Zentralzahler erhaltene Datei zu erzeugen. 

Siehe Kapitel Lese CSV . 

### Checkbox Belegnummer numerisch 
Wenn Sie diese Checkbox aktivieren, werden führende Buchstaben und Nullen aus der Belegnummer entfernt. 

Tooltip: 

Führende Buchstaben und Nullen aus der Belegnummer entfernen. 

### Checkbox Prüfe Belegnummer 2 
Diese Checkbox wird nur sichtbar, wenn die Checkbox Lese CSV aktiviert ist! Dazu muss in den Parametern der Fibu, Registerkarte Debitoren+Kreditoren die Checkbox Belegnr 2 für Debitoren bzw. Belegnr. 2 für Kreditoren gesetzt sein. 

Wenn Sie diese Checkbox aktivieren, wird auch die Belegnummer 2 geprüft.
