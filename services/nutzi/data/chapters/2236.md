# Parameter Webshop Registerkarte E-Mail 
Parameter Webshop Registerkarte E-Mail 

Hier werden alle grundlegenden Einstellungen getroffen, um den SMTP-Dienst (http://de.wikipedia.org/wiki/Simple_Mail_Transfer_Protocol) für den Versand von E-Mails zu nutzen. Die Einstellung auf dieser Registerkarte wird für alle Arten von E-Mails verwendet, welche vom Webshop generiert werden.
