# Call erzeugen 
Call erzeugen 

Dieser Dialog öffnet sich in Auftrag, Angebot und Gutschrift über Optionen >> Generieren >> Call erzeugen . 

Wenn Sie diesen Dialog öffnen, sind die wichtigsten Daten des aktuellen Vorgangs voreingestellt, wie z.B. die Kundennummer und Projektnummer. Diese Informationen können nicht geändert werden. Jedoch ist es möglich, z.B. den Ansprechpartner, die Kategorie oder die Call-Art zu wählen. 

Damit kann der Innendienst-Sachbearbeiter auf einfachem, schnellem Weg dem Aussendienst-Mitarbeiter eine Mitteilung machen, die dieser im Rahmen der Besuchsberichte wiedererhält. 

Der erzeugte Call wird in den Verlauf eingetragen und kann bei Bedarf darüber aufgerufen werden. Es wird keine Wiedervorlage erzeugt! 
## Steuerelemente des Formulars

Tooltip: 

In diesem Dialog kann der Innendienst-Sachbearbeiter auf einfachem, schnellem Weg dem Aussendienst-Mitarbeiter eine Mitteilung machen, die dieser im Rahmen der Besuchsberichte wieder erhält. 
Für mehr Informationen drücken Sie F1.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern!

### Feld Call-Betreff 
In diesem Feld kann ein Betreff für den Call gepflegt werden. Dies erleichtert die Suche nach einem speziellen Call. 

Diese Betreff-Zeile wird auch im Navigator auf der Registerkarte Termine eingeblendet, wenn auf der Registerkarte Allgemein ein Eintrag erfolgte. 

Tooltip: 

Angabe des Call-Betreffs (worum gehts?)

Nach Drücken des Buttons Neues Th. oder Neue Antw. erfassen Sie in diesem Feld den Text. 

Dieses Feld steht auch zur Verfügung, um einen selektierten Eintrag der Tabelle Übersicht zu bearbeiten.
