# Kapitel NV.ERP.Base.Common.cdNV4_Param.shtCrossDocking

Wenn Sie diese Checkbox aktivieren, wird beim Wareneingang der auftragsbezogen bestellten Ware der Cross Docking Prozess initialisiert. D.h. die Ware wird direkt in den Warenausgang gebracht. 

Beim I-Punkt-Verfahren wird sie direkt ins Tourenfach gelegt. Beim Packplatz wird die Ware in den Packbereich gestellt, damit es zusammen mit der restlichen Ware verpackt werden kann. 

Tooltip: 

Ist dieser Parameter aktiviert, so kann in der Einlagerung die Ware direkt in den Warenausgang verschoben werden.
