# Kapitel NV.ERP.Base.Common.cdSystemMM.shtPrintPackingLists

Mit diesem Parameter kann festgelegt werden, dass nach dem Druck der Pickliste aus dem Vorschlag auch gleich die Packlisten, der im Pickpool selektierten Aufträge, gedruckt werden.
