# CRM Übersicht Registerkarte Projekte 
CRM Übersicht Registerkarte Projekte 

Auf dieser Registerkarte werden je nach Selektion die entsprechenden Vorgänge zu einem Projekt angezeigt. 
### Liste Projektstatus 
Hier stehen die Projektstatus 

- Offen - Erledigt 
zur Auswahl zur Verfügung. 

Tooltip: 

Angabe des Positionsstatus 

### Checkbox Drucken 
Wenn Sie diese Checkbox aktivieren, werden die Informationen gedruckt. 

### Tabelle 

### Tabellenspalte Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Tabellenspalte Projektname 
Hier wird der Projektname angegeben. 

Tooltip: 

Name des Projekts 

### Tabellenspalte Projektart 
Hier wählen Sie die Projektart aus. 

Tooltip: 

Auswahl der Projektart 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Erfassung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der Erfassung eingetragen. 

Tooltip: 

Datum und Uhrzeit der Erfassung 

### Tabellenspalte Änderung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der letzten Änderung eingetragen. 

Tooltip: 

Datum und Uhrzeit der letzten Änderung 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Top Projekt 
Wenn Sie diese Checkbox aktivieren, wird in nachfolgenden Vorgängen nach Top-Projekten selektiert: 

- Projektänderung - Angebot, Auftrag, Gutschrift - Kondition - Bestellung, Anfrage, Belastung - Rahmenauftrag - Call - Besuchsbericht - Projektaufgabe - Service - Wiedervorlage zum Projekt 

Tooltip: 

Aktivitäten verfolgen in: 
- Projektänderung 
- Angebot / Auftrag / Gutschrift 
- Kondition 
- Bestellung / Anfrage / Belastung 
- Rahmenauftrag 
- Call 
- Besuchsbericht 
- Projektaufgabe 
- Service 
- Wiedervorlage zum Projekt
