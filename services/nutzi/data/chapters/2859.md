# Kalender 
Kalender 

Dieser Dialog öffnet sich im Betriebskalender >> Stamm >> Kalender . 

Das Menü Kalender zeigt Ihnen einerseits die bereits angelegten Kalender an und bietet andererseits die Möglichkeit einen neuen Kalender anzulegen oder zu kopieren. Es muss wenigstens ein Kalender angelegt werden. 

Ein Kalender kann nur gelöscht werden, wenn keine Schichten, Pausen etc. dazu angelegt sind.

### Button Kopieren 

Mit diesem Button können Sie einen Kalender kopieren. Es öffnet sich der Dialog Kalender kopieren . Damit vereinfachen sich die folgenden Einstellungen: 

- Übernahme eines Kalenders für ein neues Jahr - Kopieren eines Kalenders als Basis für einen neu zu erstellenden Kalender 
Im 1. Fall muss nur ein neues Jahr angegeben werden. Die Kalendernummer bleibt gleich. 

Im 2. Fall wird die Kalendernummer geleert (das System vergibt automatisch eine neue Nummer) oder eine nicht benutzte Nummer wird vorgegeben. Optional kann auch noch das Jahr geändert werden. 

Eine weitere Option erlaubt es zu unterscheiden, ob die Ressourcen- oder Ressourcengruppen-abhängigen Datensätze (abweichende Kalendertage oder abweichende Pausen) ebenfalls mit kopiert werden sollen. 

Geben Sie dazu in die Spalte ‚Kalendernr.‘ die Nummer des neuen Kalenders ein. In die Spalte ‚Jahr‘ tragen Sie das Jahr ein, für den der Kalender gelten soll. In der Spalte ‚Kalender‘ weisen Sie dem Kalender einen Namen zu. Die Spalten ‚Erfassung‘ und ‚Sachbearbeiter‘ werden mit dem Erfassungsdatum und dem Sachbearbeiter für den Kalender automatisch belegt. 

Siehe Kapitel Kalender kopieren .
