# Kundeninformation 
Kundeninformation 

Dieser Dialog öffnet sich in Auftrag und Kundenstamm über den Button neben dem Feld Suchname bzw. Kunde . In diesem Dialog erhalten Sie detaillierte Informationen zum Kunden, z.B. seine Kontaktdaten, welche Artikel er am häufigsten kauft, welchen Umsatz der Kunde macht. Mit diesen Informationen kann dann u.a. der Vertrieb aktiv werden, um z.B. dem Kunden bei einem Besuch Alternativartikel oder mögliche Sortimentserweiterungen (Zubehör-Artikel) vorzustellen oder ihn evtl. für eine Messe/Ausstellung mit ins Boot zu holen. 

Auf den einzelnen Registerkarten erhalten Sie detaillierte Informationen zum Kunden, aufgeteilt nach Kontaktdaten, Verkaufs-, Einkauf- und Umsatzdaten.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

In diesem Dialog bezieht sich die Auswahl in diesem Feld nur auf das Einlesen von Einträgen in den Registerkarten Verkauf und Einkauf . Die Anzahl bezieht sich hierbei immer auf die jeweilige Belegart, z.B. 2 Aufträge, 2 Angebote. 

Voraussetzung: eine der Checkboxen auf der Registerkarte ist aktiviert! 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
