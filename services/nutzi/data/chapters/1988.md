# BDE Buchung Registerkarte Zeit 
BDE Buchung Registerkarte Zeit 

Auf dieser Registerkarte werden alle Arbeitszeiten bzw. Produktionszeiten erfasst. Hierbei wird zwischen Rüstzeit, Bearbeitungszeit und Nachbearbeitung unterschieden. 

Jedes Ereignis wird mit Personalnummer, Zeit und Grund in einem BDE-Journal festgehalten. 

Das Buchen von Rüstzeiten ist optional, d.h. die Rüstzeit kann auch unter Bearbeitungszeit verbucht werden, um den BDE-Aufwand zu reduzieren. 

Nachbearbeitungen können jederzeit nach Abschluss der Bearbeitung stattfinden. Dies verändert die Nachkalkulation des Betriebsauftrags. 

Durch die laufende Anpassung der Reservierungen und Status durch BDE-Buchungen kann der Fertigungsdisponent zeitnah den Fertigungsstatus und –Fortschritt sowohl im Betriebsauftrag als auch im graphischen Leitstand verfolgen. 

Für einen besseren Überblick erfolgt eine Prüfung auf offene Arbeitsgänge, wenn ein Mitarbeiter einen neuen Arbeitsgang beginnen will. D.h. wenn ein Mitarbeiter einen neuen Arbeitsgang starten will (über den Button Rüstbeginn , Bearb. Beginn bzw. Nachbearb .) und er ist noch in einem anderen Arbeitsgang aktiv, erscheint eine Meldung, die ihn darauf hinweist und fragt, ob er diesen neuen Arbeitsgang wirklich starten will. Außerdem werden alle offenen Arbeitsgänge angezeigt. 

Die jeweiligen Start- bzw. Endtermine werden gespeichert. So ist eine Differenzierung für die jeweiligen Arbeitsgangzeiten möglich. 

Das Arbeiten an einem 2. Arbeitsgang wird nicht verhindert! 
### Button Rüstbeginn 

Mit diesem Button wird der Beginn der Arbeitsvorbereitung gebucht. Diese Rüstzeit wird in der Regel einmalig gebucht. Es ist keine zwingende Buchung einer Rüstzeit notwendig. 

### Button Rüstende 

Mit diesem Button wird das Ende der Arbeitsvorbereitung gebucht. Der Button ist nur zugänglich wenn Rüstbeginn gebucht wurde. 

### Button Bearb.Beginn 

Mit diesem Button wird der Beginn der Produktion bzw. des Arbeitsgangs gebucht. Die Bearbeitung kann beliebig oft erfolgen, für die Kostenberechnung wird die Gesamtzeit errechnet. 

### Button Bearb.Ende 

Mit diesem Button wird das Ende der Produktion bzw. des Arbeitsgangs gebucht. 

Plausibilitätsprüfung: 

Beim Buchen über den Button Bearb. Ende wird geprüft, ob die gemeldete Menge mit der Menge des Vorgängers übereinstimmt. Bei einer Differenz erfolgt eine Abfrage: 

„Die gemeldete Menge weicht von der Menge des Vorgängers ab. Soll trotzdem gebucht werden? JA / NEIN". 

Bei JA , wird die Buchung ausgeführt, bei NEIN kann erneut die Menge eingegeben werden. 

### Button Unterbrechen 

Mit diesem Button wird eine Unterbrechung der Produktion bzw. des Arbeitsgangs gebucht. Der Button ist nur während einer laufenden Produktion zugänglich. Nach der Unterbrechung kann der Arbeitsgang direkt beendet werden (Button Bearb. Ende ). 

### Button Fortsetzen 

Mit diesem Button kann ein Arbeitsgang weiterbearbeitet werden. 

### Button Nachbearb. 

Mit diesem Button wird eine Nachbearbeitung eines Arbeitsgangs gestartet. Ein Arbeitsgang kann beliebig oft nachbearbeitet werden, für die Kostenberechnung wird die Gesamtzeit errechnet. Eine Nachbearbeitung kann nur nach erfolgter Bearbeitung gebucht werden. 

### Button Nachb. Ende 

Mit diesem Button wird das Ende der Nachbearbeitung gebucht. 

### Feld Datum/Zeit 
Hier wird das Datum der Buchung angegeben. Beim Öffnen des Dialogs wird das Feld mit der aktuellen Uhrzeit vorbelegt. Dieser Wert kann überschrieben werden, um vergessene Buchungen nachzuholen. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld Zeit 
Hier wird die aktuelle Uhrzeit vorbelegt. 

### Feld Personalnummer 
In diesem Feld wird der Sachbearbeiter angegeben, welcher die Buchung durchführt. 

Tooltip: 

Angabe der Personalnummer 

### Feld Grund 
Hier können Kommentare zur Buchung angegeben werden, z.B. bei der Buchung einer Unterbrechung. 

### Feld Menge gut 
In dieses Feld geben Sie die "Gut"-Stückzahl ein.

Akzeptierte Menge aus der Bestellposition / Bestellpositionen 

### Feld Ausschuss 
Hier geben Sie die Ausschussmenge ein. 

Tooltip: 

Angabe der Ausschussmenge

Hier werden ein Datum und eine Uhrzeit angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Vorg.gemeldet 
Hier wird die gemeldete Gut-Stückzahl des vorangegangenen Arbeitsgangs mit der aktivierten Checkbox PM erfassen (aus den Dialogen Arbeitsgangstamm und Arbeitsgang ändern ) eingetragen. 

Tooltip: 

Vorgänger gemeldet 

### Feld Bereits gemeldet 
Hier wird die Summe der gemeldeten Mengen des Arbeitsganges gemeldet. Und zwar Gutstückzahlen und Ausschuss. 

Eine Meldung der Menge kann wie folgt erfolgen: 

Buchungsart Menge gut Menge Ausschuss

| --- | --- | --- |
| Rüsten unterbrechen | - | x |
| Rüsten Ende | - | x |
| Bearbeitung unterbrechen | x | x |
| Bearbeitung Ende | x | x |
| Nacharbeit unterbrechen | - | - |
| Nacharbeit Ende | - | - |

Angabe der Menge
