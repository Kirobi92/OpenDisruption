# Kundeninformation Registerkarte Zahlungsverhalten 
Kundeninformation Registerkarte Zahlungsverhalten 

Diese Registerkarte bietet dem Sachbearbeiter eine schnelle Übersicht, wie das Zahlverhalten des selektierten Kunden ist. 

Aus der Combobox Jahr wählen Sie das Jahr, das Sie betrachten möchten. Im linken Teil der Registerkarte werden die Soll- und Istwerte des gewählten Jahres plus 2 Vorjahre angezeigt. Auf der rechten Seite werden diese Werte grafisch dargestellt. 

Basis für die Berechnung dieser Durchschnittswerte sind die einzelnen Belege über das ganze Jahr oder eines Monats hinweg. 

Beispiel: 
Durchschnitt wird wie folgt berechnet: 

(Summe Zahltage aller X Belege) geteilt durch X (Anzahl Belege) 

### Combobox Jahr 
Bis zu dem hier angegebenen Jahr wird rückwirkend das Zahlungsverhalten angezeigt. 

Tooltip: 

Bis zu diesem Jahr drei Jahre rückwirkend wird das Zahlungsverhalten angezeigt. 

### Tabelle 

### Tabellenspalte Periode

### Tabellenspalte Ist Jahr 
Hier wird die durchschnittliche Zahldauer angegeben, gerechnet auf die Nettofälligkeit. 

Tooltip:

Hier wird die durchschnittliche Soll-Zahldauer angegeben, gerechnet auf die Nettofälligkeit.

### Tabellenspalte Ist Vorjahr 
Hier wird die durchschnittliche Ist-Zahldauer des Vorjahrs angegeben, gerechnet auf die Nettofälligkeit. 

Tooltip: 

Tatsächliche durchschnittliche Zahldauer im angegebenen Jahr, gerechnet auf das Belegdatum.

### Tabellenspalte Ist Vorvorjahr

Durchschnittliche Soll-Zahldauer im angegebenen Jahr, gerechnet auf die Nettofälligkeit.
