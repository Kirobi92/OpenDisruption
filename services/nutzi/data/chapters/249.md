# Formularsteuerung 
Formularsteuerung 

Dieser Dialog öffnet sich über den Button Steuerung im Dialog Drucken . 

Auf verschiedenen Ebenen kann das zu druckende Formular ergänzt oder ersetzt werden. So kann z.B. für einen bestimmten Kunden ein bestimmter Formularvordruck verwendet werden oder aber für eine ganze Abteilung wird immer ein abweichender Drucker genutzt. 

Die hier eingetragenen Formularsteuerungen haben absolute Priorität gegenüber den Einträgen in den Angebots- und Auftragsarten. 

Es läuft keine Verbuchungslogik für Folgeformulare ab. Es können lediglich zusätzliche Reports gedruckt werden und der Initialdruck definiert die Daten für ALLE Reports. Für die Folgereports erfolgt keine Aktualisierung der Daten. 

Im oberen Bereich wird das zu übersteuernde Formular ausgewählt. Neben dem Formular, für das die Steuerung gelten soll, müssen Sie noch festlegen, für welchen Kunden, Sachbearbeiter oder welche Abteilung die Steuerung ausgeführt werden soll. Die Kunden- oder Lieferantennummer hat die höchste Priorität, gefolgt vom Sachbearbeiter oder der Abteilung! Es gelten keine weiteren Prioritäten! 

Sobald ein zweiter Datensatz angelegt wird, prüft dieser, ob dadurch eine Endlos-Schleife entstehen kann. Dabei wird immer über den Formularnamen und den Schlüssel geprüft; es ist egal, ob die Formulare zusätzlich oder ersetzend gedruckt werden. 

Prioritäten: 
Die Kunden- oder Lieferantennummer hat die höchste Priorität, gefolgt vom Sachbearbeiter oder der Abteilung! Es gelten keine weiteren Prioritäten! 

Siehe Kapitel Abteilungen . 

Siehe Kapitel Sachbearbeiter . 

Im unteren Bereich legen Sie fest, welche Funktionalität beim Druck des oben ausgewählten Formulars ausgeführt werden soll. 

Aus der Combobox Formular Neu können Sie optional ein Formular auswählen, welches das ursprüngliche Formular ersetzen soll. Des Weiteren können Sie den gewünschten Drucker und die Anzahl der Kopien festlegen. 

Die hier eingetragenen Formularsteuerungen haben absolute Priorität gegenüber den Einträgen in den Auftragsarten. 

Über die Formularsteuerung wird vom System intern eine Übersicht der zu druckenden Belege angelegt. Wenn dann ein zu druckendes Formular bereits enthalten ist, wird dieses nicht nochmals in die Übersicht eingefügt. Diese Übersicht wird dann anschließend an den Printserver gesendet und von diesem ausgedruckt. 

Für die Eindeutigkeit gilt: Drucker + Formularname müssen beide eindeutig sein. 

Normalerweise unterdrückt eNVenta doppelte Reports!!!

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
