# Kapitel NV.ERP.Base.Common.cdSystemBase.lngILN

Die International Location Number (für EDI / Gateway) wird angegeben. Diese Nummer dient der Identifikation zwischen Geschäftspartnern. 

Tooltip: 

Geben Sie die ILN-Nummer für EDI/Gateway an.
