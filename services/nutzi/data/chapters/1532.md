# Paketdiensthistorie 
Paketdiensthistorie 

Dieser Dialog öffnet sich in der Kommissionierung >> Optionen >> PService Schnittstelle >> Paketdiensthistorie . Hier können Sie anhand der Selektionskriterien eine Liste für Pakete erstellen, die über die Versender-Schnittstelle verschickt wurden. 

Es werden nur Vorgänge berücksichtigt, in deren Versandart der Parameter Überg. Versender gesetzt ist. 

Siehe Kapitel Versandarten . 

Die Felder im Kopf des Dialogs dienen der Selektion. 

Die Paketdiensthistorie steht Ihnen nur zur Verfügung, wenn die Module Versand bzw. Logistik lizenziert sind. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Paketdiensthistorie 
Tooltip: 

In diesem Dialog können Sie anhand der Selektionskriterien eine Liste für Pakete erstellen, die über die Versender-Schnittstelle verschickt wurden. 
Für mehr Informationen drücken Sie F1.

### Tabellenspalte Verarbeitet 
Wenn das Versandgut beim Fremdprogramm angekommen ist und der Datensatz verarbeitet wurde, erscheint in dieser Spalte das Kennzeichen für die Rückmeldung. Das Kennzeichen ist in diesem Fall eine 1. Ist die Spalte leer, wurde der Datensatz nicht bearbeitet. 

### Tabellenspalte Eingepflegt 
In dieser Spalte erhalten Sie ein Kennzeichen dafür, dass relevante Daten (wie z.B. Paketnummer, Gewicht, Verpackungsart) in eNVenta ERP bearbeitet wurden, die vom Fremdprogramm zurückgekommen waren. Das Kennzeichen ist auch hier eine 1. Ist die Spalte leer, wurde der Datensatz nicht bearbeitet.

### Tabellenspalte Anzahl hin 
In dieser Spalte steht, wie viele Pakete aus Ihrem Haus an das Fremdprogramm gesendet wurden bzw. wie viele Pakete vom Fremdprogramm weitergeschickt wurden. Die Spalte dient lediglich Protokollzwecken. 

### Tabellenspalte Anzahl zurück 
In dieser Spalte steht, wie viele Pakete aus Ihrem Haus an das Fremdprogramm gesendet wurden bzw. wie viele Pakete vom Fremdprogramm weitergeschickt wurden. Die Spalte dient lediglich Protokollzwecken. 

### Tabellenspalte Gewicht Hin 
In diesen Spalten steht einmal das Gewicht des Versandguts, das in Ihrem Haus ermittelt wurde und in der Spalte ‚Gewicht Rück’ das Gewicht, welches eventuell vom Fremdprogramm überschrieben wurde. 

Tooltip: 

Angabe des Gewichts 

### Tabellenspalte Versandcode hin 
Interner Code für den Versand zum Kunden und eine entsprechende Rückmeldung vom Fremdprogramm für Protokollzwecke. 

### Tabellenspalte Versandcode rück 
Interner Code für den Versand zum Kunden und eine entsprechende Rückmeldung vom Fremdprogramm für Protokollzwecke. 

### Tabellenspalte Nachnahme 
Kennzeichen, ob das Paket per Nachnahme versendet wurde. 

Für die Verarbeitung der Nachnahmegebühr beim Fremdprogramm kann für eine Nachnahme-Sendung, die z.B. 5 Pakete hat, die gesamte Nachnahmegebühr auf das 1. Paket berechnet werden. Für die anderen 4 Pakete wird keine Nachnahmegebühr ausgewiesen.
