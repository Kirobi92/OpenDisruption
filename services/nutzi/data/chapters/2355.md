# Factoring Parameter Registerkarte Obligos 
Factoring Parameter Registerkarte Obligos 

Diese Registerkarte beinhaltet die Vertragsnummer und eine Beschreibung des Factoringvertrags. Da es möglich ist, mehrere Verträge über bestimmte Kunden abzuschließen, erhalten Sie in dieser Tabelle eine Übersicht über alle Verträge. 

Nachträgliches Ändern der Obligo-Nummer 

Wenn die Obligo- bzw. Vertragsnummer im Nachhinein geändert wird, wird geprüft ob sie bereits von einem oder mehreren Kunden verwendet wird. Ist dies der Fall, kommt eine entsprechende Meldung, die darauf hinweist, dass beim Speichern die Kunden mit aktualisiert werden und ob mit diesem Vorgang fortgefahren werden soll. Bei Nein werden sämtliche Änderungen, die Obligos betreffen, verworfen und bei Ja wird eine interne Liste mitgeführt, welche die Obligo-Nummern auf den betroffenen Kunden beim Speichern der Parameter ändert.
