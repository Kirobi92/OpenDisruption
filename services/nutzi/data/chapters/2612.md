Finanzbuchhaltung / Budget generieren 

# Budget generieren Registerkarte Selektiertes Jahr 
Budget generieren Registerkarte Selektiertes Jahr 

Diese Registerkarte zeigt den Saldo der selektierten Konten / Perioden an. In den Spalten daneben kann das Budget editiert werden. 
### Button Setze auf 0 

Mit diesem Button werden die Budgets zurückgesetzt. 

Tooltip: 

Budgets zurücksetzen 

### Button Modifikation 

Mit diesem Button wird das Budget gemäß der Modifikation (d.h. den Werten in den Felder Betrag und / oder Prozent ) geändert. 

Tooltip: 

Budget gemäß Modifikation ändern 

### Tabelle 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung
