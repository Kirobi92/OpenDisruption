# Kostenartenauswahl 
Kostenartenauswahl 

Mit dem Button neben dem Feld K/L-Nummer im Dialog Buchen Kore öffnet sich der Dialog Kostenartenauswahl . 

Die Felder Kostenartennr , Bezeichnung und die Comboboxen Gruppe 1-5 können zur Selektion verwendet werden. Eine sogenannte „Like-Abfrage“ ist ebenfalls möglich. 

Um die gesuchte Kostenart nun in den Buchungsdialog zu übernehmen, markieren Sie die entsprechende Zeile der Tabelle durch einen Mausklick und betätigen den Button oder führen einen Doppelklick aus. 

Beim Ändern eines Eintrags erscheint nun eine Abfrage, falls Sie den Dialog schließen wollen, ohne vorher gespeichert zu haben. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Kostenartenauswahl 
Tooltip: 

In diesem Dialog wählen Sie eine Kostenart aus. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche nach Kostenarten bzw. Leistungen, die den eingegebenen Suchkriterien entsprechen. Die Suche kann hier eingeschränkt werden auf Kostenarten im Unterschied zu Leistungen. Ebenso ist eine Suche nach Kostenartennummer, Bezeichnung und Gruppe 1-3 möglich. 

Wenn Sie keine Suchkriterien eingeben, werden alle vorhandenen Kostenarten und Leistungen angezeigt. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter .

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen
