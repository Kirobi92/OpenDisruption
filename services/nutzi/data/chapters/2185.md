# Serviceeinsatz Registerkarte Objekt 
Serviceeinsatz Registerkarte Objekt 

Wenn Sie im Kopf des Dialogs ein Serviceobjekt eingetragen haben, werden auf dieser Registerkarte die Detailinformationen dazu angezeigt. Nur die Felder Wartungsmemo und Servicememo sind zugänglich und entsprechender Text kann eingetragen werden. 
### Feld Vertragsart 
Hier wird die Vertragsart des eingetragenen Serviceobjekts angezeigt. 

Tooltip: 

Geben Sie der Vertragsart eine Bezeichnung. 

### Feld Garantiebeginn 
Ist dem Serviceobjekt ein Garantiebeginn hinterlegt, wird dieses Datum hier angezeigt. 

Tooltip: 

Beginn der Garantiezeit 

### Feld Garantieende 
Ist dem Serviceobjekt ein Garantieende hinterlegt, wird dieses Datum hier angezeigt. 

Tooltip: 

Ende der Garantiezeit 

### Feld Kulanzende 
Wenn für die Garantiezeit eine Kulanz eingeräumt wurde, wird das Datum des Kulanzendes hier angezeigt. 

Tooltip: 

Ende der Kulanzzeit 

### Feld Inbetriebnahme 
Hier wird die Inbetriebnahme des Serviceobjekts angezeigt. 

Tooltip: 

Datum der Inbetriebnahme 

### Feld Letzte Wartung 
Hier wird das Datum der letzten Wartung angezeigt. 

Tooltip: 

Datum der letzten Wartung 

### Feld Wartungsperiode 
Ist für das Serviceobjekt eine Wartungsperiode definiert, wird diese hier angezeigt. 

Tooltip: 

Zeigt die Wartungsperiode. 

### Feld Wartungsmemo 
Hier wird das Wartungsmemo aus dem Serviceobjekt angezeigt. 

Tooltip: 

Zeigt das Wartungsmemo aus dem Serviceobjekt. 

### Feld Servicememo 
Hier wird das Servicememo aus dem Serviceobjekt angezeigt. Diesen Text können Sie hier ergänzen. 

Tooltip: 

Zeigt das Servicememo aus dem Serviceobjekt.
