# BMEcat - Preiskonfiguration 
BMEcat - Preiskonfiguration 

Dieser Dialog öffnet sich im BMEcat - Katalogexport über die Buttons bzw. auf der Registerkarte Exportoptionen . 

Hier können Sie Preise konfiguriren, die zusammen mit dem Katalog exportiert werden sollen. 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

Meldungen 

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Bezeichnung 
Hier wird der Name des Preistyps angezeigt, wenn ein Preistyp ausgewählt ist. 

Tooltip: 

Bezeichnung der Preis-Konfiguration 

### Feld Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Button Suchen 

Mit diesem Button können Sie nach Kunden suchen. 

### Feld Firma 1 
Der 1. Adresszeile der Firma wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Combobox Preis-Typ 
Hier können Sie einen Preis-Typ angeben, z.B. Listenpreis Netto oder Kundenpreis. 

### Combobox Preisgruppe 
Hier können Sie eine Brutto-Preisgruppe auswählen, die im zugrundeliegenden Vorgang herangezogen wird und bestimmt, auf welchen Verkaufspreis zugegriffen werden soll. 

Siehe Kapitel Preisgruppen . 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

### Combobox Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Tooltip: 

Angabe der Währung 

### Checkbox Bruttopreis 
Wenn Sie diese Checkbox aktivieren, zieht der Bruttopreis. 

### Checkbox Inkl. Fracht 
Wenn Sie diese Checkbox aktivieren, versteht sich der Preis inklusive Fracht (diese Option betrifft nur die Angabe in der Ausgabe, nicht die Preisermittlung!). 

Tooltip: 

Inklusive Fracht (Option betrifft nur die Angabe in der Ausgabe, nicht die Preisermittlung)

Tooltip: 

Inklusive Versicherung (Option betrifft nur die Angabe in der Ausgabe, nicht die Preisermittlung)

Inklusive Verpackung (Option betrifft nur die Angabe in der Ausgabe, nicht die Preisermittlung)
