# Tourenstamm Registerkarte Kunden 
Tourenstamm Registerkarte Kunden 

Auf dieser Registerkarte können Kunden angelegt werden. Bei der Zuordnung eines Kunden zu einer Tour kann nicht nur die Kundennummer, sondern auch die Postleitzahl / der Ort hinterlegt werden. So wird dieser Ort mit dieser Tour nur beliefert, wenn dies auch für den hinterlegten Kunden passt. Alle anderen Postleitzahlen der Tour sind davon nicht betroffen. 
### Tabelle 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Firma 1 
Der 1. Adresszeile der Firma wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Firma 2 
Die 2. Adresszeile wird angegeben. 

Tooltip: 

Zeile 2 des Firmennamens. 

### Tabellenspalte PLZ 
Die Postleitzahl des Orts wird angegeben. 

Tooltip: 

Postleitzahl des Datensatzes 

### Tabellenspalte Ort 
Hier wird der Ort angegeben. 

Tooltip: 

Angabe des Orts 

### Tabellenspalte Fahrtzeit (hh:mm) 
Hier wird die Fahrzeit auf Basis der Startzeit hinterlegt. 

Tooltip: 

Zeit, die angibt, wann die Lieferung ankommt, relativ zum Abfahrtszeitpunkt. 

### Button Hinzufügen 

Mit diesem Button können Sie der Tabelle einen weiteren Eintrag hinzufügen. 

### Button Löschen 

Dieser Button entfernt einen Eintrag aus der Tabelle. 

### Button Suchen 

Mit diesem Button können Sie einen Kunden suchen.
