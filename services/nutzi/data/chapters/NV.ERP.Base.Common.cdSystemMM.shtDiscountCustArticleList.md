# Kapitel NV.ERP.Base.Common.cdSystemMM.shtDiscountCustArticleList

Wenn Sie diesen Parameter aktivieren, wird die Artikelliste des im Feld Rabattkunde im Menü Kunde (Registerkarte Warenwirtschaft ) hinterlegten Kunden herangezogen. Bei dieser Option kann der Kunde nicht alle Waren bestellen, sondern nur Artikel, welche entsprechend in der Liste gepflegt wurden. Die Steuerung, ob diese Option zieht, wird entsprechend im Kundenstamm hinterlegt. Bei der Artikellistung über Rabattkunde kann die Listung auf einem Kunden erfolgen und dieser wird entsprechend in anderen / mehreren Kundenstammsätzen als Rabattkunde hinterlegt, z.B. eine Zentrale und mehrere Filialen. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, wird die Artikelliste aus dem Feld 'Rabattkunde" des hinterlegten Kunden herangezogen.
