# Vertreterbesuche 
Vertreterbesuche 

Dieser Dialog öffnet sich in Kunde und Adresse über das Menü Optionen . Hier können Sie festlegen, dass ein Kunde / eine Adresse von mehreren Vertretern mit unterschiedlichen Rhythmen besucht wird. 

In der Tabelle werden zum selektierten Vertreter die Firmen aufgelistet und wann sie zuletzt besucht wurden (Spalten 'Letzter Besuch'). In der Spalte 'Rhythmus' wird angezeigt, in welchem Rhythmus ein Besuch erfolgt. 

Dieser Vertreter wird dann in den Besuchsvorschlägen (in den Methoden 1-4) berücksichtigt. 

Siehe Kapitel Besuchskalender Registerkarte Vorschläge .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Felder für eine neue Eingabe. 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

Hier wird das Datum des letzten Besuchs beim Kunden eingetragen. Das Besuchsdatum wird automatisch aus dem letzten Besuchsbericht übernommen. Dieses Feld kann auch manuell gepflegt werden.
