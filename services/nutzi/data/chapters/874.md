# Einmalkunde löschen 
Einmalkunde löschen 

Dieser Dialog öffnet sich im Kunde >> Optionen >> Einmalkunden löschen . Hier können Sie sich alle Einmalkunden anzeigen lassen. 

Einmalkunden, für die nie ein Beleg erfasst wurde und die noch nie bebucht wurden, werden komplett gelöscht und können hier nicht mehr hergestellt werden. 

Falls für einen Einmalkunden, den Sie löschen möchten, noch offene OPs bestehen, kann er nicht gelöscht werden. Sie erhalten eine entsprechende Meldung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Einmalkunden löschen 
Tooltip: 

In diesem Dialog können Sie sich alle Einmalkunden anzeigen lassen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Menü Fenster-Information 
Über dieses Menü werden Ihnen Informationen zum Dialog angezeigt. Siehe nachfolgendes Beispiel: 

Meldungen 

### Button Suche 

Mit diesem Button lesen Sie die Einmalkunden in die Tabelle ein. 

### Button Neu 

Dieser Button leert die Felder. 

### Button Löschen 

Mit diesem Button entfernen Sie Einmalkunden aus der Tabelle, die nicht gelöscht werden können. Der Grund dafür wird in der Spalte 'Info' angezeigt. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. 

### Button Prüfen 

Wenn Sie diesen Button drücken, wird in der Spalte 'Info' der Grund eingelesen, der ein Löschen des Kunden nicht erlaubt. 

Tooltip: 

Prüft ob Kunden gelöscht werden können. 

### Button Starten 

Wenn Sie diesen Button drücken, werden alle Kunden gelöscht, die im Kundenstamm als Einmalkunde gekennzeichnet sind. 

Allerdings werden nur Einmalkunden ohne offene Vorgänge gelöscht. 

Tooltip: 

Löscht alle löschbaren Kunden. 

### Button Wiederherstellen 

Dieser Button öffnet den Dialog Kundenauswahl zur Suche von Einmalkunden. Durch markieren dieser Kunden im Dialog Kundenauswahl und klick auf den Button erfolgt die Wiederherstellung. 

Siehe Kapitel Kundenauswahl . 

Tooltip: 

Öffnet die Kundenauswahl, um einzelne gelöschte Kunden wiederherzustellen.

### Feld Löschbar

### Tabelle Einmalkunde 

In dieser Tabelle werden die Einmalkunden aufgeführt. Gelöscht werden können nur diejenigen, die keinen Eintrag in der Spalte 'Info' haben.
