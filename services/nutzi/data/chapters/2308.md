# Zahlungsarten im Webshop Registerkarte Allgemein

E-Commerce Katalog / Zahlungsarten im Webshop 

# Zahlungsarten im Webshop Registerkarte Allgemein 
Zahlungsarten Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die Zahlungsarten im Detail. 
### Button Neu 

Mit diesem Button können Sie eine weitere Zahlungsart anlegen. 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag aus der Tabelle. 

### Tabelle 

### Tabellenspalte Zahlungstyp 
Hier wird die Zahlungsart angegeben, die dem Element zugeordnet ist. 

Tooltip: 

Die Zahlungsart, welche dem Element zugeordnet ist. 

### Tabellenspalte Zahlungsbedingung 
Hier geben Sie eine feste Zahlungsbedingung ein. Diese Zahlungsbedingung wird in den Auftrag übernommen. Wenn Sie hier keinen Eintrag vornehmen, so wird die Zahlungsbedingung aus dem Kundenstamm gezogen. 

Tooltip: 

Geben Sie hier eine feste Zahlungsbedingung ein. Diese Zahlungsbedingung wird in den Auftrag übernommen. Wenn Sie hier keinen Eintrag vornehmen, so wird die Zahlungsbedingung aus dem Kundenstamm gezogen. 

### Tabellenspalte Min. Bestellwert 
Hier geben Sie den Minimalwert für den Warenkorb, für dieses Zahlungsprofil-Element an. 

Tooltip: 

Minimalwert für den Warenkorb für dieses Zahlungsprofilelement. 

### Button Nach oben 

Mit diesem Button können Sie einen Eintrag der Tabelle nach oben verschieben. 

Tooltip: 

Nach oben verschieben 

### Button Nach unten verschieben 

Mit diesem Button können Sie einen Eintrag der Tabelle nach unten verschieben. 

Tooltip: 

Nach unten verschieben
