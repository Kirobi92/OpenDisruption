# Auftrag Registerkarte Bestellinfo 
Registerkarte Bestellinfo 

Diese Registerkarte zeigt im Fall einer auftragsbezogenen Bestellung (und nur in diesem Fall wegen Laufzeiten) die zugehörige Bestellposition, die Bestellnummer (Belegnummer), das Bestelldatum, die Artikelnummer, den Status der Bestellposition, die Bestätigungsnummer und die bestätigte Menge an. Wurde die Position noch nicht bestätigt, so ist „unbestätigt“ eingetragen und angezeigt. 

Damit bietet diese Registerkarte eine komfortable Übersicht für den Verkäufer, ob eine Bestellung für eine Auftragsposition vorhanden ist und welche Daten dazu vorhanden sind. 

Tooltip: 

Bestellinfo 
### Tabelle 

### Tabellenspalte Beleg 
Hier wird die Belegnummer aus dem Einkauf (Bestellung, Anfrage, Belastung) angezeigt. 

Tooltip: 

Hier wird die Bestellnummer definiert. 

### Tabellenspalte Pos 
Hier wird die Positionsnummer angegeben. Diese Nummer muss je Vorgang eindeutig sein! 

Tooltip: 

Positionsnummer 

### Tabellenspalte Bestelldatum 
Hier wird das Bestelldatum angegeben. 

Tooltip: 

Angabe des Bestelldatums 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer der Position angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Status 
Hier wird der Status der Position angegeben. 

1-offen 
2-offen, zur Lieferung freigegeben 
3-offen, zur Faktura freigegeben 
4-fakturiert 
5-geliefert (ohne Berechnung) 
6-storniert 

Tooltip: 

Angabe des Positionsstatus 

### Tabellenspalte Bestätigt 
Hier wird die gelieferte Menge angegeben. D.h. die tatsächlich durch den Lieferscheindruck als gelieferte, im Auftrag eingetragene, Menge. 

Tooltip: 

Menge geliefert: 

Die tatsächlich durch den Lieferscheindruck als gelieferte, im Auftrag eingetragene Menge. 

### Tabellenspalte Bestätigungsstatus 
Tooltip: 

Lieferbestätigungsstatus: 

0/Null = unbestätigt 
1 = teilbestätigt, ein Termin 
2 = teilbestätigt, mehrere Termine 
3 = vollbestätigt, ein Termin 
4 = vollbestätigt, mehrere Termine 

### Tabellenspalte Best. Nummer

Nummer der Bestellbestätigung aus der Bestellung.

Tooltip:
