# Kapitel NV.ERP.Base.Common.cdSystemBase.shtCurrentYear

Hier wird das fürs gesamte System gültige Buchungsjahr ausgewählt. Im Dialog Buchen in der Finanzbuchhaltung kann diese Einstellung jederzeit für den entsprechenden Arbeitsplatz geändert werden und ist nur solange gültig, bis eNVenta ERP verlassen wird. 

Bei abweichendem Wirtschaftsjahr ist das aktuelle Buchungsjahr immer das Jahr, in dem das Geschäftsjahr endet. 

Tooltip: 

Wählen Sie das gültige Buchungsjahr.
