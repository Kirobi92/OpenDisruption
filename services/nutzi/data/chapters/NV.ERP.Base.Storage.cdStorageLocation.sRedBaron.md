# Kapitel NV.ERP.Base.Storage.cdStorageLocation.sRedBaron

Hier wählen Sie einen Drucker aus. Dieser Drucker befindet sich in der Regel bei der Lagerleitung. Hier werden zum einen verschiedene, wichtige Dokumente für den Lagerleiter gedruckt (z.B. Endkontrolllisten usw.). 

Ansonsten dient dieser Drucker für manche Vorgänge als 'Fallback'-Drucker. D.h., wenn sonst kein Drucker hinterlegt ist, gibt es keine Fehlermeldung, sondern es wird an diesem Drucker ausgedruckt. 

Tooltip: 

Wählen Sie den Drucker für wichtige Dokumente des Lagerleiters.
