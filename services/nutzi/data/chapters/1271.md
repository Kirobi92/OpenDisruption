# Belastung erstellen 
Belastung erstellen 

Dieser Dialog öffnet sich über den Button Belastung im Dialog Retoure . Es ist möglich, mehrere Belastungen aus einer Retoure zu erstellen, die an verschiedene Lieferanten gesendet werden. Dafür können diese Positionen den entsprechenden Lieferanten zugeordnet werden. 

Zu beachten ist, dass in der Tabelle für Lieferanten unterschiedliche Lieferanten angezeigt werden. Dies liegt daran, dass dies die Lieferanten sind, die normalerweise als Lieferant der verknüpften Bestellposition der Retoure hinterlegt sind oder der Hauptlieferant des Artikels, falls ersteres nicht gefüllt ist. Den Lieferanten für eine der Positionen können Sie ändern und zwar über den -Button neben dem Feld Lieferant . 

Diese Belastung ist dann im Einkauf im Menü Belastung ersichtlich. 

Nach der Erstellung der Belastung wird die Nummer der Belastung auf die Retourenposition eingetragen, damit dies nachvollziehbar ist. Ebenso kann dadurch keine doppelte Belastung an den Lieferanten erfolgen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Belastung erstellen 
Tooltip: 

In diesem Dialog können Sie eine Belastungsanzeige an den Lieferanten erstellen, um z.B. die Retoure an den Lieferanten abwickeln zu können. 
Für mehr Informationen drücken Sie F1.

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Combobox Belastungsart 
In dieser Combobox kann die Art der Belastung angegeben werden. Durch diese Angabe ergeben sich die Prozesse analog einer manuellen Belastung an den Lieferanten. Über die Lieferantennummer und die Projektnummer werden die Daten an die erzeugte Belastung weitergegeben. 

Tooltip: 

Geben Sie die Einkaufsart an. 

### Checkbox Als Wertbelastung 
Über diese Checkbox kann der Sachbearbeiter entscheiden, ob die Belastung mit Lagerbuchung oder ohne Lagerbuchung erfolgt. 

Tooltip: 

Markiert diese Belastung als eine Wertbelastung (unabhängig von der tatsächlichen Belastungsart). 

### Combobox Projekt 
Steht die Belastung im Zusammenhang mit einem Projekt, wählen Sie hier das entsprechende Projekt aus. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein.
