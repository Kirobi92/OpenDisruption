# Verwaltung des Standard-Kalkulationsschemas 
Kalkulationsschema 

In der Artikelkalkulation öffnet sich über den Button Schema der Dialog Kalkulationsschema . Das Kalkulationsschema zeigt auf, aus welchen Kostenbestandteilen (z.B. Herstellkosten, Selbstkosten, Gemeinkosten) sich die Kosten des Produkts zusammensetzen. Hier können Standardschemas für Kalkulationen erstellt und verwaltet werden. Ein Schema setzt sich hierbei immer aus mehreren Faktoren zusammen, welche in der Registerkarte Allgemein ausgewählt werden können. 

Zum Anlegen eines neuen Schemas geben Sie in das Feld Schemanr eine neue Nummer und in das Feld Name eine möglichst sprechende Bezeichnung ein. Dann fügen Sie dem neuen Schema Faktoren hinzu und zwar über den Button Faktor . 

Werden Faktoren geändert, die den Kalkulationsschemas zugeteilt sind, ändern sich sämtliche Schemas automatisch mit, in denen diese Faktoren zugeteilt sind.

Für mehr Informationen drücken Sie F1.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Auswahl 

Mit diesem Button übernehmen Sie das gewählte Schema in den zugrunde liegenden Dialog. 

Tooltip: 

Schema Übernehmen 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Schema 
Hier kann die Nummer eines Schemas eingetragen werden. Einmal zur Suche und auch zur Anlage eines neuen Schemas. 

Wird keine Nummer eingetragen, vergibt eNVenta automatisch die nächste freie Nummer für ein neues Schema. 

### Feld Name 
Die Bezeichnung des Schemas wird automatisch eingelesen, wenn die Schemanummer eingetragen wurde. 

Der Datensatz kann auch über die Bezeichnung gesucht und eingelesen werden. 

### Feld Erfassung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der Erfassung eingetragen. 

Tooltip: 

Datum und Uhrzeit der Erfassung 

### Feld Änderung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der letzten Änderung eingetragen. 

Tooltip:

### Feld Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
