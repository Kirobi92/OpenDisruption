# B2B Rechnungen 
B2B Rechnung 

Der Dialog B2B Rechnung öffnet sich in der Rechungsprüfung über Optionen >> B2B Rechnungen . Dieser Dialog vereinfacht die Prüfung von importierten, fehlerhaften und/oder problematischen B2B-Rechnungseingängen (z.B. INVOIC-EDI-Nachrichten). 

Die Felder im Kopf des Dialogs werden aus der zugrundeliegenden, importierten Rechnung voreingestellt. Die entsprechenden Informationen werden auf den diversen Registerkarten dargestellt. 

Fehlende oder fehlerhafte Informationen werden auf der Registerkarte Positionen farbig hervorgehoben. Konnte z.B. die Artikelnummer nicht ermittelt werden, so stellt dies einen Fehler dar und wird entsprechend mit roter Hintergrundfarbe angezeigt. Sollte ein Feld nicht gefüllt sein, aber es würde helfen, so wird dies als Hinweis angesehen und das Feld mit gelbem Hintergrund angezeigt. 

Wenn Sie die Daten haben, können Sie diese direkt im Dialog ergänzen bzw. korrigieren. Nach der Speicherung der Daten können Sie den Bearbeitungsvorgang zum Import etc. wieder starten. 

Achtung: Der Dialog sollte nur dem Administrator bzw. berechtigten Personen zugänglich sein. Hier können Eingangsrechnungen manipuliert werden, was durch den Grundsatz der ordnungsgemäßgen Buchführung überwacht werden muss. 

Bei fehlenden Informationen haben Sie die Möglichkeit über das Menü den Vorgang auszudrucken und an den Absender zurück zu senden, mit der Bitte um Ergänzung der fehlenden Daten, wie Sie auf dem Druckbeleg ausgewiesen sind.

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.

### Combobox Rechnungslager 
Hier wählen Sie eine Lager-ILN aus. 

Mit der Übermittlung der Lager-ILN kann gesteuert werden, an welchen Standort Ware gebracht werden soll. Deshalb ist es notwendig, dass die Lager-ILN in den Bestellungen per EDI an den Lieferanten übermittelt wird. 

Da der Großteil der Rechnungen bereits per EDI übermittelt werden, können fehlgeschlagene Rechnungsprüfungen anhand der Lager-ILN eingegrenzt werden. 

Tooltip: 

Rechnungslager
