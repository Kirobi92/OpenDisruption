# BMECat - Import Registerkarte Allgemein 
BMEcat Import Registerkarte Allgemein 

Auf dieser Registerkarte werden allgemeine Einstellungen vorgenommen. 
### Feld Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

Tooltip: 

Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Feld Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

Tooltip: 

Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Combobox Katalogtyp 
Der Katalogtyp aus dem zugrundeliegenden Vorgang wird angezeigt. 

Tooltip: 

Lieferanten- oder Kundenkatalog 

### Feld Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Button Lieferant auswählen 

Mit diesem Button kann ein Lieferant gesucht werden. 

Tooltip: 

Lieferant auswählen 

### Checkbox Artikel 
Wenn Sie Artikel importieren möchten, aktivieren Sie diese Checkbox. 

Tooltip: 

Artikel importieren 

### Checkbox Katalog 
Wenn Sie einen Katalog importieren möchten, aktivieren Sie diese Checkbox. 

Tooltip: 

Katalog importieren 

### Checkbox Klass. System 
Wenn Sie ein Klassifikationssystem importieren möchten, aktivieren Sie diese Checkbox. 

Tooltip: 

Klassifikationssystem importieren 

### Combobox Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Feld Erstellt am 
Hier wird das Datum der Erstellung angegeben. 

### Checkbox Validieren 
Hier wird die Validierung der XML-Datei nach BMEcat-Standard angegeben. 

Tooltip: 

Validierung der XML-Datei nach BMEcat-Standard 

### Feld Datenbereich 
Die importierten Daten gehören zu einem Datenbereich, der hier angegeben ist. Es werden nur Daten des selben Bereichs miteinander aktualisiert. 

Damit wird ein versehentliches Überschreiben verhindert. 

Tooltip: 

Die importierten Daten gehören zu einem Datenbereich. Es werden nur Daten desselben Bereichs miteinander aktualisiert. 

Es wird empfohlen, mindestens einen Datenbereich je Datenlieferant zu nutzen. Wird kein Datenbereich definiert, setzt das System automatisch die Importformat-ID als Bereich. 

### Feld Info 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen. 

### Feld Generator Info 
Hier wird eine Information zum Generator des Imports angegeben, mit dem der Katalog erstellt wurde. 

Tooltip: 

Information zum Generator 

### Feld Basis-Pfad 
Hier geben Sie das Mime-Basisverzeichnis an, auf welches sich die relativen Pfadangaben in MIME_SOURCE beziehen. 

Tooltip: 

Mime-Basisverzeichnis, auf welches sich die relativen Pfadangaben in MIME_SOURCE beziehen.
