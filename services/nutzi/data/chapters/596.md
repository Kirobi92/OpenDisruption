# Referenz Links 
Referenz Links 

eNVenta ERP bietet die Möglichkeit, Dokumenten-Links sowie URL‘s als Info in eNVenta ERP zu hinterlegen, um 2 Vorgänge/Referenzen miteinander zu verknüpfen, z.B. einen Call mit einem anderen Call oder einer URL. Ein weiteres Beispiel wäre im Artikelstamm über eine URL-Verknüpfung z.B. „PDF-Dokumente“ aus einem Sharepoint zu einem Artikel zu referenzieren (Werk- bzw. Prüfzeugnisse). Dies erspart Ihnen lange Archiv-Recherchen. Ebenso können Sie auch direkte Links zu den Archiv-Dokumenten erstellen. 

Der Dialog Referenz Links öffnet sich u.a. in den Menüs Artikel, Auftrag, Angebot, Call-Center, Projekte, Chargen und Archiv-Mappen über den Button (bzw. <Umschalt+Ctrl+L> ). Der Dialog wird dabei immer im Help-Frame angezeigt. 

Wird der Dialog Referenz Links über den Dialog Archiv-Mappe (Anzeige der Ergebnisse der Archivsuche) geöffnet, wird der Dialog modal angezeigt, die Funktionalität ist genau gleich. 

In den Projekten und im Auftrag öffnet sich der Dialog Referenz Links auch über Optionen >> Element verlinken bzw. Position verlinken . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Referenz-Links 
Tooltip: 

In diesem Dialog können Dokumenten-Links sowie URL‘s als Info in eNVenta ERP hinterlegt werden, um 2 Vorgänge/Referenzen miteinander zu verknüpfen. 
Für mehr Informationen drücken Sie F1.

### Button Drucken 

Wenn Sie eine Position der Tabelle markieren, kann das Zieldokument dazu über diesen Button gedruckt werden (das gilt für Archiv-Dokumente und URLs). Wenn die Position kein druckbares Dokument enthält, z.B. Call-Referenz, so erscheint eine entsprechende Hinweismeldung. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten.

### Feld Kurztext 
Hier werden das Programm und die Nummer angezeigt, aus dem heraus der Dialog Referenz Links geöffnet wurde. 

Tooltip: 

Generierter Anzeigetext oder Referenztext. 

### Feld Programm 
Hier wird nochmals das zugrundeliegende Programm angezeigt. 

Tooltip: 

Art des Datensatzes, von welchem der Link ausgeht. 

### Feld Referenz 
Hier wird die Referenz zum Programm bzw. die Vorgangsnummer angezeigt. Diese Nummer dient als Quelle für neu zu erstellende Links bzw. Suchkriterium. 

Tooltip: 

Pfad bzw. URL des verlinkten Inhalts. Kann als Suchkriterium dienen. 

### Tabelle Referenzen 

In der Tabelle werden sämtliche bereits angelegte Referenzen zum ausgewählten Vorgang aufgelistet. Mit einem Doppelklick auf einen Eintrag können Sie in den entsprechenden Vorgang verzweigen. Je nachdem, ob der Doppelklick auf die Quelle oder das Ziel durchgeführt wurde, wird das entsprechende Objekt geöffnet. 
### Tabellenspalte Typ 
Tooltip: 

Name des Referenz-Link-Typens. 

### Tabellenspalte Programm 
Hier wird das Programm angegeben. D.h. die Art des Datensatzes, von welchem der Link ausgeht. 

Tooltip: 

Art des Datensatzes, von welchem der Link ausgeht. 

### Tabellenspalte Referenz 
Hier wird eine Referenz angegeben. 

Tooltip: 

Pfad bzw. URL des verlinkten Inhalts. Kann als Suchkriterium dienen. 

### Tabellenspalte Link Version 
Hier wird die Version angegeben, falls die Versionierung für den gewählten Link-Typen aktiv ist. 

Tooltip: 

Angabe der Version, falls die Versionierung für den gewählten Link-Typen aktiv ist. 

### Tabellenspalte Sprache 
Hier kann zu jedem Link eine Sprache ausgewählt werden. 

So kann der automatische Druck von Referenz Links in Abhängigkeit der Kundensprache ausgelöst werden. Wenn mehrere Links des gleichen Typs hinterlegt sind, können diese nach Sprache priorisiert werden. 

Links ohne Sprache werden als Standard genutzt. 

Sonderfall: Beim automatischen Druck von Dokumenten, die bei gleichem Typ mehrfach in der gleichen Sprache vorliegen und nicht versioniert sind, wird nur der letzte in der Liste gedruckt, falls dieser noch nicht gedruckt wurde. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Tabellenspalte Auslaufdatum 
Hier wird das Auslaufdatum der Gültigkeit des Datensatzes angegeben. 

Tooltip: 

Dieses Datum gibt an, wann ein Referenzlink ungültig wird.
