# Ressourcen für den Arbeitsgang 
BA-AG-Ressourcen 

Der Button Ressourcen im Betriebsauftrag auf der Registerkarte Arbeitsplan öffnet den Dialog BA-AG-Ressourcen . Hier sehen Sie in einer Übersicht die Zuordnungen eines Arbeitsgangs (mit Ressource) zu einem Betriebsauftrag. Die Zuordnungen können im Arbeitsplanstamm vorbelegt werden. Eine Zuordnung kann hinzugefügt, geändert oder gelöscht werden. 

### Ressourcen ändern 
Solange ein Arbeitsgang nicht abgeschlossen ist (Status 6) können Ressourcen hinzugefügt werden. Da zu diesem Zeitpunkt möglicherweise schon ein Teil der Arbeit getan ist, kann der Anteil der Ressource beschränkt werden, d.h. auf einen Wert unter 100% gesetzt werden. 

Findet jetzt eine Neuterminierung statt, wird nur noch der notwendige Anteil eingeplant. 

Dieser Mechanismus kann auch genutzt werden, um zu jedem Zeitpunkt die Arbeit auf mehrere (alternative) Ressourcen zu verteilen oder einen Ressourcenwechsel zu erreichen. Dazu muss der Anteil für diese Ressourcen so aufgeteilt werden, dass zusammen 100% Anteil entstehen. 

Die Verantwortung liegt hier beim Disponenten. Das System kann nicht beurteilen, ob Parallelität, Aufteilung oder Austausch beabsichtigt ist. 

### Ressourcen löschen 
Sobald ein Arbeitsgang terminiert ist (Status > 1) kann eine Ressource nicht mehr gelöscht werden. 

Es ist jedoch möglich, den Anteil einer Ressource bis auf 0% herunter zu setzen. Dadurch bleibt die Referenz erhalten, bereits durchgeführte Arbeiten (Reservierungen) können abgeschlossen werden. Bei einer Neuterminierung wird jedoch keine Kapazität für diese Ressource eingeplant. 

Wurde bei den Ressourcenzuordnungen eine Änderung vorgenommen und gespeichert, wird automatisch eine Neuterminierung des Betriebsauftrags ab dem geänderten Arbeitsgang durchgeführt.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Dieser Button öffnet den Dialog Ressourcenauswahl , aus der Sie eine Ressource übernehmen können. Änderungen an der Ressource sind nicht möglich. 

Siehe Kapitel Werkzeuge / Ressourcenauswahl . 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

### Tabellenspalte Zeit vor (hh,mm) 
Angabe der Zeit vor Verwendung der Ressource. Die Angabe wird aus dem Ressourcenstamm übernommen. 

### Tabellenspalte Zeit nach (hh,mm) 
Angabe der Zeit nach der Verwendung der Ressource. Die Angabe wird aus dem Ressourcenstamm übernommen. 

### Tabellenspalte Belastung % 
Angabe der maximalen Belastung der Ressource in %. Die Angabe wird aus dem Ressourcenstamm übernommen. 

### Tabellenspalte Unbegrenzte Kapazität 
Angabe, ob es sich um eine Ressource mit unbegrenzter Kapazität handelt. Die Angabe wird aus dem Ressourcenstamm übernommen. 

### Tabellenspalte Anteil% 
Falls ein Arbeitsgang mehr als eine Ressource verwendet, wird in dieser Spalte der prozentuale Anteil der Ressource am Arbeitsgang eingetragen. Ist die Spalte leer, wird der Wert in dieser Spalte auf 100% gesetzt. So erhalten Sie eine differenzierte Angabe der verwendeten Zeit pro Ressource.
