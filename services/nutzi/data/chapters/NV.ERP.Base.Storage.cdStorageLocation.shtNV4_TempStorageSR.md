# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtNV4_TempStorageSR

Hier wählen Sie das Lager aus, auf welches die Retoure gebucht werden soll, bevor sie auf den Lagerauftrag umgebucht wird. Bei diesem Lager handelt es sich um ein statisches Lager. 

Tooltip: 

Definiert das temporäre Lager für Retouren
