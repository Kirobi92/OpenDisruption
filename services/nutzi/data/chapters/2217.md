# Call Registerkarte Intern 
Call Registerkarte Intern 

Auf dieser Registerkarte ist es möglich, Kommentare zu erfassen, die nur für Mitarbeiter in Ihrem Haus bestimmt sind. Diese Kommentare werden in das Feld Intern Neu geschrieben. Beim Speichern werden diese Kommentare in das Feld Intern geschrieben und gespeichert. Der aktuellste Eintrag steht an oberster Stelle. 

Ein Sachbearbeiter kann seinen eigenen Eintrag nochmals bearbeiten, wenn es der oberste und damit letzte Eintrag in der Tabelle Übersicht ist. 

Über den Button wird der Kommentar automatisch in der Datenbank gespeichert. 

Tooltip: 

Html Editor 
### Button Neu 

Über diesen Button können Sie einen neuen internen Eintrag erstellen. 

Tooltip: 

Generiert einen neuen, internen Eintrag. 

### Button Html-Editor 

Mit diesem Button können Sie Html-Text bearbeiten. 

Außerdem können Sie einen Eintrag, der bereits abgeschlossen ist, genauer betrachten / lesen. Der Html-Editor öffnet sich dann nur mit der Registerkarte Vorschau . So können umfangreiche Texte und Bilder besser betrachtet werden. 

Änderungen sind nicht mehr möglich. 

Die Registerkarten Bearbeiten bzw. Quelltext werden weiterhin nur in änderbaren Einträgen angezeigt. 

Siehe Kapitel Html-Editor . 

Tooltip: 

Html-Editor zur Bearbeitung von Html-Texten. 

### Checkbox Themen und Antworten anzeigen 
Wenn Sie diese Checkbox aktivieren, werden auch Thema- und Antwort-Einträge angezeigt. 

Tooltip: 

Thema- und Antwort-Einträge ebenfalls anzeigen
