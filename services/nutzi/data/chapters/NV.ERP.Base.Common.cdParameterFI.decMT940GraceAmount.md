# Kapitel NV.ERP.Base.Common.cdParameterFI.decMT940GraceAmount

Hier können Standardwerte für die Auszug-/Kassenimport-Bearbeitung definiert werden. Diese Angaben werden in die Auszug-/Kassenimport-Bearbeitung übernommen und können dort temporär überschrieben werden.
