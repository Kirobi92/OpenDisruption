# EDI Auftrag / Auftragsänderung Registerkarte EDI Nachricht 
EDI Auftrag / Auftragsänderung Registerkarte EDI Nachricht 

Auf dieser Registerkarte erhalten Sie Informationen bezüglich der EDI Nachricht. 
### Button Neu mappen 

Mit diesem Button werden die Daten anhand des geänderten Mappings neu verarbeitet. 

### Button Formatieren 

Mit diesem Button werden die Daten im Feld Datei Text formatiert dargestellt. 

### Feld Transferdatum 
Hier wird das Datum des letzten Transfers angezeigt. 

Tooltip: 

Angabe des Transferdatums 

### Feld Dateiname 
Hier wird der Dateiname angezeigt. 

Tooltip: 

Angabe des Serverpfads 

### Feld EDI Nachricht Nr. 
Hier wird die eindeutige Nummer der EDI Nachricht angezeigt. 

Tooltip: 

EDI Nachricht Nr. 

### Feld Nachricht 
Hier wird der Typ der Nachricht angezeigt, z.B. ORDERS. 

### Feld Dateitext 
In diesem Feld wird der gesamte Datei-Text angezeigt. Es wird der Originalinhalt, welcher übermittelt wurde, angezeigt und kann geändert werden damit der Import funktioniert. Die Original-Datei wird hierbei nicht verändert. 

### Button Original Wiederherstellen 

Mit diesem Button lässt sich die Original-Datei wiederherstellen.
