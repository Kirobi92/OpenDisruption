# Parameter Drag and Drop Registerkarte Allgemein 
Parameter Drag Drop Registerkarte Allgemein 

Auf dieser Registerkarte können Sie steuern, was innerhalb eNVenta möglich sein soll, wie z.B. die Option Referenz Links zu erstellen oder zu deaktivieren. 
### Checkbox Deaktiviere Archivmappe einfügen 
Wenn Sie diese Checkbox aktivieren, wird die Option deaktiviert, eine von extern gezogene Datei dem Ziel als Archivmappe hinzuzufügen. 

Tooltip: 

Deaktiviert die Option, eine von extern gezogene Datei dem Ziel als Archivmappe hinzuzufügen. 

### Checkbox Deaktiviere Referenz-Link erstellen 
Wenn Sie diese Checkbox aktivieren, wird die Option deaktiviert Referenz-Links innerhalb von eNVenta zu erstellen. 

Tooltip: 

Deaktiviert die Option Referenz-Links innhalb von eNVenta zu erstellen. 

### Checkbox Deaktiviere Referenz-Link zu Datei erstellen 
Wenn Sie diese Checkbox aktivieren, wird die Option deaktiviert, einen Referenz-Link zu einer von extern auf das Ziel gezogenen Datei zu erstellen. 

Tooltip: 

Deaktiviert die Option, einen Referenz-Link zu einer von extern auf das Ziel gezogenen Datei zu erstellen. 

### Checkbox Deaktiviere Referenz-Link zu URL erstellen 
Wenn Sie diese Checkbox aktivieren, wird die Option deaktiviert, einen Referenz-Link zu einer von extern auf das Ziel gezogenen URL (z.B. Browser-Link) zu erstellen. 

Tooltip: 

Deaktiviert die Option, einen Referenz-Link zu einer von extern auf das Ziel gezogenen URL (z.B. Browser-Link) zu erstellen.
