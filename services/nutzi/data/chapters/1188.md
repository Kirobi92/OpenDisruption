# Neukalkulation 
Neukalkulation 

Die Preise und Rabatte einer oder mehrerer Positionen können jederzeit neu kalkuliert werden. Rufen Sie dazu das Menü im Auftrag >> Optionen >> Neukalkulation auf oder drücken Sie die Tastenkombination <Strg> + <K> . Es erscheint dieser Auswahl-Dialog, in dem Sie Ihre individuelle Neukalkulation gestalten können. Bitte überprüfen Sie zuvor die bereits eingestellten Optionen. 

Beim Neukalkulieren von Verkaufspreisen wird auch der Parameter Kein Rabatt aus dem Artikelstamm in die Auftragsposition übernommen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Neukalkulation 
Tooltip: 

In diesem Dialog können Sie jederzeit Preise und Rabatte mehrerer Positionen neu kalkulieren. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Checkbox Preise neu kalkulieren 
Wenn Sie diese Checkbox aktivieren, werden die Preise und Rabatte neu ermittelt. Dabei werden natürlich alle Konditionen des Kunden und alle Sonderpreise berücksichtigt. 

Die Assortierung wird nur berücksichtigt, wenn Sie alle Positionen neu kalkulieren. 

Tooltip: 

Der Verkaufspreis und die Rabatte / Zuschläge aus den Konditionen werden neu kalkuliert. 
MTZs werden neu ermittelt. 

### Checkbox EK neu kalkulieren 
Mit dieser Option werden nur die Einkaufspreise neu ermittelt. 

### Checkbox R/Z neu kalkulieren 
Wenn Sie diese Checkbox aktivieren, werden alle Positionsrabatte/-zuschläge der Auftragsposition entfernt und komplett neu ermittelt. Bei der Neukalkulation wird mit der jeweils aktuellen Notierung (und ggf. Basis) neu berechnet. 

Manuelle Änderungen an den Rabatten gehen hierdurch verloren. 

Tooltip: 

VK Rabatte / Zuschläge und MTZs neu kalkulieren 

### Checkbox Stücklisten rekursiv kalkulieren 
Mit dieser Option können die Stücklisten neu kalkuliert werden. Hierbei werden der kalkulierte Verkaufspreis und der kalkulierte Einkaufspreis in die Position eingetragen. 

Diese Option kann nur in Verbindung mit der Option Preise und Rabatte aktualisieren genutzt werden. 

Tooltip: 

VK von Stücklisten rekursiv kalkulieren 

### Checkbox Porto neu kalkulieren 
Wenn Sie diese Checkbox aktivieren, wird auch das Porto neu kalkuliert. 

### Checkbox MTZ Refresh 
Mit dieser Option kann die MTZ-Aktualisierung gestartet werden. Er wirkt sich nur auf die MTZ aus, alle anderen Positionsrabatte bleiben unberücksichtigt. 

Diese Checkbox kann nur dann aktiviert werden, wenn weder die Checkbox Preise Neu Kalk noch R/Z Neu Kalk aktiviert ist! 

### Checkbox Alle Positionen 
Wenn diese Checkbox aktiviert ist, werden alle Positionen des Auftrags neu kalkuliert. Nur wenn diese Checkbox gesetzt ist, wird die Assortierung berücksichtigt.
