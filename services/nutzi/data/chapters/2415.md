# Offene Posten Debitoren Registerkarte Selektionen 1 
Offene Posten Debitor Registerkarte Selektionen 1 

Auf dieser Registerkarte steht Ihnen der 1. Teil der Selektionen zur Verfügung. 
### Dock Panel Offene Posten 

Über die Radiobuttons nur offene , rückwirkend , gelöschte und alle auf der Registerkarte Selektionen 1 steuern Sie, welche OPs ausgewählt werden sollen. 

| offene | noch nicht ausgebuchte OPs |
| --- | --- |
| rückwirkend | OPs aus einer zurückliegenden Periode |
| ausgeglichene | ausgeglichene OPs |
| alle | offene sowie ausgeglichene OPs |

### Radiobutton Offene 
Wenn Sie diesen Radiobutton aktivieren, erhalten Sie eine Anzeige aller offenen OPs. 

### Radiobutton Ausgeglichene 
Wenn Sie diesen Radiobutton aktivieren, erhalten Sie eine Anzeige aller ausgeglichenen OPs. 

### Radiobutton Alle 
Wenn Sie diesen Radiobutton aktivieren, erhalten Sie eine Anzeige aller OPs. 

### Radiobutton Rückwirkend 
Wenn Sie diesen Radiobutton aktivieren, wird die Checkbox Teilzahlungen ausblenden auf unzugänglich gesetzt. Die Selektion Rückwirkend weist zur gewählten Periode immer alle OPs aus, d.h. die Teilzahlungs-OPs sind in der Tabelle enthalten. Die Spalten für die OP-Zusatz-Informationen ( Bereits bezahlt, Skto gebucht, Ausb. gebucht und Noch offen ) werden ausgeblendet, weil diese Informationen immer den aktuellen OP-Stand ausweisen und nicht zur rückwirkenden Periode berechnet werden.

### Checkbox Konten ohne OPs 
Wenn Sie diese Checkbox aktivieren, werden Konten ohne offene Posten angezeigt. Für diese Konten kann ebenfalls eine Saldenbestätigung über den Button Bestätigung erstellt werden. 

Tooltip: 

Konten ohne offene Posten anzeigen 

### Checkbox Ausgleichszahlungen ausblenden 
Wenn diese Checkbox 
- aktiviert ist, werden Teilzahlungs-OPs in einigen Dialog-Tabellen (z.B. Einzugsvorschlag, Zahlungsvorschlag, Mahnwesen, OP-Ausbuchung, Bonität, OP-Aufstellung, OP Debitor, Weitere OPs Debitoren, OP Debitor Auswahl, OP Kreditor, Weitere OPs Kreditoren, OP Kreditor Auswahl) ausgeblendet. - deaktiviert ist, werden die Teilzahlungs-OPs weiterhin in den Dialog-Tabellen ausgewiesen. 
Im Dialog Offene Posten Debitoren und Offene Posten Kreditor werden bei Auswahl Offene Posten Ausgeglichene oder Alle auch die Voll-Ausgleiche ausgeblendet, denn alle relevanten Details der Ausgleichsbuchung (egal ob Teilzahlung oder Vollzahlung) werden auf dem Ursprungs-OP in den Spalten der OP-Zusatz-Infos - Bereits bezahlt - Stko. gebucht - Ausb.gebucht - Noch offen 
summiert ausgewiesen.
