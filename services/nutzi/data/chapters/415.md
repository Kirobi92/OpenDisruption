# Html Report Designer Registerkarte Kopf 
Html Report Designer Registerkarte Kopf 

Auf dieser Registerkarte definieren Sie die Kopfdaten eines Html Reports. Der Html Report hat immer ein festes Schema für die gesamte Anzeige. So stehen im Kopfbereich immer 4 Spalten zur Verfügung, welche von links nach rechts angezeigt werden. Die Breite der jeweiligen Spalten wird automatisch ermittelt und richtet sich nach dem angezeigten Inhalt. Hat eine Spalte also keinen Inhalt, so ist die Breite der Spalte 0 und die restlichen Spalten nutzen die verbleibende Breite. 

Unterhalb der 4 Spalten befindet sich immer ein Bereich „quer“, welcher über die gesamte Breite geht. Es empfiehlt sich daher entsprechend längere Texte in diesem Bereich zu schreiben. 

Zum Hinzufügen eines Feldes muss dieses im linken Bereich markiert und entsprechend auf der rechten Seite eingefügt werden. Dies erfolgt mittels der Buttons und . 

Der Aufbau der linken Seite ist die Objektstruktur zum jeweiligen Datenobjekt, welches im Kopfbereich hinterlegt ist. Die Objektstruktur wächst hier beim Customizing mit und stellt daher alle notwendigen Informationen zur Verfügung. 

Nach dem Zuordnen einer Eigenschaft kann diese im rechten Bereich entsprechend den Wünschen angepasst werden. Soll eine abweichende Beschriftung zum Feld angegeben werden, so tragen Sie dies in der Spalte ‚Label‘ ein, ansonsten wird die Standard-Bezeichnung der Eigenschaft angezeigt. 

In der Spalte ‚Format‘ können Sie eine Formatierung der Werte hinterlegen, z.B. ##,00 für Zahlen mit 2 Nachkommastellen. 

In der Spalte ‚Ausrichtung‘ geben Sie an, ob die Anzeige der Werte links-, rechtsbündig oder zentriert erfolgen soll. Wenn Sie keine Angaben machen, sind Zahlen standardmäßig rechtsbündig und Texte linksbündig angeordnet. 

Über die Checkbox in der Spalte 'Ausblenden' werden die Werte trotz Zuordnung nicht im Html Report angezeigt. Die ausgeblendeten Reports werden aber analysiert und der Platz entsprechend reserviert. 

Im Bereich unterhalb der Zuordnung sehen Sie immer die Vorschau für den Bereich des Html Reports. Hierbei werden entsprechende Testdaten/Blindtexte angezeigt.
