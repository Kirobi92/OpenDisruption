Allgemeine Funktionalitäten in eNVenta ERP / Drucken in eNVenta ERP 

# Word- und Office-Vorlagen 
MS Word- / OO Writer-Vorlage 

Dieser Dialog öffnet sich über einen Doppelklick auf einen Word-Eintrag im Dialog Vorlagenstamm . Hier definieren Sie die Dokumentvorlagen für Word- und Office-Dokumente. Dies ist auch in verschiedenen Sprachen möglich. 

Die Dokumentvorlagen werden zum Druckzeitpunkt gefüllt. 

Mit dem Button wird eine neue Vorlage für eine Sprache erstellt. Die Sprache wird aus der Combobox ausgewählt. Pro Sprache ist eine Vorlage möglich. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld MS Word- / OO Writer-Vorlage 
Tooltip: 

In diesem Dialog definieren Sie die Dokumentvorlagen für Word- und Office-Dokumente. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Neuer Satz 

Mit diesem Button wird eine neue Vorlage für eine bestimmte Sprache erzeugt. 

Tooltip: 

Erstellt eine neue Vorlage für eine bereits bestehende Sprache. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Bearbeiten 

Ist bereits eine Vorlage für eine Sprache vorhanden, so kann diese mit einem Klick auf diesen Button geöffnet werden. Es öffnet sich der Dialog Word-Assistent . 

Siehe Kapitel Word-Assistent . 

Ist noch keine Vorlage für die benötigte Sprache vorhanden, so erscheint eine Rückfrage, ob Sie eine neue Vorlage erstellen möchten. Klicken Sie auf Ja wird eine Word-Datei geöffnet und Sie können die neue Vorlage bearbeiten, indem Sie aus dem Dialog Word-Assistent Einträge anklicken und in die leere Word-Vorlage einfügen. 

Das "leere" Dokument bleibt geöffnet, bis Sie die Vorlage nach Ihren Wünschen eingerichet haben. Dann speichern und schließen Sie diese Vorlage und schließen den Word-Assistenten. 

Tooltip: 

Vorlage bearbeiten oder neu erstellen 

### Button Hochladen 

Mit diesem Button können bereits existierende Word-Vorlagen auf den Dokumenten-Server geladen werden. Von hier aus ist dann eine Weiterverarbeitung möglich. 

Tooltip: 

Eine bereits existierende Vorlage wird zum Server übertragen. 

### Feld Vorlage 
Diese Vorlagennummer wird aus dem Dialog Dokument Vorlagen voreingestellt. 

### Feld Beschreibung 
Die Beschreibung der Vorlage wird automatisch zur Vorlagennummer eingelesen. 

### Combobox Archiv-Zuordnung 
Diese Combobox steht zur Archivierung von Word-Dokumenten zur Verfügung. Hier wählen Sie den Archivbereich aus. 

### Tabelle 

Die Tabelle enthält alle bereits angelegten Vorlagen. 
### Tabellenspalte Sprache 
In dieser Spalte wählen Sie die Sprache aus. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Tabellenspalte Format Info 
Hier geben Sie der Vorlage einen Namen bzw. beschreiben, für was die Vorlage gedacht ist. Z.B. ein Anschreiben, eine Einladung. 

### Tabellenspalte Version 
Wenn Sie eine Vorlage überarbeiten oder ergänzen, so können Sie diese Änderung entsprechend kennzeichen und finden so immer genau die richtige Vorlage. 

### Tabellenspalte Dokument Hochgeladen 
Wenn Sie die Checkbox in dieser Spalte aktivieren, wird die zugrundeliegende Datei, z.B. DOT oder ODT gezogen. 

Ohne ein hinterlegtes Dokument kann die Vorlage nicht gespeichert werden. 

### Tabellenspalte Größe (MB) 
Der Wert in dieser Spalte wird automatisch gezogen und hängt ab von der zugrundeliegenden Datei. 

### Tabellenspalte Datum 
Das Datum wird automatisch vom System eingetragen. 

### Tabellenspalte Dateityp 
Der Dateityp wird automatisch vom System eingetragen.
