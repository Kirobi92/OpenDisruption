Stammdaten / Kalkulationsgruppen 

# Kalkulationsgruppen Registerkarte Allgemein 
Kalkulationsgruppe Registerkarte Allgemein 

Auf dieser Registerkarte werden die Details zur Kalkulationsgruppe festgelegt. Dies umfasst die beteiligten Artikelkalkulationen sowie deren Reihenfolge. 
### Tabelle 

### Tabellenspalte Pos 
In dieser Spalte wird die Reihenfolge der beteiligten Artikelkalkulationen angezeigt. Diese kann über die Buttons und geändert werden. 

Tooltip: 

Positionsnummer 

### Tabellenspalte Kalkulation 
Hier wird die Nummer der Artikelkalkulation angegeben. 

Tooltip: 

Artikelkalkulationsnummer 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Erfassung 
Hier wird vom Programm automatisch das Datum der Erfassung eingetragen. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Tabellenspalte Erfasser 
Hier wird der Sachbearbeiter angegeben, der den Datensatz erfasst hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der den Datensatz erfasst hat. 

### Tabellenspalte Änderung 
Hier wird vom Programm automatisch das Datum der letzten Änderung eingetragen. 

Tooltip: 

Datum der letzten Änderung 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Button Satz hinzufügen 

Dieser Button öffnet den Dialog Auswahl Artikelkalkulation zur Selektion einer Artikelkalkulation. Mit dieser werden neue Positionen für die Kalkulationsgruppe erzeugt. 

Siehe Kapitel Auswahl Artikelkalkulation . 

Tooltip: 

Neue Zeile 

### Button Verschieben 

Mit diesem Button können Sie innerhalb der Tabelle eine Zeile nach oben verschieben. 

Tooltip: 

Nach oben verschieben 

### Button Verschieben 

Mit diesem Button können Sie innerhalb der Tabelle eine Zeile nach unten verschieben. 

Tooltip: 

Nach unten verschieben 

### Button Löschen 

Mit diesem Button können Sie einen Eintrag der Tabelle entfernen. 

Tooltip: 

Zeilen löschen
