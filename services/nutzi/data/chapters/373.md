# Artikel Cockpit Registerkarte Artikelstammdaten / Stückliste 
Registerkarte Artikelstammdaten/Stückliste 

Auf dieser Registerkarte werden Artikelstammdaten bezüglich Stückliste angegeben. 
### Combobox Artikeltyp 
Hier wird der Artikeltyp angegeben. 

Tooltip: 

Angabe des Artikeltyps 

### Combobox Autokalk. AStckl. 
Hier wird die automatische Kalkulation von Handelsstücklisten im Auftrag angegeben. 

Tooltip: 

Automatische Kalkulation von Handelsstücklisten im Auftrag 

### Combobox Stücklisten Kalk. 
Hier wird vorbelegt, wie der GEK der Komponenten einer Produktionsstückliste (Kennzeichen 2 in Feld Stckliste ) bei einer Zerlegung im Wareneingang ermittelt werden soll. 

0 - Anteilige Verteilung nach Wert 
1 - Anteilige Verteilung nach Gewicht 

Diese Angabe kann durch eine direkte Prozentangabe in der Produktionsstückliste selbst überschrieben werden. Dort muss die Summe aller Prozentangaben 100% ergeben. 

Tooltip: 

Stücklisten Preis Kallkulation 

### Checkbox Fixes Artikel-Set 
Wenn Sie diese Checkbox aktivieren, ist eine als Set definierte Handelsstückliste im Auftrag nach Auflösung nur im Ganzen änderbar. Die einzelnen Komponentenmengen sind unzugänglich.
