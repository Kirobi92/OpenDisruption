# CSC - Registerkarte Changes

Wissenswertes über eNVenta ERP / Client Side Customizing (CSC) 

# CSC - Registerkarte Changes 
Auf dieser Registerkarte werden alle gespeicherten Änderungen angezeigt. 

Hier können Sie einzelne Änderungen wieder entfernen, wenn Sie die Checkbox in der Spalte 'Delete' aktivieren und anschließend den Button drücken. 

Im unteren Bereich haben Sie die Möglichkeit über einen Filter nach einem Property oder Control zu filtern. Mit dem Button löschen Sie den Filter wieder.
