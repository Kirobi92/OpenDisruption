# Kapitel NV.ERP.Base.Common.cdSystemMM.shtProjectNoProposal

Wenn Sie diese Checkbox aktivieren, wird die entsprechende Checkbox ( Kein Vorschlag ) im Projekt voreingestellt und auf jeden Kunden, der beteiligt ist, übernommen. Wird die Kennzeichnung entfernt, wird sie auch aus den zugeordneten Kunden entfernt. 

Die Einstellung auf Kundenebene hat Priorität vor der Checkbox im Dialog Projekt . 

Tooltip: 

Vorbelegung für die Checkbox "Kein Vorschlag" im Projekt.
