# Kapitel NV.ERP.Base.Customer.cdCustomer.shtTourSeqNumber

Hier geben Sie für die Tour eine Stopp-Nr. innerhalb der Tour ein. Damit definieren Sie, an wievielter Stelle der Kunde beliefert wird. 

Tooltip: 

Fortlaufende Nummer, wann der Stopp innerhalb der Tour angefahren wird.
