# Lagerauftrag 
Lagerauftrag 

Der Dialog Lagerauftrag steht zur Verfügung, um Lageraufträge für eine Umlagerung, Inventur oder bei Fehlerkorrektur zu erstellen. 

### Umlagerung mit Behältern 
Die Umlagerung kann auch über Behälter vorgenommen werden. Dazu wird bei der Buchung der Entnahme des Umlagerungsauftrags eine Behälterzuordnung in den Prozess eingepflegt. Dieser Behälter muss immer leer sein. 

Direkt nach der Auslagerung wird die aktuelle Umlagerung beendet. Bei der automatisierten Umlagerungsreihenfolge wird die nächste Auslagerung geladen. 

Die Buchung endet nach der Behälterbuchung. 

Durch Scannen des Behälters wird automatisch der Zulagerungslagerauftrag wieder geladen. Jetzt kann die Platzeinlagerung durchgebucht werden. Die Ware befindet sich solange auf dem Behälter. 

#### Unterschied zwischen manueller und vorgeschlagener Umlagerung 
Die vorgeschlagene Umlagerung erzeugt nach dem Auslagerungslagerauftrag automatisch einen Einlagerungsauftrag, auf dessen Basis gearbeitet werden kann. 

Die manuelle Umlagerung erzeugt erst zum Zeitpunkt der Buchung einen Lagerauftrag. Die manuelle Umlagerung erzeugt einen Folgelagerauftrag, welcher dann auf den ZZZZZ Platz lautet. 

Die manuelle Umlagerung kann vom Umlagerungsdialog aus erreicht werden.

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
