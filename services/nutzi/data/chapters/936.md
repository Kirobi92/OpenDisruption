Projekt Management / Aufgabenstamm 

# Aufgabenstamm Registerkarte Ressourcen 
Aufgabenstamm Registerkarte Ressourcen 

Auf dieser Registerkarte können Sie einer Aufgabe mehrere Ressourcen zuordnen. Genauso können Einträge beliebig gelöscht werden. 

Eine Ressource kann nur einmal einer Aufgabe zugeordnet werden. 

Eine Ressource kann in mehreren Aufgaben enthalten sein. 

In der Projektaufgabe wird die Ressourcenzuordnung aus dem Aufgabenstamm übernommen. Die Ressourcenzuordnung kann in der Projektaufgabe beliebig verändert werden. 
### Tabelle Ressourcen 

### Tabellenspalte Ressource 
Hier wird eine Standard-Ressource angegeben. 

Tooltip: 

Angabe der Ressource 

### Tabellenspalte Bezeichnung 
Hier wid der Name der Ressource angegeben. 

Tooltip: 

Name der Ressource 

### Button OK 

Mit diesem Button speichern Sie die Eingaben. 

### Button Satz hinzufügen 

Mit diesem Button können Sie einen weiteren Eintrag in der Tabelle einfügen. 

### Button Löschen 

Mit diesem Button löschen Sie einen Eintrag der Tabelle.
