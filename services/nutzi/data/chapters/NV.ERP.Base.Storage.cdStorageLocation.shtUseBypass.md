# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtUseBypass

Wenn Sie diesen Parameter aktivieren, wird für die Einlagerung ermittelt, wie viele Rückstände für den Artikel (welcher eingelagert werden soll) im Lager vorhanden sind. Der Sachbearbeiter kann dann entscheiden, ob er auf den vorgeschlagenen Platz oder auf einen Bypass einlagern will. 

Tooltip: 

Ermittelt die Artikelrückstände für die Einlagerung im Lager.
