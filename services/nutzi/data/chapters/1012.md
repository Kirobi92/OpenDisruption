# Retourenarten verwalten Registerkarte Druck 
Retourenarten Registerkarte Druck 

Auf dieser Registerkarte können Formulare und Drucker für die Retouren festgelegt werden. Damit stehen hier alle Möglichkeiten der eNVenta -Druckbelege zur Verfügung. 

Die Retourenbestätigung kann gedruckt werden, um dem Kunden eine Bestätigung der Anlage einer Retoure zukommen zu lassen (Bsp.: „Beziehen Sie Ihre Retourensendung auf die Nummer…“ ) 

Ist die Abschlusslogik in der Retoure aktiviert, so muss die Retoure abgeschlossen sein, damit der Report gedruckt werden kann. 

Die Retouren-Eingangsbestätigung kann erst gedruckt werden, wenn die Ware eingegangen ist. Dies dient als Bestätigung an den Endkunden, dass die Ware ins Haus gelangt ist. 

Was den Druck von Logos etc. auf den Reports betrifft, so ist folgendes zu beachten: 

Es können alle gängigen Bild-Formate (bmp, gif, jpeg, png, tiff, …) ausgewählt werden. 

JPEG, GIF und PNG werden 1:1 gespeichert und auch so an den Report übergeben. 

Alle anderen nicht internetfähigen Formate (bmp, tiff, …) werden verlustfrei in ein PNG umgewandelt. 

Empfehlung: 

Verwenden Sie PNG als Format. 

Vermeiden Sie nach Möglichkeit Bilder auf Subreports.
