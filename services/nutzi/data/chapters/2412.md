# Belegauskunft 
Belegauskunft 

Im Buchen , Menü Buchen finden Sie den Eintrag Belegauskunft . Die Belegauskunft soll eine flexible Auskunft über Belege zu einem Debitor bzw. Kreditor geben. So können z.B. alle Zahlungen zu einer Belegnummer gesammelt betrachtet werden. 

Die Belegauskunft lässt sich im Dialog Buchen auch über den Shortcut <Strg + Shift + B> öffnen. 

Zunächst selektieren Sie im Kopf des Dialogs nach Debitoren oder Kreditoren. Dann geben Sie eine Kontonummer ein oder suchen diese über den Button . 

Die Selektion kann weiter eingeschränkt werden über Belegnummer (evtl. plus Checkbox Exakt ) und Belegdatum von / bis . Die Suche bei Belegnummer erfolgt über Belegnummer und Belegnummer Verbucht . 

Außerdem kann eine Gegenkontonummer eingegeben werden oder über den Button gesucht werden. 

Es werden alle Zahlungsbewegungen des Journals aufgezeigt. Trifft ein Suchkriterium zu, so werden alle zugehörigen Journalsätze in der Ergebnistabelle angezeigt. 

Hinweis: 
Die Anzeige ist sortiert nach dem Belegdatum. Erfolgten viele Teilzahlungen kann es sein, dass die Übersicht darunter leidet, da viele Zahlungen auf die gleiche Belegnummer erfolgen. 

Tipp: 
Für eine bessere Übersicht der Kontobewegungen des Kunden können Sie die Konto Info , Registerkarte Zahlungen nutzen. In der Konto Info können Sie die Zahlungsverwendung erstellen, z.B. für Ihren Steuerberater. 

Ist dem Debitor oder Kreditor eine Fremdwährung hinterlegt, so erweitert sich die Tabelle um die Spalten 'Kurs', 'Betrag', 'Abzüge', 'Saldo'. 

Ein Drilldown auf die einzelne Zeile ist möglich, sie reicht bis in die Warenwirtschaft bzw. das Archiv.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme).
