# Kapitel NV.ERP.Base.Common.cdSystemMM.shtDiscountSplit

Wenn Sie diesen Parameter aktivieren, werden wertmäßige Rabatte bei Teillieferungen anteilsmäßig auf die beiden Positionen gesplittet. 

Ist der Parameter nicht aktiv, so bleibt der Rabatt auf der Ursprungsposition bestehen und Nachlieferungen werden ohne Rabatte berechnet.
