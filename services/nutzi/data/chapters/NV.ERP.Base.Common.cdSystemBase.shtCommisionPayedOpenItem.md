# Kapitel NV.ERP.Base.Common.cdSystemBase.shtCommisionPayedOpenItem

Ist dieser Parameter gesetzt, wird in der Vertreterabrechnung im Verkauf eine Steuerung aktiviert, mit der Sie die Auswertung auf Nur bezahlte Rechnungen setzen können. Dazu ist zwingend die Checkbox Nur bezahlte Rechnungen im Dialog Provisionsauswertung zu aktivieren. 

Weitere Voraussetzung: die Finanzbuchhaltung in eNVenta muss lizenziert sein! 

Tooltip: 

Aktiviert eine Steuerung in der Vertreterabrechnung auf 'Nur bezahlte Rechnungen'.
