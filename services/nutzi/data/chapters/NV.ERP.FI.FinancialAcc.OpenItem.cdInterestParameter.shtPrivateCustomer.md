# Kapitel NV.ERP.FI.FinancialAcc.OpenItem.cdInterestParameter.shtPrivateCustomer

Wenn Sie diesen Radiobutton aktivieren, wird der Zinsparameter für einen Privatkunden generiert. Ein Kunde wird im Kundenstamm als Privatkunde gekennzeichnet.
