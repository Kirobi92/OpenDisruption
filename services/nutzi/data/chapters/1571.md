# Lagerort-Einstellungen 
Lagereinstellungen 

Unter System >> Logistik finden Sie den Eintrag Lagerort-Einstellungen . Es öffnet sich der Dialog Lagereinstellungen für die Verwaltung / Erzeugung von Picklisten im Logistik-Modul. Diese Picklisten dienen der Optimierung einer Kommissionierung, damit ein Lagermitarbeiter mehrere Aufträge gleichzeitig kommissionieren kann. Über die Anzahl Vorgänge kann angegeben werden, wie viele Aufträge ein Mitarbeiter gleichzeitig bearbeiten kann. 

Das Gewicht und das Volumen grenzen dies ein, hierfür müssen aber die Daten entsprechend in den Stammdaten gepflegt sein. 

Alle vorhandenen Einstellungen werden in der Tabelle aufgelistet. Pro Lager kann eine Einstellung gespeichert werden. Bei Markierung einer Einstellung werden die Daten automatisch eingelesen. 

Bei den Angaben zu Anzahl Vorgänge, Volumen und Gewicht gibt es keine Priorität, der Wert welcher zuerst erreicht wird, beendet die Suche nach weiteren Vorgängen im Pickpool.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
