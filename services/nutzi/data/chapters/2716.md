# Neue Auswertungsstruktur 
Neue Auswertungsstruktur 

Der Button Neuer Report im Menü Auswertungsstruktur Kore öffnet den Dialog Neue Auswertungsstruktur. Hier geben Sie im Feld Bezeichnung eine „sprechende“ Bezeichnung für die neue Auswertungsstruktur an. Diese ist dann in den verschiedenen Auswahlfeldern sichtbar. 

Über die Radiobuttons steuern Sie, von welchem Typ die neue Auswertungsstruktur ist. Wählen Sie zwischen Kostenarten, Kostenstellen und Kostenträger . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Neue Auswertung 
Tooltip: 

In diesem Dialog legen Sie eine neue Auswertungsstruktur an. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Bezeichnung 
Hier geben Sie eine Bezeichnung für die neue Auswertung ein. Diese ist dann in den verschiedenen Auswahlfeldern sichtbar. 

### Dock Panel Auswertungstyp 

Über diese Radiobuttons steuern Sie die Art der Auswertungsstruktur. 

### Field Panel Auswertung kopieren 

Mit der Checkbox Kopieren von erstellen Sie eine Kopie einer bestehenden Auswertungsstruktur. Wählen Sie hierzu aus der Combobox eine gespeicherte Struktur aus. Wenn Sie die Checkbox mit Details aktivieren, werden auch die zugeordneten Kostenarten, Kostenstellen oder Kostenträger auf die neue Struktur übernommen. 
### Checkbox Basis-Zuordnungen 
Tooltip: 

Wenn diese Checkbox gesetzt ist, werden die Basis-Zuordnungen (Konten, Steuerschlüssel, Kostenstellen, ...) transferiert.
