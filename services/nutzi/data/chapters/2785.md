# Parameter MIS Registerkarte Datenquellen 
Parameter MIS Registerkarte Datenquellen 

Auf dieser Registerkarte geben Sie die Service Adresse des MISRemoteService ein. Dies ist der Server, auf welchem die eNVenta_DW-Datenbank läuft. Der Name ist immer MISRemoteService. 

Anschließend wählen Sie im Bereich Quelle Ihre Business-Datenbank aus. Wählen Sie hierzu zuerst den Datenbank Typ SQL-Server oder Oracle und anschließend den Provider. Für eine Oracle-Datenbank benutzen Sie den Provider „Oracle Provider for OLE DB“, für die SQL-Server Datenbank den Provider „Microsoft OLE DB Provider for SQL Server“. Darunter geben Sie den Namen an, sowie die Daten zur Verbindung zum Datenbank-Server. Bei Oracle ist der Datenbank-Name als Server einzutragen. 

Im Bereich Ziel geben Sie die Verbindung zur DataWarehouse-Datenbank an. Hier ist der Datenbank-Typ immer SQL-Server und der Provider „Microsoft OLE DB Provider for SQL Server“. Daneben geben Sie die Daten für den Login am Datenbank-Server an sowie den Datenbanknamen „eNVenta_DW“.
