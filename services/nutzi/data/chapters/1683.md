# Endkontrolle 
Endkontrolle 

Dieser Dialog öffnet sich im Navigator >> Endkontrolle . Hier kann die Endkontrolle für Lagerpositionen/-aufträge durchgeführt werden. 

Eine Endkontrolle der kommissionierten Waren nach der Verpackung macht nur Sinn im Zusammenhang mit der Verpackung während des Kommissionierens. 

Bei einer Verpackung im Anschluss an die Kommissionierung ist es besser, die Endkontrolle vor dem Verpacken durchzuführen. 

Der Prozess dazu sieht folgendermaßen aus: 

- Nach Abschluss der Kommissionierung am Scanner (im Dialog Behälterbuchung ) erfolgt ein Hinweis auf die Endkontrolle bei der Behälterzuordnung. - Buchung der Behälter auf Lagerplätze. - Der Endkontrollbeleg wird mit der letzten Buchung ausgelöst. Statt Packstücknummern können auch Kommissionierbehälter ausgegeben werden. - Der Lagerleiter prüft die Ware und kann sie auf 2 Arten freigeben: 
- Umlagerung der Ware auf einen anderen Platz - Bearbeitung der Endkontrolle am Scanner 
Des Weiteren kann eine Endkontrolle ausgelöst werden, wenn Ware nicht ausreichend am Lager verfügbar ist. 

Wird eine Endkontrolle ausgelöst, so erhält der Sachbearbeiter bei der Behälterbuchung den Platz KLÄRUNG . Die anderen Buchungen erhalten dann wieder den Platz des anderen Behälters (insofern dieser in einem gekennzeichneten Klärungslagerbereich steht). 

Wird am Packplatz eine Behälternummer eines Auftrags mit Kontrollbedarf angescannt, so wird eine Meldung ausgegeben: „Der Auftrag muss endkontrolliert werden, bitte bringen Sie den Behälter in den Klärungslagerbereich.“ 

Ein berechtigter Sachbearbeiter kann die Endkontrolle ganz einfach als I.O. bestätigen, indem er die Ware via Behälterbuchung aus dem Klärungsbereich in einen anderen Bereich ausbucht. 

Das Handling der Mindermengen von Ware durch die Endkontrolle wird durch die Checkbox Mindermengen im Lagerorte auf der Registerkarte LVS Entnahme gesteuert. 

Kommt es während der Kommissionierung zu einer Mengenminderung der Entnahme aufgrund fehlender Bestände, welche nicht durch die Zukommissionierung aus anderen Beständen ausgeglichen werden kann, so wird der Auftrag erst mal nicht zum Verpacken freigegeben. 

Der Sachbearbeiter muss nun die Ware in die Endkontrolle bringen. Hier wird durch entsprechend geschultes Personal entschieden, ob die Ware trotzdem geliefert werden soll (evtl. auch in Rücksprache mit Verkäufer und Kunde), oder ob gewartet werden soll, bis der Auftrag wieder mit Ware versorgt werden kann.

### Button Drucken 

Dieser Button startet den Druck. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button OK 

Wenn Sie diesen Button drücken, ist die Kommissionierung fehlerfrei durchgelaufen. Diese Information wird im Protokoll gespeichert. 

### Button Fehler 

Wenn Sie diesen Button drücken, ist die Kommissionierung mit Fehlern durchgelaufen (mindestens 1 Fehler ist aufgetreten). Diese Information wird im Protokoll gespeichert. 

### Feld Barcode / Kontrollnr. 
In dieses Feld tragen Sie einen der drei Barcodes von der Endkontrolle ein: 

- Barcode 1 --> Auftragsnr + .+ O (alle in Ordnung) oder - Barcode 2 --> Auftragsnr - Barcode 3 --> Auftragsnr + .+ F (Fehler in mindestens 1 Position) 
So kann der Vorgang geladen und bearbeitet werden.
