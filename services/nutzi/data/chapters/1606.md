# Lagerzone Registerkarte Kommissionierbehältertypen 
Lagerzone Registerkarte Kommissionierbehälter 

Auf dieser Registerkarte können Standard-LHM-Typen angelegt und verwaltet werden. Mit Klick auf den Button kann ein LHM-Typ angelegt werden. 

In der ersten Spalte wird unmittelbar eine fortlaufende Nummer vom System generiert. 
### Tabelle 

### Tabellenspalte LHM Typ 
Aus der Combobox in dieser Spalte wählen Sie einen LHM-Typen aus. 

Tooltip: 

Ladehilfsmittel-Typ. 

### Button Hinzufügen 

Mit Klick auf diesen Button können Sie einen weiteren LHM-Typen in der Tabelle erfassen. 

Die erste Spalte erhält unmittelbar eine fortlaufende Nummer vom System. 

### Button Entfernen 

Mit diesem Button wird eine Zeile der Tabelle entfernt.
