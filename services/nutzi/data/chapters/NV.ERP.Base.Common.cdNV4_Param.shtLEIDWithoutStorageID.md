# Kapitel NV.ERP.Base.Common.cdNV4_Param.shtLEIDWithoutStorageID

Wenn Sie diese Checkbox aktivieren, wird ein Ladehilfsmittel ohne Lagernummer erzeugt. Ist die Checkbox nicht gesetzt, wird die Lagernummer zum LHM generiert. 

Tooltip: 

Ladehilfsmittel werden ohne Lagernummer in der ID erzeugt. Dadurch sind sie lagerübergreifend einsetzbar.
