# Lieferant Registerkarte Logistik 
Lieferant Registerkarte Logistik 

Auf dieser Registerkarte definieren Sie Einstellungen zu Dispo-Kreis, Stammtour und Beistell-Materiallager. 
### Button TE-Zuordnung 

Mit diesem Button können Sie dem Projekt Transportgeräte zuordnen. 

Eine Zuordnung auf Bestellanschrift "sticht" eine Zuordnung auf Lieferantenebene. 

Diese Zuordnungen werden automatisch in die Bestellung übernommen und sorgen dort je nach Konfiguration für eine abweichende Tourfindung bei einer Abholung. 

Siehe Kapitel Erweiterte Technische Entladung . 

Tooltip: 

Technische Entladungszuordnung 

### Combobox Dispo-Kreis 
Aus dieser Combobox wählen Sie den Dispositionskreis zur regionalen Abgrenzung. Die Anlage der Dispositionskreise erfolgt in einer Pickliste. 

Tooltip: 

Angabe des Dispositionskreises zur regionalen Abgrenzung 

### Combobox Stammtour 
Hier geben Sie an, mit welcher Tour (Route) (z.B. täglich, wöchentlich) der Kunde bzw. Lieferant immer (regelmäßig) beliefert werden soll. Die Touren, die zur Verfügung stehen, werden im Tourenstamm gepflegt. 

Siehe Kapitel Tourenstamm . 

Tooltip: 

Angabe der Stammtour 

### Combobox Beistell'M'Lager 
Hier wird das Materiallager für Beistellungen angegeben. Dabei handelt es sich um das Quell-Lager des Beistellauftrags. Wenn der Lieferant des dazugehörigen EK-Prozesses dieses Lager hinterlegt hat, wird der Beistellauftrag für dieses Lager erfasst und entsprechend von da abgebucht.
