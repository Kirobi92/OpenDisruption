# Aufgabe auswählen 
Aufgabe auswählen 

Das Menü Aufgabe auswählen zeigt in einer Übersicht alle bereits angelegten Aufgaben an. 

Dieser Dialog öffnet sich im Aufgabenstamm Registerkarte Allgemein über den -Button neben dem Feld Aufgabe Nr. bzw. im Projekt Registerkarte Phasen über den Button Neu , wenn der Menüpunkt Aufgabe markiert ist. 

### Anlegen einer neuen Aufgabe 
Eine neue Aufgabe kann angelegt werden, wenn der Eintrag „Aufgaben“ einer Phase ( Projekt Registerkarte Phase ) markiert ist und Sie den Button Neu drücken. Mit dem Button Neu öffnet sich der Dialog Aufgabe auswählen , wo Sie die passende Aufgabe übernehmen. Siehe nachfolgendes Beispiel: 

Neue Aufgabe anlegen - Beispiel 

Aus dem Dialog Aufgabe auswählen selektieren Sie die benötigte Aufgabe. Diese kann entsprechend Ihrer Bedürfnisse geändert werden. 

## Steuerelemente des Formulars 

### Bezeichnungsfeld Aufgabe auswählen 
Tooltip: 

In diesem Dialog werden alle bereits angelegten Aufgaben zu einem Projekt angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog.
