# Reproduktionsaufträge 
Repr. Aufträge 

Dieser Dialog öffnet sich im Buchen Kore >> Stamm >> Reproduktionsaufträge . 

Kostenreproduktionsaufträge dienen der einfachen Fortschreibung von Kosten/Leistungen auf andere Kostenarten oder Leistungsarten. Hiermit kann aus dem BAB ein Kostenbereich / Fragment selektiert werden und dessen Wert als eine „virtuelle“ Kosten-/Leistungsart auf eine „virtuelle“ Kostenstelle kopiert werden. 

Damit verschiedene Kosten- und Leistungsarten, die auf einer Quell-Kostenstelle vorhanden sind, über mehrere Umlagedefinitionen verschieden umgelegt werden können, steht die Spalte 'Umlage' zur Verfügung. Wenn "Von Quell-Kst." und "Bis Quell-Kst." identisch sind, wird die Checkbox in der Spalte 'Umlage' editierbar. 

Bei Reproduktion-Buchen wird bei Reproduktionsaufträgen mit aktivierter Checkbox Umlage eine zusätzliche Buchung für die Umlage durchgeführt. Dabei wird dann der negierte Buchwert der regulären Reproduktion verwendet und auf die Quell-Nr. und die Quell-Kostenstelle gebucht. Buchungen von Planwerten werden analog umgelegt. Diese 'Umlagen'-Buchung erhält denselben Buchungstext und dieselbe Herkunft wie reguläre Reproduktionen. 

Über das Feld Bezeichnung und die Combobox Gruppe können Sie die Tabelle für die Suche einschränken. 

In die Spalte ‚Bezeichnung‘ geben Sie eine „sprechende“ Bezeichnung des Reproduktionsauftrages ein. 

Über den Quelltyp wird gesteuert, ob eine Kostenart (KA) oder eine Leistungsart (L) gebucht wird. 

In der Spalte ‚Zielnummer‘ wählen Sie die Kostenart oder Leistungsart aus. 

In den Spalten ‚Quell-Nr. von‘ und ‚Quell-Nr. bis‘ wählen Sie Kostenarten aus, von welchen der Saldo übernommen werden soll. Zusätzlich können Sie dies über die Quell-Kostenstellen weiter einschränken. 

In die Spalte ‚Prozent‘ geben Sie den Prozentsatz ein auf den Saldo der Kosten / Leistungen, welcher auf die Ziel-Kosten / -Leistungen gebucht werden soll (z.B. vom Ursprung 30% oder 200%). 

Zusätzlich können Sie am Ende der Tabelle den Reproduktionsauftrag einer Reproduktionsgruppe zuweisen, um diese später gesamt zu verbuchen.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
