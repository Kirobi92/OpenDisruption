# Kostenarten-/Leistungsverteilung 
Kostenarten-/Leistungsverteilung 

Diesen Dialog finden Sie im Buchen Kore >> Auswertungen >> Kostenarten/Leistungsverteilung . Hier wird angezeigt, wie Kostenarten bzw. Leistungen verteilt sind. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Kostenarten-/Leistungsverteilung 
Tooltip: 

Dieser Dialog zeigt, wie Kostenarten bzw. Leistungsarten verteilt sind. 
Für mehr Informationen drücken Sie F1.

### Checkbox Kostenarten 
Falls Sie eine Verteilung auf Kostenarten wünschen, aktivieren Sie diese Checkbox. 

### Checkbox Leistungen 
Falls Sie eine Verteilung auf Leistungen wünschen, aktivieren Sie diese Checkbox. 

### Checkbox Nicht verteilte anzeigen 
Wenn Sie diese Checkbox aktivieren, werden zusätzlich Kostenarten in der Tabelle angezeigt, welche der Selektion entsprechen, aber noch keine Verteilung auf Kosten- oder Leistungsart besitzen. 

### Checkbox Nur nicht verteilte anzeigen 
Wenn Sie diese Checkbox aktivieren, werden nur 'nur nicht verteilte' Kostenarten angezeigt. Dies ist dann von Vorteil, wenn Sie prüfen möchten, ob alle Kostenarten korrekt zugeordnet wurden.
