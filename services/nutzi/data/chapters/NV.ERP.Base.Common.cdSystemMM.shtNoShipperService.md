# Kapitel NV.ERP.Base.Common.cdSystemMM.shtNoShipperService

Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Speditionstouren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Keine Spedition“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Tooltip: 

Keine Lieferung per Spedition
