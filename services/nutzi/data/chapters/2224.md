# Call-Statistik 
Call-Statistik 

Dieser Dialog öffnet sich im Call >> Optionen >> Call-Statistik . Dieser Dialog gibt Ihnen Auskunft über die Dauer, die Gründe und die Kosten Ihrer Telefonate. Diese Informationen sind aufgeschlüsselt nach Mitarbeitern, Kunden, Lieferanten oder anderen von Ihnen gewünschten Parametern und dargestellt in übersichtlichen Grafiken. 

Hier steht Ihnen eine umfangreiche Auswertung aller erfassten Calls zur Verfügung. Sie haben hier die Möglichkeit, nach verschiedenen Kriterien eine Statistik zu erstellen. Dazu stehen die Felder im Kopf des Dialogs zur Verfügung.

Tooltip:

### Field Panel Auswahl des Zeitraums 

In diesem Bereich können Sie über einen Zeitraum suchen. Dazu stehen die Radiobuttons, bzw. die Felder Vom und Bis zur Verfügung. 
### Bezeichnungsfeld Zeitraum 
Tooltip: 

Geben Sie den Zeitraum für die Statistik an. 

### Radiobutton Periode 
Wenn Sie diesen Radiobutton drücken, können Sie aus der danebenstehenden Combobox den Zeitraum auswählen. 

### Combobox Auswahl 
Diese Combobox bietet die Möglichkeit über Tag, Monat, Quartal und Jahr zu selektieren. 

### Radiobutton Von / Bis 
Wenn Sie diesen Radiobutton aktivieren, geben Sie in die Felder Von / Bis einen individuellen Zeitraum ein. 

### Feld von 
Hier werden ein Datum und eine Uhrzeit angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld bis 
Um die Suche über einen Datumsbereich einzuschränken, wird hier ein Bis-Datum angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Combobox Periode 
Zur Selektion kann hier eine Periode ausgewählt werden. Die Periode gibt das Intervall innerhalb des Zeitraums an; also pro Monat oder pro Tag. 

Hier erfolgt die Auswertung über die Felder Von … Bis Datum . 

Wird nur das Bis-Datum gefüllt, wird der Zeitraum geleert, das Von – Bis Datum bestimmt dann den Zeitraum. 

Sollte bei der Umsatzliste ein Zeitraum größer 1 Jahr ausgewählt werden, so wird der Vorzeitraum zum Datum von/bis minus Auswertungszeitraum gerechnet. 

Beispiel: 
Umsatzliste von 01.01.2019 - 31.12.2021 
Vorjahreszeitraum ist nun 01.01.2014 - 31.12.2018 

Tooltip: 

Hier wählen sie die Periode aus. 

### Checkbox Periodenanfang frei 
Mit dieser Checkbox können Sie den Anfang der Periode frei definieren. 

Bei nicht-aktiver Checkbox wird das eingegebene Datum der Periode angepasst. 

Beispiel: 
Bei der Periode "Woche" wird als Startdatum ein Mittwoch angegeben. Nun zeigt eNVenta dennoch den Zeitraum zwischen dem Montag davor und dem Sonntag danach an. 

Bei aktiver Checkbox wird das Startdatum nicht angepasst. 

Beispiel: 
Bei der Periode "Woche" wird als Startdatum ein Mittwoch angegeben. Nun zeigt eNVenta den Zeitraum zwischen dem Mittwoch und dem Dienstag danach. 

Beispiel: 
Periode Woche, Von-Datum 23.04. --> es wird die Woche von Mittwoch, den 23.04. bis Mittwoch, den 30.04. gerechnet und nicht die KW!
