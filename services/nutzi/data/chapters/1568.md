# Tourenauswertung 
Tourenauswertung 

Über Navigator >> Versand >> Tourenauswertung öffnet sich der Dialog Tourenauswertung . Über diesen Dialog ist es möglich Touren anzeigen zu lassen und auszuwerten. Über die Felder im Kopf können Sie selektieren, welche Daten in der Tabelle angezeigt werden sollen. Mit einem Doppelklick in eine Zeile der Tabelle öffnet sich der Dialog Tourenauswertung Detail . 

Siehe Kapitel Tourenauswertung Detail . 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing

Siehe Kapitel Client Side Customizing .

Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist.

### Button Drucken 

Über diesen Button kann ein Report gedruckt werden. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Neuer Satz 

Mit diesem Button können Sie der Tabelle einen neuen Eintrag hinzufügen. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button SQL-Suche 

Mit diesem Button starten Sie die Suche per SQL. 

Siehe Kapitel Suchbedingung / Suche mit SQL . 

### Button Einstellungen 

Dieser Button öffnet den Dialog Einstellungen , um weitere Einstellungen für die Arbeit mit einem bestimmten Programmteil vorzunehmen. 

Siehe Kapitel Einstellungen .

### Feld Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf .
