Finanzbuchhaltung / Debitoren Factoring 

# Debitoren Factoring Registerkarte Kundendaten 
Debitoren Factoring Registerkarte Kunden Daten 

Diese Registerkarte enthält Kunden, für die das Factoring aktiv ist und noch nicht gemeldet wurden. 
### Tabelle 

### Tabellenspalte Factoring Status 
Zur Verfügung stehen N für neu, A für Änderung und L für Löschung. 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben. 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Konto 
Hier geben Sie die Kontonummer ein, unter welcher der Kunde in der Buchhaltung als Debitor geführt wird. 

Bei der Neuanlage wird das Feld Konto automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt, kann alternativ aber manuell vergeben werden. 

Siehe Kapitel Konto Info . 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Grundeinstellungen eNVenta ERP / Parameter Allgemein . 

Das Führen mehrerer Debitoren mit identischer Kontonummer ist nicht erlaubt. 

Tooltip: 

Das Konto zum Datensatz.

### Tabellenspalte USt-IdNr. 
Hier wird die Umsatzsteuer-Identifikationsnummer des Kunden angegeben. 

Ist die Funktion System >> Parameter Allgemein >>UST-ID via WS prüfen aktiviert, so wird die Umsatzsteuer-Identifikationsnummer bei der Eingabe in den Dialogen Kunde, Lieferant, Angebot, Auftrag, Gutschrift, Bestellung etc. per Webservice geprüft. 

Achtung: Diese Prüfung ist rechtlich nicht bindend. Eine rechtlich bindende Prüfung der Umsatzsteuer-Identifikationsnummer kann nur schriftlich bei der zuständigen Behörde erfolgen. 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Umsatzsteuer-Identifikationsnummer
