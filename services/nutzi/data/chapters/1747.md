# Bestellungen Registerkarte Abschluss 
Registerkarte Abschluss 

Um die Abschluss-Logik zu aktivieren, wird in den Parametern des Einkaufs die Checkbox Bestellabschluss aktiviert. Beim Speichern der Bestellung wird automatisch die Gesamtsperre gesetzt, sie kann nicht manuell verändert werden. In der Bestellung wird die Registerkarte Abschluss aktiv. 

Folgendes wird dabei geprüft: 

- Mindestbestellwert des Lieferanten. Wenn nicht erreicht, wird eine Warnung ausgegeben. - Frachtzuschlag ab Wert. Wenn nicht vorhanden, wird eine Information ausgegeben. - Positionsmenge. Wenn diese nicht der auftragsbezogenen Menge entspricht, wird eine Warnung ausgegeben. - Soll-Liefertermin. Wenn dieser vor der Wiederbeschaffungszeit liegt, wird eine Warnung ausgegeben. - Bestelladresse. Wenn diese nicht korrekt ist, erscheint eine Messagebox, die Sie darauf hinweist und zur Korrektur auffordert. - Lieferadresse. Wenn diese nicht korrekt ist, erscheint eine Messagebox, die Sie darauf hinweist und zur Korrektur auffordert. 
Gibt es keine Fehler, kann mit dem Button Abschluss die Bestellung freigegeben werden. 

Die Sperre kann manuell nicht aufgehoben werden; sie kann nur über den Button Abschluss aufgehoben werden. 

Der EDI-Versand einer Bestellung kann auf 2 verschiedene Arten ausgeführt werden: 

- Der Bestellabschluss im Menü Parameter Einkauf (Registerkarte Allgemein ) ist aktiviert: Die Bestellung wird erfasst und wenn sie vollständig ist, wird über den Button Abschluss auf der gleichnamigen Registerkarte die Bestellung abgeschlossen und der Versand per EDI angestoßen. Auf der Registerkarte Abschluss wird der Button Start Proz sichtbar. Über diesen Button können eine Bestellung oder einzelne Positionen der Bestellung nachträglich geändert und versendet werden. Zusätzlich erfolgt eine Abfrage, ob die Bestellung wirklich per EDI versendet werden soll, damit nicht aus Versehen ein doppelter Versand erfolgt (z.B. EDI und Fax-Versand). Diese Meldung erscheint nur, wenn in den EDI Parametern die Checkbox Bestellmeldung aktiviert ist. - Der Bestellabschluss im Menü Parameter Einkauf (Registerkarte Allgemein ) ist nicht aktiviert: Damit eine Bestellung nicht sofort nach dem Speichern versendet wird, muss auf der Registerkarte Kopfdaten die Checkbox Kein EDI aktiviert werden. Jetzt bearbeiten Sie die Bestellung, bis diese vollständig und für den EDI-Versand bereit ist. Dann deaktivieren Sie die Checkbox Kein EDI , speichern die Bestellung mit dem Button und lösen so den Versand der Bestellung per EDI aus. Diese Vorgehensweise gilt auch für Änderungen an einer bestehenden Bestellung bzw. einzelner Positionen.

### Button Abschluss 

Gibt es keine Fehler, wird der Vorgang mit diesem Button freigegeben. 

### Button Start Proz. 

Über diesen Button ist es möglich einen Vorgang oder einzelne Positionen daraus nachträglich zu ändern und zu versenden. 

Zusätzlich erfolgt eine Abfrage, ob die Bestellung wirklich per EDI versendet werden soll, damit nicht aus Versehen ein doppelter Versand erfolgt (z.B. EDI und Fax-Versand). Diese Meldung erscheint nur, wenn in den EDI Parametern die entsprechende Checkbox aktiviert ist.

### Feld Freig. Sachb. 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat.
