# Kapitel NV.ERP.Base.Common.cdSystemMM.shtPServiceAutoResponse

Wenn dieser Parameter aktiviert ist, werden bei der Übermittlung von Daten an Paketdienstleister automatisch die vorhandenen Rückmeldedaten des Paketdienstleisters eingelesen.
