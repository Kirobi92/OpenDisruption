# Format Stückliste 
Format Stückliste 

eNVenta ERP bietet die Möglichkeit, die Stücklisten auch auf den auftragsbegleitenden Formularen (Angebot ... Rechnung) auszuweisen. Da dies jedoch nicht immer gewünscht und vor allem von Formularart zu Formularart verschieden sein kann, gibt es eine flexible Möglichkeit, dies festzulegen. Unter dem Menü System >> Verkauf >> Format >> Format Stücklisten können Sie die Parameter für den Druck von Stücklisten anlegen. 

Grundsätzlich steht in jedem Auftrags- oder Angebotsformular das Feld Stückliste zur Verfügung. Das Feld enthält die komplette, fertig formatierte Stückliste als Langtext. In Crystal Reports kann für jedes Formular entschieden werden, ob das Feld gedruckt werden soll oder nicht. 

Zur Festlegung eines Druckformats gehen Sie folgendermaßen vor: 

Wählen Sie zuerst das Formular, indem Sie den gewünschten Radiobutton anklicken. Das aktuelle Format wird im Feld Key2 angezeigt. Ändern Sie jetzt die Definition im Feld Zeile . Per Doppelklick in die Tabelle daneben können Sie ein Property auswählen, damit der entsprechende Wert dann auf dem Report gedruckt wird. Ebenso können Sie einen fixen Text hinterlegen, z.B. "Stückliste bestehend aus" (ohne Anführungszeichen). 

Erfolgt der Druck des Felds in einer Proportionalschrift, kann es, bedingt durch unterschiedlich breite Zeichen, zu Verschiebungen von Zeile zu Zeile kommen, d.h. die Felder stehen nicht exakt untereinander. 

Um die Felder im Druck exakt untereinander zu stellen, können Formatierungsoptionen gesetzt werden, um das Ergebnis zu verbessern: 

z.B. [name,R15] 
R = Rechtsbündig (auch L = Linksbündig ist möglich) 
15 = minimale Länge des Feldes (falls die Länge des übergebenen Feldwerts kleiner als diese Zahl ist, wird mit Leerzeichen aufgefüllt) 

So ein Eintrag könnte dann folgendermaßen aussehen: 

[QuantityOrdered,R10] [QuantityUnitSales,L8] [ArticleID,L20] [ArticleName,L30] 

Wenn Sie Feldnamen manuell in das Feld eintragen, achten Sie unbedingt darauf, dass die Schreibweise exakt eingehalten wird, da sonst das Feld beim Druck nicht erkannt wird. 

In den Zusatzformularen können KEINE Stücklisten gedruckt werden, obwohl dort ebenfalls das Feld Stückliste vorhanden ist.

### Dock Panel Formular 

In diesem Bereich wählen Sie das Formular aus. 

Über den Radiobutton Freies Formular kann für den Druck auch ein individuelles Formular gewählt werden. Dieses steht dann im Auftrag unter Menü Druck >> Formulare / Listen (Dialog Drucken ) zur Verfügung. 

Siehe Kapitel Druck Formulare / Listen . 

### Feld Zeile 
In dieser Zeile kann für das Druckformat auch ein beliebiger freier Text eingegeben werden, der ebenfalls im Druck erscheint. Dieser Text kann max. 200 Zeichen umfassen. 

Bitte beachten Sie, dass eine zu lange Textzeile beim Druck automatisch auf die im Formular festgelegte Breite umgebrochen wird. 

### Feld Schlüssel 1 
Anzeige, dass die Einstellungen nur für Stücklisten gelten. 

### Feld Schlüssel 2 
Wenn Sie ein Formular gewählt haben, wird hier das aktuelle Format angezeigt. Jetzt können Sie im Feld Zeile die Definition ändern.
