# Leistungen erfassen 
Leistungen erfassen 

Der Button Leistungserf. im Dialog Aufgabe ändern auf der Registerkarte Termine öffnet diesen Dialog zu einem neu angelegten Termin und erlaubt die Erfassung von Leistungen für den selektierten Termin. 

Beim Öffnen des Dialogs werden automatisch folgende Leistungen entsprechend der Termindaten eingetragen: 

- Leistung für die Arbeitszeit - Leistung für die Reisezeit - Leistung für die Überstunden - Leistung für das Fahrzeug 
Pro Termin können mehrere Leistungen gebucht werden und die Summe der Leistungsmenge dieser Buchungen wird im selektierten Termin als Arbeitszeit gespeichert. Die Ist-Zeit der Projektaufgabe selbst wird auch entsprechend angepasst. Umgekehrt findet jedoch kein Abgleich statt. D.h. wenn die Arbeitszeit in einem Termin geändert wird, hat das keine Auswirkungen auf die erfassten Leistungen. Bei der ersten Leistungserfassung für einen Termin wird die Arbeitszeit als Leistungsmenge vorbelegt. Ab der zweiten Leistungserfassung für einen Termin findet keine Vorbelegung mehr statt. 

Damit ein Sachbearbeiter die Berechtigung zum Ändern des Status hat, muss er einer Berechtigungsgruppe zugeordnet sein, für welche die Checkbox LE genehmigen aktiviert ist. Entsprechend muss ein Benutzer einer Berechtigungsgruppe zugeordnet sein, die LE ausweisen aktiviert hat, um das Kennzeichen für das Ausweisen einer Leistung sehen und ändern zu können. Dieses Kennzeichen kann auch auf der Projektaufgabe gesetzt werden und gilt dann als Vorbelegung für die Leistungen. 

Im Sachbearbeiterstamm gibt es außerdem die Checkbox Genehm.zwang LE. Wenn diese aktiv ist, werden alle erfassten Leistungen (zum Projekttermin) dieses Sachbearbeiters standardmäßig auf den Status Zu prüfen gesetzt. 

Siehe Kapitel Berechtigungsgruppe . 

Die Leistungserfassung über den Button funktioniert solange, bis alle 4 Leistungstypen erfasst wurden. Danach müssen die Leistungen ggf. einzeln modifiziert werden, indem sie z.B. mit einem Doppelklick in der Leistungstabelle unterhalb der Termine geöffnet werden. Hier kann dann auch der Status der Leistungserfassung geändert werden. 

Bevor Sie speichern, sind zwingend noch die Leistungsartennummer und die Leistungsmengen zu vervollständigen. 

Die genehmigten Leistungen werden dann auf der Registerkarte Termine in der unteren Tabelle angezeigt. 
## Steuerelemente des Formulars

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
