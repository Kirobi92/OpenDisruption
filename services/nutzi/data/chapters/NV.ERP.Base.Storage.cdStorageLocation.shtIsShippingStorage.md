# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtIsShippingStorage

Wenn Sie diesen Parameter aktivieren, ist das Lager als Versandlager gekennzeichnet. 

Ware wird immer vom Versandlager geliefert. Ist dort nicht ausreichend Bestand, wird die Ware vom Bezugslager auf das Versandlager umgelagert. 

Tooltip: 

Im Postleitzahlenstamm können nur Lager ausgewählt werden welche als Versandlager gekennzeichnet sind. Hiermit werden z.B. Streckenlager, Abhollager von der Zuordnung PLZ und Versandlager ausgeschlossen.
