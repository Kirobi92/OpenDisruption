# Lagerauftrag Registerkarte Positionen

Lagerverwaltungssystem (LVS) / Lagerauftrag 

# Lagerauftrag Registerkarte Positionen 
Lagerauftrag Registerkarte Positionen 

Auf dieser Registerkarte werden die Details des Lagerauftrags gepflegt. 
### Button Pos. Neu 

Dieser Button öffnet den Dialog Manuelle Lagerauftragsgenerierung . So kann der authorisierte Lagermitarbeiter eine Umlagerung oder Inventurbuchung durchführen. 

Siehe Kapitel Manuelle Lagerauftragsgenerierung . 

Tooltip: 

Mit diesem Button können Sie einen manuellen Lagerauftrag erfassen. 

### Button Buchen 

Dieser Button bucht die Lagerauftragsposition. D.h. die Position wird abgeschlossen und jede dahinter hängende Logik wird ausgeführt. 

Tooltip: 

Führt Lagerbuchungen für die ausgewählten Lagerauftragspositionen durch. 

### Button Korrektur 

Dieser Button setzt den Status der gewählten Position zurück, um Korrekturen an der Position (bei Fehlbuchungen) vornehmen zu können. 

Tooltip: 

Setzt eine LA-Position in den Anfangszustand zurück. 

### Button Storno 

Dieser Button storniert eine Lagerauftragsposition. Sie kann dann nicht mehr bearbeitet werden. Wenn es sich um eine Kommissionierung handelt, die in Bearbeitung ist, wird eine Umlagerung erzeugt, der den vorherigen Vorgang rückgängig macht. 

Tooltip: 

Storniert die Lagerauftragsposition. Kein weiteres Bearbeiten mehr möglich! 

### Button Behälterzuord. 

Dieser Button zeigt die durchgeführten Buchungen zur ausgewählten Lagerauftragsposition an. 

Siehe Kapitel Lagerauftragsjournal im LVS . 

Tooltip: 

Ordnet der LA-Position den Standardcontainer zu. Ist die LA-Position noch nicht gebucht, wird zusätzlich die Lagerbuchung durchgeführt! 

### Button Journal 

Dieser Button zeigt die durchgeführten Buchungen zur ausgewählten Lagerauftragsposition an. 

Siehe Kapitel Lagerauftragsjournal im LVS . 

Tooltip: 

Hier sehen Sie alle Lagerbuchungen, die durch diese Lagerauftragsposition durchgeführt wurden. 

### Button Platzzuordnungen 

Dieser Button ermöglicht eine Übersicht der Platzzuordnungen der ausgewählten Lagerauftragsposition im chaotischen Lager. 

Siehe Kapitel Plätze Auslagerung . 

Tooltip: 

Hier sehen Sie alle Platzzuordnungen für diese Lagerauftragsposition. 

### Button Bestand 

Dieser Button öffnet die Bestandsübersicht zu dem selektierten Artikel. 

Siehe Kapitel Bestandsanzeige . 

Tooltip: 

Hier sehen Sie den Bestand des Artikels der ausgewählten Lagerauftragsposition. 

### Button Referenz 

Dieser Button öffnet den zugrundeliegenden Vorgang. 

Tooltip: 

Hier öffnen Sie den zugrundeliegenden Beleg (Auftrag / Bestellung), die dieser LA-Position zugeordnet ist. 

### Tabelle Positionen 

Diese Tabelle enthält alle Positionen gemäß den gewählten Kriterien.

In dieser Tabellenspalte wird der Pick-Status des Lagerauftrags angegeben:

•Bearbeitung 
•erledigt

### Tabellenspalte Gesperrt 
Wenn Sie diese Checkbox aktivieren, ist der Kommissioniervorgang gesperrt. Sie wird gesetzt, wenn die Auftragsposition "liefergesperrt" ist. 

Tooltip: 

Aktiviert: Kommissioniervorgang ist gesperrt.
