# Artikelbildverwaltung Registerkarte Datenbankbilder exportieren 
Artikelbildverwaltung Registerkarte Datenbankbilder exportieren 

Auf dieser Registerkarte können Sie Artikelbilder aus der Tabelle Artikel in eine Datei ([präfix] + Artikelnr + [suffix] + ".png") exportieren. 
### Button Bilder exportieren 

Über diesen Button können Datenbankbilder exportiert werden. 

### Feld Pfad 
Hier geben Sie den lokalen oder den Server-Pfad ein, unter dem die exportierten Bilder gespeichert werden. 

Tooltip: 

Lokaler- oder Serverpfad, unter dem die exportierten Bilder gespeichert werden. 

### Button Öffnen 

Mit diesem Button können Sie den Pfad suchen. Es öffnet sich der Dialog Ordner auswählen . 

Siehe Kapitel Ordner auswählen . 

### Feld Präfix 
Hier können Sie ein Präfix eingeben, d. h. was vor der Artikelnummer angefügt werden soll. 

Tooltip: 

Legen Sie hier fest, was vor der Artikelnummer angefügt werden soll. 

### Feld Suffix 
Hier können Sie ein Suffix eingeben, d.h. was nach der Artikelnummer dem Dateinamen hinzugefügt werden soll. 

Tooltip: 

Legen Sie hier fest, was nach der Artikelnummer dem Dateinamen hinzugefügt werden soll. 

### Combobox Dateityp 
Hier wählen Sie den Dateityp aus. 

### Checkbox Auto. Referenz 
Wenn Sie diese Checkbox aktivieren, werden die Artikelbilder automatisch als Dateizuordnung hinzugefügt. Die exportierten Bilder werden als Default-Bild angelegt. Dazu wird die Referenz Link-ID des Bildes in die Tabelle Artikel im Feld Bilddatei gespeichert.
