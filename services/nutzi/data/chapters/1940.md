# Nummernkreise Produktion 
Nummernkreise Prod. 

Dieser Dialog öffnet sich über System >> Produktion >> Nummernkreise . Hier werden alle für die Produktion relevanten Nummernkreise hinterlegt. Als Bezeichnung in den Format-Feldern bedeutet: 

N = Nummer (für jede Stelle ein N, bei einer 5-stelligen Zahl also NNNNN) 
J = Jahr 
M = Monat 
C = Counter 

Um einen Überlauf von Nummernkreisen zu vermeiden und einen "Neustart" bei einem Jahreswechsel zu ermöglichen, gibt es folgende Möglichkeit: 

Wenn es gewünscht ist, dass der Counter (Zähler) für jedes neue Jahr oder jeden neuen Monat immer wieder bei 1 anfängt, wird an das Format-Pattern am Ende ein C angehängt. Z.B. JJJJMMNNNNNNNNNN C . 

Das C markiert, dass es sich um einen Counter handelt. Wenn der Counter beim Ermitteln der Zahl des letzten ermittelten Werts im Jahr / Monatsteil nicht dem aktuellen Jahr / Monat entspricht, wird wieder bei 1 angefangen. 

N = Nummer (für jede Stelle ein N, bei einer 5-stelligen Zahl also NNNNN) 
J = Jahr (entweder 2- oder 4-stellig) 
M = Monat (2-stellig) 
T = Tag (2-stellig) 
C = Kennung für Zähler 

Ist im Format ein MM für den Monat enthalten, so erfolgt immer bei Monatswechsel die Nummerierung von Beginn an, unabhängig ob ein JJ für die Jahreszahl vorhanden ist. Wenn nur pro Jahr ein Neubeginn erfolgen soll, darf die Kennung MM für Monat nicht Bestandteil des Formats sein. 

Gibt es in einem Nummernkreis keine Möglichkeit mehr eine eindeutige Nummer zu generieren, erscheint eine entsprechende Fehlermeldung. Diese Meldung kommt, wenn keine Nummer mehr frei ist bzw. bricht nach 10.000 Versuchen das ganze ab. 

Notieren Sie sich deshalb die anfangs eingetragenen Startwerte und überprüfen Sie anhand der Notizen regelmäßig, ob ein Nummernkreis an seine obere Grenze zu laufen droht. In diesem Fall droht ein Systemstillstand!

### Tabellenspalte Format 
Hier wird das Zahlenformat für die jeweiligen Nummernkreise festgelegt. Hier sind als Standardwerte ausschließlich numerische Werte vorgegeben. Es ist davon abzuraten, andere als numerische Nummernkreise zu verwenden. 

N = Nummer (für jede Stelle ein N, bei einer 5-stelligen Zahl also NNNNN) 
J = Jahr (entweder 2- oder 4-stellig) 
M = Monat (2-stellig) 
T = Tag (2-stellig) 
P = Prüfziffer nach Modulo10 (1-stellig, muss letztes Zeichen im Format sein) 

Insgesamt maximal 18-stellig 

C als Kennung für Zähler: 
Ein C wird wie ein N interpretiert, bewirkt jedoch, dass bei Monats- /Jahreswechsel die Nummer auf 1 zurückgesetzt wird. 
Enthält eine Formatvorgabe mindestens ein C, gilt: 
- M enthalten – Nummer = 1 bei Monatswechsel - J enthalten – Nummer = 1 bei Jahreswechsel

### Tabellenspalte Nummer 
In dieser Spalte werden jeweils die Startwerte für die Nummernkreise festgelegt. Dabei ist unbedingt zu beachten, dass die Nummernkreise, d.h. die Anzahl der vergebenen Stellen groß genug ist, damit es nicht zu einem Überlaufen des Nummernkreises kommt. Ist ein Nummernkreis fast gefüllt, kann der Startwert jederzeit angepasst werden, indem z.B. eine weitere Stelle hinzugefügt bzw. der Startwert eines neuen, bisher nicht belegten Nummernbereichs eingetragen wird. Im fortlaufenden Betrieb wird dann bei der Vergabe der nächsten Nummer der Wert im entsprechenden Feld des Dialogs um 1 Einheit hochgesetzt, so dass hier immer die zuletzt vergebenen und nicht die anfangs festgelegten Nummern sichtbar sind. 

Tooltip: 

Geben Sie den Startwert eines Nummernkreises ein.
