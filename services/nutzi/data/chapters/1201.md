# Position splitten 
Position splitten 

In Auftrag, Angebot und Gutschrift steht im Menü Optionen die Funktion Position splitten zur Verfügung. So kann bei einem zu geringen Bestand auf einem Lager, der Rest einer bestellten Menge von einem anderen Lager bezogen werden. 

Bei einem Splitt wird das Bezugslager entfernt, damit nicht erneut ein automatischer Umlagerungsprozess erzeugt wird. 

Um den Dialog zu öffnen, ist es notwendig die Position in der Tabelle zu markieren. Sie erhalten einen besseren Überblick über gesplittete Positionen. 

In das Feld Splittmenge tragen Sie die Menge ein. Die ursprüngliche Menge wird entsprechend reduziert. Hat eine Position eine Chargen- oder eine Lagerplatzzuordnung werden auch diese Daten übernommen / gesplittet. 

Teilgelieferte Auftragspositionen können ebenfalls gesplittet werden, ohne dass der „neuen“ Splittposition der Auftragsbezug verloren geht. Dabei darf der Splitt die vorhandene Menge nicht überschreiten. 

Ebenso ist es möglich eine Auftragsposition zu splitten, welche bestellt, aber noch nicht geliefert wurde. In diesem Fall kann auch nur die Menge im Feld Menge geliefert maximal abgesplittet werden. 

Sind mehrere Chargen/Plätze zugeteilt, entscheidet das System über die neuen Zuordnungen. Wenn möglich wird dabei vermieden, dass in der neuen Position mehrere Chargen / Plätze zugeteilt sind. 

Das voreingestellte Lager im Feld Lageränderung kann mit dem Button gelöscht werden. 

Ein Splitt ist nur für Positionen im Status < 3 möglich. Wenn für eine Position bereits eine Packliste/ein Lieferschein gedruckt wurde, darf nur maximal die Restmenge (bestellt minus geliefert) gesplittet werden. 

In der Tabelle zur Verfügbarkeit werden das Lager und der Bestand angezeigt. 

Der Dialog kann auch über den Shortcut <Strg + Umschalt + S> geöffnet werden. 

Ein Splitt für teilgelieferte Auftragspositionen ist möglich, wenn die auftragsbezogene Bestellung vorhanden und der Wareneingang für diese Bestellung noch nicht gebucht wurde. Die Splittposition des Auftrags hat keinen Bezug zur ursprünglichen Bestellposition; beim Wareneingang allerdings kann über die Lieferzuteilung dann ggf. die Ware zur Lieferung freigegeben werden. 

Dabei darf der Splitt die vorhandene Menge nicht überschreiten. 

Ebenso ist es möglich eine Auftragsposition zu splitten, welche bestellt, aber noch nicht geliefert wurde. In diesem Fall kann auch nur die Menge im Feld Menge geliefert maximal gesplittet werden.

### Checkbox Versandlager fix 
Wenn Sie diese Checkbox aktivieren, wird bei nicht ausreichender Menge über ein Bezugslager umgelagert und vom Versandlager geliefert. 

Die Checkbox Uml. Bezugslager (uB) (Dialog Verfügbarkeit Steuerung ) wird vorbelegt, wenn im Dialog Parameter Verkauf (Registerkarte Allgemein ) die Checkbox Vorgabe ist Umlagerung Bezugslager gesetzt ist und es auf Belegart-, Sachbearbeiter-, Kunden- oder Lagerebene keine Gegen-Einwände gibt (CheckBox Kein Bezugslager ). 

Grund ist, dass die Belieferung des Kunden nur über das Versandlager erfolgen soll. In diesem Fall darf alternativ zum Versandlager nur eine Belieferung des Kunden durch Streckenbestellung oder Interner Umlagerung erfolgen. 

Siehe Kapitel Versand- und Bezugslager .
