# Anfragearten Registerkarte Rollenrechte 
Anfragearten Registerkarte Rollenrechte 

Diese Registerkarte dient dazu, den Anfragearten und damit den Benutzern entsprechende Rollen/Rechte zuzuordnen. Auf diese Weise können Sie einschränken, dass die Benutzer nur bestimmte Einkaufarten betrachten und/oder einfügen können. 

Auch wenn für einen Sachbearbeiter die Rollensperrungen eingerichtet wurden, werden diese Aufträge in den Statistiken berücksichtigt. 

Hat ein Sachbearbeiter mehrere Rollen zugeordnet, so genügt es wenn bereits für eine Rolle der Zugriff gesperrt wurde. 
### Button Neu 

Dieser Button fügt eine neue Zeile in die Tabelle ein. 

### Button Löschen 

Dieser Button löscht eine Zeile der Tabelle und damit eine Rollensperrung. 

### Tabelle 

### Tabellenspalte Rolle 
Hier wählen Sie die Rolle aus, für welche Sie die Berechtigung einschränken möchten. Dazu wählen Sie aus der Combobox in der Spalte ‚Rolle‘ eine Rolle aus und können dazu die Checkboxen in den Spalten ‚Ansicht sperren‘ und ‚Speichern sperren‘ aktivieren. Es stehen alle Rollen aus dem Authentication Service zur Verfügung. 

Tooltip: 

Eine Rolle wird einem Sachbarbeiter zugeordnet. 

### Tabellenspalte Ansicht Sperren 
Beim Setzen dieser Checkbox kann der Sachbearbeiter keine Bestellungen dieser Anfrageart betrachten. Daher wird auch automatisch die Checkbox Speichern sperren gesetzt. 

Ist die Checkbox Speichern sperren gesetzt, kann der Sachbearbeiter die Bestellungen dieser Anfrageart betrachten, allerdings keinerlei Änderungen vornehmen. 

Tooltip: 

Der Sachbearbeiter kann diesen Vorgang nicht sehen. 

### Tabellenspalte Speichern sperren 
Wenn Sie die Checkbox in dieser Spalte aktivieren, kann der Sachbearbeiter keine Anfragearten bearbeiten. 

Tooltip: 

Der Sachbearbeiter kann die Vorgänge ansehen, aber nicht ändern.
