# Sachbearbeiter- Scanner Registerkarte Restmengenhandling

Lagerverwaltungssystem (LVS) / Sachbearbeiter- Scanner 

# Sachbearbeiter- Scanner Registerkarte Restmengenhandling 
Sachbearbeiter Scanner Registerkarte Restmengenhandling 

Auf dieser Registerkarte können dem Sachbearbeiter verschiedene „Rechte“ gegeben werden, mit denen er in der Kommissionierung reagieren kann, wenn er nicht die komplette Pickmenge entnehmen kann. 
### Checkbox 1 Rest später 
Wenn Sie diese Checkbox aktivieren, wird die Restmenge zu einem späteren Zeitpunkt durch diesen Kommissionierer gepickt. 

Tooltip: 

Die Restmenge wird zu einem späteren Zeitpunkt durch diesen Kommissionierer durchgeführt. 

### Checkbox 2 Fehlmenge 
Wenn Sie diese Checkbox aktivieren, wird eine Lagerkorrekturbuchung durchgeführt und ein neuer Entnahmeversuch initialisiert. 

Tooltip: 

Es wird eine Lagerkorrekturbuchung durchgeführt und ein neuer Entnahmeversuch initialisiert. 

### Checkbox 3 Mindermenge OK 
Wenn Sie diese Checkbox aktivieren, wird beim Kunden keine Rückstandslieferung über die Fehlmenge erzeugt. 

Tooltip: 

Beim Kunden wird keine Rückstandslieferung über die Fehlmenge erzeugt. 

### Checkbox 4 Rest Zuladung 
Wenn Sie diese Checkbox aktiveren, wird die noch nicht gepickte Menge als Zuladungspick durchgeführt (Menge wird entnommen in ein Versandgut gebucht und das Versandgut auf den Platz gelegt). 

Tooltip: 

Die noch nicht gepickte Menge wird als Zuladungspick durchgeführt (Menge wird entnommen, in ein Versandgut gebucht und das Versandgut auf den Platz gelegt) 

### Checkbox 5 Neue Behälter 
Wenn Sie diese Checkbox aktivieren, wird die Restmenge in ein neues LHM gelegt. 

Tooltip: 

Die Restmenge wird in ein neues LHM gelegt.
