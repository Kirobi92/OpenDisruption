# Kapitel NV.ERP.Base.Customer.cdCustomer.lngCentralPayerID

Das Feld Zentralzahler ist wie das Feld Verband (Registerkarte Warenwirtschaft ) für die Verbandsabrechnung zuständig. Es bietet eine zusätzliche Auswahlmöglichkeit in der Verbandsabrechnung für Debitoren, ohne dass die Funktionalität des Felds Verband zur Verfügung steht. (Änderung der Rechnungsanschrift)
