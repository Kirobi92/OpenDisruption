Finanzbuchhaltung / Umsatzbetrachtung 

# Umsatzbetrachtung Registerkarte Selektionen 1 
Umsatzbetrachtung Registerkarte Selektionen 1 

Auf dieser Registerkarte definieren Sie die Einstellungen für die Auswertung. 

Die Selektionskriterien Gruppe, Vertreter, Verbände und Währung beziehen sich auf die Einträge und Zuordnungen aus dem Kunden- bzw. Lieferant. 

Sämtliche Selektionskriterien können einzeln oder in Kombination miteinander verwendet werden. 
### Dock Panel Kontentyp 

Über diesen Bereich legen Sie fest, für welche Art Personenkonto die Umsatzbetrachtung erzeugt werden soll. 
### Radiobutton Debitoren 
Wurde als Personenkontenbereich Debitoren ausgewählt, steht Ihnen die Combobox Vertreter als weiteres Selektionskriterium zur Verfügung. 

### Radiobutton Kreditoren 
Wenn Sie diesen Radiobutton aktivieren, erfolgt die Suche auf den Bereich Kreditoren. 

### Field Panel Angabe der Suchkriterien 

Mit den Radiobuttons Kontonummer , Kontobezeichn . und Suchname steuern Sie die Art der Suchkriterien in den Feldern Von ... Bis . 
### Radiobutton Konto 
Wenn Sie diesen Radiobutton aktivieren, erfolgt die Auswahl auf die Kontonummer. 

### Radiobutton Kontobez. 
Wenn Sie diesen Radiobutton aktivieren, erfolgt die Auswahl auf die Kontobezeichnung. 

Tooltip: 

Kontobezeichnung 

### Radiobutton Suchname 
Wenn Sie diesen Radiobutton aktivieren, erfolgt die Auswahl auf den Suchnamen. 

### Feld Konto von 
Hier wird die Kontonummer-von zum Datensatz angezeigt. 

Tooltip: 

Das Konto von zum Datensatz. 

### Button Suche 

Mit diesem Button können Sie ein Konto suchen. 

Siehe Kapitel Kontoauswahl . 

### Feld Konto bis 
Hier wird die Kontonummer-bis zum Datensatz angezeigt. 

Tooltip: 

Das Konto bis zum Datensatz. 

### Button Suche 

Mit diesem Button können Sie ein Konto suchen. 

Siehe Kapitel Kontoauswahl . 

### Feld Kontobez. von 
Die Kontobezeichnung-von wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung von 

### Feld Kontobez. bis 
Die Kontobezeichnung-bis wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung bis 

### Feld Suchname von 
Hier wird der Suchname angegeben. Außerdem kann hier eine Belegnummer zu einem Vorgang eingetragen werden. Wenn der Vorgang existiert, wird der Vorgang geladen. 

Tooltip: 

Angabe des Suchnamens 

### Feld Suchname bis 
Hier wird der Suchname angegeben. Außerdem kann hier eine Belegnummer zu einem Vorgang eingetragen werden. Wenn der Vorgang existiert, wird der Vorgang geladen. 

Tooltip: 

Angabe des Suchnamens 

### Combobox Jahr

### Combobox Periode von 
Hier wird eine Periode-von angegeben. 

### Combobox Periode bis 
Hier wird eine Periode-bis angegeben. 

### Combobox Gruppe 
Hier können Sie Kunden- oder Lieferantengruppe auswählen. 

### Combobox Gruppe 
Hier können Sie über Kunden- bzw. Lieferantengruppe selektieren. 

### Dock Panel Beträge in 

Um die Umsatzbetrachtung auch in Fremdwährung drucken zu können, stehen die Radiobuttons Beträge in LW / FW zur Verfügung. 

Standardmäßig ist der Radiobutton LW gesetzt. Wenn Sie über die Einstellung FW suchen, werden die Beträge entsprechend in Fremdwährung angezeigt. 

Allerdings kann dann keine Gesamtsumme über alle Konten angezeigt werden, da verschiedene Währungen nicht addiert werden können. 

Die Einzelsummen pro Konto werden weiterhin angezeigt. 
### Radiobutton Beträge in 
Tooltip: 

Selektion über die Landeswährung 

### Radiobutton FW 
Tooltip: 

Fremdwährung
