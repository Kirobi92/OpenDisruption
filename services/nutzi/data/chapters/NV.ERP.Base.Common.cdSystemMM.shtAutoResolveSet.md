# Kapitel NV.ERP.Base.Common.cdSystemMM.shtAutoResolveSet

Wenn Sie diesen Parameter aktivieren, werden Artikel Sets bei Abschluss des Auftrags automatisch aufgelöst. Dieser Parameter kann nur gesetzt werden, wenn der Auftragsabschluss aktiviert ist. 

Tooltip: 

Artikel-Sets werden bei Abschluss des Auftrags automatisch aufgelöst.
