# Favoritenlisten zusammenführen und erweitern 
In eNVenta werden die verschiedenen Quellen für Favoriten aus Navigator und Favoriten im Verlauf zu einer Liste zusammengefasst. Dies bedeutet, bereits im Navigator können Favoriten auf Vorgänge und nicht nur Dialoge angelegt werden. Durch die Kombination mit dem automatischen Start von Favoriten kann z.B. direkt beim Öffnen von eNVenta ein Auftrag gestartet werden. 

Neben den Dialogen und einzelnen Vorgängen können auch Abfragen, welche als Einstellungen gespeichert sind, als Favorit angelegt werden und auch über den automatischen Start beim Öffnen von eNVenta gestartet werden, z.B: die Positionsübersicht im Verkauf mit rückständigen Lieferungen oder MIS-Abfragen aus der Online-Analyse. 

Um ein Zoom-Element als Favoriten zu definieren, gehen Sie folgendermaßen vor: 

Sie markieren das gewünschte Zoom-Element im Dialog Zoom-Elemente und übernehmen es in den Dialog Einstellungen mit dem Button . Hier klicken Sie die gewünschte Kategorie an, speichern die Einstellung und schließen den Dialog. Nun steht der Eintrag im Dialog eNVenta Zoom im Bereich Favoriten zur Verfügung. 

Das Zoom-Element Favoriten kann auch alle Arten von Einträgen anzeigen und per Drag&Drop empfangen. Sobald ein Eintrag im Zoom-Element hinzu kommt oder gelöscht wird, aktualisiert sich auch der Baum im Navigator und umgekehrt. 

Elemente aus dem Navigator können ebenfalls per Drag & Drop auf die Favoriten gezogen werden. Dies ist allerdings erst dann möglich, wenn bereits ein Favorit vorhanden ist, da sonst keine Drop-Area zur Verfügung steht.
