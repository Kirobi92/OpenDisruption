# CRM Übersicht Registerkarte Mahnverlauf 
CRM Übersicht Registerkarte Mahnverlauf 

Auf dieser Registerkarte werden je nach Selektion die entsprechenden Mahnläufe angezeigt. 
### Combobox Jahr bis 
Hier wird das (Geschäfts-) Jahr angegeben. 

### Checkbox Drucken 
Wenn Sie diese Checkbox aktivieren, werden die Informationen gedruckt. 

### Tabelle 

### Tabellenspalte Verarb. Datum 
Tooltip: 

Verarb. Datum 

### Tabellenspalte Max. MS 
Hier wird die aktuelle Mahnstufe der Position angegeben. 

Tooltip: 

Angabe der aktuellen Mahnstufe 

### Tabellenspalte Gesamtbetrag 
Hier wird der reale Fremdwährungsbetrag angegeben. 

Tooltip: 

Summe der Beträge aus den Mahnvorschlägen 

### Feld Jahr bis 
Hier wird das (Geschäfts-) Jahr angegeben. 

### Feld Mahnungen 
Hier wird die Anzahl der Mahnungen angegeben. 

Tooltip: 

Gesamtanzahl Mahnungen 

### Feld Mahnsumme 
Hier wird die Summe aller Beträge aus Mahnungen angegeben. 

Tooltip: 

Summe der Mahnungen 

### Feld Stufe 1 | 2 
Hier wird die Anzahl der Mahnungen in Stufe 1 angegeben. 

Tooltip: 

Anzahl Mahnungen Stufe 1 bezogen auf das Jahr 

### Feld Stufe 2 
Hier wird die Anzahl der Mahnungen in Stufe 2 angegeben. 

Tooltip: 

Anzahl Mahnungen Stufe 2 bezogen auf das Jahr 

### Feld Stufe 3 | >3 
Hier wird die Anzahl der Mahnungen in Stufe 3 angegeben. 

Tooltip: 

Anzahl Mahnungen Stufe 3 bezogen auf das Jahr 

### Feld Stufe > 3 
Hier wird die Anzahl der Mahnungen in Stufe > 3 angegeben. 

Tooltip: 

Anzahl Mahnungen Stufe 4 aufwärts bezogen auf das Jahr
