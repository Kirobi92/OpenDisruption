# Schlagwort-Zuordnung 
Schlagwort-Zuordnung 

Der Button Schlagworte im Dialog Archiv-Zuordnung öffnet diesen Dialog. Hier erfolgt die Zuordnung, welche Daten aus dem Objekt an welches Objekt des logischen Archivs geknüpft werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Schlagwort-Zuordnung 
Tooltip: 

In diesem Dialog erfolgt die Zuordnung, welche Daten aus dem Objekt an welches Objekt des logischen Archivs geknüpft werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .
