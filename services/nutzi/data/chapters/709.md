# Global Trade Item Number (GTIN) 
Die Global Trade Item Number (GTIN) kommt überall dort zum Einsatz, wo Artikel automatisch erfasst und verarbeitet werden und gekennzeichnet werden müssen. Mit der GTIN lassen sich Produkte und Dienstleistungen weltweit überschneidungsfrei identifizieren. 

Die GTIN fungiert als Zugriffsschlüssel auf die in Datenbanken hinterlegten Produktinformationen, wie Bezeichnung, Gewicht, Gebindegröße oder Warengruppe. 

Diese GTIN wird die EAN ablösen. Aus diesem Grund gibt es in einigen Dialogen das zusätzliche Feld / Spalte 'GTIN'. Dies sind beispielsweise 

- Artikeleinheiten - Parameter Verkauf Registerkarte Art-Suchen - BMEcat Import Registerkarte Erweitert - Artikel Imp. Parameter - Artikel Schattenstam m 
Die GTIN steht überall zur Verfügung, wo auch die EAN zur Verfügung steht. Es wurde mit Absicht nicht die EAN ersetzt, sondern eine parallele Pflegemöglichkeit geschaffen. Die GTIN umfasst auch die EAN, doch wurde in der Vergangenheit keine Prüfung auf die EAN vorgenommen, wodurch auch fehlerhafte Daten in der Datenbank vorhanden sein könnten. Die GTIN selbst wird nicht auf die korrekte Syntax geprüft, es wird nur sichergestellt, dass nur Ziffern eingegeben werden dürfen und ob die Länge korrekt ist. 

Sie können die GTIN aus der EAN über einen entsprechend definierten Datenexport und Datenimport über den Gateway Adapter nachpflegen. Alternativ sprechen Sie mit Ihrem Systembetreuer, wenn Sie die Daten automatisiert übernommen haben möchten.
