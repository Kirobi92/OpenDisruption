# Rückmeldung von Fahraufträgen 
Rückmeldung Fahrauftrag 

In der Tourendisposition gibt es das Menü Vorgänge und hier wiederum den Eintrag Rückmeldung Fahrauftrag . Dieser Dialog öffnet sich nur, wenn in der Tourendisposition eine Tour selektiert wurde. 

Es werden Daten zur Tour (Abfahrtzeit, Ankunftszeit, Kilometer), einzelne Stopps (Anzahl Hübe, Standzeit, Wartezeit …) und die Lademitteldaten zu einzelnen Stopps zurückgemeldet. 

Der Fahrauftrag kann über die Tournummer (Feld Tour Nr. ) oder die Ladelistennummer (Feld Fahrauftrag ) gesucht werden. Dabei werden auch die dazugehörenden Haltepunkte geladen. Sie werden in der Tabelle auf der Registerkarte Stopps angezeigt. 

Außerdem kann eine Pause in Minuten (m) angegeben werden. 

Wenn Ware nicht an den Kunden übergeben werden kann, so können Sie dies im Dialog Rückmeldung Fahrauftrag in der Tabelle eingeben. Dazu stehen die Spalten ‚Retoure‘ und ‚Retoure‘ zur Verfügung. Wenn Sie in diese Spalten Werte eingeben, wird beim Speichern eine Retoure generiert. 

Dazu sind im Dialog Parameter VS / LO noch entsprechende Vorgaben zu definieren. 

Siehe Kapitel Parameter Versand / Logistik . 

Wenn Sie die Anzahl der gefahrenen Kilometer im Kopf des Dialogs eintragen, springt der Cursor anschließend automatisch in das Feld Haltepunkt auf der gleichnamigen Registerkarte.

### Feld Fahrtzeit (m)

Tooltip: 

Geben Sie die Fahrtzeit in Minuten an. 

### Feld Pause (m) 
Hier kann die Pause in Minuten eingetragen werden. 

Tooltip:

### Feld Abfahrt 
Hier wird ein Startdatum für die Selektion angegeben. 

Das Feld kann schon im Voraus gefüllt werden, im Format 24:00h. Die Ankunft wird ebenfalls in diesem Format eingetragen. 

Tooltip: 

Geben Sie die Abfahrtszeit ein. 

### Feld Ankunft 
Hier wird ein Bis-Datum angegeben. 

Das Format für die Zeit wird angegeben wie folgt: 24:00h. 

Tooltip: 

Geben Sie die Ankunftszeit ein. 

### Feld Kilometer Start 
Für die Kilometer-Messung kann hier der Kilometerstand bei Start der Fahrt eingetragen werden. 

Tooltip: 

Kilometerstand bei Abfahrt 

### Feld Kilometer Ende 
Für die Kilometer-Berechnung kann hier der Kilometerstand bei Ende der Fahrt eingetragen werden. 

Tooltip: 

Kilometerstand bei Ankunft

Tooltip:
