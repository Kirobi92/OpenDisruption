Finanzbuchhaltung / Factoring Parameter 

# Factoring Parameter Registerkarte Logo 
Factoring Parameter Registerkarte Logo 

Hier können Sie das Logo der Factoring Bank hinterlegen. Es wird dann auf dieser Registerkarte angezeigt und steht für den Druck auf Reports zur Verfügung. 
### Abbildung Factoringgeber Logo 
Tooltip: 

Logo des Factoringgebers erfassen 

### Button Suche 

Mit diesem Button können Sie ein Logo suchen. 

### Button Löschen 

Mit diesem Button können Sie das Logo entfernen.
