Grundeinstellungen für eNVenta ERP / Sachbearbeiterstamm 

# Sachbearbeiterstamm Registerkarte Archivierungsgruppen 
Sachbearbeiter Registerkarte Archivierungsgruppen 

Auf dieser Registerkarte ordnen Sie dem Mitarbeiter eine Archiv-Berechtigungsgruppe zu. 
### Button Neu 

Klicken Sie auf diesen Button, um eine neue Zeile hinzuzufügen und wählen in der Combobox die passende Archiv-Berechtigungsgruppe aus. Sie können neben einer Druckergruppe auch eine Abteilung angeben, welcher wiederum Drucker zugeordnet werden können. 

### Button Löschen 

Damit entfernen Sie eine Archiv-Berechtigungsgruppe aus der Tabelle.

### Tabellenspalte Archiv-Berechtigungsgruppe 
Hier wird die Archiv-Berechtigungsgruppennummer eingetragen.
