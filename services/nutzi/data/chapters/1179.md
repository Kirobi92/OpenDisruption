# Produktionsauftrag erstellen 
Prod.-Auftrag erstellen 

Dieser Dialog öffnet sich im Auftrag >> Optionen >> Generieren >> Produktionsauftrag und steht nur zur Verfügung, wenn es sich bei der Artikelnummer der Position um eine Produktionsstückliste handelt. Außerdem müssen die Positionen vorher gespeichert werden. Ist dies der Fall, können Sie angeben, welche Menge des Artikels produziert werden soll. 

Bei der Generierung eines Produktionsauftrags aus dem Auftrag heraus, wird die Variante des Arbeitsplans genommen, welche im Dialog angegeben ist. 

(Dieser Dialog öffnet sich auch aus dem Dialog Stücklisten über den Button Produzieren , wenn das Modul Produktion freigeschaltet ist.) 

Der Dialog öffnet sich auch über die Tastenkombination <Strg> + <F> . 

Parallel kann ein neutraler Produktionsauftrag, ohne Bezug zu einem Verkaufsauftrag, erstellt werden. 

Wird aus einem Kundenauftrag ein Produktionsauftrag generiert, so wird das Lieferdatum aus der Auftragsposition an den obersten Betriebsauftrag übergeben. 

Ist bereits ein Produktionsauftrag zu dem Artikel erzeugt worden und Sie rufen die Option Produktionsauftrag erneut auf, so wechseln Sie diesmal direkt in den Dialog Produktionsauftrag in der Produktion. 

Wenn Sie aus einer Position eines Verkaufsauftrags einen Produktionsauftrag oder eine Simulation erstellen und diesen Produktionsauftrag dann in den „Echtbetrieb“ übernehmen, erhalten Sie eine Meldung, falls in der Zwischenzeit aus dem Verkaufsauftrag bereits ein produktiver Produktionsauftrag generiert wurde. Auf diese Weise wird verhindert, dass für die gleiche Auftragsposition mehrere Produktionsaufträge generiert werden.

### Combobox Variante 
Hier wird die Stücklistenvariante angegeben, welche produziert wird. Bei Angabe einer entsprechenden Variante bei der Komponente wird diese in den Betriebsauftrag übernommen. Falls keine entsprechende Variante vorhanden ist, wird die Variante 0 für diese Komponente übernommen. 

Da ein Variantenwechsel auch in der Auftragsposition gespeichert wird, wird diese geänderte Variante hier angezeigt.

### Dock Panel Modus 

Dieser Bereich definiert, über welche Ebenen die Stückliste aufgelöst werden soll und der entsprechende Produktionsauftrag inkl. Betriebsaufträgen generiert wird. 

- Nur erste Ebene - Nur die oberste Ebene der Stückliste wird produziert, unabhängig der verfügbaren Menge. - Nur Fehlbestände - Es werden nur Ebenen der Stückliste produziert, welche einen Fehlbestand aufweisen. - Alle Ebenen - Alle Ebenen der Stückliste werden produziert, unabhängig der verfügbaren Menge. 

### Dock Panel Terminierung 

In diesem Bereich legen Sie fest, wie die Ressourcen für den Betriebsauftrag terminiert werden sollen. Zur Verfügung stehen 

- Keine - Vorwärts - Rückwärts 
Voraussetzung hierfür ist, dass ein Arbeitsplan für die zu produzierenden Artikel besteht.
