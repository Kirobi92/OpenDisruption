E-Commerce Katalog / Parameter Webshop / Parameter Webshop Registerkarte E-Mail 

# Parameter Webshop Registerkarte Passwort 
Registerkarte Passwort 

Auf dieser Registerkarte nehmen Sie Einstellungen für die Passwort-E-Mail vor. 
### Feld Vorlage 
Angabe der E-Mail-Vorlage für die Passwort-E-Mail. Es kann eine bestehende Vorlage ausgewählt werden. 

Siehe Kapitel Dokument-Vorlagen . 

Vorlagenart hat zwingend E-Mail zu sein. 

### Button Suche 

Über diesen Button können Sie eine Vorlage suchen. 

Siehe Kapitel Dokument Vorlagen . 

### Combobox Sprache 
Aus dieser Combobox können Sie eine Sprache auswählen. 

### Feld Voransicht 
Hier wird der Inhalt der E-Mail (aus der Vorlage) angezeigt.
