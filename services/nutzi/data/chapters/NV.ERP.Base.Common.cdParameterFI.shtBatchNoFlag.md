# Kapitel NV.ERP.Base.Common.cdParameterFI.shtBatchNoFlag

Wenn Sie diesen Parameter aktivieren, wird in vielen Dialogen das Feld Stapelnummer sichtbar und Sie können den Buchungen eine Stapelnummer mitgeben. 

Stapelnummer bezeichnet in diesem Zusammenhang eine Nummer, die Sie einer bestimmten Anzahl an Belegen zuordnen, die Sie buchen möchten. 

Solange die Stapelnummer zugeordnet ist, kann die Buchung nicht in die Kostenrechnung übernommen werden. 

Diese Stapelnummer steht dann als Suchkriterium zur Verfügung. 

Storno und Annullierung können anhand der Stapelnummer ausgeführt werden. 

Die Stapelnummer steht in folgenden Dialogen zur Verfügung: 

- Journal - Verbandsabrechnung Debitor - Verbandsabrechnung Kreditor - Auszug-/Kassenimport - ZV verarbeiten (im Zahlungsvorschlag ) - BEZ verarbeiten (im Einzugsvorschlag ) - OP-Übernahme - Zinsrechnung - OP-Verrechnung - Kursbewertung - Pauschalwertberichtigung
