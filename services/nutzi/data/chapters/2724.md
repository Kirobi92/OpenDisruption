# Kostenauswertungen Registerkarte KA/L <> KSt / KTr 
Kostenauswertungen Registerkarte KA/L <> KSt / KTr 

Auf dieser Registerkarte werden die Gesamtwerte der gewählten Sicht/Bereich (also ob Kostenarten, Kostenstellen oder Kostenträger) angezeigt. 

Sie können wahlweise die Nummer oder das hinterlegte Kürzel im Spaltenkopf anzeigen lassen. Dafür stehen die Radiobuttons Nummer und Kürzel zur Verfügung. 
### Button Kompl. Exp. 

Tooltip: 

Kompletter Export aller Daten nach MS Excel / OO Calc 

### Radiobutton Nummer 
Tooltip: 

Zeige im Spaltenkopf 

### Radiobutton Kürzel 
Tooltip: 

Zeige das Kürzel im Spaltenkopf
