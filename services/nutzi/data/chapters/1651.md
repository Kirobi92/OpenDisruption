# Kontrollcenter Lager Registerkarte Fehler

Lagerverwaltungssystem (LVS) / Kontrollcenter Lager 

# Kontrollcenter Lager Registerkarte Fehler 
Kontrollcenter Lager Registerkarte Fehler 

Hier werden alle Picks angezeigt, welche zu irgendeinem Zeitpunkt eine Fehlermeldung generiert haben. 
### Button Lagerauftrag 

Dieser Button öffnet den Lagerauftrag. 

Siehe Kapitel Lagerauftrag . 

### Tabelle 

### Tabellenspalte Lagerauftrag 
In dieser Spalte wird die Lagerauftragsnummer angezeigt. 

Tooltip: 

Angabe der Lagerauftragsnummer 

### Tabellenspalte Fixpos 
In dieser Spalte wird die Fixposnummer angezeigt. 

Tooltip: 

Eindeutige Positionsnummer des Datensatzes. 

### Tabellenspalte Entnahmeart 
Hier wird der Kommissionierungstyp angezeigt: 
- Entnahme - Umlagerung - Einlagerung - Inventur - Fehlerkorrektur 

Tooltip: 

Hier wird der Kommissionierungstyp angezeigt: 
•Entnahme 
•Umlagerung 
•Einlagerung 
•Inventur 
•Fehlerkorrektur 

### Tabellenspalte Status 
In dieser Tabellenspalte wird der Pick-Status des Lagerauftrags angegeben: 
- offen - Bearbeitung - erledigt - storniert. 

Tooltip: 

Status der Lagerauftragsposition: 
•offen 
•Bearbeitung 
•erledigt 
•storniert 

### Tabellenspalte Gesperrt 
Wenn Sie diese Checkbox aktivieren, ist der Kommissioniervorgang gesperrt. Sie wird gesetzt, wenn die Auftragsposition "liefergesperrt" ist. 

Tooltip: 

Aktiviert: Kommissioniervorgang ist gesperrt. 

### Tabellenspalte Artikel 
In dieser Spalte wird die Artikelnummer angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Fehlertext 
In dieser Spalte wird der Fehlertext der Lagerauftragsposition angezeigt. 

Tooltip: 

Angabe eines Fehlertexts 

### Tabellenspalte Quelllager 
Hier wird das Quell-Lager angegeben. Es werden die Nummer und der Name des Lagers angezeigt. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Quell-Platz 
In dieser Spalte wird der Platz angezeigt, von dem entnommen wird. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Ziellager 
Hier wird das Ziellager angegeben. Es werden die Nummer und der Name des Lagers angezeigt. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Ziel-Platz 
In dieser Spalte wird der Platz angezeigt, auf den eingelagert/umgelagert werden soll. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Menge Soll 
In dieser Spalte wird die Zielmenge angezeigt, die gepickt werden soll. 

Tooltip: 

Angabe der Zielmenge 

### Tabellenspalte Pickmenge 
In dieser Spalte wird die gepickte Menge angezeigt. 

Tooltip: 

Angabe der Pickmenge 

### Tabellenspalte Auftrag 
In dieser Spalte wird die Belegnummer des Referenztyps angezeigt. 

Tooltip: 

Angabe der Belegnummer 

### Tabellenspalte Pick-Sachbearbeiter 
In dieser Spalte wird der Sachbearbeiter angezeigt, der die Ware verpackt hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Referenztyp
