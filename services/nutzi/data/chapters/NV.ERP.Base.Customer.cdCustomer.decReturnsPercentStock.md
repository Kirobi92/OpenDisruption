# Kapitel NV.ERP.Base.Customer.cdCustomer.decReturnsPercentStock

In diesem Feld kann für die Rücknahme von Lagerware ein Prozentsatz eingetragen werden, um den Gutschriften bei Rücknahme der Ware gemindert werden. 

Tooltip: 

Hier kann ein Prozentsatz für die Rücknahme von Lagerware eingetragen werden. 
Um diesen Prozentsatz wird die Gutschrift bei Rücknahme der Ware gemindert.
