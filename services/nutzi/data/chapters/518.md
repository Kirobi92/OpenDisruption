# Währungshistorie 
Währungshistorie 

Dieser Dialog öffnet sich im Dialog Währung über den Button Historie . Hier können Sie die Kursänderungen zu einer Währung nachverfolgen. Die Währungsnummer und der Gültigkeitszeitraum werden angezeigt. Bei Änderung des Kurses zu einer Währung wird automatisch ein Datensatz in die Währungshistorie geschrieben. Es wird immer der genaue Kurs innerhalb des Gültigkeitszeitraums eingetragen. Ebenso wird der Sachbearbeiter angezeigt, welcher die Kursänderung vorgenommen hat. 

Gibt es in der Währungshistorie bereits einen Eintrag mit diesem Datum, wird der Kurs überschrieben, d.h. es gibt nur einen Tageskurs. 

Gibt es in der Währungshistorie bereits einen Eintrag, der größer als dieses Datum ist (vordatiert), wird die Änderung abgelehnt. Kurse die vordatiert sind, müssen zuerst gelöscht werden. 

Kurse mit Datum kleiner gleich dem Tagesdatum können nicht gelöscht werden. 

Wird in der Währungstabelle ein neuer Kurs eingetragen, so wird ein entsprechender Datensatz in der Kurshistorie eingetragen. Wird der Kurs in der Währungstabelle anschließend angepasst, so wird dieser Kurs auf dem neuen Eintrag in der Kurshistorie upgedatet. 

Wird bei einer Kursänderung festgestellt, dass es bereits einen Kurseintrag gibt mit einem späteren Gültig von- (also ein Kurs in der Zukunft) wird einfach eine neue Zeile generiert, die als Gültig von- das entsprechende Datum (nach Standardlogik) bekommt und als Gültig bis- das Datum des Eintrages mit dem späteren Gültig von- . 

Im Datensatz wo bisher das Gültig bis- dem Eintrag Gültig von- entsprach (späteres Gültig von- ), wird das Gültig bis- entsprechend auf das Gültig von- des neuen Eintrages angepasst.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button.

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Combobox Währung

Tooltip: 

Angabe der Währung 

### Tabelle 

In dieser Tabelle werden die Kursänderungen zu einer Währung in einer Übersicht angezeigt. 

Der Eintrag in der Spalte ‚Gültig von‘ kann geändert werden. Damit haben Sie die Möglichkeit, eine Kursänderung unmittelbar nach der Änderung zurück- oder vorzudatieren. Wenn Sie in die Bezeichnung der Spalte klicken, öffnet sich eine Kalenderansicht. 

Bei jeder Änderung wird automatisch geprüft, dass es immer nur einen Tageskurs geben kann.

### Tabellenspalte Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind!
