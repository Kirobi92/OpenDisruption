# Mietobjekt Registerkarte Allgemein 
Mietobjekt Registerkarte Allgemein 

Auf dieser Registerkarte wird das Mietobjekt spezifiziert. 
### Feld Ressource 
Hier wird eine Standard-Ressource angegeben. 

Dies gilt nur, wenn als Mietobjekt-Typ Standard gewählt ist, ansonsten ist dieses Feld unzugänglich. 

Tooltip: 

Angabe der Ressource 

### Button Suche 

Mit diesem Button können Sie eine Ressource suchen. 

Siehe Kapitel Ressourcenauswahl . 

Tooltip: 

Ressourcenauswahl 

### Feld Ressourcenbez. 
Der Name der Ressource wird automatisch eingelesen. 

### Combobox Lager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Aus diesem Lager wird das Mietobjekt für die erste Vermietung genommen. 

Tooltip: 

Aus diesem Lager wird das Mietobjekt bei der ersten Vermietung entnommen. 

### Combobox Lagernr. alt 
Auf das hier ausgewählte Lager wird das Mietobjekt gebucht, wenn es nicht mehr vermietet werden kann (z.B. weil es alt oder kaputt ist). 

Tooltip: 

In dieses Lager wird das Mietobjekt gebucht, wenn es nicht mehr vermietet werden kann. 

### Combobox Ziellager 
Hier wählen Sie das Lager aus, auf welches das Mietobjekt nach Rückgabe aus der Vermietung (über die Retoure) gebucht wird. Aus diesem Lager wird es dann auch weiter vermietet. 

Tooltip: 

In dieses Lager wird das Mietobjekt nach Rückgabe aus der Vermietung (Retoure) gebucht. 
Das Mietobjekt wird bei der nächsten Vermietung aus diesem Lager vermietet (s.a. Checkbox "Lager schalten für Gebraucht"). 

### Checkbox Lager schalten für Gebraucht 
Wenn Sie diese Checkbox aktivieren, wird das gewählte Ziellager das Entnahmelager für alle weiteren Vermietungen dieses Mietobjekts. 

Tooltip: 

Wenn gesetzt, ist das unter Ziellagernr gewählte Lager 
Entnahmelager für alle weiteren Vermietungen des Mietobjektes. 
Diese Checkbox wird automatisch gesetzt, wenn ein 
Mietobjekt zum ersten Mal zurückgebucht wird. 

### Combobox Externes ML 
Aus dieser Combobox wählen Sie das externe Mietlager aus. Falls Ihr Unternehmen nicht genug Bestand für das Mietobjekt hat, wird die Fehlmenge von diesem externen Lager gebucht. 

Tooltip: 

Externes Mietlager für Massenmietobjekte. 
Von dem hier hinterlegten Lager werden etwaige 
Fehlbestände bei einer Vermietung gebucht, 
wenn eine Kooperationsvereinbarung mit einem 
anderen Lieferanten für dieses Mietobjekt besteht. 
Es wird ein unendlicher Bestand auf diesem Lager angenommen. 

### Feld Seriennummer 
Hier wird manuell eine Seriennummer eingetragen. Diese Seriennummer ist losgelöst von der Standard-Seriennummernverwaltung in eNVenta . 

Siehe Kapitel Parameter Miete . 

Tooltip: 

Die hier hinterlegte Seriennummer kann alternativ zur Seriennummernverwaltung 
in eNVenta ERP verwendet werden. (Siehe Parameter Miete) 

### Combobox Einheit 
Falls einem Artikel mehrere Verkaufseinheiten hinterlegt sind, können Sie hier die Einheit wählen, in der vermietet werden soll. 

Tooltip: 

Wenn mehrere Verkaufseinheiten beim Artikel hinterlegt sind, 
kann hier die Einheit, in der vermietet werden soll, ausgewählt werden. 

### Feld Qualität 
In diesem Feld können Sie die Qualität (Güte) eines zurückgekommenen Mietobjekts eintragen. U.U. ist dieses Feld bereits aus der Retoure vorbelegt. Diese Information wird dann auch in der Tabelle des Dialogs Vermietung Registerkarte Positionen angezeigt. 

Tooltip: 

Angabe der Bezeichnung 

### Feld Rüstzeit in Tagen 
Hier geben Sie die Anzahl Tage ein, die ein Mietobjekt nach Rückgabe nicht wieder vermietet werden kann. Damit haben Sie die Möglichkeit, das Mietobjekt für eine Weitervermietung "aufzubereiten" (z.B. putzen, warten). 

Tooltip: 

Anzahl Tage, die ein Mietobjekt nach der Rückgabe nicht erneut vermietet werden kann. (z.B. Inspektion, Wartung etc.) 

### Checkbox Rüstzeit unverbindlich 
Wenn diese Checkbox inaktiv ist, erscheint eine Fehlermeldung, dass die Rüstzeit nicht eingehalten wurde. 

Wenn diese Checkbox aktiv ist, erscheint ein Hinweis, dass die Rüstzeit beachtet werden sollte. 

Tooltip: 

Checkbox = inaktiv: Eine Fehlermeldung erscheint, dass die Rüstzeit nicht eingehalten wurde. 
Checkbox = aktiv: Ein Hinweis erscheint, dass die Rüstzeit beachtet werden sollte. 

### Checkbox Mietkaution 
Wenn Sie diese Checkbox aktivieren, so ist für dieses Mietobjekt eine Kaution zu hinterlegen. 

eNVenta erwartet eine Kaution im Dialog Vermietung , wenn im Kundenstamm die Checkbox Mietkaution ebenfalls gesetzt ist. 

Tooltip: 

Kennzeichnet mietkautionspflichtige Artikel. 
eNVenta ERP erwartet eine Kaution im Vermietdialog, 
wenn im Kundenstamm die Checkbox 'Mietkaution' ebenfalls gesetzt ist. 

### Checkbox Monat mit 30 Tagen 
Wenn Sie diese Checkbox aktivieren, wird ein Monat immer mit 30 Tagen gerechnet. 

Tooltip: 

Monat kalkulatorisch 30 Tage 
Wenn gesetzt, wird ein Monat immer als ein Zeitraum von 30 Tagen angesehen. 
Wenn nicht gesetzt, richtet sich die Länge eines Monats nach dem tatsächlichen Datum. 
Beispiel: 01.02.2019 – 28.02.2019 repräsentiert bei gesetzter Checkbox keinen kompletten Monat. 
Dient der Vorbelegung bei der Neuanlage von Mietobjekten. Kann je Mietobjekt geändert werden. 

### Checkbox Nur Werktags 
Wenn Sie diese Checkbox aktivieren, wird die Miete nur für Werktage berechnet. Wochenenden werden nicht berechnet! 

Tooltip: 

Wenn gesetzt, wird die Miete nur für Werktage erhoben. Wochenenden bleiben ausgenommen. 

### Checkbox Keine Ausfalltage 
Wenn Sie diese Checkbox aktivieren, werden für dieses Mietobjekt keine Ausfalltage berücksichtigt (z.B. bei Versicherungen: das Mietobjekt ist auch dann versichert, wenn es nur "steht"). 

Tooltip: 

Wenn gesetzt, werden solche Mietobjekte bei der Erfassung von Ausfalltagen nicht berücksichtigt. (z.B. Versicherungen)
