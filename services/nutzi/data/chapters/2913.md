# X Mapping Test 
X Mapping Test 

Der Dialog X Mapping Test öffnet sich über den Button Test im Dialog X Mapping Editor und kann zum Testen des Mappings genutzt werden. Der Dialog öffnet sich standardmäßig im Helpframe und dient dem Testen eines Mappings. Aus diesem Grund reagiert der Import bzw. Export immer direkt auf die gespeicherten Änderungen eines Mappings. 

Für den Export müssen zuerst Export-Daten bereitgestellt werden, z.B. per SQL-Suche oder auch indem die Eigenschaften manuell auf einen bestimmten Wert gesetzt werden. Oder es wird beispielsweise bei ORDERS Export eine Bestellung generiert und diese wird per Drag&Drop übernommen. Dies ist in der Registerkarte Daten vorzunehmen. 

Über den Button Format kann die generierte Datei formatiert angezeigt werden, dies dient der besseren Lesbarkeit der Daten. 

Für den Import kann ein X-Inhalt in das Feld eingefügt werden. Nach dem Import können die Daten betrachtet werden. Das Ergebnis des Imports wird dann in der Registerkarte Daten angezeigt.

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind.

### Button Nächster Satz
