# Kapitel NV.ERP.Base.Common.cdSystemMM.shtSerialNoStrict

Bei aktivierter Seriennummern-Zuordnung wird bei der Fakturafreigabe geprüft, ob für die Artikel eine Seriennummer eingetragen ist. Fehlt diese, bleibt die Position im Status 2 und eine entsprechende Meldung wird angezeigt. 

Achtung: Die Checkbox SN Prüfung bei Lieferschein auf der Registerkarte Druck kann nicht gesetzt werden, wenn die Strikte Seriennummern-Zuordnung nicht aktiv ist! 

Tooltip: 

Wenn dieser Parameter aktiviert ist, wird bei Fakturafreigabe geprüft, 
ob für die Artikel eine Seriennummer eingetragen ist. 
Zusätzlich kann die Seriennummer Prüfung beim 
Lieferscheindruck aktiviert werden im Reiter Druck.
