# Übersicht Rechnungsprüfung 
Übersicht Rechnungsprüfung 

Dieser Dialog öffnet sich in der Rechnungsprüfung >> Vorgänge >> Übersicht Rechnungsprüfung . Hier werden Ihnen sämtliche zu prüfende Rechnungen in einer Übersicht angezeigt. Dazu geben Sie im Kopf des Dialogs Ihre Selektionskriterien ein. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Übersicht Rechnungsprüfung 
Tooltip: 

In diesem Dialog werden Ihnen sämtliche zu prüfende Rechnungen in einer Übersicht angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Checkbox Prüfungsfehler 
Wenn Sie diese Checkbox aktivieren, werden auch fehlerhafte Rechnungen selektiert. 

### Dock Panel Art 

In diesem Bereich können Sie die Selektion auf die Belegart eingrenzen. 

Wenn Sie die Checkbox Ohne Positionsbezug aktivieren, werden auch Positionen ohne Bezug zu einer Rechnung eingelesen. Dies vor dem Hintergrund, dass ja die Möglichkeit besteht, Daten von extern einzulesen (z.B. über Proxess), die noch keinen Bezug zu einer Rechnung haben. 

Wird diese Checkbox aktiviert, werden bestimmte Felder deaktiviert und bei der Suche nicht berücksichtigt. 
### Checkbox Mit Bestellpos 
Wenn Sie diese Checkbox aktivieren, werden Rechnungsprüfungen mit zugeordneten Bestellpositionen selektiert.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
