# Offene Posten Debitoren Registerkarte Selektionen 3

Finanzbuchhaltung / Offene Posten Debitoren 

# Offene Posten Debitoren Registerkarte Selektionen 3 
Offene Posten Debitor Registerkarte Selektionen 3 

Auf dieser Registerkarte stehen nochmals weitere Selektionskriterien zur Verfügung. 
### Feld Kostenträger 
Zur Selektion über Kostenträger wählen Sie hier einen Eintrag aus. 

Tooltip: 

Angabe der Kostenträgernummer 

### Button Suche 

Mit diesem Button können Sie Kostenträger suchen. 

Siehe Kapitel Kostenträgerauswahl in der Finanzbuchhaltung . 

### Feld Kostenstelle 
Zur Selektion über eine Kostenstelle wählen Sie hier einen Eintrag aus. 

Tooltip: 

Angabe der Kostenstellennummer 

### Button Suche 

Mit diesem Button suchen Sie nach Kostenstellen. 

Siehe Kapitel Kostenstellenauswahl in der Finanzbuchhaltung . 

### Combobox Rechtsform 
Zur Selektion über eine Rechtsform wählen Sie hier einen Eintrag aus. 

Tooltip: 

Angabe der Unternehmen-Rechtsform 

### Combobox Factoringgeber 
Hier wird die Factoring Bank angegeben. 

Tooltip: 

Angabe des Factoringgebers 

### Combobox Angekaufte OP 
In Zusammenhang mit dem Factoring können Sie aus dieser Combobox folgende Einträge auswählen: - Angekaufte OPs: Nur => alle OPs mit Fact. Rückm.dtm gefüllt. - Angekaufte OPs: Ohne => alle OPs mit Fact. Rückm.dtm leer. 

### Combobox Gemeldeter OP 
In Zusammenhang mit dem Factoring können Sie hier folgende Einträge auswählen: 

- Gemeldete OPs: Nur => alle OPs mit Meld. Fact OP gefüllt. - Gemeldete OPs: Ohne => alle OPs mit Meld. Fact OP leer. 

### Combobox Niederlassung 
Zur Selektion über die Niederlassung, wählen Sie hier einen Eintrag aus. 

Tooltip: 

Angabe der Niederlassung 

### Checkbox Kontoauszug 
Wenn Sie diese Checkbox aktivieren, werden Datensätze von Kunden eingelesen, die auf der Registerkarte Rechnungswesen ebenfalls die Checkbox Kontoauszug aktiviert haben. 

### Checkbox Restkennzeichen 
Wenn Sie diese Checkbox aktivieren, werden offene Posten in die Tabelle eingespielt, die in der OP-Anlage die Kennzeichnung Rest aus Wawi erhalten haben. 

Tooltip: 

Angabe des Rest-OPs 

### Checkbox Stornos ausblenden 
Wenn Sie diese Checkbox aktivieren, werden Storno- und stornierte Buchungen ausgeblendet. 

Diese Checkbox wird zugänglich, wenn die Checkbox Hervorhebung von Stornobuchungen im Dialog Parameter Fibu , Registerkarte Allgemein gesetzt ist. 

Siehe Kapitel Parameter Fibu . 

Tooltip: 

Storno- und stornierte Buchungen ausblenden 

### Tabelle 

In dieser Tabelle kann für die Anzeige noch das Gebiet / Bundesland als Selektionskriterium gewählt werden. 
### Tabellenspalte Gebiet 
Hier kann das Gebiet bzw. das Bundesland angegeben werden. 

Tooltip: 

Angabe eines Gebiets / Bundeslands
