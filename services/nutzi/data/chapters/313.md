# Seriennummern Erfassung

Allgemeine Funktionalitäten in eNVenta ERP / Lagerverwaltung 

# Seriennummern Erfassung 
Seriennr erfassen 

Der Button Erfassen im Dialog Seriennr Zuordnung öffnet diesen Dialog. 

Im oberen Bereich legen Sie fest, zu welchem Artikel und/oder zu welchem Lieferanten bzw. zu welcher Bestellung Sie die Seriennummern erfassen möchten. 

Im unteren Bereich legen Sie die Seriennummer fest. 

Nachdem Sie die Buchung abgeschlossen und die Lagerbuchung verlassen haben, können Sie anhand der Seriennummernverwaltung Ihre Aktionen aufrufen, kontrollieren und bearbeiten. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Seriennr erfassen 
Tooltip: 

In diesem Dialog legen Sie fest, zu welchem Artikel und/oder zu welchem Lieferanten bzw. zu welcher Bestellung Sie die Seriennummern erfassen möchten. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt.

### Feld Erfasst 
Hier wird automatisch die Anzahl der erfassten Seriennummern (im gleichnamigen Feld) angezeigt, bzw. automatisch aktualisiert. 

Tooltip: 

Angabe der Menge 

### Feld Seriennummern 
Wenn Sie hier eine Seriennummer eingeben und weiter schalten, wird die Seriennummer in eine Textliste eingetragen. Mit dem Button werden alle Seriennummern in die Datenbank eingetragen. Mögliche Fehler werden als Meldung in einer Liste angezeigt.
