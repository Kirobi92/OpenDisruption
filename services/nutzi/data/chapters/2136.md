# Opportunitystufen Registerkarte Allgemein 
Opportunitystufe Registerkarte Allgemein 

Hier geben Sie die Opportunitystufe vor; sie steuert den Vertriebsprozess durch: 

- Tage gelb : erste Warnfrist zur Erreichung der nächsten Prozess-Stufe - Tage rot : zweite Warnfrist zur Erreichung der nächsten Prozess-Stufe. - Wahrscheinlichkeit in % (zur Bewertung des Forecasts) 
### Feld Tage Gelb 
Hier geben Sie die erste Warnfrist zur Erreichung der nächsten Prozessstufe an. 

Tooltip: 

Warnfrist zur Erreichung der nächsten Prozessstufe 

### Feld Tage Rot 
Hier geben Sie die zweite Warnfrist zur Erreichung der nächsten Prozessstufe an. 

Tooltip: 

Warnfrist zur Erreichung der nächsten Prozessstufe 

### Feld Wahrscheinl. % 
Hier geben Sie einen Prozentsatz für die Wahrscheinlichkeit ein, dass der Vertriebsprozess Erfolg hat. 

Tooltip: 

Wahrscheinlichkeit in %, dass der Vertriebsprozess Erfolg hat. 

### Combobox Sachbearb. WV 
Über diese Combobox können Sie einen Sachbearbeiter auswählen, an den die Wiedervorlage gesendet werden soll. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Tage WV 
Für die Erstellung der Wiedervorlage geben Sie hier eine Anzahl Tage ein. 

Tooltip: 

Geben Sie an, wann die WV erstellt werden soll. 

### Feld WV Text 
Für die Wiedervorlage kann hier ein individueller Text eingetragen werden. 

Tooltip: 

Eingabe eines individuellen Wiedervorlagetexts.
