# Archiv Parameter 
Archiv Parameter 

Dieser Dialog öffnet sich über System >> Allgemein >> Archiv >> Parameter . Hier wählen Sie das Barcode-Prüfverfahren aus. 

Wenn Sie Proxess benutzen, erscheint der Warnhinweis wegen des Barcodes nur, wenn die Checkbox Warn. mehrf. BC auch aktiviert ist. Dabei wird in allen Dokumententypen geprüft, ob der Barcode bereits benutzt wurde. Im Warnhinweis wird dann angegeben, dass ein Querverweis erzeugt wird, wenn der Barcode nochmal verwendet wird. 

Über diesen Dialog wird auch das Testverfahren für die Barcodes eingestellt.

### Combobox Barc. Prüfverf. 
Diese Combobox steht zur Verfügung, um Eingabefehler beim Archivieren zu vermeiden. Hier wählen Sie das passende Verfahren aus. Momentan erfolgt die Prüfung nur in Verbindung mit einem Proxess-Archivsystem. 

Das Barcode-Prüfverfahren ist in eNVenta aufgrund der Vorgaben von Proxess fix programmiert. 

Wenn Sie nun im Dialog Archiv-Mappe einfügen das Feld Barcode verlassen, wird geprüft ob die Eingabe entsprechend des eingestellten Prüfziffern-Verfahrens richtig ist. 

Bei einer Falscheingabe erhalten Sie eine Meldung. 

Tooltip: 

Festlegung eines Dokumenten-Management-Systems (DMS). 
Damit wird die Barcodenummer anhand von DMS-Ziffern auf ihre Korrektheit hin überprüft. 

### Checkbox Warn. mehrf. BC 
Wenn Sie Proxess benutzen, erscheint der Warnhinweis wegen des Barcodes nur, wenn diese Checkbox auch aktiviert ist. Dabei wird in allen Dokumententypen geprüft, ob der Barcode bereits benutzt wurde. Im Warnhinweis wird dann angegeben, dass ein Querverweis erzeugt wird, wenn der Barcode nochmal verwendet wird. 

Tooltip: 

Warnhinweis bei mehrfacher Verwendung desselben Barcodes (nur Proxess)
