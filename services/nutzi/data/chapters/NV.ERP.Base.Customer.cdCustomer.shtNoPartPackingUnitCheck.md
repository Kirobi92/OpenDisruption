# Kapitel NV.ERP.Base.Customer.cdCustomer.shtNoPartPackingUnitCheck

Wenn Sie diese Checkbox aktivieren, werden für den Kunden in den Vorgängen keine Zuschläge für Anbrüche von Mengeneinheiten berücksichtigt. 

Tooltip: 

Kein Anbruch R/Z: 
Für den Kunden werden in den Vorgängen keine Zuschläge für Anbrüche von Verpackungseinheiten berücksichtigt, wenn dieser Parameter aktiviert ist.
