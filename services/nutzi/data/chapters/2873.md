# Alternativressourcen 
Alternativressource 

Der Button Alternativen in der Ressource öffnet den Dialog Alternativressource . Hier können Sie alternative Ressourcen definieren, für den Fall, dass es bei einer Ressource zu einem Engpass kommt. Dies ist insbesondere in der Produktion hilfreich, um eine möglichst schnelle Auftragsbearbeitung zu gewährleisten. 

Die Alternativzuordnung gilt in beide Richtungen, d.h. von Ressource 1 nach Ressource 2 und umgekehrt. 

Für beide Richtungen kann allerdings ein getrennter Eignungsgrad festgelegt werden. Damit kann gesteuert werden, welche der möglichen Alternativen angeboten oder auch automatisch einkalkuliert werden sollen. 

Der Eignungsgrad ist frei definierbar, z.B. von 1 bis 100 oder wie Schulnoten von 1 – 6.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
