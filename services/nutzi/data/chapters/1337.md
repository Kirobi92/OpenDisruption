# Merkmalsauswahl 
Merkmalsauswahl 

Dieser Dialog öffnet sich in der Merkmal Klasse über den Button neben der linken Tabelle. Hier können Sie nach Merkmalen suchen. Dafür stehen zur Selektion auch die Felder im Kopf des Dialogs zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Merkmalsauswahl 
Tooltip: 

In diesem Dialog können Sie nach Merkmalen suchen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Klassifikationssystem 

### Checkbox Kein Druck 
Wenn Sie diese Checkbox aktivieren, dient dieses Merkmal der Recherche in eNVenta , wird aber nicht gedruckt bzw. an BMECat übergeben. 

### Checkbox Kein Webshop 
Wenn Sie diese Checkbox aktivieren, dient dieses Merkmal der Recherche in eNVenta , wird aber nicht im Webshop eNVenta genutzt. 

### Checkbox Nur Werte 
Über diese Checkbox haben Sie die Möglichkeit die Einschränkung bei der Auswahl der Werte zu einem Merkmal zu beeinflussen. Ist diese Checkbox gesetzt, wird die Wert-Auswahl auf die ausgewählten Werte eingeschränkt. 

Tooltip: 

Es können nur die Werte von dem Merkmal ausgewählt werden. 

### Checkbox Kein ML 
Wenn Sie diese Checkbox aktivieren, kann das Merkmal nicht in mehreren Sprachen gepflegt werden. 

Tooltip: 

Nicht Mehrsprachig: 
"Kein ML" sorgt dafür, dass die Merkmale nicht in anderen Sprachen angezeigt werden. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.

### Tabellenspalte Mussfeld 
Wenn Sie diese Checkbox aktivieren, wird das Merkmal zum Pflichtmerkmal. D.h. der Artikel kann nur gespeichert werden, wenn dieses Merkmal mit einem Wert versehen wurde. 

Tooltip: 

Definiert das Merkmal als Pflichteinstellung. 

### Tabellenspalte Format 
Hier wird ein Format angegeben (optional) - nur für Ganzzahl, Kommazahl, Formatierungs-Ausdruck. 
Der Ausdruck wird bei der String-Umwandlung des Werts als Formatierungsangabe genutzt. 

Beispiele: 
Zahlen: 
"#,##0.00" -> "1.000,00" 
"€ #,##0.00" -> "€ 1.000,00" 
"#.000 mm" -> "10000.000 mm" 
Datum: 
"yyyy.MM.dd" -> "2019.03.31" 

Tooltip: 

(Optional) Nur für Ganzzahl, Kommazahl, Datum als Formatierungs-Ausdruck. 
Der Ausdruck wird bei der String-Umwandlung des Werts als Formatierungsangabe genutzt. 
Beispiele: 

Zahlen 
"#,##0.00" -> "1.000,00" 
"€ #,##0.00" -> "€ 1.000,00" 
"#.000 mm" -> "10000.000 mm" 
Datum: 
"yyyy.MM.dd" -> "2019.01.31"
