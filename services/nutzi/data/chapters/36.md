# Registerkarte Belegfolge 
Registerkarte Belegfolge 

Diese Registerkarte ist mit dem Symbol gekennzeichnet und ist rechts vom Vorgangs-Dialog angeordnet. Sämtliche Vorgänge, die in einer Abhängigkeit zueinander stehen, können gebündelt betrachtet werden. Hiermit haben Sie auch die Möglichkeit Modul-übergreifend Verkaufs- und Einkaufsbelege zu einem Prozess zu betrachten. Hierbei ist es auch unerheblich aus welchem Vorgang heraus die Belegfolge gestartet wird. Die Daten werden immer komplett angezeigt, z.B. aus der Rechnungsprüfung heraus bis zurück zum Angebot. Diese Anzeige wird passend beim Öffnen bzw. Laden eines Vorgangs aktualisiert, damit die Daten stimmig zum aktuellen Vorgang betrachtet werden können. Für die schnellere Übersicht wird dabei der aktuelle Vorgang farblich hervorgehoben. 

Die Belegfolge lässt sich nur öffnen, wenn der Sachbearbeiter berechtigt ist, den entsprechenden Dialog einzusehen (z.B. einen Auftrag, eine Bestellung oder einen Betriebsauftrag). 

Zu einem Vorgang werden die wichtigsten Informationen in der Belegfolge angezeigt: Belegart, Erfassungsdatum, Status, letzter Sachbearbeiter etc. 

In dieser Belegkette werden zusätzlich zu den Vorgängen die Druckbelege in einer Übersicht dargestellt; zusammen mit der entsprechenden Belegnummer (Rechnungsnummer, Lieferscheinnummer…). Dies ist besonders dann hilfreich, wenn es für einen Vorgang mehrere Belege des gleichen Typs gibt. Auf diese Weise erhalten Sie mit wenigen Schritten einen kompletten Überblick über die Belege eines Vorgangs. Die Druckbelege haben ein entsprechendes Symbol davor, um diese zu erkennen. 

Mit Doppelklick auf einen Beleg gelangen Sie in den entsprechenden Vorgang. Bei einem Druckbeleg wird dieser als HTML-Report angezeigt. 

Auf diesem Dialog kann auch per Drag&Drop gearbeitet werden. Wird ein Beleg eingefügt oder verschoben, wird die Belegfolge neu ermittelt. 

Zusätzlich zur Anzeige der Belegfolge in tabellarischer Ansicht gibt es diese Anzeige auch auf einem Zeitstrahl im Dialog Belegfolge . 

Der Dialog Belegfolge öffnet sich über den Button .

## Steuerelemente des Formulars 

### Button Zeitstrahl 

Dieser Button öffnet den Dialog Belegfolge , wo die Belege graphisch angezeigt werden. Alle Vorgänge einer Belegkette werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

Tooltip: 

Graphische Darstellung der Belegfolge in einem Zeitstrahl wird geöffnet. 

### Button Aktualisieren 

Mit diesem Button wird die Anzeige der Belege aktualisiert. 

Tooltip: 

Die Belegfolge wird aktualisiert. 

### Bezeichnungsfeld Aktuelle Information 
Hier werden die aktuelle Datenbank und der aktuelle Benutzer angegeben.
