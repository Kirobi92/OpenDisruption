# Kapitel NV.ERP.Base.Customer.cdCustomer.sDeliveryInfo

In diesem Feld können Informationen zum Ziel der Lieferung eingetragen werden. Z.B. dass Ware morgens nicht vor 7.30h angeliefert werden soll oder dass nur ein bestimmter Mitarbeiter die Ware annehmen darf.
