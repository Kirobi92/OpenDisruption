# Produktionsarten Registerkarte Tabelle 
Produktionsart Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Produktionsarten aufgelistet. Ein Doppelklick in eine Zeile übernimmt die Produktionsart in die Felder im Kopf des Dialogs bzw. in die Felder auf den entsprechenden Registerkarten. 
### Tabelle 

### Tabellenspalte Produktionsart 
Hier wird die Produktionsart angegeben. 

Tooltip: 

Geben Sie die Nummer der Produktionsart ein. 

### Tabellenspalte Bezeichnung 
Die Bezeichnung der Produktionsart wird angezeigt. 

Tooltip: 

Name der Produktionsart 

### Tabellenspalte Sachbearbeiter 
Hier wird der Sachbearbeiter angegeben. 

Tooltip: 

Angabe des Sachbearbeiters 

### Tabellenspalte Erfassung 
Hier geben Sie das Erfassungsdatum in der Produktionsart an. 

Tooltip: 

Erfassungsdatum in der Produktionsart 

### Tabellenspalte Änderung 
Hier wird das Änderungsdatum in der Produktionsart angegeben. 

Tooltip: 

Änderungsdatum in der Produktionsart 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
