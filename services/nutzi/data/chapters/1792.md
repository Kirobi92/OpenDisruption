# Anfrage Registerkarte Anfragen 
Registerkarte Anfragen 

Auf dieser Registerkarte erhalten Sie eine Übersicht über Anfragen zum Artikel. 
### Tabelle 

### Tabellenspalte Beleg 
Hier wird die Belegnummer aus dem Einkauf (Bestellung, Anfrage, Belastung) angezeigt. 

Tooltip: 

Hier wird die Bestellnummer definiert. 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Status 
Hier wird der Status der Position angegeben. 

1-offen 
2-offen, zur Lieferung freigegeben 
3-offen, zur Faktura freigegeben 
4-fakturiert 
5-geliefert (ohne Berechnung) 
6-storniert 

Tooltip: 

Angabe des Positionsstatus 

### Tabellenspalte Bestelldatum 
Hier wird das Bestelldatum angegeben. 

Tooltip: 

Angabe des Bestelldatums 

### Tabellenspalte Menge bestellt 
Hier wird die bestellte Menge angegeben. 

Tooltip: 

Angabe der Menge, welche bestellt wurde. 

### Tabellenspalte Einheit EK 
Hier wird die Beschreibung der Einheit aus der Pickliste angezeigt. Dies ist vor allem für die Reports eine Verbesserung, da Abkürzungen leicht missverstanden werden können. (Für die korrekte Funktion ist in einigen Dialogen das interne Property für die Anzeige der Einheit angepasst worden). Bei vorhandenem Customizing mit dem Feld Einheit ist dies ggf. nachzuziehen. 

Diese Beschreibungen werden in der Pickliste gepflegt. Ist in der Pickliste keine Beschreibung gepflegt, so wird wie bisher die Bezeichnung in eNVenta angezeigt. 

Tooltip: 

Einheit EK
