# Behälterbuchung 
In diesem Dialog können Sie Kommissionierbehälter auf einen Lagerplatz buchen bzw. auf einen anderen Lagerplatz umbuchen. 

Dabei wird der Bestand nicht auf diesen Lagerplatz gebucht. Einzig das Feld Lagerplatz auf dem Behälter gibt Auskunft über den Standort. 

Existiert für die Auftrags- bzw. Tour-Kombination bereits ein Behälter der "abgestellt" wurde, so gibt das System beim Eingabefeld den Platz des bisherigen Behälters als erwarteten Wert an.

### Button Barcode 

Wird dieser Button gedrückt, so wird, wenn eine Kamera am Gerät vorhanden ist, die Barcodeerkennung aktiviert.

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Behälterinfo 
Dieses Feld gibt Auskunft über die Zusammensetzung dieses Kommissionierbehälters: 
- Ist es der einzige Behälter, so erscheint die Information "Einzelbehälter". - Ist der Behälter einer von mehreren Behältern zu einem Auftrag/Tour und ein anderer Behälter ist noch nicht abgestellt, so erscheint "Mehrbehälter". - Ist der Behälter der letzte Behälter einer Auftrags-/Tour-Kombination und alle anderen Behälter sind bereits abgestellt, so erscheint "Letzter Behälter".
