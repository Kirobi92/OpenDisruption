# Kapitel NV.ERP.Base.Common.cdParameterFI.decMT940GracePercent

Hier kann die Angabe der Karenz in % erfolgen. Der Prozentsatz wird auf den Auszug-/Kassenimport-Zahlbetrag (OP-Betrag abzüglich Skonto +/- Karenzbetrag) für die Automatikfunktion angewandt. Diese Angaben werden in die Auszug-/Kassenimport-Bearbeitung übernommen und können dort temporär überschrieben werden. 

Wenn Karenzprozent und -betrag angegeben werden, so ist dies eine UND-Abfrage, also beide Kriterien müssen erfüllt sein. 

Tooltip: 

Hier können Standard-Prozentwerte für den Auszug-/Kassenimport definiert werden. Diese Angaben werden in den Dialog 'Auszug-/Kassenimport' übernommen und können dort temporär überschrieben werden.
