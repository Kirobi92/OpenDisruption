# Kennzahlenberechnung 
Kennzahlenberechnung 

Mit Klick auf den Button Ausd. Editor im Dialog Elementberechnung - Auswahl öffnet sich der Dialog Kennzahlenberechnung , mit welchem sich eigene Kennzahlen berechnen lassen. 

Bei der Berechnung stehen alle Möglichkeiten des Microsoft SQL Server Analysis Services zur Verfügung. Diese sind in den Registerkarten Vorlagen , Metadaten und Funktionen einzeln aufrufbar und lassen sich zu einem komplexen Ausdruck zusammenfassen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Kennzahlenberechnung 
Tooltip: 

In diesem Dialog lassen sich eigene Kennzahlen berechnen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
