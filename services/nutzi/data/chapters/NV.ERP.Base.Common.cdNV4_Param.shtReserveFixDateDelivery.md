# Kapitel NV.ERP.Base.Common.cdNV4_Param.shtReserveFixDateDelivery

Ist dieser Schalter gesetzt, wird ein sogenannter Fixterminauftrag erzeugt. D.h. es wird ein Liefertermin in der Zukunft mit der Datumseinstellung des Fixtermins erzeugt. Diese Checkbox sollte standardmäßig gesetzt sein. 

Tooltip: 

Ist dieser Parameter gesetzt, so wird beim Wareneingang automatisch ein Auslagerungsauftrag erzeugt, welcher die Ware in einen speziellen Lagerbereich umlagert.
