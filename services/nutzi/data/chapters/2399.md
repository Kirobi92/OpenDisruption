# Belegübersicht 
Belegübersicht 

Dieser Dialog öffnet sich in der Konto-Info auf der Registerkarte Konto Info oder im Kontoblatt , Registerkarte Pro Tag und alle weiteren Registerkarten mit Verdichtung über einen Doppelklick in eine Zeile. Hierdurch weist die Belegübersicht nun alle Buchungen aus, die in den vorausgehenden Eingrenzungen, z.B. Monatssalden der Konto Info oder Verdichtung pro Tag des Kontoblatts stecken. 

Beim Aufruf aus der Konto Info sind nie die Saldovortragsbuchungen in der Periode 1 eingelesen, da in der Konto Info die Bewegungen (Rechnung/Gutschrift, OP-Ausgleich) ausgewiesen werden. 

Wenn Sie den Button drücken, ist diese Einstellung verworfen. Jetzt gelten die Dialog-Selektionsfelder Datum von / bis : Alle Buchungen werden selektiert, die den Zeitraum erfüllen - hierzu gehören auch die Saldovortragsbuchungen. 

Mit dem Doppelklick auf eine Zeile öffnet sich das Journal . 

Das Feld Konto ist editierbar. D.h. Sie können auf einfache Weise auch ein anderes Konto innerhalb des gleichen Kontotyps Debitor, Kreditor bzw. Sachkonto betrachten. 

Das Feld Von Datum / Bis Datum kann ebenfalls editiert werden, d.h. auch Zeiträume aus anderen Perioden können betrachtet werden.
