# Konfiguration der Webshop Benutzer Registerkarte Merkliste 
Benutzerverwaltung Registerkarte Merkliste 

Auf dieser Registerkarte sehen Sie alle Artikel des Webshop-Benutzers, welche er auf seiner Merkliste hat. Diese können auch über den Webservice verwaltet werden. So können auch Kunden der SIC die Merklistenfunktion nutzen. 

D.h. Merklisten können angezeigt, angelegt, gelöscht, hinzugefügt und gelöscht werden. 
### Tabelle 

### Tabellenspalte Bezeichnung 
Hier wird der Name zur Artikelnummer angezeigt. 

Tooltip: 

Zeigt den Artikelnamen. 

### Button Löschen 

Mit diesem Button können Sie einen Eintrag aus der Tabelle löschen. 

Tooltip: 

Artikel entfernen
