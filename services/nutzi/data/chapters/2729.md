# Kostenstellenverteilung 
Kostenstellenverteilung 

Dieser Dialog öffnet sich im Buchen Kore >> Auswertung >> Kostenstellenverteilung . 

Die Kostenstellenverteilung zeigt auf einen Blick an, wo die Kostenstelle bei den Kostenarten / Leistungen bzw. Kostenstellen als empfangende Kostenstelle eingebunden ist. 

Sie enthält im Allgemeinen eine sehr große Zahl von Datensätzen. Zwangsläufig muss die Auswahl der Datensätze eingeschränkt werden. Dies erreichen Sie über die Vorgabe der Kostenstellen-Nummer bzw. der Kostenstellen-Gruppe. Eine zusätzliche Einschränkung der in der Tabelle dargestellten Daten wird Ihnen durch die Unterteilung in Haupt- und Hilfskostenstellen ermöglicht.

### Checkbox Haupt-KSt 
Falls Sie eine Verteilung auf Hauptkostenstellen wünschen, aktivieren Sie diese Checkbox. 

Tooltip:

### Checkbox Hilfs-KSt 
Falls Sie eine Verteilung auf Hilfskostenstellen wünschen, aktivieren Sie diese Checkbox. 

Tooltip: 

Es werden die Buchungen mit Typ HK (Hilfskostenstellen) und VK (Vorkostenstellen) selektiert. 

### Checkbox Nicht verteilte anzeigen 
Über diese Checkbox werden zusätzlich Kostenstellen in der Tabelle angezeigt, welche der Selektion entsprechen, aber noch keine Verteilung auf Kosten- oder Leistungsart besitzen. 

### Checkbox Nur nicht verteilte anzeigen 
Über diese Checkbox werden nur Kostenstellen angezeigt, die noch nicht verteilt sind. Dies ist dann von Vorteil, wenn Sie prüfen möchten, ob alle Kostenstellen korrekt zugeordnet wurden.
