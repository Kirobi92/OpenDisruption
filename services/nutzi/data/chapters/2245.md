# Katalogverwaltung Registerkarte Texte

E-Commerce Katalog / Katalogverwaltung 

# Katalogverwaltung Registerkarte Texte 
Katalogverwaltung Registerkarte Texte 

Auf dieser Registerkarte können anhand der ausgewählten Sprache für die im TreeView markierte Position (Gruppe oder Produkt) eine Bezeichnung und eine Beschreibung erfasst werden. Pro Sprache können eine Bezeichnung und ein Langtext angegeben werden. 
### Combobox Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Feld Bezeichnung 
Hier geben Sie dem Text einen Namen. 

Tooltip: 

Angabe der Bezeichnung 

### Feld Navname 
Hier geben Sie einen Namen ein, der dann im Shop im Navigationsbaum angezeigt wird. 

Tooltip: 

Bitte geben Sie einen Namen ein, welcher im Shop im Navigationsbaum angezeigt wird. 

### Feld Kurzbeschreib. 
Hier kann eine Kurzbeschreibung der Gruppe angegeben werden. 

Tooltip: 

Eine Kurzbeschreibung der Gruppe wird angegeben. 

### Feld Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Button Html-Editor 

Über diesen Text können Sie Html-Text bearbeiten. 

Siehe Kapitel Html Editor . 

Tooltip: 

Html-Editor zur Bearbeitung von Html-Texten. 

### Button Satz hinzufügen 

Wenn Sie diesen Button drücken, können Sie neuen Text für den Strukturbaum anlegen. 

Tooltip: 

Neuer Text 

### Button Löschen 

Wenn Sie diesen Button drücken, können Sie Text wieder löschen. 

Tooltip: 

Text löschen
