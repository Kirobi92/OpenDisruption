# Artikel Schattenstamm Registerkarte Merkmale 
Registerkarte Merkmale 

Auf dieser Registerkarte werden Merkmale angezeigt. 

Im Import der Merkmale wird erst versucht, ein passendes, bereits existierendes Merkmal zu finden, bevor ein neues Merkmal geschaffen wird. Die Ausprägung wird dann mit dem alten bzw. neuen Merkmal verknüpft und später dem Artikel bereitgestellt. 
### Tabelle 

### Tabellenspalte Haupt-Importschlüssel 
Hier wird der Haupt-Importschlüssel angegeben. 

Tooltip: 

Angabe des Haupt-Importschlüssels 

### Tabellenspalte Merkmalsname 
Hier wird der Name des Merkmals angegeben. 

Tooltip: 

Angabe des Merkmalnamens 

### Tabellenspalte Merkmalswert 
Tooltip: 

Wert des Merkmals 

### Tabellenspalte Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Tabellenspalte Klasse 
Hier wird die Merkmalsklasse angegeben. 

### Tabellenspalte Klassifikationssystem 
Hier wird das Klassifikationssystem angegeben. Klassifikationssysteme werden im gleichnamigen Dialog verwaltet. 

Siehe Kapitel Klassifikationssystem . 

Tooltip: 

Klassifikationssystem 

### Tabellenspalte Zusätzliche Eigenschaften 
Hier wird die zusätzliche Eigenschaft angegeben. Die Eigenschaften stammen aus hierarchisch späteren Paketen und werden hier gesammelt, sofern noch kein dediziertes Feld existiert.
