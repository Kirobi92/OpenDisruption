# Saldenauswertung Registerkarte Forecast 
Saldenauswertung Registerkarte Forecast 

Auf dieser Registerkarte erhalten Sie eine Prognose für weitere Auswertungen der Salden. 

Wenn die Checkbox Budgetvergleich gesetzt ist, werden zu den Istwerten immer die entsprechenden Budgetwerte angezeigt. 

Die Werte in den Budget-Spalten kommen aus dem jeweiligen Konto.
