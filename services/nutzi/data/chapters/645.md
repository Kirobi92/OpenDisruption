# Html Text-Wizard Registerkarte Text 
Text-Wizard Registerkarte Text 

In den Feldern Feld Start und Feld Ende können Sie die Definition der Feldbegrenzer ändern. Dies ist notwendig bei E-Mail-Dokumenten im Html-Format (siehe unten). Hierzu überschreiben Sie die bereits eingetragenen Begrenzungen mit den von Ihnen im Feld-Begrenzer ausgewählten Zeichen. Geben Sie einfach jeweils ein anderes Zeichen für den Beginn und das Ende als Feldbegrenzer innerhalb der Syntax an. Sobald Sie das Feld verlassen, werden Sie gefragt, ob Sie alle bisherigen Zeichen im Text durch die neuen ersetzen wollen. 

Wenn Sie die Einstellung der Feldbegrenzer in Dokumenten ändern wollen, die bereits im Html-Format bestehen, sollten Sie die Abfrage nach Änderung aller Zeichen im Text ablehnen! Ansonsten werden auch die Zeichen für das Html-Format geändert, was ja vermieden werden soll. Das hat allerdings zur Folge, dass Sie Ihre Änderungen manuell vornehmen müssen. 

Wenn Sie mit Html arbeiten möchten, sollten Sie folgende Feldbegrenzer verwenden: 

<Feld> statt dem bisherigen </Feld/>, da es sonst Probleme mit den Ende-Tags im Html gibt (z.B. </table>).
