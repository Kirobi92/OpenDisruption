# Beistellauftrag erstellen 
Beistellauftr. erstellen 

Dieser Dialog öffnet sich über den Button Beistellung in der Retoure. Hier können Sie Ware an einen Lieferanten retournieren. 

Es ist möglich einen Beistellauftrag aus einer Retoure zu erstellen, der an verschiedene Lieferanten gesendet werden. Dafür können diese Positionen den entsprechenden Lieferanten zugeordnet werden. Hintergrund ist, dass in einer Retoure mehrere Lieferanten vorkommen können, welche unterschiedliche „Kunden“ für den Beistellungsprozess haben. Dieser Kunde und seine zugehörige Auftragsart wird pro „reine“ Positionsgruppe gezogen, wenn es eine gibt. Ansonsten wird der Standardkunde aus dem Dialog Parameter Verkauf Registerkarte Retoure und die angegebene Auftragsart gezogen. 

Die Generierung eines Beistellauftrags bietet folgende Vorteile: 

- separater Prozess, unabhängig von der Nachbestellung beim Lieferanten - logistisch einheitlich zu allen anderen Lieferprozessen mit der Möglichkeit, Packlisten und Lieferscheine zu erzeugen 
Dazu muss die Position, für die ein Beistellauftrag generiert werden soll, in der Retoure auf den Status ‚erhalten‘ gesetzt sein. 

Eine eventuelle Chargennummer oder Seriennummer der Retourenposition wird automatisch zugeordnet. 

Die Nummer des Beistellauftrags wird in der gleichnamigen Spalte auf der Registerkarte Allgemein angezeigt.

### Feld Auftraggeber 
Hier geben Sie den Lieferanten an, für welchen die Beistellung bestimmt ist. Dieser wird als Auftraggeber übernommen. So können die Daten des Lieferanten für das Reporting verwendet werden, um dies für den Lieferschein zu verwenden. Die Kundennummer aus dem Dialog Beistellauftrag ist vorbelegt aus dem Dialog Parameter Verkauf und dient als Platzhalter, um nicht für jeden Lieferanten auch eine Kundennummer anlegen zu müssen.
