# OP-Verrechnung Registerkarte Salden-Verrech. Kreditoren 
OP-Verrechnung Registerkarte Salden-Verrech. Kreditoren 

Auf dieser Registerkarte werden die Konten der Debitoren eingelesen und zwar unter Berücksichtigung von Skonto, Karenzbetrag und Karenzprozent je Konto- / Belegnummer. 
### Button Löschen 

Über diesen Button können Sie einzelne, markierte Zeilen aus der Selektion löschen. 

### Tabelle 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Firma 1 
Die Firmenbezeichnung wird automatisch vom System eingetragen, nachdem die Lieferantennummer eingetragen wurde. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Firma 2 
Die 2. Adresszeile wird angegeben. 

Tooltip: 

Zeile 2 des Firmennamens.

Tooltip: 

Angabe des Haben-Betrags
