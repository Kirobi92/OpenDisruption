# Kapitel NV.ERP.MM.Sales.cdOrderType.shtNoRoutePlanning

Wenn Sie diese Checkbox aktivieren, können bestimmte Auftragsarten von der Tourenplanung ausgeschlossen werden. 

Beispiel: Auftragsarten für Gesellschaftsabrechnungen, die lediglich der Abrechnung dienen und bei denen keine Waren bewegt werden. 

Tooltip: 

Ist dieser Parameter gesetzt, wird keine Tourenplanung für diesen Beleg durchgeführt.
