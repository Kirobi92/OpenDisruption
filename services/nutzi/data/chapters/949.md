# Projektschema Registerkarte Phasen

Projekt Management / Projektschema 

# Projektschema Registerkarte Phasen 
Projektschema Registerkarte Allgemein 

Auf dieser Registerkarte definieren und verwalten Sie Phasen zu einem Projektschema. 

Eine Mehrfachzuordnung von Phasen zu einem Schema ist möglich. 

Die Bezeichnung der Phase kann allerdings nur im Projekt geändert werden. 
### Button Neu 

Dieser Button stellt in der Tabelle eine neue Zeile bereit und öffnet den Dialog Auswahl Phase , um eine Phase auszuwählen. 

Siehe Kapitel Phase auswählen . 

### Button Löschen 

Dieser Button löscht eine Zeile der Tabelle. 

### Button Aufgaben 

Über diesen Button können Sie der Phase eine Aufgabe zuordnen. 

Siehe Kapitel Aufgaben zu Projektphasen . 

### Tabelle 

In dieser Tabelle werden die Phasen definiert. 
### Tabellenspalte Pos 
Hier wird die Positionsnummer angegeben. Diese Nummer muss je Vorgang eindeutig sein! 

Tooltip: 

Positionsnummer 

### Tabellenspalte Phase 
Hier wird eine Projektphase angegeben. 

Tooltip: 

Angabe der Projektphase 

### Tabellenspalte Bezeichnung 
Hier wird der Name der Phase angegeben. 

Tooltip: 

Angabe der Phasenbezeichnung 

### Tabellenspalte Budget 
Hier wird das Projekt-Budget angegeben 

Tooltip: 

Angabe des Projekt-Budgets 

### Tabellenspalte Tagessatz 
Hier wird der Tagessatz für die Abrechnung angegeben. 

Tooltip: 

Angabe des Tagessatzes 

### Button Nach oben 

Mit diesem Button können Sie einen Eintrag der Tabelle nach oben verschieben. 

### Button Nach unten verschieben 

Mit diesem Button können Sie einen Eintrag der Tabelle nach unten verschieben.
