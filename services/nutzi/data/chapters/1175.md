# Tourenauswahl im Auftrag 
Tourenauswahl 

Wenn die Tourenplanung aktiviert ist, öffnet sich dieser Dialog im Auftrag über den Button neben dem Feld Lieferdatum . 

Hier werden die Bedingungen für die Tourenplanung angezeigt und die daraus resultierenden Touren. Der Sachbearbeiter kann die Tour manuell übersteuern. Wird eine Tour manuell ausgewählt, wird das Kennzeichen 'MT' gesetzt. 

Siehe Kapitel Auftrag , Absatz Kürzel für Tourenermittlung . 

Hier wird die Tourenplanung für den Kopf ausgeführt und das Ergebnis angezeigt. 

Zusätzlich werden die möglichen Touren mit angezeigt, welche laut Tourenplanung ermittelt wurden. 

Der Sachbearbeiter kann nur direkt eine der möglichen Touren manuell auswählen, alternativ kann er aus der manuellen Tour über den Button Automatisch die Tourenplanung noch mal durchführen lassen. 

Mit der Checkbox Tourenplanung inkl. Pos werden die Ergebnisse der Plannung automatisch auf die Detailpositionen übertragen. 

Das heißt, wird die manuelle Auswahl geändert oder die Automatik aktiviert, so wird automatisch die Position eingestellt. 

Ist eine Sondertour für die aktuellen Einstellungen notwendig, so ist es nicht möglich für den Sachbearbeiter eine Tour auszuwählen. Dies ist dann Aufgabe des Tourenverantwortlichen. 

Hinweis: Eine Tourenplanung, die am überschrittenen Kreditlimit des Kunden im Auftrag gescheitert ist, führt automatisch zu einer Neueinplanung der Auftragspositionen, wenn der Grund der Kreditlimitsperre nicht mehr gegeben ist.

### Checkbox Kein Paketdienst 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Paketdienst-Touren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Kein Paketdienst“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Beim Kopieren eines Angebots / Auftrags wird diese Einstellung übernommen. 

Tooltip: 

Keine Lieferung per Paketdienst 

### Checkbox Kein Overnight 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Overnight-Touren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Kein Overnight“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Tooltip: 

Keine Lieferung per Overnight 

### Checkbox Keine Spedition 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Speditionstouren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Keine Spedition“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg.
