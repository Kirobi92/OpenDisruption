# Vermietungsposition Registerkarte Schaden 
Vermietungsposition Registerkarte Schaden 

Im Schadensfall geben Sie auf dieser Registerkarte die wichtigen Daten zu diesem Vorgang ein. 
### Checkbox Schaden 
Liegt ein Schadensfall vor, ist diese Checkbox bereits aus der Vermietung vorbelegt. 

### Feld Schadensdatum 
Das Datum der Schadensmeldung wird eingetragen. 

### Feld Güte Info 
Hier wird der Zustand des Artikels angezeigt. 

Tooltip: 

Zustand des Artikels 

### Feld Qualität 
Die Qualität des Artikels kann eingetragen werden.
