# Lagerplätze Registerkarte Platztypen 
Lagerplätze Registerkarte Platztypen 

Auf dieser Registerkarte können Platztypen definiert werden. Z.B. Gitterbox, Palettenplatz. 
### Tabelle 

### Tabellenspalte Platztyp 
Hier wählen Sie den Typ des Lagerplatzes aus. Zur Verfügung stehen: 

- Blocklagerplatz - Erstlagerplatz - Normal - Reserveplatz - Systemplatz - Systemplatz für Auslagerung - Tourenlagerplatz - Zweitlagerplatz
