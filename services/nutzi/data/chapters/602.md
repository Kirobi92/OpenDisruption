Grundeinstellungen für eNVenta ERP / Referenz-Link ändern 

# Referenz-Link ändern Registerkarte Gespeicherte Links 
Neuer Referenz-Link Registerkarte Gespeicherte Links 

Auf dieser Registerkarte sehen Sie alle zuletzt gespeicherten Links. Diese können aufgerufen werden (mit Doppelklick wird die Registerkarte Links erstellen/bearbeiten wieder aktiviert) und die Referenz, die geändert werden soll, ist schon zur Bearbeitung eingeblendet. 

Wenn die Erstellung eines Referenz-Links über den Button abgebrochen wurde, so stehen diese Einträge noch immer in der Tabelle. Erst bei Verlassen von eNVenta ERP werden diese nicht gespeicherten Referenz-Links entfernt. 
### Tabelle 

### Tabellenspalte Programm 
Hier wird das Programm angegeben. D.h. die Art des Datensatzes, von welchem der Link ausgeht. 

Tooltip: 

Art des Datensatzes, von welchem der Link ausgeht. 

### Tabellenspalte Link-Eigentümer 
Hier wird eine Referenz angegeben. 

Tooltip: 

ID des Datensatzes, von welchem die Verlinkung ausgeht. 

### Tabellenspalte Programm 
Hier wird das Programm angegeben. D.h. die Art des Datensatzes, von welchem der Link ausgeht. 

Tooltip: 

Art des Datensatzes, von welchem der Link ausgeht. 

### Tabellenspalte Referenz 
Hier wird eine Referenz angegeben. 

Der Pfad bzw. die URL des verlinkten Inhalts kann als Suchkriterium dienen. 

Tooltip: 

Pfad bzw. URL des verlinkten Inhalts. Kann als Suchkriterium dienen. 

### Tabellenspalte Typ 
Tooltip: 

Name des Referenz-Link-Typens. 

### Tabellenspalte Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an.
