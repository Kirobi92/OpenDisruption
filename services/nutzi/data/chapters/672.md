# EK-Rabattgruppen-Zuordnung Registerkarte EK-Rabattgruppen 
EK-Rabattgruppen-Zuordnung Registerkarte EK-Rabattgruppen 

Auf dieser Registerkarte sehen Sie die Artikel, Artikel-Bereiche und Artikel-Gruppen, die dem Lieferanten / der EK-Rabattgruppe zugeordnet sind. 

Tooltip: 

Existierende Rabattgruppen 
### Tabelle 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte EK-Rabattgrp. 
Für die Preisfindung eines Artikels kann statt der Artikelgruppe auch eine EK-Rabattgruppe herangezogen werden. 

Tooltip: 

Angabe eines EK-Rabattgruppenkürzels 

### Tabellenspalte Bezeichnung 
Hier wird der Name der Rabattgruppe angegeben. 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein.
