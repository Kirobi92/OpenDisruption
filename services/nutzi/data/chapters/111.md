# Definieren von Zusatzfeldern 
Zusatzfelder definieren 

Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. Diese Datensätze werden in einer separaten Tabelle gespeichert und sind über die Spalte 'FSRowID' mit dem konkreten Datensatz verknüpft. Die Zusatzfelder dienen der reinen Speicherung von Informationen, es erfolgt keine Abfrage über Logiken für diese Felder. 

Der Form-Menü-Button ( ) verfügt in bestimmten Dialogen (z.B. Kunde) auch über den Eintrag Zusatzfelder konfigurieren . Hierüber können Sie in Dialogen weitere Felder mit Datenbankbezug anlegen, ohne dass die Datenbanktabelle angepasst werden muss. Diese Felder dienen nur der Information und werden beispielsweise nicht in weitere Vorgänge übernommen. Diese Zusatzfelder werden, wenn definiert, in den entsprechenden Dialogen auf einer zusätzlichen Registerkarte Zusatzfelder dargestellt. Außerdem können Sie über den Fokus visualisiert werden. Die Einstellungen dazu nehmen Sie im Dialog Einstellungen vor. 

Die Daten bleiben auch bei einer Historisierung erhalten. 

Je nach Dialog, aus welchem die Option aufgerufen wird, wird der aktuelle Tabellenname angezeigt. Diese Zuordnung steuert die entsprechende Speicherung in der Datenbank. Hierbei erfolgt die Speicherung der Zusatzfelder (Definition und Werte) nicht in der angegebenen Tabelle, sondern in einer eigenständigen Tabelle, welche über die FSRowID referenziert wird.

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Aufräumen 

Wenn Sie auf diesen Button klicken, wird überprüft, ob Zusatzfelder von bereits gelöschten Daten noch bestehen. Falls dem so ist, werden diese Werte ebenfalls bereinigt.
