Finanzbuchhaltung / Debitoren Factoring 

# Debitoren Factoring Registerkarte Verfügbarkeitsliste 
Debitoren Factoring Registerkarte Verfügbarkeitsliste 

Auf dieser Registerkarte werden zu jedem Kunden, der in die Tabelle eingelesen wird, Daten zur Kreditversicherung eingelesen. 

Es werden nur Kunden eingelesen, bei denen im Kundenstamm die Registerkarte Versicherung gefüllt ist. 
### Tabelle 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben. 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Konto 
Hier geben Sie die Kontonummer des Kunden ein. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Bezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Tabellenspalte PLZ 
Die Postleitzahl des Orts wird angegeben. 

Tooltip: 

Postleitzahl des Datensatzes 

### Tabellenspalte Ort 
Hier wird der Ort angegeben. 

Tooltip: 

Angabe des Orts 

### Tabellenspalte Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Tooltip: 

Angabe der Währung 

### Tabellenspalte Summe OP 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Vers. Betrag 
Hier wird der Betrag aus der Kreditversicherung angegeben. 

Tooltip: 

Hier wird der Betrag aus der Kreditversicherung angezeigt. 

### Tabellenspalte Überschreitung WKV 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags
