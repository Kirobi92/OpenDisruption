# Werkzeug Registerkarte Tabelle

Produktion (PPS) / Werkzeug 

# Werkzeug Registerkarte Tabelle 
Werkzeug Registerkarte Tabelle 

Diese Registerkarte zeigt in einer Übersicht alle bereits angelegten Werkzeuge. 

Der Status aus dieser Tabelle wird in die BA-AG Ressourcen übermittelt. 
### Tabelle 

### Tabellenspalte Werkzeug 
Hier wird die Werkzeugnummer angegeben. 

Tooltip: 

Angabe der Werkzeugnummer 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung des Werkzeugs angegeben. 

Tooltip: 

Name des Werkzeugs 

### Tabellenspalte Werkzeugart 
Hier wird die Werkzeugart angegeben. Dies können z.B. eine Druckgussform oder ein Stanzwerkzeug sein. 

Tooltip: 

Angabe der Werkzeugart 

### Tabellenspalte Ressource 
Hier wird eine Standard-Ressource angegeben. 

Tooltip: 

Angabe der Ressource 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
In diesem Feld wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
