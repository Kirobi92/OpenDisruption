# GAP Position ändern Registerkarte Retoure 
Position Registerkarte Retoure 

Wenn Sie die Checkbox Detailinfos in Vorgängen im Dialog Parameter Verkauf , Registerkarte Retoure aktivieren, wird diese Registerkarte zugänglich und die Ursprungsretoure innerhalb der Prozesse angezeigt. Die Informationen aus den Vorgängen werden in den entsprechenden Feldern angezeigt. 
### Feld Retoure 
Hier wird die Retourennummer angegeben. 

### Feld Menge Retoure 
Hier wird die Retourenmenge angegeben. 

Tooltip: 

Angabe der Menge 

### Feld Menge Retoure 
Tooltip: 

Angabe der Menge 

### Feld Info 
Hier wird eine Text-Information angegeben. 

Tooltip: 

Angabe einer Information 

### Feld Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Feld Bezeichnung 
Die Bezeichnung dient der einfachen Identifikation des Artikels und kann auch zur Suche verwendet werden. 

Bei Eingabe der Artikelnummer wird das Feld Bezeichnung automatisch mit dem Namen des Artikels gefüllt. 

Tooltip: 

Artikelbezeichnung 

### Feld Bestellnummer 
Hier wird die Lieferanten-Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikel-Bestellnummer 

### Feld Chargen 
Hier wird die Chargennummer des Retourenartikels angezeigt. 

### Feld Seriennummern 
Hier wird die Seriennummer des Retourenartikels angezeigt. 

### Feld Auftrag 
Hier wird die Nummer des Belegs aus den Belegarten Auftrag, Gutschrift, Bestellung und Belastung (Wawi) angegeben. 

Tooltip: 

Angabe der Belegnummer 

### Feld Auftragsart 
Wenn dieses Feld gefüllt ist, wird in der Auftragserfassung die hier hinterlegte Auftragsart voreingestellt. Sie kann jederzeit geändert werden. 

Dieses Feld steht in direktem Zusammenhang mit dem Parameter Auftragsart fix im Dialog Auftragsarten . 

Tooltip: 

Eingabe der Auftragsart 

### Feld Lieferdatum 
Hier wird der Liefertermin angegeben. 

Tooltip: 

Angabe des Liefertermins 

### Feld Kommission 
Hier wird die Kommissionsnummer aus dem Kundenauftrag angezeigt. 

Tooltip: 

Hier wird die Kommissionsnummer aus dem Kundenauftrag angezeigt. 

### Feld Beleg 
Hier wird die Belegnummer aus dem Einkauf (Bestellung, Anfrage, Belastung) angezeigt. 

Tooltip: 

Hier wird die Bestellnummer definiert. 

### Feld Bestellart 
Hier wird die Einkaufsart angegeben. 

Tooltip: 

Geben Sie die Einkaufsart an. 

### Feld Wareneingang 
Hier wird die Wareneingangsnummer angegeben. 

### Feld Eingangsdatum 
Hier wird das Wareneingangsdatum angegeben. 

### Feld LS Ursprungsbest. 
Hier wird die Lieferscheinnummer der Ursprungsbestellung einer Retoure angegeben. 

Tooltip: 

Die Lieferscheinnummer der Ursprungsbestellung einer Retoure. 

### Feld LS-Datum 
Hier wird das Datum des Lieferscheins angegeben. 

Tooltip: 

Datum des Lieferscheins 

### Feld Rechnung 
Hier wird die Rechnungsnummer angegeben. 

Tooltip: 

Angabe der Rechnungsnummer 

### Feld Fakturadatum 
Hier wird das Datum der Rechnung angegeben. 

Tooltip: 

Angabe des Fakturadatums 

### Feld BN Lieferant 
Hier wird die Bearbeitungsnummer des Lieferanten angegeben. 

Tooltip: 

Bearbeitungsnummer Lieferant 

### Feld RC-Code 
Hier wird der Retouren-Code angezeigt. 

Siehe Kapitel RC-Codes .
