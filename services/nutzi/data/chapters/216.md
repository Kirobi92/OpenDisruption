# Suche mit Merkmalen 
Suche mit Merkmalen 

Diese Funktion erreichen Sie über das Menü Optionen >> Mit Merkmalen . Die Option wird aktiviert und steht nun bei der Suche im Stamm-Dialog oder bei der Suche mit SQL zur Verfügung und kann um die Suche mit Partnern erweitert werden. 

Das Feld Bezeichnung beinhaltet die Bezeichnung der gewählten Abfrage. Wird eine neue Abfrage erfasst, ist das Feld Bezeichnung leer und Sie können einen individuellen Namen vergeben. 

Wenn eine bereits bestehende Abfrage geändert wird, steht in diesem Feld der Name der Abfrage. 

Markieren Sie das gewünschte Merkmal und übernehmen Sie es mit dem Button in das Auswahlfeld. Sie können auch mehrere Merkmale mit AND oder OR verknüpfen. Über den Button Abfragen können Sie Ihre Merkmalsabfrage speichern. Mit dem Button starten Sie die Suche.
