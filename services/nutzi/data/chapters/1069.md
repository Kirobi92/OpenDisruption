# Konfiguration von Versionierungen 
Konfiguration Version. 

Dieser Dialog öffnet sich im Dialog Versionierung über den Button Konfig , wo Sie die Einzelheiten für die Versionierung festlegen. 

Im oberen Bereich des Dialogs (Registerkarte Report-Felder ) stehen alle Felder der Report-Struktur zur Verfügung. Per Button oder per Button Hinzufügen wird das gewählte Report-Feld der Versionierung (Registerkarte Versionierung ) hinzugefügt. 

Nur wenn sich eines der angegebenen Felder für das Reporting geändert hat, erfolgt die Erstellung einer neuen Version des Druckbelegs. Auf diese Weise ist ein Nachdruck des Originals möglich, wenn es keine relevante Änderung gibt oder eben eine neue Druckversion des Dokumentes bei relevanter Änderung. 

Werden hier Felder aus Listen (Collections) hinzugefügt, beispielsweise aus der Positions-Collection so wird der Wert jeder einzelnen Position (jeder einzelne Collection-Eintrag) geprüft. So kann beispielsweise überwacht werden, ob es eine Mengen- oder Preisänderung an dem Beleg gab. 

Des Weiteren kann durch das Hinzufügen der FixPosNr aus der Positions-Collection überwacht werden, ob Positionen hinzugefügt bzw. gelöscht wurden. 

Sind keine Felder für die Versionierung gepflegt, so wird die Checkbox in der Spalte ‚Version. Aktiv‘ (im Dialog Versionierung ) deaktiviert.
