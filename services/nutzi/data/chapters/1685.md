# Endkontrollprotokoll 
Endkontrollprotokoll 

Dieser Dialog öffnet sich in der Endkontrolle >> Optionen >> Endkontrollprotokoll . Hier können alle erledigten, ungeprüften und Fehler-korrigierten Endkontrollen eingesehen werden. Dabei kann nach von / bis Datum, Sachbearbeiter, Lagerauftragsnummer und Endkontrollnummer selektiert werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Endkontrollprotokoll 
Tooltip: 

In diesem Dialog können alle erledigten, ungeprüften und Fehler-korrigierten Endkontrollen eingesehen werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Siehe Kapitel Zusatzfelder definieren .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen

Hier wird der Status der Position angegeben.

3-offen, zur Faktura freigegeben 
4-fakturiert 
5-geliefert (ohne Berechnung) 
6-storniert

### Tabellenspalte Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen.
