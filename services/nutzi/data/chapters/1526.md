# Fahrzeugstamm Registerkarte Tabelle 
Fahrzeugstamm Registerkarte Tabelle 

Auf dieser Registerkarte werden alle vorhandenen Fahrzeuge mit Kennzeichen, Ladegewicht, Volumen, Max. Breite und Min. Breite angezeigt. 

Wenn Sie auf einen Eintrag doppelklicken, werden die Daten in die Kopffelder und die Registerkarte Allgemein übernommen. 
### Tabelle 

### Tabellenspalte Fahrzeug 
Hier wird die Nummer des Fahrzeugs angegeben. 

Tooltip: 

Angabe der Fahrzeugnummer 

### Tabellenspalte Fahrzeug 
Hier wird der Name des Fahrzeugs angegeben. 

Tooltip: 

Angabe von Kennzeichen und Fahrzeugtyp 

### Tabellenspalte Kfz-Kennzeichen 
Hier wird das KFZ-Kennzeichen des Fahrzeugs angegeben. 

### Tabellenspalte Ladegewicht 
Hier wird das tatsächlich verladene Gewicht der Tour angegeben. 

Tooltip: 

Das tatsächlich verladene Gewicht der Tour wird angegeben. 

### Tabellenspalte Volumen 
Hier wird das tatsächlich verladene Volumen der Tour angegeben. 

Tooltip: 

Das tatsächlich verladene Volumen der Tour wird angegeben. 

### Tabellenspalte Breite max. 
Hier wird die maximale Breite angegeben. 

Bei der Berechnung der Maßeinheiten werden die Maßeinheiten aus dem Artikelstamm in Systemeinheiten umgerechnet. 

Tooltip: 

Angabe der Breite 

### Tabellenspalte Länge (max.) 
Hier wird die maximale Länge angegeben. 

Bei der Berechnung der Maßeinheiten werden die Maßeinheiten aus dem Artikelstamm in Systemeinheiten umgerechnet. 

Tooltip: 

Angabe der Länge 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Dieses Feld zeigt die Anzahl der tatsächlich eingelesenen Datensätze.
