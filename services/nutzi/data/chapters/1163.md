# Lageranschriften 
Lageranschriften 

Dieser Dialog öffnet sich im Auftrag Registerkarte Anschriften im Bereich Lieferanschrift über den 2. -Button neben dem Feld Firma 1 . Hier lassen sich auf einfache Weise Umlagerungen, z.B. in Abhollägern durchführen. 

Änderungen in den Lageranschriften können Sie nur im Dialog Lagerort vornehmen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Lageranschriften 
Tooltip: 

In diesem Dialog lassen sich auf einfache Weise Umlagerungen, z.B. in Abhollägern durchführen. 
Für mehr Informationen drücken Sie F1.

### Feld Transport Distanz 
Hier können Sie den Transportweg bis zur Grenze in % angeben. Dies vor dem Hintergrund, dass es sich um einen statistischen Wert für eine ungefähre Hochrechnung handelt. Deshalb muss der Sachbearbeiter hier den wahrscheinlichsten bzw. durchschnittlich zutreffenden Prozentsatz hinterlegen. 

Beispiel: 
Transportweg bis Adresse = 100 km 
Weg bis Grenze = 40 km 
=> Transport Distanz = 40 

Tooltip: 

Der anteilige Transportweg bis zur Grenze in Prozent
