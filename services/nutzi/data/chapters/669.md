# EK-Rabattgruppen-Zuordnung Registerkarte Artikel 
EK-Rabattgruppen-Zuordnung Registerkarte Artikel 

Um einer Lieferanten-Rabattgruppe Artikel zuzuordnen, wählen Sie jetzt auf der Registerkarte Artikel aus der linken Tabelle einen Artikel oder auch mehrere aus und ordnen Sie diese der rechten Tabelle Rabattgruppen mit dem Button zu. Der Button macht diese Zuordnung wieder rückgängig. 

Tooltip: 

Zuordnung von Artikeln 
### Feld Hauptlieferant 
Hier wird die Nummer des Hauptlieferanten für den Artikel angegeben. Dieser Lieferant wird dann im Bestellwesen als Lieferant vorgeschlagen. 

Tooltip: 

Nummer des Hauptlieferanten des Artikels. 

### Button Suche 

Mit diesem Button können Sie nach Lieferanten suchen. 

### Tabelle 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung dient der einfachen Identifikation des Artikels und kann auch zur Suche verwendet werden. 

Bei Eingabe der Artikelnummer wird das Feld Bezeichnung automatisch mit dem Namen des Artikels gefüllt. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte EK-Rabattgrp. 
Für die Preisfindung eines Artikels kann statt der Artikelgruppe auch eine EK-Rabattgruppe herangezogen werden. 

Tooltip: 

Angabe eines EK-Rabattgruppenkürzels 

### Tabellenspalte Bezeichnung 
Hier wird der Name der Rabattgruppe angegeben. 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Button Suche 

Mit diesem Button können Sie Artikel suchen. 

Tooltip: 

Auswahl der Artikel über die Artikelsuche
