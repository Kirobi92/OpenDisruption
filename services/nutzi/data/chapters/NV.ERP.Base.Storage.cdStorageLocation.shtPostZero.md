# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtPostZero

Ist dieser Schalter gesetzt, wird eine Buchung der Menge 0 im Lagerauftrag ermöglicht. Diese Funktion steht für das 1-stufige LVS zur Verfügung. 

Ist dieser Schalter nicht gesetzt, so wird in diesem Moment keine Buchung vorgenommen. 

Wenn die Checkboxen 

- Prüfziffer am Lagerplatz buchen? - Lager LHM fähig - 0 buchen 
nicht gesetzt sind, läuft im Programm das 2-stufige LVS. 

Tooltip: 

Ermöglicht eine Buchung der Menge 0 im Lagerauftrag.
