# Artikel Registerkarte Produktion 
Artikel Registerkarte Produktion 

Da es Länder in Europa gibt, welche Rohmaterial, Halbfabrikate und Endprodukte per Gesetz im System getrennt auf verschiedenen Lagern führen müssen, steht die Registerkarte Produktion zur Verfügung. 

Außerdem ist es möglich hier einen Artikel als sogenanntes Kleinteil zu definieren. Lesen Sie dazu das Kapitel Flexibles Produktionslager . 

Diese Registerkarte steht nur zur Verfügung, wenn das Modul Produktion lizenziert ist. 
### Combobox Materiallager 
Hier wählen Sie das Materiallager aus. 

Tooltip: 

Definiert das Lager 

### Feld Opt. Losgröße 
Hier wird eine Menge eingetragen, die beim Generieren eines Betriebsauftrags gezogen wird. 

Wird hier nichts eingetragen, wird im Betriebsauftrag mit Menge 1 weitergearbeitet. 

Die optimale Losgröße wird im Dialog Prod.-Auftrag erstellen angezeigt. 

Siehe Kapitel Produktionsauftrag erstellen . 

Tooltip: 

Angabe der Menge 

### Checkbox Kleinteil 
Wenn Sie diesen Parameter aktivieren, kennzeichnen Sie den Artikel als Kleinteil und er wird auf ein Lager für Kleinteile eingelagert. 

Organisatorisch sollte das Kleinteilelager in der Nähe der Produktion liegen, damit weite Wege vermieden werden, um z.B. kurzfristig Unterlagscheiben zu beschaffen. 

Damit haben Sie die Möglichkeit Material für einen Betriebsauftrag von mehreren Lagern zu entnehmen. 

So sparen Sie sich die Umlagerung für die Produktion vom Materiallager zum Produktionslager und das Kleinteilelager ist immer gefüllt. Dies ist sichergestellt über den Bestellautomaten, den Bestellvorschlag etc. 

Von diesem Kleinteilelager wiederum kann dann nur eine begrenzte Anzahl an Kleinteilen geholt werden, damit es nicht sofort leer ist. 

Siehe Kapitel Flexibles Produktionslager . 

Tooltip: 

Ein Artikel kann als Kleinteil definiert werden. 
Damit sparen Sie sich die Umlagerung vom Materiallager zum Produktionslager. 

### Feld Max. Menge KT 
Hier geben Sie die Menge für das Kleinteil an, um eine entsprechende Menge in der Produktion zur Verfügung zu haben. 

Tooltip: 

Max. Menge Kleinteil 
Max. Menge pro BA-Position, bis wann die Entnahme aus dem Kleinteillager erfolgen soll.
