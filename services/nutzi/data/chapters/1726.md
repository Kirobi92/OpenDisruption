# EK Konditionen in der Disposition 
EK Konditionen 

Dieser Dialog öffnet sich über den Button Kondition im Dialog Disposition , die Artikeldaten werden übergeben. 

Wenn Sie den Dialog EK Konditionen aus der Bestellung (über den Button Kondition auf der Registerkarte Positionen ) öffnen, erscheint er im Help Frame. 

Hier erhalten Sie eine Übersicht, wie der Preis für den Artikel zustandegekommen ist und welche Rabatte für die Position gezogen wurden. Dieser Dialog dient nur der Information. 

Eine genaue Beschreibung der Felder und Checkboxen entnehmen Sie bitte dem Kapitel Rabatt Einkauf . Mit dem Button neben dem Feld Rabatt-Nr. können Sie in die Rabattverwaltung verzweigen. 

Siehe Kapitel Rabatt Einkauf .

### Feld Menge / VP 
Es ist möglich, für ein und denselben Artikel unterschiedliche Mengeneinheiten für das Lager und den Einkauf zu führen. Beispielsweise können Sie einen Artikel in Stück am Lager führen, jedoch kistenweise einkaufen. Der Umrechnungsfaktor von der Lagereinheit zur Einkaufseinheit wird in das Feld Faktor eingetragen. Standardmäßig enthält das Feld den Wert 1 (Lagereinheit = Einkaufseinheit). 

Beispiel: 
Lager = 1 Kanister mit 100 kg, Einkauf = 1 kg, Faktor = 0,01. 

Die beiden Felder Einheit Verp. und Menge / VP müssen gemeinsam mit Werten gefüllt werden. Falls eines der Felder nicht gefüllt wird, erscheint eine Meldung. 

Tooltip: 

Inhalt der kleinsten Einkaufs-Verpackungseinheit in Lagereinheit
