# Anfragespiegel Registerkarte Positionen 
Anfragespiegel Registerkarte Positionen 

Diese Registerkarte enthält die Daten entsprechend der Selektion im Kopf des Dialogs. 
### Button Stückliste 

Ist der angefragte Artikel einer Position ein Stücklistenartikel, so kann die Stückliste über diesen Button angezeigt werden. Hierzu muss zuvor die entsprechende Anfrageposition selektiert werden. 

### Button Bestwert 

Dieser Button markiert die günstigsten Positionen. 

Tooltip: 

Die besten Ergebnisse werden markiert. 

### Tabelle 

Diese Tabelle zeigt die Lieferanten und die entsprechenden Belegnummern. 
### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip:

### Tabellenspalte Beleg

Hier wird die Bestellnummer definiert.

Hier wird der Nettobetrag angegeben.

In dieser Spalte wird der Positionsstatus angegeben. Zur Verfügung stehen: 

1 - Offen 
2 - Bestätigt 
3 - Wareneingang erfolgt 
4 - Rechnungsprüfung durchgeführt 
5 - Lieferung ohne Berechnung 
6 – Storniert

2 - Gesendet 
3 - Geantwortet

5 - Abgelehnt

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Maximale Anzahl der Positionen

Anzahl der gefundenen Positionen
