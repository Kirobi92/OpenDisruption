# Kundennachfrage 
Kundennachfrage 

Die „Kundennachfrage“ ist ein weiteres Informationselement: Auf einem Zeitstrahl wird die Historie des Kunden in Bezug auf einen Artikel angezeigt. So wird auf einen Blick sichtbar, ob auf Anfragen auch Käufe gefolgt sind, welche Mengen jeweils geordert wurden und ob ein Artikel regelmäßig bezogen wird. 

Mit diesem Element kann der Vertrieb im Beratungsgespräch unterstützt werden. In einer Beratung ist auf einen Blick erkennbar, ob der Kunde den Artikel bereits einmal gekauft hat oder nicht, bzw. ob er dies regelmäßig tut. 

Auf einer Zeitleiste werden die Vorgänge des Kunden der letzten 12 Monate, der letzten Quartale oder der letzten Wochen zu dem aktuellen Artikel angezeigt. Folgende Vorgänge werden unterstützt: 

- Aufträge - Angebote - Gutschrift - Produktseitenaufrufe ( eGate )
