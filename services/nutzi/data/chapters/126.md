eNVenta ERP / Job Zeitplan 

# Job Zeitplan Registerkarte Ausführung 
Job Zeitplan Registerkarte Ausführung 

Auf dieser Registerkarte legen Sie die Details für die Ausführung des Jobs fest. 
### Radiobutton Sofort 
Wenn Sie diesen Radiobutton aktivieren, können Sie keine weiteren Einstellungen vornehmen, es werden nur das aktuelle Datum und die Uhrzeit in das Feld Nächster Lauf eingetragen, sobald Sie auf klicken. 

Nur Jobs mit einem Eintrag im Feld Nächster Lauf werden verarbeitet. 

Das Anlegen von Sofort-Jobs ist an anderer Stelle im Text/Kapitel beschrieben. 

### Radiobutton Zeitplan 
Wenn Sie diesen Radiobutton aktivieren, können Sie mehrere Zeitpläne hinterlegen, z.B. getrennt nach Wochentag. 

Klicken Sie auf den Button , um einen neuen Zeitplan anzulegen. Dann werden weitere Felder sichtbar und zugänglich. 

### Radiobutton Folge-Job 
Wenn Sie diesen Radiobutton aktivieren, wird der angezeigte Job zu einem Folge-Job. D.h. er wird immer automatisch nach Ausführung des zugrunde liegenden Jobs (also der, welcher im Dialog Job Übersicht aktiviert ist) ausgeführt. Mit dieser Funktion können sogenannte Hierarchien in den Jobs (oder auch Abhängigkeiten) angelegt werden. Eine Anlage dieser Jobs im Job Zeitplan erübrigt sich! 

Außerdem erscheint das Feld Jobzeitplan-ID zur Eingabe eines Jobs. Dieser Job kann über den daneben stehenden Button gesucht werden. Es öffnet sich der Dialog Job-Auswahl . 

Siehe Kapitel Job-Auswahl . 

### Feld Jobplan 
Die Nummer des Jobplans wird eingetragen. 

### Button Suche 

Dieser Button öffnet den Dialog Job Auswahl zur Auswahl eines Jobs. Hier werden alle Jobs angezeigt, die in der Job Übersicht vorhanden sind. 

### Feld Bezeichnung 
Wenn Sie die Nummer des Jobplans eingetragen haben, wird dieses Feld automatisch mit dem Namen gefüllt. 

### Feld Status 
Der Status des Jobplans wird angezeigt.

Angabe des Status

### Tabellenspalte Zeitplan

### Tabellenspalte Nächster Lauf

Eingabe von Datum und Uhrzeit 

### Button Satz hinzufügen 

Um einen neuen Zeitplan anzulegen, klicken Sie diesen Button. Weitere Felder werden eingeblendet. 

### Button Löschen 

Wenn Sie einen Zeitplan löschen möchten, klicken Sie diesen Button. 

### Combobox Intervall 
Wählen Sie in dieser Combobox aus, welche Regel Sie anlegen wollen. Zur Verfügung stehen: 
- Einmal - Täglich - Wöchentlich - Monatlich 
Je nach gewähltem Intervall erhalten Sie unterhalb der Tabelle andere Felder zur Eingabe. 

Wenn Sie Einmal auswählen, können Sie ein Datum und eine Uhrzeit anlegen, zu dem der Lauf einmalig starten soll, analog zur Option Sofort , nur eben zu einem anderen gewählten Zeitpunkt. 

Wenn Sie die Option Täglich wählen, geben Sie einen Datumsbereich an, für welchen dies gilt. Anschließend geben Sie an, in welchem Rhythmus der Job laufen soll und optional eingegrenzt auf einen Zeitraum, z.B. nur während der normalen Arbeitszeit oder nur nachts. 

Wenn Sie die Option Wöchentlich angeben, können Sie die Angaben aus Täglich zusätzlich noch auf die Wochentage einschränken, z.B. nur Montag, Mittwoch und Freitag. 

Über das Intervall Monatlich haben Sie die Möglichkeit Jobs monatlich bzw. auch quartalsweise zu definieren. 

Das funktioniert folgendermaßen: 
Werden zum Intervall Monatlich ein oder mehrere Monate aktiviert, so zählt der eingetragene Tag nur für die angegebenen Monate. In den anderen Monaten wird der Job nicht ausgeführt. Ist kein Monat aktiviert, gelten alle Monate. 

Sie können mehrere parallele Zeitpläne anlegen, das System ermittelt immer automatisch den Zeitpunkt für den nächsten Lauf. Klicken Sie einfach auf den Button , um einen weiteren Zeitplan anzulegen. 

### Feld Datum von 
Hier kann für den Start des Jobs ein Datum eingetragen werden. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld Datum bis 
Hier kann für das Ende des Jobs ein Datum eingetragen werden. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Feld Alle .. Minuten 
Hier können Sie angeben, wie oft (in Minuten) der Job ausgeführt werden soll. 

### Feld Zeit von (hh:mm) 
Hier wird die Uhrzeit angegeben, zu dem ein Job startet. 

### Feld Zeit bis (hh:mm) 
Für Jobs, die öfters am Tag ausgeführt werden, können Sie hier angeben, um welche Uhrzeit der Job das letzte Mal laufen soll. 

Tooltip: 

Zeigt die Zeit-bis an. 

### Feld Tag 
Hier geben Sie einen Tag des Monats ein, an dem der Job laufen soll. 

### Field Panel Wochentage 

Wenn Sie das Intervall Wöchentlich gewählt haben, können Sie hier über die Wochentage die Jobs weiter eingrenzen. 

### Field Panel Monat 

Wenn Sie das Intervall Monatlich und einen bestimmten Tag ausgewählt haben, dann hier noch einen oder mehrere Monate aktivieren, wird der Job nur an diesem Tag in diesen Monaten ausgeführt.
