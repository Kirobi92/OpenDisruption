# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtNV4_IgnoreMaxStockLevel

Wenn Sie diesen Parameter aktivieren, wird der Maximalbestand auf diesem Lager ignoriert. Es kann also unbeschränkt eingebucht werden. 

Tooltip: 

Ignoriert den Maximalbestand auf diesem Lager.
