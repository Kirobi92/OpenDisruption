# Aktionsdefinition Registerkarte Wiedervorl. 
Aktionsdefinition Registerkarte Wiedervorlage 

Auf dieser Registerkarte definieren Sie Ihre Einstellungen, wie Sie es aus dem bereits bekannten Dialog Wiedervorlage erfassen kennen. 
### Checkbox Aktiv 
Diese Checkbox ist standardmäßig gesetzt. Im Einzelfall kann sie auch deaktiviert werden. Dann wird keine Aktion angestoßen. 

### Feld Vorgang 
Hier kann ein Betreff angegeben werden. 

Dieser wird dann in der Wiedervorlage als "Vorgang" angezeigt. 

Wird dieses Feld leer gelassen, füllt eNVenta das Feld in der Wiedervorlage automatisch mit dem Referenztext. 

Tooltip: 

Hier kann ein Betreff angegeben werden. 
Dieser wird dann in der Wiedervorlage als "Vorgang" angezeigt.

### Feld Nachricht 
In diesem Feld definieren Sie eine Nachricht, die in den generierten Wiedervorlagen angezeigt werden soll. 

Tooltip: 

Definieren Sie eine Nachricht, die in den generierten Wiedervorlagen angezeigt werden soll. 

### Feld Tage WV 
Hier geben Sie die Anzahl Tage nach dem auslösenden Event an, zu dem eine Wiedervorlage generiert wird. 

Tooltip: 

Anzahl der Tage nach dem auslösenden Event, an dem die Wiedervorlage generiert wird. 

### Combobox Termin-Typ 
Hier geben Sie den Termin-Typ an. Zur Verfügung stehen: 
- zur Bearbeitung - zur Info - zur Genehmigung 

### Combobox Priorität 
Hier geben Sie die Priorität des Termins ein. Zur Verfügung stehen: 
- normal - wichtig - sehr wichtig 

Tooltip: 

Priorität des Termins 

### Feld Warn. n. Tagen 
Hier geben Sie die Anzahl der Tage nach der Wiedervorlage an, an dem die Warnung generiert wird. 

Tooltip: 

Anzahl der Tage nach der Wiedervorlage, an dem die Warnung generiert wird. 

### Combobox Abteilung 
Soll die Wiedervorlage an eine Abteilung gesendet werden, wählen Sie diese hier aus. 

Eine Wiedervorlage kann entweder an eine Abteilung oder einen Empfänger gesendet werden; beides ist nicht möglich! 

Tooltip: 

Hier kann definiert werden, an welche Abteilung die Wiedervorlage generiert werden soll. 
Es kann entweder direkt eine Abteilung oder ein Platzhalter für eine Abteilung angegeben werden. 

### Button Verzweigen 

Mit diesem Button gelangen Sie in den Text-Wizard . Dort kann der Abteilung über definierte Logik eine Wiedervorlage generiert werden. 

Siehe Kapitel Text-Wizard . 

### Combobox Empfänger 
Soll die Wiedervorlage an eine Person (Empfänger) gesendet werden, wählen Sie diese hier aus. 

Eine Wiedervorlage kann entweder an einen Empfänger oder eine Abteilung gesendet werden; beides ist nicht möglich! 

Tooltip: 

Hier kann definiert werden, an welchen Emfänger die Wiedervorlage generiert werden soll. 
Es kann entweder direkt ein Emfänger oder ein Platzhalter für den Emfänger angegeben werden.
