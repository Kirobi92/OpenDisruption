# Parameter Office Registerkarte Paketstraße 
Parameter Office Registerkarte Paketstraße 

Unter Paketstraße versteht man den Weg, den ein Paket im Unternehmen zurücklegt, bevor es extern weiter versandt wird. So lässt sich bei Verzögerungen in der Weiterleitung eines Pakets auch leichter nachverfolgen, wo das Paket liegen geblieben sein könnte. 

Diese Registerkarte steht exklusiv im Zusammenhang mit der Funktion Paketstraße im Modul LVS . In der rechten Tabelle können Sie Paketstraßen definieren. 

Siehe Kapitel Paketstraße . 

Tooltip: 

Paketstraße 
### Feld Paketstraße 
Hier wird die Nummer der Paketstraße angezeigt. 

Tooltip: 

Zeigt die Paketstraßennummer 

### Feld Paketstraße 
Hier wird der Name der Paketstraße angezeigt. 

Tooltip: 

Zeigt die Bezeichnung der Paketstraße an. 

### Combobox Waagenart 
Hier wählen Sie die Art der Waage aus. 

eNVenta unterstützt folgende Waagen: 

- Hersteller: Bosche 
Typ: WAI - Hersteller: Bizerba 
Typ: Industrie-Terminal ST 

Tooltip: 

Angabe der Waagenart

Hier wird ein (Standard-)Drucker angegeben.

Hier können Sie die IP-Adresse der Waage angeben.

V-LOG Etikettendrucker 

### Button Einfügen 

Mit diesem Button können Sie eine neue Paketstraße in die Tabelle eingeben. Die Nr. für die Paketstraße wird automatisch vergeben. Die Bezeichnung können Sie individuell wählen. Dazu geben Sie noch die Com-Port-Waage und den Drucker an. 

### Button Entfernen 

Mit diesem Button können Sie eine Paketstraße aus der Tabelle entfernen.
