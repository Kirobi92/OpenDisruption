# Kapitel NV.ERP.Base.Common.cdSystemMM.shtCollectiveDeliveryNote

Wenn dieser Parameter gesetzt ist, werden bei allen Aufträgen des Kunden mit Kennzeichen ‚Sammelrechnung‘ alle Positionen in Lieferschein und Packliste angedruckt. 

In diesem Fall kann im Auftrag keine abweichende Lieferanschrift hinterlegt werden. Der Vergleich der Lieferanschrift bezieht sich immer auf Firma 1 und Ort der Lieferanschrift bzw. Rechnungsanschrift, wenn das Feld Firma 1 der Lieferanschrift leer ist. 

Achtung: Dieser Parameter sollte nicht gesetzt sein, wenn mit einem zweiteiligen oder dreiteiligen Verbund gearbeitet wird! 

Ist die Checkbox Nur Packlistenpositionen auf Lieferschein im Dialog Parameter Versand / Logistik gesetzt, werden beim Druck des Lieferscheins in der Kommissionierung nur die Positionen der aktuell am Bildschirm angezeigten Packliste gedruckt. Dies ist unabhängig davon, ob der Parameter Sammellieferschein / -Packliste hier gesetzt ist.
