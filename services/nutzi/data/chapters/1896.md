# Steuerschlüssel in der Rechnungsprüfung ändern 
Steuerschlüssel ändern 

Der Button neben den Feldern Fracht und Nebenkosten bzw. der Button Steuerschl . in der Rechnungsprüfung auf der Registerkarte Allgemein öffnet den Dialog Steuerschlüssel ändern . Hier können Sie einen anderen Steuerschlüssel auswählen. Bei Übernahme dieses Steuerschlüssels wird der Datensatz automatisch aktualisiert. 

Bei der Änderung der Steuerschlüssel auf Positionsebene werden die Aufwandskonten automatisch neu ermittelt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Steuerschlüssel ändern 
Tooltip: 

In diesem Dialog können Sie den Steuerschlüssel für eine Rechnungsprüfung ändern. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
