# Verbandsabrechnung Debitor Registerkarte Direktverbuchung 
Registerkarte Direktverbuchung 

Auf dieser Registerkarte werden die Daten für die Direktverbuchung erfasst. 
### Feld Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Dieser Button öffnet den Dialog Kontoauswahl , um komfortabel das gesuchte Konto zu wählen. 

Siehe Kapitel Kontoauswahl . 

### Feld Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Feld Belegnummer 
Hier wird die Belegnummer angegeben. 

Tooltip: 

Angabe der Belegnummer 

### Checkbox Aus Vorschlag 
Wenn Sie diese Checkbox aktivieren, wird die originale Belegnummer aus dem Abrechnungsvorschlag übernommen. 

### Feld Belegnummer 2 
Für eine Direktverbuchung kann hier die Belegnummer 2 eingetragen werden. 

Tooltip: 

Angabe der Belegnummer 2 

### Checkbox Aus Vorschlag 
Wenn Sie diese Checkbox aktivieren, wird die originale Belegnummer aus dem Abrechnungsvorschlag übernommen.
