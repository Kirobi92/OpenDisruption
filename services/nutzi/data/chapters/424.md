In dieser Registerkarte kann dann das Feld zugewiesen werden, welches die Verbindung darstellt. 
### Button Neue Zeile 

Mit diesem Button fügen Sie eine neue Zeile in die Tabelle ein. 

### Button Lösche Zeile 

Mit diesem Button löschen Sie eine Zeile der Tabelle. 

### Button -> Expert 

Mit diesem Button können Kriterien der Tabelle in das Experten-Register verschoben werden, damit das SQL nachgearbeitet werden kann.
