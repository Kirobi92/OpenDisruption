# Kapitel NV.ERP.MM.Sales.cdOrderType.lngRetirementAccountID

Dieses Feld ist nur zugänglich, wenn die Checkbox Interner Warenabgang gesetzt ist. 

Hier geben Sie eine Kontonummer für den internen Warenabgang an. 

In einem solchen Fall handelt es sich um ein Pflichtfeld! 

Ein Speichern der Auftragsart ohne angegebene Kontonummer wird verhindert. 

Tooltip: 

Das Konto zum Datensatz.
