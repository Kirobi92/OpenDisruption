# Übersicht über bestehende Rahmenbestellungen / Rahmenverträge 
Rahmenbestellungen 

Dieser Dialog öffnet sich über den Button Historie in der Rahmenbestellung . Hier werden alle bestehenden Rahmenverträge zum ausgewählten Lieferanten angezeigt. 

Ist kein Lieferant ausgewählt, werden über den Button alle bestehenden Rahmenverträge in die Tabelle eingespielt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Rahmenbestellungen 
Tooltip: 

In diesem Dialog werden alle bestehenden Rahmenverträge zum ausgewählten Lieferanten angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Feld Lieferant 
Die Lieferantennummer wird aus der Rahmenbestellung voreingestellt. Es ist möglich den Lieferanten zu überschreiben, um Rahmenbestellungen eines anderen Lieferanten zu betrachten. 

Tooltip: 

Angabe der Lieferantennummer 

### Button Suche 

Mit diesem Button können Sie nach Lieferanten suchen. 

Siehe Kapitel Lieferantenauswahl . 

### Feld Anschrift 
Hier wird die Adresse angegeben. 

Tooltip: 

Angabe der Anschrift 

### Combobox Status 
Zur Selektion kann hier ein Status ausgewählt werden. 

Tooltip: 

Status der Rahmenbestellung. 

1 = offen 
2 = abgeschlossen (alle Positionen sind im Status 2) 

### Tabelle Rahmenbestellungen 

Die Tabelle zeigt die Rahmenbestellungen zum selektierten Lieferanten an. 
### Tabellenspalte Vertrag 
Hier wird die Vertragsnummer angegeben. 

Tooltip: 

Angabe der Vertragsnummer 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Status der Rahmenbestellung. 

1 = offen 
2 = abgeschlossen (alle Positionen sind im Status 2) 

### Tabellenspalte Datum von 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Tabellenspalte Datum bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein.

Angabe des Ansprechpartners

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
