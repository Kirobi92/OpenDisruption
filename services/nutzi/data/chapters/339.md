# Anleitung zum Datenimport unter Verwendung der Set-Funktion 
Das Set wird verwendet, wenn aus einer Datenzeile, welche in der Importdatei vorhanden ist, mehrere Datensätze gebildet werden müssen, um diese importieren zu können. Als Beispiel dient hier der Import mehrerer Preise auf einmal in verschiedene Preisgruppen. 

Die Importdatei sieht z.B. wie folgt aus: 

Artikelnr; Preis1; Preis2; Preis3 

Demo-Artikel; 10,00; 20,00; 30,00; 

Generell legen Sie als erstes das Import-Format im Dialog Datenimport an. 

Hier wird bereits pro Preis ein zusätzliches Feld für die Preisgruppe und die „Ab Menge“ angelegt, da diese als Pflichteingaben in der Preisgruppen-Verwaltung vorhanden sind. Dies wird bereits durch das System bestimmt und beim Import müssen diese Felder gefüllt werden. Falls diese Werte nicht in unserer Datei vorhanden sind, werden diese per Standard-Wert gefüllt. 

Als nächstes definieren Sie im Dialog Datenimport Set das Set. 

Anschließend müssen manuell die Felder angelegt werden (unten links), welche importiert werden sollen, also hier was in die Preisgruppe geschrieben werden soll. Anschließend müssen die Werte aus den einzelnen Spalten der Import-Datei diesen Feldern zugewiesen werden, damit während des Imports die einzelnen Zeilen generiert werden. 

Hierfür legen Sie in der rechten Tabelle jeweils eine Zeile für jeden zu importierenden Datensatz an.
