# Kapitel NV.ERP.MM.Sales.cdOrderType.shtTargetStorageID

Hier wählen Sie das Ziellager aus. Die Ware wird dann auf dieses Lager geliefert. 

Bei der Faktura eines Auftrags wird der Warenabgang direkt wieder als Warenzugang auf das Ziellager gebucht. 

Umlagerungen sollten besser über den Umlagerungsprozess ausgeführt werden! Der Vorteil ist u.a.: Für die Einlagerung kann der bestehende Prozess verwendet werden. 

Für das Mietobjekt gilt: auf dieses Lager wird das Mietobjekt nach Rückgabe aus der Vermietung gebucht.
