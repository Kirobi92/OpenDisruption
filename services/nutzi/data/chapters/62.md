# Zoom-Item Einstellungen - Beispiel 
Zur besseren Veranschaulichung, wie die Anlage eines Zoom-Elements aussehen kann, wird dies nachfolgend anhand eines Beispiels bildlich gezeigt. Wir haben uns für das Beispiel Kritische Kunden entschieden. 

Um ein Zoom-Element das erste Mal anzulegen, drücken Sie im Dialog eNVenta Zoom den Button . Nach der Auswahl eines Zoom-Elements erscheint dann automatisch der Dialog Einstellungen zum Zoom-Element. 

Um aus einem bereits angelegten Zoom-Element in die Einstellungen zu gelangen, drücken Sie den Button , rechts in der Leiste. Siehe nachfolgende Abbildung. 

## Einstellungen 
Einstellungen Registerkarte Allgemein 

Auf der Registerkarte Allgemein werden die Einstellungen für das Zoom-Element angezeigt, wie z.B. der Titel, wie oft das Element aktualisiert wird etc. Der Titel sollte durch eine möglichst sprechende Bezeichnung ersetzt werden, damit der Sachbearbeiter sofort den Zweck des Zoom-Elements erkennt. 

In der Tabelle werden die verschiedenen Parameter des Zoom-Elements angezeigt. Je nach Zoom-Element stehen verschiedene Parameter zur Selektion zur Verfügung. Hierbei kann es sich um ein Eingabefeld, eine Auswahlliste oder nur eine Checkbox handeln. Weitere Parameter können im Customizing hinzugefügt werden. 

Die verschiedenen Möglichkeiten der Einstellungen werden in der Registerkarte Beschreibung detaillierer erläutert.

## Beschreibung des Zoom-Elements 
Sie erhalten eine Liste der Kunden, deren Kreditlimit überschritten ist oder die im Kundenstamm (Kundenstatus) auf Liefersperre bzw. Gesamtsperre gesetzt sind. 
Kunden mit einer kritischen Situation können zusammengestellt und einfach überblickt werden. Insbesondere die finanziell problematischen Kunden mit den höchsten Kreditlimitüberschreitungen können sofort erkannt werden. 
Sie erhalten eine sortierte Liste nach Kundenstatus und Limitdifferenz. 
Es werden der Betrag der Kreditlimitüberschreitung und auch die prozentuale Überschreitung dargestellt: 
- Spalte 'KS' (Kundensperre):  rot = Gesamtsperre, gelb = Liefersperre - Spalte 'Limitdifferenz': gelb = Kreditlimitüberschreitung Über Parameter kann auf Kundengruppen, Vertreter, interne Vertreter und/oder Gebiete eingeschränkt werden. 
Der Aufruf erfolgt im Kundenstamm mit Doppelklick auf die Spalten 'Kundennr', 'Firma1', 'KS'. 
Der Aufruf der Bonitätsinformation (Dialog Bonität ) erfolgt mit Doppelklick auf die Spalten 'Limitdifferenz' und '%'. 
Die Aktualisierung empfehlen wir im Standard einmal täglich. Zugangsberechtigt sind nur die Benutzer, die auch das Recht für den Dialog Kundenbonitätsliste (frmCustomerSolvencyList) besitzen. 
## Aktualisierung des Zeitplans 
Auf der Registerkarte Aktualisierung Zeitplan wird der Zeitplan aktualisiert. Dazu ist es notwendig, dass ein Job-Server eingerichtet ist. Über den Zeitplan definieren Sie wann und wie häufig die Zahlen neu berechnet werden sollen. Im Standard erfolgt die Aktualisierung der Zahlen immer um 03:00 Uhr nachts, da hier die Auslastung der Server entsprechend gering ist. Je nach Komplexität der Berechnung kann der Vorgang pro Zoom-Element entsprechend andauern und daher erfolgt die Berechnung nicht alle x Minuten, sondern nur über die Einstellung des Zeitplans.
