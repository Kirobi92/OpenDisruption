# Retoure Registerkarte Kopfdaten 
Retoure Registerkarte Kopfdaten 

Auf dieser Registerkarte können bis zu 3 Rabatte bzw. Zuschläge prozentual oder absolut mit Text angegeben werden. Wird aus dieser Retoure eine Gutschrift generiert, werden diese Rabatte/Zuschläge entsprechend in die Gutschrift übernommen. 
### Feld Kundenstatus 
Hier wird der Kundenstatus angegeben. 

Tooltip: 

Angabe des Kundenstatus 

### Feld Sperrgrund 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Sperrgrund zum Kundenstatus 

### Combobox Thekengruppe 
Hier wird eine Thekengruppe angegeben. 

### Checkbox Lager --> Benutzer 
Wenn Sie diese Checkbox aktivieren, wird das Lager bei der Einlagerung auf das am Benutzer hinterlegte Lager gedreht. 

Tooltip: 

Das Lager wird bei der Einlagerung auf das Lager gedreht, das dem Benutzer hinterlegt ist. 

### Checkbox Versandauflösung 
Wenn Sie diese Checkbox aktivieren, werden die eingelagerten Mengen für rückständige Aufträge verwendet. 

Siehe Kapitel Versandauflösung . 

Tooltip: 

Die eingelagerten Mengen werden für rückständige Aufträge verwendet.
