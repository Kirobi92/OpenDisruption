# IBAN + BIC Update 
IBAN + BIC Update 

Dieser Dialog öffnet sich im Bankenstamm >> Optionen >> IBAN + BIC . Für den Zahlungsverkehr mit IBAN und BIC können Sie hier die entsprechenden Daten automatisch ergänzen. 

Für die Berechnung der IBAN-Nummern nutzt eNVenta ein Standardverfahren (DE, zwei Prüfziffern, gefolgt von der Bankleitzahl und der Kontonummer). Allerdings gibt es Kreditinstitute (z.B. Deutsche Bank, Commerzbank), die eine eigene Berechnungsmethode einsetzen. Bitte prüfen Sie das! 
## Steuerelemente des Formulars 

### Bezeichnungsfeld IBAN + BIC Update 
Tooltip: 

In diesem Dialog können Sie die entsprechenden Daten für den Zahlungsverkehr mit IBAN und BIC verwalten. 
Für mehr Informationen drücken Sie F1.

### Button Import 

Dieser Button öffnet den Dialog Datenimport . Wenn Sie hier über das Format IBAN suchen, werden die passenden Werte in die Tabelle eingelesen. Die Werte werden per CSV aus einer Bankleitzahlen-Datei eingelesen. 

Sind für ein und die gleiche BLZ mehrere BICs vorhanden (z.B. Deutsche Bank, Apotheker-Bank) können die Postleitzahl und evtl. der Ort in die Suche mit einbezogen werden. 

Im Zweifelsfall wird die BIC aus dem Satz mit dem Merkmal = 1 verwendet. Beim Typ BANK wird die IBAN aus BLZ und Kontonummer errechnet und eingetragen. 

Kann kein passender Datensatz zugeordnet werden, erhalten Sie eine Meldung.
