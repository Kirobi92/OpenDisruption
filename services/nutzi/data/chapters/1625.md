# Neue Position in der Tableauverwaltung 
Neue Position 

Der Button Neu in der Tableauverwaltung auf der Registerkarte Allgemein öffnet diesen Dialog. Hier können Positionen einem Tableau zugeordnet werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Neue Position 
Tooltip: 

In diesem Dialog können Positionen einem Tableau zugeordnet werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Combobox LHM Typ 
Hier wird der Typ des Ladehilfsmittels angegeben. 

Tooltip: 

Ladehilfsmittel-Typ. 

### Combobox Behälter Links 
Auflistung aller Lagerplätze, die sich auf dem Tableau befinden. Hier wird eine Tableaupositionsnummer ausgewählt, die direkt links vom aktuellen Lagerplatz steht (Grundlage der X-Koordinate). 

Tooltip: 

Wählen Sie die Tableau-Positions-Nummer aus, die sich links von dem Lagerplatz befindet. 

### Combobox Behälter Unten 
Auflistung aller Lagerplätze, die sich auf dem Tableau befinden. Hier wird eine Tableaupositionsnummer angegeben, die direkt unterhalb des aktuellen Lagerplatzes steht (Grundlage der Y-Koordinate). 

### Dock Panel Orientierung 

Hier definieren Sie, ob der Behälter im Verhältnis zu seiner eingegebenen Länge/Breite im Hochformat oder Querformat eingelagert wird. 
### Radiobutton Hochformat 
Das Tableau wird mit derselben Längen-, Höhen- und Breitenangabe von dem ausgewählten LHM angelegt. Die Länge und Breite des LHM's werden bei der Größe des Tablars vertauscht. 

### Radiobutton Querformat 
Das Tableau wird mit derselben Höhenangabe von dem ausgewählten LHM angelegt. Die Länge und Breite des LHM’s werden bei der Größe des Tablars vertauscht. 

### Checkbox Horizontal auffüllen 
Wenn diese Checkbox aktiviert wird, wird das Tableau horizontal aufgefüllt. 

### Feld Anzahl 
Wenn Sie hier eine Ziffer eintragen, lassen sich mehr als ein Behälter erzeugen.
