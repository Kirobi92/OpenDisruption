# GAP Position ändern Registerkarte Lagerauftrag

Verkauf / Gutschriftsabrechnungsverfahren / GAP Position ändern 

# GAP Position ändern Registerkarte Lagerauftrag 
Position Registerkarte Lagerauftrag 

Gilt nur wenn das LVS lizenziert ist! 

Auf dieser Registerkarte werden Informationen zum Lagerauftrag angezeigt. 

Die gelieferte Menge eines LVS-Auftrags wird mit Druck des Lieferscheins angepasst. Die kommissionierte Menge wird auf dieser Registerkarte angezeigt. 
### Feld Menge komm. 
Hier wird die kommissionierte Menge angezeigt. 

Tooltip: 

Menge geliefert: 

Die tatsächlich durch den Lieferscheindruck als gelieferte, im Auftrag eingetragene Menge. 

### Feld Menge offen 
Hier wird die offene, d.h. verfügbare Menge angezeigt. 

Tooltip: 

Menge geliefert: 

Die tatsächlich durch den Lieferscheindruck als gelieferte, im Auftrag eingetragene Menge. 

### Combobox Versandlager 
Hier wird das Versandlager angezeigt. 

Tooltip: 

Definiert das Lager 

### Checkbox Zuladung 
Wenn es möglich ist, diesen Artikel "zuzuladen", aktiviereren Sie diese Checkbox. 

### Tabelle Lagerauftragspositionen 

### Tabellenspalte Status 
In dieser Tabellenspalte wird der Pick-Status des Lagerauftrags angegeben: 
- offen - Bearbeitung - erledigt - storniert. 

Tooltip: 

Status der Lagerauftragsposition: 
•offen 
•Bearbeitung 
•erledigt 
•storniert 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Gesperrt 
Wenn Sie diese Checkbox aktivieren, ist der Kommissioniervorgang gesperrt. Sie wird gesetzt, wenn die Auftragsposition "liefergesperrt" ist. 

Tooltip: 

Aktiviert: Kommissioniervorgang ist gesperrt. 

### Tabellenspalte Pickmenge 
In dieser Spalte wird die gepickte Menge angezeigt. 

Tooltip: 

Angabe der Pickmenge 

### Tabellenspalte Menge Soll 
In dieser Spalte wird die Zielmenge angezeigt, die gepickt werden soll. 

Tooltip: 

Angabe der Zielmenge 

### Tabellenspalte Pick-Sachbearbeiter 
In dieser Spalte wird der Sachbearbeiter angezeigt, der die Ware verpackt hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Erfassung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der Erfassung eingetragen. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Tabellenspalte Änderung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der letzten Änderung eingetragen. 

Tooltip: 

Datum der letzten Änderung 

### Tabellenspalte Quelllager 
Hier wird das Quell-Lager angegeben. Es werden die Nummer und der Name des Lagers angezeigt. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Quell-Platz 
In dieser Spalte wird der Platz angezeigt, von dem entnommen wird. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Ziellager 
Hier wird das Ziellager angegeben. Es werden die Nummer und der Name des Lagers angezeigt. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Ziel-Platz 
In dieser Spalte wird der Platz angezeigt, auf den eingelagert/umgelagert werden soll. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Kommiss. Etikettendrucker 
In dieser Spalte werden das Druckdatum und die Uhrzeit des Kommissionierungsetikettes angezeigt. 

### Tabellenspalte Sachbearbeiter 
In dieser Spalte wird der Sachbearbeiter angezeigt, der den Lagerauftrag als letztes bearbeitet hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Pickinfo 
In dieser Spalte wird die Pick-Information angezeigt. 

### Tabellenspalte Lagerauftrag 
Hier wird die Lagerauftragsnummer angegeben. 

Tooltip: 

Angabe der Lagerauftragsnummer 

### Tabellenspalte Fixpos 
Hier wird eine fixe, eindeutige Positionsnummer des Datensatzes angegeben. 

Tooltip: 

Eindeutige Positionsnummer des Datensatzes.
