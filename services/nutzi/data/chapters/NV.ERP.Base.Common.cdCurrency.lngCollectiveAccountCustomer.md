# Kapitel NV.ERP.Base.Common.cdCurrency.lngCollectiveAccountCustomer

Hier wird das entsprechende, zur Währung gehörende Kundensammelkonto hinterlegt. 

Es ist notwendig für jede Währung ein eigenes Sammelkonto für Debitoren (Forderungen aus Lieferungen und Leistungen) anzugeben, damit bei der Neuanlage eines Debitors das Sammelkonto währungsabhängig automatisch zugeordnet werden kann. Die Sammelkonten sind im Sachkontenstamm in der entsprechenden Währung zu führen! 

Tooltip: 

Geben Sie das zur Währung gehörende Kundensammelkonto ein.
