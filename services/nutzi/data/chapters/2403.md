# Stapel Bearbeitung 
Stapel Bearbeitung 

Dieser Dialog öffnet sich im Journal über den Button Stapelbearbeitung . Es werden alle gefundenen Stapelnummern in die Tabelle eingelesen. 

Wenn Sie auf eine Zeile in der Tabelle der Stapelbearbeitung einen Doppelklick ausführen, öffnet sich der Dialog Journal und zeigt alle Buchungen des Stapels an. 

Über den Button Definitiv setzen wird die Stapelnummer entfernt. Erst jetzt können die Datensätze an die Kostenrechnung übergehen. 

Über den Button können Sie den markierten Stapel mit allen enthaltenen Buchungen löschen. Diese Buchungen sind dann aus dem Journal und den Offenen Posten gelöscht. 

Siehe Kapitel Journal . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Stapelbearbeitung 
Tooltip: 

In diesem Dialog werden alle zu einem Stapel gefundenen Belege angezeigt zur Übergabe an die Kostenrechnung. 
Für mehr Informationen drücken Sie F1.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Über diesen Button können Sie den markierten Stapel mit allen enthaltenen Buchungen löschen. Diese Buchungen sind dann aus dem Journal und den Offenen Posten gelöscht. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Definitiv setzen 

Mit diesem Button entfernen Sie die Stapelnummer von der Buchung, so dass diese Buchung in die Kostenrechnung übernommen werden kann. 

Tooltip: 

Setzt den Stapel definitiv, sodass diese Buchungen in die KORE übernommen werden können.
