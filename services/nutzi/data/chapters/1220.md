# ABC Analyse 
ABC Analyse 

Dieser Dialog öffnet sich im Auftrag >> Statistik >> ABC Analyse . Hier können ABC-Analysen für Kunden und Artikel erstellt werden. Bei der Analyse werden nur die Umsätze berücksichtigt, welche entsprechend angegeben werden. Die Ergebnisse der Analyse werden direkt in den Kunden- bzw. Artikelstamm zurückgeschrieben. 

Kopfrabatte werden hierbei nicht berücksichtigt. 

Beispiel für eine prozentuale Berechnung: 
Sie haben in den letzten 6 Monaten einen Gesamtumsatz von 100.000,-- € gemacht. 

Die Verteilung wird angegeben: 
A-Artikel: 80 % 
B-Artikel: 15% 
C-Artikel: 5% 

Dann wird folgendes gemacht: Im Hintergrund wird eine Umsatzliste generiert und nach Artikelnummern gruppiert. Der Artikel mit dem besten Umsatz steht an 1. Stelle. Dann wird dessen Umsatz von den 100.000,-- € Gesamtumsatz abgezogen und der Artikel wird als A-Artikel gekennzeichnet. Dann kommt der 2. Artikel, dessen Umsatz wird vom verbliebenen Rest-Gesamt wieder abgezogen und als A-Artikel gekennzeichnet. Dies wird solange fortgeführt, bis 80% des Gesamtumsatzes (hier also 80.000,-- €) erreicht sind. Dann wird entsprechend mit den B-Artikeln weitergerechnet. 

Kunden bzw. Artikel, die in keine dieser Kategorien fallen, erhalten das Kennzeichen D. 

Es können nur %-Werte oder absolute Werte angegeben werden; eine Mischung beider Möglichkeiten ist ausgeschlossen. 

Bei der Angabe von absoluten Werten werden Kunden mit einem Umsatz größer der Angabe als A-, B- oder C-Kunden bzw. Artikel eingetragen. 

Beispiel für eine Berechnung nach Betrag: 

Je Kunde/Artikel wird die Summe der Beträge/Erträge aller fakturierten Aufträge und Gutschriften im angegebenen Zeitraum ermittelt. 

Wenn der Umsatz/Ertrag des Artikels/Kunden "A Betrag ab" echt überschreitet, so bekommt der Artikel/Kunde das Kennzeichen A. 

Wenn "A Betrag ab" nicht überschritten wird, aber "B Betrag ab", so bekommt der Artikel/Kunde das Kennzeichen B, etc. 

Beispiel: 
| Artikel | Umsatz | Ertrag |
| --- | --- | --- |
| E | 300.000,-- | 80.000,-- |
| F | 400.000,-- | 101.000,-- |
| G | 70.000,-- | 3.000,-- |
| H | 0 | 0 |

A Betrag ab 100.000,-- 
B Betrag ab 4.000,-- 
C Betrag ab 1.000,-- 

-> Auf Umsatz eingestellt: 
E bekommt A 
F bekommt A 
G bekommt B 
H bekommt D 

-> Auf Ertrag eingestellt: 
E bekommt B 
F bekommt A 
G bekommt C 
H bekommt D 

### Divers- und Beschaffungsartikel 
Eine ABC Analyse kann auch für Divers- und Beschaffungsartikel durchgeführt werden. Diese machen in manchen Branchen einen erheblichen Teil des Umsatzes aus. 

Begründung 
Haben Unternehmen einen hohen Anteil an Diversartikeln - wie es zum Beispiel in der SHK-Branche bei Badmöbeln etc. üblich ist - und haben diese Diversartikel einen hohen Wert, so geht die Aussagekraft der ABC-Analyse gegen Null, weil am Ende nur noch Diversartikel in der Rangfolge berücksichtigt werden. 

Für Beschaffungsware gilt im Grunde genommen das Gleiche. In der Regel will das Unternehmen Lagerware qualifizieren. Beschaffungsware kann etwas zum Erkenntnisgewinn beitragen, muss es aber nicht. 

Ferner gibt es Unternehmen, die Artikel mit geringem Wert in hoher Stückzahl verkaufen. Die quantitative Unterscheidung ist möglich. 

Hinzu kommt, dass ein Artikel - bezogen auf seine Filiale - im Ranking unterschiedlich hoch bewertet werden kann; im einen Lager ist es ein Top-Produkt und im anderen ein Ladenhüter. 

Dafür stehen die Funktionalitäten Lager , Lagergruppe , Keine Diversartikel und Keine Beschaffungsartikel zur Verfügung. D.h. Sie können die ABC-Analyse auf Lager bzw. Lagergruppen einschränken und Divers- bzw. Beschaffungsartikel von der Bewertung ausnehmen.

### Combobox AG / HG 
Hier wird eine Artikelgruppe angegeben. Sie kann alphanumerisch sein und darf max. 8 Zeichen haben. Artikelgruppen werden im Menü Stamm >> Artikelgruppen angelegt. 

Siehe Kapitel Artikelgruppen . 

Diese Combobox ist nur zugänglich, wenn der Radiobutton Artikel aktiviert ist. 

Wenn eine Artikelgruppe angegeben wird, dann werden nur Artikel bewertet, die in der gewählten Artikelgruppe oder Artikelhauptgruppe sind. Bei der Ermittlung des Umsatzes / Ertrags werden nur Auftragspositionen mit Artikeln der vorgegebenen Artikelgruppe oder Artikelhauptgruppe berücksichtigt. 

Das hat den Vorteil, dass innerhalb der Artikelgruppen eigene ABC-Hierarchien möglich sind. Auf diese Weise kann z.B. ein Kleinteil in seiner Artikelgruppe als A-Artikel bewertet und entsprechend disponiert werden. Im Kontext des gesamten Artikelspektrums wäre diese Einstufung nicht möglich und der Artikel würde vom System immer als D-Artikel verwaltet werden. 

Tooltip: 

Wenn gegeben, so werden nur Artikel dieser Artikel(haupt)gruppe upgedated. 
Es werden nur Umsätze / Erträge eingeschränkt auf Artikel dieser Artikel(haupt-)gruppe berücksichtigt. 
Wenn zusätzlich ein Lager / eine Lagergruppe angegeben wurde, so erfolgt ein Update auf dem Lager und nicht 
im Artikelstamm. 

### Combobox Lager 
Um die ABC Analyse auf ein Lager zu beschränken, wählen Sie hier das entsprechende Lager aus. 

Tooltip: 

Wenn gegeben, so erfolgt ein Update auf der Tabelle Lager. 
Die Umsätze / Erträge werden auf das Lager eingeschränkt. 

### Combobox Lagergruppe 
Um die ABC Analyse auf eine Lagergruppe zu beschränken, wählen Sie hier die entsprechende Lagergruppe aus. 

Tooltip: 

Wenn gegeben, so erfolgt ein Update auf der Tabelle Lager für alle Läger der Lagergruppe. 
Die Umsätze / Erträge werden auf das jeweilige Lager beschränkt.
