# Kapitel NV.ERP.Base.Common.cdSystemMM.lngCustomerIDPOM

Hier kann eine Kundennummer eingetragen werden, die im Fall von Materialrücksendung oder -beistellung gezogen werden soll. Bei der Materialrücksendung an einen Lieferanten muss aus logistischen Gründen immer ein Verkaufsauftrag angelegt werden, damit die Kommissionierungs-Prozesse entsprechend gesteuert werden. 

Ist dem Lieferanten keine Kundennummer hinterlegt, so wird der hier eingestellte Dummy-Kunde gezogen. 

Tooltip: 

Standard-Kunde für die Generierung der Beistellung in Betriebsaufträgen
