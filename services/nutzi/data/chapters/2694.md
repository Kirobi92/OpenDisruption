# Planwerte importieren 
Planwerte Import 

Dieser Dialog öffnet sich im Buchen Kore >> Buchen >> Planwerte importieren . Hier haben Sie die Möglichkeit Planwerte aus einer CSV-Datei zu importieren. Dabei muss jedoch ein bestimmtes Format eingehalten werden. Eine Excel-Vorlage kann zur Verfügung gestellt werden. 

Die Radiobuttons Kostenstelle und Kostenträger stehen für den Import von Planwerten für Kostenstelle und Kostenträger zur Verfügung. Dafür gibt es in der Tabelle auf der Registerkarte Planzahlen die geänderten Spalten ‚KST / KTR‘. 

Wenn Planwerte für Kostenstellen und Kostenträger importiert werden, so sind 2 verschiedene Dateien zu verwenden. Der Aufbau für diese Dateien ist gleich. 

In der Dialog-Spalte ‚KST / KTR‘ werden je nach gewähltem Radiobutton die aus der Importdatei eingelesenen Kostenstellen- oder Kostenträgernummern ausgewiesen. 

Wird der Import für den gleichen Zeitraum wiederholt / nochmals durchgeführt, werden nur die gleichlautenden Planwerte von Kostenart bzw. Leistung und Zieltyp gelöscht, das heißt, es wird nicht kumuliert. 

Kostenträger, die ohne Betrag in der Import-Datei stehen, werden auf 0 gesetzt.

### Button Neu 

Dieser Button leert die Felder. 

### Button Import 

Mit diesem Button werden Planwerte in die Kore übernommen. 

Die csv-Datei enthält eine Plausibilitätsprüfung: Die Spalten 'Gesamt Var./Fix', 'Gesamt Var.' and 'Gesamt Fix' müssen rechnerisch korrekt die Summe der Monatswerte enthalten, ansonsten wird dieser Button nicht zugänglich. 

### Button Löschen 

Mit diesem Button werden Planwerte gelöscht. 

Es werden alle Planwerte des selektierten Zeitraums gelöscht. 

Tooltip: 

Planwerte löschen 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Radiobutton Kostenstelle 
Wenn Sie diesen Radiobutton aktivieren, werden Planwerte für Kostenstellen importiert. 

### Radiobutton Kostenträger 
Wenn Sie diesen Radiobutton aktivieren, werden Planwerte für Kostenträger importiert. 

### Combobox Jahr 
Hier wählen Sie das Jahr aus, für welches die Planzahlen importiert werden. 

### Combobox Periode von 
Hier wählen Sie die Periode innerhalb des Buchungsjahres aus, für welche die Planzahlen importiert oder gelöscht werden sollen. 

### Combobox bis 
Hier wählen Sie die Periode innerhalb des Buchungsjahres aus, bis zu welcher die Planzahlen importiert oder gelöscht werden sollen. 

### Feld Pfad / Datei 
Hier geben Sie das Verzeichnis und den Dateinamen der Datei ein. 

### Button Suche 

Über diesen Button wählen Sie die Import-Datei aus. 

### Feld Trennzeichen 
Zeichen, welches die einzelnen Datenfelder trennt. In der Regel: „ ; „ 

### Feld Dezimaltrennz. 
Angabe des Dezimal-Trennzeichens innerhalb eines Datenfeldes. 

### Checkbox Nur Kostenarten 
Wenn Sie diese Checkbox aktivieren, werden nur Kostenarten importiert. 

### Checkbox Nur Leistungen 
Wenn Sie diese Checkbox aktivieren, werden nur Leistungen importiert.
