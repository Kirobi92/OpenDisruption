# Angebotsarten Registerkarte Allgemein 
Angebotsarten Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie allgemeine Einstellungen. 

Wenn die Checkbox inkl. MwSt. aktiviert ist, ist diese Belegart vornehmlich für Endkunden gedacht. 
### Combobox Lager 
Hier wählen Sie das Lager aus, das für diese Belegart genutzt werden soll. 

Das hier ausgewählte Lager überschreibt das Standardlager, das dem Sachbearbeiter zugeordnet ist. 

Tooltip: 

Wählen Sie das Lager aus. 
Diese Lagereinstellung überschreibt das Standardlager, das dem Sachbearbeiter zugeordnet ist. 

### Combobox Versandweg 
Hier kann für die Belegart ein Versandweg vorbelegt werden. Dafür wählen Sie aus dieser Combobox einen Eintrag aus. 

Diese Belegart kann im Vorgang übersteuert werden. 

Tooltip: 

Hier ordnen Sie einen Versandweg zu. 

### Feld Bindefrist 
Die Bindefrist, die Sie hier einstellen, hat Vorrang vor dem Standardwert aus den Parametern Verkauf. Diese Bindefrist kann manuell angepasst werden. 

Die Bindefrist wird in den einzelnen Beleg übernommen und kann dort ebenfalls wieder überschrieben werden, z.B. bei Sondervereinbarung mit dem Kunden. 

Beim Kopieren eines Angebots in ein anderes Angebot oder einen Auftrag bleibt die Bindefrist innerhalb derselben Angebotsart erhalten. Wird eine andere Angebotsart in der Zielkopie verwendet, gelten die dort hinterlegten Bindefristen. 

Tooltip: 

Hier können Sie eine belegspezifische Bindefrist hinterlegen. 
Diese überschreibt die in den Verkaufsparametern 
gesetzte "Standard-Bindefrist". 

### Checkbox Inklusive Mehrwertsteuer 
Diese Option hat zur Folge, dass erst ein Bruttopreis errechnet wird, der anschließend mit der Menge multipliziert wird. Die Preise auf den einzelnen Formularen sind dann Bruttopreise. 

Tooltip: 

Wenn aktiviert, ist die MwSt.in den Positionen der Angebotsart bereits enthalten. Diese Angebotsart ist für Endkunden einzustellen. 

### Checkbox Kein Sonderpreis 
Wenn Sie diesen Parameter aktivieren, wird kein Sonderpreis gezogen. Dies betrifft sowohl die Sonderpreisverwaltung als auch die Sonderpreise in der Konditionsstaffel und dem Artikelstamm. 

D.h. diese Einstellung wird in den Vorgang übernommen, die Checkbox Kein Sonderpreis auf der Registerkarte Kopfdaten wird automatisch aktiviert. 

Tooltip: 

Wenn aktiviert, wird kein Sonderpreis gezogen. Dies betrifft sowohl die Sonderpreisverwaltung als auch die Sonderpreise in der Konditionsstaffel und dem Artikelstamm. 

### Checkbox Kreditlimit Prüfung 
Wenn Sie diesen Parameter aktivieren, wird bei Erstellung des Belegs das Kreditlimit des Kunden geprüft. 

Dies geschieht nur zur Information und hat keine Auswirkungen. 

Tooltip: 

Kreditlimitprüfung 
Die Kreditlimitprüfung im Beleg erfolgt nur bei einfacher Prüfung. 
D.h. wenn die 'erweiterte Prüfung' unter 
"System >> Allgemein >> Parameter >> Wawi" 
nicht aktiv ist. 

### Checkbox Ausstellung 
Wenn aktiviert, wird die Angebotsart in der Ausstellungsübersicht angezeigt. 

Tooltip: 

Wenn aktiviert, wird die Angebotsart in der Ausstellungsübersicht angezeigt.
