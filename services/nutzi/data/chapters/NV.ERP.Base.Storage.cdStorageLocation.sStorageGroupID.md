# Kapitel NV.ERP.Base.Storage.cdStorageLocation.sStorageGroupID

Hier wählen Sie eine Lagergruppe aus. Damit kann ein Lager einer Lagergruppe zugwiesen werden. 

Diese Lagergruppe wiederum kann einen gemeinsamen Bestand bilden. 

Läger können in einer Lagergruppe zusammengefasst werden. Bestandsanzeige und Bedarfsermittlung kann dann über diese Gruppe erfolgen. 

Hinweis: 

Für die Bedarfsermittlung können Sie im Bestellvorschlag mehrere Läger auf einmal auswählen. 

Siehe Kapitel Bestellvorschlag . 

Tooltip: 

Wählen Sie eine Lagergruppe aus.
