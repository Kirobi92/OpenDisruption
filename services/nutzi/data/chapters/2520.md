# Saldenlisten Registerkarte Selektionen 1 
Saldenlisten Registerkarte Selektionen 1 

Auf dieser Registerkarte legen Sie Selektionskriterien an. 

Die Liste wird nach aufsteigenden Kontonummern sortiert, die Sachkonten werden aufsteigend zuerst nach Bilanz, dann G+V-Konten gelistet. 
### Dock Panel Kontentyp 

In diesem Bereich wird Ihnen angezeigt, welche Art von Konten für die Selektion der Saldenlisten zur Verfügung steht. Wenn Sie Sachkonten aktivieren, wählen Sie bitte aus, ob Bilanz und/oder G+V-Konten (siehe Sachkontenstamm) angezeigt werden sollen. 

Wenn Sie den Radiobutton Debitoren oder Kreditoren aktiviert haben, stehen rechts die Combobox Kundengruppe bzw. Lieferantengruppe , das Feld Sammelkonto und die Checkbox Nur Guthaben zur Verfügung. 

Zusätzlich kann bei gesetztem Radiobutton Debitoren oder Kreditoren auch über die Combobox Verband selektiert werden. (Die Selektionskriterien nur und keine sind auch hier standardmäßig enthalten.) 

### Combobox Jahr / Periode 
Hier wählen Sie das Jahr aus. 

### Combobox Periode 
Hier wählen Sie den Zeitraum aus. 

### Checkbox Mit Vorjahreswerten 
Wenn Sie diese Checkbox aktivieren, werden auch Vorjahreswerte selektiert. 

### Feld Konto von 
Hier geben Sie die Kontonummer ein, ab der Sie selektieren möchten. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Über diesen Button können Sie ein Konto suchen. 

Siehe Kapitel Kontoauswahl . 

### Feld Konto bis 
Hier geben Sie die Kontonummer ein, bis zu der Sie selektieren möchten. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Über diesen Button können Sie ein Konto suchen. 

Siehe Kapitel Kontoauswahl . 

### Feld Kostenstelle 
Hier können Sie die Selektion auf eine Kostenstelle eingrenzen. 

Tooltip: 

Angabe der Kostenstellennummer 

### Button Suche 

Mit diesem Button können Sie eine Kostenstelle suchen. 

Siehe Kapitel Kostenstellenauswahl . 

### Checkbox Bilanz 
Durch deaktivieren dieser Checkbox können Sie beim Selektieren der Saldenliste Sachkonten die Auswahl direkt auf die entsprechenden Kontotypen einschränken. 

### Checkbox G + V 
Durch deaktivieren dieser Checkbox können Sie die Selektion der Saldenliste Sachkonten direkt auf die entsprechenden Kontotypen einschränken. 

### Checkbox Mit Zwischenabschluss 
Wenn Sie diese Checkbox aktivieren, werden die Zwischenabschlussbuchungen berücksichtigt. 

### Feld Sammelkonto 
Hier können Sie nach dem Sammelkonto zu Personenkonten selektieren. 

### Checkbox Nur Guthaben 
Wenn Sie diese Checkbox aktivieren, werden nur Konten mit Guthaben angezeigt. 

### Checkbox Mit Nullsalden 
Wenn Sie diese Checkbox aktivieren, werden auch Konten angezeigt, deren Saldo 0 ist. 

### Checkbox Ohne Buchung 
Wenn Sie diese Checkbox aktivieren, werden auch Konten ohne Buchung angezeigt.
