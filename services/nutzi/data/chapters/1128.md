# Rahmenbestellungen für Spezifikationsaufträge 
Rahmenbestellungen 

Falls ein Lieferant, bei dem bestellt wird, mehrere Standorte hat und auch von unterschiedlichen Standorten liefert und die Ware berechnet, kann für diesen Lieferanten ein „Verband“ angelegt werden und die Standorte werden innerhalb des Verbands verknüpft. 

Bei der Anlage der Position in der Rahmenbestellung muss es sich um eine Artikelgruppe handeln, da die genaue Dimension des Artikels der Bestellung zum Zeitpunkt des Globalabschlusses nicht feststeht. 

Innerhalb der Artikelgruppe muss es einen Artikel geben, welcher das Kennzeichen „Ohne Spezifikation“ im Artikelstamm gesetzt hat. 

Wenn nun ein Kunde einen Abruf aus diesem Globalabschluss tätigt, so muss für diesen Kunden ein Rahmenauftrag angelegt werden. 

Innerhalb dieses Rahmenauftrags muss die gleiche Artikelgruppe angelegt werden, wie in der zugehörigen Rahmenbestellung. Dazu steht im Dialog Rahmenauftrag auf der Registerkarte Positionen die weitere Registerkarte Abrufe zur Verfügung. Hier erhalten Sie Informationen zu Abrufen aus diesem Rahmenauftrag. Es werden Informationen wie Artikelnummer, Abrufmenge, Liefertermin etc. angezeigt. 

Erstellen Sie hierfür einen Abruf auf einen Artikel „Ohne Spezifikation“ über die Gesamtmenge, welche der Kunde abnehmen möchte. 

Anschließend markieren Sie den Abruf und klicken in den Dialog Rahmenabrufe auf den Button Rahmenb ., um die zugehörige Rahmenbestellung (im Dialog Rahmenbestellungen , der Verweis dafür steht weiter unten im Text) auswählen zu können. 

Markieren Sie die Rahmenbestellung und klicken auf den Button um die Verbindung herzustellen. 

Die Hintergrundfarbe der Abrufposition macht kenntlich, dass diese Position nur der Verbindung zwischen Rahmenauftrag und Rahmenbestellung dient. 

Teilt der Kunde Ihnen nun den genauen Artikel mit, welchen er benötigt, so legen Sie einen 2. Abruf über den genauen Artikel an und die Menge, welche abgerufen wird. 

Dieser Abruf muss nun in eine Auftragsposition übernommen werden, damit die Verbindung von der Rahmenbestellung bis zum endgültigen Auftrag vorhanden ist. 

Das Verhalten im Feld Menge für Artikel „ohne Spezifikation“ ändert sich. Sobald der Abruf mit dem Verkaufsauftrag verknüpft ist, wird die Menge des Abrufs mit der Artikelposition „ohne Spezifikation“ um die 2. Abrufmenge reduziert.

## Steuerelemente des Formulars 

### Bezeichnungsfeld Rahmenbestellungen 
Tooltip: 

In diesem Dialog wählen Sie für einen Spezifikations-Rahmenabruf die zugehörige Rahmenbestellung aus. 
Für mehr Informationen drücken Sie F1.

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter .
