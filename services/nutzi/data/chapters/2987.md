# Intrastat - Warennummer Registerkarte Tabelle 
Warennummer Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Warennummern angezeigt. 
### Tabelle 

### Tabellenspalte Warennummer 
Hier wird die Warennummer angegeben. 

Tooltip: 

Geben Sie die Warennummer ein. 

### Tabellenspalte Bezeichung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Geben Sie hier eine Bezeichnung für die Warennummer ein. 

### Tabellenspalte Bes. Maßeinheit 
Hier wird die besondere Maßeinheit für Intrastat angegeben. Diese kann an der Warennummer hinterlegt werden. 

Die Menge einer Intrastatposition wird immer in der Maßeinheit ihrer Warennummer angegeben. Wurde keine Maßeinheit hinterlegt, so wird stattdessen die Masse der Intrastatposition in kg angegeben. 

Der Umrechnungsfaktor zur Umrechnung der besonderen Maßeinheit einer Warennummer in die Lagereinheit eines Artikels muss für jeden Artikel der Warennummer gepflegt werden.

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Dieses Feld zeigt die Anzahl der tatsächlich eingelesenen Datensätze.
