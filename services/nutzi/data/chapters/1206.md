# Parameter Scanner Registerkarte DataFile 
Parameter Scanner Registerkarte DataFile 

Auf dieser Registerkarte nehmen Sie Einstellungen für den Import von Daten aus dem Offline Scanner vor. 

Das Feld X-Mapping Setting ID steht zur Verfügung, um ein Setting auszuwählen. Mit dem -Button kann das Setting editiert und bearbeitet werden. 

Dann ist hier noch die Codierung anzugeben. 

Jetzt wählen Sie noch die Übertragungsart aus. Es kann zwischen Server und lokal gewählt werden. Allerdings ist Server die empfohlene Einstellung, so dass die Daten zentral und von mehreren Anwendern (Scanner) aktualisiert werden. 

Als Dateiname wird hier der absolute Pfad der Datei angegeben. 
### Combobox X-Mapping ID 
Hier wird der Identifizierer eines Mappings angegeben, das mit dem X-Mapper erstellt wurde. 

Tooltip: 

Identifizierer eines Mappings, das mit dem X-Mapper erstellt wurde. 

### Button Öffnen 

Mit diesem Button kann das Setting editiert und bearbeitet werden. 

### Combobox Codierung 
Hier wird die Zeichencodierung angegeben. 

Siehe Kapitel Formate für Exportdateien - Encodings . 

Tooltip: 

Zeichenkodierung 

### Combobox Übertragungsart 
Für den Import aus CSV- oder XML-Dokumenten (für den Import über den Offline Scanner) wählen Sie die Übertragungsart aus. Es kann zwischen Server und lokal gewählt werden. Allerdings ist Server die empfohlene Einstellung, so dass die Daten zentral und von mehreren Anwendern (Scanner) aktualisiert werden. 

### Feld Dateiname 
Hier wird der Dateiname angegeben. 

Für den Import aus CSV- oder XML-Dokumenten für den Import über den Offline-Scanner geben Sie den absoluten Pfad der Datei an.
