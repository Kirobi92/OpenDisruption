# Offene Posten Debitoren Registerkarte Selektionen 2

Finanzbuchhaltung / Offene Posten Debitoren 

# Offene Posten Debitoren Registerkarte Selektionen 2 
OP Debitor Registerkarte Selektionen 2 

Diese Registerkarte bietet weitere Selektionsmöglichkeiten für die Suche nach offenen Posten. 

Zum Thema Mahnsperre und Einzugssperre lesen Sie bitte das Kapitel Einschränken von OP-Sperren über das Datum . 
### Feld Betrag von 
Zur Selektion kann hier ein Betrag ausgewählt werden. 

Tooltip: 

Selektion über 'Betrag von' 

### Feld bis 
Zur Selektion über einen "Betrags-Bereich" kann hier ein Betrag ausgewählt werden. 

Tooltip: 

Selektion über 'Betrag bis' 

### Checkbox S/H berücks. 
Wenn Sie diese Checkbox nicht aktivieren, wird die Spalte ‚Betrag‘ im OP mit positivem bzw. negativem Wert eingetragen. Das Soll-/Haben-Kennzeichen bleibt unberücksichtigt. 

Wenn Sie diese Checkbox aktivieren, wird bei einem positiven Wert nach einer Rechnung gesucht und bei einem negativen Wert nach der Zahlung oder der Gutschrift. 

Tooltip: 

Soll/Haben-Kennzeichen berücksichtigen 

### Combobox Datum 
Hier wählen Sie aus, was für ein Datum zur Selektion genutzt werden soll. Zur Verfügung stehen: 

- Belegdatum - Buchungsdatum - Einzugssperre bis - Löschdatum - Mahnsperre bis - Nettofällig - Valutadatum 

### Feld von 
Wenn Sie in der Combobox Datum einen Eintrag ausgewählt haben, werden die Felder von und bis zugänglich. 

Hier geben Sie jetzt ein Datum für die Selektion ein. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld bis 
Wenn Sie in der Combobox Datum einen Eintrag ausgewählt haben, werden die Felder von und bis zugänglich. 

Hier geben Sie jetzt ein Bis-Datum für die Selektion über einen Datumsbereich ein. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Combobox OP-Kennz. 
Zur Selektion kann hier ein OP-Kennzeichen ausgewählt werden. 

Tooltip: 

Angabe des OP-Kennzeichens 

### Combobox Zahlung 
Zur Selektion über eine Zahlungsbedingung wählen Sie hier einen Eintrag aus. 

Tooltip: 

Zahlungsbedingung 

### Feld Projekt 
Zur Selektion über eine Projektnummer, wählen Sie hier einen Eintrag aus. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Über diesen Button können Sie ein Projekt suchen. 

Siehe Kapitel Projektauswahl . 

### Combobox Mahngruppe 
Zur Selektion über eine Mahngruppe, wählen Sie hier einen Eintrag aus. 

Tooltip: 

Angabe der Mahngruppe 

### Feld Mahnstufe 
Dieses Feld dient der Selektion nach Mahnstufe. 

Tooltip: 

Angabe der aktuellen Mahnstufe 

### Combobox Verband 
Über diese Combobox können Sie die OPs nach Verbänden selektieren. Die Selektionskriterien nur und keine sind standardmäßig enthalten. Wird im Kundenstamm im Feld Verband eine Kontonummer hinterlegt, erscheint diese ebenfalls als Verbandsnummer in der Combobox zur Auswahl. 

### Combobox Zentralzahler 
Aus dieser Combobox können Sie einen Zentralzahler zur Selektion auswählen. 

### Checkbox Nachnahme 
Wenn Sie diese Checkbox aktivieren, werden nur offene Posten eingelesen, die auf Nachnahme gesetzt sind. 

Tooltip: 

Zahlung erfolgt per Nachnahme
