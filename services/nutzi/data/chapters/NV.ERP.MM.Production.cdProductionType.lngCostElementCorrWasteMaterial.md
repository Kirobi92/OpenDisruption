# Kapitel NV.ERP.MM.Production.cdProductionType.lngCostElementCorrWasteMaterial

Hier geben Sie die Kostenart für Schrott ein. 

Damit kann Ausschuss-Material (Schrott) korrekt erfasst und verbucht werden. So kann die Berechnung der IST-Kosten bzw. der tatsächlichen Herstellkosten korrekt ausgeführt werden. 

Tooltip: 

Kostenart für den Materialanteil für den erstellten Schrott, mit dem der Kostenträger entlastet werden soll.
