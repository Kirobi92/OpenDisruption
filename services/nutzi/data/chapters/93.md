# Meldungen 
Meldungen 

Wenn es bei der Verarbeitung von Vorgängen in eNVenta zu Problemen kommt, werden Sie mit einer sogenannten PostMessage (Dialog Meldungen ) darauf hingewiesen. Meldungen können als Information, Warnung oder Fehler ausgegeben werden. Dies kann in Einzelfällen definiert werden (z.B. im Zusammenhang mit einer Abschlussprüfung). 

Es ist möglich, dass zu einem Eintrag in diesem Dialog noch weitere Informationen hinterlegt sind. 

Die Registerkarten zeigen mehr Informationen zu Fehler, Stack Trace und Details. So können Sie besser nachvollziehen, wo im Code die Meldung erzeugt wurde. 

Mit dem Button kann die Meldung ausgedruckt werden. Bei einer großen Anzahl von Meldungen kann es hilfreich sein, die einzelnen Probleme bzw. Fehler zu bearbeiten ohne jedes Mal die Meldung wieder aufrufen zu müssen. 

Schließen Sie den Dialog Meldungen mit dem Button . 

eNVenta ERP öffnet sich. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Meldungen 
Tooltip: 

Für neue Termine in Ihrer Wiedervorlage werden Sie beim Start von eNVenta ERP und bevor sich der eNVenta ERP-Dialog öffnet, mit einer Meldung darauf hingewiesen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Menü Erweitert 
Über dieses Menü wird ein zusätzliches Multiline-Feld eingeblendet, welches den Titeltext der Nachricht enthält. Das ist besonders bei langen Texten direkt im Titel praktisch, da Sie hier nicht in der Tabelle nach rechts scrollen müssen. Außerdem können Sie daraus auch Texte kopieren - das geht in der Tabelle nicht. 

Dieselbe Funktion wird aufgerufen, wenn Sie einen Doppelklick auf eine Zeile in der Tabelle machen.
