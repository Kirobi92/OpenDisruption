# Retoure Registerkarte Bezug 
Registerkarte Bezug 

Auf dieser Registerkarte lässt sich eine Verbindung zu einer Bestellung herstellen, d.h. die Ursprungsbestellung wird angezeigt. Dazu geben Sie entweder eine Bestellnummer ein oder suchen diese über den Button (öffnet den Dialog Auswahl Bestellung ). Falls bereits eine Verbindung zu einer Bestellung besteht (aufgrund auftragsbezogener Bestellung), wird automatisch die entsprechende Bestell- und Lieferantennummer angezeigt. 

Wenn hier ein Lieferant eingetragen ist, wird bei Erstellung einer Belastung oder einer Anfrage (über die gleichnamigen Buttons im Kopf des Dialogs) dieser Lieferant voreingestellt. 
### Feld Bestellung 
Hier wird die Nummer der Bestellung angegeben. 

Tooltip: 

Hier wird die Bestellnummer definiert. 

### Button Verzweigen 

Dieser Button verzweigt in die zugrundeliegende Bestellung. 

Tooltip: 

Bestellung aufrufen 

### Button Suche 

Mit diesem Button können Sie eine Bestellung suchen. Es öffnet sich der Dialog Auswahl Bestellung . 

Siehe Kapitel Auswahl Bestellung . 

Tooltip: 

Bestellung auswählen 

### Feld Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Button Hinzufügen 

Wenn Sie diesen Button drücken, wird der Lieferant in die Registerkarte Beteiligte eingetragen. 

Tooltip: 

Hinzufügen zu Beteiligten 

### Feld Lieferdatum 
Hier wird der Liefertermin angegeben. 

Tooltip: 

Angabe des Liefertermins
