# Wochentage 
Wochentage 

Der Button Wochentage im Kopf des Dialogs Kalender Schicht öffnet den Dialog Wochentage . Hier kann für einen bestimmten Wochentag die Arbeitszeit bestimmt werden, falls dieser von den Standardzeiten, die in der Schicht selber festgelegt wurden, abweicht. Z.B. Samstag und Sonntag. 

Dabei muss der Wochentag festgelegt werden: 

- 1 = Montag - 2 = Dienstag - 3 = Mittwoch - 4 = Donnerstag - 5 = Freitag - & = Samstag - 7 = Sonntag 
Wenn für einen Kalender der abweichende Wochentag nicht angegeben ist, so kommt, falls vorhanden, der entsprechende Eintrag aus Kalender 0 zur Anwendung.

### Tabellenspalte Tag 
Die Bezeichnung für diesen Tag wird automatisch eingelesen. 

### Tabellenspalte Zeit von (hh:mm) 
Um die Dauer der Arbeitszeit an diesem Wochentag festzulegen, kann durch Angabe in dieser Spalte der Beginn der Arbeitszeit festgelegt werden. Die Angabe ersetzt den normalen Kalender für diesen Tag! 

Um z.B. festzulegen, dass am Sonntag nicht gearbeitet wird, empfiehlt es sich im Basiskalender für Sonntag eine Arbeitszeit 00:00 Uhr - 00:00 Uhr anzugeben. 

Sonderfall: 

Eine Schicht kann sich auch in den nächsten Tag hineinziehen. In diesem Fall ist die Zeit Bis kleiner als die Zeit Von . 

### Tabellenspalte Zeit bis (hh:mm) 
Um die Dauer der Arbeitszeit an diesem Wochentag festzulegen, kann durch Angabe in dieser Spalte das Ende der Arbeitszeit festgelegt werden. Die Angabe ersetzt den normalen Kalender für diesen Tag! 

Um z.B. festzulegen, dass am Sonntag nicht gearbeitet wird, empfiehlt es sich im Basiskalender für Sonntag eine Arbeitszeit 00:00 Uhr - 00:00 Uhr anzugeben. 

Sonderfall: 

Eine Schicht kann sich auch in den nächsten Tag hineinziehen. In diesem Fall ist die Zeit Bis kleiner als die Zeit Von . 

Tooltip: 

Zeigt die Zeit-bis an.
