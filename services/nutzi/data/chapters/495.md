# Archivsuche Registerkarte Ergebnis 
Archivsuche Registerkarte Ergebnis 

Zum Öffnen wird immer das Standard-Programm für den jeweiligen Datei-Typ herangezogen, der auf dem Client hinterlegt ist. Wer also zum Anzeigen von Tiff-Bildern das Programm „Irfan-View“ benutzt, bekommt das Bild mit „Irfan-View“ angezeigt. Wer den „Standard-Windows-Viewer“ benutzt, bekommt das Bild mit diesem Programm geöffnet usw. 

Es sind alle gängigen Dateitypen erlaubt; also auch Word, Excel, Tiff, Gif, JPG, PDF usw. 

Speichern von Word-Dokumenten: 

Die Word-Dokumente, die aus dem System erstellt werden, werden auch archiviert. Daher öffnet sich, nachdem das Dokument aufgerufen ist, sofort ein „Speichern unter“-Dialog. Hier bitte das Dokument zwischenspeichern, damit es anschließend automatisch in das jeweilige Archiv importiert werden kann. 

Am Dokument kann nichts mehr geändert werden! Das erste Dokument wird archiviert. 
### Tabelle 

In dieser Tabelle sind die ersten beiden Spalten fest vorgegeben. Alle weiteren Spalten können im Dialog Logische Archive konfiguriert werden. Dazu wählen Sie im Dialog Logische Archive das verwendete Archiv aus. Dann tragen Sie bei den Schlagworten (auf der Registerkarte Schlagworte ) in der Spalte 'Reihenfolge' eine Ziffer ein. Damit legen Sie fest, welche Schlagworte und an welcher Stelle diese Schlagworte unter dieser Tabelle angezeigt werden sollen. 
### Tabellenspalte Phys. Archiv 
Diese Spalte steht immer in dieser Tabelle zur Verfügung. Dies ist vom System so festgelegt.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
