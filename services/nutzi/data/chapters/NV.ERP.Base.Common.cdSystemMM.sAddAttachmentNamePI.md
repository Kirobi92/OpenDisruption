# Kapitel NV.ERP.Base.Common.cdSystemMM.sAddAttachmentNamePI

Hier geben Sie die Formel zur Ermittlung des Dateinamens ein bei der Generierung einer E-Mail für den Proformarechnungsdruck. Es wird immer fix der Wert "Proformarechnung" verwendet, gefolgt von der Formel als Dateiname.
