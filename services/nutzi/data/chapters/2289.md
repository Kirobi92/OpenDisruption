# Zeichen Codierung 
Zeichencodierung 

Dieses Menü finden Sie in der Katalogverwaltung >> Optionen >> Suchoptionen >> Volltextsuche >> Zeichen Codierung . Hier werden die Buchstaben und Zeichen mit Nummern belegt, sie werden codiert. 

Für die Interpretation von Sonderzeichen im Webshop oder BMECat müssen diese im ASCII-Code (http://de.wikipedia.org/wiki/ASCII-Code) angezeigt werden. Der ASCII-Code ist eine standardisierte Norm, welche international gilt. Für jedes ASCII-Zeichen kann die jeweilige Zeichen-Art ausgewählt werden. Diese wird dann von der entsprechenden Dritt-Software interpretiert.
