# Disposition Registerkarte Disposition 
Disposition Registerkarte Disposition 

Anhand des ausgewählten Artikels und Lieferanten werden die Felder dieser Registerkarte mit den dazugehörenden Daten aus Rabattverwaltung, Artikel- und Lieferantenstamm gefüllt und können entsprechend modifiziert werden. Die vorgegebenen Werte gehen mit dem Button Dispo in den Dispo-Pool ein und werden von dort in die Bestellung übergeben. 
### Feld Menge bestellt 
Hier geben Sie die zu disponierende Menge des Artikels ein. 

Erst nach Eingabe der Menge findet die automatische Preisermittlung statt! 

Tooltip: 

Geben Sie für die automatische Preisermittlung die zu disponierende Menge ein. 

### Combobox Einheit 
Hier kann die Einheit ausgewählt werden. 

Tooltip: 

Einkaufeinheit 

### Combobox Einkaufsart 
Hier wählen Sie die Einkaufart aus. 

Dieser Eintrag ist wichtig für die weitere Abwicklung der Bestellung und die Auswahl des richtigen Bestellformulars. Nach der Eingabe der Lieferantennummer wird die Einkaufart des Lieferanten voreingestellt. 

Tooltip: 

Hier wählen Sie die Einkaufsart aus. Sie ist wichtig für die weitere Abwicklung der Bestellung und die Auswahl des richtigen Bestellformulars. 

### Feld Lieferdatum 
Hier wird der Liefertermin angegeben. 

Tooltip: 

Geben Sie den Liefertermin ein. 

### Feld Wunschdatum 
Hier geben Sie den Wunschtermin für die Lieferung ein. 

Tooltip: 

Geben Sie den Wunschtermin ein. 

### Feld KW 
Hier wird die Kalenderwoche angegeben. 

Bei Eingabe einer Kalenderwoche wird der Termin automatisch auf den ersten Tag dieser Kalenderwoche gesetzt. 

Bei Angabe eines Lieferdatums wird die Kalenderwoche hier automatisch angegeben. 

Tooltip: 

Hier kann die Kalenderwoche angegeben werden. 

### Feld Preis 
Hier geben Sie den Preis an, mit dem disponiert werden soll. 

Tooltip: 

Geben Sie den Preis ein, mit dem disponiert werden soll. 

### Combobox Währung 
Hier wird die Währung angezeigt, kann aber überschrieben werden. 

Tooltip: 

Angabe der Währung 

### Feld EK pro 
Hier wird der Einkaufspreis je Anzahl Lagereinheiten angeben. Der Einkaufspreis kann sich auf 10, 100, 1000 oder eine andere Anzahl Einheiten des Artikels beziehen. Standardmäßig wird hier eine 1 eingetragen. 

Tooltip: 

Geben Sie die Preiseinheit ein. 

### Feld Faktor EK 
Hier geben Sie einen Faktor ein. 

Tooltip: 

Geben Sie einen Faktor ein. 

### Feld Nebenkosten 
Hier werden die Nebenkosten aus dem Artikelstamm voreingestellt; diese können überschrieben werden. 

Tooltip: 

Nebenkosten werden angezeigt und können überschrieben werden. 

### Combobox Lager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager 

### Feld WBZ 
Die Wiederbeschaffungszeit wird angezeigt. Mehr zum Thema Wiederbeschaffungszeit (WBZ) und deren Berechnung erfahren Sie im Kapitel Wiederbeschaffungszeit . 

Siehe Kapitel Wiederbeschaffungszeit . 

Tooltip: 

Zeigt die Wiederbeschaffungszeit. 

### Feld WBZ StdAbw. 
Hier wird die Standardabeichung der WBZ angezeigt, d.h. die Zeit um welche die Wiederbeschaffungszeit durchschnittlich abweicht. 

Tooltip: 

Zeigt die WBZ-Standardabweichung an, d.h. die Zeit, um welche die Wiederbeschaffungszeit durchschnittlich abweicht. 

### Feld Kostenstelle 
Hier wird die Kostenstellennummer angegeben. 

Tooltip: 

Die Kostenstellennummer kann angegeben werden. 

### Feld Kostenträger 
Hier wird die Kostenträgernummer angegeben. 

Tooltip: 

Die Kostenträgernummer kann eingegeben werden. 

### Tabelle Rabatte 

Bis zu 6 Rabatte/Zuschläge können zu einer Disposition angegeben werden. Aufgrund der Konditionen und der Rabattgruppe schlägt das Programm bis zu 6 Rabatte/Zuschläge vor. Der Konditionstext kann angegeben werden und wird in die Bestellung übernommen. Nach Eingabe der Bestellmenge werden die Rabatte automatisch berechnet und angezeigt. 

Tooltip: 

Für die Disposition können hier Rabatte eingegeben werden. 
### Tabellenspalte Rabatttext 
Hier wird ein Text zum Grundrabatt angegeben. 

Tooltip: 

Angabe des Rabatttexts 

### Tabellenspalte Rabatt % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Rabattbetrag 
Hier wird der Rabattbetrag angegeben. 

Tooltip: 

Angabe des Rabattbetrags 

### Button Suche 

Dieser Button öffnet den Dialog Teuerungszuschlag zur Selektion eines MTZs. 

Siehe Kapitel Teuerungszuschlag . 

### Button Satz hinzufügen 

Dieser Button fügt eine neue Rabattzeile in die Tabelle ein. Diese Zeile steht dann zur Bearbeitung zur Verfügung. 

Tooltip: 

Neue Rabattzeile hinzufügen 

### Button Löschen 

Mit diesem Button löschen Sie einen Rabatt aus der Tabelle. 

### Feld Bestellnummer 
Hier wird die Artikel-Bestellnummer beim (Haupt-)Lieferanten angegeben. 

Tooltip: 

Tragen Sie die Artikel-Bestellnummer beim (Haupt-)Lieferanten ein. 

### Feld Bestelltext 
Hier können Sie einen individuellen Text zur Bestellung eintragen. 

Tooltip: 

Geben Sie einen internen Bestelltext ein. 

### Button Html-Editor 

Dieser Button öffnet den Html Editor zur Bearbeitung von Html-Texten. 

Siehe Kapitel Html Editor . 

Tooltip: 

Html-Editor zur Bearbeitung von Html-Texten. 

### Feld Poswert FW 
Hier wird der Preis angegeben. 

Tooltip: 

Preis 

### Feld Info 
In dieses Feld können Sie artikelbezogene Informationen eingeben. Diese werden in der Bestellung im Tabellenfeld Info zu dieser Artikelposition angezeigt. 

Tooltip: 

Geben Sie eine artikelbezogene Information ein. 

### Feld Rabatt-Info 
Der Text des Infofelds aus der Rabattverwaltung wird angezeigt. 

Tooltip: 

Info-Text aus Rabattverwaltung wird angezeigt.

Hier wird die Vertragsnummer angegeben.

Angabe der Vertragsnummer

Hier wird der Liefertermin angegeben. 

Tooltip: 

Angabe des Liefertermins 

### Feld Menge

Tooltip: 

Angabe der Menge
