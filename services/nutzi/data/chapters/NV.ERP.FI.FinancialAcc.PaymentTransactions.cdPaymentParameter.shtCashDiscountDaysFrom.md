# Kapitel NV.ERP.FI.FinancialAcc.PaymentTransactions.cdPaymentParameter.shtCashDiscountDaysFrom

Über dieses Feld können Sie die Berücksichtigung der Skonti in Tagen im Zahlungsverkehr festlegen. 

Tooltip: 

Ausgehend vom Tagesdatum erhöhen/verringern die hier erfassten Tage das Datum im Vorschlagsdialog.
