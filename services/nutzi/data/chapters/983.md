# Parameter Preisfindungscache 
Parameter Preisfindungscache 

Unter System >> Verkauf >> Parameter Preisfindungscache steht der gleichnamige Dialog zur Verwaltung der Preisgruppen für den Cache der Preisfindung zur Verfügung. 

Beim Einlesen der Daten werden die Daten gebündelt abgefragt und in den Cache geladen. 

Für den jeweiligen Vorgang wird hinterlegt, welche Preisgruppen-Daten in den Cache geladen werden. Hierbei werden die entsprechenden Preise einmalig in den Arbeitsspeicher geladen, damit diese während der Konditionierung nicht aus der Datenbank geladen werden müssen. Die Auswahl der Vorgänge ist fix hinterlegt und kann nur durch entsprechende Programmierung angepasst werden. 

Die größere Steigerung der Performance für den Massenexport Datanorm bzw. BMECat-Export erfolgt mit der Aktivierung des Preisfindungscache. Für die einzelne Auftragsposition ist die Ermittlung der Preise nur in Relation schneller, die Effekte wirken erst beim Export vieler Daten im bemerkbaren Bereich. 

Achtung: Jeder Cache benötigt einen entsprechenden Arbeitsspeicher auf dem Broker und sollte daher auch nicht übermäßig genutzt werden. Erfahrungsgemäß genügen ca. 2-4 GB zusätzlicher Arbeitsspeicher. Bei einer mittleren Datenbank sollte der Verkaufsparameter „Preisfindungscache“ aktiviert sein.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie den Vorgang zur Tabelle Einschränkungen hinzu und können die Preisgruppen in der Spalte ‚Werte‘ eingeben. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Preisgruppen-Analyse 

Durch Markierung eines Vorgangs und anschließenden Klick auf diesen Button öffnet sich der Analyse-Dialog. 

Siehe Kapitel Preisgruppen-Analyse .
