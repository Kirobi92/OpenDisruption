# Stückliste produzieren 
Stückliste produzieren 

Dieser Dialog öffnet sich aus dem Dialog Stückliste über den Button Produzieren , falls das Modul Produktion nicht lizenziert ist. 

Die Produktion einer Stückliste ist nur für Produktionsstücklisten möglich und wird über den Button Produzieren ausgelöst. 

Gehen Sie bitte folgendermaßen vor: 
Geben Sie zunächst die Artikelnummer, die zu produzierende Variante und die zu produzierende Menge (Anzahl) ein. Legen Sie auch das Lager fest, für das die Buchungen ausgeführt werden sollen. 

Im Dialog Bestand können Sie jederzeit die einzelnen Buchungen eines Artikels nachvollziehen. 

Siehe Kapitel Bestandsanzeige . 

Wenn Sie mit dem Modul Produktion arbeiten, erhalten Sie die Meldung, dass ein Produktionsauftrag mit Nr. xxx angelegt wurde. Diesen Produktionsauftrag finden Sie in der Produktion im Menü Produktionsauftrag . 

Die Verfügbarkeit wird hier nicht geprüft! D.h. wenn kein ausreichender Lagerbestand vorhanden ist, führt die Buchung bei den entsprechenden Komponenten zu einem negativen Lagerbestand. 

Ist das Modul Produktion dagegen lizenziert, öffnet sich der Dialog Produktionsauftrag erstellen . 

Siehe Kapitel Produktionsauftrag erstellen . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Stückliste produzieren 
Tooltip: 

In diesem Dialog starten Sie die Produktion der Stückliste. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Combobox Variante 
Hier wählen Sie die gewünschte Variante der Produktionsstückliste aus. 

### Feld Menge 
Hier geben Sie die Menge ein, die produziert werden soll. 

Zur Buchung einer Zerlegung geben Sie einfach eine negative Menge ein. 

### Combobox Lager 
Hier wählen Sie das Lager aus, für das die Buchungen ausgeführt werden sollen. 

### Checkbox Nur 1. Ebene 
Wenn Sie diese Checkbox aktivieren, wird nur die 1. Ebene der Stückliste produziert. Alle darunter "hängenden" Stücklisten werden ignoriert.
