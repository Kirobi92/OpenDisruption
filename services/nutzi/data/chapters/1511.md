# Parameter Versand / Logistik Registerkarte Gefahrgut 
Parameter VS / LO Registerkarte Gefahrgut 

Auf dieser Registerkarte nehmen Sie Einstellungen zum Gefahrgut vor. 
### Feld Flammgrenze 
Für die Gefahrguttexte gibt es den Parameter Flammgrenze . Hier kann die Gradzahl eingegeben werden, ab der die Waren entflammbar sind. Ab da muss zusätzlich zum Gefahrgutbild das Flammbild angedruckt werden. 

Tooltip: 

Hier kann die Gradzahl eingegeben werden, ab der die Waren entflammbar sind. Ab da muss zusätzlich zum Gefahrgutbild das Flammbild angedruckt werden. 

### Feld Flammbild 
Hier weisen Sie eine Bilddatei für die Flammgrenze zu. 

Tooltip: 

Zuweisung der Bilddatei für die Flammgrenze.
