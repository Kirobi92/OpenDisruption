# Konfiguration der Webshop Benutzer Registerkarte Allgemein 
Benutzerverwaltung Registerkarte Allgemein 

Auf dieser Registerkarte nehmen Sie die Einstellungen zum Kunden vor. Das Feld Passwort wird aus dem Kundenstamm vorbelegt. Aus den Comboboxen Preislevel , Lagerlevel und Sprache wählen Sie die Einträge aus, die dem Benutzer zugeordnet werden sollen. 

Im Bereich Opt-In Registrierung wählen Sie den entsprechenden Status aus. Wenn dieser bestätigt ist, ist das Feld Reg ID leer. 
### Feld Passwort 
Dieses Feld wird mit dem Passwort aus dem Kundenstamm vorbelegt. 

### Combobox Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Combobox Preislevel 
Wählen Sie aus der Combobox aus, welchen Preis der Benutzer angezeigt bekommen soll. Zur Verfügung stehen: 

| Kein Preis | im Webshop werden keine Preise angezeigt |
| --- | --- |
| Standardpreis | Standardpreise werden angezeigt |
| Kundenspezifisch | auf Kundenbasis wird eine Preiskalkulation durchgeführt |

### Combobox Lagerlevel 
Wählen Sie aus dieser Combobox, wie der Bestand für den Benutzer im Webshop angezeigt werden soll. Zur Verfügung stehen: 

| nichts anzeigen | keine Anzeige, ob Bestand vorhanden oder verfügbar ist |
| --- | --- |
| ist Ladeartikel | Anzeige, dass es sich um einen Lagerartikel handelt |
| ist verfügbar | Anzeige, ob die Menge verfügbar ist |
| Bestand | Anzeige des genauen Lagerbestands |

### Checkbox OP anzeigen 
Wenn Sie diese Checkbox aktivieren, wird der OP im Webshop angezeigt. 

D.h. ein Warenkorb wird bestellt, in eNVenta übernommen und ein Auftrag generiert. Der Auftrag wird fakturiert und in die Finanzbuchhaltung ( OP-Übernahme ) übernommen. Der Auftrag wird gebucht. 

Wenn Sie sich anschließend wieder neu im Webshop anmelden, wird der Dialog Offene Posten angezeigt und weist die entsprechende Position als OP aus. 

### Combobox Zahlungsprofil 
Hier wählen Sie das Zahlungsprofil aus. Diese Zahlungsprofile (Zahlungsarten) werden im Dialog Zahlungsarten definiert. 

Siehe Kapitel Zahlungsarten im Webshop . 

### Combobox Versandprofil 
Hier wählen Sie ein Versandprofil auswählen. Diese Profile werden im Dialog Versandprofile definiert. 

Siehe Kapitel Versandprofile im Webshop . 

### Combobox Status 
Hier wählen Sie den entsprechenden Status aus. 

### Checkbox Zugang gesperrt 
Wenn Sie diese Checkbox aktivieren, wird für einen Benutzer der Zugang zum Webshop gesperrt. 

### Feld Gesperrt 
Hier steht das Datum, an dem der Benutzer gesperrt wurde. 

### Feld Gesperrt durch 
Hier steht der Sachbearbeiter, der den Benutzer gesperrt hat. 

### Checkbox Hauptbenutzer 
Wenn Sie diese Checkbox aktivieren, kann der User, der dieses Recht hat, anderen Benutzern seines Unternehmens Webshop-Rechte setzen. Dieser Masterbenutzer sieht dann unter ‚Benutzerprofil‘ die Daten der anderen Benutzer seines Unternehmens. 

### Checkbox Gast 
Wenn diese Checkbox aktiviert ist, erscheint beim Login des Gastbenutzers die Meldung „Gastzugang“, um dem Benutzer zu verdeutlichen, dass er Gastkunde ist. 

Ein Gastbenutzer kann nicht bestellen. Bei Klick auf Bestellen oder Anfragen erscheint der Login-Dialog, damit der Benutzer sich mit einem anderen Login anmelden kann, welcher dann den Warenkorb bestellen darf.
