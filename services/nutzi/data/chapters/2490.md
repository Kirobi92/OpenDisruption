# Jahresabschluss 
Jahresabschluss 

Im Buchen >> Abschluss >> Abschluss finden Sie den Dialog Jahresabschluss . 

eNVenta ERP lässt ein Arbeiten in mehreren Geschäftsjahren (rückwirkend wie auch vorwärts) generell zu. Ein Zwang, ein Geschäftsjahr abzuschließen, besteht nicht. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Jahresabschluss 
Tooltip: 

In diesem Dialog wählen Sie das Jahr aus, das Sie abschließen wollen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Jahr öffnen 

Mit diesem Button wird ein neues Geschäftsjahr eröffnet und alle Stammdaten werden in das neue Geschäftsjahr kopiert. 

Nach dem Eröffnen des neuen Geschäftsjahrs muss das Programm beendet werden. Wenn Sie sich anschließend wieder anmelden, sind die Umstellungen erfolgt. 

Beim Eröffnen eines neuen Buchungsjahres wird in den eNVenta ERP -Parametern das aktuelle Jahr automatisch auf das neu eröffnete Buchungsjahr gesetzt. 

### Button Abschluss 

Über diesen Button wird das Jahr endgültig abgeschlossen und steht für Buchungsvorgänge nicht mehr zur Verfügung. 

Dabei wird immer zuerst das älteste noch offene Jahr abgeschlossen, unabhängig von dem Jahr, das in der Combobox ausgewählt worden ist. 

Der Ausdruck der Auswertungen für abgeschlossene Jahre ist weiterhin möglich. 

Ein Zwang, das Geschäftsjahr abzuschließen, besteht nicht. 

### Button Vortrag 

Über diesen Button wird der Saldenvortrag der Bilanzkonten (Aktivierung EB-Buchung im Sachkontenstamm Registerkarte Abschluss erforderlich) im nächsten Geschäftsjahr generiert. Der Vorgang kann beliebig wiederholt werden (z.B. nach Abschlussbuchungen). 

### Combobox Jahr 
Hier wählen Sie das Jahr aus, das abgeschlossen werden soll.
