# Datenexport 
Datenexport 

Dieser Dialog steht zum Export von Stammdaten (z.B. aus dem Kundenstamm) zur Verfügung. So ist es möglich, dass andere Programme diese Daten verarbeiten können. 

Der Dialog kann aus verschiedenen Vorgängen aufgerufen werden oder aus modalen Dialogen über den Button (z.B. im Dialog Anlagenklassen >> CSV-Export). 

Der aufrufende Dialog steuert, welche Formate und eNVenta -Felder (auf der Registerkarte Format , Hierarchie-Baum im linken Bereich) zur Verfügung gestellt werden. 

Der Datenexport erhält nur die Liste an Kunden, die auch in der Registerkarte Tabelle angegezeigt wurden. 

Die "Unterauswahl" auf die Ansprechpartner wird nur für die aktuelle Suche hinterlegt und steht nur im Dialog Kunde zur Verfügung. 

Werden mehrere Ansprechpartner für den Datenexport markiert, können die Merkmale des einzelnen Ansprechpartners nicht exportiert werden! 

Die zusätzliche Übertragung einer Fixposnr zusätzlich zur Positionsnr von Bestellungen ist möglich. Die Struktur gilt für 
- ORDERS-Nachrichten - ORDCHG-Nachrichten - DESADV-Nachrichten - INVOIC-Nachrichten 
Der Dialog Datenexport enthält in der Warenwirtschaft zum Teil andere Funktionalitäten als in der Finanzbuchhaltung! 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Datenexport 
Tooltip: 

Dieser Dialog steht zum Export von Stammdaten (z.B. aus dem Kundenstamm) zur Verfügung. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Referenz-Link 

Dieser Button steht zur Verfügung, um Dokumenten-Links sowie URL‘s als Info in eNVenta ERP zu hinterlegen oder um 2 Vorgänge/Referenzen miteinander zu verknüpfen. 

Siehe Kapitel Referenz-Links . 

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Auswahl 

Dieser Button übernimmt Ihre Einstellungen in den zugrunde liegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Exportieren 

Mit diesem Button starten Sie den Export der Daten. 

### Button Laden 

Über diesen Button können Sie ein externes Format (XML-Datei) laden, um in eNVenta zu speichern und für den Export Ihrer Daten zu verwenden. 

Tooltip: 

Format aus Datei laden 

### Button Speichern 

Dieser Button speichert das Format in einer Datei, so dass Sie die Format-Datei extern verwenden können (z.B. für Ihren Support oder Administrator). 

Tooltip: 

Format in Datei speichern 

### Feld Format 
Unter diesem Schlüssel ist Ihr Format gespeichert. Entweder wählen Sie ein Format über Registerkarte Tabelle aus oder Sie geben den Schlüssel hier ein und suchen (F2) danach.; gibt es mehrere Treffer werden diese in die Registerkarte Tabelle gelistet. 

### Feld Beschreibung 
In diesem Feld kann ein individueller Text zur Beschreibung eingetragen werden. 

Tooltip: 

Angabe der Format-Beschreibung
