# Parameter Kasse Registerkarte Signierung AT 
Parameter Kasse Registerkarte Signierung AT 

Auf dieser Registerkarte nehmen Sie die grundlegenden Einstellungen für die Registrierungspflicht für die Kassen vor. 

Mehr zur Registrierungspflicht erfahren Sie im Kapitel Verfahren zur Registrierkassenpflicht . 
### Button Protokoll 

Mit diesem Button werden die Ergebnisse des gewählten Log Levels aufgelistet. Dies dient der internen Verwendung. Es öffnet sich der Dialog Protokoll Signierung . 

Siehe Kapitel Protokoll Signierung . 

Tooltip: 

Signaturprokoll einsehen 

### Button Test 

Mit diesem Button kann der Test des Webdienstes gestartet werden. 

Tooltip: 

Verbindung zum Signaturprovider testen 

### Feld URL 
Hier tragen Sie die URL ein. Diese kommt vom Provider A-Trust und ist die Adresse des Webservice. 

Tooltip: 

Angabe der URL des Signatur-Providers 

### Feld Benutzer 
Hier wird der Benutzer eingetragen. 

Tooltip: 

Angabe des Benutzers beim Signaturprovider 

### Feld Passwort 
Hier geben Sie das Passwort an. 

Tooltip: 

Angabe des Passworts beim Signaturprovider 

### Combobox Log Level 
Hier wählen Sie aus, was protokolliert werden soll: Info, Warnung oder Fehler. 

Tooltip: 

Auswahl, was protokolliert werden soll.
