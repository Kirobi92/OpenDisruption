# WB Index (Wiederbeschaffungsindex) 
WB Index 

Dieses Menü öffnet sich über Anlage >> Stamm >> WB Index . Hier können beliebige Wiederbeschaffungsindexe für Anlagen erfasst werden. D.h. dass für eine bestimmte Maschine ein Index festgelegt wird, um nach der Abschreibung dieser Maschine genügend finanzielle Mittel für die Neuanschaffung zur Verfügung zu haben. 

Wiederbeschaffungswert Index 

Der Index ist frei definierbar. 

Es kann z.B. der Verbraucherpreisindex (VPI) des Statistischen Bundesamts abgebildet werden. Verbraucherpreisindex und Inflationsrate - Statistisches Bundesamt (destatis.de) 

Ausgangspunkt dieser Betrachtung ist immer das jeweilige Basisjahr (derzeit 2015), in dem der Warenkorb erstellt wurde. Das aktuelle Gewichtungsschema des Warenkorbes wird durch das Statistische Bundesamt unter dem Titel „Wägungsschema für das Basisjahr 2015“ veröffentlicht. Üblicherweise alle fünf Jahre wird ein neuer Verbraucherpreisindex mit einem neuen Basisjahr eingeführt. Dadurch sind die zu langen Reihen verketteten Verbraucherpreisindizes unterschiedlicher Basisjahre jedoch nur eingeschränkt miteinander vergleichbar. 

Das Statistische Bundesamt veröffentlicht auch Zeitreihen zum Verbraucherpreisindex ab 1948 im Internet[2] und ab 1991 als Online-Datenbank mit Diagramm. 

Quelle: Verbraucherpreisindex für Deutschland – Wikipedia 

Beispiel: 
Anlage wird 2018 für 10.000 gekauft. Diese Anlage hat im Jahr 2019 einen Wiederbeschaffungswert von 10.144,51 – Details siehe nachfolgende Abbildung zum Verbraucherpreisindex. 

Berechnung 2019: 
Betrag / VPI Quelljahr * VPI Zieljahr 
10000 / 103,8 * 105,3 
= 10144,51 

Berechnung 2020: 
Betrag / VPI Quelljahr * VPI Zieljahr 
10144,51/ 105,3 * 106,999 (aktuell angenommener Wert) 
= 10308,19
