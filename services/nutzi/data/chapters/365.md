# Auftrag Cockpit Registerkarte Bestellung 
Auftrag Cockpit Registerkarte Bestellung 

Auf dieser Registerkarte werden die wesentlichen Daten, der mit der Position verbundenen Bestellung, angezeigt. 
### Tabelle 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung dient der einfachen Identifikation des Artikels und kann auch zur Suche verwendet werden. 

Bei Eingabe der Artikelnummer wird das Feld Bezeichnung automatisch mit dem Namen des Artikels gefüllt. 

Tooltip: 

Artikelbezeichnung

5 - Lieferung ohne Berechnung 
6 – Storniert

### Tabellenspalte Bestätigungsstatus 
Hier wird der Bestätigungsstatus angezeigt, wenn eine Strecken-Einkaufsart gewählt ist und die Mehrfachbestätigungs-EK-Logik im Dialog Parameter Einkauf gesetzt ist. Der Wert in dieser Spalte wird dann bei einer Änderung aktualisiert. 

Folgende Status stehen zur Verfügung: 

0/Null = unbestätigt 
1 = teilbestätigt, ein Termin 
2 = teilbestätigt, mehrere Termine 
3 = vollbestätigt, ein Termin 
4 = vollbestätigt, mehrere Termine 

Siehe Kapitel Parameter Einkauf .
