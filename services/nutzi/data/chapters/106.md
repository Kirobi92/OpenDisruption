### Registerkarte Administration 

Einstellungen Registerkarte Administration 

Diese Registerkarte ist nur im Administrations-Modus verfügbar. 

Hier werden alle verfügbaren Rollen des Authentication Services aufgelistet. Hier geben Sie an, ob das Zoom-Element mit seinen Einstellungen für bestimmte Rollen sichtbar sein soll. 

Über die Checkbox Öffentlich steuern Sie, ob das nun angelegte Zoom-Element für die Benutzer zugänglich sein soll. Dies sehen Sie nun in Ihrer Auswahl der Zoom-Elemente unterhalb der allgemeinen Auswahl dieses Zoom-Elements. Die Benutzer können das Zoom-Element mit seinen Einstellungen übernehmen, dessen Einstellungen aber nicht selbst bearbeiten. 

Über die Checkbox Automatisch sichtbar bekommen die Benutzer, welche das Recht auf dieses Zoom-Element haben, beim Start von eNVenta dieses Zoom-Element automatisch angezeigt. Eine manuelle Auswahl ist dann nicht notwendig bzw. wird auch gar nicht in der Auswahl der Zoom-Elemente angeboten. 
### Checkbox Öffentlich 
Damit das Zoom-Element für andere Benutzer verfügbar wird, muss diese Checkbox gesetzt werden. 

Das Zoom-Element kann in der Administrator-Ansicht solange konfiguriert und getestet werden, bis alles passt. 

Anschließend kann es mit dem Setzen der Checkbox veröffentlicht publiziert werden. 

Es ist auch möglich, die Checkbox zu einem späteren Zeitpunkt zu deaktivieren, wenn z.B. Nacharbeiten notwendig sind. 

Dadurch wird das Element bei der nächsten Anmeldung nicht mehr angeboten. 

### Checkbox Automatisch sichtbar 
Hier wird festgelegt, ob das Zoom-Element automatisch sichtbar sein soll, sofern es öffentlich ist. 

- Checkbox gesetzt : Dieses Element wird allen berechtigten Benutzern bei der nächsten Anmeldung angezeigt. Es kann vom Benutzer manuell ausgeblendet und auch wieder hinzugefügt werden. - Checkbox nicht gesetzt : Jeder berechtigte Benutzer kann dieses Element bei Bedarf manuell hinzufügen. 

### Tabelle Rollen 

Jedes Zoom-Element besitzt bereits eine Berechtigungskontrolle. Es kann nur von Benutzern verwendet werden, die das entsprechende Recht dazu besitzen. Wie die Prüfung erfolgt, ist in der Regel in der Beschreibung des Elements zu finden. 

Hier können die Rollen definiert werden, für die dieses Zoom-Element sichtbar sein soll. Das macht z.B. bei allgemeinen Zoom-Elementen Sinn, die durch die eingestellten Parameter nur für bestimmte Benutzer sichtbar sein sollen. 

Wenn keine Rolle ausgewählt ist, dann ist das Element für alle berechtigten Benutzer sichtbar. 
### Tabellenspalte Rolle 
Hier wird für einen Sachbearbeiter eine Rolle (Rechte) für einen bestimmten Vorgang zugeordnet bzw. angezeigt.
