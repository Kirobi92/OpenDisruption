# Merkmale Registerkarte Allgemein 
Merkmale Registerkarte Allgemein 

Auf dieser Registerkarte werden die Details des Merkmals gepflegt. 
### Combobox Einheit 
Hier wählen Sie die Einheit des Merkmals aus; z.B. m, km, Stück, Paket. 

### Combobox Datentyp 
Hier wählen Sie den Datentyp aus. Dieses Feld darf nicht leer sein! 

Der Typ darf nach dem Speichern nicht mehr geändert werden. 

Unterstützt werden: Ganzzahl, Kommazahl, Zeichenfolge, Text, Html Text, Datum 

- Zeichenfolge: maximal 250 Zeichen. 

Tooltip: 

Typ des Merkmals. 
Der Typ darf nach dem Speichern nicht mehr geändert werden. 
Unterstützt werden: Ganzzahl, Kommazahl, Zeichenfolge, Text, Html Text, Datum 
- Zeichenfolge: maximal 250 Zeichen. 

### Feld Format 
Wenn ein Datentyp ausgewählt wurde, kann hier das Format eingetragen werden. Allerdings nur für Ganzzahl, Kommazahl, Formatierungs-Ausdruck. 

Der Ausdruck wird bei der String-Umwandlung des Werts als Formatierungsangabe genutzt. 

Beispiele: 

Zahlen 
"#,##0.00" -> "1.000,00" 
"€ #,##0.00" -> "€ 1.000,00" 
"#.000 mm" -> "10000.000 mm" 
Datum: 
"yyyy.MM.dd" -> "2019.03.31" 

Tooltip: 

(Optional) Nur für Ganzzahl, Kommazahl, Datum als Formatierungs-Ausdruck. 
Der Ausdruck wird bei der String-Umwandlung des Werts als Formatierungsangabe genutzt. 
Beispiele: 

Zahlen 
"#,##0.00" -> "1.000,00" 
"€ #,##0.00" -> "€ 1.000,00" 
"#.000 mm" -> "10000.000 mm" 
Datum: 
"yyyy.MM.dd" -> "2019.01.31" 

### Feld Länge (max.) 
Wenn 'Zeichenfolge' gewählt ist, geben Sie hier die Länge ein. 

Tooltip: 

Nur für Zeichenfolge, Max. Länge der Zeichenfolge 

### Feld Wert min. 
Für Ganzzahl und Kommazahl geben Sie hier den kleinsten erlaubten Wert ein. 

Tooltip: 

(Optional) Nur für Ganzzahl, Kommazahl, kleinster erlaubter Wert. 

### Feld Max 
Für Ganzzahl und Kommazahl geben Sie hier den größten erlaubten Wert ein. 

Tooltip: 

(Optional) Nur für Ganzzahl, Kommazahl, größter erlaubter Wert. 

### Button A-Z 

Wenn Sie diesen Button drücken, werden die Werte nach dem Alphabet sortiert. 

Tooltip: 

Sortiert die Werte nach dem Alphabet 

### Button Satz hinzufügen 

Mit diesem Button können Sie einen neuen Eintrag in die Tabelle einfügen. 

### Button Nach oben verschieben 

Mit diesem Button kann ein Eintrag noch oben verschoben werden. 

Tooltip: 

Mit diesem Button werden Einträge nach oben verschoben. 

### Button Nach unten verschieben 

Mit diesem Button verschieben Sie einen Eintrag nach unten. 

Tooltip: 

Mit diesem Button werden Einträge nach unten verschoben. 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag aus der Tabelle. 

### Checkbox Nur Werte 
Wenn Sie diese Checkbox aktivieren, können nur Werte vom Merkmal ausgewählt werden. 

Tooltip: 

Es können nur die Werte von dem Merkmal ausgewählt werden. 

### Checkbox Mussfeld 
Wenn Sie diese Checkbox aktivieren, wird das Merkmal zum Pflichtmerkmal. D.h. der Artikel kann nur gespeichert werden, wenn dieses Merkmal mit einem Wert versehen wurde. 

Tooltip: 

Definiert das Merkmal als Pflichteinstellung. 

### Checkbox Kein Druck 
Wenn Sie diese Checkbox aktivieren, kann das Merkmal nicht gedruckt werden. 

Hinweis: Diese Checkbox wird im BMEcat Export nicht berücksichtigt, da der Export kein Druck ist. 

### Checkbox Kein Webshop 
Wenn Sie diese Checkbox aktivieren, wird das Merkmal nicht im Webshop angezeigt. 

### Checkbox Kein ML 
Wenn Sie diese Checkbox aktivieren, wird das Merkmal nicht in anderen Sprachen gepflegt. Dies betrifft nur den Datentyp Zeichenfolge . Es gibt keine Auswirkungen auf andere Datentypen. 

Tooltip: 

Nicht Mehrsprachig: 
"Kein ML" sorgt dafür, dass die Merkmale nicht in anderen Sprachen angezeigt werden. 
Das gilt nur für den Datentyp 'Zeichenfolge'.
