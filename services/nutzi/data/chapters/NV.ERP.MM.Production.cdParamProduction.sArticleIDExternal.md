# Kapitel NV.ERP.MM.Production.cdParamProduction.sArticleIDExternal

Hier kann ein Dienstleistungsartikel eingetragen bzw. über den Button ausgewählt werden. Dabei ist es sinnvoll, dass diesem Artikel kein Lager zugeordnet ist, der Artikel als Serviceartikel ausgewiesen ist und weder ein Chargen- noch ein Serienartikel ist. 

Diese Einstellung kann im Arbeitsgang übersteuert werden. 

Dieser Artikel wird für das Bestellen von Arbeitsgängen benötigt. 

Tooltip: 

Hier kann ein sogenannter Dienstleistungsartikel eingetragen werden.
