Finanzbuchhaltung / Konto Information 

# Konto Information Registerkarte OP-Aufstellungen 
Konto Info Registerkarte OP-Aufstellungen 

Auf dieser Registerkarte erhalten Sie eine Liste mit offenen Posten. 

In der rechten Tabelle erhalten Sie Detailinformationen zu einem markierten Eintrag der linken Tabelle. 
### Tabelle 

### Tabellenspalte S 
Hier wird der Verarbeitungsstatus angegeben. Zur Verfügung stehen: 

- = offen 
OK = erledigt 
VB = verbucht 
X = Fehler 

Tooltip: 

Verarbeitungstatus 
- = offen 
OK = erledigt 
VB = verbucht 
X = Fehler 

### Tabellenspalte Gesamt FW 
Hier wird der Gesamtbetrag in Fremdwährung angegeben. 

Tooltip: 

Angabe des Gesamtbetrags in Fremdwährung 

### Tabellenspalte Zahlbetrag FW 
Hier wird der Betrag in Fremdwährung angegeben. 

Tooltip: 

Angabe des Betrags in Fremdwährung 

### Tabellenspalte Währung 
Hier wird das Währungskürzel angegeben. 

Tooltip:
