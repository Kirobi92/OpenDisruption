# Nachrichten-Aufträge 
Nachrichten-Aufträge 

Im Navigator auf der Registerkarte Jobs steht Ihnen im Menü Druck der Eintrag Nachrichten-Aufträge zur Verfügung. Mit einem Klick auf diesen Menüeintrag öffnet sich der Dialog Nachrichten-Aufträge . Hier werden die Aufträge zur Übermittlung von Nachrichten (ähnlich den Druck-Aufträgen) angezeigt und auch, ob der Nachrichtenversand ausgeführt werden konnte oder nicht (Spalte ‚Status‘). 

Siehe Kapitel Nachrichtenjournal . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Nachrichten-Aufträge 
Tooltip: 

In diesem Dialog werden die Aufträge zur Übermittlung von Nachrichten (ähnlich den Druck-Aufträgen) angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

Dieser Button öffnet den Dialog Nachrichtenjournal . Hier wird das Journal für den ausgewählten Nachrichten-Auftrag angezeigt. 

Außerdem ist es möglich, mehrere Zeilen zu markieren und dann mittels Klick auf diesen Button kann der erneute Versand der E-Mails ausgelöst werden. Der Versand erfolgt erst nach einer Sicherheitsabfrage.

### Tabelle 

In dieser Tabelle werden die Nachrichten-Aufträge angezeigt. 

Hier können mehrere Zeilen markiert und dann kann mittels Klick auf den Button Erneut senden der erneute Versand der E-Mails ausgelöst werden. Der Versand erfolgt erst nach einer Sicherheitsabfrage. 

Für eine verbesserte Filterung im Dialog kann nach dem Status gefiltert werden.
