# Paketstraße 
Paketstraße 

Die Paketstraße beschreibt den Weg, den ein Paket nimmt, bis es das Werk verlässt; verschiedene Arbeitsschritte für den Versand einer Ware werden festgelegt. Das Kapitel steht für das 1-stufige LVS zur Verfügung. Dazu gehören: 

- Bestimmung der Versandart, falls nicht bereits im Kundenstamm oder Auftrag vorbelegt - Kommissionierung - Versandbearbeitung inkl. Wiegen - Lieferpapiere ausstellen - Versandaufkleber anbringen - etc. 
Wenn Ihnen keine Paketstraße zugeordnet ist, erhalten Sie eine Meldung. 

### Mehrzonen-Paket 
Ein Mehrzonen-Paket ist ein Paket in der Paketstraße, welches aus Waren mehrerer Zonen besteht. Das Etikett wird dann als Mehrzonen-Paket gekennzeichnet. 

Der letzte Mitarbeiter, der Ware für diese Paketware hinzu lagert, erhält die Information, dass das Mehrzonen-Paket jetzt komplett ist und geliefert werden kann. Bis zu diesem Zeitpunkt wird bei dem Versuch, das Paket an der Paketstraße zu bearbeiten, eine Fehlermeldung generiert. 

Bei der Verarbeitung der Ware an der Paketstraße ist zwingend sämtliche zugepackte Ware zu erfassen! Nur wenn alle Waren zum Mehrzonen-Paket bestätigt wurden, kann dieses Paket weiterverarbeitet werden. 

Bei einem Auftrag mit Zahlungsbedingung Nachnahme wird auch die Rechnung sofort gedruckt.

### Button Druck V-LOG 

Dieser Button überträgt die erstellten Pakete an V-Log, aber noch nicht an den Paketdienstleister. Nach der erfolgreichen Übermittlung an V-Log wird das Versandetikett am entsprechenden Drucker ausgedruckt und muss auf das Paket geklebt werden. Zu diesem Zeitpunkt wird auch die externe Paketnummer in eNVenta eingetragen und die Recherche/ das Tracking kann später über diese Nummer erfolgen. 

Nach dem erfolgreichen Druck des V-Log-Etiketts wird nun automatisch der Lieferschein gedruckt. Dafür hinterlegen Sie im Dialog Paketstraße den Lieferschein-Drucker. 

### Button Splitt
