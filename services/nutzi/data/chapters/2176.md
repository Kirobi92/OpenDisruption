# Parameter Service Erinnerung Registerkarte Allgemein 
Parameter Service Erinnerung Registerkarte Allgemein 

Auf dieser Registerkarte werden allgemeine Einstellungen bezüglich der Erinnerungen an Wartungen vorgenommen. 
### Feld E-Mail-Vorlage 
Hier wird eine E-Mail-Vorlage die Erinnerung an eine Wartung angegeben. 

### Button Suche 

Mit diesem Button können Sie eine E-mail-Vorlage auswählen. Es öffnet sich der Dialog Vorlagenauswahl . 

Siehe Kapitel Dokument erstellen (Vorlage auswählen) . 

Tooltip: 

Dokumentvorlage Suchen 

### Button Verzweigen 

Dieser Button öffnet den Dialog Dokumentvorlagen . 

Siehe Kapitel Dokument Vorlagenstamm . 

Tooltip: 

Dokumentvorlagen Verwalten 

### Feld E-Mail für Test 
Hier kann eine E-Mail-Adresse für einen Test angegeben werden. 

Tooltip: 

Angabe der E-Mail-Adresse. 
Bei der Angabe mehrerer Adressen, müssen diese durch Semikolon getrennt sein. 

### Feld Fax-Vorlage 
Hier wird eine Fax-Vorlage die Erinnerung an eine Wartung angegeben. 

### Button Suche 

Mit diesem Button können Sie eine Fax-Vorlage auswählen. Es öffnet sich der Dialog Vorlagenauswahl . 

Siehe Kapitel Dokument erstellen (Vorlage auswählen) . 

Tooltip: 

Dokumentvorlage Suchen 

### Button Verzweigen 

Dieser Button öffnet den Dialog Dokumentvorlagen . 

Siehe Kapitel Dokument Vorlagenstamm . 

Tooltip: 

Dokumentvorlagen Verwalten 

### Combobox Reportname 
Hier wird der Reportname ausgewählt. 

Tooltip: 

Auswahl des Reports für die Erinnerung 

### Combobox Erinnerungsweg 
Hier wird der Erinnerungsweg angegeben. Zur Verfügung stehen: 

- E-Mail, Fax, Druck - E-Mail, Druck, Fax - Fax, E-Mail, Druck - Fax, Druck, E-Mail - Druck, E-Mail, Fax - Druck, Fax, E-Mail 

Tooltip: 

Legt die Reihenfolge der Benachrichtigungsart vor. 

### Combobox Drucker 
Hier wird ein (Standard-)Drucker angegeben. 

Tooltip: 

Angabe des (Standard-)Druckers 

### Feld Karenztage 1 
Hier können Karenztage angegeben werden für Erinnerungen der Stufe 1. 

### Feld Karenztage 2 
Hier können Karenztage angegeben werden für Erinnerungen der Stufe 2. 

### Feld Karenztage 3 
Hier können Karenztage angegeben werden für Erinnerungen der Stufe 3.
