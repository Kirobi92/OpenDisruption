# Job-Verwaltung 
Job-Verwaltung 

Dieser Dialog öffnet sich über den Button oder im Navigator >> System >> LVS >> Job für Lagerauftragsstorno . 

Haben Sie im zugrundeliegenden Dialog z.B. Massendruck bereits Parameter oder Werte vorbelegt, so werden sie an die Job-Verwaltung übergeben. 

Ein Job kann nicht erstellt werden, wenn der Job-Server nicht lizensiert ist. Auch beim Sofort-Job in den diversen Dialogen erscheint eine Meldung, falls der Job-Server nicht lizensiert ist. 

Klicken Sie auf den Button um einen neuen Job anzulegen. Hier wird dann ein Eintrag in der oberen Tabelle generiert, wobei die Bezeichnung (gleichnamiges Feld) des aufgerufenen Dialogs als Bezeichnung voreingestellt wird. Diese Bezeichnung können Sie überschreiben. 

Wenn Sie einen neuen Job speichern, ist zwingend ein Zeitplan zu hinterlegen. 

Siehe Kapitel Job Zeitplan . 

Des Weiteren kann eine Job-Server Gruppe direkt bei der Einrichtung eines Jobs angegeben werden. Dazu stehen im Dialog Job-Verwaltung die Combobox Job-Server Grp. und in der Job Übersicht die Spalte ‚Job-Server Grp‘ zur Verfügung. 

Beim Speichern des Jobs wird dieser automatisch aktiviert, damit der Job-Server diesen sofort verarbeiten kann (Status = aktiv ). Sie sehen diesen Job dann auch sofort im Dialog Job-Übersicht .

### Feld Bezeichnung 
In diesem Feld wird der Name des zugrundeliegenden Dialogs angezeigt. Diese Bezeichnung kann überschrieben werden. 

### Tabelle Parameter 

In der Tabelle werden alle Parameter zum Job Server-„Objekt“ angezeigt bzw. verwaltet. Bei Aufruf zur Einrichtung eines Jobs werden hierzu alle verwendbaren Parameter übergeben. 
### Tabellenspalte Parameter 
In dieser Spalte wird die Bezeichnung des Parameters angezeigt. 

Haben Sie im zugrundeliegenden Dialog z.B. Massendruck bereits Parameter angelegt, werden diese hier angezeigt. 

### Tabellenspalte Wert 
In dieser Spalte wird der entsprechend übergebene Wert aus dem aufrufenden Dialog automatisch eingetragen. Der Wert kann manuell geändert werden, es erfolgt aber keine Validierung zum Zeitpunkt der Angabe, sondern erst bei Ausführung! Um den Parameter zu bearbeiten gibt es den Button Bearbeiten im Kopfbereich. In diesem Fall wird der Vorgang zum Job Server aufgerufen und dort können die Parameter angepasst werden – damit erhöht sich die Sicherheit, dass nur valide Parameter eingetragen werden. Über den Button Job Server im aufgerufenen Dialog wird der aktualisierte Parameter übernommen. 

### Tabellenspalte Name 
Hier wird der technische Name des Parameters angezeigt. 

### Combobox Job-Server Grp. 
Hier können Sie eine Job-Server Gruppe auswählen. 

### Feld Letzter Lauf 
Hier wird das Datum des letzten Laufs des Jobs angezeigt. 

### Feld Nächster Lauf 
Hier wird das Datum angezeigt, an dem der Job das nächste Mal laufen wird. 

### Feld Zeitplan 
In diesem Feld wird der Zeitplan angezeigt, der dem Job hinterlegt ist. 

Über den danebenstehenden Button gelangen Sie in den Dialog Job-Zeitplan zur Definition eines Zeitplans. 

### Button Job Planen 

Wenn Sie auf diesen Button drücken, öffnen Sie den Dialog Job Zeitplan , um eine Planung für den Job vorzunehmen. 

Siehe Kapitel Job Zeitplan .
