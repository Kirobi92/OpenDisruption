# Kapitel NV.ERP.Base.Common.cdSystemMM.shtCostObjectConstraint

Wenn Sie diesen Parameter aktivieren, ist jeder Auftragsposition zwingend ein Kostenträger zuzuweisen. Vorausgesetzt die Kostenrechnung ist lizenziert! 

Beim Speichern von Auftragspositionen wird dann geprüft, ob ein Kostenträger zugeordnet ist.
