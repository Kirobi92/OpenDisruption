Allgemeine Funktionalitäten in eNVenta ERP / Suchfunktionen in eNVenta ERP 

# Auswahl von Adressen, Kunden und Lieferanten 
AKL-Auswahl 

In diesem Dialog haben Sie die Möglichkeit, mit verschiedenen Kriterien eine Adresse, einen Kunden oder Lieferanten zu suchen. Um die Auswahl detaillierter zu gestalten, können Sie dazu noch die Checkboxen Adressen , Kunden und (nur im Call-Center) Lieferanten aktivieren. 

Im Standard erfolgt die Suche auf die Rechnungsanschrift. Bei der Suche nach Kunden und/oder Adressen kann dies mit den Radiobuttons des Suchmodus auf die Lieferanschrift umgestellt werden. 

Sie können eines oder mehrere der oberen Felder ausfüllen. Um den Suchvorgang zu starten, betätigen Sie den Button . Haben Sie keines der Felder vorbelegt, wird die Suche über die kompletten Stammdaten ausgeführt. 

Eine generelle Suche kann zu langen Wartezeiten führen. 

Nur wenn zu dem vorgegebenen Suchnamen mehrere Kunden existieren, werden automatisch auch die Adressen berücksichtigt. 

Wird im Such-Dialog eine Adresse ausgewählt, erscheint eine Abfrage, ob diese in einen Kunden umgewandelt werden soll. 

Siehe Kapitel Adressumwandlung .

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.

### Tabellenspalte GEO-Koordinaten 
Hier werden geografische Daten zur besseren Orientierung angegeben. Damit lassen sich Besuche o.ä. besser planen und Sie können die Karte zur besseren Orientierung und Festlegung einer Besuchs-Reihenfolge nutzen. 

Das Format der GEO-Koordinaten lautet: 
XX.yyyyyy,AA.bbbbbb 

Beispiel für Stockach, Nissen&Velten: 
47.848029,09.005334 

Tooltip:

### Tabellenspalte Transport Distanz 
Hier können Sie den Transportweg bis zur Grenze in % angeben. Dies vor dem Hintergrund, dass es sich um einen statistischen Wert für eine ungefähre Hochrechnung handelt. Deshalb muss der Sachbearbeiter hier den wahrscheinlichsten bzw. durchschnittlich zutreffenden Prozentsatz hinterlegen. 

Beispiel: 
Transportweg bis Adresse = 100 km 
Weg bis Grenze = 40 km 
=> Transport Distanz = 40
