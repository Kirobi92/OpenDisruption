# Kontoauswahl 
Kontoauswahl 

Im Buchen auf der Registerkarte Allgemein öffnet sich die Kontoauswahl automatisch, wenn bei Eingabe von Buchstaben bzw. Ziffern in den Feldern Soll und Haben und anschließendem Klick auf die Enter -Taste, mehrere Treffer möglich sind. 

In der Tabelle erscheinen sämtliche Konten entsprechend der Logik des eingestellten Buchungskreises. Die Einstellungen im Dialog Parameter Fibu , Bereich Kontensuche nach spielen hierbei ebenfalls eine Rolle. Ist dort nichts ausgewähl, werden das Feld Multisuche und die Checkbox Beliebig nicht angezeigt. 

Für alle 3 Registerkarten gilt: Wenn es einen exakten Treffer gibt, wird nur dieser eingelesen. 

Wenn Sie F2 (Speichern) drücken, werden mögliche Treffer, beginnend mit ... nachgelesen. Ggf. ist eine erweiterte Suche über die Checkbox Beliebig möglich. 

Sie können entweder das gewünschte Konto aus der Tabelle auswählen oder geben auf den Registerkarten Sachkonto , Debitor oder Kreditor Ihre Suchkriterien ein. Anschließend klicken Sie auf den Button . In der Tabelle erscheinen nun alle Konten, die den gewählten Suchkriterien entsprechen. 

Mit einem Doppelklick auf die Zeile mit dem gewünschten Konto übernehmen Sie dieses direkt in den Dialog Buchen . Sie können die Zeile auch markieren und das Konto durch einen Klick auf den Button übernehmen. 

Siehe Kapitel Parameter Finanzbuchhaltung .
