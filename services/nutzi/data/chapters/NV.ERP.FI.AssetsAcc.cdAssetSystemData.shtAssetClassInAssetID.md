# Kapitel NV.ERP.FI.AssetsAcc.cdAssetSystemData.shtAssetClassInAssetID

Wenn Sie diesen Parameter aktivieren, wird beim Anlegen einer neuen Anlage die Anlagenklasse vor das Anlagenformat gesetzt. 

Das funktioniert folgendermaßen: 

Sie legen eine neue Anlage an, ohne manuell eine Nummer zu definieren. Wenn alle Daten eingefügt sind und Sie diese Eingaben speichern, wird der Nummer (welche hier laut Parametereinstellung 5-stellig ist) eine Anlagenklasse (z.B. 30) vorangestellt. Die Nummer könnte beispielsweise so aussehen: 3000695. Leere Stellen werden mit 0 aufgefüllt. 

Siehe Kapitel Anlagenklasse .
