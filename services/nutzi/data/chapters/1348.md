# Gutschriftsabrechnung Registerkarte EDI Nachricht 
Gutschriftsabrechnung Registerkarte EDI-Nachricht 

Diese Registerkarte steht zur Verfügung, wenn Daten über den EDI automatisch importiert wurden. 

Hier kann der Inhalt der Datei betrachtet werden bzw. nochmals importiert werden, um z.B. nachträglich die Positionserkennung zu erhöhen. 
### Button Verarbeiten 

Wenn Sie diesen Button drücken, wird die EDI-Nachricht erneut geladen und die Positionsdaten werden gesetzt. 

Tooltip: 

Lädt die EDI-Nachricht erneut und setzt die Positionsdaten. 

### Feld Dateitext 
In diesem Feld wird der Text der Datei angezeigt.
