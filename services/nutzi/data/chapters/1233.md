# Angebot Registerkarte Logistik 
Registerkarte Logistik 

Auf dieser Registerkarte weisen Sie dem Vorgang Informationen für den Transport zu. 
### Combobox Stammtour 
Hier geben Sie an, mit welcher Tour (Route) (z.B. täglich, wöchentlich) der Kunde bzw. Lieferant immer (regelmäßig) beliefert werden soll. Die Touren, die zur Verfügung stehen, werden im Tourenstamm gepflegt. 

Siehe Kapitel Tourenstamm . 

Tooltip: 

Es kann eine Tour bzw. Route hinterlegt werden, mit welcher die Auslieferung erfolgt. 

### Feld Tour lfd Nummer 
Hier wird für die Tour eine Stopp-Nr. innerhalb der Tour angegeben. Damit definieren Sie, an wievielter Stelle der Kunde beliefert wird. 

Tooltip: 

Angabe des Stopps innerhalb einer Tour 

### Combobox Tourzyklus 
Wenn Sie hier den Tourzyklus angeben, wird bei der Generierung von Touren der Stammtour ein Tourzyklus zugeordnet (z.B. Vormittag, Nachmittag oder 1. und 2. Tour eines Tages). 

Siehe auch Kapitel Tourzyklus . 

Tooltip: 

Angabe, mit welchem Tourzyklus die Auslieferung erfolgt. 

### Combobox Dispo-Kreis 
Aus dieser Combobox wählen Sie den Dispositionskreis zur regionalen Abgrenzung. Die Anlage der Dispositionskreise erfolgt in einer Pickliste. 

Tooltip: 

Angabe des Dispositionskreises zur regionalen Abgrenzung 

### Feld Fixtermin 
Hier können Sie für die Lieferung ein festes Datum eintragen. 

### Feld Früh. Uhrzeit 
Hier können Sie eine Uhrzeit für den frühesten Anlieferungstermin eingeben. 

### Feld Spät. Uhrzeit 
Hier können Sie eine Uhrzeit für den spätesten Anlieferungstermin eingeben. 

### Checkbox Kein Paketdienst 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Paketdienst-Touren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Kein Paketdienst“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Beim Kopieren eines Angebots / Auftrags wird diese Einstellung übernommen. 

Tooltip: 

Keine Lieferung per Paketdienst

### Checkbox Kein Fuhrpark 
Wenn dieser Parameter aktiviert ist, findet keine Lieferung über den Fuhrpark statt. 

Tooltip: 

Keine Lieferung per Fuhrpark
