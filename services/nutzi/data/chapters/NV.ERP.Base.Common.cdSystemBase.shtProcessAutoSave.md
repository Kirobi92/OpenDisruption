# Kapitel NV.ERP.Base.Common.cdSystemBase.shtProcessAutoSave

Wenn Sie diese Checkbox aktivieren, werden mit Klick auf den Button Drucken ( ) und Bestellen (in Auftrag und Angebot) die Datensätze automatisch gespeichert. 

D.h. mit Klick auf den Button wird bei Auswahl von Einzeldruck von Einkaufs- und Verkaufsbelegen automatisch gespeichert. Bei der Speicherung werden die gewohnten Routinen durchlaufen und bei fehlerhaften Daten wird eine entsprechende Fehlermeldung angezeigt. 

Tooltip: 

Änderungen automatisch speichern bei Auswahl von Einzeldruck von Ein- und Verkaufsbelegen.
