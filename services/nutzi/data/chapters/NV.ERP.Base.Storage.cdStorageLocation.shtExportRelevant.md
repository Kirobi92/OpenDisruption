# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtExportRelevant

Dieses Kennzeichen legt fest, ob das Lager für zollrelevante Geschäftsvorfälle im Exportbereich wichtig ist. 

Eine entsprechende Prüfung erfolgt bei aktiver ATLAS-Schnittstelle in der Kommissionierung! 

Tooltip: 

Kennzeichnet das Lager für zollrelevante Geschäftsvorfälle beim Export.
