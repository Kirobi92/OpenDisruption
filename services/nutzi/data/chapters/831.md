# Adressauswahl 
Adressauswahl 

Dieser Dialog öffnet sich im Dialog Adressverknüpfung über den Button neben der Tabelle Adresse . 

In diesem Dialog kann eine Vielzahl von Ansprechpartner verwaltet werden. D.h. eine Vielzahl an Rechnungsanschriften, Lieferanschriften und Rechnungspostanschriften (Zusatzanschriften). 

Des Weiteren können den Kontaktpersonen eine oder mehrere Zusatzanschriften zugeordnet werden. Dabei ist allerdings eine Anschrift als Hauptanschrift für den Ansprechpartner zu definieren. Die Adressinformationen dieser Hauptanschrift werden dann in die entsprechenden Felder des Ansprechpartners übernommen und können nicht geändert werden. (So wird eine Doppelpflege verhindert!) 

Folgende Adresstypen stehen zur Verfügung: 

AN = Angebotsanschrift 
BA = Bestellanschrift 
AB = Auftragsbestätigungsanschrift 
RA = Rechnungsanschrift 
RP = Rechnungspostanschrift 
LA = Lieferanschrift 
LE = Leistungsempfängeranschrift

### Checkbox (AN) Angebotsanschrift 
Wenn Sie diese Checkbox aktivieren, handelt es sich um eine Angebotsanschrift. D.h. eine abweichende Anschrift, an welche das Angebot gesendet wird. 

Diese Adresse hat sonst keine weitere spezielle Funktion. 

Tooltip: 

Angebotsanschrift (AN): 
Abweichende Anschrift, an welche das Angebot gesendet wird. 
Ansonsten hat diese Adresse keine weitere spezielle Funktion. 

### Checkbox (BA) Bestellanschrift 
Wenn Sie diese Checkbox aktivieren, handelt es sich um eine Bestellanschrift. D.h. eine abweichende Anschrift, an welche die Bestellung gesendet wird. 

Diese Adresse hat sonst keine weitere spezielle Funktion. 

Tooltip: 

Bestellanschrift (BA): 
Abweichende Anschrift, wo der Besteller sitzt. 
Ansonsten hat diese Adresse keine weitere spezielle Funktion. 

### Checkbox (AB) Auftragsbestätigungsanschrift 
Wenn Sie diese Checkbox aktivieren, handelt es sich um eine Auftragsbestätigungsanschrift. D.h. eine abweichende Anschrift, an welche die Auftragsbestätigung gesendet wird. 

Diese Adresse hat sonst keine weitere spezielle Funktion. 

Tooltip: 

Auftragsbestätigungsanschrift (AB): 
Angabe, wohin die Auftragsbestätigung gesendet wird. 
Ansonsten hat diese Adresse keine weitere spezielle Funktion. 

### Checkbox (RA) Rechnungsanschrift 
Wenn Sie diese Checkbox aktivieren, handelt es sich um eine Rechnungsanschrift. D.h. eine abweichende Anschrift, an welche die Rechnung gesendet wird. 

Es kann unterschiedliche Rechnungsanschriften zu einer juristischen Person geben. 

Die abweichende Rechnungsanschrift hat keine Auswirkung auf das Mahnwesen. 

Die Mahnungen laufen immer auf den Kundenstammsatz, nicht auf die abweichende Rechnungsanschrift! Ebenso ZM u.v.a. 

Relevant für die Finanzbuchhaltung sind die Rechnungsanschrift und Lieferanschrift aus den Kopfdaten! 

Tooltip: 

Rechnungsanschrift (RA): 
Unterschiedliche Rechnungsanschriften zu einer juristischen Person. 
ACHTUNG: Die abweichende Rechnungsanschrift hat keine Auswirkung auf das Mahnwesen. 
Die Mahnungen laufen immer auf den Kundenstammsatz, nicht auf die abweichende RA! Ebenso ZM u.v.a. 
Relevant für die Finanzbuchhaltung ist die Rechnungsanschrift und Lieferanschrift aus den Kopfdaten! 

### Checkbox (RP) Rechnungspostanschrift 
Wenn Sie diese Checkbox aktivieren, handelt es sich um eine Rechnungspostanschrift. D.h. eine abweichende Anschrift, an welche die Rechnungen gesendet wird. 

Dies ist eine abweichende Rechnungsanschrift für Rechnungen. 

Die abweichende Rechnungsanschrift hat keine Auswirkung auf das Mahnwesen. 

Die Mahnungen laufen immer auf den Kundenstammsatz, nicht auf die abweichende Rechnungspostanschrift! 

Es ist als reine Postanschrift zu verstehen. 

Tooltip: 

Rechnungspostanschrift (RP): 
Abweichende Rechnungsanschrift für Rechnungen. 
ACHTUNG: Die abweichende Rechnungsanschrift hat keine Auswirkung auf das Mahnwesen. 
Die Mahnungen laufen immer auf den Kundenstammsatz, nicht auf die abweichende RP! 
Es ist als reine Postanschrift zu verstehen. 

### Checkbox (LA) Lieferanschrift 
Wenn Sie diese Checkbox aktivieren, handelt es sich um eine Lieferanschrift. D.h. eine abweichende Anschrift, an welche die Lieferung gesendet wird. 

Diese Adresse hat sonst keine weitere spezielle Funktion. 

Tooltip: 

Lieferanschrift (LA): 
Angabe einer abweichenden Lieferanschrift. 
Diese kann im Auftrags-/Angebots-/Gutschriftkopf unter der Registerkarte „Anschriften“ ausgewählt werden 
und ist nicht unter der Registerkarte „Zusätzliche Anschriften“ auswählbar! 

### Checkbox (LE) Leistungsempfängeranschrift 
Wenn Sie diese Checkbox aktivieren, handelt es sich um eine Leistungsempfängeranschrift. D.h. eine abweichende Anschrift des Leistungsempfängers. 

Diese Adresse hat sonst keine weitere spezielle Funktion. 

Der Leistungsempfänger ist im Property oSalesProcessAddress_ARA zu finden. 

Tooltip: 

Leistungsempfängeranschrift (LE)
