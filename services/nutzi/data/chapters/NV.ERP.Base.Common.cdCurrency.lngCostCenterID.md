# Kapitel NV.ERP.Base.Common.cdCurrency.lngCostCenterID

Hier kann eine Kostenstelle für Kursdifferenzen ausgewählt werden, die durch das Modul Kursbewertung entstehen. 

Diese Kostenstelle wird verwendet, wenn auf den entsprechenden Kursgewinn- bzw. Kursverlustkonten keine Kostenstellenverteilung oder keine 100%-Verteilung hinterlegt ist. 

Siehe Kapitel Kursbewertung .
