# Konfiguration der Webshop Benutzer Registerkarte Tabelle 
Benutzerverwaltung Registerkarte Tabelle 

Auf dieser Registerkarte werden alle Webshop-Benutzer aufgelistet und stehen für eine Auswahl zur Verfügung. 
### Tabelle 

### Tabellenspalte Benutzer 
Hier wird der Webshop-Benutzer angegeben. 

Tooltip: 

Angabe des Webshop-Nutzers 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Partner 
Die Nummer des Kontakts / Ansprechpartners wird angegeben. 

Tooltip: 

Nummer des Kontakts/Partners 

### Tabellenspalte Firma 1 
Der 1. Adresszeile der Firma wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Partner 
Hier wird der Name des Ansprechpartners angegeben. 

Tooltip: 

Name des Ansprechpartners 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Dieses Feld zeigt die Anzahl der tatsächlich eingelesenen Datensätze.
