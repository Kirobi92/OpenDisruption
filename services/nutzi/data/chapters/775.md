# Gutscheinartikel 
Gutschein - / Gratisartikel 

Dieser Dialog öffnet sich im Artikel >> Optionen >> Gutscheinartikel . 

Kunden können im Shop Gutscheine einlösen. Dies können zum einen gekaufte Wertgutscheine sein oder zum anderen kostenlos verteilte Gutscheine, welche zum Kauf anregen sollen. Diese Gutscheine/Gratisartikel können an eine Bedingung geknüpft sein (z.B. bestimmter Artikel oder Warenwert). 

Bei diesen Gutscheinen handelt es sich nicht um erworbene Gutscheine an der Kasse, sondern um Gutschein-Codes, welche zum Hinzufügen eines bestimmten Artikels in den Warenkorb führen. 

Der Gutschein-Code ist ein optionales Feld. Wenn ein Gutschein-Code angegeben wird, muss geprüft werden, welche Artikel in den Gutschein-Bereich fallen (Warenwert von / bis). Es werden alle Artikel eingefügt, welche in den Wertebereich fallen. 

Grundsätzlich gibt es 2 Möglichkeiten, die Artikel über einen Gutscheincode (z.B. Coupon hinzufügen oder diese für bestimmte Bestellarten freischalten) zu referenzieren. 

Für einzelne Gutscheine können anhand des Artikelstamms und des Kundenstamms individuelle Einschränkungen vorgenommen werden. Beispiel: nur Kundengruppe Stahl . 

Beim Absenden des Warenkorbs / in der Abschlusslogik des Auftrags wird noch einmal überprüft, ob die Bedingungen für die Gratis- / Gutscheinartikel erfüllt sind. 

Im Webshop wird der Kunde auf der Warenkorb-Seite auf Gratisartikel hingewiesen, wenn seine Bestellpositionen die Bestellung eines solchen erlauben. Hier werden nur Artikel vorgeschlagen, welche KEINEN Gutscheincode haben. In einem extra Eingabefeld kann der Benutzer einen Gutschein-Code angeben. Erfüllt er die Bedingungen für den Gutschein, wird dieser in den Warenkorb eingefügt.

### Feld Gültig von 
Hier geben Sie das Start-Datum eines Gültigkeitszeitraums ein. 

### Feld Gültig bis 
Hier geben Sie das Ende-Datum eines Gültigkeitszeitraums ein. 

### Feld Gutschein Code 
Wenn ein Gutschein-Code angegeben wird, muss geprüft werden, welche Artikel in den Gutschein-Bereich fallen (Warenwert von / bis). Es werden alle Artikel eingefügt, welche in den Wertebereich fallen. 

Wenn dieses Feld leer bleibt, ist der Artikel ein Gutschein- bzw. Gratisartikel. 

### Checkbox Exklusiv 
Wenn diese Checkbox aktiviert ist, kann pro Warenkorb nur ein Gutschein eingelöst werden. 

### Checkbox Mehrere Artikel 
Wenn diese Checkbox gesetzt ist, können mehrere Gutscheine pro Warenkorb eingelöst werden.

### Dock Panel Einschränkung 

Über diese Radiobuttons können die Gutscheine auf eine bestimmte Gruppe eingeschränkt werden. 
### Radiobutton Pro Kunde 
Wenn Sie diesen Radiobutton aktivieren, werden die Gutscheine auf einen Kunden eingeschränkt. 

### Radiobutton Pro Bestellung 
Wenn Sie diesen Radiobutton aktivieren, werden die Gutscheine auf eine Bestellung eingeschränkt. 

### Radiobutton Pro Ansprechpartner 
Wenn Sie diesen Radiobutton aktivieren, werden die Gutscheine auf einen Ansprechpartner eingeschränkt. 

### Combobox Artikelgruppe 
Hier wählen Sie eine Artikelgruppe aus. 

Damit ist das Einfügen eines Gutschein- / Gratisartikels nur möglich, wenn die entsprechende Artikelgruppe bereits Bestandteil der Bestellung ist. 

### Feld Artikel von 
Statt Warenwert von / bis können Sie hier eine Artikelnummer eingeben. 

Wenn Sie hier einen Artikel eintragen, wird das Einfügen eines Gutschein- / Gratisartikels nur möglich, wenn der entsprechende Artikel bereits Bestandteil der Bestellung ist. 

Tooltip: 

Angabe der Artikelnummer 

### Feld Artikel bis 
Statt Warenwert von / bis können Sie hier eine Artikelnummer eingeben. 

Wenn Sie hier einen Artikel eintragen, wird das Einfügen eines Gutschein- / Gratisartikels nur möglich, wenn der entsprechende Artikel im gewählten Artikelbereich bereits Bestandteil der Bestellung ist. 

Tooltip:
