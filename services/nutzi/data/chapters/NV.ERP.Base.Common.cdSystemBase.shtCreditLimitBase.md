# Kapitel NV.ERP.Base.Common.cdSystemBase.shtCreditLimitBase

Für das Kreditlimit kann definiert werden, ob es auf den Netto- oder Bruttopreis definiert wird. Dazu stehen die Radiobuttons exkl. MwSt. bzw. inkl. MwSt. zur Verfügung. Standardmäßig ist exkl. MwSt. eingestellt. 

Für weitere Details bezüglich Kreditlimit lesen Sie Kapitel Kreditlimit - Allgemeine Informationen . 

Tooltip: 

Definieren Sie, ob das Kreditlimit auf Brutto- oder Nettopreis berechnet werden soll.
