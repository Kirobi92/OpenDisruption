# Prognosestatistik 
Prognosestatistik 

Die Prognosestatistik öffnet sich im Dialog Prognosegenerator über den Button Statistik und bietet einen Überblick über die prozentualen Anteile der einzelnen Verfahren an der Summe der berechneten Artikel mit Qualitätsbewertung. Die dargestellten Verfahren entsprechen dem Standard aus den Lehrbüchern. 

Im Verfahren Sporadischer Verlauf werden Artikel, die nur gelegentliche Lagerbewegungen aus den Abverkäufen oder der Produktion aufweisen, dargestellt. Artikel ohne Lagerbewegungen werden im Verfahren Inaktiver Artikel dargestellt. Für diese Artikel ist ein automatisches Bestellwesen nicht praktikabel. Deshalb werden auch keine Varianzen berechnet. 

Wie bereits im vorhergehenden Kapitel beschrieben, wird für jeden Artikel eine Qualitätskennzahl für das zugewiesene Verfahren ermittelt. Diese werden in 3 Spalten zusammengefasst.

### Registerkarte Allgemein 

Prognosestatistik Registerkarte Allgemein 

Diese Registerkarte steht zur Verfügung, um die Artikel anzuzeigen, für welche das ermittelte Verfahren angewendet wurde. Die Tabelle wird gefüllt, wenn Sie auf der Registerkarte Allgemein auf das entsprechende Prognoseverfahren einen Doppelklick machen. So können Sie sich zum Beispiel alle A-Artikel an einem Lagerort mit dem Prognoseverfahren Gleitender Durchschnitt anzeigen lassen. 

Wenn Sie einen Doppelklick auf einen Artikel ausführen, wird der Prognosemonitor geöffnet und zeigt die Daten zum Artikel an. 
### Tabelle 

### Tabellenspalte Prognoseverf. 
Hier wird das Verfahren angegeben. 

### Tabellenspalte Anteil % 
Angabe, welcher Anteil an Artikeln aus der Prognosestatistik zu dieser Gruppe gehört. 

### Tabellenspalte Varianz% 0-9 
In dieser Spalte wird pro Verfahren der prozentuale Anteil der Artikel mit guter Qualität (Qualitätskennzahl 0 – 9) angezeigt, d.h. das ermittelte Verfahren ist optimal geeignet. 

### Tabellenspalte Varianz% 10-19 
Artikel mit mittlerer Qualität (Qualitätskennzahl 10 – 19). 

### Tabellenspalte Varianz% 20-99 
Artikel mit schlechter Qualität (Qualitätskennzahl 20 – 99).
