# AES-Parameter 
Parameter AES 

Dieser Dialog öffnet sich im Kommissionieren >> Optionen >> AES-Schnittstelle >> AES-Parameter . Hier stellen Sie die grundlegenden Parameter für die Definition und Pflege des AES ein. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Parameter AES 
Tooltip: 

In diesem Dialog stellen Sie die grundlegenden Parameter für die Definition und Pflege des AES ein. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Speichern 

Mit diesem Button speichern Sie die Einstellungen. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Standard 

Dieser Button belegt die systemseitig notwendigen Felder vor, welche immer einen vorgegebenen Wert haben. 

Der Button wird nur dann genutzt, wenn die Felder aus Versehen manuell überschrieben wurden. 

Tooltip: 

Belegt die systemseitig notwendigen Felder vor. 

### Feld AES Partner 
Hier geben Sie die AES-Partnernummer an. Diese Nummer wird von BEX vergeben. 

Tooltip: 

Geben Sie die AES-Partnernummer ein. 

### Feld AES Kunde 
Die AES-Nummer des Kunden aus eNVenta ERP wird eingetragen. 

Tooltip: 

Geben Sie die AES-Kundennummer ein. 

### Feld AES Mandant 
Hier geben Sie den AES-Mandanten an. Diese Nummer wird von BEX vergeben. 

Tooltip: 

Geben Sie den AES-Mandanten ein. 

### Feld AES Benutzer 
Hier geben Sie den AES-Benutzernamen ein. Diese Nummer wird von BEX vergeben. 

Tooltip: 

Geben Sie den AES-Benutzer ein. 

### Checkbox Automatische Übergabe an AES 
Wenn Sie diese Checkbox aktivieren, werden die Daten automatisch an AES übergeben. 

Tooltip: 

Wenn aktiviert, werden die Daten automatisch an AES übergeben. 

### Checkbox Vor Übertragung AES-Vorgang aktualisieren 
Wenn Sie diese Checkbox aktivieren, werden die Versanddaten vor der endgültigen Datenübertragung nochmals ermittelt, validiert und zusammengestellt. 

Tooltip: 

Wenn aktiviert, werden die Versanddaten vor der endgültigen Datenübertragung nochmals ermittelt, validiert und zusammengestellt. 

### Feld URL 
Die Service-Adresse wird eingetragen. Über den Button Default kann diese ggf. wiederhergestellt werden. 

Tooltip: 

Geben Sie die Service-Adresse ein. 

### Feld Benutzer 
Dieser Eintrag ergibt sich aus den AES-Mandanten-Einstellungen. 

Tooltip: 

Der Mandant wird angezeigt. 

### Feld Passwort 
Diese Nummer wird von BEX vergeben und ist für den Service gedacht. 

Tooltip: 

Geben sie ein Passwort an. 

### Feld Zertifikat 
Der Name der Zertifikatsdatei wird eingetragen. Er ist für die sichere Kommunikation mit dem Service über https erforderlich. Über den Button Default kann diese ggf. wiederhergestellt werden. 

Tooltip: 

Geben Sie die Zertifikats-Datei an. 

### Feld Timeout in ms 
Der Eintrag hier definiert die Zeit in Millisekunden, nach der ein Kommunikationsversuch mit dem Service abgebrochen wird, wenn der Service bis dahin keine Antwort gegeben hat. Dieser Wert kann manuell eingestellt werden. 

Tooltip: 

Geben Sie die Zeit in Millisekunden an, nach denen der Kommunikationsversuch mit dem Service abgebrochen wird. 

### Checkbox Über Proxy 
Wenn die Kommunikation mit dem AES -Service über einen Proxy erfolgen soll, muss diese Checkbox gesetzt sein. Dann kann der Proxy über die Felder URL , Port , Domain , Benutzer , Passwort eingestellt werden. 

Tooltip: 

Aktivieren Sie die Checkbox, wenn der Service über einen Proxy erfolgen soll. 

### Feld URL 
Hier wird die URL für die Internetadresse angegeben. 

Tooltip: 

Angabe der URL für die Internetadresse 

### Feld Domain 
Hier wird die URL für die Internetadresse angegeben. 

Tooltip: 

Angabe der URL für die Internetadresse
