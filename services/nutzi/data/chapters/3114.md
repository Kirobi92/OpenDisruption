# Import Editor - Preisgruppen Registerkarte Preisgruppen 
Import Editor - Preisgruppen Registerkarte Preisgruppen 

Auf dieser Registerkarte werden Informationen der Import Daten und des Artikelschattenstamms nebeneinander gestellt. Die Daten können verglichen und angepasst werden. 

Hier kann auch ein Gültigkeitsdatum bzw. ein Gültigkeitszeitraum (Felder Gültig ab - Gültig bis ) für Preise angegeben werden. 
### Feld Preisgruppe 
Hier wird die Preisgruppe angegeben

### Feld Rabatt 
Hier wird ein prozentualer Rabatt oder Zuschlag angegeben. Ein positiver Wert bedeutet Rabatt, ein negativer Wert bedeutet Zuschlag.
