# Webshop Registerkarte Warenkörbe

E-Commerce Katalog / Webshop 

# Webshop Registerkarte Warenkörbe 
Webshop Registerkarte Warenkörbe 

Auf dieser Registerkarte werden Warenkörbe aufgelistet. 
### Tabelle 

### Tabellenspalte Auftragsart 
Hier wird die Auftragsart aus dem Warenkorb angegeben. 

Tooltip: 

Eingabe der Auftragsart 

### Tabellenspalte Webshop 
Hier wird die Webshop ID angegeben. 

Tooltip: 

Angabe der Webshop ID 

### Tabellenspalte Warenkorbnr 
Hier wird die Warenkorbnummer angegeben. 

Tooltip: 

Angabe der Warenkorbnummer 

### Tabellenspalte Benutzer 
Hier wird der Webshop-Benutzer angegeben. 

Tooltip: 

Angabe des Webshop-Nutzers 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Firma 2 
Die 2. Adresszeile wird angegeben. 

Tooltip: 

Zeile 2 des Firmennamens. 

### Tabellenspalte Erfassung 
Hier wird vom Programm automatisch das Datum der Erfassung eingetragen. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

### Button Auswählen 

Mit Klick auf diesen Button wird aus dem Warenkorb ein Auftrag bzw. ein Angebot erstellt und es öffnet sich entweder der Dialog Auftrag oder Angebot . 

Die Entscheidung über Auftrag und Angebot wurde vom Benutzer im Webshop getroffen und kann nicht geändert werden. 

### Button Löschen 

Dieser Button löscht einen markierten Warenkorb. 

### Feld Memo 
Dieses Feld zeigt das Memo aus dem Warenkorb an und wird aus dem Webshop übernommen. Dieses Memo wird in den Auftrag oder das Angebot übernommen. 

### Tabelle Positionen 

Diese Tabelle zeigt die Positionen eines markierten Warenkorbs an. Die angezeigten Werte werden immer aktualisiert, können aber nicht geändert werden. 
### Tabellenspalte Pos 
Die Nummer der Position wird angezeigt. 

Tooltip: 

Positionsnummer 

### Tabellenspalte Artikel 
Die Artikelnummer wird angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Postext 
Zu einer Position kann ein bis zu 2000 Zeichen langer Zusatztext eingegeben und in den Formularen gedruckt werden. 

Bitte achten Sie darauf, einen doppelten Druck des Textes in den Formularen zu verhindern, indem Sie nicht den Artikeltext, sondern nur den Positionstext drucken. 

Beachten Sie außerdem, dass alle Änderungen im Dialog Position auch im zugrundeliegenden Vorgang mit dem Button gespeichert werden müssen. 

Tooltip: 

Zusätzlicher Text zur Position 

### Tabellenspalte Menge 
Die Menge des Artikels aus dem Webshop wird angezeigt. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Einheit VK 
Die Verkaufseinheit wird angezeigt. 

Tooltip: 

Verkaufseinheit 

### Tabellenspalte PosBetrag 
Hier wird der Positionsbetrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Bezeichnung 
Der Name des Artikels wird eingetragen. 

Tooltip: 

Artikelbezeichnung

Zusätzlicher Text zur Position
