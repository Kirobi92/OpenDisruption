# Kapitel NV.ERP.Base.Customer.cdCustomer.shtOpenItemCompensation

Wenn Sie diese Checkbox aktivieren, kann der Kunde an der OP-Verrechnung teilnehmen. 

Voraussetzung für eine OP-Verrechnung ist, dass der Kunde im Kundenstamm auf der Registerkarte Allgemein eine Lieferanten- bzw. Kundennummer zugeteilt bekommen hat. 

Siehe Kapitel Verrechnung von offenen Posten .
