# Kapitel NV.ERP.Base.Common.cdParameterFI.shtCashDiscGracePeriod

Hier wird dem Kunden eine zusätzliche Karenzzeit gewährt (Skonto fällig + Anzahl Tage). 

Das Programm ermittelt bei Zahlung innerhalb dieser Karenz den Skonto automatisch.
