# Neue GAP Position 
Neue GAP Position 

Dieser Dialog öffnet sich über den Button Neu in der Gutschrifsabrechnung auf der Registerkarte Allgemein . 

Hier können Sie neue Positionen für die Gutschriftsabrechnung erfassen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Neue Position GAP 
Tooltip: 

In diesem Dialog können Sie neue Positionen für die Gutschriftsabrechnung erfassen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren . 

### Menü Fenster-Information 
Über dieses Menü werden Ihnen Informationen zum Dialog angezeigt. Siehe nachfolgendes Beispiel: 

Meldungen 

### Button Auswahl 

Mit diesem Button übernehmen Sie den Artikel in die Rechnungsprüfung. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Auftrag 
In diesem Feld wird die Auftragsnummer aus dem Dialog Gutschriftsabrechnung angezeigt. 

Tooltip: 

Angabe der Belegnummer 

### Feld Auftragsart 
In diesem Feld wird die Auftragsart aus der Gutschriftsabrechnung angezeigt. 

### Feld Artikel 
In diesem Feld wird die Artikelnummer des markierten Eintrags in der Tabelle der Gutschriftsabrechnung angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Feld Artikel 
In diesem Feld können Sie eine Artikelnummer erfassen, die in die Gutschriftsabrechnung aufgenommen werden soll. 

Tooltip: 

Angabe der Artikelnummer 

### Button Suche 

Mit diesem Button können Sie einen Artikel suchen. 

Siehe Kapitel Artikelauswahl . 

### Feld Bezeichnung 
Bei Eingabe der Artikelnummer wird dieses Feld automatisch mit dem Namen des Artikels gefüllt. 

Tooltip: 

Artikelbezeichnung 

### Feld Menge bestellt 
Hier wird die bestellte Menge eingetragen. 

Tooltip: 

Angabe der Menge, welche bestellt wurde. 

### Feld Mengeneinheit Verkauf 
Hier wird die VK-Mengeneinheit des Artikels angezeigt. 

Tooltip: 

Einheit VK 

### Feld Preis 
Hier wird der Preis eingetragen. 

Tooltip: 

Preis 

### Feld Währungskürzel 
Hier wird das Währungskürzel angezeigt. 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Feld VK pro 
Hier wird der VK pro des Artikels angezeigt. 

Tooltip: 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- je 1 Einheit 
- je 10 Einheiten 
- je 100 Einheiten 

### Feld Mengeneinheit Verkauf 
Hier wird die VK-Mengeneinheit angezeigt. 

Tooltip: 

Einheit VK
