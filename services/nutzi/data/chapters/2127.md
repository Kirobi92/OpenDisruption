# Dublettenprüfung 
Dublettencheck 

Dieser Dialog öffnet sich in der Adresse >> Optionen >> Dublettenprüfung . 

Mit der Dublettenprüfung kann anhand der Prüfgenauigkeit ein Vergleich in Adresse und Kunde auf doppelte Einträge durchgeführt werden. Sie erhalten eine Textdatei, in der alle Dubletten aufgelistet werden. 

In das Feld Von Datum wird das Erfassungsdatum des Adressenstammsatzes eingetragen. 

Wenn die Anwendung aus dem Application-Broker gestartet wird, speichert eNVenta die Datei unter “C:WINDOWSsystem32inetsrv“ (und zwar auf dem Application-Broker). 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Dublettenprüfung 
Tooltip: 

In diesem Dialog können Adressen und Kunden auf doppelte Einträge geprüft werden. 
Für mehr Informationen drücken Sie F1.
