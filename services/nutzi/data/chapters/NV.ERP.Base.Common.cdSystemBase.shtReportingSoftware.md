# Kapitel NV.ERP.Base.Common.cdSystemBase.shtReportingSoftware

Hier können Sie die Reporting-Software (Crystal Reports oder DevExpress) auswählen, die für den Druck verwendet werden soll. Falls DevExpress ausgewählt ist, aber kein DevExpress-Report vorhanden ist, findet ein Fallback auf CrystalReports statt. 

Beispiel: 
Ein typischer Anwendungsfall ist die Übertragung von fertig generierten Lieferscheinen im Rahmen des Intercompany-Prozesses für Streckenlieferungen. Ein weiterer Anwendungsfall findet sich in der Lieferantenerklärung, die beim Rechnungsdruck mitgeliefert wird. Dazu muss eine PDF aus einem Report generiert werden. 

Tooltip:
