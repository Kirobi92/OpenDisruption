# Kapitel NV.ERP.Base.Common.cdSystemMM.shtOfferCompletion

Wenn Sie diesen Parameter aktivieren, wir das Angebot nach dem Speichern auf Status abgeschlossen gesetzt (Registerkarte Abschluss ). Dieser Parameter kann nur gesetzt werden, wenn der Auftragsabschluss aktiviert ist. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, wird das Angebot nach dem Speichern auf Status 'abgeschlossen' gesetzt.
