# RP Positionen bearbeiten Registerkarte Rahmen-Info 
Position Registerkarte Rahmen-Info 

Auf dieser Registerkarte erhalten Sie Informationen bezüglich einer Rahmenvereinbarung. 

Hat eine Bestellposition einen Rahmenvertragsbezug wird der automatisch generierte Rahmenvertragstext aus der Rahmenposition und/oder dem Rahmenabruf separat im Feld Beschreibung auf dieser Registerkarte eingefügt. Zusätzlich werden Informationen aus dem Rahmenvertrag wie Vertragsnummer, Lieferdatum, Abrufmenge und Vertragspreis angezeigt. 

Ist ein Rahmenvertragsbezug vorhanden, sieht man das auch daran, dass die Registerkarte mit einem Sternchen versehen ist. 

WICHTIG: 

Der autogenerierte Text 'Abruf aus RahmenvertragXY' wird zusätzlich nur noch auf den Einkaufsreports ausgegeben. 

Wenn Sie Ihre Reports selbst verwalten, müssen Sie in folgenden Reports das Property sOutlAgreeCallText hinzufügen: 

- rptPurchaseDocumentFree - rptPurchaseDocument - rptGoodsReceipt - rptSalesDocumentFree - rptSalesDocument - rptSalesDocBacklog - rptNV4_PickOrderLabel 
### Feld Vertrag 
Hier wird die Vertragsnummer angegeben. 

Tooltip: 

Angabe der Vertragsnummer 

### Feld Lieferdatum 
Hier wird der Liefertermin angegeben. 

Tooltip: 

Angabe des Liefertermins 

### Feld Termin 
Hierbei handelt es sich um ein Textfeld, in welchem ein beliebiger Hinweistext als "Stichtag" angegeben werden kann. 

Tooltip: 

Textfeld zur Angabe eines Stichtags 

### Feld Menge 
Hier wird die Menge angegeben. 

Tooltip: 

Angabe der Menge 

### Feld EK 
Hier wird der Einkaufspreis angegeben. 

Tooltip: 

Anzeige des Betrags 

### Feld EK pro 
Hier wird der Einkaufspreis je Anzahl Lagereinheiten angeben. Der Einkaufspreis kann sich auf 10, 100, 1000 oder eine andere Anzahl Einheiten des Artikels beziehen. Standardmäßig wird hier eine 1 eingetragen. 

Tooltip: 

Einkaufspreis je Anzahl Lagereinheiten. 

### Feld Bestelltext 
Wenn ein Rahmenvertragsbezug besteht, wird der automatisch generierte Rahmenvertragstext aus der Rahmenposition und/oder dem Rahmenabruf hier angezeigt.
