# Simulation Registerkarte Arbeitsplan 
Registerkarte Arbeitsplan 

Hier werden die Arbeitspläne angezeigt. 
### Tabelle 

### Tabellenspalte AP Pos 
Hier wird die Positionsnummer angegeben. Diese Nummer muss je Vorgang eindeutig sein! 

Tooltip: 

Positionsnummer 

### Tabellenspalte Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Tabellenspalte Datum/Zeit von 
Hier werden ein Datum und eine Uhrzeit angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Tabellenspalte Datum/Zeit bis 
Um die Suche über einen Datumsbereich einzuschränken, wird hier ein Bis-Datum angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit
