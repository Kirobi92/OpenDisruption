# Projekt Registerkarte Anschrift 
Projekt Registerkarte Anschrift 

Auf dieser Registerkarte kann eine beliebige Anschrift zum Projekt hinterlegt werden. Die Anschrift ist frei wählbar, sie muss nicht im Kundenstamm vorhanden sein. 

Sollte die eingegebene Adresse unvollständig sein, werden Sie mit einer Meldung daraufhingewiesen. 

Mit Auswahl aus den Comboboxen Zahlung und Lager lassen sich für das Projekt eine Zahlungsbedingung und ein Lager hinterlegen, die automatisch in den Auftrag übernommen werden. Dies gilt auch bei der Erfassung eines Angebots, einer Gutschrift oder eines Rahmenauftrags. 

Sie können auch einen Kostenträger eintragen / auswählen. Ist ein Kostenträger eingetragen, wird dieser Kostenträger in die Auftragsposition übernommen. Vorhandene Positionen werden auf eine geänderte Projektnummer geprüft und ggf. wird die Kostenträgernummer angepasst. Bei einer manuellen Änderung der Kostenträger bleiben diese Änderungen gespeichert.

### Feld GEO-Koordinaten 
Hier werden geografische Daten zur besseren Orientierung angegeben. Damit lassen sich Besuche o.ä. besser planen und Sie können die Karte zur besseren Orientierung und Festlegung einer Besuchs-Reihenfolge nutzen. 

Das Format der GEO-Koordinaten lautet: 
XX.yyyyyy,AA.bbbbbb 

Beispiel für Stockach, Nissen&Velten:

### Feld USt-IdNr. 
Hier wird die Umsatzsteuer-Identifikationsnummer des Kunden angegeben.

### Feld Kreditlimit 
Hier kann ein Standard-Kreditlimit für den Kunden (in System-Währung) festgelegt werden. Falls der Betrag überschritten wird, erhalten Sie eine Meldung, wenn Sie einen Auftrag zu diesem Kunden erfassen. Sie erhalten die Möglichkeit, eine Begründung für die Überschreitung einzugeben. 

Die absoluten Kopfrabatte werden in der Kreditlimitprüfung nicht berücksichtigt. 

Das Kreditlimit ist immer in "Haus-Währung" anzugeben!

Zeigt das aktuelle Kreditlimit des Projektes.

### Checkbox Top Projekt 
Wenn Sie diese Checkbox aktivieren, wird das Projekt als Top-Projekt gekennzeichnet. D.h. es handelt sich um ein Projekt, das vom Sachbearbeiter besondere Aufmerksamkeit erfordert, bzw. gewährleistet eine genaue Beobachtung des Projektfortschritts. 

Ein Projektbezug kann in folgenden Dialogen angegeben werden: 

- Projektänderung - Angebot, Auftrag, Gutschrift - Kondition - Bestellung, Anfrage, Belastung - Rahmenauftrag - Call - Besuchsbericht - Projektaufgabe - Service - Wiedervorlage zum Projekt 
Sobald in einem dieser Dialoge mit Bezug auf ein Projekt eine Modifikation stattfindet, wird das aktuelle Datum im Projekt in das Feld letzte Aktivität geschrieben. 

Modifikation bedeutet, dass z.B. bei einem Auftrag mit Projektbezug ein Lieferschein gedruckt wurde, aber auch, dass im Auftragskopf ein Lieferdatum modifiziert wurde. 

Tooltip: 

Aktivitäten verfolgen in: 
- Projektänderung 
- Angebot / Auftrag / Gutschrift 
- Kondition 
- Bestellung / Anfrage / Belastung 
- Rahmenauftrag 
- Call 
- Besuchsbericht 
- Projektaufgabe 
- Service 
- Wiedervorlage zum Projekt
