# Tableauverwaltung Registerkarte Allgemein

Lagerverwaltungssystem (LVS) / Tableauverwaltung 

# Tableauverwaltung Registerkarte Allgemein 
Tableauverwaltung Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie ein Tablar im Einzelnen. 
### Button Neu 

Dieser Button öffnet den Dialog Neue Position , um eine neue Position anzulegen. Mit der neuen Tableau-Position wird ein tatsächlicher Lagerplatz in der Zone angelegt. 

Siehe Kapitel Neue Position in der Tableauverwaltung . 

Tooltip: 

Neuer Lagerplatz auf dem Tablar. 

### Button Löschen 

Dieser Button löscht eine markierte Zeile der Tabelle. 

Tooltip: 

Position löschen 

### Button Drehen 

Mit diesem Button kann der ausgewählte Lagerplatz auf dem Tableau gedreht werden. 

Tooltip: 

Drehen des Lagerplatzes auf dem Tableau. 

### Button Umlagerung 

Dieser Button öffnet den Dialog Umlagerung zu einem markierten Eintrag der Tabelle. 

Siehe Kapitel Umlagerung in der Tableauverwaltung . 

### Tabelle Tablarpositionen 

Diese Tabelle enthält die Informationen entsprechend der Selektionskriterien. 
### Tabellenspalte Tablarpos 
In dieser Spalte wird die Positionsnummer des Lagerplatzes angezeigt. 

### Tabellenspalte LHM Typ 
In dieser Spalte wird der Ladehilfsmittel-Typ angegeben. 

Tooltip: 

Ladehilfsmittel-Typ. 

### Tabellenspalte Start Position X 
In dieser Spalte wird die Startposition der X-Koordinate auf dem Tableau des Lagerplatzes in der Einheit der System Parameter angegeben. 

### Tabellenspalte Start Position Y 
In dieser Spalte wird die Startposition der Y-Koordinate auf dem Tableau des Lagerplatzes in der Einheit der System Parameter angegeben. 

### Tabellenspalte Breite 
In dieser Spalte wird die Breite des Lagerplatzes auf dem Tableau in der Einheit der System Parameter angegeben. 

Tooltip: 

Breite des Tablars in der Einheit der System-Parameter 

### Tabellenspalte Länge 
In dieser Spalte wird die Länge des Lagerplatzes auf dem Tableau in der Einheit der System Parameter angegeben. 

Tooltip: 

Länge des Tablars in der Einheit der System-Parameter 

### Tabellenspalte Höhe 
In dieser Spalte wird die Höhe des Lagerplatzes auf dem Tableau in der Einheit der System Parameter angegeben. 

Tooltip: 

Angabe der Höhe 

### Combobox Überlappung 
Diese Combobox ermöglicht die Verschiebung eines LHMs nach links, rechts oben oder unten, um die Überlappung von 2 LHMs aufzulösen. 

- oben - nieder - links - rechts 

Tooltip: 

Lagerplatz wird auf den nächsten freien Platz in der ausgewählten Richtung verschoben. 

### Button Verschieben 

Dieser Button führt die zuvor angegebene Aktion aus. 

Tooltip: 

Position verschieben
