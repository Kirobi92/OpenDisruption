# Import Editor für Artikel Registerkarte Rabatt 2 
Import Editor - Artikel Registerkarte Rabatt 2 

Auf dieser Registerkarte werden weitere Rabatt-Informationen der Import Daten und des Artikelschattenstamms nebeneinandergestellt. Dabei geht es um die Pflege der Felder Menge / VP 1-5 in der Rabattverwaltung Einkauf auf der Registerkarte Verpackungseinheiten . Die Daten können verglichen und angepasst werden. 

Beim Datenimport werden diese Felder dann automatisch gefüllt. 
### Feld Menge / VP (R) 
Es ist möglich, für ein und denselben Artikel unterschiedliche Mengeneinheiten für das Lager und den Verkauf zu führen. Beispielsweise können Sie einen Artikel in Stück am Lager führen, jedoch kistenweise verkaufen. Der Umrechnungsfaktor von der Lagereinheit zur Verkaufseinheit wird in das Feld Faktor eingetragen. Standardmäßig enthält das Feld den Wert 1 (Lagereinheit = Verkaufseinheit). 

Beispiel: 
Lager = 1 Kanister mit 100 kg, Verkauf = 1 kg, Faktor = 0,01. 

Die beiden Felder Einheit Verp. und Menge / VP müssen gemeinsam mit Werten gefüllt werden. Falls eines der Felder nicht gefüllt wird, erscheint eine Meldung.

### Feld Gültig ab (ERG) 
ERG = Einkauf Rabatt Gültigkeit 

Diese Spalte wird nur angezeigt, wenn es auch einen Wert gibt. Die Spalte enthält die Einkauf Rabatt Gültigkeit für den Import. 

Tooltip: 

Geben Sie die Einkauf Rabatt Gültigkeit an. 

### Feld Beginn der Gültigkeit
