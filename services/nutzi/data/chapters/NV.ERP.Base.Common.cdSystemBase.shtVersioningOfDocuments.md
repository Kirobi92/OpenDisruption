# Kapitel NV.ERP.Base.Common.cdSystemBase.shtVersioningOfDocuments

Wenn Sie diesen Parameter setzen, wird die Beleg-Versionierung in eNVenta aktiviert und im Dialog Einzeldruck der Button Version. angezeigt, um die Parametrisierung der Beleg-Versionierung vornehmen zu können. 

Bei der Beleg-Versionierung erfolgt pro Druck, je nach Parametrisierung, die Erstellung einer neuen Druckversion. Bei aktiver Archivierung wird jede Druckversion archiviert. 

Siehe Kapitel Konfiguration von Versionierungen . 

Tooltip: 

Aktiviert die Versionierung von Druckbelegen 
(Verkauf, Einkauf, Retoure)
