# Gutschrift für eine Retoure kann noch vor Wareneingang der Retoure erstellt werden!

Verkauf 

# Wareneingang in der Retoure buchen 
Wareneingang buchen 

Wenn die Retoure bereits in die Datenbank eingefügt ist und wenn mindestens 1 Position noch nicht im Wareneingang gebucht ist, öffnet sich über den Button Eingang im Kopf des Dialogs Retoure der Dialog Wareneingang buchen . 

Der Wareneingang kann ein- oder zweistufig gebucht werden. 

Bei einer 2-stufigen Wareneingangsbuchung muss das QS-Lager angegeben werden. 

Mit Klick auf den Button ungeprüft wird eine Lagerbuchung in das gewählte QS-Lager vorgenommen und der Dialog automatisch geschlossen. In der Position wird bei den entsprechenden Positionen das Kennzeichen ‚QS‘ gesetzt. 

Mit Klick auf den Button geprüft wird ebenfalls eine Lagerbuchung vorgenommen und der Dialog geschlossen. Die Ware wird bei QKZ 1 auf das gewählte Lager der Position gebucht und bei einer QKZ größer 1 auf das gewählte Nachbearbeitungslager. 

### Gutschrift vor Wareneingang der Retoure erstellen 
D.h. nach Erstellung einer Gutschrift ist es möglich die 2. Stufe im Wareneingang zu buchen. Dies erfolgt wie üblich auf das Warenlager oder das Nachbearbeitungslager. Dazu prüft eNVenta bei Erstellung der Gutschrift, ob es sich bei der Buchung auf ein Lager, um eine einstufige oder zweistufige Buchung handelt und welche Stufe bereits gebucht wurde. Wurde erst die Stufe 1 gebucht, kann das Buchen der Stufe 2 neu auch erfolgen, nachdem die Gutschrift erstellt wurde. In beiden Fällen wird die Retoure in den Status beendet gesetzt, nachdem die Gutschrift erstellt oder der Button Beenden gedrückt wurde. 

Bei Klick auf den Button Beenden erfolgt eine Abfrage, falls der Status QS gesetzt ist. Dadurch wird verhindert, dass der Wareneingang ohne Stufe 2 abgeschlossen wird. Der Wareneingang setzt dann den Status der Position auf erledigt . 

Bei Erstellung der Gutschrift wird der Status der Retourenposition auf beendet gesetzt, wenn noch der QS Status gesetzt ist (statt auf erledigt ). 

Im Wareneingang können Sie noch auf den Eingangsbutton klicken, wenn der Status beendet oder erledigt ist und der QS-Status gesetzt ist. 

Weiterhin wird nun im Wareneingang der Stufe 2 der Beleg so abgeschlossen, wie es die Gutschriftserzeugung gemacht hätte. D.h. Positionen im Status beendet und mit Gutschrift werden in Status erledigt verschoben und wenn alle Positionen im Status erledigt sind, wird die Retoure abgeschlossen (Kopf-Status 3).
