# Gutscheinverwaltung Registerkarte Allgemein

Kasse / Gutscheinverwaltung 

# Gutscheinverwaltung Registerkarte Allgemein 
Gutscheinverwaltung Registerkarte Allgemein 

Auf dieser Registerkarte werden die Informationen entsprechend Ihrer Selektionen angezeigt. 
### Tabelle 

### Tabellenspalte Gutschein 
Hier wird die Nummer des Gutscheins angegeben. 

Tooltip: 

Angabe der Gutscheinnummer 

### Tabellenspalte Gutscheinart 
Hier wird die Art des Gutscheins angegeben. 

Tooltip: 

Angabe der Gutscheinart 

### Tabellenspalte Status 
In dieser Spalte wird der Status des Gutscheins eingetragen. So können Sie sehen, ob der Gutschein aktiviert ist oder evtl. schon teilweise ausgeschöpft. 

### Tabellenspalte Aktivierer 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der den Gutschein aktiviert hat. 

### Tabellenspalte Aktivierungsdatum 
In dieser Spalte wird das Aktivierungsdatum angezeigt. 

### Tabellenspalte Währung 
Hier wird das ISO-Kürzel des international standardisierten Währungskürzels angegeben, da dieses für den Zahlungsverkehr verwendet wird. 

Tooltip: 

Angabe des ISO-Kürzels 

### Tabellenspalte Initialwert 
Hier wird der Initialwert aus dem Gutschein angezeigt. 

Tooltip: 

Wert des Gutscheins bei Erstellung 

### Tabellenspalte Restwert 
Hier wird der Restwert aus dem Gutschein angezeigt. 

Tooltip: 

Wert des Gutscheins nach Verwendung 

### Tabellenspalte Ersteller 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der den Gutschein erstellt hat. 

### Tabellenspalte Abschlussdatum 
Hier wird das Datum eingetragen, an dem der Gutschein ungültig wird. 

### Tabellenspalte Gelöscht 
Hier wird das Gelöscht-Datum angezeigt. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl 
Hier wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt. 

Tooltip: 

Anzahl
