# Kapitel NV.ERP.Base.Common.cdParameterFI.shtCreateOpenItemTaxSetup

In diesem Bereich können Steuerschlüssel (Inland, EU-Ausland und Nicht-EU-Länder) für die Anlage offener Posten für Debitoren-/Kreditoren-Ausgleich und Debitoren-/Kreditoren-Rückzahlung als Standard hinterlegt werden. 

Diese Felder können auch leer bleiben! 

Wollen Sie die alte Logik, Steuerschlüssel aus dem letzten OP übernehmen / beibehalten, so wählen Sie den entsprechenden Radiobutton.
