# Kapitel NV.ERP.Base.Common.cdPaymentCondCustomer.shtActualDays3

Hier können bis zu 3 Skonto-Fristen und Zeiträume innerhalb einer Zahlungsbedingung festgelegt werden. 

In Tage 1, Tage 2, Tage 3 legen Sie die Anzahl Tage fest, wann der Beleg Skonto-fällig wird. Z.B. Tage 1 –7-, Tage 2 –10-, Tage 3 –14-. (Formel: Belegdatum + Tage 1 + Tage 2 + Tage 3). 

Wenn Sie ein Datum über 'Tag d. Monats" eingeben, ist in 'Tage1' zwingend eine 0 einzugeben. 

Tooltip: 

Geben Sie für die Skontofrist eine Anzahl an Tagen ein.
