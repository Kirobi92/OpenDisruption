# Neuer Referenz-Link 
Neuer Referenz-Link 

Dieser Dialog öffnet sich über den Button Neuer Link im Dialog Referenz Links . Hier können Sie weitere Links anlegen bzw. erhalten eine Übersicht über bereits gespeicherte Links. Dafür stehen die beiden Registerkarten zur Verfügung. 

Verlinkung von Elementen: 
Wenn Sie z.B. zu einem Artikel als Referenz eine Datei setzen möchten, gehen Sie wie folgt vor: 

- Artikel im Artikelstamm aufrufen - Button Referenz-Link drücken - es öffnet sich der Dialog Neuer Referenz-Link - in diesem Dialog rechts den Button Url drücken - Url-Pfad eingeben - Dateityp festlegen (z.B. ein Word-Dokument .doc) - mit dem Button einfügen 
Nun ist das Dokument mit dem Artikel verlinkt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Neuer Referenz-Link 
Tooltip: 

In diesem Dialog können Sie weitere Links anlegen bzw. erhalten eine Übersicht über bereits gespeicherte Links. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
