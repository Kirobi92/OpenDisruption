# Bestellwertrabatte 
Bestellwertrabatte 

Dieser Dialog öffnet sich mit dem Button B-Wertrabatte im Kopf des Dialogs Lieferant oder mit dem Button Bestellwertrabatte in der Disposition . Hier können Sie Frachtfreigrenzen, Mindermengenaufschläge und Rabatte zuordnen. Die Daten hierfür werden in die Kopfrabatte der Bestellung übertragen, sofern angegeben. 

Sie können zu einem Lieferanten eintragen, ab welchem Wert (Spalte 'Betrag von') ein Frachtaufschlag, ein Mindermengenaufschlag und / oder ein Rabatt zugeordnet werden soll. 

Aus der Bestellwertetabelle wird nur ein Wert also Mindermengenzuschlag, Rabatt oder Frachtzuschlag übernommen, wenn diese in unterschiedlichen Einträgen in der Tabelle sind. Es wird der nächst kleinere Mindermengenzuschlag genommen, der von der Summe aller Bestellpositionen nicht überschritten wird. 

Dazu ein Beispiel: 
Nehmen wir an zu einem Lieferanten sieht die Bestellwertrabatte wie folgt aus: 

Lieferant Betrag von Frachtaufschlag Mindermengenzuschlag Rabatt 
| | | |
| --- | --- | --- |
| xyz | 0,1 | -50 |
| xyz | 30 | 20 |
| xyz | 50 | |

Nun wird eine Bestellung für den Lieferanten erfasst. Der Wert der erfassten Position beträgt ohne Kopfrabatte und andere Zuschläge, die noch auf dem Beleg gesetzt werden können, 25. Hier wird nun geprüft, welcher Wert bei der Neukalkulation/ Erfassung übernommen werden muss. Das verwendete Schema lautet: ist 25 >= 50 nein, ist 25 >= 30 nein, ist 25 >= 0,1 Ja -> In die Kopfdaten (Registerkarte Kopfdaten ) wird -50 in den 1. Kopfrabatt geschrieben. 

Der Frachtzuschlag wird nicht übernommen, auch wenn der Betrag kleiner 30 ist. 

Bei einer Bestellung im Wert von 35 geht das System folgendermaßen vor: 
Ist 35 >= 50 nein, ist 35 >= 30 Ja, wird wird 20 in die Kopfdaten, Feld Fracht geschrieben. Hier wird die -50 aus dem Mindermengenzuschlag nicht mit reingeschrieben, da schon vorher ein passender Eintrag gefunden wurde, der die Bedingung erfüllt hat. 

Um die beiden Einträge aus der Tabelle übernehmen zu können, müssten die Einträge inder Tabelle wie folgt aussehen: 

Lieferant Betrag von Frachtzuschlag Mindermengenzuschlag Rabatt 
| | | | | |
| --- | --- | --- | --- | --- |
| xyz | 0,1 | 25 | -50 | |
| xyz | 30 | 20 | -25 | 20 |
| xyz | 50 | | | |

Hier wird jetzt für die 25 Bestellwert wie oben 25 in das Feld Fracht geschrieben und -50 in den 1. Kopfrabatt. Beim Bestellwert von 30 wird 20 in das Feld Fracht geschrieben, -25 in den 1. Kopfrabatt und 20 in den 2. Kopfrabatt.

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken.

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
