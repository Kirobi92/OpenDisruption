Allgemeine Funktionalitäten in eNVenta ERP / Datentransfer (Import und Export von Daten) 

# Server Datei Auswahl 
Server Datei Auswahl 

Dieser Dialog öffnet sich 
- im Dialog Datenimport über den Button Laden , - im Dialog Artikelbildverwaltung über den Button neben dem Feld Exportpfad - und in den Kasseneinstellungen auf der Registerkarte Datenübergabe über den Button . 
Hier können Sie eine Server-Datei oder einen Pfad auswählen und in den zugrundeliegenden Vorgang übernehmen. 

Nach Auswahl einer Verbindung werden die Dateien und Verzeichnisse unten aufgelistet. 

### Java-Erweiterungen 
Die aufgerufene URL wurde um einen weiteren optionalen Parameter erweitert, mit welchem der zu verwendende Dateiauswahldialog definiert werden kann. 

Um den Java-Dialog weiterhin verwenden zu können geben Sie als Java-Argument die Option „DfileSelectOld“ an. 

Client Launcher Configuration 

Mit Hilfe des Parameters kann zwischen dem Dateiauswahl-Dialog des Betriebssystems (Default) und dem „alten“ Java-Dialog unterschieden werden. 

Achtung: Diese Funktion ist nicht für den Dauereinsatz gedacht. Sollten Sie auch den alten Dialog noch nutzen, stellen Sie bitte die Aufrufstellen entsprechend um.

### Button Öffnen 

Mit diesem Button kann direkt der Dialog zur Verwaltung der Verbindungen aufgerufen werden ( Verbindungsmanager ). 

Der Button ist nur sichtbar, wenn der Benutzer auch die entsprechenden Speicherrechte für den Dialog hat. 

Beispiel: 
Der Sachbearbeiter befindet sich in einem Ordner, von dem er zwei Dateien gern direkt im Dialog Server Datei Auswahl hochladen möchte. Damit er nicht, wie beim Hochladen-Button, erneut zu seinen Dateien navigieren muss, markiert er beide Dateien, zieht sie auf die Tabelle und lässt sie „fallen“. Beide Dateien werden direkt hochgeladen. 

Tooltip:

### Feld Dateiname 
Hier geben Sie einen Dateinamen ein. 

Um die Auswahl der Datei komfortabler zu machen bzw. bei einer sehr großen Menge an Dateien die Suche zu beschleunigen, kann in dieses Feld eine Zeichenkette gefolgt von * eingetragen werden. Wenn Sie das *-Symbol eingegeben haben und Enter drücken, wird automatisch an die richtige Stelle in der Tabelle gescrollt und nur Dateien mit dem Suchbegriff werden angezeigt. 

Mit dem *-Symbol kann die Suche gestartet werden. Z.B. *.jpg findet alle Bilddateien mit dieser Endung im Verzeichnis und listet diese auf. Durch Leeren des Felds Dateiname wird der Filter zurückgesetzt. 

Tooltip: 

* als Platzhalter zum Filtern der Dateien benutzen
