# Serviceübersicht Registerkarte Allgemein 
Serviceübersicht Registerkarte Allgemein 

Auf dieser Registerkarte werden alle Serviceeinsätze angezeigt, die den Selektionskriterien entsprechen. 
### Tabelle 

### Tabellenspalte Einsatz 
Hier wird die Nummer des Serviceeinsatzes angegeben. 

Tooltip: 

Angabe der Einsatznummer 

### Tabellenspalte Bezeichnung 
Hier wird der Name der Einsatzart angegeben. 

Tooltip: 

Angabe des Namens der Einsatzart 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Datum/Zeit von 
Hier werden ein Datum und eine Uhrzeit angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Tabellenspalte Datum/Zeit bis 
Um die Suche über einen Datumsbereich einzuschränken, wird hier ein Bis-Datum angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Tabellenspalte Monteur 
Hier wird der zuständige Monteur angezeigt. 

### Tabellenspalte Serviceobjekt 
Hier wird die Nummer des Serviceobjekts angegeben. 

Tooltip: 

Angabe der Serviceobjekt-Nummer 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein.
