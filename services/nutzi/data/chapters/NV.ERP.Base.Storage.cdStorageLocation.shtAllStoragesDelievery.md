# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtAllStoragesDelievery

Ist dieser Parameter gesetzt, werden alle Zuordnungen in der Tabelle Lieferlager ignoriert. 

Alle vorhandenen Läger sind dann automatisch Bezugsläger für dieses Lager. 

Ausnahme: Sperrläger.
