# Interne Gutschriftssuche 
Gutschriftssuche (intern) 

Der Dialog Gutschriftssuche (intern) öffnet sich in der Gutschriftsabrechnung über den Button neben dem Feld Gutschrift, int . 

Hier können interne Gutschriften (der Kunde bekommt Geld) ausgewählt und in eine Gutschriftsabrechnung übernommen werden. Dies kann erfolgen, wenn der Kunde eine bestehende Gutschrift in Abzug bringen möchte. 

Es können hierbei nur Gutschriften ausgewählt werden, welche noch nicht fakturiert wurden. Sonst könnte es passieren, dass der Kunde das Geld doppelt verrechnet bekommt. Bei der Faktura der Gutschriftsabrechnung wird hierbei die interne Gutschrift automatisch mit fakturiert. 
## Steuerelemente des Formulars

Tooltip: 

In diesem Dialog können interne Gutschriften (der Kunde bekommt Geld) ausgewählt und in eine Gutschriftsabrechnung übernommen werden.

### Tabellenspalte USt-IdNr. 
Hier wird die Umsatzsteuer-Identifikationsnummer des Kunden angezeigt. 

Tooltip: 

Umsatzsteuer-Identifikationsnummer 

Ist die Funktion "System->Parameter->USt-IdNr. via WS prüfen" aktiviert, so 
wird die Umsatzsteuer Identifikationsnummer bei der Eingabe in den Dialogen 
Kunde, Lieferant, Angebot, Auftrag, Gutschrift, Bestellung, etc. per 
WebService geprüft. 

Achtung: Diese Prüfung ist rechtlich nicht bindend. Eine rechtlich bindende 
Prüfung der Umsatzsteuer Identifikationsnummer kann nur schriftlich bei der 
zuständigen Behörde erfolgen.
