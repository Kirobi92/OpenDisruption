# Manuelle Umbuchungen in der Gesellschaftsverwaltung 
Die Gesellschaftsverwaltung für eine gemeinsame Warenwirtschaft steht nur zur Verfügung, wenn die entsprechende Lizenz freigeschaltet ist! 

Eine manuelle Umbuchung von Lager auf Lager wird nicht unterstützt! 

Dies muss über entsprechende Umlageraufträge realisiert werden. 

Manuelle Umbuchungen innerhalb eines Lagers sind von der internen Verrechnung ausgeschlossen.
