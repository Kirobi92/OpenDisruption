# Kontoauswahl Registerkarte Sachkonto 
Kontoauswahl Registerkarte Sachkonto 

Im Buchen auf der Registerkarte Allgemein öffnet sich die Kontoauswahl automatisch, wenn bei Eingabe von Buchstaben bzw. Ziffern in den Feldern Soll und Haben und anschließendem Klick auf die Enter -Taste, mehrere Treffer möglich sind. 

Auf dieser Registerkarte können Sie nach Sachkonten suchen. 
### Feld Konto 
Hier geben Sie für die Suche die Kontonummer eines Sachkontos ein. 

Tooltip: 

Das Konto zum Datensatz. 

### Feld Kontobez. 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. Wenn Sie das %-Zeichen voranstellen, kann nach einem zusammenhängenden Teil der Kontobezeichnung gesucht werden. 

Tooltip: 

Kontobezeichnung 

### Combobox Jahr 
Hier stehen offene Geschäftsjahre zur Auswahl. 

Begründung: Je Sachkonto und Jahr können unterschiedliche Daten im Konto vorhanden sein. 

### Feld Multisuche 
Über dieses Feld lässt sich die Suche erweitern. Dazu muss allerdings in den Parametern der Finanzbuchhaltung im Bereich Kontensuche nach mindestens eine der Checkboxen Teil d. Kontonr. , Firma1 , Firma2 , Firma3 oder Suchname aktiviert sein. 

Siehe Kapitel Parameter Fibu Registerkarte Allgemein . 

Tooltip: 

Suche gemäß Parameter 

### Checkbox Beliebig 
Wird diese Checkbox gesetzt, wird nach einem Datensatz gesucht, bei dem an beliebiger Stelle in dieser Position - innerhalb der „Fundstelle“ - diese Eingabe vorhanden ist. 

Tooltip: 

Suche an beliebiger Position innerhalb der Fundstelle. 

### Combobox Kontoklasse 
Hier können Sie eine Kontoklasse zur Selektion auswählen. 

Tooltip: 

Angabe der Kontoklasse 

### Combobox Steuerart 
Aus dieser Combobox können Sie eine Steuerart auswählen. 

### Combobox Währung 
Zur Selektion kann hier eine Währung ausgewählt werden. 

Tooltip: 

Angabe der Währung 

### Combobox Niederlassung 
Hier können Sie eine Niederlassung auswählen. Dann erfolgt die Suche niederlassungsbezogen. 

Tooltip: 

Angabe der Niederlassung
