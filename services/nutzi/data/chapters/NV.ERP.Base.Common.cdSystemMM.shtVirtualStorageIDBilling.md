# Kapitel NV.ERP.Base.Common.cdSystemMM.shtVirtualStorageIDBilling

Hier geben Sie ein Lager für das Lieferschein-Identitätspreisverfahren ein. Durch Setzen dieses Systemparameters erfolgen die Ermittlung des (lagerspezifischen) GEK’s und die bestands- und wertmäßige Reduzierung schon beim Lieferscheindruck. 

Damit das funktioniert, ist zwingend die Checkbox Lief. Identität zu setzen! 

Siehe Kapitel Lieferschein Identitätspreisverfahren . 

Tooltip: 

Das Lager, das Sie hier auswählen, definiert das Lager für das Lieferschein-Identitätspreisverfahren.
