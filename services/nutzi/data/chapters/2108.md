# Aktivität bearbeiten Registerkarte Eigenschaften 
Aktivität bearbeiten Registerkarte Eigenschaften 

Auf dieser Registerkarte definieren Sie die einzelnen Aktivitäten der Aktionsschritte. Diese können hier auch geändert werden. 
### Combobox Aktivitätstyp 
In dieser Combobox kann festgelegt werden, um was für einen Kontakt es sich handelt. Es gibt sechs verschiedene Kontaktarten (Postsendung, Fax, E-Mail, Telefonat, Besuch und Intern). Je nach zugewiesener Art stehen bei der Ausführung der Aktivität entsprechende Funktionen zur Verfügung. 

Telefon in der Aktivität ist zugänglich 
| 1. Postsendung: | ermöglicht die Erstellung eines Serienbriefs |
| --- | --- |
| 2. Fax: | ermöglicht die Erstellung eines Serienfaxes |
| 3. E-Mail: | es kann eine Serien-E-Mail erstellt werden |
| 4. Telefonat: | es soll eine Telefonaktion durchgeführt werden; der Button |
| 5. Besuch: | bei der Aktivität handelt es sich um einen Besuch des Teilnehmers |
| 6. Intern: | es erfolgt eine interne Aktivität |

Tooltip: 

Angabe des Aktivitätstyps 

### Feld Nachfolgeaktion 
Hier geben Sie die Nachfolgeaktion ein. 

Tooltip: 

Angabe einer Aktionsnummer 

### Button Suche 

Über diesen Button können Sie eine Aktion suchen. 

Siehe Kapitel Aktionsauswahl . 

### Feld Bezeichnung 
Hier wird die Bezeichnung der Nachfolgeaktion angezeigt. 

Tooltip: 

Name der Aktion 

### Combobox Werbemittel 
Der Werbemittelcode aus den Vorlagedaten der Aktion wird übernommen und kann hier ggf. geändert werden. Neben dem Werbemittelcode wird auch die Bezeichnung als zusätzliche Information angezeigt. 

Ist ein Kunde Teilnehmer an einer Vertriebsaktion und wurde für ihn eine bestimmte Aktivität eingerichtet, kann bei der Auftragserfassung automatisch ein Werbemittelcode zugeordnet werden. 

Die Zuordnung erfolgt unabhängig vom Status der Aktivität. 

Tooltip: 

Angabe des Werbemittelkennzeichens 

### Combobox Nächster Schritt 
Hier kann die Folgeaktivität des aktuellen Aktionsschrittes hinterlegt werden. 

Tooltip: 

Nummer der Aktivität 

### Combobox Zuständig 
Hier kann der zuständige Benutzer für die Aktivität hinterlegt werden. Dieser steht als Suchkriterium auf der Registerkarte Teilnehmer in der Aktion zur Verfügung. 

Tooltip: 

Angabe des zuständigen Sachbearbeiters 

### Feld Vorlage 
Hier kann die Nummer einer Dokumentvorlage eingetragen werden. 

Tooltip: 

Geben Sie die Nummer der Vorlage an. 

### Button Suche 

Mit diesem Button kann eine Datei an die Aktivität angehängt werden, z.B. Anhang für E-Mail. Es öffnet sich der Dialog Vorlagenauswahl . 

Siehe Kapitel Dokument erstellen (Vorlagenauswahl) . 

Tooltip: 

Vorlagenauswahl 

### Button Verzweigen 

Mit diesem Button gelangen Sie in einen Dialog, welcher Ihnen je nach ausgewähltem Kontakttyp die entsprechende Vorlagenauswahl anbietet. 

Tooltip: 

Vorlage Bearbeiten 

### Button Löschen 

Mit diesem Button wird die eingetragene Vorlage gelöscht. 

Tooltip: 

Vorlage Entfernen 

### Button Öffnen 

Dieser Button öffnet das Menü Dokument-Vorlagen , um eine neue Vorlage direkt hier anlegen zu können. 

Siehe Kapitel Vorlagenstamm . 

Tooltip: 

Dokument Vorlagen 

### Feld Beschreibung 
In diesem Feld wird der Typ der Datei angezeigt. 

Tooltip: 

Geben Sie die Beschreibung des Typs ein. 

### Feld Vorlagenart 
Dieses Feld informiert Sie über die Art der Vorlage, die Sie ausgewählt haben. 

Tooltip: 

Art der Vorlage 

### Feld Versandkosten 
Bei diesen Kosten handelt es sich um durchschnittliche Versandkosten pro Teilnehmer. Diese stehen in der Aktion für die Kalkulation zur Verfügung. 

Tooltip: 

Durchschnittl. Versandkosten pro Teilnehmer 

### Feld Materialkosten 
Hier werden die durchschnittlichen Materialkosten pro Teilnehmer angegeben. Diese stehen in der Aktion für die Kalkulation zur Verfügung. 

Tooltip: 

Durchschnittl. Materialkosten pro Teilnehmer 

### Feld Abwickl.Kosten 
Hier werden die durchschnittlichen Kosten pro Teilnehmer angegeben. Diese stehen in der Aktion für die Kalkulation zur Verfügung. 

Tooltip: 

Durchschnittliche Kosten pro Teilnehmer 

### Feld Teilnehmer-% 
Hier werden durchschnittliche Kosten pro Teilnehmer in % angegeben. Diese stehen in der Aktion für die Kalkulation zur Verfügung. 

Tooltip: 

Durchschnittliche Kosten pro Teilnehmer in % 

### Checkbox Wiedervorlage aktivieren 
Mit diesem Parameter kann eine automatische Terminierung der Nachfolgeaktivität in die Wiedervorlage gelegt werden. 

Tooltip: 

Aktiviert automatische Terminierung der Nachfolgeaktivität 

### Feld Tage 
Wenn Sie die Checkbox Wiedervorlage aktivieren gesetzt haben und hier eine Anzahl an Tagen eingeben, wird nach Verstreichen dieser Tage eine Wiedervorlage generiert. 

Tooltip: 

Angabe der Tage, wann eine Wiedervorlage generiert werden soll. 

### Combobox An Sachbearb 
In diesen Feldern wird alternativ festgelegt, für welchen Sachbearbeiter der Termin in der Wiedervorlage gelten soll. Ist keines der Felder gefüllt, wird ein genereller Termin generiert. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Combobox An Abteilung 
In diesen Feldern wird alternativ festgelegt, für welche Abteilung der Termin in der Wiedervorlage gelten soll. Ist keines der Felder gefüllt, wird ein genereller Termin generiert. 

Tooltip: 

Zeigt die Abteilung, die die Wiedervorlage erhält.
