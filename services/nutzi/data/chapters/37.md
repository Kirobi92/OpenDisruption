# Anpassung eNVenta Layout 
Mit eNVenta haben Sie die Möglichkeit die Anwendung an Ihre Bedürfnisse anzupassen (Ein- und Aus-Sliden von Dialogen). Wenn Sie einen breiten Monitor oder auch 2 Monitore verwenden, können Sie eNVenta so anpassen, dass Sie alle notwendigen Informationen auf einen Blick sehen. 

Mit dem Klick auf die rechte Maustaste auf den Titel eines Dialogs öffnen Sie das Untermenü, um die Dialoge an eine andere Stelle zu docken oder auch mittels der Option In neuen Fenster öffnen komplett aus eNVenta loszulösen – die Anzeige auf zwei parallelen Bildschirmen ist damit möglich: Auf einem Bildschirm sehen Sie den Auftrag und auf einem zweiten Bildschirm sehen Sie den Artikelstamm, um weitere Informationen für die schnelle Artikelrecherche zu erhalten. 

Eine weitere Option zur Optimierung der Oberfläche ist die Möglichkeit einen Dialog automatisch ein- und ausblenden zu lassen. In diesem Fall werden die Dialoge nur angezeigt, wenn Sie mit der Maus auf das entsprechende Symbol bzw. den Dialogtitel zeigen. 

Durch einen Klick auf die Pinnadel in einem Dialog bzw. durch die rechte Maustaste auf eine Registerkarte können Sie Dialoge automatisch ein- und ausblenden bzw. sliden lassen. 

Dies bedeutet, erst wenn die Maus auf dem Dialog bzw. Symbol ist erscheint der Dialog.
