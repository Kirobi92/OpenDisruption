# Nachträgliche Erstellung einer Zahlungsmitteilung 
Um einem Kunden bzw. Lieferanten mitzuteilen, welche Zahlung wie verwendet wurde, kann dem Kunden/Lieferanten eine Mitteilung gesendet werden. Diese enthält eine Aufstellung ausgebuchter OPs. Diese Zahlungsverwendung kann auch nachträglich erstellt werden. Dafür stehen in den Dialogen Konto-Info, Journal und Buchen weitere Buttons zur Verfügung. 

Ob im Druckdialog die Radiobuttons Email und Faxen sichtbar sind, hängt davon ab, ob dem Kunden eine E-Mail-Adresse oder eine Faxnummer hinterlegt ist. 

Folgende Reihenfolge wird angewendet: 

- E-Mail-FiBu/Fax-FiBu - E-Mail/Fax-Nr. und - E-Mail Avis/FaxAvis. 
Auf der Zahlungsmitteilung wird differenziert dargestellt, was ausgebucht worden ist und was an Skonto abgezogen wurde. Wird der Ausbuchungsbetrag keinem OP zugewiesen, wird er auf die ausgeglichenen OPs aufgeteilt. 

Im Dialog Konto Info steht der Button zur Verfügung. Auf der Registerkarte Zahlungen werden die mit einer Ausgleichsbuchung (DA/DZ oder KA/KZ) über ein Geldkonto ausgeglichenen OPs visualisiert. Diese Daten bieten die Basis für die nachträgliche Erstellung einer Zahlungsmitteilung. 

Siehe Kapitel Konto Info . 

Im Dialog Journal steht der Button Zahlgsverw . zur Verfügung. Über diesen Button kann bei Debitoren der Druck einer Zahlungsverwendung angestoßen werden, wenn eine druckfähige Buchung (Buchungskreis DA/DZ oder KA/KZ über ein Geldkonto) selektiert ist. 

Siehe Kapitel Journal . 

Im Dialog Buchen steht der Button zur Verfügung. Mit diesem kann bei Debitoren der Druck einer Zahlungsmitteilung angestoßen werden, nachdem die Selektion einer druckfähigen Buchung (Buchungskreis DA/DZ oder KA/KZ über ein Geldkonto) aus der Tabelle der letzten Buchungen erfolgte. 

Siehe Kapitel Buchen .
