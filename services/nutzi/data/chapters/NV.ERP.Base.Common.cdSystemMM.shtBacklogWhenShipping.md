# Kapitel NV.ERP.Base.Common.cdSystemMM.shtBacklogWhenShipping

Wenn Sie diesen Parameter aktivieren, wird bei Lieferscheindruck eine Position mit der Restmenge gebildet. 

Ist dieser Parameter nicht gesetzt, wird eine Position mit Restmenge beim Rechnungsdruck gebildet. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, wird bei Lieferscheindruck eine Position mit der Restmenge gebildet (sonst bei Rechnungsdruck).
