# Kasseneinstellungen Registerkarte Tabelle 
Kasseneinstellungen Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Kassen aufgelistet. 
### Tabelle 

### Tabellenspalte Kasse 
Hier wird die Nummer der Kasse angegeben. 

Tooltip: 

Definiert die Kassennummer 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung der Kasse angegeben. 

Tooltip: 

Hier wird die Kassenbezeichnung angegeben. 

### Tabellenspalte Std. Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind! 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Tabellenspalte Aktueller Kassennutzer 
In diesem Feld wird der Sachbearbeiter angezeigt, der auf die Kasse angemeldet ist. Solange ein Benutzer eingetragen ist, kann kein anderer Benutzer diese Kasse verwenden. Um einen Sachbearbeiter aus dieser Kasse zu entfernen bzw. abzumelden, klicken Sie auf den Button . Eine Fehlermeldung erscheint, wenn Sie sich an der Kasse anmelden wollen und der vorhergehende „Kassennutzer“ sich nicht abgemeldet hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der an der Kasse angemeldet ist. 

### Tabellenspalte Gesperrt 
Wenn diese Checkbox gesetzt ist, so kann diese Kasse im Menü Kassenauswahl nicht aufgerufen werden. In allen anderen Dialogen (z.B. für Auswertungen) ist die Kasse allerdings vorhanden. 

Tooltip: 

Kasse kann nicht aufgerufen werden. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
