# Betriebskalender Registerkarte Tabelle 
Kalender Schicht Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Kalender angezeigt. 
### Tabelle 

### Tabellenspalte Kalender 
Hier wird die Nummer des Kalenders angegeben. 

Tooltip: 

Angabe der Kalender-Nr. 

### Tabellenspalte Kalendername 
Hier wird die Bezeichnung des Kalenders angegeben. 

Tooltip: 

Angabe des Kalender-Namens 

### Tabellenspalte Jahr 
Hier wird das Jahr angegeben, für welches der Kalender gültig ist. 

Tooltip: 

Angabe des Jahrs 

### Tabellenspalte Schicht 
Hier wird die Schicht angegeben. 

Tooltip: 

Angabe der Schicht 

### Tabellenspalte Schichtname 
Hier wird der Name der Schicht angegeben. 

Tooltip: 

Name der Schicht 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die Summe aller tatsächlich gefundenen Datensätze anzeigen.
