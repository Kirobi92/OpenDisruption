# Webshop Registerkarte Aktive Sitzungen

E-Commerce Katalog / Webshop / Webshop Registerkarte Übersicht 

# Webshop Registerkarte Aktive Sitzungen 
Registerkarte Aktive Sitzungen 

Diese Registerkarte bietet einen Überblick über Besucher des Webshops. 
### Button Details 

Mit diesem Button können Sie sich weitere Informationen zur Session ansehen. 

Tooltip: 

Session Details 

### Button Beenden 

Mit diesem Button wird eine Session aus der Tabelle entfernt. 

Tooltip: 

Session beenden 

### Tabelle 

### Tabellenspalte Benutzer 
Hier wird der Webshop-Benutzer angegeben. 

Tooltip: 

Angabe des Webshop-Nutzers 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Button Akt. 

Mit diesem Button werden die Daten aktualisiert. 

Tooltip: 

Aktualisieren 

### Checkbox Gastbenutzer 
Wenn Sie diese Checkbox aktivieren, werden Sessions von Gast-Nutzern angezeigt. 

Tooltip: 

Sitzungen von Gast-Benutzern anzeigen 

### Feld Anzahl max. 
Hier wird die maximale Anzahl an Datensätzen angezeigt, die gefunden wurden.
