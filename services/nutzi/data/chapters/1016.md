# Kostenstellenzuordnung Registerkarte Allgemein 
Kostenstellenzuordnung Registerkarte Allgemein 

Auf dieser Registerkarte wird die Kostenstelle für die Auftragserfassung angegeben, welche bei den angegebenen Parametern gezogen werden soll. 
### Feld Kostenstelle 
Hier wählen Sie die Kostenstelle aus, die zugeordnet werden soll. 

Tooltip: 

Angabe der Kostenstellennummer 

### Button Suche 

Dieser Button öffnet den Dialog Kostenstellenauswahl zur Selektion einer Kostenstelle. 

Siehe Kapitel Kostenstellenauswahl . 

### Feld Bezeichnung 
Hier wird automatisch die Bezeichnung zur selektierten Kostenstelle angezeigt. 

Tooltip: 

Angabe der Bezeichnung
