# Katalogverwaltung 
Katalogverwaltung 

Dieses Menü dient der Erstellung von Print-Katalogen sowie zur Bestückung des Webshops. Es können mehrere Kataloge parallel verwaltet werden, um z.B. unterschiedliche Daten für den Printkatalog und den Webshop zu pflegen. Die wichtigsten Einstellungen eines Katalogs können innerhalb von diesem kopiert werden. 

### Katalogübersicht 
Über die 2 TreeViews können die Gruppenhierarchie und die Details einer Gruppe getrennt verwaltet werden. Dadurch ergibt sich eine Verbesserung der Geschwindigkeit in der Navigation innerhalb eines Katalogs. 

Auch ein Verschieben oder Kopieren von einem Element des unteren TreeViews in den oberen TreeView ist möglich. So kann eine Artikelzuordnung durch klicken und ziehen auf ein Produkt des oberen TreeViews kopiert werden. Ebenso kann der Text von einer Gruppe auf eine andere Gruppe übernommen werden. 

Im TreeView werden alle vorhandenen Kataloge angezeigt. Alle Operationen werden innerhalb des TreeViews in Echtzeit angezeigt. Durch einfaches Auf- und Zuklappen der Kataloge erscheinen die Untergruppen, Produkte und Artikel. 

### Protokollierung 
Beim Zuordnen eines Artikels zu einer Produktgruppe werden das Erfassungsdatum, der Erfasser sowie das Änderungsdatum und der Sachbearbeiter der Änderung auf Datenbankebene (Tbl ECOMMGRPART) gespeichert. So kann immer nachverfolgt werden, wer und wann welche Änderung stattgefunden hat.

### Baumansicht Katalog 

In diesem TreeView sehen Sie alle Kataloge, welche angelegt sind sowie die Gruppenhierarchie bis auf die Produktebene. 

Mit dem Tastatur-Kürzel <ALT + Cursor> kann ein Element verschoben werden. 

Über die Tastatur-Kürzel <Strg + K>, <Strg + G>, <Strg + P>, <Strg + L> können zudem neue Kataloge / Gruppen / Produkte / Papierkörbe erstellt werden. 

Mit den Tastatur-Kürzeln <Strg + Shift + X>, <Strg + Shift + C>, <Strg + Shift + V>, <Strg + Shift + L> kann das ausgewählte Element ausgeschnitten, kopiert, eingefügt oder verknüpft werden. 

<Strg + Entf> löscht das ausgewählte Baumelement.

### Menü Optionen / Alle Pfade neu berechnen 
Über dieses Menü wird die gesamte Struktur aller Kataloge geprüft und die Nummern werden aufsteigend neu vergeben. Dies ist in Ausnahmefällen notwendig, wenn sehr viele Änderungen ohne Zwischenspeicherungen vorgenommen wurden. Erst nach der Neuberechnung sind alle Pfade für den Webshop erreichbar.
