# Ereignisdefinition Registerkarte Exists 
Ereginisdefinition Registerkarte Exists 

In der Registerkarte Exists können ähnlich wie im Dialog Suchen mit SQL Abfragen auf die Datenbank formuliert werden. Hiermit ist es möglich Querverbindungen zwischen 2 Tabellen zu prüfen 

In der Combobox Datentabelle kann die Tabelle ausgewählt werden, auf welcher der Exist ausgeführt wird. 

Hier werden nur Tabellen angezeigt, deren Komponenten auch im Dialog Komponenten erfasst wurden. 

Darunter werden sämtliche Datenbank-Spalten der Tabelle angezeigt. 

Auf der Registerkarte Operatoren können dann einzelne Spalten abgefragt werden. 

Die Registerkarte Verknüpfung bietet die Möglichkeit eine Verknüpfung auf die Komponente zu erstellen, die im Kopfbereich ausgewählt ist. 

Im Feld auf der rechten Seite wird dann das Statement angezeigt. 

Beispiel: 

In der Artikelgruppe des Artikels soll eine Merkmalsklasse ausgewählt sein. 
### Button Hinzufügen 

Mit diesem Button kann eine neue Abfrage auf die Datenbank erfasst werden. 

Tooltip: 

Hinzufügen eines neuen Exists auf eine Datenbanktabelle. 

### Button Löschen 

Mit diesem Button löschen Sie ein Exist auf einer Datenbanktabelle. 

Tooltip: 

Löschen eines Exists auf einer Datenbanktabelle. 

### Combobox Datentabelle 
Hier wählen Sie die Datenbank aus, auf welcher Exists ausgeführt werden. 

Hier werden nur Tabellen angezeigt, deren Komponenten auch im Dialog Komponenten erfasst wurden.
