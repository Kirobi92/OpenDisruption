# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtAddCheckDigit

Wird dieser Parameter gesetzt, erhalten alle Lagerplätze eine dreistellige Prüfziffer, welche per Random Zahlengenerator ermittelt wird. Diese Prüfziffer wird am Lagerplatz im Barcode mit angedruckt. Diese Funktion steht für das 1-stufige LVS zur Verfügung. 

Bei der Einlagerung müssen nun die komplette Platznummer und die Prüfziffer eingegeben werden. 

Bei der Auslagerung sind „Platznummer + Prüfziffer“ oder nur die Prüfziffer möglich. 

Tooltip: 

Alle Lagerplätze erhalten eine dreistellige Prüfziffer, welche per Random Zahlengenerator ermittelt wird.
