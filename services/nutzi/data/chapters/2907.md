# X-Mapping Editor Registerkarte X Vorlage

Werkzeuge / EDI X Mapping / X-Mapping Editor 

# X-Mapping Editor Registerkarte X Vorlage 
Registerkarte X Vorlage 

Auf dieser Registerkarte kann ein fest definierter Wert angegeben werden, welcher in die Datei geschrieben wird. 

Beispiel: 

<!-- generiert mit eNVenta ERP von Nissen & Velten Software GmbH --> 

HINWEIS FÜR DEN EXPORT 

Für den Export empfiehlt es sich eine "Vorlage" für den spezifischen Knoten zu definieren, wenn die Bedingungen nicht absolut sind (z.B. bei Verwendung des "nicht gleich" Operators). Die Vorlage muss natürlich den Bedingungen des Knotens entsprechen. 

Siehe Registerkarte X Knoten. 
### Feld Vorlage 
Die Vorlage für das Erzeugen des Elements während des Exports. 

Es ist daher möglich einen eigenen "Konstruktor" zu definieren, wie das entsprechende Element initial auszusehen hat. Zum Beispiel können dadurch statische- / default-Werte gesetzt oder Kommentare definiert werden. 

Eine Vorlage könnte z.B. so aussehen: 

<Article note="Dieser Knoten wurde vom eNVenta XML-Mapper generiert" > 
<!-- Ein statischer Kommentar, der bei jedem exportierten 'Article'-Knoten vorhanden sein wird. --> 
<ID>default ID</ID> 
<Name>default Name</Name> 
<!-- Ein weiterer Kommentar am Schluss --> 
</Article>
