# Replikationsdefinition Registerkarte Ausdrücke 
Replikationsdefinition Registerkarte Ausdrücke 

Auf dieser Registerkarte werden Ausdrücke definiert, die für den ganzen Datensatz entscheiden, ob eine Replikation erfolgt oder nicht. Sie können für jede Tabelle definieren, welche Daten repliziert werden sollen. Hierfür können Sie entsprechende Einschränkungen vornehmen. Über einen Formeleditor können komplexe Bedingungen gebaut werden. 
### Feld Allgemein 
Allgemeingültiger Ausdruck, der sowohl beim Insert als auch beim Update verwendet wird. 

Das Ergebnis dieses Ausdrucks muss "bool"( Ja oder Nein ) sein. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor . 

### Feld Insert 
Dieser Ausdruck wird verwendet, um zu prüfen, ob der Datensatz beim Einfügen repliziert werden soll. 

Wenn zusätzlich der ‚Allgemeine Ausdruck‘ definiert wurde, dann wird der Datensatz nur repliziert, wenn beide Ausdrücke "wahr" liefern. 

Das Ergebnis dieses Ausdrucks muss "bool" sein. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor . 

### Feld Update 
Dieser Ausdruck wird verwendet, um zu prüfen, ob der Datensatz beim Ändern repliziert werden soll. 

Wenn zusätzlich der ‚Allgemeine Ausdruck‘ definiert wurde, dann wird der Datensatz nur repliziert, wenn beide Ausdrücke "wahr" liefern. 

Das Ergebnis dieses Ausdrucks muss "bool" sein. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor .
