# Benutzerprofil Registerkarte Zahlung 
Benutzerprofil Registerkarte Zahlung 

Auf dieser Registerkarte können Sie Einstellungen für die Zahlung vornehmen. 
### Feld Kontonummer 
Hier geben Sie eine Kontonummer ein. 

### Feld BLZ 
Hier geben Sie die Bankleitzahl ein. 

Tooltip: 

Angabe der Bankleitzahl / SWIFT 

### Feld Bankname 
Hier geben Sie den Namen der Bank ein. Dieser wird automatisch eingetragen, wenn Sie eine BLZ eingetragen haben. 

### Feld IBAN 
Für den Zahlungsverkehr mit IBAN geben Sie hier die entsprechende Nummer ein. 

Tooltip: 

IBAN (International Bank Account Number) 

### Feld BIC 
Für den Zahlungsverkehr mit BIC geben Sie hier die entsprechende Nummer ein.
