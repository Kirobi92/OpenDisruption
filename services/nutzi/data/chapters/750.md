### Tabellenspalte Artikel Kalkulationsgruppen 
Hier wird die Artikel Kalkulationsgruppe angezeigt. 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angezeigt. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Preisgruppe 
Hier können Sie eine Brutto-Preisgruppe auswählen, die im zugrundeliegenden Vorgang herangezogen wird und bestimmt, auf welchen Verkaufspreis zugegriffen werden soll. 

Diese Preisgruppe wird auch zur Ermittlung des Basispreises herangezogen, sollte dieser nicht aus den Einkaufs-Konditionen ermittelt werden können. 

Preisgruppen werden im gleichnamigen Dialog angelegt. 

Siehe Kapitel Preisgruppen . 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

Diese Preisgruppe wird auch zur Ermittlung des Basispreises herangezogen, sollte dieser nicht aus den Einkaufs-Konditionen ermittelt werden können. 

### Tabellenspalte Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Tooltip: 

Angabe der Währung 

### Tabellenspalte Sachbearbeiter 
Der Sachbearbeiter, der den Datensatz zuletzt bearbeitet hat, wird automatisch vom System eingetragen. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Änderung 
Das Datum der letzten Änderung wird automatisch vom System eingetragen. 

Tooltip: 

Datum und Uhrzeit der letzten Änderung 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
In diesem Feld wird die tatsächlich gefundene Anzahl an Datensätzen angezeigt.
