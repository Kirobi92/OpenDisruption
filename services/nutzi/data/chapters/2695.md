# Planwerte importieren Registerkarte Planwerte 
Planwerte Import Registerkarte Planzahlen 

Diese Registerkarte enthält die Planzahlen gemäß Ihrer Selektion. 

Es werden nur Daten einer csv-Datei eingelesen! 

Plausibilitätsprüfung: Die Spalten 'Gesamt Var/Fix', 'Gesamt Var' und 'Gesamt Fix' müssen rechnerisch korrekt die Summe der Monatswerte enthalten, ansonsten wird der Button Import ( F3 ) nicht zugänglich! 
### Button Löschen 

Dieser Button löscht eine selektierte Zeile aus der Tabelle. 

Tooltip: 

Selektierte Zeilen löschen 

### Tabelle 

Die importierten Zeilen verschwinden aus der Tabelle. Wenn es beim Import zu Fehlern gekommen ist, sehen Sie in jeder Zeile an 1. Stelle den Grund des Fehlers.
