# Lagerzone Registerkarte Allgemein 
Lagerzone Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie einzelne Lagerbereiche. 

Bei der Anzeige der Felder wird differenziert, wenn der Dialog mit und ohne LVS Lizenz geöffnet wird. Wenn es sich um ein Lager mit chaotischer Lagerführung handelt, nicht aber um ein LVS-Lager, so werden nur die Felder angezeigt, welche für die Verwaltung der Lagerzone notwendig sind. 

Bei Aufruf des Dialogs ohne LVS Lizenz stehen nur die Felder Zone und Bezeichnung zur Verfügung. 

Bei Aufruf des Dialogs mit LVS Lizenz stehen folgende Felder und Registerkarten zur Verfügung: 

- Zone - Bezeichnung - Ist Sonderzone - Bypass Bereich - Klärungsbereich - Packbereich - Schnittbereich - Keine I Punkt Benutzung - Entnahme gesperrt - Artikel-Etikettendruck - Keine Prüfung auf offene Aufträge - Behälterliste nicht drucken - Max. Aufträge - Max. Picks - Min. Picks - Registerkarte Skills - Registerkarte Sonderzonen - Registerkarte Kommissionier Behältertypen - Registerkarte Automatisierung

### Checkbox Ist Sonderzone 
Wenn Sie diese Checkbox aktivieren, wird die selektierte Lagerzone zu einer Sonderzone. In diesem Fall wird die Registerkarte Sonderzone eingeblendet. 

### Checkbox Bypassbereich 
Plätze, die in diesem Bereich liegen, werden als Bypassplätze behandelt. D.h. diese Plätze werden dann durch die spezielle Entnahmestrategie Komplett / Teil von Bypasslagerplatz ermittelt. 

### Checkbox Klärungsbereich 
Wenn Sie diese Checkbox aktivieren, wird die Zone als Klärungsbereich ausgewiesen. D.h. die Ware, die hier abgelegt wird, muss noch "endkontrolliert" werden. 

Tooltip: 

Zone, in welche Ware abgelegt werden kann, die noch endkontrolliert werden muss. 

### Checkbox Packbereich 
Wenn Sie diese Checkbox aktivieren, aktivieren Sie den Bereich in dem die Packplätze liegen. Hier gelten besondere Bedingungen für die Buchung von Kommissionierbehältern. 

Tooltip: 

Lagerzone wird als Packbereich definiert. 

### Checkbox Schnittbereich 
Wenn Sie diese Checkbox aktivieren, wird der Schnittbereich zugeordnet. 

Bei der Chargenzuordnung von Schnittartikeln mit dem Kennzeichen „Schnitt nur in bestimmten Zonen“ müssen zusätzliche Bedingungen an die Chargenzuordnung erfüllt sein. 

Zuerst müssen Chargenbestände in der Schnittzone zugeteilt werden. Ist dies nicht möglich, werden erst dann Chargen verwendet, auf welche bereits Zuordnungen existieren. 

Ist auch dies nicht möglich, so findet die Zuordnung auf eine neue Charge statt. 

Innerhalb einer der Möglichkeiten wird zuerst geprüft, ob es eine passende Charge gibt, die nicht geschnitten werden muss. Anschließend werden die längeren Chargen gewählt. 

Tooltip: 

Lagerzone wird als Schnittbereich definiert.

### Feld Max. Aufträge 
Maximale Anzahl von Kundenaufträgen, welche durch einen Kommissionierer in dieser Zone auf einmal gepickt werden dürfen. Nur diese werden ihm dann auch zugeteilt. 

### Feld Max. Picks 
Hier geben Sie die maximale Anzahl von Picks ein, welche durch einen Kommissionierer in dieser Zone auf einmal gepickt werden können. Ist diese Grenze erreicht, so werden dem Kommissionär keine weiteren Aufträge zugeteilt. 

Es werden allerdings auch keine Kundenaufträge innerhalb einer Zone getrennt. D.h. ist die Maximalgrenze innerhalb eines Auftrags erreicht, so wird der komplette Auftrag nicht mehr zugeteilt. 

### Feld Min. Picks 
Hier geben Sie die kleinste Anzahl an Picks ein, für die es lohnt in eine Zone zu gehen. Ist dieser Wert nicht erreicht, werden Picks in dieser Zone nicht zugeteilt. Der Sachbearbeiter, welcher versucht diese Zone zu übernehmen, erhält eine Hinweismeldung. Befindet sich ein Auftrag im Eilt- oder Express-Status, so wird diese Minimum-Beschränkung nicht angewendet. Befinden sich nur noch Aufträge in Zonen unterhalb des Minimums im Pool, so wird die Beschränkung nicht beachtet und der Pick zugeteilt. 

### Checkbox Entnahme gesperrt 
Wenn Sie diese Checkbox aktivieren, können in dieser Zone Picks der Priorität Normal nicht gepickt werden. 

Befinden sich Picks der Priorität Eilig oder Express in der Zone, wird das Picken für diese Vorgänge nicht verhindert. 

Auf diese Art und Weise kann der Kunde seine Ware an der Theke aus eben dieser Zone erhalten. 

### Checkbox Artikel-Etikettendruck 
Wenn Sie diese Checkbox aktivieren, wird bei der Kommissionierung ein Etikett für den Artikel gedruckt. 

Tooltip: 

Artikeletikettendruck bei der Kommissionierung 

### Checkbox Keine Prüfung auf offene Aufträge 
Wenn Sie diese Checkbox aktivieren, werden Lagerplätze in diesem Packbereich bei einem Mehrbehältervorgang für Aufträge nicht gesperrt. Es können alle Behälter auf einem Lagerplatz abgestellt werden. 

Tooltip: 

Es erfolgt keine Sperrung der Lagerplätze bei einem Mehrbehältervorgang in dieser Lagerzone (Packbereich). 

### Checkbox Behälterliste nicht drucken 
Wenn Sie diese Checkbox aktivieren, wird beim Abstellen des letzten Behälters eines Mehrbehälter-Vorgangs keine Behälterliste gedruckt. 

Tooltip: 

Bei einem Mehrbehältervorgang wird keine Behälterliste in der Lagerzone gedruckt.
