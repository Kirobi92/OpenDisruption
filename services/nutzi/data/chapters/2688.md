# Kostenstellenauswahl 
Kostenstellenauswahl 

Dieser Dialog öffnet sich im Buchen Kore über den Button Kostenst . auf der Registerkarte Allgemein . Im Dialog Kostenstellenauswahl gibt es die Tabelle Kostenstellen und eine Tabelle Auswahl . Die Felder Kostenstellennr , Bezeichnung und die Comboboxen Gruppe 1-5 können zur Selektion verwendet werden. Eine sogenannte „Like-Abfrage“ ist ebenfalls möglich. 

Um der Kostenart für diese Buchung eine bzw. mehrere Kostenstellen neu zuzuordnen, gehen Sie bitte wie folgt vor: 

- Markieren Sie in der linken Tabelle die Kostenstelle, die Sie zuordnen möchten durch Mausklick in die entsprechende Zeile. - Übertragen Sie nun die markierte Kostenstelle durch Klick auf den -Button in die rechte Tabelle. - Erfolgt die Umlage der Kostenart über einen Schlüssel, so sehen Sie in den Spalten 'Anteil' und 'Umlagekürzel' die entsprechenden Werte der Kostenstelle, wie sie im Kostenstellenstamm hinterlegt sind. 

Siehe Kapitel Kostenstellen . 
Ist im Kostenartenstamm die prozentuale Umlage gewählt, so geben Sie bitte die entsprechenden Anteile im Feld Prozent ein. Dabei ist darauf zu achten, dass sich insgesamt 100 Prozent ergeben. 

Siehe Kapitel Kostenarten . 

Beim Ändern eines Eintrags erscheint nun eine Abfrage, falls Sie den Dialog schließen wollen, ohne vorher gespeichert zu haben. 

Zurück im Dialog Buchung sehen Sie die zugeordneten Kostenstellen. Wenn Sie sich nun doch für die Standardumlage, wie im Kostenartenstamm hinterlegt, entscheiden, so kann diese sehr einfach über den Button Standard ausgewählt werden.
