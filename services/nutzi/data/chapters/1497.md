# Gutschein 
Gutschein 

Dieser Dialog öffnet sich über den Button im Dialog Gutscheinverwaltung oder über den Button Gutschein in der Kasse, allerdings nur wenn die Kasseneinstellung Gutscheine an der Kasse erstellen freigeschaltet ist. 

Siehe Kapitel Kasseneinstellungen . 

Mit dem Anlegen des Gutscheins ist dieser nicht automatisch aktiviert, dazu muss die Checkbox Aktivieren angeklickt werden. 

In der Kasse wird ein Gutschein erst „aktiv“, wenn er bezahlt ist. 

Solange die Checkbox Aktivieren nicht gesetzt ist, ist eine endgültige Löschung des Datensatzes möglich. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Gutschein 
Tooltip: 

In diesem Dialog können Gutscheine angelegt werden. 
Für mehr Informationen drücken Sie F1.

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Feld Gutschein 
Hier wird die Nummer des Gutscheins eingetragen. 

Tooltip: 

Die eindeutige Bezeichnung des Gutscheins wird angegeben. 

### Combobox Gutscheinart 
Hier wählen Sie eine Gutschriftsart aus. 

Tooltip: 

Angabe der Gutscheinart 

### Feld Initialwert 
Hier wird der Preis angegeben. 

Tooltip: 

Wert des Gutscheins bei Erstellung 

### Combobox Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Die Systemwährung (Euro für die deutsche Version) MUSS in der Tabelle unter der Währungsnummer 1 und dem Kurs 1,0 angegeben werden! 

Tooltip: 

Geben Sie eine eindeutige, fortlaufende Nummer ein. 

### Checkbox Aktivieren 
Wenn Sie diese Checkbox anklicken, wird der Gutschein aktiviert, d.h. er kann genutzt werden. 

### Feld Status 
Hier wird der Status des Gutscheins angezeigt. 

### Feld Restwert 
Hier geben Sie den Restwert aus dem Gutschein zur Selektion ein. 

Tooltip:
