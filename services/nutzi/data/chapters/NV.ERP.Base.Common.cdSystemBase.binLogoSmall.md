# Kapitel NV.ERP.Base.Common.cdSystemBase.binLogoSmall

Es können alle möglichen Grafiken ausgewählt werden (bmp, gif, jpg, png, tiff, …). Der -Button ermöglicht die Suche über eine Dateiauswahl. 

Die Bilder werden intern im PNG-Format gespeichert (komprimiert ohne Verluste) und auch in diesem Format an den Report übergeben. 

Die Standardmaße des Logos sind 3,66 x 1,82 cm. 

Tooltip: 

Das Seitenverhältnis dieses Logos entspricht 2:1.
