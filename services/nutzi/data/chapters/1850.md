# Ändern der Inventurpositionen 
Inventurposition ändern 

Über den Button Ändern im Dialog Inventur , auf der Registerkarte Inventurliste , öffnet sich der Dialog Inventurposition ändern . Hier können Sie durch Eingabe der Artikelnummer direkt auf den Datensatz zugreifen, ohne dass Sie sich in der Tabelle bewegen müssen. Ist bereits eine Zeile der Tabelle (Registerkarte Inventurliste ) markiert, werden die Daten dieser Zeile übernommen. 

In den Feldern Artikelnr und Lagerplatz tragen Sie Ihre Selektionskriterien ein, falls diese nicht durch die markierte Zeile der Tabelle übernommen wurden. Durch Klick auf den Button werden alle Positionen innerhalb der Inventur selektiert und können nacheinander über die Schnellerfassung bearbeitet werden. 

Anschließend können Sie in der rechten Spalte die Ergebnisse der Zählung eintragen. Hierbei sind nur die tatsächlich benötigten Felder ( mind. Inventur Ist, Bew. Prozent und Bewertungs-EK ) aktiv. Mit Klick auf den Button Übern.> ( <Strg> + <R> ) werden die Daten in die Inventurtabelle übertragen. 

Diese Art der Erfassung ist besonders dann geeignet, wenn die gezählten Daten nicht als Liste vorliegen, sondern beispielsweise als besprochenes Tonband. 

Wird bei einem bereits erfassten Artikel ein weiterer Eintrag vorgenommen, erhalten Sie die Möglichkeit, die neue Menge zur alten zu addieren, zu subtrahieren oder die bisherige Menge zu überschreiben. Geben Sie hierfür im Feld neben „Inventur Ist“ entsprechend ein „+“ ein, um die Menge zu addieren, ein „-“ um die Menge zu subtrahieren oder keine Eingabe, um die bisherige Inventur-Istmenge zu ersetzen. Die Operation wird bei Klick auf den Button Übern.> ( <Strg> + <R> ) durchgeführt. 

Über die Buttons und können Sie innerhalb der Suchkriterien navigieren. Sollte die Istmenge noch nicht gespeichert sein, so erscheint eine Meldung. 

Die Angaben im Dialog Inventurposition ändern werden automatisch auf die Inventur-Positionstabelle übertragen.
