# Kapitel NV.ERP.MM.CashDesk.cdCashDeskParameters.shtLetterHead

Wenn Sie diese Checkbox aktivieren, wird das Logo auf die Listen gedruckt. In Auftrags- und Bestellformularen wird zusätzlich noch eine kleine Absenderzeile über der Adresse gedruckt. 

Tooltip: 

Wenn aktiviert, wird das Logo auf den Beleg gedruckt.
