Lagerverwaltungssystem (LVS) / Thekenerfassung 

# Thekenerfassung Registerkarte LVS 
Registerkarte LVS 

Auf dieser Registerkarte werden in einer Übersicht die einzelnen Lageraufträge angezeigt. Jede Entnahme eines Artikels von einem Platz bzw. von einem LHM in einer Charge wird als 1 Lagerauftrag dargestellt. D.h. es gibt mindestens so viele Lageraufträge wie Positionen mit Lagerhaltung, evtl. sogar mehr. 

Über die einzelnen Buttons auf der Registerkarte können Sie z.B. die Abholung zu einem Auftrag auslösen. D.h. aus dem Auftrag werden sofort die entsprechenden Lageraufträge erstellt und auch sofort die Etiketten auf einem speziellen Drucker erzeugt. 
### Button Lagerauftrag 

Dieser Button öffnet den Dialog Lagerauftrag zur Selektion nach einem Lagerauftrag. 

Siehe Kapitel Lagerauftrag . 

### Button Etiketten 

Dieser Button öffnet den Dialog Etikettendruck zum Nachdruck der Lagerauftragsetiketten. Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Siehe Kapitel Etikettendruck im LVS . 

### Button Umlagern 

Dieser Button öffnet den Dialog LA – generieren . Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Siehe Kapitel Lagerauftrag generieren . 

### Button Liefern 

Mit diesem Button wird für den Auftrag bzw. für die markierten Auftragspositionen ein Lagerauftrag für das LVS-geführte Lager erstellt. Mit einem Lagerauftrag wird die Kommissionierung abgewickelt. Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Button Zuw. Scanner 

Über diesen Button können die Picks des Auftrags einem Scanner zur Kommissionierung zugewiesen werden (ähnlich der Scanner-Zuordnung im Thekenprozess). 

Diese Funktion könnte z.B. genutzt werden, wenn ein Auftrag gezielt kommissioniert werden muss, weil der Kunde den Auftrag "spontan" abholen möchte. 

Die angemeldeten Scanner-User in einem Lager werden, entsprechend der Lagernummer im Auftragskopf, vorgeschlagen. 

Tooltip: 

Möglichkeit Lageraufträge direkt einem Scanner zuzuordnen.
