# Kapitel NV.ERP.Base.Common.cdShippingCondition.shtNV4_ContainerCommissioned

Dieser Parameter bestimmt, dass die Versandart nicht auf einen bestimmten Behältertyp festgelegt ist. 

Tooltip: 

Wenn aktiviert, ist die Versandart nicht auf einen bestimmten Behältertyp festgelegt.
