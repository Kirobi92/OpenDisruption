# Import Items erzeugen Registerkarte Allgemein

Initialisierung / Import Items erzeugen 

# Import Items erzeugen Registerkarte Allgemein 
Basis Dialog Dokumente Registerkarte Allgemein 

Hier geben Sie allgemeine Informationen für den Export ein. 
### Feld Dateiname 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Feld Zielverzeichnis 
Hier geben Sie das Zielverzeichnis für den Export ein. 

### Checkbox CSV Matching 
Wenn Sie diese Checkbox aktivieren, wird die Registerkarte CSV Matching zugänglich.
