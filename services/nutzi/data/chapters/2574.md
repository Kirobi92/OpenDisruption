# Weitere Offene Posten Debitoren 
Weitere OPs Debitoren 

Dieser Dialog öffnet sich im Bankeinzug über den Button Weitere OPs . 

Hier haben Sie die Möglichkeit über die Auswahlfelder Kontonr, Belegnummer und Gruppe weitere OPs anzuzeigen und durch markieren und den Button oder Doppelklick in den Vorschlag zu übernehmen. 

Die wichtigsten OP-Daten werden in der Tabelle angezeigt. Je nachdem aus welchem Dialog Sie den Dialog Weitere OPs Debitoren öffnen, sehen Sie in den rechten Spalten, ob der OP in einem anderen Vorschlagsdialog enthalten ist. Z.B. Mahnlauf, Zinslauf oder Verbandsabrechnung. Markieren Sie einen OP und klicken den Button OP Detail , haben Sie die Möglichkeit, Daten eines OPs zu verändern. (Siehe Beschreibung zum Button Detail .)

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
