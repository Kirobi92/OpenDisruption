# Anfrage aus der Retoure erstellen 
Anfrage erstellen 

Der Button Anfrage im Kopf des Dialogs Retoure öffnet den Dialog Anfrage erstellen . Hier können Sie eine Anfrage an einen Lieferanten generieren. Sie können die Einkaufart, den Lieferanten und evtl. die Projektnummer eingeben. Diese Referenz wird zur Retourenposition eingetragen. D.h. im Dialog Retoure auf der Registerkarte Allgemein steht die Spalte ‚Anfrage‘ zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Anfrage erstellen 
Tooltip: 

In diesem Dialog können Sie aus der Retoure eine Anfrage an einen Lieferanten generieren. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt.
