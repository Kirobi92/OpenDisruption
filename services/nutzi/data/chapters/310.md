# Seriennummern 
Seriennummern 

Eine Seriennummer ist eine eindeutige Identifizierung eines Produkts durch den Hersteller. Mithilfe dieser können dem Kunden besondere Serviceleistungen oder Produktrückrufe ermöglicht werden. 

Im Verkauf und Einkauf im Menü Lager steht der Eintrag Seriennummer zur Verfügung. Hier findet die Erfassung der Nummern und auch die Zuordnung bei Abgang statt. Die Erfassung und die Suche finden über die Registerkarten im Dialog statt. Im Kopf des Dialogs stehen diverse Felder zur Selektion bereit. 

In diesem Dialog kann eine Recherche über die dem System bekannten Seriennummern zur Artikel erfolgen. 

Wird für einen Artikel manuell eine Seriennummer zugeordnet, wird nicht nur im Feld Belegnummer eine „0“ eingetragen, sondern zusätzlich im Feld Firma 1 der Text „manuelle Lagerbuchung“. 

Aus diesem Grund können in der Tabelle 140 Zeichen eingetragen werden. Damit kann der Text der manuellen Lagerbuchung vollständig angezeigt werden.

### Combobox Status 
Diese Combobox steht zur Selektion über den Status zur Verfügung. Folgende Einträge stehen zur Verfügung: 

- Frei: Seriennummer ist erfasst und keinem Vorgang zugeordnet, z.B. aus dem Warenein-gang oder dem Produktionsauftrag heraus. - Repariert (Lieferant sendet die Ware zurück): Ware, welche zuvor den Status Zurückgesendet hatte. Wird im Wareneingang wieder der Bestellung zugeordnet. - Ausgebucht: Verkauft an Kunden. Status wird bereits ab Zeitpunkt der Zuordnung zum Beleg gesetzt. - Zurückgesendet: Ware wurde an den Lieferanten zurückgesendet (via Belastung oder Materialbeistellungsauftrag). - Reklamiert: Kunde sendet die Seriennummer an Händler zurück (via Retoure). - Verbaut: Wenn die Seriennummer als Komponente im Betriebsauftrag zugeordnet wurde. - Objekt zugeordnet: Entfällt bzw. wurde ersetzt durch den Status Ausgebucht bei Verkauf bzw. Frei bei Wareneingang. - Zuordnung entfernt: Entfällt 

Tooltip: 

Status Seriennummer: 
1 - Frei 
2 - Repariert (bei Eingang von Status Reklamiert) 
3 - Ausgebucht (bei Ausgang) 
4 - Zurückgesendet (bei Ausgang über Belastung/Materialrücksendung) 
5 - Reklamiert (bei Eingang in Retoure) 
6 - Verbaut (bei Ausgang über Produktion)
