# Skontierfähiger Wert 
Skontierfähiger Wert 

Im Buchen >> Optionen finden Sie den Eintrag Skontierfähiger Wert . Hier kann für Rechnungen, die manuell gebucht werden, ein vom Rechnungsbetrag abweichender skontierfähiger Betrag erfasst werden. 

Vorgehensweise: 
Erfassen Sie eine Debitoren- oder Kreditorenrechnung. Vor Betätigen des Buttons rufen Sie das Menü Skontierfähiger Wert auf. Erfassen Sie im Feld Skontierf. Wert den entsprechenden Betrag und bestätigen die Eingabe mit dem Button . Danach schließen Sie Ihre Buchung ebenfalls über den Button ab. 

Für die Ermittlung des Skontoabzugs wird im Feld Skontierfähig der Betrag aus dem OP herangezogen und nicht der Gesamtrechnungsbetrag. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Skontierfähiger Wert 
Tooltip: 

In diesem Dialog kann ein abweichender, skontierfähiger Betrag erfasst werden. 
Für mehr Informationen drücken Sie F1.
