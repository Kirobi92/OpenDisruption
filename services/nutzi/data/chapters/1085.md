# Textpositionen 
Textposition 

Möchten Sie im Vorgang (z.B. Auftrag, Bestellung) einen Text als Position einfügen, so wählen Sie in der Combobox Positionsart auf der Registerkarte Positionen den Eintrag Text . Anschließend geben Sie wie gewohnt (nach Klicken des Buttons Neu ) einen Artikel in die Tabelle ein. Es öffnet sich automatisch der Dialog Textposition , in dem Sie einen beliebigen Text eingeben können oder wählen dort über den Button neben dem Feld Text einen Textbaustein aus. Übernommene Textbausteine werden dann an den vorhandenen Text angefügt. 

Siehe Kapitel Textbausteine . 

Im Bereich Drucken können Sie festlegen, für welchen Bereich der Textbaustein zur Verfügung stehen soll. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Textposition 
Tooltip: 

In diesem Dialog können Sie einen beliebigen Text eingeben, der dann im Vorgang angezeigt wird. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
