# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtStorageIDExternalType

Gilt für Cross Company: 
Wenn dieser Parameter aktiviert ist, kann im Feld Lager(gruppe)n nur die Nummer einer Lagergruppe eingetragen werden. Die Nummer der Lagergruppe muss vom verbundenen Unternehmen bekannt gemacht sein. 

In diesem Zusammenhang ist im Dialog Lagergruppen die Checkbox in der Spalte 'CC' aktiviert werden. 

Siehe Kapitel Lagergruppen . 

Tooltip: 

Handelt es sich bei der Fremdlagernummer um eine Lagergruppe?
