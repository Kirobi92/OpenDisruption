# Suchbedingung / Suche mit SQL Registerkarte Expertenmodus 
Suchbedingung Registerkarte Expertenmodus 

Auf dieser Registerkarte können detaillierte SQL-Statements geschrieben werden. Bitte beachten Sie hierbei, dass unter Umständen keine genaue Syntax-Prüfung vorgenommen werden kann. 

Alle angegebenen Kriterien werden mit AND verknüpft. Wenn ein Feld mehrfach vorkommt, ist dieses zwingend mit OR zu verknüpfen. Dafür ist bei der Zusammenstellung der Bedingung unbedingt nach Feldern zu gruppieren. 
### Button Hinzufügen 

Mit diesem Button übernehmen Sie einen Eintrag aus dem Feld Felder in das rechte Feld. 

### Button AND 

Wenn Sie diesen Button drücken, wird eine Add-Verknüpfung hinzugefügt. 

Tooltip: 

Add Verknüpfung hinzufügen 

### Button OR 

Wenn Sie diesen Button drücken, wird eine OR-Verknüpfung hinzugefügt. 

Tooltip: 

OR Verknüpfung hinzufügen 

### Combobox Datentabelle 
In dieser Combobox steht die zugrundeliegende Datentabelle zur Verfügung. 

### Liste Spalten 
In diesem Feld werden alle Felder der Datenbank angezeigt. 

Tooltip: 

Select Felder 

### Dock Panel Operator 

In diesem Bereich stehen die Operatoren zur Auswahl zur Verfügung. 

Tooltip: 

Select Operatoren 

### Feld Wert 
Hier wird der Wert für die Select-Bedingung angegeben. 

Tooltip: 

Wert für Select-Bedingung 

### Feld Gewählte Information 
In diesem Feld wird die zusammengestellte Suchbedingung angezeigt. 

Tooltip: 

Select Statement
