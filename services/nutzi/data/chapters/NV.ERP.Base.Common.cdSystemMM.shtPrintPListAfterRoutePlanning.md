# Kapitel NV.ERP.Base.Common.cdSystemMM.shtPrintPListAfterRoutePlanning

Wenn Sie diese Checkbox aktivieren, darf eine Packliste (mit Versandweg Zufuhr) erst gedruckt werden, wenn der Artikel einer Tour zugeordnet worden ist. Diese Einstellung wirkt auch im Massendruck für Packlisten. 

Tooltip: 

Wenn der Schalter gesetzt ist, wird das Drucken der Packlisten erst nach erfolgreicher Tourenplanung ermöglicht.
