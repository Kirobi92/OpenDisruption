# Teilnehmeraktivität Registerkarte Steuerung 
Aktivität- Teilnehmer Registerkarte Steuerung 

Auf dieser Registerkarte kann der zukünftige Verlauf der Marketing-Aktion für diesen Teilnehmer festgelegt werden. 

Alle vorgenommenen Einstellungen gelten nur für die aktuelle, kundenbezogene Aktivität. 
### Feld Folgeaktionsnr. 
Dieses Feld wird automatisch aus dem Aktivitätsschritt vorbelegt. 

Ist eine Nachfolgeaktion eingetragen, werden für den Teilnehmer weiterhin die noch offenen Aktivitäten berücksichtigt. 

Siehe Kapitel Aktionsübersicht . 

Tooltip: 

Angabe einer Aktionsnummer 

### Button Suche 

Mit diesem Button können Sie eine Folge-Aktion suchen. 

Siehe Kapitel Aktionsteilnehmer eintragen . 

### Feld Name der Aktion 
Hier wird die Bezeichnung der Aktion angegeben. 

Tooltip: 

Name der Aktion 

### Combobox Nächster Schritt 
Hier können Sie den nächsten Schritt für die Aktivität auswählen. 

Tooltip: 

Wählen Sie den nächsten Aktivitätsschritt 

### Combobox Zuständig 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Angabe des zuständigen Sachbearbeiters 

### Checkbox WV für Nachfolgeschritt 
Mit diesem Parameter kann eine automatische Terminierung der Nachfolgeaktivität in die Wiedervorlage aktiviert werden. 

Tooltip: 

Terminierung der Nachfolge-Aktivität 

### Feld Anzahl Tage 
In diesem Feld können die Tage für die Terminierung der Wiedervorlage hinterlegt werden. Ist dieses Feld nicht gefüllt, wird das Tagesdatum als Termin genommen. 

Tooltip: 

Anzahl Tage für die Terminierung der Wiedervorlage 

### Combobox An Benutzer 
In diesem Feld wird alternativ festgelegt, für wen der Termin in der Wiedervorlage gelten soll. Ist dieses Feld nicht gefüllt, wird ein genereller Termin generiert. Ist dieses Feld in der Marketing-Aktion im Dialog Aktivität bearbeiten vorbelegt, wird die Einstellung automatisch übernommen. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Combobox An Abteilung 
In diesem Feld wird alternativ festgelegt, für welche Abteilung der Termin in der Wiedervorlage gelten soll. Ist dieses Feld nicht gefüllt, wird ein genereller Termin generiert. Ist dieses Feld in der Marketing-Aktion im Dialog Aktivität bearbeiten vorbelegt, wird die Einstellung automatisch übernommen. 

### Feld Information 
Hier kann eine interne Information für die Wiedervorlage eingetragen werden.
