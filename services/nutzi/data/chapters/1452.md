# Folgeauftrag generieren 
Folgeauftrag gener. 

Dieser Dialog öffnet sich in der Vermietung >> Optionen >> Folgeauftrag gener . Hier können Folgeaufträge für mehrere Vermietungen generiert werden. 

Den Folgeauftrag für eine einzelne Vermietung generieren Sie erneut über den Button Auftrag . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Folgeauftrag generieren 
Tooltip: 

In diesem Dialog können Folgeaufträge für mehrere Vermietungen generiert werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
