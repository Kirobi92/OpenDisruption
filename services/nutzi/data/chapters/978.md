# Parameter Verkauf Registerkarte KSt-Zuweisung 
Parameter Verkauf Registerkarte KST-Zuweisung 

Auf dieser Registerkarte können die Kriterien für die automatische Kostenstellenzuordnung (= Kostenstellenzuweisung) festgelegt werden. Es können maximal zwei Kriterien miteinander kombiniert werden. Diese Kriterien müssen festgelegt werden, bevor über den Menüpunkt System >> Kostenstellenzuordnung eine Zuordnung der Kostenstellen entsprechend den Kriterien erfolgt. 

Diese Parameter sollten nach der Hinterlegung der Kostenstellen nicht mehr geändert werden, da ansonsten keine durchgängigen Daten an die Kostenrechnung übergeben werden. 

Wollen Sie dennoch eine Änderung dieser Einstellung vornehmen, dann werden sämtliche bisher vorgenommenen Zuordnungen gelöscht. 

Es können höchstens 2 Parameter in Kombination aktiviert werden.
