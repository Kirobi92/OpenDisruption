# Kapitel NV.ERP.Base.Common.cdSystemMM.shtCostCenterConstraint

Wenn Sie diesen Parameter aktivieren, ist jeder Auftragsposition zwingend eine Kostenstelle zuzuweisen. Vorausgesetzt die Kostenrechnung ist lizenziert. 

Beim Speichern eines Auftrags oder einer Gutschrift wird dann geprüft, ob eine Kostenstelle zugeordnet ist. 

Auf der Registerkarte KST-Zuweisung können die Kriterien für die automatische Kostenstellenzuordnung (= Kostenstellenzuweisung) festgelegt werden. 

Kostenstellen werden Artikeln bzw. Kunden im Dialog Kostenstellenzuordnung zugeordnet. 

Siehe Kapitel Kostenstellenzuordnung .
