# Status-Information Registerkarte Bedingungen 
Status Information Registerkarte Bedingungen 

Auf dieser Registerkarte werden die Bedingungen der Replikationseinstellungen dargestellt. 
### Tabelle 

### Tabellenspalte Component-Typ 
Hier werden die Components angegeben, die benutzt werden. 

### Tabellenspalte Zusätzliche Such-Bedingung 
Hier wird eine zusätzliche Such-Bedingung angegeben. 

### Tabellenspalte Mandanten-Key 
Hier wird der Zugriff des Mandanten auf die Component / Tabelle angegeben. Diese kann z.B. genutzt werden, um Tabellen zu teilen (z.B. Pickliste).
