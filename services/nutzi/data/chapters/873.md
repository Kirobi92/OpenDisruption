# Zuordnungen Zahlungsbedingungen 
Zuordnungen Zahlungsbedingungen 

Dieser Dialog öffnet sich im Kunde >> Optionen >> Zuordnungen Zahlungsbedingungen . Dieser Dialog steht zur Verfügung, um Stahl- und SHK-Artikel gesondert in Aufträgen erfassen zu können oder auch um z.B. für Projektaufträge abweichende Zahlungsziele zu verwalten. 

Hier können Sie unterschiedliche Zahlungsbedingungen eines Kunden verschiedenen Auftragsarten zuordnen. Dann können diese, z.B. in Abhängigkeit des gewählten Artikels (ob Stahl oder SHK), genutzt werden. Die Zuordnung einer Zahlungsbedingung zur Auftragsart übersteuert die Einstellung aus dem Kundenstamm. 

Diese Zahlungsbedingungen wiederum werden unterschiedlichen Auftragsarten zugeordnet, je nach Verwendungszweck. 

Wird ein Auftrag erfasst, für den es einen Zahlungsbedingungseintrag der Kombination Kunde + Auftragsart gibt, wird die hier hinterlegte Zahlungsbedingung im Auftrag eingetragen. Sie überschreibt somit die Zahlungsbedingung, die im Kundenstamm auf der Registerkarte Warenwirtschaft , Combobox Zahlung hinterlegt ist. 

Pro Kombination Kunde + Auftragsart ist nur eine Zahlungsbedingung möglich.
