# Wechsel-Verwaltung Registerkarte Bankdaten 
Wechsel - Verwaltung Registerkarte Bankdaten 

Auf dieser Registerkarte geben Sie Daten zur Bank ein. 
### Domizilbank 

In diesem Bereich tragen Sie die Bankverbindung des Wechselausstellers ein. 
### Feld IBAN 
Hier wird die IBAN eingetragen. 

Eine Suche der Bankdaten über die IBAN ist möglich. 

Tooltip: 

IBAN (International Bank Account Number) 

### Feld Bezeichnung 
Hier wird der Name der Domizil-Bank angegeben. 

### Combobox Land 
Hier wird das Länderkennzeichen zum Datensatz angegeben. 

Tooltip: 

Länderkennzeichen des Datensatzes 

### Feld PLZ 
Die Postleitzahl des Orts wird angegeben. 

Tooltip: 

Postleitzahl des Datensatzes 

### Feld Ort 
Hier wird der Ort angegeben. 

Tooltip: 

Angabe des Orts 

### Feld Straße 
Die Straße zur Firma wird angegeben. 

Tooltip: 

Angabe des Straßennamens 

### Button Suche 

Mit diesem Button können Sie eine Bank suchen. 

Siehe Kapitel Bankenauswahl . 

### Feld Bank KtoNr. 
Hier wird die Bank-Kontonummer angegeben. 

Tooltip: 

Bank Kontonummer 

### Diskontbank 

Geben Sie einen Wechsel an Ihre Hausbank, können Sie die entsprechenden Daten in diesem Bereich eintragen. 
### Feld IBAN
