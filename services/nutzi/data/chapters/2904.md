# X-Mapping Editor Registerkarte Hinweise 
X-Mapping Editor Registerkarte Hinweise 

Sollte zu einer Zuordnung ein Hinweis hinterlegt sein, so kann dies hier angezeigt bzw. hinterlegt werden. Dies ist ein Freitext, welcher für den Manager des X Mappings entsprechende Hinweise beinhalten kann. 

Bei vorhandenen Fehlern im Mapping wird hier eine Fehlermeldung statt dem im Klartext erfassten Hinweis angezeigt. 

Tooltip: 

Hier können Sie Hinweise bzw. Bemerkungen zu dem gewählten Mapping in Klartext erfassen. 

Bei vorhandenen Fehlern im Mapping wird hier eine Fehlermeldung statt dem im Klartext erfassten Hinweis angezeigt.
