# Kapitel NV.ERP.Base.Common.cdSystemBase.shtDateInRangeDays

Wenn Sie hier eine Anzahl Tage eingeben, haben Sie eine bessere Kontrolle, ob ein Rechnungs- oder Belegdatum korrekt eingetragen ist, bzw. in einem „sinnvollen“ Zeitrahmen liegt. 

Die erfasste Anzahl an Tagen wird vom Systemdatum subtrahiert oder addiert. Liegt das Buchungs-/Belegdatum außerhalb dieses Bereichs erhalten Sie einen Hinweis. Dieser Hinweis kann jederzeit ignoriert werden. 

Dieser Parameter wird in folgenden Dialogen berücksichtigt: 

- Einkauf: Rechnungsprüfung , Felder R-Datum / Valuta , Registerkarte Kopfdaten Feld Buchungsdatum - Verkauf : Massendruck , Felder Valuta und Druckdatum - Fibu: Übernahme , Feld Buchungsdatum - Fibu: Kreditorenvorerfassung (bei Datenübernahme) - Kore: Übernahme Feld Buchungsdatum
