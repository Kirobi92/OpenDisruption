Finanzbuchhaltung / Konto Information 

# Konto Information Registerkarte Zinsrechnungen 
Konto Info Registerkarte Zinsrechnungen 

Auf dieser Registerkarte erhalten Sie eine Auflistung der Zinsrechnungen des Debitors. 
### Tabelle 

### Tabellenspalte Zinslauf 
Hier wird der Zinslauf angegeben. 

Tooltip: 

Angabe des Zinslaufs 

### Tabellenspalte S 
Hier wird der Verarbeitungsstatus angegeben. Zur Verfügung stehen: 

- = offen 
OK = erledigt 
VB = verbucht 
X = Fehler 

Tooltip: 

Verarbeitungstatus 
- = offen 
OK = erledigt 
VB = verbucht 
X = Fehler 

### Tabellenspalte Verarbeitungsdatum 
Hier wird ein Verarbeitungstag angegeben. 

Tooltip: 

Angabe des Verarbeitungstags 

### Tabellenspalte Rechn.-Datum 
Hier wird das Datum der Rechnung angegeben. 

Tooltip: 

Angabe des Fakturadatums 

### Tabellenspalte Gesamt 
Hier wird der Gesamtbetrag angegeben. 

Tooltip: 

Angabe des Gesamtbetrags 

### Tabellenspalte Zinsbetrag 
Hier wird der Zinsbetrag angegeben. 

Tooltip: 

Angabe des Zinsbetrags 

### Tabellenspalte Währung 
Hier wird das Währungskürzel angegeben. 

Tooltip: 

Währungskürzel
