# Kapitel NV.ERP.MM.Sales.cdOrderType.shtIsTransferSalesOrder

Tooltip: 

Kennzeichnet die Belegart als Belegart für Umlagerungen. So gekennzeichnete Belegarten können im Dialog Lagerorte, Registerkarte Automatisierung für interne Umlagerungen ausgewählt werden. 
Bei Nutzung von Multicompany in den Parametern Gesellschaft ist das Kennzeichen für die Belegarten für interne und externe Umlagerungen notwendig. Nur so gekennzeichnete Belegarten können dort ausgewählt werden.
