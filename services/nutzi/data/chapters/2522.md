# Konten Rechnen 
Konten Rechnen 

Dieser Dialog öffnet sich in der Saldenliste >> Optionen >> Konten Rechnen . 

Bei evtl. Differenzen in den Saldenlisten besteht im Menü Konten Rechnen die Möglichkeit, diese Konten neu zu rechnen. 

Zuerst wählen Sie aus, ob es sich um Sachkonten, Debitoren oder Kreditoren handelt. Falls Ihnen die Kontonummer bekannt ist, geben Sie diese in das Feld Konto ein. 

Sie können den OP-Journal-Abgleich in Landeswährung oder Fremdwährung ausführen. Dazu aktivieren Sie den entsprechenden Radiobutton. 

Mit bestätigen Ihrer Eingaben über den Button werden Journalbuchungen für die betreffenden Konten neu hochgerechnet. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Konten Rechnen 
Tooltip: 

In diesem Dialog können Konten neu berechnet werden, z.B. bei Differenzen in den Saldenlisten. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist.

Mit diesem Button leeren Sie die Felder.

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Radiobutton OP/Journal Abgleich 
Wenn Sie diesen Radiobutton aktivieren, werden Personenkonten mit Differenzen zwischen OP und Journal ausgewiesen. Falls Differenzen vorhanden sind, erscheint eine entsprechende Meldung, die den Differenzbetrag anzeigt. 

### Dock Panel Kontentyp 

Hier wählen Sie den Kontotyp aus. 

### Combobox Jahr 
Hier wird das (Geschäfts-) Jahr angegeben. 

### Radiobutton Landeswährung 
Wenn Sie diesen Radiobutton drücken, wird der OP-Journal-Ausgleich in Landeswährung ausgeführt. 

### Radiobutton Fremdwähr. 
Wenn Sie diesen Radiobutton drücken, wird der OP-Journal-Ausgleich in Fremdwährung ausgeführt. 

### Checkbox Alle Konten 
Wenn Sie diese Checkbox aktivieren und den Button betätigen, werden Ihnen sämtliche Konten in der Tabelle angezeigt. 

### Button Suche 

Über diesen Button können Sie ein Konto suchen. 

Siehe Kapitel Kontoauswahl .
