# Kasseneinstellungen Registerkarte Registrierkasse 
Kasseneinstellungen Registerkarte Registrierkasse 

Auf dieser Registerkarte nehmen Sie die Einstellungen für die Registrierkasse vor; auch in Bezug auf die Kassensicherheitsverordnung. 
### Checkbox KassenSichV - Pflicht 
Wenn diese Checkbox aktiviert ist, unterliegt die gewählte Kasse der Kassensicherungsverordnung (KassenSichV). Diese Einstellung ist wesentlich für die Arbeit mit DSFinV-K! 

Wenn die Checkbox aktiviert ist, wird die Combobox TSE zugänglich und Sie können eine eindeutige ID für eine technische Sicherheitseinrichtung zuweisen. Die ID führt zu einer eindeutigen Identifikation und Zuordnung von Transaktionen auf eine TSE. 

Tooltip: 

Festlegung, ob diese Kasse der Kassensicherungsverordnung (KassenSichV) unterliegt. 

### Combobox TSE 
Hier wählen Sie eine eindeutige ID für eine technische Sicherheitseinrichtung aus. 

Die ID führt zur eindeutigen Identifikation und Zuordnung von Transaktionen auf eine TSE. 

Tooltip: 

Eindeutige ID für eine technischen Sicherheitseinrichtung. 
Die ID führt zur eindeutigen Identifikation und Zuordnung von Transaktionen auf eine TSE.
