# Einzugsparameter 
Einzugsparameter 

Dieser Dialog öffnet sich im Navigator >> System >> Finanzbuchhaltung >> Einzugsparameter und im Bankeinzug über den Button Parameter . Hier werden die grundsätzlichen Selektionskriterien im Bereich ‚Debitoren‘ hinterlegt (analog den Zahlungsparametern). Auch hier wird aufgrund der Fälligkeit der offenen Posten und der hier hinterlegten Kriterien ein Vorschlag im Bankeinzug erzeugt. 

Die allgemeinen Einstellungen für einen Versand der Anschreiben per Email können hier ebenfalls definiert werden.

### Combobox Zahlbed. Teilz. 
Für die Anlage eines OPs aus einer Teilzahlung können Sie eine Zahlungsbedingung dazu hinterlegen. Dafür steht diese Combobox zur Verfügung. Hier wird dann eine Zahlbedingung ohne Skonto ausgewählt. 

Ist hier eine Zahlungsbedingung hinterlegt, wird für die OP-Anlage aus der Teilzahlung diese Zahlungsbedingung genutzt. 

Ist hier keine Zahlungsbedingung hinterlegt, wird der Sachbearbeiter, wenn er eine Teilzahlung durchführt, mit einer Meldung darauf hingewiesen, dass dieses Feld nicht gefüllt ist. Wird diese Meldung ignoriert, wird für die OP-Anlage die Zahlungsbedingung aus dem Kundenstamm benutzt. 

Die Teilzahlung wird immer auf dem Ursprungs-OP verdichtet in den OP-Zusatz-Infos „Bereits bezahlt, Skto. gebucht, Ausb. gebucht, Noch offen“. In manchen Vorschlagsdialogen werden diese OP-Zusatz-Infos auch noch weiter zusammengefasst auf Erforderlich und Noch offen. In den Vorschlagsdialogen ist die Zahlungsbedingung des Ursprungs-OP ausschlaggebend. 

Sie erhalten einen Hinweis, wenn Sie die Zahlungsbedingung zu Teilzahlungen ändern. 

Tooltip: 

Wählen Sie eine Zahlungsbedingung für die Anlage eines OPs aus einer Teilzahlung.

### Feld Druck Avis ab 
Mit einem Eintrag in diesem Feld (Anzahl der OPs) können für Bankeinzug und Zahlungsverkehr die Einstellungen aus den Parametern übersteuert werden. Auf diese Weise kann definiert werden, ob und ab welcher Position ein Avis gedruckt werden soll. 

Das Avis wird innerhalb einer Kontonummer, gruppiert nach Zahlungsverbindung, gedruckt. 

In diesem Fall kann festgelegt werden, ab welcher Anzahl offener Posten ein Zahlungsavis beim Bankeinzug gedruckt wird. 

Wurde in diesem Feld kein Eintrag vorgenommen, ermittelt eNVenta ERP anhand der Länge des Verwendungszwecks automatisch den Avis-Druck. 

Der Wert 0 oder Null in diesem Feld bewirkt, dass bei ausreichend Platz im Verwendungszweck kein Avis erstellt wird. Ansonsten wird evtl. bereits schon ab einem OP ein Avis erstellt. 

Im Zahlweg der Firmen-Bankverbindung wird die Länge des Verwendungszwecks hinterlegt (in der gleichlautenden Tabellenspalte). 

Siehe Kapitel Zahlweg . 

Tooltip: 

Druck Avis ab ... OPs 

### Feld Fällig bis Tage 
Über dieses Feld können Sie festlegen, dass Rechnungen automatisch im Bankeinzug erscheinen, die während der hier eingetragenen Periode fällig werden. 

Tooltip: 

Ausgehend vom Tagesdatum erhöhen/verringern die hier erfassten Tage das Datum im Vorschlagsdialog. 

### Checkbox Eingabe des Zahlbetrags 
Wenn Sie diesen Parameter aktivieren, wird der Zahlbetrag eingegeben. 

- Vollausgleich : Zahlbetrag = OP-Betrag abzüglich Skonto - Teilzahlung : Zahlbetrag zuzügliche Skonto = neuer OP 

Tooltip: 

Eingabe des Zahlbetrags: 
- Vollausgleich: Zahlbetrag = OP-Betrag abzüglich Skonto 
- Teilzahlung: Zahlbetrag zuzüglich Skonto = Neuer OP 

### Checkbox Gutschrift sofort fällig 
Ist diese Checkbox gesetzt, sind Gutschriften immer sofort fällig und werden im Einzugsvorschlag sofort berücksichtigt, unabhängig von der Zahlungsbedingung (Fälligkeit). 

Für die Guthabenauszahlung hat diese Checkbox keine Relevanz. Die Gutschriften werden strikt nach Fälligkeiten berücksichtigt. 

Tooltip: 

Gutschriften sind sofort fällig.
