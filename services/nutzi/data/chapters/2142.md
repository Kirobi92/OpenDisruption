# Adresse 
Adresse 

Das Menü Adresse ist vom Aufbau her dem Menü Kunde sehr ähnlich. 

Der Adress-Stamm bietet die Möglichkeit, parallel zu den Kunden die Interessenten zu erfassen und zu verwalten. Die Eingabe der Daten in die vorgesehenen Felder der Bildschirmmaske kann in beliebiger Reihenfolge vorgenommen werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Adresse 
Tooltip: 

In diesem Dialog können Interessenten erfasst werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Feld USt-IdNr. 
Hier wird die Umsatzsteuer-Identifikationsnummer des Kunden angegeben. 

Ist die Funktion System >> Parameter Allgemein >>UST-ID via WS prüfen aktiviert, so wird die Umsatzsteuer-Identifikationsnummer bei der Eingabe in den Dialogen Kunde, Lieferant, Angebot, Auftrag, Gutschrift, Bestellung etc. per Webservice geprüft. 

Achtung: Diese Prüfung ist rechtlich nicht bindend. Eine rechtlich bindende Prüfung der Umsatzsteuer-Identifikationsnummer kann nur schriftlich bei der zuständigen Behörde erfolgen. 

Siehe Kapitel Parameter Allgemein .

### Menü Optionen / Export als Outlook-Kontakte 
Um Kontaktdaten in Microsoft Office zu exportieren, steht Ihnen im Menü Optionen der Eintrag Export als Outlook-Kontakte zur Verfügung. Dabei werden alle Einträge der Registerkarte Tabelle übertragen. 

Beim Export werden Sie gefragt, ob Sie die Ansprechpartner mit exportieren möchten. Diese werden als zusätzliche Kontakte ins Outlook exportiert. Die Einträge stehen Ihnen im Menü Kontakte im Outlook zur Verfügung. 

Wenn der Export erfolgreich war, erhalten Sie eine Meldung. 

Enthält eine Tabelle mehr als 30 Einträge, werden Sie mit einer Messagebox über die Dauer der Laufzeit informiert, falls der Export mehr als 2 Minuten dauert.
