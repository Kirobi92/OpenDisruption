# Archiv-Zuordnung 
Archiv-Zuordnung 

Dieser Dialog öffnet sich im Dialog Archiv Suche über den Button Arc. Zuordnung . Hier werden die logischen Archive zugeordnet, die in diesem Dialog zur Verfügung stehen sollen. 

So können Sie zum Beispiel im Call-Center das logische Archiv „CallCenter“ als „Std-Einfügen-Archiv“ heranziehen, gleichzeitig zur Suche aber auch das logische Archiv „Verkauf“ mit einbinden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Archiv-Zuordnung 
Tooltip: 

In diesem Dialog werden die logischen Archive zugeordnet, die in der Archivsuche zur Verfügung stehen sollen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Mit diesem Button lässt sich ein Datensatz kopieren. Wenn ein Eintrag kopiert wird, werden auch die zugeordneten Schlagworte und die Zuordnung zum physikalischen Archiv mitkopiert. Dies erleichtert es bei mehreren Formularen, z.B. im Auftragswesen, pro Formular die korrekte Archivierung einzustellen. 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Schlagworte 

Dieser Button öffnet den Dialog Schlagwort-Zuordnung , um Schlagworte zuzuordnen. 

Siehe Kapitel Schlagwort-Zuordnung . 

### Button Mapping 

Dieser Button öffnet den Dialog Log. Archiv-Zuordnung zur Zuordnung eines Archivs. 

Siehe Kapitel Mapping / Verbinden eines logischen Archivs mit den Archiven aus dem Dokumenten-Server .
