# Kapitel NV.ERP.Base.Common.cdSystemMM.shtTotalValuesRounding

Mit dieser Checkbox wird ein alternatives Positionskalkulationsverfahren aktiviert: 

Falls die Checkbox gesetzt, so werden vorab alle Prozentrabatte in den Preis eingerechnet und mit diesem der Positionsbetrag ermittelt. D.h., alle Rabatte werden abgezogen und das Endergebnis gerundet. 

Andernfalls wird schrittweise diskontiert, wobei für alle Prozentrabatte im Hintergrund ein monetärer Äquivalentbetrag ermittelt wird. Nach jedem Rabattabzug wird das Ergebnis gerundet und mit dem gerundeten Zwischenergebnis weitergerechnet.
