# Zusätzliche Anschriften Registerkarte Allgemein 
Zusätzliche Anschriften Registerkarte Allgemein 

Auf dieser Registerkarte legen Sie weitere Informationen an, wie z.B. die GEO-Koordinaten oder das MwSt-Kennzeichen. 

Pro Adresstyp kann eine Adresse des Kunden als "Standard" (gleichnamige Checkbox) markiert werden. 
### Checkbox Standard 
Wenn Sie diese Checkbox aktivieren, kann die Adresse als "Standard" gekennzeichnet werden. 

Diese Standard-Adresse wird beim Speichern eines Auftrags automatisch als "zusätzliche" Adresse eingetragen.
