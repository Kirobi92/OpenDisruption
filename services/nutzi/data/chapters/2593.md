# Verarbeitete Mahnvorschläge löschen 
Mahnvorschläge löschen 

Dieser Dialog öffnet sich im Mahnwesen >> Optionen >> Verarbeitete Mahnvorschläge löschen . Hier können verarbeitete Mahnvorschläge endgültig aus der Datenbank gelöscht werden. Alle erfolgreich verarbeiteten Vorschläge werden in der Datenbank gespeichert, damit diese jederzeit geprüft werden können. Aber es besteht die Möglichkeit automatisch alle verarbeiteten Vorschläge komplett aus der Datenbank zu entfernen. 

Sie können entweder ganz allgemein über den Button suchen oder in der Combobox Sachbearbeiter einen Sachbearbeiter auswählen, dessen Vorschläge gelöscht werden sollen. Ebenso kann über die Vorschlagsnummer gesucht werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Verarbeitete Mahnvorschläge löschen 
Tooltip: 

In diesem Dialog können alte Mahnvorschläge endgültig aus der Datenbank gelöscht werden. 
Für mehr Informationen drücken Sie F1.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Combobox Sachbearbeiter 
Aus dieser Combobox wählen Sie den Sachbearbeiter aus, dessen Mahnvorschläge gelöscht werden sollen. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
