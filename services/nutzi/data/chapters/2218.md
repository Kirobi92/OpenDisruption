# Call Registerkarte Tabelle 
Call Registerkarte Tabelle 

Auf dieser Registerkarte erscheinen sämtliche Calls, die zum ausgewählten Suchkriterium im Kopf der Maske erfasst wurden. 
### Tabelle 

### Tabellenspalte Call 
Hier wird die Nummer des Calls angegeben. 

Tooltip: 

Angabe der Call-Nummer 

### Tabellenspalte Call-Betreff 
Hier wird der Betreff des Calls angegeben (Thema). 

Tooltip: 

Angabe des Call-Betreffs (worum gehts?) 

### Tabellenspalte AKL-Typ 
Hier wird der AKL-Typ angegeben. 

Zur Verfügung stehen: 
A = Adresse 
K = Kunde 
L = Lieferant 
V = Vertreter 
P = Kontakt 

Tooltip: 

AKL-Typ: 
A = Adresse 
K = Kunde 
L = Lieferant 
V = Vertreter 
P = Kontakt 

### Tabellenspalte AKL-Nummer 
Hier wird eine Adress-/Kunden- oder Lieferantennummer angegeben. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Tabellenspalte Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Call-Art 
Hier wird die Call-Art angegeben. 

Tooltip: 

Call-Art 

### Tabellenspalte Kategorie 
Hier wird der Name der Call Kategorie angegeben. 

Tooltip: 

Angabe der Call-Kategorie 

### Tabellenspalte Priorität 
Hier wird die Call-Priorität angegeben. 

Tooltip: 

Angabe der Call-Priorität 

### Tabellenspalte Status 
Hier wird der Status des Calls angegeben. 

Tooltip: 

Angabe des Call-Status 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
In diesem Feld wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
