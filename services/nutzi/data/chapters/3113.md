# Import Editor - Preisgruppen 
Import Editor - Preisgruppen 

Wenn Sie den Button Bearbeiten im Dialog Import Bearbeiten auf der Registerkarte Preisgruppe klicken, öffnet sich dieser Dialog. In diesem Dialog können auf 'Staging'-Ebene die Daten nochmals geprüft ober nachbearbeitet werden, bevor sie in den Artikel Schattenstamm übernommen werden. 

Über den Status auf der rechten Seite wird angezeigt, welche Werte geändert bzw. ignoriert werden. Beim Speichern im Dialog werden die Änderungen zurückgeschrieben. 

Es ist besonders wichtig, die Daten hier in diesem Dialog zu prüfen / nachzubearbeiten! Denn wenn die Daten erst mal im Artikel Schattenstamm stehen, sind sie ins Echtsystem eingebunden und bei einem weiteren Import ins Staging ( Artikel Import Manager ) fließen auch diese (evtl. falschen oder unvollständigen) Daten in den Vergleich mit ein. Das sollte nicht geschehen! 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Import Editor - Preisgruppen 
Tooltip:

Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt.
