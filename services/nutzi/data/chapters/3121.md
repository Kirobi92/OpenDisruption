# Leistungsstatistik Registerkarte Statistik 
Leistungsstatistik Registerkarte Statistik 

Diese Registerkarte wird entsprechend der Selektionskriterien mit Daten gefüllt. 
### Feld Titel 
Hier wird der Titel der Leistungsstatistik angegeben. 

Tooltip: 

Angabe des Titels der Leistung 

### Feld Von / Bis 
Nachdem Sie im Kopf des Dialogs einen Zeitraum ausgewählt haben und den Button drücken, wird dieser Datumsbereich hier eingetragen. 

### Feld Von / Bis VZ 
Nachdem Sie einen Zeitraum im Kopf des Dialogs ausgewählt haben und den Button klicken, wird hier die entsprechende Vorperiode automatisch eingetragen. 

Sollte bei dem Auswertungszeitraum ein Zeitraum größer 1 Jahr ausgewählt werden, so wird der Vorzeitraum(VZ) zum Datum von/bis minus dem Auswertungszeitraum. 

Beispiel: 
Auswertungszeitraum von 01.01.2016 - 31.12.2020 
Vorjahreszeitraum ist nun 01.01.2011 - 31.12.2015 

Tooltip: 

Zeigt das von/bis Datum des Vorzeitraums (Vergleichszeitraum). 

Sollte beim Auswertungszeitraum ein Zeitraum größer 1 Jahr ausgewählt werden, so wird der Vorzeitraum(VZ) zum Datum von/bis minus des Auswertungszeitraums gerechnet. 
z.B. 
Auswertungszeitraum von 01.01.2016 - 31.12.2020 
Vorjahreszeitraum ist nun 01.01.2011 - 31.12.2015
