# Update-Skripte 
Update-Script 

Dieser Dialog öffnet sich im Dialog Update-Assistent über den Button Details . Hier können die Update-Scripte auf die jeweilige DB angepasst werden. Mit dem Button Ausführen wird der Vorgang durchgeführt. 

Mit dem Button Ausgeführt wird die Durchführung manuell bestätigt. Der Haken in der Spalte ‚Ausgeführt‘ wird gesetzt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Update-Script 
Tooltip: 

In diesem Dialog können die Update-Scripte auf die jeweilige DB angepasst werden. 
Für mehr Informationen drücken Sie F1. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. 

### Button Ausführen 

Dieser Button führt den gewählten Befehlsblock aus. 

Tooltip: 

Führt den ausgewählten Befehlsblock aus. 

### Button Ausgeführt 

Dieser Button setzt den aktuellen Befehlsblock auf ausgeführt. 

Tooltip: 

Setzt den aktuellen Befehls-Block auf 'ausgeführt'. 

### Tabelle 

### Tabellenspalte Ausgeführt 
Mit dem Button Ausgeführt wird die Durchführung manuell bestätigt und der Haken in dieser Spalte wird gesetzt. 

### Tabellenspalte Datum 
Hier wird das Ausführungsdatum angezeigt. 

### Tabellenspalte Mandanten-Key 
Hier geben Sie den Mandantenschlüssel an, wenn mit Mandanten-Datenbanken gearbeitet wird. 

Tooltip: 

Angabe des Mandanten-Schlüssels 

### Tabellenspalte Typ 
Ist das Update-Skript vom Typ DML (DML = Data Manipulation Language), wird dies in dieser Spalte angezeigt. 

### Tabellenspalte Tabelle 
Hier wird die betroffene Datenbanktabelle angezeigt. 

Tooltip: 

Angabe der DB-Tabelle 

### Tabellenspalte Info 
Hier wird ein Text zur Information angezeigt, z.B. neue Felder, Update Bestandskunden. 

Tooltip: 

Angabe einer Information 

### Tabellenspalte Script ID 
Hier wird die Skript-ID ausgewiesen.
