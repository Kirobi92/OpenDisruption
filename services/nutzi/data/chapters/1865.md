# Wareneingang Registerkarte Zerlegte SN 
Wareneingang Registerkarte Zerlegte SN 

Enthält der Wareneingang Stücklisten mit Seriennummern und Sie möchten eine Position zerlegen, erhalten Sie die Meldung, dass die Seriennummern zu prüfen sind. Dafür nutzen Sie diese Registerkarte. Für die Zerlegung der Stücklisten muss die 'Menge_Zerlegen' in der Bestellposition vor dem Wareneingang entsprechend gepflegt sein. 

Auf der Registerkarte Zerlegte SN sehen Sie die Artikel und die originale Stückliste, die zerlegt wurde. Die Seriennummern werden über den Button Seriennr. generiert. 
### Button Aktualisieren 

Mit diesem Button können Sie für einen Wareneingang prüfen, ob noch Zuordnungen getätigt werden müssen. Es erscheinen nur Positionen der Stücklisten, welche noch keine vollständige Seriennummern-Zuordnung haben. 

Tooltip: 

Mit diesem Button werden alle auf der Registerkarte 'Zerlegte SN' aufgelisteten Positionen neu geladen. 

### Button Seriennummer 

Mit diesem Button werden die Seriennummern generiert. 

Tooltip: 

Öffnet die Seriennummernerfassung für jede aufgelistete Position im Grid.
