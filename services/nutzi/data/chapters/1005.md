# Gutschriftsarten Registerkarte Druck 
Gutschriftsarten Registerkarte Druck 

Zu jeder Gutschriftsart können eigene Formularvarianten erstellt und auf der Registerkarte Druck hinterlegt werden. Verknüpfen Sie die Formulare mit einem Windows-Druckertreiber. Die Drucke werden dann automatisch auf die gewünschten Drucker verteilt. So können beispielsweise die Packliste und/oder der Lieferschein automatisch zu dem im Lager stehenden Drucker geschickt werden. Oder die Auftragsbestätigung wird mit einem Faxdruckertreiber verknüpft und somit automatisch per Fax versendet. 

Was den Druck von Logos etc. auf den Reports betrifft, so ist folgendes zu beachten: 

Es können alle gängigen Bild-Formate (bmp, gif, jpeg, png, tiff, …) ausgewählt werden. 

JPEG, GIF und PNG werden 1:1 gespeichert und auch so an den Report übergeben. 

Alle anderen nicht internetfähigen Formate (bmp, tiff, …) werden verlustfrei in ein PNG umgewandelt. 

Empfehlung: 

Verwenden Sie PNG als Format. 

Vermeiden Sie nach Möglichkeit Bilder auf Subreports.
