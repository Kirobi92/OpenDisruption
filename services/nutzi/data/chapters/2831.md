# Preisoptimierung Registerkarte Konversionsratenanalyse 
Preisoptimierung Registerkarte Konversionsratenanalyse 

Hier werden die Artikelvorschläge nach dem Verfahren der Konversionsratenanalyse angezeigt. Die Funktionalität ist analog der Registerkarte Absatz-Trendanalyse . Anstatt der Balkengraphik der Absatzmengen wird hier der Verlauf der letzten Konversionsraten visualisiert. Es ist möglich, dass Artikel auf beiden Registerkarten angezeigt werden, wenn sie jeweils die Kriterien erfüllen. 
### Tabelle 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung dient der einfachen Identifikation des Artikels und kann auch zur Suche verwendet werden. 

Bei Eingabe der Artikelnummer wird das Feld Bezeichnung automatisch mit dem Namen des Artikels gefüllt. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Artikelgruppe 
Hier wird der Name der Artikelgruppe angegeben. 

Tooltip: 

Bezeichnung der Artikelgruppe 

### Tabellenspalte Konversionsraten 
Hier wird die Konversionsrate angegeben. 

### Tabellenspalte Trendstärke 
Hier wird die Konversion-Trendstärke angegeben. 

### Tabellenspalte Hauptlieferant 
Hier wird die Nummer des Hauptlieferanten für den Artikel angegeben. Dieser Lieferant wird dann im Bestellwesen als Lieferant vorgeschlagen. 

Tooltip: 

Nummer des Hauptlieferanten des Artikels. 

### Tabellenspalte Firma 1 
Die Firmenbezeichnung wird automatisch vom System eingetragen, nachdem die Lieferantennummer eingetragen wurde. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Preisänderung 
Hier wird das Datum angegeben. 

Tooltip: 

Datum der letzten Preisoptimierung 

### Tabellenspalte Betrag Erfolg 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Agent ID 
Hier wird der Agent angegeben. 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status

Hier wird das Datum angegeben.
