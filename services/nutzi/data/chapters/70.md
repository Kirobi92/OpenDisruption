# Beispiel für ein Fokus-Element 

## Parkplatz 
Mit Hilfe des Fokus-Elements Ordner können individuelle organisatorische Hilfsmittel in Form von Ordnern und Ablagen für Vorgänge und Objekte geschaffen werden. Objekte und Vorgänge werden einfach per Drag&Drop auf den Ordner gezogen und damit für eine spätere Verwendung gespeichert (geparkt). Die Vorgänge bzw. Objekte können aus dem Dialog, aus der Wiedervorlage oder sogar aus dem Verlauf heraus in das Fokus-Element übernommen werden. Auf diese Weise kann z.B. die Wiedervorlage aufgeräumt werden und Sie behalten trotzdem einen wichtigen Vorgang im Blick. Die Elemente im Ordner bleiben auch nach Beendigung von eNVenta ERP erhalten und können dann am nächsten Tag wiederverwendet werden. 

Auf diese Weise lassen sich beispielsweise zu bearbeitende Vorgänge und Stammdaten vorübergehend in einen dazu vorgesehenen Ordner (Parkplatz) ablegen. Sie können mehrere solche Fokus-Elemente vom Typ „Ordner“ anlegen und diese entsprechend für Ihren Verwendungszweck benennen. 

Die Objekte können Sie per Doppelklick jederzeit aufrufen oder auch in weiteren Vorgängen verwenden. Es besteht z.B. die Möglichkeit, den Kunden, der auf dem Parkplatz liegt, in eine geöffnete Marketing-Aktion hineinzuziehen (Drag&Drop). Oder es können ein Auftrag, eine Auftragsposition hinzugefügt werden. 

Sollten Sie ein Objekt nicht mehr im Ordner benötigen, so können Sie diesen Vorgang über den Button neben dem Fokus-Element löschen. 

Achtung: NICHT den Button Schließen ( X ) oben rechts verwenden! Damit würden Sie das Fokus-Element mit allen Vorgängen löschen. 

Soll eine Ablage nur temporär genutzt werden, um sich einen Vorgang oder Objekt zu merken, kann auf eine Speicherung der Daten verzichtet werden. Hierzu muss in den Einstellungen des Fokus-Elements die Checkbox nicht speichern aktiviert werden. Mit dem Beenden von eNVenta ERP werden diese Objekte dann aus dem Ordner gelöscht. 

## Vorgehensweise 
Gehen Sie dazu auf die Registerkarte Fokus . 

Hier drücken Sie den Button . Es öffnet sich der Dialog Fokus-Elemente . Hier wählen Sie Ordner aus und übernehmen diese Auswahl mit dem Button . 

Danach öffnet sich der Dialog Einstellungen Fokus Ordner/Ablage . 

Hier können Sie dem Ordner einen Titel geben, z.B. Parkplatz.
