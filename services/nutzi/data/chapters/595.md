# Referenz-Link-Typen 
Referenz-Link-Typen 

Dieses Menü finden Sie unter System >> Allgemein . Hier legen Sie Ihre Referenz-Link-Typen an, welche den einzelnen Referenz-Links zugeordnet werden können. Auf diese Weise können Sie leicht ersichtlich zwischen Werkszeugnissen, Prüfzeugnissen und weiteren Dokumenttypen unterscheiden. 

Referenz-Links stehen Ihnen u.a. in den Menüs Call-Center, Projekte, Chargen, Auftrag, Angebot, Gutschrift und in den Archiv-Mappen zur Verfügung. 

Der Dialog Referenz-Links öffnet sich in den einzelnen Menüs über den Button (bzw. <Umschalt+Ctrl+L> ). 

Siehe Kapitel Referenz-Links .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

### Tabellenspalte Verw.zweck 
Hier können Sie einen Verwendungszweck auswählen. 

Wenn Sie hier "Bild" auswählen, werden diese Typen von eNVenta erkannt und können so von Bildern anderer Links unterschieden werden. 

Für ein Bild ist der Verwendungszweck ‚Bild‘ unbedingt zu setzen. Nur so bleiben die Daten bei der Speicherung korrekt sichtbar und werden entsprechend gespeichert. 

Die Liste der Verwendungszwecke ist durch das Programm vorgegeben und kann durch Customizing erweitert werden. 

Tooltip: 

Der Verwerwendungszweck wird zum Teil innerhalb von eNVenta genutzt und ausgewertet. 

### Tabellenspalte Archiv 
Über die Combobox in dieser Spalte ist es möglich, Dokumenten bestimmter Referenz-Link-Typen automatisch ein Archiv zuzuordnen. 

Ist ein Archiv ausgewählt, so wird die Datei automatisch ohne folgende Benutzer-Interaktion in das Archiv eingefügt. 

Druck und Mailversand 
Wenn ein automatischer Druck aktiviert wird und die Daten befinden sich im Archiv, so werden die Daten aus dem Archiv aufgerufen und an den Drucker übergeben. 

Bei automatischem Mailversand muss das Archiv als Anhang zur Mail hinzugefügt werden.
