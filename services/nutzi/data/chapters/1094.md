# Positionsfreigabe 
Positionsfreigabe 

Dieser Dialog öffnet sich über den Button Freigabe im Auftrag auf der Registerkarte Positionen . 

Mit den zur Verfügung stehenden Buttons haben Sie verschiedene Möglichkeiten der Freigabe von Positionen. 

Für manche Prozesse kann es sinnvoll sein, Artikel-Set-Positionen in den Status 3 (Fakturafreigabe) zu setzen, obwohl diese zu diesem Zeitpunkt noch nicht geliefert sind. Dazu kann eine teilweise oder vollständige Fakturafreigabe von Artikel-Set-Komponentenpositionen zu jeder Zeit erlaubt werden. Die Fakturafreigabe erfolgt dann mit der Lieferung der letzten Komponentenposition. 

Ebenso ist es nun möglich eine Auftragsposition zu splitten, welche bestellt, aber noch nicht geliefert wurde. In diesem Fall kann auch nur die Menge im Feld Menge geliefert maximal gesplittet werden. 

Bei der Lieferfreigabe der Auftragspositionen findet auch eine Prüfung hinsichtlich der Verfügbarkeit der beauftragten Menge statt, wenn dies in der Auftragsart eingestellt ist. Ansonsten findet die Prüfung bei der Erfassung der Positionen statt. 

Eine Lieferfreigabe kann nicht mehr zurückgenommen werden, wenn der Lieferschein bereits gedruckt wurde. 

Zur Erzeugung einer Teillieferung gibt es eine weitere Möglichkeit, die vor allem für Abrufaufträge von Interesse ist: 

Jede Artikelposition enthält 2 Mengenfelder: 
- bestellte Menge - gelieferte Menge 
Diese beiden Felder finden Sie im Dialog Position (Button Ändern im Auftrag auf der Registerkarte Positionen ). Bei der Erfassung der Auftragsposition werden Menge_bestellt und Menge_geliefert gleichgesetzt. Die Menge "zu liefern" kann nachträglich noch angepasst werden.

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Lieferung 

Dieser Button gibt alle selektierten Positionen zur Lieferung frei. 

### Button Lieferung alle 

Dieser Button gibt alle Positionen des Auftrags zur Lieferung frei. 

### Button Faktura 

Dieser Button gibt alle selektierten Positionen zur Faktura frei. 

Ein Prozess-Administrator kann die Fakturafreigabe des Auftrags bereits vor Buchung des Wareneingangs durchführen. Ebenso ist dies möglich bei einer Einkaufsart ohne Lagerbuchung oder Bestellposition mit Status > 3. 

### Button Faktura alle 

Dieser Button gibt alle Positionen des Auftrags zur Faktura frei. 

### Button Aufheben 

Dieser Button nimmt die Freigabe der selektierten Positionen um eine Stufe zurück. 

### Checkbox Nur gelieferte 
Ist dieser Parameter gesetzt, können nur Positionen, die bereits geliefert wurden, zur Fakturierung freigegeben werden. Dieser Parameter ist beim Öffnen des Dialogs immer gesetzt, kann aber jederzeit verändert werden.
