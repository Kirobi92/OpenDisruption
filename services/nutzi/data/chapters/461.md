# eNVenta Init Registerkarte Export 
eNVenta Init Registerkarte Export 

Auf dieser Registerkarte stehen Ihnen die beiden Buttons zum Starten des Exports zur Verfügung. 
### Button Rep. Export 

Mit diesem Button wird der Export gestartet. Dabei werden eine XML-Datei und pro Checkbox eine CSV-Datei erstellt. Je nach Auswahl der Menge kann der Export einige Minuten andauern. 

Tooltip: 

Start Export der Html Reports 

### Button Konfig. Exp. 

Mit diesem Button starten Sie den Export der Parameter Konfigurationsdaten . 

Siehe Kapitel Parameter Konfigurationsdaten . 

Tooltip: 

Start Export der Parameter Konfigurationsdaten
