Produktion (PPS) / Betriebsauftrag 

# Betriebsauftrag Registerkarte Arbeitsplan 
Betriebsauftrag Registerkarte Arbeitsplan 

Auf dieser Registerkarte werden alle Arbeitsgänge aus dem Arbeitsplan angezeigt und zwar in der Reihenfolge, wie diese abgearbeitet werden. Ein Arbeitsplan kann pro Stückliste im Arbeitsplanstamm hinterlegt werden und wird bei Einfügen des Betriebsauftrags automatisch eingelesen. 

Siehe Kapitel Arbeitsplan . 

Eine Terminierung kann erst erfolgen, wenn ein Arbeitsplan erstellt wurde. 
### Button Neu 

Dieser Button öffnet den Dialog Auswahl Arbeitsgang , um einen Arbeitsgang auszuwählen. 

Siehe Kapitel Arbeitsgang auswählen . 

### Button Ändern 

Mit diesem Button können Sie den Arbeitsgang ändern. 

Siehe Kapitel Arbeitsgang ändern . 

### Button Löschen 

Mit diesem Button können Sie einen Arbeitsgang löschen. 

### Button Alternative 

Mit diesem Button können Sie ebenfalls einen Arbeitsgang auswählen. 

Siehe Kapitel Arbeitsgang auswählen . 

### Button Ressourcen 

Mit diesem Button öffnet sich der Dialog BA-AG Ressourcen und bietet eine Übersicht über Zuordnungen zu einem Arbeitsgang. 

Siehe Kapitel Ressourcen für den Arbeitsgang . 

### Button Material 

Über diesen Button können Sie Material zuordnen. 

Siehe Kapitel Materialzuordnung . 

### Button BDE 

Über diesen Button erhalten Sie eine Übersicht zu den Betriebsdaten. 

Siehe Kapitel BDE Buchung . 

### Button Disposition 

Über diesen Button wird die Generierung der Bestellung oder Disposition ausgelöst. Dazu öffnet sich der Dialog Arbeitsgänge disponieren . 

Siehe Kapitel Arbeitsgänge bestellen/disponieren . 

### Tabelle 

### Tabellenspalte AP Pos 
Hier wird die Positionsnummer angegeben. Diese Nummer muss je Vorgang eindeutig sein! 

Tooltip: 

Positionsnummer 

### Tabellenspalte Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Rüstzeit (mm,Is) 
Hier wird einmalig für einen Arbeitsgang die zu berücksichtigende Zeit in (Echt-)Minuten und Industriesekunden an. 

Rüstzeit (mm=Echtminute, Is=Industriesekunde) 

Zur Information: 

1 Industriesekunde (Is) = 0,006 Echtminuten (mm) 
(z.B: 8 Stunden 45 Minuten = 525 Echtminuten = 87500 Industriesekunden)

### Tabellenspalte Überlappung % 
Hier geben Sie an, ob sich der Arbeitsgang mit dem vorhergehenden Arbeitsgang überlappen darf. Dies dient der Durchlaufzeitverkürzung. 

Tooltip: 

Angabe, ob sich der Arbeitsgang mit dem vorhergehenden Arbeitsgang überlappen darf. 

### Tabellenspalte Faktor Vorlaufzeit 
Hier geben Sie den Faktor an, mit welchem die Vorlaufzeit berücksichtigt wird. Dies ermöglicht eine feinere Gliederung der Produktion pro Betriebsauftrag. 

### Tabellenspalte Faktor Übergang 
Für die Manipulation, der in der Ressourcenverwaltung hinterlegten Übergangszeiten von einer Ressource auf eine andere, kann hier ein Faktor für den Übergang angegeben werden. 

### Tabellenspalte Faktor Nachlaufzeit 
Hier geben Sie den Faktor an, mit welchem die Nachlaufzeit berücksichtigt wird. Dies ermöglicht eine feinere Gliederung der Produktion pro Betriebsauftrag. 

### Tabellenspalte Kostensatz 
Hier wird der Kostensatz für den Arbeitsgang angegeben. 

Tooltip: 

Angabe des Kostensatzes 

### Tabellenspalte BDE Zwang 
Wenn Sie diese Checkbox aktivieren, erfolgt bei Durchlauf des Arbeitsgangs eine BDE-Zeiterfassung. 

Tooltip: 

Aktiviert den BDE-Zwang 

### Tabellenspalte Meilenstein 
Hier wird ein Arbeitsgang als Meilenstein gekennzeichnet. 

Tooltip: 

Kennzeichnet den Arbeitsgang als Meilenstein
