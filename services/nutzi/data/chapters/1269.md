# Retoure Registerkarte Abschluss 
Retoure Registerkarte Abschluss 

Auf dieser Registerkarte schließen Sie die Retourenpositionen ab. Nach Abschluss der Retourenpositionen können keine weiteren Prozesse zu dieser Retoure erstellt werden. 

Voraussetzung für die Retourenabschlussprüfung ist, dass in den Parametern des Verkaufs die Checkbox Retourenabschluss auf der Registerkarte Allgemein gesetzt wird. 

Siehe Kapitel Parameter Verkauf Registerkarte Allgemein . 

Über den Retourenabschluss wird gesteuert, ob der Druck der Belege zur Abholung bzw. Eingangsbestätigung der Retourenpositionen möglich ist. Nur wenn die Positionen abgeschlossen sind, kann die Retourenmenge auf den Belegen bestätigt werden. Die Eingangsbuchung einer Retoure ist unabhängig vom Retourenabschluss. 

Erst nach Abschluss der Retoure ist es möglich eine Gutschrift für den Kunden zu erstellen. Gleiches gilt auch für die Freigabe der Retourenpositionen. Beides soll sicherstellen, dass keine Änderungen an der retournierten Menge mehr vorgenommen werden. 

Bei Retourenabschluss werden folgende Dinge geprüft: 
- Bei gesetzter Option "Abholung" wird geprüft, ob die Adresse der Abholung korrekt ist. - Es wird geprüft, ob die Menge der Retoure größer ist als die Menge_geliefert im Ursprungsauftrag. 
Beide Prüfungen werden als Warnung angezeigt. 

Um auch nach Freigabe einer Retoure (Button Abschluss auf der gleichnamigen Registerkarte) Positionen hinzuzufügen, steht Ihnen der Button Start Proz . zur Verfügung. Für neu hinzugekommene Positionen wird eine neue Belegkette (Verbundprozess) generiert. Dies kann mehrmals geschehen, solange die Retoure nicht vollständig abgeschlossen ist. 

### Button Abschluss 

Dieser Button schließt den Vorgang ab. 

Tooltip: 

Vorgang abschließen 

### Button Start Proz. 

Dieser Button steht zur Verfügung, um auch nach Freigabe einer Retoure neue Positionen hinzuzufügen. 

### Feld Freig. Sachb. 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Freigabedatum 
Hier werden ein Datum und eine Uhrzeit angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit
