# LHM-Typverwaltung

Lagerverwaltungssystem (LVS) 

# LHM-Typverwaltung 
LHM-Typen 

Dieser Dialog öffnet sich im Artikel >> Stamm >> LHM-Typverwaltung und im Lagerauftrag >> Stamm >> LHM-Typen . Hier können Sie Ladehilfsmittel definieren und verwalten. Über die Comboboxen und Felder im Kopf des Dialogs können Sie gezielt ein Ladehilfsmittel suchen. 

Über die Checkbox Intern kann gesteuert werden, ob dieses LHM später auch für das Erzeugen von Packstücken verfügbar ist. 

Ist ein LHM mit Intern gekennzeichnet, wird es nur im Lager verwendet. 

Sobald einmal eine Buchung mit der Emballage stattgefunden hat, ist der Artikel mit seiner „ID“ verbraucht. 

#### Löschen eines LHM-Typen 
Ein LHM-Typ kann gelöscht werden, in dem Sie die Checkbox in der Spalte 'Gelöscht' aktivieren. Gelöschte LHM-Typen stehen dann in anderen Dialogen nicht mehr zur Verfügung. 

Wenn Sie die Checkbox Gelöschte im Kopf des Dialogs aktivieren, können Sie aber weiterhin angezeigt werden. 

Gelöschte Ladehilfsmittel bleiben in der Datenbank mit einem Gelöscht-Kennzeichen stehen. So stehen sie nach wie vor in der Historie oder für Auswertungen zur Verfügung. 

Damit ein LHM-Typ gelöscht werden darf, erfolgt eine Prüfung, ob dieser Typ noch in einer Zuordnung (LHM-LHM-Zuordnung, Artikel-LHM-Zuordnung, LVS Parameter) verwendet wird.
