# Kapitel NV.ERP.Base.Common.cdSystemBase.shtWPPriceOnPurchaseByOrder

Falls diese Checkbox gesetzt ist, so werden im Fall einer auftragsbezogenen Beschaffung - auch bei lagerspezifischem GEK - EK und NK aus Bestellung, Konfektionierung oder Produktion an die Auftragsposition zurück übertragen. 

Ist diese Checkbox nicht gesetzt, so wird immer der lagerspezifische GEK and die Auftragsposition zurück übertragen.
