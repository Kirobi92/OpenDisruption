# Anfrage Registerkarte Anschriften 
Anfrage Registerkarte Anschriften 

Diese Registerkarte ermöglicht die Erfassung und Änderung der Bestell- und Lieferanschrift. 

In der Bestellanschrift wird die Anschrift aus dem Lieferantenstamm automatisch übernommen. In der Lieferanschrift kann manuell eine abweichende Lieferanschrift angegeben werden, um z.B. eine Filiale direkt zu beliefern. 

Grundsätzlich können Bestell- und Lieferanschrift jederzeit überschrieben oder verändert werden, ohne dass damit eine Veränderung im Stammdatensatz stattfindet. 
### Feld Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben.

### Feld GEO-Koordinaten 
Hier werden geografische Daten zur besseren Orientierung angegeben. Damit lassen sich Besuche o.ä. besser planen und Sie können die Karte zur besseren Orientierung und Festlegung einer Besuchs-Reihenfolge nutzen. 

Das Format der GEO-Koordinaten lautet:

Beispiel für Stockach, Nissen&Velten:

### Feld Transport Distanz 
Hier können Sie den Transportweg bis zur Grenze in % angeben. Dies vor dem Hintergrund, dass es sich um einen statistischen Wert für eine ungefähre Hochrechnung handelt. Deshalb muss der Sachbearbeiter hier den wahrscheinlichsten bzw. durchschnittlich zutreffenden Prozentsatz hinterlegen. 

Beispiel: 
Transportweg bis Adresse = 100 km 
Weg bis Grenze = 40 km 
=> Transport Distanz = 40 

Tooltip: 

Der anteilige Transportweg bis zur Grenze in Prozent 
Bsp: 
Transportweg bis Adresse = 100 km 
Weg bis Grenze = 40 km 
=> Transport Distanz = 40 

### Checkbox Adresse gültig 
Diese Checkbox fixiert die von Ihnen eingetragene Adresse, auch wenn diese nicht über den Street Code zur Verfügung steht. Es ist ja möglich, dass sich die Adresse, z.B. in einem Neubaugebiet befindet und die Deutsche Post ihren Street Code noch nicht aktualisiert hat. 

Wird eine Adresse nachträglich geändert, so wird die Checkbox Adresse gültig automatisch wieder geleert, damit eine erneute Validierung der Adresse erfolgen kann. 

Siehe Kapitel Adressprüfung in eNVenta .

Diese Checkbox fixiert die von Ihnen eingetragene Adresse.

### Feld Industrievertreter 
Hier wird die Nummer des Industrievertreters angegeben. 

Ein Industrievertreter entspricht ungefähr einem Vertreter aus dem Verkauf. 
In diesem Fall handelt es sich um die Nummer des Industrievertreters. Die Zuordnung der Industrievertretung wird manuell vorgenommen. In diesem Fall werden im Zuordnungsdialog nur die angezeigt, die auch dem Lieferanten zugeordnet sind. 

Angelegt wird die Industrievertretung im Lieferant unter Optionen >> Industrievertretung . 

Siehe Kapitel Industrievertretung . 

Tooltip: 

Angabe der Lieferantennummer

### Combobox Partner 
Hier wird ein Ansprechpartner angegeben. 

Siehe Kapitel Ansprechpartner . 

In diesem Fall kann aus der Lieferanschrift des Kunden (aus dem zugrundeliegenden Auftrag) ein Ansprechpartner selektiert werden. Dazu muss das Feld Firma 1 im Bereich Lieferanschrift gefüllt sein. Dabei wird der Ansprechpartner aus dem Auftrag übernommen und kann in der Bestellung geändert werden.
