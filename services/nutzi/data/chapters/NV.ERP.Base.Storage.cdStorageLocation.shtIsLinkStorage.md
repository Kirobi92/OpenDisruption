# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtIsLinkStorage

Wenn Sie diesen Parameter aktivieren, wird das gewählte Lager zu einem Link-Lager. Auf einem Link-Lager können Artikel mit unterschiedlichen Qualitäten gelagert werden. 

Hierbei ist es erforderlich pro Qualität jeweils ein virtuelles Lager zu erstellen. Diese Lager werden mit dem eigentlichen Lager über die Comboboxen Link-Lager 1 bis 3 verknüpft. Die virtuellen Lager werden mit dem Parameter „Link-Lager“ gekennzeichnet. 

Qualität 1 (wiederverkäuflich) entspricht dem Link-Lager 1 

Qualität 2 (zu prüfen) entspricht dem Link-Lager 2 (ggf. als Sperrlager zu definieren) 

Qualität 3 (Schrott) entspricht dem Link-Lager 3 

Es ist darauf zu achten, dass jedes LVS-Lager seine eigenen virtuellen Link-Lager hat. 

Siehe Kapitel Link-Lager . 

Tooltip: 

Wenn gesetzt, wird zwischen den verschiedenen Qualitäten differenziert.
