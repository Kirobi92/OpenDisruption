# Archiv-Dokument bearbeiten 
Archiv-Dok. bearbeiten 

Dieser Dialog öffnet sich, wenn Sie eine Datei bearbeiten möchte, die in der Archivsuche liegt. Gehen Sie auf die Registerkarte Ergebnis und markieren Sie die Archiv-Mappe. Damit wird rechts in der unteren Tabelle, die eingefügte Datei eingelesen. Nun wird der Button Dok. Bearb. aktiv. Wenn Sie ihn drücken, erscheint der Dialog Archiv-Dok. bearbeiten . Mit Schließen des Dialogs Archiv-Dok. bearbeiten wird die Datei gespeichert und geschlossen. Siehe nachfolgendes Beispiel. 

Archiv-Dok. bearbeiten - Beispiel 

In diesem Dialog können Sie das Archiv-Dokument bearbeiten. Beim Schließen des Fensters wird das Dokument gespeichert und archiviert. 

Wichtig: Beenden Sie Word nicht, bevor Sie diesen Dialog schließen! Das Speichern der Word-Datei und das Schließen von Word wird beim Schließen dieses Dialogs vreanlasst.
