# Initialisierung

# Initialisierung 
Im Navigator >> System >> Initialisierung stehen Ihnen Menüs zur Verfügung, um mit einer Initialisierung die Grunddaten für die Nutzung von eNVenta ERP anzulegen. Dazu gehören die Dialoge eNVenta Init und Import Items erzeugen . 

Im Regelfall wird die Initialisierung (gemeint ist hier die Zuweisung von Anfangswerten) einmalig gemacht und zwar, wenn Sie die Arbeit mit eNVenta ERP aufnehmen. Bei der Initialisierung geben Sie Ihre Firmendaten, die wichtigsten Sachkonten usw. an. Zusätzlich können Sie auch die Sachkontenrahmen initialisieren, welche mit eNVenta ERP mitgeliefert werden. Siehe auch: 

- eNVenta Init - Import Item Auswahl - Import Items erzeugen - Parameter Konfiguration
