# Rahmenauftrag Registerkarte Texte 
Rahmenauftrag Registerkarte Texte 

Auf dieser Registerkarte kann Text eingetragen werden, differenziert nach Memo-, Start- und Endetext. 

Bei der Generierung eines Auftrags aus dem Rahmenauftrag heraus (Button Generiere oder Menüeintrag Optionen >> Periodisch generieren ) werden der Auftrags-Starttext und Auftrags-Endetext automatisch übernommen. 

Die Texte werden NICHT übernommen, wenn zu einem bereits bestehenden Auftrag eine Position aus dem Rahmenauftrag hinzugefügt wird. 
### Feld Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Feld Auftragsmemo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen. 

### Button Suche 

Dieser Button öffnet den Dialog Textbausteine zur Selektion eines bereits angelegten Textbausteins. 

Siehe Kapitel Textbausteine . 

### Feld Auftrag-Starttext 
Hier kann ein individueller Text als Starttext im Vorgang eingetragen werden. Dieser kann auf den Formularen angedruckt werden. 

Start-Texte sind keine Positionsdaten, sondern Kopfdaten! Deshalb werden sie nur dann übernommen, wenn auch der Kopf des Auftrags aus dem Rahmenauftrag erzeugt wird.
