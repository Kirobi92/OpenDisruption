# Passwort ändern 
Passwort ändern 

Jeder Benutzer des Programms hat die Möglichkeit jederzeit sein Passwort zu ändern. Dies hat unbedingt von Zeit zu Zeit zu geschehen, um Missbrauch zu vermeiden. 

Vor allem der Datenbankadministrator SYSADM hat zwingend das Standardpasswort zu ändern, da bei Schulungen von eNVenta ERP mit dem Standardpasswort des SYSADM gearbeitet wird, um alle Funktionalitäten im Zugriff zu haben. 

Nach Aufruf des Dialogs über das Menü System >> Allgemein >> Passwort ändern muss zunächst das alte Passwort eingegeben werden. Diese zusätzliche Identifizierung stellt sicher, dass kein anderer Benutzer eine unbewachte Gelegenheit zur Passwortänderung nutzt. 

Schließlich wird das neue Passwort zweimal angegeben. Die Eingabe kann 50 Zeichen umfassen. Nur wenn beide Eingaben genau übereinstimmen, wird das neue Passwort akzeptiert. Auf diese Weise kann ein unbemerkter Schreibfehler vermieden werden.
