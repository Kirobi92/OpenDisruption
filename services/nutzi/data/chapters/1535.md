# PTV Depot 02 
PTV Depot02 

Dieser Dialog öffnet sich in der Kommissionierung >> Optionen >> PTV-Schnittstelle >> PTVDepot02 . Nach dem Import der Rückmeldedaten aus PTV konnten evtl. nicht alle Sendungen einem Fahrzeug zugeordnet werden. Z.B. weil es auch günstiger ist eine Sendung über einen Frachtdienstleister zu versenden. 

In diesem Fall werden die Daten nach dem Rückimport auf das Depot02 gesetzt. Hier können Sie für alle Lieferscheine der Tabelle eine neue Versandart vergeben, welche einen Frachtdienstleister hinterlegt hat. Die Einstellung der Versandart ist nur temporär und wird nach der Fakturafreigabe zurückgesetzt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld PTV Depot 02 
Tooltip: 

Nach dem Rücktransport der Daten auf das Depot02 können hier alle Lieferscheine eine neue Versandart erhalten. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
