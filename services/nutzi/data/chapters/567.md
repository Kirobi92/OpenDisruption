# PROXESS Registerkarte Rechnungsprüfung

Grundeinstellungen für eNVenta ERP / PROXESS 

# PROXESS Registerkarte Rechnungsprüfung 
PROXESS Registerkarte Rechnungsprüfung 

Auf dieser Registerkarte geben Sie den Quell-, Ziel- und/oder Fehlerpfad ein. Die -Buttons stehen zur Suche zur Verfügung. 
### Button Ausführen 

Mit diesem Button wird die Datei aus dem Quellpfad eingelesen und die XML-Dateien werden verarbeitet. 

Tooltip: 

Proxess Import starten 

### Feld Quellpfad 
Hier geben Sie den Quellpfad an. D.h. von welchem Server kommt die Datei. 

### Button Suche 

Mit diesem Button können Sie einen Ordner auswählen, um den Pfad des Servers zu hinterlegen. 

Siehe Kapitel Ordner auswählen . 

### Feld Zielpfad 
Hier geben Sie den Zielpfad ein. D.h. wo soll die Datei gespeichert werden, wenn der Import technisch in Ordnung ist. 

### Button Suche 

Mit diesem Button können Sie einen Ordner auswählen, um den Pfad des Servers zu hinterlegen. 

Siehe Kapitel Ordner auswählen . 

### Feld Fehlerpfad 
Hier geben Sie den Fehlerpfad ein. D.h. wo soll die Datei abgelegt werden, wenn beim Import ein Fehler auftritt. 

Dieser gesonderte Pfad "sammelt" fehlerhafte Dateien und erspart eine lange Suche nach fehlerhaften Dateien im Zielpfad. 

### Button Suche 

Mit diesem Button können Sie einen Ordner auswählen, um den Pfad des Servers zu hinterlegen. 

Siehe Kapitel Ordner auswählen . 

### Combobox Wiedervorlage 
In dieser Combobox wählen Sie aus, welcher Sachbearbeiter eine Wiedervorlage für den erfolgten Import erhalten soll. Im Fall eines Fehlers wird eine Wiedervorlage mit Priorität "sehr wichtig" erfasst und bei einer Warnmeldung, z.B. Preis außerhalb der Gültigkeit, eine Wiedervorlage mit Priorität "wichtig". 

Sie können entweder einen Sachbearbeiter oder eine Abteilung auswählen! 

Tooltip: 

Wiedervorlage an Sachbearbeiter bei Importfehler von Proxess Rechnungsdateien. 

### Combobox Abteilungsnr. 
Soll die Wiedervorlage für importierte Rechnungen nicht nur an einen einzelnen Sachbearbeiter gehen, sondern alternativ auch an eine Abteilung, wählen Sie hier die entsprechende Abteilung aus. Im Fall eines Fehlers wird eine Wiedervorlage mit Priorität "sehr wichtig" erfasst und bei einer Warnmeldung, z.B. Preis außerhalb der Gültigkeit, eine Wiedervorlage mit Priorität "wichtig". 

Wenn Sie eine Abteilung auswählen, kann verhindert werden, dass bei Urlaub oder Krankheit eines einzelnen Sachbearbeiters, die Wiedervorlage unbearbeitet bleibt. 

Sie können entweder einen Sachbearbeiter oder eine Abteilung auswählen! 

### Checkbox Alle Mandanten 
Wenn Sie diese Checkbox aktivieren, werden nicht nur die Dateien für den eigenen Mandanten importiert, sondern auch für alle anderen. 

Ist die Checkbox nicht aktiv, werden nur XML-Rechnungen des eigenen Mandanten importiert, die anderen Dateien verbleiben im Import-Ordner. 

Tooltip: 

Der Import wird auch für andere Mandanten durchgeführt.
