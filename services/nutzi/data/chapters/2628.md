# Absender im Auszug-/Kassenimport 
Absender 

Über den Button Absender im Dialog Auszug-/Kassenimport öffnet sich der Dialog Absender . 

Im Kopf des Dialogs können Sie die Selektion für Ihre Daten einstellen. 

Die Auszug-/Kassenimport Absender-Tabelle speichert die Daten von Bankleitzahl und Kontonummer zu jedem Debitor bzw. Kreditor. Auf diese Weise wird die Trefferquote für die Automatik in der Auszug-/Kassenimport-Erkennung erhöht. 

Hat nun ein Debitor bzw. Kreditor eine Änderung seiner Kontoverbindung mitgeteilt, so kann der alte Datensatz gelöscht werden, damit dies bei der nächsten Erkennung entsprechend berücksichtigt wird.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

### Checkbox Exakt 
Wenn Sie diese Checkbox aktivieren, erfolgt die Suche genau nach der eingegebenen Kontonummer. 

Tooltip: 

Wenn diese Checkbox gesetzt ist, muss der Treffer 1:1 dem gesuchten Begriff entsprechen. 
Ist die Checkbox deaktiv, erfolgt die Suche 
- ohne Beachtung der Groß-/Kleinschreibung 
- nach Übereinstimmung an beliebiger Stelle
