# Preisänderungsagent 
Preisänderungsagent 

Dieser Dialog öffnet sich, wenn Sie im Dialog Preisoptimierung auf der Registerkarte Automatisch auf den Button neben der Tabelle Agent klicken. 

Hier wird der Preisoptimierungsagent konfiguriert. Die Angabe von Artikelgruppen ist optional. Alle anderen Parameter müssen angegeben werden. Das stellt die notwendige Absicherung der Arbeit des Agenten sicher. 

Die Checkbox Aktiv ist Voraussetzung dafür, dass der Agent Buchungen durchführen kann. Das zeitweilige Deaktivieren ist damit jederzeit möglich. 

Ein einzelner Agent kann mit dem Button Start manuell einmalig gestartet werden.
