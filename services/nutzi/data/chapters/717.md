# Artikelgruppen Registerkarte Text 
Artikelgruppe Registerkarte Text 

Auf dieser Registerkarte können Warengruppen-spezifische Texte, auch in Fremdsprachen, angelegt werden. Dieses Feld hat informativen Charakter und kann auf Formulare gedruckt werden. 
### Combobox Sprache 
Hier wählen Sie die gewünschte Sprache aus. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Feld Text 
Hier geben Sie einen individuellen Text ein.
