# Kapitel NV.ERP.Base.Common.cdCurrency.shtCurrencyID

Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Die Systemwährung (Euro für die deutsche Version) MUSS in der Tabelle unter der Währungsnummer 1 und dem Kurs 1,0 angegeben werden! 

Tooltip: 

Geben Sie eine eindeutige, fortlaufende Nummer ein.
