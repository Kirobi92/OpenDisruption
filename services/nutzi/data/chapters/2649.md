# Wartungshistorie 

Wartungshistorie 

Dieser Dialog bietet eine gute Übersicht über bereits erfolgte Wartungen bzw. eignet sich sehr für die Planung von Wartungen in der Zukunft. Dieser Dialog öffnet sich, wenn Sie im Anlagenstamm den Button Wart. Hist. drücken. 

Über die Felder Anlage und Bezeichnung kann auch nach Wartungsaufträgen anderer Anlagen gesucht werden. 

Es gibt 2 Arten von Wartungen: 

- Planwartung verwalten: Diesen Satz kennzeichnen die Checkbox in der Spalte 'Planwartung' und die Angaben zum Intervall. Beim Speichern einer Planwartung wird eine Wiedervorlage an den Sachbearbeiter erstellt. Durch die Spalten zum Intervall wird die Wiederholung definiert. Werden in einem Satz mit Intervall die Spalte ‚Wartungsdatum‘ und ‚Betrag‘ erfasst, wird dieser Satz auf „Wartung erledigt“ gesetzt. Es wird automatisch ein neuer Satz mit den fortgeschriebenen Werten aus Erledigungsdatum und Intervall erstellt und gleichzeitig die Wiedervorlage an den Sachbearbeiter. - Sonstige Wartungen dokumentieren: (Einmalige Wartungen dokumentieren, es erfolgt keine geplante Wiederholung dieser Wartung.) Ein neuer Satz wird hinzugefügt ( F4 ), bei dem aber gleichzeitig auch die Spalten ‚Wartungsdatum‘ und ‚Betrag‘ gefüllt sind. In diesem Fall wird in der Spalte ‚Intervall‘ der leere Eintrag ausgewählt. 
Die Tabelle gibt Auskunft über das Datum der Wartung, die ausgeführte Maßnahme, die Nummer des Lieferanten, der die Wartung durchgeführt hat, den nächsten Wartungstermin, etc. 

Die Spalte ‚Wartungstermin‘ zeigt das Datum der geplanten Wartung. 

Die Spalte ‚Vorlauf‘ gibt an, für wie viele Tage vor dem Wartungstermin die Wiedervorlage an den Mitarbeiter datiert ist. 

Die Spalte ‚Wartungsdatum‘ zeigt den Tag an, an dem die Wartung ausgeführt wurde. 

Wenn in der Spalte ‚Verlinkt‘ die Checkbox aktiviert ist, ist bereits ein Referenz Link hinterlegt.

### Button Löschen 

Mit diesem Button können Sie einen Eintrag löschen. 

- Unerledigte Wartungen (Checkbox Wartung erledigt ist nicht aktiv) werden gelöscht. - Erledigte Wartungen (Checkbox Wartung erledigt ist aktiv) werden wieder aktiviert. Das heißt, es werden die Felder Wartungsdatum und Betrag gelöscht und die Checkbox Wartung erledigt deaktiviert. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
