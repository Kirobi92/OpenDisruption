# Kapitel NV.ERP.Base.Customer.cdCustomer.shtNoRemain

Ist dieser Parameter aktiviert, werden bei Teillieferungen keine Restmengen gebildet. 

Dieser Parameter hat Vorrang vor dem Parameter Kein Rückstand aus den Verkaufsparametern. 

Siehe Kapitel Parameter Verkauf . 

Tooltip: 

Keine Bildung von Restmengen bei Teillieferung.
