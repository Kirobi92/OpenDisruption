# Kapitel NV.ERP.MM.Production.cdParamProduction.shtHalfFinishedProducts

Wenn Sie diesen Parameter setzen und dazu noch der Parameter Perm. Leist-Buch gesetzt wird, ist eine Auswertung für Halbfertigwaren gewährleistet. 

Zuverlässige Werte können nur für die Betriebsaufträge gewährleistet werden, die nach der Aktivierung des Parameters freigegeben wurden. Denn ab diesem Zeitpunkt erfolgen die Prüfungen auf Datenkonsistenz. 

Zur Erinnerung: die Checkbox Perm. Leist-Buch ist dafür verantwortlich, dass jede BDE-Buchung bezüglich Arbeitsleistung über das Leistungs-Journal in die Kostenrechnung übertragen wird. Das Leistungs-Journal dient als Quelle für die Ermittlung der geleisteten Arbeitsleistung für das Halbfertigprodukt.
