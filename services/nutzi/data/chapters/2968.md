# EDI Parameter Registerkarte Allgemein 
EDI Parameter Registerkarte Allgemein 

Auf dieser Registerkarte wählen Sie aus der Combobox einen Sachbearbeiter aus, an den die Wiedervorlage gesendet werden soll, falls beim Import oder Export Fehler erscheinen. Dies ist besonders dann interessant, wenn die Verarbeitung per Job Server läuft und kein Sachbearbeiter dies manuell überprüft. 

Die Parameter für die Wiedervorlage gelten nur für das alte Inhouse Format und nicht für das XML-Format! 
### Combobox Wiedervorlage 
Aus dieser Combobox wählen Sie den Sachbearbeiter aus, der die Wiedervorlage erhalten soll.
