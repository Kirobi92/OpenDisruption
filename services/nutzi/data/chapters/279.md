# Chargenrecherche Registerkarte Bestand 
Chargenrecherche Registerkarte Bestand 

Auf dieser Registerkarte sehen Sie den Bestand zu den vorhandenen Chargenartikeln der Selektion. Mit der Checkbox Alle Chargen werden alle Chargenartikel angezeigt, d.h. auch Chargen mit Bestand 0 werden angezeigt. 

Nach einem Wechsel auf diese Registerkarte, z.B. von der Registerkarte Verwendung , ist die Suche mit F2 oder dem Button erneut auszuführen. Erst dann wird die Tabelle gefüllt. 
### Checkbox Alle Chargen 
Wenn Sie diese Checkbox aktivieren, werden auch Chargen ohne Bestand angezeigt. 

Tooltip: 

Falls gesetzt, werden alle entsprechenden Chargen selektiert, unabhängig davon ob diese einen Bestand haben oder nicht. 

### Tabelle Chargenbestand 

Diese Tabelle zeigt Detailinformationen zu vorhandenen Chargenartikeln. 
### Tabellenspalte Charge 
Hier wird die Chargennummer angegeben. 

Die Chargenverwaltung steht nur zur Verfügung, wenn sie lizenziert ist. Dies erkennen Sie, wenn im Dialog Konfiguration die Checkbox aktiv gesetzt ist. 

Siehe Kapitel Konfiguration der Parameter .

### Tabellenspalte Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit.
