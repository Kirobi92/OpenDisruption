# Positionssplitt in der Rechnungsprüfung 
RP Positionssplitt 

Dieser Dialog öffnet sich mit dem Button Splitt auf der Registerkarte Allgemein in der Rechnungsprüfung . Hier können Sie die gewählte Position der Rechnungsprüfung teilen (z.B. bei einer Teillieferung). Es entsteht eine neue Position in der bezogenen Bestellung. (Da die neue Position nicht Teil dieser Rechnungsprüfung ist, erscheint sie nicht. Sie ist dann nur in der Bestellung enthalten!) 

Für diese Position ist der Wareneingang bereits erfolgt, das bedeutet Status 3. So ist es möglich, diese Position in einer weiteren/späteren Rechnungsprüfung zu diesem Wareneingang zu verarbeiten. Deshalb ist auch nur das Feld Menge berechnet zugänglich. Geben Sie die Teilmenge ein. Diese neu generierte Position wird dann aus der Rechnungsprüfung herausgenommen und kann mit einer anderen Rechnungsprüfung weiter bearbeitet werden. 

Die Mengen der gesplitteten Positionen müssen der Wareneingangsmenge entsprechen. 

Beim Splitt von Positionen einer Rechnungsprüfung werden alle Konditionen gesplittet, d.h. %-Rabatte/Zuschläge werden 1:1 übernommen; absolute Rabatte werden anhand der Mengenverteilung gesplittet.
