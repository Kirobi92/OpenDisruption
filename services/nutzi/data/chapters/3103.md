# Datenimport bearbeiten Registerkarte Fremdartikelnummer 
Import Bearbeiten Registerkarte Fremdartikelnummern 

Auf dieser Registerkarte werden die Fremdartikelnummern eingelesen. Die eingelesenen Daten können nach unterschiedlichen Kriterien gefiltert werden. 
### Button Bearbeiten 

Wenn Sie diesen Button drücken, können Artikelmerkmale bearbeitet werden. 

Tooltip: 

Artikel Merkmale bearbeiten 

### Button Fremdart. Verarbeiten 

Wenn Sie diesen Button drücken, werden ausgewählte Fremdartikelnummern in den Schattenstamm übernommen. 

Tooltip: 

Ausgewählte Fremdartikelnummer in den Schattenstamm übernehmen 

### Feld SQL-Information 
Tooltip:

### Tabelle 

### Tabellenspalte Importschlüssel 
Hier wird der Import-Schlüssel angegeben. 

Tooltip: 

Der "Import-Schlüssel" ist ein generierter Wert, der als eindeutige ID (Referenzschlüssel) zur Referenzierung des Datensatzes dient. Er wird aus den Haupteigenschaften des Datensatzes zusammengesetzt. Jeder Datensatz benötigt einen eigenen, eindeutigen Import-Schlüssel. Es wird kein neuer Datensatz angelegt, wenn es bereits einen Datensatz mit demselben Import-Schlüssel gibt, sondern der bereits vorhandene überschrieben/aktualisiert. 

### Tabellenspalte Haupt-Importschlüssel 
Hier wird der Haupt-Importschlüssel angegeben. 

Tooltip: 

Angabe des Haupt-Importschlüssels 

### Tabellenspalte Fremdartikelnummer 
Hier wird die Fremdartikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer
