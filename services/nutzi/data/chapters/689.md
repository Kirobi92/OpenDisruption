# Artikel Registerkarte LVS 
Artikel Registerkarte LVS 

Auf der Registerkarte LVS definieren Sie pro Artikel nachfolgende Einstellungen für das Lagerverwaltungssystem. 
### Checkbox LHM Zuordnungszwang 
Wenn Sie diese Checkbox aktivieren, wird der Artikel zwingend einem LHM zugeordnet. 

Tooltip: 

Entscheidung, ob der LHM-Zuordnungszwang greifen soll, der gesetzte Haken bejaht dies. 

### Combobox LHM Typ 
Hier wird definiert, welches LHM auf den Lagerplatz passt. 

Tooltip: 

Standard LHM Typ für den Wareneingang 

### Feld Dezimalstellen 
Da kein Komma als Trennung zwischen Vor- und Nachkommastellen eingetragen werden kann, geben Sie hier die Nachkommastellen an. 

### Feld Zählzeitraum 
Gilt nur, wenn das LVS lizenziert ist! 

Hier wird ein Zeitraum für die permanente Inventur angegeben, wie oft ein Artikel im Jahr zu zählen ist. Ist nichts gepflegt, muss der Artikel alle 12 Monate gezählt werden. 

Tooltip: 

Zählzeitraum (alle x Monate) 

### Feld Zählbestand 
Gilt nur, wenn das LVS lizenziert ist! 

Hier wird der Zählbestand angegeben, ab dessen Unterschreitung der Artikel für die permanente Inventur gezählt werden soll. 

Tooltip: 

Bestand, ab welchem ein Lagerplatz für diesen Artikel gezählt werden soll. 

### Checkbox Keine Rückstandserinnerung 
Wenn diese Checkbox aktiviert wird, wird bei einem Rückstand des Artikels keine Meldung ausgegeben. 

Tooltip: 

Dieser Schalter steuert, ob der verantwortliche Sachbearbeiter im Falle eines Rückstands eine Wiedervorlage erhält. 

### Checkbox Bypassfähig 
Wenn Sie diese Checkbox aktivieren, kann der Artikel in einem Bypass-Lagerbereich gelagert werden. 

### Checkbox Zuladungsartikel 
Ist diese Checkbox gesetzt, so ist der Artikel ein Zuladungsartikel. Er wird nicht kommissioniert, sondern vom System zum Beginn der Kommissionierung automatisch vom Platz ausgebucht. Der Fahrer erhält dann eine spezielle Liste (Etiketten) mit welcher er dann erfährt, wo noch Ware abzuholen ist. Dies wird normalerweise bei Artikeln gemacht, deren Vorkommissionierung sinnlos oder nur umständlich zu machen wäre (Große Tanks, Kies usw.). 

### Checkbox Strikte Chargentrennung 
Wenn Sie diese Checkbox aktivieren, wird der Artikel strikt nach Chargen getrennt. Damit wird bei der Entnahme sichergestellt, dass nicht unterschiedliche Chargen entnommen werden. Damit ist die Lagerhaltung auf dem Platz/LHM zwingend chargenrein durchzuführen. 

### Feld Lagerauftrag 
Hier geben Sie die Anzahl für die Kommissionierungsetiketten ein. 

Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Feld Verkauf 
Hier geben Sie die Anzahl für die Verkaufsetiketten ein. 

Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Checkbox Endkontrolle 
Wenn Sie diese Checkbox aktivieren, wird eine Endkontrolle durchgeführt. Mit Setzen dieser Checkbox wird das Feld Grund zugänglich. 

### Feld Grund 
Geben Sie hier einen Grund (individuelle Eingabe) für die Endkontrolle ein. Dieses Feld ist nur zugänglich, wenn die Checkbox Endkontrolle aktiviert ist.
