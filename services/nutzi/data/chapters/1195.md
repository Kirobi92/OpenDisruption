# EDI Auftrag / Auftragsänderung 
EDI Auftrag / Auftragsänderung 

Dieser Dialog öffnet sich, wenn Sie im Dialog EDI Auftrag auswählen einen Doppeklick auf einen Eintrag der Tabelle ausführen. 

Hier werden die B2B-Tabellen (Datenbanktabellen) visualisiert. Sie erhalten eine Ansicht auf die Ebene zwischen EDI Import und dem eNVenta Auftrag . 

Wenn der Auftag nicht automatisch angelegt werden kann, können die Daten über diesen Dialog korrigiert werden. Anschließend wird dann der Auftrag angelegt, wenn Sie auf den Button Auftrag übernehmen klicken. 

Es wird der Originaltext angezeigt und fehlerhafte Einträge werden rot gekennzeichnet, wenn z.B. auf Basis der importierten Daten keine Artikelnummer ermittelt werden konnte. 

Felder zur Korrektur sind nur zugänglich, wenn diese in den Prozessregeln auch für die Ermittlung aktiviert wurden, z.B. Kunden-Artikelnummer, EAN oder GTIN.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Auftrag übernehmen 

Mit diesem Button wird der Auftrag erzeugt. Bei der Übernahme bzw. Generierung werden die kompletten Prozessregeln analog dem EDI-Import ausgeführt. Hierdurch kann ggf. die EDI-Nachricht wieder nicht angelegt werden, kann aber direkt in diesem Dialog wieder nachkorrigiert werden. 

Der Import eines Auftrags ist nur einmalig möglich! 

Bei der Erkennung eines wiederholten Imports erscheint eine Hinweismeldung und bietet die Möglichkeit, die Aktion abzubrechen. 

Tooltip: 

Ein neuer Auftrag wird anhand der Daten generiert. 

### Button Prozessregeln 

Mit diesem Button gelangen Sie in den Dialog Prozessregeln , um sich die nötigen Prozessregeln auszusuchen bzw. zu korrigieren, damit zukünftige Importe korrekt laufen und keine Nacharbeit mehr notwendig ist. 

Siehe Kapitel Prozessregeln . 

### Button Objekt-Editor 

Mit diesem Button gelangen Sie in den Dialog Objekt-Editor . Die importierte Datei wird als Objekt angezeigt und kann bearbeitet werden. Hierzu wird der Import komplett neu durchgeführt und ggf. die Daten hierdurch in der B2B-Tabelle aktualisiert. 

Siehe Kapitel Objekt-Editor . 

### Button XML-Mapping 

Mit diesem Button gelangen Sie in den Dialog X-Mapping Editor . Das Mapping stellt Datentransformationen (Dezimal, Datum, etc.) und weitere Funktionalitäten bereit. Sollte es einen Fehler im Mapping geben so kann das Mapping korrigiert werden damit zukünftige Importe korrekt laufen und keine Nacharbeit mehr notwendig ist. 

Siehe Kapitel X-Mapping Editor .
