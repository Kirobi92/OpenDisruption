# Simulation Registerkarte Kalkulation 
Simulation Registerkarte Kalkulation 

Auf dieser Registerkarte wird Ihnen die Kalkulation der Simulation angezeigt. 

Es ist möglich für einen Auftrag die Produktionssimulation für verschiedene Mengen zu kalkulieren, um so die Auswirkungen auf die Herstellkosten des Produkts erkennen zu können. Auf diese Weise können die Herstellkosten optimal ermittelt werden. 

Dazu generieren Sie aus dem Auftrag heraus über die Option PPS Simulation eine Simulation und öffnen anschließend den Dialog Simulation . 

Die Vergleichsmengen können direkt auf der Registerkarte erfasst werden (in den Feldern Menge 2 und Menge 3 ) und führen zu einer sofortigen Neukalkulation der Daten in der Tabelle. Dazu stehen für bis zu 2 Vergleichsmengen die zusätzlichen Spalten ‚Menge Soll‘, ‚Betrag Soll‘ und ‚Gesamt Soll‘ zur Verfügung. Die Anzeige im Feld SK / ME Soll (unterhalb der Tabelle) erfolgt je Kalkulation. 
### Button EK buchen 

Wenn in einer Simulation die Kalkulation eines Betriebsauftrags gewählt ist, der direkt einer Kundenauftragsposition zugeordnet ist, ist dieser Button zugänglich. 

Dieser Button ermöglicht es, zur Vorabkalkulation und Preisbildung von Angeboten und Aufträgen, die Kalkulationsergebnisse aus einer Produktionssimulation in die Angebots- und Auftragspositionen zu übernehmen. 

Bereits fakturierte Auftragspositionen können nicht mehr geändert werden! 

Tooltip: 

Kalkulierter EK wird in Auftrags- bzw. Angebotsposition übertragen.
