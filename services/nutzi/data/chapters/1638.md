# Sachbearbeiter- Scanner Registerkarte Scannereinstellungen

Lagerverwaltungssystem (LVS) / Sachbearbeiter- Scanner 

# Sachbearbeiter- Scanner Registerkarte Scannereinstellungen 
Sachbearbeiter Scanner Registerkarte Scannereinstellungen 

Auf dieser Registerkarte legen Sie Details zum Scannen für den einzelnen Sachbearbeiter fest. Dazu nutzen Sie die zur Verfügung stehenden Checkboxen. 
### Checkbox Behälter muss jedes Mal angescannt werden. 
Wenn Sie diese Checkbox aktivieren, wird festgelegt, dass der Kommissionierer den Kommissionierbehälter 
• bei jedem Pick scannen muss (aktiv) 
• er nur beim ersten Pick zu einem Auftrag scannen muss (inaktiv). 

Tooltip: 

Mit diesem Schalter kann festgelegt werden, ob der 
Kommissionierer den Kommissionierbehälter 
• bei jedem Pick scannen muss (aktiv) 
• er nur beim ersten Pick zu einem Auftrag scannen muss (inaktiv) 

### Checkbox Kein Scan von Platz 
Wenn Sie diese Checkbox aktivieren, wird festgelegt, dass der Kommissionierer den Platz scannen muss(deaktiv). Wenn im Lagerort mit LHM’s gearbeitet wird, sollte dieser Parameter deaktiviert sein. 

Tooltip: 

Mit diesem Schalter kann festgelegt werden, ob der Kommissionierer den 
Platz scannen muss (inaktiv). Wenn im Lagerort mit LHM’s gearbeitet wird, sollte dieser Parameter deaktiviert sein. 

### Checkbox Kein Scan von Artikel 
Wenn Sie diese Checkbox aktivieren, wird festgelegt, dass der Kommissionierer den Artikel bei jedem Pick scannen muss (aktiv). 

Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Tooltip: 

Mit diesem Schalter kann festgelegt werden, ob der Kommissionierer den 
Artikel bei jedem Pick scannen muss (aktiv). 
Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Checkbox Kein Scan von Chargen 
Wenn Sie diese Checkbox aktivieren, wird festgelegt, dass der Kommissionierer die Charge bei jedem Pick auswählen muss (aktiv). 

Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Tooltip: 

Mit diesem Schalter kann festgelegt werden, ob der Kommissionierer den 
Artikel bei jedem Pick scannen muss (aktiv). 
Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Checkbox Menge vorschlagen 
Wenn Sie diese Checkbox aktivieren, wird festgelegt, dass dem Kommissionierer die Einlagerungs-, Umlagerungs- oder Pickmenge automatisch vorbelegt wird. 

Tooltip: 

Mit diesem Schalter kann festgelegt werden, ob dem Kommissionierer die 
Einlagerungs-, Umlagerungs- oder Pickmenge automatisch vorbelegt wird. 

### Checkbox Endkontrolle durch Behälterbuchung 
Wenn Sie diese Checkbox aktivieren, wird die Endkontrolle automatisch durch eine Umlagerung aus dem Klärungsbereich an den Packplatz als erledigt kennzeichnet. 

Tooltip: 

Die Endkontrolle wird automatisch durch eine Umlagerung aus dem Klärungsbereich an den Packplatz als 'erledigt' kennzeichnet.
