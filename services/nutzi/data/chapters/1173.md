# Bestellauskunft 
Bestellauskunft 

Über das Menü Optionen >> Auskunft >> Bestellauskunft (oder <Strg> + <B> ) im Auftrag erhalten Sie einen Dialog mit einer Anzeige aller diesem Auftrag zugeordneten Bestellungen. 

Der Programmteil kann darüber hinaus verwendet werden, um gezielt Abfragen über den aktuellen Bestellbestand von Lieferanten und Artikeln durchzuführen. Auf diese Weise kann aus der Auftragsbearbeitung heraus dem Kunden eine jederzeit aktuelle Auskunft über die Lieferbereitschaft gegeben werden. 

Der obere Teil des Dialogs enthält alle Auswahlfelder. Nachdem Sie alle gewünschten Suchkriterien eingegeben haben, betätigen Sie den Button , um das Ergebnis anzuzeigen. 

Es werden alle Bestellpositionen zu dem Artikel mit ihrem Lieferstatus, Termin und der Bestellmenge angezeigt. 

Sie können nach Lieferantennummer, Artikelnummer, Belegnummer und Auftragsnummer suchen. Außerdem ist eine Einschränkung auf Datum, sowie die Auswahl eines Sachbearbeiters oder einer Abteilung möglich. 

Je nachdem aus welchem Dialog die Bestellauskunft geöffnet wird, können Bestellpositionen per Doppelklick in den zugrundeliegenden Vorgang übernommen werden. 

Weitere Funktionsweisen des Doppelklicks beim Aufruf aus: 

- dem Auftrag - der Dialog Bestellauskunft wieder geschlossen. - dem Wareneingang - die Bestellposition übernommen. - der Bestellung - die entsprechende Bestellung geöffnet. 
In Artikelstamm, Inventur und Belegjournal dient die Bestellauskunft lediglich der Information.
