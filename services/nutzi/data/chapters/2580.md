# Druck der Avise und Mahnungen 
Drucken 

Wenn Sie im Bankeinzug den Modus Zeigen benutzen, wird ein Feld eingeblendet, indem Sie die Anzahl der Avise festlegen können. Diese Funktion kann genutzt werden, um bei einer Vielzahl von Belegen, die Ansicht im Darstellungsprogramm, z.B. Internet Explorer, einzuschränken. 

Eine Mahnung (Zahlungsavis, Einzugsavis, Zinsrechnung, Kontoauszug, Differenzmeldung, ...) kann in der Sprache des Personenkontos ausgegeben werden. Deshalb ist jede Mahnung (...) ein eigenständiger Druckjob und öffnet bei der Auswahl "Zeigen" jeweils eine neue Registerkarte im Browser. (Bis der Browser abstürzen würde). 

Deshalb ist dieses Feld mit 2 vorbelegt und auf maximal 50 eingeschränkt.

### Dock Panel Modus 

Hier wählen Sie aus, was genau mit dem Beleg geschehen soll. Z.B. ob er ausgedruckt werden soll oder als E-Mail versendet werden soll. 

Die Felder für den Fax-Service werden nicht angezeigt, wenn im Dialog Parameter Nachrichten kein Fax-Service definiert ist. Somit können Sachbearbeiter auch nur Dienste/Modus auswählen, welche auch tatsächlich bedient werden. 

### Checkbox Kein Archiv 
Wenn Sie diese Checkbox aktivieren, wird das Formular nicht archiviert. 

### Checkbox Briefkopf 
Diese Checkbox wird aus dem Dialog Parameter Allgemein (Registerkarte Allgemein ) vorbelegt und kann hier im Einzelfall deaktiviert werden. 

Erläuterung: 

Im Rahmen von Geschäftsprozessen ist es immer wieder notwendig, Belege physisch oder digital als Ausdrucke bereitzustellen. Aus rechtlichen Gründen muss dies bei offiziellen Dokumenten mit bestimmten Informationen (Anschrift, BLZ, …) – in Kopf- und Fußzeile - geschehen. Diese Informationen werden im Folgenden als Briefkopf bezeichnet. 

Dies können Kunden nutzen, die ihre Firmeninformationen direkt mit dem Druck des Belegs auf ein Blanko-Papier drucken möchten. 

Die Checkbox Briefkopf dient in den Dialogen als Vorbelegung und im automatisierten Druck als Einstellung für den Druck. 

Im Einzelfall kann in den Druck-Dialogen der einzelnen Vorgänge die Checkbox Briefkopf deaktiviert werden, falls Sie ausnahmsweise auf ein Briefpapier drucken möchten. 

Tooltip: 

Wenn aktiv, wird der Briefkopf z.B. Logo, Absender auf den Beleg gedruckt.
