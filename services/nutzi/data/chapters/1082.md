# Einmalkunden reaktivieren 
Um einen nicht physisch gelöschten Einmalkunden wieder zu reaktivieren, steht im Dialog Kunde erfassen der Button Gelöschte K. zur Verfügung. Mit diesem Button öffnet sich der Dialog Kundenauswahl und dort ist die Checkbox Nur gel. Kunden (in roter Schrift) bereits aktiviert. Diese Checkbox ist unzugänglich, d.h. in diesem Fall wird nur nach gelöschten Kunden gesucht! 

Der Button liest alle gelöschten Einmalkunden in die Tabelle ein. Die Kundennummer wird reaktiviert. D.h. auf diese Weise ist es nicht erforderlich, einen bereits gelöschten Einmalkunden neu anzulegen. 

Mit dem Button wird der Kunde in den Auftrag eingelesen.
