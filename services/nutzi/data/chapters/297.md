# Lagerorte Registerkarte Automatische Scannerzuweisung

Allgemeine Funktionalitäten in eNVenta ERP / Lagerverwaltung / Lagerorte 

# Lagerorte Registerkarte Automatische Scannerzuweisung 
Lagerorte Registerkarte Aut. Scannerzuweisung 

Auf dieser Registerkarte können Sie den Vorgang und die Reihenfolge festlegen, wie er dem Scanner zugeordnet werden soll. 
### Button Plus 

Mit diesem Button können Sie der Tabelle einen Eintrag hinzufügen. 

### Button Löschen 

Dieser Button löscht einen Eintrag aus der Tabelle.
