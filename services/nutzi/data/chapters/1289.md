# Rahmenauftrag Registerkarte Kopfdaten 
Rahmenauftrag Registerkarte Kopfdaten 

Auf dieser Registerkarte können weitere Informationen zum Rahmenvertrag, insbesondere für die periodische Fakturierung, festgelegt werden. 
### Combobox Bestellart 
Hier wird die Bestellart angegeben. Dazu gehören z.B. die Bestellarten Anfrage , per Mail , telefonisch . 

Tooltip: 

Bestellart zur Übernahme in Auftrag 

### Feld Bestellnummer 
Hier wird die Bestellnummer des Lieferanten angezeigt. 

Tooltip: 

Lieferanten-Artikelnummer zur Übernahme in Auftrag 

### Feld Bestelldatum 
Hier wird das Datum der Bestellposition zur Übernahme in den Auftrag angezeigt. 

Tooltip: 

Bestelldatum zur Übernahme in Auftrag 

### Combobox Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Tooltip: 

Angabe der Währung 

### Feld Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Diese Nummer wird dann in den Aufträgen, die aus dem Rahmenauftrag generiert werden, zugeordnet. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Mit diesem Button können Sie nach Projekten suchen. Es öffnet sich der Dialog Projektauswahl . 

Siehe Kapitel Projektauswahl . 

### Feld Kürzel RA 
Hier kann ein Kürzel für den Rahmenauftrag eingegeben werden. 

Hier kann z.B. eine ID des Kunden angegeben werden, unter welcher er den Rahmenvertrag führt. 

Tooltip: 

Kürzel Rahmenauftrag 

### Feld Abschlussdatum 
In diesem Feld wird das Abschluss-Datum des Rahmenvertrags angegeben. Dieses muss nicht zwingend mit der Laufzeit des Rahmenauftrags übereinstimmen. 

Tooltip: 

Abschlussdatum: 

In das Feld Abschlussdatum geben Sie das Datum 
über den Abschluss der Vereinbarung an. 

### Checkbox Periodische Abrufe 
Wenn Sie diese Checkbox aktivieren, erfolgt die automatische Generierung von Aufträgen zeitgesteuert über die vereinbarten Perioden. 

Bei der periodischen Generierung von Abrufen können nur Rahmenverträge berücksichtigt werden, wenn eine Zahlungsbedingung hinterlegt wird. 

Siehe Absatz Menü Optionen / Periodisch generieren. 

### Feld Startdatum 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld Menge 
Hier wird die Abrufmenge für die periodische Fakturierung eingetragen. 

Tooltip: 

Angabe der Menge 

### Feld Termin 
Dieser Termin entspricht dem Tag des Monats, an dem die Fakturierung durchgeführt werden soll. Z. B. 1, 5, 15... 

Tooltip: 

Tag an dem, die Fakturieren durchgeführt werden soll 

### Feld Vorlauftage 
In diesem Feld geben Sie an, wie viel Tage vor dem Lieferdatum des Abrufs, der Auftrag generiert wird. 

Diese Vorlauftage werden auf das aktuelle Datum gerechnet. Hierbei werden auch Wochenenden und Feiertage berücksichtigt. Ist dieses errechnete Datum größer (also weiter in der Zukunft) als das Lieferdatum eines Abrufs (Spalte ‚Lieferdatum‘ im Dialog Rahmenabrufe), wird ein neuer Auftrag generiert. In diesem Fall darf allerdings die Checkbox in der Spalte ‚Ohne Spez.‘ im Dialog Rahmenabrufe nicht gesetzt sein. 

Weiterhin ist zu beachten, dass Lagernummer und Auftragsnummer angegeben sind. 

Siehe Kapitel Rahmenabrufe in der Rahmenbestellung . 

Tooltip: 

Generierung von Aufträgen über Job-Server 

Anzahl der Tage eines Zeitraums. 
In dem angegebenen Zeitraum vor dem Liefertermin eines Rahmenabrufs, wird ein Auftrag für diesen Rahmenabruf generiert. 
Hierbei werden nur Werktage eingerechnet. 

### Dock Panel Periode 

In diesem Bereich legen Sie die Zeitabstände für die Fakturierung fest.
