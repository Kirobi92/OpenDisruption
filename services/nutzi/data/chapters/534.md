# Sachbearbeiter Abwesenheiten für den Administrator 
Sachbearb. Abwesenheit 

Unter System >> Allgemein finden Sie den Eintrag Sachbearbearbeiter Abwesenheit . In diesem Menü kann nur der Administrator die Abwesenheiten (z.B. Urlaub, Krankheit) verwalten. Dadurch hat der Administrator den Überblick über Abwesenheiten und die Stellvertretung kann geregelt werden. 

Diese Pflege erfolgt im Regelfall zentral über eine Zeiterfassungs-Software und wird dann per CSV-Datei in eNVenta ERP importiert. Der CSV-Import erfolgt über den Datenimport Adapter . 

Mit dieser Funktionalität ist das aufwändige Kontrollieren der Wiedervorlagen nicht mehr notwendig. Die Wiedervorlagen werden automatisch an den Kollegen weitergesendet, der die Vertretung übernommen hat. Damit fällt die Recherche nach Wiedervorlagen weg und die Gefahr, dass eine wichtige Mitteilung verloren geht. Die Wiedervorlagen bleiben beim Original-Empfänger und werden zusätzlich dem Stellvertreter angezeigt. 

Um Wiedervorlagen des „Eigners“ von den eigenen (d.h des Stellvertreters) unterscheiden zu können, werden diese in dunkelblauer Farbe angezeigt. Neu hinzugekommene Wiedervorlagen werden in grün angezeigt. 

Bitte beachten Sie, dass es im Navigator >> Datei ebenfalls einen Eintrag Abwesenheiten gibt, dort aber keine Namen angezeigt werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Sachbearb. Abwesenheit 
Tooltip: 

In diesem Dialog werden die Abwesenheiten von Mitarbeitern, wie z.B. Urlaub und Krankheit, verwaltet. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Neuer Satz 

Mit diesem Button können Sie der Tabelle einen neuen Eintrag hinzufügen. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
