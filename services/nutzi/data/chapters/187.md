# Datumsfelder 
Generell kann in allen Datumsfeldern das Datum in vereinfachter Form eingegeben werden. Fehlende Werte werden aus dem aktuellen Tagesdatum ergänzt. 

Nachfolgend einige Beispiele, die dem Tagesdatum 11.10.2019 als Grundlage dienen. 

Beispiele: 

| 20 oder 02 | 20.10.2021 oder 02.10.2021 |
| --- | --- |
| 0110 | 01.10.2021 |

usw. 

Geben Sie einen * ein, wird das Tagesdatum zusammen mit der aktuellen Uhrzeit eingetragen. 

* *+x darf auf keinen Fall mit Leerzeichen geschrieben werden!!! ) *-x darf auf keinen Fall mit Leerzeichen geschrieben werden!!! )
