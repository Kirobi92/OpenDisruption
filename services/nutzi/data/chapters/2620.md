# Zahlstellenverbindungen 
Zahlstellen Verb. 

Dieser Dialog öffnet sich im Bankenstamm über den Button Zahlstellen . Hier werden der Typ, die Bankenkennung und die Währung für die Bank hinterlegt, geändert oder gelöscht. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Zahlstellen 
Tooltip: 

In diesem Dialog werden der Typ, die Bankenkennung und die Währung für die Bank verwaltet. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Tabelle 

### Tabellenspalte Typ 
Hier wird der Typ der Zahlstellenverbindung angegeben. Dies ist ein Pflichtfeld. Gültige Werte: 

BLZ - Bankleitzahl 
SWIFT - Swiftcode 

### Tabellenspalte BLZ / SWIFT 
Hier geben Sie den Bank-Code an. Dies ist eine Pflichtangbe. Gültige Werte sind abhängig vom Eintrag im Feld Typ . Entweder 
Typ = 'BLZ', dann Bankleitzahl der Bank 
Typ = 'SWIFT', dann Swiftcode der Bank. 

Tooltip: 

Angabe der Bankleitzahl / SWIFT 

### Tabellenspalte Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen.
