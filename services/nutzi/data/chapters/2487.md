# Bilanz 
Bilanz 

Im Buchen >> Abschluss >> Auswertungen finden Sie den Dialog Bilanz . Über dieses Menü drucken Sie die Bilanz für ein von Ihnen gewähltes Jahr und die vorgegebenen Kriterien. 

Die Anwendung kann auch über Buchen >> Stamm >> Auswertungsstruktur, Registerkarte Bilanz und von dort mit dem Button Drucken geöffnet werden. Aus dieser Anwendung wird die aktuell geöffnete Auswertung in den Druckdialog übernommen. 

In der Standard-Bilanz kann auch über Datum bzw. Periode selektiert werden. Je nach Einstellung ändern sich die Bezeichnungen der Felder bzw. Comboboxen zu diesen Radiobuttons.

### Button Drucken 

Wenn Sie diesen Button drücken, öffnet sich der Dialog Meldungen und listet Ihnen nicht zugeordnete Konten auf. Wenn Sie den Button Ignorieren drücken, wird der Druck fortgesetzt, mit Abbrechen wird der Druck abgebrochen. 

Im oberen Bereich des Dialogs Meldungen steht der Grund (z.B. nicht oder falsch zugeordnet) und im unteren Bereich sind die entsprechenden Konten aufgelistet. 

### Button Suche 

Mit diesem Button werden zu den Selektionskriterien alle bereits journalisierten Buchungen in die Tabelle eingelesen, um dann zu drucken. 

### Button Neu 

Dieser Button leert die Felder.

### Combobox Jahr 
Hier wählen Sie das Jahr, für das die Bilanz gedruckt werden soll. 

### Field Panel Periode 

Hier haben Sie 2 Möglichkeiten den Druck der Bilanz zu steuern. Einmal nach Datum und einmal nach Periode, Reports werden angepasst. 

Datum: 
Wenn der Radiobutton Datum aktiviert ist, ändert sich die Bezeichnung der Felder in Von Datum / Bis Datum . So erhalten Sie eine tagesgenaue Auswertung. 

Periode: 
Wenn der Radiobutton Periode gewählt ist, wählen Sie aus der Combobox den Zeitraum, für den die Bilanz gedruckt werden soll. Druck der Bilanz nur für die ausgewählte Periode bzw. von Jahresbeginn bis einschließlich der gewählten Periode. 
### Feld Datum von 
Für eine Selektion geben Sie hier das gewünschte Datum ein. 

Tooltip:

### Feld Datum bis 
Für die Selektion über einen Datumsbereich (Periode) geben Sie hier das Bis-Datum ein. 

Tooltip: 

Geben Sie ein Bis-Datum ein.

Hier wird eine Periode-von angegeben. 

### Combobox Periode bis 
Hier wird eine Periode-bis angegeben. 

### Checkbox Mit Kontonummern 
Wenn Sie diese Checkbox aktivieren, wird die Kontonummer angezeigt. 

### Checkbox Mit Zwischenabschluss 
Bei Aktivierung dieser Checkbox werden die Buchungen aus dem Buchungskreis Zwischenabschluss-Buchungen berücksichtigt. 

### Checkbox Mit Vorjahreswerten 
Wenn Sie diese Checkbox aktivieren, werden Vorjahreswerte zum entsprechend ausgewählten Zeitraum mit angedruckt. 

### Checkbox Mit Nullsalden 
Wenn Sie diese Checkbox aktivieren, werden auch Konten mit Null-Salden angezeigt. 

### Checkbox Verdichten 
Wenn Sie diese Checkbox aktivieren, werden nur die Summenzeilen angezeigt. 

### Checkbox 0-Werte drucken 
Wenn Sie diese Checkbox aktivieren, werden auch Auswertungen mit 0-Werten angezeigt. 

### Checkbox FW-Beträge drucken 
Wenn Sie diese Checkbox aktivieren, werden die Beträge zusätzlich in Fremdwährung gedruckt. Ist der Saldo NULL wird auch das Währungskürzel nicht angezeigt.
