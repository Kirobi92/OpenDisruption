# Kostenarten 
Kostenarten 

Der Kostenartenstamm erlaubt die Erfassung und Verwaltung der Kostenarten für die Kostenrechnung. Soweit Kostenarten mit den Aufwandskonten aus der Finanzbuchhaltung übereinstimmen, können Sie auch direkt von dort übernommen werden. 

Im Kostenartenstamm können Angaben zur Umlage der Kostenart auf Kostenstellen gemacht werden. Dies ist jedoch nicht zwingend erforderlich. Auch wenn Sie an dieser Stelle eine Standardumlage eingeben, werden bei der Übernahme von Buchungen aus der Finanzbuchhaltung oder Warenwirtschaft auf jeden Fall die dort bei der Buchung angegebenen Kostenstellen berücksichtigt. Nur wenn beim Buchen keine Kostenstelle angegeben wurde, erfolgt die Umlage so, wie sie im Kostenartenstamm in der Kostenrechnung definiert wurde. 

Sie erreichen den Kostenartenstamm über den Navigator Kostenrechnung >> Kostenarten oder in der Menüleiste der meisten Programmmodule über das Menü Stamm . 

### Ändern/Löschen einer Kostenart 
Sie können die beschriebenen Einstellungen zu Kostenarten jederzeit ändern und Ihre Änderungen mit dem Button speichern. 

Das Löschen einer Kostenart mit dem Button ist nur möglich, solange noch keine Buchung auf die Kostenart in der Kostenrechnung erfolgt ist. Wurden schon Buchungen ausgeführt, akzeptiert das Programm die Löschung nicht. 

Aus dem Programmmodul Kostenrechnung heraus kann nicht nachgeprüft werden, ob auf diese Kostenart bereits in der Finanzbuchhaltung gebucht wurde. Auch eine Überprüfung der im Sachkontenstamm hinterlegten Kostenarten erfolgt nicht. Solche Buchungen würden dann aber bei der Übernahme in die Kostenrechnung ( Kore-Übernahme ) mit einem entsprechenden Fehlertext abbrechen.

### Checkbox Gesperrt 
Wenn Sie diese Checkbox aktivieren, ist die Kostenart gesperrt und wird im Buchen Kore nicht mehr angezeigt. Ein manuelles Buchen in der Kostenrechnung ist dann nicht möglich und eine Übernahme wird prinzipiell verweigert; es erscheint eine Fehlermeldung. 

Tooltip: 

Aktiviert: Setzt den Datensatz auf 'gesperrt'. 

### Feld Bezeichnung 
Dieses Feld wird nach Eingabe einer Kostenartennummer automatisch mit dem Namen gefüllt. 

Wenn Sie eine neue Kostenart anlegen, geben Sie hier die passende Bezeichnung ein.
