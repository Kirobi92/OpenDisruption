# Zusatzpositionen 
Zusatzpositionen 

Im Artikelstamm unter dem Menü Optionen finden Sie den Eintrag Zusatzpositionen . 

In diesem Menü definieren Sie zusätzliche Positionen zu einem Artikel, z.B. für Leergut, Verpackungsmaterialpositionen, Anarbeitungspositionen, Materialbearbeitung (z.B. Schnittkosten) oder Kostenzuschläge für Sperrgut (Transportkosten). 

Nachdem Sie im Artikelstamm eine Hauptposition eingetragen haben, können Sie über die Zusatzpositionen weitere (Zusatz-) Positionen in den Artikelstamm übernehmen. 

Beim Öffnen des Dialogs Zusatzpositionen wird die Artikelnummer aus dem Artikelstamm voreingestellt. In das Feld Zusatzposition wird die Zusatzposition eingetragen und auf der Registerkarte Allgemein geben Sie noch die Faktoren für Verkauf und Menge ein. Bei der Berechnung der Mengen und Preise mit Faktoren wird die Menge des Artikels mit dem Faktor multipliziert, um die Menge für die Zusatzposition zu ermitteln. Sind VK fix und Menge Fix definiert, haben diese Werte Priorität. 

Die Zusatzposition wird genau in der Menge generiert, die Sie eingeben, unabhängig von einem evtl. Aufmaß! 

Im Fall von Faktor Menge wird die Menge für die Zusatzposition genau in dem Verhältnis zur Originalposition generiert, wie definiert, unabhängig von den beiden zugrundeliegenden Mengeneinheiten!

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind.

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern.

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
