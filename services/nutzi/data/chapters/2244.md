# Katalogverwaltung Registerkarte Allgemein 
Katalogverwaltung Registerkarte Allgemein 

Auf dieser Registerkarte haben Sie die Möglichkeit, bereits angelegte Gruppen bzw. Produkte mit Hilfe der Radiobuttons Verschieben, Kopieren, Verknüpfen zu verschieben, kopieren oder zu verknüpfen. Wenn Sie z.B. die Gruppe ‚Arbeit‘ verschieben möchten, öffnen Sie die Baumstruktur soweit als nötig, klicken den Radiobutton an und verschieben die Gruppe mit der Maus. 

Drag & Drop Operationen: 

Mit Hilfe der Maus können die verschiedenen Ebenen eines Katalogs bearbeitet werden. Hierfür können diese durch einfaches Ziehen des Elements verschoben, kopiert oder verknüpft werden. Vor Durchführung der Aktion ist die entsprechende Art der Operation auszuwählen, die Auswahl bleibt dauerhaft aktiv. 

Gruppen und Produkte können nur innerhalb eines Katalogs verschoben/verknüpft werden. 

Für das Verschieben, Kopieren oder Verknüpfen gibt es einige Einschränkungen: 

Gruppen, Produkte und Artikel können nur innerhalb eines Kataloges verschoben werden. 

Ein Katalog kann nicht verschoben werden. 

Ist eine Operation für ein Element des Katalogs nicht möglich, wird der entsprechende Radiobutton deaktiviert. 

Wenn Sie eine Gruppe oder ein Produkt gleich über den Button entfernen, so sind alle Verknüpfungen mit betroffen und sind ein für alle Mal gelöscht!

### Combobox Merkmalsklasse 
Für die markierte Gruppen- oder Produktebene kann eine Zuordnung zu einer SML bzw. einer SML-Sicht gewählt werden. Über diese SML- (Sicht) ist eine entsprechende Suche und Anzeige im Web-Shop möglich. 

Wird nichts ausgewählt, wird im Web-Shop generell die Artikelbezeichnung angezeigt, ansonsten die hinterlegte Sachmerkmalleiste bzw. Sicht.

### Button Erben 

Mit diesem Button kann die SML-Einstellung der übergeordneten Ebene bzw. aller übergeordneten Ebenen auf die derzeitig aktive Gruppe / Produkt übernommen werden. Ist der Radiobutton Alle Ebenen aktiv, so wird die erste gefundene SML(Sicht) übernommen.

### Radiobutton Eine Ebene 
Wenn Sie diesen Radiobutton aktivieren, wird die Operation für die Buttons Erben oder Vererben für die nächste Ebene eingestellt. 

### Radiobutton Alle Ebenen 
Wenn Sie diesen Radiobutton aktivieren, wird die Operation für die Buttons Erben oder Vererben für alle Ebenen eingestellt. 

### Button Vererben 

Mit diesem Button kann die SML-Einstellung der untergeordneten Ebene bzw. aller untergeordneten Ebenen vererbt werden. 

Tooltip: 

Mit diesem Button kann die SML-Einstellung der untergeordneten Ebene bzw. aller untergeordneten Ebenen vererbt werden. 

### Checkbox Kein Preis 
Wenn Sie diese Checkbox aktivieren, wird kein Preis genannt. Im Webshop erscheint dann der Satz Preis auf Anfrage . 

### Checkbox Aktion 
Wenn diese Checkbox aktiviert wird, kann der Artikel im Webshop gesondert gestaltet werden. Im Standard ist dies allerdings nicht zu erkennen.
