# Zusatzlieferanten 
Zusatzlieferanten 

Wenn Sie im Dialog Bestellen/Disponieren den Button Anfragen aktivieren, öffnet sich der Dialog Zusatzlieferanten . Hier werden die ausgewählten Lieferanten angezeigt. Sie können auch über die Buttons im Kopf des Dialogs weitere Lieferanten auswählen. Die erste Anfrage wird generiert und für alle ausgewählten Lieferanten kopiert. 

### Serienanfrage 
Beim Anfragen aus dem Auftrag bzw. Angebot heraus, erscheint nach dem Klick auf den Button Anfragen automatisch der Dialog Zusatzlieferanten . 

Hier werden die Lieferanten erfasst, an welche die (Serien-) Anfrage gestellt wird. Mit Betätigung des -Buttons wird je Lieferant eine Anfrage erstellt. Bei Klick auf den Button wird eine Anfrage, wie im Dialog zuvor eingestellt, erzeugt und keine Serienanfrage. Diese Anfragen werden über eine Serienanfragenummer (logisch) zusammengefasst. Das bedeutet, die Serienanfrage existiert nicht als Datensatz, sondern stellt nur einen logischen Verbund (Klammer) mehrerer Anfragen dar. Hierzu wird jede Anfrage einer Serie mit einer Serienanfragennummer versehen, welche im Anfragestamm als Feld sichtbar ist. 

Bei der Erzeugung der Anfragen wird immer dann, wenn mehr als ein Lieferant betroffen ist, eine neue "Anfragen-Vergleichsnummer" erzeugt und in jede erzeugte Anfrage eingetragen. Dazu gibt es einen extra Nummernkreis.
