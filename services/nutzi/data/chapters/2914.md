# X Mapping Test Registerkarte Import

Werkzeuge / EDI X Mapping / X Mapping Test 

# X Mapping Test Registerkarte Import 
X Mapping Test Registerkarte Import 

Für den Import kann auf dieser Registerkarte ein X-Inhalt in das Feld eingefügt werden. Nach dem Import können die Daten betrachtet werden. Das Ergebnis des Imports wird dann in der Registerkarte Daten angezeigt. 
### Button Import 

Mit diesem Button wird der Import gestartet. Nach dem Import können die Daten betrachtet werden. Das Ergebnis des Imports wird dann in der Registerkarte Daten angezeigt. 

### Button Format 

Über den Button Format kann die generierte Datei formatiert angezeigt werden, dies dient der besseren Lesbarkeit der Daten. 

### Button Datenansicht 

Dieser Button zeigt den zu importierenden Text in einer Baumstruktur an, sowie diese in der Mapping-Konfiguration definiert wurde. 

Tooltip: 

Zeigt den zu importierenden Text in einer Baumstruktur an, 
sowie diese in der Mapping-Konfiguration definiert wurde. 

### Button Zurücksetzen 

Mit diesem Button wird das Import-Objekt zurückgesetzt, d.h. es wird ein neues Objekt erzeugt. 

Tooltip: 

Setzt das Import-Objekt zurück, d.h. es wird ein neues Objekt erzeugt.
