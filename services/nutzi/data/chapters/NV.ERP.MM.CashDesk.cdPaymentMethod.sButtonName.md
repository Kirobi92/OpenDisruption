# Kapitel NV.ERP.MM.CashDesk.cdPaymentMethod.sButtonName

Angabe der Bezeichnung, welches Label für den Button im Dialog Bezahlen verwendet wird. 

Die sichtbaren Buttons werden an den Positionen 1-12 im Dialog Bezahlen angezeigt: Hierbei geht es immer von rechts nach links und von oben nach unten. 

Wurde eine Positionsnr nicht vergeben, so wird der entsprechende Button ausgeblendet. Entsprechend ist eine individuelle Einstellung möglich, um die Bedienung zu vereinfachen. 

Siehe Kapitel Bezahlen . 

Tooltip: 

Legt die Beschriftung des Buttons im Bezahlen-Dialog fest. (max. 20 Zeichen)
