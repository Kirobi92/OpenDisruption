# Kapitel NV.ERP.MM.Production.cdParamProduction.shtStorageIDProduction

Wählen Sie hier das Produktionslager aus. Zwischenprodukte und produzierte Teile werden auf diesem Lager deponiert. Chaotische Lager stehen nicht zur Auswahl. Nach Abschluss der Produktion werden dann die produzierten Teile auf das QS-Lager bzw. Produktionslager gebucht. 

Dies kann im Betriebsauftrag übersteuert werden. 

Das Produktionslager darf kein Chaotisches Lager sein. 

Tooltip: 

Bei Produktion aus dem Verkaufsauftrag erfolgt die Buchung auf das Lager der Auftragsposition.
