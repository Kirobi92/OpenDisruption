# Sachbearbeiterstamm Registerkarte Einkauf

Grundeinstellungen für eNVenta ERP / Sachbearbeiterstamm 

# Sachbearbeiterstamm Registerkarte Einkauf 
Sachbearbeiter Registerkarte Einkauf 

Auf dieser Registerkarte legen Sie einkaufsrelevante Funktionalitäten für den Sachbearbeiter fest. 
### Combobox Einkaufsart 
Hier wird die Einkaufsart angegeben. 

Diese Einkaufsart wird in Bestellungen / Dispositionen für den Sachbearbeiter voreingestellt, kann aber geändert werden. 

Tooltip: 

Geben Sie die Einkaufsart an. 

### Combobox Lager WE 
Hier ist zwingend das Lager auszuwählen, auf welches der Sachbearbeiter im Wareneingang standardmäßig den Wareneingang bucht. 

Die Streckenlieferung zu einem normalen Wareneingang wird gedreht. 

Diese Einstellung gilt im Zusammenhang mit dem LVS. 

Tooltip: 

Lagernummer kann auf die Bestellposition und Auftragsposition gedreht werden, wenn dieser Benutzer einen Wareneingang verbucht. 

### Combobox RP Max. Status 
Aus dieser Combobox wählen Sie den Berechtigungsstatus für die Rechnungsprüfung aus. Je nach gewähltem Status hat ein Mitarbeiter das Recht eine Rechnungsprüfung zu prüfen, zu genehmigen, freizugeben oder auch abzulehnen. Hierbei gilt: Das höhere Recht beinhaltet automatisch auch die niedrigeren Rechte. Z.B. das Recht der Ablehnung einer Rechnung beinhaltet auch das Recht der Freigabe der Rechnungsprüfung. In dieser Combobox stehen folgende Status zur Verfügung: 

- leer - Genehmigung - Freigabe - Ablehnung 
Je nach Auswahl darf ein Mitarbeiter eine Rechnung in der Rechnungsprüfung nur erfassen (keine Auswahl), genehmigen (d.h. zustimmen, dass die Rechnung in Ordnung ist), freigeben (und damit die Rechnung zur Zahlung freigegeben) oder ablehnen . 

Siehe Kapitel Rechnungsprüfung . 

Tooltip: 

Angabe des Status der Rechnungsprüfung
