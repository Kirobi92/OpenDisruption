# Kapitel NV.ERP.MM.Sales.cdOrderType.shtNoSpecialPrice

Wenn Sie diese Checkbox aktivieren, wird kein Sonderpreis gezogen. Dies betrifft sowohl die Sonderpreisverwaltung als auch die Sonderpreise in der Konditionsstaffel und dem Artikelstamm.Diese Einstellung wird dann in den Auftrag übernommen, d.h. der Parameter Kein Sonderpreis auf der Registerkarte Kopfdaten wird automatisch aktiviert. 

Ist der Schalter nicht gesetzt und es wird ein Sonderpreis gezogen, so wird dieser im Auftrag ( rot ) markiert.
