# Massenkalkulation 
Massenkalkulation 

Im Betriebsauftrag >> Stamm steht der Eintrag Massenkalkulation zur Verfügung. Da sich Herstellkosten von Baugruppen und Produkten ständig ändern und um dafür die Kalkulationsgrundlage zeitnah aktualisieren zu können, kann mit diesem Dialog eine Massenkalkulation für Produktionsstücklisten durchgeführt werden. 

Das Ergebnis der Kalkulation kann hierbei wahlweise im Artikelstamm in das Feld Letzter EK oder in das Feld Kalkulatorischer EK übernommen werden. Dafür klicken Sie einen entweder den Radiobutton Letzter EK (LEK) oder Kalk. EK (KEK) an. 

Als Kalkulationsmenge wird die optimale Losgröße aus dem Artikelstamm verwendet. Wenn nicht vorhanden, dann Menge 1. 

Grundsätzlich können nur Produktionsstücklisten in Variante 0 kalkuliert werden.
