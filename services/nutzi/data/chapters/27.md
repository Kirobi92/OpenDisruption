eNVenta ERP / Update-Assistent 

# Update-Assistent Registerkarte Datenbank-Trigger 
Update-Assistent Registerkarte Datenbank-Trigger 

Auf dieser Registerkarte werden Datenbanken und Trigger angezeigt. Diese können über die entsprechenden Buttons aktiviert oder deaktiviert werden. 
### Button Ignorieren 

Wenn Sie diesen Button klicken, werden aktive Datenbank-Trigger beim Update ignoriert. Dann kann das Update ausgeführt werden. 

Es wird nicht empfohlen diesen Button zu drücken, da dies die Laufzeit massiv verlängert! 

Tooltip: 

Ignorieren und Update mit aktivierten Triggern ausführen. 

### Button Deaktivieren 

Wenn Sie diesen Button drücken, werden die ausgewählten Trigger deaktiviert. 

Tooltip: 

Deaktivieren Sie die ausgewählten Trigger. 

### Button Alle Deaktiv. 

Wenn Sie diesen Button drücken, werden alle Trigger deaktiviert. 

Tooltip: 

Alle Trigger deaktivieren 

### Button Aktivieren 

Wenn Sie diesen Button drücken, werden die ausgewählten Trigger aktiviert. 

Tooltip: 

Aktivieren Sie die ausgewählten Trigger aktivieren. 

### Button Alle Aktivieren 

Wenn Sie diesen Button drücken, werden alle Trigger aktiviert. 

Tooltip: 

Alle Trigger aktivieren
