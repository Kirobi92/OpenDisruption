# Suchen eines Inventursatzes

Einkauf 

# Inventurscan 
Inventurscan 

Dieser Dialog ist für mobile Scanner gedacht, über welche die Barcodes der Plätze und/oder Artikel direkt eingescannt werden und dann später ausgewertet werden können. Damit der mobile Scanner diesen Dialog nutzen kann, muss ein entsprechender Broker veröffentlicht werden, in welchem der Start-Workflow angepasst wird auf NV.ERP.Navigator.wflStartScanner. 
### Suchen eines Inventursatzes über den Scanner 
Das Feld Matchcode dient als universelles Suchfeld für die jeweiligen Inventurpositionen. Die Suchreihenfolge ist wie folgt: 

Suchreihenfolge Chaoslager 
- Barcode - Lagerplatz - EAN des Artikels - Artikelnr 
Suchreihenfolge statisches Lager 
- Barcode - EAN des Artikels - Artikelnr - Lagerplatz 
Falls mit der Suche ein eindeutiger Inventursatz gefunden werden konnte, wird dieser in den Dialog eingespielt und der Benutzer kann den Datensatz bearbeiten (falls der Status der Inventurposition dies zulässt). Wurde bereits eine Menge zu diesem Inventursatz erfasst, wird diese angezeigt. 

Konnte durch den Scan keine eindeutige Zuordnung gefunden werden (z.B. Scan eines Lagerplatzes, auf dem der Artikel für die Inventur liegt), öffnet sich automatisch der Dialog Inventur - Positionswahl . Hier kann der Benutzer nun den passenden Datensatz auswählen. 

Siehe Kapitel Inventur - Positionsauswahl . 

Über die Buttons Vor und Zurück kann durch die verschiedenen Suchergebnisse navigiert werden. Über den Button Auswahl wird der Datensatz übernommen. 

Konnte kein passender Datensatz gefunden werden, wird der Benutzer gefragt, ob er eine neue Inventurposition erstellen möchte. 

Eingabereihenfolge 
Der Scanner führt den Benutzer über die Cursorsteuerung zu den benötigten Eingabefeldern. Der Ablauf ist wie folgt: 

Chaotisches Lager: 
- Lagerplatz - Artikelnummer - Chargennummer (nur falls es sich um einen Chargenartikel handelt) - Istmenge 
Statisches Lager: 
- Artikelnummer - Chargennummer (nur falls es sich um einen Chargenartikel handelt) - Istmenge 
Nach Eingabe der Istmenge wird der Cursor automatisch auf den ‚Speichern-Button‘ ( ) nach der Istmenge gesetzt. Mit Drücken dieses Buttons wird der Datensatz gespeichert. Durch Drücken der Enter-Taste wird der Datensatz in der Inventur gespeichert. Der Cursor steht wieder automatisch im Feld Matchcode . 

Mit Klick auf den Button Neu wird der komplette Dialog geleert. 

Mit Klick auf den Button Pos.Neu wird eine neue Zeile in der Inventur angelegt. 
### Artikelbild 
Sowohl aus dem Dialog Inventur – Zählen als auch aus dem Dialog Inventur –Positionswahl kann das Artikelbild der aktuellen Inventurposition betrachtet werden. Der Dialog wird per Klick auf den -Button neben der Artikelbezeichnung aufgerufen. 

Siehe Kapitel Artikelbild . 
### Protokollierung der Inventur-Scans 
Jeder Scan einer Inventurposition wird protokolliert. Um die Protokolldaten einsehen zu können, muss der Dialog Protokoll Scanner aufgerufen werden. 

Siehe Kapitel Protokoll Scanner . 

Wurde in der Inventur bereits eine Inventurposition ausgewählt, werden die Werte dieser Position in den Dialog als Vorbelegung übernommen. 
## Steuerelemente des Formulars 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. 

### Feld Inventur 
Hier wird die Nummer der zugrundeliegenden Inventur angezeigt. 

Tooltip: 

Angabe der Inventurnummer 

### Feld Lager 
Hier wird das zugrundeliegende Lager angezeigt. 

Tooltip: 

Anzeige des ausgewählten Lagers 

### Feld Suchname 
Hier geben Sie eine beliebige Bezeichnung für den Artikel (z.B. Artikelname, Artikelnummer, EAN, etc.) zur Selektion ein. Kann der Scan keine eindeutige Zuordnung finden, öffnet sich mit der Tab-Taste der Dialog Inventur - Positionsauswahl . 

### Feld Status 
Hier wird die Gewichtseinheit des Artikels angezeigt. 

Tooltip: 

Angabe des Status-Texts 

### Feld Lagerplatz 
Hier wird der Lagerplatz angezeigt, auf dem der Artikel liegt. 

Tooltip: 

Angabe des Lagerplatzes 

### Feld Artikel 
Hier geben Sie die Artikelnummer zur Selektion ein. 

Wurde das Feld Matchcode gefüllt, wird die Artikelnummer hier automatisch angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Button Artikelbild 

Mit diesem Button verzweigen Sie in den Dialog Artikelbild . 

Siehe Kapitel Artikelbild . 

### Feld Bezeichnung 
Hier wird die Artikelbezeichnung angezeigt. 

Tooltip: 

Artikelbezeichnung 

### Feld Ein. Lager 
Hier wird die Einheit des Artikels angezeigt. 

Tooltip: 

Angabe der Lagereinheit. Dazugehörige Pickliste heißt „Mengeneinheit“. 

### Feld Charge 
Ist der eingelesene Artikel ein Chargenartikel, wird die Chargennummer angezeigt. 

Wenn Sie eine neue Position einfügen möchten, können Sie den Artikel auch über die Chargennummer erfassen. 

Tooltip: 

Angabe der Chargennummer 

### Feld Sollmenge 
Hier wird die Sollmenge des Artikels angezeigt. 

Tooltip: 

Angabe der Menge 

### Feld Ist-Menge 
Hier tragen Sie die IST-Menge des Artikels ein. 

Tooltip: 

Angabe der Menge 

### Button Pos. Neu 

Mit diesem Button können Sie eine neue Inventurposition anlegen. 

### Button Neu 

Mit diesem Button leeren Sie das Feld Matchcode für eine Neueingabe.
