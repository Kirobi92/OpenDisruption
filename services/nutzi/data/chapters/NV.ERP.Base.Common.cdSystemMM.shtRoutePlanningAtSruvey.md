# Kapitel NV.ERP.Base.Common.cdSystemMM.shtRoutePlanningAtSruvey

Wenn Sie diese Checkbox aktivieren, wird bereits bei Auftragserfassung die beste mögliche Tour ermittelt und im Auftrag auf der Registerkarte Logistik in das Feld Tour eingetragen. 

Des Weiteren gilt: Ist diese Checkbox gesetzt, so wird der Auftrag, wenn er bereits eingetourt war, beim Setzen der Liefersperre im Auftragskopf automatisch ausgetourt! 

Beispiel: 
Die Packliste für einen Auftrag ist bereits gedruckt, dann wird der Auftrag auf Liefersperre gesetzt. Der Auftrag wird aus der Tour, die bereits bei Auftragserfassung ermittelt wurde, automatisch ausgetourt. 

Tooltip: 

Beim Speichern einer Auftragsposition erfolgt eine Tourenplanung.
