# Rabatt Verkauf Registerkarte Staffel 
Rabatt Verkauf Registerkarte Staffel 

Auf dieser Registerkarte können Sie Rabatt- oder Preisstaffeln festlegen. 

Werden eine Staffelstufe oder eine Gültigkeit geändert, werden die Felder Sachbearbeiter und Änderungsdatum aktualisiert. Damit dies sofort angezeigt werden kann, muss die Rabattverwaltung komplett neu eingelesen werden, wodurch die Anzeige automatisch auf die Registerkarte Allgemein springt. 

Je Stufe kann nur entweder Rabatt, Preis oder Abschlag angegeben werden! Innerhalb der Staffel ist aber eine Mischung der Varianten möglich. 
### Combobox Gültig von 
Das Datum, das Sie hier auswählen, bestimmt den Beginn der Gültigkeit. 

### Tabelle Der Gültigkeiten 

### Tabellenspalte Menge 
Für jede Stufe, die verwendet wird, muss angegeben werden, ab welcher Menge/Wert/Gewicht des Artikels die Kondition gilt. Dieser Wert ist von der Staffelung abhängig, die Sie in der Registerkarte Allgemein (Bereich Staffelung nach ) eingestellt haben. 

Geben Sie hier den Schwellwert ein (Positionsmenge, Positionswert oder Positionsgewicht), ab dem diese Staffelposition gezogen wird. Falls die jeweilige Positionseinheit von der der Staffel abweicht, findet zuvor eine entsprechende Umrechnung statt. (Die Staffel bezieht ihre Einheiten direkt aus dem Artikelstamm (Einheit VK des Rabattsatzes). Für die Beträge gilt die Systemwährung. 

Tooltip: 

Schwellwert der Position (Positionsmenge, Positionswert oder Positionsgewicht), ab dem diese Staffelposition gezogen wird. 
Falls die jeweilige Positionseinheit von der der Staffel abweicht, findet zuvor eine entsprechende Umrechnung statt. 
(Die Staffel bezieht ihre Einheiten direkt aus dem Artikelstamm (EinheitVk). Für Beträge gilt Systemwährung.) 

### Tabellenspalte Rabatt % 
Hier geben Sie einen prozentualen Rabatt oder Zuschlag ein. Ein positiver Prozentwert bedeutet einen Rabatt, ein negativer Prozentwert einen Zuschlag. Der Prozentsatz wird entweder auf den Basispreis berechnet (falls angegeben) oder auf den Verkaufspreis des Artikels aus dem Artikelstamm. 

Das bedeutet, in der Auftragsposition werden sowohl dieser Staffelrabatt als auch der Grundrabatt gezogen. 

Tooltip: 

Angabe eines prozentualen Rabatts / Zuschlags 

### Tabellenspalte Preis 
Der hier angegebene Preis ist ein absoluter Preis und wird als Verkaufspreis in die Angebots- bzw. Auftragsposition übernommen. 

Tooltip: 

Preis 

### Tabellenspalte Abschlag 
Hier geben Sie einen absoluten Abschlag pro Mengeneinheit ein. Wird das Feld ohne Angabe von pro Menge verwendet, handelt es sich um einen absoluten Zu- oder Abschlag, der sich nicht auf die Menge bezieht. Ein positiver Wert bedeutet einen Abschlag, ein negativer Wert einen Zuschlag. So kann beispielsweise ein Zuschlag von 5,-- Euro pro 100 kg des Artikels berechnet werden. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Abschlag pro 
Hier geben Sie eine Mengeneinheit für den absoluten Zu- oder Abschlag ein. 

Wenn für ‚Abschlag Pro‘ ein Wert hinterlegt ist, orientiert sich der kalkulierte Positions-Abschlagsbetrag an der Positionsmenge der betroffenen Verkaufsposition. Hierfür wird die Positionsmenge durch den ‚Abschlag Pro‘ dividiert und der Staffel-Abschlagsbetrag mit dem Ergebnis multipliziert. Liegen die Mengen in unterschiedlichen Einheiten vor, wird zuvor entsprechend umgerechnet. (Die Mengeneinheit der Staffel entspricht der Einheit VK aus dem Artikelstamm). 

Wenn kein ‚Abschlag Pro‘ hinterlegt ist, wird der Staffel-Abschlagsbetrag 1 zu 1 auf die Verkaufsposition übertragen. 

Beispiel: 

| Auftrag | 1000 kg |
| --- | --- |
| Zuschlag | 5,-- Euro pro 100 kg |
| Summe Zuschlag | = 50,-- Euro |

Tooltip: 

Wenn für 'Abschlag Pro' ein Wert hinterlegt ist, so bezieht sich der Abschlag auf diese Menge in "Einheit Abschl. pro". 
Wenn die Einheit fehlt, so bezieht sich der Pro auf die Verkaufsstammeinheit des Positionsartikels im Verkauf 
bzw. auf die Positionsmengeneinheit im Einkauf. 
D.h. für die Berechnung des Rabattbetrags wird der Abschlagsbetrag auf die Positionsmenge umgerechnet. 
Hierfür wird die Positionsmenge durch den 'Abschlag Pro' dividiert und der Staffel-Abschlagsbetrag mit dem Ergebnis multipliziert. 
Liegen die Mengen in unterschiedlichen Einheiten vor, wird zuvor entspr. umgerechnet. 

Wenn kein 'Abschlag Pro' hinterlegt ist, wird der Staffel-Abschlagsbetrag 1 zu 1 auf die Position übertragen. 

### Tabellenspalte Einheit Abschl. pro 
Hier kann eine Einheit ausgewählt werden. Dies ist die Einheit EK aus Rabatt Einkauf , wenn sie gegeben ist und sonst die Lagereinheit. Die Spalte kann bei fehlendem Artikelbezug auch leer sein. Der Abschlag_pro bezieht sich in diesem Fall dann auf die Positionsmengeneinheit des Einkaufsbelegs. 

Tooltip: 

Als Einheit für den Abschlag pro zieht die Verkaufsstammeinheit des Artikels. 

### Tabellenspalte Mengenskaliert 
Wenn ein Rabattbetrag auf Basis eines Abschlags entsteht, so wird die Checkbox in dieser Spalte aktiviert. Der Rabattbetrag bezieht sich dann auf die Menge der Position. Ändert sich diese, so wird der Betrag neu berechnet. 

Tooltip: 

Wenn gesetzt, so wird der aus dem Abschlag errechnete Rabattbetrag an der Position 
als "Mengenskaliert" markiert. Damit hängt der Rabattbetrag von der Kalkulationsmenge ab. 

### Tabellenspalte Sonderpreis 
Hier können Sie einen Sonderpreis mit Gültigkeitszeitraum hinterlegen. Dieser Sonderpreis hat Priorität gegenüber Preisen, Rabatten und Abschlag für die Staffelstufe, wenn diese im Gültigkeitszeitraum liegt. 

Der Sonderpreis wird um die hinterlegten Rabatte (Grundrabatt1,2) ergänzt, das bedeutet die Betrachtungsweise ist brutto . 

Tooltip: 

Angabe eines Sonderpreis
