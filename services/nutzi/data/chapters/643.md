# Html Text-Wizard Registerkarte Felder 
Text-Wizard Registerkarte Felder 

Bietet ein Dialog oder auch eine Funktion mehrere Positionslisten an, werden diese im Text-Wizard auf dieser Registerkarte angeboten. 

Die Registerkarte Felder bietet Ihnen alle zur Verfügung stehenden Felder für den Aufbau einer Vorlage an. Welche Felder dies sind, hängt davon ab, aus welchem Modul heraus der Text-Wizard geöffnet wird. 
### Button Hinzufügen 

Mit diesem Button fügen Sie die gewählten Felder in die Vorlage ein.
