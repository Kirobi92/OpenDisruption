# Parameter Office Registerkarte Anbindung

Grundeinstellungen für eNVenta ERP / Parameter Office 

# Parameter Office Registerkarte Anbindung 
Parameter Office Registerkarte Anbindung 

Auf dieser Registerkarte legen Sie fest, ob Sie in der Textverarbeitung mit Microsoft Word oder Open Office arbeiten möchten. Für die Tabellenverarbeitung geben Sie ebenfalls an, ob Sie Microsoft Excel oder Open Office Calc verwenden möchten. 

Es werden die Microsoft-Versionen ab 2003 unterstützt, sowie OpenOffice ab Version 2.1. 

Grundsätzlich muss die Datei „soffice.exe“ (nur Windows) bzw. „soffice“ im OpenOffice-Verzeichnis angegeben werden. 

Standard-Pfade (OpenOffice 2.1): 

Windows: C:ProgrammeOpenOffice.org 2.1programsoffice.exe 

Macintosh: 
/Applications/OpenOffice.org 2.1.app/Contents/MacOS/program/soffice 

Linux: 
(Applicationsoffice.) 

Anwendungsbeispiel: 
Eine bedingte Textabfrage mit Benutzerfeldern könnte z.B. so aussehen: 
- Bedingung: oPCSInfo.sCompany2 == “[Firma2]“ - Dann: leer - Sonst: gefüllt 
### Dock Panel OpenOffice Writer verwenden 

Hier wählen Sie aus, ob die Textverarbeitung mit Microsoft Word oder OpenOffice Writer vorgenommen werden soll. 

### Dock Panel OpenOffice Calc verwenden 

Hier wählen Sie aus, ob die Tabellenverarbeitung mit Microsoft Excel oder OpenOffice Calc vorgenommen werden soll. 

### Checkbox Datenexport mit Systemanwendung öffnen 
Tooltip: 

Die exportierten Tabellendaten aus eNVenta in eine MS Excel-Mappe werden 
von der im Betriebssystem festgelegten Standardanwendung geöffnet. 
Dies kann ein anderes Programm sein als Excel. 

### Feld OpenOffice-Pfad 
Dieses Feld wird zugänglich, wenn Sie OpenOffice Writer oder OpenOffice Calc aktiviert haben. Hier geben Sie den Pfad für OpenOffice ein. 

### Button Suche 

Mit diesem Button können Sie den Pfad suchen.
