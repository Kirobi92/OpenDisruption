# Bankverbindungen der Kunden bzw. Lieferanten Registerkarte International 
Bankverbindung Registerkarte International 

Auf dieser Registerkarte definieren Sie die Einstellungen für internationale/außer-europäische Zahlungsvorgänge. 

Aus der Spalte 'Spesenregelung' können Sie auswählen, wie die Kosten des Geldtransfers verteilt werden sollen. Es stehen Ihnen 3 Verteilungsarten zur Verfügung: 

- Auftraggeber / Begünstigter: Verteilung 50% Auftraggeber/50% Begünstigter - Auftraggeber: Verteilung 100% Auftraggeber - Begünstigter: Verteilung 100% Begünstigter 
Wählen Sie keine der 3 Verteilungsarten aus, wird automatisch die Verteilungsart ‚Auftraggeber/Begünstigter‘ hinterlegt. 
### Tabelle 

### Tabellenspalte Nr. 
Hier wird die Standard-Bankverbindung angezeigt. Gibt es keine speziell gekennzeichnete Bankverbindung, wird die jüngste Bankverbindung zum Kunden bzw. Lieferanten angezeigt. Für diesen Vorgang kann die Bankverbindung auch geändert werden. 

### Tabellenspalte Gesellschaft 
Hier wird die Gesellschaftsnummer für das Verfahren 'Multicompany' angegeben. 

Siehe Kapitel Gesellschaftsverwaltung für Multicompany . 

Tooltip: 

Gesellschaft für das Verfahren "Eine Warenwirtschaft". 

### Tabellenspalte IBAN 
Hier wird die IBAN eingetragen. 

Tooltip: 

IBAN (International Bank Account Number) 

### Tabellenspalte Nummer 
Hier wird die Kontonummer der Zahlungsverbindung angegeben. 

Tooltip: 

Kontonummer der Zahlungsverbindung 

### Tabellenspalte Bankname 
Hier wird der Name der Bank angegeben. 

Tooltip: 

Angabe des Banknamens 

### Tabellenspalte BIC 
Hier wird die BIC-Nummer eingetragen. 

Tooltip: 

Bank Identifier Code 

### Tabellenspalte BLZ / SWIFT 
Hier geben Sie den Bank-Code an. Dies ist eine Pflichtangbe. Gültige Werte sind abhängig vom Eintrag im Feld Typ . Entweder 
Typ = 'BLZ', dann Bankleitzahl der Bank 
Typ = 'SWIFT', dann Swiftcode der Bank. 

Tooltip: 

Angabe der Bankleitzahl / SWIFT 

### Tabellenspalte Währung 
Hier wird das Währungskürzel angegeben. 

Tooltip: 

Währungskürzel 

### Tabellenspalte Standardverbindung 
Wenn die Checkbox in dieser Spalte gesetzt ist, wird diese Bankverbindung beim Debitor, Kreditor oder im Unternehmen als Standardbankverbindung verwendet. 

Tooltip: 

Angabe, ob Standard-Bankverbindung
