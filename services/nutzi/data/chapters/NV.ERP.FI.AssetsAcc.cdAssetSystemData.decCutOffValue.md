# Kapitel NV.ERP.FI.AssetsAcc.cdAssetSystemData.decCutOffValue

Hier kann ein Erinnerungswert für die Anlagegüter vorbelegt werden. Dieser Wert kommt nicht zum Tragen, wenn die Anlagenklasse oder das Anlagegut einen Wert erhält. 

Ein Erinnerungswert ist ein Restwert auf ein Anlagegut, für welchen das System keine weitere Abschreibung errechnet. Damit bleiben bereits abgeschriebene Anlagen in der Bilanz. 

Für spezielle Bewertungsansätze (z.B. Vermögensrecht in manchen Ländern) kann es gewünscht sein, die Abschreibung bei einem bestimmten Wert enden zu lassen. 

Tooltip: 

Hier kann ein Erinnerungswert für Anlagegüter eingetragen werden.
