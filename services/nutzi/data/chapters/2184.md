# Serviceeinsatz Registerkarte Kopfdaten 
Serviceeinsatz Registerkarte Kopfdaten 

Auf dieser Registerkarte steht Ihnen ein Feld zur Verfügung in das Sie eine Beschreibung oder einen Kommentar zum Serviceeinsatz eintragen können. Dieser Text wird im Auftrag auf der Registerkarte Texte im Feld Memo angezeigt. 
### Feld Grund/Aufgabe 
Hier können Sie eine Beschreibung oder einen Kommentar zum Serviceeinsatz erfassen. 

Tooltip: 

Beschreibung zum Serviceeinsatz 

### Combobox Phase 
Hier können Sie zusätzlich zur Projektnummer auch die Projektphase für den Serviceeinsatz eintragen. Wird keine Projektphase angegeben, so werden alle Serviceeinsätze in der ersten Projektphase berücksichtigt. 

Tooltip: 

Zusätzliche Angabe der Projektphase. 

### Checkbox Letzte Wartung 
Ist dieser Parameter gesetzt, erscheint auf der Registerkarte Positionen in der Spalte ‚Bearbeitet‘ das letzte Wartungsdatum, wenn der Status auf 2 gesetzt ist. 

Tooltip: 

Wenn aktiviert, wird das Datum der letzten Wartung auf der Registerkarte 'Positionen' angezeigt. 

### Feld Erinnerungsstufe 
Hier kann für den Serviceeinsatz eine Erinnerungsstufe angegeben werden. Auf diese Weise können Kunden rechtzeitig eine Erinnerung an eine Wartung etc. erhalten (per E-Mail). Diesen Erinnerungen können Termine, Karenztage und entsprechend formulierte Erinnerungstexte enthalten. 

Diese werden im Dialog Parameter Service Erinnerung definiert.
