# Mahngruppen Einkauf 
Mahngruppen Einkauf 

Dieser Dialog öffnet sich über den Button Gruppen im Dialog Mahnwesen Einkauf . Dieser Dialog gibt Ihnen einerseits einen Überblick über bereits vorhandene Mahngruppen für Lieferanten und bietet andererseits die Möglichkeit neue Mahngruppen anzulegen. 

Pro Mahngruppe können unterschiedliche Mahnparameter hinterlegt werden, um z.B. die unterschiedlichen Lieferzeiten und evtl. Karenzzeiten zu berücksichtigen. 

Damit ein Lieferant im Mahnwesen berücksichtigt werden kann, muss er einer Mahngruppe zugeordnet werden. Die Einstellungen dazu nehmen Sie im Lieferantenstamm auf der Registerkarte Warenwirtschaft vor. 

Siehe Kapitel Lieferantenstamm Registerkarte Warenwirtschaft . 

Standardmäßig gibt es nur eine Mahngruppe, in der alle Lieferanten eingebunden sind. 

Um ein selektives Mahnwesen zu ermöglichen, können Lieferanten aber auch verschiedenen Mahngruppen zugeteilt werden.
