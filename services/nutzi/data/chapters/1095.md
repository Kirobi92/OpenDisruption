# Mengenänderungen im Auftrag 
Soll nur eine Teilmenge ausgeliefert werden, kann der gewünschte Wert direkt in der Spalte ‚Menge’ in der Positionstabelle eingegeben werden (nur bei bereits gespeicherten Positionen). Alternativ kann im Dialog Position die gelieferte Menge auf den gewünschten Wert geändert werden. 

Bei der Fakturierung der Position wird automatisch eine neue offene Restposition erzeugt und dem Auftrag hinzugefügt. Diese kann dann wiederum teilgeliefert werden usw. In die neu entstandene Position können Sie natürlich auch einen neuen Liefertermin eintragen und weitere Änderungen vornehmen. 

Achten Sie bitte darauf, dass die beiden Checkboxen Restpos bei Lieferung bzw. Restpos mit Status 2 aus den Systemparametern des Verkaufs gesetzt sind. Je nach Einstellung hat dies Auswirkungen auf die Mengenänderungen. 

Auf diese Weise können beliebig viele Abrufe durchgeführt werden, bis die Restmenge 0 erreicht und der Auftrag vollständig fakturiert ist. 

Vorteil hierbei ist auch, dass Sie anhand der einzelnen Teilpositionen genau die Abrufhistorie des Artikels in der Übersicht haben. 

Hilfreich ist dabei auch die Spalte 'R' in der Positionstabelle. Sie enthält ein R (Rückstand), wenn die gelieferte Menge kleiner als die bestellte Menge ist, also ein Lieferrückstand besteht. Ist das Kennzeichen Kein Rückstand in den Parametern des Verkaufs eingeschaltet, soll also keine Restlieferung erfolgen, zeigt die Spalte ein S (S= streichen) an. Mit Hilfe der Spalte 'R' erkennen Sie sofort, welche Positionen des Auftrags Rückstände enthalten. 

Ein Restkennzeichen ‚R‘ auf einer Position mit Untermenge bedeutet, dass, sobald eine entsprechende Restposition generiert werden würde (also beim Lieferschein- oder Rechnungsdruck), eine solche auch tatsächlich angelegt wird. Dementsprechend bedeutet ‚S‘, dass zum genannten Zeitpunkt eben keine Restposition generiert wird. NACH dem Generierungstermin der Restposition wird nun das Restkennzeichen vom System automatisch auf ‚S‘ gesetzt, um so zu verhindern, dass diese Quellposition z.B. bei einem weiteren Lieferscheindruck nochmals berücksichtigt wird. 

Siehe Kapitel Parameter Verkauf Registerkarte Allgemein . 

Der Dialog Position enthält auch die Option, die Erzeugung einer Restposition zu unterbinden. Dazu wird die Checkbox Rest streichen aktiviert. 

Siehe Kapitel VK-Positionen bearbeiten . 

Bei der Generierung einer Restposition wird diese mit einem speziellen Kennzeichen versehen. Sie enthält im Datenbankfeld RestAusPos die Positionsnummer der Originalposition. In den Auftragsformularen steht dieses Feld zur Verfügung und kann zur Kennzeichnung von Restpositionen (z.B. Hinweis: Rückstandsposition) verwendet werden.
