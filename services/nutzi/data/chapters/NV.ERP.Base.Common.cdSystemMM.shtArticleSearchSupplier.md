# Kapitel NV.ERP.Base.Common.cdSystemMM.shtArticleSearchSupplier

Wenn Sie diese Checkbox aktivieren, können Sie die Suche nach Artikeln im System gezielt über die Lieferanten-Bestellnummer des Rabatt-Einkaufs steuern. 

Tooltip: 

Hier kennzeichnen Sie, ob bei der Artikelsuche die Bestellnummer des Lieferanten aus dem Rabatt-Einkauf verwendet werden soll.
