# Lieferbedingungen 
Lieferbedingung 

Unter dem Menü Stamm >> Lieferbedingungen , z.B. im Kunden- oder Lieferantenstamm finden Sie den Dialog zur Pflege der Lieferbedingungen. Hier definieren Sie Lieferbedingungen für Ihr Unternehmen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Lieferbedingung 
Tooltip: 

In diesem Dialog können Lieferbedingungen definiert werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report .

Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl.

Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Sprache 

Über diesen Button können auch Lieferbedingungen in Fremdsprachen hinterlegt werden. Dazu markieren Sie die Zeile, für die Sie einen Text in einer Fremdsprache hinterlegen möchten und klicken auf diesen Button. Es öffnet sich der Dialog Fremdsprachentexte . 

Siehe Kapitel Fremdsprachentexte . 

### Tabelle Lieferbedingungen 

### Tabellenspalte Nummer 
Jede Lieferbedingung erhält eine Nummer, mit der sie eindeutig identifiziert und zugeordnet werden kann. Diese Nummer wird manuell vergeben und darf nicht doppelt belegt sein. 

Wird hier nichts eingetragen, erhalten Sie eine Fehlermeldung. 

Tooltip: 

Eindeutige ID der Lieferbedingung. Der Wert ist nachträglich nicht änderbar. 
Der Wert muss zwischen 1 und 999 liegen. 

### Tabellenspalte Text 
Der Text für die Lieferbedingung kann mit Hilfe des Report-Writers in alle auftragsbegleitenden Formulare an beliebiger Stelle integriert werden. 

Tooltip: 

Angabe der Vereinbarung zur Lieferung. 
Steuert unter anderem die Angabe von Portokosten oder Franko-Lieferung. 

### Tabellenspalte Franko 
Ist diese Checkbox gesetzt, erfolgt die Lieferung Franko. Eventuell im Auftrag angegebene Porto- und Verpackungskosten werden nicht auf den Warenwert aufgeschlagen, sondern sind im Warenwert enthalten. Das Franko-Kennzeichen wird in Angebot und Auftrag übernommen und kann dort geändert werden. 

Tooltip: 

Wenn dieser Parameter gesetzt ist, erfolgt die Lieferung franko. 

### Tabellenspalte Porto 
Hier kann eine Porto- bzw. Frachtpauschale angegeben werden, die automatisch bei dieser Lieferbedingung in das Angebot bzw. den Auftrag übernommen wird. Eine Änderung ist jederzeit möglich. 

Tooltip: 

Geben Sie eine eine Porto- bzw. Frachtpauschale ein.
