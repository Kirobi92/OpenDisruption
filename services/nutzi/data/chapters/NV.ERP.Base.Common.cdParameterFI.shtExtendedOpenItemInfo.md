# Kapitel NV.ERP.Base.Common.cdParameterFI.shtExtendedOpenItemInfo

Wenn Sie diesen Parameter aktivieren, werden in den nachfolgend aufgeführten Vorschlägen weitere Spalten angeboten, die genau informieren, in welchen Vorschlägen ein OP enthalten ist: 

- OP Debitoren - OP Kreditoren - OP Ausbuchung - OP Verrechnung - Belegauskunft - OP Suche - Bonitätsinformation - Bankeinzug - Zahlungsverkehr - Mahnen - Zinsrechnung - Verbandsabrechnung 

Tooltip: 

Info, ob ein OP in einem Zahlungs-, Mahn-, Verbandsabrechnungs- oder Zinsrechnungs-Vorschlag enthalten ist.
