# Kapitel NV.ERP.Base.Common.cdSystemBase.shtLuceneSearchMaxHitCount

Hier geben Sie die maximale Anzahl an Treffern ein, die in der Ergebnisliste von Lucene angezeigt werden sollen.

Maximale Trefferanzeige: 4000
