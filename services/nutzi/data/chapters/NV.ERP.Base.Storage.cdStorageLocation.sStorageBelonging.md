# Kapitel NV.ERP.Base.Storage.cdStorageLocation.sStorageBelonging

Dieses Feld wird gefüllt, wenn Sie im Feld Auswahl einen Lagerort markiert haben und hängt von den Einstellungen für das gewählte Lager ab. Es gibt u.a. den Eintrag Eigenes Lager , CC Lager etc.
