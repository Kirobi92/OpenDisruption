# Umsatzlisten Registerkarte Statistik 
Umsatzlisten Registerkarte Statistik 

Auf dieser Registerkarte wird das Ergebnis Ihrer Auswertung angezeigt. 

Über einen Klick mit der rechten Maustaste in eine Zelle können Sie einen Datenexport ins Excel starten. 
### Feld Titel 
Hier wird der Titel der Leistungsstatistik angegeben. 

Tooltip: 

Angabe des Titels der Leistung 

### Feld Von / Bis 
Dieses Feld zeigt das Von/Bis-Datum des gewählten Zeitraums. 

### Feld Von / Bis VZ 
Hier wird das Von/Bis-Datum des Vorzeitraums (Vergleichszeitraum) angezeigt. 

Sollte bei dem Auswertungszeitraum ein Zeitraum größer 1 Jahr ausgewählt werden, so wird der Vorzeitraum(VZ) zum Datum von/bis minus dem Auswertungszeitraum ausgewertet. 

Beispiel: 

Auswertungszeitraum von 01.01.2017 - 31.12.2021 
Vorjahreszeitraum ist nun 01.01.2012 - 31.12.2016 

Tooltip: 

Zeigt das von/bis Datum des Vorzeitraums (Vergleichszeitraum). 

Sollte beim Auswertungszeitraum ein Zeitraum größer 1 Jahr ausgewählt werden, so wird der Vorzeitraum(VZ) zum Datum von/bis minus des Auswertungszeitraums gerechnet. 
z.B. 
Auswertungszeitraum von 01.01.2016 - 31.12.2020 
Vorjahreszeitraum ist nun 01.01.2011 - 31.12.2015 

### Tabelle 

### Tabellenspalte Gruppe 
Diese Spalte wird je nach Auswahlkriterium ausgeblendet. In dieser Spalte ist die Nummer der Gruppe eingetragen. 

### Tabellenspalte Gruppenname 
Diese Spalte wird je nach Auswahlkriterium ausgeblendet. In dieser Spalte ist die Bezeichnung der Gruppe eingetragen. 

### Tabellenspalte Schlüssel 
Hier wird die Kurzbezeichnung entsprechend der Auswahl in der Combobox Aufschlüsseln angezeigt. 

### Tabellenspalte Name 
Die Bezeichnung zum gewählten Schlüssel wird angezeigt. 

### Tabellenspalte Umsatz 
Der Umsatz aus der gewählten Periode wird angezeigt. 

### Tabellenspalte Ertrag 
Der Ertrag aus der gewählten Periode wird angezeigt. 

### Tabellenspalte Ertrag % 
Der Ertrag wird in % dargestellt. 

### Tabellenspalte Umsatz VZ 
Hier wird der Umsatz aus dem Vorjahr angezeigt. Die Anzeige bezieht sich auf die Vorjahres-Periode. 

### Tabellenspalte % 
Der Umsatz des Vorjahrs wird in % dargestellt. 

### Tabellenspalte Ertrag VZ 
Hier wird der Ertrag aus dem Vorjahr angezeigt. Die Anzeige bezieht sich auf die Vorjahres-Periode. 

### Tabellenspalte % 
Der Ertrag aus dem Vorjahr wird in % dargestellt. 

### Tabellenspalte Menge 
Die umgesetzte Menge wird eingetragen. 

### Tabellenspalte Menge VZ 
Die Menge aus dem Vorjahr wird angezeigt. Die Anzeige bezieht sich ebenfalls auf die Vorjahres-Periode. 

### Tabellenspalte Anzahl 
Diese Spalte enthält die Zahl der enthaltenen Belegpositionen, d.h. Anzahl Verkäufe/Lieferungen. 

Durch Sortierung nach dieser Spalte können Renner/Penner ermittelt oder eine Lageroptimierung vorgenommen werden.
