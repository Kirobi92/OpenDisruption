# Benutzerprofil 
Benutzerprofil 

Dieser Dialog öffnet sich über den Button Benutzerprofil im Benutzerkonto und listet die Informationen zu einem Benutzer auf. Einen Gastkunden erkennen Sie am Minus-Zeichen vor der Nummer. Auf den einzelnen Registerkarten können Sie die Daten zu einem Benutzer ergänzen. Grundsätzlich werden auf der linken Seite immer die Daten aus der Benutzeranmeldung im Webshop angezeigt. Diese dienen der Orientierung bei der Anlage des Benutzers als Kunde im Kundenstamm. 

Bei einem Benutzer, der bereits im Menü Kunde angelegt ist, werden die Daten aus dem Kunden gezogen. 

Einem angelegten Kunden können mehrere Benutzer zugewiesen werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Benutzerprofil 
Tooltip: 

In diesem Dialog finden Sie die Daten zu einem Benutzer. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Profil -> Kunde 

Wenn Sie diesen Button drücken, wird der Kunde mit den Daten aus dem Internetprofil aktualisiert. 

Achtung: Dabei gehen Altdaten verloren! 

Tooltip: 

Klicken Sie hier, um den Kunden mit den Daten aus dem Internetprofil zu aktualisieren. (Altdaten gehen dadurch verloren!) 

### Button Kunde -> Profil 

Wenn Sie diesen Button drücken, werden die Kundendaten in das Internetprofil übernommen. 

Tooltip: 

Klicken Sie hier, um die Daten aus dem Kunden in das Internetprofil zu übernehmen.
