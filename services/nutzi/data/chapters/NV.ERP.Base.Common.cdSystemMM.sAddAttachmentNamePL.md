# Kapitel NV.ERP.Base.Common.cdSystemMM.sAddAttachmentNamePL

Hier geben Sie die Formel zur Ermittlung des Dateinamens ein bei der Generierung einer E-Mail für den Packlistendruck. Es wird immer fix der Wert "Packliste" verwendet, gefolgt von der Formel als Dateiname. 

Tooltip: 

Formel zur Generierung einer E-Mail für den Druck der Packliste.
