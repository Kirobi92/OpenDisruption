# Datenimport bearbeiten Registerkarte Einheiten 
Import Bearbeiten Registerkarte Einheiten 

Auf dieser Registerkarte werden die Einheiten eingelesen. Die eingelesenen Daten können nach unterschiedlichen Kriterien gefiltert werden. 
### Button Bearbeiten 

Mit diesem Button können die Artikel-Einheiten bearbeitet werden. 

Tooltip: 

Artikel Einheiten bearbeiten 

### Button Einheit Verarbeiten 

Mit diesem Button werden ausgewählte Einheiten in den Schattenstamm übernommen. 

Tooltip: 

Ausgewählte Einheiten in den Schattenstamm übernehmen 

### Feld SQL-Information 
Tooltip: 

SQL Bedingung aktiv 

### Tabelle 

### Tabellenspalte Importschlüssel 
Hier wird der Import-Schlüssel angegeben. 

Tooltip: 

Der "Import-Schlüssel" ist ein generierter Wert, der als eindeutige ID (Referenzschlüssel) zur Referenzierung des Datensatzes dient. Er wird aus den Haupteigenschaften des Datensatzes zusammengesetzt. Jeder Datensatz benötigt einen eigenen, eindeutigen Import-Schlüssel. Es wird kein neuer Datensatz angelegt, wenn es bereits einen Datensatz mit demselben Import-Schlüssel gibt, sondern der bereits vorhandene überschrieben/aktualisiert. 

### Tabellenspalte Haupt-Importschlüssel 
Hier wird der Haupt-Importschlüssel angegeben. 

Tooltip: 

Angabe des Haupt-Importschlüssels 

### Tabellenspalte Einheit VK 
Hier wird die Verkaufseinheit angegeben. 

Tooltip: 

Verkaufseinheit 

### Tabellenspalte Faktor VK 
Hier wird ein Umrechnungsfaktor von Lagereinheit zu Verkaufseinheit (bezogen auf den Preis) angegeben, um den in Relation korrekten Verkaufspreis zu ermitteln. 

Tooltip: 

Angabe des Umrechnungsfaktors von Lager- zu Verkaufseinheit, bezogen auf den Preis. 

### Tabellenspalte Standardeinheit 
Wenn Sie diese Checkbox aktivieren, wird die Standardeinheit voreingestellt. Diese Einheit wird bei der Erfassung einer Position im Auftrag voreingestellt. Diese Einstellung kann über die entsprechende Combobox geändert werden. 

### Tabellenspalte GTIN 
Hier wird die Global Trade Item Number angegeben. Die Global Trade Item Number (GTIN) ist eine von der GS1 verwaltete und vergebene Identifikationsnummer, mit der Produkte und Packstücke weltweit eindeutig identifiziert werden können. 

Die GTIN muss eindeutig sein und der Aufbau der GTIN wird beim Speichern geprüft. 

Die GTIN wird im ASCII-EDI nicht berücksichtigt! 

Siehe Kapitel Global Trade Item Number . 

Tooltip: 

Global Trade Item Number 

### Tabellenspalte EAN 
Hier wird die EAN-Nummer des Artikels angegeben. EAN bedeutet European Article Number und bezeichnet eine internationale, eindeutige Kennzeichnung für Produkte. Der EAN Code wird meist als maschinenlesbarer Strichcode mit Zahlen auf der Verpackung einer Ware außen gut sichtbar angebracht, um von Barscannern / Scannerkassen schnell eingelesen werden zu können. 

Tooltip: 

Die EAN-Nummer wird angegeben. Diese EAN-Codenummer dient als Suchfeld in der Auftrags- und Bestellerfassung. 

### Tabellenspalte Einheit Typ 
Hier wird der Typ der Einheit angegeben. 
V = Verkaufseinheit 
E = Einkaufseinheit 

Tooltip: 

V: Verkaufseinheit 
E: Einkaufseinheit 

Im Falle des X-Mapper Imports: 
A: Verkaufs- und Einkaufseinheit
