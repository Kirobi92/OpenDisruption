# AES-Übertragung 
Datenübertragung AES 

Dieser Dialog öffnet sich in der Kommissionierung >> Optionen >> AES-Schnittstelle >> AES-Übertragung . Über dieses Menü kann eine weitere Bearbeitung der AES -Daten erfolgen. Die Daten können das erste Mal an AES übertragen, aktualisiert oder storniert werden. Ebenso ist eine Löschung der Daten möglich, wenn noch keine Übergabe an AES erfolgt ist. Welche Aktion für den einzelnen Lieferschein möglich ist, bestimmt der Zollstatus aus AES . 

Es können Lieferscheine gelöscht werden, welche bereits an AES übertragen wurden. In diesem Fall werden die Lieferscheine zusätzlich mit gelbem Hintergrund angezeigt. 

Existiert eine Rechnung zum Lieferschein, wird diese ebenso an AES übertragen. 

Bei besonderen Artikeln müssen eine Zolleinheit und ein entsprechender Umrechnungsfaktor im Artikel angegeben sein. 

Siehe Kapitel Artikelstamm .

### Button Senden 

Mit diesem Button werden die selektierten Vorgänge an AES übertragen. Sinn des Sendevorgangs ist es, die jeweiligen Daten zur Validierung und Nacherfassung ans AES zu senden, so dass im Anschluss von dort aus dieser Vorgang an die Zollbehörde übermittelt werden kann. 

### Button Storno 

Dieser Button schickt einen sogenannten Löschsatz ans AES mit dem Ziel, dort diesen Vorgang zu stoppen bzw. zu löschen. Ob die Stornierung Erfolg hat, hängt vom Zollstatus ab. 

### Button PlausiStat 

Dieser Button informiert Sie in einem Info-Dialog über den Plausibilitätsstatus.

### Checkbox Vor Übertragung AES-Vorgang aktualisieren 
Diese Checkbox wird automatisch aus den Parametern vorbelegt. Ist sie gesetzt, werden unmittelbar vor dem Übertragen der Daten an AES , die zu übertragenden Elementardaten des Lieferscheins nochmals ermittelt, validiert, verdichtet und gespeichert. 

Diese Option ist hilfreich, wenn eine längere Zeitspanne zwischen der Datenerfassung und dem eigentlichen Absenden liegt. Somit können nachgeschaltete Änderungen an einem Lieferschein nochmals berücksichtigt werden. 

Wird zeitnah gesendet und kann daher eine nachträgliche Lieferscheinmanipulation ausgeschlossen werden, ist es sinnvoll diese Checkbox NICHT zu setzen. 

Tooltip: 

Wenn aktiviert, werden die zu übertragenden Elementardaten des Lieferscheins vor Übertragung nochmals ermittelt, validiert, verdichtet und gespeichert.
