# Anlagenklasse Detail 
Anlagenklasse Detail 

Dieser Dialog öffnet sich über den Button Details im Dialog Anlagenklassen . Hier werden die Anlagenklassen um alle abschreibungsrelevanten Informationen / Felder ergänzt und eine weitere Vordefinition ist möglich. Alle abschreibungsrelevanten Konten, AfA-Methoden und sonstige Einstellungen können eingetragen werden. Diese Einstellungen sind dann im Anlagenstamm auf den Registerkarten voreingestellt. Dies vereinfacht die Neuanlage ähnlicher Anlagen. Die Daten können geändert werden. 

Die Übersicht über die Bereichskennzeichen findet auch in einer Tabelle statt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Anlagenklasse Detail 
Tooltip: 

In diesem Dialog werden Anlagenklassen um alle abschreibungsrelevanten Informationen / Felder ergänzt und eine weitere Vordefinition ist möglich. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
