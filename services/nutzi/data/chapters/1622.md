# Tableauverwaltung 
Tableauverwaltung 

Dieser Dialog öffnet sich im Lagerauftrag >> Stamm >> Tableauverwaltung . Hier werden die einzelnen Tableaus angelegt bzw. verwaltet. 

Das verwendete Lagersystem ist ein automatisches Hochregallagersystem, welches die eingelagerten Waren an einer zentralen Entnahmestelle (= Entnahmeöffnung) ausgibt. Hierbei lagern die Waren auf Regalbrettern (sog. Tablare bzw. Tableaus). Diese können innerhalb des Containers mit einem Shuttle optimiert ein- und umgelagert werden. 

#### Tableauverwaltung Typen 
Wenn Sie den Button Typen anklicken, ändern sich die Felder des Dialogs. In der Tableau-Typverwaltung können Beispiele/Baupläne für konkrete Tableaus zur Verfügung gestellt und bei der Erstellung der einzelnen Tableaus verwendet werden. 

Als Lagerbereich erhalten diese dann das Kennzeichen „T“ für Typ als Unterscheidungsmerkmal zu den konkreten Tableaus, die dann auch konkrete Lagerbereiche in diesem Feld hinterlegt haben.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Typen 

Dieser Button ändert die Einstellungen im Dialog auf Typen .
