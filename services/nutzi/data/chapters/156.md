# Restmengenhandling 

### Funktionen 
Im Restmengenhandling kann man verschiedene Optionen auswählen, warum Sie eine geringere Menge eingegeben haben, als erwartet war. 

Welche Einstellungen ein Sachbearbeiter hat, kann unterschiedlich sein. Dies kann in der Scannersachbearbeitereinstellung konfiguriert werden. 

Folgende Optionen stehen maximal zur Verfügung: 

1 Rest später 2 Fehlmenge 3 Mindermenge OK 
ACHTUNG: Sie verändern damit die gelieferte Menge des Auftrags. 4 Rest Zuladung 5 Neue Behälter 
| Wählen Sie diese Option, um den Rest des Picks zu einem späteren Zeitpunkt zu bearbeiten. Dies können Sie selbst tun oder jemand anderem überlassen. Überlassen Sie es jemand anderem, so wird diese neue Entnahme wieder in den Pick Pool zurückgelegt. |
| --- |
| Wählen Sie diese Option, wenn nicht ausreichend Ware am Platz vorhanden ist. Dadurch wird der Bestand korrigiert. Das System versucht dann automatisch einen weiteren Bestand zur Entnahme zu finden. Diese neue Entnahme wird dann versucht in den Kommissionierprozess einzubinden. |
| Wählen Sie diese Option, wenn die kommissionierte Menge so in Ordnung ist (Beispiel eine Packungseinheit enthält 100 Stk., kommissioniert werden sollen 103 Stk.). |
| Wählen diese Option, wenn der Rest bei der Aufladung des Transportfahrzeuges direkt entnommen werden soll. |
| Wählen Sie diese Option, wenn Sie den Rest in einen neuen Kommissionierbehälter packen wollen. Sie erhalten den nachfolgenden Vorgang dann direkt auf den Scanner und müssen dann nur noch die Menge eingeben und den neuen Behälter scannen. |

## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Barcode 

Wird dieser Button gedrückt, so wird, wenn eine Kamera am Gerät vorhanden ist, die Barcodeerkennung aktiviert. 

Dabei werden folgende Barcodetypen erkannt: Code128, EAN8, EAN13.

Dieser Button öffnet das Einstellungsmenü für den Scanner. 

### Button Aktualisieren
