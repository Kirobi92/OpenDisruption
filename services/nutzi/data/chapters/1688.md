# Abschluss Scan 
Abschluss Scan 

Damit die Pakete von V-Log an den Paketdienstleister übermittelt werden können, gibt es die Funktion Abschlussscan . Hier wird der entsprechende Dienstleister ausgewählt und anschließend werden alle externen Paketnummern, welche auf dem Versandetikett angedruckt sind, angescannt. Gibt es für den Kunden an dem Tag keine Tourenbelieferung, erscheint in dem Dialog die Meldung OK und die Daten werden an den Paketdienstleister übermittelt. 

Gibt es für den Kunden an dem Tag noch eine Lieferung per Tour, dann erscheint ein roter Hinweistext, dass der Kunde auch per Tour beliefert wird (Doppelbelieferungsschutz, hierzu mehr in Kapitel Doppelte Anlieferung ausschließen ). Die Daten werden nicht an den Paketdienstleister übermittelt und das Paket wird auf die entsprechende Tour umgebucht. 

Siehe Kapitel Doppelte Anlieferung ausschließen . 

Siehe Kapitel Doppelbelieferungsschutz .
