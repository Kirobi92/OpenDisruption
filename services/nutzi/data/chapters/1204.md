# Parameter Scanner Registerkarte TCP/IP 
Parameter Scanner Registerkarte TCP/IP 

Auf der Registerkarte TCP/IP werden die IP-Adresse und der Port zum ausgewählten Arbeitsplatz angezeigt. Hier wird geprüft, ob die Ziel-IP erreichbar ist und ob Daten abgerufen werden können. 
### Feld IP-Adresse 
Hier geben Sie die IP-Adresse an. 

Tooltip: 

Die IP des Scanners / des zugehörigen Cradles. 

### Feld Port 
Hier geben Sie den Port an. 

Tooltip: 

Der Port, an den der Scanner Daten schickt.
