# Gutschriftsarten Registerkarte Allgemein 
Gutschriftsarten Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die allgemeinen Einstellungen für die Gutschriftsarten. 
### Combobox Lager 
Hier wählen Sie das Lager aus, das für diese Belegart genutzt werden soll. 

Das hier ausgewählte Lager überschreibt das Standardlager, das dem Sachbearbeiter zugeordnet ist. 

Tooltip: 

Nummer des Lagers 

### Checkbox Strecke 
Im Rahmen eines Streckengeschäfts können auch Gutschriften erfasst werden. 

Auswählbare Gutschriftsarten auf Basis einer solchen Retoure können nur als "Strecke" geflaggte Belegarten sein. Außerdem ist die Gutschriftsart standardmäßig vom Typ "Wertgutschrift". 

Hierbei gilt für die Gutschrift: 

- Die Gutschrift muss vom Typ „Strecke“ sein. - Die Gutschrift wird automatisch als "Wertgutschrift" gekennzeichnet. - Wurde die Retoure mit Lagerbuchung erfasst, so ist die Gutschrift kalkulatorisch so zu handhaben wie eine Warenrückgabe von Lagerware. (EK und VK aus dem Auftrag) - Wurde die Retoure ohne Lagerbuchung erfasst, so ist die Gutschrift kalkulatorisch so zu handhaben wie eine "Rechnungskorrektur", also EK auf 0,-. 
Es gilt: 
Diese besonderen Gutschriftsarten stehen nur in dem beschriebenen Kontext zur Verfügung. 

In den Gutschriftsarten wird bei Strecke die Checkbox Wertgutschrift automatisch gesetzt. Die Gutschriftsart ist in diesem Fall automatisch vom Typ Wertgutschrift . 

Die Handhabung dieser Gutschrifts- und Retourenarten sind in der Erweiterten Konditionsfindung ebenso zu handhaben! 

Tooltip: 

Kennzeichnet die Belegart als Streckenprozess 
(Lieferung von Lieferant direkt zu Kunde). 

### Checkbox Wertgutschrift 
Wenn Sie diesen Parameter aktivieren, wird aus jeder Gutschrift mit dieser Gutschriftsart eine Wertgutschrift. 

Tooltip: 

Wenn aktiviert, wird aus jeder Gutschrift mit dieser Gutschriftsart eine Wertgutschrift. 

### Combobox Auftragsart RK 
Über diese Combobox ordnen Sie eine Auftragsart zu, aus deren Rechnungsnummernkreis die Rechnungsnummer für die Fakturierung der Gutschrift gezogen wird. 

Tooltip: 

Zugeordnete Auftragsart, aus deren Rechnungsnummernkreis 
die Rechnungsnummer für die Fakturierung der Gutschrift gezogen wird. 

### Combobox Zahlung 
Hier wird die Zahlungsbedingung angegeben. Es werden Nummer und Text angezeigt. 

Der Eintrag, den Sie hier auswählen, wird bei jeder neuen Gutschrift (mit dieser Gutschriftsart) automatisch eingetragen. 

Die Zahlungsbedingung, die Sie hier auswählen, hat Vorrang vor der Zahlungsbedingung aus dem Kundenstamm. 

Tooltip: 

Wählen Sie die Zahlungsbedingung zur Gutschriftsart aus. 

### Checkbox Inklusive Mehrwertsteuer 
Diese Option hat zur Folge, dass erst ein Bruttopreis errechnet wird, der anschließend mit der Menge multipliziert wird. Die Preise auf den einzelnen Formularen sind dann Bruttopreise. 

Tooltip: 

Wenn aktiviert, ist die MwSt. in der Gutschrift enthalten. 

### Checkbox Kein Intrastat 
Wenn Sie diese Checkbox aktivieren, erfolgt keine Berücksichtigung in der Intrastat. 

Wenn die Checkbox nicht gesetzt ist, ist zwingend ein Land anzugeben. Voraussetzung: Dieses Land ist in der Ländertabelle für Intrastat freigegeben. 

Tooltip: 

Wird nicht in der Intrastat berücksichtigt, wenn Checkbox aktiv ist.
