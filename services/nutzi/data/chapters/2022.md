# Arbeitsgangübersicht 
Arbeitsgangübersicht 

Dieser Dialog öffnet sich im Betriebsauftrag >> Optionen >> Arbeitsgangübersicht . Hier erhalten Sie entweder generell eine Übersicht über Arbeitsgänge aller Betriebsaufträge oder, wenn Sie im Kopf des Dialogs Ihre Selektionskriterien eingeben, gezielte Informationen zu einem bestimmten Betriebsauftrag. 

Das Datum und der Buchungsgrund aus dem BDEJournal werden angezeigt. So ist sofort ersichtlich, wann Arbeitsgänge effektiv zu bearbeiten sind. Diese Terminierung wird im Betriebsauftrag auf der Registerkarte Terminierung festgelegt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Arbeitsgangübersicht 
Tooltip: 

In diesem Dialog erhalten Sie entweder generell eine Übersicht über Arbeitsgänge aller Betriebsaufträge oder gezielte Informationen zu einem bestimmten Betriebsauftrag. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
