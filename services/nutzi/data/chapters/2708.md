# Umlage-Detail 
Umlage-Detail 

Mit einem Doppelklick in eine Zeile oder durch Klick auf den Button im Dialog Umlagen öffnet sich der Dialog Umlage-Detail . Hier können nun die Werte der einzelnen Umlageschlüssel zentral eingegeben bzw. gepflegt werden. Darüber hinaus können Perioden-Werte angegeben werden. D.h. also, es können auch während des Jahrs Änderungen der Umlageschlüssel bzw. flexible Umlageschlüssel abgebildet werden. 

Die Jahre in den Umlage-Details entsprechen nicht den Jahren, die in der Finanzbuchhaltung angelegt wurden. Es werden lediglich die Jahre angezeigt, für die schon mal Umlagedetails erfasst wurden. Um z.B. das Jahr 2018 zur Verfügung zu haben, muss mindestens 1 Umlagedetail für das Jahr 2018 angelegt werden (manuell oder über den Vortrag -Button). Beim nächsten Öffnen des Dialogs kann anschließend das Jahr 2018 in der Combobox ausgewählt werden. 

Das Bearbeiten der Tabelle (also Zeile einfügen, ändern oder löschen) erfolgt in bekannter Weise. 

Das Kürzel wird aus dem zugrunde liegenden Dialog übernommen, kann aber auch geändert werden. Für jedes Jahr kann hier ein neuer Umlageschlüssel definiert werden, um z.B. Preissteigerungen zu berücksichtigen. Der Wert der Umlage ist manuell zu ermitteln und hier einzutragen. Zusätzlich ist es möglich eine Umlage einer Kostenstelle zuzuordnen. 

Um eine Auswahl für die Tabelle zu treffen, stehen Ihnen zusätzlich die Comboboxen Kürzel, Kostenstellennr und Jahr zur Verfügung. 

Die Summenzeile wird in der Tabelle grau hinterlegt. 

Gesperrte Objekte werden in eckigen Klammern angezeigt!

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Vortrag 1:1 

Wenn Sie nach Markieren der gewünschten Zeilen diesen Button drücken, werden die Verteilung und die Werte des Umlageschlüssels 1:1 auf das nächste Geschäftsjahr kopiert.
