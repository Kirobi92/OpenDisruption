# Weitere Offene Posten Kreditor 
Weitere OP's Kreditoren 

Dieser Dialog öffnet sich im Zahlungsvorschlag über den Button Weitere OP . Hier haben Sie die Möglichkeit über die Auswahlfelder Kontonr, Belegnummer und Gruppe weitere OPs anzuzeigen und durch markieren und den Button oder Doppelklick in den Vorschlag zu übernehmen. 

Die wichtigsten OP-Daten werden in der Tabelle angezeigt. Je nachdem aus welchem Dialog Sie den Dialog Weitere OPs Kreditoren öffnen, sehen Sie in den rechten Spalten, ob der OP in einem anderen Vorschlagsdialog enthalten ist. Z.B. Verbandsabrechung. Markieren Sie einen OP und klicken den Button OP Detail , haben Sie die Möglichkeit, Daten eines OPs zu verändern. (Siehe Beschreibung in OP Kreditoren , Button Detail ).

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
