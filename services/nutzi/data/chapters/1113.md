# Auftragsstückliste 
Auftragsstückliste 

Dieser Dialog öffnet sich mit dem Button Stückliste auf der Registerkarte Positionen im Auftrag . Hier erhalten Sie eine Übersicht über die Komponenten der Artikelposition. Dieser Dialog ist nur zugänglich, wenn es sich bei der Artikelposition um eine Handels- oder Produktionsstückliste handelt. 

Mit einem Doppelklick auf eine Artikelposition werden rechts auf der Registerkarte Komponente Informationen zum Artikel eingespielt. 

Die automatische Kalkulation einer Handelsstückliste im Auftrag erfolgt nur, wenn im Dialog Parameter Verkauf die Checkbox Autom. Kalkulation Auftragsstückliste auf der Registerkarte Allgemein gesetzt ist und im Artikelstamm (Registerkarte Verkauf ) ein Eintrag aus der Combobox Autokalk AStckl gewählt ist. 

Mehr zum Thema Stücklisten erfahren Sie im Kapitel Stücklistentypen .

### Button Drucken 

Dieser Button startet den Druck der Auftragsstückliste. 

### Button Suche 

Mit diesem Button können Sie temporäre Änderungen verwerfen. Das betrifft alle Änderungen auf jeder Ebene. 

So ersparen Sie sich das Schließen und Wieder-Öffnen des Dialogs. 

### Button Speichern 

Dieser Button speichert die Eingaben. 

### Button Neuer Satz 

Mit diesem Button können Sie eine neue Komponente erzeugen; ein neuer Eintrag wird automatisch im TreeView erzeugt und der Cursor steht sofort auf der Registerkarte Komponente im Feld Artikelnr . zur Eingabe bereit. 

Bei einem erneuten Klick auf den Button bzw. F4 wird die letzte Komponente gespeichert, falls noch nicht passiert. 

### Button Löschen 

Mit diesem Button kann ein Datensatz gelöscht werden. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Var. Einzeln
