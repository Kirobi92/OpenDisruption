# Rabattpflege Registerkarte Staffel 
Registerkarte Staffel 

Auf dieser Registerkarte werden Staffeln aufgelistet / hinzugefügt, die bearbeitet werden können. 
### Tabelle 

### Tabellenspalte Intern ID 
Hier wird eine interne, laufende Nummer angegeben. 

Tooltip: 

Interne laufende Nummer 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Menge 
Hier wird die Menge angegeben. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Rabatt % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Preis 
Ein Preis, der hier angegeben wird, wird in den Rabatt Verkauf übernommen. 

Wenn vom Sachbearbeiter keine Gültigkeit in der Staffel eingetragen wird, wird in das Gültig von -Feld der 01.01.1900 eingetragen. Diese Staffelsätze werden ebenfalls in den Rabatt Verkauf übernommen, allerdings ohne Gültigkeit. 

Tooltip: 

Preis 

### Tabellenspalte Abschlag 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Abschlag pro 
Hier wird die Menge angegeben. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Sonder VK 
Dieses Feld ermöglicht die Festlegung eines Sonderpreises für einen bestimmten Gültigkeitszeitraum. Der Sonderpreis gilt generell für alle Kunden und wird immer in der Auftragsverarbeitung verwendet, sobald er gepflegt ist. Sonderpreise können auch kundengruppenbezogen festgelegt werden. 

Hinweis: es gibt die Funktionen Sonder VK und Sonderpreis . Dies sind zwei verschiedene Lösungsansätze für den selben Zweck. 

Sonderpreis und Sonder-VK schließen weitere Rabatte aus. 

Der Sonder VK wird meistens genutzt für eine zeitliche, der Sonderpreis für eine mengenmäßige Begrenzung. 

Sie sind grundsätzlich nicht darauf ausgelegt, gemeinsam verwendet zu werden - falls doch, so gewinnt prinzipiell immer der Sonderpreis. 

Einzige Ausnahme hiervon ist die Bestpreisfindung: Wenn ein Sonder VK einen günstigeren Preis hat, wird dieser dann trotzdem gewählt. 

Die Bestpreisfindung kann im Dialog Parameter Verkauf auf der Registerkarte Allgemein aktiviert werden. 

Siehe Kapitel Parameter Verkauf . 

Davon abgesehen ist der Sonder VK nicht immer spezifischer als der Sonderpreis - tatsächlich bietet letzterer die Option, durch Angabe einer Kundengruppe spezifischer zu sein. 

Dies gilt nur für den VK 1. 

Sonderpreise werden im gleichnamigen Kapitel angelegt. Siehe Kapitel Sonderpreise . 

Tooltip: 

Angabe eines Sonderpreis
