# Sequentieller Wareneingang

Mobile Datenerfassung / Lagerprozesse 

# Sequentieller Wareneingang 
Der sequentielle Wareneingang ist ein Teil des kompletten Wareneingangs. Der Prozess beinhaltet den Weg der Ware nach der Erstannahme (Wareneingang Stufe 1) bis zur Einlagerung auf den Lagerplatz / Ladehilfsmittel. 

Hierbei wird, je nachdem ob die Checkbox Benutze Behälter für den Wareneingang im Dialog Lagerorte gesetzt ist, auf ein Ladehilfsmittel oder auf einen Lagerplatz gebucht. 

Der sequentielle Wareneingang beginnt damit, dass zuerst eine lagergeführte Bestellposition / eine Retourenposition ermittelt wird, für die bereits der Wareneingang Stufe 1 gebucht wurde. Anschließend werden weitere Informationen gesammelt: 

Die Qualität der Ware, die Menge, die Chargennummer (wenn es sich um einen Chargenartikel handelt) und zum Schluss der Lagerplatz / Ladehilfsmittel. 
### Prozessschritte 
Der sequentielle Wareneingang kann in folgende Teilprozesse unterteilt werden: 

- Auswahl der Wareneingangsposition - Auswahl der Qualität - Auswahl der Menge - Auswahl der Charge - Auswahl des Lagerplatzes / Ladehilfmittels 

### Auswahl der Wareneingangsposition 
Mit dem Dialog Sequentieller Wareneingang wird eine Bestellposition ermittelt, die der Sachbearbeiter einlagern möchte. Hierzu muss eine eindeutige Bestellposition ausgewählt werden, für die der Wareneingang Stufe 1 bereits gebucht wurde. Diese kann der Sachbearbeiter über folgende Kriterien filtern und ermitteln: 
- Bestellnummer/ Retourennr - Bestellnummer + Positionsnummer/ Retourennr + Positionsnr - Artikelnummer - Wareneingangsnummer 

### Auswahl der Qualität 
Im nächsten Schritt wird vom Sachbearbeiter, im Dialog Seq. Wareneingang: Qualität die Qualität der Ware beurteilt und eingetragen. Hier kann später in verschiedene Qualitätslager eingelagert werden, sofern diese im Dialog Lagerorte definiert sind. 
### Auswahl der Menge 
Im Dialog Seq. Wareneingang: Menge wird nun vom Sachbearbeiter die Menge erwartet, die er einlagern möchte. Hierbei darf die Menge nicht größer sein, als die Menge die bei der Stufe 1 Buchung angegeben wurde. 
### Auswahl der Charge 
Wenn es sich um einen Chargenartikel handelt, muss nun eine Chargennummer eingegeben werden. Die Eingabe erfolgt über den Dialog Seq. Wareneingang: Charge 
### Auswahl des Lagerplatzes / Ladehilfsmittel 
Zum Schluss wird nun der Lagerplatz / das Ladehilfsmittel ausgewählt auf dem eingelagert werden soll. Im Dialog Seq. Wareneingang: Lagerplatz werden, sofern auf einen Lagerplatz eingelagert werden soll, Lagerplätze gesucht. Diese werden mit folgender Priorisierung ermittelt: 

- Erstlagerplatz für diesen Artikel - Zweitlagerplatz für diesen Artikel - Lagerplatz mit Bestand für diesen Artikel Siehe auch: 

- Sequentieller Wareneingang - Seq. Wareneingang: Lagerplatz - Seq. Wareneingang: Menge - Seq. Wareneingang: Qualität - Seq. Wareneingang: Charge
