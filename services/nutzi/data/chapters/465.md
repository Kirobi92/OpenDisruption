# Import Items erzeugen Registerkarte Tabellen 
Basis Dialog Dokumente Registerkarte Tabelle 

Diese Registerkarte enthält eine Übersicht aller bisher ausgewählten Tabellen, von welchen bereits Spalten übernommen wurden. Über den Button auf der rechten Seite kann eine Einschränkung der Daten vorgenommen werden. Hierfür öffnet sich beim Klick auf diesen Button der Dialog Suchbedingung – hier kann analog der Suche mit SQL die Bedingung festgelegt werden, nach welcher die CSV-Dateien für die Tabelle gefüllt werden. 

Wird eine Tabelle markiert, werden auf der Registerkarte Spalten nur die Spalten der gewählten Tabelle angezeigt.
