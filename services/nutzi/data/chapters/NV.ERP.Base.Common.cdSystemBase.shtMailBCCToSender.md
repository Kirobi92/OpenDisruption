# Kapitel NV.ERP.Base.Common.cdSystemBase.shtMailBCCToSender

Wenn Sie diesen Parameter aktivieren, ist es möglich den Benutzer beim Email-Versand (Dialog E-Mail senden ) über das Feld BCC in Kopie zu nehmen. Dies erfolgt dann immer, wenn der Versand von Emails über SMTP erfolgt und dies nicht im Dialog Parameter Sachbearbeiter übersteuert wird. Damit ist eine Dokumentation des erfolgreichen Versandes feststellbar (bei einem Versand über das lokale Outlook wäre dies über den Ordner "gesendete Objekte" möglich). 

Siehe Kapitel Parameter Sachbearbeiter . 

Tooltip: 

Per Parameter kann jede E-Mail an den „Versender“ per BCC geschickt werden, 
damit der Versender auch über den Versand informiert wird.
