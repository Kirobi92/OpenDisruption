# Wiederkehrende Buchung anlegen / verwalten Registerkarte Gegenbuchungen 
WdkB Verwalten Registerkarte Gegenbuchungen 

Auf dieser Registerkarte legen Sie Kontonummer, Betrag, Kostenstelle, Kostenträger und Steuerschlüssel für die Gegenbuchung(en) fest. 

Wurden mehrere Gegenbuchungen erfasst, muss die Korrektur-Buchungszeile erfasst werden, bevor der Button betätigt wird. 
### Button Neue Zeile 

Mit diesem Button legen Sie eine neue Buchung (Gegenbuchung) an. 

### Button Zeile lösch. 

Durch markieren einer Zeile in der Tabelle und betätigen des Buttons Zeile lösch. kann eine Gegenbuchung gelöscht werden.
