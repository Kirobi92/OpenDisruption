# Kapitel NV.ERP.Base.Customer.cdCustomer.shtNoInvPrintAtShip

Wenn Sie diese Checkbox aktivieren, steuern Sie dass im Versand- bzw. Logistikmodul mit der Freigabe keine Rechnungen für diesen Kunden gedruckt werden. Ist diese Einstellung im Kommissionieren aus dem Kundenstamm vorbelegt, wird die Checkbox im Kommissionieren deaktiviert und kann auch nicht manuell gesetzt werden. 

Dieser Parameter ist nur relevant, wenn im Versand der Parameter Rechnung nach Freigabe drucken gesetzt ist. 

Siehe Kapitel Parameter VS / LO . 

Tooltip: 

Kein Rechnungs-Druck im Versand
