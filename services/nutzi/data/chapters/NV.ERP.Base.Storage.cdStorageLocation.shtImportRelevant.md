# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtImportRelevant

Dieses Kennzeichen legt fest, ob das Lager für zollrelevante Geschäftsvorfälle im Importbereich wichtig ist. Die Funktion wird nicht unterstützt bzw. ist nur zur Info. 

Tooltip: 

Kennzeichnet das Lager für zollrelevante Geschäftsvorfälle beim Import.
