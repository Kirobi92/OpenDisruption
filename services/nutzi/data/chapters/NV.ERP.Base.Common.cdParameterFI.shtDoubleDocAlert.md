# Kapitel NV.ERP.Base.Common.cdParameterFI.shtDoubleDocAlert

Hier stehen Ihnen die Einträge keine Warnung , alle OPs und nur offene OPs zur Verfügung. So können Sie die Doppelbelegwarnung für Personenkonten auch für bereits gezahlte Rechnungen aktivieren oder sie ganz ausschalten. 

Die Prüfung bezieht sich auf Belegnummer / Belegnummer 2 je Debitor bzw. Kreditor.
