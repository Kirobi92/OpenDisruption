# Kapitel NV.ERP.Base.Storage.cdStorageLocation.dtNV4_TimePrecommission

Hier geben Sie eine Uhrzeit ein, zu der die Vorkommissionierung auf diesem Lager startet. Alles was bis zu diesem Zeitpunkt nicht gepickt ist, wird erst in die nächste Tour eingeplant. Diese Funktion steht für das 2-stufige LVS zur Verfügung.
