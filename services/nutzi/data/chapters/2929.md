# Datenexport Adapter 
Datenexport Adapter 

Wenn Sie im Gateway Adapter Kontext die Zeile mit der Adapter Nr. Datenexport markiert haben und den Button drücken, öffnet sich die Maske Datenexport Adapter . In diesem Dialog definieren Sie die Export-Jobs. 

Dieser Adapter nutzt keine Ereignisse. 

Im Menü Datenexport Adapter legen Sie zum Adapter die Komponente fest und definieren, ob die Übertragung per Server oder Client erfolgen soll. 

Sie können beliebig viele Export-Jobs über einen Adapter-Kontext definieren. Im Gateway wird der Job wie alle anderen über den Job-Server ausgeführt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Datenexport Adapter 
Tooltip: 

In diesem Dialog definieren Sie die Export-Jobs. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Combobox Typ 
Hier wählen Sie den Typ aus. 

Die verfügbaren Typen im Gateway-Adapter für Datenimport/-export sind fest im System vorgegeben und können ggf. durch SQL-Script ergänzt werden. 

Die im Standard enthaltenen Typen sind: 

'NV.ERP.Base.Article.cdArticleColl' 
'NV.ERP.Tools.ArticleConditionImport.cdDIArticleColl' 
'NV.ERP.Tools.ArticleConditionImport.cdDIArticlePriceGroupColl' 
'NV.ERP.Tools.ArticleConditionImport.cdDIArticleUnitsColl' 
'NV.ERP.Tools.ArticleConditionImport.Import.cdDIVersionColl' 
'NV.ERP.FI.FinancialAcc.OpenItem.cOpenItemCustomerColl' 
'NV.ERP.FI.FinancialAcc.OpenItem.cOpenItemSupplierColl' 
'NV.ERP.Base.Common.Interfaces.PurchaseOrderExport.cPurchaseOrderExportColl' 

Beispiel: 

INSERT INTO Gw_DataExportComponents (BranchKey, ComponentName, ComponentType) 
VALUES (<BK>, 'Bestell Export', 'NV.ERP.Base.Common.Interfaces.PurchaseOrderExport.cPurchaseOrderExportColl');

### Combobox Export Format 
Hier wählen Sie das Format für den Export aus. 

### Button Öffnen 

Dieser Button öffnet den Dialog Datenexport . Beim Export-Format können Sie zwischen CSV und ASII wählen. Bei CSV werden die einzelnen Datenfelder durch ein Separator-Zeichen getrennt, bei ASCII werden die Felder mit einer festen Länge exportiert.
