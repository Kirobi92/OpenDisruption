# Aufgabenübersicht 
Aufgaben-Übersicht 

In der Projektübersicht auf der Registerkarte Allgemein gibt es den Button Aufgaben . Wenn Sie diesen betätigen, öffnet sich der Dialog Aufgaben-Übersicht . In diesem Dialog werden Aufgaben angezeigt, die einem Projekt zugeordnet sind. 

Sie können diesen Dialog auch komfortabel über den Navigator , Menüpunkt Projekt aufrufen. 

Ohne einen Eintrag in der Tabelle der Projektübersicht markiert zu haben, werden im Dialog Aufgaben-Übersicht sämtliche Projekt-Aufgaben aufgelistet. Wenn Sie ein Projekt markieren und den Button Aufgaben drücken, werden nur Aufgaben zu diesem Projekt angezeigt. 

Dieses Menü lässt sich auch im Projekt über das Menü Optionen öffnen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Aufgabenübersicht 
Tooltip: 

In diesem Dialog können Sie sich zu einem Projekt die zugeordneten Aufgaben anzeigen lassen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Drucken 

Mit diesem Button starten Sie den Druck, um die Informationen als Liste (in Querformat) zu drucken. Die Selektionskriterien werden übergeben. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Wenn Sie eine Aufgabe markieren und diesen Button drücken, gelangen Sie im Dialog Aufgabe ändern und können die Aufgabe gemäß Ihren Wünschen bearbeiten. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Button SQL-Suche 

Mit diesem Button können Sie eine Suche per SQL starten. 

Siehe Kapitel Suchbedingung . 

### Button Einstellungen 

Hier definieren Sie Einstellungen für die Arbeit mit einem bestimmten Programmteil.

Die Klassifizierung wird auch entsprechend in der Aufgaben-Übersicht angezeigt und als Selektionskriterium zur Verfügung gestellt.

### Tabellenspalte Ist-Zeit (hh,mm) 
Hier kann die tatsächlich benötigte Zeit manuell eingetragen werden oder sie wird aufgrund der Leistungserfassung automatisch addiert und hier eingetragen. 

Bei jeder Leistungserfassung auf die Projektaufgabe wird die Ist-Zeit entsprechend erhöht. Die Erhöhung der Ist-Zeit erfolgt nur bei Leistungsarten mit dem Leistungstyp "Projektmanagement" um z.B. Fahrzeiten / Fortbildung anderer Kollegen nicht zu berücksichtigen.
