# Stücklisten Registerkarte Komponente 2 
Stückliste Registerkarte Komponente 2 

Auf dieser Registerkarte werden weitere Informationen angegeben. 
### Feld Änderung 
Hier wird vom Programm automatisch das Datum der letzten Änderung eingetragen. 

Tooltip: 

Datum der letzten Änderung 

### Feld Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Code 1 
Dieses Feld steht zur individuellen Eingabe zur Verfügung. Es kann per Reportprogrammierung eingebunden werden, hat aber keine Funktionalität.
