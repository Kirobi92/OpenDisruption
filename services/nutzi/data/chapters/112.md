# Definieren von Zusatzfeldern Registerkarte Allgemein 
Zusatzfelder definieren Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die Details für Zusatzfelder. 

Alle zusätzlichen Felder werden im zugrundeliegenden Dialog auf einer eigenen Registerkarte angezeigt. 
### Feld Feldname 
Hier geben Sie an, wie das Zusatzfeld heißen soll. Beispiel: Belegfeld 2, Zusatzfeld 2. 

Tooltip: 

Angabe des Feldnamens 

### Combobox Datentyp 
Hier wählen Sie aus, welche Art von Wert das Feld haben kann; eine Zahl oder einen Text. Zur Verfügung stehen: 

- Datum/Zeit - Zeichenkette - Ganzzahl - Gleitkommazahl 

Tooltip: 

Angabe des Datentyps 

### Feld Reg. Ausdruck 
Hier geben Sie die zulässigen Werte an, z.B. 4 Zeichen bei einer Zeichenkette, aber keine Sonderzeichen. 

Tooltip: 

Regulärer Ausdruck 

### Button Öffnen 

Dieser Button öffnet den Dialog Regulärer Ausdruck , um Einzelheiten für die Zusatzfelder zu definieren, wie z.B. ob die Felder auch Buchstaben enthalten dürfen. 

Siehe Kapitel Regulärer Ausdruck . 

### Feld Formatierung 
Hier definieren Sie, wie die eingegebenen Werte dargestellt werden. Z.B. ##,0## für Werte mit mind. 1 Nachkommastelle oder ##,000 immer mit 3 Nachkommastellen. 

Tooltip: 

(Optional) Nur für Ganzzahl, Kommazahl, Datum als Formatierungs-Ausdruck. 
Der Ausdruck wird bei der String-Umwandlung des Werts als Formatierungsangabe genutzt. 
Beispiele: 

Zahlen 
"#,##0.00" -> "1.000,00" 
"€ #,##0.00" -> "€ 1.000,00" 
"#.000 mm" -> "10000.000 mm" 
Datum: 
"yyyy.MM.dd" -> "2019.01.31" 

### Feld Beschreibung 
Hier geben Sie eine erweiterte Beschreibung an, welche Felder im Zusatzfeld gepflegt werden sollen. 

Tooltip: 

Angabe einer Beschreibung
