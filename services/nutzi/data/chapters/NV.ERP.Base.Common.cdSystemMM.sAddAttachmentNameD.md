# Kapitel NV.ERP.Base.Common.cdSystemMM.sAddAttachmentNameD

Hier geben Sie die Formel zur Ermittlung des Dateinamens ein bei der Generierung einer E-Mail für den Druck einer Belastung. Es wird immer fix der Wert "Belastung" verwendet, gefolgt von der Formel als Dateiname.
