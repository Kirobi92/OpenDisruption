# Sachkontenstamm Registerkarte Tabelle 
Sachkonto Registerkarte Tabelle 

Auf dieser Registerkarte werden alle vorhandenen Sachkonten angezeigt. 

Siehe Kapitel Allgemeine Informationen in eNVenta ERP / Registerkarte Tabelle . 
### Tabelle 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Tabellenspalte Steuerart 
Hier wird die Steuerart angegeben (V = Vorsteuer, M = Umsatzsteuer). 

Tooltip: 

Angabe der Steuerart 
V = Vorsteuer, M = Mehrwertsteuer (Umsatzsteuer) 

### Tabellenspalte Steuerschlüssel 
Hier wird der Steuerschlüssel angegeben. 

Tooltip: 

Angabe des Steuerschlüssels 

### Tabellenspalte Bilanz / G+V 
Hier wird die Zuordnung des Sachkontos zu Bilanz oder G&V angegeben. Gültige Werte sind: 
B - Bilanzkonto 
G - G&V - Konto 
Nur gültig, wenn Feld Typ = 'S' (Sachkonto) 

Wenn B, kann das Konto auch OP-führend (Opfuehrend=1) sein. 

Tooltip: 

Sachkonto für Bilanz oder G&V 

### Tabellenspalte Kontogruppe 
Hier wird eine Kontogruppe angegeben. Diese werden im Dialog Kontogruppen verwaltet. 

Siehe Kapitel Kontogruppen . 

Tooltip: 

Angabe der Kontogruppe 

### Tabellenspalte Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind! 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Tabellenspalte Geldkonto 
Wenn die Checkbox in dieser Spalte gesetzt ist, handelt es sich um ein Geldkonto (Bank/Kasse). Gültige Werte sind: 
0 - kein Geldkonto 
1 - Geldkonto 

Tooltip: 

Kennzeichnung als Geldkonto 

### Tabellenspalte Kasse 
Wenn gesetzt, handelt es sich um ein Kassenkonto (Dadurch kein Haben-Saldo möglich). Gültige Werte:0 - kein Kassenkonto 1 - Kassenkonto Kann nur gesetzt sein, wenn Feld Geldkonto ebenfalls gesetzt ist. 

### Tabellenspalte Kostenrechnung 
Wenn die Checkbox in dieser Spalte gesetzt ist, ist die Kostenrechnung für das Konto aktiviert. Gültige Werte: 
0 - Keine Kostenrechnung 
1 - Kostenrechnung. 

Nur wenn dieses Kennzeichen gesetzt ist, können auch die Felder Kstzwang und Ktrzwang gesetzt sein. 

Tooltip: 

Kostenrechnung zum Konto 

### Tabellenspalte Kostenstellenzwang 
Wenn die Checkbox in dieser Spalte gesetzt, ist Kostenstellenzwang für das Konto aktiviert. Gültige Werte: 
0 - Kein Kostenstellenzwang 
1 - Kostenstellenzwang 

Nur wenn das Kennzeichen Kostenrechnung gesetzt ist, kann dieses Kennzeichen gesetzt sein. 

Tooltip: 

Aktiviert Kostenstellenzwang für das Konto 

### Tabellenspalte Kostenart 
Hier wird eine Kosten- oder Leistungsartennummer für die Kostenrechnung angegeben. 

Tooltip: 

Angabe der Kosten- oder Leistungsartennummer 

### Tabellenspalte Kostenträgerzwang 
Wenn die Checkbox in dieser Spalte gesetzt ist, ist der Kostenträgerzwang für das Konto aktiviert. Gültige Werte: 
0 - Kein Kostenträgerzwang 
1 - Kostenträgerzwang 

Nur wenn das Kennzeichen Kostenrechnung gesetzt ist, kann dieses Kennzeichen gesetzt sein. 

Tooltip: 

Aktiviert Kostenträgerzwang für das Konto 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
