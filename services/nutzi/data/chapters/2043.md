# Simulation 
Simulation 

In diesem Dialog kann ein Produktionsauftrag simuliert werden. Die Simulation kann pro Auftragsposition erfolgen, vorausgesetzt es handelt sich dabei um einen Stücklistenartikel. Wie im Produktionsauftrag stehen auch hier die Möglichkeiten zur Terminierung, Kalkulation etc. zur Verfügung. 

Wird für eine Simulation eine Terminierung ausgelöst, so wechselt die Anzeige automatisch auf die Registerkarte Kalkulation und eine Aktualisierung wird ausgelöst. 

Wenn ein Artikel Teil einer Simulation im PPS ist, kann trotzdem die Eigenschaft "Charge" im Artikelstamm geändert werden. 

Wenn eine Simulation in einen Betriebsauftrag überführt wird, so erfolgt eine erneute Überprüfung der Chargen-Kennzeichnung im Betriebsauftrag.

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Referenz-Link 

Dieser Button öffnet den Dialog Referenz Links . 

Referenz Links bieten die Möglichkeit Dokumenten-Links bzw. URLs als Info zu hinterlegen, um 2 Vorgänge / Referenzen miteinander zu verknüpfen. Dies erspart Ihnen lange Archiv-Recherchen.

### Button Drucken 

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.

### Feld Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Erfassung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der Erfassung eingetragen. 

Tooltip: 

Datum und Uhrzeit der Erfassung 

### Baumansicht Übersicht Betriebsaufträge 

In diesem Feld erhalten Sie eine Übersicht aller Betriebsaufträge, welche im simulierten Produktionsauftrag enthalten sind. Mit einem Doppelklick auf den Betriebsauftrag kann dieser geöffnet werden. 

Im Betriebsauftrag selbst stehen alle Möglichkeiten zur Verfügung, analog zu einem „produktiven“ Betriebsauftrag.
