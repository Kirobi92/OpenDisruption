# Katalogauswahl 
Katalogauswahl 

Dieser Dialog öffnet sich im BMECat Katalogexport über den Button neben den Feldern Gruppe / Katalog im Kopf des Dialogs. Hier können Sie einen Katalog auswählen oder eigene Kataloge und Gruppen zusammenstellen. 

Um einen eigenen Katalog oder eigene Gruppen zusammenzustellen, gehen Sie folgendermaßen vor: 

Geben Sie im obersten Bereich des Dialogs eine Gruppe oder eine Bezeichnung zur Selektion ein. 

Wenn Sie jetzt auf den Button klicken, wird die obere Tabelle gefüllt. Hier markieren Sie den gewünschten Katalog. In der Tabelle darunter wird der Katalog geöffnet und seine Gruppen werden angezeigt. 

Sie haben jetzt die Möglichkeit einen Katalog selbst aus mehreren Katalogen und Gruppen für den Export zusammenzustellen. Die Gruppen, die in einen Katalog sollen, werden in der Tabelle Gewählte Kataloggruppen zusammengestellt. Mit dem Button oder Drag&Drop werden sie in diese Tabelle eingefügt. 

Mit dem Button neben der Tabelle lässt sich ein Eintrag wieder entfernen. 

Das System vergibt dann einen automatisch generierten Namen. Wenn so ein Katalog importiert wird, heißt dieser standardmäßig "Custom Multi Group". Der Name kann allerdings in der Katalogverwaltung neu vergeben werden. 

Der Export selbst funktioniert wie gehabt. Es werden weiterhin Bilder, Dokumente, Merkmale etc. normal exportiert.
