# Kapitel NV.ERP.FI.FinancialAcc.Posting.cdDiffMessageProposalParam.shtPooled

Wenn Sie diesen Parameter aktivieren, werden alle Differenzen je Konto in einem Schreiben zusammengefasst. 

Tooltip:
