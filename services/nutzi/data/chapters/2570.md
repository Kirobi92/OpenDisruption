# Bankeinzug Registerkarte Disposition

Finanzbuchhaltung / Bankeinzug 

# Bankeinzug Registerkarte Disposition 
Bankeinzug Registerkarte Disposition 

Auf dieser Registerkarte wählen Sie zunächst eine oder mehrere Hausbanken aus, über die der Bankeinzug erfolgen soll. 

Die ausgewählte(n) Bank(en) erscheinen in der Tabelle auf der Registerkarte Disposition . Hier haben Sie die Möglichkeit, durch Eintrag in der Spalte ‚Limit’, eine Höchstgrenze für den gesamt zu zahlenden Betrag pro Bank festzulegen. 
### Button Neue Zeile 

Mit diesem Button öffnet sich der Dialog Bankverbindungen , um die gewünschte Bank auszuwählen. 

Siehe Kapitel Bankverbindung . 

Tooltip: 

Neue Zeile einfügen 

### Button Lösche Zeile 

Mit diesem Button können Sie eine bereits gewählte Hausbank wieder entfernen. 

Tooltip: 

Zeilen löschen 

### Button Zahlweg 

Wird über eine inländische oder ausländische Bank ein Einzugsvorschlag erstellt, werden nach Betätigen des Buttons Zahlweg die entsprechenden Zahlweg-Einstellungen, die bei dieser Bank hinterlegt sind, angezeigt. 

Tooltip: 

Verwalten der Zahlwege 

### Button Zuordnen 

Mit diesem Button können Sie nach der erfolgreichen Generierung eines Einzugsvorschlags einen/mehrere offene Posten einer bestimmten Hausbank zuordnen (manuelle Zuteilung). Hierzu markieren Sie den/die gewünschten OPs auf der Registerkarte Vorschlag . Danach wählen Sie die neue Dispositionsbank auf der Registerkarte Disposition aus, markieren diese in der Tabelle und betätigen den Button Zuordnen . 

Tooltip: 

Selektierte Vorschlagssätze einer Disposition zuordnen. 

### Tabelle 

Dieses Feld weist die Summe des Zahlbetrags des gesamten Vorschlags in Landeswährung aus, egal welche Währung das Personenkonto oder das Dispositionskonto hat. 

Dieses Feld gibt einen schnellen Überblick, in welcher Höhe die Liquidität beeinflusst wird. 

Sätze mit der Information „OP nicht verrechenbar“ sind nicht eingerechnet. 
### Tabellenspalte Bankname 
Hier wird der Bankname angegeben. 

Zur Information: Mit einem Doppelklick auf einen Eintrag in dieser Spalte erreichen Sie den Dialog Bankverbindung . Und im Dialog Bankverbindung öffnet der Button SEPA Daten den Dialog SEPA Daten . 

### Tabellenspalte BLZ / SWIFT 
Hier geben Sie den Bank-Code an. Dies ist eine Pflichtangbe. Gültige Werte sind abhängig vom Eintrag im Feld Typ . Entweder 
Typ = 'BLZ', dann Bankleitzahl der Bank 
Typ = 'SWIFT', dann Swiftcode der Bank.

### Tabellenspalte Kreditlinie 
Hier kann ein Standard-Kreditlimit für den Kunden (in System-Währung) festgelegt werden. Falls der Betrag überschritten wird, erhalten Sie eine Meldung, wenn Sie einen Auftrag zu diesem Kunden erfassen. Sie erhalten die Möglichkeit, eine Begründung für die Überschreitung einzugeben. 

Die absoluten Kopfrabatte werden in der Kreditlimitprüfung nicht berücksichtigt. 

Das Kreditlimit ist immer in "Haus-Währung" anzugeben! 

Tooltip: 

Angabe des gewährten Kreditlimits
