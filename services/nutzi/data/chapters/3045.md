# Status-Information Registerkarte Trace 
Status Information Registerkarte Trace 

Auf dieser Registerkarte werden die Lade-Bedingungen angegeben, die abgesetzt sind. 

Diese Registerkarte muss zum Tracen geöffnet sein. Daten werden nur zur Laufzeit angegeben; es erfolgt keine Speicherung. 
### Tabelle 

### Tabellenspalte Event 
Hier wird das Event angegeben. 

### Tabellenspalte Objekt-Typ 
Hier wird der Objekt-Typ angegeben. 

### Tabellenspalte Info 
Hier wird die Ladebedingung angegeben.
