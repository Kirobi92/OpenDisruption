# Zahlungsparameter 
Zahlungsparameter 

Dieser Dialog öffnet sich im Navigator >> System >> Finanzbuchhaltung >> Zahlungsparameter . Hier hinterlegen Sie die grundsätzlichen Selektionskriterien für den Zahlungsverkehr im Bereich ‚Kreditoren‘. Anhand der Fälligkeit der offenen Posten und der hier hinterlegten Kriterien werden diese dann zur Zahlung im Zahlungsverkehr vorgeschlagen. 

Diese Einstellungen gelten für alle Kreditoren.

### Combobox Zahlbed. Teilz. 
Ebenso wie in den Einzugsparametern kann für die Anlage eines OPs aus einer Teilzahlung eine Zahlungsbedingung hinterlegt werden. 

Ist hier eine Zahlungsbedingung hinterlegt, wird für die OP-Anlage diese Zahlungsbedingung aus der Teilzahlung genutzt. 

Ist hier keine Zahlungsbedingung hinterlegt, wird der Sachbearbeiter mit einer Meldung darauf hingewiesen, wenn er eine Teilzahlung durchführt. Wird diese Meldung ignoriert, wird für die OP-Anlage die Zahlungsbedingung aus dem Lieferantenstamm benutzt. 

Die Teilzahlung wird immer auf dem Ursprungs-OP verdichtet in den OP-Zusatz-Infos „Bereits bezahlt, Skto. gebucht, Ausb. gebucht, Noch offen“. In manchen Vorschlagsdialogen werden diese OP-Zusatz-Infos auch noch weiter zusammengefasst auf Erforderlich und Noch offen. In den Vorschlagsdialogen ist die Zahlungsbedingung des Ursprungs-OP ausschlaggebend. 

Sie erhalten einen Hinweis, wenn Sie die Zahlungsbedingung zu Teilzahlungen ändern. 

Tooltip: 

Für die Anlage eines OPs aus einer Teilzahlung kann eine Zahlungsbedingung hinterlegt werden.

### Feld Druck Avis ab 
Mit einem Eintrag in diesem Feld (Anzahl der OPs) können für Bankeinzug und Zahlungsverkehr die Einstellungen aus den Parametern übersteuert werden. Auf diese Weise kann definiert werden, ob und ab welcher Position ein Avis gedruckt werden soll. 

Das Avis wird innerhalb einer Kontonummer gruppiert nach Zahlungsverbindung gedruckt. 

D.h. hier kann festgelegt werden, ab welcher Anzahl offener Posten ein Zahlungsavis beim automatischen Zahlungsverkehr gedruckt wird. 

Wurde in diesem Feld kein Eintrag vorgenommen, ermittelt eNVenta ERP anhand der Länge des Verwendungszwecks automatisch den Avis-Druck. 

Tooltip: 

Hier kann definiert werden, ob und ab welcher Position ein Avis gedruckt werden soll. 
Einstellungen aus den Parametern werden übersteuert. 

### Feld Fällig bis Tage 
Über dieses Feld können Sie einen Zeitraum festlegen, damit automatisch auch die Rechnungen, die erst in diesem Zeitraum zur Zahlung fällig werden, im Zahlvorschlag erscheinen. 

Der Zeitraum ermittelt sich anhand Ihrer vorgegebenen Fälligkeit im Zahlungsverkehr zuzüglich der Tage, die in den Zahlungsparametern eingetragen sind. 

Tooltip: 

Ausgehend vom Tagesdatum erhöhen/verringern die hier erfassten Tage das Datum im Vorschlagsdialog.

### Checkbox Eingabe des Zahlbetrags 
Wenn Sie diesen Parameter aktivieren, wird der Zahlbetrag eingegeben. 

- Vollausgleich : Zahlbetrag = OP-Betrag abzüglich Skonto - Teilzahlung : Zahlbetrag zuzügliche Skonto = neuer OP 

Tooltip: 

Eingabe des Zahlbetrags: 
- Vollausgleich: Zahlbetrag = OP-Betrag abzüglich Skonto 
- Teilzahlung: Zahlbetrag zuzüglich Skonto = Neuer OP 

### Checkbox Gutschrift sofort fällig 
Ist diese Checkbox gesetzt, sind Gutschriften immer sofort fällig und werden im Zahlungsverkehr sofort berücksichtigt, unabhängig von der Zahlungsbedingung (Fälligkeit). 

Tooltip:
