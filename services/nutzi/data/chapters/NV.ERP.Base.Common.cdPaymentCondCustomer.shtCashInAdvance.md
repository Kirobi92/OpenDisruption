# Kapitel NV.ERP.Base.Common.cdPaymentCondCustomer.shtCashInAdvance

Hier können für die Zahlungsbedingung Vorauskasse Einstellungen definiert werden. 

Für einen Kunden, für den die Zahlungsbedingung Vorauskasse gesetzt ist, muss im Auftrag manuell eine Zahlungsnummer generiert werden. 

Mehr zum Thema Zahlungsnummer erfahren Sie im Kapitel Auftrag Registerkarte Kopfdaten / Feld Zahlungsnr . 

Siehe Kapitel Auftrag Registerkarte Kopfdaten . 

Tooltip: 

Hier definieren Sie Einstellungen für die Zahlungsbedingung 'Vorkasse '.
