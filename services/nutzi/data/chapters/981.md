# Parameter Verkauf Registerkarte Gutschriftsabrechnung 
Parameter Verkauf Registerkarte Gutschriftsabrechnung 

Auf dieser Registerkarte nehmen Sie die Einstellungen für die Gutschriftsabrechnung vor. 
### Combobox Ausgleichslager 
Hier geben Sie das Ausgleichslager für das Gutschriftsabrechnungsverfahren ein. Neue Positionen der Gutschrift werden automatisch auf dieses virtuelle Lager erfasst, damit die Bestände korrekt abgebucht werden. 

Tooltip: 

Ausgleichslager für das Gutschriftsverfahren 

### Combobox Auftragsart 
Hier geben Sie die Auftragsart an für die Generierung eines Gegen-Auftrags aus der Gutschriftsabrechnung heraus. 

Damit Auftragsarten hier zur Auswahl zur Verfügung stehen, müssen folgende Voraussetzung erfüllt zu sein: 

- Die Auftragsart des Gegenauftrags darf die Einstellung Lagerbuchung nicht aktiviert haben. - Die Fakturafreigabe muss aktiviert sein. - Aufträge mit einer Auftragsart 'ohne Fakturierung' sind vom Gutschriftsverfahren ausgeschlossen. 

Tooltip: 

Auftragsart für die Gegen-Rechnungen im Gutschriftsverfahren 

### Combobox Rechnung intern 
Hier geben Sie den Report für die interne Rechnung an, welche bei der Faktura der Gutschriftsabrechnung generiert wird. Diese muss abweichend von der offiziellen Rechnung sein, da es sich um einen internen Beleg handelt. 

Tooltip: 

Reportformular für den internen Rechnungsdruck bei der Fakturierung von Gutschriftsanzeigen.
