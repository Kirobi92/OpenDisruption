# Lieferbedingung 
Lieferbedingung übernehmen 
Soll die ermittelte / angegebene Lieferbedingung übernommen werden? 

Ermittlung Lieferbedingung (inaktiv) 
Wie soll die entsprechende Lieferbedingung im System identifiziert werden? 

LB-Nr. 
Anhand der eigenen Lieferbedingung-ID eine entsprechende Lieferbedingung im System suchen. 

LB-Text 
Anhand des Lieferbedingungtextes eine entsprechende Lieferbedingung im System suchen. 

vom Kundenstamm 
Die definierte Lieferbedingung des Kunden verwenden. 

vom Lieferantenstamm 
Die definierte Lieferbedingung des Lieferanten verwenden.
