# Rechnungsprüfung Registerkarte Kopfdaten 
Rechnungsprüfung Registerkarte Kopfdaten 

Auf dieser Registerkarte werden weitere Informationen angegeben. 
### Feld Bankverbindung 
Die Nummer der Bankverbindung wird aus dem Lieferantenstamm voreingestellt, kann aber geändert werden. 

Tooltip: 

Bankverbindung aus dem Lieferantenstamm 

### Feld Bankname 
Hier wird der Name der Bank angegeben. 

Tooltip: 

Angabe des Banknamens 

### Button Suche 

Mit diesem Button kann eine Bankverbindung gesucht werden. 

Siehe Kapitel Bankverbindung Lieferant . 

### Checkbox Keine Habenstellung 
Wenn Sie diese Checkbox aktivieren, landen die Daten nicht zur Übernahme im Belegjournal der Finanzbuchhaltung, sondern werden in der Datenbank-Tabelle Sonderjournal gespeichert. Diese Tabelle dient nur zur Dokumentation der Kreditoren-Rechnung. 

Tooltip: 

Keine Habenstellung 

### Checkbox Keine Zentralregulierung 
Diese Checkbox ist nur sichtbar, wenn es sich bei dem Lieferanten um einen Verbandslieferanten handelt. 

Das Kennzeichen Keine ZR hat folgende Wirkung: 

- Der OP wird im Zahlungslauf für den Verbandslieferanten berücksichtigt. - Der OP wird von der Verbandsabrechnung ausgeschlossen. 

Tooltip: 

Keine Zentralregulierung (Verbandsabrechnung bzw. Zentralzahlung) 

### Feld Industrievertreter 
Hier wird die Nummer des Industrievertreters angegeben. 

Ein Industrievertreter entspricht ungefähr einem Vertreter aus dem Verkauf. 

Tooltip: 

Angabe der Lieferantennummer 

### Button Suche 

Mit diesem Button kann nach einem Industrievertreter gesucht werden. 

Siehe Kapitel Industrievertretung . 

### Checkbox OP auf Industrie-Vertreter 
Wenn diese Checkbox gesetzt wird, wird der zu erzeugende OP (Fibu) auf den Industrie-Vertreter gebucht. 

Bei der Übergabe an die Finanzbuchhaltung wird die aktuellste Bankverbindung des Industrievertreters übergeben. 

Tooltip: 

Offener Posten wird auf den Industrie-Vertreter erzeugt. 

### Feld Buchungsdatum 
Hier wird das Buchungsdatum angegeben. 

Tooltip: 

Angabe des Buchungsdatums 

### Feld R.-Beleg-Typ 
In diesem Feld wird angezeigt, um welchen Rechnungstypen einer EDI-Rechnung es sich handelt. 

Bei diesem Rechnungstyp handelt es sich um den Rechnungstyp INVOIC aus der eingelesenen EDI-Datei. 

Der Rechnungstyp wird in der EDI Datei INVOIC im Satz HD51 im Feld Beleg-Typ an Stelle 5 - 7 mit einem Code geliefert. Aufgrund dieses Codes wird der Rechnungstyp (wie folgt) visualisiert: 

Rechnungstyp: 

- 380 Original Handelsrechnung - 385 Original Sammelrechnung - 480 Kopie Handelsrechnung - 485 Kopie Sammelrechnung - 580 Duplikat Handelsrechnung - 585 Duplikat Sammelrechnung - 680 Ersatz Handelsrechnung - 685 Ersatz Sammelrechnung 
Gutschriftstyp: 

- 83 Original Wertgutschrift - 84 Wertbelastung - 381 Gutschrift - 483 Kopie Wertgutschrift - 484 Kopie Gutschrift - 583 Duplikat Wertgutschrift - 584 Duplikat Gutschrift - 683 Ersatz Wertgutschrift - 684 Ersatz Gutschrift

### Feld Rechnungstyp 
Hier wird der Typ der Rechnungsprüfung automatisch angezeigt. Z.B. Eingang , bei Eingang einer Rechnung. 

Tooltip: 

Typ der Rechnung 

### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen.
