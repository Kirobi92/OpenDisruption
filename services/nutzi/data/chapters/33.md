# Registerkarte Telefon 
Registerkarte Telefon 

Auf dieser Registerkarte werden die notwendigen Daten zu einem Anrufer angezeigt. Sie gelangen auf diese Registerkarte, wenn Sie auf das Symbol klicken. 

Es gibt eine Möglichkeit, Kundendaten während eines eingehenden Anrufs aufzurufen: Bei entsprechender Einstellung in Framework Studio ruft der Webservice bei einem eingehenden Anruf die Daten des Tapi-Services auf. Der Info-Button wird ausgeblendet. 

Unter System >> Parameter Telefon muss der Webservice zuvor für jeden Arbeitsplatz aktiviert werden, damit Sie diese Logik nutzen können. 

Außerdem wird automatisch auf die Registerkarte Anruf gewechselt, wenn ein neuer Anruf eingeht oder wenn ein Anruf beendet wird. Dies hat allerdings zur Folge, dass die Anrufliste auf der gleichlautenden Registerkarte erst aktualisiert wird, wenn Sie auf diese Registerkarte wechseln. 

Siehe Kapitel Parameter Telefon . 
## Steuerelemente des Formulars 

### Button CRM Übersicht 

Mit diesem Button gelangen Sie in den Dialog CRM Übersicht . Dort erhalten Sie eine effiziente Übersicht über CRM-Aktivitäten etc. 

Siehe Kapitel CRM Übersicht . 

### Button Info 

Wenn Sie diesen Button aktivieren, werden die Informationen (Kunde, Adresse, Lieferant) zu einem ankommenden Anruf angezeigt, wenn die eingehende Telefonnummer in eNVenta ERP hinterlegt ist. 

### Feld AKL-Nummer 
Hier wird eine Adress-/Kunden- oder Lieferantennummer angegeben. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Feld Firma 
Der Name der Firma wird angezeigt. 

### Feld Partner 
Der Name der Kontaktperson wird angezeigt. 

### Feld Beginn 
Der Beginn (also die Uhrzeit) des Gesprächs wird eingetragen. 

Tooltip: 

Beginn 

### Feld Dauer 
Die Dauer des Gesprächs wird automatisch berechnet. 

Tooltip: 

Dauer in Minuten 

### Button Abheben 

Mit diesem Button können Sie einen eingehenden Anruf am Bildschirm annehmen. Der Griff zum Telefonhörer erübrigt sich. 

### Button Auflegen 

Mit diesem Button können Sie einen Anruf am Bildschirm beenden. Der Griff zum Telefonhörer erübrigt sich. 

### Button Rückruf 

Dieser Button ist zugänglich zu einem aktuellen Anruf / einem markierten Eintrag. Mit diesem Button starten Sie den Rückruf zur angezeigten Telefonnummer. 

### Button Call Center 

Dieser Button öffnet den Dialog Call , zur Erfassung eines Calls. Dabei werden die relevanten Daten voreingestellt. 

### Button Stamm 

Dieser Button ist aktiv, wenn der eingehende Anruf oder der markierte Eintrag ein Kunde, Lieferant oder eine Adresse ist. Es öffnet den Dialog Kunde , Lieferant oder Adresse , damit Ihnen sofort alle wichtigen Informationen zu diesem Anrufer zur Verfügung stehen. Dabei werden die relevanten Daten voreingestellt. 

### Button Bestellung 

Dieser Button ist aktiv, wenn der eingehende Anruf oder der markierte Eintrag ein Lieferant ist. Es öffnet sich der Dialog Bestellung zur Erfassung einer Bestellung. Dabei werden die relevanten Daten voreingestellt. 

### Button Auftrag 

Dieser Button ist aktiv, wenn der eingehende Anruf oder der markierte Eintrag ein Kunde ist. Es öffnet sich der Dialog Auftrag zur Erfassung eines Auftrags. Dabei werden die relevanten Daten voreingestellt. 

### Button Angebot 

Dieser Button ist aktiv, wenn der eingehende Anruf oder der markierte Eintrag ein Kunde ist. Es öffnet sich der Dialog Angebot zur Erfassung eines Angebots. Dabei werden die relevanten Daten voreingestellt. 

### Button Anruf 

Mit diesem Button starten Sie den Telefonanruf. 

Hierbei wird die Referenz der markierten Zeile übernommen. 

### Button Call 

Dieser Button öffnet den Dialog Call zur Erfassung eines Calls. Dabei werden die relevanten Daten voreingestellt. 

Hierbei wird die Referenz der markierten Zeile übernommen. 

Tooltip: 

Call Center öffnen 

### Button Anruf+Call 

Dieser Button baut einen Anruf auf und öffnet gleichzeitig den Dialog Call. 

Hierbei wird die Referenz der markierten Zeile übernommen. 

Tooltip: 

Anruf & Call Center öffnen 

### Bezeichnungsfeld Aktuelle Information 
Hier werden die aktuelle Datenbank und der aktuelle Benutzer angegeben. 

Tooltip: 

Aktuelle DB und aktueller Benutzer
