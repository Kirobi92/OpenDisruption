Grundeinstellungen für eNVenta ERP / Massenänderung 

# Massenänderung Registerkarte Vorlage 
Massenänderung Registerkarte Vorlage 

Auf dieser Registerkarte definieren Sie die Einstellungen für die Massenänderung. 
### Button Löschen 

Mit diesem Button kann ein Eintrag der Tabelle gelöscht bzw. von der Änderung ausgeschlossen werden. 

Tooltip: 

Selektierte Zeilen löschen 

### Button Zurücksetzen 

Mit diesem Button kann ein Eintrag wieder in den Ursprungszustand zurückgesetzt werden. 

Tooltip: 

Auf Ursprungszustand zurücksetzen
