# Projektphase ändern 
Projektphase ändern 

Dieser Dialog öffnet sich mit einem Doppelklick auf eine Phase im Dialog Projekt Registerkarte Phasen . Hier können Sie die Änderungen durchführen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Projektphase ändern 
Tooltip: 

In diesem Dialog können Sie Änderungen an einer Projektphase vornehmen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Speichern 

Mit diesem Button speichern Sie die Eingaben. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Bezeichnung 
Der Name der Phase wird voreingestellt. 

Tooltip: 

Angabe der Phasenbezeichnung 

### Feld Datum/Zeit von 
Geben Sie hier das Start-Datum für die Phase ein. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Datum/Zeit bis 
Für die zeitliche Begrenzung der Phase geben Sie hier ein Bis-Datum ein. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Combobox Bereich 
Hier wählen Sie den Bereich aus, z.B. Rechnungswesen oder Warenwirtschaft. 

### Feld Budget 
Die geplanten Tage werden angegeben. Der Wert wird aus dem Phasenstamm übernommen. 

Tooltip: 

Angabe der geplanten Tage 

### Feld Tage geplant 
Angabe der geplanten Zeit in Tagen. Diese Angabe wird aus dem Phasenstamm übernommen und ist unabhängig von den Einträgen in den Feldern Datum/Zeit von … bis . 

### Feld Tagessatz 
Angabe des Tagessatzes für die geplanten Tage. Der Wert wird aus dem Phasenstamm übernommen. 

Tooltip: 

Angabe des Tagessatzes 

### Feld Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Erfassung 
Hier wird vom Programm automatisch das Datum der Erfassung eingetragen. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Feld Änderung 
Hier wird vom Programm automatisch das Datum der letzten Änderung eingetragen. 

Tooltip: 

Datum der letzten Änderung 

### Feld Beschreibung 
Die Beschreibung der Projektphase wird aus dem Phasenstamm übernommen; er kann hier auch geändert werden. 

Tooltip: 

Angabe einer Beschreibung
