# Übersicht Terminänderung 
Übersicht Terminänderung 

Dieser Dialog öffnet sich in der Planung Betriebsaufträge >> Optionen >> Übersicht Terminänderung und in der Plantafel >> Optionen >> Übersicht Terminaufträge . 

Hier wird ein Protokoll geführt für jede Terminierungsänderung eines Betriebsauftrags. Dabei werden Zeitpunkt, Sachbearbeiter und neue/alte Start- und Endetermine des Betriebsauftrags festgehalten. Ein entsprechender Report kann gedruckt werden. 

Die Protokollierung der Daten erfolgt immer, wenn eine neue Terminierung inkl. Ressourcen erfolgt. 

Im Kopf des Dialogs legen Sie die Selektionskriterien für die Suche fest. Diese können auch über den Button SQL-Suche bzw. den Button Einstellungen festgelegt werden. 

Der Button Löschen auf der Registerkarte Allgemein steht zur Verfügung, um markierte Tabelleneinträge zu löschen. 

Aber Achtung: Markierte Einträge werden ohne vorherige Prüfung oder Rückfrage gelöscht. 

In diesem Zusammenhang gibt es im Dialog Planung Betriebsaufträge die Checkbox K.mehrstufige PA . Wenn Sie diese Checkbox aktivieren, werden nur Betriebsaufträge ohne Produktionsauftrag und Betriebsaufträge mit Produktionsaufträgen, die nur einen Betriebsauftrag enthalten, angezeigt.
