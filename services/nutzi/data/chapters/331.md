# Datenexport Registerkarte Allgemein

Allgemeine Funktionalitäten in eNVenta ERP / Datentransfer (Import und Export von Daten) / Datenexport 

# Datenexport Registerkarte Allgemein 
Datenexport Registerkarte Allgemein 

Auf dieser Registerkarte legen Sie allgemeine Einstellungen für den Export fest. 
### Combobox Formatart 
Hier wählen Sie aus, ob der Datentransfer über den neuen X-Mapper oder die alte Formatdefinition abgewickelt werden soll. 

Wenn Sie hier X-Mapper auswählen, werden auf der Registerkarte nur noch Felder angezeigt, die für den X-Mapper relevant sind. So kann das Mapping der Daten und Dateien über den X-Mapper erfolgen. 

Aus diesem Grund wird auch die Registerkarte Format ausgeblendet. 

Tooltip: 

Hier wird bestimmt, ob der Datentransfer über den neuen X-Mapper oder die alte Formatdefinition abgewickelt werden soll. 

### Combobox X-Mapping ID 
Hier wird der Identifizierer eines Mappings angegeben, das mit dem X-Mapper erstellt wurde. 

Tooltip: 

Identifizierer eines Mappings, das mit dem X-Mapper erstellt wurde. 

### Combobox Verbindung 
Hier wählen Sie die Verbindung aus, über die die exportierten Daten gespeichert werden (z.B. Server oder lokales Laufwerk). 

Tooltip: 

Angabe des Typs 

### Feld Datei 
Hier hinterlegen Sie, wohin die exportierten Daten gespeichert werden (z.B. Laufwerk C oder Desktop). 

### Button Suche 

Über diesen Button kann der Pfad für die CSV-Datei gesucht werden. 

### Combobox Format 
Hier wählen Sie den Formattyp für die Exportdatei aus, entweder CSV oder ASCII. Dazu definieren Sie noch Dezimalzeichen, Separator und/oder Kopfzeile, je nachdem was die Gegenstelle benötigt. Dies wird zusammen mit der Formatdefinition abgespeichert. 

### Combobox Codierung 
Hier wählen Sie die Codierung der Zeichen aus, die den Anforderungen des Empfängers entsprechen (Codierung nach ISO 8859). 

Siehe Kapitel Formate für Exportdateien - Encodings . 

### Checkbox Daten anfügen 
Beim Export wird die bestehende Datei immer überschrieben. Es muss die bestehende Datei ergänzt werden. 

Tooltip: 

Daten anfügen an eine vorhandene Datei. 

### Feld Anzahl Sätze 
Nach dem Export wird in diesem Feld die Anzahl der exportierten Datensätze angezeigt. 

### Feld Anzahl Fehler 
Falls beim Export Fehler auftreten, wird hier die Anzahl der Fehler angezeigt.
