# Import Items erzeugen Registerkarte Spalten 
Basis Dialog Dokumente Registerkarte Spalten 

Auf dieser Registerkarte werden alle Export-Spalten(Properties) in einer Tabelle dargestellt. Um eine Spalte zu einem späteren Zeitpunkt mit Daten zu initialisieren, aktivieren Sie die Checkbox in der Spalte ‚Platzhalter‘. Es öffnet sich eine zweite Tabelle in der alle verfügbaren Platzhalter aufgelistet sind. Durch Doppelklicken wird einer Spalte ein gewünschter Platzhalter zugewiesen. 

Diese Daten werden beim Import in die Datenbank automatisch gefüllt.
