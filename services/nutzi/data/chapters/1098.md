# VK-Positionen bearbeiten Registerkarte Logistik 
Position Registerkarte Logistik 

Auf dieser Registerkarte können Einstellungen bezüglich der Logistik, also Versandweg, Transportweg etc. vorgenommen werden. 

Erfassung Gefahrgut 

Bei der Erfassung einer Auftragsposition werden immer die Punkte berechnet, falls es sich um einen Gefahrgutartikel handelt (ADR-Zuordnung vorhanden). Die Berechnung erfolgt immer anhand der ‚Menge zu liefern‘. Das Ergebnis wird pro Auftragsposition gespeichert und wird hier angezeigt. 

Auf dieser Registerkarte stehen entsprechende Funktionalitäten für die Gefahrgutverwaltung zur Verfügung. Dies sind: 

- Checkbox ADR Position - ADR Transporttyp - ADR Punkte und - Button ADR Info 
Die Checkbox ADR Position wird automatisch gesetzt, wenn es sich um ein Gefahrgut handelt. 

Aus der Combobox ADR Transporttyp wählen Sie die Art des ADR Transports aus (EQ, LQ, Transport). Im Standard ist immer die Option ADR Transport nach Punkten hinterlegt. 

Sie haben manuell die Möglichkeit die Angabe auf „Freigestellte Menge (EQ)“ oder „Begrenzte Mengen (LQ)“ zu ändern. Diese Möglichkeiten stehen nur zur Verfügung, wenn die Stammdaten gemäß Tabelle A eine entsprechende Zuordnung erlauben. Bei der Angabe von E0 als freigestellte Menge ist die Angabe „Freigestellte Menge (EQ)“ nicht erlaubt. Wenn keine Angabe „Begrenzte Mengen (LQ)“ vorhanden ist, kann diese Option nicht ausgewählt werden. 

Für die vereinfachte Berechnung erfolgt bei Angabe von Gefahrgütern der Beförderungskategorie 0 die Berechnung fix auf 9999 Punkte, wodurch der Transport gemäß ADR ohne Ausnahmegenehmigung erfolgen muss. 

Achtung: eNVenta prüft diese Einstellung nicht nach – die Anpassung obliegt dem Sachbearbeiter gemäß den Vorschriften der ADR.

### Feld Wunschdatum 
Wünscht der Kunde eine frühere Lieferung, wird der entsprechende Wunschtermin in das Feld Wunschdatum eingetragen. Im Programmteil Liefervorschlag kann das Programm gegebenenfalls bei frühzeitiger Lieferung den Wunsch des Kunden berücksichtigen und eine frühere Lieferung vorschlagen. 

Dieses Wunschdatum wird in der Tourendisposition nicht berücksichtigt; dort wird der Liefertermin zugrundegelegt. 

Existieren 2 Positionen mit unterschiedlichem Lieferdatum, so werden diese Positionen zum „kleineren“ Datum berücksichtigt (um Kosten zu senken). 

Wenn kein Wunschdatum auf der Position vorhanden ist (z.B. Parameter Kein Termin aus Kalenderwoche ), so wird das Kopf-Lieferdatum als Wunschdatum der Position übertragen und damit die Tourenplanung ausgeführt.

### Checkbox Liefersperre 
Wenn Sie diese Checkbox aktivieren, wird eine Liefersperre für diese Auftragsposition gesetzt. 

Diese Sperre kann gesetzt werden, so lange der Status der Position < 3 ist. 

Solange die Sperre gesetzt ist, kann für die Position weder eine Packliste noch ein Lieferschein gedruckt werden. 

Wenn die Auftragsposition ein Beschaffungsartikel ist, gilt folgendes: 

Hat eine Auftragsposition mit einem 'Beschaffungsartikel' die "Liefersperre" gesetzt, wird dieser Artikel nicht bei der Berechnung eines Bestellvorschlages / im Bestellautomat berücksichtigt. 

Die Liefersperre im Auftrag bewirkt, dass der Beschaffungsartikel auch erst dann bestellt wird, wenn alle Unklarheiten mit dem Kunden geklärt werden konnten und die Liefersperre dann entsprechend entfernt wird.

### Combobox Bezugslager 
Hier wählen Sie das Bezugslager aus, d.h. das Lager von welchem die Ware per Umlagerungsprozess (Umlagerungsauftrag und -bestellung) bezogen werden soll. 

Beispiel: Der Kunde möchte seine Ware an einem bestimmten Lager (Lager 1) abholen. Dazu ist es notwendig, die Ware vorher mittels Umlagerung vom Lager 2 zu beschaffen. 

Die Generierung von Umlagerungsauftrag und -bestellung erfolgt automatisch beim Auftragsabschluss und wird als Verbundobjekt unterhalb der Auftragsposition angezeigt. (Registerkarte Verbund im Auftrag auf der Registerkarte Positionen ) 

Es stehen nur die Lager zur Verfügung, die als Bezugslager definiert sind. Von diesem Bezugslager muss die Ware per Umlagerung beschafft werden. 

Falls keine Bezugslager-Zuordnung besteht, dann zieht die Einstellung alle Lager liefern im Dialog Lagerorte (d.h. alle vorhandenen Lager sind automatisch Bezugslager für dieses Lager. AUSNAHME: Sperrlager!). 

Es ist notwendig, dass eine der beiden genannten Einstellungen gesetzt ist. 

Bezugslager werden im gleichnamigen Dialog einem Versandlager zugeordnet.

### Feld ADR Punkte 
Hier werden die ADR Punkte angegeben. 

Zur Berechnung der ADR Punkte wird der stoffspezifische Faktor identifiziert und multipliziert mit der Menge. 

Die Summe aller berechneten Einzelwerte der geladenen gefährlichen Güter darf dann den Wert 1000 nicht übersteigen, sonst ist der Transport "kennzeichnungspflichtig", d.h. Fahrzeug und Fahrer müssen alle ADR-Transportvorschriften berücksichtigen. 

In Zusammenhang mit ADR lesen Sie das Kapitel Gefahrgutverwaltung .

### Checkbox Kein Paketdienst 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Paketdienst-Touren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Kein Paketdienst“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Beim Kopieren eines Angebots / Auftrags wird diese Einstellung übernommen.
