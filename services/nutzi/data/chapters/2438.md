# OP-Altersgliederung Registerkarte Selektionen 3 
OP-Altersgliederung Registerkarte Selektionen 3 

Für die Multisuche stehen für Debitoren auf der Registerkarte Selektionen 3 die Tabellen Gruppe, Vertreter und Gebiet für eine Selektion zur Verfügung. 

Für die Multisuche steht für Kreditoren auf der Registerkarte Selektionen 3 die Tabelle Gruppe für eine Selektion zur Verfügung. 

Multisuche: Sie können einzelne Sätze selektiv oder blockweise markieren.
