# Artikel Cockpit Registerkarte Preisfindung 
Registerkarte Preisfindung 

Auf dieser Registerkarte werden Daten bezüglich der Preisfindung angegeben. 
### Combobox Einheit VK 
Hier wird die Einheit für den Verkauf ausgewählt. 

Tooltip: 

Verkaufseinheit 

### Feld Faktor VK 
Hier wird ein Umrechnungsfaktor von Lagereinheit zu Verkaufseinheit (bezogen auf den Preis) angegeben, um den in Relation korrekten Verkaufspreis zu ermitteln. 

Tooltip: 

Faktor VK gibt an, wieviel Lagereinheiten 1 VK-Mengeneinheit entspricht. 

Beispiel: EinhL = Karton, EinhVK = Stück, FaktorVK = 0,01 => 
1 Stück entspricht 0,01 Kartons bzw. 
1 Karton enthält 100 Stück 

### Feld Umrechnungsfaktor 
Umrechnungsfaktor von der Verkaufs- zur Lagereinheit. 

Wird ein Faktor VL eingegeben, so wird nur der Faktor VK umgerechnet und automatisch eingefügt, der eingegebene Wert bleibt erhalten. 

Umgekehrt gilt: bei der Eingabe des Faktors VK wird nur der Wert in Faktor VL automatisch angepasst. 

Vergleichen Sie hierzu noch die Beschreibung zu Feld Faktor VK . 

Beispiel: 
Lager = 1 Kanister mit 100 kg, Verkauf = 1 kg, Faktor = 100. 

### Combobox Artikelgruppe 
Hier wird eine Artikelgruppe angegeben. Sie kann alphanumerisch sein und darf max. 8 Zeichen haben. Artikelgruppen werden im Menü Stamm >> Artikelgruppen angelegt. 

Siehe Kapitel Artikelgruppen . 

Tooltip: 

Angabe der Artikelgruppe 

### Combobox VK-Rabattgruppe 
Für die Preisfindung eines Artikels kann statt der Artikelgruppe auch eine Rabattgruppe herangezogen werden. 

### Combobox Artikelkalk. Gr. 
Wenn Sie aus dieser Combobox einen Eintrag auswählen, wird der Artikel einer Artikel Kalkulationsgruppe zugeordnet. 

Siehe Kapitel Artikel Kalkulationsgruppe . 

Tooltip: 

Artikelkalkulationsgruppe 

### Combobox Kum Gruppe 
Hier wählen Sie die Kumulationsgruppe für die warengruppenübergreifende Rabattierung (Assortierung) aus. 

### Feld VK pro 
Hier wird eine Preiseinheit angegeben. Dieser Preis kann sich auf 10, 100, 1000 oder eine andere Anzahl Artikeleinheiten beziehen. Standard ist der Wert 1. 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- 1 Einheit - je 10 Einheiten - je 100 Einheiten 

Tooltip: 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- je 1 Einheit 
- je 10 Einheiten 
- je 100 Einheiten 

### Feld Min. Rohertrag 
Hier wird der minimale Rohertrag des Artikels angezeigt. 

Tooltip: 

Minimaler Rohertrag des Artikels 

### Feld Max. Rohertrag 
Hier wird der maximale Rohertrag des Artikels angezeigt. 

Tooltip: 

Maximaler Rohertrag des Artikels 

### Combobox Einheit Verp. 
Hier wählen Sie die Einheit für Verpackung aus. 

Tooltip: 

Angabe der Verpackungseinheit 

### Feld Menge Verp. 
Es ist möglich, für ein und denselben Artikel unterschiedliche Mengeneinheiten für das Lager und den Verkauf zu führen. Beispielsweise können Sie einen Artikel in Stück am Lager führen, jedoch kistenweise verkaufen. Der Umrechnungsfaktor von der Lagereinheit zur Verkaufseinheit wird in das Feld Faktor eingetragen. Standardmäßig enthält das Feld den Wert 1 (Lagereinheit = Verkaufseinheit). 

Beispiel: 
Lager = 1 Kanister mit 100 kg, Verkauf = 1 kg, Faktor = 0,01. 

Die beiden Felder Einheit Verp. und Menge / VP müssen gemeinsam mit Werten gefüllt werden. Falls eines der Felder nicht gefüllt wird, erscheint eine Meldung. 

### Combobox Anbruch 
Mit einer Auswahl aus dieser Combobox können Anbruch-Varianten bereits auf Artikelebene definiert werden. 

Dazu sind zwingend auch die Felder Einheit Verp. und Menge / VP zu füllen.

### Combobox Keine Rückn. 
Hiermit kann der Artikel von der Rücknahme ausgeschlossen werden. 

Nur Sachbearbeiter mit Berechtigungsgruppe Rücknahme erlaubt dürfen solche Artikel zurücknehmen. 

Tooltip: 

Hiermit kann der Artikel von der Rücknahme ausgeschlossen werden. 
Nur Sachbearbeiter mit Berechtigungsgruppe -> 'Rücknahme erlaubt' dürfen solche Artikel zurücknehmen. 

### Combobox Runden Preis 
Hier kann das gewünschte Rundungsverfahren eingestellt werden. Die Rundung erfolgt automatisch bei jeder Änderung des Preisfaktors im Gesamtwert. Eine Besonderheit sind die Bruttorundungsverfahren. Die gewünschte Rundung entsteht erst nach Aufschlag des Steuersatzes. Dies ist ein Pflichtfeld 

Tooltip: 

Standardmäßig wird der Preis auf 3 nachkommastellen gerundet. 
Sollte ein Rundungsverfahren selektiert sein wird der Preis nach diesem gerundet. 

### Feld Min. M. Auftrag 
Hier wird die Mindestmenge des Artikels pro Position angegeben, bezogen auf die Standard-Verkaufseinheit. Eine Unterschreitung führt zu einer Meldung im Abschluss, wenn die Abschlussprüfung im Dialog Parameter Verkauf (Registerkarte Allgemein ) gesetzt wurde. Damit vermeiden Sie Fehler beim Einlesen der Daten über Schnittstellen (wie z.B. Eingabe der Menge 1000 statt der Menge 10). 

Wird eine abweichende Verkaufseinheit gewählt, so ist es erforderlich diese vor dem Vergleich entsprechend umzurechnen. 

Siehe Kapitel Parameter Verkauf . 

Tooltip: 

Mindest Auftragsmenge 

### Feld Max. M. Auftrag 
Hier wird die maximale Menge des Artikels pro Position angegeben, bezogen auf die Standard-Verkaufseinheit. Eine Überschreitung führt zu einer Meldung im Abschluss, wenn die Abschlussprüfung im Dialog Parameter Verkauf (Registerkarte Allgemein ) gesetzt wurde. Damit vermeiden Sie Fehler beim Einlesen der Daten über Schnittstellen (wie z.B. Eingabe der Menge 1000 statt der Menge 10). 

Wird eine abweichende Verkaufseinheit gewählt, so ist es erforderlich diese vor dem Vergleich entsprechend umzurechnen. 

Siehe Kapitel Parameter Verkauf .

Maximale Auftragsmenge 

### Feld Min. absolut 
Hier wird der absolute Mindestrohertrag angegeben. 

Tooltip: 

Min. Rohertrag absolut 

### Checkbox Kein Skonto 
Wenn diese Checkbox aktiviert ist, kann der Artikel kein Skonto erhalten. 

Tooltip: 

Wenn aktiviert, wird kein Skonto gezogen. 

### Checkbox Kein Rabatt 
Wenn Sie diese Checkbox aktivieren, werden vom System keine Rabatte angewendet. Dies kann im Vorgang durch den Sachbearbeiter übersteuert werden. 

Tooltip: 

Wenn gesetzt, so werden keine Rabatte oder Zuschläge aus den Konditionen gezogen. 

### Checkbox Seriennummer 
Wenn diese Checkbox aktiviert ist, handelt es sich bei dem Artikel um einen Serienartikel. 

Tooltip: 

Kennzeichnet den Artikel als Seriennummernartikel. 

### Checkbox Chargen 
Wenn Sie diese Checkbox aktivieren, wird der Artikel als Chargenartikel gekennzeichnet. 

Die Chargenverwaltung steht nur zur Verfügung, wenn sie lizenziert ist. Dies erkennen Sie, wenn im Dialog Konfiguration die Checkbox aktiv gesetzt ist. 

Siehe Kapitel Konfiguration der Parameter . 

Wenn die Chargenverwaltung zu diesem Zeitpunkt nicht aktiviert ist, erscheint eine Hinweismeldung. 

Tooltip: 

Definiert einen Artikel als Chargen-geführt (auch Los oder Serie genannt).
