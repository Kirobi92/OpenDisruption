# Fehlerarten 
Fehlerarten 

In diesem Dialog hinterlegen und verwalten Sie alle möglichen Fehlerarten, die beim Verpacken oder Versenden der Ware vorkommen können. Diese Fehlerarten spielen eine Rolle bei der Endkontrolle. 

Die Nummer der Fehlerart und der Sachbearbeiter werden vom System automatisch eingetragen, die Spalten ‚Name‘ und ‚Beschreibung‘ werden vom Sachbearbeiter gefüllt. 

Siehe Kapitel Endkontrolle . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Fehlerarten 
Tooltip: 

In diesem Dialog hinterlegen und verwalten Sie alle möglichen Fehlerarten, die beim Verpacken oder Versenden der Ware vorkommen können. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Tabelle 

### Tabellenspalte Fehlerart 
In dieser Spalte wird die Nummer der Fehlerart angezeigt; bei einem neuen Eintrag wird vom System automatisch eine Nummer generiert. 

### Tabellenspalte Name 
In dieser Spalte wird der Name der Fehlerart angezeigt; oder bei einem neuen Eintrag eine Fehlerart eingetragen. 

### Tabellenspalte Beschreibung 
In dieser Spalte wird die Beschreibung der Fehlerart angezeigt; oder bei einem neuen Eintrag eine neue Beschreibung eingetragen. 

### Tabellenspalte Sachbearbeiter 
In dieser Spalte wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz angelegt bzw. zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
