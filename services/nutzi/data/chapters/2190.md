# Service Druckbelege 
Service Druckbelege 

Der Button Belege im Serviceeinsatz öffnet diesen Dialog mit einer Historie der Druckbelege für Formulare, die bereits zu diesem Einsatz gedruckt worden sind. Dabei werden der Formularname, das Druckdatum und der Sachbearbeiter angegeben. 

Wenn Sie in der Tabelle einen Eintrag markieren, können Sie diesen ein weiteres Mal mit dem Button drucken. Über den Modus und / oder das Format können Sie vorgeben, wie der Druck staffinden soll. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Service Druckbelege 
Tooltip: 

In diesem Dialog werden bereits gedruckte Formulare zu einem Serviceeinsatz angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Dock Panel Format 

Über diese Radiobuttons wählen Sie das Druckformat aus. 

### Dock Panel Modus 

Wählen Sie hier, ob Sie einen Ausdruck auf Papier möchten oder der Beleg auf dem Bildschirm angezeigt werden soll. 

Über den Radiobutton E-Mail kann ein Druckbeleg auch als E-Mail-Anhang versendet werden. 

Wenn beim E-Mail-Versand mehrere Referenz-Links angehängt werden, werden die Dateinamen automatisch durchnummeriert, z.B. „_1“, „_2“, wenn diese den gleichen Dateinamen besitzen.
