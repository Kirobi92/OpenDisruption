# Leistungsarten Registerkarte Allgemein 
Leistungsarten Registerkarte Allgemein 

Auf dieser Registerkarte müssen alle Felder gefüllt werden. Zur Auswahl stehen Ihnen zu jedem Feld die -Buttons zur Verfügung, um die entsprechende Kostenart, Kostenstelle oder Kostenträger zu suchen. 

Wenn die Leistungsart einen Kostenträger hat, dann hat dieser Priorität. 
### Feld KA Entlastung 
Hier wird eine Kosten- oder Leistungsartennummer für die Kostenrechnung angegeben. 

Tooltip: 

Kostenart für Entlastung. 

### Button Suche 

Mit diesem Button können Sie eine Kostenart zur Entlastung suchen. 

Siehe Kapitel Kostenartenauswahl . 

### Feld Name 
Hier wird der entsprechende Name angezeigt. 

### Feld KA Belastung 
Hier wird eine Kosten- oder Leistungsartennummer für die Kostenrechnung angegeben. 

Tooltip: 

Kostenart für Belastung. 

### Button Suche 

Mit diesem Button können Sie eine Kostenart zur Belastung suchen. 

Siehe Kapitel Kostenartenauswahl . 

### Feld Name 
Hier wird der entsprechende Name angezeigt. 

### Feld KSt Belastung 
Hier wird die Kostenstellennummer für die Belastung angegeben. 

Tooltip: 

Kostenstelle für Belastung 

### Button Suche 

Mit diesem Button können Sie eine Kostenstelle suchen. 

Siehe Kapitel Kostenstellenauswahl . 

### Feld Name 
Hier wird der entsprechende Name angezeigt. 

### Feld KTr Belastung 
Hier wird der Kostenträger für die Belastung angegeben. 

Tooltip: 

Kostenträger für Belastung 

### Button Suche 

Mit diesem Button können Sie einen Kostenträger suchen. 

Siehe Kapitel Kostenträgerauswahl . 

### Feld Name 
Hier wird der entsprechende Name angezeigt. 

### Feld Faktor 
Für die Berechnung kann hier ein Faktor eingetragen werden. 

Tooltip: 

Angabe des Faktors 

### Combobox Leistungstyp 
Hier kann eine Leistungsart ausgewählt werden. 

Tooltip: 

Kategorisierung von Leistungsarten. 

### Combobox Leistungseinheit 
Hier kann eine Leistungseinheit ausgewählt werden. 

Tooltip: 

Einheit der Leistungsart
