# Packstückrückscan 
Packstückrückscan 

Dieser Dialog öffnet sich im Dialog Lieferschein scannen >> Optionen >> Packstückrückscan . 

Beim Wareneingang kann ein Mitarbeiter durch Scannen der Packstücke die 2. Stufe des Wareneingangs verbuchen und automatisch diese Packstücke auf eine neue Tour zur Auslieferung verladen (Cross-Docking: Die Ware wird nicht eingelagert, sondern direkt auf eine andere Tour verladen.) Voraussetzung hierbei ist, dass alle Auftragspositionen den Packstücken bereits bei der Versendung vom Bezugslager zugeordnet wurden. Sollte keine Positionszuordnung vorhanden sein, so erfolgt kein Cross-Docking. Dies muss erfolgen damit die Mengenprüfung korrekt ausgeführt werden kann. 

Diese Funktion steht nur bei einem 3-teiligen-Verbund (auftragsbezogene Umlagerung) zur Verfügung. Der Wareneingang der 1. Stufe wird hierbei automatisch durch den Lieferscheindruck im Umlagerungsauftrag erzeugt – die 2. Stufe wird in diesem Dialog ausgeführt. 

Die Verbuchung der 2. Stufe erfolgt erst nachdem alle Packstücke zur Lieferung erfolgreich gescannt wurden. Nach dem Scannen erfolgt im Hintergrund automatisch die Erzeugung von Packliste und Lieferschein für das Cross-Docking. Nach dem Scannen eines Packstücks wird hierzu das Feld automatisch geleert, damit das nächste Packstück angescannt werden kann. 

Nach Eingabe der Lagernummer kann der Sachbearbeiter die Packstücknummer anscannen. Ist das Lager nicht das Ziellager dieses UA-/UB-Packstücks, kommt es zu einer Fehlermeldung. Ansonsten wird der Cross-Docking-Vorgang ausgelöst.
