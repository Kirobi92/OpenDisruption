# Thekenerfassung Registerkarte Meldungen 
Registerkarte Meldungen 

Wenn beim ORDERS-Import (Auftragsimport) Meldungen in das Gateway geschrieben werden, werden diese im angelegten Auftrag auf der Registerkarte Meldungen angezeigt. Hierbei werden alle Meldungen aus dem Mapping und den Prozessregeln angezeigt, damit der Sachbearbeiter im Verkauf diese Meldungen entsprechend prüfen kann. Ist dazu noch eine entsprechende Abschlussprüfung (EDI-Fehler in Auftragsposition) angelegt, kann der Auftrag auch nicht automatisch abgeschlossen werden. 

Die Meldungen können markiert werden und über den Button erledigt entsprechend als erledigt markiert werden. 

Wird eine Meldung markiert, so werden im Feld unten links die Details der Meldung angezeigt. Im Feld rechts daneben können Notizen zur jeweiligen Meldung hinterlegt werden. 

Über die Abschlussprüfung Remarks in Auftrag wird geprüft, ob es in einem Auftrag importierte Meldungen gibt. Sobald es Meldungen gibt, muss sich ein Sachbearbeiter den Auftrag und die dazu gehörenden Meldungen ansehen. 

Hinweis: Die Abschlussprüfungen sind nicht Bestandteil des Standards, diese stehen mit dem Großhandels-Package zur Verfügung. 

Diese Meldungen können entweder als Info, Warnung oder Fehler ausgegeben werden. 
### Button Erledigt 

Mit diesem Button wird der Status der ausgewählten Meldung auf Erledigt gesetzt. 

Tooltip: 

Ändert den Status der ausgewählte Meldung auf 'erledigt'.
