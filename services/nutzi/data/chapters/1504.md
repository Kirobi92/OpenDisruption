Kasse / Kassensicherungsverordnung TSE 

# DSFinV-K Export Parameter 
DSFinV-K Exp. Param. 

Der Button Parameter im Dialog DSFinV-K Export öffnet den Dialog DSFinV-K Exp. Param . Hier können Sie die Parameter für den Export über den X-Mapper definieren. Für jede Tabelle wird ein entsprechendes Mapping mitgeliefert. Der Dateiname kann geändert werden, doch sollte dies nur auf Anforderung des Prüfers gemacht werden, da der Dateiname in der DSFinV-K vorgegeben ist. 

DSFinV-K bedeutet: Digitale Schnittstelle der Finanzverwaltung für Kassensysteme 

Beim Export nach DSFinV-K werden alle Dateien exportiert, welche die Kennung Export haben und zusätzlich die geforderte Datei Index.xml. 

Über den Button X-Mapping gelangen Sie in den X-Mapping Editor für eine markierte Zeile. 

Siehe Kapitel X-Mapping Editor .

### Button Suche 
Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 
Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 
Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button X-Mapping 

Über diesen Button gelangen Sie in den X-Mapping Editor für eine markierte Zeile. 

Siehe Kapitel X-Mapping Editor .
