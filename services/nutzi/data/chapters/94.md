# Wv-Übersicht 
Wv-Übersicht 

Der Button im Navigator auf der Registerkarte Termine öffnet den Dialog Wv-Übersicht und bietet die Möglichkeit alle im System befindlichen Wiedervorlagen aufzurufen. Die Auswahl der Wiedervorlagen kann über die Felder und Comboboxen im Kopf des Dialogs eingegrenzt werden. 

Einzelne Vorgänge können über einen Doppelklick in eine Zeile der Tabelle aufgerufen werden. 

Die Rückwärtsterminierung eines in der Zukunft liegenden Termins bis zum aktuellen Tagesdatum ist möglich. 

Dabei muss berücksichtigt werden, dass dann das Tagesdatum ausgeschrieben werden muss (die Eingabe des * reicht hier nicht aus!), da die Uhrzeit ebenfalls berücksichtigt wird. 

Offene Wiedervorlagen können bereits in der Tabelle geändert werden. 

Je nach Priorität wird die Spalte ‚Termin‘ farblich gekennzeichnet. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Wv-Übersicht 
Tooltip: 

Dieser Dialog bietet die Möglichkeit alle im System befindlichen Wiedervorlagen aufzurufen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Drucken 

Dieser Button druckt den Inhalt zur Übersicht. 

Tooltip: 

Auslösen des Drucks 

### Button Suche 

Mit diesem Button suchen Sie nach Informationen; entsprechend Ihrer Selektionskriterien. 

### Button Neu 

Dieser Button leert die Datenfelder. 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Neuer Satz 

Mit diesem Button können Sie der Tabelle einen neuen Eintrag hinzufügen. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Combobox Sachbearbeiter 
Hier wählen Sie den Sachbearbeiter aus, dessen Datensätze Sie ansehen möchten. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Combobox Abteilung 
Hier wählen Sie die Abteilung aus, dessen Datensätze Sie ansehen möchten. 

### Feld Vorgang 
In diesem Feld wird der Vorgang (Call oder Projekt) mit Nummer und Namen eingetragen, den Sie ansehen möchten. 

### Feld Nachricht 
In diesem Feld wird die Nachricht, d.h. die Information aus der Wiedervorlage eingetragen, die Sie ansehen möchten. 

### Combobox Priorität 
Hier geben Sie die Priorität des Termins ein. Zur Verfügung stehen: 
- normal - wichtig - sehr wichtig 

Tooltip: 

Priorität des Termins 

### Combobox Termin-Typ 
Hier geben Sie den Termin-Typ an. Zur Verfügung stehen: 
- zur Bearbeitung - zur Info - zur Genehmigung 

### Feld Termin von 
Hier geben Sie ein Datum ein, ab dem Sie suchen möchten. 

### Feld Termin bis 
Hier geben Sie ein Datum ein, bis zu dem Sie suchen möchten. 

### Field Panel Status 

Zur Selektion können Sie in diesem Bereich nach Status selektieren. Zur Verfügung stehen: 
- offen - erledigt - gelöscht 
Der Status offen ist voreingestellt. 
### Checkbox Offen 
Diese Checkbox ist voreingestellt; es werden nur offene Calls in die Tabelle eingelesen. 

### Checkbox Erledigt 
Wenn Sie diese Checkbox aktivieren, werden nur erledigte Termine in die Tabelle eingelesen. 

### Checkbox Gelöscht 
Wenn Sie diese Checkbox aktivieren, werden nur gelöschte Termine in die Tabelle eingelesen. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Tabelle Termine 

Diese Tabelle wird entsprechend Ihrer Selektionskriterien mit detaillierten Informationen/Daten gefüllt. 
### Tabellenspalte Termin 
Hier können Sie eingeben, wann die Wiedervorlage beim Empfänger ankommen soll. Das Tagesdatum ist voreingestellt. 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Vorgang 
Hier wird der betroffene Vorgang angegeben. 

### Tabellenspalte Abteilung 
Hier wird die Abteilung des Sachbearbeiters angegeben. 

### Tabellenspalte Priorität 
Hier geben Sie die Priorität des Termins ein. Zur Verfügung stehen: 
- normal - wichtig - sehr wichtig 

Tooltip: 

Priorität des Termins 

### Tabellenspalte Termin-Typ 
Hier geben Sie den Termin-Typ an. Zur Verfügung stehen: 
- zur Bearbeitung - zur Info - zur Genehmigung 

### Tabellenspalte Nachricht 
Hier wird der Text aus dem Feld Nachricht angezeigt. 

### Feld Nachricht 
In diesem Feld können Sie sich die Nachrichtentexte ausführlich anzeigen zu lassen. Dazu markieren Sie eine Zeile der Tabelle. Die Nachricht, die erfasst wurde, wird angezeigt. 

Außerdem haben Sie über das Feld. Max. Anzahl die Möglichkeit nur eine bestimmte Anzahl an Terminen in der Tabelle anzeigen zu lassen.
