# Versandprofile im Webshop Registerkarte Allgemein 
Versandprofile Registerkarte Allgemein 

Auf dieser Registerkarte nehmen Sie die detaillierten Einstellungen für die Versandart vor. 
### Button Neu 

Mit diesem Button können Sie eine weitere Versandart anlegen. 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag aus der Tabelle. 

### Tabelle 

### Tabellenspalte Versandtyp 
Hier können Sie dem Versandprofil einen Versandtypen zuweisen. 

### Tabellenspalte Versandart 
Hier wird die Nummer der Versandart angegeben. 

Tooltip: 

Art, wie die Sendung vom Absender zum Empfänger gelangt. 

### Tabellenspalte Versender 
Hier kann der zuständige Versender eingestellt werden. Die Versender pflegen Sie im Dialog Picklisten . 

Beim Ändern des Versenders werden automatisch alle Versandcodes zur Versandart gelöscht. 

Siehe Kapitel Picklisten . 

Tooltip: 

Geben Sie den zuständigen Versender ein. 

### Tabellenspalte Lieferbedingung 
Hier wird die Lieferbedingung zum Versandprofil angegeben. 

Tooltip: 

Geben Sie hier eine Lieferbedingung ein. 

### Tabellenspalte Versandkosten kalkulieren 
Hier wierden die Versandkosten angegeben. 

Tooltip: 

Angabe der Versandkosten 

### Button Nach oben 

Mit diesem Button können Sie einen Eintrag der Tabelle nach oben verschieben. 

Tooltip: 

Nach oben verschieben 

### Button Nach unten verschieben 

Mit diesem Button können Sie einen Eintrag der Tabelle nach unten verschieben. 

Tooltip: 

Nach unten verschieben
