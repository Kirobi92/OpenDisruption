# Auswertungsstruktur Registerkarte Saldenauswertung 
Auswertungsstruktur Registerkarte Saldenauswertung 

Die Registerkarte Saldenauswertung gibt Ihnen eine Vielzahl von Möglichkeiten, Saldenstrukturen zu definieren und auszuwerten. U.a. sind dies betriebswirtschaftliche Auswertungen wie BWA, Erlösstrukturen, Kostenstrukturen, Wareneinkaufsstrukturen, Haushaltsquerschnitte u.v.m. 

Im Gegensatz zu den anderen Auswertungen kann in der Saldenauswertung ein Konto mehrfach verschiedenen Details zugeordnet werden. 

Ein bereits zugeordnetes Konto ist in der rechten Tabelle farbig markiert. 

Um in der Auswertungsstruktur einen Rechnungslegungskreis anzugeben, müssen Sie mit dem Button Neue Auswertung den Dialog Neue Auswertung öffnen. Nur hier können Sie einen Rechnungslegungskreis zuordnen. 

Beim Kopieren einer Auswertung ist der Rechnungslegungskreis nicht identisch, im Dialog Neue Auswertung ist die Checkbox mit Details unzugänglich, d.h. der Baum wird kopiert, die Konten aber nicht. 

Der Rechnungslegungskreis wird in der Auswertungsstruktur in einer unzugänglichen Combobox angezeigt. Die Auswertungsstruktur zeigt in der Tabelle der Konten (rechte Seite, Mitte) nur die zum Rechnungslegungskreis zugeordneten Konten. 

Beim Abrufen der Auswertung (Saldenauswertung, Bilanz oder G + V) erfolgt eine Prüfung auf falsch zugeordnete Konten. 

Beispiel „Erlösstruktur“: 

Über den Button Neue Auswertung legen Sie eine neue Struktur (Auswertung) an. Bitte vergeben Sie hier den Namen „Erlösstruktur“. 

Nun müssen Sie diese Erlösstruktur markieren und über den Button Neue Stufe eine Stufe hinzufügen. Nennen Sie diese neu hinzugefügte Stufe nun Ergebnis und geben in das Feld Summentext auch das Wort 'Ergebnis' ein. Anschließend setzen Sie noch die Checkbox Nur Summe , damit auch nur die Summe und der Summentext ausgegeben werden. Innerhalb dieser Stufe zeigen wir nun unsere Erlöse und Kosten auf! Markieren Sie die Stufe „Ergebnis“ und klicken Sie auf den Button Neue Stufe und benennen diese Stufe als 'Erlöse'. Das gleiche machen Sie mit den Kosten. Damit über die einzelnen Stufen eine Summe gebildet werden kann, muss auch hier der Summentext und die Checkbox Nur Summe angegeben werden. Für die Zuordnung der Konten benötigt die jeweilige Stufe (Erlöse/Kosten) ein Detail. Über markieren der entsprechenden Stufe und Klick auf Detail wird dieses hinzugefügt und kann benannt werden. Hier können Sie jetzt alle Erlös- bzw. Kostenkonten von rechts nach links in die Baumstruktur schieben, so dass diese dem entsprechenden Detail zugeordnet werden. 
### Combobox Rechn.leg.kreis 
Hier kann der passende Rechnungslegungskreis ausgewählt werden. 

Tooltip: 

Angabe des Rechnungslegungskreises 

### Feld Zeilentext 
In diesem Feld legen Sie den Namen für eine Stufe, einen Text oder ein Detail fest. 

Tooltip: 

Zeilentext 

### Feld Summentext 
In diesem Feld können individuelle Summentexte eingegeben werden. Wird hier nichts eingetragen, steht in der Auswertung (im Dialog und im Report) kein Text beim Ergebnis. 

Tooltip: 

Der Summentext wird in der Auswertung in Spalte Bezeichnung auf der Summenzeile ausgewiesen. 

### Button Hinzufügen 

Tooltip: 

Automatisches Füllen des Feldes Summentext. 

### Feld 100% Basis 
In diesem Feld haben Sie die Möglichkeit bis zu 5 Stufen (1 bis 5) als Basen festzulegen. In der Saldenauswertung ( Buchen >> Auswertungen >> Sachkonten ) erzeugen Sie hiermit je Stufe 1-5 Vergleichsspalten. In jeder der Vergleichsspalten werden alle Auswertungsstufen auf die jeweilige 100%-Basis berechnet. 

### Feld 100% Basistext 
Hier wird der Text der gewählten Stufe angegeben. 

Tooltip: 

Basis festlegen 

### Button Suche 

Wenn Sie diesen Button drücken, wird immer die nächste Zeile mit einer Basis in der Auswertungs-Struktur gesucht. Ist die Suche unten angekommen, wird von oben gesucht. 

Tooltip: 

Zur nächsten 100% Basis springen 

### Checkbox MIS Auswertung 
Wenn Sie diese Checkbox aktivieren, können Sie den einzelnen Auswertungsstufen Kennziffern für das MIS zuordnen. Dazu steht Ihnen die Combobox MIS Referenz zur Verfügung. 

Tooltip: 

Angabe der MIS-Auswertung 

### Button Öffnen 

Dieser Button öffnet die MIS Referenz Texte. 

Siehe Kapitel MIS Referenz Texte . 

### Radiobutton Stufe 
Innerhalb einer Stufe werden alle Werte der Konten in der Auswertung zu einem Ergebnis zusammengerechnet. 

Sie können beliebig viele Stufen ineinander verschachteln. 

### Radiobutton Text 
Diese Positionen im Strukturaufbau sind reine Texte und dienen dem Layout der Auswertung. 

### Radiobutton Detail 
Nur einem Detail können Konten zugeordnet werden! 

D.h. wenn Sie einer Stufe Konten zuordnen wollen, müssen Sie zuvor innerhalb der Stufe ein Detail einfügen. 

### Radiobutton Basis 
Die Basis bilden die Konten. Anhand ihrer Werte erfolgen die Berechnungen innerhalb einer Stufe. 

### Combobox MIS Referenz 
Hier können Sie einen MIS Referenz Text auswählen. 

Siehe Kapitel MIS Referenz Text . 

Tooltip: 

MIS Referenznummer 

### Tabelle Konten 

In dieser Tabelle werden alle Konten angezeigt, die im Sachkontenstamm auf der Registerkarte Abschluss als Bilanz-Konto gekennzeichnet sind. Alle Konten, die noch in der Tabelle stehen, sind in der Auswertungsstruktur nicht zugeordnet. Die Tabelle ist numerisch sortiert, kann jedoch auch durch Klick auf den Spaltennamen nach weiteren Spalten sortiert werden. 
### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Tabellenspalte Bilanz / G+V 
Hier wird die Zuordnung des Sachkontos zu Bilanz oder G&V angegeben. Gültige Werte sind: 
B - Bilanzkonto 
G - G&V - Konto 
Nur gültig, wenn Feld Typ = 'S' (Sachkonto) 

Wenn B, kann das Konto auch OP-führend (Opfuehrend=1) sein. 

Tooltip: 

Sachkonto für Bilanz oder G&V 

### Checkbox Check Zuordnungen 
Wenn diese Checkbox aktiviert ist, wird im Dialog Saldenauswertung für diese Auswertung geprüft, ob alle Konten zugewiesen sind. 

Tooltip: 

Prüfe auf vollständige Zuordnung aller Konten 

### Feld Konto 
In diesem Feld können Sie eine Kontonummer eingeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Mit diesem Button sucht eNVenta ERP das eingegebene Konto im Strukturaufbau und in der Kontentabelle und markiert es. Ist die Platzierung außerhalb des Anzeigebereichs, muss manuell hin geblättert/gescrollt werden. 

Tooltip: 

Suche Konto 

### Dock Panel Zuordnung 

Sind im Bereich Zuordnung beide Unterstufen mit dem Radiobutton Addieren gekennzeichnet, ermittelt sich das Ergebnis aus den Erlösen, abzüglich der gewährten Skonti. Wird bei der zweiten Unterstufe der Radiobutton Subtrahieren gesetzt, ermittelt sich das Ergebnis aus den Erlösen, zuzüglich der gewährten Skonti. 

Für den Aufbau einer Auswertung, deren Ergebnis nur auf Grund der Saldenwerte ermittelt werden soll, werden alle Stufen mit dem Radiobutton Addieren gekennzeichnet.
