# Parameter - Auftragsimport 
Parameter - Auftragsimport 

Dieser Dialog öffnet sich im Gateway >> Optionen >> Parameter - Auftragsimport . Hier werden die Parameter für das Anlegen der Aufträge aus ORDERS-Import hinterlegt. 

Es ist möglich, verschiedene Profile zu hinterlegen und somit auch für einzelne Referenzen unterschiedliche Einstellungen zu machen. 

Siehe Kapitel EDI-Teilnehmer . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Parameter - Auftragsimport 
Tooltip: 

In diesem Dialog werden die Parameter für das Anlegen der Aufträge aus ORDERS-Import hinterlegt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Tabelle 

### Tabellenspalte Import Profil 
In dieser Spalte geben Sie dem neuen Profil einen Namen. 

### Tabellenspalte Fehlende Preise ergänzen 
Wenn Sie die Checkbox in dieser Spalte setzen, wird bei nicht vorhandenen Preisen eine Preisermittlung durchgeführt. 

### Tabellenspalte Preise u .Kond erm. 
Wenn Sie die Checkbox in dieser Spalte aktivieren, werden die Preise immer neu ermittelt und nicht 1:1 aus der Import-Datei übernommen. 

### Tabellenspalte EK ergänzen 
Wenn Sie die Checkbox in dieser Spalte aktivieren, werden die Einkaufspreise aus dem System in den generierten Auftrag übernommen. 

### Tabellenspalte Artikel Verfügbarkeitsprüfung 
Wenn Sie die Checkbox in dieser Spalte nicht setzen, wird eine Verfügbarkeitsprüfung unterbunden. Diese Checkbox ist standardmäßig zu setzen, da es in den meisten Fällen gewünscht ist, dass eine Verfügbarkeitsprüfung stattfindet.
