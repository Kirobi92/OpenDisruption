# X Mapping Test Registerkarte Export 
X Mapping Test Registerkarte Export 

Für den Export müssen Sie auf dieser Registerkarte zuerst Export-Daten bereitstellen, z.B. per SQL-Suche oder/auch indem die Eigenschaften manuell auf einen bestimmten Wert gesetzt werden. Oder bei ORDERS Export z.B. eine Bestellung erzeugen und per Drag&Drop in den Test ziehen. Dies ist in der Registerkarte Daten vorzunehmen. 
### Button Format 

Über den Button Format kann die generierte Datei formatiert angezeigt werden, dies dient der besseren Lesbarkeit der Daten.
