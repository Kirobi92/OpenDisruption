# Kapitel NV.ERP.MM.Sales.cdOrderType.shtNoCostAcc

Mit diesem Parameter kann pro Auftragsart festgelegt werden, ob der Wareneinsatz an die Schnittstelle der Kostenrechnung übergeben werden soll. 

Wird die Checkbox aktiviert, so werden für die Auftragspositionen weiterhin die Kostenstellen und / oder Kostenträger an die Fibu und somit auch an die Kostenrechnung übergeben. 

Ist die Checkbox Int. Warenabgang auf der Registerkarte Interner Warenabgang gesetzt, ist diese Checkbox zwingend zu setzen! 

Tooltip: 

Wenn aktiviert, wird der Wareneinsatz an die Schnittstelle der Kostenrechnung übergeben. 
Der Schalter 'Keine Sollstellung' wirkt sich jetzt nur noch auf die Bereitstellung der Daten für die Fibu-Schnittstelle (Belegjournal) aus.
