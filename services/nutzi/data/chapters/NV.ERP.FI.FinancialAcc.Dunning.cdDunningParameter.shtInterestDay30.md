# Kapitel NV.ERP.FI.FinancialAcc.Dunning.cdDunningParameter.shtInterestDay30

Wenn Sie diesen Parameter setzen, wenn die Mahnzinsen erst ab dem 30. Tag der Fälligkeit berechnet werden. 

Tooltip: 

Wenn Sie diesen Parameter setzen, werden die Mahnzinsen erst ab dem 30. Tag der Fälligkeit berechnet.
