Grundeinstellungen für eNVenta ERP / IKOMSYS - Schnittstelle EUROBAUSTOFF 

# IKOMSYS - Schnittstelle EUROBAUSTOFF Registerkarte Zentrallagerbestellung 
IKOMSYS Registerkarte Zentrallagerbestellung 

Auf dieser Registerkarte nehmen Sie weitere Einstellungen für das Versenden der Bestellung an das EBZL vor. 
### Field Panel Wiedervorlagen 

In diesem Bereich legen Sie fest, an welchen Sachbearbeiter eine Wiedervorlage erzeugt werden soll. 
### Combobox Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Checkbox WV an Erfasser 
Wenn Sie diese Checkbox aktivieren, erhält der Erfasser des Datensatzes eine Wiedervorlage. 

Tooltip: 

Wiedervorlage an Erfasser der Bestellung. 

### Checkbox Nur Fehler 
Wenn Sie diese Checkbox aktivieren, erhält der Sachbearbeiter nur bei Fehlern eine Wiedervorlage. 

Tooltip: 

Wiedervorlage wird nur bei aufgetretenen Fehlern erstellt. 

### Tabelle Lager 

In dieser Tabelle ordnen Sie eine EBS-Kundennummer einem Lager zu. Mehrere Kunden können einem Lager zugeordnet werden. 
### Tabellenspalte Lager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte EBS-Kunde 
Hier wird die EUROBAUSTOFF-Kundennummer angegeben. 

Tooltip: 

EUROBAUSTOFF-Kundennummer 

### Button Satz hinzufügen 

Mit diesem Button können Sie der Tabelle einen weiteren Eintrag hinzufügen. 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag aus der Tabelle. 

### Tabelle Lieferanten 

In dieser Tabelle ordnen Sie einen Lieferanten einem Zentrallager zu. Diese Lager werden in einer Pickliste gepflegt. 
### Tabellenspalte Zentrallager 
Diese Spalte enthält die Mandanten ID. 

Tooltip: 

EUROBAUSTOFF Lagernummer 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Lieferant 
Die Firmenbezeichnung wird automatisch vom System eingetragen, nachdem die Lieferantennummer eingetragen wurde. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Button Satz hinzufügen 

Mit diesem Button können Sie der Tabelle einen weiteren Eintrag hinzufügen. 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag aus der Tabelle.
