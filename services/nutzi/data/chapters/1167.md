# Preispflege Registerkarte Tabelle 
Preispflege Registerkarte Tabelle 

Diese Registerkarte zeigt alle vorhandenen Datensätze zu den im Kopf des Dialogs selektierten Einstellungen an. 

### Tabelle 

### Tabellenspalte Preispflegetyp 
Hier wird der Preispflegetyp angezeigt. 

### Tabellenspalte Status 
Hier wird der Name des Status angezeigt. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Artikel von 
Hier wird die Von-Artikelnummer angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Artikel bis 
Hier wird die Bis-Artikelnummer angezeigt. 

Tooltip: 

Angabe der Bis-Artikelnummer 

### Tabellenspalte Artikelgrp. von 
Hier wird die Von-Artikelgruppe angezeigt. 

Tooltip: 

Angabe der Artikelgruppe 

### Tabellenspalte Artikelgrp. bis 
Hier wird die Bis-Artikelgruppe angezeigt. 

Tooltip: 

Angabe der Artikelgruppe 

### Tabellenspalte Ab Stichtag 
In dieser Spalte wird der festgelegte Stichtag angezeigt. 

### Tabellenspalte Bis Stichtag 
Wenn ein Datumsbereich als "Stichtag" angelegt ist, wird hier das Bis-Datum angezeigt. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Dieses Feld zeigt die Anzahl der tatsächlich eingelesenen Datensätze.
