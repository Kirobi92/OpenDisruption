# Automatischer Druck mit Referenz Links 
Auto Druck Ref Links 

Um mit Hilfe von Referenz-Links automatisch Dokumente zu drucken, gibt es unter System >> Allgemein das Menü Auto Druck Referenz Links . 

In diesem Dialog nehmen Sie die Einstellungen dafür vor: zusammen mit einem Standardformular aus Verkauf, Einkauf oder Produktion (Druck von Betriebsaufträgen) können verlinkte Dokumente mit gedruckt oder versendet werden. Die jeweils hinterlegten Referenz-Links werden dann immer anschließend an den normalen Druck auf dem in der Auto Druck Konfiguration hinterlegten Drucker gedruckt. 

Es werden auch angepasste Reports im Dialog zur Definition des Auto-Drucks von Referenz Links angezeigt und können für den automatischen Ausdruck definiert werden.

### Tabelle Dokumente 

### Tabellenspalte Einkauf/Verkauf 
Der Eintrag in dieser Spalte dient hauptsächlich zur Steuerung der restlichen Einstellungen. Hier kann angegeben werden, für welchen Bereich eine automatische Druckregelung hinterlegt werden soll. 

Es stehen die Bereiche Einkauf, Produktion, Verkauf und Retoure zur Verfügung. 

### Tabellenspalte Reportname 
Hier kann angegeben werden, bei welchem Report Referenzdokumente mit gedruckt werden sollen. Welche Reports hier angeboten werden, hängt von der Auswahl in der Spalte 'Einkauf/Verkauf' ab. Je nach Einstellung werden unterschiedliche Reports angeboten. Es werden dabei alle Reports der Standard-Schnittstelle zum Druck der Verkaufs- bzw. Einkaufsbelege angezeigt. 

### Tabellenspalte Link-Typ 
Hier kann der Referenz-Link-Typ angegeben werden, welcher mit dieser Regel gedruckt wird.

Hier wird der Typ der Referenz-Links angegeben, welcher automatisch mit gedruckt werden soll. So können Sie einen eigenen Referenz-Typ „Werkszeugnis“ erstellen, damit dieser dann entsprechend mit gedruckt wird, wenn in einem Auftrag enthalten.

Angabe des Referenz-Link-Typs 

### Tabellenspalte Belegart 
Soll die Regel nur im Fall von bestimmten Belegarten (Einkaufs- / Auftragsarten) ziehen, so kann dies hier festgelegt werden. Die angebotenen Möglichkeiten werden, wie schon bei der Reportauswahl, durch die Auswahl in der Spalte 'Einkauf/Verkauf' gesteuert. 

### Tabellenspalte AKL-Typ 
Soll die Einstellung nur bei bestimmten Kunden/ Adressen / Lieferanten ziehen, so kann das hier eingestellt werden. Wenn eine Eingabe erfolgt, muss auch die Spalte 'AKL-Nummer' gefüllt werden. Die beiden Spalten werden beim Speichern des Vorgangs im Falle von Neuanlage und Änderung geprüft. Dabei wird kontrolliert, ob zur angegebenen Kombination ein gültiger Datensatz existiert. Ansonsten wird eine Fehlermeldung ausgegeben.

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Tabellenspalte Drucker 
Hier ist der Drucker eingetragen, an den das Formular gesendet wurde. 

Diese Kriterien können beliebig festgelegt werden. Allerdings müssen mindestens die Angaben Einkauf/Verkauf, Link-Typ und Formularname gemacht werden. 

Die Verlinkung erfolgt auf Beleg- oder Positionsebene. Bei Teillieferungen etc. werden nur Links der beteiligten Positionen berücksichtigt. 

Tooltip: 

Angabe des (Standard-)Druckers
