# Kundenstatistik Registerkarte Statistik 
Kundenstatistik Registerkarte Statistik 

Diese Tabelle gibt Auskunft über die Umsätze aus dem gewählten Zeitraum und die prozentualen Veränderungen gegenüber dem Vorjahreszeitraum. 

Die Anzeige der Prozentwerte erfolgt mit 2 Nachkommastellen. 
### Feld Titel 
Hier wird der Titel der Leistungsstatistik angegeben. 

Tooltip: 

Angabe des Titels der Leistung 

### Feld Von / Bis 
Dieses Feld zeigt das Von/Bis-Datum des gewählten Zeitraums. 

Tooltip: 

Von-Bis-Datum 

### Button Nach oben skalieren 

Aus der Statistik eines Monats / einzelner Kalenderwochen kann mit diesem Button wieder hochskaliert werden, wenn zuvor nach unten skaliert wurde. 

Tooltip: 

Wenn Sie diesen Button drücken, können Sie die Statistik wieder auf eine höhere Ebene skalieren (wenn diese zuvor herunterskaliert war). 

### Button Verschieben 

In der Statistik (Artikel-, Lieferanten- und Kundenstatistik) kann die Statistik für ein bestimmtes Jahr generiert werden. 

Es kann ein bestimmter Monat ausgewählt werden. Wenn Sie dann diesen Button drücken, können Sie sich ihre Statistik auf den einzelnen Monat herunterskalieren. Es wird Ihnen die Statistik einzelner Kalenderwochen innerhalb des gewählten Monats angezeigt. Daraus wiederum kann in eine einzelne Woche herunterskaliert werden. 

Tooltip: 

Wenn Sie diesen Button drücken, können Sie die Statistik (eine Ebene darunter) genauer betrachten.
