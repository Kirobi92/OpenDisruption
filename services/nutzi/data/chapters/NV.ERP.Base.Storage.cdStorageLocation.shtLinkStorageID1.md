# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtLinkStorageID1

Wenn Sie den Parameter Link-Lager aktiviert haben, wählen Sie hier ein Link-Lager aus. 

Über die Link-Lager 1-3 können bis zu 3 abweichende Qualitätsstufen definiert werden! 

Link-Lager werden für die Einlagerung von Artikeln mit unterschiedlichen Qualitätsstufen eingerichtet. Dies sind virtuelle Lager. Bei einer zu geringen Verfügbarkeit eines Artikels kann dann auf eine der anderen Qualitäten zurückgegriffen werden. 

Tooltip: 

Definiert das Link-Lager
