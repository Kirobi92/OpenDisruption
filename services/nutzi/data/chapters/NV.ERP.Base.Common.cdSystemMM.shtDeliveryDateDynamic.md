# Kapitel NV.ERP.Base.Common.cdSystemMM.shtDeliveryDateDynamic

Wenn Sie diesen Parameter aktivieren, wird das Lieferdatum im Auftrag dynamisch aus dem Tourenplan ermittelt. Dafür gibt es in Angebot (Registerkarte Anschriften ), Auftrag (Registerkarte Anschriften ) und Kunde (Registerkarte Logistik ) die Combobox Tourzyklus . 

Beim Kopieren eines Angebots in einen Auftrag wird der Tourzyklus übergeben. 

Die Stammdaten für die Anschriften und der Tourenstamm müssen gepflegt sein. Ansonsten ist das Lieferdatum statisch. D.h. es basiert auf der Eingabe.
