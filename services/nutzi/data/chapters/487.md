# Auswahl logischer Archive 
Ausw. Logisches Archiv

## Steuerelemente des Formulars 

### Bezeichnungsfeld Ausw. Logisches Archiv 
Tooltip: 

In diesem Dialog kann ein logisches Archiv ausgewählt werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Button Neues Archiv 

Mit diesem Button können Sie ein logisches Archiv neu erstellen, um dieses dann verwenden zu können. Das neue logische Archiv verwendet als Basis das zuvor angegebene physische Archiv mit dessen Schlagwörtern. 

Wird ein logisches Archiv selektiert, werden automatisch die fehlenden logischen Schlagwörter hinzugefügt und den physikalischen Schlagwörtern zugeordnet. Dabei werden auch die bereits vorhandenen logischen Schlagwörter geprüft. Falls es bereits das "korrekte" logische Schlagwort gibt (d.h. der Name und der Datentyp stimmen mit dem physikalischen Schlagwort überein) und dieses noch keine Zuordnung zum betroffenen physikalischen Archiv hat, wird dieses Schlagwort verwendet (gemappt) und kein neues erzeugt. 

Tooltip: 

Legen Sie ein neues logisches Archiv an.
