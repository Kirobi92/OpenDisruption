# Daten nach Excel exportieren

Allgemeine Funktionalitäten in eNVenta ERP / Datentransfer (Import und Export von Daten) 

# Daten nach Excel exportieren 
Wenn Sie aus einer beliebigen Tabelle einen Export der Daten zu Microsoft Excel vornehmen möchten, markieren Sie die Zeile und klicken dann mit der rechten Maustaste auf einen Spalteneintrag. Es öffnet sich das Kontextmenü. Hier klicken Sie auf den Eintrag „Datenübergabe an MS Excel / OO Calc“ oder „Datenübergabe an MS Excel / OO Calc mit Dezimalsummen“. So werden die Daten der Tabelle einmal mit und einmal ohne Dezimalsummen übergeben. 

Anschließend öffnet sich Microsoft Excel und zeigt den Inhalt der gesamten Tabelle an. Der Name des exportierten Dialogs inkl. Datenbank und Jahr wird angezeigt, die Tabelle inkl. Überschrift exportiert. Dieser Export kann je nach Datenmenge eine Weile dauern. 

Um den Datenexport sehr großer Datenmengen nach Excel zu ermöglichen, sind folgende Scripts auszuführen: 

INSERT INTO Gw_DataExportComponents (BranchKey, ComponentName, ComponentType) 
VALUES (<BK>, 'OP-Debitoren Export', 'NV.ERP.FI.FinancialAcc.OpenItem.cOpenItemCustomerColl'); 

INSERT INTO Gw_DataExportComponents (BranchKey, ComponentName, ComponentType) 
VALUES (<BK>, 'OP-Kreditoren Export', 'NV.ERP.FI.FinancialAcc.OpenItem.cOpenItemSupplierColl'); 

Das Skript ist auf die Komponente bezogen und muss so individuell (in der Regel von unserem Consulting) angepasst werden. 

Anschließend kann der Datenexport über das Gateway angestoßen werden. 

Während der Export an Excel läuft, darf der Dialog / das Formular nicht geschlossen werden, die Übertragung würde in diesem Fall abbrechen. Um dies zu verhindern wird während des Exports die Sanduhr angezeigt. 

Die Summen aller Spalten werden mit Dezimal-Werten berechnet und unterhalb der letzten Zeile eingefügt. Das Wort „Summe“ wird in die erste Spalte eingetragen. Wenn hier Dezimal-Werte eingetragen sind, wird deren Summe gebildet. 

Das Wort „Summe“ und die Summen selbst werden fett dargestellt. 

Es finden keine besonderen Berechnungen statt, Excel summiert hier nur die Beträge automatisch auf. 

Ebenso können Sie den Inhalt einer einzelnen Zeile nach Excel exportieren. Dazu markieren Sie den Inhalt einer Zeile mit <Strg + C> (rechte Maustaste) und platzieren den Cursor in der Excel-Tabelle in die entsprechende Spalte und setzen den Zeileninhalt mit <Strg + V> ein.
