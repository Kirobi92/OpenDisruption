# Historisierungsübersicht Registerkarte Allgemein

Administration / Daten Historisierung / Historisierungsübersicht 

# Historisierungsübersicht Registerkarte Allgemein 
Historisierung Übersicht Registerkarte Allgemein 

Auf der Registerkarte Allgemein werden die Tabellen angezeigt, welche historisiert werden können. Wenn Sie eine Zeile markieren, werden im Kopf des Dialogs die entsprechenden Felder gefüllt. 
### Tabelle 

### Tabellenspalte Aktivieren 
Wenn diese Checkbox aktiviert ist, wird die Historisierung für diese Tabelle ausgeführt. 

Tooltip: 

Wenn dieser Parameter aktiviert ist, wird die Historisierung für diese Tabelle ausgeführt. 

### Tabellenspalte Tabelle 
Hier wird der technische Name der Tabelle angegeben. 

Tooltip: 

Technischer Name der Tabelle 

### Tabellenspalte Komponente 
Hier wird der technische Name der Helper-Komponente angegeben. 

Tooltip: 

Der technische Name der Helper-Komponente. 

### Tabellenspalte Komponentenname 
Hier wird der Name der Komponente angegeben. 

Tooltip: 

Der Name der Komponente wird angegeben. 

### Tabellenspalte Anzahl Produktiv 
Hier wird die Anzahl der produktiven Datensätze angegeben. 

Tooltip: 

Anzahl der produktiven Datensätze 

### Tabellenspalte Anzahl Historie 
Hier wird die Anzahl der historisierten Datensätze angegeben. 

Tooltip: 

Anzahl der historisierten Datensätze 

### Tabellenspalte Datum bis 
Bis zu dem hier angegebenen Datum ist definiert, dass jüngere Datensätze im Produktivteil sind, die älteren in der Historie. 

Tooltip: 

Bis zu diesem Datum ist definiert, dass jüngere Datensätze im Produktivteil sind, ältere in der Historie. 

### Tabellenspalte Transferdatum 
Datensätze, welche ein Datum haben das älter ist als dieses Datum, wurden bereits in die Historie ausgelagert. 

Tooltip: 

Datensätze welche älter sind, wurden bereits in die Historie ausgelagert. 

### Tabellenspalte Historienfeld 
Hier wird definiert, welches Datumsfeld zur Historisierung herangezogen wird (Erfassungsdatum, Änderungsdatum). 

Tooltip: 

Definiert, welches DB-Datumsfeld zur Historisierung herangezogen wird (Erfassungsdatum, Änderungsdatum). 

### Tabellenspalte Null historisieren 
Wenn diese Checkbox aktiviert ist, werden NULL-Datumswerte berücksichtigt und historisiert. 

Tooltip: 

Wenn die Checkbox gesetzt ist, werden NULL Datumswerte berücksichtigt und historisiert. 

### Tabellenspalte Where-Bedingung 
Hier werden nur die Datensätze angegeben, welche dieses Kriterium erfüllen. Diese werden dann zur Historisierung zur Verfügung gestellt. 

Tooltip: 

Nur Datensätze, welche dieses Kriterium erfüllen, stehen zur Historisierung zur Verfügung. 

### Tabellenspalte Plausibilitätsprüfung 
Daten, welche der hier eingestellten Prüfung entsprechen, werden übernommen. 

Tooltip: 

Die Daten werden unter Berücksichtigung der eingestellten Prüfung übernommen. 

### Tabellenspalte Protokollierung 
Hier wird die Protokollierung angegeben. 

Tooltip: 

Es kann eine Protokollierung konfiguriert werden. 

### Tabellenspalte Datenverbindung 
Hier wird die Datenverbindung angegeben. 

Tooltip: 

Angabe der Datenverbindung
