# ADR Info 
ADR Info 

Der Button ADR Info im Dialog VK-Positionen bearbeiten auf der Registerkarte Logistik öffnet den Dialog ADR Info . 

In diesem Dialog werden sämtliche Informationen zum Gefahrgut angezeigt, wie z.B. die UN-Nummer und die verwendete Gefahrgutbeschreibung. Die Daten stammen aus der ADR Tabelle A . 

Bei Aufruf des Dialogs im Auftrag über Menü Optionen >> Auskunft >> ADR Info erfolgt die Anzeige gruppiert nach UN-Nummer. 

Der Dialog ADR Info kann außerdem aus Auftragsposition, Auftrag, Packstücke, Kommissionieren und Tourendisposition aufgerufen werden.

### Tabellenspalte Beförderungskategorie 
Hier wird die Ziffer angegeben, welche die Beförderungskategorie angibt, der der Stoff/Gegenstand für Zwecke der Freistellungen in Zusammenhang mit Mengen, die je Beförderungseinheit befördert werden, zugeordnet ist. 

- 0 = Höchstzulässige Gesamtmenge je Beförderungseinheit = keine Freistellung erlaubt (0kg) - 1 = Höchstzulässige Gesamtmenge je Beförderungseinheit = 20 kg bzw. 50 kg für die UN-Nummern 0081, 0082, 0084, 0241, 0331, 0332 und 0482 - 2 = Höchstzulässige Gesamtmenge je Beförderungseinheit = 333 kg - 3 = Höchstzulässige Gesamtmenge je Beförderungseinheit = 1000 kg - 4 = Höchstzulässige Gesamtmenge je Beförderungseinheit = unbegrenzt 

Tooltip:

- 0 = Höchstzulässige Gesamtmenge je Beförderungseinheit = keine Freistellung erlaubt (0kg) 
- 1 = Höchstzulässige Gesamtmenge je Beförderungseinheit = 20 kg bzw. 50 kg für die UN-Nummern 0081, 0082, 0084, 0241, 0331, 0332 und 0482 
- 2 = Höchstzulässige Gesamtmenge je Beförderungseinheit = 333 kg 
- 3 = Höchstzulässige Gesamtmenge je Beförderungseinheit = 1000 kg 
- 4 = Höchstzulässige Gesamtmenge je Beförderungseinheit = unbegrenzt 

### Tabellenspalte Tunnelbeschränkungscode 
Hier wird der Code angegeben, der darstellt, welche Tunnel der Transport nicht durchfahren darf. 

- A = keine Beschränkung für gefährliche Güter - B = Durchfahrtsverbot mit gefährlichen Gütern, die zu einer sehr großen Explosion führen können. - C = gleiche Einschränkung wie bei B plus gefährliche Güter, die zu einer großen Explosion oder einem umfangreichen Freiwerden giftiger Stoffe führen können. - D = entspricht C zusätzlich gefährlicher Güter, die zu einem großen Brand führen können. - E = Durchfahrtsverbot für alle gefährlichen Güter, mit Ausnahmen von UN2919, UN3291 und UN3373
