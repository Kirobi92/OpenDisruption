# Kapitel NV.ERP.Base.Common.cdSystemMM.shtAllowInventoryWorthCorrFix

Tooltip: 

Falls gesetzt, dann können Inventurbewertungen zu entsprechenden Wertkorrektur-Buchungen im Lager führen. 
Falls nicht gesetzt, so finden niemals Wertkorrektur-Buchungen statt. 
Außerdem bezieht sich dann eine Bewertung einer Inventurposition auf ihren Lagerbestand anstatt auf ihre Ist-Menge.
