# Materialposition 
Materialposition 

Über den Button Ändern im Betriebsauftrag auf der Registerkarte Positionen öffnet sich das Menü Materialposition . Hier können Sie die Werte eines bereits eingefügten Artikels ändern. 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern!

### Checkbox Kleinteil 
Wenn Sie diesen Parameter aktivieren, kennzeichnen Sie den Artikel als Kleinteil und er wird auf ein Lager für Kleinteile eingelagert. 

Organisatorisch sollte das Kleinteilelager in der Nähe der Produktion liegen, damit weite Wege vermieden werden, um z.B. kurzfristig Unterlagscheiben zu beschaffen. 

Damit haben Sie die Möglichkeit Material für einen Betriebsauftrag von mehreren Lagern zu entnehmen. 

So sparen Sie sich die Umlagerung für die Produktion vom Materiallager zum Produktionslager und das Kleinteilelager ist immer gefüllt. Dies ist sichergestellt über den Bestellautomaten, den Bestellvorschlag etc. 

Von diesem Kleinteilelager wiederum kann dann nur eine begrenzte Anzahl an Kleinteilen geholt werden, damit es nicht sofort leer ist. 

Siehe Kapitel Flexibles Produktionslager . 

Tooltip: 

Ein Artikel kann als Kleinteil definiert werden. 
Damit sparen Sie sich die Umlagerung vom Materiallager zum Produktionslager.
