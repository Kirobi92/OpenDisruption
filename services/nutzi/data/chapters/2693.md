# Planwerte ändern 
Planwerte ändern 

Dieser Dialog öffnet sich im Buchen Kore >> Buchen >> Planwerte ändern . Hier kann auf Kostenarten-, Leistungsarten- bzw. Kostenstellenebene die Planung eingegeben oder geändert werden. Die Selektion hierfür erfolgt über die Radiobuttons Kostenarten, Leistungen, Kostenstellen . Danach geben Sie entsprechend die Nummer ein oder suchen sie mit dem -Button. Haben Sie den Radiobutton Kostenarten oder Leistungen ausgewählt, so öffnet sich mit dem -Button entweder der Dialog Kostenartenauswahl oder der Dialog Kostenstellenauswahl . 

Neben dem Feld Nummer wird das Jahr ausgewählt, für welches die Planwerte geändert werden sollen. 

Mit dem Speichern der geänderten Planwerte, sind die Daten in das Kore_Journal gebucht. 

Grundsätzlich gilt sowohl beim Anlegen als auch beim Ändern von Planwerten, dass danach die Umlagen erneut gebucht werden müssen, damit die Umlagen auf dem korrekten (aktuellen) Stand sind.
