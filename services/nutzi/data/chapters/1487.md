# Einlage 
Einlage 

Dieser Dialog öffnet sich im Kassieren >> Kassieren >> Einlage und ermöglicht das Hinzufügen von Bargeld jeder Währung zur Kasse. Der Betrag kann in der Stückelung oder als Gesamtbetrag angegeben werden. 

Die Felder Gegenkonto, Steuerschlüssel, Kostenstelle, Kostenträger und Projekt spielen auch eine Rolle bei der automatischen Verbuchung bzw. dem CSV-Export. Die Anzeige dieser Felder hängt mit dem Wert zusammen, der beim Einlagegrund (z.B. Trinkgeld, Abschlagszahlung und Rückgeld aus Auszahlungen) eingestellt wird. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Einlage 
Tooltip: 

In diesem Dialog können Sie Bargeld jeder Währung zur Kasse hinzufügen.

### Feld Kassierer 
Hier wird der Kassierer angezeigt. 

Ist der Sachbearbeiter freigegeben, kann in diesem Feld der Sachbearbeiter geändert werden. Ansonsten ist dieses Feld nicht editierbar. 

### Combobox Währung 
In dieser Combobox kann die Währung, für welche eine Einlage durchgeführt werden soll, ausgewählt werden. Je nach Auswahl der Währung werden die Geldeinheiten eingelesen. 

Tooltip: 

Angabe der Währung 

### Combobox Einlagengrund 
In diesem Feld muss ein Grund für eine Einlage angegeben werden, damit diese vernünftig verbucht werden kann. Es werden dabei alle Einlagefälle angeboten. 

### Feld Verw.zweck 
Um eine Geldeinlage besser „verstehen“ zu können, geben Sie in dieses Feld einen Buchungstext ein (mit max. 80 Zeichen; kein Pflicht-Feld!). Der Text wird an die Finanzbuchhaltung über die Auszug-/Kassenimport -Funktion übergeben, damit die Daten immer nachvollziehbar bleiben. 

Bei der Übergabe der Daten an die Finanzbuchhaltung wird der jeweilige Verwendungszweck angegeben, damit beim Ausgleich der Buchungen der konkrete Grund bekannt ist und nicht manuell zusätzliche Papiere an die Finanzbuchhaltung übergeben werden müssen. 

Tooltip: 

Angabe des Verwendungszwecks
