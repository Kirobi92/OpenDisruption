# Daten Umsetzung Registerkarte Manuelle Umsetzung 
Daten Umsetzung Registerkarte Manuelle Umsetzung 

Auf dieser Registerkarte können die genannten 1:1 Umsetzungen manuell durch Eingabe in die Spalten ‚Externer Wert‘ und ‚Interner Wert‘ hinterlegt werden. 
### Button Auswählen 

Mit diesem Button übernehmen Sie eine Vorlage aus der Datenumsetzung. 

Tooltip: 

Aus Datenumsetzungsvorlagen übernehmen 

### Button Satz hinzufügen 

Mit diesem Button können Sie der Tabelle eine neue Zeile hinzufügen. 

### Button Löschen 

Mit diesem Button können Sie eine markierte Zeile der Tabelle löschen.
