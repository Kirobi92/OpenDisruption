# Kommissionieren Registerkarte Zusatzfelder 
Registerkarte Zusatzfelder 

Auf dieser Registerkarte stehen u.a. die Felder Code 1 – 5 zur Verfügung. Diese Felder können individuell für das Reporting genutzt werden. 
### Feld Code 1 
Dieses Feld steht zur individuellen Eingabe zur Verfügung. Es kann per Reportprogrammierung eingebunden werden, hat aber keine Funktionalität.

### Checkbox Stückliste anzeigen 
Um Stücklisten besser erkennen zu können, können Sie die Kopfposition einer Stückliste anzeigen lassen. Dazu aktivieren Sie diese Checkbox. 

Dabei ist zu beachten: Wenn Sie den Stücklistenkopf markieren, werden alle Bearbeitungs-Buttons für diese Position deaktiviert. 

Beim Druck des Lieferscheins aus der Kommissionierung heraus wird die Checkbox entsprechend geprüft und der Lieferschein enthält zusätzlich zu den Komponenten auch den Stücklisten-Kopf. 

Tooltip: 

Stücklistenkopf beim Laden anzeigen.
