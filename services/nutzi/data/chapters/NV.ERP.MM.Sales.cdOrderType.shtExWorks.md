# Kapitel NV.ERP.MM.Sales.cdOrderType.shtExWorks

Wenn dieser Parameter aktiviert ist, dürfen mit dieser Auftragsart nur Artikel geliefert werden, die direkt vom Werk kommen. 

Die Kennzeichnung ab Werk kann einem Artikel auch bereits im Artikelstamm (Registerkarte Verkauf ) zugewiesen werden.
