# Marketing-Aktion Registerkarte Allgemein 
Marketing-Aktion Registerkarte Allgemein 

Auf dieser Registerkarte werden die Aktionsschritte zur Marketing-Aktion hinterlegt. Dazu klicken Sie auf den Button Hinzufügen . Dieser Button öffnet den Dialog Aktivität Auswahl . Hier wählen Sie nacheinander die gewünschten Aktivitätsschritte aus. 

Die gewählte Aktivität wird in die Tabelle eingefügt und erhält automatisch eine fortlaufende Nummer, die in der Spalte ‚Aktivitätsnummer’ eingetragen wird. Aktionsschritte, die bereits unterhalb der markierten Zeile vorhanden sind, werden um einen Schritt nach hinten verschoben. 

In der Tabelle kann dann noch der Haken in der Spalte 'Wiedervorlage' gesetzt werden, um sich selbst (zur Erinnerung) oder einem anderen Sachbearbeiter eine Wiedervorlage zu senden. 

Mit einem Doppelklick auf einen Eintrag der Tabelle öffnet sich der Dialog Aktivität-Teilnehmer . Dort können Sie auf der Registerkarte Allgemein im Feld Information einen freien Text einfügen, wie z.B. "am 15.12. telefonisch erreicht; vor einer weiteren Kontaktaufnahme erst einmal das Video senden, dann nach-telefonieren". 

Wenn alle Informationen gefüllt sind, schließen Sie für den Teilnehmer die Aktivität ab. 

Ist eine Aktivität abgeschlossen und Sie öffnen die Marketing-Aktion nochmals, sind die "abgeschlossenen" Teilnehmer nicht mehr in der Tabelle zu sehen; ihr Aktionsschritt steht auf "Aktionsende". 

Siehe Kapitel Aktivität - Teilnehmer . 
### Button Hinzufügen 

Dieser Button öffnet den Dialog Aktivität Auswahl . Hier kann gezielt nach Aktivitätstypen oder Beschreibungen gesucht werden. Anschließend kann das Suchergebnis in die Marketing-Aktion übernommen werden. 

Mit dieser Funktionalität wird auch jedes Mal eine neue E-Mail-Vorlage angelegt, da davon ausgegangen wird, dass Vorlagen auch je Aktion individuell angepasst werden. Das heißt auch, dass bei Übernahme einer neuen Aktion in eine Marketing-Aktion intern die Vorlage kopiert wird, damit sie in der Aktion verändert werden kann, ohne den Stamm zu tangieren. 

Siehe Kapitel Aktivität auswählen . 

Tooltip: 

Suche nach Aktivitätstypen oder Beschreibungen. 
Achtung: nach dem Hinzufügen von einer Aktivität wird diese automatisch gespeichert und die Teilnehmer und Teilnehmeraktivitäten werden ebenso aktualisiert und gespeichert. 

### Button Bearbeiten 

Dieser Button oder ein Doppelklick in eine Zeile öffnet den Dialog Aktivität bearbeiten , in der die gewünschte Änderung vorgenommen werden kann.

Tooltip: 

Änderung vornehmen 

### Button Löschen 

Mit diesem Button kann eine markierte Zeile (Aktionsschritt) aus der Registerkarte entfernt werden, wenn für die nachfolgenden Schritte noch keine Buchung vorgenommen wurde. 

Tooltip: 

Entfernen einer Tabellenzeile 

### Button Status 

Mit diesem Button erhalten Sie die Information wie viel Prozent der Aktion bereits ausgeführt sind. 

Tooltip: 

Statusinformation zur Aktion

Der Name des Aktivitätstyp (z.B. Adresse prüfen) wird angezeigt. 

Tooltip:

### Tabellenspalte Ausgeführt in % 
In dieser Spalte wird angezeigt, für wie viel Prozent der Teilnehmer die Aktivität bereits ausgeführt wurde. 

Diese Spalte wird nur gefüllt, wenn eine Ermittlung über den Button Status durchgeführt wurde. 

### Tabellenspalte Wiedervorlage 
Hier wird angezeigt, ob beim Ausführen des Aktionsschritts eine automatische Wiedervorlage an den im Dialog Aktivität bearbeiten hinterlegten Sachbearbeiter / Abteilung erstellt wird.
