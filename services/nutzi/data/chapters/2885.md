# Konfiguration Transformer Registerkarte Allgemein 
Konfiguration Transformer Registerkarte Allgemein 

Auf dieser Registerkarte nehmen Sie die allgemeinen Einstellungen vor. 
### Feld Quell-Objekt 
Hier wird das Quellobjekt des Transformers angegeben. 

Tooltip: 

Das Quellobjekt des Transformers. 

### Button Suche 

Dieser Button öffnet den Dialog Auswahl Komponente . 

Tooltip: 

Das Quellobjekt des Transformers auswählen 

### Feld Ziel-Objekt 
Hier wird das Zielobjekt des Transformers angegeben. 

Tooltip: 

Das Zielobjekt des Transformers wird angegeben. 

### Button Suche 

Dieser Button öffnet den Dialog Auswahl Komponente . 

Tooltip: 

Das Zielobjekt des Transformers wird ausgewählt. 
Nur transformierbare Objekte werden angezeigt. 

### Combobox Ziel-Methode 
Diese Combobox hält alle Transformationsmethoden bereit, die an diesem Zielobjekt zum Aufruf zur Verfügung stehen 

Tooltip: 

Die Transformationsmethode am Zielobjekt. 

### Feld Suchen 
Tooltip: 

Angabe der Properties des Quellobjekts 

### Baumansicht Struktur 

In dieser Baumstruktur ist es möglich, Struktur-Definitionen zu übernehmen. 

Tooltip: 

Angabe der Properties des Quellobjekts 

### Button Öffnen 

Tooltip: 

Ordnen Sie ein Property einem Parameter zu. 

### Tabelle 

In dieser Tabelle können nicht nur Properties angegeben werden, sondern auch ein Ausdruckseditor oder ein Fixwert. Dafür stehen die gleichnamigen Spalten zur Verfügung. 

Tooltip: 

Zuordnung der Eigenschaften zu den Methodenparametern
