# BMECat - Import Registerkarte Gruppen 
BMECat Import Registerkarte Klass. System / Gruppen 

Hier werden die Gruppen des Klassifikationssystems angegeben. 
### Tabelle 

### Tabellenspalte ID 
Hier wird der eindeutige Identifikator einer Gruppe angegeben. 

Tooltip: 

Eindeutiger Bezeichner einer Gruppe. 

### Tabellenspalte Name 
Hier wird der eindeutige Name der Gruppe innerhalb des Klassifikationssystems angegeben. 

Tooltip: 

Gibt den eindeutigen Namen der Gruppe innerhalb des Klassifikationssystems an. 

### Tabellenspalte Obergruppe 
Hier wird der Verweis auf die Obergruppe dieser Gruppe angegeben. 

Tooltip: 

Verweis auf die Obergruppe dieser Gruppe 

### Feld ID 
Hier wird der eindeutige Identifikator einer Gruppe angegeben. 

Tooltip: 

Eindeutiger Bezeichner einer Gruppe. 

### Feld Name 
Hier wird der eindeutige Name der Gruppe innerhalb des Klassifikationssystems angegeben. 

Tooltip: 

Gibt den eindeutigen Namen der Gruppe innerhalb des Klassifikationssystems an. 

### Feld Synonyme 
Hier werden die Synonyme der Gruppe innerhalb des Klassifikationssystems angegeben. 

Tooltip: 

Beschreibt die Synonyme der Gruppe innerhalb des Klassifikationssystems 

### Tabelle Merkmale 

### Tabellenspalte ID 
Hier wird die Referenz auf den eindeutigen Identifikator eines Merkmals angegeben. 

Tooltip: 

Referenz auf den eindeutigen Identifikator eines Merkmals 

### Tabellenspalte Muss 
Hier wird das Kennzeichen angegeben, ob das Merkmal verpflichtend ist. 

Tooltip: 

Kenner, ob das Merkmal verpflichtend ist 

### Tabellenspalte Datentyp 
Hier wird der Datentyp angegeben. 

Tooltip: 

Angabe des Datentyps 

### Tabellenspalte Einheit 
Hier wird die Maßeinheit des Merkmals angegeben. 

Tooltip: 

Maßeinheit des Merkmals
