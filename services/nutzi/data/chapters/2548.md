# Mandatsliste 
Mandatsliste 

Dieser Dialog öffnet sich im Buchen >> Auswertungen >> Personenkonten . Hier können Sie sich zu einem oder mehreren Kunden die hinterlegten Mandate anzeigen lassen. 

Die Daten können über die Felder im Kopf des Dialogs selektiert werden. 

Erteilte Bankeinzüge können in ein SEPA-Mandat umgewandelt werden. Das Umstellungsdatum hierfür wird dem Feld Datum ab im Dialog Mandatsliste entnommen. Die Umstellung erfolgt über das Menü Optionen >> Mandatsgenerierung . Die angelegten Mandate erhalten das Datum „Gültig ab“ der Umstellung. 

Der Umstellungsbrief ist als weiterer Report im Dialog Mandatsliste vorhanden. Der Druck erfolgt für alle selektierten Tabellensätze. Im Umstellungsbrief wird das Datum Gültig ab des Mandats ausgewiesen. Der Umstellungsbrief soll Ihrem Kunden die Überprüfung seiner Daten ermöglichen. 

Das Mandat wird generiert, wenn 

- der Kunde die Zahlungsbedingung BEZ hat. - dieser Kunde die Checkbox Abbuch. Auftrag (Registerkarte Rechnungswesen ) nicht gesetzt hat. - eine Bankverbindung mit IBAN + BIC vorhanden ist, für die es noch kein Mandat gibt.
