# Kopieren / Verschieben von Projektaufgaben 
Kopieren / Verschieben 

Dieser Dialog öffnet sich im Projekt auf der Registerkarte Phasen mit dem Button Kopieren . Hier kann der Sachbearbeiter mittels Checkboxen entscheiden, welche Elemente auf die Ziel-Projektaufgabe übernommen werden. Die Projektaufgabe kann so auch in ein anderes Projekt oder eine andere Projektphase kopiert werden. 

Dies erspart Ihnen den Aufwand, jedes Mal eine neue Aufgabe manuell anzulegen und alle Einzelheiten wieder eingeben zu müssen. 

Folgende Einstellungen sind möglich, die per Client Side Customizing vorbelegt werden können: 

- Datum/Zeit - Sollzeit - Verantwortlich - Ressourcen - Checkliste (nur Kennungen, ohne Verarbeitungsdatum und Sachbearbeiter) - Referenz Links - Termine 
Die Originalbezeichnung wird übernommen und kann dann ggf. angepasst werden. Beim Speichern wird sichergestellt, dass die neue Projektaufgabe keine leere Bezeichnung hat.
