# Artikelauswahl Registerkarte Merkmalfilter 
Registerkarte Merkmalfilter 

Auf dieser Registerkarte können Sie einen Merkmalfilter setzen. 
### Button Filter entf. 

Mit diesem Button kann der Suchfilter zurückgesetzt werden. 

Tooltip: 

Filtereinstellungen entfernen. 

### Combobox Klasse 
Nach Selektion einer Klasse aus dieser Combobox werden alle Merkmale dieser Klasse angezeigt. 

### Button Merkmalsklasse suchen 

Mit diesem Button können Sie nach Klassen suchen. 

### Tabelle 

Nach einer Auswahl in der Combobox Klasse werden in der Tabelle alle Merkmale dieser Klasse angezeigt. 

Damit können jetzt auch freie Werte für z.B. Wertebereiche (Spalte 'Wert von' / 'Wert bis') eingegeben werden. Alternativ kann ein Wert aus der Combobox ausgewählt werden. Die Combobox enthält nur Werte, welche auch im Suchergebnis vorkommen. Wurde bereits nach einem Wert gefiltert, werden die Werte für alle Merkmale, nach denen nicht gefiltert wurde, auf das Ergebnis eingeschränkt. 

In der Tabelle werden alle Merkmale angezeigt, die mindestens eine Zuweisung von einem Artikel aus der Ergebnismenge haben. 
### Tabellenspalte Merkmal 
Hier wird der Name des Merkmals angezeigt. 

Tooltip: 

Merkmalname 

### Tabellenspalte Wert von 
Hier wird ein Startwert angegeben. 

### Tabellenspalte Wert bis 
Hier wird ein Bis-Wert angegeben. 

### Tabellenspalte Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung
