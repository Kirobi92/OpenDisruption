# Assistent für Wegeoptimierung 
Weg. Opt. Assistent 

Der Button Assistent im Dialog Wegeoptimierung öffnet die Maske Weg. Opt. Assistent . Hier können Sie ein Schema für die Wegeoptimierung generieren und auf mehrere Gänge übertragen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Wegeoptimierung-Assistent 
Tooltip: 

In diesem Dialog können Sie ein Schema für die Wegeoptimierung generieren. 
Für mehr Informationen drücken Sie F1.

### Feld Ebene 
Über dieses Feld legen Sie fest, welche Ebene berücksichtigt wird. 

### Feld 1. Wegeopt. 
Ein Eintrag in diesem Feld bestimmt die erste Nummer, welche bei der Generierung als erstes vergeben werden soll. Sie ermittelt sich automatisch anhand der aktuell höchsten Nummer, kann aber auch von Hand überschrieben werden. 

Tooltip: 

Startwert der Wegeoptimerungs-Nummern 

### Feld Schrittweite 
Die Angabe in diesem Feld ist der Abstand, der zwischen zwei Wegeoptimierungsnummern verwendet wird. Diese Schrittweite kann variabel definiert werden. Die Schrittweite wird mit 5 vorbelegt, damit eine spätere Reorganisation der Lagerplätze möglich ist. (Beträgt der Abstand 2, so können Sie nur eine Wegeoptimierungsnummer dazwischen packen, bevor Sie die komplette Wegeoptimierung erneut machen müssen.) 

Tooltip: 

Schrittweite, die auf die nächste Wegeoptimierungsnummer addiert wird. 

### Combobox Sortierrichtung 
Die Zuordnung der Wegeoptimierung erfolgt je nach der Sortierrichtung. 

Wenn die Sortierrichtung „Absteigend“ ausgewählt wurde, erfolgt die Zuteilung der Wegeoptimierungsnummer ausgehend vom Lagerplatz mit der höchsten Lagerplatznummer (99-99-99-99-99) zur kleinsten Lagerplatznummer (01-01-01-01-01). 

Wenn die Sortierrichtung „Aufsteigend“ ausgewählt wurde, erfolgt die Zuteilung der Wegeoptimierungsnummer ausgehend vom Lagerplatz mit der niedrigsten Lagerplatznummer (01-01-01-01-01) zur größten Lagerplatznummer (99-99-99-99-99).

### Tabelle 

Bei der Erzeugung der Wegeoptimierung gibt es verschiedene Möglichkeiten. Da diese meist mit gegenüberliegenden Regalen arbeiten, kann bei der Schemadefinition bestimmt werden, wie in diesen Regalen (Gängen) gearbeitet werden soll. Diese konkreten Gänge können dann auch bei der Erzeugung aus dem Schema heraus definiert werden. Für diese Kombination werden die Wegeoptimierungen dann durchgeführt. Ist das Schema komplett durchlaufen, so wird die Wegeoptimierung automatisch bei der nächsten Gang-Paarung weiter durchgeführt. Die Nummer wird dabei erhöht, so dass die erste Gang-Paarung auch die niedrigste Nummer erhält.
