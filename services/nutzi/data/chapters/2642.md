# Anlagenstamm Registerkarte Kapitalverzinsung 
Anlage Registerkarte Kapitalverzinsung 

Auf dieser Registerkarte definieren Sie die Kapitalverzinsung für diese Anlage. 

Im Menü Stamm >> Kapital Verz. Art sind alle angelegten Kapitalverzinsungsarten aufgelistet. 

Wählen Sie hierzu eine Kapitalverzinsungsart aus und geben eine Zins-Kontonummer ein, auf welches die Zinsen gebucht werden sollen. Mit dem Button öffnet sich die Kontensuche. Nach Eingabe der Kontonummer wird automatisch die Kontobezeichnung eingelesen. 
### Combobox Kap. Verz. Art 
Hier wird die Kapitalverzinsungsart angegeben. 

Tooltip: 

Kapitalverzinsungsart 

### Feld Zins-Kontonr. 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Button Suche 

Mit diesem Button können Sie ein Konto suchen. 

Siehe Kapitel Kontoauswahl . 

### Feld Kontobez. 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Feld Prozent 
In diesem Feld geben Sie die Höhe der Zinsen für die Kapitalverzinsung an. 

### Feld Letzte Verzins. 
Die Anzeige der letzten Verzinsung ist wichtig, wenn Sie eine kalkulatorische AfA durchführen möchten. Dies darf erst erfolgen, wenn die Verzinsung vorausgegangen ist. 

Das geht nur bei einer manuellen Umstellung der Abschreibungsmethode in der Spalte ‚Steuerrechtlich‘. 

Tooltip: 

Letzte Verzinsung 

### Feld WB Wert 
Hier wird der Wiederbeschaffungswert angezeigt. Die Berechnung erfolgt über das Menü Buchen >> Berechne WB-Wert . 

Siehe Kapitel WB Werte . 

Tooltip: 

Wiederbeschaffungswert 

### Feld Jahr 
Hier wird das (Geschäfts-) Jahr angegeben. 

Tooltip: 

Wiederbeschaffungsjahr 

### Feld Berechnet am 

Tooltip: 

Wiederbeschaffungswert berechnet am 

2022-12-23
