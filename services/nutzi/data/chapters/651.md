# Text-Wizard Registerkarte Text 
Text-Wizard Registerkarte Text 

Auf dieser Registerkarte wird der Textaufbau angezeigt. 

In die beiden Felder Feld Start und Feld Ende unten auf der Registerkarte werden die Bezeichner der Variablen eingetragen. 

Hier können Sie auch die Definition der Feldbegrenzer ändern. Dies ist notwendig bei E-Mail-Dokumenten im Html-Format (siehe unten). Hierzu überschreiben Sie die bereits eingetragenen Begrenzungen mit den von Ihnen im Feld-Begrenzer ausgewählten Zeichen. Geben Sie einfach jeweils ein anderes Zeichen für den Beginn und das Ende als Feldbegrenzer innerhalb der Syntax an. Sobald Sie das Feld verlassen, werden Sie gefragt, ob Sie alle bisherigen Zeichen im Text durch die neuen ersetzen wollen. 

Wenn Sie die Einstellung der Feldbegrenzer in Dokumenten ändern wollen, die bereits im Html-Format bestehen, lehnen Sie die Abfrage nach Änderung aller Zeichen im Text ab! Ansonsten werden auch die Zeichen für das HTML-Format geändert, was ja vermieden werden soll. Das hat allerdings zur Folge, dass Sie Ihre Änderungen manuell vornehmen müssen. 

Wenn Sie mit Html arbeiten möchten, verwenden Sie folgende Feldbegrenzer: 

<Feld> statt dem bisherigen </Feld/>, da es sonst Probleme mit den Ende-Tags im Html gibt (z.B. </table>) 

Hier ein Beispiel mit abweichenden Feldbegrenzern. 

### Feld Feld Start 
In diesem Feld geben Sie die Bezeichner der Variablen für den Start ein. 

### Feld Feld Ende 
In diesem Feld geben Sie die Bezeichner der Variablen für das Ende ein.
