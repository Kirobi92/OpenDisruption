# Offene Posten Kreditor Registerkarte Selektionen 2 
Offene Posten Kreditor Registerkarte Selektionen 2 

Diese Registerkarte bietet weitere Möglichkeiten, offene Posten zu Kreditoren zu selektieren. 

Zum Thema Zahlsperre lesen Sie bitte das Kapitel Einschränken von OP-Sperren über das Datum . 
### Feld Betrag von 
Zur Selektion kann hier ein Betrag ausgewählt werden. 

Tooltip: 

Angabe des Betrags 

### Feld bis 
Zur Selektion kann hier ein Betrag ausgewählt werden. 

Tooltip: 

Angabe des Betrags 

### Checkbox S/H berücks. 
Wenn Sie diese Checkbox nicht aktivieren, wird die Spalte ‚Betrag‘ im OP mit positivem bzw. negativem Wert eingetragen. Das Soll-/Haben-Kennzeichen bleibt unberücksichtigt. 

Wenn Sie diese Checkbox aktivieren, wird bei einem positiven Wert nach einer Rechnung gesucht und bei einem negativen Wert nach der Zahlung oder der Gutschrift. 

Tooltip: 

Soll/Haben-Kennzeichen berücksichtigen 

### Combobox Datum 
Hier wählen Sie aus, was für ein Datum zur Selektion genutzt werden soll. Zur Verfügung stehen: 

- Belegdatum - Buchungsdatum - Löschdatum - Nettofällig - Valutadatum - Mahnsperre bis 

### Feld von 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Combobox OP-Kennz. 
Zur Selektion kann hier ein OP-Kennzeichen ausgewählt werden. 

Tooltip: 

Angabe des OP-Kennzeichens 

### Combobox Zahlung 
Hier wird die Zahlungsbedingung angegeben. Es werden Nummer und Text angezeigt. 

Tooltip: 

Zahlungsbedingung 

### Feld Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Über diesen Button können Sie ein Projekt suchen. 

Siehe Kapitel Projektauswahl . 

### Combobox Verband 
Über diese Combobox können Sie die OPs nach Verbänden selektieren. Die Selektionskriterien nur und keine sind standardmäßig enthalten. Wird im Kundenstamm im Feld Verband eine Kontonummer hinterlegt, erscheint diese ebenfalls als Verbandsnummer in der Combobox zur Auswahl. 

### Combobox Zentralzahler 
Über diese Combobox können Sie nach Zentralzahler selektieren. 

### Checkbox Zahlsperre 
Wenn Sie diese Checkbox aktivieren, werden nur offene Posten eingelesen, die auf Zahlsperre gesetzt sind. 

Tooltip: 

Diesen OP nicht im Zahlungsvorschlag berücksichtigen. 

### Checkbox Kein Kontoauszug 
Wenn Sie diese Checkbox aktivieren, wird der OP, der mit Kein Kontoauszug gekennzeichnet ist, selektiert. 

Tooltip: 

Diesen OP nicht im Kontoauszug berücksichtigen. 

### Checkbox Keine Zentralregulierung 
Wenn diese Kennzeichnung aktiviert ist, findet keine Zentralregulierung (Verbandsabrechnung bzw. Zentralzahlung) statt. 

Tooltip: 

Keine Zentralregulierung (Verbandsabrechnung bzw. Zentralzahlung)
