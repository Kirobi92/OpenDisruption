# Kreditversicherungsliste Registerkarte Kunden

Finanzbuchhaltung / Sicherheiten-Verwaltung / Kreditversicherungsliste 

# Kreditversicherungsliste Registerkarte Kunden 
Kreditversicherungsliste Registerkarte Kunden 

Auf dieser Registerkarte definieren Sie weitere Informationen zum Kunden. 
### Radiobutton Konto 
Durch aktivieren dieses Radiobuttons haben Sie die Möglichkeit in den Datenfeldern Von...Bis Ihre Daten über die Eingabe der Kontonummer zu selektieren. 

### Radiobutton Kontobez. 
Durch aktivieren dieses Radiobuttons haben Sie die Möglichkeit in den Datenfeldern Von...Bis Ihre Daten über die Eingabe der Bezeichnung zu selektieren. 

Tooltip: 

Kontobezeichnung 

### Radiobutton Suchname 
Durch aktivieren dieses Radiobuttons haben Sie die Möglichkeit in den Datenfeldern Von...Bis Ihre Daten über die Eingabe des Suchnamens zu selektieren. 

### Feld Konto von 
Hier wird die Kontonummer-von zum Datensatz angezeigt. 

Tooltip: 

Das Konto von zum Datensatz. 

### Button Suche 

Mit diesem Button können Sie nach Kunden suchen. 

Siehe Kapitel Kundenauswahl . 

### Feld Konto bis 
Hier wird die Kontonummer-bis zum Datensatz angezeigt. 

Tooltip: 

Das Konto bis zum Datensatz. 

### Button Suche 

Mit diesem Button können Sie nach Kunden suchen. 

Siehe Kapitel Kundenauswahl . 

### Feld Kontobez. von 
Die Kontobezeichnung-von wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung von 

### Feld Kontobez. bis 
Die Kontobezeichnung-bis wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung bis 

### Feld Suchname von 
Hier wird der Suchname angegeben. Außerdem kann hier eine Belegnummer zu einem Vorgang eingetragen werden. Wenn der Vorgang existiert, wird der Vorgang geladen. 

Tooltip: 

Angabe des Suchnamens 

### Feld Suchname bis 
Hier wird der Suchname angegeben. Außerdem kann hier eine Belegnummer zu einem Vorgang eingetragen werden. Wenn der Vorgang existiert, wird der Vorgang geladen. 

Tooltip: 

Angabe des Suchnamens 

### Checkbox Inland 
Über diese Checkbox können Sie Ihre Daten nach dem MwSt.-Kennzeichen aus dem Kundenstamm selektieren. 

### Checkbox Ausland 
Über diese Checkbox können Sie Ihre Daten nach dem MwSt.-Kennzeichen aus dem Kundenstamm selektieren, wobei EU und Ausland zusammengefasst werden. 

### Combobox Rechtsform 
Hier wählen Sie die Rechtsform des Kunden aus. Diese werden in einer Pickliste gepflegt. 

Tooltip: 

Angabe der Unternehmen-Rechtsform 

### Checkbox Kein neg. Sald. 
Durch aktivieren dieser Checkbox werden keine negativen Salden von Kunden angezeigt. 

Tooltip: 

Keine negativen Salden anzeigen 

### Checkbox Null-Salden 
Durch aktivieren dieser Checkbox werden auch Kunden berücksichtigt, die keine OPs haben. 

### Checkbox Vers.-Prüfung 
Aktivieren Sie diese Checkbox, werden nur Kunden berücksichtigt, die bei Aktivierung der Checkbox zu prüfende Aufträge (im Kundenstamm auf der Registerkarte Versicherung ) auf den Status nicht versichert gesetzt wurden.
