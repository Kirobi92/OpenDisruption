# Bestellungen Registerkarte Texte 
Registerkarte Texte 

Auf der Registerkarte Texte können Memo-, Start- und Endetext eingetragen werden. 

Das Memo hat keine Auswirkungen auf die Formulare. Die Start- und Endetexte werden auf den bestellbegleitenden Papieren angedruckt. 

Ist für den Start- oder Endetext eine Automatik hinterlegt, so wird der entsprechende Textbaustein beim Einfügen des Auftrags automatisch gesetzt. 

Siehe Kapitel Textbausteine . 
### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen.
