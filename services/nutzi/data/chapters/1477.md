# Währungseinheiten 
Währungseinheiten 

Dieser Dialog öffnet sich im Navigator über System >> Kasse >> Währungseinheiten oder über Kassieren >> Stamm . Hier definieren Sie die Stückelungen für einzelne Währungen. Nicht jede Währung verwendet die gleichen Geldeinheiten ( Beispiel: In der Schweiz ist das kleinste Geldstück ein 5-Rappen-Stück, in Deutschland das 1-Cent-Stück). Diese werden im Fall einer Zahlungsart mit Geldstückelung bei der Einlage, Entnahme und Abrechnung geladen und dem Benutzer zur Eingabe angeboten. Gespeichert wird nur die Endsumme. 

Bei Auswahl eines Eintrags in der Combobox Währung werden die vorhandenen Geldeinheiten der Währung eingelesen.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Combobox Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen.
