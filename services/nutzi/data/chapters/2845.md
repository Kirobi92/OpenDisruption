# Artikelempfehlungen 
Artikelempfehlungen 

Die Ergebnisse der Produktempfehlungen werden als Element im Beratungsassistenten zur Verfügung gestellt. Hierzu muss das Element Artikelempfehlungen eingerichtet werden. 

Das Element zeigt Artikelempfehlungen, die zu diesem Kunden und dem aktuellen Artikel passen. Die Auswahl passender Artikel basiert auf der Konfiguration des gewählten Empfehlungsagenten. Dabei kann es sich um Artikel handeln, die der Kunde bereits in der Vergangenheit dazu gekauft hat, Kaufgewohnheiten ähnlicher Kunden oder passende Artikel aus der Gesamthistorie. 

Die Darstellung der Ergebnisse erfolgt als Tabelle, sortiert nach Häufigkeit oder Marge absteigend. Bei Markierung einer Zeile wird das Standard-Artikelbild sowie der zugehörige Artikeltext unterhalb der Tabelle angezeigt. 

Die Aktualisierung der Anzeige erfolgt aus einem Verkaufsvorgang heraus, wenn eine neue Artikelposition markiert wird bzw. der Mini-Button B neben der Tabelle geklickt wird. 

Die empfohlenen Artikel können mittels Drag&Drop in den Verkaufsvorgang übernommen werden. 

Ziel / Nutzen 
Das Element liefert dem Berater im Gespräch mit dem Kunden Hinweise für zusätzliche Verkaufschancen. Potentiell für diesen Kunden interessante Artikel können aktiv angesprochen werden. 

Darstellung 
Klickt der Benutzer auf eine Empfehlungszeile, werden im unteren Teil des Elements das Artikelbild und die Artikelbeschreibung angezeigt. 

Parameter 
Agent: Titel des Empfehlungsagenten 

Level (1-3) 
1 = Kunde hat zum aktuellen Artikel auch diese Artikel gekauft (Kunde) 
2 = ähnliche Kunden kauften zu diesem Artikel auch (Cluster) 
3 = Insgesamt häufig zu diesem Artikel nachgefragte Produkte (Total) 

Max. Anzahl: 
Maximale Anzahl vorgeschlagener Artikel 

Drill Down 
Keiner. Per Drag&Drop kann ein Eintrag auf einen Auftrag oder ein Angebot gezogen werden. Der Artikel wird als Position eingefügt. 

Aktualisierung 
Die Aktualisierung der Daten erfolgt nur durch Anforderung des Benutzers. In Angebot/Auftrag/Gutschrift kann über den Mini-Button B eine Aktualisierung der Daten zur markierten Position ausgelöst werden. 
Alternativ kann auch eine Auftragsposition oder ein anderes Datenobjekt, das eine Kundenreferenz enthält, per Drag&Drop auf die Kundenbeobachtung gezogen werden.
