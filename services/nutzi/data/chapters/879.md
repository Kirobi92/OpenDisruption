# Vertreter Registerkarte Provision 
Vertreter Registerkarte Provision 

Auf dieser Registerkarte legen Sie die Daten für die Berechnung der Provision fest. 
### Dock Panel Provisionsart 

Hier legen Sie fest, ob die Provision auf Umsatz oder Ertrag berechnet werden soll: Auftragswert (Nettoumsatz) oder Rohertrag (exklusive Steuer). 

### Feld Provision % 
Hier wird der Provisionssatz für die Vertreterabrechnung in % angegeben. 

Bitte beachten Sie die Provisionshierarchie. 

Tooltip: 

Angabe eines Provisionssatzes für die Vertreterabrechnung. 

### Feld Provisionsfaktor 
Hier tragen Sie den Gewichtungsfaktor für die Provisionierung ein. Die Provisionen können von Vertreter zu Vertreter unterschiedlich gewichtet werden. Normalerweise wird hier der Wert 1 angegeben. Bei doppelter Provision wird 2 und bei halber Provision 0,5 angegeben usw. 

### Feld Von PLZ 
Hier haben Sie die Möglichkeit, einen Vertreter auf einen bestimmten Postleitzahlen-Bereich festzulegen (Anfangs-PLZ). Dieses Feld ist ein reines Informationsfeld ohne weitere Funktion. 

### Feld PLZ bis 
Hier haben Sie die Möglichkeit, einen Vertreter auf einen bestimmten Postleitzahlen-Bereich festzulegen (End-PLZ). Dieses Feld ist ein reines Informationsfeld ohne weitere Funktion. 

### Feld Bankanschrift 
Hier geben Sie den Namen der Bank des Vertreters ein. 

Tooltip: 

Angabe der Bank-Anschrift 

### Feld Bank KtoNr. 
Hier geben Sie die Kontonummer des Vertreters ein. 

Tooltip: 

Bank Kontonummer 

### Feld BLZ 
Hier geben Sie die Bankleitzahl des Vertreters ein. 

Tooltip: 

Angabe der Bankleitzahl / SWIFT 

### Bezeichnungsfeld SEPA 
Tooltip: 

Single Euro Payments Area 

### Feld IBAN 
Hier geben Sie die IBAN-Nummer ein. 

Tooltip: 

IBAN (International Bank Account Number) 

### Feld BIC 
Hier geben Sie optional die BIC ein. 

Tooltip: 

Bank Identifier Code
