# Serviceobjekt Registerkarte Begründungen 
Serviceobjekt Registerkarte Begründung 

Hier werden die Begründungen zur Deaktivierung der Serviceobjecte angezeigt. Diese werden im Dialog Deaktivierung eines Serviceobjekts angegeben. 
### Tabelle 

### Tabellenspalte Deaktivierung 
Hier wird angezeigt, ob es sich um eine Aktivierung oder um eine Deaktivierung handelt. 

Tooltip: 

Zeigt an, ob es sich um eine Aktivierung oder um eine Deaktivierung handelt. 

### Tabellenspalte Erfassung 
Hier wird vom Programm automatisch das Datum der Erfassung eingetragen. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Tabellenspalte Erfasser 
Hier wird der Sachbearbeiter angegeben, der den Datensatz erfasst hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der den Datensatz erfasst hat. 

### Tabellenspalte Begründung 
Hier wird die Begründung angezeigt, warum das Serviceobjekt aktiviert oder deaktiviert wurde. 

Tooltip: 

Begründung, wieso das Service Objekt akt- bzw. deaktiviert wurde.
