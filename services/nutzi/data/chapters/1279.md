# Beschaffungsartikel in der Retoure 
Um Beschaffungsartikel zu retournieren, ist folgendes zu beachten: Der Beschaffungsartikel kann in der Retoure erfasst werden. Die Freigabe dieser Retourenposition erfolgt erst, wenn für diesen Beschaffungsartikel eine Belastung an den Lieferanten erfasst wurde. Wenn diese Belastung rechnungsgeprüft ist, wird die Retoure automatisch freigegeben. 

Da nicht jeder Sachbearbeiter einen Beschaffungsartikel so ohne weiteres freigeben darf, sind folgende Einstellungen nötig: 

- Im Dialog Parameter Verkauf Registerkarte Retoure die Checkbox Beschaffungsartikel Retoure prüfen setzen. - Da nicht jeder Sachbearbeiter eine Rechnungsprüfung freigeben soll, wird im Dialog Berechtigungsgruppe auf der Registerkarte Allgemein die Checkbox Besch.-Artikel Retouren Ber. gesetzt. 
Offene Belastungen werden im Dialog Rechnungsprüfung auf der Registerkarte Offene Belastungen angezeigt.
