# Artikel Schattenstamm Registerkarte Rabatt Einkauf Gültigkeiten 
Artikel Schattenstamm Registerkarte Rabatt Einkauf Gültigkeiten 

Auf dieser Registerkarte werden Einkaufsrabattgültigkeiten angezeigt. 

Von hier werden die Rabatte in die Rabattverwaltung des Einkaufs übernommen. 
### Tabelle 

### Tabellenspalte Haupt-Importschlüssel 
Hier wird der Haupt-Importschlüssel angegeben. 

Tooltip: 

Angabe des Haupt-Importschlüssels 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Rabatt 
Hier wird eine Rabattnummer angegeben. 

### Tabellenspalte Einheit EK 
Hier wird die Einkaufeinheit angegeben. 

Tooltip: 

Einkaufeinheit 

### Tabellenspalte Gültig von 
Das Datum, das Sie hier auswählen, bestimmt den Beginn der Gültigkeit. 

### Tabellenspalte Basispreis 
Hier wird der Basispreis angegeben. 

Tooltip: 

Angabe des Basispreises 

### Tabellenspalte Rabatt 1 % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Rabatt 2 % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Preisgruppe 
Hier können Sie eine Brutto-Preisgruppe auswählen, die im zugrundeliegenden Vorgang herangezogen wird und bestimmt, auf welchen Verkaufspreis zugegriffen werden soll. 

Siehe Kapitel Preisgruppen . 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

### Tabellenspalte PG-Datum von 
Hier wird das Preisgruppendatum angegeben. 

### Tabellenspalte Standardstaffel 
Hier wird die Standardstaffel angegeben. Dabei bedeutet 
1 - aktiviert 
0 - deaktiviert 

Tooltip: 

Angabe einer Standardstaffel 

### Tabellenspalte Zusätzliche Eigenschaften 
Hier wird die zusätzliche Eigenschaft angegeben. Die Eigenschaften stammen aus hierarchisch späteren Paketen und werden hier gesammelt, sofern noch kein dediziertes Feld existiert. 

Tooltip: 

Diese Eigenschaften stammen aus hierarchisch späteren Paketen und werden hier gesammelt, sofern noch kein dediziertes Feld existiert.
