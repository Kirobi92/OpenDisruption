# Kunde Registerkarte Memo 
Kunde Registerkarte Memo 

Auf dieser Registerkarte können für einzelne Vorgänge Informationen zum Kunden eingetragen werden, wie z.B. ein Memo für den Auftrag oder eine Buchung. Wird dann der Kunde in einem entsprechenden Vorgang wie z.B. im Auftrag erfasst, öffnet sich nach Drücken des Buttons Enter das hinterlegte Memo. 

Siehe Kapitel Memo . 
### Feld Allgemein 
Dieses Memo wird nur im Kundenstamm angezeigt und kann per Reportprogrammierung jederzeit angedruckt werden. 

Tooltip: 

Kunden-Memo 

### Feld Aufträge 
Dieses Memo wird angezeigt, wenn der Kunde in Angebot, Auftrag oder Gutschrift ausgewählt wird. 

Tooltip: 

Angabe des Auftrags-Memos 

### Feld Buchung 
Dieses Memo wird angezeigt, wenn der Kunde beim Buchen in der Finanzbuchhaltung ausgewählt wird. 

Das Buchungsmemo ist jahresabhängig, da es in den Kontenstamm geht. Daher ist je nach gewähltem Buchungsjahr ein unterschiedlicher Inhalt möglich. 

Tooltip: 

Enthält interne Textinformationen. 

### Feld Versand 
Hier kann eine Information für den Versand eingetragen werden. 

Wenn im Dialog Versandanschrift ein Memo erfasst ist und die Versandadresse als Standard (Checkbox Standard ) gekennzeichnet ist, so übersteuert dies im Auftrag das Versandmemo aus dem Kundenstamm! 

Siehe Kapitel Versandanschrift . 

Nach einem entsprechenden Customizing kann diese Information auf den Versandbelegen angedruckt werden.
