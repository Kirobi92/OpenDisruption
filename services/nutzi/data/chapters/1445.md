# Vermietungsposition Registerkarte Ausfall 
Vermietungsposition Registerkarte Ausfall 

Auf dieser Registerkarte können Sie im Falle eines Ausfalls der Vermietungsposition entsprechende Informationen eingeben. Diese Ausfalltage werden bei der Erstellung der Schlussrechnung berücksichtigt, sofern das Mietobjekt Ausfalltage zulässt. 
### Checkbox Keine Ausfalltage 
Je nachdem, um welche Art von Mietobjekt es sich handelt, können dafür keine Ausfalltage angerechnet werden. Z.B. Versicherung. 

Tooltip: 

Wenn gesetzt, werden solche Mietobjekte bei der Erfassung von Ausfalltagen nicht berücksichtigt. (z.B. Versicherungen) 

### Feld Tage 
Falls die Vermietungsposition ausgefallen ist, geben Sie hier die Anzahl Tage ein. 

Tooltip: 

Angabe der Ausfalltage 

### Combobox Grund 
Aus dieser Combobox können Sie den Grund für den Ausfall auswählen. Diese Gründe werden in einer Pickliste gepflegt. 

Siehe Kapitel Picklisten . 

Tooltip: 

Grund für den Ausfall 

### Feld Memo 
Wenn es für einen Ausfall mehrere Gründe gibt, können Sie hier weitere Gründe eintragen. 

Tooltip: 

Enthält interne Textinformationen.
