# Einsatzarten 
Einsatzarten 

Unter dem Menü System >> Service >> Einsatzarten definieren Sie die Einsatzarten für Ihre Serviceverträge (wie z.B. Inspektion, Regelwartung). 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Einsatzarten 
Tooltip: 

In diesem Dialog definieren Sie Einsatzarten für Serviceverträge. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

Meldungen 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Artikel 

Dieser Button öffnet das Menü Einsatzart-Artikel . Wenn Sie eine Zeile der Tabelle markieren, bevor Sie diesen Button drücken, wird die Einsatzart in der Combobox des Dialogs Einsatzart-Artikel voreingestellt. 

Siehe Kapitel Einsatzart-Artikel . 

Tooltip: 

Zeigt die Artikel zur Einsatzart. 

### Tabelle 

### Tabellenspalte Einsatzart 
Geben Sie eine Nummer für die Einsatzart ein. 

Tooltip: 

Geben Sie der Einsatzart eine Nummer. 

### Tabellenspalte Bezeichnung 
Geben Sie der Einsatzart einen Namen. 

Tooltip: 

Angabe des Namens der Einsatzart 

### Tabellenspalte Sammelrechnung 
Wenn Sie die Checkbox in dieser Spalte aktivieren, erhält der Kunde eine Sammelrechnung. 

Siehe Kapitel Sammelrechnung . 

Tooltip: 

Kunde erhält Sammelrechnung 

### Tabellenspalte Letzte Wartung 
Wenn Sie die Checkbox in dieser Spalte aktivieren, wird Ihnen im Serviceeinsatz in der Spalte ‚Bearbeitet‘ das Datum der letzten Wartung eingetragen. Dazu ist notwendig im Serviceeinsatz auf der Registerkarte Kopfdaten ebenfalls die Checkbox Letzte Wartung zu setzen. 

Tooltip: 

Setzt das Datum der letzten Wartung 

### Tabellenspalte Auftragsart 
Aus dieser Combobox ist zwingend eine Auftragsart auszuwählen. (Diese werden in den Auftragsarten im Verkauf angelegt.) 

Siehe Kapitel Verkauf / Auftragsarten . 

Tooltip: 

Zeigt die Nummer der gewählten Belegart aus der Tabelle. 

### Tabellenspalte Angebotsart 
Es kommt immer wieder mal vor, dass ein Serviceeinsatz mit einer Reparatur verbunden ist. Damit für solch eine Reparatur auch erst einmal ein Kostenvoranschlag für den Kunden gestellt werden kann, wählen Sie hier die erforderliche Angebotsart zuzuweisen. 

Siehe Kapitel Serviceeinsatz , Absatz Kostenvoranschlag im Serviceeinstz. 

Tooltip: 

Angabe der Belegart
