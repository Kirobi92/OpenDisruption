# Bestellautomat Registerkarte Bestellungen 
Bestellautomat Registerkarte Bestellungen 

Auf dieser Registerkarte werden die generierten Bestellungen angezeigt und können per Doppelklick auf die gewünschte Zeile aufgerufen werden. Voraussetzung hierfür ist, dass im Kopf des Dialogs im Feld Benutzer ein Benutzer eingetragen ist. 
### Tabelle 

### Tabellenspalte Bestellung 
Hier wird die Nummer der Bestellung angegeben. 

Tooltip: 

Hier wird die Bestellnummer definiert. 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Firma 2 
Die 2. Adresszeile wird angegeben. 

Tooltip: 

Zeile 2 des Firmennamens. 

### Tabellenspalte Ort 
Hier wird der Ort angegeben. 

Tooltip: 

Angabe des Orts 

### Tabellenspalte Nettobetrag LW 
Hier wird der Nettobetrag angegeben. 

Tooltip: 

Angabe des Netto-Betrags
