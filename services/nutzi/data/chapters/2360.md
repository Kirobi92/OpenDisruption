# Rechnungslegungskreise in der Finanzbuchhaltung 
Rechnungslegungskreise 

Im Menü Buchen bzw. Sachkontenstamm >> Stamm >> Rechnungslegungskreise steht der Dialog Rechnungslegungskreise zur Verfügung. Hier erfassen und verwalten Sie die Rechnungslegungskreise. 

Durch eine weitere Datenstruktur kann jedes Sachkonto einer Kontogruppe zugeordnet werden, wobei eine Kontogruppe zu beliebig vielen Rechnungslegungskreisen gehören kann. 

So ist es möglich auf einfache Weise Auswertungen über einen bestimmten Rechnungslegungskreis zu erstellen. In der Auswertungsstruktur stehen dann nur die Konten zur Verfügung, die in der/den hinterlegten Kontogruppen zum selektierten Rechnungslegungskreis fallen. 

Bei der Ermittlung eines Ergebnisses in einer Bilanz-Auswertung werden dann nicht mehr einfach alle G&V-Konten herangezogen, sondern nur die G&V-Konten, die zum Rechnungslegungskreis der Auswertung gehören. Wenn einer Auswertung kein Kreis zugewiesen ist, stehen nach wie vor alle Konten zur Verfügung und für die Berechnung eines Ergebnisses werden weiterhin alle G&V-Konten verwendet. 

Beim Start einer Auswertung wurde bereits geprüft, ob alle Konten des Abschlusstyps (Bilanz oder G&V) in der Auswertung enthalten sind (bzw. ob sie korrekt zugeordnet sind) und falls nötig eine entsprechende Warnmeldung ausgegeben. Nun wird auch berücksichtigt, ob die Auswertung einem Rechnungslegungskreis zugeordnet ist. 

Außerdem kann nun einer Idea-Export-Definition ein Rechnungslegungskreis zugewiesen werden, sodass nur die Daten für den entsprechenden Kreis exportiert werden. Dies betrifft die Daten aus den Tabellen Konto, Journal, Anlagenstamm und Anlagenjournal . 

Alle genannten Zuordnungen sind freiwillig. Wenn Sie darauf verzichten, funktionieren alle Auswertungen und der IDEA-Export wie gewohnt. 

Diese Rechnungslegungskreise spielen in folgenden Vorgängen eine Rolle: 

- Sachkontenstamm : Combobox Kontogruppen bzw. Spalte 'Kontogruppe' (Registerkarte Tabelle ) - Kontogruppe im Buchen / Sachkonto >> Menü Stamm - Auswertungsstruktur Registerkarte Bilanz, G+V, Saldenauswertung : Combobox Rechn.leg.kreis - IDEA Export : Combobox Rechn.leg.kreis
