# Scanner - Buchen

Lagerverwaltungssystem (LVS) 

# Scanner - Buchen 
Buchen 

Dieser Dialog öffnet sich im Lagerauftrag >> Scanner >> Scanner Buchen . 

Dieser Dialog wird bei der zweistufigen Kommissionierung, der Einlagerung, welche über den Dialog Wareneingang erstellt wurde und der Umlagerung verwendet. 

## Steuerelemente des Formulars 

### Bezeichnungsfeld Buchen 
Tooltip: 

Dieser Dialog wird bei der zweistufigen Kommissionierung, der Einlagerung, welche über den Dialog 'Wareneingang' erstellt wurde und der Umlagerung verwendet. 
Für mehr Informationen drücken Sie F1. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Feld Barcode 
Hier wird der Barcode des zu bearbeitenden Lagerauftrags eingelesen. Danach wird automatisch die Lagerauftragsposition eingelesen und der Dialog gemäß der bisher durchgeführten Eingaben eingestellt. Der Barcode setzt sich wie folgt zusammen: 

Lagerauftragsnummer minus FixPosNummer 

Bei der ersten Lagerbuchung einer Lagerauftragsposition erhält der Sachbearbeiter die oben angezeigte Kombination. 

Tooltip: 

Angabe des Barcodes 

### Feld LA 
Hier wird die Statusinformation des Lagerauftrags angezeigt. Möglich sind hierbei: 
- offen - Bearbeitung - erledigt - storniert 
Weiterhin kann hier die Art des Lagerauftrags abgelesen werden: 
- Entnahme - Einlagerung - Umlagerung 

### Feld Lager von 
Im Fall eines Lagerauftrags mit einem chaotischen Lager als Von Lager/Nach Lager muss hier der Lagerplatz eingescannt werden um nachzuweisen, dass an der richtigen Position ein- bzw. ausgelagert wird. Wird hier ein falscher Wert eingetragen, wird eine Fehlermeldung angezeigt und der Scan des richtigen Lagerplatzes gefordert. Nach Abschluss dieses Scans wird das nächste Feld freigegeben. 

### Feld Lagerplatz 
Hier wird angegeben, auf welchem Lagerplatz die Packstücknummer zu finden ist. 

Tooltip: 

Angabe des Lagerplatzes 

### Feld Lager nach 
Im Fall eines Lagerauftrags mit einem chaotischen Lager als Von Lager/Nach Lager muss hier der Lagerplatz eingescannt werden um nachzuweisen, dass an der richtigen Position ein- bzw. ausgelagert wird. Wird hier ein falscher Wert eingetragen, wird eine Fehlermeldung angezeigt und der Scan des richtigen Lagerplatzes gefordert. Nach Abschluss dieses Scans wird das nächste Feld freigegeben. 

### Feld Lagerplatz 
Hier wird angegeben, auf welchem Lagerplatz die Packstücknummer zu finden ist. 

Tooltip: 

Angabe des Lagerplatzes 

### Feld Tour 
Hier wird die Bezeichnung der Tour angezeigt. 

### Feld Artikel 
Hier wird die Bezeichnung des Artikels angezeigt. 

### Feld Pickmenge 
Hier wird die Menge angegeben, die vom Sachbearbeiter gepickt, eingelagert oder umgelagert wird. Wird hier eine geringere Menge eingegeben als ursprünglich mitgeteilt, so wird eine Korrekturlogik angestoßen, die weitestgehend im Hintergrund abläuft. 

Tooltip: 

Angabe der Zielmenge 

### Feld Pickmenge 
In dieser Spalte wird die gepickte Menge angezeigt. 

Tooltip: 

Angabe der Pickmenge 

### Button Starten 

Mit diesem Button wird die Korrekturlogik angestoßen. 

### Feld Auftrag 
Hier werden Informationen des dazugehörigen Auftrages angezeigt. 

### Button Neu 

Mit diesem Button wird der gesamte Dialog zurückgesetzt / geleert. 

### Button Korrektur 

Dieser Button dient dazu, eine bereits verbuchte oder geöffnete Position zurückzusetzen, um Fehlbuchungen zu beheben.
