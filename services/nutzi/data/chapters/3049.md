# Replikationsservice Registerkarte Ereignisprotokoll 
Replikations-Service Registerkarte Ereignisprotokoll 

Der Replikationsservice hat keine Benutzer-Oberfläche. Meldungen über Informationen oder Fehler werden in ein Ereignisprotokoll geschrieben. Diese Meldungen werden hier angezeigt. 

Die neusten Meldungen stehen oben. 
### Tabelle 

### Tabellenspalte Erfassung 
Datum und Zeit der Meldung werden angezeigt. 

Tooltip: 

Zeigt das Erfassungsdatum 

### Tabellenspalte Grund 
Der Grund der Meldung wird angezeigt: 
0 = Information 
3 = Fehler 

### Tabellenspalte Mandanten-Key 
Hier wird der Mandanten-Key angezeigt. 

Tooltip: 

Angabe des Mandanten-Schlüssels 

### Tabellenspalte Replik.-Gruppe 
Wenn die Fehler einem Mandanten zugeordnet werden können, weil es z.B. beim Starten eines Prozessors oder bei der Verarbeitung ein Ereignis eintritt, dann wird das hier angegeben. 

Bei den anderen Meldungen wie z.B. Starten oder Herunterfahren des Services, wird hier "-------" angegeben. 

Tooltip: 

Replikations-Gruppe: Zusammenfassung mehrerer Tabellen zu einer Gruppe. 

### Tabellenspalte Text 
Hier wird der Meldungs-Text angezeigt. 

### Feld Beschreibung 
Hier werden Detail-Informationen zur Meldung angezeigt. Dazu muss die entsprechende Meldung ausgewählt werden. 

Tooltip: 

Angabe einer Beschreibung
