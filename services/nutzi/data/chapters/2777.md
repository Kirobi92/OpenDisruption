# Kostenträger Registerkarte Tabelle 
Kostenträger Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Kostenträger angezeigt. 
### Tabelle 

### Tabellenspalte Kostenträger 
Hier wird die Kostenträgernummer angegeben. 

Tooltip: 

Angabe der Kostenträgernummer 

### Tabellenspalte Bezeichnung 
Der Name des Kostenträgers wird angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Kürzel 
Hier wird ein Kürzel angegeben. 

### Tabellenspalte Abteilung 
Hier wird der Name der Abteilung des Ansprechpartners beim Kunden angegeben. 

Tooltip: 

Angabe der Abteilung des Ansprechpartners beim Kunden. 

### Tabellenspalte Zuständig 
Hier kann der für die Kostenstelle zuständige Abteilungsleiter o.ä. eingetragen werden. 

### Tabellenspalte Erfassung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der Erfassung eingetragen. 

Tooltip: 

Datum und Uhrzeit der Erfassung 

### Tabellenspalte Änderung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der letzten Änderung eingetragen. 

Tooltip: 

Datum und Uhrzeit der letzten Änderung 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Gesperrt 
Wenn Sie diese Checkbox aktivieren, werden nur gesperrte Datensätze gesucht und angezeigt. 

Tooltip: 

Aktiviert: Setzt den Datensatz auf 'gesperrt'. 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die Anzahl der tatsächlich gefunden Datensätze angezeigt.
