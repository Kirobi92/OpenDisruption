# Team Viewer 
Über den Button ? im Kopf des eNVenta Start-Dialogs (Navigator) kann über den Menüeintrag TeamViewer der Team Viewer geöffnet werden. 

Bei einem Telefonat mit einem eNVenta -Anwender (z.B. mit dem Support) kann dieser komfortabel und ohne Schwierigkeiten eine TeamViewer-Sitzung starten. Da eNVenta aber nicht nur von N&V eingesetzt wird, wird der Link zum N&V TeamViewer über einen Parameter individualisiert. Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein im Feld TeamViewer-URL vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein .
