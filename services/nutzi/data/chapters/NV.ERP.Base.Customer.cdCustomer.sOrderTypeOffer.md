# Kapitel NV.ERP.Base.Customer.cdCustomer.sOrderTypeOffer

Hier können Sie die Angebotsart (z.B. für Brutto-Angebote) für den Kunden auswählen. Diese Angebotsart wird im Angebot voreingestellt; sie kann dort überschrieben werden. Die Angebotsarten werden im gleichnamigen Dialog angelegt. 

Siehe Kapitel Angebotsarten .
