# Erfolgsrechnung (G+V) 
G + V 

Im Buchen >> Abschluss >> Auswertungen finden Sie das Menü G+V . Über dieses Menü drucken Sie die Erfolgsrechnung (G+V) für einen von Ihnen vorgegebenen Zeitraum, mit den von Ihnen vorgegebenen Kriterien. 

Die Anwendung kann auch über Buchen >> Stamm >> Auswertungsstruktur und Auswahl der entsprechenden Registerkarte und von dort mit dem Button Drucken geöffnet werden. Aus dieser Anwendung wird die aktuell geöffnete Auswertung in den Druckdialog übernommen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld G + V 
Tooltip: 

Über diesen Dialog drucken Sie die Erfolgsrechnung (G+V) für einen von Ihnen vorgegebenen Zeitraum, mit den von Ihnen vorgegebenen Kriterien, aus. 
Für mehr Informationen drücken Sie F1.

### Combobox Jahr 
Hier wählen Sie das Jahr, für das die G+V gedruckt werden soll. 

### Dock Panel Selektion nach 

Hier haben Sie 2 Möglichkeiten den Druck der G + V zu steuern. Einmal nach Datum und einmal nach Periode, Reports werden angepasst. 

Datum: 
Wenn der Radiobutton Datum aktiviert ist, ändert sich die Bezeichnung der Felder auf Von Datum / Bis Datum . So erhalten Sie eine tagesgenaue Auswertung. 

Periode: 
Wenn der Radiobutton Periode aktiviert ist, wählen Sie aus der Combobox den Zeitraum, für den die G+V gedruckt werden soll. Druck der G+V nur für die ausgewählte Periode bzw. von Jahresbeginn bis einschließlich der gewählten Periode.

### Checkbox Mit Kontonummern 
Wenn Sie diese Checkbox aktivieren, wird die Kontonummer angezeigt. 

### Checkbox Mit Zwischenabschluss 
Bei Aktivierung dieser Checkbox werden die Buchungen aus dem Buchungskreis Zwischenabschluss-Buchungen berücksichtigt. 

### Checkbox Mit Vorjahreswerten 
Wenn Sie diese Checkbox aktivieren, werden Vorjahreswerte zum entsprechend ausgewählten Zeitraum mit angedruckt 

### Checkbox Mit Nullsalden 
Wenn Sie diese Checkbox aktivieren, werden auch Konten mit Null-Salden angezeigt. 

### Checkbox Mit Saldovortrag 
Diese Checkbox wird nur bei Aktivierung des Radiobuttons kumuliert aktiv. 

Die Aktivierung dieser Checkbox ist nur notwendig, wenn innerhalb eines Geschäftsjahrs mit eNVenta ERP gestartet wird und die bestehenden Salden der G+V-Konten über Saldovortragskonten (Summenvortragskonto) gebucht werden. 

### Checkbox Verdichten 
Wenn Sie diese Checkbox aktivieren, werden nur die Summenzeilen angezeigt. 

### Checkbox 0-Werte drucken 
Wenn Sie diese Checkbox aktivieren, werden auch Auswertungen mit 0-Werten angezeigt. 

### Checkbox FW-Beträge drucken 
Wenn Sie diese Checkbox aktivieren, werden die Beträge zusätzlich in Fremdwährung gedruckt. Ist der Saldo NULL wird auch das Währungskürzel nicht angezeigt.
