# Artikelauswahl Registerkarte Artikeldienste 
Artikelauswahl Registerkarte Artikeldienste 

Diese Registerkarte unterstützt die Recherche mit verschiedenen Providern. 

Da die Services unterschiedliche Informationen unterstützen, ist zwingend erforderlich, dass der Inhalt von dem jeweiligen Provider bereitgestellt wird. 

Ein Doppelklick auf einen Eintrag der Tabelle öffnet den Dialog Produktdetails . 

Siehe Kapitel Produktdetails . 
### Combobox Datenlieferant 
Aus dieser Combobox wählen Sie den Datenlieferanten. 

Tooltip: 

Auswahl des Datenlieferanten 

### Feld Suchen 
In dieses Feld geben Sie einen Suchbegriff für die Recherche ein. 

Tooltip: 

Eingabe des Suchbegriffs 

### Checkbox Auslaufartikel anzeigen 
Wenn Sie diese Checkbox aktivieren, werden auch Auslaufartikel angezeigt. 

Tooltip: 

Auslaufartikel anzeigen 

### Tabelle 

### Tabellenspalte Vorschau 
In dieser Spalte wird ein Bild zum Produkt angezeigt. 

### Tabellenspalte Produktinfo 
In dieser Spalte wird eine Information zum Produkt angezeigt, wie z.B. Name und Maße. 

### Tabellenspalte Preis 
In dieser Spalte wird der Preis angezeigt, wenn er zur Ansicht freigeschaltet ist. 

### Tabellenspalte Verfügbar 
In dieser Spalte wird die Verfügbarkeit angezeigt, wenn er zur Ansicht freigeschaltet ist. 

### Tabellenspalte Beschreibung 
In dieser Spalte wird eine weitere Beschreibung des Produkts angezeigt. 

### Tabellenspalte Auslaufdatum 
Wenn dem Produkt ein Auslaudatum gesetzt ist, wird es in dieser Spalte angezeigt. 

### Feld Anzahl max. 
Hier geben Sie die Anzahl an Datensätzen an, die eingelesen werden sollen. 

Tooltip: 

Max. Anzahl an Datensätzen, die eingelesen werden.
