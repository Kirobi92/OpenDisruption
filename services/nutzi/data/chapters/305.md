Allgemeine Funktionalitäten in eNVenta ERP / Lagerverwaltung / Lagerplätze 

# Lagerplätze Registerkarte Generieren 
Lagerplätze Registerkarte Generieren 

Auf dieser Registerkarte können Sie entweder einzelne Plätze anlegen oder über Generieren ganze Gruppen von Lagerplätzen automatisch vom Programm einrichten lassen. Wichtig ist, dass sowohl Platz von als auch Platz bis im gleichen Format eingegeben werden. Nur der letzte numerische Teil der Lagerplatzbezeichnung wird automatisch hochgezählt. 

Beispiel: 
Von Platz R01F013 
Bis Platz R01F030 

eNVenta ERP generiert in diesem Fall genau 18 Lagerplätze. 

Es empfiehlt sich, die Lagerplätze in einem Format zu erfassen, das eine alphabetische Sortierung in der richtigen Reihenfolge zulässt. Damit wird eine optimierte Kommissionierung möglich. 
### Button Generieren 

Dieser Button generiert die Lagerplätze. 

### Feld Lagerplatz von 
Geben Sie den Lagerplatz ein, der generiert werden soll. 

Um ganze Gruppen von Lagerplätzen automatisch einzurichten, geben Sie außerdem einen Platz bis ein. 

Tooltip: 

Geben Sie den Lagerplatz ein, der generiert werden soll. 

### Feld Lagerplatz bis 
Um ganze Gruppen von Lagerplätzen automatisch einzurichten, geben Sie außerdem einen Platz bis ein. 

Tooltip: 

Hier geben Sie einen Platz-bis ein, wenn ganze Gruppen von Lagerplätzen angelegt werden sollen. 

### Checkbox Fixplatz 
Wenn Sie diese Checkbox aktivieren, können in bestimmten Bereichen, z.B. Picklager, die Lagerplätze als Fixplätze gekennzeichnet werden. 

Tooltip: 

Wenn aktiviert, können in bestimmten Zonen, z.B. Picklager, die Lagerplätze als Fixplätze gekennzeichnet werden.
