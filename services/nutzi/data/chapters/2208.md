# Serviceübersicht 
Serviceübersicht 

Dieser Dialog bietet Ihnen einen Überblick über sämtliche Einsätze. Zur Selektion der Daten stehen die Felder im Kopf des Dialogs zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Serviceübersicht 
Tooltip: 

Dieser Dialog bietet Ihnen einen Überblick über sämtliche Einsätze. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

Meldungen 

### Button Drucken 

Mit diesem Button starten Sie den Druck. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

### Button Neu 

Mit diesem Button leeren Sie die Felder. 

### Button Auswahl 

Mit diesem Button übernehmen Sie einen selektierten Datensatz in den zugrunde liegenden Dialog.

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Feld Kunde 
Zur Selektion eines Serviceeinsatzes geben Sie hier die Kundennummer ein. 

Tooltip: 

Angabe der Kundennummer 

### Button Suche 

Mit diesem Button können Sie Kunden suchen. 

Siehe Kapitel Kundenauswahl . 

### Feld Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Datum/Zeit von 
Zur Selektion der Daten können Sie hier Datum und Zeit eingeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld Datum/Zeit bis 
Zur Selektion der Daten über einen Datumsbereich können Sie hier ein Ende-Datum eingeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Combobox Einsatzart 
Zur Selektion können Sie hier die Einatzart auswählen. 

Tooltip: 

Angabe der Einsatzart 

### Combobox Monteur 
Hier können Sie den Monteur auswählen. 

Tooltip: 

Angabe des Monteurs 

### Feld Projekt 
Hier können Sie ein Projekt auswählen, wenn der gesuchte Serviceeinsatz einen Projektbezug hat. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Mit diesem Button können Sie ein Projekt suchen. 

Siehe Kapitel Projektauswahl . 

### Feld Bezeichnung 
Der Name zur Projektnummer wird eingetragen. 

Tooltip: 

Name des Projekts 

### Combobox Status 
Zur Selektion können Sie hier einen Status auswählen. 

Tooltip: 

Angabe des Positionsstatus 

### Combobox Sachbearbeiter 
Hier können Sie zur Selektion einen Sachbearbeiter auswählen. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
