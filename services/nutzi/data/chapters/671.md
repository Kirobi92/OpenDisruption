# EK-Rabattgruppen-Zuordnung Registerkarte Artikelgruppe 
EK-Rabattgruppen-Zuordnung Registerkarte Artikel-Gruppe 

Um einer Lieferanten-Rabattgruppe Artikelgruppen zuzuordnen, wählen Sie auf der Registerkarte Artikel-Bereich aus der linken Tabelle eine Artikelgruppe aus und ordnen Sie diese der rechten Tabelle Rabattgruppen mit dem Button zu. Der Button macht diese Zuordnung wieder rückgängig. 

Um mehrere Artikelgruppen einer EK-Rabattgruppe zuzuordnen, markieren Sie die Einträge in der Tabelle und weisen diese mit dem Button einer Einkauf-Rabattgruppe zu.

### Tabellenspalte Artikelgruppe

Hier wird der Name der Rabattgruppe angegeben.
