# Kopieren eines Kunden 
eNVenta bietet die Möglichkeit einen Kunden-Datensatz zu kopieren. Die Ausgangssituation ist ein Kundenstammsatz, welcher einen ähnlichen Aufbau darstellt. Sie können für eine einfache Stammdatenanlage Basis-Kunden anlegen. Diese versehen Sie mit relevanten Kriterien wie Zahlungsbedingungen und können anschließend die Vorbelegungen mit den neuen Kundendaten pflegen. Ein Beispiel für diese Funktion wären ‚Diverse Kunden‘. 

Mit dem Button kann der Kopiervorgang angestoßen werden. 

Nun wird das Feld Kundennummer automatisch geleert. Im Feld Suchname ändern oder ergänzen Sie den Suchnamen für den Neukunden. Mit dem Button speichern Sie den Vorgang und das System vergibt automatisch eine neue Kundennummer. 

Die Versicherungsdaten, wie z.B. Gesellschaft, Beurteilungsart und Versicherungsnummer werden beim Kopieren gelöscht!
