# Fokus Kontakte 
Fokus Kontakte 

Das Fokus Element Kontakte zeigt auf einen Blick und ohne weitere Mausklicks, ob, wann und mit wem die letzten Kontakte zu diesem Kunden/Lieferanten/Adresse stattgefunden haben. Der direkte Aufruf per Doppelklick ist möglich. 

Das Element wird bei jeder Änderung im offenen Vorgang automatisch aktualisiert.
