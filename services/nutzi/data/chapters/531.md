# Druckergruppen Registerkarte Abteilungsdruckergruppen 
Druckergruppen Registerkarte Abteilungsdruckergruppen 

Auf dieser Registerkarte ordnen Sie den Abteilungen Drucker zu. 

Sie können neben einer Druckergruppe auch eine Abteilung angeben, welcher wiederum Drucker zugeordnet werden können. Druckergruppen und Abteilungen werden in der gleichen Combobox angezeigt. 

Jeder Sachbearbeiter bekommt automatisch die Drucker seiner Abteilung zugeordnet, hierfür muss keine separate Zuweisung erfolgen. 

Siehe Kapitel Abteilungen .
