# Kapitel NV.ERP.Base.Common.cdParameterFI.shtTaxAuditYearTo

Wählen Sie über diese Combobox das Endedatum für den Zeitraum aus, den Sie prüfen möchten. 

Tooltip: 

Das Datum, das Sie hier auswählen, bestimmt das Endedatum für den zu prüfenden Zeitraum.
