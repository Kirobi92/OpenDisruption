# Verbandsabrechnung Debitor Registerkarte Nicht zuordenbare Zahlungen 
Verbandsabrechnung Registerkarte Nicht zuordenbare Zahlungen 

Ist auf dieser Registerkarte z.B. ein Eintrag ohne Kundennummer, so können Sie über einen Doppelklick in die entsprechende Zeile versuchen, doch noch eine Kundennummer zuzuordnen. Dieser Vorgang kann erfolgreich sein, wenn in der Zwischenzeit eine Mitgliedsnummer für diesen Eintrag angelegt wurde. Dann wird die Kundennummer auch in die gleichnamige Spalte eingetragen. Ein Doppelklick auf diesen Spalteneintrag öffnet den Kundenstamm. 

Auch hier steht die Spalte ‚Mitgliedsnr‘ zur Verfügung. 

Ist ein Wert in der Datei negativ, wird er auch so in der Registerkarte dargestellt. Im Datenimport kann keine Manipulation erfolgen. Erst durch die Zuordnung zu den 3 verschiedenen Ausbuchungskonten des Verbands / Zentralzahlers kann eine Negierung erfolgen. 
### Button Löschen 

Mit diesem Button kann ein Eintrag in der Tabelle gelöscht werden. 

### Button Halbautom. 

Über diesen Button starten Sie die halbautomatische Zuordnung der Datensätze. 

Mit der Halbautomatik haben Sie die Möglichkeit Datensätze zu buchen, bei denen das Personenkonto eindeutig ist, aber mit dem OP kein eindeutiger Treffer möglich ist. ( Beispiel: Ein Verband gibt einen anderen Rechnungsbetrag an - die Belegnummer führt zur Zuordnung in der Halbautomatik.) 

Findet die Halbautomatik einen solchen Datensatz, werden die Daten in den Dialog OP – Ausbuchung eingelesen. 

Dieser Dialog ist Ihnen aus der manuellen OP-Verbuchung bekannt. 

Wird der Dialog OP – Ausbuchung aus der Halbautomatik heraus geöffnet, steht Ihnen hier der Button Stopp zur Verfügung. Dieser bricht den aktuellen Vorgang des Einlesens ab. Alle bis dahin gebuchten Vorgänge bzw. OPs sind gebucht, die entsprechenden Sätze sind auch aus der Vorschlagstabelle gelöscht. 

### Button Zuordnen 

Mit diesem Button können Sie versuchen, markierte Zeilen erneut zuzuordnen.
