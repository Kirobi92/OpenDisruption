# Kapitel NV.ERP.Base.Common.cdParameterFI.shtTaxAuditYearFrom

Wählen Sie über diese Combobbox das Startdatum für den Zeitraum aus, den Sie prüfen möchten. 

Tooltip: 

Das Datum, das Sie hier auswählen, bestimmt das Startdatum für den zu prüfenden Zeitraum.
