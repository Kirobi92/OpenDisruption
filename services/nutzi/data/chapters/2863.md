# Archivpool Registerkarte Fehler 
Archiv-Pool Registerkarte Fehler 

Diese Registerkarte zeigt alle Druckbelege an, welche nicht korrekt archiviert werden konnten. Für jeden Beleg wird der Fehlertext in der Tabelle angezeigt. 

Wenn Sie einen Eintrag anklicken, erscheint unten der komplette Fehlertext. 

Eine Archivierung wird ausgeführt, unabhängig davon ob der zugeordnete Channel über eine aktive Verbindung zum Archivsystem verfügt oder nicht. Diese Belege wurden im Fehlerprotokoll aufgelistet und es war nicht möglich, diese Belege erneut zu archivieren. 

Um das zu vereinfachen, ist es möglich über den Button -> Offen die markierten Belege wieder in die Registerkarte Offen zu übernehmen. So kann die Verarbeitung der Reports erneut gestartet werden. 
### Button Details 

Über diesen Button öffnen Sie den Dialog Archiv-Pool Eintrag , um alle Daten des Belegs einsehen zu können. 

Siehe Kapitel Archiv-Pool Eintrag . 

### Button -> Offen 

Über diesen Button werden markierte Belege wieder in die Registerkarte Offen verschoben.

### Tabellenspalte Druck-Status 
Hier wird der Druck-Status angegeben. Dieser gibt Auskunft über den Verarbeitungsstatus des Dokuments: 

| -1 | Druckauftrag nicht auffindbar |
| --- | --- |
| 0 | Transaktion nicht abgeschlossen |
| 1 | Druck wurde vom PrintService noch nicht begonnen |
| 2 | Beleg wird gerade gedruckt |
| 3 | Beleg ist erfolgreich gedruckt bzw. verarbeitet worden |
| -3 | Beleg erfolgreich in der Datenbank erstellt und korrekt verarbeitet |
| 4 | Beleg ist fehlerhaft verarbeitet |
