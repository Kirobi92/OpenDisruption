# AES-Journal 
Übertragungsjournal 

Dieser Dialog öffnet sich in der Kommissionierung >> Optionen >> AES-Schnittstelle >> AES-Journal . Dieses Menü dient zur Recherche historischer oder nur erfasster Exportvorgangsdaten. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Übertragungsjournal 
Tooltip: 

In diesem Dialog erhalten Sie einen Überblick über historische oder nur erfasste Exportvorgangsdaten. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Feld Lieferschein 
Zur Selektion der Daten geben Sie in dieses Feld die Nummer des Lieferscheins ein, den Sie übertragen möchten. 

Tooltip: 

Geben Sie den Lieferschein an zur Daten-Übertragung. 

### Feld Lieferschein bis 
Zur Selektion der Daten über einen Bereich, geben Sie hier eine weitere Lieferscheinnummer ein. 

Tooltip: 

Geben Sie den Bis-Lieferschein an zur Daten-Übertragung. 

### Feld Erfassung ab 
Wenn Sie hier ein Datum eingeben, kann nach diesem Datum selektiert werden. 

Tooltip: 

Geben Sie ein Erfassungsdatum ein. 

### Feld Übertragung ab 
Zur Selektion kann hier das Datum der letzten Übertragung eingegeben werden. 

Tooltip: 

Geben Sie ein Übertragungsdatum ein. 

### Feld Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Geben Sie die Kundennummer ein. 

### Button Suche 

Mit diesem Button können Sie einen Kunden suchen.

### Button Entfernen 

Mit diesem Button entfernen Sie die Kundennummer. 

### Feld Firma 1 
Hier wird die Zeile 1 des Firmennamens angezeigt.

Zeile 1 des Firmennamens. 

### Feld TIN (Zollnr) 
Für eine Anbindung an AES wird hier die TIN-Nummer des Kunden angegeben. Diese wird von der Zollbehörde vergeben. 

Sie ist notwendig für Rücksendungen und ist eine Pflichtangabe. 

Tooltip: 

Angabe der TIN 

### Combobox Best.-Land 
Wählen Sie aus dieser Combobox ein Land aus, nachdem Sie selektieren möchten. 

Tooltip: 

Wählen Sie das Bestimmungsland aus. 

### Combobox Deklaration 
Wählen Sie aus dieser Combobox eine Deklaration für die Selektion aus. Zur Verfügung stehen EU und EX. 

Tooltip: 

Wählen Sie eine Deklaration für die Selektion aus. Zur Verfügung stehen EU und EX. 

### Radiobutton Alle Lieferscheine 
Wenn Sie diesen Radiobutton aktivieren, werden alle Lieferscheine in der Tabelle angezeigt. 

### Radiobutton Keine gelöschten 
Wenn Sie diesen Radiobutton aktivieren, werden gelöschte Lieferscheine nicht in der Tabelle angezeigt. 

### Radiobutton Nur gelöschte 
Wenn Sie diesen Radiobutton aktivieren, werden nur gelöschte Lieferscheine in der Tabelle angezeigt. 

Es können Lieferscheine gelöscht werden, welche bereits an AES übertragen wurden. In diesem Fall werden die Lieferscheine zusätzlich mit gelbem Hintergrund angezeigt. 

### Combobox Sachbearbeiter 
Wählen Sie aus dieser Combobox den Sachbearbeiter, dessen Vorgänge Sie in der Tabelle sehen möchten. 

Tooltip: 

Wählen Sie einen Sachbearbeiter. 

### Combobox Status 
Wählen Sie aus dieser Combobox einen Eintrag aus, nach dem selektiert werden soll. Zur Verfügung stehen: 

- erfasst - gesendet - erneut gesendet - storniert 

Tooltip: 

Wählen Sie einen Status zur Selektion.
