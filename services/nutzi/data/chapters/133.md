# Layout von Dialogen bearbeiten 
Mit der Docking-Windows-Technologie lassen sich verschiedene Dialoge in eNVenta ERP umgestalten, z.B. neben- und übereinander fixieren, um sie so gleichzeitig auf Breitbildmonitoren oder auf 2 nebeneinanderstehenden Monitoren darstellen zu können. Auf diese Weise werden große grafische Darstellungen wie z.B. der Produktionsleitstand oder der Projektleitstand von eNVenta ERP noch ergonomischer nutzbar. Im Telefonvertrieb z.B. können sich Mitarbeiter auf einem Monitor die Stammdaten und auf einem zweiten die am häufigsten gekauften Artikel des Kunden parallel anzeigen lassen. 

Alle Forms, die zur Entwicklungszeit einem normalen Frame zugeordnet wurden, können am Client zur Laufzeit aus dem Frame herausgelöst und neu angedockt werden. Sie können entscheiden wie und wo Sie Forms an die Stamm-Dialoge andocken / anlegen möchten. Dafür können Sie einfach eine Registerkarte ziehen und dann beliebig platzieren. 

Um ein gedocktes Form wieder zurück in das ursprüngliche Frame zu verschieben, wird das Kontextmenü verwendet, welches durch Rechtsklick auf die Titelleiste geöffnet wird. 

Beim Schließen von eNVenta ERP werden die vorgenommenen Änderungen gespeichert, damit beim nächsten Aufruf des Programms die Einstellungen wieder geladen werden können. 

Im Navigator können Sie unter dem Menüpunkt Datei >> Layout zurücksetzen die gesamten Einstellungen wieder löschen. 

Es öffnet sich eine Message Box. Wenn Sie diese mit Ja schließen, wird gelöscht! 

Ein sofortiger Neustart ist sinnvoll, damit alle neu vorgenommenen Einstellungen dann auch wieder gespeichert werden. 

Anschließend werden Sie gleich noch gefragt, ob die Anwendung beendet werden soll. Egal, wie Sie sich dort entscheiden, es wird gelöscht.
