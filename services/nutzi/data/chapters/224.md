# Artikelauswahl Registerkarte Artikelinfo 
Registerkarte Artikelinfo 

Auf dieser Registerkarte erhalten Sie detaillierte Informationen zum Artikel. 
### Feld Bezeichnung 
In diesem Feld wird die Bezeichnung des Artikels angezeigt, den Sie in der Tabelle markiert haben. 

Tooltip: 

Artikelbezeichnung 

### Feld Beschreibung 
In diesem Feld wird die Beschreibung des Artikels angezeigt, den Sie in der Tabelle markiert haben. 

Tooltip: 

Angabe einer Beschreibung 

### Feld Artikelgruppe 
Hier wird die Artikelgruppe angezeigt, die dem Artikel zugeordnet ist. 

Tooltip: 

Bezeichnung der Artikelgruppe 

### Combobox Lager 
Hier wird das Standardlager für den Artikel angezeigt. Sie können aber hier das Lager für eine Bestandsübersicht ändern. 

Tooltip: 

Definiert das Lager 

### Feld Bestand effektiv 
Hier wird der Bestand des Artikels auf dem ausgewählten Lager angezeigt. 

Tooltip: 

Dies ist der Wert, der sich errechnet aus dem Wert im Feld Bestand abzüglich der bereits kommissionierten Ware (Lieferschein gedruckt, aber noch nicht lagerverbucht. Datentechnisch gesehen, sind dies die Reservierungen in Stufe 2.) 

### Feld Verfügbar 
Die verfügbare Menge wird angezeigt. 

Tooltip: 

Angabe der Menge 

### Feld Bestellt 
Die bestellte Menge wird angezeigt. 

Tooltip: 

Hier wird die bestellte Menge angezeigt. 

### Feld Zuteilbar 
Hier wird die zuteilbare Menge angegeben. 

Tooltip: 

Angabe der Menge
