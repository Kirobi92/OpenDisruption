# OP Kreditor Detail 
OP Kreditor Detail 

Dieser Dialog öffnet sich im Dialog Offene Posten Kreditor über einen Doppelklick in eine Zeile, den Button Details oder den Button . Er bietet eine Übersicht über sämtliche Buchungen zu diesem bestimmten Beleg. 

Werden hier Einträge verändert, so müssen diese über den Button gespeichert werden. 

Wenn das Valutadatum eines OPs (im Dialog OP-Kreditoren Detail ) geändert wird, wird geprüft, ob das Valutadatum vor, gleich bzw. nach dem Belegdatum ist. 

Ist das Valutadatum vor dem Belegdatum, erscheint eine Meldung und gibt einen Hinweis darauf, dass jetzt die Fälligkeiten (Nettofälligkeit und Skontofällig 1-3) anhand des Belegdatums berechnet werden. Wenn das Valutadatum älter bzw. gleich ist, erhalten Sie eine Meldung mit der Frage, ob die Fälligkeiten anhand des Valutadatums neu berechnet werden sollen. 

### Minderungen bzw. Erhöhungen aus Rechnungen in Bezug auf das Anlagengut 
Wird eine Kreditor-Rechnung bzw. Kreditor-Gutschrift in der Finanzbuchhaltung von eNVenta oder in der Anlagenbuchhaltung von eNVenta eingebucht, wird diese automatisch in die Spalte ‚Anlage/Lfd.Nr.‘ eingetragen. Dazu ist es erforderlich, dass das Fibu-Anlagekonto im Sachkontenstamm immer die Kennzeichnung Anlagekonto und im Feld Sktokto.direkt das Anlagekonto enthält. 

Die Tabellenspalte ‚Anlage/Lfd.Nr.‘ und die Tabellenspalte ‚Sktokto. direkt‘ können nachträglich erfasst werden. 

In der Finanzbuchhaltung wird bei kreditorischer Ausgleichsbuchung, z.B. 
- Buchen , Buchungskreis KA, KZ - Zahlungsverkehr - Auszug-/Kassenimport - Buchen , Menü OP , OP-Verrechnung - Buchen , Menü OP , OP Kreditor , Registerkarte OP-Verrechnung gebuchter Skontoabzug oder gebuchte Ausbuchung automatisch in die Anlagenbuchhaltung gebucht, wenn 
- im OP Kreditor Detail in der Tabellenspalte ‚Anlage/Lfd.Nr.‘ diese erfasst ist, - sowie das Anlagekonto in Sktokto. direkt enthalten ist. - Das Skonto- oder Ausbuchungskonto (= das Anlagekonto) muss im Sachkontenstamm die Kennzeichnung Anlagekonto enthalten und - muss identisch sein mit dem Anlagekonto in der Anlage zum Bereichskennzeichen Fibu-Rechn.Kr. im Dialog Parameter Allgemein . 
Die automatische Buchung in die Anlagenbuchhaltung meldet eine Bestätigung mit den Angaben: 
- Anlage/Lfd.Nr. - Bereichskz. und - Betrag.

### Tabellenspalte Zahlsperre 
Wenn Sie die Checkbox in dieser Spalte aktivieren, wird für den OP eine Zahlsperre gültig. Der OP wird vom Zahlungsverkehr ausgeschlossen. Der hier eingetragene Wert wird auch an den Dialog Offene Positen Kreditor übergeben. 

Damit nicht für jeden neuen OP eine Zahlsperre gesetzt werden muss, gibt es im Lieferantenstamm auf der Registerkarte Rechnungswesen die Checkbox Zahlungssperre . 

Auf bereits in einem Zahlungsvorschlag enthaltene OPs wirkt sich das Setzen dieser Checkbox nicht aus. 

Siehe Kapitel Lieferantenstamm Registerkarte Rechnungswesen . 

Tooltip: 

Diesen OP nicht im Zahlungsvorschlag berücksichtigen. 

### Tabellenspalte Zahlsperre bis 
Wenn Sie in diese Spalte ein Datum setzen, wird der Haken in der Spalte 'Zahlsperre' automatisch mit gesetzt. So ersparen Sie sich die Mühe, regelmäßig gesperrte OPs zu überwachen, um sie wieder zu entsperren. 

Wenn Sie das Sperr-Datum löschen, wird immer auch die Checkbox in der Spalte 'Zahlsperre' entfernt. 

Das heißt, möchten Sie aus einer befristeten Zahlsperre eine unbefristete Zahlsperre machen, müssen Sie nach dem Entfernen des Datums in der Spalte ‚Zahlperre bis‘ die Checkbox in der Spalte 'Zahlsperre' aktivieren. 

Wird ein Sperr-Datum gesetzt, erfolgt im Feld OP-Memo automatisch ein Eintrag mit folgenden Informationen: 

- Tagesdatum - Sachbearbeiter - Name des Sperrkennzeichens - Datum der Sperre (entfällt, wenn nur das Sperr-Kennzeichen gesetzt wird) 
Bitte beachten: Wird beim Erfassen einer Sperre / eines Sperr-Datums zusätzlich ein individuelles OP-Memo auf einen Satz erfasst, muss ein "Zwischen"-Speichern erfolgen, ansonsten wird durch Aktivieren der Sperre oder Erfassen des Sperr-Datums das individuelle OP-Memo überschrieben. Zuvor vorhandene OP-Memos bleiben erhalten. 

Tooltip: 

Geben Sie ein Bis-Datum ein.
