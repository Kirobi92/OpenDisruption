# Datenimport bearbeiten Registerkarte Merkmalsklassenzuordnung 
Import Bearbeiten Registerkarte Merkmale 

Auf dieser Registerkarte werden die Merkmale eingelesen. Die eingelesenen Daten können nach unterschiedlichen Kriterien gefiltert werden. 
### Button Bearbeiten 

Wenn Sie diesen Button drücken können Sie die Artikelmerkmalsklasse bearbeiten. Es öffnet sich der Dialog Import Editor - Merkmalsklassenzuordnung . 

Tooltip: 

Artikel Merkmalsklassenzuordnung bearbeiten 

### Button Klassenzuordnung Verarbeiten 

Mit diesem Button werden ausgewählte Merkmale in den Schattenstamm übernommen. 

Tooltip: 

Ausgewählte Merkmalsklassenzurodnung in den Schattenstamm übernehmen

### Tabelle 

### Tabellenspalte Importschlüssel 
Hier wird der Import-Schlüssel angegeben. 

Tooltip: 

Der "Import-Schlüssel" ist ein generierter Wert, der als eindeutige ID (Referenzschlüssel) zur Referenzierung des Datensatzes dient. Er wird aus den Haupteigenschaften des Datensatzes zusammengesetzt. Jeder Datensatz benötigt einen eigenen, eindeutigen Import-Schlüssel. Es wird kein neuer Datensatz angelegt, wenn es bereits einen Datensatz mit demselben Import-Schlüssel gibt, sondern der bereits vorhandene überschrieben/aktualisiert. 

### Tabellenspalte Haupt-Importschlüssel 
Hier wird der Haupt-Importschlüssel angegeben. 

Tooltip: 

Angabe des Haupt-Importschlüssels

Siehe Kapitel Klassifikationssystem . 

Tooltip: 

Klassifikationssystem
