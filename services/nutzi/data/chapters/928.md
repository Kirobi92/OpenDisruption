# Projektphasen Registerkarte Allgemein

Projekt Management / Projektphasen 

# Projektphasen Registerkarte Allgemein 
Projektphasenstamm Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie allgemeine Einstellungen zur Phase. 
### Feld Budget 
Hier definieren Sie ein Budget in Tagen. D.h. wie lange die Phase bearbeitet werden kann. 

Tooltip: 

Angabe des Projekt-Budgets in Tagen 

### Feld Tage geplant 
Hier geben Sie für die Phase die geplanten Tage ein. 

### Feld Tagessatz 
Hier wird der Tagessatz für die Abrechnung angegeben. 

Tooltip: 

Angabe des Tagessatzes 

### Combobox Bereich 
Über diese Combobox lassen sich Phasen filtern. Dies dient der übersichtlicheren Darstellung bei größeren Projekten. 

### Feld Beschreibung 
Hier können Sie der Phase einen individuellen Text hinzufügen. 

Tooltip: 

Angabe einer Beschreibung
