# Kapitel NV.ERP.Base.Common.cdSystemMM.shtStorageIDCentral

Hier wählen Sie das Zentrallager aus. Dieses Zentrallager spielt eine Rolle bei der Umlagerung von Artikeln. Im Bestellvorschlag, wo auch Umlagerungen disponiert werden, wird das Zentrallager eingetragen, falls kein Umlagerungslager in der Position des Bestellvorschlags bzw. in den Lagerorten (auf der Registerkarte Disposition ) angegeben wurde. Wenn ein Zentrallager eingerichtet wurde, werden alle Außenlager von dort aus bestückt, wenn keine abweichenden Regelungen getroffen worden sind. 

Siehe Kapitel Lagerorte Registerkarte Disposition , Absatz Beispiele für Umlagerungs-Szenarien. 

Tooltip: 

Für die Umlagerung von Artikeln wählen Sie hier das Zentrallager.
