# Tourenauswertung Detail 
Tourenauswertung Detail 

Dieser Dialog öffnet sich mit einem Doppelklick in eine Zeile der Tabelle im Dialog Tourenauswertung . Hier lassen sich Touren detaillierter auswerten. Es kann hier noch über Kunden bzw. Kundengruppen eingeschränkt und nach Stopps, Belegen und Positionen (Checkboxen Stopps anzeigen, Belege anzeigen und Positionen anzeigen ) gefiltert werden. 

Viele Kosten können nur auf Stoppebene sinnvoll ermittelt werden. Aus diesem Grund wird die Darstellung nicht rein auf Lieferscheinebene angezeigt, sondern 3-stufig: 

Stufe 1: Betrachtung auf Stoppebene 
Stufe 2: Betrachtung auf Belegebene 
Stufe 3: Betrachtung auf Positionsebene 

Die Ergebnisse werden in der Tabelle dargestellt. 

Die Elemente der Stufe 1 werden dabei Dunkelgrau dargestellt. 
Die Elemente der Stufe 2 werden dabei in Hellgrau dargestellt. 

Abschließend soll zum Schluss ein Summenelement geführt werden. Dieses wird Gelb hinterlegt. 

Die Tabelle enthält u.a. Ertrag, Kosten pro Kilometer, Ertrag* ohne Lieferkosten. 

*WS-Package: Der Ertrag wird nach EKF-Stufe / KEK-Stufe ermittelt. 

Kilometerkosten können auf die Auftragsposition herunter gerechnet werden. Dies ist möglich, indem Sie das Ergebnis der Berechnung (Kosten-Kilometer-Lieferschein) / Anzahl der Positionen im Lieferschein berücksichtigen. 

Die Kosten pro Stopp bzw. pro Position werden nach folgender Formel berechnet: 

Kosten pro km = Gesamtkosten Fuhrpark / Gesamt-km 

Grundkosten je Tour = z.B. LKW-Kosten pro Tag 

Tourkosten = Grundkosten je Tour + (Anzahl Stopps x Stoppkosten) + Tourenlänge x (Kosten pro km) 

Die Kosten pro Stopp werden anhand der Einstellungen zur Kundenrentabilität berechnet.
