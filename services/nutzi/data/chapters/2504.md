# Neue Auswertung 
Neue Auswertung 

Um eine neue Auswertungsstruktur anzulegen, drücken Sie im Dialog Ausw. Struktur den Button Neue Auswertung . Damit öffnet sich der Dialog Neue Ausw. Struktur . 

Im Feld Bezeichnung vergeben Sie den Namen für die neue Auswertung (z.B. G+V-Test). 

Wollen Sie eine Auswertung komplett neu erstellen (ohne Strukturvergabe), bestätigen Sie dies mit dem Button . Sie gelangen wieder in den Dialog Ausw. Struktur und können mit dem Aufbau der neuen Auswertung beginnen. 

Je nachdem, welcher Radiobutton gewählt wird, erstellen Sie eine Bilanz, eine G+V, eine UVA oder eine Saldenauswertung. 

Sie haben auch die Möglichkeit, die Struktur einer bereits vorhandenen Auswertung zu übernehmen. Hierzu aktivieren Sie nach Vergabe des Namens für die Auswertung die Checkbox Kopieren von im Feld Auswertung kopieren . Jetzt können Sie in der darunterliegenden Combobox eine Auswertung auswählen, deren Struktur Sie übernehmen wollen. 

Aktivieren Sie die Checkbox mit Detail , werden die Struktur der Auswertung und alle Kontozuordnungen übernommen.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern!

### Checkbox Kopieren von 
Wenn Sie diese Checkbox aktivieren, können Sie eine Auswertung auswählen, um deren Struktur zu kopieren bzw. zu übernehmen.
