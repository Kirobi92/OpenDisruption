# BMECat - Import Registerkarte Käufer 
BMEcat Registerkarte Käufer 

Die Felder auf dieser Registerkarte zeigen die Informationen aus dem BMEcat an. Diese Informationen sind optional, wenn sie nicht vorhanden sind, bleibt die Registerkarte deaktiviert. 
### Feld GLN/ILN 
Hier wird die globale Identifikationsnummer für EDI-Teilnehmer (für den Austausch von Geschäftsdokumenten) angegeben, früher: ILN.
