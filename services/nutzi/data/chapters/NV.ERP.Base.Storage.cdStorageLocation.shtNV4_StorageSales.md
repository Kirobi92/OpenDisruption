# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtNV4_StorageSales

Wenn Sie diesen Parameter aktivieren, wird bei einer Umlagerung in dieses Lager automatisch ein Verkaufsetikett gedruckt. 

Tooltip: 

Aktiviert bei einer Umlagerung auf dieses Lager automatisch den Druck eines Verkaufsetiketts.
