# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtUseContainersForGoodsReceipt

Wenn Sie diesen Parameter aktivieren, ist die Ware im sequentiellen Wareneingang unbedingt auf einen Behälter zu buchen. 

Es besteht die Möglichkeit statt des Lagerauftrags auch das LHM anzuscannen. 

Tooltip: 

Ist dieser Parameter aktiviert, muss die Ware im sequentiellen Wareneingang auf einen Behälter gebucht werden. Nicht gültig für den sequentiellen Wareneingang am Scanner!
