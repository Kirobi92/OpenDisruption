# Lagerauftrag generieren Registerkarte Auftragspositionen 
LA - generieren Registerkarte Auftragspositionen 

Auf dieser Registerkarte werden alle Positionen eines Auftrags aufgelistet. 
### Tabelle 

### Tabellenspalte Beleg 
In dieser Spalte wird die zugrundeliegende Auftragsnummer angezeigt. 

Tooltip: 

Angabe der Belegnummer 

### Tabellenspalte Pos 
Hier wird die Positionsnummer angezeigt. 

Tooltip: 

Positionsnummer 

### Tabellenspalte Status 
Hier wird der Status der Auftragsposition angezeigt. Zur Verfügung stehen: 

- Offen - Offen, zur Lieferung freigegeben - Offen, zur Faktura freigegeben - Fakturiert - Geliefert - Storniert 

Tooltip: 

Positionsstatus 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer aus dem Auftrag angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Hier wird die Artikelbezeichnung angezeigt. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Versandlager 
Hier wird das Lager der Auftragsposition angezeigt. 

Tooltip: 

Versandlagernr. 

### Tabellenspalte Lieferschein 
Falls bereits eine Lieferscheinnummer existiert, wird diese angezeigt. 

Tooltip: 

Angabe der Lieferscheinnummer 

### Tabellenspalte Menge 
Hier wird die Menge der Auftragsposition angezeigt. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Bestellt für LA 
Hier wird die bestellte Menge angegeben. 

Tooltip: 

Angabe der Menge, welche bestellt wurde.
