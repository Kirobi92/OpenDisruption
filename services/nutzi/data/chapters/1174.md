# Toureninformation 
Toureninformation 

In Auftrag, Angebot und Gutschrift unter Optionen >> Auskunft finden Sie den Eintrag Toureninformation . 

Mit Hilfe dieses Dialogs sehen Sie die nächsten verfügbaren Touren, beginnend ab dem aktuellen Zeitpunkt. Sie sehen hierbei, welche Tour bereits wie stark ausgelastet ist und können ggf. durch Einplanung Ihres Auftrags eine gleichmäßige Auslastung berücksichtigen. 

Bei Markierung einer Tour sehen Sie die bereits eingeplanten Stopps und können daher auch in etwa einschätzen, wann der LKW beim Kunden sein wird, bevor Sie den Auftrag abschließen. Wenn Sie eine genauere Berechnung der Ankunftszeit sehen möchten, so ist dies bereits an der Auftragsposition ersichtlich. 

Im Dialog Tour - Zuordnung Postleitzahlen werden die Zeiten hinterlegt mit welchen die geschätzten Ankunftszeiten berechnet werden. 

Siehe Kapitel VK-Positionen bearbeiten . 

Siehe Kapitel Zuordnung von Postleitzahlen zu einer Tour . 

Es werden alle Touren aufgelistet, deren Startdatum nach der aktuellen Uhrzeit zum aktuellen Datum starten. Hierbei werden nur Touren angezeigt, welche bereits eingeplante Vorgänge haben. Keine leeren Touren! 

Gibt es zu einer angezeigten Tour bereits eine Zuordnung zu diesem Auftrag, wird der Name der Tour grün hinterlegt. 

Wenn keine Routen mit Standarddaten gefunden werden (d.h. in den Kopfdaten ist eine Route festgelegt), dann werden Routen anhand der Postleitzahl berechnet. Dabei hat die Postleitzahl der Lieferanschrift höhere Priorität als die Postleitzahl der Rechnungsanschrift.
