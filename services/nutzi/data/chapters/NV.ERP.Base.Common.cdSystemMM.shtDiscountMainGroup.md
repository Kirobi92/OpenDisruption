# Kapitel NV.ERP.Base.Common.cdSystemMM.shtDiscountMainGroup

Hier kann die Anzahl der Stellen festgelegt werden, welche die Hauptgruppe der VK-Rabattgruppen bestimmen. 

So haben Sie die Möglichkeit für die Preisfindung von Hauptartikelebene auf Rabattgruppenebene umzuschalten. 

Damit können Sie dann Konditionen optional nur für VK Rabatthauptgruppenebene angeben. 

Die Funktion ist analog der Konditionierung für Artikel-Hauptgruppen.
