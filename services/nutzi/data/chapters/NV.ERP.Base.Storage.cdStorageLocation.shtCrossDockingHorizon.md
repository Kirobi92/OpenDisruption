# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtCrossDockingHorizon

Hier kann angegeben werden, für wie viele Tage in die Zukunft zum Lieferdatum der Dialog Cross Docking bei der Einlagerung von auftragsbezogener Ware geöffnet werden soll. Ist kein Eintrag vorhanden, wird zum Lieferdatum + 2 Tage der Dialog Cross Docking geöffnet. 

Diese Funktion steht für das 1-stufige LVS zur Verfügung. 

Siehe Kapitel Cross Docking . 

Tooltip: 

Abstand Lieferdatum vom aktuellen Datum fürs Crossdocking.
