# Sequentieller Wareneingang im LVS

Lagerverwaltungssystem (LVS) 

# Sequentieller Wareneingang im LVS 
Seq Wareneingang 

Dieser Dialog öffnet sich im Lagerauftrag >> Scanner >> Seq. Wareneingang . Nach dem Verbuchen der ersten Stufe kann mit diesem Dialog der Wareneingang sequentiell durchgeführt werden. Hierzu werden nach Scannen des Artikels die potentiellen Wareneingänge angeboten. Nach Auswahl und Eingabe der Menge wird die Teilmenge dann wareneingangsverbucht. Dieses Kapitel steht für das 2-stufige LVS zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Seq. Wareneingang 
Tooltip: 

Dieser Dialog steht zur Verfügung, um nach dem Verbuchen der ersten Stufe des Wareneingangs den Wareneingang sequentiell durchzuführen. 
Für mehr Informationen drücken Sie F1. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. 

### Feld Artikel 
Hier wird die Artikelnummer für den Wareneingang eingetragen. 

Tooltip: 

Angabe der Artikelnummer 

### Feld Bezeichnung 
Hier wird die Bezeichnung des Artikels angezeigt. 

### Tabelle 

### Tabellenspalte Beleg 
In dieser Spalte wird die Belegnummer des Lieferanten angezeigt. 

Tooltip: 

Hier wird die Bestellnummer definiert. 

### Tabellenspalte Fixpos 
In dieser Spalte wird die Fixposnummer angezeigt. 

Tooltip: 

Eindeutige Positionsnummer des Datensatzes. 

### Tabellenspalte Restmenge 
In dieser Spalte wird die restliche Menge angezeigt. 

Tooltip: 

Angabe der Menge 

### Feld Menge 
Hier wird die Menge angegeben. 

Tooltip: 

Angabe der Menge 

### Button Neu 

Mit diesem Button leeren Sie die Felder. 

### Button Buchen 

Mit diesem Button wird der Wareneingang gebucht.
