# Auftrag Registerkarte Texte 
Registerkarte Texte 

Auf dieser Registerkarte können Memo, Start- und Endetext (zum Druck von Belegen an den Kunden) eingetragen werden. 
### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen. 

### Button Suche 

Dieser Button öffnet den Dialog Textbausteine zur Selektion eines Textbausteins. 

Siehe Kapitel Textbausteine . 

Tooltip: 

Auswahl der Textbausteine öffnen 

### Feld Starttext 
Hier geben Sie einen sogenannten Starttext ein. Dieser Starttext (ein Begrüßungstext) wird nur in Angebot und Auftragsbestätigung gedruckt. 

Ist dem Starttext eine Automatik hinterlegt, so wird der entsprechende Textbaustein beim Einfügen des Belegs automatisch gesetzt. 

Siehe Kapitel Textbausteine . 

### Button Suche 

Dieser Button öffnet den Dialog Textbausteine zur Selektion eines Textbausteins. 

Siehe Kapitel Textbausteine . 

Tooltip: 

Auswahl der Textbausteine öffnen 

### Button Html-Editor 

Dieser Button öffnet den Dialog HTML-Editor zur Bearbeitung von HTML-Texten. 

Siehe Kapitel HTML-Editor . 

Tooltip: 

Html-Editor öffnen, um die Texte zu formatieren. 

### Feld Endtext 
Hier geben Sie einen sogenannten Endetext ein. D.h. ein Schlusssatz, mit dem der Text auf dem Beleg endet. 

Ist dem Endetext eine Automatik hinterlegt, so wird der entsprechende Textbaustein beim Einfügen des Auftrags automatisch gesetzt. 

Siehe Kapitel Textbausteine . 

### Button Suche 

Dieser Button öffnet den Dialog Textbausteine zur Selektion eines Textbausteins. 

Siehe Kapitel Textbausteine . 

Tooltip: 

Auswahl der Textbausteine öffnen 

### Button Html-Editor 

Dieser Button öffnet den Dialog Html Editor zur Bearbeitung von Html-Texten. 

Siehe Kapitel Html Editor . 

Tooltip: 

Html-Editor öffnen, um die Texte zu formatieren. 

### Feld Versand 
In diesem Feld stehen Informationen, die im Kundenstamm auf der Registerkarte Memo im Feld Versand oder in den Versandanschriften als Memo hinterlegt sind. 

Wenn im Dialog Versandanschrift ein Memo erfasst ist und die Versandadresse als Standard (Checkbox Standard ) gekennzeichnet ist, so übersteuert dies im Auftrag das Versandmemo aus dem Kundenstamm! 

Siehe Kapitel Versandanschrift . 

### Feld Pickinfo 
Dieses Feld steht für eine Anzeige am Scanner zur Verfügung. Hier können Informationen für den Lagermitarbeiter eingetragen werden.
