# Parameter Kasse 
Parameter Kasse 

Dieser Dialog öffnet sich im Navigator über System >> Kasse >> Parameter Kasse . Hier können Sie Kürzel für Gutscheine und Rechnungen definieren. Diese Kürzel dürfen keine Leerzeichen enthalten und werden stellvertretend als Artikelnummern im Kassiervorgang genutzt. 

Sind bereits Einträge von Kürzeln im Kassenjournal vorhanden, können keine Änderungen an Kürzeln mehr vorgenommen werden. Dies dient der Sicherheit. Des Weiteren dürfen Kürzel nicht gleich lauten wie Artikelnummern. 

Ebenso können Sie die Artikelnummern für die temporäre Nutzung von Gutschein und Rechnung in der Kasse angeben. Hier müssen zwingend Artikelnummern angegeben werden, welche so nicht im Artikelstamm existieren.
