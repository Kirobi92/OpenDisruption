# Versions-Info 
Versions-Info 

Dieser Dialog öffnet sich im Kopf des eNVenta Start-Dialogs (Navigator) über das Fragezeichen (ganz oben). 

Im Dialog Versions-Info werden Ihnen der Name des verwendeten Packages und die Package-Version angezeigt, die Sie verwenden. Außerdem wird Ihnen angezeigt, wie lange die Lizenz noch gültig ist (als Datum und auch in Tagen). 

Außerdem werden auch Informationen zur Java Runtime Version und FS Version angezeigt. 

Unterhalb der Java Runtime Version wird auch die Version des Crystal Reports angezeigt. 

Die Spalte ‚Kompiliert‘ zeigt an, wann die Version kompiliert wurde und die Spalte ‚Bugfix‘ gibt an, ob es sich um eine Bugfix-Version handelt. Der Stand des Bugfixes kann über das Datum zugeordnet werden. 

Um Lizenzen besser überwachen zu können, gibt es das Feld Lizenz-Information . Neben der Angabe der installierten Packages wird auch die Angabe der Lizenzierung angezeigt. 

### Ausgabe der eNVenta Version in Service Host 
Viele Kunden nutzen eNVenta -Services wie z.B. den JobServer, den WebAccess und viele weitere. Damit beim Aktualisieren der Anwendung (z.B. ServiceRelease einspielen) die Service-Hosts nicht vergessen werden, kann an jedem Service dessen Stand abgefragt werden. 

So kann einfach erkannt werden, ob der Service einen anderen Stand als die Anwendung hat. 

Um den genauen Versionsstand eines Webservice zu ermitteln, kann dies mittels der Methode GetServiceInfo abgefragt werden.

### Feld Versions-Details 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Außerdem werden auch die Datenbank und der Datenbank-Typ angezeigt. 

Beispiel-Info: 
Datenbank: DEV01 
Datenbank-Typ: SqlServer (FrameworkSystems.FrameworkDataProvider 3.8.0.0 | System.Data 4.0.0.0 (4.0.30319.17929)) 

Framework Studio Version: 3.8 Beta3 Intern. 
Java Runtime Version: 1.8.0_05-b13. 

Diese Informationen werden auch bei der Funktion „E-Mail an den Support“ übermittelt, damit möglichst alle Daten vorhanden sind und Ihnen gezielt geholfen werden kann.
