# Übersicht Terminänderung Registerkarte Allgemein

Produktion (PPS) / Übersicht Terminänderung 

# Übersicht Terminänderung Registerkarte Allgemein 
Übersicht Terminänderung Registerkarte Allgemein 

Auf dieser Registerkarte sehen Sie die Informationen entsprechend Ihrer Selektionen. 
### Button Löschen 

Mit diesem Button werden markierte Tabelleinträge entfernt. 

Tooltip: 

Selektierte Zeilen löschen 

### Tabelle 

### Tabellenspalte Betriebsauftrag 
Hier wird die Nummer des Betriebsauftrags angegeben. 

Tooltip: 

Angabe der Betriebsbauftragsnr. 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Firma 1 
Die Firmenbezeichnung wird automatisch vom System eingetragen, nachdem die Lieferantennummer eingetragen wurde. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Alter Starttermin 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Tabellenspalte Alter Endetermin 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Tabellenspalte Neuer Starttermin 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Tabellenspalte Neuer Endetermin 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Tabellenspalte Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Tabellenspalte Änderung 
Hier werden vom Programm automatisch das Datum und die Uhrzeit der letzten Änderung eingetragen. 

Tooltip: 

Datum und Uhrzeit der letzten Änderung 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
