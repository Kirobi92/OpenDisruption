# Definieren von Zusatzfeldern Registerkarte Tabelle 
Zusatzfelder definieren Registerkarte Tabelle 

Auf dieser Registerkarte sehen Sie die bereits angelegten Eigenschaften. 
### Tabelle 

### Tabellenspalte Feldname 
Hier wird der Feldname angegeben. 

Tooltip: 

Angabe des Feldnamens 

### Tabellenspalte Datentyp 
Hier wird der Datentyp angegeben. 

Tooltip: 

Angabe des Datentyps 

### Tabellenspalte Formatierung 
Hier wird die Art der Formatierung angegeben. 

(Optional) Nur für Ganzzahl, Kommazahl, Datum als Formatierungs-Ausdruck 

Der Ausdruck wird bei der String-Umwandlung des Werts als Formatierungsangabe genutzt. 

Beispiele: 
Zahlen: 
"#,##0.00" -> "1.000,00" 
"€ #,##0.00" -> "€ 1.000,00" 
"#.000 mm" -> "10000.000 mm" 
Datum: 
"yyyy.MM.dd" -> "2017.01.31"

### Tabellenspalte Regulärer Ausdruck 
Hier wird der reguläre Ausdruck angegeben. D.h. Details der Zusatzfelder. Wie z.B. ob Buchstaben enthalten sein dürfen, die maximale Länge etc. 

Siehe Kapitel Regulärer Audruck . 

Tooltip: 

Regulärer Ausdruck 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Dieses Feld zeigt die Anzahl der tatsächlich eingelesenen Datensätze.
