# Kapitel NV.ERP.Base.Common.cdParameterFI.sIdeaBufferPath

Hier geben Sie den Pfad für den Idea Export ein, um Dateien temporär auf dem Broker zu speichern. 

Tooltip: 

Dieser Pfad (auf dem Broker) wird beim Idea Export benötigt, um Dateien temporär auf dem Broker zu speichern.
