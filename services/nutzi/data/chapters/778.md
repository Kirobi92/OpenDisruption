# Gutscheinartikel Registerkarte Tabelle 
Gutschein - / Gratisartikel Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Datensätze angezeigt. 
### Tabelle 

### Tabellenspalte Gutschein Nr. 
Hier wird die Gutscheinnummer angezeigt. 

### Tabellenspalte Gutschein Code 
Hier wird der Gutscheincode angezeigt. 

### Tabellenspalte Artikelgruppe 
Hier wird die Artikelgruppe angezeigt. 

### Tabellenspalte Kundengruppe 
Hier wird die Kundengruppe angezeigt. 

### Tabellenspalte Einschränkung 
Hier wird angezeigt, auf was der Gutschein eingeschränkt ist. 

### Tabellenspalte Nur Neukunden 
Wenn die Checkbox in dieser Spalte gesetzt ist, gilt der Gutschein nur für Neukunden. 

### Tabellenspalte Exklusiv 
Wenn die Checkbox in dieser Spalte aktiviert ist, kann pro Warenkorb nur ein Gutschein eingelöst werden. 

### Tabellenspalte Gültig von 
Hier geben Sie das Start-Datum eines Gültigkeitszeitraums ein. 

### Tabellenspalte Gültig bis 
Hier geben Sie das Ende-Datum eines Gültigkeitszeitraums ein. 

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
In diesem Feld wird die Anzahl der tatsächlich gefundenen Datensätze angezeigt.
