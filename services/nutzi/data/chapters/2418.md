# Offene Posten Debitoren Registerkarte Selektionen 4 
Offene Posten Debitor Registerkarte Selektionen 4 

Auf dieser Registerkarte stehen weitere Selektionsmöglichkeiten zur Verfügung. 
### Checkbox Einzugssperre 
Wenn Sie diese Checkbox aktivieren, werden nur offene Posten eingelesen, die auf Einzugssperre gesetzt sind. 

### Checkbox Mahnsperre 
Wenn Sie diese Checkbox aktivieren, werden nur offene Posten eingelesen, die auf Mahnsperre gesetzt sind. 

### Checkbox Kein Zinslauf 
Wenn Sie diese Checkbox aktivieren, wird der OP in Zinsrechnungen nicht berücksichtigt. 

Tooltip: 

Diesen OP nicht in Zinsrechnungen berücksichtigen. 

### Checkbox Kein Kontoauszug 
Wenn Sie diese Checkbox aktivieren, wird der OP, der mit Kein Kontoauszug gekennzeichnet ist, selektiert. 

Tooltip: 

Diesen OP nicht im Kontoauszug berücksichtigen. 

### Checkbox Kein Monatsauf. 
Wenn Sie diese Checkbox aktivieren, wird der OP, der mit Kein Monatsaufsteller gekennzeichnet ist, selektiert. 

Tooltip: 

Diesen OP nicht im Monatsaufsteller berücksichtigen. 

### Checkbox Keine Zentralregulierung 
Wenn Sie diese Checkbox aktivieren, findet keine Zentralregulierung statt. 

Tooltip: 

Keine Zentralregulierung (Verbandsabrechnung bzw. Zentralzahlung) 

### Checkbox Keine Durchschnittsberechnung 
Wenn Sie diese Checkbox aktivieren, findet keine Durschnitts-Außenstanddauerberechnung statt. 

Tooltip: 

Keine Durchschnitts-Außenstanddauerberechnung
