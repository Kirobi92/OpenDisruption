# Kapitel NV.ERP.MM.CashDesk.cdCashDeskParameters.shtAllowCreateCreditVoucher

Wenn dieser Parameter gesetzt ist, kann ein Gutschein an der Kasse erstellt und auch verkauft werden. Ist dieser Parameter nicht gesetzt, so können nur bereits existierende Gutscheine als Zahlungsmittel in der Kasse verwendet werden.
