# Produktionsauftrag 
Produktionsauftrag 

Ein Produktionsauftrag umfasst einen kompletten mehrstufigen Produktionsablauf und beinhaltet beliebig viele hierarchisch organisierte Betriebsaufträge. Ein Produktionsauftrag entspricht in der Regel einer kompletten Stückliste. 

Ein Produktionsauftrag kann direkt aus einem Kundenauftrag heraus generiert werden oder aus dem Artikel. Wurde der Produktionsauftrag aus einem Kundenauftrag generiert, so ist die Referenz im Kopf ersichtlich. 

### Erzeugen eines Produktionsauftrags 
Es gibt 2 Möglichkeiten, einen Produktionsauftrag zu erzeugen: 

- Aus einem bereits erfassten Kundenauftrag. Dazu öffnen Sie im Auftrag im Menü Optionen >> Generieren >> Produktionsauftrag erstellen (es öffnet sich der Dialog Prod.-Auftrag erstellen ). Ist bereits ein Produktionsauftrag für die Position vorhanden, wird dieser automatisch aufgerufen. - Aus der Stamm-Stückliste. Es entsteht ein kundenneutraler Produktionsauftrag. 
Der Generierungsdialog Prod. Auftrag erstellen bietet folgende Varianten: 

- „Nur erste Ebene“: In diesem Fall wird der Produktionsauftrag generiert und der Betriebsauftrag nur für die oberste Ebene der Stückliste. Weitere Auflösungsschritte können dann zu einem späteren Zeitpunkt im Produktionsauftrag oder Betriebsauftrag vorgenommen werden. - „Nur Fehlbestände“: Die Stückliste wird über alle Ebenen aufgelöst. Dabei wird für jede Baugruppe mit Fehlmenge ein Betriebsauftrag generiert. Dies ist nur für eine Standard-Produktion sinnvoll. Individuelle Baugruppen können nicht erkannt werden. - „Alle Ebenen“: Es werden grundsätzlich für alle Ebenen der Stückliste die Betriebsaufträge für alle Baugruppen vollständig generiert. 
Für die Fälle 2 und 3 besteht zusätzlich die Option, eine automatische Terminierung durchzuführen. Dabei ist nur eine Vorwärtsterminierung mit Startdatum „Heute“ möglich. 

Wird aus einem Kundenauftrag ein Produktionsauftrag generiert, so wird das Lieferdatum aus der Auftragsposition an den obersten Betriebsauftrag übergeben. 

Bei der Generierung eines Produktionsauftrags aus dem Auftrag heraus, wird die Variante des Arbeitsplans genommen, welche im Dialog angegeben ist. Dies gilt auch für eine Simulation. 

Ist der Produktionsstart zu einem späteren Termin gewünscht, hat dies im Produktionsauftrag festgelegt zu werden.

### Button Drucken 

Mit diesem Button können Sie den Produktionsauftrag sowie eine Liste der Betriebsaufträge mit Strukturinformationen an den Report übergeben. 

Ist der Produktionsauftrag eine Simulation, muss der Druck als solcher gekennzeichnet sein. 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Über diesen Button wird der komplette mehrstufige Produktionsablauf mit allen Betriebsaufträgen gelöscht. Alle abhängigen Reservierungen und Betriebsaufträge werden gelöscht. Bereits in Arbeit befindliche Betriebsaufträge werden geparkt und abgeschlossen. In diesem Fall erfolgt keine Löschung des Betriebsauftrags im Datenbestand. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
