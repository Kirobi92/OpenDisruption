# Material disponieren Registerkarte Bestellen 
Material disponieren Registerkarte Bestellen 

Ähnlich wie beim „Bestellen aus dem Auftrag“ kann aus dem Betriebsauftrag eine Disposition/Bestellung ausgelöst werden. Das markierte Material muss dazu mit dem Button in die Liste der vorzunehmenden Dispositionen eingetragen werden. Menge und Termin können geändert werden. Über die Combobox Einkaufart wird die Einkaufart für die Bestellung eingestellt. 

Die generierten Dispositionen oder Bestellpositionen sind direkt mit der Materialposition des Betriebsauftrags verknüpft. Es können auch Baugruppen bestellt werden. 

Die Registerkarte Bestellen enthält in der oberen Tabelle die Komponenten, welche zu bestellen sind. 

Die Artikelnummer und die Bezeichnung werden aus dem Betriebsauftrag übernommen. Die benötigte Menge für den Betriebsauftrag wird automatisch in die Spalte ‚Bestellt‘ übernommen und kann ggf. angepasst werden. Die Spalte ‚Einheit EK‘ zeigt die Mengeneinheit an. In der Spalte ‚Liefertermin‘ kann ein Datum vorgegeben werden oder wird durch die Lieferantenauswahl automatisch gefüllt. In der Spalte ‚Lieferant‘ wird die Lieferantennummer angegeben. Eine Angabe kann über die Lieferantenauswahl (Button L-Auswahl ) oder über den Lieferantenvorschlag (Button L-Vorschlag ) erfolgen. Wenn ein Hauptlieferant im Artikel hinterlegt ist, wird dieser automatisch übernommen. 

Die untere Tabelle zeigt eine Übersicht aller generierten Bestellungen zum Betriebsauftrag. Durch einen Doppelklick in die markierte Zeile kann die zugehörige Bestellung geöffnet werden. 

Um für die Fremdfertigung einer Baugruppe Material beistellen zu können, muss aus der generierten Bestellung heraus ein Beistellauftrag erzeugt werden ( Einkauf >> Bestellung >> Optionen ).

### Button Disponieren 

Dieser Button erstellt eine Disposition für die markierte Zeile. Nach erfolgter Disposition erscheint die Zeile nicht mehr in der Tabelle und das Materiallager wird aktualisiert. 

Tooltip: 

Erstellt eine Disposition 

### Button Bestellen 

Dieser Button erstellt für die ausgewählte Zeile eine Bestellung. Nach erzeugter Bestellung wird die markierte Zeile gelöscht und in die untere Tabelle eingetragen. Das Produktionslager wird aktualisiert. Bei der Erzeugung der Bestellung werden offene Rahmenbestellverträge berücksichtigt. Wenn kein offener Abruf vorhanden ist, wird automatisch ein neuer Abruf generiert. 

Ist der Betriebsauftrag einem Projekt zugeordnet, wird das Projekt bei einer direkten Bestellung in die Bestellung übernommen. 

Tooltip: 

Erstellt eine Bestellung

### Combobox Einkaufsart 
In dieses Feld wird die Einkaufsart aus dem Sachbearbeiterstamm übernommen, wenn entsprechend vorhanden. Sollte im Sachbearbeiterstamm keine Einkaufsart definiert sein, so wird die erste gefundene Einkaufsart vorbelegt. Die Vorbelegung kann vor Auslösung der Disposition/Bestellung manuell geändert werden. 

Tooltip: 

Geben Sie die Einkaufsart an.
