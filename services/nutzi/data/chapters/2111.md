# Merkmale für Teilnehmer zuweisen 
Merkmale zuweisen 

Der Dialog Merkmale zuweisen (Aktion) öffnet sich in der Marketing Aktion auf der Registerkarte Teilnehmer über den Button Merkmale und in der Aktivität – Teilnehmer über den Button Partner Merkmale . 

So können Sie komfortabel allen aktuell eingelesenen Teilnehmern ein Merkmal zuweisen. Es kann entweder ein Merkmal für Kunde oder Partner gewählt werden. 

In diesem Dialog können Merkmale nicht bearbeitet werden! 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Merkmale zuweisen 
Tooltip: 

In diesem Dialog können Sie allen aktuell eingelesenen Teilnehmern ein Merkmal zuweisen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
