# Online Analyse Abfragen 
Online Analyse Abfragen 

Der Button Abfragen in der Online Analyse öffnet den Dialog Online Analyse Abfragen . Hier können gespeicherte Abfragen exportiert oder auch Abfragen aus anderen Systemen importiert werden. Diese Abfragen werden als XML-Dateien gespeichert. 

Die Abfragen sind auf die verschiedenen Cubes aufgeteilt. Klappen Sie einen Cube auf, sehen Sie alle dazu gespeicherten Abfragen. 

Der Pfeilbutton neben einem Feld öffnet den Dialog Filter . Mit diesem Dialog können Sie die Anzeige der Daten nach verschiedenen Kriterien einschränken. 

Aktivieren Sie die Checkbox Filter so stehen Ihnen im Kopfbereich verschiedene Möglichkeiten zur Filterung zur Verfügung. 

Wählen Sie zuerst den Filtertyp aus. Sie können hier zwischen einer Filterung auf Elementeigenschaften (Bezeichnung Filter), Element (Wert Filter) oder Anzahl der Einträge (Top N) auswählen. 

Je nach Auswahl sehen Sie die verschiedenen Möglichkeiten der Filter-Einstellungen. 

Der Bezeichnungsfilter erlaubt die Filterung nach Elementeigenschaften. Jedes Element besitzt in der OLAP-Datenbank einen eindeutigen Schlüssel, ein Artikel besitzt z.B. eine Artikelnummer. Manche Elemente sind über zusammengesetzte Schlüssel zu finden, z.B. bei Einsatz des Mandanten-Managements. In diesem Fall haben Sie neben Key 0 auch den Key 1 als Suchmöglichkeit. Über den Filter NAME können Sie auf die Bezeichnung filtern, wie diese in der Pivot Tabelle steht. 

Als Suchkriterium stehen Ihnen verschiedene Möglichkeiten zur Verfügung, u.a. Gleich, Ungleich, Größer, Kleiner, Zwischen,… 

Bei einer Filterung nach Wert können Sie die verschiedenen Kennzahlen auswählen, nach denen gefiltert werden soll. 

Bei der Filterung nach Anzahl Werte können Sie z.B. nur die ersten 10 Einträge angeben, die ersten 10 % aller Einträge, usw. 

Unterhalb der Filterung haben Sie die Möglichkeit direkt die Auswahl der Werte einzuschränken. Aktivieren Sie hierzu die Checkbox Element Auswahl – es werden Ihnen nun alle Werte der Dimension aufgelistet, welche bereits in der Pivot Tabelle angezeigt werden. Durch aktivieren der Checkbox bleiben diese Dimensionen in der Tabelle angezeigt. 

Über die Checkbox Anzahl Elemente einschränken geben Sie die maximale Anzahl der anzuzeigenden Elemente an. Bei der Angabe von -1 werden alle Werte angezeigt. 

Auch eine Kombination aus Filter, Element Auswahl und Anzahl Elemente ist möglich. 

Bei Klick auf den Button OK wird die Filterung übernommen und die Anzeige wird aktualisiert. 

Wenn Sie mit der rechten Maustaste auf eine Kennzahl oder Dimension klicken, so können Sie die Feldeinstellung beeinflussen. Sie können hier automatisch Summenzeilen, Anzeige des Durchschnitts usw. anzeigen lassen.

Tooltip: 

In diesem Dialog können gespeicherte Abfragen exportiert oder auch Abfragen aus anderen Systemen importiert werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
