# Kapitel NV.ERP.Base.Common.cdSystemMM.shtOrderOnItemsListHeader

Wenn diese Checkbox nicht gesetzt ist, so werden VK-Artikel-Sets komponentenweise bestellt. 

Wennn diese Checkbox gesetzt ist, erfolgt eine Bestellung über die Headerposition des Sets. In diesem Falle muss das Set im Auftrag zuvor aufgelöst werden. Außerdem sind dann keinerlei mengenerhöhende Änderungen am Set mehr möglich. 

Tooltip: 

Hier geht es um das Verhalten bei Bestellungen aus dem Auftrag: 
Ist die Checbox nicht gesetzt, so werden VK-Artikel-Sets komponentenweise bestellt. 
Andernfalls erfolgt eine Bestellung über die Kopf-Position des Sets. 
In diesem Fall muss das Set im Auftrag zuvor aufgelöst werden. 
Außerdem sind dann keinerlei mengenerhöhende Änderungen am Set mehr möglich.
