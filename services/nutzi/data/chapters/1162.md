# Versandkosten kalkulieren 
Versandkosten kalkulieren 

In diesem Dialog können Sie Versand- bzw. Zufahrtskosten festlegen. Dieser Dialog öffnet sich im Auftrag >> Stamm >> Versandkosten . Pro Auftragsart können, abhängig vom Auftragswert, mehrere Kostenpauschalen angelegt werden. 

Die Zuweisung zum Auftrag erfolgt mit Einfügen des Auftrags. Eine automatische Neuberechnung bei Änderung des Auftrags ist nicht möglich. 

Die Versandkostenpauschale hat nach den Versandkosten aus den Modulen Versand und Logistik und dem manuellen Eintrag die niedrigste Priorität.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

### Tabellenspalte Fragen 
Wird die Checkbox in dieser Spalte gesetzt, erscheint in der Auftragserfassung eine Abfrage, ob die Versandkostenpauschale in den Auftrag übernommen werden soll. 

Tooltip: 

Kennzeichnung, ob in der Auftragserfassung eine Abfrage 
zur Übernahme der Pauschale als Portokosten erfolgen soll. 

Wenn nicht gesetzt, wird die Pauschale immer gezogen.
