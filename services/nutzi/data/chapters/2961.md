# Verkaufsdatenbericht 
Verkaufsdatenbericht 

Wenn der Nachrichtentyp SLSRPT aktiviert ist, öffnet sich über den Button im Dialog EDI Teilnehmer neben dem Begriff Art. ausschluss der Dialog Verkaufsdatenbericht . 

In diesem Dialog sind die ausgewählten Artikel und Artikelgruppen aufgelistet. 

Für jede Position werden die Verkaufsdaten ermittelt. 

Die Combobox Intervall bietet verschiedene Zeiträume für die Selektion in der Tabelle, ebenso wie das Feld Letzter Versand . Das Feld Lieferantennr ist aus den EDI-Teilnehmern voreingestellt. 

Wenn Sie im Dialog Verkaufsdatenbericht mit dem Button eine neue Zeile in die Tabelle einfügen, in der Spalte ‚Kategorietyp‘ Artikel auswählen und dann in die Spalte ‚Kategorie Wert‘ klicken, öffnet sich der Dialog Artikelauswahl . 

Wenn Sie dasselbe für eine Artikelgruppe machen, öffnet sich der Dialog Artikelgruppenauswahl . 

Siehe Kapitel Artikelauswahl . 

Siehe Kapitel Artikelgruppenauswahl .

### Button Job-Server 

Mit diesem Button können Sie Jobs konfigurieren. Wenn der Job läuft, erzeugt er lediglich ein Event. Die Verabeitung dieses Events erfolgt jedoch asynchron; nämlich erst beim Lauf des EDI-Adapters. 

Siehe Kapitel Job Verwaltung . 

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Neuer Satz 

Mit diesem Button können Sie der Tabelle einen neuen Eintrag hinzufügen. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Freigeben 

Wenn Sie diesen Button drücken, wird ein Gateway-Event registriert. Dieses Event wird erst beim regulären Durchlauf eines zuständigen EDI-Adapter-Kontexts verarbeitet. 

Tooltip: 

Ein Vermerk für den Export eines Verkaufsdatenberichts wird für das Gateway erzeugt. 
Der Bericht wird für den Versand per EDI freigegeben.

### Feld Zeitraum von 
Hier geben Sie das Datum einer Rechnung ein, ab dem die Verkaufsdaten exportiert werden sollen. 

HINWEIS 

Beim Export der EDI Nachricht, wird dieses Datum automatisch durch das "Zeitraum bis" + 1 Tag ersetzt. 
Wenn kein "Zeitraum bis"-Datum angegeben wurde, wird es durch das Datum des EDI-Exports ersetzt. 

Tooltip: 

Es werden die Verkaufsdaten exportiert, deren Rechnungsdatum ab diesem Datum (inklusive) ist. 

HINWEIS 
Beim Export der EDI Nachricht, wird dieses Datum automatisch durch das "Zeitraum bis" + 1 Tag ersetzt. 
Wenn kein "Zeitraum bis"-Datum angegeben wurde, wird es durch das Datum des EDI-Exports ersetzt. 

### Feld Zeitraum bis 
Wenn dieses Feld leer ist, werden die Verkaufsdaten exportiert, deren Rechnungsdatum kleiner als das aktuelle Datum ist - also bis zu und inklusive dem gestrigen Tag, vom Zeitpunkt des EDI Exports gesehen. 

HINWEIS 
Beim Export der EDI Nachricht wird dieses Datum automatisch geleert. 
Das eingetragene Datum gilt also nur für den nächsten EDI-Export.

### Feld Letzter Versand 
Zur Selektion können Sie hier das entsprechende letzte Versand-Datum eintragen. 

Tooltip: 

Das Datum, an dem der letzte Export des Verkaufsdatenberichts per EDI erfolgte. 

### Feld Lieferant 
Dieses Feld ist aus dem Dialog EDI-Teilnehmer voreingestellt und zeigt die Lieferantennummer.
