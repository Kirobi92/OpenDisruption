# Kosten reproduzieren 
Reproduktion - Buchen 

Dieser Dialog öffnet sich im Buchen Kore >> Buchen >> Kosten reproduzieren . 

Über die Comboboxen Jahr und Periode wählen Sie die Buchungsperiode aus, für welche die Kosten bzw. Leistungen einer Kosten- bzw. Leistungsart reproduziert werden sollen. Aus dem Tagesdatum werden das aktuelle Jahr und die aktuelle Periode bereits vorbelegt. Reproduktion bedeutet - der Saldo einer Kosten- bzw. Leistungsart wird auf eine andere Kosten- bzw. Leistungsart übertragen und kann automatisch angepasst werden. Der Saldo der Kosten- bzw. Leistungsart wird aus dem Kore-Journal berechnet. 

Siehe Kapitel Reproduktionsaufträge . 

Das Buchungsdatum wird automatisch mit dem letzten Tag der gewählten Periode vorbelegt. 

Entweder muss die Checkbox Ist-Werte oder Plan-Werte aktiviert werden. 

Nach der erfolgten Reproduktion erhalten Sie eine Meldung über die Anzahl der Buchungen, die erfolgreich reproduziert wurden.

Tooltip: 

In diesem Dialog können Kosten bzw.Leistungen einer Kosten- oder Leistungsart reproduziert werden.

### Combobox Jahr 
Hier wählen Sie das Geschäftsjahr aus, für welches die Kosten einer Kosten- oder Leistungsart reproduziert werden sollen.

Hier wählen Sie die entsprechende Periode aus.

Die Bezeichnung der Periode wird angezeigt. 

Tooltip: 

Angabe der Bezeichnung

Tooltip: 

Angabe des Buchungsdatums 

### Checkbox Nur löschen 
Wenn Sie diese Checkbox aktivieren, werden bereits gebuchte Reproduktionen der Periode wieder aus dem System gelöscht. 

### Checkbox Ist-Werte 
Wenn Sie diese Checkbox aktivieren, werden die Salden der Ist-Werte der Kosten- bzw. Leistungsarten als Berechnungsgrundlage genommen. 

### Checkbox Planwerte 
Wenn Sie diese Checkbox aktivieren, werden die Salden der Planwerte der Kosten- bzw. Leistungsarten als Berechnungsgrundlage genommen. 

### Combobox Gruppe 
Hier können Sie Reproduktionsgruppen auswählen, in welchen mehrere Reproduktionsschlüssel zusammengefasst werden. Diese werden im Dialog Reproduktionsgruppen angelegt. 

Siehe Kapitel Reproduktionsgruppen . 

### Tabelle Schlüssel 

Hier werden alle Reproduktionsaufträge angezeigt, welche angelegt wurden. Ist eine Reproduktionsgruppe ausgewählt, werden nur die Schlüssel angezeigt, welche dieser Gruppe zugeordnet sind. Aus den sichtbaren Schlüsseln können nun eine oder mehrere ausgewählt werden, welche beim Klick auf den Button berechnet werden.
