# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtUseResolveStorage

Wenn dieser Parameter gesetzt wird, wird die Klärungslagerlogik genutzt. 

Ansonsten wird bei der Bestandskorrektur die Ware nur ausgebucht. Eine nachträgliche Überprüfung ist nicht mehr möglich. 

Tooltip: 

Aktiviert die Klärungslagerlogik
