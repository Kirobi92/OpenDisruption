# Kapitel NV.ERP.Base.Common.cdSystemMM.shtShippingExtProgramID

Hier geben Sie an, welche Schnittstelle für Paketdienstleister genutzt wird. Es stehen nur die Einträge leer und MHP V-LOG Webservice zur Verfügung. 

Tooltip: 

Angabe der Schnittstelle, welche für Paketdienstleister genutzt wird. Entweder CSV-Schnittstelle oder Webservice für die Kommunikation von V-LOG.
