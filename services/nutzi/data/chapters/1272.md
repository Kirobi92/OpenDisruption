# Bestellung erstellen 
Bestellung erstellen 

Dieser Dialog öffnet sich über den Button Bestellung in der Retoure. Hier haben Sie die Möglichkeit für die markierte Position eine Bestellung zu generieren. Sie können hier die Einkaufart, den Lieferanten und eventuell ein Projekt, das zum Lieferanten hinterlegt ist, eingeben/auswählen. Mit dem Button wird die Bestellung generiert. 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Combobox Einkaufsart 
Hier kann entschieden werden, mit welchem Einkaufsprozess die Wiederbeschaffung der Ware erfolgt. So kann z.B. durch Auswahl einer Einkaufart eine Streckenbestellung erzeugt werden, damit der Kunde direkt vom Werk beliefert wird. Ebenso kann die Ware vom eigenen Lager beliefert werden und über die Bestellung das eigene Lager wieder aufgefüllt werden. 

In der erzeugten Bestellung stehen die gleichen Möglichkeiten wie bei einer manuellen Bestellung zur Verfügung. 

Tooltip: 

Geben Sie die Einkaufsart an. 

### Combobox Projekt

Tooltip:
