# Umsatzbetrachtung Registerkarte Selektionen 2 
Umsatzbetrachtung Registerkarte Selektionen 2 

Auf dieser Registerkarte definieren Sie weitere Einstellungen für die Auswertung. 

Die Selektionskriterien Code 1, Code 2 und Code 3 beziehen sich auf die Einträge und Zuordnungen aus dem Kunden bzw. Lieferant . 

### Feld Code 1 
Dieses Feld steht zur individuellen Eingabe zur Verfügung. 

Hier können Daten erfasst werden, für die kein spezielles Feld vorgesehen ist. Die Felder können ebenfalls zur Kundensuche verwendet werden. Jedes Feld kann maximal 80 Zeichen enthalten. 

### Feld Code 2 
Dieses Feld steht zur individuellen Eingabe zur Verfügung. 

Hier können Daten erfasst werden, für die kein spezielles Feld vorgesehen ist. Die Felder können ebenfalls zur Kundensuche verwendet werden. Jedes Feld kann maximal 80 Zeichen enthalten. 

### Feld Code 3 
Dieses Feld steht zur individuellen Eingabe zur Verfügung. 

Hier können Daten erfasst werden, für die kein spezielles Feld vorgesehen ist. Die Felder können ebenfalls zur Kundensuche verwendet werden. Jedes Feld kann maximal 80 Zeichen enthalten. 

### Dock Panel Konten mit Umsatz 0 

Über die Radiobuttons Konten mit Umsatz 0, Ohne, Mit und Nur können Sie steuern, dass Personenkonten, welche die entsprechende Einstellung nicht erfüllen, bei der Selektion nicht berücksichtigt werden. 

### Combobox Niederlassung 
Außerdem besteht die Möglichkeit eine Auswertung auf Niederlassung zu beschränken. Wählen Sie dazu in der Combobox Niederlassung einen Eintrag aus. Wenn ein Sachbearbeiter einer Niederlassung zugeordnet ist, kann er sich trotzdem die Umsätze aller Kreditoren ansehen. Sowie er den Radiobutton Kreditoren anklickt, wird die Combobox Niederlassung auf unzugänglich gesetzt und damit „ausgeschaltet“.
