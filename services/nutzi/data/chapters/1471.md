# Kasseneinstellungen Registerkarte Bezahlen 
Kasseneinstellungen Registerkarte Bezahlen 

Damit die Kasse möglichst nur mit der Tastatur gesteuert werden kann, können auf dieser Registerkarte verschiedene Shortcuts hinterlegt werden. 

Für die Währungen können bis zu 3 verschiedene Shortcuts angelegt werden. Die Shortcuts sind immer <Alt+Ziffer> der gewählten Währung. 

Neben den Shortcuts für die Währungen können auch Shortcuts für die Zahlungsarten hinterlegt werden. Es können bis zu 9 Shortcuts für insgesamt 9 Zahlungsarten hinterlegt werden. 

Pro Shortcut ordnen Sie eine Zahlungsart zu. Die Reihenfolge richtet sich nach der Bedienung Ihrer Kasse, z.B. Shortcut Zahlungsart 1 ist Barzahlung. Im Bezahlvorgang der Kasse kann die Zahlungsart dann mit dem Shortcut F1-F12 gewählt werden. 

Siehe Kapitel Zahlungsart . 
### Combobox [Alt]+[1] 
Mit Hilfe dieser Combobox wird die 1. Währung (Standard-Währung) für die Tastenkombination <Alt+1> festgelegt. 

Tooltip: 

Währung wählen mit Ctrl1 

### Combobox [Alt]+[2] 
Mit Hilfe dieser Combobox wird eine 2. Währung für die Tastenkombination <Alt+2> festgelegt. 

Tooltip: 

Währung wählen mit Ctrl2 

### Combobox [Alt]+[3] 
Mit Hilfe dieser Combobox wird eine 3. Währung für die Tastenkombination <Alt+3> festgelegt. 

Tooltip: 

Währung wählen mit Ctrl3 

### Checkbox Skonto auf Rechnungen erlauben 
Wenn Sie diese Checkbox aktivieren, kann beim Kassieren einer Rechnung ein Skontosatz im Dialog Bezahlen eingegeben werden. 

Achtung: Wenn auf der Rechnung bereits Skonto berücksichtigt wurde, wird der eingegebene Skontosatz zusätzlich abgezogen.
