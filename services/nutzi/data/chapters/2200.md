# Serviceobjekt Registerkarte Allgemein 
Serviceobjekt Registerkarte Allgemein 

Auf dieser Registerkarte werden allgemeine Informationen zum Objekt erfasst. 
### Feld Auftrag 
Hier geben Sie die Nummer eines Auftrags ein. 

Tooltip: 

Angabe der Belegnummer 

### Button Suche 

Mit diesem Button können Sie nach Aufträgen suchen. 

Siehe Kapitel Auftragsauswahl . 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Auftrag. 

### Feld Menge 
Hier geben Sie die Menge ein. Diese Menge hat nur informative Bedeutung. 

Tooltip: 

Angabe der Menge 

### Feld H. Prod. Nr. 
Hier wird die Herstellernummer eingetragen. 

Tooltip: 

Hersteller Produkt-Nummer 

### Combobox Kalk. Basis 
Hier wird die Kalkulationsbasis des Artikels zur Ermittlung des Ertrags angegeben. 

Tooltip: 

Angabe der Kalkulationsbasis zur Ermittlung des Ertrags. 

### Combobox Zuständig 
Hier kann der zuständige Monteur ausgewählt werden. Die Auswahl kann je nach Wohnort, Fähigkeiten etc. erfolgen. 

Tooltip: 

Auswahl des zuständigen Monteurs 

### Combobox Service Einsatzart 
Hier wird die Einsatzart angegeben. 

Tooltip: 

Angabe der Einsatzart 

### Feld Rechnungsmemo 
Hier wird ein individueller Text zur internen Information bezüglich der Rechnung angegeben. 

Tooltip: 

Enthält interne Textinformationen zur Rechnung. 

### Feld Apparatememo 
Hier wird ein individueller Text zur internen Information zu einem Apparat angegeben. 

Tooltip: 

Enthält interne Textinformationen zu einem Apparat. 

### Feld Servicememo 
Hier wird das Servicememo aus dem Serviceobjekt angezeigt. Diesen Text können Sie hier ergänzen. 

Tooltip: 

Enthält interne Textinformationen bezüglich des Services.
