# Kapitel NV.ERP.Base.Common.cdSystemBase.shtBarcodeTestMethod

Hier geben Sie die Methode zur Prüfung des Barcodes bei der Abgabe zur Verschlagwortung für die Archivierung (im Dialog Archiv-Mappe einfügen ) ein. 

Tooltip: 

Festlegung eines Dokumenten-Management-Systems (DMS). 
Damit wird die Barcodenummer anhand von DMS-Ziffern auf ihre Korrektheit hin überprüft.
