# Kapitel NV.ERP.Base.Common.cdSystemMM.shtStartProzWithSave

Ist dieser Parameter gesetzt, so wird nach einem ersten Abschluss bei jedem Klick auf automatisch auch Start Proz. durchgeführt. Das hat den Vorteil, dass z.B. nachträglich erfasste Sets automatisch aufgelöst werden. (vgl. Checkbox Auto. Setauflösung ). Es werden aber auch automatische Bestellvorgänge im Hintergrund ausgelöst, wenn z.B. ein Beschaffungsartikel nachträglich erfasst wurde. 

Achtung: Ist diese Funktion automatisiert, wird mit jedem Speichern die Funktion ausgeführt. Dabei werden z.B. bei neuen Streckenpositionen mit jedem Speichern die Bestellpositionen geneneriert. Dies kann umgangen werden, wenn man erst nach vollständiger Erfassung speichert. 

Tooltip: 

Nach Auftragsabschluss wird mit dem Speichern des Auftrages automatisch der Button "Start Proz" aktiviert. 
Damit werden mit dem Speichern ggfs. notwendige Verbundobjekte und Stücklistenauflösungen vorgenommen. 
Achtung: Ist diese Funktion automatisiert, wird mit jedem Speichern die Funktion ausgeführt. 
Dabei werden z.B. bei neuen Streckenpositionen mit jedem Speichern die Bestellpositionen geneneriert. 
Dies kann umgangen werden, wenn Sie erst nach vollständiger Erfassung speichern.
