# Parameter Auswahlliste 
Parameter Auswahlliste 

Dieser Dialog öffnet sich in verschiedenen Dialogen mit einem Doppelklick in eine Zeile des Dialogs (z.B. Parameter Abschluss ). Je nachdem, welcher Dialog zugrundeliegt, ändert sich die Überschrift des Dialogs. Hier können Sie mehrere Belegarten auswählen und übernehmen. 

Button Alle : es werden alle Einträge markiert. 
Button Keine : an allen Einträgen werden die Haken wieder entfernt. 
Button Aktivieren : Damit wird der Haken für einen markierten Eintrag gesetzt. 
Button Deaktivieren : Damit wird der Haken an einem markierten Eintrag wieder entfernt. 

Falls eine Belegart gelöscht wird, wird geprüft, ob zu dieser Belegart die Abschlusslogik konfiguriert wurde. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Parameter Auswahlliste 
Tooltip: 

In diesem Dialog können Sie mehrere Belegarten auswählen und übernehmen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Auswahl 

Mit diesem Button übernehmen Sie eine oder auch mehrere markierte Belegarten in den zugrunde liegenden Vorgang. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Alle 

Wenn Sie diesen Button drücken, werden alle Einträge ausgewählt. 

Der Button ist nur sichtbar, wenn der Dialog eine Mehrfach-Auswahl zulässt. 

### Button Keine 

Bei einer Einfach-Auswahl wird mit diesem Button die Auswahl des Eintrages entfernt und der Dialog wird automatisch geschlossen. 

Wenn der Dialog eine Mehrfach-Auswahl zulässt, dann werden alle Einträge deaktiviert. 

### Button Aktivieren 

Mit diesem Button werden alle in der Tabelle markierten Einträge ausgewählt. 

Der Button ist nur sichtbar, wenn der Dialog eine Mehrfach-Auswahl zulässt. 

Tooltip: 

Mit diesem Button werden markierte Einträge ausgewählt. 

### Button Deaktivieren 

Mit diesem Button werden alle in der Tabelle markierten Einträge deaktiviert. 

Der Button ist nur sichtbar, wenn der Dialog eine Mehrfach-Auswahl zulässt. 

Tooltip: 

Mit diesem Button werden markierte Einträge deaktiviert. 

### Feld Suchbegriff 
Hier geben Sie einen Begriff ein, um die Suche einzuschränken und damit schneller zu einer Auswahl zu gelangen. 

Der Filter kann gesetzt werden auf 
- Artikelgruppen im Zoom-Element Umsatz / Ertrag - Marketing Aktion im Zoom-Element Meine Verkaufsaktion - Sachbearbeiter im Zoom-Element Offene Aufträge - Kunden im Zoom-Element Offene Aufträge - Kundengruppen im Zoom-Element Kritische Kunden - Projekte im Zoom-Element Überfällige Projekt ToDo‘s - Lagerorte im Zoom-Element Kritische Auftragspositionen - Ressourcengruppen im Zoom-Element Engpässe Ressourcen 

### Button Suche 

Dieser Button löst die Suche aus. 

### Tabelle 

In dieser Tabelle werden die auswählbaren Einträge angezeigt. 

- Einfach-Auswahl: 
Der markierte Eintrag stellt die Auswahl dar. Durch einen Doppelklick auf eine Zeile wird der entsprechende Eintrag ausgewählt und der Dialog automatisch geschlossen. Die Auswahl kann mit dem Button Keine aufgehoben werden. - Mehrfach-Auswahl: 
Mithilfe der Checkbox in der ersten Spalte können Einträge ausgewählt oder deaktiviert werden. Mit den Buttons über der Tabelle kann auch eine Auswahl für mehrere Einträge gleichzeitig getroffen werden. Die getroffene Auswahl wird beim Beenden des Dialogs mit dem Button Auswahl übernommen.
