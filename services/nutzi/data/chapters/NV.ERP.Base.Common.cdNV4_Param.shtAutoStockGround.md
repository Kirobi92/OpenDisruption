# Kapitel NV.ERP.Base.Common.cdNV4_Param.shtAutoStockGround

Aus dieser Combobox wird die automatische Platzzuordnungslogik für chaotische, nicht-LVS-Lager gewählt. Je nach Auswahl zieht dann die automatische Lieferfreigabe aus dem Dialog Parameter Verkauf , Combobox Autom. Plätze . 

Siehe Kapitel Parameter Verkauf . 

Tooltip: 

Mit dieser Checkbox wird die automatische Platzzuordnungslogik für chaotische, nicht-LVS-Lager bestimmt.
