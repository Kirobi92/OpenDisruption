# Projekt Registerkarte Intern 
Projekt Registerkarte Intern 

Auf dieser Registerkarte können Memos zum Projekt hinterlegt werden. Es ist möglich, Kommentare zu erfassen, die nur für Mitarbeiter in Ihrem Haus bestimmt sind. Diese Kommentare werden in das Feld Text eingegeben. Nach dem Speichern wird der Text in das Feld Intern gestellt. Die Kommentare werden chronologisch gespeichert, d.h. der aktuellste Eintrag steht an oberster Stelle. Die bereits erfassten Kommentare können nachträglich nicht mehr geändert werden. 
### Feld Text 
In dieses Feld wird Text für den internen Gebrauch geschrieben. Nach dem Speichern wird dieser Text in fas Feld Intern verschoben. 

### Feld Intern 
Hier steht der Text, nachdem er im Feld Text erfasst und gespeichert wurde.
