# Import ADR Tabelle 
Import ADR Tabelle 

Dieser Dialog öffnet sich über den Button Import ADR Tabelle im Dialog ADR Tabelle A , um die Informationen aus einer elektronisch vorhandenen ADR Tabelle (csv-Datei) für den Import zu verarbeiten. 

Der Import bietet 2 Modi an: 
- Neu Importieren: Alles in der Tabelle löschen und neu importieren. - Aktualisieren: Anhand der Schlüsselkriterien der Tabelle wird geprüft, ob bereits ein Eintrag existiert oder nicht. Existiert ein Eintrag, so wird dieser aktualisiert. Fehlende Daten bzw. neue UN-Nummern werden entsprechend ergänzt. (Schlüsselfelder sind UN-Nummer, Bezeichnung und Verpackungsgruppe.) 
Achtung: Wenn alle Datensätze neu importiert werden, müssen die Artikelzuordnungen neu vorgenommen werden. Beim Import erfolgt keine automatische Zuordnung der Artikel zu UN-Nummern. 

Im Feld Pfad geben Sie an, wo die zu importierende Datei liegt. Hierbei wird der Server Datei-Manager verwendet. 

Im Feld Mapping hinterlegen Sie das Mapping für den Import der Tabelle A. Dieses Mapping müssen Sie zuvor im X-Mapping Editor hinterlegen. Es wird kein Mapping mitgeliefert, da es keine normierte Datei für den Import der Tabelle A gibt. 

In der Combobox Codierung geben Sie die Codierung der zu importierenden Datei an. 

Nachdem Sie alle Angaben getätigt haben, können Sie mittels Klick auf den Button Import den Importvorgang starten.
