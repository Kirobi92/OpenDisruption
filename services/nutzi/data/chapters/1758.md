# Rahmenübersicht in der Bestellung auswählen 
Rahmenübersicht 

Wenn Sie im Dialog Bestellung einen Artikel eingeben, für den beim angegebenen Lieferanten mehrere Rahmenbestellungen angelegt sind, erscheint der Dialog Rahmenübersicht . Hier können Sie sich eine Rahmenbestellung aussuchen, von der dieser zu bestellende Artikel abgerufen werden soll. Dies geschieht mit Markieren der entsprechenden Zeile und dem Button . 

Sind im Bestellvorschlag Artikel in der Spalte ‚RA’ mit einem Häkchen gekennzeichnet, so bestehen für diese Artikel ein oder auch mehrere Rahmenaufträge. Mit dem Button RahmenA lässt sich die Rahmenübersicht aufrufen, in der Sie eine Übersicht erhalten. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Rahmenübersicht 
Tooltip: 

In diesem Dialog können Sie sich eine Rahmenbestellung aussuchen, von der dieser zu bestellende Artikel abgerufen werden soll. 
Für mehr Informationen drücken Sie F1.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button.
