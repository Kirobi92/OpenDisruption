# Umsatzbetrachtung Registerkarte Umsatzrelevant

Finanzbuchhaltung / Umsatzbetrachtung 

# Umsatzbetrachtung Registerkarte Umsatzrelevant 
Umsatzbetrachtung Registerkarte Umsatzrelevant 

Auf dieser Registerkarte verwalten Sie umsatzrelevante Daten. 
### Checkbox Alle umsatzrelevanten Konten 
Wenn Sie diese Checkbox aktivieren, werden alle umsatzrelevanten Konten in die obere Tabelle eingelesen. 

### Tabelle 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Tabellenspalte Saldo FW 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind! 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Tabellenspalte Saldo FW 
Hier wird der Betrag in Fremdwährung angegeben. 

Tooltip: 

Angabe des Betrags in Fremdwährung 

### Button Satz hinzufügen 

Mit diesem Button fügen Sie eine neue Zeile in die Tabelle ein. 

Tooltip: 

Konto hinzufügen 

### Button Löschen 

Mit diesem Button löschen Sie einen Eintrag. 

Tooltip: 

Selektierte Konten entfernen
