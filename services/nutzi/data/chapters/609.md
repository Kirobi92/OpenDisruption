# Update Prüfung Registerkarte Expertenmodus

Grundeinstellungen für eNVenta ERP / Update Prüfung 

# Update Prüfung Registerkarte Expertenmodus 
Update Registerkarte Experten-Modus 

Auf der Registerkarte Experten-Modus werden alle Update-Scripts angezeigt, die existieren. 

Für ein notwendiges DB-Update markieren Sie die Zeile und drücken den Button Ausführen . Das Update wird ausgeführt. 

### Button Details 

Mit dem Button Details wird das Update-Script geöffnet. 

Siehe Kapitel Update-Script . 

### Button Ausführen 

Dieser Button führt das ausgewählte Skript aus. 

Tooltip: 

Führt das ausgewählte Skript aus. 

### Button RowVersion 

Mit Klick auf den Button Row Version erhalten alle Tabellen eine neue Spalte ‚FSROWVERSION‘. Diese Spalte kann manuell gepflegt werden. 

Tooltip: 

FSRowVersion-Spalte manuell in allen Tabellen pflegen. 

### Tabelle 

### Tabellenspalte Package-Name 
Hier wird der Name des Packages angegeben. 

Tooltip: 

Angabe des Package-Namens 

### Tabellenspalte Version 
Hier wird die Version des Packages angegeben. 

Tooltip: 

Angabe der Package-Version 

### Tabellenspalte Script 
Hier wird der Name des Skripts angegeben.
