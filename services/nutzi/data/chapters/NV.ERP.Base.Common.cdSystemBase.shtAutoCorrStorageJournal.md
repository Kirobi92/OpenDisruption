# Kapitel NV.ERP.Base.Common.cdSystemBase.shtAutoCorrStorageJournal

Hier aktivieren Sie eine Kennzeichnung für die Autokorrektur im Lagerjournal. D.h. bei einer Lagerbuchung mit einem lagerspezischen GEK wird am Ende jeder Buchung geprüft, ob der Wert im Lagerjournal mit dem Wert in der Tabelle Lager übereinstimmt. Wenn nicht, erfolgt die automatische Korrektur. Dabei wird der Wert in der Tabelle Lager als korrekt angenommen und die Differenz (Korrektur) wird als eigenständige Buchung im Lagerjournal vermerkt.
