# Feste Schlagwort-Werte 
Feste Schlagwort-Werte 

Dieser Diaolog öffnet sich über System >> Allgemein >> Archiv >> Feste Schlagwort-Werte . Es gibt die Schlagwort-Typen "Text", "Ganzzahl", "Datum", "Dezimalzahl" und "Feste Werte". Logische Schlagwörter, die den Typ "Feste Werte" haben, können dann nur über eine Combobox mit den Werten aus dem Dialog Feste Schlagwort-Werte befüllt werden. 

Dies gilt sowohl für das Einfügen von Archiv-Mappen, als auch für die Suche im Archiv. Als feste Schlagwort-Werte können nur Zeichenfolgen eingegeben werden. Entsprechend sind diese Werte dann 100% kompatibel zu physischen Text-Schlagwörtern. 

Bei den festen Schlagwort-Werten kann über die Checkbox Nicht anzeigen definiert werden, ob diese für die Befüllung der Archiv-Mappe bzw. Archivsuche zur Verfügung stehen. 

Das Schlagwort kann auf ein logisches Archiv festgelegt werden. Dazu steht die Spalte ‚Archiv‘ zur Verfügung. Wenn ein festes Schlagwort ohne individuelle Archivzuordnung eingefügt wird, wird dies für alle Archive zur Verfügung gestellt. 

Wenn Global aktiv ist, wird das Feld bei dem logischen Archiv (in der Spalte 'Global' ist die Checkbox gesetzt) unterdrückt. Auf diesem Weg lassen sich beide Möglichkeiten abbilden: 

- Schlagwort nur für bestimmte Archive freischalten - Globales Schlagwort für bestimmte Archive deaktivieren 
Die Schlagwörter werden nach Archivzuordnung sortiert. Die globalen stehen ganz oben, gefolgt von den aktivierten Archiv-spezifischen, den deaktivierten Archiv-spezifischen und dann die nicht aktivierten. 

Ist Global am festen Schlagwort-Wert gesetzt und ein Archiv gewählt, wird der Schlagwort-Wert bei diesem gewählten Archiv nicht angezeigt. 

Ist Global deaktiviert, wird der Schlagwort-Wert nur auf dem gewählten Archiv angezeigt. 

Wenn Sie in die Spalte ‚Archiv‘ doppelklicken, nachdem Sie eine Zeile markiert haben, die mit Global gekennzeichnet ist, öffnet sich der Dialog Parameter Auswahlliste . Dort entscheiden Sie, welche Schlagwärter ausgeblendet werden. 

Siehe Kapitel Parameter Auswahlliste .
