# Leistungsgruppen 
Leistungsgruppen 

Dieser Dialog öffnet sich im Buchen Kore >> Stamm >> Leistungsgruppren . Über dieses Menü können Sie für Auswertungszwecke Leistungsarten in Gruppen unterteilen. Sie haben die Möglichkeit, eine Struktur mit einer Hauptgruppe (Gruppe 1) und 2 Untergruppen (Gruppen 2 und 3) anzulegen. 

Wenn Sie die Kostenrechnung neu einrichten, legen Sie die Struktur fest, bevor Sie mit dem Erfassen der Leistungsarten beginnen. Damit können Sie im Leistungsartenstamm direkt die korrekten Gruppenzuordnungen vornehmen. Die Einstellungen können aber auch später jederzeit geändert werden. 

Im oberen Teil des Dialogs legen Sie zunächst über die Radiobuttons fest, welchen Gruppenkreis (Gruppen 1 – 3) Sie bearbeiten möchten. 

Wenn Sie Änderungen am Kürzel oder der Bezeichnung bei bereits bestehenden Gruppen vornehmen, beachten Sie bitte, dass diese schon im Leistungsartenstamm hinterlegt sein können. Korrigieren Sie die Eingaben dort entsprechend, damit die Zuordnungen nicht verloren gehen. Dasselbe gilt, wenn Sie Gruppen löschen. 

Wenn Sie die Gruppenstruktur für Ihre Auswertungen nicht nutzen möchten, so legen Sie bitte trotzdem eine Gesamtgruppe in Gruppe 1 an, die Sie dann bei allen Leistungen hinterlegen können, da es sich dabei um ein Muss-Feld handelt.

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
