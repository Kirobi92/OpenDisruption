# Lagerplätze Registerkarte Bestände 
Lagerplätze Registerkarte Bestände 

Auf dieser Registerkarte erhalten Sie eine Übersicht, welche Artikel auf diesem Platz eingelagert sind und welchen Bestand diese Artikel haben. Falls der Artikel über eine Chargennummer verfügt, wird auch diese angezeigt. 

Wenn Sie in eine Zeile der Tabelle klicken, werden auf der Registerkarte Bestände die derzeit eingelagerten Artikel mit ihren Beständen angezeigt. 

Die untere Tabelle zeigt Informationen (Zu- bzw. Abgänge) zur Platzzuordnung. 
### Tabelle 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angezeigt. 

Tooltip: 

Artikelnummer eingeben 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Bestand 
Hier wird der aktuelle Bestand laut Lager angegeben, ohne dass eventuell vorhandene Reservierungen, offene Lagerbuchungen und Bestellungen berücksichtigt werden. 

Tooltip: 

Buchbestand des Artikels wird angezeigt. 

### Tabellenspalte Charge 
Die Chargennummer wird angezeigt. 

Tooltip: 

Angabe der Chargennummer
