# Rabatt Verkauf Registerkarte Gültigkeit 
Rabatt Verkauf Registerkarte Gültigkeit 

Für eine komfortable Rabattverwaltung ist es mit dieser Registerkarte möglich, Gültigkeitszeiträume für Rabatte zu definieren. 

Außerdem kann nach Datum (absteigend) sortiert werden. Dazu klicken Sie auf die Spaltenüberschrift. 

Werden eine Staffelstufe oder eine Gültigkeit geändert, werden die Felder Sachbearbeiter und Änderungsdatum aktualisiert. Damit dies sofort angezeigt werden kann, muss die Rabattverwaltung komplett neu eingelesen werden, wodurch die Anzeige automatisch auf die Registerkarte Allgemein springt. 

Folgende Werte werden übergeben: 
- Gültig ab - Basispreis - Rabatt 1% - Rabatt 2% - Preisgruppe - Standardstaffel 
Die Standardstaffeln müssen dafür auch mit einem Gültig ab erweitert werden. 

Bei einem Zugriff wird immer der Datensatz gesucht, der dem Tagesdatum am nächsten ist. 

Beispiel: Es wurden Rabatte zum 01.10.2018, 01.11.2018 und 01.12.2018 angelegt. Ist das Tagesdatum der 27.11.2018, wird der Rabatt vom 01.11.2018 verwendet! 

Damit nicht zwangsweise immer eine neue Staffel angelegt werden muss, nur, weil z.B. ein neuer Basispreis wirksam werden soll, aktivieren Sie für die Tabelle die Checkbox in der Spalte ‚Standardstaffel’. Ist diese Checkbox gesetzt, wird dieses Gültigkeitsdatum in der Combobox der Registerkarte Staffel nicht angeboten, sondern bei Zugriff die Standardstaffel verwendet. 

Wird eine Gültigkeit als „zu löschen“ markiert, erscheint eine Hinweismeldung, falls noch eine Rabattstaffel zur Gültigkeit vorhanden ist. Der Vorgang kann an dieser Stelle korrekt abgebrochen werden. Beim Löschen der Gültigkeit wird die entsprechende Staffel komplett mit gelöscht. 

Lesen Sie dazu bitte noch den Absatz Hinweis zur Nutzung der Registerkarte Allgemein bzw. Gültigkeit im allgemeinen Dokumentationsteil. 
### Tabelle 

### Tabellenspalte Gültig von 
Hier geben Sie an, ab wann der Rabatt gültig ist. 

### Tabellenspalte Gültig bis 
Hier geben Sie an, bis wann der Rabatt gültig ist. 

### Tabellenspalte Basispreis 
Hier wird der Basispreis angegeben. 

Tooltip: 

Angabe des Basispreises 

### Tabellenspalte Rabatt 1 % 
Hier kann ein Rabatt in % eingetragen werden. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Rabatt 2 % 
Hier kann ein 2. Rabatt in % eingetragen werden. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Preisgruppe 
Wenn eine Preisgruppe angegeben ist, so wird der Preis aus der Preisgruppe in die Konditionierung übernommen. Angegebene Rabatte in der Gültigkeit werden ebenfalls auf den Preis aus der Preisgruppe angewendet. Ein angegebener Basispreis hat eine höhere Priorität als die Preisgruppe. 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

### Tabellenspalte Standardstaffel 
Wenn Sie die Checkbox in dieser Spalte aktivieren, wird das Gültigkeitsdatum in der Combobox der Registerkarte Staffel nicht angeboten; es wird die Standardstaffel verwendet. 

Tooltip: 

Angabe einer Standardstaffel 

### Button OK 

Mit diesem Button speichern Sie die neue Gültigkeit. 

### Button Satz hinzufügen 

Mit diesem Button können Sie eine weitere Gültigkeit anlegen. 

### Button Löschen 

Mit diesem Button löschen Sie eine markierte Zeile.
