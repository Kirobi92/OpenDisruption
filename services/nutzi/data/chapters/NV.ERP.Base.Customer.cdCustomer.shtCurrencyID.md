# Kapitel NV.ERP.Base.Customer.cdCustomer.shtCurrencyID

Hier können Sie die Währung des Kunden festlegen. Wurde zum Kunden bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung des Kunden geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen.
