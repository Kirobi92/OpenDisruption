# Skalierung setzen 
Skalierung setzen 

Der Button Skalierung im Dialog Parameter Sachbearbeiter öffnet diesen Dialog. 

Damit kann jeder Sachbearbeiter die Ansicht der eNVenta -Anwendung für sich anpassen. Der Dialog kann größer oder auch kleiner gestellt werden. Damit ändern sich auch die Schriftgröße der Labels und der eingetragenen Werte. 

Für die Skalierung werden alle Buttons, Felder, Eingaben etc. größer angezeigt, ohne dass sich die Felder überlappen. 

Dafür drücken Sie einen der Buttons oder geben einen individuellen Wert in das Feld Anderer Wert ein. Danach schließen Sie den Dialog und starten die Anwendung neu. Beim erneuten Öffnen der Anwendung erscheint diese in der gewählten Größe. Diese Einstellung bleibt solange gespeichert, bis Sie diese Einstellung wieder ändern. 

Die Einstellungen werden pro PC und Benutzer gespeichert.
