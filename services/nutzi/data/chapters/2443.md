# OP-Verrechnung Registerkarte Beleg-Verrech. Debitoren 
OP-Verrechnung Registerkarte Beleg-Verrech. Debitoren 

Auf dieser Registerkarte werden Belege eingelesen, die den Selektionskriterien im Kopf des Dialogs entsprechen. So kann eine Verrechnung auf Beleg-Ebene vorgenommen werden. 

Verbuchung der offenen Posten 

Wird der Button (Buchen) geklickt, so werden alle offenen Posten der aktiven Registerkarte automatisch über das Ausbuchungskonto ausgeglichen. Offene Skonti auf den einzelnen Konten werden berücksichtigt und eine Skonto-Rückrechnung entsprechend durchgeführt. 

Es erfolgt automatisch eine entsprechende Steuerkorrektur, gemäß den in den OPs hinterlegten Steuerschlüsseln. 

Vollautomatische Verrechnung 

Erfolgte die vollautomatische OP-Verrechnung im Dialog Buchen nicht, haben Sie die Möglichkeit die Beleg-Verrechnung über den Dialog OP-Verrechnung , Registerkarte Beleg-Verrech. Debitoren oder Beleg-Verrech. Kreditoren durchzuführen. 

Hierbei haben Sie auch die Möglichkeit, Abweichungen über die Felder Grenzbetrag oder Grenzprozent zu verwenden. Sind beide Felder gefüllt, muss beides zutreffen. 

Die Belegnummer und das Buchungsdatum für die Buchung der Verrechnung erfassen Sie im Dialog OP-Verrechnung . 

Die Transitkontonummer wird aus dem Feld Autom. OP-Verrech. ( Parameter Fibu , Registerkarte Konten ) vorbelegt oder muss erfasst werden. 

Im Dialog werden zu den Selektionskriterien alle OPs mit gleicher Belegnummer eines Debitors oder Kreditors eingelesen. 

In der Spalte ‚Skonto‘ wird immer der Skonto ausgewiesen. Die Fälligkeit des Skontos wird nicht berücksichtigt. 

In der Spalte ‚Differenz‘ wird Ihnen unter Abzug der Spalte ‚Skontos‘ die verbleibende Abweichung der OPs zur jeweiligen Belegnummer, Grenzbetrag und Grenzprozent angezeigt. 

Wenn Sie Verarbeiten wählen, werden die OPs über das Transitkonto verrechnet. Der Skonto und die Differenz werden miteinander verrechnet und über Skonto ausgebucht. 

Sie können auch Belegnummern löschen, dann erfolgt dieses Mal keine Verrechnung dieser OPs. 
### Button Löschen 

Über diesen Button können Sie einzelne, markierte Zeilen aus der Selektion löschen.
