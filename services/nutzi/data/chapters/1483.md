# Belegauswahl in der Kasse 
Belegauswahl 

Dieser Dialog öffnet sich über den Button Rechnung im Kopf des Dialogs Kassieren . Hier wählen Sie die Rechnung aus, die Sie kassieren möchten. 

Der Betrag der Rechnung bzw. Gutschrift wird als Position in den Kassiervorgang eingetragen. Hierfür wird die „Schatten-Artikelnr B1000“ verwendet und der Gesamtwert der Rechnung bzw. Gutschrift übernommen. 

Beim Kassieren auf Rechnung und/oder Lieferschein erscheint der Druck-Dialog (Menü Druck der Auftragsbelege ). Mit öffnen des Druck-Dialogs wird der Beleg abgeschlossen. Ansonsten kann die Rechnung nicht an der Kasse gedruckt werden und der Auftrag muss manuell geöffnet werden.

### Field Panel Auswahl 

Die Auswahl, welches Skonto gezogen werden soll, kann mit Hilfe eines Klicks auf den passenden Radiobutton ausgeführt werden. Das System rechnet dann automatisch den Rechnungsbetrag aus. Es wird automatisch der aktuell gültige Skontowert vorbelegt. Dies kann z.B. hilfreich sein, wenn ein Kunde plötzlich doch bezahlen will und Sie diesem nun aus Kulanz einen höheren Skontobetrag gutschreiben. 

Wenn Sie den Radiobutton Kein Skonto aktivieren, wird kein Skonto berücksichtigt.
