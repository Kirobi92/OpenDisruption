# Online Analyse Registerkarte Abfragen verwalten 
Online Analyse Registerkarte Abfragen verwalten 

Auf dieser Registerkarte werden die Abfragen verwaltet. 
### Button Speichern 

Über diesen Button speichern Sie auf der Registerkarte Abfragen verwalten die aktuelle Abfrage, welche in der Registerkarte Analyse gerade aktiv ist. Hierzu werden die Felder Bezeichnung und Beschreibung aktiv, um die notwendigen Daten zu hinterlegen. 

### Button Satz hinzufügen 

Mit diesem Button wird ein entsprechender neuer Eintrag erzeugt und mit einer Bezeichnung gespeichert. 

### Button Löschen 

Mit diesem Button löschen Sie einen Eintrag. 

### Feld Bezeichnung 
In diesem Feld wird der Eintrag aus der Tabelle 'Bezeichnung' eingetragen. 

### Checkbox Benutzerkontext 
Durch Aktivierung dieser Checkbox kann die Abfrage so eingeschränkt werden, dass nur Sie selbst diese Abfrage zur Auswahl haben und nicht alle anderen Sachbearbeiter. 

### Feld Beschreibung 
Hier kann ein erläuternder Text zur Abfrage hinterlegt werden. 

### Button Html-Editor 

Mit diesem Button können Sie Html-Texte formatieren. 

Siehe Kapitel Html Editor . 

Tooltip: 

Html-Editor zur Bearbeitung von Html-Texten. 

### Button Neu 

Über diesen Button legen Sie eine neue Zeile an und können anschließend die Rolle auswählen, für welche Sie die Berechtigung einschränken möchten. Dazu wählen Sie aus der Combobox in der Spalte ‚Rolle‘ eine Rolle aus und können dazu die Checkboxen in den Spalten ‚Ansicht sperren‘ und ‚Speichern sperren‘ aktivieren. Es stehen alle Rollen aus dem Authentication Service zur Verfügung. 

Tooltip: 

Neu Rolle bzw. Berechtigung für Zoom 

### Button Löschen 

Mit diesem Button können Rollensperrungen gelöscht werden.
