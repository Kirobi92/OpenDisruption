# Call-Statistik Registerkarte Statistik 
Call-Statistik Registerkarte Statistik 

Je nachdem, welche Kriterien festgelegt wurden, wird die Statistik mit dem Button gestartet. Die Werte werden Ihnen in dieser Registerkarte angezeigt. 

Nach einer Änderung der Auswahlkriterien ist die Statistik unbedingt immer neu zu starten. Die Aktualisierung erfolgt nicht automatisch. 

Wenn Sie einen Eintrag auf der Registerkarte Statistik markieren, werden Ihnen die detaillierten Informationen dazu auf der Registerkarte Details angezeigt. 

Wichtiger Hinweis: 

Ein Druck der Statistik ist erst möglich, wenn Sie auf die Registerkarte Chart gewechselt haben! 
### Feld Titel 
Hier wird der Titel der Leistungsstatistik angegeben. 

Tooltip: 

Angabe des Titels der Leistung 

### Feld Von / Bis 
Hier wird der Zeitraum angezeigt, den Sie im Kopf des Dialogs definiert haben. 

Tooltip: 

Von-Bis-Datum 

### Tabelle 

### Tabellenspalte Periode 
In dieser Spalte wird die Periode / der Zeitraum angezeigt. Der Eintrag hängt davon ab, was Sie als Zeitraum in der gleichnamigen Combobox ausgewählt haben. 

Beispiel: 
Wenn Sie Jahr ausgewählt haben, werden in dieser Spalte die einzelnen Monate aufgelistet. 

### Tabellenspalte Anzahl 
Hier wird die Anzahl der Calls angezeigt. 

### Tabellenspalte Dauer 
Hier wird die Dauer des Vorgangs angegeben. 

### Tabellenspalte Anzahl VJ 
Hier wird die Anzahl der Calls aus dem Vorjahr angezeigt. 

Tooltip: 

Anzahl der Calls aus Vorjahr. 

### Tabellenspalte Dauer VJ 
Hier wird die Dauer des Vorgangs aus dem Vorjahr angezeigt. 

Tooltip: 

Dauer des Vorgangs im Vorjahr. 

### Button Nach oben skalieren 

Wenn Sie diesen Button drücken, können Sie die Statistik wieder auf eine höhere Ebene skalieren (wenn diese zuvor herunterskaliert war). 

Tooltip:

### Button Nach unten skalieren 

Wenn Sie diesen Button drücken, können Sie die Statistik (eine Ebene darunter) genauer betrachten. 

Tooltip: 

Wenn Sie diesen Button drücken, können Sie die Statistik (eine Ebene darunter) genauer betrachten.
