# Kasseneinstellungen Registerkarte Druckeinstellungen 
Kasseneinstellungen Registerkarte Druckeinstellungen 

Auf dieser Registerkarte geben Sie an, welche Einstellungen beim Druck gelten sollen. 

Das Signal zum Öffnen der Kassen-Schublade stellen Sie direkt am Drucker ein (nicht in Escape-Sequenzen). 
### Button Bon Customize 

Dieser Button öffnet den Dialog Report Customizing und bietet die Möglichkeit während der Laufzeit weitere Reports anzulegen. 

Siehe Kapitel Report Customizing .

Bon Customize 

### Button Customize

### Combobox Formular 
Hier wählen Sie ein Formular für den Bon-Druck aus. 

### Combobox Drucker 
Hier kann der Drucker für den Bondruck eingerichtet werden. Dieser wird über den Printserver angesprochen. 

Wenn kein Drucker hinterlegt ist, wird automatisch der Standarddrucker des Sachbearbeiters verwendet. 

Tooltip: 

Angabe des Druckernamens 

### Feld Exemplare 
Mit Hilfe dieser Option kann die Anzahl der zu druckenden Kassenzettel eingestellt werden. 

Hinweis: Bei Eingabe einer 0 wird ein Ausdruck erstellt. 

Tooltip: 

Anzahl Bons für den Druck. 

### Checkbox Automatisch drucken 
Ist diese Option ausgewählt, wird automatisch der Bon bei Abschluss des Bezahl-Vorgangs an den Druckservice übergeben. Ist diese Option nicht aktiviert, kommt der Druckauswahldialog bei Abschluss des Bezahl-Vorgangs. 

Beim Nachdruck eines Bons über den Dialog Bonnachdruck öffnet sich trotz gesetzter Checkbox der Druck-Dialog. So können Sie entscheiden, wie der Nachdruck erfolgen soll. Z.B. als PDF oder E-Mail. 

Tooltip: 

Es wird kein Druckdialog geöffnet, sondern der Beleg wird sofort gedruckt. 

### Checkbox Briefkopf 
Diese Checkbox wird aus dem Dialog Parameter Allgemein (Registerkarte Allgemein ) vorbelegt und kann hier im Einzelfall deaktiviert werden. 

Erläuterung: 

Im Rahmen von Geschäftsprozessen ist es immer wieder notwendig, Belege physisch oder digital als Ausdrucke bereitzustellen. Aus rechtlichen Gründen muss dies bei offiziellen Dokumenten mit bestimmten Informationen (Anschrift, BLZ, …) – in Kopf- und Fußzeile - geschehen. Diese Informationen werden im Folgenden als Briefkopf bezeichnet. 

Dies können Kunden nutzen, die ihre Firmeninformationen direkt mit dem Druck des Belegs auf ein Blanko-Papier drucken möchten. 

Beispielsweise wird im Kommissionieren der Lieferschein über LS-Drucken gedruckt. Deshalb gibt es hier die Checkbox Briefkopf . D.h. in Druckdialogen steht die Checkbox Briefkopf zur Verfügung. Diese wird durch den Sachbearbeiter gesteuert und durch das Client-Side Customizing vorbelegt. Möglicherweise erübrigt sich dann für Sie eine entsprechende Einstellung im Client Side Customizing. 

Im Einzelfall kann in den Druck-Dialogen der einzelnen Vorgänge die Checkbox Briefkopf deaktiviert werden, falls Sie ausnahmsweise auf ein Briefpapier drucken möchten. 

Die Checkbox Briefkopf dient in den Dialogen als Vorbelegung und im automatisierten Druck als Einstellung für den Druck. 

Weitere Beispiele: Kassenbon drucken, Auftragsbestätigung, Betriebsauftrag etc. 

Tooltip: 

Wenn aktiviert, wird das Logo auf den Beleg gedruckt.
