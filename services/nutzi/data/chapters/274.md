# Chargenauswahl 
Chargenauswahl 

Der Dialog Chargenauswahl öffnet sich beispielsweise im Betriebsauftrag auf der Registerkarte Positionen über den Button Charge . Der Artikel aus dem zugrundeliegenden Vorgang ist voreingestellt, ebenso die dort angegebene Menge. 

Hier werden zum ausgewählten Artikel sämtliche Chargen angezeigt, die zu diesem Artikel angelegt wurden (in der Tabelle). 

Pro Position in einem Betriebsauftrag kann nur eine Chargennummer zugeordnet werden. Sind mehrere Chargen für eine Position notwendig, ist diese unbedingt zu splitten. 

Weitere Informationen finden Sie im Kapitel Produktion / Material splitten . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Chargenauswahl 
Tooltip: 

In diesem Dialog werden zum ausgewählten Artikel sämtliche Chargen angezeigt, die zu diesem Artikel angelegt wurden (in der Tabelle). 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter .
