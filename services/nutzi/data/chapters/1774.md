# Positionspflege im Einkauf 
Positionspflege 

Damit im Einkauf (Bestellung, Belastung und Anfrage) mehrere Positionen auf einfache Weise und zur gleichen Zeit geändert werden können, steht der Dialog Positionspflege zur Verfügung. Dieser Dialog öffnet sich, wenn eine oder mehrere Positionen in der Tabelle markiert werden und Sie den o.g. Dialog über Optionen >> Positionspflege öffnen. 

Die geänderten Daten werden mit dem Button in den zugrundeliegenden Vorgang übernommen und erst hier können die Änderungen auch gespeichert werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Positionspflege 
Tooltip: 

In diesem Dialog können mehrere Positionen auf einfache Weise und zur gleichen Zeit geändert werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
