# Rabattpflege Registerkarte Gültigkeit 
Registerkarte Gültigkeit 

Auf dieser Registerkarte werden die Gültigkeiten eingetragen / angezeigt, die angepasst werden können. 
### Tabelle 

### Tabellenspalte Intern ID 
Hier wird eine interne, laufende Nummer angegeben. 

Tooltip: 

Interne laufende Nummer 

### Tabellenspalte Gültig von 
Für die Suche über einen Datumsbereich geben Sie hier einen Gültigkeitsbeginn ein. 

### Tabellenspalte Gültig bis 
Für die Suche über einen Datumsbereich geben Sie hier ein Gültigkeitsende ein. 

### Tabellenspalte Basispreis 
Hier wird der Basispreis angegeben. 

Tooltip: 

Angabe des Basispreises 

### Tabellenspalte Rabatt 1 % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Rabatt 2 % 
Hier wird der Rabatt-/Zuschlagsprozentsatz angegeben. 

Zuschläge haben negative Vorzeichen, Rabatte positive. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Tabellenspalte Preisgruppe 
Hier können Sie eine Brutto-Preisgruppe auswählen, die im zugrundeliegenden Vorgang herangezogen wird und bestimmt, auf welchen Verkaufspreis zugegriffen werden soll. 

Siehe Kapitel Preisgruppen . 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

### Button Satz hinzufügen 

Mit diesem Button öffnet sich eine neue Zeile und eine neue Gültigkeit kann angelegt werden. 

Tooltip: 

Neue Zeile 

### Button Löschen 

Tooltip: 

Zeilen löschen
