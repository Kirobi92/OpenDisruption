# Artikel Registerkarte Bild 
Artikel Registerkarte Bild 

Auf der Registerkarte Bild kann jedem Artikel ein Bild zugeordnet werden. Hierbei unterstützt die Software folgende Bildformate: 

- BMP - GIF - PNG - JPG - JPEG - WMF - TIF 
WICHTIGER HINWEIS: Artikelbilder die größer als 1024x1280px sind, werden bei der Darstellung in eNventa automatisch verkleinert. Dies ist ein ressourcenaufwändiger Prozess, der jedes Mal stattfindet, wenn das Bild aufgerufen wird. 

Daher wird empfohlen keine hochauflösenden Bilder zu verwenden, welche diesen Wert überschreiten. 

Wenn einem Artikel hochauflösende Bilder zugeordnet werden sollen, ist es besser diese dem Bild mit einem Referenz-Link zuzuordnen. 

Alle Bilder werden in der Datenbank gespeichert. 

Die Zuordnung zu den Artikeln wird dabei nicht berührt und bleibt somit immer aktuell. 

Die Zuordnung von Bildern kann z.B. für folgende Aufgaben interessant sein: 

- Einfache Identifizierung der Artikel über ein Bild. - Gestaltung von Druckerzeugnissen durch Einbindung von Bildern. Beispielsweise könnten die Bilder der angebotenen Produkte mit in eine Bestellung aufgenommen werden. 
Gehen Sie bei der Zuordnung eines Bilds folgendermaßen vor: 

Zuerst wird der gewünschte Artikel gesucht und im Artikel angezeigt. 

Über den Button Zuordnen öffnet sich der zugehörige Dialog Dateizuordnung . Es werden alle bereits angelegten Dateien angezeigt. 

Wenn Sie einem Artikel ein Bild zuweisen, bekommt das Bild zwingend den Namen des Artikels. Sollen es mehrere Bilder sein, so haben Sie dem Namen entweder ein Präfix oder Suffix hinzuzufügen. 

Siehe Kapitel Dateizuordnung . 
### Button Zuordnen 

Mit diesem Button können Sie weitere Bilddateien hinzufügen. 

Siehe Kapitel Dateizuordnung . 

Tooltip: 

Weitere Dateien hinzufügen 

### Tabelle 

In diesem Feld wird der Artikel in einem kleinen Bild dargestellt. 

### Feld Pfad 
In diesem Feld wird der Serverpfad angezeigt. 

### Button Standardbild 

Mit diesem Button ist es möglich, ein bestimmtes Artikelbild als Standard zu definieren. Dieses Standardbild wird in der Tabelle der Bilder mit einem blauen Hintergrund farblich hervorgehoben. 

Tooltip: 

Als Standardbild setzen 

### Abbildung Artikelbild 
In diesem Feld wird der Artikel als Bild dargestellt. 

### Feld Def. Bildpfad 
In diesem Feld wird zur Information der Pfad zur Standard-Bilddatei eingetragen bzw. angezeigt. Das bedeutet, das angegebene Bild erscheint überall als 1. bzw. Standardbild zum Artikel. 

Tooltip: 

Der Pfad zur Standardbilddatei wird angegeben.
