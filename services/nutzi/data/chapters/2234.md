# Parameter Webshop Registerkarte Sprachen 
Parameter Webshop Registerkarte Sprachen 

Auf dieser Registerkarte können Sie definieren, welche Sprachen im Webshop angeboten werden sollen. Hier ist zu beachten, dass zwingend eine Sprache als Standard vorbelegt wird. 

Wird eine Ressource in der Sprache nicht gefunden, so wird die Standardsprache gezogen. Kann diese wiederum nicht gezogen werden, so wird stattdessen die numerische ID ausgegeben. 

Iso-Kürzel werden im Sprach-Stamm gepflegt ( System >> Allgemein >> Sprachen ). Es ist wichtig, dass das ISO-Kürzel korrekt gepflegt ist, damit die Umsetzung von dem ISO-Kürzel auf die eNVenta ERP -interne Sprach-ID erfolgen kann. 

Siehe Kapitel Sprachen . 
### Tabelle 

### Tabellenspalte Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Tabellenspalte Iso Kürzel 
Hier wird das ISO-Sprachkennzeichen angegeben. Example: de, en. 

Tooltip: 

Angabe des ISO-Kürzel der Sprache 

### Button Satz hinzufügen 

Über diesen Button können weitere Sprachen hinzugefügt werden. 

### Button Löschen 

Über diesen Button kann eine Sprache aus der Tabelle gelöscht werden.
