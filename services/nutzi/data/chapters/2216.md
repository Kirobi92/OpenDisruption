# Call Registerkarte Allgemein 
Call Registerkarte Allgemein 

Auf dieser Registerkarte werden allgemeine Call-Informationen/Daten erfasst. 

Für Thema und Antwort können mehrere Einträge getrennt voneinander erfasst werden. Diese können auch später noch/wieder geändert werden. Dafür stehen die Tabelle Übersicht und das Feld Text zur Verfügung. 
### Button Neues Thema 

Wenn Sie diesen Button aktivieren, wird in der Tabelle Übersicht eine neue Zeile zur Eingabe bereitgestellt. 

Die Zeile wird dann mit dem Begriff Thema gekennzeichnet. 

Tooltip: 

Erfassen Sie ein eues Thema. 

### Button Neue Antwort 

Wenn Sie diesen Button aktivieren, wird in der Tabelle Übersicht eine neue Zeile zur Erfassung einer Antwort bereitgestellt. Die Zeile wird dann mit dem Begriff Antwort gekennzeichnet. 

Tooltip: 

Neue Antwort erfassen 

### Button Text löschen 

Mit diesem Button wird eine Zeile der Tabelle Übersicht mit dem -Kennzeichen versehen. 

Tooltip: 

Text löschen 

### Feld Call-Betreff 
In diesem Feld kann ein Betreff für den Call gepflegt werden. Dies erleichtert die Suche nach einem speziellen Call. 

Diese Betreff-Zeile wird auch im Navigator auf der Registerkarte Termine eingeblendet, wenn auf der Registerkarte Allgemein ein Eintrag erfolgte. 

Tooltip: 

Angabe des Call-Betreffs (worum gehts?) 

### Tabelle 

In der Tabelle stehen alle Vorgänge zum Call in chronologischer Reihenfolge zur Verfügung. Der aktuellste Eintrag steht an oberster Stelle. So haben Sie die aktuellsten Informationen sofort im Blick. 

Jeder Tabelleneintrag wird gekennzeichnet mit Thema / Antwort, Datum, Zeit und Kunden - bzw. Firmenstamm . 

### Feld Text 
Nach Drücken des Buttons Neues Th. oder Neue Antw. erfassen Sie in diesem Feld den Text. 

Dieses Feld steht auch zur Verfügung, um einen selektierten Eintrag der Tabelle Übersicht zu bearbeiten. 

### Button Suche 

Mit diesem Button können Sie einen Textbaustein auswählen. 

Siehe Kapitel Textbausteine . 

### Button Html-Editor 

Mit diesem Button können Sie Html-Text bearbeiten. 

Siehe Kapitel Html Editor . 

Tooltip: 

Html-Editor zur Bearbeitung von Html-Texten. 

### Feld Start 
Dieses Feld zeigt den Beginn der Bearbeitung des Calls an. 

Tooltip: 

Beginn der Call-Bearbeitung 

### Feld Dauer 
Dieses Feld ist ein Eingabefeld, um die Dauer manuell zu erfassen. 

Tooltip: 

Dauer eines Vorgangs (z.B. Call-Erfassung oder Besuch) 

### Feld Dauer ges. 
Dieses Feld zeigt die gesamte Dauer an, welche für diesen Call benötigt wurde (bis zum Abschluss des Calls). 

Tooltip: 

Gesamtdauer der Call-Bearbeitung 

### Feld Gebühr 
Dieses Feld ermöglicht eine manuelle Gebührenerfassung. 

Tooltip: 

Manuelle Erfassung einer Gebühr 

### Feld Gebühr ges. 
Dieses Feld zeigt die Gesamtgebühren des Calls an; es werden alle Bearbeitungszeiten summiert. 

Tooltip: 

Gesamtgebühren des Calls.
