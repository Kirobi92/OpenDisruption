# Simulation Registerkarte Materialbedarf 
Simulation Registerkarte Materialbedarf 

Auf dieser Registerkarte wird für den im Baum aktivierten Betriebsauftrag und seine Unterstufen eine Auflösung des Materials vorgenommen / simuliert. Auf diese Weise kann bereits während der Vorbereitung von Betriebsaufträgen der zu erwartende Bedarf eines Artikels ermittelt werden. 

Da es bei einer Simulation keine Reservierungen des Materials gibt, muss das Ergebnis um die Bedarfsmenge der Simulation korrigiert werden. 

Der Materialbedarf kann nur für aufgelöste (generierte) Betriebsaufträge ermittelt werden. Wenn Unteraufträge nicht generiert sind, können diese nicht berücksichtigt werden, bzw. kann der Materialbedarf nur auf Baugruppenebene ermittelt werden.

### Tabellenspalte Meldebestand 
Hier wird eine Menge für den Meldebestand angegeben. Dieser Meldebestand ist wichtig für die Lagerdisposition. Wird der Meldebestand unterschritten, entsteht je nach Dispositionsverfahren ein Bestellvorschlag. 

Wird der Meldebestand versehentlich kleiner angegeben als der Mindestbestand, so erscheint eine Meldung. 

Tooltip: 

Wird die hier angegebene Menge erreicht, wird eine Meldung ausgegeben.
