# Lademittelzuordnung 
Artikel / LHM-Zuordnung 

Dieser Dialog öffnet sich über den Button LHM im Artikelstamm. Hier können Sie den Artikeln Ladehilfsmittel zuordnen. 

Die LHMs werden bei der Lagerung und Festlegung von möglichen Lagermengen im Lager benötigt. Diese Einteilung gibt an, welche Menge eines Artikels auf einen bestimmten Lagerplatz eingelagert werden kann. Dies ist für die Bestimmung des Lagerplatzes im Lager wichtig. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Artikel / LHM-Zuordnung 
Tooltip: 

In diesem Dialog können Sie den Artikeln Ladehilfsmittel zuordnen. 
Für mehr Informationen drücken Sie F1.

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button.
