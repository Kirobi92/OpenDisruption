# Auswertungsstruktur 
Auswertungsstruktur Fibu 

Dieser Dialog öffnet sich im Buchen >> Stamm >> Auswertungsstruktur Fibu . Der Aufbau einer solchen Struktur sieht wie eine Baumstruktur aus. Jeder Baum ist eine Auswertung und jeder Baum hat natürlich auch diverse Äste. Der oberste Ast ist die Stufe, danach kommt das Detail und unter dem Detail sitzen dann die entsprechenden zugewiesenen Konten. Dazwischen können sowohl Leerzeilen als auch Texte liegen. Die Summen und Ergebnisse werden über die Stufen und Zuordnungen gesteuert. 

Nachdem Sie auf der entsprechenden Registerkarte eine Auswertung (Combobox Auswertung ) ausgewählt haben, wird auf der linken Seite des Dialogs Auswertungsstruktur Fibu die Struktur der Auswertung mit den zugeordneten Konten dargestellt. Mit einem Doppelklick auf die Ebene oder Klick auf das „+“-Symbol vor einer Ebene öffnen Sie die Ebene und sehen die untergeordneten Ebenen dazu. 

Auf der rechten Seite des Dialogs erfolgt im Wesentlichen der Aufbau der Auswertung. 

Über die Auswahl der Registerkarte bestimmen Sie, was für eine Auswertung aufgebaut oder angepasst werden soll. Sie können hier wählen zwischen Bilanz, G+V, UVA und Saldenauswertung. Der Aufbau der Struktur erfolgt für jede Art der Auswertung ähnlich. 

Über die Combobox Rechn.leg.kreis können Sie der jeweiligen Auswertung einen bestimmten Rechnungslegungskreis zuweisen. 

Um in der Auswertungsstruktur einen Rechnungslegungskreis anzugeben, müssen Sie mit dem Button Neue Auswertung den Dialog Neue Auswertung öffnen. Nur hier können Sie einen Rechnungslegungskreis zuordnen. 

Beim Kopieren einer Auswertung ist der Rechnungslegungskreis nicht identisch. Im Dialog Neue Auswertung ist jetzt die Checkbox mit Details unzugänglich, d.h. der Baum wird kopiert, die Konten aber nicht. 

Der Rechnungslegungskreis wird in der Auswertungsstruktur in einer unzugänglichen Combobox angezeigt. Die Auswertungsstruktur zeigt in der Tabelle der Konten (rechte Seite, Mitte) nur die zum Rechnungslegungskreis zugeordneten Konten. 

Beim Abrufen der Auswertung (Saldenauswertung, Bilanz oder G + V) erfolgt eine Prüfung auf falsch zugeordnete Konten. 

Im Feld Zeilentext legen Sie den Namen für eine Stufe, einen Text oder ein Detail fest. 

Summentexte können individuell im Feld Summentext erfasst werden. Wird hier nichts eingetragen, steht in der Auswertung (im Dialog und im Report) kein Text beim Ergebnis. 

Wenn Sie den Button neben dem Feld 100% Basis-Text drücken, wird immer die nächste Zeile mit einer Basis in der Auswertungs-Struktur gesucht. Ist die Suche unten angekommen, wird von oben gesucht. 

Alle Auswertungen, die mit Prozentwerten dargestellt werden sollen, müssen in der Registerkarte Saldenauswertung angelegt werden. 

Wird eine Auswertung über alle Mandanten gestartet, so werden nur Kontonummern eingelesen, keine Kontobezeichnungen. 

Wichtiger Tipp: Speichern Sie immer wieder während des Aufbaus der Auswertung!

### Button Löschen 

Gelöscht wird, was in der Tabelle markiert ist. Dabei ist folgendes zu beachten: 

- Markierung der obersten Ebene: Die gesamte Auswertung wird gelöscht. - Markierung einer Ebene mit nachgeordneten Ebenen: Ab der Markierung wird der gesamte untergeordnete Zweig gelöscht. - Markierung eines Kontos: Das markierte Konto wird links im Strukturbaum entfernt und wieder in der Kontentabelle (rechts) aufgenommen. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern!

Tooltip: 

Item entfernen 

### Button Nach links übertragen 

Mit diesem Button (Hinzufügen) können Sie die Konten der Kontentabellen (rechts) einer Detail-Ebene hinzufügen. Sie haben die Möglichkeit in der Tabelle mehrere Konten blockweise oder selektiv zu markieren und zuzuordnen. Hat die gewählte Ebene nicht die Eigenschaft ‚Detail‘, kann der Button nicht genutzt werden. Das Hinzufügen erfolgt in der gewählten (Detail-) Ebene am Ende, die numerische Einsortierung erfolgt nicht. Soll das Konto numerisch einsortiert sein, müssen Sie das Konto mit den Buttons an die gewünschte Position schieben.
