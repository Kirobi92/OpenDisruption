# Kapitel NV.ERP.MM.Sales.cdOrderType.lngNumRangeNumber

Hier wird eine auf Auftragsartebene abweichende Nummerierung eingegeben. Dies ist eine spezielle Variante zum Thema Nummernkreise. Sie kann z.B. genutzt werden, wenn bestimmte Vorgänge anhand ihres Nummernkreises erkannt werden sollen/müssen. 

Beispiel: Bei SHK z.B. die Badwelt.
