# Kundenmietobjekt Registerkarte Allgemein 
Kundenmietobjekt Registerkarte Allgemein 

Auf dieser Registerkarte können Kundenpreise für das Mietobjekt festgelegt werden. 
### Combobox Periode 
Hier wird die Abrechnungsperiode angegeben. Diese definiert das Abrechnungsintervall, wenn die Checkbox Dynamische Abrechnungsperioden nicht gesetzt ist. 

Ist die Checkbox gesetzt, definiert die Combobox das "Start-Abrechnungsintervall". So kann per Vorfakturierung z.B. eine Wochenabrechnung erfolgen, das Mietobjekt aber erst nach einem Monat zurückgegeben werden. 

Die nächste Rechnung würde dann nach dem nächst höheren Intervall gestellt werden. 

Als Abrechnungsperiode stehen folgende Einträge zur Verfügung: 

- Tag - Woche - Monat - 0,5 Tag 
Die Abrechnungsperiode kann bereits im Kundenmietobjekt vorgegeben werden. 

Siehe Kapitel Kundenmietobjekt .

### Checkbox Dynamische Abrechnungsperioden 
Wenn Sie diese Checkbox aktivieren, werden die Abrechnungsperioden dynamisch ermittelt. D.h. wenn diese Checkbox gesetzt ist, wird der Eintrag in der Combobox Periode als Startintervall für die erste Faktura genutzt. Das tatsächliche Mietintervall wird anhand der tatsächlichen Vermietdauer ermittelt und zum Zeitpunkt der jeweiligen Faktura die jeweils passende Abrechnungsperiode mit dem zur Periode hinterlegten Betrag ermittelt und berechnet.
