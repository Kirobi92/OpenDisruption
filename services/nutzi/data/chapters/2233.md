# Parameter Webshop Registerkarte Ressourcen 
Parameter Webshop Registerkarte Ressourcen 

Hier definieren Sie Ressourcen für den Webshop. 

Die Angabe des Bildpfades wird als Präfix genutzt für die Bildpfade der Artikel aus der Katalogverwaltung. 

Der Bildpfad ist eine Ergänzungsmöglichkeit für Bildpfade, welche in der Katalogverwaltung gepflegt werden. 

Beispiel: 
Das Bild liegt unter 
http://meineDomain.de/shop/images/haushaltsgeraete/backofen.jpg. 

Der Bildpfad in der Katalogverwaltung ist angegeben als 
haushaltsgeraete/backofen.jpg 

der Shop ist installiert unter: 
http://meineDomain.de/shop/ 

damit fehlt der Unterordner für die Bilder in der Pfadangabe: 
[Domain] + [Bildpfad] + [Pfad aus Katalog] 
[http://meineDomain.de/shop/] + [images/] + [haushaltsgeraete/backofen.jpg] 

Die Domain wird nicht explizit angegeben, sondern vom Webshop automatisch ermittelt. 

Der Pfad ist relativ zum Wurzelverzeichnis des Webshops. 

Sie können anstatt des relativen Pfads auch einen absoluten Pfad angeben, welcher dem Pfad aus der Katalogverwaltung vorangestellt wird. Der eNVenta ERP-Webshop erkennt dies entsprechend. 

Die Standard-Erweiterung für Bilder ist images/. 

Beispiel: 
[Bildpfad] + [Pfad aus Katalog] 
[c:images] + [haushaltsgeraete/backofen.jpg] 

Das ist insbesondere dann nützlich, wenn mehrere Webshop-Instanzen dieselben Bilder verwenden sollen. Wenn sich der Bild-Pfad allerdings außerhalb des Shop-Verzeichnisses befindet, müssen dem IIS-Benutzer aus dem Application-Pool des Webshops Leserechte auf das Verzeichnis gegeben werden. 

Der Netzwerkbenutzer der Webshop-Konfiguration benötigt entsprechende Leserechte für das Verzeichnis. 
### Feld Bildpfad 
Hier geben Sie den Pfad zur Bilddatei an. 

Tooltip: 

Der Pfad zu der Bilddatei wird angegeben. 

### Feld Dokumentpfad 
In diesem Feld muss entweder ein absoluter Pfad ("c:\...") oder ein relativer Pfad ("documents/") eingegeben werden. 

Tooltip: 

In diesem Feld muss entweder ein absoluter Pfad ("c:\\...") oder ein relativer Pfad ("documents/") eingegeben werden.
