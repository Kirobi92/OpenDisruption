# Kapitel NV.ERP.FI.FinancialAcc.PaymentTransactions.cdPaymentParameter.decMinAmountOI

Über dieses Feld können Sie eine Untergrenze festlegen, ab der ein offener Posten erst im Zahlvorschlag angezeigt wird. 

Tooltip: 

Legen Sie eine Untergrenze fest, ab der ein offener Posten erst im Zahlvorschlag angezeigt wird.
