# Auftragsarten Registerkarte Rollenrechte 
Auftragsarten Registerkarte Rollenrechte 

Diese Registerkarte dient dazu, den Auftragsarten und damit den Benutzern entsprechende Rollen/Rechte zuzuordnen. Auf diese Weise können Sie einschränken, dass die Benutzer nur bestimmte Auftragsarten betrachten und/oder einfügen können. Da jeder Sachbearbeiter einer Rolle zugeteilt ist, wird darüber die Berechtigung der Auftragsart zugeteilt, z.B. ist so sichergestellt, dass kein Auszubildender im Verkauf Projektaufträge erfassen kann. 

Werden keine Rollenrechte vergeben, sehen alle Sachbearbeiter alle Aufträge. 

Auch wenn für einen Sachbearbeiter die Rollensperrungen eingerichtet wurden, werden diese Aufträge in den Statistiken berücksichtigt. 

Hat ein Sachbearbeiter mehrere Rollen zugeordnet, so genügt es wenn bereits für eine Rolle der Zugriff gesperrt wurde. 
### Button Neu 

Über diesen Button legen Sie eine neue Zeile an und können anschließend die Rolle auswählen, für welche Sie die Berechtigung einschränken möchten. Dazu wählen Sie aus der Combobox in der Spalte ‚Rolle‘ eine Rolle aus und können dazu die Checkboxen in den Spalten ‚Ansicht sperren‘ und ‚Speichern sperren‘ aktivieren. Es stehen alle Rollen aus dem Authentication Service zur Verfügung. 

### Button Löschen 

Mit diesem Button können Rollensperrungen gelöscht werden. 

### Tabelle Rollenrechte 

### Tabellenspalte Rolle 
Hier geben Sie die Rolle an. 

Tooltip: 

Eine Rolle wird einem Sachbarbeiter zugeordnet. 

### Tabellenspalte Ansicht Sperren 
Beim Setzen der Checkbox in dieser Spalte kann der Sachbearbeiter keine Aufträge dieser Auftragsart betrachten. Daher wird auch automatisch die Checkbox in der Spalte 'Speichern Sperren' gesetzt. 

Tooltip: 

Der Sachbearbeiter kann diesen Vorgang nicht sehen. 

### Tabellenspalte Speichern sperren 
Ist die Checkbox in dieser Spalte gesetzt, kann der Sachbearbeiter die Aufträge dieser Auftragsart betrachten, allerdings keinerlei Änderungen vornehmen. 

Tooltip: 

Der Sachbearbeiter kann die Vorgänge ansehen, aber nicht ändern.
