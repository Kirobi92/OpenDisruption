# EDI Auftrag / Auftragsänderung Registerkarte Artikel Informationen 
Registerkarte Artikel Information 

Auf dieser Registerkarte wird der gefundene Artikel mit weiteren Informationen angezeigt. 
### Feld Gefun. Artikelnr. 
Hier wird die Artikelnummer angezeigt. 

Tooltip: 

Zeigt die Artikelnummer. 

### Button Öffnen 

Mit diesem Button gelangen Sie in den Artikelstamm. 

### Feld Bezeichnung 
Hier wird der Name zur Artikelnummer angezeigt. 

Tooltip: 

Zeigt den Artikelnamen. 

### Feld Auskunft 
Hier wird eine Meldung zum Artikel angezeigt.
