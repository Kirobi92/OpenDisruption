# Gutscheinverwaltung 
Gutscheinverwaltung 

In diesem Menü haben Sie die Möglichkeit Gutscheine zu verwalten. D.h. Sie können sich eine Übersicht über bereits ausgestellte und verbuchte Gutscheine ansehen oder auch neue anlegen. 

Die Suche können Sie über die Kopfdaten einschränken. 

Die Gutscheinart bezieht sich auf die Zahlungsarten wie sie im Menü Zahlungsarten definiert sind. 

Siehe Kapitel Zahlungsarten . 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Checkbox Nicht aktiviert 
Wenn Sie diese Checkbox aktivieren, ist der Gutschein erfasst, kann aber an der Kasse noch nicht zum Bezahlen genutzt werden. 

Die Checkbox kann auch zur Selektion genutzt werden. 

### Checkbox Aktiviert 
Wenn Sie diese Checkbox aktivieren, ist der Gutschein erfasst. D.h. im Dialog Gutschein ist die Checkbox Aktivieren noch nicht gesetzt. 

### Checkbox Benutzt 
Wenn Sie diese Checkbox aktivieren, ist ein Teilbetrag des Gutscheins bereits verwendet worden. 

### Checkbox Geschlossen 
Wenn Sie diese Checkbox aktivieren, ist der verfügbare Betrag = 0. 

### Checkbox Gelöscht 
Wenn Sie diese Checkbox aktivieren, wird der Gutschein in der Tabelle markiert und über den Button gelöscht. Allerdings wird der Datensatz nicht wirklich gelöscht, sondern der Status auf gelöscht =1 gesetzt. 

Ist die Checkbox aktiviert, erscheinen in der Tabelle über den Button auch die gelöschten (dazu wird in die Tabelle eine Spalte eingeblendet, sie dient nur der Anzeige, eine Änderung ist nicht möglich).
