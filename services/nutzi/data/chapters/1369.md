# Ausstellungsübersicht 
Ausstellungsübersicht 

Im Navigator >> Verkauf finden Sie den Eintrag Ausstellungsübersicht . Hier werden nur Angebote angezeigt, die mit Ausstellung gekennzeichnet sind und Sie erhalten eine Übersicht über Ausstellungsberatungen / Angebote. 

Im Kopf des Dialogs stehen Felder zur Selektion der Daten zur Verfügung. Die Tabelle wird entsprechend der Selektionskriterien gefüllt. 

Zur Suche von Daten können Sie auch den Button SQL Suche nutzen. 

Selektionen lassen sich über den Button Einstellungen speichern. 

Die Felder unterhalb der Tabelle zeigen die Auswertungen (Summen) an, wie z.B. die Anzahl der Angebote, die Summe der Beratungsdauer. 

Ein Doppelklick auf einen Tabelleneintrag öffnet den zugrundeliegenden Vorgang (Angebot). 

Die Auswertung kann auf einem Report gedruckt werden, welcher die Selektionen und den Inhalt der Tabelle enthält. 

Dazu ist es notwendig eine Angebotsart mit der Kennzeichnung Ausstellung zu definieren. Dafür aktivieren Sie die gleichnamige Checkbox auf der Registerkarte Allgemein im Dialog Angebotsarten . 

Die Beratungsdauer wird im Angebot auf der Registerkarte Kopfdaten im Feld Beratungsdauer angezeigt. Voraussetzung: die zugrundeliegende Angebotsart ist mit Ausstellung gekennzeichnet. Die Zeit wird angegeben mit hh:mm. 

Siehe Kapitel Angebotsarten . 

Siehe Kapitel Angebot . 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .
