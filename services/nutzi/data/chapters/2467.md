# Differenzmeldung 
Differenzmeldung 

Im Buchen >> OP öffnet sich die Differenzmeldung . Über dieses Menü steuern Sie den Ausdruck der Differenzmeldungen, die Sie beim Ausgleich der offenen Posten erzeugt haben. 

Es werden alle Daten des Ausgleichs an die Differenzmeldung übergeben, sowie alle Daten der ausgeglichenen OPs pro Bezahlvorgang. 

Pro Zahlungsvorgang wird eine Tabelle auf dem Report ausgegeben. 

Es wird immer das höchste Skontodatum angedruckt und der Skontobetrag, welcher am Tag der Ausgleichszahlung gezogen werden dürfte. Die Differenz zwischen Zahlbetrag und Restbetrag wird korrekt ausgewiesen. 

Die gelöschten Differenzmeldungen stehen Ihnen zum Druck nicht mehr zur Verfügung. 

Des Weiteren ist eine Archivierung der Differenzmeldung möglich. Damit die Archivierung korrekt funktioniert, wird jetzt pro Differenzmeldung ein Report gedruckt. Dabei wird nach Kundennummer, Zahldatum und Text gruppiert. 

Differenzmeldungen können auch für Rücklastschriften erstellt werden. 

Nach Beendigung des Drucks erfolgt über einen Dialog automatisch die Aufforderung zu entscheiden, ob die gedruckten Differenzmeldungen als verarbeitet gekennzeichnet werden sollen oder weiter als Vorschlag zur Verfügung stehen.
