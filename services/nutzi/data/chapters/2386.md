# Kontoauswahl Registerkarte Kreditor 
Kontoauswahl Registerkarte Kreditor 

Im Buchen auf der Registerkarte Allgemein öffnet sich die Kontoauswahl automatisch, wenn bei Eingabe von Buchstaben bzw. Ziffern in den Feldern Soll und Haben und anschließendem Klick auf die Enter -Taste, mehrere Treffer möglich sind. 

Auf dieser Registerkarte können Sie nach Kreditorenkonten suchen. Die oberen Felder dienen der Selektion. 
### Feld Konto 
Hier geben Sie für die Suche die Kontonummer eines Kreditors ein. 

Tooltip: 

Das Konto zum Datensatz. 

### Feld Kontobez. 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. Wenn Sie das %-Zeichen voranstellen, kann nach einem zusammenhängenden Teil der Kontobezeichnung gesucht werden. 

Tooltip: 

Kontobezeichnung 

### Feld Jahr 
Hier wird das Jahr angegeben. Auf dieser Registerkarte ist das Feld Jahr nicht änderbar. 

Begründung: Wird ein neues Personenkonto angelegt, wird das Konto automatisch für jedes offene Geschäftsjahr angelegt. Änderungen im Personenkonto sind für alle Geschäftsjahre identisch. 

### Feld Multisuche 
Über dieses Feld lässt sich die Suche erweitern. Dazu muss allerdings in den Parametern der Finanzbuchhaltung im Bereich Kontensuche nach mindestens eine der Checkboxen Teil d. Kontonr. , Firma1 , Firma2 , Firma3 oder Suchname aktiviert sein. 

Siehe Kapitel Parameter Fibu Registerkarte Allgemein . 

Tooltip: 

Suche gemäß Parameter 

### Checkbox Beliebig 
Wird diese Checkbox gesetzt, wird nach einem Datensatz gesucht, bei dem an beliebiger Stelle in dieser Position - innerhalb der „Fundstelle“ - diese Eingabe vorhanden ist. 

Tooltip: 

Suche an beliebiger Position innerhalb der Fundstelle. 

### Feld Suchname 
Hier wird der Suchname angegeben. Außerdem kann hier eine Belegnummer zu einem Vorgang eingetragen werden. Wenn der Vorgang existiert, wird der Vorgang geladen. Die Selektion erfolgt auf das gleichnamige Feld im Lieferantenstamm. 

Tooltip: 

Angabe des Suchnamens 

### Feld Firma 1 
Hier geben Sie die Zeile 1 des Firmennamens an. Die Suche erfolgt auf das gleichnamige Feld im Lieferanten . 

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Firma 2 
Hier können Sie die Zeile 2 des Firmennamens eingeben. Die Suche erfolgt auf das gleichnamige Feld im Lieferanten . 

Tooltip: 

Zeile 2 des Firmennamens. 

### Feld PLZ 
Hier kann zur Selektion die PLZ eingegeben werden. Die Suche erfolgt auf das gleichnamige Feld im Lieferantenstamm. 

Tooltip: 

Postleitzahl des Datensatzes 

### Feld Land 
Hier kann zur Selektion das Land eingegeben werden. Die Suche erfolgt auf das gleichnamige Feld im Lieferantenstamm. 

Tooltip: 

Hier wird das Land des Firmensitzes angegeben. 

### Combobox Lieferantengr. 
Hier können Sie zur Selektion der Daten eine Lieferantengruppe auswählen. 

Tooltip: 

Angabe der Lieferantengruppe 

### Feld Sammelkonto 
Zur Selektion kann hier auch das Sammelkonto eingetragen werden. 

### Combobox Währung 
Zur Selektion kann hier eine Währung ausgewählt werden. 

Tooltip: 

Angabe der Währung
