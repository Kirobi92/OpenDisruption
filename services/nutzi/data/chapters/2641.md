# Anlagenstamm Registerkarte Kostenstellen 
Anlage Registerkarte Kostenstellen 

Auf dieser Registerkarte besteht die Möglichkeit, pro Anlagegut mehrere Kostenstellen zu hinterlegen (in Prozentangaben). Die Hinterlegung von Kostenstellen dient der Verrechnung von Anlagekosten auf verschiedene Kostenstellen. 

Nach Abschluss der Erfassung wird geprüft, ob die Summe aller Prozentangaben 100 % ergibt. 

Einer bereits gebuchten Anlage ohne Kostenstelle kann nachträglich eine Kostenstelle zugeordnet werden. 

In diesem Fall kann nur EINE Kostenstelle gewählt werden! 
### Tabelle 

### Tabellenspalte Kostenstelle 
In dieser Spalte wird die Kostenstelle angegeben. 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung der Kostenstelle angegeben. 

### Tabellenspalte Wert in % 
Hier wird der Wert in % angegeben. 

### Button Neu 

Mit diesem Button öffnet sich der Dialog KST Auswahl . Von dort übernehmen Sie eine oder mehrere Kostenstellennummern (blockweise oder selektiv). 

Siehe Kapitel Kostenstellenauswahl . 

Tooltip: 

Fügt eine Kostenstelle hinzu. 

### Button Löschen 

Mit diesem Button löschen Sie eine Zeile der Tabelle. 

Tooltip: 

Dieser Button löscht eine Kostenstelle. 

### Button Tauschen 

Über diesen Button öffnet sich der Dialog KST Tausch für den Austausch einer Kostenstelle. 

Siehe Kapitel Kostenstellen tauschen . 

Tooltip: 

Tauscht die ausgewählte KSt gegen eine andere.
