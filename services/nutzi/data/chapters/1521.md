# Neuer Vorgang AES 
Neuer Vorgang AES 

Dieses Menü dient der Erfassung eines neuen Exportvorgangs nach AES auf der Basis eines vorhandenen Lieferscheins. Der Aufruf erfolgt über den Button AES im Dialog Kommissionieren oder automatisch nach dem Lieferscheindruck, wenn der Parameter Automat. Übergabe an AES gesetzt ist. 

Werden mehrere Lieferscheine aus einem Kommissioniervorgang erzeugt, wird dieser Dialog für jeden Lieferschein neu geöffnet. 

Ein AES-Vorgang kann nur für Lieferadressen erfasst werden, für die im Menü Länder das Kennzeichen Export-relevant gesetzt ist. 

Siehe Kapitel Länder . 

Der eigentliche Datenexport erfolgt zweistufig. Im ersten Schritt (Button Erfassen ) wird der Lieferschein ausgewertet, d.h. die zugehörigen Adress-, Artikel- und Verpackungsdaten werden validiert, verdichtet und schließlich entsprechend gespeichert. 

Im zweiten Schritt (Button Senden ) können die vorvalidierten Daten per Webservice an den AES-Zielserver gesendet werden. 

Die Felder Lieferschein, Anschrift, Lieferanschrift, (ISO-Landeskürzel), Kunde, TIN (Zollnr) werden aus dem Kunden- bzw. Lieferantenstamm vorbelegt und können nicht geändert werden. 

Wenn Sie alle notwendigen Daten angegeben haben, klicken Sie auf den Button Erfassen . Die Daten sind nun in der Datenbank soweit gespeichert, dass der Export an den AES-Client gestartet werden kann. Sie können dies direkt über den Button Senden starten oder später als Sammelübergabe im Menü Optionen >> AES-Schnittstelle >> AES-Übertragung . 

Siehe Kapitel AES-Übertragung . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Neuer Vorgang AES 
Tooltip: 

In diesem Dialog erfassen Sie einen neuen Exportvorgang nach AES. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Tooltip: 

Angabe der Anschrift 

### Combobox Best.-Land 
Hier wählen Sie das Land aus. Das Bestimmungsland wird mit dem ISO-Länderkürzel der Empfängeradresse vorbelegt. 

Wenn (in Sonderfällen) das Bestimmungsland nicht mit dem Land aus der Empfängeradresse übereinstimmt, kann es in der Combobox Bestimmungsland überschrieben werden. 

Tooltip: 

Wählen Sie das Bestimmungsland aus. 

### Feld TIN des Kunden 
Für eine Anbindung an AES wird hier die TIN-Nummer des Kunden angegeben. Diese wird von der Zollbehörde vergeben. 

Sie ist notwendig für Rücksendungen und ist eine Pflichtangabe.
