# Stücklisten Registerkarte Komponente 
Stückliste Registerkarte Komponente 

Auf dieser Registerkarte erfassen Sie die Positionen zur Stückliste. 
### Button Bestand 

Dieser Button bietet einen Überblick über den Bestand des Artikels. 

Siehe Kapitel Bestandsanzeige . 

### Feld Artikel 
Hier kann der Stückliste ein weiterer Artikel (eine Komponente der Stückliste) zugeordnet werden. 

Wenn Sie im Baum auf der linken Seite einen Artikel markieren, wird seine Artikelnummer hier eingetragen. 

Tooltip: 

Angabe der Artikelnummer 

### Button Suche 

Mit diesem Button können Sie Stücklisten-Artikel suchen. 

Siehe Kapitel Artikelauswahl . 

### Feld Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingelesen, nachdem Sie die Artikelnummer eingetragen haben. 

Tooltip: 

Artikelbezeichnung 

### Feld Pos 
Geben Sie hier bitte die Positionsnummer ein. Die Komponente wird anhand der Positionsnummer an der richtigen Stelle in die Stückliste einsortiert. 

Tooltip: 

Positionsnummer 

### Combobox Variante 
Standardmäßig ist hier eine 0 eingetragen. Die Variante 0 wird als Normal-Wert angesehen. Falls Sie eine Position erfassen möchten, die nur in bestimmten Fällen Gültigkeit haben soll, dann vergeben Sie eine Variantennummer größer 0. 

Wenn für eine Komponente mehrere Varianten existieren, so müssen diese die gleiche Positionsnummer in der Stückliste besitzen, wenn die Varianten untereinander ausgetauscht werden sollen. 

### Feld Menge 
Legen Sie hier die Menge fest, mit der die Komponente in die Stückliste eingehen soll. Die Menge wird immer in Lagereinheit angegeben! 

Tooltip: 

Legen Sie hier die Menge fest, mit der die Komponente in die Stückliste eingehen soll (in Lagereinheit). 

### Feld Menge pro 
Hier wird die Menge pro Lagereinheit angegeben. 

Tooltip: 

Menge pro Lagereinheit 

### Feld Lagereinheit 
Hier wird die Lagereinheit angegeben, d.h. die Einheit in welcher der Artikel auf dem Lager geführt wird. 

Tooltip: 

Angabe der Lagereinheit. Dazugehörige Pickliste heißt „Mengeneinheit“. 

### Feld Disposchwund 
In diesem Feld legen Sie den prozentualen Aufschlag bei der Disposition fest, d.h. die zur Disposition vorgeschlagene Menge entspricht der tatsächlich benötigten Menge + Schwund. 

Tooltip: 

Prozentualer Aufschlag bei der Disposition, d.h. die zur Disposition vorgeschlagene Menge entspricht der tatsächlich benötigten Menge + Schwund. 

### Feld Materialschwund 
Hier können Sie einen voraussichtlichen Materialverlust eingeben. Dadurch können Sie sicherstellen, dass bei der Produktion ein eventuell vorkommender Materialverlust gedeckt ist. 

Tooltip: 

Angabe eines voraussichtl. Materialverlusts. 

### Feld Datum von 
Soll eine Komponente erst ab einem bestimmten Zeitpunkt in die Stückliste einfließen, nur für einen bestimmten Zeitraum gelten, geben Sie hier das Start-Datum ein. 

Tooltip: 

Soll eine Komponente erst ab einem bestimmten Zeitpunkt in die Stückliste einfließen oder 
nur für einen bestimmten Zeitraum gelten, geben Sie hier das Start-Datum ein. 

### Feld Datum bis 
Soll eine Komponente nur bis zu einem bestimmten Zeitpunkt in die Stückliste einfließen, nur für einen bestimmten Zeitraum gelten, geben Sie hier das Ende-Datum ein. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Feld Kalk. Anteil GEK 
Im Zusammenhang mit den Einkaufsstücklisten steht Ihnen dieses Feld zur Verfügung. Hier können Sie einen prozentualen Wert eingeben. Dieser Wert wird beim Generieren der Einkaufsstückliste übernommen und kann für die Bestellposition angepasst werden. Auf Ebene der Bestellposition ist sicherzustellen, dass die Summe immer 100% ergibt. 

Siehe Kapitel Einkaufsstückliste . 

Tooltip: 

Verteilung des GEKs anhand des Kalkulationsanteils auf die Stücklisten Komponenten bei Zerlegung. 

### Checkbox Komponente drucken 
Wenn Sie diese Checkbox aktivieren, wird der Druck der Stücklisten-Komponente aktiviert. 

Tooltip: 

Aktiviert den Druck der Stücklistenkomponente.
