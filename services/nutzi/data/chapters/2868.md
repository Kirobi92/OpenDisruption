# Ressource Registerkarte Leistungserfassung 
Ressource Registerkarte Leistungserfassung 

Auf dieser Registerkarte können einer Ressource verschiedene Leistungsarten zugeordnet werden. 
### Combobox Leistungsart 
Hier wählen Sie die Leistungsart Arbeitszeit aus. 

Tooltip: 

LA-Nr Arbeitszeit 

### Feld Artikel Arb.Zeit 
Für die Leistungserfassung geben Sie hier eine Artikelnummer ein, welche für die Arbeitszeit generiert wurde. 

Tooltip: 

Artikelnummer für Arbeitszeit

Mit diesem Button können Sie einen Artikel suchen.
