# Kapitel NV.ERP.Base.Common.cdParameterFI.shtDebFactoringActive

eNVenta unterstützt das verdeckte Factoring (auch stilles Factoring genannt). 

Verdecktes Factoring bedeutet, dass offene Forderungen an eine Factoringbank verkauft werden, ohne dass der Debitor darüber informiert wird. Diese Factoring-Variante wird einerseits genutzt, um die Beziehungen zu den Kunden nicht zu belasten und andererseits, um Abtretungsverbote zu umgehen. Das Mahnwesen bleibt im Unternehmen angesiedelt. 

Um am stillen Verfahren teilzunehmen hat ein Unternehmen zwingend bestimmte Voraussetzungen zu erfüllen und auch die Besonderheiten dieser Factoring-Variante sind zu berücksichtigen. 

Mit dieser Checkbox wird das Factoring freigeschaltet (aktiviert). Nach dem nächsten Start von eNVenta steht dann 
- im Navigator >> System >> Fibu die Auswahl Factoring Parameter - im Dialog Kunde die Registerkarte Factoring - im Buchen >> OP die Auswahl Factoring zur Verfügung. 

Tooltip: 

Aktiviert das Factoring
