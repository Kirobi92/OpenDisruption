# BMECat - Import Registerkarte Verkäufer 
BMEcat Import Registerkarte Verkäufer 

Die Felder auf dieser Registerkarte zeigen die Informationen aus dem BMECat an. Handelt es sich um einen Lieferantenkatalog, hat der Sachbearbeiter hier zusätzlich die Lieferantennummer anzugeben. 
### Feld Lieferant 
Wenn es sich um einen Lieferantenkatalog handelt, hat der Sachbearbeiter hier zusätzlich die Lieferantennummer anzugeben. 

Tooltip: 

Lieferantennummer 

### Feld GLN/ILN 
Hier wird die globale Identifikationsnummer für EDI-Teilnehmer (für den Austausch von Geschäftsdokumenten) angegeben, früher: ILN. 

Tooltip: 

Globale Identifikationsnummer für EDI-Teilnehmer (für den Austausch von Geschäftsdokumenten), früher: ILN
