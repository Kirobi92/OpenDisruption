# CRM Übersicht Registerkarte Zahlungsverhalten 
CRM Übersicht Registerkarte Zahlungsverhalten 

Auf dieser Registerkarte wird je nach Selektion das Zahlungsverhalten (in Tagen) angezeigt. Da es sich hierbei um einen durchschnittlichen Wert handelt, werden die Ergebnisse auf ganze Zahlen gerundet dargestellt. 
### Combobox Jahr 
Hier wird das (Geschäfts-) Jahr angegeben. 

### Checkbox Drucken 
Wenn Sie diese Checkbox aktivieren, werden die Informationen gedruckt. 

### Tabelle 

### Tabellenspalte Periode 
Hier werden die Perioden zum gewählten Jahr angezeigt. 

Tooltip: 

Bezugszeitraum 

### Tabellenspalte Ist Jahr 
Hier wird das tatsächliche Zahlungsverhalten aus dem aktuellen Jahr angegeben, gerechnet auf die Nettofälligkeit. 

Tooltip: 

Tatsächliche durchschnittliche Zahldauer im angegebenen Jahr, gerechnet auf das Belegdatum. 

### Tabellenspalte Soll 
Hier wird die durchschnittliche Soll-Zahldauer im angegebenen Jahr angezeigt, gerechnet auf die Nettofälligkeit. 

Tooltip: 

Durchschnittliche Soll-Zahldauer im angegebenen Jahr, gerechnet auf die Nettofälligkeit. 

### Tabellenspalte Ist Vorjahr 
Hier wird das tatsächliche Zahlungsverhalten aus dem Vorjahr angegeben (gerechnet auf die Nettofälligkeit). 

Tooltip: 

Tatsächliche durchschnittliche Zahldauer im angegebenen Jahr, gerechnet auf das Belegdatum. 

### Tabellenspalte Soll - 1 
Hier wird die durchschnittliche Soll-Zahldauer angegeben, gerechnet auf Nettofälligkeit. 

Tooltip: 

Durchschnittliche Soll-Zahldauer im angegebenen Jahr, gerechnet auf die Nettofälligkeit. 

### Tabellenspalte Ist Vorvorjahr 
Hier wird das tatsächliche Zahlungsverhalten von vor 2 Jahren angegeben (gerechnet auf die Nettofälligkeit). 

Tooltip: 

Tatsächliche durchschnittliche Zahldauer im angegebenen Jahr, gerechnet auf das Belegdatum. 

### Tabellenspalte Soll - 2 
Hier wird die durchschnittliche Soll-Zahldauer angegeben, gerechnet auf Nettofälligkeit. 

Tooltip: 

Durchschnittliche Soll-Zahldauer im angegebenen Jahr, gerechnet auf die Nettofälligkeit.
