# Marketing-Aktion Registerkarte Teilnehmer 
Marketing-Aktion Registerkarte Teilnehmer 

Diese Registerkarte enthält alle Adressen und Kunden, die an der Marketing-Aktion teilnehmen, bzw. werden hier die Kunden, Adressen etc. eingefügt, die an der Marketing-Aktion teilnehmen sollen. Um Teilnehmer in diese Registerkarte einzufügen, ist es notwendig, den Kundenstamm bzw. Adressstamm zu öffnen und dort über das Menü Optionen >> Aktionsteilnehmer eintragen jeden Teilnehmer einzeln zu übernehmen. Deshalb ist es sehr wichtig, dass die Stammdaten sehr gut gepflegt sind. 

Mit Teilnehmer ist der Ansprechpartner beim Kunden, Interessenten gemeint. Das bedeutet auch, dass Sie mehrere Teilnehmer eines Kunden / einer Adresse in die Aktion einfügen können. 

Über die Comboboxen Letzte Aktivität bzw. Nächst. Aktivität werden die einzelnen Aktivitäten für die Teilnehmer zur Bearbeitung festgelegt. 

In der Tabelle dieser Registerkarte kann nach jeder Spalte selektiert werden. 

Siehe Kapitel Aktionsteilnehmer eintragen . 
### Button Suche 

Mit diesem Button werden alle Teilnehmer in die Tabelle eingelesen. 

Tooltip: 

Liest Teilnehmer ein. 

### Button Leeren 

Dieser Button leert die Tabelle. 

Tooltip: 

Mit diesem Button leeren Sie die Tabelle. 

### Button Löschen 

Dieser Button löscht einzelne Teilnehmer aus der Tabelle. 

Tooltip: 

Löscht einen Teilnehmer 

### Button Ausblenden 

Dieser Button entfernt den Aktionsteilnehmer für die nächste Aktivität in dieser Marketing-Aktion. 

Tooltip: 

Entfernt den Aktionsteilnehmer für die nächste Aktivität. 

### Button Ausführen 

Mit diesem Button verzweigt das Programm in den Dialog Aktivität . 

Siehe Kapitel Aktivität . 

Tooltip: 

Öffnet die Aktivität. 

### Button Übergabe 

Mit diesem Button öffnet sich der Dialog Aktionsauswahl . Es können alle in der Tabelle enthaltenen Teilnehmer an eine weitere Marketing-Aktion übergeben werden. 

Siehe Kapitel Aktionsübersicht . 

Tooltip: 

Übergibt Teilnehmer an eine weitere Marketing-Aktion. 

### Button Serie 

Dieser Button öffnet den Dialog Serienausführung . Dort können einzelne Aktivitäten der Marketing-Aktion gestartet / gedruckt werden. Dieser Button ist nur zugänglich, wenn die Teilnehmer über die "nächste Aktivität" gesucht worden sind. 

Der Serienversand per Email erfolgt über Outlook, so dass Sie hier eine gute Möglichkeit haben, Ihre "Serie" zu überwachen. 

Siehe Kapitel Serienausführung . 

Tooltip: 

Öffnet die Serienausführung. 

### Button Verlauf 

Dieser Button öffnet den Dialog Teilnehmeraktivitätsauswahl . Dort bekommen Sie eine Übersicht über Teilnehmer der Marketing-Aktion. 

Siehe Kapitel Teilnehmeraktivitätsauswahl . 

Tooltip: 

Zeigt Teilnehmer in einer Übersicht. 

### Button Merkmal 

Dieser Button öffnet den Dialog Merkmale zuweisen . Dort können Sie Teilnehmern Merkmale zuordnen. 

Siehe Kapitel Merkmale für Teilnehmer zuweisen . 

Tooltip: 

Ordnet den Teilnehmern Merkmale zu. 

### Combobox Letzte Aktivität 
Mit Auswahl aus dieser Combobox können Sie in der Tabelle der Registerkarte nur Teilnehmer anzeigen lassen, die zuletzt an der ausgewählten Aktivität teilgenommen haben. 

Tooltip: 

Zeigt Teilnehmer der zuletzt ausgewählten Aktion. 

### Combobox Nächste Aktivität 
Über diese Combobox wählen Sie alle Teilnehmer aus, für welche die nächste Aktivität durchgeführt werden soll. 

Tooltip: 

Zeigt Teilnehmer für die nächste Aktivität. 

### Combobox Sachbearbeiter 
Für die Suche der Teilnehmer kann hier der zuständige Benutzer hinterlegt werden. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Feld Teilnehmer Anz. 
In diesem Feld werden die Einträge der Tabelle addiert. 

Tooltip: 

Anzahl der Teilnehmer der Aktion gemäß der Selektion 

### Feld Anzahl max. 
In diesem Feld legen Sie fest, wie viele Teilnehmer aus der Tabelle am nächsten Serienlauf maximal teilnehmen sollen. Dies ermöglicht Ihnen eine Ausführung der Serienverarbeitung in mehreren Läufen. 

Wenn bei einem Serienlauf für E-Mails einige Teilnehmer keine Email-Adresse haben, werden am Ende alle Fehler auf einmal ausgegeben. Der automatische Ablauf wird nicht unterbrochen. 

Tooltip: 

Anzahl der Teilnehmer für den nächsten Serienlauf. 

### Tabelle 

Mehrere Teilnehmer können in dieser Tabelle markiert und anschließend gelöscht oder an eine andere Marketing-Aktion/Aktivität übergeben werden. 
### Tabellenspalte AKL-Nummer 
Hier wird die Nummer des Teilnehmers angezeigt. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Tabellenspalte Firma 1 
Hier wird der Name der Firma wird angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Typ 
Kennzeichnung des Aktionsteilnehmers. A = Adresse, K = Kunde, L = Lieferant. 

### Tabellenspalte Ort 
Hier wird die Anschrift des Teilnehmers angezeigt. 

Tooltip: 

Angabe des Orts 

### Tabellenspalte PLZ 
Mit Klick auf den Spaltenkopf ‚PLZ’ ist es möglich, die Teilnehmer der Marketing-Aktion nach Postleitzahlen zu sortieren. Hier erfolgt die Übergabe an Serien-Aktionen immer entsprechend der Sortierung in der Tabelle. 

Tooltip: 

Postleitzahl des Datensatzes 

### Tabellenspalte Partner 
Hier können Sie einen Ansprechpartner auswählen, der im gleichnamigen Menü hinterlegt wurde. 

Tooltip: 

Nummer des Kontakts/Partners 

### Tabellenspalte E-Mail 
Hier wird die E-Mail-Adresse angezeigt. 

Tooltip: 

Angabe der E-Mail-Adresse. 
Bei der Angabe mehrerer Adressen, müssen diese durch Semikolon getrennt sein. 

### Tabellenspalte Letzter Schritt 
Hier wird die zuletzt für den Teilnehmer ausgeführte Aktivität eingetragen. 

### Tabellenspalte Nächster Schritt 
Hier tragen Sie den nächsten Aktivitätsschritt ein. 

### Tabellenspalte Variante 
Dient als Kennzeichen, wenn eine Aktivität mehrmals durchgeführt wird. 

### Tabellenspalte Sachbearbeiter 
Hier wird der verantwortliche Mitarbeiter angezeigt.
