# CRM Übersicht Registerkarte Umsatz 
CRM Übersicht Registerkarte Umsatz 

Auf dieser Registerkarte werden je nach Selektion die entsprechenden Umsätze angezeigt. Diese können auf entsprechende Zeiträume eingeschränkt werden. 
### Dock Panel Periode 

Über diese Radiobuttons legen Sie fest, welche Periode angezeigt wird. 

### Combobox Jahr bis 
Hier wählen Sie aus, bis zu welchem Jahr die Daten eingelesen werden sollen. 

### Checkbox Drucken 
Wenn Sie diese Checkbox aktivieren, werden die Informationen gedruckt. 

### Tabelle 

### Tabellenspalte Periode 
Je nachdem welchen Radiobutton Sie aktiviert haben, wird hier die entsprechende Periode angezeigt. 

Tooltip: 

Angabe der Periode 

### Tabellenspalte Umsatz 
Hier wird der Umsatz in der gewählten Periode angegeben. 

### Tabellenspalte Ertrag 
Hier wird der Ertrag in der gewählten Periode angegeben. 

### Tabellenspalte Marge 
Hier wird die Marge in der gewählten Periode angegeben. 

### Tabellenspalte Umsatz VJ 
Hier wird der Vorjahres-Umsatz in der gewählten Periode angegeben. 

### Tabellenspalte Diff. % 
Hier wird die prozentuale Differenz in der gewählten Periode angegeben.
