# Parameter Kasse Registerkarte Allgemein 
Parameter Kasse Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die Kasseneinstellungen, wie z.B. Kürzel für Gutscheine und Rechnungen. 
### Feld Gutscheinkürzel 
Hier geben Sie ein Gutscheinkürzel für die Kasse an. 

Tooltip: 

Geben Sie ein Kennzeichen für Gutscheine ein. 

### Feld Rechnungskürzel 
Hier geben Sie ein Kennzeichen ein, mit dem die Rechnungen in der Artikelliste und im Kassenjournal identifiziert werden können. 

Tooltip: 

Geben Sie ein Kennzeichen für Rechnungen ein. 

### Feld OP-Kürzel 
Hier geben Sie ein Kürzel für einen OP ein. Dieses Kürzel wird in das Feld Artikel . im Dialog Kassieren eingetragen. 

Dieses Kürzel bezieht sich auf die Möglichkeit, im Dialog Kassieren einen OP über den Button OP auszuwählen. Damit ist es möglich an der Kasse auch einen OP, der in der Fibu manuell erzeugt wurde, an der Kasse auszuwählen und über Auszug-/Kassenimport auszugleichen. Das geht z.B. für das Kassieren einer Rücklastschrift oder Zinsrechnung. 

Beispiel: 

Ein Kunde (Handwerker) kauft im Laden ein und an der Kasse fällt ihm ein, dass noch eine Rechnung (z.B. eine Zinsrechnung) offen ist, die er hier und jetzt gleich mit bezahlen möchte. 

Tooltip: 

Kennzeichen, mit welchem offene Posten (Fibu) in der Artikelliste und im Kassenjournal identifiziert werden können. 

### Combobox Steuerschlüssel Rechnung 
Hier geben Sie den Steuerschlüssel an, für das Kassieren bestehender Rechnungen an der Kasse. 

Tooltip: 

Geben Sie den Steuerschlüssel für Rechnungen ein. 

### Checkbox Stornogrund 
Wird dieser Parameter aktiviert, muss bei einem Storno eine Begründung eingegeben werden. 

Die Kasseneinstellungen werden pro Kasse vorgenommen, die Kassenparameter gelten für alle Kassen. 

Tooltip: 

Wenn gesetzt, muss bei Storno eine Begründung eingegeben werden. 

### Checkbox Eingeschränkte Kassenauswahl 
Wenn Sie diese Checkbox aktivieren, werden Kassen für bestimmte Abteilungen oder Sachbearbeiter festgelegt. Auf diese Weise sind für die Sachbearbeiter nur die zugewiesenen Kassen, z.B. der eigenen Filiale, zugänglich und keine fremden Kassen, was zu Fehldaten führen könnte. 

Wenn Sie diese Einstellung gespeichert haben und den Dialog Kasseneinstellungen öffnen, werden auf der Registerkarte Allgemein die Combobox Abteilung und die Listbox Sachbearbeiter angezeigt und zugänglich. 

Siehe Kapitel Kasseneinstellungen . 

Tooltip: 

Nur Kassen zur Auswahl stellen, die dem Sachbearbeiter oder dessen Abteilung zugeordnet sind.
