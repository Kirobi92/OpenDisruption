# BDE Journal Zeit 
BDE Journal Zeit 

Diesen Dialog finden Sie im Betriebsauftrag >> Optionen >> BDE Journal Zeit . In diesem Dialog können zeitbezogene Arbeitsgangdaten recherchiert werden. 

Solange der Betriebsauftrag noch nicht abgeschlossen ist, können der Grund und die gebuchte Zeit geändert werden. 

Für das Ändern der Zeit gilt: 

- Je nach Typ der Buchungszeile wird die Zeit von oder die Zeit bis neu bestimmt. - Es darf keine Zeitüberschneidung mit vorheriger oder nachfolgender Buchung stattfinden. - Eine neue Reservierung für diesen Zeitraum muss vorgenommen werden. - Die benötigte Zeit wird neu berechnet und ausgewiesen. 
Bei Abschluss des Arbeitsganges durch den Wareneingang wird in der Spalte 'Menge gut' die Summe der akzeptierten Menge aus der Bestellposition / Bestellpositionen eingetragen. In 'Ausschuss' wird die Differenz zwischen bestellter Menge und gelieferter Menge eingetragen. Diese Mengen können nachträglich korrigiert werden. 

Die Änderung einer Buchungszeile wird in den Spalten ‚Sachbearbeiter‘ und ‚Änderung‘ gespeichert. 

Die Zeit wird nicht für jede Zeile errechnet, sondern nur für die markierte Zeile (unterhalb der Tabelle), zusammen mit der Zeitspanne (Felder Datum/Zeit von-bis , Kapazität ).

### Button Pos. löschen 

Mit diesem Button ist es möglich, geparkte Betriebsaufträge zu löschen, da diese die BDE-Buchungen verhindern bzw. erschweren. 

Der Button wird erst zugänglich, wenn es sich bei der Position in der Tabelle um eine geparkte Position handelt. 

Bevor die Position tatsächlich gelöscht wird, erscheint noch eine entsprechende Abfrage. Wird diese Meldung bestätigt, wird der markierte Betriebsauftrag gelöscht und das BDE Journal wird aktualisiert. 

Tooltip: 

Position Löschen - nur mit Buchungsart="Geparkt / Term. aufgehoben".
