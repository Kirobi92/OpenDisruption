# Vermietung Registerkarte Artikelinfo

Miete / Vermietung / Vermietung Registerkarte Positionen 

# Vermietung Registerkarte Artikelinfo 
Registerkarte Artikelinfo 

Auf dieser Registerkarte werden die Artikelnummer und die Mietobjektnummer angezeigt. 
### Feld Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Artikelstamm. 

### Feld Mietobjekt 
Hier wird die Nummer des Mietobjekts angegeben. 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in das Mietobjekt. 

### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Mietobjekt-Memo
