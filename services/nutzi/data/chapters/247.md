# Text-Automatisierung 
Für die Text-Automatisierung können Zahlen- und Datumswerte formatiert werden. Das funktioniert, indem in einem Feld - durch Semikolon getrennt - das Format angegeben wird. 

Beispiel: 

Callnr ohne Format: </oObj.lngCallID/> => 2017093636 
Callnr mit Format: </oObj.lngCallID ; #,##0/> => 2.017.093.636 
Datum mit Format: </oObj.dtEntryDate ; d. MMMM yyyy, h:mm/> => 3. Februar 2017, 7:15
