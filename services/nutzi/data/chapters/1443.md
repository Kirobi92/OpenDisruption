# Vermietungsposition Registerkarte Preise 
Vermietungsposition Registerkarte Preise 

Auf dieser Registerkarte definieren Sie die Preise für die Position. 
### Feld Mindestmiete 
Hier wird der Preis angegeben. 

Tooltip: 

Mindestmiete, wenn DA nicht gesetzt, sonst Tagespreis. 

### Feld Folgemiete 
Hier wird die Folgemiete angegeben. 

Tooltip: 

Folgemiete, wenn DA nicht gesetzt, sonst Wochenpreis. 

### Feld 0,5 Tagespreis 
Da manche Mietobjekte kurzfristig eingesetzt werden können, geben Sie hier den Preis für die Vermietung für einen halben Tag ein. 

### Feld Tagespreis 
Hier geben Sie den Preis ein, der berechnet wird, wenn das Mietobjekt tageweise vermietet wird. 

### Feld Wochenpreis 
Hier geben Sie den Preis ein, der berechnet wird, wenn das Mietobjekt wochenweise vermietet wird. 

### Feld Monatspreis 
Hier geben Sie den Preis ein, der berechnet wird, wenn das Mietobjekt einen Monat oder mehr vermietet wird. 

### Feld Mind.-Mietdauer 
Hier geben Sie eine Anzahl an Tagen ein, die als Mindestmietdauer gerechnet wird. Das Mietobjekt kann zwar früher zurückgegeben werden, die Berechnung erfolgt aber auf den Mindest-Mietzeitraum. 

Tooltip: 

Mindestmietdauer 
Angabe in Tagen. Ein solches Mietobjekt kann zwar innerhalb der Mindestmietzeit zurückgegeben werden 
– es wird aber immer die Miete für die Mindestmietdauer berechnet. 

### Feld Preis MMD 
In diesem Feld können Sie einen abweichenden Preis für die Mindestmietdauer eintragen. 

Tooltip: 

Hier kann ein abweichender Preis für die Mindestmietdauer des MOs hinterlegt werden, 
wenn im Fall der Abrechnungsperiode 'Monat' die Mindestmietdauer < der Abrechnungsperiode in Tagen ist. 

### Feld Verlustpreis 
Hier geben Sie den Preis ein, der dem Kunden bei Verlust in Rechnung gestellt werden soll. 

Tooltip: 

Preis 

### Feld Verlust - % MP 
Hier wird der Verlustanteil in % aus der bezahlten Miete eingetragen. 

Tooltip: 

Anteil aus bezahlter Miete(Verlust) 

### Feld Preis Rückkauf 
Falls die Position zurückgekauft werden soll, definieren Sie hier den Verkaufspreis. 

Tooltip: 

Preis 

### Feld Kaufen - % MP 
Hier wird der Kaufanteil in % aus der bezahlten Miete eingetragen. 

Tooltip: 

Anteil aus bezahlter Miete(Kaufen) 

### Feld Rabatt 1 % 
Hier kann ein Rabatt in % vergeben werden. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Feld Rabatttext 1 
Hier kann ein Rabatttext eingetragen werden. 

Tooltip: 

Angabe des Rabatttexts 

### Feld Rabatt 2 % 
Hier kann ein Rabatt in % vergeben werden. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Feld Rabatttext 2 
Hier kann ein Rabatttext eingetragen werden. 

Tooltip: 

Angabe des Rabatttexts 

### Feld Rabatt 3 % 
Hier kann ein Rabatt in % vergeben werden. 

Tooltip: 

Angabe eines Rabatt-/Zuschlagsprozentsatzes 

### Feld Rabatttext 3 
Hier kann ein Rabatttext eingetragen werden. 

Tooltip: 

Angabe des Rabatttexts 

### Checkbox Dynamische Abrechnungsperioden 
Wenn Sie diese Checkbox aktivieren, werden die Abrechnungsperioden dynamisch ermittelt. D.h. wenn diese Checkbox gesetzt ist, wird der Eintrag in der Combobox Abrechn.Per. als Startintervall für die erste Faktura genutzt. Das tatsächliche Mietintervall wird anhand der tatsächlichen Vermietdauer ermittelt. 

Tooltip: 

Dynamische Abrechnungsperioden 
Wenn gesetzt, dient die Combobox 'Abrechn. Periode' als Startintervall für die erste Faktura. 
Anhand der tatsächlichen Vermietdauer wird dann das tatsächliche Mietintervall ermittelt. 
Zum Zeitpunkt der jeweiligen Faktura wird die jeweils passende Abrechnungsperiode 
mit dem zur Periode hinterlegten Betrag ermittelt und berechnet. 

### Feld MB pro ZE 
Hier wird die Zeiteinheit für die Mietbeträge angegeben. Der Mietbetrag wird durch die Zeiheinheit dividiert berechnet. 

Ein Mietbetrag pro Tag hätte hier z.B. eine 1 stehen. 

Tooltip: 

Mietbetrag pro Zeiteinheit 
Definiert die Zeiteinheit für die Mietbeträge. Der Mietbetrag wird durch die Zeiteinheit dividiert berechnet. 
Ein Mietbetrag pro Tag hätte hier z.B. eine 1 stehen.
