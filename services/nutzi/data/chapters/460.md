# eNVenta Init Registerkarte Fibu Export 
eNVenta Init Registerkarte Fibu Export 

Auf dieser Registerkarte können Daten einer Finanzbuchhaltung in ein Import-Item exportiert werden. Der Eintrag im Feld Beschreibung bildet den späteren Dateinamen des zu erzeugenden Import-Items. Über das Feld Verzeichnis kann gesteuert werden, in welches Verzeichnis das Import-Item abgelegt wird. 

Das Verzeichnis befindet sich auf dem Broker. Alle Dateien werden auf dem Broker gespeichert, dementsprechend benötigt der Sachbearbeiter Schreibrechte auf dem entsprechenden Zielverzeichnis. 

Über die Checkboxen kann festgelegt werden, welche Bereiche exportiert werden. Abgesehen von den Konten werden alle Daten, welche die Tabelle enthält, exportiert. 
### Button Start Export 

Mit diesem Button wird der Export gestartet. Dabei werden eine XML-Datei und pro Checkbox eine CSV-Datei erstellt. Je nach Auswahl der Menge kann der Export einige Minuten andauern. 

Tooltip: 

Start Fibu Export 

### Feld Dateiname 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Feld Verzeichnis 
Hier geben Sie das Verzeichnis für den Export ein. 

### Feld Jahr 
Wenn dieses Feld gefüllt wird, können nur Konten aus dem angegebenen Jahr exportiert werden.
