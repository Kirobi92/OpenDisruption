# Feldmapping 
Feldmapping 

Dieser Dialog öffnet sich im BMEcat – Katalogexport über den Button Feldmapping . Hier kann der Benutzer Felder für den Katalog (Standard- und UDX-Filter) definieren, die am späteren BMEcat-Produkt gesetzt werden. Das Mapping ist pro Katalog zu definieren. 

Hier ist es möglich, ein Mapping für die Spalten aus der Artikelliste frei zu bezeichnen. Die Felder können hinzugefügt aber auch entfernt werden. 

- Der Feldname ist frei definierbar. - Die Komponente wird ausgewählt. Die zugrundeliegende Liste ist fix und enthält nur die Werte Artikellistung bzw. Artikel. - Je nach Auswahl der Komponente steht eine Liste mit möglichen Eigenschaften aus der jeweiligen Komponente zur Verfügung. 
Der Export kann nur stattfinden, wenn ein Artikel bzw. eine Artikelliste einem Kunden zugeordnet ist (im Dialog Artikellistung ). Dazu ist es notwendig, die Checkbox Filter Artikellistung auf der Registerkarte Exportoptionen zu setzen. 

Siehe Kapitel Artikellistung .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Auswahl 

Mit diesem Button übernehmen Sie ein Feldmapping in den BMEcat - Katalog . 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
