# Besuchsbericht Registerkarte Ziel/Thema 
Registerkarte Ziel/Thema 

Auf dieser Registerkarte geben Sie interne Textinformationen zum Besuch ein: Welches Thema wurde behandelt, ist das Ziel erreicht etc.? 
### Feld Besuchsinfo 
In diesem Feld geben Sie Informationen zum Besuch ein: Welches Thema wurde behandelt, ist das Ziel erreicht etc.? 

Tooltip: 

Enthält interne Textinformationen. 

### Button Textbausteine 

Über diesen Button gelangen Sie in den Dialog Textbausteine . 

Damit besteht die Möglichkeit Textbausteine für den Besuchsbericht zu verwenden. Bei manchen Kunden entstehen weitere Aufgaben nach Besuchen, wie z.B. Verteilung von Aufgaben an andere Kollegen, die Versendung von Fragenkatalogen für die Bedarfsermittlung. Sind beispielsweise solche Fragenkataloge als Textbausteine vorhanden, können Sie bequem ausgewählt, an den Besuchsbericht angehängt und versendet werden. 

Je nachdem, was ausgewählt ist, wird der Text auf die entsprechende Stelle übernommen. Wird im Nachhinein weiterer Text ausgewählt, wird der Text hinten angehängt. Leerzeichen müssen beachtet werden. 

Siehe Kapitel Textbausteine .
