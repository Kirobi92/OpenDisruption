# Scheckerfassung 
Scheckerfassung 

Dieser Dialog öffnet sich im Dialog Ausbuchung über den Button Scheck . Hier erfassen Sie in den entsprechenden Feldern Schecknummer, Kontonummer und Bankleitzahl des Schecks. 

Haben Sie alle Daten erfasst, bestätigen Sie diese mit dem Button und die Daten werden im Menü Buchen >> Auswertungen >> Personenkonten >> Scheckeinreichung gespeichert. 

Siehe Kapitel Scheckeinreichung . 

Wenn ein Scheck doppelt erfasst wird, erhalten Sie einen Hinweis, dass dieser Scheck bereits erfasst wurde und ob Sie trotzdem neu speichern wollen. Betätigen Sie den Button OK , werden die Daten erneut gespeichert. 

Mit dem Button Abbrechen verlassen Sie den Hinweis ohne Speicherung der Daten.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Scheckeinreichung 

Mit diesem Button gelangen Sie in den Dialog Scheckeinreichung . 

Siehe Kapitel Scheckeinreichung . 

### Feld Schecknummer 
Hier geben Sie die Nummer des Schecks ein. 

### Feld IBAN 
Hier geben Sie die IBAN-Nummer des Schecks ein. 

Tooltip: 

IBAN (International Bank Account Number) 

### Feld BIC 
Hier geben Sie die BIC-Nummer des Schecks ein. 

Tooltip: 

Bank Identifier Code 

### Feld Scheckkontonr. 
Hier geben Sie die Kontonummer ein, die auf dem Scheck vermerkt ist. 

### Feld BLZ 
Hier geben Sie die Bankleitzahl ein, die auf dem Scheck vermerkt ist. 

Tooltip: 

Angabe der Bankleitzahl / SWIFT 

### Feld Betrag 
Dieses Feld wird automatisch mit dem erfassten Betrag aus dem Dialog Buchen gefüllt, kann aber jederzeit überschrieben werden. 

Tooltip: 

Scheckbetrag kann nicht negativ sein 

### Feld Text 
In diesem Feld werden die Kundennummer und der Firmenname des Scheckausstellers automatisch eingetragen. Dieses Feld kann jedoch jederzeit überschrieben werden.
