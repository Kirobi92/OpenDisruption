# Menge ändern in der Kommissionierung 
Menge ändern 

Durch Anklicken des Buttons Menge können Sie eine Mengenänderung für den im Dialog Kommissionieren markierten Artikel vornehmen. Diese Änderung wird nach Betätigen des Buttons durchgeführt. 

Im oberen Bereich des Dialogs erscheint die Auftragsreferenz der markierten Zeile in der Kommissionierung. Es werden die Belegnummer, die Artikelnummer und die Positionsnummer angezeigt, um eine einfache Übersicht zu erhalten. Dies ist besonders bei Sammelbelegen hilfreich, um eine genaue Unterscheidung zu erhalten. 

Sie können zwischen Lagereinheiten und Verkaufseinheiten wählen. Sobald Sie den gewünschten Radiobutton aktivieren, werden die Mengen im Dialog umgerechnet. 

Bei Änderung der Menge eines Chargenartikels öffnet sich nach der Übernahme automatisch der Dialog Chargenabgang , um die Zuordnung anzupassen. Wurde der Verkaufsparameter Automatische Chargenzuordnung gesetzt, so erscheint dieser Dialog nicht und die Anpassungen werden vom System vorgenommen. 

Verwaltung von Fehlmengen bei Komplettlieferung 

In der Praxis kommt es immer wieder vor, dass die zu kommissionierende Menge vom Lagermitarbeiter nicht kommissioniert werden kann, weil entweder der Bestand innerhalb von eNVenta ERP vom tatsächlichen Lagerbestand abweicht oder aber die zu kommissionierende Menge in Teilen defekt oder nicht verwendbar ist. 

Im Standard wird die tatsächlich kommissionierte Menge im Versandmodul durch Anpassung der Menge im Dialog Menge ändern korrigiert. 

Verwendet der Kunde den Job-Server, um eine automatische Lieferzuteilung zu machen, so wird direkt, nachdem der Sachbearbeiter den Auftrag gepickt hat, die Restmenge erneut zugeteilt, weil eNVenta ERP von einem ausreichenden Bestand ausgeht. 

Um dies zu verhindern, gibt es den Versandparameter Fehlmengenberichte . Damit wird es möglich, die Ware auf ein Klärungslager umzubuchen. 

Daher wird bei einer Reduktion der Menge in oben genanntem Dialog ein Klärungslager angeboten, auf den die Fehlmenge gebucht werden kann. Die Fehlmenge wird dabei bereits als Vorschlag vorbelegt, so dass der Mitarbeiter nur noch die Umbuchung durchführen muss. 

Klärungslager können im Dialog Lagergruppen in einer Spalte hinzugefügt werden. Klärungslager sind vom Typ her Sperrlager und kein chaotisches Lager. Nur solche Lager kommen als Klärungslager in Betracht. 

Befindet sich die Ware auf dem Klärungslager, so wird erst einmal geprüft, ob sie tatsächlich nicht vorhanden ist oder vom Kommissionär schlicht nicht gefunden wurde. In einem solchen Fall ist die gefundene Menge wieder auf das Lager zurückzubuchen, von der sie stammte (manueller Prozess). Muss die Ware "abgeschrieben" werden, ist dies ebenfalls über einen manuellen Prozess (oder die Inventur) machbar. 

In diesem Zusammenhang gibt es im Dialog Menge ändern die Checkbox Korrekturbuchung und die Combobox Klärungslager . 

Nachdem die Menge im Dialog Menge ändern angepasst wurde und dazu die Checkbox Korrekturbuchung aktiviert wurde, öffnet sich der Dialog Korrekturbuchung . 

Siehe Kapitel Korrekturbuchung (Lagerbewegung).

### Feld Neue Menge 
In diesem Feld geben Sie eine neue Liefermenge ein. Das Feld ist vorbelegt mit der aktuellen Auftragsmenge. 

Wenn im Kommissionieren im Lagerfach keine Ware gefunden wird, kann auch eine Korrekturbuchung mit 0 erfolgen. Das System verhält sich dann so, als ob der Button Rückstand verwendet worden wäre. 

Des Weiteren wird die Korrekturbuchung, die hier angestoßen wird, ausgeführt. 

Aufgrund dieser Korrekturbuchung wird die Position wieder in Status 1 gesetzt, die Packlistennummer und die Positionszuordnungen werden gelöscht. Sobald der Bestand im Lager wieder aufgefüllt ist, kann z.B. über die Lieferzuteilung der Status der Position wieder erhöht werden, damit die Ware kommissioniert werden kann. 

Tooltip: 

Geben Sie eine neue Liefermenge an.
