# Kapitel NV.ERP.Base.Common.cdSystemMM.shtNoBacklog

Wenn Sie diesen Parameter aktivieren, wird bei Teillieferungen die Restmenge gestrichen. Das Kennzeichen wird bei Lieferung bzw. Faktura automatisch auf der entsprechenden Auftragsposition auf "S" (wie streichen) gesetzt.
