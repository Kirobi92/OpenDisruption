# Druck-Aufträge 
Druck-Aufträge 

Zum einen gibt es im Navigator auf der Registerkarte Jobs die Tabelle für eigene Druck-Aufträge und zum anderen den Menüpunkt Druck , wo Sie das Menü Druck-Aufträge finden. Hier werden für jeden Sachbearbeiter die Druckaufträge und deren Status angezeigt. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Druck-Aufträge 
Tooltip: 

In diesem Dialog werden für jeden Sachbearbeiter die Druckaufträge und deren Status angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Aufräumen 

Dieser Button räumt die Informationen auf dem Monitor auf. D.h. alle abgearbeiteten Druckaufträge verschwinden aus der Tabelle. 

Siehe Kapitel Druck-Infos aufräumen . 

### Button Fehler buchen 

Mit diesem Button werden markierte Zeilen mit den Status Fehler und Timeout „verbucht“. Der Status wird dadurch geändert auf Fehler gebucht und der Druck-Auftrag wird aus dem Speicher gelöscht. Die verbuchten Fehler werden nicht im Navigator angezeigt. 

Tooltip: 

Fehlerhafte Druck-Aufträge verbuchen. 

### Combobox Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Diese Combobox wird nur angezeigt, wenn die Einstellung "Druck-Aufträge aller Benutzer" freigeschaltet ist. Ansonsten wird die Combobox ausgeblendet und im Hintergrund mit dem aktuellen Benutzer gefüllt, damit nur die eigenen Druck-Aufträge gesehen und verwaltet werden können. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Checkbox Nur fehlerhafte 
Wenn Sie diese Checkbox aktivieren, werden nur Druck-Aufträge mit den Status Fehler , Timeout und Fehler gebucht angezeigt. Diese Checkbox ist beim Öffnen des Dialogs automatisch gesetzt und die entsprechenden Datensätze werden eingelesen. 

### Combobox Druck-Server 
Aus dieser Combobox können Sie den Print-Server auswählen, dessen Druckaufträge Sie in der Tabelle ansehen möchten. 

### Combobox Drucker 
Hier wird ein (Standard-)Drucker angegeben. 

Tooltip: 

Angabe des (Standard-)Druckers 

### Combobox Titel 
Über diese Combobox können Sie gezielt nach Formular-Typen suchen (z.B. Lieferschein, Rechnung). Alle momentan in der Druckliste verfügbaren Titel werden angeboten (unabhängig vom Sachbearbeiter). 

### Tabelle 

### Tabellenspalte Druckdatum 
Das Datum des Drucks wird angezeigt. 

### Tabellenspalte Titel 
In dieser Spalte wird der Formular-Typ zuzüglich einer evtl. Dokumentennummer angezeigt. Diese Dokumentennummer wird nicht von allen Programmteilen gefüllt. 

Tooltip: 

Angabe des Titels 

### Tabellenspalte Druck-Status 
Der Druck-Status wird angezeigt. 

### Tabellenspalte Drucker 
Der hinterlegte Drucker wird angezeigt. 

Tooltip: 

Angabe des (Standard-)Druckers 

### Tabellenspalte Programm 
Die Spalten ‚Programm‘ und ‚Referenz‘ geben die Referenz auf einen Vorgang an, für die dieser Druck-Auftrag erzeugt wurde. Diese Referenz wird nur von einigen Programmteilen gefüllt. Wenn eine Referenz vorhanden ist, wird diese mit einem Doppelklick in die Zeile geöffnet. 

Tooltip: 

Art des Datensatzes, von welchem der Link ausgeht. 

### Tabellenspalte Referenz 
Die Spalten ‚Programm‘ und ‚Referenz‘ geben die Referenz auf einen Vorgang an, für die dieser Druck-Auftrag erzeugt wurde. Diese Referenz wird nur von einigen Programmteilen gefüllt. Wenn eine Referenz vorhanden ist, wird diese mit einem Doppelklick in die Zeile geöffnet. 

Tooltip: 

Pfad bzw. URL des verlinkten Inhalts. Kann als Suchkriterium dienen. 

### Tabellenspalte Sachbearbeiter 
Der zuständige Sachbearbeiter wird angezeigt. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
