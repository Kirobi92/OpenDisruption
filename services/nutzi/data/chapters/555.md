Grundeinstellungen für eNVenta ERP / Parameter Allgemein 

# Parameter Allgemein Registerkarte Logo 
Parameter Allgemein Registerkarte Logo 

Diese Registerkarte bietet die Möglichkeit, dynamische Bilder zu verwalten. Diese können dann in den Reports mit angedruckt werden. 

Bei Abweichungen von den Standardmaßen kann das Bild verzerrt dargestellt werden. Bearbeiten Sie ihr Logo gegebenenfalls in einem Bildbearbeitungsprogramm. 

Um eventuelle Fehlermeldungen beim Druck zu verhindern, erhält jeder Sachbearbeiter beim Start von eNVenta ERP eine Meldung, wenn eines der Logos nicht gepflegt ist. Sind beide Logos vorhanden, erscheint die Meldung nicht mehr. 

Was den Druck von Logos etc. auf den Reports betrifft, so ist folgendes zu beachten: 

Es können alle gängigen Bild-Formate (bmp, gif, jpeg, png, tiff, …) ausgewählt werden. 

JPEG, GIF und PNG werden 1:1 gespeichert und auch so an den Report übergeben. 

Alle anderen nicht internetfähigen Formate (bmp, tiff, …) werden verlustfrei in ein PNG umgewandelt. 

Empfehlung: 

Verwenden Sie PNG als Format. 

Vermeiden Sie nach Möglichkeit Bilder auf Subreports. 

### Abbildung Logo groß 
Es können alle möglichen Grafiken ausgewählt werden (bmp, gif, jpg, png, tiff, …). Der -Button ermöglicht die Suche über eine Dateiauswahl. 

Die Bilder werden intern im PNG-Format gespeichert (komprimiert ohne Verluste) und auch in diesem Format an den Report übergeben. 

Möchten Sie das Logo entsprechend austauschen, so beachten Sie bitte diese Größe im Original-Bild und füllen dies ggf. durch einen transparenten Bereich auf, falls Ihr Logo kleiner ausfallen sollte. 

Tipp zur Vorgehensweise: 

Es kann eine Word-Vorlage verwendet werden, in der Sie Ihr Logo an die entsprechende Stelle platzieren. 

Öffnen Sie die mitgelieferte Word-Datei. Diese finden Sie als Ressource im Broker. 

Per Rechtsklick -> "Bild ändern" öffnet sich der Explorer und Sie können das vorhandene Bild mit Ihrem Logo ersetzen. 

Per Rechtsklick -> "Größe und Position" können Sie die Größe und Position des Bildes definieren. 

Die definierte Word-Vorlage muss anschließend noch in eine Bild-Datei umgewandelt werden: 

Dafür öffnen Sie folgende Webseite: http://www.zamzar.com/ 

Schritt 1: Über „durchsuchen“ öffnet sich der Explorer, indem Sie ihre Word-Vorlage auswählen, die konvertiert werden soll. 

Schritt 2: Wählen Sie das Format aus, in das die Vorlage umgewandelt werden soll. 
Am besten sollte das PNG-Format verwendet werden. 

Schritt 3: Fügen Sie ihre E-Mailadresse ein. 

Schritt 4: Konvertieren Sie die Datei. 

Anschließend erhalten Sie eine E-Mail mit dem Downloadlink. Über den Link wird die Webseite von Zamar geöffnet, auf der Sie ihr Logo downloaden können. 

Die Standardmaße des LogoBig oder LogoBigAlternativ sind 21,00 x 4,40 cm. 

Tooltip: 

Das Seitenverhältnis dieses Logos entspricht 21:4,4. 

### Button Suche 

Dieser Button ermöglicht eine Suche über eine Dateiauswahl. 

### Abbildung Logo klein 
Es können alle möglichen Grafiken ausgewählt werden (bmp, gif, jpg, png, tiff, …). Der -Button ermöglicht die Suche über eine Dateiauswahl. 

Die Bilder werden intern im PNG-Format gespeichert (komprimiert ohne Verluste) und auch in diesem Format an den Report übergeben. 

Die Standardmaße des Logos sind 3,66 x 1,82 cm. 

Tooltip: 

Das Seitenverhältnis dieses Logos entspricht 2:1. 
Standardabmessung im Reporting 
Breite: 3,66 cm 
Höhe: 1,82 cm 

### Button Suche 

Dieser Button ermöglicht eine Suche über eine Dateiauswahl. 

### Feld Kopftitel 
Hier wird der Titel der Kopfleiste in der Anwendung angegeben. 

Tooltip: 

Dieser Titel wird in der Kopfleiste der Anwendung angezeigt. 

### Steuerelement Titel-Farbe 

Die hier hinterlegte Farbe wird für die Einfärbung der Titelleisten im Navigator und in den Haupt-Forms verwendet. 

Damit können unterschiedliche eNVenta -Umgebungen für eine bessere Orientierung gekennzeichnet werden, um so z.B. Test- von Produktiv-Umgebungen besser unterscheiden zu können. 

Weil in Mandanten-Umgebungen die Parameter mandantenspezifisch gespeichert werden, können so auch Mandanten besser unterschieden werden. 

Eine Änderung dieses Parameters wirkt sich erst nach einem Neustart der Anwendung aus. 

Tooltip: 

Die Titelleiste wird mit dieser Farbe eingefärbt. 
Wirkt erst nach einem Neustart. 

### Button Löschen 

Entfernt die Titel-Farbe. 
Das Ändern der Farbe wirkt sich erst nach einem Neustart der Anwendung aus. 

Tooltip: 

Farbe entfernen
