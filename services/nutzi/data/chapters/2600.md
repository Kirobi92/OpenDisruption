# Zahlvorschlag verarbeiten Registerkarte Zahlung 
ZV Verarbeiten Registerkarte Zahlung 

Auf dieser Registerkarte sehen Sie in der Tabelle, wie die OPs für die Zahlung aufbereitet werden. Es kann hier nichts geändert werden. 

Wurde die Verarbeitung mittels eines zu druckenden Formulars gewählt, gibt es nach dem Formulardruck noch die Möglichkeit für den Druck von sogenannten AVIS-Schreiben. Dazu klicken Sie die Checkbox in der Spalte ‚Zahlungsavis’ an. 

AVIS-Schreiben teilen dem Kreditor mit, welche offenen Posten mit diesem Zahlungslauf ausgeglichen werden sollen. Handelt es sich hier um einen größeren Zahlungsvorschlag, reicht der Platz für die Belegnummern auf dem Formular nicht aus. In diesem Fall kann ein separates AVIS-Schreiben gedruckt werden.
