# Besuchsplan Registerkarte Allgemein 
Besuchsplan Registerkarte Allgemein 

Auf dieser Registerkarte werden Ihnen nach Eingabe der Selektionskriterien im Kopf des Dialogs die entsprechenden Datensätze in die Tabelle eingelesen. Es besteht die Möglichkeit, mehrere Besuche für einen Kunden zu erfassen. 
### Button Einzelauswahl 

Bei der manuellen Planung kann über diesen Button ein Kunde oder eine Adresse (Interessent) aus dem Dialog AKL-Auswahl gewählt werden. 

Siehe Kapitel Auswahl Adressen, Kunden und Lieferanten . 

Tooltip: 

Auswahl eines einzelnen Kunden oder Lieferanten 

### Button Mehrfachauswahl 

Alternativ zur Einzelwahl können auch mehrere Kunden auf einmal übernommen werden. Hierzu kann mit diesem Button der Dialog AKL-Auswahl nach Vertreter geöffnet werden. 

Siehe Kapitel Teilnehmer Auswahl . 

Tooltip: 

Auswahl mehrerer Kunden oder Lieferanten 

### Button Löschen 

Mit diesem Button können Sie einzelne Zeilen aus der Tabelle löschen. 

- Zum Löschen einer einzelnen Zeile wird die Zeile durch Anklicken/Markieren ausgewählt. Mit Klick auf den Button Löschen wird die Zeile mit einem versehen. - Zum Löschen mehrerer einzelner Zeilen werden die gewünschten Zeilen durch halten von <Strg> und Klicken mit der Maus ausgewählt. Mit Klick auf den Button Löschen wird die Zeile mit einem versehen. - Zum Löschen mehrerer Zeilen als Block untereinander wird die Zeile am Blockanfang angeklickt und dann während dem Halten von <Shift> die letzte Zeile des Blocks gewählt. Mit Klick auf den Button Löschen wird die Zeile mit einem versehen. 
Die Zeilen werden erst endgültig gelöscht, wenn Sie den Button drücken. 

Tooltip: 

Löschen eines Tabelleneintrags 

### Button Projekt + 

Über diesen Button können Sie dem Kunden ein Projekt zuordnen. Zu diesem Zweck öffnet sich der Dialog Projektauswahl . 

Siehe Kapitel Projektauswahl . 

Tooltip: 

Projekt zuordnen 

### Button Projekt - 

Dieser Button entfernt das Projekt aus der gleichnamigen Spalte. 

Tooltip: 

Projektzuordnung löschen

### Tabellenspalte AKL-Nummer 
Die Nummer der Adresse, des Kunden oder Lieferanten wird angezeigt. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Tabellenspalte AKL-Typ 
Hier wird der Typ zur AKL-Nummer angezeigt. Zur Verfügung stehen: 

- A = Adresse - K = Kunde - L = Lieferant 

Tooltip: 

AKL-Typ: 
A = Adresse 
K = Kunde 
L = Lieferant 
V = Vertreter 
P = Kontakt

Tooltip:

### Tabellenspalte Ort 
Der Ort des Firmensitzes wird angezeigt. 

Tooltip: 

Angabe des Orts 

### Tabellenspalte Ansprechpartner 
Der Ansprechpartner zur AKL-Nummer wird angezeigt. 

Tooltip: 

Nummer des Kontakts/Partners 

### Tabellenspalte Projekt 
Falls dieser Vorgang einem Projekt zugeordnet werden soll, kann ein Projekt über den Button Projekt+ ausgewählt und übernommen werden. 

Das Feld Projekt ist Teil des Schlüssels für einen Besuchsplan. Durch das Entfernen der Projektnummer wird auch der entsprechende Besuchsbericht aktualisiert, falls bereits vorhanden. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein.

### Tabellenspalte Status 
Aus dieser Combobox kann der Status des Besuchs ausgewählt werden. Zur Verfügung stehen: 
- geplant - vereinbart - bestätigt - besucht - abgesagt . 
Für eine komfortablere Bearbeitung eines Eintrags innerhalb des Besuchsplans ist es so möglich, den Status anzupassen. So ist auch klar erkennbar, ob der Termin geplant, vereinbart oder vom Kunden auch bestätigt wurde. 

Tooltip: 

Angabe des Positionsstatus
