# PTV-Parameter 
Parameter PTV 

Dieser Dialog öffnet sich in der Kommissionierung >> Optionen >> PTV-Schnittstelle >> PTV-Parameter . Um die PTV-Schnittstelle aus dem PTV heraus nutzen zu können, müssen Sie in diesem Dialog zuerst ein paar Einstellungen vornehmen. 

Nur wenn Sie die Pflichteinstellungen getätigt haben, kann die PTV-Schnittstelle komplett funktionieren! 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Parameter PTV 
Tooltip: 

In diesem Dialog nehmen Sie Einstellungen zur PTV-Schnittstelle vor. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Speichern 

Mit diesem Button speichern Sie die Eingaben. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Export Pfad 
Hier hinterlegen Sie den Pfad, den das PTV-Tool für die Schnittstellenkommunikation zu PTV vorsieht. 

Tooltip: 

Geben Sie den Pfad ein für die Schnittstellen-Kommunikation. 

### Button Suche 

Mit diesem Button können Sie den Pfad suchen. 

### Button Entfernen 

Wenn Sie diesen Button drücken, wird der Pfad aus dem Feld entfernt. 

### Feld Adressdatei 
Tragen Sie den Dateinamen (ohne Endung) ein, welcher vom PTV-System für den Export der Adressdaten vorgesehen ist. 

Tooltip: 

Geben Sie die Adressdatei ein für den Export der Adressdaten. 

### Feld Auftragsdatei 
Tragen Sie den Dateinamen (ohne Endung) ein, welcher vom PTV-System für den Export der Versanddaten vorgesehen ist. 

Tooltip: 

Geben Sie die Auftragsdatei ein für den Export der Adressdaten. 

### Feld Import Pfad 
Hier hinterlegen Sie den Pfad, den das PTV-Tool für die Schnittstellenkommunikation von PTV vorsieht. 

Tooltip: 

Geben Sie den Import-Pfad an. 

### Button Suche 

Mit diesem Button können Sie den Pfad suchen. 

### Button Entfernen 

Wenn Sie diesen Button drücken, wird der Pfad aus dem Feld entfernt. 

### Feld Import-Datei 
Tragen Sie den Dateinamen (ohne Endung) ein, welcher vom PTV-System für den Import der PTV-Ergebnisse vorgesehen ist. 

Tooltip: 

Geben Sie die Importdatei an. 

### Feld Max. Alter LSch 
Mit dieser Option können Sie eine Einschränkung bei der Übertragung alter Lieferscheine machen. D.h. Angabe des maximalen Alters der Lieferscheine in Tagen. 

Tooltip: 

Wie viele Tage dürfen die Lieferscheine maximal alt sein, um übertragen zu werden (optional)? 

### Checkbox PTV über Gateway 
Wenn diese Checkbox gesetzt ist, wird die PTV Logik über den Gateway Adapter abgebildet. 

Tooltip: 

Ist dieser Parameter aktiv, so wird die PTV Logik über den Gateway Adapter abgebildet. 

### Combobox Kundenevent 
Dies ist die eindeutige ID dieser Konfiguration (GUID) für den Dateinamen. Ist noch keine ID gesetzt, handelt es sich noch um einen neuen, ungespeicherten Datensatz. 

Tooltip: 

Das Event, mit welchem die Kunden Adressübertragung an PTV stattfindet. 

### Combobox Lieferanten Event 
Tooltip: 

Das Event, mit welchem die Lieferanten-Adressübertragung an PTV stattfindet. 

### Combobox Lieferanschrift-Event 
Tooltip: 

Das Event, mit welchem die Liefer-Adressübertragung an PTV stattfindet. 

### Combobox Bestellanschrift-Event 
Tooltip: 

Das Event, mit welchem die Bestell-Adressübertragung an PTV stattfindet.
