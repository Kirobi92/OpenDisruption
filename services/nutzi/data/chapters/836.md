# Bankverbindungen der Kunden bzw. Lieferanten Registerkarte Standard 
Bankverbindung Registerkarte Standard 

Auf dieser Registerkarte definieren Sie die Einstellungen für inländische Zahlungsvorgänge. 

Zu einer IBAN-Bankverbindung (in den Bankverbindungen für Kunden und Lieferanten) wird auch der Name der Bank änderbar vorbelegt. Sollte aus dem Bankenstamm keiner vorhanden sein, können Sie hier nun einen erfassen. Damit ist bei der Auswahl einer Bankverbindung die Wiedererkennung vereinfacht. 
### Tabelle 

### Tabellenspalte Nr. 
Hier wird die Standard-Bankverbindung angezeigt. Gibt es keine speziell gekennzeichnete Bankverbindung, wird die jüngste Bankverbindung zum Kunden bzw. Lieferanten angezeigt. Für diesen Vorgang kann die Bankverbindung auch geändert werden. 

### Tabellenspalte Gesellschaft 
Hier wird die Gesellschaftsnummer für das Verfahren 'Multicompany' angegeben. 

Siehe Kapitel Gesellschaftsverwaltung für Multicompany . 

Tooltip: 

Gesellschaft für das Verfahren "Eine Warenwirtschaft". 

### Tabellenspalte Typ 
Wenn Sie eine neue Zeile zur Bearbeitung öffnen, wählen Sie hier den Typ aus. Zur Verfügung stehen Bank und IBAN . 

### Tabellenspalte IBAN 
Hier geben Sie die IBAN-Nummer ein. 

Tooltip: 

IBAN (International Bank Account Number) 

### Tabellenspalte QR-IBAN 
Tooltip: 

Die IBAN Nr. ist eine QR-IBAN Nummer. 

### Tabellenspalte Nummer 
Hier tragen Sie die Kontonummer des Kunden ein. 

Tooltip: 

Kontonummer der Zahlungsverbindung 

### Tabellenspalte Bankname 
Hier geben Sie den Namen der Bank ein. 

Tooltip: 

Angabe des Banknamens 

### Tabellenspalte BIC 
Hier geben Sie die BIC-Nummer ein. 

Tooltip: 

Bank Identifier Code 

### Tabellenspalte BLZ / SWIFT 
Hier geben Sie Bankleitzahl / SWIFT. 

Tooltip: 

Angabe der Bankleitzahl / SWIFT 

### Tabellenspalte Währung 
Hier geben Sie die Währung ein, die auf diesem Konto geführt wird. 

Tooltip: 

Währungskürzel 

### Tabellenspalte ESR-Konto 
Hier geben Sie die ESR-Kontonummer ein; diese wird nur für den Schweizer Zahlungsverkehr benötigt. 

### Tabellenspalte Standardverbindung 
Wenn Sie die Bank als Standard-Bankverbindung führen möchten, aktivieren Sie die Checkbox in dieser Spalte. 

Es kann nur eine Bankverbindung als Standardverbindung definiert werden. 

Tooltip: 

Angabe, ob Standard-Bankverbindung
