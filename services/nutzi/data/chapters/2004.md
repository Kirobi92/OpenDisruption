# Plantafel Registerkarte Übersicht 
Plantafel Registerkarte Übersicht 

Diese Registerkarte gibt Ihnen eine Übersicht über verplante Ressourcengruppen. 
### Feld Betriebsauftrag 
Hier wird die Nummer des Betriebsauftrags angegeben. 

Tooltip: 

Angabe der Betriebsbauftragsnr. 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Betriebsauftrag. 

### Feld Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status 

### Feld Startdatum 
Hier wird ein Startdatum für die Selektion angegeben. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Feld Datum bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Feld Produktion 
Hier wird die Nummer des Produktionsauftrags angegeben. 

Tooltip: 

Angabe des Produktionsauftrags 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Produktionsauftrag, falls vorhanden. 

### Feld Status 
Der Name des Status wird angegeben. 

Tooltip: 

Angabe des Status
