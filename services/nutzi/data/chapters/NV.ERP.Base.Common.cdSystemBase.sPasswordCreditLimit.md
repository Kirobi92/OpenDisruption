# Kapitel NV.ERP.Base.Common.cdSystemBase.sPasswordCreditLimit

Wird hier ein Passwort hinterlegt, kann ein Auftrag bei Überschreitung des Kreditlimits nur gespeichert werden, wenn im Vorgang das Passwort eingegeben wird. 

Bei striktem Kreditlimit kann über das Passwort eine Auftragserfassung ermöglicht werden. 

Tooltip: 

Passwort für die Freigabe eines Auftrags trotz Überschreitung des Kreditlimits.
