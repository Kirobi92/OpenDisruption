# Einkaufsarten Registerkarte Druck 
Einkaufsarten Registerkarte Druck 

Zu jeder Einkaufart können eigene Formularvarianten erstellt werden. Verknüpfen Sie die Formulare mit einem Windows-Druckertreiber. Die Formulare werden dann automatisch auf die gewünschten Drucker verteilt. So kann beispielsweise die Bestellung mit einem Faxdruckertreiber verknüpft und somit automatisch per Fax versendet werden. 

Auch die Anzahl der Kopien kann je Formular vorgegeben werden. Dies ist vor allem bei der Nutzung von Laser- und Tintenstrahldruckern von Vorteil, da hier keine Durchschläge möglich sind. 

Was den Druck von Logos etc. auf den Reports betrifft, so ist folgendes zu beachten: 

Es können alle gängigen Bild-Formate (bmp, gif, jpeg, png, tiff, …) ausgewählt werden. 

JPEG, GIF und PNG werden 1:1 gespeichert und auch so an den Report übergeben. 

Alle anderen nicht internetfähigen Formate (bmp, tiff, …) werden verlustfrei in ein PNG umgewandelt. 

Empfehlung: 
Verwenden Sie PNG als Format. 

Vermeiden Sie nach Möglichkeit Bilder auf Subreports.

### Checkbox Sofortdruck 
Wenn Sie diese Checkbox aktivieren, wird sofort nach Drücken des Buttons und der Speicherung der erfassten Bestellung der Druck-Dialog angezeigt. Wurde in der Einkaufart für ein oder mehrere Formulare eine Anzahl Kopien in das Feld Anz . eingetragen, startet der Druck sofort ohne jeden weiteren Tastendruck! 

Tooltip: 

Wenn aktiviert, wird unmittelbar nach dem Speichern der Druck-Dialog geöffnet und der Druck kann sofort erfolgen.
