# Kapitel NV.ERP.Base.Common.cdSystemBase.shtCountryVersion

Hier kann festgelegt werden, welche Länderversion von eNVenta ERP genutzt werden soll. Je nach Wahl werden in einigen Programmbereichen (z.B. Finanzbuchhaltung) andere Berechnungen bzw. Vorgehensweisen ausgeführt. Zur Verfügung stehen: 

- Deutschland - Schweiz - Österreich 

Tooltip: 

Hier legen Sie die Länderversion für die Arbeit in eNVenta fest. Zur Verfügung stehen 
- Deutschland 
- Schweiz 
- Österreich
