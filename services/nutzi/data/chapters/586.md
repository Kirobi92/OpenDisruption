Grundeinstellungen für eNVenta ERP 

# SMTP Versand für E-Mails 
eNVenta bietet die Möglichkeit den Versand von E-Mails per SMTP zu versenden. Der zentrale Versand über einen SMTP-Server bietet sich für Massen-Mailing an und wenn die Benutzer kein lokales Outlook haben. Daher ist die Einrichtung spätestens notwendig, wenn systemseitig automatisch E-Mails versendet werden sollen. 

SMTP bedeutet Simple Mail Transfer Protocol und ist ein Protokoll der Internet-Protokollfamilie, das zum Austausch von E-Mails in Computernetzen dient. Es wird vorrangig zum Einspeisen und Weiterleiten von E-Mails verwendet. 

Zusätzlich zur Basic Authentication wird auch die OAuth2 Authentication für den Mail-Versand über SMTP unterstützt. 

Wenn Sie nichts auswählen, ist das Verhalten analog zu Basic Authentication. 

Wenn Sie oAuth für Exchange Online auswählen und speichern, werden zusätzlich die Einträge für Anwendung-ID und Verzeichnis-ID angezeigt. 

Verwendung findet der E-Mail Versand per SMTP z.B. bei Abschluss des Auftrags. Hier kann der Kunde automatisch via E-Mail über SMTP benachrichtigt werden, wenn der Druck des Lieferscheins und der Druck der Rechnung erfolgt ist. 

Um den Versand der E-Mails per SMTP nutzen zu können, ist es notwendig einige Einstellungen in eNVenta vorzunehmen. Dazu gehören die Systemeinstellung, die Autorisierung einzelner Mitarbeiter, deren PCs und eine bestimmte E-Mail-Adresse. 

Die Einrichtung der Verbindung zum SMTP-Server erfolgt über den Dialog Parameter Nachrichten . 

Parameter Nachrichten 

Der Versand ist über einen lokalen SMTP-Server und über Office365 möglich. Hierzu ist es notwendig smtp.office365.com als Host anzugeben. Ebenso muss der Benutzer eine gültige Berechtigung zum Versand haben! 

Wenn Sie die Verbindung eingerichtet haben, können Sie pro Benutzer zwischen dem Mail-Versand via Outlook und SMTP-Server wählen. 

Beim Versand via Outlook hat der Benutzer eine Kopie der gesendeten Mail in seinem Postfach, bei einem Versand via SMTP fehlt dies. In diesem Fall erhält der Benutzer die E-Mail als BCC-Empfänger, wenn im Dialog Parameter Allgemein konfiguriert. 

Die Einstellung für den Mailversand erfolgt im Dialog Parameter Office , Registerkarte SMTP . 

Parameter Office 

Es gibt hier folgende Möglichkeiten: 

- Systemeinstellung -> Die Einstellung aus Parameter Nachrichten wird übernommen. - Ja -> Der Versand erfolgt zwingend über SMTP. - Nein -> Der Versand erfolgt immer über das lokale Outlook.
