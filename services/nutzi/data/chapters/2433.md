# Offene Posten Sachkonten 
Offene Posten Sachkonten 

Dieser Dialog öffnet sich im Buchen >> OP >> OP Sachkonten . Hier können Sie Sachkonten, welche im Sachkontenstamm auf der Registerkarte Allgemein als OP-Führend aktiviert sind, verwalten. 

Siehe Kapitel Sachkontenstamm Registerkarte Allgemein . 

In der Regel handelt es sich bei diesen Sachkonten um sogenannte Transit- oder Verrechnungskonten. 

Beispiele hierfür sind die Transitkonten im Bankeinzug und Zahlungsverkehr und für reine Sachkontenbuchungen das Konto Geldtransit oder Scheckein- bzw. –ausgang. 

In Zusammenhang mit der Multisuche steht die obere Tabelle für eine Konten-Auswahl zur Verfügung (Kontonummer und Kontobezeichnung). So kann die Auswahl einzeln, selektiv oder blockweise erfolgen. Für die markierten Konten werden auf der Registerkarte Allgemein dann die OPs angezeigt. 

Es stehen Ihnen die Selektionsfelder Konto, Buchungsdatum, Belegnummer, Rückwirkend, Buchungsjahr und Bis Periode zur Verfügung.

### Button Drucken 

Mit diesem Button starten Sie den Druck. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in allen Feldern, nicht gespeicherte Änderungen gehen verloren. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert bzw. verarbeitet die Eingaben. Vor einer endgültigen Speicherung prüft das System die Eingabe. Wenn fehlende oder fehlerhafte Daten gefunden werden, erhalten Sie eine Meldung und die Eingabe muss entsprechend korrigiert bzw. vervollständigt werden. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Einstellungen 

Über diesen Button definieren Sie Einstellungen für einen bestimmten Programmteil. 

Siehe Kapitel Einstellungen .

### Checkbox Exakt 
Wenn Sie diese Checkbox aktivieren, wird exakt nach der eingegebenen Belegnummer selektiert. 

Tooltip: 

Wenn diese Checkbox gesetzt ist, muss der Treffer 1:1 dem gesuchten Begriff entsprechen. 
Ist die Checkbox deaktiv, erfolgt die Suche 
- ohne Beachtung der Groß-/Kleinschreibung 
- nach Übereinstimmung an beliebiger Stelle 

### Checkbox Rückwirkend 
Über diese Checkbox ist es möglich, auch Sach-OP-Listen für bereits abgelaufene Zeiträume aufzurufen. 

### Combobox Buchungsjahr
