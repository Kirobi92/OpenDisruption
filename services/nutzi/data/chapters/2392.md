# Konto Information Registerkarte Zahlungen 
Konto Info Registerkarte Zahlungen 

Diese Registerkarte weist in der oberen Tabelle die Zahlungen von Kunden bzw. Lieferanten (auch in Fremdwährung) aus und informiert in der unteren Tabelle über die Verwendung der Zahlung. Über den Button kann die Zahlungsverwendung gedruckt werden. 

Zur Selektion können Sie ein Datum oder einen Datumsbereich eingeben. 

Obere Tabelle: 

Hier werden grundsätzlich alle Buchungen ausgewiesen, die über den Buchungskreis DA oder DZ bzw. KA oder KZ gebucht wurden, z.B. über Debitor-Ausgleich, Einzugsvorschlag, Zahlungsvorschlag, wenn das Gegenkonto als Geldkonto ( Sachkonto , Registerkarte Allgemein ) gekennzeichnet ist. 

Ist das Transitkonto ( Parameter Finanzbuchhaltung , Registerkarte Konten ) als Geldkonto ( Sachkontenstamm , Registerkarte Allgemein ) gekennzeichnet, wird dieser Satz hier ebenfalls ausgewiesen. Automatisiert wird beispielweise die Rücklastschrift auf Voll-Ausgleich über Transitkonto gebucht. 

Siehe Kapitel Parameter Finanzbuchhaltung . 

Siehe Kapitel Sachkonto . 

Für die markierte Zahlung kann eine Zahlungsverwendung erstellt werden. Für die nachträgliche Erstellung der Zahlungsverwendung ist die tatsächliche Reihenfolge der Buchung ausschlaggebend, um die Spalten 'Erforderlich' und 'noch offen' nachträglich zu ermitteln. 

Untere Tabelle 'Verwendung': 

Hier wird die Verwendung des Zahlungsbetrages ausgewiesen. Sie können entnehmen, in welcher Höhe auf den jeweiligen Offenen Posten eine Zahlung (z.B. Voll-Ausgleich, Teil-Zahlung, Rest-Zahlung) gebucht wurde, sowie die Höhe eines gebuchten Skontos und/oder einer gebuchten Ausbuchung. 

Die Konto Info weist zum gewählten Kundenkonto bzw. Liefeantenkonto die Zahlungseingänge und –ausgänge auf. Wenn Sie einen Zahlungseingang auswählen, wird die Verwendung zum Zeitpunkt der Buchung aufgelistet: Die bebuchten offenen Posten sind ausgewiesen und Sie können entnehmen, ob es eine Voll-Zahlung, Teil-Zahlung oder Rest-Zahlung war. 

Diese Informationen können Sie über den Report Zahlungsverwendung auch an Ihren Kunden bzw. Lieferanten versenden. 

Rücklastschriften weisen in dieser Tabelle generell keine Info aus. Für Rücklastschriften öffnet sich beim Buchen immer die Differenzmeldung Detail . 

Siehe Kapitel Differenzmeldung Detail . 

Report Zahlungsverwendung 
Die Datenübergabe aus der unteren Tabelle, der Zahlungsverwendung, enthält alle Daten der ausgewählten Zahlung erweitert und alle Angaben, die z.B. der Steuerberater zur Verbuchung benötigt. 

Konto Info Registerkarte Zahlungen Datenübergabe 

Hierfür markieren Sie in der oberen Tabelle die Zahlung, in der unteren Tabelle wird die Zahlungsverwendung dargestellt. Die Datenübergabe erfolgt durch Klick mit der rechten Maustaste in die untere Tabelle an beliebiger Stelle. Wählen Sie aus dem Kontextmenü die Auswahl „Datenübergabe an MS Excel / OO Calc“. Die Dialogtabelle weist bereits die Dezimalsummen aus. 

Zeilen-Hintergrundfarbe: 

hellgrau: Zahlung, gebucht auf einen vorhandenen offenen Posten, als Voll-/Teil-/Rest-Zahlung. 
dunkelgrau: Summenzeile 
türkis: In dieser Buchung neu angelegter OP. 
hellgelb: Dieser OP resultiert aus einer Rücklastschrift.

### Dock Panel Datumstyp 

Über diesen Bereich können Sie die Selektion auf Datumstypen einschränken. Das Buchungsdatum ist voreingestellt. 

Wenn der Radiobutton Tagesdatum aktiviert ist, wird die Spalte ‚Tagesdatum‘ in der Tabelle angezeigt und zwar mit Uhrzeit (z.B. 08:00).
