# IDEA Export Registerkarte Tabellen 
IDEA Export Registerkarte Tabellen 

Diese Registerkarte wird zugänglich, nachdem Sie die Registerkarte Medien geöffnet und hier einen Datenträger / Unterordner ausgewählt haben. 

Im rechten Teil dieser Registerkarte legen Sie noch weitere Details für die Tabelle fest: 

- Die Bezeichnung und die URL können frei gewählt werden. - Für den Datenexport können Trennzeichen, Text-Hüllen (um z.B. zusammengehörenden Text zu „fixieren“) und Datensatz-Trenner (sie steuern das Format der CSV-Datei) definiert werden. - Im Feld SQL-Statement kann pro Tabelle ein SQL-Statement hinterlegt werden. 
### Button Eingrenzen 

Beim Aufruf des Dialogs Suchbedingung erscheint eine Abfrage, die anzeigt ob auf dem TreeView die passende Tabelle ausgewählt ist (es war sonst nicht möglich an die nötigen Informationen für den Dialog Suchbedingung heranzukommen). 

Des Weiteren sind die Felder Bezeichnung auf der Registerkarte Spalten sowie Tabellen unzugänglich, da aufgrund evtl. Änderungen der Export scheitert, weil mit diesen Bezeichnungen die Selects für die Datenbank aufgebaut wurden und diese aber auch für das SQL gebraucht werden. 

Sobald das Where-Statement zusammengebaut wurde und an den Dialog zurück geschickt wird, wird diese gleich konvertiert, so dass es beim Export zur Verfügung steht. Es ist nämlich nicht möglich die Konvertierung erst während des Exports zu machen, da Informationen über das Objekt fehlen.
