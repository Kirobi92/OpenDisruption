# Html Report Designer Registerkarte Links 
Html Report Designer Registerkarte Links 

Auf dieser Registerkarte steuern Sie, ob weitere Reports im aktuellen Report integriert oder verlinkt werden sollen. Diese Subreports werden dynamisch angezeigt, je nach Berechtigung des Sachbearbeiters. 

Der hinzuzufügende Subreport muss bereits entsprechend vorhanden sein und kann über den Button ausgewählt werden. 

In der Spalte ‚Report Auswahl‘ steht die Bezeichnung des Reports. In der Spalte ‚Titel‘ kann eine abweichende Bezeichnung für die Anzeige im Html Report angegeben werden, z.B. Wiedervorlagen. 

Über die Checkbox in der Spalte 'Eingebettet' wird gesteuert, ob der Inhalt des Subreports im Html Report integriert angezeigt oder der Titel des Subreport als Link zum Report angezeigt wird. Über die Spalte ‚Report Vorlage‘ kann dem Subreport eine abweichende Vorlage zugeordnet werden. Dies übersteuert dann die eigentliche Report-Vorlage des Sub-Reports, wenn diese eigenständig angezeigt würde. 

Über die Spalte ‚Max. Zeilen‘ geben Sie bei Listen-Reports an, wie viele Zeilen angezeigt werden sollen, z.B. 5 für die letzten 5 Wiedervorlagen, damit nicht alle Wiedervorlagen angezeigt werden.

### Tabellenspalte Eingebettet 
Über die Checkbox in dieser Spalte wird gesteuert, ob der Inhalt des Subreports im Html Report integriert angezeigt oder der Titel des Subreport als Link zum Report angezeigt wird. Über die Spalte ‚Report Vorlage‘ kann dem Subreport eine abweichende Vorlage zugeordnet werden. Dies übersteuert dann die eigentliche Report-Vorlage des Sub-Reports, wenn diese eigenständig angezeigt würde. 

### Tabellenspalte Reportvorlage 
Dies ist die eindeutige ID dieser Konfiguration (GUID) für den Dateinamen. Ist noch keine ID gesetzt, handelt es sich noch um einen neuen, ungespeicherten Datensatz.
