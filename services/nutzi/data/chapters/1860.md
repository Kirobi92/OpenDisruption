# Inventur - Artikelbild 
Inventur - Artikelbild 

Dieser Dialog öffnet sich im Dialog Inventurscan über den Button . Hier werden Ihnen Detailinformationen zum Artikel und das Artikelbild angezeigt. 
## Steuerelemente des Formulars 

### Feld Artikel 
Hier wird die Artikelnummer angezeigt. 

Tooltip: 

Zeigt die Artikelnummer. 

### Feld Lagereinheit 
Hier wird die Lagereinheit des Artikels angezeigt. 

Tooltip: 

Angabe der Lagereinheit. Dazugehörige Pickliste heißt „Mengeneinheit“. 

### Feld Artikelbezeichnung 
Hier wird der Name zur Artikelnummer angezeigt. 

Tooltip: 

Zeigt den Artikelnamen.
