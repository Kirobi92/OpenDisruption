# Gruppentitel ändern 
Gruppentitel ändern 

Wenn Sie in einem Vorgang (z.B. Auftrag, Bestellung) auf der Registerkarte Positionen den Radiobutton Gruppentitel aktivieren und anschließend den Button Neu drücken, haben Sie die Möglichkeit, verschiedene Auftragspositionen in einer Gruppe zusammen zu fassen. 

Für diese können hier in diesem Dialog ein Titel und eine Beschreibung hinterlegt oder aus bereits vorhandenen Stammdaten durch Auswahl in der Combobox Titel übernommen werden. Das kann sinnvoll sein, um z.B. eine bessere Übersicht bei einer Vielzahl von Positionen zu gewähren. 

Ist für den Start- oder Endetext eine Automatik hinterlegt, so wird der entsprechende Textbaustein beim Einfügen des Auftrags automatisch gesetzt. 

Siehe Kapitel Textbausteine .

### Checkbox Kein Einzelpreis 
Wenn Sie diese Checkbox aktivieren, werden in den Formularen (d.h. von Angebot bis Rechnung) keine Einzelpreise eingetragen. Die Funktion für das Unterdrücken des Einzelpreises kann genutzt werden, um detailliertere Preise gegenüber dem Kunden nicht kenntlich zu machen, z.B. bei einem Artikel-Set. 

Bei der Funktion Kein Einzelpreis werden auch alle Positionen der Gruppe automatisch zu einer Positionsgruppe zusammengefasst. 

### Checkbox Kein Gruppenpreis 
Wenn Sie diese Checkbox aktivieren, wird im Formular (d.h. von Angebot bis Rechnung) kein Gruppenpreis eingetragen. 

### Abbildung Bilddatei 
Hier wird die ausgewählte Bilddatei angezeigt. 

### Button Suche 

Mit diesem Button können Sie für den Gruppentitel eine Bilddatei auswählen und anhängen. 

Das ausgewählte Bild wird analog zum Artikelstamm in der Datenbank gespeichert. 

### Button Löschen 

Mit diesem Button können Sie die Bilddatei wieder entfernen.
