# Intrastat Firmen / Firmenstamm für Meldungen 
Intrastat Firmen 

Wenn Sie diesen Dialog aus dem Dialog Intrastat >> Stamm >> Intrastat Firmen aufrufen, heißt er Intrastat Firmen . 

Wenn Sie den Dialog aus dem Navigator >> System >> Finanzbuchhaltung >> Firmenstamm für Meldungen öffnen, heißt er Firmenstamm für Meldungen ! 
### Intrastat Firmen 
Dieser Dialog öffnet sich im Dialog Intrastat >> Stamm >> Intrastat Firmen . Hier werden die Daten des Unternehmens angegeben, welches zur Abgabe der Intrastat-Meldung verpflichtet ist. Auf diese Weise ist es auch möglich, mehrere Mandanten oder Gesellschaften zu einer Meldung zusammenzufassen. 

Im Dialog Intrastat Firmen sind zwingend folgende Felder zu füllen, falls im jeweiligen Land erforderlich: 

- Vollständige Adresse (Firma 1, Firma 2, Firma 3, Firma 4, Straße, PLZ, Ort, PLZ, PF) - Umsatzsteueridentifikationsnummer - Intrastat BuFa-Nr. - Bundesland - Land (Das Land wird bei der Intrastat-Meldung geprüft, ob es für Intrastat freigegeben ist, oder nicht.) 

### Firmenstamm für Meldungen 
Firmenstamm für Meldungen 

Es kommt vor, dass große Unternehmen mit Niederlassungen im europäischen Ausland dort UVA-Meldungen abgeben. Dafür werden Finanzamtsdaten des jeweiligen Landes benötigt. Diese Daten sind im Dialog Firmenstamm für Meldungen zu erfassen. Der Dialog steht im Navigator unter System >> Finanzbuchhaltung zur Verfügung. 

Der erste Satz (System Firmenstamm) weist immer die Daten des Firmenstamms informativ aus. Die weiteren Sätze befüllen Sie für die anderen Länder (nicht Systemland) mit den relevanten Daten für die jeweilige UVA des anderen Landes. Für jedes Land kann nur ein Satz angelegt werden.

### Button Suche 

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten.
