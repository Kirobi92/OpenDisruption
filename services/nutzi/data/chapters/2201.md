# Serviceobjekt Registerkarte Standort 
Serviceobjekt Registerkarte Standort 

Auf dieser Registerkarte sehen Sie, wo das Serviceobjekt zu finden ist. 
### Feld Lieferanschrift 
Hier wird die Anschriftsnummer des Kunden eingetragen. 

Tooltip: 

Angabe der Lieferanschrift 

### Button Suche 

Mit diesem Button können Sie eine Versandanschrift suchen. 

Siehe Kapitel Versandanschriften . 

### Feld Adresse 
Die Adresse wird automatisch zur Anschriftnummer eingetragen. 

### Feld Standplatz 
In diesem Feld können Sie bei Bedarf z.B. noch das Stockwerk und eine Zimmernummer eingeben. 

Tooltip: 

Angabe des Standplatzes 

### Combobox Kontakt 
Hier kann der Ansprechpartner ausgewählt werden. 

Tooltip: 

Nummer des Kontakts/Partners 

### Combobox Gebiet 
Hier wird ein Gebiet / Bundesland zu Auswertungszwecken angegeben. 

Gebiete / Bundesländer werden dem Kunden im Kundenstamm auf der Registerkarte Logistik zugewiesen. 

Die Einträge selbst werden in der Pickliste gepflegt. 

Tooltip: 

Angabe eines Gebiets / Bundeslands
