# Generierung einer Bestellung in der Rahmenbestellung 
Mit Klick auf den Button Generiere im Dialog Rahmenbestellungen werden alle erfassten, noch offenen Abrufe zur gesamten Rahmenbestellung in eine Bestellung zusammengefasst. Hierfür müssen Sie zuerst über eine entsprechende Dialogbox die Einkaufart und die Lagernummer festlegen. Bei der Generierung der Bestellung wird automatisch eine neue Bestellnummer aus dem Nummernkreis vergeben. Nach erfolgreicher Generierung werden alle Rahmenabrufe, die in die Bestellung eingegangen sind, um die Belegnummer, Belegart und Positionsnummer aktualisiert. Auf diese Weise ist immer nachvollziehbar, in welche Bestellungen die Rahmenabrufe eingegangen sind. 

Wenn zu einer Position alle Abrufe generiert wurden, wird diese Position automatisch abgeschlossen. Änderungen für diese Position können dann nicht mehr vorgenommen werden. 

Wenn keine offenen Positionen mehr existieren, erscheint eine Hinweismeldung, dass die Rahmenbestellung abgeschlossen werden kann. Wenn Sie diese Meldung mit Ja bestätigen, wird der Status der Rahmenbestellung auf 2 (= abgeschlossen) gesetzt.
