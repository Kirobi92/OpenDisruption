# Nummernkreise im Einkauf 
Nummernkreise Einkauf 

Dieser Dialog öffnet sich im Navigator >> Einkauf >> Nummernkreise . eNVenta ERP bietet Ihnen die Möglichkeit, Nummernkreise, z.B. für Lieferanten- und Bestellnummern zu hinterlegen. Dies hat den Vorteil, dass vom Programm automatisch Nummern vergeben werden. Jedes Mal, wenn Sie z.B. eine neue Bestellung erfassen, wird die nächsthöhere Nummer ermittelt und der Bestellung zugewiesen. 

Als Bezeichnung in den Format-Feldern bedeutet: 
N = Nummer (für jede Stelle ein N, bei einer 5-stelligen Zahl also NNNNN) 
T = Tag 
J = Jahr 
M = Monat 
C = Counter 

Um einen Überlauf von Nummernkreisen zu vermeiden und einen "Neustart" bei einem Jahreswechsel zu ermöglichen, gibt es folgende Möglichkeit: 

Wenn es gewünscht ist, dass der Counter (Zähler) für jedes neue Jahr oder jeden neuen Monat immer wieder bei 1 anfängt, wird an das Format-Pattern am Ende ein C angehängt. Z.B. JJJJMMNNNNNNNNNN C . 

Das C markiert, dass es sich um einen Counter handelt. Wenn der Counter beim Ermitteln der Zahl des letzten ermittelten Werts im Jahr / Monatsteil nicht dem aktuellen Jahr / Monat entspricht, wird wieder bei 1 angefangen. 

N = Nummer (für jede Stelle ein N, bei einer 5-stelligen Zahl also NNNNN) 
J = Jahr (entweder 2- oder 4-stellig) 
M = Monat (2-stellig) 
T = Tag (2-stellig) 
C = Kennung für Zähler 

Ist im Format ein MM für den Monat enthalten, so erfolgt immer bei Monatswechsel die Nummerierung von Beginn an, unabhängig ob ein JJ für die Jahreszahl vorhanden ist. Wenn nur pro Jahr ein Neubeginn erfolgen soll, darf die Kennung MM für Monat nicht Bestandteil des Formats sein. 

Nach Aufruf des Programmteils Nummernkreise enthält das Feld Nummer immer die zuletzt vergebene Nummer des jeweiligen Nummernkreises. 

Gibt es in einem Nummernkreis keine Möglichkeit mehr eine eindeutige Nummer zu generieren, erscheint eine entsprechende Fehlermeldung. Diese Meldung kommt, wenn keine Nummer mehr frei ist bzw. bricht nach 10.000 Versuchen das ganze ab. 

Wichtig: Notieren Sie sich deshalb die anfangs eingetragenen Startwerte und überprüfen Sie anhand der Notizen regelmäßig, ob ein Nummernkreis an seine obere Grenze zu laufen droht. 

Die maximale Länge einer Nummer mit allen Bestandteilen beträgt 10 Stellen. 

Es gelten folgende Regeln: 
1.	Jeder Bestandteil darf nur einmal im Format vorkommen. 
2.	Das Format muss einen fortlaufenden Teil mit mindestens einem N enthalten. 

Beispiele: 
JJNNNN 
TTMMJJNNN 
NNNN 
NNNNJJ 

unzulässige Beispiele: 
JJNNNTTNN 
JJJJJNNNN 
TMMJJ 

Die Nummernkreise für das Lieferantenkonto und das V-Lieferantenkonto betreffen die Buchhaltung und müssen auf die Kontenkreise der verwendeten Buchhaltung abgestimmt sein. Bei Verwendung der Finanzbuchhaltung von eNVenta ERP sind die Nummernkreise bei Änderung automatisch auch in der Finanzbuchhaltung eingestellt. 

Lieferant: 
Nummer für (Normal-)Lieferanten. 

Bestellung: 
Belegnummer bei Bestellungen. 

Lieferantenkonto: 
Kontonummer des Lieferanten. Wird in eNVenta ERP-Finanzbuchhaltung verwendet. 

Es wird eine Kontonummer für das Personenkonto aus diesem Nummernkreis gezogen. Wenn der Parameter Kunde/Lieferant Konto gleich im Dialog Parameter Allgemein , Registerkarte Allgemein , gesetzt ist, wird automatisch die Lieferantennummer auch als Kontonummer gesetzt, unabhängig vom Nummernkreis. 

Aus Gründen der Übersichtlichkeit empfiehlt es sich, dass die Nummernkreise für den Lieferanten und das Lieferantenkonto übereinstimmen. 

Einmallieferant: 
Nummer für Einmallieferanten. 

Einmallieferant Konto: 
Kontonummer für Einmallieferanten. 

Es wird eine Kontonummer für das Personenkonto aus diesem Nummernkreis gezogen. Wenn der Parameter Kunde/Lieferant Konto gleich im Dialog Parameter Allgemein , Registerkarte Allgemein , gesetzt ist, wird automatisch die Lieferantennummer auch als Kontonummer gesetzt, unabhängig vom Nummernkreis. 

Aus Gründen der Übersichtlichkeit empfiehlt es sich, dass die Nummernkreise für den Einmallieferanten und seine Kontonummer übereinstimmen. 

NB-Lieferant: 
Nummer für Verbandslieferanten. Beim Einfügen des Lieferanten wird anhand des Lieferantentyps "Verbandslieferant" eine Nummer aus diesem Nummernkreis gezogen. 

NB-Lieferkonto: 
Nummer für Verbandslieferanten. Beim Einfügen des Lieferanten wird anhand des Lieferantentyps "Verbandslieferant" eine Kontonummer für das Personenkonto aus diesem Nummernkreis gezogen. Wenn der Parameter Kunde/Lieferant Konto gleich im Dialog Parameter Allgemein , Registerkarte Allgemein , gesetzt ist, wird automatisch die Lieferantennummer auch als Kontonummer gesetzt, unabhängig vom Nummernkreis. 

Anfrage: 
Belegnummer bei Anfragen. 

Wareneingang: 
Nummer für Wareneingänge. 

Inventur: 
Nummer für Inventuren. 

Einkauf-Retouren: 
Nummer bei Retouren. 

EDI Transfer Nr.: 
Nummer für den EDI Transfer. 

Anfragen-Vergleichsnummer: 
Nummer für Anfragen bei verschiedenen Lieferanten, um diese vergleichen zu können 

Artikelnummer: 
Nummer für einen neuen Artikel 

Rahmenvertrag EK:

Nummer für Artikel

### Tabellenspalte Format 
Hier wird das Zahlenformat für die jeweiligen Nummernkreise festgelegt. Hier sind als Standardwerte ausschließlich numerische Werte vorgegeben. Es ist davon abzuraten, andere als numerische Nummernkreise zu verwenden. 

N = Nummer (für jede Stelle ein N, bei einer 5-stelligen Zahl also NNNNN) 
J = Jahr (entweder 2- oder 4-stellig) 
M = Monat (2-stellig) 
T = Tag (2-stellig) 
P = Prüfziffer nach Modulo10 (1-stellig, muss letztes Zeichen im Format sein) 

Insgesamt maximal 18-stellig 

C als Kennung für Zähler: 
Ein C wird wie ein N interpretiert, bewirkt jedoch, dass bei Monats- /Jahreswechsel die Nummer auf 1 zurückgesetzt wird. 
Enthält eine Formatvorgabe mindestens ein C, gilt: 
- M enthalten – Nummer = 1 bei Monatswechsel - J enthalten – Nummer = 1 bei Jahreswechsel

### Tabellenspalte Nummer 
In dieser Spalte werden die jeweiligen Startwerte für die Nummernkreise festgelegt. Dabei ist unbedingt zu beachten, dass die Nummernkreise groß genug gewählt werden. D.h. richten Sie sich am Besten einen großen Spielraum zwischen Start- und Endwert ein, damit möglichst viele Geschäftsvorfälle getätigt werden können, bevor ein neuer Nummernkreis angelegt werden muss. 

Wenn die Differenz zwischen Start- und Endwert zu klein gewählt wird, kann es vorkommen, dass Ihnen der Nummernkreis innerhalb kürzester Zeit überläuft , weil er voll ist. In diesem Fall droht u.U. ein Systemstillstand! 

Der Nummernkreis für die Vergabe von Artikelnummern aus dem Artikel Schattenstamm ist hier ebenfalls vorhanden und kann geändert werden. 

Tooltip: 

Geben Sie den Startwert eines Nummernkreises ein.
