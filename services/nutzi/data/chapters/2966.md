# Inventory Report - Adapter definieren 
Bevor Sie die Arbeit mit dem Menü Gateway Adapter Kontext beginnen, sind zwingend die notwendigen Adapter zu definieren. Diese Einträge stehen dann im Dialog Gateway Adapter Kontext in der Spalte ‚Adapter Nr‘ zur Verfügung. 

Der Eintrag erfolgt über Menü Optionen >> Gateway Adapter Kontext und muss zwingend den Namen „AdapterEDI“ haben. 

In den Picklisten ( System >> Allgemein >> Picklisten ) sind im Vorfeld nachfolgende Einstellungen zu hinterlegen: 

- EDI Message Format - EDI Message Type - Laendereinstellungen (Cultureinfo) 
Beim Lieferant bzw. Kunde >> Registerkarte Allgemein 

- ILN 
### Typ ORDERS 
Folgende Versandarten erwartet der E/D/E im ORDERS Import: 

10 - Paket Standard (Lieferung per UPS) 
11 - Paket Express 9:00 (Lieferung per UPS bis 09:00 Uhr des Folgetages) 
12 - Paket Express 10:30 (Lieferung per UPS bis 10:30 Uhr des Folgetages) 
13 - Paket Express 12:00 (Lieferung per UPS bis 12:00 Uhr des Folgetages) 
20 - Schnellvers. Sped. (Lieferung per Spedition für zum Beispiel gefährliche Güter oder grosse Güter (Wassertank) 
30 - Standardversand - Standardlieferung per E/D/E-LKW auch genannt Tour oder Route 

Wird keine Versandart mitgegeben, geht der E/D/E davon aus, dass die bestellte Ware mit der Standardtour geliefert wird. 

Für den ORDERS-Export müssen die Events unterhalb der Tabelle aktiviert werden. Dies sind folgende Events: 

- Bestellabschluss ausgelöst - Bestellung aktualisiert - EDI Versand Bestelländerung ausgelöst (wenn der Disponent eine Abfrage erhalten soll) 
Für INVRPT muss zusätzlich noch das Event „Lagerbestandsbericht ausgelöst“ aktiviert sein.

### Lieferanten-/Kundendaten parametrisieren 
Werkzeuge >> Gateway >> Optionen >> EDI Teilnehmer 

Nachricht INVRPT 
Import / Export I / E 
Nachrichtenformat EDI 
| a: Lieferant bzw. Kunde auswählen | Die ILN muss hinterlegt sein. |
| --- | --- |
| b: Nachrichten Auswahl | Dem Lieferanten bzw. dem Kunden die entsprechende Nachricht zuordnen, hier wird unterschieden zwischen Import/Export, dem Nachrichtentyp und dem Nachrichtenformat. |
| c: Art. Auswahl | Hier werden die Artikeldaten nach unterschiedlichen Kriterien innerhalb eines Lagers selektiert; die hier selektierten Artikel werden in den Lagerbestandbericht übernommen. |
| d: | Hier wird die FTP Verbindung hinterlegt. |

Nachrichtentyp und Nachrichtenformat müssen zuvor in einer Pickliste angelegt werden. 

### Job generieren 
Nachdem alle Parameter bzgl. Pfade, Artikel, Lieferanten/Kunden gesetzt sind, kann ein Job generiert werden. Hierzu bitte den entsprechenden Button in der Kopfzeile anklicken; Dialog Job-Verwaltung öffnet sich. 

Siehe Beschreibung Jobs/Job-Server . 

### Jobs kontrollieren 
Werkzeuge >> Gateway >> Optionen >> Gateway Protokoll 

Über das Gateway Protokoll können die Jobs auf Fehler, Warnung etc. kontrolliert werden.
