# Kapitel NV.ERP.FI.AssetsAcc.cdAssetParameter.shtNotInFinAcc

Wenn die Checkbox in dieser Spalte aktiviert ist, werden die Daten nicht an die Finanzbuchhaltung übergeben. 

Tooltip: 

Wenn aktiviert, werden die Daten nicht an die Fibu übergeben.
