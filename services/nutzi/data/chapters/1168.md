# Etikettendruck im Auftrag 
Etikettendruck 

Dieser Dialog öffnet sich im Auftrag >> Druck >> Etikettendruck . Hier können Sie für einzelne Auftragspositionen Etiketten drucken. Der Prozess muss manuell angestoßen werden. 

Die Anzahl der Etiketten wird mit der Liefermenge aus der Auftragsposition vorbelegt. 

Im Artikelstamm auf der Registerkarte Verkauf gibt es das Feld Anzahl Etiketten . Hier geben Sie die entsprechende Anzahl an Etiketten für den Druck ein. Bei der Eingabe von 0 wird nie ein Etikett gedruckt, bei der Eingabe von 1 wird immer genau 1 Etikett gedruckt – unabhängig der Menge im Auftrag. 

Der zu verwendende Report wird im Kundenstamm auf der Registerkarte Warenwirtschaft hinterlegt. Dieser Report basiert auf dem Report Etikettendruck und kann über den Druck -Dialog, Button Customize , angepasst werden.
