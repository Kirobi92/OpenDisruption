# Chargen Registerkarte Allgemein 
Charge Registerkarte Allgemein 

Hier geben Sie die Daten für die Verschlagwortung an. 
### Feld Prüfzeugnis 
Geben Sie hier eine evtl. Prüfzeugnisnummer ein. 

Tooltip: 

Angabe der Prüfzeugnisnummer 

### Feld Zeugnisversion 
Die Nummer der Zeugnisversion wird eingetragen. 

### Combobox Lager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager 

### Feld Lagerplatz 
Hier wird der Lagerplatz ausgewählt. 

Tooltip: 

Angabe des Lagerplatzes 

### Combobox Einheit Charge 
Aus dieser Combobox können Sie die Maßeinheit des Artikels wählen. 

### Feld Code 1 
Dieses Code-Feld können Sie frei definieren. 

### Feld Code 2 
Dieses Code-Feld können Sie frei definieren. 

### Feld Code 3 
Dieses Code-Feld können Sie frei definieren. 

### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen.
