# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtLEAbleStorage

Wenn dieser Parameter gesetzt ist, hat das Lager die Voraussetzung erfüllt, dass hier mit Ladehilfsmitteln gearbeitet werden kann. Diese Funktion steht für das 1-stufige LVS zur Verfügung. 

Tooltip: 

Wenn aktiviert, kann mit Ladehilfsmitteln gearbeitet werden.
