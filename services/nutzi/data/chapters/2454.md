# Verbandsabrechnung Beispiel 
Nachfolgend zeigen wir an einem Beispiel, wie Verbandsabrechnungen verarbeitet werden können. Wir unterscheiden hierbei nicht nach Debitoren oder Kreditoren! 

Die beiden Varianten sind: - Button Buchen - Button Direktverbuchung 
Verbandsabrechnung 

Button Buchen = OPs umbuchen 

Die OPs werden vom Teilnehmer auf den Verband bzw. Zentralzahler umgesetzt. Das heißt, beim Teilnehmer ist der OP ausgeglichen, beim Verband bzw. Zentralzahler ist der OP offen. 

Journal 

Button Direktverbuchung = OP ausbuchen 

Verbandsabrechnung Button Direktverbuch. 

Die OPs werden beim Teilnehmer ausgeglichen, unter Beachtung von Skonto und Ausbuchung 1 bis 3. Es handelt sich hierbei i.d.R. um Gebühren. Die Ausbuchung 1 wird auf das im Verband bzw. Zentralzahler hinterlegte Verband-Ausbuchungskonto Konto 1 gebucht. Die Ausbuchung 2 etc. 

Der Saldo dieses OPs (nach Abzug von Skonto und Ausbuchung 1, 2 und 3) wird auf das Sachkonto (Registerkarte Direktverbuchung ) gebucht. Die Zahlung des Verbands bzw. Zentralzahlers wird dann gegen dieses Sachkonto gebucht.
