# Sequentieller Wareneingang 
In diesem Dialog wird eine Bestellposition für den Sequentiellen Wareneingang ermittelt. Zulässig sind nur Positionen für die bereits der Wareneingang Stufe 1 verbucht wurde. 

Die gewünschte Person kann über verschiedene Filter Eingaben ausgewählt werden. Zulässig dafür sind: 

- Bestellnummer - Retourennummer - Bestellnummer + Positionsnummer - Retourennummer + Positionsnummer - Artikelnummer - Wareneingangsnummer 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt.

### Button Barcode 

Wird dieser Button gedrückt, so wird, wenn eine Kamera am Gerät vorhanden ist, die Barcodeerkennung aktiviert. 

Dabei werden folgende Barcodetypen erkannt: Code128, EAN8, EAN13. 

### Button Einstellungen 

Dieser Button öffnet das Einstellungsmenü für den Scanner. 

### Button Aktualisieren 

Dieser Button aktualisiert das Ergebnis der Liste.
