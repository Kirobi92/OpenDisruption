# To-Do Liste 
To-Do Liste 

Dieser Dialog öffnet sich über den Button To-Do Liste im Dialog Projekt und im Dialog Call . Hier können Sie allen Beteiligten in einem Projekt jederzeit einen aktuellen Stand des Projekts bzw. der Aufgaben präsentieren (also eine Art to-do-Liste). 

Eine To-Do Liste kann nur angelegt werden, wenn ein Projekt existiert. 

Wird ein Projekt gelöscht, ist zwingend auch die To-Do Liste zu löschen! 
## Steuerelemente des Formulars 

### Bezeichnungsfeld To-Do Liste 
Tooltip: 

In diesem Dialog können Sie allen Beteiligten in einem Projekt jederzeit einen aktuellen Stand des Projekts bzw. der Aufgaben präsentieren (also eine Art to-do-Liste). 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren . 

### Menü Fenster-Information 
Über dieses Menü werden Ihnen Informationen zum Dialog angezeigt. Siehe nachfolgendes Beispiel: 

Meldungen 

### Button Drucken 

Mit diesem Button starten Sie den Druck. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

### Button Neu 

Mit diesem Button leeren Sie die Felder. 

### Button Speichern 

Mit diesem Button speichern Sie die Eingaben. 

### Button Neuer Satz 

Mit diesem Button können Sie ein neues ToDo anlegen; eine weitere Zeile wird in der Tabelle generiert. 

### Button Löschen 

Mit diesem Button können Sie einen Eintrag der Tabelle löschen.

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Feld Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Projektnummer eingeben 

### Button Suche 

Mit diesem Button können Sie ein Projekt suchen. 

Siehe Kapitel Projektauswahl . 

### Feld Projektname 
Hier wird der Projektname angegeben. 

Tooltip: 

Name des Projekts eingeben 

### Combobox Bereich 
Aus dieser Combobox wählen Sie den Bereich aus, für welchen die ToDo Liste erstellt wird. 

Tooltip: 

Bereich für das ToDo auswählen 

### Feld Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Beschreibung des ToDo's 

### Combobox Klasse 
Hier wählen Sie eine Klasse / Zuordnung zur ToDo Liste aus. Diese werden im Dialog ToDo Klassen definiert. 

Siehe Kapitel ToDo Klassen . 

Tooltip: 

Klasse auswählen 

### Feld Priorität 
Hier wird die Priorität angegeben, z.B. eilig. 

Tooltip: 

Priorität von 1 bis 9 

### Liste Status 
Zur Selektion können Sie einen oder mehrere Status anklicken. 

Tooltip: 

Status auswählen 

### Feld Position 
Hier wird eine fixe, eindeutige Positionsnummer des Datensatzes angegeben. 

Tooltip: 

Eindeutige Positionsnummer des Datensatzes. 

### Combobox Erfasser 
Hier wird der Sachbearbeiter angegeben, der den Datensatz erfasst hat. 

Tooltip: 

Erfasser des ToDos 

### Combobox Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zuständiger Sachbearbeiter 

### Feld Erledigt vom 
Für die Selektion von abgeschlossenen ToDo Listen können Sie hier ein Datum eingeben. 

Tooltip: 

Abschlussdatum des ToDo's 

### Feld Erledigt bis 
Für die Selektion von abgeschlossenen ToDo Listen in einem Datumsbereich können Sie hier ein Bis-Datum eingeben. 

Tooltip: 

Abschlussdatum bis 

### Combobox Intern/Extern 
Für die Selektion der ToDo Listen können Sie hier auswählen, ob es sich um einen internen oder externen Vorgang handelt. 

Tooltip: 

Interner oder extermer Vorgang? 

### Checkbox Nur kritische To-do's 
Wenn Sie diese Checkbox aktivieren, erfolgt die Selektion über kritische ToDo Listen. 

Tooltip: 

Wenn aktiviert, werden kritische ToDo-Listen gesucht.
