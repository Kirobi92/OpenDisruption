# Kapitel NV.ERP.Base.Common.cdCountry.shtImportRelevant

Dieses Kennzeichen legt fest, ob das Land für zollrelevante Geschäftsvorfälle im Importbereich wichtig ist. Dieses Kennzeichen hat nur informativen Charakter.
