# EDI Adapter verwalten 
EDI Adapter 

Dieser Dialog öffnet sich im Dialog Gateway Adapter Kontext , wenn Sie in der oberen Tabelle eine Zeile mit einem EDI Adapter markieren und den Button rechts der Tabelle klicken. 

Es ist möglich, EDI-Dateien per Drag & Drop zu importieren. Dafür stehen 2 Möglichkeiten zur Verfügung: 

- Klicken Sie die EDI-Datei im Datei-Explorer von Windows an und ziehen diese bei gedrückter Maus auf die Protokoll-Tabelle im EDI Adapter. - Markieren Sie den EDI-Nachrichten-Text in einem Texteditor, wie z.B. dem Notepad und ziehen ihn bei gedrückter Maus auf die Protokoll-Tabelle im EDI Adapter. 
Zur Funktionsweise lesen Sie bitte das Kapitel EDI Adapter .

### Checkbox Daten-Rollback (Test-Modus) 
Wenn Sie diese Checkbox aktivieren, werden keine Daten in die Datenbank geschrieben. Auf diese Weise kann ein Import oder Export wiederholt gestartet werden. 

Tooltip: 

Es werden keinerlei Daten in der Datenbank verändert. Allerdings werden die EDI-Dateien trotzdem erstellt bzw. verschoben. 

### Checkbox Nur Adapter-Kontext spezifisch 
Wenn diese Checkbox aktiviert ist, werden nur die EDI-Nachrichten-Konfigurationen berücksichtigt, die explizit für den aktuell betroffenen Adapterkontext definiert wurden. Andernfalls werden auch zusätzlich diejenigen berücksichtigt, die keinem Adapterkontext zugeordnet sind. 

Diese Option wird nicht gespeichert und wird daher beim Ausführen durch einen Job nicht berücksichtigt. 

Bei der Verarbeitung durch einen Job verhält es sich also so, als ob die Checkbox deaktiviert wäre. 

Siehe dazu Spalte 'Adapter Kontext Nr' im Dialog EDI-Teilnehmer .

E = Export / I = Import 

### Tabellenspalte Schritt 
Tooltip: 

Manche Verarbeitungen werden in mehreren Schritten durchgeführt. 
U.A. bei dem Export im Adapter (der Anwendung) "cMessageConverterEDI" wird das Laden und Versenden in 2 Schritten angezeigt.

Tooltip: 

Zeigt an welches Modul des Adapters ausgeführt wurde.
