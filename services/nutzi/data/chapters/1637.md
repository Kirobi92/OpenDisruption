# Sachbearbeiter- Scanner Registerkarte Zonen & Skills 
Sachbearbeiter Scanner Registerkarte Zonen & Skills 

Diese Registerkarte steht nur für LVS-Scanner Benutzer zur Verfügung. Für Benutzergruppen ist diese Registerkarte nicht verfügbar. 

Um in bestimmten Lagerbereichen arbeiten zu können, bzw. zu dürfen, benötigt ein Sachbearbeiter bestimmte Fähigkeiten und Berechtigungen (sogenannte Skills). Dazu gehört z.B. der Gabelstapler-Führerschein in einer Hochregalzone. Um diese Informationen zu pflegen, steht Ihnen die Registerkarte Zonen & Skills zur Verfügung. Sie enthält 2 Tabellen: Gesperrte Zonen und Fähigkeiten . 
### Tabelle Gesperrte Zonen 

Ist eine Zone für einen Mitarbeiter von den Skills her zugänglich, soll aber in einer bestimmten Zone nicht arbeiten dürfen, kann der Lagerleiter diese bestimmte Zone für den Mitarbeiter sperren. 
### Tabellenspalte Zone 
In dieser Combobox wird die Zonennummer ausgewählt / angezeigt. 

Tooltip: 

Angabe der Lagerzone 

### Button Hinzufügen 

Mit diesem Button können Sie einen weiteren Eintrag anlegen. 

### Button Löschen 

Mit diesem Button kann ein Eintrag gelöscht werden. 

### Tabelle Fähigkeiten 

Hier können die Fähigkeiten (Skills) eines Mitarbeiters eingetragen werden. 
### Tabellenspalte Skill 
In dieser Combobox wird der Skill ausgewählt. 

### Button Hinzufügen 

Mit diesem Button können Sie einen weiteren Eintrag anlegen. 

Tooltip: 

Neue Fähigkeit 

### Button Löschen 

Mit diesem Button kann ein Eintrag gelöscht werden. 

Tooltip: 

Mit diesem Button löschen Sie eine Fähigkeit.
