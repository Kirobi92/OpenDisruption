# Zusätzliche Anschriften Registerkarte Memo 
Zusätzliche Anschriften Registerkarte Memo 

Auf dieser Registerkarte kann eine interne Information zur Adresse eingefügt werden. 
### Feld Memo 
Wenn Sie diese Checkbox aktivieren, wird die Versandadresse als Standard-Versandadresse des Kunden gekennzeichnet. Diese hat Priorität bei der Auftragserfassung! 

Wenn Sie der Versandanschrift eine Information / ein Memo mitgeben möchten, ist das unbedingt hier im Feld Memo zu erfassen. Dieses wird dann im Auftrag gezogen und übersteuert das Versandmemo, das im Kundenstamm angegeben ist. 

Tooltip: 

Enthält interne Textinformationen.
