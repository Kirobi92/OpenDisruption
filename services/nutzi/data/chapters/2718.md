# Dimensionen 
Kore Dimensionen 

Dieser Dialog öffnet sich im Buchen Kore >> Stamm >> Dimensionen . 

Zur Verbesserung der Auswertungsmöglichkeiten gibt es die flexible Möglichkeit, Buchungen verschiedenen Dimensionen zuzuordnen. Diese Zuordnungen können beim manuellen Buchen von Kostenrechnungen vorgenommen werden. Diese Zuordnungen sind dann in den verschiedenen Auswertungen als Auswahlkriterium vorhanden. 

Sollen die Dimensionen auch bei der Kore-Übernahme automatisch gefüllt werden, so muss dies ausprogrammiert werden. Hierfür steht eine Schnittstelle bereit. Haben Sie hierzu Fragen, wenden Sie sich bitte an Ihren Administrator oder an die Nissen & Velten Software GmbH. 

Für jede Dimension, welche Sie benutzen möchten, vergeben Sie eine eindeutige Bezeichnung. Diese erscheint im Dialog Buchen Kore und Verteilung (im Dialog Buchen in der Finanzbuchhaltung) am Ende der Tabelle als Spaltenname. 

Über die Checkbox Numerisch wird eingestellt, dass nur Zahlen in diese Dimensionen eingetragen werden können. 

Über die Checkbox Aktiv wird eingestellt, dass die Dimension während des Buchens angezeigt und gespeichert wird. Auch stehen in den Auswertungen nur aktive Dimensionen zur Verfügung. 

Sie können maximal 10 verschiedene Dimensionen pflegen.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert. 

### Button Schließen 

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Checkbox Numerisch 
Wenn Sie diese Checkbox aktivieren, wird eingestellt, dass nur Zahlen in diese Dimensionen eingetragen werden können. 

Tooltip: 

Wenn aktiviert, werden nur Zahlen in diese Dimensionen eingetragen. 

### Checkbox Aktiv 
Tooltip: 

Wenn aktiviert, werden Dimensionen während des Buchens angezeigt und gespeichert.
