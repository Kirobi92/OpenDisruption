# Besuchsstatistik 
Besuchsstatistik 

Der Dialog Besuchsstatistik öffnet sich im Besuchsbericht >> Statistik >> Besuchsstatistik und bietet Ihnen eine Vielzahl von Möglichkeiten für die Auswertung der erfassten Besuche. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Besuchsstatistik 
Tooltip: 

Dieser Dialog bietet Ihnen eine Vielzahl von Möglichkeiten für die Auswertung der erfassten Besuche. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Dock Panel Selektion 

Für eine gezielte Auswertung können Sie die Selektion über diese Radiobuttons weiter einschränken. 

### Field Panel Auswahl des Zeitraums 

In diesem Bereich kann die Selektion zeitlich eingeschränkt werden. Entweder mit Auswahl aus der Combobox oder über die Felder Vom / Bis . 
### Bezeichnungsfeld Zeitraum 
Tooltip: 

Geben Sie den Zeitraum für die Statistik an. 

### Radiobutton Periode 
Wenn Sie diesen Radiobutton drücken, können Sie aus der danebenstehenden Combobox den Zeitraum auswählen. 

### Combobox Auswahl 
Zur Selektion über einen Zeitraum stehen die Einträge 
- Tag - Woche - Monat - Quartal - Jahr 
zur Verfügung. Dieser Zeitraum gibt die Spanne der Analyse an. 

Wird Woche ausgewählt, werden in der Tabelle die einzelnen Kalenderwochen aufgelistet. Wird dazu die Checkbox Period. frei aktiviert, erfolgt die Selektion genau zu dem Termin, der im Feld Vom/Bis eingetragen ist und nicht mehr die komplette Kalenderwoche (Montag-Sonntag). 

### Radiobutton Von / Bis 
Wenn Sie diesen Radiobutton aktivieren, geben Sie in die Felder Von / Bis einen individuellen Zeitraum ein. 

### Feld von 
Hier werden ein Datum und eine Uhrzeit angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Feld bis 
Um die Suche über einen Datumsbereich einzuschränken, wird hier ein Bis-Datum angegeben. 

Tooltip: 

Eingabe von Datum und Uhrzeit 

### Combobox Periode 
Hier kann ein Zeitraum für die Auswertung ausgewählt werden. 

Tooltip: 

Hier wählen sie die Periode aus. 

### Checkbox Periodenanfang frei 
Mit dieser Checkbox können Sie den Anfang der Periode frei definieren. 

Bei nicht-aktiver Checkbox wird das eingegebene Datum der Periode angepasst. 

Beispiel: 
Bei der Periode "Woche" wird als Startdatum ein Mittwoch angegeben. Nun zeigt eNVenta dennoch den Zeitraum zwischen dem Montag davor und dem Sonntag danach an. 

Bei aktiver Checkbox wird das Startdatum nicht angepasst. 

Beispiel: 
Bei der Periode "Woche" wird als Startdatum ein Mittwoch angegeben. Nun zeigt eNVenta den Zeitraum zwischen dem Mittwoch und dem Dienstag danach.
