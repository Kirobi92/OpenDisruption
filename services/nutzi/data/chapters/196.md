# Diagramm Einstellungen Registerkarte Farben

Allgemeine Funktionalitäten in eNVenta ERP / Diagramm Einstellungen 

# Diagramm Einstellungen Registerkarte Farben 
Diagramm Einstellungen Registerkarte Farben 

Auf dieser Registerkarte stellen Sie die farbliche Darstellung der Grafik ein. 
### Combobox Schema 
Wählen Sie aus der Combobox das Schema für die Anzeige der Grafik aus. 

### Combobox Skalierung 
Wählen Sie aus dieser Combobox die Skalierung aus. 

### Combobox Skal. Tiefe 
Wählen Sie aus dieser Combobox die Skalierungstiefe aus. 

### Steuerelement Anfangsfarbe 

Wählen Sie hier eine Anfangsfarbe aus. Wenn Sie auf das farbige Kästchen klicken, öffnet sich eine Farbpalette. 

### Steuerelement Endfarbe 

Wählen Sie hier eine Endfarbe aus. Wenn Sie auf das farbige Kästchen klicken, öffnet sich eine Farbpalette. 

### Steuerelement Umrand.farbe 

Wählen Sie eine Farbe für die Umrandung aus. Wenn Sie auf das farbige Kästchen klicken, öffnet sich eine Farbpalette. 

### Checkbox Farbe nach Zeilen 
Wenn Sie diese Checkbox aktivieren, erscheint jeder Balken im Diagramm mit einer anderen Farbe. Der Farbverlauf liegt zwischen Anfangs- und Endfarbe. 

### Checkbox Umrandung 
Wenn Sie diese Checkbox aktivieren, erhalten Sie eine Umrandung. 

### Checkbox Graustufen 
Wenn Sie diese Checkbox aktivieren, werden die Werte im Diagramm ohne Farben, nur in verschiedenen Grautönen, angezeigt. 

### Checkbox Kantenglättung 
Wenn Sie eine Kantenglättung wünschen, aktivieren Sie diese Checkbox. 

### Feld Transparenz 
Tragen Sie in dieses Feld einen Wert für die Transparenz ein.
