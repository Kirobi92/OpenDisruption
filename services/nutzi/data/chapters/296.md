# Lagerorte Registerkarte Endkontrolle

Allgemeine Funktionalitäten in eNVenta ERP / Lagerverwaltung / Lagerorte 

# Lagerorte Registerkarte Endkontrolle 
Lagerorte Registerkarte Endkontrolle 

Auf dieser Registerkarte können Sie eine Fehlerart für einen bestimmten Zeitraum von der Endkontrolle deaktivieren. Dazu wählen Sie aus der Combobox FehlerartNr eine Fehlerart aus und geben im Feld Deaktiviert bis ein Datum ein. Diese Registerkarte steht für das 1-stufige LVS zur Verfügung. 
### Feld Deaktiviert bis 
Hier geben Sie das Datum ein, bis zu dem der Fehler deaktiviert wird. 

Tooltip: 

Bestimmt das Datum, bis zu dem der Fehler deaktiviert wird. 

### Combobox Fehlerart 
Hier wählen Sie die Fehlerart aus, welche deaktiviert wird. 

Tooltip: 

Wählen Sie die Fehlerart, die deaktiviert wird. 

### Feld Memo 
Hier geben Sie ein internes Memo ein, wenn Sie die Endkontrolle im LVS deaktivieren. 

Tooltip: 

Interne Information, wenn Sie die Endkontrolle im LVS-Lager deaktivieren.
