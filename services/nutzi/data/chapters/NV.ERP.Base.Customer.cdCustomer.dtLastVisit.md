# Kapitel NV.ERP.Base.Customer.cdCustomer.dtLastVisit

Hier wird das Datum des letzten Besuchs beim Kunden eingetragen. Das Besuchsdatum wird aus dem letzten Besuchsbericht übernommen. Dieses Feld kann auch manuell gepflegt werden. 

Nur im Zusammenhang mit dem Modul Vertrieb zu pflegen.
