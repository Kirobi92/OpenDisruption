# Kapitel NV.ERP.Base.Common.cdParameterFI.shtAlterationRestriction

Wenn Sie diesen Parameter aktivieren, können Daten, die aus der Warenwirtschaft kommen, nicht annulliert, storniert oder geändert werden. 

Tooltip: 

Buchungen aus der Faktura- und Einkaufübernahme nicht ändern/annullieren.
