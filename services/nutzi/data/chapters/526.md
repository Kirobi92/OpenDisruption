# Import Straßen 
Import Straßen 

Dieser Dialog öffnet sich über den Button Import im Dialog Postleitzahlen . 

Hier geben Sie den Pfad an, wo die Datei (ASCII-Datei) liegt, die Sie von der Deutschen Post erhalten haben und importieren müssen. 

In der Combobox Mapping geben Sie das zu verwendende Mapping an, um die Daten aus der ASCII-Datei nach eNVenta zu importieren. Über den Button neben der Combobox gelangen Sie in den Mapping-Dialog, um das mitgelieferte Mapping zu importieren. Das Mapping wird als Ressourcen-Datei mit dem eNVenta -Package ausgeliefert. 

Über das Encoding steuern Sie das Einlesen der ASCII-Datei. Hierbei wird empfohlen den Standard-Eintrag „CR437“ beizubehalten. 

Zum Starten des Imports der StreetCode-Daten klicken Sie auf den Button Import . 

Die Auslieferung der Daten ist nicht Bestandteil von eNVenta . Sie müssen die Daten selbst bei der Deutschen Post lizenzieren.
