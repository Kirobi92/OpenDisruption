# Kapitel NV.ERP.FI.FinancialAcc.OpenItem.cdFlatRateValueAdjustmentParam.decStatus1Percent

Pauschalwertbertichtigung: Das Feld größer wird automatisch mit dem letzten Eintrag der Anzahl Tage gefüllt. Über das Feld Prozent haben Sie die Möglichkeit, einen weiteren Berechnungssatz für die Pauschalwertberichtigung zu hinterlegen. 

Tooltip: 

Geben Sie einen Prozentwert für die Bewertung auf Kundenstatus ein.
