# Html Report Designer Registerkarte Allgemein 
Html Report Designer Registerkarte Allgemein 

Diese Registerkarte steuert das allgemeine Aussehen der Reports über die zugeordneten Vorlagen. 
### Combobox Reportvorlage 
Hier wählen Sie die benötigte Vorlage aus. Je nach gewählter Vorlage stehen die weiteren Eigenschaften der Reports zur Verfügung. So kann die Eigenschaft „Zeile“ im Report nur für entsprechende Listen-Reports verwendet werden. 

Tooltip: 

Report Vorlage für Kopf und Fuss 

### Button Öffnen 

Mit diesem Button öffnen Sie den Dialog Report Vorlage . Dort kann auch eine neue Report Vorlage angelegt werden. 

Siehe Kapitel Report Vorlage . 

### Feld Maximale Zeilen 
In diesem Feld definieren Sie, wie viele Zeilen ein Report beinhalten soll. Gibt es mehr Zeilen, z.B. Artikeldatensätze, so werden nur die ersten Zeilen entsprechend auf dem Report angezeigt. 

### Checkbox Standard 
Wenn Sie diese Checkbox aktivieren, wird dieser Report beim Öffnen des Html Reports zum Datenobjekt angezeigt. 

Siehe Kapitel Html Report Auswahl . 

Tooltip: 

Bei mehreren passenden Reports wird dieser bevorzugt. 

### Checkbox Veröffentlichen 
Wenn Sie diese Checkbox aktivieren, steht der Report den Benutzern zur Verfügung. Auf diese Weise können Sie einen Report in Ruhe definieren und nach erfolgreicher Erstellung und Prüfung des Reports kann dieser veröffentlicht werden. 

Tooltip: 

Der Report wird dem Anwender zur Auswahl angeboten.
