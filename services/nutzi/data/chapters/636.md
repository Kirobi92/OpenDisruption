# Html Editor Registerkarte Quelltext 
Html Editor Registerkarte Quelltext 

Diese Registerkarte zeigt den Html-Code an, dieser kann bearbeitet werden. Bearbeiten Sie den Html-Code nur, wenn Sie ganz sicher sind, es finden keine Prüfungen der Syntax statt. 

Bitte schreiben Sie hier nur Html, das alternativ auch mit dem Editor (d.h. mit den Buttons, Comboboxen etc. der Registerkarten Format und Einfügen ) erzeugt werden könnte. 

Beim Speichern des Textes wird der Quelltext in der Datenbank gespeichert, damit dieser Text von allen Stellen innerhalb von eNVenta ERP gleich interpretiert wird.
