# Gutschriftsabrechnung Registerkarte Kopfdaten 
Gutschriftsabrechnung Registerkarte Kopfdaten 

Auf der Registerkarte Kopfdaten werden Informationen des Kunden angezeigt. Das Mwst.-Kennzeichen und die Ust-IdentNr. werden hierbei automatisch eingetragen. 

Es kann ein abweichendes Buchungsdatum angegeben werden. Sollte keines angegeben werden, so wird das Tagesdatum der Faktura als Buchungsdatum übernommen. 

Sollte ein Gegenauftrag bei der Faktura der Gutschrift erstellt worden sein, so wird die Auftragsnummer angezeigt und der Gegen-Auftrag kann entsprechend geöffnet werden. 
### Feld MwSt.-Kz 
Hier wird das Mehrwertsteuerkennzeichen des Kunden angegeben. 

Tooltip: 

Mehrwertsteuer-Kennzeichen 

### Feld USt-IdNr. 
Hier wird die Umsatzsteuer-Identifikationsnummer des Kunden angezeigt. 

Tooltip: 

Umsatzsteuer-Identifikationsnummer 

Ist die Funktion "System->Parameter->USt-IdNr. via WS prüfen" aktiviert, so 
wird die Umsatzsteuer Identifikationsnummer bei der Eingabe in den Dialogen 
Kunde, Lieferant, Angebot, Auftrag, Gutschrift, Bestellung, etc. per 
WebService geprüft. 

Achtung: Diese Prüfung ist rechtlich nicht bindend. Eine rechtlich bindende 
Prüfung der Umsatzsteuer Identifikationsnummer kann nur schriftlich bei der 
zuständigen Behörde erfolgen. 

### Feld Belegnr. G-Auftr. 
Hier wird die Belegnummer des Gegenauftrags angezeigt. 

Tooltip: 

Auftragsnummer des Gegenauftrages 

### Button Öffnen 

Sollte ein Gegenauftrag bei der Faktura der Gutschrift erstellt worden sein, so wird die Auftragsnummer im Feld Belegnr. angezeigt und der Gegen-Auftrag kann über diesen Button geöffnet werden. 

### Feld Buchungsdatum 
Hier kann das Buchungsdatum angegeben werden. Sollte keines angegeben werden, so wird das Tagesdatum der Faktura als Buchungsdatum übernommen. 

Tooltip: 

Angabe des Buchungsdatums 

### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen.
