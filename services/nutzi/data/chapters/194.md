# Diagramm Einstellungen Registerkarte Y-Achse 
Registerkarte Y-Achse 

Hier nehmen Sie die Einstellungen für die Y-Achse vor. 
### Checkbox Sichtbar 
Je nach Registerkarte wird die entsprechende Achse mit diesem Parameter sichtbar geschaltet. 

### Combobox Linienart 
Diese Combobox bietet verschiedene Möglichkeiten, die Linienart festzulegen. 

### Feld Linienstärke 
Hier geben Sie eine Ziffer ein für die Linienstärke. 

### Combobox Markierungstyp 
Als Markierungstyp wählen Sie entweder Prozent oder Datenintervall aus. 

### Feld Prozent 
Wenn Sie als Markierungstyp Prozent ausgewählt haben, geben Sie einen Wert ein. 

### Feld Intervall 
Wenn Sie als Markierungstyp Intervall ausgewählt haben, geben Sie einen Wert ein.
