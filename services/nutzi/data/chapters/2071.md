# CRM Übersicht Registerkarte Offene Posten 
CRM Übersicht Registerkarte Offene Posten 

Auf dieser Registerkarte werden je nach Selektion die entsprechenden offenen Posten angezeigt. 
### Checkbox Drucken 
Wenn Sie diese Checkbox aktivieren, werden die Informationen gedruckt. 

### Feld Offene Aufträge 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Feld (davon freigeg.) 
Hier werden freigegebene Auftragspositionen angegeben. 

Tooltip: 

Freigegebene Auftragspositionen 

### Feld Gesamtobligo 
Hier wird der Gesamt-Obligo angegeben (gesamte offene Posten + gelieferte, aber noch nicht fakturierte Lieferscheine (Lieferscheindruck ist erfolgt)). 

Tooltip: 

Angabe der gesamten offenen Posten + gelieferte, aber noch nicht fakturierte Lieferscheine (Lieferscheindruck ist erfolgt). 

### Feld OP-Übernahme 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Feld Summe OP 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Feld (davon fällig) 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Feld Darl. + Wechsel 
Hier wird der Betrag angegeben. 

Tooltip: 

Summe aus Darlehen und Wechsel 

### Feld Gesamt 
Hier wird der Gesamtwert angegeben: 

Kundenwerte: offene Aufträge + Summe OP + OP-Übernahme + Darlehen + Wechsel 
Projektwerte: offene Aufträge + Summe OP + OP-Übernahme 

Tooltip: 

Kundenwerte: offene Aufträge + Summe OP + OP-Übernahme + Darlehen + Wechsel 
Projektwerte: offene Aufträge + Summe OP + OP-Übernahme 

### Feld Vers. Betrag 
Hier wird der Betrag aus der Kreditversicherung angegeben. 

Tooltip: 

Angabe des Betrags der Versicherung 

### Feld Vers. noch frei 
Hier wird die Differenz zwischen Versicherungsbetrag und Summe OP + OP-Übernahme + freig. A-Positionen angegeben. 

Tooltip: 

Differenz zwischen Vers.-Betrag und Summe OP + OP-Übernahme + freig. A-Positionen 

### Feld Off. Auftr. KL rel. 
Hier werden offene Aufträge angegeben, die Kreditlimit-relevant sind. 

Tooltip: 

Offene Aufträge, KL relevant 
(die offenen Auftrags-Anteile, die durch ein belegspezif. KL 
vollumfänglich abgedeckt sind, werden intern auf 0 gesetzt) 

### Feld (KL. gesperrt) 
Hier wird das belegspezifische Kreditlimit angezeigt, das gesperrt ist. 

Tooltip: 

Angabe des belegspezifischen Kreditlimits, gesperrt 

### Feld Std. KL Kunde 
Hier kann ein Standard-Kreditlimit für den Kunden (in System-Währung) festgelegt werden. Falls der Betrag überschritten wird, erhalten Sie eine Meldung, wenn Sie einen Auftrag zu diesem Kunden erfassen. Sie erhalten die Möglichkeit, eine Begründung für die Überschreitung einzugeben. 

Die absoluten Kopfrabatte werden in der Kreditlimitprüfung nicht berücksichtigt. 

Das Kreditlimit ist immer in "Haus-Währung" anzugeben! 

Tooltip: 

Standardkreditlimit des Kunden 

### Feld (Liefersperre) 
Hier wird der Betrag angegeben. 

Tooltip: 

Summe Aufträge mit Liefersperre 

### Feld Sonder KL Kunde 
Hier kann ein Standard-Kreditlimit für den Kunden (in System-Währung) festgelegt werden. Falls der Betrag überschritten wird, erhalten Sie eine Meldung, wenn Sie einen Auftrag zu diesem Kunden erfassen. Sie erhalten die Möglichkeit, eine Begründung für die Überschreitung einzugeben. 

Die absoluten Kopfrabatte werden in der Kreditlimitprüfung nicht berücksichtigt. 

Das Kreditlimit ist immer in "Haus-Währung" anzugeben! 

Tooltip: 

Angabe des gewährten Kreditlimits 

### Feld KL-Projekt 
Hier kann ein Standard-Kreditlimit für das Projekt (in System-Währung) festgelegt werden. 

Tooltip: 

Angabe des Projekt-Kreditlimits 

### Feld KL belegspez. 
Hier wird das belegspezifische Kreditlimit angegeben. 

Tooltip: 

Kreditlimit belegspezifisch 

### Feld KL Summe 
Hier wird die Summe des Kreditlimits angezeigt. Es setzt sich wie folgt zusammen: 

Kundenwerte: Standard-KL Kunde + Sonder-KL Kunde 

Projektwerte: Kreditlimit aus Projekt 

Tooltip: 

Kundenwerte: Std. KL Kunde + Sonder KL Kunde + KL belegspez. 
Projektwerte: KL Projekt 

### Feld - KL wirksam 
Hier wird der Kreditlimit-wirksame Betrag angegeben: Summe OP + OP-Übernahme + offene Aufträge, KL relevant. 

Tooltip: 

Kreditlimitwirksam: Summe OP + OP-Übernahme + offene Aufträge, KL relevant - KL belegspez (fakturiert) 

### Feld Betrag 
Hier wird der Betrag angegeben. 

Tooltip: 

KL Summe - KL wirksam 

### Feld Letzte Zahlung

Tooltip: 

Anzahl der verbuchten Mahnungen

Tooltip: 

Angabe der aktuellen Mahnstufe 

### Feld Ø Zahlungsdauer

Durchschnittliche Zahlungsdauer 

### Feld Status

Tooltip: 

Angabe des Kundenstatus 

### Feld Sperrgrund

Tooltip: 

Sperrgrund zum Kundenstatus
