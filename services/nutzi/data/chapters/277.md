# Chargenrecherche Registerkarte Verwendung 
Chargenrecherche Registerkarte Verwendung 

Diese Registerkarte gibt Ihnen Auskunft über die Verwendung der Chargen im Bereich Verkauf . Nachdem Sie die Felder der Registerkarte gefüllt haben (dadurch lässt sich die Suche noch etwas einschränken), werden über den Button die Daten in die Tabellen eingespielt. 
### Feld Kunde 
Hier wird die Kundennummer zur Selektion angegeben. 

Tooltip: 

Angabe der Kundennummer 

### Button Suche 

Dieser Button öffnet den Dialog Kundenauswahl zur Suche eines Kunden. 

Siehe Kapitel Kundenauswahl . 

### Feld Firma 1 
Hier wird die 1. Adresszeile der Firmenadresse angezeigt. 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Feld Beleg 
Geben Sie eine Belegnummer für die Suche ein. 

Tooltip: 

Angabe der Belegnummer 

### Feld Faktura von 
Geben Sie ein Fakturadatum für die Selektion ein. 

Tooltip: 

Angabe des Fakturadatums 

### Feld Faktura bis 
Geben Sie ein Fakturadatum für die Selektion über einen Datumsbereich ein. 

Tooltip: 

Angabe des Ende-Datums

### Tabellenspalte Auftragsart 
Wenn dieses Feld gefüllt ist, wird in der Auftragserfassung die hier hinterlegte Auftragsart voreingestellt. Sie kann jederzeit geändert werden. 

Dieses Feld steht in direktem Zusammenhang mit dem Parameter Auftragsart fix im Dialog Auftragsarten .
