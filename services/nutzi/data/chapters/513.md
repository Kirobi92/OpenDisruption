# Gesellschafter 
Gesellschafter 

Dieser Dialog öffnet sich im Firmenstamm über den Button Gesellschaft . 

Hier können Sie die Gesellschafter und deren Daten anlegen. 

Folgende Daten des Gesellschafters sind zu definieren: 

- Name des Gesellschafters - 13-stellige Steuernummer des Gesellschafters - Steuer ID - Steuer W-ID - Gewinnverteilungsschlüssel Gesellschafter - Rechtsform des Gesellschafters - Beteiligungsschlüssel des Gesellschafters 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Gesellschafter 
Tooltip: 

In diesem Dialog können Sie die Gesellschafter und deren Daten anlegen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Tabelle 

### Tabellenspalte Name 
Hier wird der Name des Gesellschafters eingetragen. 

Tooltip: 

Name des Gesellschafters 

### Tabellenspalte Gesellschafterschlüssel 
Hier definieren Sie einen sogenannten Gesellschafterschlüssel (ein unternehmensbezogenes, betriebsinternes Zuordnungsmerkmal). 

Tooltip: 

Gesellschafterschlüssel, unternehmensbezogenes / betriebsinternes Zuordnungsmerkmal. 

### Tabellenspalte Gesellschaftergruppe 
Hier wählen Sie die Gesellschaftergruppe aus. Zur Verfügung stehen Voll- und Teilhafter. 

Vollhafter: Unbeschränkte persönliche Haftung mit dem gesamten Vermögen. 

Teilhafter: Die Haftung ist auf den Betrag einer bestimmten Vermögenseinlage beschränkt. 

Tooltip: 

Gesellschaftergruppe (Kapitalkontenentwicklung für Personenhandelsgesellschaften und andere Mitunternehmerschaften, Vollhafter). 

### Tabellenspalte Nummer des Beteiligten 
Hier geben Sie die Nummer des Beteiligten aus der Festellungserklärung ein. 

Der Begriff Feststellungserklärung bezeichnet die Erklärung zur gesonderten Feststellung von Grundlagen für die Einkommensbesteuerung. 

Tooltip: 

Nummer des Beteiligten aus Feststellungserklärung (Vordruck FB) 

### Tabellenspalte Zeichnernummer 
Falls vorhanden, geben Sie hier die Zeichnernummer aus der Feststellungserklärung an. 

Tooltip: 

Zeichnernummer (soweit vorhanden) aus Feststellungserklärung (Vordruck FB) 

### Tabellenspalte Steuernummer 
Hier geben Sie die 13-stellige Steuernummer des Gesellschafters an. 

Tooltip: 

13stellige Steuernummer des Gesellschafters 

### Tabellenspalte Steuernummer 
Hier wird die persönliche Steuernummer angegeben. 

### Tabellenspalte Unterjähriges Änderungsdatum 
Hier können Sie ein unterjähriges Änderungsdatum für den Gewinnverteilungsschlüssel der Gesellschafter auswählen. 

Tooltip: 

Unterjähriges Änderungsdatum, Gewinnverteilungsschlüssel Gesellschafter 

### Tabellenspalte Früherer Verteilungsschlüssel 
Falls erforderlich, können Sie hier den früheren Verteilungsschlüssel zur Gewinnverteilung eintragen. 

Tooltip: 

Früherer Verteilungsschlüssel, Gewinnverteilungsschlüssel Gesellschafter 

### Tabellenspalte Rechtsform 
Hier wählen Sie die Rechtsform des Gesellschafters aus (z.B. Personengesellschaft, natürliche Person - Privatvermögen). 

Tooltip: 

Rechtsform des Gesellschafters 

### Tabellenspalte Natürliche Personen GmbH 
Wenn der Gesellschafter eine "natürliche Person" einer GmbH ist, aktivieren Sie die Checkbox in dieser Spalte. 

Tooltip: 

Natürliche Personen als Gesellschafter einer GmbH 

### Tabellenspalte Sonderbilanz benötigt 
Wenn diese Checkbox aktiviert ist, wird eine Sonderbilanz benötigt. 

Tooltip: 

Sonderbilanz benötigt 

### Tabellenspalte Ergänzungsbilanz benötigt 
Wenn diese Checkbox aktiviert ist, wird eine Ergänzungsbilanz benötigt. 

Tooltip: 

Eine Ergänzungsbilanz wird benötigt.
