# Import Item Auswahl 
Import Item Auswahl 

Dieser Dialog öffnet sich mit dem Button im Dialog eNVenta Init auf der Registerkarte Import . 

Hier können Sie ein oder mehrere Import-Schemata eines bestimmten Verzeichnisses auswählen und in den Dialog eNVenta ERP Init übernehmen. 

Das Verzeichnis befindet sich auf dem Broker. Wenn Sie hierauf keinen Zugriff haben, wenden Sie sich bitte an Ihren Administrator. 

Der Dialog prüft alle im Verzeichnis vorhandenen XML-Dateien auf ihr Schema und auf die referenzierte CSV-Datei. Ist das Schema korrekt und existiert die CSV-Datei, listet der Dialog den Import Item in der Tabelle auf. 

Auf der rechten Seite des Dialogs werden alle evtl. vorhandenen Probleme gemeldet, z.B. wenn Sie Daten einer Tabelle importieren möchten, aber in der Zieltabelle bereits Einträge vorhanden sind. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Import Item Auswahl 
Tooltip: 

In diesem Dialog können Sie ein oder mehrere Import-Schemata eines bestimmten Verzeichnisses auswählen und in den Dialog 'eNVenta ERP Init' übernehmen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter .
