# Gültigkeitsänderung 
Gültigkeitsänderung 

Dieser Dialog öffnet sich im Dialog Preisgruppen Details über den Button Gültigk. Änd . Hier kann das Datum für die Gültigkeiten, die in der Zukunft liegen, geändert werden. Dies ist für mehrere Artikel einer Preisgruppe möglich. Dazu geben Sie das alte Datum von/bis ein; nur diese Artikel werden dann mit einem neuen Datum versehen. 

Aktuell gültige oder in der Vergangenheit liegende Datumsangaben werden nicht geändert! Ist ein Preis aktuell gültig, kann nur das Bis-Datum geändert werden. Sind Preise noch nicht gültig, können das Von- und das Bis-Datum geändert werden. 

Ist im Dialog Preisgruppen Details ein Lieferant gewählt worden, wird dieser hier voreingestellt. 

Das Von- und Bis-Datum im Bereich Ist ist mit den Informationen aus den Preisgruppen Details voreingestellt.
