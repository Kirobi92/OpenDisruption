# Leistungsabrechnung 
Leistungsabrechnung 

Dieser Dialog öffnet sich im Projekt >> Optionen >> Leistungsabrechnung . 

Über diesen Dialog können erfasste Projektleistungen eingelesen werden, um daraus einen Auftrag zu generieren. Der Dialog besitzt Selektionsmöglichkeiten für die Kundennummer, die Projektnummer, die Projektphase, die Projektaufgabe, Start- und Enddatum (der Leistung) sowie für den Status der Leistung und der Projektaufgabe. 

Des Weiteren kann über Checkboxen auf die Leistungen eingegrenzt werden, die ausgewiesen werden sollen bzw. die fakturierbar sind. Beim Initialisieren des Dialogs sind diese Checkboxen aktiv. Der Status der Leistung ist auf Genehmigt vorbelegt, der Status der Aufgabe auf Beendet . 

Wenn der Dialog geöffnet wird, werden standardmäßig alle Selektionsfelder gefüllt, soweit dies möglich ist. Wenn z.B. eine Aufgabe selektiert wurde und dann die Leistungsabrechnung geöffnet wird, wird automatisch auf diese Aufgabe eingegrenzt. Dasselbe gilt für die Phase eines Projekts. Wenn dem Projekt nur genau ein Kunde zugeordnet ist, wird auch die Kundennummer vorbelegt. Beim Einlesen von Leistungen werden die zu prüfenden Leistungen gelb markiert. Leistungen, die einen Fehler bei der Auftragsgenerierung verursachen würden, werden rot eingefärbt. 

In diesem Dialog kann ebenfalls der Status einer Leistung geändert werden, wenn der Sachbearbeiter die entsprechenden Berechtigungen hat. 

Siehe Kapitel Berechtigungsgruppe . 

Wenn Sie einen Doppelklick auf eine Leistung im Grid machen, wird die zugehörige Projektaufgabe geöffnet (falls es sich um eine Leistung für einen Projekttermin handelt). Jede Leistung hat eine Checkbox Auswahl . Nur die Leistungen, die dort einen Haken gesetzt haben, werden für eine Auftragsgenerierung verwendet (sofern diese den Status Genehmigt haben und als 'auszuweisen' gekennzeichnet sind). Die Auswahl können Sie über den Button Markieren setzen oder entfernen. 

Je nachdem in welcher Spalte Sie den Cursor stehen haben, öffnet sich mit einem Doppelklick der zugrundeliegende Vorgang. Dies können der Auftrag, die Projektaufgabe, der Call etc. sein. 

Wenn Sie einen Doppelklick in die Spalte ‚Leistungsmenge‘ machen, öffnet sich die Leistungserfassung zu dieser Buchung.

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken.

Dieser Button schließt den Dialog.

Mit diesem Button können Sie für eine Projektaufgabe eine Auswahl setzen oder entfernen. 

Tooltip:

### Button Auftrag erzeugen 

Um einen Auftrag zu erzeugen, müssen Sie auf diesen Button klicken. Evtl. erscheint dann noch eine Meldung, dass zuerst eine Auftragsart gewählt werden muss, welche über eine Combobox im Dialog gewählt werden kann. 

Bei der Generierung eines Auftrags aus einem Projekt, bei dem mehrere Kunden beteiligt sind, erscheint eine Abfrage, welcher Kunde in den Auftrag eingetragen werden soll. In diesem Zusammenhang gibt es zur Selektion eines Kunden im Projekt , Registerkarte Beteiligte , die Spalte ‚Keine Abrechnung‘ in der Tabelle Kunden . 

Tooltip: 

Auftrag erzeugen
