# Merkmale 
Merkmale 

Dieser Dialog öffnet sich über den Button Merkmale im Dialog Merkmalsverwaltung oder im Artikel , Registerkarte Merkmale Button Merkmale . Jedes Merkmal hat einen eindeutigen Datentyp für die Werte und jeder Wert hat ebenfalls einen eindeutigen Datentyp. 

Hier werden Merkmale erstellt und bearbeitet. 

Es können Werte ohne Klassenzuordnung gespeichert werden. 

Jedes Merkmal hat einen eindeutigen Datentyp für die Werte und jeder Wert hat ebenfalls einen eindeutigen Datentyp. 

Ein Merkmal kann z.B. die Farbe oder das Gewicht sein. 

Im oberen Bereich der Merkmale geben Sie die Grunddaten des Merkmals an. Diese dienen der Suche nach Merkmalen, wie auch der Anzeige und dem Druck der Merkmale. 

Die Merkmalnummer können Sie manuell angeben oder wird beim Speichern des Merkmals automatisch vergeben. 

Im Feld Bezeichnung geben Sie eine möglichst sprechende Bezeichnung für das Merkmal an. Analog können Sie eine Kurzbezeichnung für das Merkmal angeben, dieses steht dann z.B. beim Export zur Verfügung. 

Die Felder Bezeichnung , Kurzbezeichnung und Beschreibung sind durch Wechsel der Sprache entsprechend in mehreren Sprachen pflegbar.

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind.

Dieser Button startet die Suche nach Daten; entweder ganz allgemein oder wenn Sie im Kopf des Dialogs entsprechende Selektionskriterien festgelegt haben, gezielt.

Dieser Button leert die Daten in den Kopf-Feldern.

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Kopieren 

Dieser Button startet die Kopierfunktion. Es werden sämtliche Felder kopiert, welche keine Schlüsselfelder sind. 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Vorheriger Satz 

Mit diesem Button können Sie rückwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Nächster Satz 

Mit diesem Button können Sie vorwärts blättern. Die Reihenfolge beim Blättern bezieht sich auf die Registerkarte Tabelle , sofern diese vorhanden ist. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. In manchen Fällen werden Sie gefragt, ob Sie die Eingaben / Änderungen noch vorher speichern möchten. 

### Button Wertestamm
