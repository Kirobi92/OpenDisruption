# Wiederholungsdruck der Bestellbelege 
Druckbelege 

Den Wiederholungsdruck können Sie über den Button Belege aktivieren. Es erscheint eine Liste der bereits zu dieser Bestellung gedruckten Formulare. Durch Markierung des gewünschten Belegs und Klick auf den Button ist ein nochmaliger Druck jederzeit möglich. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Druckbelege 
Tooltip: 

In diesem Dialog können Sie einen Wiederholungsdruck starten. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Drucken 

Dieser Button startet den Druck. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Button B löschen 

Dieser Button löscht eine Auftragsbestätigung, um anschließend den Auftrag neu zu generieren und erneut eine Bestätigung zu drucken. 

Tooltip: 

Das Bestelldokument wird gelöscht. 
Voraussetzung ist, dass noch kein Wareneingang erfolgt ist und es keine Bestellbestätigung gibt. 

### Button Ref. Journal 

Mit diesem Button öffnen Sie den Dialog Ref Link Druck Journal . Hier werden Referenz-Druckbelege angezeigt. 

Siehe Kapitel Referenz Link Druck Journal. 

### Button Druck-Vers. 

Über diesen Button öffnet sich der Dialog Druck-Versionen , falls die Beleg-Versionierung aktiviert ist. Damit erhalten Sie eine Übersicht über abgelegte Dokumente. 

Siehe Kapitel Druck-Versionen . 

Siehe Kapitel Parameter Allgemein, Registerkarte Wawi, Checkbox Beleg-Vers .

### Checkbox Briefkopf 
Wenn Sie diese Checkbox aktivieren, wird das Logo auf die Listen gedruckt. In Auftrags- und Bestellformularen wird zusätzlich noch eine kleine Absenderzeile über der Adresse gedruckt. 

Tooltip: 

Wenn aktiv, wird der Briefkopf z.B. Logo, Absender auf den Beleg gedruckt. 

### Dock Panel Modus 

Hier wählen Sie den Druckmodus aus. 
### Radiobutton E-Mail 
Über diesen Button kann ein Druckbeleg auch als Email-Anhang versendet werden. 

Wenn beim Email-Versand mehrere Referenz-Links angehängt werden, werden die Dateinamen automatisch durchnummeriert, z.B. „_1“, „_2“, wenn diese den gleichen Dateinamen besitzen. 

### Dock Panel Format
