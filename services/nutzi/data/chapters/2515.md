# Kontoblätter Registerkarte Pro Tag 
Kontoblatt Registerkarte Pro Tag 

Diese Registerkarte zeigt Einträge pro Tag. 
### Tabelle 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Tabellenspalte Buchungsdatum 
Hier wird das Buchungsdatum angegeben. 

Tooltip: 

Angabe des Buchungsdatums 

### Tabellenspalte Belegdatum 
Hier wird das Belegdatum angegeben. 

Tooltip: 

Angabe des Belegdatums 

### Tabellenspalte Betrag Soll 
Hier wird der Soll-Betrag angegeben. 

Tooltip: 

Angabe des Soll-Betrags 

### Tabellenspalte Betrag Haben 
Hier wird der Haben-Betrag angegeben. 

Tooltip: 

Angabe des Haben-Betrags 

### Tabellenspalte Währung 
Hier wird das Währungskürzel angegeben. 

Tooltip: 

Währungskürzel 

### Tabellenspalte Betrag Soll FW 
Hier wird der Soll-Betrag in Fremdwährung angegeben. 

Tooltip: 

Angabe des Fremdwährungsbetrags im Soll 

### Tabellenspalte Betrag Haben FW 
Hier wird der Haben-Betrag in Fremdwährung angegeben. 

Tooltip: 

Angabe des Fremdwährungsbetrags im Haben 

### Tabellenspalte Jahr 
Hier wird das (Geschäfts-) Jahr angegeben. 

### Tabellenspalte Periode 
Hier wählen Sie einen Zeitraum aus. 

### Tabellenspalte Niederlassung 
Hier wird die Niederlassungsnummer angegeben. 

Tooltip: 

Angabe der Niederlassung
