# Html Report Designer Registerkarte Zeile 
Html Report Designer Registerkarte Zeile 

Auf dieser Registerkarte werden die Eigenschaften der Reports immer für eine Tabellenansicht im Html Report definiert. Die Breite der Spalten richtet sich hierbei immer nach dem Inhalt der jeweiligen Spalte und der Gesamtanzahl der Spalten. Die Einstellungen zu Label, Format und Ausrichtung ist analog der Funktionen in der Registerkarte Kopf . 

Über die Spalte ‚Total‘ kann für das Datenfeld ein Gesamtwert gebildet werden. Es sind die Funktionen 
"SUM" - Summierung der Datenfelder 
"AVG" - Durchschnitt der Werte 
"COUNT" - Anzahl der Zeilen 

möglich. Die Funktionen können nur für numerische Werte gesetzt werden. 

Die Registerkarte Sortierung erlaubt es, die Sortierreihenfolge der Tabellenzeilen zu bestimmen. Dabei können auch Datenfelder (Properties) aus untergeordneten Objekten verwendet werden. 

Ein Sortierdatenfeld kann auch als "Gruppe" markiert werden. Die Zeilen der Tabelle werden nach diesem Datenfeld gruppiert. Das Datenfeld wird als Gruppenüberschrift angezeigt. Sind Spalten mit Total-Werten vorgesehen, so werden automatisch auch Gruppentotale gebildet und angezeigt. 

Hinweis: es ist nur ein Gruppenfeld möglich. 

Hinweis: die Sortierung findet auf Brokerebene statt, nicht auf der Datenbankebene. 

In Kombination mit der Maximalanzahl Zeilen (Registerkarte Allgemein ) können so sehr gezielt eingeschränkte Dateninhalte dargestellt werden.
