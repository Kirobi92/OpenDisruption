# Kapitel NV.ERP.Base.Common.cdPaymentCondCustomer.shtCashOnDelivery

Ist bei einer Zahlungsbedingung Nachnahme gesetzt, werden alle Belege mit dieser Zahlungsbedingung nicht im Mahnwesen bzw. Bankeinzug berücksichtigt. 

Skonto-Fälligkeiten und Skonto-Sätze werden in den Spalten Tage 1/Skonto 1, Tage 2/Skonto 2, Tage 3/Skonto 3 eingetragen. 

Tooltip: 

Zahlungsbedingung 'Nachnahme' wird aktiviert.
