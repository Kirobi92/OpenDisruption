# Packlistendruck / Seriendruck in der Tourendisposition 
Seriendruck 

Die Dialoge Packlistendruck und Seriendruck basieren auf ein und demselben Form und werden daher gemeinsam in diesem Kapitel beschrieben. 

Im Menü Packlistendruck können Packlisten gedruckt werden. Sie können hier u.a. das Format wählen und die Anzahl der Kopien. 

Der Button Druck Bel. in der Tourendisposition erlaubt den Massendruck von Lieferscheinen zu einer Tour. Es öffnet sich der Dialog Seriendruck . 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Seriendruck 
Tooltip: 

In diesem Dialog können Packlisten bzw. Lieferscheine per Massendruck gedruckt werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Checkbox Briefkopf 
Diese Checkbox wird aus dem Dialog Parameter Allgemein (Registerkarte Allgemein ) vorbelegt und kann hier im Einzelfall deaktiviert werden. 

Erläuterung: 

Im Rahmen von Geschäftsprozessen ist es immer wieder notwendig, Belege physisch oder digital als Ausdrucke bereitzustellen. Aus rechtlichen Gründen muss dies bei offiziellen Dokumenten mit bestimmten Informationen (Anschrift, BLZ, …) – in Kopf- und Fußzeile - geschehen. Diese Informationen werden im Folgenden als Briefkopf bezeichnet. 

Dies können Kunden nutzen, die ihre Firmeninformationen direkt mit dem Druck des Belegs auf ein Blanko-Papier drucken möchten. 

Die Checkbox Briefkopf dient in den Dialogen als Vorbelegung und im automatisierten Druck als Einstellung für den Druck. 

Im Einzelfall kann in den Druck-Dialogen der einzelnen Vorgänge die Checkbox Briefkopf deaktiviert werden, falls Sie ausnahmsweise auf ein Briefpapier drucken möchten. 

Tooltip: 

Wenn aktiv, wird der Briefkopf z.B. Logo, Absender auf den Beleg gedruckt.
