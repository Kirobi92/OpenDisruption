# Übersicht Betriebsaufträge 
Übersicht Betriebsaufträge 

Das Menü Übersicht Betriebsaufträge bietet die Möglichkeit nach Betriebsaufträgen zu suchen. Die Felder im Kopf des Dialogs können zur Selektion genutzt werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Übersicht Betriebsaufträge 
Tooltip: 

In diesem Dialog können Sie Betriebsaufträge suchen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.

### Button Drucken 

Mit diesem Button können Sie entweder eine "Betriebsauftrag Übersicht" oder eine Materialbereitstellungsliste drucken. 

Da für manche Betriebsaufträge eine große Anzahl an Materialien benötigt wird, diese aber nicht immer gleichzeitig zur Verfügung stehen, können Sie über das Drucken der Materialbereitstellungsliste immer sofort erkennen, welche Materialien noch fehlen. D.h. für den selektierten Betriebsauftrag werden die Materialbereitstellungsscheine gedruckt. 

Siehe Kapitel Druck der Formulare und Listen .
