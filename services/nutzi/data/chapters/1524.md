# Fahrzeugstamm Registerkarte Allgemein 
Fahrzeugstamm Registerkarte Allgemein 

Auf dieser Registerkarte geben Sie die Details zum Fahrzeug ein. 
### Feld Paletten max. 
Hier geben Sie die maximale Anzahl an Paletten ein, die geladen werden dürfen. 

Tooltip: 

Maximale Anzahl Paletten 

### Feld Kosten je Km 
Hier geben Sie ein, was das Fahrzeug je Kilometer kostet. 

Tooltip: 

Kosten des Fahrzeugs pro Km. 

### Feld Kosten je Std. 
Hier geben Sie die Kosten des Fahrzeugs pro Betriebsstunde ein. 

Tooltip: 

Kosten je Betriebsstunde 

### Feld Beschreibung 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Feld Ladegewicht 
Hier geben Sie für das Fahrzeug das zulässige Ladegewicht ein. 

Für das Ladegewicht hat das Programm die Gewichtseinheit kg vorgesehen. 

Tooltip: 

Ladegewicht in kg 

### Feld Volumen 
Hier geben Sie für das Fahrzeug das zulässige Ladevolumen ein. 

Für das Ladevolumen hat das Programm die Volumeneinheit m³ vorgesehen. 

Tooltip: 

Volumen in m³ 

### Feld Länge (max.) 
Hier können Sie die maximale Länge in Meter eintragen (interessant für die Toureninformation). 

Tooltip: 

Max. Länge in m. 

### Feld Breite max. 
Hier können Sie die maximale Breite in Meter eintragen (interessant für die Toureninformation). 

Tooltip: 

Max. Breite in m.
