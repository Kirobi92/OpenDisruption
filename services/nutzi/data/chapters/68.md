# Fokus-Element Organisation 

## Ordner 

### Inhalt 
Mit Hilfe dieses Elements können individuelle organisatorische Hilfsmittel in Form von Ordnern und Ablagen für Vorgänge und Objekte geschaffen werden. Objekte und Vorgänge werden einfach per Drag&Drop auf den Order gezogen und auch per Drag&Drop weiterverwendet. 
### Ziel / Nutzen 

Es lassen sich beispielsweise zu bearbeitende Vorgänge und Stammdaten vorübergehend in einen dazu vorgesehenen Ordner ablegen. Nach Bearbeitung werden diese wieder gelöscht. Soll eine Ablage nur temporär genutzt werden, um sich einen Vorgang oder ein Objekt zu merken, kann auf eine Speicherung der Daten verzichtet werden (siehe Parameter).  Eingetragene Objekte können z.B. auch als Selektionskriterien auf einen Dialog gezogen werden. 
### Darstellung 
Titelliste der enthaltenen Objekte/Vorgänge 
### Parameter 
Über den Titel des Ordners kann der organisatorische Zweck festgelegt werden. 
Standardmäßig werden die Einträge gespeichert. Für eine sehr temporäre Nutzung kann der Parameter "Nicht speichern" gesetzt werden. 
### Drill Down 
Aufruf des Vorgangs/Dokuments 
### Aktualisierung 
Manuell per Drag&Drop. Löschen von Elementen mit Button "X" möglich. 
### Berechtigung 
Zugangsberechtigt sind alle Benutzer, die auch das Recht für den Dialog Fokus Ordner/Ablage (frmDashItemObjectFolder) besitzen.
