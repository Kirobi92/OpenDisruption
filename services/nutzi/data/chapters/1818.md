# Rahmenbestellungen Registerkarte Positionen 
Rahmenbestellung Registerkarte Positionen 

Auf dieser Registerkarte werden die Artikelpositionen für die Rahmenbestellung eingetragen. 
### Button Neu 

Mit diesem Button fügen Sie eine neue Zeile in die Tabelle ein. 

### Button Löschen 

Wenn Sie diesen Button drücken, wird die markierte Zeile mit dem -Zeichen gekennzeichnet. 

### Button Preise 

Dieser Button öffnet den Dialog EK Konditionen . Hier erhalten Sie eine Übersicht über Konditionen zum gewählten Artikel. 

Siehe Kapitel EK Konditionen in der Disposition . 

Tooltip: 

Mit diesem Button erhalten Sie eine Übersicht 
über Konditionen zum gewählten Artikel. 

### Button Bestand 

Dieser Button öffnet den Dialog Bestand und bietet einen Überblick über den Bestand des Artikels. 

Siehe Kapitel Bestandsanzeige . 

Tooltip: 

Über diesen Button erhalten Sie einen Überblick über den Bestand des Artikels. 

### Button Abrufe 

Dieser Button öffnet den Dialog Rahmenabrufe zum Abrufen von Rahmenbestellungen. 

Siehe Kapitel Rahmenabrufe in der Rahmenbestellung . 

Tooltip: 

Mit diesem Button können Sie Rahmenbestellungen abrufen. 

### Radiobutton Artikel 
Wenn Sie diesen Radiobutton aktivieren, wird die Rahmenbestellung auf Artikelebene angelegt. 

### Radiobutton Artikelgruppe 
Wenn Sie diesen Radiobutton aktivieren, wird die Rahmenbestellung auf Artikelgruppenebene angelegt. 

### Feld Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind! 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Tabelle 

### Tabellenspalte PosArt 
Hier wird das Kennzeichen A für Artikel oder W für Artikelgruppe bei der Neuanlage einer R-Position eingetragen, je nachdem welchen Radiobutton Sie aktiviert haben. 

Tooltip: 

Kennzeichen für Artikel oder Artikelgruppe wird angezeigt. 

### Tabellenspalte Status 
Status der Position. Alle Rahmenbestellpositionen erhalten bei der Erfassung den Status 1 (offen). In dem Moment, in dem alle Bestellpositionen geliefert sind, wird der Status automatisch auf 2 (beendet) gesetzt. 

Tooltip: 

Status der Position: 

1 = offen 
2 = beendet (alle Bestellpositionen sind geliefert) 

### Tabellenspalte Artikelgruppe 
Hier tragen Sie die Artikelgruppe ein, für die der Rahmen erfasst wird. 

Tooltip: 

Angabe der Artikelgruppe 

### Tabellenspalte Artikel 
Hier tragen Sie die Artikelnummer ein, für die der Rahmen erfasst wird. Diese kann bei einer Warengruppenposition allerdings nicht eingegeben werden. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung wird aus dem Artikelstamm oder über die Kundenartikelnummer nachgelesen und kann manuell geändert werden. 

Dies gilt nicht für Warengruppenpositionen. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Gesamtmenge 
Hier wird die vertraglich vereinbarte Gesamtmenge der Rahmenbestellposition eingetragen. 

Tooltip: 

Die gesamte Menge der Position wird eingetragen. 

### Tabellenspalte Abgerufen 
Hier wird die Summe der abgerufenen Mengen aus Bestellungen angezeigt. Offene Abrufe sind hier nicht enthalten. 

Tooltip: 

Die bereits abgerufene Menge der Position wird eingetragen. 

### Tabellenspalte Global gesamt 
Die Spalte zeigt die Gesamtmenge der Abrufe aus dem Global-Prozess an. 

Eine Position aus einem Rahmenauftrag kann direkt über den Globalabschluss mit einer Rahmenbestellung verbunden werden. Dies ist nur möglich für Artikelgruppenpositionen. 

Siehe Kapitel Globalabschluss und Strecke . 

Tooltip: 

Zeigt die Gesamtmenge der Abrufe aus dem Global-Prozess an. 

### Tabellenspalte Preis 
Der Preis wird aus der Lieferantenkondition gezogen und dient hier als Vorschlag. Der Preis kann entsprechend der Vereinbarung überschrieben werden. 

Tooltip: 

Preis der Rahmenposition
