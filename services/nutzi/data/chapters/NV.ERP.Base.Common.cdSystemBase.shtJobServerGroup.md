# Kapitel NV.ERP.Base.Common.cdSystemBase.shtJobServerGroup

Hier können Sie eine Job-Server Gruppe auswählen. Wenn der Benutzer oder der Job keiner Job-Server-Gruppe zugeordnet ist, so werden alle neu angelegten Jobs automatisch dieser Gruppe zugewiesen. 

Über die Job-Server-Gruppen können die verschiedenen Jobs mehreren verschiedenen Job-Servern zugewiesen werden, um die Last entsprechend zu verteilen. 

Tooltip: 

Mit Auswahl aus dieser Combobox werden alle neu angelegten Jobs automatisch dieser Gruppe zugewiesen.
