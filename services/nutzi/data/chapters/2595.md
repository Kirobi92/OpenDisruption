# Zahlungsverkehr Registerkarte Disposition 
Zahlungsvorschlag Registerkarte Disposition 

Auf der Registerkarte Disposition wählen Sie zunächst eine oder mehrere Hausbanken aus, über die der Zahlungslauf erfolgen soll. 

Die ausgewählte(n) Bank(en) erscheinen in der Tabelle auf der Registerkarte Disposition . Hier haben Sie die Möglichkeit, durch direkten Eintrag in der Spalte 'Limit', eine Höchstgrenze für den gesamt zu zahlenden Betrag pro Bank festzulegen. 

Ist für das Buchen einer Kreditoren-Rechnung keine Bankverbindung ausgewählt, wird die Bankverbindung aus dem Kundenstamm gezogen. 

Entfernen Sie eine Dispositionsbank ohne zuvor eine neue Dispositionsbank ausgewählt zu haben, wird der gesamte Vorschlag gelöscht. 

Werden über die gleiche Bank mehrere EU-Standard-Zahlungen an einem Tag ausgeführt, muss für jeden Zahlungslauf eine fortlaufende Tagesnummer vergeben werden. Im Menü Zahlvorschlag verarbeiten wird hierzu bei EU-Standardzahlungen das Datenfeld Lfd. Tagesnummer aktiv. Hier vergeben Sie manuell die entsprechende Nummer. 
### Button Neue Zeile 

Mit diesem Button öffnet sich der Dialog Bankverbindung . Hier wählen Sie die gewünschte Bank durch Doppelklick mit der Maus oder markieren und den Button .

### Button Zuordnen 

Mit dem Button Zuordnen können Sie nach der erfolgreichen Generierung eines Zahlungsvorschlags einen/mehrere offene Posten einer bestimmten Hausbank zuordnen (manuelle Zuteilung). 

Hierzu markieren Sie den/die gewünschten OPs auf der Registerkarte Vorschlag . Danach wählen Sie die neue Dispositionsbank auf der Registerkarte Disposition aus, markieren diese in der Tabelle und betätigen den Button Zuordnen . 

Tooltip: 

Selektierte Vorschlagssätze einer Disposition zuordnen. 

### Tabelle Banken 

Die Tabelle enthält die Banken entsprechend der Selektionen. 
### Tabellenspalte Bankname 
Hier wird der Bankname angegeben. 

### Tabellenspalte BLZ / SWIFT 
Hier geben Sie den Bank-Code an. Dies ist eine Pflichtangbe. Gültige Werte sind abhängig vom Eintrag im Feld Typ . Entweder 
Typ = 'BLZ', dann Bankleitzahl der Bank 
Typ = 'SWIFT', dann Swiftcode der Bank. 

Tooltip: 

Angabe der Bankleitzahl / SWIFT 

### Tabellenspalte Bankverb. Whrg. 
Hier wird das Währungskürzel der Bankverbindung angegeben. 

Tooltip: 

Währung der Bankverbindung

Hier wird der festgelegte Betrag angegeben. 

Tooltip:

Tooltip: 

Angabe des Betrags in Fremdwährung 

### Tabellenspalte Saldo

Tooltip: 

Angabe des Betrags 

### Tabellenspalte Kreditlinie 
Hier kann ein Standard-Kreditlimit für den Kunden (in System-Währung) festgelegt werden. Falls der Betrag überschritten wird, erhalten Sie eine Meldung, wenn Sie einen Auftrag zu diesem Kunden erfassen. Sie erhalten die Möglichkeit, eine Begründung für die Überschreitung einzugeben. 

Die absoluten Kopfrabatte werden in der Kreditlimitprüfung nicht berücksichtigt. 

Das Kreditlimit ist immer in "Haus-Währung" anzugeben! 

Tooltip: 

Angabe des gewährten Kreditlimits
