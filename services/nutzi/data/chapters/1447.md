# Vermietung zuweisen 
Vermietung zuweisen 

Dieser Dialog öffnet sich in der Retoure Registerkarte Allgemein über den Button Verm. zuweisen . Hier können Mietobjekte, die zurückkommen (z.B. von einer Baustelle) einem Auftrag / einer Vermietung zugeordnet werden. 

Es werden nur Mietobjekte angezeigt. Hier können auch Teilrückgaben realisiert werden. eNVenta führt dann entsprechende Positionssplitts in der Retoure durch. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Vermietung zuweisen 
Tooltip: 

In diesem Dialog können Mietobjekte, die zurückkommen (z.B. von einer Baustelle) einem Auftrag / einer Vermietung zugeordnet werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
