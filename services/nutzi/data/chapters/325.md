Allgemeine Funktionalitäten in eNVenta ERP / Datentransfer (Import und Export von Daten) 

# Verbindungsauswahl 
Verbindungsauswahl 

Dieser Dialog öffnet sich im Dialog Server Datei Auswahl über den Button neben dem Feld Datei . Hier stehen sämtliche Verbindungen zur Verfügung, die in den Datei Auswahl-Dialog übernommen werden können. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Verbindungsauswahl 
Tooltip: 

In diesem Dialog stehen sämtliche Verbindungen zur Verfügung, die in den Datei Auswahl-Dialog übernommen werden können. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Feld Verbindungsname 
Hier wird der Name der Verbindung angegeben. 

Tooltip: 

Angabe des Verbindungsnamens 

### Button Öffnen 

Mit diesem Button kann direkt der Dialog zur Verwaltung der Verbindungen aufgerufen werden ( Verbindungsmanager ). 

Der Button ist nur sichtbar, wenn der Benutzer auch die entsprechenden Speicherrechte für den Dialog hat. 

Beispiel: 
Der Sachbearbeiter befindet sich in einem Ordner, von dem er zwei Dateien gern direkt im Dialog Verbindungsauswahl hochladen möchte. Damit er nicht, wie beim Hochladen-Button, erneut zu seinen Dateien navigieren muss, markiert er beide Dateien, zieht sie auf die Tabelle und lässt sie „fallen“. Beide Dateien werden direkt hochgeladen.
