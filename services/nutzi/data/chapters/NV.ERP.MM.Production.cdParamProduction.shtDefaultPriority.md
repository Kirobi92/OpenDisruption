# Kapitel NV.ERP.MM.Production.cdParamProduction.shtDefaultPriority

Hier können Sie eine Ziffer eingeben, um die Priorität festzulegen. Diese Priorität spielt eine Rolle bei der Neuplanung von Betriebsaufträgen. (Dialog Neuplanung ) 

Ist im Betriebsauftrag keine Priorität gesetzt (Registerkarte Kopfdaten , Feld Priorität ), zieht die Priorität, die Sie hier definiert haben.
