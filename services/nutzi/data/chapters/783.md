# Zusatzpositionen Registerkarte Allgemein 
Zusatzposition Registerkarte Allgemein 

Auf dieser Registerkarte legen Sie die Einstellungen für die Zusatzposition fest. 
### Feld Faktor VK 
Dieser Faktor gibt an, um das Wievielfache der VK der Originalposition multipliziert wird, um den VK der Zusatzposition zu erhalten. 

Tooltip: 

Dieser Faktor gibt an, um das wievielfache der VK der Originalposition 
multipliziert wird, um den VK der Zusatzposition zu erhalten. 

### Feld VK fix 
Hier können Sie einen fixen Verkaufspreis eintragen. Die komplette VK-Preisbasis (Währung, VKpro, Mengeneinheit VK) wird dargestellt. 

Sind VK fix und Menge fix definiert, haben diese Werte Priorität. 

Tooltip: 

Fix vorgegebener Netto-Preis, bezogen auf den Artikelstamm-Preisbasis-VK. 

### Feld Faktor Menge 
Hier geben Sie einen Umrechnungsfaktor von Lager- zu Verkaufseinheit ein, bezogen auf die Menge. 

Tooltip: 

Dieser Faktor gibt an, um das wievielfache die Menge der Originalposition 
multipliziert wird um die Menge der Zusatzposition zu erhalten. 

### Feld Menge fix 
Hier kann eine fixe Menge eingetragen werden. 

Sind VK fix und Menge fix definiert, haben diese Werte Priorität. 

Tooltip: 

Angabe der Menge 

### Feld Mengeneinheit 
Hier wird die Mengeneinheit angezeigt. 

### Feld Pos 
In dieses Feld tragen Sie ein, an welcher Position der Zusatzartikel stehen soll. Wenn Sie mehrere Zusatzpositionen zu einem Artikel generieren, können Sie auf diese Weise die Reihenfolge beeinflussen. 

Tooltip: 

Positionsnummer 

### Combobox Gruppentyp 
Aus dieser Combobox wählen Sie einen Gruppentyp aus. Damit wird ein Zusatzartikel automatisch mit dem Hauptartikel verbunden. 

Die Typen haben folgende Bedeutung: 

- Komplett Rechnung: (nur Prüfung bei Rechnungsdruck, alle Positionen der Gruppe müssen auf die gleiche Rechnung gedruckt werden) - Komplett: (nur Packlistendruck bzw. Lieferscheindruck und Rechnung, wenn alle Positionen freigegeben sind) - Synchron : (komplett + führende Position gibt Status für alle anderen Positionen vor) - Synchron Menge: (komplett + führende Position gibt Status und Menge für alle anderen Positionen vor) 
In Ergänzung zu den Gruppentypen erhalten Sie weitere Informationen im Kapitel Gruppentypen . 

### Checkbox Automatisch 
Wenn Sie diese Checkbox aktivieren, wird die Zusatzposition beim Speichern des Auftrags automatisch, ohne Rückfrage, in den Auftrag eingefügt. Die Zusatzpositionen werden immer unterhalb der zugeordneten Artikelposition angefügt, evtl. vorhandene andere Positionen werden entsprechend verschoben. 

Tooltip: 

Ist dieses Kennzeichen gesetzt, so wird die Zusatzposition immer hinzugefügt 
(entweder nach Rückkehr aus dem Auswahldialog oder beim Speichern, wenn kein Auswahldialog erschienen ist). 

### Checkbox Pflichtposition 
Wenn Sie diese Checkbox aktivieren, werden bestimmte Positionen, sogenannte Pflichtpositionen (z.B. darf ein Wasserzähler nicht ohne Eichgebühr verkauft werden) standardmäßig nicht aus einer Positionsgruppe herausgelöst. Verfügt der Benutzer über eine spezielle Berechtigung, dann kann die Position doch entkoppelt werden. 

Ist die Checkbox gesetzt, kann sie im Auftrag nicht entkoppelt werden (Sie erhalten eine Fehlermeldung). 

Diese Checkbox kann nur gesetzt werden, wenn auch die Checkbox Automatisch dazu gesetzt ist. 

Die spezielle Berechtigung, auch solche Pflichtpositionen entkoppeln zu dürfen, kann im Dialog Berechtigungsgruppe über die Checkbox Entkoppeln von Pflicht-Zusatzpos. erteilt werden. 

Wenn eine Position aus dem Verbund bereits fakturiert ist, so muss der Benutzer auch das Recht zum Entkoppeln nach Teilfaktura haben (dafür steht die gleichnamige Checkbox im Dialog Berechtigungsgruppe zur Verfügung). Hier erscheint ebenfalls eine Fehlermeldung, wenn der Sachbearbeiter nicht die notwendige Berechtigung hat. 

Siehe Kapitel Berechtigungsgruppe . 

Tooltip: 

Definiert eine Pflichtposition. Diese kann im Auftrag nicht entkoppelt werden.
