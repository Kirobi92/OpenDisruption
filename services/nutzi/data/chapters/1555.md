# Tourenstamm 
Tourenstamm 

Dieser Dialog öffnet sich in der Tourendisposition >> Tour >> Tourenstamm oder im Auftrag über das Menü Stamm . Hier können Sie Touren festlegen. Dazu geben Sie im Kopf des Dialogs eine Stammtournummer und deren Bezeichnung ein und wählen in der Combobox Tourenlager ein Lager aus, von dem Ware für die Tour entnommen werden soll. 

Voraussetzung für das Ermitteln der nächsten Tour ist, dass die Tour bereits angelegt wurde. Bei der Generierung der Tour müssen die entsprechenden Felder im Tourenstamm gefüllt bzw. übernommen werden. Dabei wird aus Wochentag und Zeit ein konkretes Datum bzw. ein konkreter Zeitpunkt berechnet. 

Es gibt verschiedene Möglichkeiten eine Tour zu berechnen: 

- Ab dem angegebenen Zeitpunkt wird die nächste mögliche Tour ermittelt und vorgeschlagen. Und zwar auf Basis der Postleitzahl und der Annahmezeit. - Die Ermittlung der Tour erfolgt über Datum und Zeit in der Zukunft, also dem Termin an dem die Lieferung erfolgen soll. Es wird die letztmögliche Tour gesucht. - Zu einem bestimmten Datum wird eine Liste möglicher Touren erstellt (mit einem Toleranzzeitraum). 
Die Methoden liefern jeweils die gefundene Tour (Tournummer) oder eine Liste von Touren. 

Bei der Ermittlung der Tour ist je nach Kennzeichnung entweder Wochentag + Zeit des Annahmeschlusses oder Wochentag + Online-Zeit des Annahmeschlusses relevant für die Entscheidung. Standardmäßig wird nur der Annahmeschluss geprüft, der Online-Annahmeschluss steht externen Programmen zur Verfügung. 

Bei der Tourenzuordnung werden die Touren-Positionen entsprechend des Prioritätenschlüssels aufsteigend sortiert. Auch bei Retouren und Bestellungen findet eine Vorbelegung des Prioritätenschlüssels statt. 

Die Übertragung auf die Touren-Positionen erfolgt beim Erstellen der Tourenpositionen. Das Rückschreiben auf die Auftragspositionen erfolgt beim Zuordnen zu einer Tour. 

Wenn die zurückgelieferte Tour dem Vorgang nicht zugeordnet werden kann, wegen max. Abladestellen, Volumen, Abmessungen, Ladehilfsmittels etc., dann muss die Methode erneut mit veränderter Zeit aufgerufen werden, um die nächste mögliche Tour zu ermitteln, bis eine passende gefunden ist. Dieser Vorgang erfolgt automatisch durch das System, wenn der Verkaufsparameter Dyn. Lieferdatum gesetzt ist. 

Ist der Tour keine Postleitzahl zugeordnet, geht das System davon aus, dass die Tour überall hin liefert! 

Siehe Kapitel Parameter Verkauf . 

### Gefahrgut 
Hier kann hinterlegt werden, ob die Tour Gefahrgut - über die 1000 Punkte Marke hinaus - transportieren darf. 

Dieser Wert zieht, so lange noch kein Fahrer zugeordnet ist. Ist ein Fahrer zugeordnet, so zieht die ADR Gefahrgutinformation des Fahrers. Beim Ändern des Fahrers einer Tour wird geprüft, ob die Änderung auf Basis der aktuellen Punktesituation auf der Tour überhaupt gültig ist. Bei der Einplanung eines Auftrags auf eine Tour werden die ADR-Punkte geprüft und ggf. wird eine andere Tour genommen, falls die 1000-Punkte-Grenze bereits erreicht ist. 

Bitte beachten Sie, dass auch Retouren und Bestellungen in die Berechnung der ADR-Punkte einbezogen werden. Auch wenn diese zwischenzeitlich abgeholt oder Ware abgeliefert wird, so wird dies nicht berücksichtigt, da nicht ausgeschlossen werden kann, dass Ware doch noch auf dem LKW geladen ist, da der Kunde evtl. nicht angetroffen wurde. 

Eine Tour kann nicht abgeschlossen werden, wenn der Fahrer nicht die notwendigen Rechte besitzt.

### Feld Tour 
In dieses Feld geben Sie zur Suche einer Tour die Nummer für eine Stammtour ein. 

Bei Neuanlage einer Tour kann diese Stamm-Tour-Nummer manuell eingegeben werden oder kann vom System generiert werden. 

Tooltip: 

Geben Sie zur Selektion die Stamm-Tour-Nummer ein. 

### Checkbox Route 
Wenn Sie diese Checkbox aktivieren, wird die Route aktiviert. Sie gibt an, wohin die LKWs jeweils fahren, d.h. sie gibt die ausgeführte Tour pro Tag an. Die Planung wann Touren gefahren werden (Wochentag und Abfahrtszeit) ergibt sich aus dem Tourenstamm. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, wird die Route aktiviert. 

### Feld Bezeichnung 
Der Name der Tour wird nach Eingabe der Stamm Tour Nummer angezeigt. 

Bei Neuanlage einer Tour, ist dieses Feld zwingend zu füllen. So kann auch eine Selektion über dieses Feld stattfinden. 

Tooltip: 

Angabe der Bezeichnung 

### Feld Gültig von 
Der Tour ist zwingend eine Gültigkeit zuzuordnen; dafür geben Sie hier ein Datum ein. 

Tooltip: 

Für die Gültigkeit der Tour ist hier zwingend ein Datum einzugeben. 

### Feld Gültig bis 
Um der Tour einen Gültigkeitsbereich zuzuordnen, geben Sie hier ein Bis-Datum ein. 

Tooltip: 

Geben Sie ein "Bis-Gültigkeitsdatum" ein. 

### Combobox Tourenlager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager, von welchem aus die Tour startet. 

### Combobox Ist Tour zu Route 
Aus dieser Combobox wählen Sie eine Tourengruppe aus. So kann eine einzelne Tour einer Tourengruppe zugewiesen werden. In der Tourengruppe stehen alle Touren zur Verfügung, welche die Kennung ist Tourengruppe gesetzt haben. Durch die Zuordnung einer Tourengruppe wird die Tour mit Ihren Daten genau dem PLZ-Gebiet der Tourengruppe zugeordnet. 

Tooltip: 

Wählen Sie zur Selektion eine Tourengruppe aus.

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat.
