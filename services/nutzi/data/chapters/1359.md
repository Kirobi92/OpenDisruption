# GAP Position ändern Registerkarte Korrektur 
GAP Position Registerkarte Korrektur 

Diese Registerkarte zur Verfügung, um eine Gutschriftsposition zu korrigieren. D.h. wenn Sie eine Gutschrift über eine Position erhalten, die im Wert nicht dem originalen Positionswert aus dem Auftrag entspricht, kann hier über einen sogenannten Gegenauftrag der fehlende Betrag nachgefordert werden. Dazu geben Sie im Bereich Preis den Preis ein, den Sie nachfordern und/oder die Menge. 

In das Feld Postext G.-Auftr. können Sie einen individuellen Text zur Korrektur eingeben. 

Mit diesem Gegenauftrag wird der Kunde wieder in Fälligkeit gesetzt. 

Um eine Rechnung zu erzeugen, ist die Abrechnungsart Autom. Druck Rechnung zu setzen. Mit der Freigabe wird der Gegenauftrag freigegeben. 

Hinweis: Im Einzelfall kann die Abrechnungsart auch auf Gutschriftsabrechnungsverfahren gesetzt werden. 
### Feld Auftrag 
Hier wird die Nummer des Belegs aus den Belegarten Auftrag, Gutschrift, Bestellung und Belastung (Wawi) angegeben.

Dieses Feld steht in direktem Zusammenhang mit dem Parameter Auftragsart fix im Dialog Auftragsarten .

### Feld Postext G-Auftr. 
Zu einer Position kann ein bis zu 2000 Zeichen langer Zusatztext eingegeben und in den Formularen gedruckt werden. 

Bitte achten Sie darauf, einen doppelten Druck des Textes in den Formularen zu verhindern, indem Sie nicht den Artikeltext, sondern nur den Positionstext drucken. 

Beachten Sie außerdem, dass alle Änderungen im Dialog Position auch im zugrundeliegenden Vorgang mit dem Button gespeichert werden müssen.
