# Datenimport bearbeiten 
Import Bearbeiten 

Dieser Dialog öffnet sich im Artikel Import Manager über den Button Bearbeiten auf der Registerkarte Allgemein . Hier können eingelesene Quelldaten bearbeitet oder ergänzt werden, bspw. können Artikel Kalkulationsgruppen eingefügt werden. 

Die eingelesenen Artikel-Datensätze können nach unterschiedlichen Kriterien gefiltert werden. Entsprechend dynamisch sind die Spalten, je nach Import-Format. Dadurch lassen sich die Artikel einfacher bearbeiten bzw. mit Daten anreichern. 

Hier werden auch die neu eingelesenen Daten mit dem vorherigen Import in dieses Data-Staging verglichen, wenn die Importschlüssel übereinstimmen (Artikeldaten alt <> Artikeldaten neu). 

Zum Bearbeiten können die entsprechenden Artikeldaten gefiltert werden. Dazu stehen auf der Registerkarte Artikel Felder und Comboboxen zur Verfügung. 

Die Einstellungen werden für das Staging im Artikel Schattenstamm genutzt. 

Siehe Kapitel Artikel Schattenstamm .

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Button Verarbeiten 

Mit diesem Button werden alle markierten Datensätze verarbeitet (Artikelstammdaten und Preise) und in den Artikel Schattenstamm übergeben. 

Tooltip: 

Übernimmt alle Importdaten in den Schattenstamm 

### Feld Hauptlieferant 
Hier wird die Nummer des Hauptlieferanten für den Artikel angegeben. Dieser Lieferant wird dann im Bestellwesen als Lieferant vorgeschlagen. 

Tooltip: 

Nummer des Hauptlieferanten des Artikels. 

### Checkbox Neu 
Wenn Sie diese Checkbox aktivieren, werden alle Datensätze eingelesen, welche erstmalig in das Staging eingelesen werden. 

### Checkbox Änderungen 
Wenn Sie diese Checkbox aktivieren, werden alle Datensätze angezeigt, welche bereits bei einem früheren Import eingelesen wurden. D.h. geänderte Artikel, die gegen den Artikelstamm geprüft werden. Es werden nur Datensätze mit Änderungen angezeigt! 

### Checkbox Ohne Änderungen 
Wenn Sie diese Checkbox aktivieren, werden alle Datensätze angezeigt, welche bei früheren Importen bereits eingelesen wurden. Es werden nur Datensätze ohne Änderungen angezeigt. 

### Checkbox Verarbeitete 
Wenn Sie diese Checkbox aktivieren, werden Datensätze eingelesen, welche aus diesem Import bereits in den Artikel Schattenstamm eingelesen wurden und welche hier noch einmal überarbeitet werden sollen, um sie dann erneut in den Artikel Schattenstamm zu übernehmen. 

Wenn Sie einen oder mehrere Datensätze in der Registerkarte Artikel markieren und einen Doppelklick ausführen oder den Button Bearbeiten aktivieren, öffnet sich der Dialog Import Editor - Artikel . 

Siehe Kapitel Import Editor - Artikel . 

### Feld Bezeichnung 
Hier wird das Importformat aus dem zugrundeliegenden Datensatz angezeigt. 

Tooltip: 

Angabe der Bezeichnung 

### Feld Import-Version 
Hier wird die Import-Version aus dem zugrundeliegenden Datensatz angezeigt. 

### Feld Datenbereich 
Hier wird der Datenbereich zu den importierten Daten angezeigt. Es werden nur Daten desselben Bereichs miteinander aktualisiert. 

Tooltip: 

Die importierten Daten gehören zu einem Datenbereich. Es werden nur Daten desselben Bereichs miteinander aktualisiert. 

Es wird empfohlen, mindestens einen Datenbereich je Datenlieferant zu nutzen. Wird kein Datenbereich definiert, setzt das System automatisch die Importformat-ID als Bereich.
