# Debitoren Factoring Registerkarte Neue OP 
Debitoren Factoring Registerkarte Neue OP 

Hier werden alle offenen Posten der Debitoren eingelesen. 

Es werden nur Daten in die Tabelle eingelesen, wenn im Dialog Factoring Parameter Registerkarte Allgemein die Checkbox OPs täglich aktiviert ist. 
### Tabelle 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben. 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Bezeichnung 
Hier wird die Bezeichnung angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Belegnummer 
Hier wird die Belegnummer angegeben. 

Tooltip: 

Angabe der Belegnummer 

### Tabellenspalte Betrag 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte S/H 
Hier wird das Soll- / Haben-Kennzeichen angegeben. 

Tooltip: 

Soll / Haben-Kennzeichnung 

### Tabellenspalte Währung 
Hier wird das Währungskürzel angegeben. 

Tooltip: 

Währungskürzel 

### Tabellenspalte Betrag FW 
Hier wird der Betrag in Fremdwährung angegeben. 

Tooltip: 

Angabe des Betrags in Fremdwährung 

### Tabellenspalte Kurs 
Hier wird der Kurs angegeben. 

Der Wechselkurs wird durch das Programm aus der Währungstabelle vorbelegt. 

Tooltip: 

Währungskurs 

### Tabellenspalte Skontierfähig 
Hier wird der skontierfähige Betrag angegeben, inkl. Steuer und in Landeswährung. 

Tooltip: 

Skontierfähiger Wert, 
inkl. Steuer, Landeswährung

### Tabellenspalte Nettofälligkeit 
Hier wird das Netto-Fälligkeitsdatum angegeben. 

Tooltip: 

Angabe der Nettofälligkeit. 
Die Nettofälligkeit ist das Datum, zu dem ein offener Posten ohne Skontoabzug zu zahlen ist. 

### Tabellenspalte Factoring-Typ 
In dieser Spalte wird der Factoring-Typ angezeigt. So kann beim Commerz-Factoring mitgeteilt werden, um welche Art OP es sich handelt.
