# Kapitel NV.ERP.Base.Common.cdSystemMM.shtDoubleSupplyProtection

Wenn Sie diesen Parameter aktivieren, wird verhindert, dass ein Kunde doppelt (bzw. 2-mal an einem Tag) beliefert wird. 

Dies könnte der Fall sein, wenn beispielsweise der Kunde an einem bestimmten Tag mit der Spedition beliefert wird. Dann soll die restliche Ware (aus Kostengründen) ebenfalls mit der Spedition geliefert werden, anstatt dem eigenen Fuhrpark oder Paketdienst. 

Tooltip: 

Die Lieferung mehrerer Aufträge über mehrere Touren an einem Tag soll verhindert werden.
