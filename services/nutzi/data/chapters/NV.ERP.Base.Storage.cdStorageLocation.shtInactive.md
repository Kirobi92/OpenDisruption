# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtInactive

Dieser Parameter wird über den Button De/aktivieren gesteuert und kann nicht direkt genutzt werden! 

Wenn das Lager deaktiviert ist, kann auch nicht mehr auf dieses Lager gebucht werden; es steht auch in keinem Vorgang mehr zur Verfügung. 

Diese Einstellung findet beispielsweise Verwendung, wenn eine Niederlassung geschlossen wird, die historischen Vorgänge aber erhalten bleiben (müssen). 

Tooltip: 

Diese Checkbox wird nur über den Button De/aktiviere gesteuert. 
Ein manuelles Setzen ist daher nicht möglich!
