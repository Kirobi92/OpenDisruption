# CSC Spalten über Client Side Customizing hinzufügen 
Es ist möglich, über das Client Side Customizing einer Tabelle weitere Spalten hinzuzufügen. So können Sie sich Tabellen zusammenstellen, wie Sie es für Ihre Arbeit brauchen. 

Nachfolgend wird am Beispiel der Tabelle im Kundenstamm die Vorgehensweie erläutert. Dazu wird die Spalte 'Versandweg' ausgewiesen mit Listenwerten und nicht mit Ziffern (IDs). 

Starten Sie im Dialog das Client Side Customizing mit Klick auf den Button . 

Hier wählen Sie den Eintrag Customizing aus. 

Es öffnet sich der Dialog Client Customizing Console . Jetzt klicken Sie irgendwo in die Zeile der Tabelle im geöffneten Dialog. 

Im Dialog Client Side Customizing Console wird grdTable markiert und darunter, in der Registerkarte Properties wird die Zeile childControl ausgewiesen. Dort klicken Sie auf den +Button. 

Jetzt beginnen Sie Ihre Anpassungen im Client Side Customizing . Folgen Sie dabei den nachfolgenden Arbeitsschritten: 

- Auswahl der Spalte im Dialog Table Column Wizard (Registerkarte Datasource selection ) und Klick auf den Button Choose . 
Table Column Wizard 

- Anschließend wird automatisch auf die Registerkarte Settings gewechselt. Hier passen Sie das Feld EditorStyle an, indem Sie den Eintrag auf Combobox ändern. 
Table Column Wizard Registerkarte Settings, Feld EditorStyle 

- Passen Sie das Feld ListDatasource an, indem Sie den Button Choose drücken. 
Table Column Wizard Registerkarte Settings, Feld List Datasource 

- Jetzt ist das Objekt zuzuweisen. Dazu wählen Sie im Dialog Datasource Picker - List Datasource das Objekt aus. 
Datasource Picker - List Datasource 

- Passen Sie das Feld List Display Attribute mittels Button Choose an. 
Table Column Wizard Registerkarte Settings, Feld List Display Attribute 

- Nun ist zwingend die Darstellung zuzuweisen. Dies nehmen Sie im Dialog Datasource Picker - List Display Member mit dem Button Choose vor. Markieren Sie die Auswahl Text und bestätigen mit Button Choose . 
Datasource Picker - List Display Member 

- Passen Sie das Feld List Value Attribute an; ebenfalls mit dem Button Choose . 
Table Column Wizard Registerkarte Settings, Feld List Value Attribute 

- Anschließend ist zwingend der Datenbank-Wert zuzuweisen. Dies nehmen Sie im Dialog Datasource Picker - List Value Member vor. Markieren Sie die Auswahl ID und bestätigen dies mit dem Button Choose . 
Datasource Picker - List Value Member 

- Passen Sie nun das Feld EditStyle im Dialog Table Column Wizard Registerkarte Settings an. Wählen Sie im Feld EditStyle den Eintrag ListValuesInput aus. 

Table Column Wizard Registerkarte Settings 

Wählen Sie ListValuesInput aus, damit die Werte ausgewiesen werden und eine freie Texteingabe nicht möglich ist. 

Deaktivieren Sie die Checkbox Enabled , damit in der Tabelle nichts geändert werden kann. 

Klicken Sie nun auf den Button Finish . Die neue Spalte ist eingefügt. 
Ende 

Speichern Sie im Dialog Client Side Customizing über den markierten Button. 

Client Side Customizing - Speichern 

Anschließend schließen Sie die Client Side Customizing Console über das X. 

Die neue Spalte befindet sich im eNVenta Dialog ganz rechts außen. Siehe nachfolgende Abbildung: 

Kunde Registerkarte Tabelle mit der Spalte 'Versandweg' 

In diesem Beispiel wird die Spalte 'Versandweg' mit der Information angezeigt, die auf der Registerkarte Logistik ausgewählt ist. 

Nicht jedes Feld, das Sie per Client Side Customizing in die Tabelle einbinden, darf auch verändert werden!
