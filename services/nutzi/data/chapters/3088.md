# Artikel Import Manager Registerkarte Allgemein 
Artikel Import Manager Registerkarte Allgemein 

Auf dieser Registerkarte stehen die Details zum Import zur Verfügung. 

Die Formate werden in alphabetischer Reihenfolge angezeigt. 
### Button Bearbeiten 

Mit diesem Button wird der Dialog Import Editor geöffnet. Hier können Daten selektiert werden, welche bearbeitet werden sollen. 

Siehe Kapitel Import Bearbeiten . 

### Button Verarbeiten 

Mit diesem Button können Sie einen Eintrag der Tabelle verarbeiten. 

### Button Löschen 

Mit diesem Button können Sie einen Eintrag der Tabelle löschen. 

### Tabelle 

### Tabellenspalte Datenlieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Tabellenspalte Import Format 
Hier wird die Bezeichnung angegeben. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Hauptdatensätze 
Hier wird angezeigt, wie viele Datensätze vom Typ der Haupt-Importkomponenten aus der jeweiligen Format-Konfiguration importiert wurden. Falls mehrere Datenquellen mit unterschiedlichen Importkomponenten angegeben sind, werden die Datensätze aller aufgeführten Typen eingerechnet. 

Die Anzahl der nur in Abhängigkeit zur Hauptkomponente importierten Sätze, wird aus der Berechnung ausgeschlossen. 

Tooltip: 

Zeigt an, wieviele Datensätze vom Typ der Haupt-Importkomponenten aus der jeweligen Format-Konfiguration importiert wurden. Falls mehrere Datenquellen mit unterschiedlichen Importkomponenten angegeben sind, werden die Datensätze aller aufgeführten Typen eingerechnet. 

Die Anzahl der nur in Abhängigkeit zur Hauptkomponente importierten Sätze wird aus der Berechnung ausgeschlossen.
