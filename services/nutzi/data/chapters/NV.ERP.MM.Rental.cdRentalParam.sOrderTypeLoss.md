# Kapitel NV.ERP.MM.Rental.cdRentalParam.sOrderTypeLoss

Falls ein Mietobjekt vom Kunden verloren geht, so ist hier die entsprechende Auftragsart auszuwählen. Das entsprechende virtuelle Mietlager sollte hier ebenfalls hinterlegt sein, damit das Mietlager um dieses Mietobjekt entlastet werden kann. 

Siehe Kapitel Auftragsarten . 

Tooltip: 

Auftragsart 'Verlust': 
Verliert der Kunde ein Mietobjekt, so muss er dieses in der Regel bezahlen. 
Damit das virtuelle Mietlager von dem Mietobjekt entlastet wird, sollte in der hier hinterlegten Auftragsart ebenfalls das virtuelle Mietlager als Lager hinterlegt werden. 
Lagerbuchung muss ebenfalls aktiviert sein.
