# Prognoseausschluss 
Prognoseausschluss 

Der Button Ausschluss im Prognosemonitor sowie die Option Prognoseausschluss im Artikelstamm öffnen den Dialog Prognoseausschluss . So können in der Prognoserechnung Kunden bzw. Kundengruppen für einzelne Artikel von der Ermittlung der Prognose ausgeschlossen werden. 

Folgende Werte sind im Dialog Prognoseausschluss einzutragen: 

- Artikelnummer (Vorbelegung aus dem Artikelstamm, wenn der Dialog über Optionen geöffnet wird) - Ausschlusselementart: - für Kunden - Für Kundengruppe - Ausschlusselement: - Kundennummer oder - Kundengruppe 
Werden ein Kunde oder eine Kundengruppe gelöscht, werden auch die abhängigen Prognosewerte gelöscht. 

Die doppelte Eingabe eines Datensatzes im Dialog Prognoseausschluss ist nicht möglich, d.h. die Eingabe wird abgewiesen und Sie erhalten eine entsprechende Meldung. 

Wenn ein Artikel hinzugefügt oder gelöscht wird, erfolgt eine Abfrage, ob alle Prognosen, die zu diesem Artikel gehören, gelöscht werden sollen. 

Empfehlung: Speichern Sie erst, wenn alle Werte gefüllt sind, ansonsten geht bei jedem Speichern eines Artikels (für den bereits eine Prognose existiert) eine Messagebox auf.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
