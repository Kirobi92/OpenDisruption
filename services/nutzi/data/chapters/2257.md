# BMECat - Import Registerkarte Artikel 
BMEcat Import Registerkarte Artikel 

Diese Registerkarte zeigt die Informationen zum Artikel an. Die Import-Artikel werden in der Tabelle angezeigt. Für jeden Artikel wird geprüft, ob es diesen Artikel bereits im Artikelstamm gibt. 
### Button Artikel prüfen 

Wenn Sie diesen Button drücken, wird die Artikelnummer mit der Lieferanten-Artikelnummer verglichen. 

Tooltip: 

Artikelnummer mit Lieferanten-Artikelnummer abgleichen. 

### Checkbox Artikeldaten 
Wenn Sie diese Checkbox aktivieren, werden Artikeldaten importiert. 

Tooltip: 

Artikeldaten importieren 

### Checkbox Preis 
Wenn Sie diese Checkbox aktivieren, werden Artikelpreis importiert. 

Tooltip: 

Artikelpreise importieren 

### Checkbox Merkmale 
Wenn Sie diese Checkbox aktivieren, werden Artikelmerkmale importiert. 

Tooltip: 

Artikelmerkmale importieren 

### Checkbox Ressourcen 
Wenn Sie diese Checkbox aktivieren, werden Artikelresourcen importiert. 

Tooltip: 

Artikelressourcen importieren 

### Tabelle 

Die Tabelle zeigt die Daten entsprechend der Selektion an. 
### Tabellenspalte Lieferanten Artikelnr 
Tooltip: 

Lieferanten Artikelnummer 

### Tabellenspalte Hersteller Art. 
Tooltip: 

Hersteller Artikelnr.
