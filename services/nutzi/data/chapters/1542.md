# Ladeliste drucken 
Ladeliste drucken 

Dieser Dialog öffnet sich in der Kommissionierung >> Optionen >> Frachtenplanung >> Ladeliste drucken oder über Tourendisposition >> Druck >> Ladeliste drucken . 

Ist die Ladeliste leer und Sie drücken den Button öffnet sich für den Druck der Ladelisten ebenfalls der Dialog Ladeliste drucken . 

Haben Sie im Menü Ladeliste eine Ladeliste ausgewählt, wird beim Öffnen des Dialogs Ladeliste drucken gleich diese Ladeliste eingefügt. Der Druckvorgang wird ausgelöst, so dass der Benutzer direkt die aktuelle Ladeliste drucken kann. 

Dies funktioniert allerdings nur, wenn die Ladeliste bereits gespeichert wurde. Ansonsten wird der Druck mit Hinweis auf die fehlende Speicherung abgebrochen. 

Des Weiteren werden 2 Formulare angeboten: 

- Eine Frachtliste : Das ist eine Ladeliste mit der Reihenfolge, wie die Güter aufgeladen werden müssen (das Versandgut, das als letztes abgeladen werden soll, muss zuerst eingeladen werden). - Eine Tourliste nach Anfahrstationen: Im Prinzip die gleichen Informationen wie in der Frachtliste nur, dass die Reihenfolge umgekehrt ist (die erste Abladestation steht ganz oben auf der Liste, die letzte ganz unten).
