# Kapitel NV.ERP.Base.Common.cdSystemMM.shtICAutoInvoicePosting

Ist dieser Parameter für eine automatische Rechnungsprüfung gesetzt, löst der IC-Adapter eine komplette Verbuchung der Rechnungsprüfung aus. Hierbei wird angenommen, dass die exportierte Rechnung des einen Mandanten immer stimmt und somit eine Freigabe der Rechnung automatisch erfolgen kann.
