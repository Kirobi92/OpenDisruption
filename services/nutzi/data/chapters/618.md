# Massenänderung 
Massenänderung 

Dieser Dialog steht Ihnen in vielen Dialogen unter Menü Optionen zur Verfügung. Dazu gehören u.a.: 
- Kunde - Mietobjekt - Bankenstamm - Suchbedingung - Arbeitsplan 
Der Dialog Massenänderung bietet die Möglichkeit, Änderungen für mehrere Datensätze einer Datenbanktabelle in eNVenta gleichzeitig und damit auch schnell vorzunehmen. Hierbei werden alle Sicherheitsprüfungen der Logikebene berücksichtigt, z.B. muss im Lieferantenstamm immer eine Lieferantennummer vorhanden sein. 

Schlüsselfelder können nicht geändert werden. 

Gehen Sie wie folgt vor: 

- Suchen Sie sich alle Daten zusammen, die Sie ändern wollen, z.B. über die Suche mit SQL . - Sie erfassen oder ändern z.B. im Feld Code des Lieferantenstamms einen Eintrag. 
Damit sich der Dialog Massenänderung öffnet, darf die vorgesehene Änderung nicht gespeichert werden!!! Stattdessen öffnen Sie den Dialog Massenänderung . - Auf der Registerkarte Vorlage sehen Sie den ursprünglichen Wert in der Spalte 'Original Wert' (falls vorhanden) und in der Spalte 'Geänderter Wert' die vorgenommene Änderung. 
Sind alle Werte in Ordnung, wechseln Sie auf die Registerkarte Vorschau .
