# Serviceleitstand 
Serviceleitstand 

In diesem Menü werden alle Ressourcen angezeigt. So erhalten Sie eine Übersicht, wie Ihre Ressourcen verplant sind. Eine Auswahl über hierarchisch angelegte Ressourcengruppen ist möglich. 

Alle Zuteilungen von Monteuren werden angezeigt, welche gleichzeitig als Ressource angelegt sind. 

Folgende Status finden Verwendung: 

Blau Gelb Grün 
| offener Serviceeinsatz |
| --- |
| freigegebener Serviceeinsatz (in Bearbeitung) |
| abgeschlossener Serviceeinsatz |

Änderungen des Einsatzes sind möglich, indem Sie den Balken im Gantt-Diagramm verschieben. 

Ein Einsatz lässt sich nur verschieben, wenn der Status des Einsatzes nicht auf ‚abgeschlossen‘ steht. Auch das Größer-Ziehen eines Balkens ist möglich. 

Bereits vorhandene Einsätze im gleichen Zeitraum werden nicht berücksichtigt. 

Wird bei der Erstellung eines Serviceeinsatzes (z.B. aus dem Call Center) kein Monteur angegeben, wird der Kunde im Serviceleitstand angezeigt. Durch vertikales Verschieben kann eine Einplanung auf einen Monteur vorgenommen werden.

Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen über diesen Button ebenfalls zur Verfügung.
