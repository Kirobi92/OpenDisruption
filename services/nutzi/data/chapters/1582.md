# LVS Parameter Registerkarte Allgemein 
LVS Parameter Registerkarte Allgemein 

Auf dieser Registerkarte werden allgemeine Einstellungen für das LVS vorgenommen. 
### Combobox Mitarbeiter 
Für jeden Scanner-Sachbearbeiter wird im Lager automatisch ein LHM mit diesem LHM-Typ angelegt. Dieses LHM kann bei der Kommissionierung zum Transport verwendet werden. 

Tooltip: 

Für jeden Scanner-Sachbearbeiter wird im Lager automatisch ein LHM mit diesem LHM-Typ angelegt. Dieses LHM kann bei der Kommissionierung zum Transport verwendet werden. 

### Combobox Standard 
Hier wählen Sie den LHM-Typ für Zuladung und Cross Docking. 

Tooltip: 

LHM Typ für Zuladung und Cross Docking. 
Dazu sollte ein eigener LHM-Typ verwendet werden. 

### Checkbox Restposition Blockieren 
Die neu generierten Restpositionen mit Bezug auf ein LVS-Lager bekommen eine Liefersperre. 

Tooltip: 

Die neu generierten Restpositionen mit Bezug auf ein LVS-Lager bekommen eine Liefersperre. 

### Combobox Automatik 
Aus dieser Combobox wird die automatische Platzzuordnungslogik für chaotische, nicht-LVS-Lager gewählt. Je nach Auswahl zieht dann die automatische Lieferfreigabe aus dem Dialog Parameter Verkauf , Combobox Autom. Plätze . 

Siehe Kapitel Parameter Verkauf . 

Tooltip: 

Mit dieser Combobox wird die automatische Platzzuordnungslogik für chaotische, nicht-LVS-Lager bestimmt. 

### Feld Zeitspanne WV 
Für die Erstellung einer automatischen Wiedervorlage kann hier ein Zeitraum eingetragen werden. Wenn die Endkontrolle mehr als x-Tage deaktiviert wurde, erhält der Sachbearbeiter eine Wiedervorlage. 

Tooltip:

### Combobox Empfänger WV 
Wenn die Endkontrolle mehr als die definierten Tage in „Zeitspanne WV“ deaktiviert wird, erfolgt eine Wiedervorlage an den Empfänger. 

Tooltip: 

Wenn die Endkontrolle mehr als die definierten Tage in „Zeitspanne WV“ deaktiviert wird, erfolgt eine Wiedervorlage an den Empfänger. 

### Feld Mengenfaktor 
Hier geben Sie einen Faktor für die Mengeneingabe an. Wird dieser Mengenfaktor bei der Eingabe überschritten ( Beispiel: Es sollen 10 gepickt werden und der Mitarbeiter gibt 10000 ein) so kommt es bei einem Faktor von 1000 automatisch zu einer Fehlermeldung. 

Tooltip:

### Combobox Belastungsart 
Hier wählen Sie die Belastungsart aus, die standardmäßig gezogen werden soll. 

Tooltip: 

Aus dieser Combobox wählen Sie die Belastungsart aus, die standardmäßig gezogen werden soll. 

### Checkbox Retouren WE Theke 
Wenn Sie diesen Parameter aktivieren, kann am Retouren-Bildschirm ein verkürzter Wareneingang durchgeführt werden. Hierbei wird ein Etikett an der Theke erzeugt, mit der die retournierte Ware wieder eingelagert werden kann. Die Retoure muss dann nicht über den Einlagerungsdialog durchgeführt werden. 

Tooltip: 

Wenn Sie diesen Parameter aktivieren, kann am Retouren-Bildschirm ein verkürzter Wareneingang durchgeführt werden. Hierbei wird ein Etikett an der Theke erzeugt, mit der die retournierte Ware wieder eingelagert werden kann. Die Retoure muss dann nicht über den Einlagerungsdialog durchgeführt werden. 

### Checkbox Nur 1 Fixplatzartikel pro Platz zuweisbar 
Wenn Sie diesen Parameter aktivieren, werden bei Neuanlage die Erst- und Zweitlagerplätze vorbelegt. Das gleiche Datenfeld wird am Lagerplatz vorbelegt. 

Tooltip: 

Vorbelegung bei Erst- und Zweitlagerplätzen bei der Neuanlage 

### Checkbox Kein Lagerplatzscan 
Wenn Sie diesen Parameter aktivieren, muss der Lagerplatz nicht gescannt werden. Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Tooltip: 

Wenn Sie diesen Parameter aktivieren, muss der Lagerplatz nicht gescannt werden. 
Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Checkbox Wareneingang mit Sammeletikett 
Wenn Sie diese Checkbox aktivieren, wird der gleichnamige Parameter im Dialog Plätze Einlagerung vorbelegt. Auftragsbezogen bestellte Ware kann gesammelt, durch das Scannen eines Sammeletiketts vereinnahmt und eingelagert werden. 

Ist diese Checkbox nicht gesetzt, wird schon das Sammeletikett nicht gedruckt. 

Siehe Kapitel Plätze Einlagerung . 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, wird der gleichnamige 
Parameter im Dialog 'Einlagerung' vorbelegt. Auftragsbezogen 
bestellte Ware kann gesammelt, durch das Scannen eines 
Sammeletiketts vereinnahmt und eingelagert werden. 
Ist diese Checkbox nicht gesetzt, wird schon das Sammeletikett 
nicht gedruckt. 

### Checkbox Lagerplatzüberschreiben erlaubt 
Wenn dieser Parameter aktiviert wird, ist das Einlagern auf einen abweichenden Lagerplatz erlaubt. 

Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Tooltip: 

Wenn dieser Parameter aktiviert wird, ist das Einlagern auf einen abweichenden Lagerplatz erlaubt. 
Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Checkbox Wareneingang mit Lieferungsauswahl 
Wenn Sie diese Checkbox aktivieren, muss jede gelieferte Position aktiv als geliefert gesetzt werden. 

Tooltip: 

Ist dieser Parameter aktiv, muss jede gelieferte Position aktiv als 'geliefert' gesetzt werden. 

### Checkbox Komplettauftrag über LA reservieren 
Wenn Sie diese Checkbox aktivieren, wird der komplette Auftrag als Lagerauftrag reserviert (Auftragstyp Komplettauftrag ). Diese Checkbox sollte standardmäßig gesetzt sein. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, wird der komplette Auftrag als Lagerauftrag reserviert. 

### Checkbox Fixterminauftrag über LA reservieren 
Ist dieser Schalter gesetzt, wird ein sogenannter Fixterminauftrag erzeugt. D.h. es wird ein Liefertermin in der Zukunft mit der Datumseinstellung des Fixtermins erzeugt. Diese Checkbox sollte standardmäßig gesetzt sein. 

Tooltip: 

Ist dieser Parameter gesetzt, so wird beim Wareneingang automatisch ein Auslagerungsauftrag erzeugt, welcher die Ware in einen speziellen Lagerbereich umlagert. 

### Checkbox Abrufauftrag über LA reservieren 
Wenn diese Checkbox gesetzt ist, wird beim Wareneingang automatisch ein Auslagerungsauftrag erzeugt, welcher die Ware in einen speziellen Lagerbereich umlagert. 

Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

Tooltip: 

Ist diese Checkbox gesetzt, so wird beim Wareneingang automatisch ein Auslagerungsauftrag erzeugt, welcher die Ware in einen speziellen Lagerbereich umlagert. 
Diese Funktion steht für das 2-stufige LVS zur Verfügung. 

### Checkbox Anscannen des Auftragsbezugs bei Auslagerung 
Wenn Sie diesen Parameter aktivieren, muss bei der Auslagerung – nach Angabe des Lagerplatzes – die Auftragsreferenz angegeben werden, welche auf dem Einlagerungsetikett angedruckt ist. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, muss bei der Auslagerung – nach Angabe des Lagerplatzes – die Auftragsreferenz angegeben werden, welche auf dem Einlagerungsetikett angedruckt ist. 

### Checkbox Erhalte Pick-Pool Einträge bei der Lagerauftragsgenerierung 
Wenn diese Checkbox aktiviert wird, bleibt der Pickpool-Eintrag erhalten bis der Lieferschein gedruckt ist. Dies ist wichtig für die Nachvollziehbarkeit und die Anzeige im Pickmonitor. 

Wenn diese Checkbox nicht gesetzt ist, wird mit Generierung des Lagerauftrags der Pick gelöscht. 

Tooltip: 

Wenn diese Checkbox aktiviert wird, wird der Eintrag im 
Pick-Pool nach Erzeugen des Lagerauftrags nicht gelöscht. 

### Checkbox Rücklagerung nach Storno 
Wenn Sie diesen Parameter aktivieren, wird nach dem Stornieren eines durchgeführten Lagerauftrags automatisch eine Umlagerung (zur "Rücklagerung", d.h. der vorherige Vorgang wird rückgängig gemacht) des Lagerauftrags erzeugt. 

Tooltip: 

Wenn diese Checkbox aktiviert wird, wird nach dem Stornieren eines durchgeführten Lagerauftrags automatisch eine Rücklagerung des Lagerauftrags erzeugt. 

### Checkbox Cross Docking 
Wenn Sie diese Checkbox aktivieren, wird beim Wareneingang der auftragsbezogen bestellten Ware der Cross Docking Prozess initialisiert. D.h. die Ware wird direkt in den Warenausgang gebracht. 

Beim I-Punkt-Verfahren wird sie direkt ins Tourenfach gelegt. Beim Packplatz wird die Ware in den Packbereich gestellt, damit es zusammen mit der restlichen Ware verpackt werden kann. 

Tooltip: 

Ist dieser Parameter aktiviert, so kann in der Einlagerung die Ware direkt in den Warenausgang verschoben werden. 

### Checkbox Wareneingang mit Lieferungsauswahl 
Wenn Sie diese Checkbox aktivieren, muss jede gelieferte Position aktiv als geliefert gesetzt werden. 

Tooltip: 

Ist dieser Parameter aktiv, muss jede gelieferte Position aktiv als 'geliefert' gesetzt werden. 

### Checkbox Scanner-gestützte Fehlerkorrektur 
Ist diese Option aktiv, so wird die Fehlerbeseitigung bei der Endkontrolle durch einen Scanner-geführten Prozess unterstützt. 

Tooltip: 

Ist diese Option aktiv, so wird die Fehlerbeseitigung bei der Endkontrolle durch einen Scanner-geführten Prozess unterstützt. 

### Checkbox LHM ohne Lagernr. 
Wenn Sie diese Checkbox aktivieren, wird ein Ladehilfsmittel ohne Lagernummer erzeugt. Ist die Checkbox nicht gesetzt, wird die Lagernummer zum LHM generiert. 

Tooltip: 

Ladehilfsmittel werden ohne Lagernummer in der ID erzeugt. Dadurch sind sie lagerübergreifend einsetzbar.
