# BMEcat - Katalogexport Registerkarte Kataloggruppen 
BMECat - Katalogexport Registerkarte Kataloggruppen 

Auf dieser Registerkarte werden die Kataloggruppen aufgelistet, die exportiert werden (Haupt- und / oder Untergruppen). 

Beschreibungen können im Feld Beschreibung hinzugefügt werden. 
### Button Laden 

Wenn Sie diesen Button drücken, werden zu exportierende Gruppen angezeigt. 

Tooltip: 

Anzeige zu exportierender Gruppen 

### Feld System ID 
Hier kann eine System ID zur Selektion eingetragen werden. 

Tooltip: 

Angabe der Katalogsystemkennung (Der Lieferant sollte für sich eine eindeutige Kennung des Kataloggruppensystems haben.) 

### Feld Name 
Hier wird der Name des Katalogs automatisch zur Nummer eingetragen. 

Tooltip: 

Name des Kataloggruppensystems 

### Feld Beschreibung 
In diesem Feld kann der Sachbearbeiter einen individuellen Text erfassen. 

Tooltip: 

Beschreibung des Kataloggruppensystems 

### Tabelle 

### Tabellenspalte ID 
Tooltip: 

Angabe der Gruppennummer 

### Tabellenspalte Name 
Tooltip: 

Gruppenname 

### Tabellenspalte Typ 
Tooltip: 

Angabe des Gruppentyps (root, node, leaf) 

### Tabellenspalte Obergruppe 
Tooltip: 

Übergeordnete Gruppe 

### Tabellenspalte Beschreibung 
Tooltip: 

Angabe der Gruppenbeschreibung
