# Datev-Parameter 
DATEV-Parameter 

Dieser Dialog öffnet sich im Dialog DATEV-Export WaWi / Fibu über den Button Parameter . Hier definieren Sie Parameter für DATEV. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld DATEV-Parameter 
Tooltip: 

In diesem Dialog definieren Sie die Parameter für DATEV. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren . 

### Menü Fenster-Information 
Über dieses Menü werden Ihnen Informationen zum Dialog angezeigt. Siehe nachfolgendes Beispiel: 

Meldungen 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Einstellungen. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Combobox Schnittst.-Typ 
Hier wählen Sie die passende Schnittstelle aus. Zur Verfügung steht Format . Wenn Sie Format auswählen, haben Sie den Vorteil, dass keine DATEV-spezifischen .exe- und .ini-Dateien benötigt werden. 

### Feld Berater 
Hier geben Sie (in Absprache mit Ihrem Steuerberater) die Beraternummer des Steuerberaters bei DATEV ein. 

### Feld Beratername 
Hier geben Sie (in Absprache mit Ihrem Steuerberater) den Namen des Steuerberaters bei DATEV ein. 

### Feld Mandant 
Hier geben Sie (in Absprache mit Ihrem Steuerberater) die Mandantennummer bei DATEV ein. 

### Feld Abr.-Jahr 
Hier wird das (Geschäfts-) Jahr angegeben. 

Für dieses Jahr wird die Übertragung der Daten an DATEV vorgenommen. 

Tooltip: 

Abrechnungsjahr 

### Button Eröffnen 

Dieser Button wird sichtbar, wenn die Finanzbuchhaltung nicht im Einsatz ist. Mit diesem Button kann ein neues Buchungsjahr eröffnet werden. 

Tooltip: 

Neues Buchungsjahr eröffnen? 

### Feld Sachk.länge 
Hier können Sie für einen einwandfreien DATEV-Export die Kontonummerlänge für das KNE-Verfahren eingeben. 

Tooltip: 

Sachkontenlänge (min. 4 - max. 8) 

### Button Sachkontenstamm 

Dieser Button wird sichtbar, wenn die Finanzbuchhaltung nicht im Einsatz ist. Mit diesem Button kann ein neues Buchungsjahr eröffnet werden. 

Tooltip: 

Sachkontenstamm aufrufen 

### Feld Letzter Export bis 
In diesem Feld wird automatisch das Datum des letzten Übertragungsvorgangs eingetragen. 

Tooltip: 

Letzter Export bis 

### Checkbox Ohne Belegnummer 2 
Wenn diese Checkbox nicht gesetzt ist, wird die Spalte ‚Belegnummer 2‘ angezeigt und in der Export-Datei in die Spalte ‚Belegfeld 2‘ geschrieben. 

Wird die Checkbox allerdings aktiviert, wird die Spalte nicht angezeigt und der Wert wird nicht in die Export-Datei geschrieben. 

Tooltip: 

Belegnummer 2 nicht exportieren 

### Checkbox Nettofälligkeit in Belegnummer 2 
Tooltip: 

Exportiere das Nettofälligkeitsdatum (in Belegnummer 2) 

### Feld Verzeichnis 
Das Verzeichnis wird über den Dateiauswahl-Dialog eingetragen. Im Verzeichnis werden alle benötigten Dateien des Datei-Exports eingetragen. 

### Button Suche 

Mit diesem Button können Sie ein Verzeichnis suchen.
