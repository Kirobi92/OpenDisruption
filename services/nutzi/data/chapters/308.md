# Chargenumbuchung 
Chargenumbuchung 

Der Dialog Chargenumbuchung kann in den Einkauf- und Verkaufbelegen über Lager >> Chargenumbuchung geöffnet werden. 

Hier wird der eingebuchte Bestand an Pseudo-Chargen auf die tatsächlichen Chargen umgebucht. Diese Umbuchung erfolgt als GEK-neutrale Lagerbuchung. 

Im Kopf des Dialogs finden Sie Selektionsfelder, um die gewünschten Chargen einzuspielen. Gleichzeitig ist eine Platzzuordnung über den Button Plätze möglich (es öffnet sich der Dialog Plätze Einlagerung ). Ist im Dialog Chargenumbuchung eine aufzulösende Charge selektiert, wird der maximale (Chargen-)Bestand im Feld Menge vorbelegt. Wenn nur eine Teilmenge umgebucht (aufgelöst) werden soll, kann dieser Wert angepasst werden. Hier kann nun auch die Ursprungscharge eingegeben werden. 

Beim Schließen des Dialogs wird sichergestellt, dass die rechte Tabelle keine ungebuchten Daten mehr enthält. 

Siehe Kapitel Pseudo Charge . 

Siehe Kapitel Plätze Einlagerung .

### Checkbox Ursprungsmenge neu = Menge 
Wenn Sie diese Checkbox aktivieren, wird die Ursprungsmenge des neu entstehenden Chargensatzes mit dessen Bestandsmenge gleichgesetzt. Es entsteht eine neue Ursprungscharge. Andernfalls verbleibt der alte Ursprungswert auf der neuen Charge und es wird damit sichtbar, dass es sich bei der durch den Split neu entstandenen Zielcharge nicht mehr um eine 'Ursprungs-Charge' handelt. 

Tooltip: 

Ist dieser Schalter aktiviert, so wird als Menge für die Umbuchung immer mit der Menge der ursprünglichen Charge durchgeführt.
