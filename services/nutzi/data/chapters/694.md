# Artikel Registerkarte Gefahrgut 
Artikel Registerkarte Gefahrgut 

Auf dieser Registerkarte können Sie sogenannte Gefahrgut-Informationen zum Artikel definieren. 
### Feld EG-Nr. 
Geben Sie für die Gefahrgutverwaltung die EG-Nummer ein. 

### Feld CAS Nummer 
Für die Gefahrgutverwaltung kann die CAS-Nummer hinterlegt werden. 

### Feld Unfallmerkblatt 
In diesem Feld kann für die Gefahrgutverwaltung die Bezeichnung des Unfallmerkblatts hinterlegt werden. 

### Feld Hilfe-Merkblatt 
Hinterlegen Sie die Bezeichnung des Hilfemerkblatts für die Gefahrgutverwaltung. 

### Feld Flammpunkt 
Falls erforderlich, geben Sie zu einem Artikel den Flammpunkt für die Gefahrgutverwaltung ein. 

### Feld WGK 
In diesem Feld kann die Wassergefährdungsklasse hinterlegt werden. 

### Combobox Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Feld ADR Technische Beschreibung 
In diesem Feld kann eine technische Beschreibung angegeben werden, die auf dem Beförderungspapier bei Angabe „N.A.G.“ in Klammern angedruckt wird. 

Das Feld Technische Beschreibung und die Combobox ADR Transporttyp werden nur angezeigt werden, wenn ADR lizenziert ist. 

Tooltip: 

Angabe der technischen Beschreibung. 
Es handelt sich um eine anerkannte chemische oder biologische Beschreibung oder eine andere Beschreibung, die üblicherweise verwendet wird. 

### Combobox ADR Transporttyp 
Für die Erfassung der Auftragsposition kann hier der ADR Transporttyp hinterlegt werden. Es stehen verschiedene Einträge zur Verfügung: 

- Kein ADR - ADR Transport nach Punkten - EQ - LQ 
Die mögliche Auswahl ergibt sich aus der Zuordnung des Artikels zu einer UN-Nummer der ADR-Tabelle A. Ist keine Zuordnung vorhanden, so kann der ADR-Transporttyp nur Kein ADR sein, ansonsten ergibt sich die mögliche Auswahl anhand der UN-Nummer. 

Bei einer Änderung der ADR-Tabelle A bzw. der Zuordnung von Artikel zur ADR-Tabelle A erfolgt eine Prüfung, ob die zugeordneten Daten im Artikelstamm noch zulässig sind. Bei einem negativen Ergebnis erhalten Sie eine entsprechende Meldung mit den Artikeln, welche so nicht mehr zulässig wären. 

Bei einem negativen Ergebnis erfolgt automatisch dazu parallel eine Umstellung der betroffenen Artikel auf die restriktivste Berechnungsmethode. Das heißt, ist der Artikel nicht mehr EQ-fähig, wird umgestellt auf ADR Transport nach Punkten. Gibt es gar keine ADR-Zuordnung mehr, wird im Artikelstamm automatisch Kein ADR gesetzt. 

Achtung: Es gibt keinen Automatismus, welcher bestehende Auftragspositionen aktualisiert. 

Tooltip: 

Art des ADR Transport 
1 - Kein ADR 
2 - ADR Transport nach Punkten 
3 - Freigestellte Mengen (EQ) 
4 - Begrenzte Mengen (LQ)
