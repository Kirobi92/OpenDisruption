# Replikationsdefinition Registerkarte Allgemein 
Replikationsdefinition Registerkarte Allgemein 

Auf dieser Registerkarte werden die Daten festgelegt. 
### Tabelle 

### Tabellenspalte Spalte 
Der Name der Spalte wird angegeben. Die Spalten werden automatisch anhand der ausgewählten Tabelle gefüllt. Sie kann nicht bearbeitet werden. 

### Tabellenspalte Neu 
Wenn die Checkbox in dieser Spalte aktiviert ist, wird die Spalte neu repliziert. 

### Tabellenspalte Insert 
Wenn die Checkbox in dieser Spalte aktiviert ist, wird der Wert beim Einfügen eines neuen Datensatzes repliziert. 

Wenn es sich bei der Spalte um einen Primärschlüssel handelt, dann wird sie in jedem Fall repliziert - unabhängig davon, was Sie hier angegeben haben. 

Sobald ein Ausdruck für „Allgemein" oder „Insert" definiert wurde, wird diese Checkbox automatisch aktiviert. 

### Tabellenspalte Update 
Diese Checkbox gibt an, ob der Wert beim Ändern eines Datensatzes repliziert werden soll. 

Sobald ein Ausdruck für "Allgemein" oder "Update" definiert wurde, wird diese Checkbox automatisch aktiviert. 

### Tabellenspalte Ausdr. 
Wenn Ausdrücke für die Replikations-Entscheidung definiert sind, dann wird in der Tabellen-Spalte "Ausdr." ein "X" angezeigt. 

Bei der Auswahl einer Tabellen-Spalte werden rechts neben dem Grid die entsprechenden Ausdrücke angezeigt. Diese können von dort aus auch mit den entsprechenden Buttons bearbeitet werden. 

### Tabellenspalte Transform 
Wenn Ausdrücke für die Transformation definiert sind, dann wird hier ein "X" angezeigt. 

Bei der Auswahl einer Tabellen-Spalte werden rechts neben dem Grid die entsprechenden Ausdrücke angezeigt. Diese können von dort aus auch mit den entsprechenden Buttons bearbeitet werden. 

### Feld Allgemein 
Das ist ein allgemeingültiger Ausdruck, der immer verwendet wird. Dieser Ausdruck wird angewendet, um zu prüfen, ob die Spalte repliziert werden soll. 

Das Ergebnis dieses Ausdrucks muss "bool" ( Ja oder Nein ) sein. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor . 

### Feld Insert 
Dieser Ausdruck wird verwendet, um zu prüfen, ob die Tabellen-Spalte beim Einfügen des Datensatzes repliziert werden soll. 

Wenn zusätzlich der ‚Allgemeine Ausdruck‘ definiert wurde, dann wird die Tabellen-Spalte nur repliziert, wenn beide Ausdrücke "wahr" liefern. 

Das Ergebnis dieses Ausdrucks muss "bool" ( Ja oder Nein ) sein. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor . 

### Feld Update 
Dieser Ausdruck wird verwendet, um zu prüfen, ob die Tabellen-Spalte beim Ändern des Datensatzes repliziert werden soll. 

Wenn zusätzlich der ‚Allgemeine Ausdruck‘ definiert wurde, dann wird die Tabellen-Spalte nur repliziert, wenn beide Ausdrücke "wahr" liefern. 

Das Ergebnis dieses Ausdrucks muss "bool" ( Ja oder Nein ) sein. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor . 

### Feld Insert 
Mit diesem Ausdruck können Sie für die Tabellen-Spalte einen anderen Wert berechnen. 

Diese Berechnung erfolgt beim Einfügen des Datensatzes und, wenn in "Transform Upd." nicht anders definiert, auch beim Ändern des Datensatzes. 

Das Ergebnis dieses Ausdrucks muss dem Datentyp der Tabellen-Spalte entsprechen. 

Dem Expression-Builder wird der Typ der Tabellen-Spalte übergeben. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor . 

### Feld Update 
Mit diesem Ausdruck können Sie einen speziellen Ausdruck für den Update-Vorgang definieren. Wenn dieser Ausdruck nicht angegeben ist, dann wird derselbe Ausdruck wie beim Insert-Vorgang verwendet. 

Das Ergebnis dieses Ausdrucks muss dem Datentyp der Tabellen-Spalte entsprechen. 

Dem Expression-Builder wird der Typ der Tabellen-Spalte übergeben. 

### Button Öffnen 

Mit diesem Button öffnet sich ein Bearbeitungs-Dialog für die Expressions. 

Siehe Kapitel Ausdruckseditor .
