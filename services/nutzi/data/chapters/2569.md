# Bankeinzug Registerkarte Selektionen 3 
Einzugsvorschlag Registerkarte Selektionen 3 

Gibt es in einem Unternehmen mehrere Sachbearbeiter, die Bankeinzüge ausführen können, so kann über das Auswahlfeld Gebiet auf dieser Registerkarte, das gewünschte Vertriebsgebiet ausgewählt werden. So kann jeder Sachbearbeiter seine zugewiesenen Vertriebsgebiete bearbeiten. Aus dem Feld können auch mehrere Einträge ausgewählt werden. 

Für einen bereits angelegten Bankeinzugsvorschlag bleibt die Selektion gespeichert. 

Diese Selektion wird auch im Mahnlisten-Druck angezeigt. 
### Tabelle 

In dieser Tabelle kann für die Anzeige noch das Gebiet / Bundesland als Selektionskriterium gewählt werden.
