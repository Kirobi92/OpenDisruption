# Registerkarte Tabelle 
Registerkarte Tabelle - Beispiel 

Die Registerkarte Tabelle beinhaltet immer alle Datensätze, die den vorgegebenen Suchkriterien entsprechen. Ist der Parameter Automatisch Tabelle zeigen im Dialog Parameter Allgemein aktiviert, wird die Registerkarte automatisch geöffnet, wenn es mehr als einen Suchtreffer gibt. Per Doppelklick auf den gewünschten Datensatz werden die Daten in die entsprechenden Felder des Dialogs eingelesen und die Registerkarte Allgemein aktiviert. Z.B. werden im Kundenstamm alle Kunden und entsprechende Ansprechpartner eingelesen. 

In sämtlichen Stamm-Dialogen wird die Suche nach Datensätzen aufgrund von Ladezeiten eingeschränkt. Dazu steht Ihnen das Feld Max. Anzahl auf der Registerkarte Tabelle zur Verfügung. Die maximale Anzahl wird im Dialog Parameter Allgemein eingestellt, sie kann aber hier geändert werden, um bei einem erneuten Laden der Daten die Ergebnismenge entsprechend vorzugeben. 

Wenn in den Stamm-Dialogen die Registerkarte Tabelle gewählt wird, wird der Button automatisch inaktiv, da nicht genau ersichtlich ist, welches der aktuelle Datensatz ist. 

Siehe Kapitel Parameter Allgemein . 

## Sortierung der Spalten / Filterung 
Alle Tabellen in eNVenta unterstützen die Client-basierende Sortierung / Filterung. 

Bei der Client-basierenden Sortierung wird die Reihenfolge durch die Tabelle direkt im Client erreicht. Hierzu kann der Sachbearbeiter nach dem Einlesen der Daten mittels Klick auf den Spaltenkopf die Sortierung in der Tabelle verändern. Das funktioniert für eine Spalte aber nur, wenn alle Datensätze an den Client übertragen wurden. Durch die Lade-Optimierung von eNVenta werden die Datensätze immer nur "portionsweise" übertragen. Das schont die Client- und die Server-Ressourcen. 

Hierbei handelt es sich um ein eNVenta Standardverhalten, damit eine ressourcenschonende und möglichst performante Benutzung von Tabellen möglich ist. Der Nachteil ist, wenn nicht alle Daten an den Client übertragen wurden, z.B. weil es mehr als 500 Einträge gibt, so ist eine Sortierung und Filterung direkt in der Tabelle nicht möglich. Eine Filterung würde das Ergebnis verfälschen, wenn z.B. per Excel-Export die Daten weitergegeben werden. 

Beispiel: 
Bei ungefilterten Inventuren kann dies den Client in die Überlastung führen (Performance und Speicher), deshalb raten wir dringend davon ab. 
### Für die Benutzung gibt es nun mehrere Möglichkeiten: 
- Nach unten Scrollen damit alle notwendigen Daten nachgeladen werden - jedes Scrollen lädt die nächste "Portion" (zumeist Schritte von 50 Datensätzen) an Daten vom Server. Wird z.B. direkt an das Ende der Tabelle gesprungen, werden alle Daten dazwischen automatisch nachgeladen. - Reduzierung der Suchergebnisse durch Verbesserung der Suchkriterien. Dies ergibt eine spezifischere Suche. - Anpassung der maximalen Anzahl in der Registerkarte Tabelle (Feld Max. Anzahl ), wenn vorhanden. Durch die Höhe der max. Anzahl Treffer werden ggf. Suchergebnisse unterdrückt und daher dann ist ggf. zwingend die Zahl hochzusetzen. 
### Vorgehensweise: 
Um zu sortieren bzw. zu filtern, klicken Sie mit der rechten Maustaste in den Spaltenkopf. Es öffnet sich ein Dialog, in dem Sie über die Sortierkriterien aufsteigend , absteigend und unsortiert sortieren oder zusätzlich, mit Auswahl eines Filters, die Sortierreihenfolge noch genauer spezifizieren können. Das Vorhandensein eines Filters wird mit einem Symbol (blaues Dreieck) im Tabellenkopf angezeigt. 

Außerdem können weitere, zusätzliche Filterungen aktiviert werden. Dazu gehen Sie auf die nächste, benötigte Spalte und machen mit der rechten Maustaste einen Doppelklick und wählen die entsprechende Sortierung oder Filterung aus. 

Eine Schnellsortierung ist möglich per Klick auf den Spaltenkopf. Hierbei gibt es nur die einfache Sortierung auf dieser gerade angeklickten Spalte. Bei mehrfachem Klicken ändert sich die Sortierreihenfolge bzw. die Sortierung wird wieder deaktiviert. 

Die Sortierung ist immer nur zur Laufzeit des Dialogs aktiv. Die Sortierung wird nicht dauerhaft gespeichert. 

Mit der Auswahl Filter zurücksetzen wird Ihre Filterauswahl wieder zurückgenommen.
