# Kuchendiagramm Registerkarte Allgemein

Allgemeine Funktionalitäten in eNVenta ERP / Kuchendiagramm 

# Kuchendiagramm Registerkarte Allgemein 
Kuchendiagramm Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie das Aussehen des Kuchens. 
### Feld Zeilenindex 
Soll das Kuchendiagramm Werte einer bestimmten Zeile darstellen, geben Sie hier die entsprechende Zeilennummer ein. 

Standardmäßig ist „0“ eingetragen, das Kuchendiagramm wird in diesem Fall über alle Spalten berechnet und angezeigt. 

### Feld Höhe, % (3D) 
Geben Sie an, wie „hoch“ der Kuchen sein soll. 

Tooltip: 

Kuchen Höhe, % (3D) 

### Feld Andere, % 
Alle Anteile unterhalb dieser Prozentangabe werden zu „Andere“ zusammengefasst. 

### Feld Start Winkel, Gr. 
Geben Sie den Winkel der Drehung des Diagramms an. 

### Feld Abbr. Abstand, % 
Geben Sie den Abstand in % ein, den die einzelnen Kuchenstücke voneinander haben sollen. 

Tooltip: 

Abbruch Abstand, % 

### Checkbox Alle Stücke herausbrechen 
Wenn Sie möchten, dass Kuchenstücke etwas voneinander abgesetzt werden, aktivieren Sie diese Checkbox. 

### Checkbox Stücke abwechselnd herausbrechen 
Wenn beispielsweise nur jedes 2. Kuchenstück herausgebrochen werden soll, setzen Sie diese Checkbox. 

### Checkbox Andere herausbrechen 
Wenn Werte in diesem Kuchen herausgebrochen werden sollen, die mit ‚Other‘ bezeichnet sind (d.h. sie sind eigentlich zu gering, um erwähnt zu werden), setzen Sie diese Checkbox.
