# Anfrage Registerkarte Kopfdaten 
Anfrage Registerkarte Kopfdaten 

Auf dieser Registerkarte geben Sie die notwendigen Daten, wie z.B. die Zahlungsart, die Art der Lieferung, die Versandart etc. ein. Die Daten in diesem Register sind mit den Einträgen aus dem Lieferantenstamm vorbelegt. 
### Feld Projekt 
Wenn der Vorgang zu einem Projekt gehört, geben Sie hier die Projektnummer an. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Button Suche 

Mit diesem Button können Sie ein Projekt suchen. 

Siehe Kapitel Projektauswahl . 

### Button Verzweigen 

Mit diesem Button gelangen Sie in die Projektverwaltung. Dort sind bereits alle Informationen zur gewählten Projektnummer voreingestellt. 

### Combobox Lieferung 
Hier wird angegeben, wie der Kunde beliefert werden soll (z.B. frei Haus oder ab Werk). Diese Einträge werden im Dialog Lieferbedingung gepflegt. 

Siehe Kapitel Lieferbedingung . 

Tooltip: 

Gibt die gewünschte Lieferbedingung an. 

### Combobox Versand 
Hier wird die Nummer der Versandart angegeben. 

Tooltip: 

Gibt die gewünschte Versandart an. 

### Checkbox Komplettlieferung gewünscht 
Wenn diese Checkbox gesetzt ist, ist eine Komplettlieferung gewünscht. 

Mit Setzen dieser Checkbox wird der Lieferant über den Wunsch nach Komplettlieferung in Kenntnis gesetzt. 

Siehe auch Kapitel Bestellungen Registerkarte Kopfdaten . 

Tooltip: 

Der Lieferant soll darüber informiert werden, dass eine Komplettlieferung gewünscht wird. 

### Feld Valutadatum 
In diesem Feld können Sie ein Valutadatum für die Rechnungsprüfung eintragen bzw. vorbelegen. Ab dem eingetragenen Valutadatum beginnt die Fälligkeit gemäß Zahlungsvereinbarung zu laufen. Wenn dies nun für eine Bestellung in 60 Tagen ist, dann tragen Sie dort ' *+60 ' ein. 

Tooltip: 

Geben Sie ein Valutadatum für die Rechnungsprüfung an. 

### Combobox Zahlung 
Hier wird die Zahlungsbedingung angegeben. Es werden Nummer und Text angezeigt. 

Tooltip: 

Gibt die gewünschte Zahlungsbedingung an. 

### Checkbox Keine Zentralregulierung 
Diese Checkbox ist nur sichtbar, wenn es sich bei dem Lieferanten um einen Verbandslieferanten handelt. 

Das Kennzeichen hat folgende Wirkung: 

- Der OP wird im Zahlungslauf für den Verbandslieferanten berücksichtigt. - Der OP wird von der Verbandsabrechnung ausgeschlossen. 

Tooltip: 

Keine Zentralregulierung (Verbandsabrechnung bzw. Zentralzahlung) 

### Feld Fracht 
Tragen Sie hier die Frachtkosten ein (vgl. Porto). 

Tooltip: 

Angabe der Frachtkosten 

### Feld Nebenkosten 
Hier können Nebenkosten eingetragen werden. 

Tooltip: 

Angabe der Nebenkosten 

### Checkbox Nicht mahnen 
Wenn Sie diese Checkbox aktivieren, wird der entsprechende Artikel/Lieferant im Mahnvorschlag des Einkaufs nicht berücksichtigt; d.h. nicht gemahnt. 

Dies macht dann Sinn, wenn z.B. Frachtkosten über eine Artikelnummer definiert sind. Diesen Artikel kann der Lieferant ja nicht liefern. 

Tooltip: 

Wenn aktiviert, wird der Vorgang nicht angemahnt. 

### Checkbox Keine WBZ aktualisieren 
Wenn die Bestellung bei der automatischen Berechnung der Wiederbeschaffungszeit nicht berücksichtigt werden soll, setzen Sie diese Checkbox. Gründe dafür können sein, dass z.B. Eilbestellungen oder Lieferschwierigkeiten eines Lieferanten die Wiederbeschaffungszeit verfälschen würden. 

Siehe Kapitel Wiederbeschaffungszeit . 

Tooltip: 

Wenn aktiviert, wird die WBZ nicht aktualisiert. 

### Checkbox Kein EDI 
Ist diese Checkbox gesetzt, wird der Versand per EDI nicht gestattet. 

Tooltip: 

Wenn aktiviert, wird der Versand per EDI nicht gestattet. 

### Checkbox Kein Intrastat 
Wenn Sie diese Checkbox aktivieren, erfolgt keine Berücksichtigung in der Intrastat. 

Wenn die Checkbox nicht gesetzt ist, ist zwingend ein Land anzugeben. Voraussetzung: Dieses Land ist in der Ländertabelle für Intrastat freigegeben. 

Tooltip: 

Wird nicht in der Intrastat berücksichtigt, wenn Checkbox aktiv ist. 

### Combobox Währung 
Hier können Sie die Währung festlegen. Wurde zum Kunden / Lieferanten bereits ein Vorgang erfasst oder eine Buchung vorgenommen, kann die Währung nicht mehr geändert werden. 

Wird die Währung geändert, so wird auch das zugehörige Sammelkonto automatisch angepasst. 

Wird hier nichts eingetragen, wird automatisch die Systemwährung eingetragen. 

Tooltip: 

Angabe der Währung 

### Feld Kurs 
Hier wird der Kurs angegeben. 

Der Wechselkurs wird durch das Programm aus der Währungstabelle vorbelegt. 

Tooltip: 

Währungskurs 

### Combobox MwSt.-Kz 
Dieses Feld legt die Steuerkategorie fest: 

- Inland - EU - Ausland 
In Abhängigkeit dieses Kennzeichens werden bei den Vorgängen (Angebot, Auftrag, Gutschrift) der entsprechende Steuerschlüssel und das zugehörige Erlöskonto voreingestellt. In der Buchhaltung wird der richtige Steuerschlüssel ermittelt. 

Tooltip: 

MwSt.-Kennzeichen des Lieferanten wird angezeigt. 

### Combobox Sprache 
Hier wird die gewünschte Sprache angegeben. 

Tooltip: 

Gibt die gewünschte Sprache an. 

### Combobox Stammtour 
Hier geben Sie an, mit welcher Tour (Route) (z.B. täglich, wöchentlich) der Kunde bzw. Lieferant immer (regelmäßig) beliefert werden soll. Die Touren, die zur Verfügung stehen, werden im Tourenstamm gepflegt. 

Siehe Kapitel Tourenstamm . 

Tooltip: 

Angabe der Stammtour des Lieferanten 

### Combobox Dispo-Kreis 
Aus dieser Combobox wählen Sie den Dispositionskreis zur regionalen Abgrenzung. Die Anlage der Dispositionskreise erfolgt in einer Pickliste. 

Tooltip: 

Der Dispositionskreis des Lieferanten wird angezeigt.
