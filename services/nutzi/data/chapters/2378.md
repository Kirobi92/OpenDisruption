# OP Anlage 
OP Anlage 

Dieser Dialog öffnet sich im Dialog Ausbuchung über den Button OP . Hier tragen Sie die Daten für eine OP-Anlage des Restbetrags ein. 

Unmittelbar beim Anlegen eines neuen OPs können ein (oder mehrere) Sperrkennzeichen und der Grund für die Sperrung gesetzt werden. Dazu stehen folgende Funktionalitäten zur Verfügung: - Combobox Sperrgrund und die Checkboxen - Einzugssperre - Mahnsperre - Kein Zinslauf - Kein Kontoauszug - Kein Monatsaufsteller - Keine D. berech. 
Diese Informationen werden anschließend in den Dialogen Offene Posten Kreditor , Offene Posten Debitoren , Ausbuchung und Bonität angezeigt und können auch zur Selektion genutzt werden.

### Button Speichern 

Mit diesem Button bestätigen Sie die eingetragenen Daten für die OP-Anlage. 

Nach bestätigen mit dem Button gelangen Sie wieder in den Dialog OP -Ausbuchung und wenn der Restbetrag auf 0,00 steht, kann die Buchung über den Button endgültig abgeschlossen werden. 

Wird eine OP-Zuordnung nicht mit dem Button abgeschlossen, werden alle Zuordnungen nach betätigen des Buttons aufgehoben.

### Tabelle OP-Zuordnung 

Im vorausgehenden Dialog OP-Ausbuchung bebuchte OPs werden hier aufgeführt. Wenn Sie einen Satz markieren, erhält diese OP Anlage die OP-Daten Steuerschlüssel, Kostenstelle, Kostenträger, Projektnummer des markierten Satzes vererbt. Daher werden ggf. diese Felder im Dialog geleert. Das Feld Ausbuchung wird ebenfalls geleert, weil an dieser Stelle eine Skonto-Buchung nicht vorgesehen ist. 

Der Bereich OP-Zuordnung wird nur sichtbar, wenn im Vorgänger-Dialog OP-Ausbuchung (DA) OPs ausgebucht wurden. 

Tooltip: 

Aus den selektierten Belegen werden relevante Daten in den neuen OP übernommen. 
(z.B. Kostenstelle, Steuerschlüssel)

### Feld Zahlungsnr. 
Hier geben Sie die Zahlungsnummer ein, die im Auftrag manuell für die Zahlungsart Vorkasse generiert worden ist. Die Liefefreigabe des Auftrags wird im Fall von Vorkasse erst entfernt, wenn die Buchung komplett abgeschlossen ist. 

Nach Eingabe der Zahlungsnummer bei der OP Anlage wird auf Basis der Zahlungsnummer nach der entsprechenden Auftragsnummer gesucht und der Auftrag auf Sperrstatus 0 (Normal) gesetzt werden. Dies wird auch im Dialog OP Ausbuchung in der Registerkarte OP-Anlagen in der Spalte ‚Auftrag freigegeben‘ angezeigt bzw. kann dort auch vor Abschluss der OP-Ausbuchung wieder manuell entfernt werden. 

Die definitive Freigabe des Auftrags findet mit Drücken des Buttons statt. Falls der Anwender anschließend die Buchung abbricht, bleibt der Auftrag freigegeben! 

Siehe Kapitel OP Ausbuchung . 

Tooltip: 

Die Zahlungsnummer (im Auftrag bei Zahlungsart Vorkasse generiert) erzeugt ggf. im Auftrag die Lieferfreigabe.
