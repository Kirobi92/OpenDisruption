# Parameter Drag and Drop 
Parameter Drag Drop 

eNVenta bietet auch die Möglichkeit externe Dateien (außerhalb von eNVenta ) nach eNVenta zu ziehen. Dies können Word-Dokumente sein, welche in das Archiv gestellt werden sollen oder auch URL-Links, welche als Referenzlink zum Vorgang gehören. 

In diesem Zusammenhang steht der Dialog Parameter Drag Drop zur Verfügung. Diesen finden Sie im Navigator >> System >> Allgemein >> Parameter Drag Drop . Hier können Sie die verschiedenen Drop-Aktionen abschalten. 

Dies funktioniert nur mit Windows-Rechnern. 

In allen Dialogen, in denen die Funktion Archiv-Mappe einfügen vorhanden ist, wird beim Ziehen eines externen Dokuments die Drop-Funktion Archiv-Mappe einfügen oder Referenz Link einfügen/erstellen angeboten. Dazu erhalten Sie beim Ziehen des Dokuments in die eNVenta -Anwendung eine Auswahl angezeigt. Sollte aufgrund der Parameter nur eine Einstellung zur Auswahl stehen, erscheint keine Auswahl und die Aktion wird sofort ausgeführt. 

Im Zusammenhang mit der globalen Suche können auch sogenannte Objekte in Vorgänge zur Weiterverarbeitung übernommen werden. 

Beispiele: 

- Artikelobjekte können als Positionen in Vorgänge eingebunden sein (z.B. Auftrag, Angebot, Gutschrift, Retouren, Anfragen, Bestellungen und Belastungen). - Ein Kundenobjekt kann in das Feld Kundennr . eines entsprechenden Vorgangsdialogs übernommen werden (z.B. Retoure, Auftrag). - Ein Lieferantenobjekt kann in das Feld Lieferantennr . eines entsprechenden Vorgangsdialogs übernommen werden (z.B. Anfrage, Bestellung). - Ein Vorgang (z.B. Auftrag) kann zur Erstellung einer Retoure in das entsprechende Feld im Dialog Retoure übernommen werden. - Objekte können in Selektionsfelder von Statistiken übernommen werden. - Objekte, wie z.B. Auftrag und Angebot können auf ein Projekt gezogen werden. - Kunden, Lieferanten und Adressen können in einem Projekt in die Tabelle Beteiligte gezogen werden. - Kunden, Projekte und Lager können in einen Rahmenauftrag gezogen werden; ebenso Artikel zur Erfassung in der Positionstabelle. - Objekte, wie z.B. Artikel, Kunde, Kundengruppe und Projekt können in den Rabatt Verkauf gezogen werden.

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
