# Server 
Job-Server Übersicht 

Dieser Dialog öffnet sich über den Navigator >> Registerkarte Jobs >> System >> Server . 

Wenn Sie einen Job-Server installieren, müssen Sie dies in den folgenden Schritten tun: 

- Job-Server publishen - die Job-Server Konfiguration öffnen und die Einstellungen vornehmen - den Haken in der Checkbox Deaktiviert entfernen - speichern - Job-Server mit dem Button Neustart neu starten 
Hier werden alle verfügbaren Job-Server angezeigt. Diese Job-Server werden bereits beim Publish (= der Veröffentlichung) von FS festgelegt. 

Dazu finden Sie ausführliche Informationen in der Dokumentation zu Framework Studio. 

Es können beliebig viele Job-Server installiert werden, allerdings müssen die Job-Server auf verschiedene Windows-Server verteilt werden (d.h. immer nur ein Job-Server pro Server!). 

Wenn Sie die Checkbox in der Spalte ‚Deaktiviert‘ anklicken, wird die entsprechende Checkbox in der Job-Server Konfiguration deaktiviert. Damit wird der Job-Server abgeschaltet. 

Siehe Kapitel Job-Server Konfiguration . 

Der Job-Server arbeitet immer in 2 Stufen. Zuerst wird ein Job eingeplant, inkl. Zeitplan, Abhängigkeiten usw. Anschließend wird der erstellte Plan (Instanzen) der Jobs vom Job-Server abgearbeitet.

### Tabelle Der Job-Server 

In dieser Tabelle werden alle verfügbaren Job-Server angezeigt. Diese Job-Server werden bereits bei der Veröffentlichung der Anwendung festgelegt. 
### Tabellenspalte Server 
In dieser Spalte wird der zugeordnete Servername angezeigt. 

### Tabellenspalte Deaktiviert 
Wenn Sie die Checkbox in dieser Spalte aktivieren, wird der Job-Server deaktiviert. 

### Tabellenspalte Gruppe 
Zusätzlich können Sie in dieser Spalte den Job-Server einer Gruppe zuordnen, damit alle Jobs für diesen Server automatisch diese Gruppenzuordnung erhalten. In diesem Fall können alle Job-Server dieser Gruppe den Job bearbeiten. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Gruppenlose Jobs 
Wenn Sie die Checkbox in dieser Spalte aktivieren, kann der Job-Server zusätzlich auch Jobs ausführen, welche keiner Gruppe zugeordnet sind. 

Geben Sie immer mindestens einen Job-Server an, der gruppenlose Jobs verarbeiten kann. 

Tooltip: 

Der Job-Server führt auch gruppenlose Jobs aus. 

### Tabellenspalte Verarb.-Einheiten 
In dieser Spalte wird die Anzahl der Einheiten angezeigt, d.h. die zu verarbeitenden Jobs. 

### Tabellenspalte Akt. Jobs 
Hier werden die aktiven Jobs angezeigt, d.h. diejenigen die im Moment verarbeitet werden. 

### Tabellenspalte Letzte Prüfung 
Hier ist das Datum der letzten Prüfung des Jobs eingetragen. 

### Tabellenspalte Status 
In dieser Spalte wird der augenblickliche Status des Job-Servers angezeigt, damit evtl. vorhandene Probleme gleich sichtbar sind.
