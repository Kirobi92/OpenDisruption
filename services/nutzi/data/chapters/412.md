# Parameter ELSTER 
Parameter ELSTER 

Im Navigator >> System >> Administration steht der Eintrag ELSTER zur Verfügung. Damit öffnet sich ein Dialog, um Parameter für das ELSTER-Verfahren zu definieren. 

Der NVERiC-Dienst hat zwingend installiert zu sein, wenn Sie die Abgabe der UVA aus eNVenta heraus vornehmen möchten. 

eNVenta übergibt die Daten an den NVERIC-Dienst und dieser übermittelt die Daten automatisch an das Finanzamt. 

### Protokollierung der ELSTER-Sendenachweise 
Für Steuerdaten, die per ELSTER erstellt und an das Finanzamt gesendet werden, kann die gesamte Datenkommunikation automatisch in einem Sendeprotokoll gespeichert werden. Dies betrifft sämtliche vom Finanzamt zur Verfügung gestellten ELSTER-Komponenten, d.h. die Ermittlung der Finanzämter, der UVA und der ZM. 

Ob die Dateien von ELSTER abgelegt und dazu noch protokolliert werden, hängt von der Log-Stufe ab. Dafür steht der Dialog Konfiguration-Protokoll (unter System >> Administration >> Protokollierung >> Konfiguration Protokollieren in eNVenta ) zur Verfügung. Siehe nachfolgende Abbildung.

### Button Datenschutz 

Im Zusammenhang mit der gültigen DSGVO gibt es die Funktion für die Datenschutzvereinbarung. Dies ist ein allgemeines Informationsschreiben über die Verarbeitung personenbezogener Daten im Besteuerungsverfahren durch die Steuerverwaltung. 

Über den Button Datenschutz können Sie das Dokument öffnen. 

Damit werden Kunden bzw. Unternehmen über die Informationsweitergabe unterrichtet. 

Bei der Installation öffnet sich einmalig die Datenschutzerklärung (https://download.elster.de/download/dokumente/Informationen_zu_Artikel_12_bis_14_Datenschutz-Grundverordnung.pdf). 

Beim Speichern einer Konfiguration mit einem vorhandenen Service Host kommt eine Abfrage, ob der Benutzer die Datenschutzvereinbarung von ELSTER akzeptiert oder nicht. Nur wenn sie akzeptiert wird, wird die Konfiguration gespeichert. Dadurch wird sichergestellt, dass eNVenta -seitig keine neue ELSTER-Konfiguration gespeichert werden kann, ohne dafür die Datenschutzvereinbarung zu akzeptieren. 

Tooltip: 

Öffnet die ELSTER Datenschutzvereinbarung im Web-Browser.

### Checkbox ELSTER-Kommunikation ausgeben 
Wenn Sie diese Checkbox aktivieren, wird eNVenta -seitig die Kommunikation mit dem ELSTER-Service protokolliert. Damit können die Fehlersuche für abgeschickte Anfragen an ELSTER und die dazu gehörenden Antworten nachvollzogen werden. Diese Daten werden protokolliert. 

Die Protokoll-Funktion kann per App-Parameter eingeschaltet werden. Zusätzlich werden die Verzeichnisse angegeben, in die das Request-/Response-XML protokolliert wird. 

Tooltip: 

Ausgabe der ELSTER-Kommunikation
