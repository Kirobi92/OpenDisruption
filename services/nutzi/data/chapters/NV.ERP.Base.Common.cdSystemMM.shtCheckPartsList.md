# Kapitel NV.ERP.Base.Common.cdSystemMM.shtCheckPartsList

Wenn Sie diesen Parameter aktivieren, erfolgt bei manueller Lieferfreigabe von bereits gespeicherten Positionen bei Handelsstücklisten eine Verfügbarkeitsprüfung für die Komponenten. Komponenten mit Fehlmengen werden in einer Liste angezeigt.
