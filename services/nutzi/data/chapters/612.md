# Update-Skripte Registerkarte Befehle 
Update-Script Registerkarte Befehle 

Wenn Sie in der Tabelle einen Eintrag markieren, werden die Befehle angezeigt, wie sie in eNVenta hinterlegt sind. Diese können manipuliert werden und mit dem Button Setzen werden diese gespeichert. 
### Button Setzen 

Mit diesem Button speichern Sie die manipulierten Befehle. 

### Feld Befehlsliste 
In diesem Feld werden die Befehle aufgelistet. 

Neue manuelle Befehle beginnen mit 'SQL-Server' bzw. 'Oracle'.
