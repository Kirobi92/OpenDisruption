# Kapitel NV.ERP.Base.Common.cdSystemMM.shtArticleSearchCustomer

Wenn Sie diese Checkbox aktivieren, steuern Sie die Suche nach Artikeln im System gezielt über die Kundenartikelnummer. 

Tooltip: 

Kennzeichen, ob bei der Artikelsuche die Kunden-Artikelnummer verwendet werden soll.
