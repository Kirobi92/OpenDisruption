# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtNV4_GenerateBacklockReminder

Wird durch das LVS automatisch eine Rückstandsposition erzeugt (Anzahl gepickt < benötigte Anzahl), wird normalerweise für den Erfasser des Auftrags eine Wiedervorlage zur Information erzeugt und gesendet. Wenn Sie dies verhindern wollen, aktivieren Sie diese Checkbox. 

Tooltip: 

Mit diesem Parameter wird der automatische Versand einer Wiedervorlage an den Erfasser des Auftrags verhindert.
