# Versionierung 
Versionierung 

Dieser Dialog öffnet sich im Dialog Einzeldruck über den Button Version . Hier können Sie festlegen, welche Belege versioniert abgelegt werden sollen. Wenn Sie einen neuen Datensatz anlegen (Button ), können Sie aus der Combobox in der Spalte ‚Reportname‘ einen Report auswählen, der standardmäßig in dieser Liste zur Verfügung steht. Wenn Sie dazu die Checkbox in der Spalte ‚Version. aktiv‘ anklicken, wird die Versionierung des gewählten Reports aktiviert. 

Der Button Konfig . oder ein Doppelklick auf einen Tabelleneintrag im Dialog Versionierung öffnet den Dialog Konfiguration Version , wo Sie die Einzelheiten für die Versionierung festlegen. 

Siehe Kapitel Konfiguration von Versionierungen .

Tooltip:

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Konfig. 

Dieser Button öffnet den Dialog Konfiguration Version , wo Sie die Einzelheiten für die Versionierung festlegen. 

Siehe Kapitel Konfiguration von Versionierungen .
