Allgemeine Funktionalitäten in eNVenta ERP / Lagerverwaltung 

# Seriennummernjournal 
Seriennummernjournal 

Der Button Seriennummernjournal im Dialog Seriennummern öffnet diesen Dialog. Ist im Dialog Seriennummern eine Zeile der Tabelle markiert, wird diese Seriennummer in das Feld Seriennummer übernommen. In der Tabelle sehen Sie die detaillierten Informationen zu dieser Seriennummer. 

Ist der Dialog leer und Sie geben eine Artikelnummer in das Feld Artikel ein, werden die dazugehörigen Seriennummern mit den Detailinformationen in die Tabelle eingespielt. 

Hier erhalten Sie eine Übersicht über die einer Seriennummer zugeordneten Aufträge und Vorgänge mit dem zugehörigen Status.

### Tabellenspalte Seriennr Status 
Hier wird der Status der Seriennummern angegeben. 

- Frei: Seriennummer ist erfasst und keinem Vorgang zugeordnet, z.B. aus dem Wareneingang oder dem Produktionsauftrag heraus. - Repariert (Lieferant sendet die Ware zurück): Ware, welche zuvor den Status Zurückgesendet hatte. Wird im Wareneingang wieder der Bestellung zugeordnet. - Ausgebucht: Verkauft an Kunden. Status wird bereits ab Zeitpunkt der Zuordnung zum Beleg gesetzt. - Zurückgesendet: Ware wurde an den Lieferanten zurückgesendet (via Belastung oder Materialbeistellungsauftrag). - Reklamiert: Kunde sendet die Seriennummer an Händler zurück (via Retoure). - Verbaut: Wenn die Seriennummer als Komponente im Betriebsauftrag zugeordnet wurde. - Objekt zugeordnet: Entfällt bzw. wurde ersetzt durch den Status Ausgebucht bei Verkauf bzw. Frei bei Wareneingang. - Zuordnung entfernt: Entfällt
