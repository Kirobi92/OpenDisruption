# Kalendertag 
Kalendertag 

Über den Button Kalendertag im Kopf des Dialogs Betriebskalender öffnet sich der Dialog Kalendertage . Ganze Tage können als arbeitsfrei definiert werden. Z.B. Feiertage. 

Dabei gilt folgende Hierarchie: 

- Angabe einer Ressource: gilt nur für die Ressource - Angabe einer Ressourcengruppe: gilt für alle Ressourcen dieser Gruppe. - Keine Angabe von Ressource oder Gruppe: gilt für alle Ressourcen. 
Es kann nur entweder Ressource oder Ressourcengruppe angegeben werden. 

Die Ressource für einen Kalendertag kann geändert werden, wenn alle relevanten Daten geladen sind (Kalendernr, Tag, Schicht). 

Bei Angabe einer Ressourcengruppe gilt die Einstellung nicht für Ressourcen einer Untergruppe. 

Wenn für einen Kalender der abweichende Kalendertag nicht angegeben ist, so kommt der entsprechende Eintrag aus Kalender 0 zur Anwendung, falls vorhanden. Somit genügt es z.B. einen Feiertag einmal in Kalender 0 festzulegen, um eine Wirksamkeit für alle Kalender zu erreichen.
