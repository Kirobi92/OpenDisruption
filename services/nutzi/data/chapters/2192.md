# Servicevertrag Registerkarte Allgemein 
Servicevertrag Registerkarte Allgemein 

Auf dieser Registerkarte legen Sie die allgemeinen Daten des Vertrags fest. 
### Feld Bezeichnung 
Geben Sie dem Vertrag einen Namen. 

Tooltip: 

Angabe eines Servicevertragsnamen 

### Feld Statusgrund 
Hier können Sie bei Bedarf einen Grund für die Statuswahl eintragen. 

Tooltip: 

Grund für die Wahl des Status 

### Combobox Bestellart 
Hier kann die Bestellart ausgewählt werden. Dazu gehören u.a. die Bestellarten Anfrage, per Mail, telefonisch . 

Tooltip: 

Geben Sie die Bestellart an. 

### Feld Bestellung 
Geben Sie hier die Bestellnummer ein. 

### Feld Bestelldatum 
Geben Sie hier ein Bestelldatum ein. 

Tooltip: 

Angabe des Bestelldatums 

### Feld Eigentümer 
Hier kann zusätzlich zum Vertragsnehmer der Eigentümer der Serviceobjekte (mit Firmenbezeichnung) hinterlegt werden. 

Tooltip: 

Angabe der Kundennummer 

### Button Suche 

Mit diesem Button können Sie Kunden suchen. 

Siehe Kapitel Kundenauswahl . 

### Button Verzweigen 

Mit diesem Button verzweigen Sie in den Kundenstamm. 

### Feld Anschrift 
Nachdem Sie den Eigentümer eingetragen haben, wird dessen Rechnungsanschrift hier angezeigt. 

Tooltip: 

Angabe der Anschrift 

### Feld Code 1 
Dieses Feld steht zur individuellen Eingabe zur Verfügung. Es kann per Reportprogrammierung eingebunden werden, hat aber keine Funktionalität. 

Tooltip: 

Feld zur individuellen Verfügung.

Tooltip: 

Feld zur individuellen Verfügung.

Dieses Feld steht zur individuellen Eingabe zur Verfügung. Es kann per Reportprogrammierung eingebunden werden, hat aber keine Funktionalität.
