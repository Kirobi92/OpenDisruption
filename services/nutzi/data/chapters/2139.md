# Opportunitytypen 
Opportunitytyp 

Dieser Dialog öffnet sich im Dialog Opportunityübersicht >> Stamm >> Opportunytypen . Hier werden die einzelnen Typen verwaltet. 

Ein Opportunitytyp ist eine Klassifikation, um was für eine Art Kontakt es sich handelt. Z.B. ein Kunde von SQL-Business auf eNVenta ERP umstellen will oder ob es ein komplett neuer Kunde. 

Bei der Prüfung zur Einordnung der Opportunities werden immer beide Kriterien (Tage und Umsatz) geprüft. Trifft kein einziges der Kriterien zu, wird der Kontakt immer mit der Stufe ‚D‘ eingeordnet. 

Außerdem werden automatisch das Erfassungsdatum und der entsprechende Sachbearbeiter eingetragen.
