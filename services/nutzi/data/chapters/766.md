# Suche der Artikel mit Merkmalen 
Suche mit Merkmalen 

Dieser Dialog öffnet sich im Artikel >> Optionen >> Suche mit Merkmalen . Hier wird links der „Klassen“-Baum angezeigt. Durch Doppelklick auf eine Klasse oder ein Merkmal wird dieses in die Filterliste rechts übernommen. In der Filterliste kann, wie in der Artikelauswahl, nach einem Wert oder Wertebereich selektiert werden. 

Ein Merkmal darf immer nur einmal im Filter enthalten sein. 

Filter ohne ausgewählten Wert werden ignoriert. 

Der Dialog bleibt geöffnet, die Suche kann jederzeit angepasst werden. Wird der Dialog neu geöffnet, werden bestehende Filter wieder angezeigt. 
### Erweiterung Merkmalsverwaltung: 
Alle Merkmale werden um eine 3. Option, die Freitexteingabe erweitert, wenn "Nur Werte" deaktiviert ist, aber Ausprägungswerte hinterlegt sind. 

Die Werte werden entsprechend der ausgewählten Sprache angezeigt und können direkt für diese Sprache gepflegt werden.
