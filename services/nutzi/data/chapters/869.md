# Export der Kundenpreise 
Export Kundenpreise 

Dieser Dialog öffnet sich im Kunde >> Optionen >> Export Kundenpreise . Über diesen Dialog kann eine einfache bzw. Standard-Kundenpreisliste erstellt und per CSV exportiert werden. 

Beim Öffnen des Dialogs ist zwingend die Kundennummer einzugeben. Das Datenexportformat inkl. Dateipfad kann angegeben werden. Das Feld Format kann manuell gefüllt werden oder über den Button (es öffnet sich der Dialog Datenexport ). 

Auf der Registerkarte Allgemein grenzen Sie die Informationen für den Export ein. Dazu stehen die Felder und Checkboxen zur Verfügung. Dabei können Sie auch festlegen, ob z.B. Bruttopreise oder Sonderpreise verwendet werden sollen. Die Tabelle enthält dann die gewählten Einstellungen zur Information. Dabei ist aber noch kein Export erfolgt! Den Export starten Sie über den gleichnamigen Button, wenn die Einstellungen zuvor gespeichert wurden. Nach dem Import erhalten Sie eine Meldung, ob der Export erfolgt ist. 

Mit dem Button Job starten starten Sie den sofortigen Export über den Job-Server. 

Siehe Kapitel Datenexport .

Dieser Button startet den Druck . Der Button wird erst aktiv, wenn Daten eingelesen sind.
