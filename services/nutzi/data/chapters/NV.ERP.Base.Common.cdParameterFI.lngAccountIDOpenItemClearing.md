# Kapitel NV.ERP.Base.Common.cdParameterFI.lngAccountIDOpenItemClearing

Hier geben Sie eine Kontonummer ein für die automatische OP-Verrechnung. 

Nur wenn dieses Feld gefüllt ist, wird automatisch die OP-Verrechnung über diese Kontonummer durchgeführt. Dieses Konto dient als Verrechnungskonto für die Verrechnung der Rechnung / Gutschrift mit dem Geldausgang / Geldeingang.
