# Html Editor Registerkarte Einfügen 
Html Editor Registerkarte Einfügen 

Da Html nur begrenzt Leerzeichen (bzw. nicht mehr als eins) zulässt und auch keinen Tabulator kennt, können auf dieser Registerkarte entsprechende Sonderzeichen über Buttons eingefügt werden: 

- löst einen Zeilenumbruch aus 

- fügt den Tabulator ein 

- fügt ein Leerzeichen ein 

Des Weiteren lassen sich Links (Button ) und Bilder (Button ) einfügen. Geben Sie dazu die gewünschte URL in das entsprechende Feld neben dem Button ein und übernehmen den Eintrag mit Klick auf den entsprechenden Button. Anschließend wird das Feld wieder geleert. 
### Button Zeilenumbruch 

Dieser Button fügt einen Zeilenumbruch am Ende der Zeile ein.
