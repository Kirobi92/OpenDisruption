# Einrichtung Lucene in eNVenta 

### Serverpfad konfigurieren 
Nach dem Publish ist es erforderlich in eNVenta ein bereits angelegtes Netzwerkverzeichnis zu benennen, in dem später der Suchindex aufgebaut wird. Der Pfad hat unbedingt, wie unten gezeigt, als Serverpfad im Verbindungsmanager hinterlegt zu werden. Als Name wurde hier Lucene gewählt. 

Sie gelangen über den Navigator Menü System >>Allgemein >> Dateiverwaltung >> Verbindungsmanager zur Einrichtung des Speicherortes. 

Als Verbindungsart geben Sie Datei an. 

Als Name wählen Sie eine möglichst sprechende Bezeichnung, z.B. Lucene 

Als Serverpfad geben Sie den Speicherort an. Der Speicherort muss identisch sein mit der Angabe der Konfiguration, welche Sie in Framework Studio hinterlegt haben. 

Hinweis: Der Speicherort sollte möglichst eine SSD-Platte oder entsprechend performant sein. 

Achten Sie darauf, dass die Pfade sich auf demselben Server befinden, auf dem der Lucene Service läuft!

### Konfiguration Lucene-Suche 
Zur Konfiguration der Suche öffnen Sie den Dialog Parameter Allgemein über den Navigator Menü System >> Allgemein >> Parameter Allgemein und wechseln in die Registerkarte Suche . 

Parameter Allgemein Registerkarte Suche 

Hier geben Sie im Feld Service Host die URL ein. Diese besteht aus dem Server (hier local-host), dem Port (hier 23456) und dem Dienstnamen (LuceneServiceHost), welche in Framework Studio definiert wurden. 

Für jeden Mandanten kann ein separater Lucene-Dienst eingerichtet werden. Hierzu müssen beim Publish des Dienstes die entsprechenden Angaben im XML-Setting vorhanden sein. 

Der Indexpfad ist bereits angelegt (siehe oben) und kann über den Button ausgewählt werden. 

Über die Angabe der maximalen Anzahl Treffer kann die Suche entsprechend eingeschränkt werden. 

Wenn Sie ein Straßenverzeichnis gekauft und eingespielt haben, so kann die Suche nach den Straßen auch in den Lucene-Index aufgenommen werden. 

Nachdem alle Angaben erfolgt sind, kann die erste Indexierung gestartet werden. Zum Starten der Indexierung klicken Sie auf den Button Reindex . Über diesen Button können Sie auch manuell die Re-Indexierung starten. Im Feld Fortschritt erfolgt die Anzeige in Prozent für den Fortschritt der Re-Indexierung bei Klick auf Button Aktualisieren rechts neben dem Feld. 

Während der Re-Indexierung ist keine Suche und Schreiben über den Luceneindex möglich. Da eine Datenspeicherung auch eine Ergänzung des Index zur Folge hat, kann eNVenta dadurch evtl. nicht nutzbar sein. Dieser Vorgang dauert mehrere Minuten bis Stunden.
