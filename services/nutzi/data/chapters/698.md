# Artikelnummer aus Nummernkreis generieren

Stammdaten 

# Kopieren eines Artikels 
Artikel kopieren 

Im Dialog Artikel öffnet sich über den Button der Dialog Artikel kopieren . Damit lässt sich auf komfortable Weise ein Artikel mit allen relevanten Daten (Auswahl über die Checkboxen) zentral kopieren. 

In das Feld Artikelnr tragen Sie die neue Artikelnummer ein. Der Status des neu generierten Artikels wird automatisch auf Normal gesetzt. 

Ausnahme: der Artikel befindet sich im Status in Vorbereitung . 

### Artikelnummer aus Nummernkreis generieren 
Wenn Sie einen Artikel kopieren und keine Artikelnummer festlegen, erscheint jedes Mal beim Speichern die Abfrage, ob eine Artikelnummer generiert werden soll. Wenn Sie diese mit Ja bestätigen, wird eine Artikelnummer aus dem Nummernkreis generiert. 

Damit bekommen Sie eine geschlossene Artikelnummernstruktur. 

Für das Kopieren stehen folgende Optionen (über die entsprechenden Checkboxen) zur Verfügung: 

- Stückliste (Komponenten + Text) - Arbeitsplan (Arbeitsgänge Ressourcenzuordnungen, Materialzuordnungen) - Rabatt Verkauf (bei Bezug zu Artikelnummer) - Rabatt Einkauf (bei Bezug zu Artikelnummer) - Artikeleinheiten Verkauf - Artikeleinheiten Einkauf - Preisgruppen Details - Sonderpreise - Rabatte / Zuschläge - Teuerungszuschlag - Zubehör-Artikel - Zusatzpositionen (Artikelstamm >> Optionen) - Sachmerkmale (Merkmalszuordnung - neue EAN generieren (mit oder ohne Prüfziffer) - Prüfziffer anhängen - Bilder - Gebinde Einkauf - Gebinde Verkauf

### Combobox Status 
Der Status, der dem Artikel hinterlegt ist, wird angezeigt. 

Tooltip: 

Folgende Status stehen zur Verfügung: 
• Normal: Alle Funktionen stehen für den Artikel zur Verfügung. 
• Gesperrt: Der Artikel kann in der Auftragsbearbeitung oder dem Bestellwesen nicht mehr erfasst und bearbeitet werden. 
• Ausgelaufen extern: Wird dieser Status gesetzt, bedeutet dies, dass der Lieferant den Artikel nicht mehr liefern kann. Eine entsprechende Meldung erscheint und weist Sie darauf hin. 
• Ausgelaufen intern: Wird dieser Status gesetzt, wird das Dispoverfahren auf Beschaffungsartikel (5) gesetzt. Status 5 verhält sich wie Status 1 (= normal). Eine entsprechende Meldung erscheint und weist Sie darauf hin. 
• In Vorbereitung: Artikel mit diesem Status können bestellt, aber nicht verkauft werden. 
Die Differenzierung der Status 'Ausgelaufen extern' und 'Ausgelaufen intern' ist im Hinblick auf das Einlesen von Daten in den Artikel Schattenstamm notwendig.

### Checkbox Sachmerkmale 
Wenn Sie diese Checkbox aktivieren, werden auch Sachmerkmale berücksichtigt. 

Damit beim Kopieren eines Artikels mit Merkmalen die korrekten Daten kopiert werden, ist eNVenta so eingerichtet, dass bei Änderungen eines Merkmalswerts auf dem ursprünglichen Artikel, dieser nicht auf den kopierten Artikel übertragen wird. 

Wenn in der Combobox Klasse im Artikelstamm auf der Registerkare Merkmale nichts ausgewählt ist, werden alle Merkmale, die für diesen Artikel einen Wert haben (klassenübergreifend) angezeigt.
