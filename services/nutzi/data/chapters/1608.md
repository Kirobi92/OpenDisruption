# Entfernen von Schnittresten aus dem Schnittbereich (Trommelumlagerung) 
Bei der Bearbeitung von eingeschränkt schnittfähigen Artikeln gibt es für diese Artikel im LVS-Lager eine besondere Art der Umlagerung. Ziel ist es, diese Artikel von Ihrem aktuellen Lagerplatz automatisch auf einen Lagerplatz im Schnittbereich des LVS-Lagers umzulagern und gleichzeitig alle weiteren offenen Chargenzuordnungen zu diesem Artikel ebenfalls anzupassen. 

Grundsätzlich orientieren sich die gesamten Vorgänge an der bestehenden Umlagerung von LHMs (Ladehilfsmitteln) – hierbei ist Voraussetzung, dass die Ware in einem LHM gelagert wird und die Lagerplätze LHM-pflichtig sind.

### Ermittlung des Auslagerungsschnittes 
Ist der Schnittartikel kommissioniert, wird durch das System geprüft, ob auf diesen Platz zu der Charge noch Platzzuordnungen bestehen. Ist dies nicht der Fall, wird der Sachbearbeiter gefragt, ob er die Restmenge auslagern möchte. Wird dies mit Nein beantwortet, geht die Kommissionierung normal weiter. Wird dies mit Ja beantwortet, öffnet sich der Umlagerungsdialog. 

### Trommelauslagerung 
Ablauf der Scannerbearbeitung 
Es wird eine neuer Menüpunkt Trommelumlagerung erstellt. Dieser fragt grundsätzlich folgende Informationen ab: 

- Lagerplatz - Ladehilfsmittel - Menge - Evtl. Ziel-Ladehilfsmittel 

### Prüfungen während des Ablaufs 
Nach Anscannen des Ladehilfsmittels wird geprüft, ob auf dem LHM nur ein Artikel liegt und ob dieser ein stückverwalteter Artikel ist. Existieren für diesen noch Chargenzuordnungen, wird geprüft, ob der Platz eine Artikelplatzzuordnung besitzt oder nicht. 

Die Mengeneingabe wird darauf geprüft, ob die gesamte Charge zugeordnet wurde. Ist dies nicht der Fall, so kommt die Aufforderung ein Ziel-Ladehilfsmittel anzugeben. 

Ansonsten wird das Ladehilfsmittel, auf dem der Artikel gelegen hat, automatisch zum Transport-Ladehilfsmittel. 

### Verbuchung 
Die Verbuchung der Auslagerung erfolgt wie die manuelle Umlagerung durch einen Umlagerungsauftrag. Der einzige Unterschied ist, dass hier bestimmte Daten schon vorab eingepflegt sind. 

### Einlagern des Schnittrestes 
Der Schnittrest erhält jetzt wie jede Umlagerung einen Zielplatz und kann auf diesen via Umlagerungsmodul verbucht werden. 

### Auslagerung des Nachschubs 
Ermittlung des richtigen Ladehilfsmittels für den Nachschub 

Wird der Rest ausgelagert, so wird automatisch ein Nachbestückungsauftrag ermittelt. Hierzu sucht das System die Charge des Artikels mit der größten Menge an Chargenzuordnungen. 

Dazu wird die Menge in der Tabelle Platzzuordnung zu einer Charge auf einem Platz summiert. Der höchste davon wird auf den frei gewordenen Platz umgelagert (Umlagerungsauftrag erstellen mit Kennzeichen 'inkl. LHM'). 

Dazu wird der Umlagerungsauftrag durchgeführt. 

Wird nur eine bestimmte Anzahl umgelagert und das LHM auf dem Platz verbleibt, so wird die Entnahmelogik der manuellen Umlagerung verwendet. 

### Umtrommeln 
Das Umtrommeln ist im eigentlichen Sinne eine Umlagerung. Der Unterschied ist, dass hier nicht das komplette LHM entnommen wird, sondern eine Teilmenge. Dadurch entsteht eine neue Chargennummer für den umgechargten Artikel.
