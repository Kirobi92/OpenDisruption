# Sachbearbeiterstamm Registerkarte Druckergruppen

Grundeinstellungen für eNVenta ERP / Sachbearbeiterstamm 

# Sachbearbeiterstamm Registerkarte Druckergruppen 
Sachbearbeiter Registerkarte Druckergruppen 

Auf dieser Registerkarte ordnen Sie dem Mitarbeiter eine Druckergruppe zu. 

Jeder Sachbearbeiter bekommt automatisch die Drucker seiner Abteilung zugeordnet, hierfür muss keine separate Zuweisung erfolgen. 
### Button Neu 

Klicken Sie auf diesen Button, um eine neue Zeile hinzuzufügen und wählen in der Combobox die passende Druckergruppe aus. Sie können neben einer Druckergruppe auch eine Abteilung angeben, welcher wiederum Drucker zugeordnet werden können. 

### Button Löschen 

Damit entfernen Sie eine Druckergruppe aus der Tabelle.
