# Kapitel NV.ERP.Base.Common.cdSystemMM.shtForceLossReason

Wenn Sie diesen Parameter aktivieren, ist zwingend ein Verlustgrund für „verlorene“ Angebote anzugeben. 

Mit dieser Information können Sie eine Auswertung generieren, um zu prüfen, warum aus einem Angebot kein Auftrag wurde. 

Siehe Kapitel Verlustgrund .
