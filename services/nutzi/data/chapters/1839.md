# Zusatzmahntext 
Zusatzmahntext 

Ist im Dialog Mahnwesen Einkauf eine Mahnposition in der Tabelle ausgewählt, so kann durch Klick auf den Button Zusatztext dieser Dialog geöffnet werden. Hier können Sie der Mahnung einen individuellen Text anhängen. Der Text wird pro Lieferant und Mahnvorschlag gespeichert, hat also keinen direkten Bezug zu der einzelnen Position (pro Mahnlauf kann zu einem Lieferanten nur ein Text eingegeben werden). 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Zusatztext 
Tooltip: 

In diesem Dialog können Sie der Mahnung einen individuellen Text anhängen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Dieser Button sucht Daten entsprechend Ihrer Einstellungen. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button speichert Ihre Eingaben. 

### Button Löschen 

Dieser Button löscht markierte Daten. 

Bitte benutzen Sie diesen Button mit Vorsicht! Nicht in jedem Fall wird vor Schließen des Dialogs gespeichert bzw. erscheint eine Meldung, die Sie zum Speichern auffordert.

Dieser Button schließt den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern!
