# Kapitel NV.ERP.FI.FinancialAcc.PaymentTransactions.cdCollectionParameter.sEmailForTest

Hier können Sie eine E-Mail-Adresse für Testfälle eingeben, ansonsten muss dieses Feld leer sein. 

In diesem Fall geht kein Einzugs-Avis per E-Mail zum Kunden. 

Tooltip: 

Hier geben Sie eine E-Mail-Adresse für Testfälle ein.
