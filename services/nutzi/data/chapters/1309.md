# Rabattpflege Registerkarte Kunden 
Registerkarte Kunden 

Hier werden Kunden eingetragen / angezeigt, für die ein Rabattsatz gepflegt werden soll. 
### Tabelle 

### Tabellenspalte Intern ID 
Hier wird eine interne, laufende Nummer angegeben. 

Tooltip: 

Interne laufende Nummer 

### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben, unter welcher der Kunde in der Warenwirtschaft geführt wird. 

Bei der Neuanlage wird das Feld Kunde automatisch aus dem zugehörigen Nummernkreis unter Berücksichtigung des Parameters Kunde = Konto aus dem Dialog Parameter Allgemein gefüllt. Die Nummer kann alternativ auch manuell vergeben werden. 

Siehe Kapitel Nummernkreise Verkauf . 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Firma 1 
Hier wird die Zeile 1 des Firmennamens angezeigt. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Kundengruppe 
Hier wird eine Kundengruppe angegeben. 

Tooltip: 

Zeigt die Kundengruppe. 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Tooltip: 

Öffnet die Kundenauswahl 

### Button Satz hinzufügen 

Mit diesem Button öffnet sich eine neue Zeile und ein neuer Kunde kann eingetragen werden. 

Tooltip: 

Neue Zeile 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag. 

Tooltip: 

Zeilen löschen
