Grundeinstellungen für eNVenta ERP 

# Update Prüfung 
Update 

Die Update-Prüfung erfolgt bei jedem Start von eNVenta ERP , unabhängig der Versionsnummer. Bei der Überprüfung werden auch die Mandanten-abhängigen Skripte geprüft. 

Aus Sicherheitsgründen gibt es in diesem Menü keine Möglichkeit das Customizing oder die Hilfe aufzurufen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Update 
Tooltip: 

In diesem Dialog erfolgt bei jedem Start von eNVenta ERP eine Update-Prüfung, unabhängig der Versionsnummer. 
Für mehr Informationen drücken Sie F1. 

### Button Schließen 

Dieser Button löscht einen Eintrag. 

### Feld Datenbank 
Hier wird die Datenbank angegeben.
