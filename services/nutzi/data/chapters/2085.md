# Besuchsplan 
Besuchsplan 

Über die Besuchsplanung soll eine regelmäßige Kundenbetreuung stattfinden. In der Besuchsvorbereitung kann sich der Außendienst-Mitarbeiter (AD) optimal auf den Besuch vorbereiten, um das bestmögliche Ergebnis zu erreichen. Die Außendienstmitarbeiter benötigen für den Kunden-Besuch vorab umfangreiche Informationen über den Kunden. Diese sind in geeigneter Weise zur Verfügung zu stellen. 

In diesem Dialog erfassen Sie Besuchspläne. Dazu ist es erforderlich, den Vertreter vorzugeben! 

Der Besuchskalender basiert primär auf Besuchsplan-Positionen, diese benötigen einen Besuchsplan (Kopf). 

Die Zuordnung, welche Kunden in den Besuchsplan eingetragen werden, kann halb-automatisch erfolgen oder manuell. 

Die Pflege der turnusmäßigen Besuche im Besuchskalender untertützt die Planung der Besuche. Mehrere Kunden können mit dem Button Mehrfachauswahl auf der Registerkarte Allgemein auf einmal übernommen werden. 

Diese Besuchstermine erscheinen dann in der Registerkarte Allgemein . 

Bei der manuellen Planung kann über den Button Einzelwahl im Besuchsplan ein Kunde oder eine Adresse (Interessent) über den Dialog AKL-Auswahl ausgewählt werden. Alternativ zur Einzelwahl können auch mehrere Kunden auf einmal übernommen werden. Hierzu kann mit dem Button Mehrfachauswahl der Dialog AKL-Auswahl nach Vertreter geöffnet werden. 

Siehe Kapitel Teilnehmer Auswahl nach Vertreterl . 

Wenn Sie aus einem Besuchsplan heraus per Doppelklick einen neuen Besuchsbericht öffnen und Sie in diesem Besuchsbericht den Ansprechpartner ändern, erscheint eine Abfrage, ob dieser Ansprechpartner in den Besuchsplan übernommen werden soll oder nicht. 

Wird diese Referenz nicht bestätigt, wird der Vorgang nicht gespeichert. So wird vermieden, dass falsche Daten entstehen.
