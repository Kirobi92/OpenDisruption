# Komponente ersetzen 
Komponente ersetzen 

Der Button Ersetzen im Dialog Verwendungsnachweis öffnet diesen Dialog. Hierbei wird die angegebene Artikelnummer in den Dialog übernommen. Sie können einen Artikel in allen entsprechenden Stücklisten austauschen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Komponente ersetzen 
Tooltip: 

In diesem Dialog können Sie eine Stücklistenkomponente ersetzen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Field Panel Komponententausch 

Über diese Checkboxen entscheiden Sie, ob der Artikel nur in Stamm-Stücklisten oder nur in Auftrags-Stücklisten ersetzt wird. 

Der Austausch in Auftragsstücklisten ist nur dann möglich, wenn noch kein Produktionsauftrag dazu existiert (bei Produktionsstücklisten). 

Bei Handelsstücklisten ist der Austausch möglich für alle Auftragspositionen ohne Eintrag von Lieferscheinnummer, Packlistennummer und Rechnungsnummer. Damit soll ausgeschlossen werden, dass laufende Kommissioniervorgänge und gelieferte Positionen betroffen sind. 

Wenn die neu einzufügende Komponente selbst eine Stückliste ist, so ist zu prüfen, ob sie nicht bereits vorhanden ist.
