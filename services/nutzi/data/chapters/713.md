# Artikeleinheiten Einkauf/Verkauf Registerkarte Tabelle 
Artikeleinheiten Registerkarte Tabelle 

Auf dieser Registerkarte werden alle bereits angelegten Artikeleinheiten zu diesem Artikel angezeigt. 

Um die Einstellungen dieser Artikeleinheiten zu überprüfen, klicken Sie einmal mit der linken Maustaste in der Liste auf den gewünschten Eintrag und Sie erhalten die Informationen zum ausgewählten Artikel auf der Registerkarte Allgemein . 
### Tabelle 

### Tabellenspalte Artikel 
Die Artikelnummer wird angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Einheit 
Die Einheit, welche dem Artikel zugeordnet ist, wird angezeigt. 

Tooltip: 

Mengeneinheit 

### Tabellenspalte Typ 
Hier wird angezeigt, ob es sich um eine Zuordnung für den Einkauf oder Verkauf handelt. 

Tooltip: 

Gibt an, für welchen Bereich eine Mengeneinheit definiert ist: 
V = Verkauf 
E = Einkauf 
Leer = Allgemein 

### Tabellenspalte Dublette 
Tooltip: 

Falls zu einem bestimmten Artikel eine bestimmte Mengeneinheit 
sowohl als Verkaufseinheit als auch als Einkaufseinheit definiert ist, 
dann spricht man von (Mengeneinheits-)Dublette. 

### Tabellenspalte Standardeinheit 
Wenn Sie diese Checkbox aktivieren, wird die Standardeinheit voreingestellt. Diese Einheit wird bei der Erfassung einer Position im Auftrag voreingestellt. Diese Einstellung kann über die entsprechende Combobox geändert werden. 

### Tabellenspalte Stammeinheit 
Tooltip: 

Stamm-Mengeneinheit (überschrieben) 

### Feld Anzahl max. 
In diesem Feld wird die maximale Anzahl an Datensätzen angezeigt, die in der Tabelle angezeigt werden. Diesen Wert stellen Sie im Dialog Parameter Allgemein ein. 

Siehe Kapitel Parameter Allgemein . 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden. 

### Feld Anzahl Sätze 
Hier wird die tatsächliche Anzahl der angezeigten Datensätze eingelesen. Dieser Wert kann den Wert aus dem Feld Max. Anzahl allerdings nicht überschreiten.
