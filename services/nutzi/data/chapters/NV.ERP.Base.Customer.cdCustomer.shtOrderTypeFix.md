# Kapitel NV.ERP.Base.Customer.cdCustomer.shtOrderTypeFix

Ist für den Kunden im Feld Auftragsart eine Auftragsart zugewiesen und der Parameter Auftragsart Fix gesetzt, so können für den Kunden nur Aufträge mit dieser Auftragsart erfasst werden. 

Der Parameter Auftragsart fix ist nur zugänglich, wenn im Feld Auftragsart eine Auftragsart zugewiesen ist. 

Tooltip: 

Ist für den Kunden im Feld 'Auftragsart' eine Auftragsart zugewiesen und der 
Parameter 'Auftragsart Fix' gesetzt, 
so können für den Kunden nur Aufträge mit dieser Auftragsart erfasst werden. 
Eine Ausnahme bilden Ausgleichsaufträge, die aus der Gutschriftsabrechnung erzeugt wurden. 
Für diese wird die unter Parameter Verkauf, Registerkarte "Gutschriftsabrechnung" 
hinterlegte Auftragsart gezogen.
