# Kostenträgerrechnung Registerkarte Daten 
Kostenträgerrechnung Registerkarte Daten 

Auf dieser Registerkarte werden die Daten in tabellarischer Form dargestellt. 

Ein Doppelklick auf einen Eintrag in der Registerkarte Daten wechselt auf die Registerkarte Kostenträgerblatt und beginnt mit diesem Kostenträger. 

Leistung aus Fertigung: Die im Dialog Parameter Produktion hinterlegte Leistungsartennummer wird hier summiert. Siehe nachfolgende Abbildung:
