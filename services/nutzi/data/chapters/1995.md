# Halbfertigwaren Registerkarte Gruppiert nach 
Halbfertigwaren Registerkarte Gruppiert nach 

Auf dieser Registerkarte können die Daten komprimiert angezeigt werden. Für die Gruppierung stehen die Funktionen / Radiobuttons 

- Produktionsart - Produktion Nr. - Artikelgruppe - Kostenträger 
zur Verfügung.
