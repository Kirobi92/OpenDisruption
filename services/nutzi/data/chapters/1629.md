# Konfigurationsschalter

Lagerverwaltungssystem (LVS) 

# Ladehilfsmittel Information 
Ladehilfsmittel Info 

Dieser Dialog öffnet sich über den Button LHM Info im Dialog Ladehilfsmittel . Hier verwalten Sie weitere Informationen zum Ladehilfsmittel. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Ladehilfsmittel Info 
Tooltip: 

In diesem Dialog verwalten Sie weitere Informationen zum Ladehilfsmittel. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren . 

### Menü Fenster-Information 
Über dieses Menü werden Ihnen Informationen zum Dialog angezeigt. Siehe nachfolgendes Beispiel: 

Meldungen 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Auswahl 

Dieser Button übernimmt eine markierte Zeile in den zugrundeliegenden Dialog. 

### Button Schließen 

Dieser Button schließt den Dialog. 

### Feld LHM Nr. von 
Hier geben Sie zur Selektion eine LHM-Nummer ein. 

### Feld LHM Nr. bis 
Für die Selektion über einen Bereich, geben Sie hier eine weitere LHM-Nummer ein. 

### Combobox Verwendung 
Hier wird das Verwendungslager angezeigt und kann geändert werden. Dies ist das Lager in welchem das Ladehilfsmittel aktuell verwendet wird. Diese Information ist wichtig, falls das LHM ohne Nummer generiert wurde. 

Tooltip: 

Verwendungslager 

### Combobox Erfassung 
Hier wird das Erfassungslager angezeigt, kann aber geändert werden. Dies ist das Lager auf welchem das Ladehilfsmittel angelegt wurde (hier steht nur eine Nummer drin, wenn das LHM mit Lagernummer generiert wurde). 

Tooltip: 

Angabe der Erfassungslagernummer 

### Feld Lagerplatz 
Hier wird angegeben, auf welchem Lagerplatz die Packstücknummer zu finden ist. 

Tooltip: 

Angabe des Lagerplatzes, auf dem die Packstücknummer ist. 

### Combobox Status 
Hier können Sie einen Status auswählen: 

- leer 
- in Einlagerung 
- in Umlagerung 
- in Kommissionierung 
- auf Platz aber leer 
- auf Platz und benutzt 

### Feld Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Button Suche 

Mit diesem Button können Sie Artikel suchen. 

Siehe Kapitel Artikelauswahl . 

### Feld Charge 
Hier wird die Chargennummer angegeben. 

Die Chargenverwaltung steht nur zur Verfügung, wenn sie lizenziert ist. Dies erkennen Sie, wenn im Dialog Konfiguration die Checkbox aktiv gesetzt ist. 

Siehe Kapitel Konfiguration der Parameter . 

Tooltip: 

Angabe der Chargennummer 

### Tabelle LHM-Informationen 

In dieser Tabelle werden die Daten entsprechend Ihrer Selektionskriterien eingelesen. 
### Tabellenspalte Verwendungslager 
Hier wird das Lager angegeben, in welchem das Ladehilfsmittel aktuell verwendet wird. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte LHM Nr. 
In dieser Spalte wird die LHM-Nummer angezeigt. 

Tooltip: 

Wird beim Anlegen des Ladehilfsmittels automatisch generiert. 

### Tabellenspalte LHM Beschreibung 
In dieser Spalte wird die Bezeichnung des LHM-Typs angezeigt. 

Tooltip: 

Ladehilfsmittel-Typ. 

### Tabellenspalte Erfassungslager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Erfassungslager 

### Tabellenspalte Lagerplatz 
In dieser Spalte wird der Lagerplatz angezeigt auf dem sich das LHM derzeit befindet. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Artikel 
In dieser Spalte wird die Artikelnummer angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Hier wird der Name zur Artikelnummer angezeigt. 

Tooltip: 

Zeigt den Artikelnamen. 

### Tabellenspalte Qualität 
In dieser Spalte wird das Qualitätskennzeichen angezeigt. Zulässige Zeichen sind 1 bis 4. Je höher der Wert, desto schlechter die Qualität. 

Tooltip: 

Qualitätskennzeichen 
Zulässige Kennzeichen sind 1 bis 4, je höher der Wert desto schlechter die Qualität. 

### Tabellenspalte Menge 
In dieser Spalte wird die Menge des Artikels auf dem LHM angezeigt. 

Tooltip: 

Angabe der Menge 

### Tabellenspalte Charge 
In dieser Spalte wird die Chargennummer angezeigt. 

Tooltip: 

Angabe der Chargennummer 

### Tabellenspalte Darunterliegendes LHM 
In dieser Spalte wird das darunterliegende LHM angezeigt. 

### Tabellenspalte Daraufgestapeltes LHM 
In dieser Spalte wird das darauf gestapelte LHM angezeigt.
