# Artikelgruppen 
Artikelgruppe 

Jeder Artikel wird einer Artikelgruppe zugeordnet. Daher ist es empfehlenswert, vor der Erfassung der Artikeldaten, die Organisation der Artikelgruppen zu planen und festzulegen. Dieser Dialog öffnet sich im Artikel >> Stamm >> Artikelgruppen . 

Eine Artikelgruppe wird oftmals auch Warengruppe oder Produktgruppe genannt und wird u.a. genutzt für Preisfindung und Auswertungen. 

Die Artikelgruppe hat eine höhere Priorität als die Artikel-Hauptgruppe. 

Artikelgruppen können Sie beispielsweise einer Lieferanten-Rabattgruppe zuordnen. Damit lässt sich der Arbeitsaufwand für die Zuordnung bzw. die Artikelkalkulation verringern. Um eine neue Artikelgruppe anzulegen, tragen Sie im Feld Artikelgruppe das Kürzel und im Feld Bezeichnung die exakte Bezeichnung der neuen Gruppe ein. 

Das Gruppenkürzel darf max. 8 Zeichen umfassen, die Gruppenbezeichnung 80 Zeichen. 

Nummernkreis für Warengruppen: 
In den Warengruppen kann ein Nummernkreis für die zugehörigen Artikelnummern festgelegt werden. Wenn dies gewünscht wird, tragen Sie bitte das Intervall in die Datenfelder Artikelnr Von .... Bis ein. 

Im Artikel erfolgt bei der Warengruppenzuordnung eine Prüfung, ob der Artikel innerhalb des hier festgelegten Nummernkreises liegt. 

Wenn kein Nummernkreis festgelegt ist, erfolgt keine Prüfung!

### Checkbox Kein E-Commerce 
Wenn Sie diese Checkbox aktivieren, steht diese Artikelgruppe im E-Commerce nicht zur Verfügung. 

Gültige Werte: 

- 0 - Datensatz steht im Internet zur Verfügung - 1 - Datensatz steht nicht im Internet zu Verfügung. 
Diese Option ist nur freigeschaltet, wenn das Modul E-Commerce lizenziert ist! 

Tooltip: 

Wenn aktiviert, wird die Artikelgruppe nicht im Webshop angezeigt. 

### Checkbox Kein Intrastat 
Wenn Sie diese Checkbox aktivieren, erfolgt keine Berücksichtigung in der Intrastat. 

Wenn die Checkbox nicht gesetzt ist, ist zwingend ein Land anzugeben. Voraussetzung: Dieses Land ist in der Ländertabelle für Intrastat freigegeben. 

Tooltip: 

Wird nicht in der Intrastat berücksichtigt, wenn Checkbox aktiv ist.
