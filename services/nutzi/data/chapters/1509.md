# Parameter Versand / Logistik Registerkarte Allgemein 
Parameter VS / LO Registerkarte Allemein 

Auf dieser Registerkarte werden allgemeine Einstellungen für Versand / Logistik angelegt. 
### Feld Nettogew. max. 
Hier kann das maximale Nettogewicht für einen Auftrag gepflegt werden. Wenn ein Wert eingetragen ist, wird beim Erfassen eines Auftrags geprüft, ob das hinterlegte, maximale Nettogewicht eingehalten wird. Nur wenn das hinterlegte Nettogewicht pro Verkaufseinheit zu groß ist, wird das Gesamtnettogewicht gelb (im Auftrag Registerkarte Positionen , Feld Nettogewicht ) markiert, da der hinterlegte Wert kleiner ist. 

Tooltip: 

Maximales Nettogewicht eines Auftrags 

### Combobox Fakturafreigabe 
Hier wählen Sie aus, nach welchem Vorgang die Rechnung freigegeben wird. Z.B. nach Lieferscheindruck , Lieferschein Scann oder Rückmeldung des Fahrauftrags . 

Wird hier Lieferscheindruck ausgewählt, wird nach erfolgreichem Lieferscheindruck automatisch die Fakturafreigabe gesetzt. 

Wird hier Lieferschein Scann ausgewählt, wird die Rechnung freigegeben, wenn im Dialog Lieferschein Scann der Lieferschein gedruckt ist. 

Siehe Kapitel Lieferscheine scannen . 

Wird hier Rückmeldung Fahrauftrag ausgewählt, wird die Rechnung freigegeben, wenn der LKW zurück ist und die Ware vom Kunden angenommen wurde.

Methode für automatische Fakturafreigabe

### Checkbox Fakturafreigabe nach LS-Druck 
Wenn Sie diesen Parameter aktivieren, wird direkt nach dem Lieferscheindruck automatisch auch die Faktura-Freigabe ausgelöst. 

Wird zum Parameter Fakturafreigabe nach LS-Druck noch der Parameter Rechnung nach Freigabe drucken gesetzt, wird durch den Druck des Lieferscheins die Freigabe erzeugt und zusätzlich die Rechnung mit den bekannten Voraussetzungen gedruckt. 

Diese Einstellung gilt nur für den Dialog Kommissionieren und kann im Versandvorgang übersteuert werden. 

Tooltip: 

Wenn Sie diese Checkbox aktivieren, wird nach dem Lieferscheindruck automatisch auch die Faktura-Freigabe ausgelöst. 

### Checkbox Rechnungsdruck nach Freigabe 
Ist dieser Parameter gesetzt, wird mit der Fakturafreigabe automatisch der Rechnungsdruck ausgeführt. 

Ist im Kundenstamm oder im Auftrag der Parameter Kein Rg.-Druck im VS gesetzt, wird der Rechnungsdruck auch bei gesetztem Parameter Rechnung nach Freigabe drucken nicht ausgeführt. 

Wenn die Lieferanschrift im Auftrag abweicht, wird auch keine Rechnung gedruckt. 

Tooltip: 

Mit diesem Parameter wird automatisch nach Fakturafreigabe der Rechnungsdruck ausgeführt. 

### Checkbox Multi-Packlisten auf einen LS / RE drucken 
Diese Option ist beschränkt auf die Kommissionierart ‚Multi-Packlisten‘ und zieht nur dann, wenn zugleich der Parameter Sammellieferschein in den Parametern des Verkaufs gesetzt ist. So werden beim späteren Rechnungsdruck aus dem Versand – sofern nötig, alle beteiligten Auftragsköpfe auf Sammelrechnung gesetzt. 

Dies zieht nur beim Rechnungsdruck über den Versand!

### Checkbox Rechnungsbriefkopf 
Diese Checkbox steht zur Verfügung, um aus der Kommissionierung heraus den Andruck des Briefkopfs auf der Rechnung zu steuern. Dafür ist allerdings Voraussetzung, dass die Checkbox k.Rg.-Druck VS im Kundenstamm bzw. Auftrag nicht gesetzt ist! 

Wenn der Parameter gesetzt ist, wird beim Rechnungsdruck aus dem Versand heraus immer der Briefkopf auf der Rechnung gedruckt (analoge Einstellung z.B. zum Massendruck Verkauf). 

Tooltip: 

Beim Druck der Rechnung wird der Rechnungsbriefkopf mitgedruckt. 

### Checkbox Packlisten mit Pickliste drucken 
Mit diesem Parameter kann festgelegt werden, dass nach dem Druck der Pickliste aus dem Vorschlag auch gleich die Packlisten, der im Pickpool selektierten Aufträge, gedruckt werden. 

Tooltip:

### Checkbox Packlistendruck bei Versandauflösung 
Wenn Sie diese Checkbox aktivieren, wird bei Buchung einer Versandauflösung im Wareneingang der Packlistendruck für ausgewählte Positionen angepasst. Mit Hilfe der Versandauflösung muss diese Ware aus dem Wareneingang nicht mehr eingelagert und später wieder ausgelagert werden. Die Ware kann sofort mit Hilfe der automatisch generierten Packliste auf einen Packplatz verbracht werden und der Kommissionierungsprozess zur Auslieferung an den Kunden durchgeführt werden. 

Die Packliste kann direkt auf dem Standarddrucker des angemeldeten Sachbearbeiters erzeugt werden. Dazu drücken Sie im Dialog Eingangsbuchung den Button Drucken . 

Siehe Kapitel Eingangsbuchung . 

Wenn diese Checkbox nicht gesetzt ist, ist die Combobox Versandart im Auftrag auf der Registerkarte Kopfdaten zugänglich. 

Tooltip: 

Wenn gesetzt, werden die Packlisten direkt gedruckt, wenn in der Versandauflösung auf 'Buchen' geklickt wird. 

### Checkbox Nur Packlistenpositionen auf Lieferschein 
Wenn dieser Parameter aktiviert ist, werden beim Druck des Lieferscheins in der Kommissionierung nur die Positionen der aktuell am Bildschirm angezeigten Packliste gedruckt, wenn zusätzlich der Parameter Sammellieferschein / -Packliste in den Verkaufsparametern gesetzt ist. 

Damit der Parameter funktioniert, muss in der Kommissionierung über die Packlistennummer gesucht werden. 

Siehe Kapitel Parameter Verkauf .

### Checkbox Fehlmengenberichte 
Wenn diese Checkbox gesetzt ist, werden bei negativen Mengenänderungen in der Kommissionierung, Berichte über Fehlmengen gedruckt, ehe sie abgeschlossen werden können. 

Tooltip: 

Wenn aktiv, werden bei negativen Mengenänderungen in der Kommissionierung, Berichte über Fehlmengen gedruckt, ehe sie abgeschlossen werden können. 

### Checkbox PosPackstück-Zuordnungspflicht 
Wenn dieser Parameter aktiviert ist, wird beim Verlassen des Dialogs Packstücke geprüft, ob es Packstücke gibt, die noch keine Positionszuordnungen haben. Falls ja, erhalten Sie eine Meldung, die Sie daraufhin weist, dass noch Zuordnungen fehlen.
