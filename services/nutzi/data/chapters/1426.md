# Mietobjekte als Zubehör 
Zubehör 

Dieser Dialog öffnet sich über den Button Zubehör im Dialog Mietobjekt . Über diesen Dialog ist es möglich z.B. bei einer Preiserhöhung eines Mietobjektes die weiteren Mietobjekte zu ermitteln, in welchen das aktuelle Mietobjekt als Zusatz enthalten ist, um auch bei diesen den Gesamtpreis der Vermietung anzupassen. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Zubehör 
Tooltip: 

In diesem Dialog können Sie Mietobjekte ermitteln, in welchen das aktuelle Mietobjekt Zusatz ist.

### Tabellenspalte Klasse 
Hier wird die Klasse des Mietobjekts angegeben. Zur Verfügung stehen: 

- Standard: Dieses MO kann nur einmal in einer Vermietung vorkommen (z.g. Bagger). Optional ist dies mit der Ressourcenverwaltung kombinierbar. - Versicherung: Dieses MO stellt eine Versicherungsposition dar. So gekennzeichnete MOs können als Zusätze bei Mietobjekten, welche versicherungspflichtig sind, hinterlegt werden und werden automatisch in die Vermietung übernommen, wenn der Kunde keine eigene Versicherung nachweist. - Paketmietobjekt: Dieses MO stellt ein "Hüllobjekt" dar. D.h. so gekennzeichnete Mietobjekte bilden den Rahmen für unter Zusätze hinterlegten Positionen und ermöglichen die Kalkulation eines Paketpreises für die Vermietung. - Massenartikel: Dieses MO kann mehrfach in einer Vermietung vorkommen. (z.B. Schalungsteile) 

Tooltip: 

Mietobjekt Klasse 
Definiert die Klasse des Mietobjektes. 
Standard – Dieses MO kann nur einmal in einer Vermietung vorkommen (z.B. Bagger). Optional mit der Ressourcenverwaltung kombinierbar. 
Versicherung – Dieses MO stellt eine Versicherungsposition dar. So gekennzeichnete MOs können als Zusätze bei MOs, 
welche versicherungspflichtig sind, hinterlegt werden und werden automatisch in die Vermietung übernommen, wenn der Kunde keine eigene Versicherung nachweist. 
Paketmietobjekt – Dieses MO stellt ein „Hüllobjekt“ dar. D.h., so gekennzeichnete MOs bilden den Rahmen für die unter Zusätze hinterlegten Positionen 
und ermöglichen die Kalkulation eines „Paketpreises“ für die Vermietung. 
Massenartikel – Dieses MO kann mehrfach in einer Vermietung vorkommen. (z.B. Schalungsteile)
