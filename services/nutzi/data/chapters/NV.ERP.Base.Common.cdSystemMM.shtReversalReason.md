# Kapitel NV.ERP.Base.Common.cdSystemMM.shtReversalReason

Wird dieser Parameter aktiviert, muss bei einem Storno eine Begründung eingegeben werden. 

Die Kasseneinstellungen werden pro Kasse vorgenommen, die Kassenparameter gelten für alle Kassen. 

Tooltip: 

Wenn gesetzt, muss bei Storno eine Begründung eingegeben werden.
