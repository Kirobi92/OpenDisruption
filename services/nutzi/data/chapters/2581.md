# Verarbeitete Vorschläge im Bankeinzug löschen 
Verarbeitet Einzugsvorschläge löschen 

Im Bankeinzug unter Optionen finden Sie den Eintrag Verarbeitete Einzugsvorschläge löschen . In diesem Dialog können alte Zahlvorschläge endgültig aus der Datenbank gelöscht werden. Normalerweise werden alle erfolgreich verarbeiteten Vorschläge in der Datenbank gespeichert, damit diese jederzeit geprüft werden können. Sie haben über diesen Dialog aber auch die Möglichkeit alle alten Vorschläge automatisch und komplett aus der Datenbank zu entfernen. 

Die gelöschten Einzugsvorschläge stehen für die Auswahl im Dialog Rücklastschriften buchen nicht mehr zur Verfügung! D.h. die automatische Verbuchung der Rücklastschrift kann nicht mehr erfolgen, es muss ggf. manuell Skonto zurückgebucht werden. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Verarbeitete Einzugsvorschläge löschen 
Tooltip: 

In diesem Dialog können alte Zahlvorschläge endgültig aus der Datenbank gelöscht werden. 
Für mehr Informationen drücken Sie F1.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
