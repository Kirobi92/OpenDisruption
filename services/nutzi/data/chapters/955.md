# Projekt-Leistungscontrolling 
Projekt-Leist. controlling 

Dieser Dialog öffnet sich im Projekt >> Optionen >> Projekt-Leistungscontrolling . Dem Dialog wird das aktuelle Projekt und ggf. die aktuell selektierte Projektphase übergeben. Außerdem kann in diesem Dialog nach Kunde, Ressource, Datum und Status der Leistungserfassung selektiert werden. 

Die Projektnummern und die Phasen werden in Tabellen angezeigt, so dass es möglich ist über mehrere Projekte bzw. Phasen zu selektieren. 

Unter den Selektionsfeldern befinden sich 2 Registerkarten. In der Aufgaben-Tabelle (obere Tabelle) werden alle selektierten Aufgaben mit den zugehörigen Controlling-Daten aufgelistet. In der Tabelle darunter werden die einzelnen Leistungserfassungen zu einer Projektaufgabe angezeigt, sobald Sie auf eine Zeile in der oberen Tabelle klicken. Die obere Tabelle zeigt außer der Kundennummer, Suchname, Projektbereich, Nummer, der Bezeichnung für das Projekt und die Aufgabe Controlling-Daten an: 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Leistungscontrolling 
Tooltip: 

In diesem Dialog werden für das Controlling die Projektleistungen angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
