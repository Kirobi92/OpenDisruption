# Gesellschaftsverwaltung in der Intrastat

Gesellschaftsverwaltung für Multicompany 

# Parameter Gesellschaft 
Parameter Gesellschaft 

Die Gesellschaftsverwaltung für eine gemeinsame Warenwirtschaft steht nur zur Verfügung, wenn die entsprechende Lizenz freigeschaltet ist! 

Dieser Dialog öffnet sich im Navigator über System >> Allgemein >> Gesellschaftsverwaltung >> Parameter Gesellschaft . In den Gesellschafter-Systemparametern werden die Grundlagen der internen „Verrechnung“ gelegt. 

Hier werden die zu verwendenden Auftrags- und Einkaufsarten für die Generierung der automatischen Belege hinterlegt: 

- Auftragsart mit Faktura (UAF) für Umlagerung zwischen den verschiedenen Gesellschaften, inkl. Berechnung - Auftragsart ohne Faktura (UA) für Umlagerung innerhalb einer Gesellschaft ohne Berechnung (interner Werksverkehr) - Wertgutschriftart für die interne Gutschrifterstellung (Retouren) - Einkaufart mit Rechnungsprüfung (UBRP) für die Umlagerung zwischen den Gesellschaften inkl. Berechnung - Einkaufart ohne Rechnungsprüfung (UB) für Umlagerungen ohne Berechnung innerhalb einer Gesellschaft (interner Werksverkehr) - Kundenursprungsauftragspositionen werden erst fakturiert, wenn die internen Ver-rechnungspreise feststehen. Dies gilt insbesondere für die Beschaffung von Diversartikeln oder Beschaffungsware. - Das System sorgt dafür, dass nur „fertige“ Positionen fakturiert werden. 
Bei der Angabe des IVP (= interner Verrechnungspreis) handelt es sich um einen prozentualen Aufschlag für Beschaffungsware, wenn diese an eine andere Gesellschaft verkauft wird. Der Aufschlag kann pro Artikelgruppe spezifiziert werden, wenn nicht erfolgt, wird dieser Wert als Default-Wert übernommen. Der IVP bezieht sich immer auf den EK (EK1). 

Mehr zum Thema Gesellschaftsverwaltung erhalten Sie im Kapitel Gesellschaftsverwaltung für Multicompany .

### Combobox Preistyp, IVP 
In dieser Combobox stehen die Einträge EK + Zuschlag und Preisfindung zur Verfügung. Wenn Sie Preisfindung auswählen, so erfolgt die Verrechnung analog der Lagerware über die Konditionierung für den Abrechnungskunden. 

So haben Sie die Möglichkeit bezüglich IVP für Beschaffungsware zwischen beiden Abrechnungsverfahren zu wählen. 

Tooltip: 

Beschaffungsware, Preistyp für den internen Verrechnungspreis (IVP): 
- 1: EK + Zuschlag 
- 2: Preisfindung 

### Feld Zuschl. IVP % 
Hier geben Sie einen Zuschlag an für den internen Verrechnungspreis (IVP): 

IVP = EK + Zuschlag 
Umlagerungen zwischen unterschiedlichen Gesellschaften werden bei Beschaffungsware mit dem IVP verrechnet. 
(Der IVP ist der VK der Umlagerungsposition) 

Tooltip: 

Zuschlag für die Kalkulation des internen Verrechnungspreises (IVP) für das Verfahren 'EK + Zuschlag': 
IVP = EK + Zuschlag 
Abrechnungspositionen zwischen unterschiedlichen Gesellschaften werden bei Beschaffungsware mit dem IVP verrechnet. 
(Der IVP ist der VK der Abrechnungsposition.)
