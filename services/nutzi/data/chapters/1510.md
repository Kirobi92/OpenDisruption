# Parameter Versand / Logistik Registerkarte Tour 
Parameter VS / LO Registerkarte Tour 

Auf dieser Registerkarte nehmen Sie die Einstellungen für Touren vor. 
### Feld Kilometer max. 
Falls der Wert, der hier in den Parametern für die Kilometeranzahl angegeben ist, nicht zu der Eingabe im Dialog Rückmeldung Fahrauftrag passt, erhalten Sie eine Meldung. Damit verhindern Sie eventuelle Fehleingaben. 

Tooltip: 

Falls die hier festgelegte km-Strecke nicht zu der Eingabe im Dialog 'Rückmeldung Fahrauftrag' passt, erhalten Sie eine Meldung. 

### Feld Warnung 
Wenn diese Zeitgrenzen festgelegt sind, wird die Startzeit in der Tourendisposition entsprechend markiert. Voraussetzung dafür ist, dass Picks vorhanden sind und noch nicht alle Picks abgeschlossen sind. 

Touren im Status Warnung werden entsprechend gelb markiert, um dem Fuhrparkleiter einen schnellen Überblick zu ermöglichen. 

Tooltip: 

Warnung xx Minuten vor Abfahrt 

### Feld Kritisch 
Wenn diese Zeitgrenzen festgelegt sind, wird die Startzeit in der Tourendisposition entsprechend markiert. Voraussetzung dafür ist, dass Picks vorhanden sind und noch nicht alle Picks abgeschlossen sind. 

Kritische Touren werden rot markiert, um dem Fuhrparkleiter einen schnellen Überblick zu ermöglichen. 

Tooltip: 

Kritisch xx Minuten vor Abfahrt 

### Combobox Paket Rückscan 
In dieser Combobox stehen folgende Optionen zur Verfügung: 
- Immer -> Es müssen immer die Lieferscheine und die Packstücke angescannt werden, um die Ware auf dem Fahrzeug zu bestätigen. - Nur bei Umlagerung -> Das Scannen der Packstücke ist nur notwendig, wenn es sich um eine Umlagerung handelt (Bezugslager am Auftrag ist gesetzt, Verbundobjekt vorhanden). - Nur Packstücke -> Es müssen immer die Packstücke angescannt werden, um die Ware auf dem Fahrzeug zu bestätigen. Das Scannen der Lieferscheine ist nicht notwendig. 
Je nachdem, was hier ausgewählt ist, wird im Dialog Lieferscheine scannen das Feld Packstück editierbar. 

Siehe Kapitel Lieferscheine Scannen . 

Tooltip: 

Angabe welche Belege/Vorgänge beim Paket Rückscan angescannt werden müssen. 

### Combobox Ret. Belegart 
Um bereits bei der Rückmeldung eines Fahrauftrags evtl. Retouren generieren zu können, steht die Combobox Retourenart zur Verfügung. Bei der Generierung der Retoure über die Rückmeldung Fahrauftrag wird die Retoure mit der entsprechenden Retourenart angegeben. 

Tooltip: 

Mit der Retourenart, die Sie hier auswählen, kann unmittelbar nach Rückmeldung eines Fahrauftrags eine Retoure mit dieser Retourenart generiert werden. 

### Feld Planungshorizont 
In dieses Feld geben Sie die Anzahl Tage ein, die ab Wunschdatum für die Tourenplanung herangezogen werden. 

Tooltip: 

Anzahl der Tage, die ab Wunschdatum für die Tourenplanung herangezogen werden 

### Checkbox Neusortierung Scans 
Wenn Sie diese Checkbox aktivieren, wird beim Speichern im Dialog Lieferscheine scannen die Reihenfolge der Stopps gemäß der Reihenfolge des Scanns der Lieferscheine angepasst. Der Fahrer hat damit die Möglichkeit, die letzte Sortierung der tatsächlichen Fahrtreihenfolge vorzunehmen, um z.B. auf eine Baustelle auf der Tour Rücksicht zu nehmen. 

Siehe Kapitel Lieferscheine scannen . 

Tooltip: 

Neusortierung beim Beladungsscan 

### Checkbox Belege hinzufügen 
Mit dieser Funktion ist es möglich, weitere Belege einer Tour über die Funktion „Lieferscheine scannen“ hinzuzufügen. Ein Fahrer scannt z.B. einen weiteren Lieferschein ein, welcher vom Vortag übriggeblieben ist oder ein Kunde wollte spontan, statt die Ware zu holen, eine Lieferung. So kann hier schnell der Lieferschein der Tour zugeordnet werden (Achtung: Zuschläge usw. erfolgen hierdurch nicht, es geht nur um die schnelle Umplanung/Einplanung eines Beleges auf die Tour). 

Tooltip: 

Nicht geladene Belege beim Scannen von Lieferscheinen hinzufügen. 

### Checkbox Tourenplanung bei Auftragserfassung 
Wenn Sie diese Checkbox aktivieren, wird bereits bei Auftragserfassung die beste mögliche Tour ermittelt und im Auftrag auf der Registerkarte Logistik in das Feld Tour eingetragen. 

Des Weiteren gilt: Ist diese Checkbox gesetzt, so wird der Auftrag, wenn er bereits eingetourt war, beim Setzen der Liefersperre im Auftragskopf automatisch ausgetourt! 

Beispiel: 
Die Packliste für einen Auftrag ist bereits gedruckt, dann wird der Auftrag auf Liefersperre gesetzt. Der Auftrag wird aus der Tour, die bereits bei Auftragserfassung ermittelt wurde, automatisch ausgetourt. 

Tooltip: 

Beim Speichern einer Auftragsposition erfolgt eine Tourenplanung. 

### Checkbox Versandarten über Touren steuern 
Wenn Sie diese Checkbox aktivieren, werden 

- im Kundenstamm die Felder Versender und Versender-ID ausgeblendet und - im Auftrag die Versandart ausgeblendet. 

Wenn diese Checkbox nicht gesetzt ist, ist die Combobox Versandart im Auftrag auf der Registerkarte Kopfdaten zugänglich und bietet so die Möglichkeit, einen Sondertransport abzubilden. 

Tooltip: 

Die Versandart im Auftrag wird anhand der ermittelten Tour rückübertragen. 

### Checkbox Packlistendruck nach Tourenplanung 
Wenn Sie diese Checkbox aktivieren, darf eine Packliste (mit Versandweg Zufuhr) erst gedruckt werden, wenn der Artikel einer Tour zugeordnet worden ist. Diese Einstellung wirkt auch im Massendruck für Packlisten. 

Tooltip: 

Wenn der Schalter gesetzt ist, wird das Drucken der Packlisten erst nach erfolgreicher Tourenplanung ermöglicht. 

### Checkbox Doppelbelieferungsschutz 
Wenn Sie diesen Parameter aktivieren, wird verhindert, dass ein Kunde doppelt (bzw. 2-mal an einem Tag) beliefert wird. 

Dies könnte der Fall sein, wenn beispielsweise der Kunde an einem bestimmten Tag mit der Spedition beliefert wird. Dann soll die restliche Ware (aus Kostengründen) ebenfalls mit der Spedition geliefert werden, anstatt dem eigenen Fuhrpark oder Paketdienst. 

Tooltip: 

Die Lieferung mehrerer Aufträge über mehrere Touren an einem Tag soll verhindert werden. 

### Checkbox DB-Schutz nur bis Verpackung 
Wenn Sie diese Checkbox aktivieren, ist der Doppelbelieferungsschutz nur bis zur Verpackung aktiv. 

Tooltip: 

Doppelbelieferungsschutz erfolgt nur bis zur Zuordnung von Artikel auf Packstück. Anschließend erfolgt keine automatische Umplanung der Sendung. 

### Checkbox Kein Overnight 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Overnight-Touren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Kein Overnight“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Tooltip: 

Keine Lieferung per Overnight 

### Checkbox Kein Paketdienst 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Paketdienst-Touren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Kein Paketdienst“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Beim Kopieren eines Angebots / Auftrags wird diese Einstellung übernommen. 

Tooltip: 

Keine Lieferung per Paketdienst 

### Checkbox Keine Spedition 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Speditionstouren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Keine Spedition“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Tooltip: 

Keine Lieferung per Spedition 

### Checkbox Kein Fuhrpark 
Wenn Sie diese Checkbox aktivieren, wird der Artikel nicht über den Fuhrpark beliefert. 

Tooltip: 

Keine Lieferung per Fuhrpark
