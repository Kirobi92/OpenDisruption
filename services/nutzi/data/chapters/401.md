# Ereignisdefinition Registerkarte Expertenmodus 
Ereignisdefinition Registerkarte Expertenmodus 

Über die Registerkarte Expertenmodus können Sie für jedes, unter "Bedingung" erfasste Feld noch weitere Einschränkungen erstellen. 

Dies ist hilfreich, falls die Möglichkeiten der Registerkarte Bedingung nicht reichen, um das Ereignis einzuschränken. 

Zur Erfassung der Bedingung dient der Ausdrucks Editor , der sich über den Button öffnet. 

Beispiel: 
Das o.g. Ereignis soll auch auftreten, wenn ein bestehender Divers-Artikel auf Nicht-Divers-Artikel gesetzt wird. Deswegen wurden im Kopfbereich und auf der Registerkarte Bedingung die Checkboxen Eingefügt UND Geändert aktiviert. 

Nun kann in der Registerkarte Expertenmodus für den Fall "Divers-Flag" eine andere Bedingung erfasst werden, als für den Fall "Artikel geändert".

Tooltip:

### Feld Einfügen Bed. 
Das Event wird nur ausgelöst, wenn in das ausgewählte Feld ein Wert eingetragen wird, aber davor kein Wert vorhanden war und die Bedingung zutrifft. 

Beispiel: Das Feld Hauptlieferant im Artikelstamm war leer, wird nun mit einer Lieferantennummer befüllt und diese lautet "70000". 

Tooltip: 

Das Event ist nur gültig, wenn ein Wert in das ausgewählte Feld eingetragen wird,

Beispiel: Das Feld "Hauptlieferant" im Artikelstamm war leer, wird nun mit einer Lieferantennr. befüllt und diese lautet "70000".

### Feld Ändern Bed. 
Das Event wird nur ausgelöst, wenn in dem ausgewählten Feld der Wert geändert wird und die Bedingung zutrifft. 

Beispiel: Das Feld Hauptlieferant im Artikelstamm hatte einen Wert, wird nun mit einer anderen Lieferantennummer befüllt und diese lautet "70000". 

Tooltip: 

Das Event ist nur gültig, wenn in dem ausgewählten Feld der Wert geändert wird und die Bedingung zutriff. 

Z.B.: Das Feld "Hauptlieferant" im Artikelstamm hatte einen Wert, wird nun mit einer anderen Lieferantennummer befüllt und diese lautet "70000".

### Feld Löschen Bed. 
Das Event wird nur ausgelöst, wenn aus dem ausgewählten Feld der vorhandene Wert gelöscht wird und die Bedingung zutrifft. 

Beispiel: Das Feld Hauptlieferant im Artikelstamm ist befüllt, wird nun geleert und der Wert war "70000".
