# Kapitel NV.ERP.Base.Common.cdSystemMM.shtICPurchPriceUpdating

Ist dieser Parameter gesetzt, wird automatisch der EK aktualisiert, wenn die Rechnungsprüfung automatisch durchgeführt wird. 

Dieser Parameter kann in der einzelnen Rechnungsprüfung übersteuert werden. 

Siehe Kapitel Rechnungsprüfung .
