# Allgemeine Pausen 
Allgemeine Pause 

Über den Button Pausen im Kopf des Dialogs Betriebskalender öffnet sich das Menü Allgemeine Pause . Hier erhalten Sie eine Übersicht über bereits angelegte Pausen oder legen zu einer Schicht die Pausen neu fest. 

Wenn für einen Kalender keinerlei Pause angegeben ist, so kommen die im Kalender 0 angegebenen Pausen zur Anwendung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Allgemeine Pause 
Tooltip: 

In diesem Dialog erhalten Sie eine Übersicht über bereits angelegte Pausen oder legen zu einer Schicht die Pausen neu fest. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!

### Tabellenspalte Schicht 
Hier wird die Schicht angegeben. 

Sonderfall: Im Fall einer Schicht, die sich in den nächsten Tag hinein zieht, kann die Pause auch über Mitternacht hinaus definiert werden. In diesem Fall ist die Zeit Bis kleiner als die Zeit Von . 

Tooltip: 

Angabe der Schicht 

### Tabellenspalte Ressourcengruppe 
Hier wird die Nummer der Ressourcengruppe angegeben. 

Tooltip: 

Angabe der Ressourcengruppe 

### Tabellenspalte Zeit von (hh:mm) 
Hier wird der Beginn der Pause angegeben. Sie gilt grundsätzlich für alle Ressourcen. Wird die Zeit nicht korrekt eingegeben, erhalten Sie einen Hinweis. 

### Tabellenspalte Zeit bis (hh:mm) 
Hier wird das Ende der Pause angegeben. Sie gilt grundsätzlich für alle Ressourcen. Wird die Zeit nicht korrekt eingegeben, erhalten Sie einen Hinweis. 

Tooltip: 

Zeigt die Zeit-bis an. 

### Tabellenspalte Grund 
Hier wird zur besseren Darstellung ein Grund für die Pause angegeben (z.B. Mittagspause). 

Tooltip: 

Grund für die Pause 

### Tabellenspalte Datum von 
Hier wird die Pause auf ein Datum oder den Beginn eines Datumsbereichs eingegrenzt. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein. 

### Tabellenspalte Datum bis 
Hier wird die Pause auf ein Datum oder einen Datumsbereich eingegrenzt. 

Tooltip: 

Geben Sie ein Bis-Datum ein. 

### Tabellenspalte Sachbearbeiter 
Hier wird vom Programm automatisch der Sachbearbeiter eingetragen, der den Datensatz zuletzt geändert hat. 

Tooltip: 

Zeigt den Sachbearbeiter, der als letztes den Datensatz bearbeitet hat. 

### Tabellenspalte Erfassung 
Hier wird vom Programm automatisch das Datum der Erfassung eingetragen. 

Tooltip:
