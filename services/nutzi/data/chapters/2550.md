# Steuerjournal 
Steuerjournal 

Dieser Dialog öffnet sich im Buchen >> Auswertungen >> Steuer >> Steuerjournal . Hier werden Ihnen detailliert alle Steuerbuchungen für Kontrollzwecke zur Verfügung gestellt. 

In diesem Menü ist es möglich nach einer UVA-Auswertungsstruktur bzw. nach den darin enthaltenen Steuerschlüsseln zu selektieren. 

Die Selektionsmöglichkeit ‚UVA' wird nur dann aktiviert, wenn im Bereich Steuerart der Radiobutton Selektion ausgewählt ist. Wenn in der Combobox Steuerschlüssel ein Steuerschlüssel ausgewählt wurde, kann keine UVA gewählt werden (und umgekehrt). Wird eine UVA (UVA-Auswertungsstruktur) gewählt, so werden nur Steuer-Buchungen geladen, welche mit Steuerschlüsseln gebucht wurden, die in der gewählten UVA-Struktur zugeordnet sind. 

Das Steuerjournal bietet auch die Möglichkeit, bestehende Fremdwährungsbeträge anzuzeigen. Zur Selektion nach Währung steht die Combobox Währung zur Verfügung. Wenn Sie eine Fremdwährung auswählen, werden die beiden Radiobuttons Beträge in LW und FW sichtbar. So legen Sie die Währung für die Darstellung der Umsätze fest.

### Dock Panel Steuerart 

Über diese Radiobuttons legen Sie fest, über welche Steuerart Sie suchen wollen. 

Die Radiobuttons Alle , MwSt. und VSt. dienen dazu einen gesamten Überblick über die Steuerbuchungen nach den 3 Kategorien 

- Alle = Umsatzsteuer und Vorsteuer - MwSt. = nur Umsatzsteuer - VSt. = nur Vorsteuer 
zu geben; somit wird keine Steuerbuchung durch unpassende Selektion übersehen. Hier ist nur der Zeitraum selektierbar und die Sortierung.

UVA und Belegnummer. Damit weist die Tabelle dann auch nur die Steuerbuchungen der eingeschränkten Selektion aus!

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme).
