# Konfiguration der Intrastat Registerkarte Allgemein 
Konfiguration Intrastat Registerkarte Allgemein 

Auf dieser Registerkarte nehmen Sie die grundlegenden Einstellung zur Konfiguration der Intrastat Meldung vor. 
### Combobox Gesellschaft 
Hier wird die Gesellschaftsnummer für das Verfahren 'Multicompany' angegeben. 

Siehe Kapitel Gesellschaftsverwaltung für Multicompany . 

Tooltip: 

Gesellschaft für das Verfahren "Eine Warenwirtschaft". 

### Combobox ISO-LKZ 
Hier wird das IsoKz des Landes angegeben. Dies wird insbesondere benötigt bei Zahlungen innerhalb der EU (IBAN / BIC) und Auslandszahlungen über SWIFT. 

Tooltip: 

Angabe des Ländercode gemäß ISO 3166-1 alpha-2 

### Combobox Intrastat Firma 
Tooltip: 

Angabe der Intrastat Geschäftseinheit 

### Feld Materialnummer 
Hier wird die Materialnummer angegeben, die vom Statistischen Bundesamt zugewiesen wurde. 

Tooltip: 

Angabe der Materialnummer, die vom Statistischen Bundesamt zugewiesen wurde 

### Feld Steuernummer 
Hier geben Sie die Steuernummer für den Intrastat-Prozess angeben. 

### Feld Untersch.nr. 
Hier wird eine 3-stellige Unterscheidungsnummer für Intrastat angegeben. 

Tooltip: 

3-stellige Unterscheidungsnummer Intrastat 

### Feld Kennnummer 
Hier wird die Kennnummer des Melders angegeben. Sie setzt sich zusammen aus: 

- Bu/Fa des Bundeslandes des Finanzamtes (2-stellig) - Steuernummer (11-stellig, bei 10-stelliger hat eine 0 angefügt zu werden) - Unterscheidungsnummer (3-stellig) 
Das Feld für die Kennnummer hat 16 Stellen und wird an die Export-Datei übertragen. 

Die Kennnummer kann manuell angepasst werden. 

Tooltip: 

Kennnummer des Melders. Setzt sich zusammen aus: 
1. Bu/Fa des Bundeslandes des Finanzamtes (2-stellig) 
2. Steuernummer (11-stellig, bei 10-stelliger muss eine 0 angefügt werden) 
3. Unterscheidungsnummer (3-stellig) 

Wird automatisch aus Bu/Fa, Steuernummer und Unterscheidungsnummer ermittelt, 
kann aber manuell geändert werden. 

### Combobox Mapping 
Hier wird der Identifizierer eines Mappings angegeben, das mit dem X-Mapper erstellt wurde. 

Tooltip: 

Identifizierer eines Mappings, das mit dem X-Mapper erstellt wurde. 

### Checkbox Meldung je Verkehrsrichtung 
Wenn Sie diese Checkbox aktivieren, erfolgt die Meldung je Verkehrsrichtung (Eingang/Versendung) in einer separaten Datei. 

Tooltip: 

Meldung erfolgt je Verkehrsrichtung (Eingang/Versendung) in separater Datei 

### Button Suche 

Mit diesem Button können Sie den erforderlichen Pfad suchen. 

### Feld Dateiendung 
Es ist möglich die Intrastat-Datei auch als CSV abzugeben. Dafür steht dieses Feld zur Verfügung. 

Ist keine Datei-Endung angegeben, wird dies beim ersten Speichern automatisch eingetragen. Dies kann anhand des Mappings (in der gleichlautenden Combobox) erfolgen: 

- Ist das Mapping vom Typ XML, ist die Datei-Endung XML. - Ist das Mapping vom Typ Text, ist die Datei-Endung CSV. 

Tooltip: 

Dateiendung sollte ohne Punkt "." eingegeben werden. 

### Feld Empfänger 
Hier wird eine genaue Beschreibung angegeben. Sie können bis zu 32000 Zeichen angeben (bei Oracle: 2000). 

Tooltip: 

Angabe einer Beschreibung 

### Combobox Intr. BuFa Nr. 
Hier wird die Bundes-Finanzamtsnummer für Intrastat angegeben. Alle Bundesländer stehen zur Verfügung. 

Die Bu/FA-Nummer hat einen 2-stelligen Schlüssel. Dieser 2-stellige Schlüssel wird für die Kennnummer des Auskunftspflichtigen benötigt. 

Bei der Bu/Fa handelt es sich grundsätzlich um die gleichen Bundesländer wie bei der Ursprungs-/Bestimmungsregion, allerdings ohne die 99 und 25. 

Dieser Eintrag ist zwingend vorzunehmen! 

Siehe Kapitel Firmenstamm . 

Tooltip: 

Bundesland des zuständigen Finanzamts für Intrastat. 

### Combobox Codierung 
Hier wird die Zeichencodierung angegeben. 

Siehe Kapitel Formate für Exportdateien - Encodings . 

Tooltip: 

Zeichenkodierung: 

Der Intrastat-Standard ist "Westeuropäisch (ISO)".
