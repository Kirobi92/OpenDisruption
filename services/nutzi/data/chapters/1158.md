# Gruppentitel Registerkarte Allgemein 
Gruppentitel Registerkarte Allgemein 

Auf dieser Registerkarte wählen Sie den entsprechenden Gruppentitel aus. 
### Feld Titel 
Hier wird der Eintrag aus dem Feld Titel im Kopf des Dialogs angezeigt. 

### Feld Beschreibung 
Hier wird die Beschreibung zum selektierten Titel angezeigt. 

Tooltip: 

Angabe einer Beschreibung 

### Button Html-Editor 

Mit diesem Button öffnet sich der Html Editor zur Bearbeitung von Html-Texten. 

Siehe Kapitel Html Editor . 

Tooltip: 

Html-Editor zur Bearbeitung von Html-Texten. 

### Checkbox Kein Einzelpreis 
Wenn Sie diese Checkbox aktivieren, werden in den Formularen (d.h. von Angebot bis Rechnung) keine Einzelpreise eingetragen. Die Funktion für das Unterdrücken des Einzelpreises kann genutzt werden, um detailliertere Preise gegenüber dem Kunden nicht kenntlich zu machen, z.B. bei einem Artikel-Set. 

Bei der Funktion Kein Einzelpreis werden auch alle Positionen der Gruppe automatisch zu einer Positionsgruppe zusammengefasst. 

### Checkbox Kein Gruppenpreis 
Wenn Sie diese Checkbox aktivieren, wird im Formular (d.h. von Angebot bis Rechnung) kein Gruppenpreis eingetragen. 

Tooltip: 

Kein Gruppenpreis
