# Auftragsstückliste Registerkarte Komponente 
Auftragsstückliste Registerkarte Komponente 

Auf dieser Registerkarte definieren Sie die Details für die Komponente, wie z.B. Menge und Positionsnummer. 
### Button Bestand 

Dieser Button öffnet den Dialog Bestand . Hier erhalten Sie eine Übersicht und sehen, ob dieser Artikel Bestand hat. 

Siehe Kapitel Bestandsanzeige . 

### Button Seriennummer 

Dieser Button öffnet den Dialog Seriennummern zuordnen . So können Sie der Komponente eine Seriennummer für eine bessere Nachverfolgung hinterlegen. 

Siehe Kapitel Seriennummern-Zuordnung . 

### Button Chargen 

Dieser Button öffnet den Dialog Chargenabgang . So können Sie der Komponente eine Chargennummer zuordnen. 

Siehe Kapitel Chargenabgang . 

### Button Plätze 

Dieser Button öffnet den Dialog Plätze Auslagerung um festzulegen, von welchem Lagerplatz der Artikel entnommen werden soll. 

Siehe Kapitel Plätze Auslagerung . 

### Button Auflösen 

Wird ein Stücklistenartikel in eine bestehende Auftragsstückliste aufgenommen, so wird nur der Stücklistenartikel selbst gespeichert. Erst wenn dieser markiert und der Button Auflösen geklickt wird, werden alle Komponenten dieser Stückliste ebenfalls in die Auftragsstückliste übernommen. 

### Feld Artikel 
Wenn Sie im Feld auf der linken Seite eine Komponente markieren, wird hier die Artikelnummer angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Button Suche 

Mit diesem Button können Sie einen Artikel suchen. 

Siehe Kapitel Artikelauswahl . 

### Feld Bezeichnung 
Die Bezeichnung des Artikels wird angezeigt. 

Tooltip: 

Artikelbezeichnung 

### Feld Pos 
Hier wird die Positionsnummer angegeben. Diese Nummer muss je Vorgang eindeutig sein! 

Tooltip: 

Positionsnummer 

### Feld Menge 
Hier wird die Menge angegeben. 

Tooltip: 

Angabe der Menge 

### Feld Menge pro 
Hier wird die benötigte Menge der Komponente, je zu produzierender Einheit, angegeben. 

### Feld Lagereinheit 
Hier wird die hinterlegte Lagereinheit angezeigt. 

Tooltip: 

Angabe der Lagereinheit. Dazugehörige Pickliste heißt „Mengeneinheit“. 

### Feld Materialschwund 
In diesem Feld kann ein prozentualer Wert für voraussichtlichen Materialverlust eingetragen werden. Dadurch wird sichergestellt, dass bei der Produktion ein evtl. vorkommender Materialverlust gedeckt ist. Die Datenbanktabelle 'Reservierung' wird entsprechend des %-Werts erhöht. 

### Checkbox Komplett drucken 
Wenn Sie diese Checkbox aktivieren, wird der Druck der Stücklisten-Komponente aktiviert. 

Tooltip: 

Aktiviert den Druck der Stücklistenkomponente.
