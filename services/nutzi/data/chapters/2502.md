# Auswertungsstruktur Registerkarte UVA 
Auswertungsstruktur Registerkarte UVA 

Auf dieser Registerkarte definieren Sie den Aufbau der Umsatzsteuer-Voranmeldung. 

Im Gegensatz zu den anderen Registerkarten werden den Details hier keine Konten, sondern Steuerschlüssel zugeordnet. 

Das Prinzip des Aufbaus ist aber das gleiche wie bei der G+V oder Bilanz. 

Somit ist gewährleistet, dass in der UVA-Excel-Auswertung die Bemessungsgrundlage und der entsprechende Umsatzsteueranteil im Formular eingetragen werden. 

Es besteht die Möglichkeit, die Auswertung der UVA auch für ein anderes Land zu erstellen. 

Wird ein Land gewählt, sollen nur Auswertungsstrukturen zur Auswahl stehen, welche für das entsprechende Land definiert wurden (also auch den entsprechenden Steuerschlüssel enthalten). 

Alle verwendeten Steuerschlüssel dieser Struktur sind in der gleichen Währung zu buchen (gleiche Währung auf dem Steuerkonto). 

Beim Speichern der Auswertungsstruktur wird geprüft, ob die Konten der zugeordneten Steuerschlüssel alle in derselben Währung verschlüsselt sind. Ist dies nicht der Fall, so werden Sie mit einer Hinweismeldung darauf aufmerksam gemacht. 

Berechnung der UVA: 

Anhand der Selektion des Landes kann entschieden werden, ob LW- oder FW-Beträge des Journals als Datenbasis für die UVA dienen. Hierbei wird das Kennzeichen Steuerabrechnung in FW aus den Länderparametern berücksichtigt. 

Es ist möglich die UVA authentifiziert, d.h. mit Zertifikat zu versenden. 

Die XML steht nur zur Laufzeit zur Verfügung und wird nicht gespeichert. 

Für den Versand per ELSTER stehen die Checkbox Mit Signatur und das Feld PIN zur Verfügung. 

Es gibt 3 mögliche ELSTER-Signaturen: 
- Per Software-Zertifikat - Per Sicherheitsstick - Per Signaturkarte 
Das Software-Zertifikat steht in Form einer PFX-Datei zur Verfügung. 

Für die Signierung über Sicherheitsstick oder Signaturkarte ist zwingend eine DLL anzugeben. 
### Combobox Land 
Hier können Sie das Land wählen, für welches Sie eine UVA ansehen möchten. 

Tooltip: 

Angabe des Ländercode gemäß ISO 3166-1 alpha-2 

### Feld Zeilentext 
In diesem Feld legen Sie den Namen für eine Stufe, einen Text oder ein Detail fest. 

Tooltip: 

Zeilentext 

### Feld Summentext 
In diesem Feld können individuelle Summentexte eingegeben werden. Wird hier nichts eingetragen, steht in der Auswertung (im Dialog und im Report) kein Text beim Ergebnis. Wird in Feld Summenzeile nichts eingetragen, wird keine Summe im Dialog und im Report ausgewiesen. 

Tooltip: 

Der Summentext wird in der Auswertung in Spalte Bezeichnung auf der Summenzeile ausgewiesen. 

### Button Hinzufügen 

Tooltip: 

Automatisches Füllen des Feldes Summentext. 

### Radiobutton Stufe 
Innerhalb einer Stufe werden alle Werte der Konten in der Auswertung zu einem Ergebnis zusammengerechnet. 

Sie können beliebig viele Stufen ineinander verschachteln. 

### Radiobutton Text 
Diese Positionen im Strukturaufbau sind reine Texte und dienen dem Layout der Auswertung. 

### Radiobutton Detail 
Nur einem Detail können Konten zugeordnet werden! 

D.h. wenn Sie einer Stufe Konten zuordnen wollen, müssen Sie zuvor innerhalb der Stufe ein Detail einfügen. 

### Radiobutton Basis 
Die Basis bilden die Konten. Anhand ihrer Werte erfolgen die Berechnungen innerhalb einer Stufe.

### Tabellenspalte Steuerschlüssel 
Hier wird der Steuerschlüssel angegeben. 

Tooltip: 

Angabe des Steuerschlüssels 

### Tabellenspalte Währung 
Geben Sie hier das Währungskürzel ein. Es wird verwendet in Auswahl, Anzeige und Druck von Währungen. Die Angabe ist nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

(Es dürfen keine Sonderzeichen z.B. € und $ verwendet werden.) 

Bitte beachten Sie, dass bei der Währung Euro sowohl das Kürzel als auch das ISO-Kürzel mit EUR zu belegen sind! 

Tooltip: 

Angabe des Währungskürzels. Wird verwendet in Auswahl, Anzeige und Druck von Währungen. 
Angabe ist nicht nachträglich änderbar und kann max. 3 Zeichen beinhalten. 

### Feld UVA-Ziffer 
In diesem Feld tragen Sie die entsprechende Ziffer aus der Umsatzsteuer-Voranmeldung ein, z.B. Steuerpflichtige Umsätze – zum Steuersatz von 19 v.H. - UVA-Ziffer 51 oder Abziehbare Vorsteuerbeträge – Vorsteuerbeträge aus Rechnungen von anderen Unternehmen - UVA-Ziffer 66. 

Für die Steuerschlüssel für den Innergemeinschaftlichen Erwerb bzw. die Steuerschlüssel - Umsätze, für die der Leistungsempfänger die Steuer nach § 13b UStG schuldet, müssen Sie das entsprechende Detail im Vorsteuerbereich auf der linken Seite markieren. Danach erhalten Sie Zugriff auf die Checkbox Umkehrung . Aktivieren Sie diese Checkbox, wird im Feld rechts der UVA-Aufbau angezeigt. 

### Checkbox Umkehrung 
Die Checkbox Umkehrung müssen Sie verwenden, wenn Sie Steuerschlüssel für den innergemeinschaftlichen Erwerb gebucht haben, die nach §13 UStG versteuert werden. Im Bereich der Vorsteuer wird nach markieren der Zeile diese Checkbox zugänglich und kann somit gesetzt werden. Im Anschluss daran müssen Sie im Feld daneben die entsprechend zum Vorsteuersatz gelistete Umsatzsteuerzeile markieren. Hierdurch wird gewährleistet, dass die Bemessungsgrundlage und der entsprechende Umsatzsteueranteil im Formular eingetragen werden. Das heißt, die Umsatzsteuer wird im Bereich der Umsatzsteuer gemeldet und gleichzeitig im Bereich der Vorsteuer wieder als Vorsteuerabzug geltend gemacht (Umkehrung Vorsteuer auf Umsatzsteuer).
