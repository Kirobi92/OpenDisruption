# NW365.Dataconnect 
NW365.Dataconnect 

Dieser Dialog öffnet sich im Navigator >> System >> Allgemein >> Schnittstellen >> NW365.Dataconnect . Hier definieren Sie die Einstellungen von eNVenta zum NW365.Dataconnect. 

Mittels der Suche über NW365.Dataconnect können Artikel mit den zugehörigen Merkmalen und Bilder, die von Nordwest gepflegt wurden, in den Artikelstamm übernommen werden. Hierbei werden alle mitgelieferten Daten aus dem Datenportal in eNVenta übernommen. 

Es erfolgt keine Prüfung auf Korrektheit der Daten, diese werden so übernommen wie sie geliefert werden. 
## Steuerelemente des Formulars

Tooltip:

### Combobox Preisgruppe 
Aus dieser Combobox können Sie eine VK-Preisgruppe auswählen, die bestimmt auf welchen Verkaufspreis zugegriffen werden soll. 

Tooltip: 

Hier wählen Sie eine Preisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

### Combobox Importschl.-Typ 
Hier wählen Sie den Typ des Importschlüssels aus, der für das Mapping des Nordwest-Lieferanten auf einen Lieferanten im Stamm verwendet werden soll (vgl. Navigator >> System >> Allgemein >> Import-Schlüssel Verwaltung ). 

Tooltip: 

Importschlüssel-Typ, um den Lieferanten zuzuordnen. 

### Feld Datenbereich 
Hier geben Sie den Datenbereich an, der beim Import eines Artikels in den Schattenstamm verwendet wird. 

Tooltip: 

Die importierten Daten gehören zu einem Datenbereich. Es werden nur Daten desselben Bereichs miteinander aktualisiert. 

Es wird empfohlen, mindestens einen Datenbereich je Datenlieferant zu nutzen. Wird kein Datenbereich definiert, setzt das System automatisch die Importformat-ID als Bereich. 

### Feld Lieferant 
Hier geben Sie die Lieferantennummer an. 

### Feld Besteller 
Hier geben Sie die Nummer des Bestellers an. 

### Checkbox Direkte Preis- und Verfügbarkeitsabfrage 
Wenn dieser Parameter gesetzt ist, werden Preis und Verfügbarkeit direkt in der Artikelauswahl geprüft. Dafür stehen auf der Registerkarte Artikeldienste die Spalten ‚Preis‘ und ‚Verfügbarkeit‘ zur Verfügung. 

Diese Suche kann die Performance beeinträchtigen. 

Tooltip: 

Wenn dieser Parameter aktiv ist, wird direkt in der Artikelauswahl der Preis und die Verfügbarkeit überprüft.
