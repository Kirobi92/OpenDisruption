# Intrastat - Einlesen der Warennummern (Sova-Leitdatei) 
Jedes Jahr stellt das Statistische Bundesamt eine Datei mit den gültigen Warennummern zur Verfügung. Die sogenannte Sova-Leitdatei. Diese hat ein definiertes Format und muss in eNVenta eingelesen werden. 

Für den Import wird die Datei XX_SOVAyyyy.txt verwendet (z.B. 18_SOVA2020.txt). Die Datei wird in einer zip-Datei zur Verfügung gestellt und zuerst durch den Benutzer entpackt.
