# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtNoNegStockValues

Wenn Sie diesen Parameter aktivieren, wird verhindert, dass in einem LVS-Lager negative Lagerplatzbestände entstehen. 

Tooltip: 

Keine negativen Lagerbestände
