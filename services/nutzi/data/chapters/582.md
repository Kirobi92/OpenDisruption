# Parameter Office Registerkarte SMTP

Grundeinstellungen für eNVenta ERP / Parameter Office 

# Parameter Office Registerkarte SMTP 
Parameter Office Registerkarte SMTP 

Auf dieser Registerkarte nehmen Sie die Einstellungen für den Email-Versand über SMTP vor. Dafür müssen Sie zuvor die Email-Adresse im Menü Sachbearbeiter hinterlegen. Diese wird dann standardmäßig als Absender verwendet. 

SMTP-Dienst (http://de.wikipedia.org/wiki/Simple_Mail_Transfer_Protocol) 

Die Einstellung wird pro PC und Benutzer vorgenommen. Benötigt ein Benutzer z.B. den E-Mail-Import, benötigt er weiterhin einen Outlook-Client und kann somit auch weiterhin diesen zum Versand von E-Mails nutzen. 

Siehe Kapitel Anbindung von Office 365 . 

Derzeit gibt es keine Option im .NET Framework, die eine TLS (Implicit SSL) Übertragung ermöglicht. 
### Radiobutton Systemeinstellung 
Wenn Sie diesen Radiobutton aktivieren, wird der SMTP-Parameter genutzt, wenn er eingestellt ist. 

### Radiobutton Ja 
Wenn dieser Radiobutton gesetzt ist, wird immer ein SMTP genutzt. Dieser muss dann auf dem Broker eingestellt sein. 

### Radiobutton Nein 
Wenn dieser Radiobutton aktiviert ist, wird niemals ein SMTP-Server genutzt.
