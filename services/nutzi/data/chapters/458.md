# eNVenta Init 
eNVenta Init 

Dieser Dialog öffnet sich über System >> Initialisierung >> eNVenta Init und steht für die Initialisierung von eNVenta ERP zur Verfügung. Bei der Initialisierung geben Sie Ihre Firmendaten, die wichtigsten Sachkonten usw. an. Zusätzlich können Sie auch die Sachkontenrahmen initialisieren, welche mit eNVenta ERP mitgeliefert werden. 

Damit Sie die vorgefertigten Html Reports nutzen können, müssen Sie diese über die Initialisierungs-Funktion von eNVenta einspielen. 

Aktivieren Sie als erstes die Option Nur Import Items , damit Sie keine bestehenden Daten überschreiben. 

Wählen Sie im Dialog Import Item Auswahl den Eintrag HtmlReports aus und klicken auf Button um die Html Reports einzulesen. 

Im Dialog eNVenta Init klicken Sie auf den Button Start Init . 

Nach dem Import der Html Reports erscheint eine Mitteilung und anschließend stehen die Reports in den Vorgängen entsprechend zur Verfügung. 

Bei einem nochmaligen Import ist folgendes zu beachten: 

- Es werden neue Datensätze ergänzt, vorhandene aber unverändert belassen. Dadurch können neue Reports ergänzt werden. - Aber: Gelöschte Datensätze (z.B. Properties) aus vorhandenen Reports werden wiederhergestellt. - Schlüsselkonflikte mit individuell ergänzten Datensätzen können nicht ausgeschlossen werden (z.B. Templates). Dies führt dazu, dass etwaige Datensätze mit gleichen Schlüsselfeldern nicht importiert werden. 
Die Registerkarte Export bietet einen Button zum Start des Report Exports an.
