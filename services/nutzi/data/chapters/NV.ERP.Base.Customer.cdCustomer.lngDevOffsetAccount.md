# Kapitel NV.ERP.Base.Customer.cdCustomer.lngDevOffsetAccount

In dieses Feld können Sie eine Kontonummer für eine standardmäßige Gegenbuchung eingeben. 

Diese Einstellung zieht ausschließlich bei manuellen Buchungen in der Finanzbuchhaltung.
