# Lagerplätze Registerkarte Artikelinfo

Allgemeine Funktionalitäten in eNVenta ERP / Lagerverwaltung / Lagerplätze 

# Lagerplätze Registerkarte Artikelinfo 
Lagerplätze Registerkarte Artikel Info 

Auf dieser Registerkarte erhalten Sie zum ausgewählten Artikel detaillierte Informationen. Die Spalten ‚Min. Menge’ und ‚Max. Menge’ sind editierbar. 

Wenn auf dem Lagerplatz keine Maximalmenge (Spalte ‚Menge Max.‘) eingegeben wurde, ist die maximale Menge manuell auszurechnen. Hierfür müssen die Nettoabmessungen aus dem Artikelstamm mit den Abmessungen vom Lagerplatz verglichen werden. 
### Feld Lagerplatz 
Hier wird die Nummer des markierten Lagerplatzes (aus der linken Tabelle) angezeigt. 

Tooltip: 

Angabe des Lagerplatzes 

### Feld Artikel 
Hier geben Sie die Nummer des Artikels ein, der auf den Lagerplatz gebucht werden soll. Anschließend drücken Sie den Button und übertragen so den Artikel in die Tabelle. Hier können Sie noch die Mindest- und Maximalmenge angeben. 

Tooltip: 

Geben Sie den Artikel für den Lagerplatz ein. 

### Feld Bezeichnung 
Hier wird der Name zur Artikelnummer angezeigt. 

Tooltip: 

Zeigt den Artikelnamen. 

### Tabelle 

### Tabellenspalte Artikel 
Hier wird die erste zugeordnete Artikelnummer angezeigt. 

Tooltip: 

Zeigt die zugeordnete Artikelnummer an. 

### Tabellenspalte Lagereinheit 
Die Lagereinheit des Artikels wird angezeigt. 

Tooltip: 

Zeigt die Lagereinheit des Artikels. 

### Tabellenspalte Faktor VK 
Der Verkaufsfaktor des Artikels wird angezeigt. 

Tooltip: 

Zeigt den VK-Faktor 

### Tabellenspalte Min. Menge 
Hier wird die Menge angezeigt, die mindestens im Lager vorhanden sein muss. 

Tooltip: 

Zeigt den Mindest-Lagerbestand. 

### Tabellenspalte Max. Menge 
Hier wird die Menge angezeigt, die maximal im Lager vorhanden sein darf. 

Tooltip: 

Zeigt die max. Lagermenge des Artikels. 

### Button OK 

Mit diesem Button speichern Sie die Einträge der Tabelle. 

### Button Satz hinzufügen 

Wenn Sie eine Artikelnummer in das Feld Artikelnr . eingetragen haben und drücken diesen Button, wird der Artikel in die Tabelle eingetragen und Sie können noch Mindest- und Maximalmenge eingeben. 

### Button Löschen 

Mit diesem Button können Sie einen Eintrag aus der Tabelle entfernen.
