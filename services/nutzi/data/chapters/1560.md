# Tourengruppenzuordnung 
Tourengruppenzuordnung 

Dieser Dialog öffnet sich im Kunde >> Optionen >> Tourengruppenzuordnung . 

Hier können Sie einem Kunden Tourengruppen zuordnen. 

Bei Zuordnung einer oder mehrerer Tourengruppen im Kundenstamm wird der Kunde nur über diese Tourengruppen beliefert. Die ermittelte Tour muss zu einer der angegebenen Tourengruppen gehören. Eine Ermittlung der verfügbaren Touren über die PLZ oder weitere Zuordnungen erfolgt in diesem Falle nicht mehr! 

Die Angabe mehrerer Tourengruppen ist alternativ zur Auswahl einer Stammtour möglich. Ist im Kundenstamm eine Stammtour vorgegeben, so hat diese Einstellung Priorität. 

Die Zuordnung von Tourenvorgaben erfolgt wie folgt: 

- Stammtour aus Kundenstamm - Routen aus Kundenstamm - Anhand PLZ-Zuordnung bzw. Tourenstamm 
Die einzelnen Stufen ergänzen sich nicht, jede Stufe wird für sich analysiert.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern!
