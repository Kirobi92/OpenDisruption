# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtNoChangeOfSGAtSP

Damit nich aus Versehen Lagerplätze bei einer Einlagerung überschrieben werden, aktivieren Sie diesen Parameter. 

Tooltip: 

Ist diese Checkbox aktiviert, so kann der Lagerplatz bei der Einlagerung nicht durch den Lagermitarbeiter geändert werden.
