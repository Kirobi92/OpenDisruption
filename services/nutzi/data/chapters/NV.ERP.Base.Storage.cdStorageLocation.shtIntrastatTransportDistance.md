# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtIntrastatTransportDistance

Hier können Sie den Transportweg bis zur Grenze in % angeben. Dies vor dem Hintergrund, dass es sich um einen statistischen Wert für eine ungefähre Hochrechnung handelt. Deshalb muss der Sachbearbeiter hier den wahrscheinlichsten bzw. durchschnittlich zutreffenden Prozentsatz hinterlegen. 

Beispiel: 
Transportweg bis Adresse = 100 km 
Weg bis Grenze = 40 km 
=> Transport Distanz = 40
