# Kopieren einer Bestellung 
Beleg kopieren 

eNVenta ERP bietet eine einfache Möglichkeit, ganze Bestellungen zu kopieren. Dazu muss die zu kopierende Bestellung in den Bestell-Dialog eingespielt und das Menü Optionen >> Beleg kopieren ausgewählt werden. 

Das Kopieren eines Belegs auf eine andere Währung funktioniert nur, wenn die Checkbox Neukalkulation aktiviert ist. Diese Neukalkulation sorgt für die Preisumstellung. Ist die Checkbox Neukalkulation nicht gesetzt, sind alle bestehenden Preise auf die abweichende Währung umzurechnen. 

Haben Sie einen Lieferanten mit einer ähnlichen oder identischen Bestellung im Bestand, kann diese für den aktuellen Lieferanten kopiert werden. Ändern Sie dazu durch Eingabe der Lieferantennummer oder mit Hilfe des Buttons den Lieferanten. 

Auch vorbereitete Musterbestellungen können so für jeden beliebigen Lieferanten übernommen werden. 

Beim Kopieren einer Bestellung bzw. Belastung wird das Wunschdatum mit dem Tagesdatum aktualisiert. Beim Kopieren einer Anfrage wird das Original-Wunschdatum nicht verändert. 

Beim Kopieren einer Bestellung wird die Einkaufart mit der Standard-Einkaufart aus dem Lieferantenstamm vorbelegt. 

Beim Kopieren eines Belegs (des Einkaufs) werden die Positionsnebenkosten auf null gesetzt. Ausnahme: Dies gilt nicht für das Kopieren einer Belastung in eine andere Belastung.

### Button Speichern 

Mit diesem Button wird der Kopiervorgang gestartet. Nach erfolgreicher Beendigung erhalten Sie eine entsprechende Meldung. Es wird automatisch eine neue Belegnummer aus dem jeweiligen Nummernkreis vergeben. Der neu generierte Beleg wird sofort in den Dialog eingespielt. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Combobox Belegart 
Über diese Combobox kann die Bestellung in eine andere Belegart kopiert werden. Sie finden in der Liste der Belegarten außer den Einkaufarten auch die Einträge Anfrage und Belastung . 

Beispielsweise kann durch die Auswahl „Anfrage“ aus der Bestellung eine Anfrage erzeugt werden. Auf diese Weise ersparen Sie sich die Arbeit, die Anfrage komplett neu zu erfassen. 

Tooltip: 

Geben Sie die Einkaufsart an. 

### Feld Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Button Suche 

Mit diesem Button können Sie einen Lieferanten suchen. 

Siehe Kapitel Lieferantenauswahl . 

### Feld Firma 1 
Die Firmenbezeichnung wird automatisch vom System eingetragen, nachdem die Lieferantennummer eingetragen wurde. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Checkbox Nur markierte Positionen 
Ist diese Checkbox aktiviert, wird der Beleg nur für die markierte Bestellposition kopiert. 

Tooltip: 

Wenn aktiviert, wird der Beleg nur für die markierten Bestellpositionen kopiert. 

### Checkbox Stornierte Positionen 
Wenn Sie diese Checkbox aktivieren, können beim Kopieren des Leitbelegs auch stornierte Positionen mit kopiert werden. 

Tooltip: 

Stornierte Positionen 

### Checkbox Serienanfrage 
Wenn Sie eine Serienanfrage kopieren, ist die Checkbox Serienanfrage automatisch gesetzt. Sie kann aber deaktiviert werden. 

Tooltip: 

Diese Checkbox ist automatisch gesetzt, wenn Sie eine Serienanfrage kopieren. 

### Checkbox Wertbelastung 
Diese Checkbox ist nur aktiv, wenn als Einkaufart „Belastung“ ausgewählt wurde. Wird diese Checkbox gesetzt, so wird die Checkbox im „Ziel-Beleg“ gesetzt. 

Tooltip: 

Aktiviert die Wertbelastung im Ziel-Beleg. 

### Checkbox Lieferanschrift aktualisieren 
Ist diese Checkbox gesetzt, wird die Lieferanschrift aktualisiert. 

Tooltip: 

Lieferanschrift aktualisieren 

### Checkbox Lager aktualisieren 
Wenn Sie diese Checkbox aktivieren, werden alle Lagerorte nach der Einkaufart aktualisiert. 

Tooltip: 

Alle Lagerorte werden nach der Einkaufsart aktualisiert. 

### Checkbox Steuer/Konto aktualisieren 
Mit Aktivierung des Parameters Steuer und Aufwandskonto neu werden die Steuerschlüssel für die Bestellpositionen aus dem Artikel bzw. den Warengruppen neu ausgelesen und die Bestellposition damit aktualisiert. 

Tooltip: 

Wenn aktiviert, werden Steuerschlüssel für die Bestellpositionen aus dem Artikel bzw. den 
Warengruppen neu ausgelesen und die 
Bestellposition damit aktualisiert. 

### Checkbox Neukalkulation 
Ist diese Checkbox aktiviert, werden die Preise und Rabatte automatisch aktualisiert. 

Tooltip: 

Wenn aktiviert, werden die Preise und Rabatte automatisch aktualisiert. 

### Checkbox Kursaktualisierung 
Wenn diese Checkbox aktiviert wird, wird der Kurs für die Kopie neu berechnet. 

Tooltip: 

Falls aktiviert, wird der Kurs für die Kopie erneut berechnet.
