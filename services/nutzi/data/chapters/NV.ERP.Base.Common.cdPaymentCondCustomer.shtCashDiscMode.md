# Kapitel NV.ERP.Base.Common.cdPaymentCondCustomer.shtCashDiscMode

Wenn Sie den Parameter aktivieren, beginnt die Skontofrist sofort, d.h. mit dem Valuta-Datum bzw. Druckdatum. 

Wenn dieser Parameter nicht gesetzt wird, gilt der ‚Tag des Monats‘ als Startdatum für die Skontofrist.
