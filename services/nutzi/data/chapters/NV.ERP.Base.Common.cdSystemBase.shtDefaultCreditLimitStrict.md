# Kapitel NV.ERP.Base.Common.cdSystemBase.shtDefaultCreditLimitStrict

Ist zusätzlich zu einem Eintrag im Feld Kreditlimit WW der Parameter Strikt gesetzt, können bei Überschreitung des Kreditlimits keine Aufträge erfasst werden. Es sei denn im Dialog Parameter Allgemein ist ein Passwort für das strikte Kreditlimit hinterlegt und dieses wird bei der Auftragserfassung eingegeben. 

Im Dialog Sachbearbeiter kann für einen Sachbearbeiter die Checkbox Striktes KL gesetzt werden. Dieses übersteuert dann die Einstellung des strikten Kreditlimits aus den allgemeinen Parametern, wenn dies nicht gesetzt ist. 

Siehe Kapitel Parameter Allgemein . 

Siehe Kapitel Sachbearbeiter . 

Striktes Kreditlimit bedeutet: 
Ist zu einem Kreditlimit (definiert im Kundenstamm) der Parameter Strikt gesetzt, können bei Überschreitung des Kreditlimits keine Aufträge erfasst werden. Nur ein Sachbearbeiter mit der entsprechenden Berechtigung kann das definierte Kreditlimit überschreiten. 

Siehe Kapitel Parameter Verkauf . 

Siehe Kapitel Sachbearbeiter . 

Tooltip: 

Wenn die Checkbox gesetzt ist, wird strikt das Standard-Kreditlimit gesetzt.
