# Daten Historisieren Registerkarte Allgemein

Administration / Daten Historisierung / Daten Historisieren 

# Daten Historisieren Registerkarte Allgemein 
Daten Historisieren Registerkarte Allgemein 

Auf dieser Registerkarte nehmen Sie die Einstellungen zur Historisierung vor. 
### Bezeichnungsfeld Tabellen 
Tooltip: 

Tabellen, welche für die Historisierung zur Verfügung stehen und aktiviert sind. 

### Tabelle 

Hier werden die Tabellen zur Historisierung vorgeschlagen. 

### Button Nach rechts übertragen 

Mit diesem Button weisen Sie eine Tabelle zu, die historisiert werden soll. 

Tooltip: 

Weisen Sie die Tabelle der aktuellen Definition zu. 

### Button Nach links übertragen 

Mit diesem Button wird die Zuordnung wieder gelöscht. 

Tooltip: 

Die Zuordnung der Tabelle zu der Historisierungsdefinition wird gelöscht. 

### Tabelle 

Hier werden die ausgewählten Tabellen angezeigt. 
### Tabellenspalte Datum bis 
Bis zu dem hier angegebenen Datum ist definiert, dass jüngere Datensätze im Produktivteil sind, die älteren in der Historie. 

Tooltip: 

Bis zu diesem Datum ist definiert, dass jüngere Datensätze im Produktivteil sind, ältere in der Historie. 

### Tabellenspalte Transferdatum 
Datensätze, welche ein Datum haben das älter ist als dieses Datum, wurden bereits in die Historie ausgelagert. 

Tooltip: 

Datensätze welche älter sind, wurden bereits in die Historie ausgelagert. 

### Tabellenspalte Anzahl Produktiv 
Hier wird die Anzahl der produktiven Datensätze angegeben. 

Tooltip: 

Anzahl der produktiven Datensätze 

### Tabellenspalte Anzahl Historie 
Hier wird die Anzahl der historisierten Datensätze angegeben. 

Tooltip: 

Anzahl der historisierten Datensätze 

### Tabellenspalte Historisierung Aktivieren 
Wenn diese Checkbox aktiviert ist, wird die Historisierung für diese Tabelle ausgeführt. 

Tooltip: 

Bei aktivierter Checkbox wird die Historisierung für diese Tabelle ausgeführt. 

### Tabellenspalte Null historisieren 
Wenn diese Checkbox aktiviert ist, werden NULL-Datumswerte berücksichtigt und historisiert. 

Tooltip: 

Wenn die Checkbox gesetzt ist, werden NULL Datumswerte berücksichtigt und historisiert.
