# Definition eines Massendrucks 
Massendruck Def. 

Der Button Neu im Dialog Massendruck öffnet diesen Dialog, um einen neuen Massendrucklauf zu definieren. 

Führen Sie jetzt mit folgenden Schritten die Definition des Drucklaufs durch: 

- Wählen Sie aus der Combobox Druckbelegart das gewünschte Formular. Es stehen Ihnen alle Standardformulare der Bestellverwaltung zur Verfügung. Alternativ dazu können auch zusätzliche, von Ihnen definierte Formulare verwendet werden. - Im nächsten Schritt können Sie die Selektionsbedingung angeben. Sie haben dazu das Auswahl-Box Spalten . Hier stehen alle verwendbaren Datenfelder aus Bestellposition, Bestellkopf, dem Lieferanten- und Artikelstamm zur Verfügung. Mit Hilfe dieser Felder, den Operatoren, dem Eintrag im Feld Wert und den logischen Verknüpfungen AND / OR können beliebige Selektionen vorgenommen und mit dem Button Dazu zusammengestellt werden. - Auch die direkte Eingabe in das Feld SELECT … WHERE ist möglich. Es besteht die gleiche Funktionalität, wie im Kapitel Suchen mit SQL erläutert wird. Bitte beachten Sie, dass die Selektionsbedingung der Schreibweise und den Vorschriften der Sprache SQL genügen muss! Wenn keine Selektionsbedingung angegeben ist, wird das Formular für alle Bestellungen oder Anfragen gedruckt, für die der jeweilige Druck noch nicht erfolgt ist. Die Anzeige in diesem Feld beruht immer auf der ausgewählten Tabelle. 
Bitte beachten Sie, dass die Selektionsbedingungen der Schreibweise und den Vorschriften der Sprache SQL genügen müssen! 

Siehe Kapitel Allgemeine Funktionalitäten in eNVenta ERP / Suchfunktionen . 

Bei Zusatzformularen wird keine Rücksicht darauf genommen, ob schon einmal ein Druck erfolgt ist! 

Die Felder Eink. Wert > / Eink. Wert < geben Ihnen die Möglichkeit, eine zusätzliche Selektion nach dem Umsatz des Belegs vorzunehmen. Tragen Sie einfach die gewünschte Zahl in eines der Felder ein. Auch eine Kombination ist möglich. 

Legen Sie jetzt die gewünschte Sortierfolge der Drucke mit der Combobox Sortierung fest. Falls Sie auf eine Sortierung verzichten, erhöht sich die Geschwindigkeit des Massendrucklaufs etwas. 

Geben Sie zuletzt in das Feld Bezeichnung eine kurze, aber präzise Erklärung des Zwecks bzw. der Selektion dieses Massendrucklaufs ein. 

Ohne Angabe einer Beschreibung kann die Definition nicht gespeichert werden! 

Im Dialog Massendruck Def. kann (analog zum Verkauf) ein Drucker hinterlegt werden. Dazu steht Ihnen die Combobox Druckername zur Verfügung. 

Es ist notwendig, hier einen Drucker auszuwählen. Dieser Drucker wird jedoch nur dann verwendet, wenn im Massendruck kein Standard-Drucker hinterlegt ist. 

Wie im Verkauf kann auch hier ein Druckmodus ausgewählt werden. Zur Verfügung stehen Druck und Email > Druck für den Versand per Email. 

In das Feld Vorlagennr geben Sie die gewünschte Dokumentvorlage ein oder wählen Sie aus dem Dialog Vorlagenauswahl aus.

### Checkbox Briefkopf 
Diese Checkbox wird aus dem Dialog Parameter Allgemein (Registerkarte Allgemein ) vorbelegt und kann hier im Einzelfall deaktiviert werden. 

Erläuterung: 

Im Rahmen von Geschäftsprozessen ist es immer wieder notwendig, Belege physisch oder digital als Ausdrucke bereitzustellen. Aus rechtlichen Gründen muss dies bei offiziellen Dokumenten mit bestimmten Informationen (Anschrift, BLZ, …) – in Kopf- und Fußzeile - geschehen. Diese Informationen werden im Folgenden als Briefkopf bezeichnet. 

Dies können Kunden nutzen, die ihre Firmeninformationen direkt mit dem Druck des Belegs auf ein Blanko-Papier drucken möchten. 

Die Checkbox Briefkopf dient in den Dialogen als Vorbelegung und im automatisierten Druck als Einstellung für den Druck. 

Im Einzelfall kann in den Druck-Dialogen der einzelnen Vorgänge die Checkbox Briefkopf deaktiviert werden, falls Sie ausnahmsweise auf ein Briefpapier drucken möchten. 

Tooltip: 

Wenn aktiv, wird der Briefkopf z.B. Logo, Absender auf den Beleg gedruckt.
