# Nachlieferung 
Auswahl Belegart 

Wenn Sie den Button Nachlief . im Dialog Retoure drücken, erscheint eine Abfrage, ob Sie eine Nachlieferung erzeugen wollen. Wenn Sie hier Ja drücken, öffnet sich der Dialog Auswahl Belegart zur Auswahl der Belegart (z.B. Auftragsart) mit der eine Nachlieferung erzeugt werden soll. Für die markierten Positionen kann ein neuer Verkaufsauftrag erstellt werden. Für die Erstellung der Nachlieferung muss die Retourenposition nicht zuerst freigegeben werden, diese kann sofort nach dem Wareneingang erstellt werden. 

Wenn trotz nicht vorhandener Freigabe einer Retoure bereits eine Vorab-Information an den Lieferanten notwendig ist, so kann über den Menüpunkt Druck >> Dokument erstellen die Benachrichtigung erfolgen. Diese Dokumente können archiviert werden, um die Nachvollziehbarkeit zu gewährleisten. 

Standardmäßig wird die Auftragsart des Kunden vorbelegt, wenn diese im Kundenstamm hinterlegt ist. 

Über die Auftragsart kann gesteuert werden, ob die Nachlieferung mit Berechnung erfolgt oder auch ohne Berechnung, z.B. weil auch keine separate Gutschrift für den Kunden erstellt wird. Die Combobox der Auftragsart ist anhand der Berechtigung des Sachbearbeiters gefüllt. 

Bei der Erstellung des Verkaufsauftrags stehen alle Funktionen wie bei anderen Aufträgen zur Verfügung. Für die Nachverfolgbarkeit wird die Auftragsnummer in die beteiligten Retourenpositionen eingetragen. Hierdurch ist zusätzlich sichergestellt, dass keine doppelte Nachlieferung erzeugt werden kann. 

Werden mehrere Positionen zur Nachlieferung herangezogen, so werden die Positionen vom Lager der 1. selektierten Position abgebucht. 

Wird die verfügbare Menge unterschritten, so wird die Position im Nachlieferungsauftrag automatisch auf Status 1 gesetzt.
