# Zusatzkosten in der Rechnungsprüfung 
Zusatzkosten RP 

Neben der eigentlichen Bestellung der Waren kann es passieren, dass noch weitere Kosten hinzukommen, welche nicht auf den gleichen Vorgang bzw. Lieferanten zurückzuführen sind. Dies können z.B. Speditionskosten oder Verzollungskosten sein, welche unter Umständen sogar erst Wochen später berechnet werden. Für die Erfassung solcher Zusatzkosten kann die ursprüngliche Rechnung aufgerufen werden und eine weitere Rechnung in Kurzfassung erfasst werden. Dafür steht Ihnen der Dialog Zusatzkosten RP zur Verfügung. Um diesen Dialog zu öffnen, muss in der Rechnungsprüfung der Button Zusatzkosten geklickt werden. 

Zusatzkosten werden netto erfasst. Dies vor dem Hintergrund, dass Kosten im Lagerjournal netto ausgewiesen werden. 

Der Button Zusatzkosten ist erst aktiv, wenn die Rechnungsprüfung bereits abgeschlossen ist! 

In den Feldern Lieferant und Rechnung werden automatisch die Daten aus der Rechnungsprüfung eingetragen. 

Über den Button kann eine neue Zeile erstellt werden, in der die Daten der Zusatzrechnung erfasst werden. 

Diese umfasst die gleichen Felder wie eine normale Rechnungsprüfung und dient der schnellen Erfassung zusätzlicher Rechnungen. Sobald die Zusatzrechnung über den Button eingefügt wurde, kann die Rechnung über den Button Position Buchen verbucht werden. D.h. die Kosten werden in die Nebenkosten im Artikelstamm bzw. die Auftragsposition geschrieben. Im Fall von Fremdfertigung werden die Kosten in den Betriebsauftrag geschrieben. 

Eine Wertermittlung für Divers-Artikel findet nicht statt. 

Zu diesem Zeitpunkt werden alle Tabellen bzw. Datensätze aktualisiert, welche mit der Ursprungs-Rechnung bzw. –Bestellung zusammenhängen. So wird z.B. die zugehörige Auftragsposition zu der Bestellung aktualisiert, indem die Nebenkosten der Position angepasst werden. 

Wenn der Parameter Lagerspez. GEK-Ermittlung im Dialog Parameter Allgemein , Registerkarte Wawi gesetzt ist, erfolgt eine Wertkorrekturbuchung ins Lagerjournal. 

Sind mehrere Positionen einer Bestellung betroffen, so werden die Kosten der Zusatzrechnung automatisch verteilt. Die Art der Verteilung (nach Wert oder Gewicht) wird in den Einkaufsparametern eingestellt. 

Ist der Parameter Keine NK-Aktualisierung in der Rechnungsprüfung aktualisiert, werden Artikel-, Auftragspositionen und Positionen im Betriebsauftrag nicht aktualisiert. 

Siehe Kapitel Parameter Einkauf .

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Button Position Buchen 

Mit diesem Button können Sie die Rechnung buchen. D.h. die Zusatzkosten werden in die Nebenkosten im Artikelstamm, die Auftragsposition bzw. in den Betriebsauftrag geschrieben. 

Bereits verbuchte Zusatzkostenpositionen (Status > 1) sind unveränderlich und können auch nicht mehr gelöscht werden.
