# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtPackagingProcess

Wenn Sie diesen Parameter aktivieren, wird der Verpackungsvorgang vom Kommissionieren gelöst. Eine separate Verpackung wird aktiv und Dialog Packplatz kommt zum Einsatz. 

Siehe Kapitel Packplatz . 

Tooltip: 

Ist dieser Parameter gesetzt, so wird die Ware nach der Kommissionierung abgestellt und zu einem späteren Zeitpunkt verpackt.
