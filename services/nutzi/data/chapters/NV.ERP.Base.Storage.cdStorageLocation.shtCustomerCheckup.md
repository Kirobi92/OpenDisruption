# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtCustomerCheckup

Ist dieser Parameter gesetzt, prüft das Programm bei der Belegerfassung, ob der Kunde auf das Lager zugelassen ist und lehnt gegebenenfalls die Erfassung ab. Die Zuordnung von Kunden zu einem oder mehreren Lagern kann im Kundenstamm vorgenommen werden. 

Tooltip: 

Wenn aktiviert, prüft das Programm bei der Belegerfassung, ob der Kunde auf das Lager zugelassen ist 
und lehnt gegebenenfalls die Erfassung ab.
