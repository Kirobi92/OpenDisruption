# Projekt Registerkarte Gewerke 
Projekt Registerkarte Gewerke 

Da ein Projekt mehrere Gewerke beinhalten kann, wie z.B. Sanitär, Heizung, Stahl und zu jedem Gewerk entsprechende Kunden zugeordnet werden können, steht diese Registerkarte zur Verfügung. Hier werden Gewerke erfasst und können den Projektteilnehmern zugeordnet werden. 

Die zur Verfügung stehenden Gewerke werden in einer Pickliste gepflegt. 

Im linken Bereich der Registerkarte wählen Sie das Gewerk aus. Sollte das gewünschte noch nicht in der Tabelle vorhanden sein, ordnen Sie es über den Button zu. 

Im rechten Teil können Sie die Projektteilnehmer zuordnen. Wenn Sie ein Gewerk markieren, werden bereits zugeordnete Projektteilnehmer zu den Gewerken angezeigt. Es ist möglich weitere Teilnehmer zuzuordnen oder auch wieder zu entfernen. 

Ein Teilnehmer kann mehreren Gewerken zugeordnet werden. 

In diesem Zusammenhang gibt es im Dialog Projektübersicht die Möglichkeit nach Gewerken zu selektieren. 

Ist ein Gewerk ausgesucht, werden alle Projekte angezeigt, in denen es eine Zuordnung eines Teilnehmers zu einem „vorgegebenen“ Gewerk gibt. 
### Tabelle Gewerke 

In dieser Tabelle stehen die bereits angelegten Gewerke zur Selektion zur Verfügung. 
### Tabellenspalte Gewerke 
Hier wird das Gewerk angegeben. 

Tooltip: 

Angabe des Gewerks 

### Tabellenspalte Top Gewerk 
Wenn Sie diesen Parameter aktivieren, wird ein Gewerk im Projekt als Top-Gewerk gekennzeichnet. Über diese Kennzeichnung kann auch die Nachverfolgung stattfinden. 

Tooltip: 

Kennzeichnung des Gewerks als Top-Gewerk 

### Button Satz hinzufügen 

Mit diesem Button kann ein Beteiligter zugeordnet werden. 

Dieser Button ist erst zugänglich, wenn das Projekt gespeichert wurde. 

### Button OK 

Mit diesem Button speichern Sie die Einstellungen. 

Dieser Button ist erst zugänglich, wenn das Projekt gespeichert wurde. 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag aus der Tabelle. 

Dieser Button ist erst zugänglich, wenn das Projekt gespeichert wurde. 

### Tabelle Teilnehmer 

Hier können Sie die Projektteilnehmer zuordnen. Wenn Sie ein Gewerk markieren, werden bereits zugeordnete Projektteilnehmer zu den Gewerken angezeigt. Es ist möglich weitere Teilnehmer zuzuordnen oder auch wieder zu entfernen. 
### Tabellenspalte AKL-Nummer 
Hier wird eine Adress-/Kunden- oder Lieferantennummer angegeben. 

Tooltip: 

Angabe der Adress-, Kunden- oder Lieferantennummer. 

### Tabellenspalte AKL-Typ 
Hier wird der AKL-Typ angegeben. 

Zur Verfügung stehen: 
A = Adresse 
K = Kunde 
L = Lieferant 
V = Vertreter 
P = Kontakt 

Tooltip: 

AKL-Typ: 
A = Adresse 
K = Kunde 
L = Lieferant 
V = Vertreter 
P = Kontakt 

### Tabellenspalte Firma 1 
Hier wird die Firmenadresse angegeben. 

Für die Eingabe der Adresse stehen bis zu 4 Zeilen zur Verfügung (Firma 1 - 4). 

Die Eingabe der Firma 1 ist Pflicht! 

Der Firmenname wird automatisch eingelesen, wenn Sie eine AKL-Nummer eingeben. 

Tooltip: 

Zeile 1 des Firmennamens. 

### Tabellenspalte Ort 
Hier wird der Ort angegeben. 

Tooltip: 

Angabe des Orts
