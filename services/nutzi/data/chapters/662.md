# Lieferant Registerkarte Logo 
Lieferant Registerkarte Logo 

Diese Registerkarte bietet die Möglichkeit Lieferanten-Logos einzubinden. 

Das Lieferanten-Logo wird beim BME-Cat Export der Katalogdaten verwendet. 
### Button Suche 

Mit diesem Button können Sie ein Logo in die Registerkarte übernehmen. 

### Button Löschen 

Mit diesem Button können Sie ein Logo entfernen.
