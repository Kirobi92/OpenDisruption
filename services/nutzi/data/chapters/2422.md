# Zinsrechnung 
Zinsrechnung 

Im Dialog Offene Posten Debitoren wird nach Markieren einer Zeile der Button Zinsen aktiv. Mit ihm öffnet sich der Dialog Zinsrechnung . Die Zinsrechnung kann auch im Buchen >> OP >> Zinsrechnung aufgerufen werden. 

In diesem Dialog können Sie Zinsen für einen Kunden anlegen und berechnen. Voraussetzung dafür ist allerdings, dass im Kunde Registerkarte Rechnungswesen die Checkbox Zinslauf aktiviert ist. Sie können die Berechnung der Zinsen auch an diversen Kriterien festmachen, z.B. an der Mahnstufe. 

In den oberen Feldern erscheint der ausgewählte Kunde. 

Ihre Auswahl können Sie durch die Felder Belegdatum von / bis, Belegnr. von / bis und die Radiobuttons Offene, Ausgeglichene und Alle einschränken. 

Die Felder Zinssatz Ford. und Zinssatz Guth. werden mit dem Zinssatz aus den Zinsparametern vorbelegt, können aber überschrieben werden. 

Das Feld Zinsdatum wird mit dem Tagesdatum vorbelegt, kann aber überschrieben werden. 

Das Feld Karenztage wird aus den Zinsparametern vorbelegt, kann aber überschrieben werden. 

In der Tabelle werden alle Belege gemäß Ihren Vorgaben im Kopf des Dialogs angezeigt. 

Bei einer späteren Neuberechnung angefallener Zinsen erscheint der OP über Zinsen dann nicht in der Tabelle; damit ist eine Zinseszinsrechnung ausgeschlossen. 

Für den Massendruck in der Zinsrechnung gibt es für den Postausgang die Möglichkeiten E-Mail, Fax und Drucken . Wird im Druckdialog zuerst E-Mail gedruckt, werden die E-Mails versandt. Der Druckdialog bleibt nach Fertigstellung weiterhin geöffnet, der Radiobutton E-Mail ist verschwunden. Jetzt können Faxe versandt oder Belege gedruckt werden. 

Der Versand von E-Mails ist nur über SMTP möglich! 

Die Fax-Nummern bzw. Email-Daten werden in Spalten im Dialog angezeigt. 

Neue OPs in einem Vorschlag sind automatisch gespeichert und damit im gleichen Dialog für einen anderen Benutzer nicht mehr verfügbar. Das betrifft: 
- Suchen über „neuen Vorschlag“ JA: ein neuer Vorschlag wird erstellt. - Suchen über „neuen Vorschlag“ NEIN: im aufgerufenen Vorschlag werden Daten hinzugefügt.

### Button Job-Server 

Dieser Button öffnet die Job-Verwaltung, wo die Jobs für die Zinsrechnung angelegt und bearbeitet werden können. Die Bezeichnung des Jobs lautet standardmäßig „Zinsrechnung" und kann überschrieben werden. 

Siehe Kapitel Job-Verwaltung .
