# Kalenderansicht

Allgemeine Funktionalitäten in eNVenta ERP 

# Kalenderansicht 

Mit einem Klick der rechten Maustaste in ein Datumsfeld wird ein Kalender angezeigt. 

Da unter Macintosh die rechte Maustaste nicht funktioniert, wird diese Kalenderansicht über die Tastatur mit der „Pfeiltaste nach unten“ aufgerufen. 

Mit Enter wird das Datum ausgewählt und zum nächsten Feld gesprungen. 

Mit Escape wird die Kalenderansicht geschlossen, ohne dass ein Datum eingetragen wird. 

Mit den Pfeiltasten (rechts/links) der Tastatur kann ein Tag vor- bzw. zurückgeschalten werden. 

Zum Datum kann auch eine Uhrzeit ausgewählt werden. Diese Einstellung erleichtert vor allem die Zusammenstellung von Touren aufgrund von Uhrzeiten. Die Anzeige der Uhrzeit erfolgt automatisch in den Feldern, in denen auch eine manuelle Eingabe der Uhrzeit zulässig ist. 

Mit den < / >-Buttons kann zum vorherigen bzw. nächsten Monat gewechselt werden. 

Durch Klick auf den Monat öffnet sich eine Monatsauswahl. 

Mit Klick auf das Jahr kann ein anderes Jahr eingestellt werden. 

Wenn Sie den Button mit dem * drücken wird das heutige Datum gesetzt, inkl. der aktuellen Uhrzeit. 

Der Button mit den Zeichen -/- leert das Feld. 

Ebenso ist eine Bedienung über die Tastatur möglich: 

Mit den Pfeiltasten der Tastatur (rechts/links) kann von einem Tag auf den nächsten oder den Vortag gewechselt werden. 

Mit den Pfeiltasten der Tastatur (hoch/runter) kann von einer Woche auf die nächste oder die Vorwoche gewechselt werden. 

Mit den Page-(= Bild-)Tasten der Tastatur kann von einem Monat auf den nächsten oder den Vormonat gewechselt werden. 

Wenn Sie auf der Tastatur die Kombination <Ctrl> + <Page up/down> nutzen, können Sie ein Jahr vor- oder zurückspringen. 

Mit den Tasten Pos1 bzw. Ende kann auf den 1. bzw. letzten Tag des Monats gesprungen werden. 

Schnelleingabe von Datum und Uhrzeit: 
Es ist möglich, ein Datum und/oder eine Uhrzeit nur durch Eingabe von Ziffern zu setzen. 

Nur Datum: 
„*“ liefert das aktuelle Datum. 
Wenn nur Teile des Datums (z.B. nur der Tag) eingegeben werden, dann werden die restlichen Teile aus dem aktuellen Tages-Datum (Monat und Jahr) übernommen. 

Je nach Einstellung der Länder kann die Reihenfolge von Tag, Monat und Jahr variieren. Dementsprechend werden die eingegebenen Ziffern anders interpretiert. 

Tag - Monat - Jahr, z.B. "de-DE" Monat - Tag - Jahr, z.B. "en-GB" Jahr - Monat - Tag, z.B. "fr-CA" 
| | | |
| --- | --- | --- |
| d | d | d |
| dd | dd | dd |
| ddMM | MMdd | MMdd |
| ddMMyy | MMddyy | yyMMdd |
| ddMMyyyy | MMddyyyy | yyyyMMdd |

Die Abkürzungen haben folgende Bedeutung: 
- d,dd - Tag - MM - Monat - yy, yyyy: - Jahr 2- bzw. 4-stellig 
Beispiele (mit Tagesdatum 24.07.2019 und der Ländereinstellung "de-DE"): 
- 3 - 03.07.2019 - 25 - 25.07.2019 - 250617 - 25.06.2019 
Nur Uhrzeit: 
"*" liefert die aktuelle Uhrzeit. 

Folgende Formate werden dabei erkannt: 
- H - HH - Hmm - HHmm - Hmmss - HHmmss 
Diese Abkürzungen haben folgende Bedeutung: 
- H, HH - Stunden in 24-Stunden-Schreibweise - mm - Minuten - ss - Sekunden 
Beispiele: 
- 5 - 05:00:00 - 1405 - 14:05:00 - 74640 - 07:46:40

Durch Eingabe wird das Datum (wie oben beschrieben) ausgewertet. Als Uhrzeit wird 00:00:00 verwendet. 

Hinweis: Es ist möglich, in einem Datumsfeld einen Wert nur für den Tag einzugeben; das System vervollständigt das Datum, passend zum aktuellen Monat / Jahr. 

Beispiel: Geben Sie im Monat Dezember, im Jahr 2019 die Ziffer 2 ein, wird der 2.12.2019 vom System vervollständigt. 

Weitere Eingabemöglichkeiten für das Datum (Spezialfälle!) 

| m: | Monatsanfang (1.) |
| --- | --- |
| m+: | Monatsanfang (1.) + Anzahl Tage |
| m-: | Monatsanfang (1.) - Anzahl Tage |
| M: | Monatsende |
| M+: | Monatsende (28/30/31) + Anzahl Tage |
| M-: | Monatsende (28/30/31) - Anzahl Tage |

Uhrzeit: 
Zudem kann die Uhrzeit angehängt werden. Diese muss mit einem Leerzeichen abgetrennt sein. Die Uhrzeit wird wie oben beschrieben ausgewertet. 

Beispiele (mit dem Tages-Datum 24.07.2019 und der Ländereinstellung "de-DE"): 

| 30 - | 30.07.2019 00:00:00 |
| --- | --- |
| 240519 - | 24.05.2019 00:00:00 |
| 17 1319 - | 17.07.2019 13:15:00 |
| 1706 5 - | 17.06.2019 05:00:00 |
