# Import Editor für Artikel Registerkarte Logistik 
Import Editor - Artikel Registerkarte Logistik 

Auf dieser Registerkarte werden Informationen der Logistik der Import Daten und des Artikelschattenstamms nebeneinandergestellt. Die Daten können verglichen und angepasst werden. 
### Checkbox Kein Paketdienst 
Wenn Sie diese Checkbox aktivieren, werden bei der Ermittlung der verfügbaren Touren keine Paketdienst-Touren berücksichtigt bzw. es können keine Touren mit diesem Transportweg angelegt werden. Die Eigenschaft „Kein Paketdienst“ kann global in den Versandparametern gesteuert werden sowie im Kundenstamm und im Artikelstamm. Sobald eine der Stammdaten die entsprechende Kennung hat, erfolgt keine Tourenermittlung über diesen Transportweg. 

Beim Kopieren eines Angebots / Auftrags wird diese Einstellung übernommen. 

Tooltip: 

Keine Lieferung per Paketdienst

### Checkbox Sperrgut 
Wenn Sie diese Checkbox aktivieren, wird der Artikel als Sperrgut-Artikel gekennzeichnet. Wenn dieser Artikel in einen Auftrag übernommen wird, ist zwingend eine Versandart zu wählen, welche Sperrgüter akzeptiert. 

Hierfür ist im Dialog Versandarten zwingend die Checkbox in der Spalte 'Sperrgut' zu aktivieren. Auf diese Weise können Sie z.B. sicherstellen, dass nicht versucht wird, Sperrgüter über einen Paketdienstleister zu versenden, sondern immer eine Spedition beauftragt werden muss. 

Im Auftrag erscheint bei der Erfassung eine entsprechende Fehlermeldung, falls versucht wird ein Sperrgut mit einer nicht zugelassenen Versandart zu versenden. 

Die Logik für Sperrgüter wird auch für den Webshop genutzt.
