# Kapitel NV.ERP.Base.Customer.cdCustomer.sOrderType

Wenn Sie hier einen Eintrag auswählen, wird in der Auftragserfassung die hier hinterlegte Auftragsart voreingestellt, sie kann aber jederzeit geändert werden. 

Auftragsarten werden im gleichnamigen Dialog angelegt. 

Siehe Kapitel Auftragsarten . 

Dieses Feld steht in direktem Zusammenhang mit dem Parameter Auftragsart fix . 

Tooltip: 

Eingabe der Auftragsart
