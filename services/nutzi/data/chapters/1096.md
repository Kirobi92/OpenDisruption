# VK-Positionen bearbeiten 
Position 

In diesem Kapitel möchten wir die Möglichkeiten des bereits mehrfach erwähnten Dialogs Position näher beschreiben. Aus Platzgründen können nicht alle Informationen zu einem Artikel in der Positionstabelle untergebracht werden. Aus diesem Grund stehen Ihnen weniger häufig genutzte Felder und Möglichkeiten in diesem Zusatz-Dialog zur Verfügung. 

Die Felder im Kopf des Dialogs werden aus dem zugrunde liegenden Vorgang voreingestellt. 

Nachdem eine Auftragsposition erfasst wurde, kann diese auch nachträglich verändert werden. Vor jeder Änderung wird hierbei geprüft, ob der Sachbearbeiter die entsprechende Berechtigung hat und ob der gesamte Auftragsprozess an den jeweiligen Stellen eine Änderung noch erlaubt. So kann auch hier keine Änderung mehr vorgenommen werden, wenn die Position bereits fakturiert ist. Der Dialog öffnet sich mit dem Button Ändern , dem Shortcut F8 oder Doppelklick in eine Zeile. 

Für Diversartikel gilt: Wenn die Checkbox Tab Rabatt1% in den Parametern des Verkaufs aktiviert ist, öffnet sich das Kalkulationsfenster nicht . 

Siehe Kapitel Parameter Verkauf Registerkarte Allgemein.

### Button Entbinden 

Ist in einem Auftrag ein Beschaffungsartikel für einen RAB (= Reservierte, Auftragsbezogene Bestellung) erfasst, dieser wird aber dringend zur Auslieferung für einen anderen Auftrag benötigt, so kann dieser Beschaffungsartikel aus dem eigentlichen Auftrag gelöst (entbunden) werden. Die entbundene Ware ist dann auch zwingend für den neuen Auftrag zu verwenden! Zu diesem Zweck steht der Button Entbinden zur Verfügung. 

Die Bestellung, der im Lager befindlichen Beschaffungsware wird vom Auftrag entbunden. Die Bestellung selbst bleibt als Lagerbestellung erhalten. 

Es ist unbedingt notwendig, dass dieser „entbundene“ Beschaffungsartikel neu bestellt wird. 

Jeder Schritt dieses Prozesses wird dokumentiert. 

Nach Drücken des Buttons Entbinden und bestätigen der Meldung wird der Auftragsbezug in der Bestellposition gelöscht. 

Das ist nur möglich, solange für die Position noch keine Packliste, Lieferschein oder Rechnung gedruckt worden ist. Es darf auch kein Verbundprozess sein. Damit ist auch sichergestellt, dass in Kommissionierung befindliche Positionen nicht mehr entbunden werden.

### Checkbox Liefersperre 
Wenn Sie diese Checkbox aktivieren, wird eine Liefersperre für diese Auftragsposition gesetzt. 

Diese Sperre kann gesetzt werden, so lange der Status der Position < 3 ist. 

Solange die Sperre gesetzt ist, kann für die Position weder eine Packliste noch ein Lieferschein gedruckt werden. 

Tooltip: 

Liefersperre 

### Checkbox Fakturasperre 
Wenn Sie diese Checkbox aktivieren, werden die Rabatte von der Faktura ausgeschlossen. 

Ist diese Checkbox zu einem Artikel gesetzt, für den eine Gutschrift erzeugt werden soll, so wird der Druck der Gutschrift verhindert. Grund dafür ist, dass in eNVenta die Möglichkeit der Teil-Gutschrift nicht vorgesehen ist!
