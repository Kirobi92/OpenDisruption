# Kapitel NV.ERP.Base.Common.cdParameterFI.shtPaymTransDocID2

Wenn Sie diesen Parameter aktivieren, wird im Zahlungsverkehr und im Auszug-/Kassenimport, Kreditoren-OPs die Belegnummer 2 verwendet.
