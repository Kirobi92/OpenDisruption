# Differenzierte Beschaffung im Bestellautomat 
Bestellvorschlagsdetails 

Mit diesem Dialog ist es möglich, pro Position im Bestellautomat zu definieren, ob diese 

- beim Lieferanten A oder B bestellt werden soll - über die Einkaufart A oder B (Strecke oder Lagereinkauf) bestellt wird - über eine entsprechende Niederlassung oder ein Zentrallager beschafft wird (Auftragsart im Bestellautomat) 
Um diese Funktionen zu nutzen, öffnen Sie diesen Dialog per Doppelklick auf einen Tabelleneintrag. 

Je nach Zuordnung generiert der Bestellautomat die jeweiligen Bestellungen. Die Besonderheit ist, dass bei Auswahl eines Streckeneinkaufs, pro Auftrag eine Bestellung generiert wird. 

Sind keine Positionsdaten eingegeben, werden die Kopfdaten als Basis verwendet.

### Combobox Einkaufsart 
Hier kann entsprechend die Einkaufart oder die Auftragsart pro Position eingestellt werden. Wenn Sie eine Auftragsart auswählen, wird statt einer Bestellung ein Umlagerungsauftrag ausgelöst. 

Sind Einkaufart UND Auftragsart auf der gleichen Position angegeben, wird ein Auftrag erzeugt, wenn die Umlagerungs-Auftragsart eingestellt ist. 

Eine Streckeneinkaufart ist nur möglich, wenn die Position einen Auftragsbezug hat. 

Im Fall einer Streckeneinkaufart muss je Auftrag eine separate Bestellung generiert werden. 

Tooltip: 

Geben Sie die Einkaufsart an. 

### Combobox Auftragsart Uml. 
Hier kann die Auftragsart für die Umlagerung gewählt werden. Wird diese wieder entfernt, so wird die Position im Bestellautomaten als manuelle Position gekennzeichnet, unabhängig vom Bestellkennzeichen welches zuvor vom System ermittelt wurde.
