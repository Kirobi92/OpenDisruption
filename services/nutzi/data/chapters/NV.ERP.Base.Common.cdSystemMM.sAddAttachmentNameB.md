# Kapitel NV.ERP.Base.Common.cdSystemMM.sAddAttachmentNameB

Hier geben Sie die Formel zur Ermittlung des Dateinamens ein bei der Generierung einer E-Mail für den Druck einer Bestellung. Es wird immer fix der Wert "Bestellung" verwendet, gefolgt von der Formel als Dateiname.
