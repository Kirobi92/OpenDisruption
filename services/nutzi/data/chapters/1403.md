# Lieferant - Gesellschaft 
Lieferant - Gesellschaft 

Die Gesellschaftsverwaltung für eine gemeinsame Warenwirtschaft steht nur zur Verfügung, wenn die entsprechende Lizenz freigeschaltet ist! 

Dieser Dialog öffnet sich im Navigator >> System >> Allgemein >> Gesellschaftsverwaltung >> Lieferant Gesellschaft . Hier kann einem Lieferanten pro Gesellschaft die Gesellschafter-spezifische „Kundennummer“ hinterlegt werden. Die Lieferanten sind nicht pro Gesellschaft einzeln zugeordnet, damit jede Gesellschaft noch einzeln bei den Lieferanten bestellen kann (aber nicht muss, aufgrund der Möglichkeit die Bedarfsmengen zu sammeln, um bessere Konditionen zu erzielen). 

Die Angaben werden beim Export einer EDI-Datei entsprechend berücksichtigt. 

Mehr zum Thema Gesellschaftsverwaltung erhalten Sie im Kapitel Gesellschaftsverwaltung für eine gemeinsame Warenwirtschaft .
