# Kreditversicherungsliste Registerkarte Versicherung 
Kreditversicherungsliste Registerkarte Versicherung 

Auf dieser Registerkarte hinterlegen Sie Informationen zur Versicherung. 
### Checkbox Ohne Versicherungsdaten 
Anzeige aller Kunden, welchen keine Versicherung im Kundenstamm zugeordnet ist. 

### Checkbox Nicht versichert 
Anzeige aller Kunden, denen eine Beurteilungsart mit Typ nicht versichert zugeordnet ist. 

### Checkbox Pauschal versichert 
Anzeige aller Kunden, denen eine Beurteilungsart mit Typ pauschal versichert zugeordnet ist. 

### Checkbox Namentlich versichert 
Anzeige aller Kunden, denen eine Beurteilungsart mit Typ namentlich versichert zugeordnet ist. 

### Combobox Beurteil.art von 
Über diese Combobox können Sie Ihre Daten nach den Beurteilungsarten selektieren. 

### Combobox Beurteil.art bis 
Über diese Combobox können Sie Ihre Daten nach den Beurteilungsarten selektieren. 

### Combobox Vers. Gesell. 
Aus dieser Combobox können Sie zur Selektion eine Versicherungsgesellschaft auswählen. 

### Combobox Jahr 
Hier wird das (Geschäfts-) Jahr angegeben. 

### Combobox Periode 
Hier wählen Sie einen Zeitraum aus. 

### Feld Datum der Meldung 
Hier wird das Datum angezeigt. 

Tooltip: 

Geben Sie ein Startdatum zur Selektion ein.
