# Volltext-Index anlegen für Microsoft SQL-Server ab 2005 
Diese Funktion steht nur dann zur Verfügung, wenn das entsprechende Feature für den SQL-Server installiert ist. 

Als erstes muss in den Einstellungen der Datenbank die Volltext-Indexierung aktiviert werden. Das ist notwendig, damit überhaupt ein Volltext-Index angelegt werden kann. 

Anschließend ist es zwingend notwendig einen neuen Volltext-Katalog anzulegen. (z.B. „Calltext“) 

Anschließend werden diesem Katalog die Tabellen / Spalten zugewiesen, die indiziert werden sollen. In dem Beispiel wird die Tabelle „CC_CallText“ ausgewählt und in dieser die Spalte „Text“. 

Der Index ist in regelmäßigen Abständen zu reorganisieren. 

Sie können für diesen Volltext-Katalog unter „Population Schedule“ eine automatische Reorganisation einschalten.
