# Extra Auftrag erzeugen 
Extra Auftrag erzeugen 

Es kommt immer wieder vor, dass bei Vermietungen nachträglich Gutschriften erstellt werden müssen. Dabei ist es nicht möglich aus dem „Auftrag“ eine Gutschrift zu erstellen. Zu diesem Zweck steht der Dialog Extra Auftrag erzeugen zur Verfügung. 

Der Dialog kann aus der Vermietung über Menü Optionen >> Extra Auftrag erzeugen geöffnet werden, wenn die Vermietung im Status Abgeschlossen, Schlussabrechnung oder Nachberechnung ist. 

Beim Öffnen des Dialogs wird die Vermietung mit allen Positionen automatisch eingelesen. Wird eine Position markiert, so können in den Feldern unterhalb der Tabelle ein Preis und Positionstext angegeben werden, welche in die generierte Gutschrift übergeben werden. Bei der Generierung der Gutschrift werden alle Positionen aus der Tabelle in die Gutschrift übernommen. 

Über die Auswahl der Belegart kann die Gutschriftsart ausgewählt werden. Beim Erzeugen der Gutschrift über den Button Speichern wird immer eine Wertgutschrift generiert – unabhängig der gewählten Gutschriftsart.

### Tabellenspalte VK pro 
Hier wird eine Preiseinheit angegeben. Dieser Preis kann sich auf 10, 100, 1000 oder eine andere Anzahl Artikeleinheiten beziehen. Standard ist der Wert 1. 

Dieser Wert spiegelt die Anzahl an Artikeleinheiten wider, auf welche sich der Verkaufspreis bezieht. 

Beispiel: 
- 1 Einheit - je 10 Einheiten - je 100 Einheiten

Zu einer Position kann ein bis zu 2000 Zeichen langer Zusatztext eingegeben und in den Formularen gedruckt werden.

Beachten Sie außerdem, dass alle Änderungen im Dialog Position auch im zugrundeliegenden Vorgang mit dem Button gespeichert werden müssen.
