Anlagenbuchhaltung / Anlagenstamm 

# Anlagenstamm Registerkarte Konten 
Anlage Registerkarte Konten 

Auf dieser Registerkarte werden alle Konten definiert, die für die einzelnen Buchungen (Zugang, AfA, Abgang) benötigt werden. Die Informationen zu den Konten werden auch hier in einer Tabelle dargestellt. 

Die Konten können in der Tabelle oder in den Feldern unterhalb der Tabelle erfasst werden. 

Ohne Anlage kann nicht gebucht werden. 

Mit einem Doppelklick in eine Tabellenzeile oder mit dem Button kann die Kontoauswahl geöffnet werden. 

Beim Abgang eines Anlageguts (Verkauf oder Verschrottung) werden folgende Buchungssätze erzeugt: 

| Erlösbetrag | >> Geld-/Forderungskonto an Erlösen aus Abgang AV |
| --- | --- |
| Zeitanteilige AfA | >> AfA an Wertberichtigungskonto |
| Restbuchwert | >> Abgang RBW an Wertberichtigungskonto |

Kontoauswahl 

Wenn Sie nur einen Teil einer Kontonummer eingeben, öffnet sich der Dialog Kontoauswahl . In der Tabelle erscheinen sämtliche Konten entsprechend der Logik des eingestellten Buchungskreises. Sie können entweder das gewünschte Konto aus der Tabelle auswählen oder geben auf den Registerkarten Sachkonto, Debitor oder Kreditor Ihre Suchkriterien ein. Anschließend drücken Sie auf den Button . In der Tabelle erscheinen nun alle Konten, die den gewählten Suchkriterien entsprechen. Mit einem Doppelklick auf die Zeile mit dem gewünschten Konto übernehmen Sie dieses direkt in den zugrundeliegenden Dialog. Sie können die Zeile auch markieren und das Konto durch einen Klick auf den Button übernehmen. 

Mit dem Button leeren Sie die Eingabefelder und die Tabelle. 

Möchten Sie kein Konto auswählen, verlassen Sie den Dialog mit Hilfe des Buttons .

### Checkbox Ausb. Wertbericht. 
Bei Aktivierung dieser Checkbox wird zusätzlich beim Abgang des Anlageguts, der Buchungssatz - Wertberichtigungskonto an Anlagekonto mit dem Wert: AHK - erzeugt und somit sichergestellt, dass der Bestand beider Konten um den Abgang korrigiert wird. (Nur von Bedeutung, wenn ein gesondertes Wertberichtigungskonto geführt wird.) 

Die Checkboxen Ausb. Wertbericht. können bereits im Menü Anlagenklasse Detail vorbelegt werden und werden dann über die Anlagenklassen in den Anlagenstamm übernommen. 

Die Wertberichtigungsbuchung erfolgt nur in der Finanzbuchhaltung, sie wird nicht ins Anlagenjournal gebucht. 

Tooltip: 

Ausbuchung Werberichtung
