# LVS Parameter Registerkarte Zusatzfelder 
Registerkarte Zusatzfelder 

Diese Registerkarte wird sichtbar, wenn Sie individuelle Zusatzfelder mit Datenbankbezug angelegt haben. Hier stehen die Felder zur Verfügung, die Sie im Dialog Zusatzfelder konfigurieren angelegt haben. 

Siehe Kapitel Zusatzfelder definieren . 
### Button Öffnen 

Mit diesem Button gelangen Sie in den Dialog Zusatzfelder definieren , um weitere Felder anzulegen. 

Siehe Kapitel Zusatzfelder definieren . 

Tooltip: 

Mit diesem Button öffnet sich der Dialog zum Editieren der Zusatzfeld-Definitionen.
