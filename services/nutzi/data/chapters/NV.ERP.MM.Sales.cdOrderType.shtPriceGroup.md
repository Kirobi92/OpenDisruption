# Kapitel NV.ERP.MM.Sales.cdOrderType.shtPriceGroup

Hier können Sie eine Preisgruppe für die Auftragsart auswählen, die bestimmt auf welchen Verkaufspreis zugegriffen wird. 

Jedem Kunden ist zwingend eine Preisgruppe zuzuordnen. 

Preisgruppen finden Verwendung in speziellen Aufträgen oder speziellen Preisermittlungen. 

Siehe Kapitel Preisermittlung im Verkauf . 

Tooltip: 

Hier wählen Sie eine Bruttopreisgruppe aus, die bestimmt auf welchen Verkaufspreis zugegriffen wird.
