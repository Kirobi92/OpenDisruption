# Kapitel NV.ERP.MM.CashDesk.cdPaymentMethod.shtBlocked

Wenn Sie eine Zahlungsart sperren möchten, aktivieren Sie diesen Parameter. Die Zahlungsart wird beim Bezahl-Vorgang nicht zur Auswahl angezeigt. 

Tooltip: 

Sperrt eine Zahlungsart
