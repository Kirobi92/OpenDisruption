# Parameter ADR 
Parameter ADR 

Im Navigator unter System >> Allgemein >> Parameter ADR öffnet sich der gleichnamige Dialog. Hier werden grundlegende Einstellungen für den Versand von Gefahrgut festgelegt. Dies sind: 

- Maximales Gewicht pro Transport: 
Das Gewicht an gefährlichen Gütern, welches maximal mit einer Transporteinheit transportiert werden darf (in Kg). Wird hier nichts gepflegt, wird von einer Menge von 8000 Kg ausgegangen. - Maximale Anzahl EQ Packstücke: 
Maximale Anzahl EQ Packstücke pro Transporteinheit (Keine Einstellung = 1000 Stück). - Belegart zum ADR Druck: 
Belegart, bei welcher der ADR Fahrbeleg automatisch mit gedruckt wird. Im Regelfall sollte dies mit dem Druck des Lieferscheins erfolgen. - Reportname: 
Angabe der Reports- als Beförderungspapier mit den Gefahrgutinformationen. Es handelt sich hierbei um einen separaten Report, welcher zum angegebenen Zeitpunkt gedruck wird. 
## Steuerelemente des Formulars 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
