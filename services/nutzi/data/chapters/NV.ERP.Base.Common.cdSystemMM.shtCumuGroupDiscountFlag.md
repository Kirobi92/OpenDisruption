# Kapitel NV.ERP.Base.Common.cdSystemMM.shtCumuGroupDiscountFlag

Wenn Sie diesen Parameter aktivieren, nutzen Sie die Möglichkeit zur Warengruppen-übergreifenden Rabattierung. Hierbei wird beim ersten Einfügen eines Auftrags geprüft, ob mehrere gleichartige Artikel (über die Assortierung-Gruppe im Artikelstamm definiert) im Auftrag vorhanden sind und ob diese zu einer Staffelstufe in der Konditionierung zusammengefasst werden können. 

Siehe Kapitel Assortierung im Rabatt Verkauf . 

Tooltip: 

Mit diesem Parameter aktivieren Sie die Warengruppen-übergreifende Rabattierung.
