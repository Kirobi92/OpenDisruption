# Jahresabschluss Registerkarte Selektiver Vortrag 
Jahresabschluss Registerkarte Selektiver Vortrag 

Diese Registerkarte steht für einen individuellen Saldenvortrag der Bilanzkonten zur Verfügung. 

Auf dieser Registerkarte werden folgende Informationen angezeigt: 

- Konto - Bezeichnung des Kontos - Schlussbestand Vorjahr, z.B. 2018 - S/H-Kennzeichen zu Schlussbestand - Eröffnungsbestand aktuelles Jahr, z.B. 2019 - S/H-Kennzeichen zu Eröffnungsbestand - Betrag für die Buchung - S/H-Kennzeichen zur Buchung - Spalte zur Aktivierung/Deaktivierung des zu buchenden Vortrags 
### Button Vortrag 

Mit diesem Button wird nach einer Abfrage der Vortrag der Salden ausgeführt. 

### Tabelle 

### Tabellenspalte Konto 
Hier wird die Kontonummer angegeben. 

Tooltip: 

Das Konto zum Datensatz. 

### Tabellenspalte Kontobezeichnung 
Die Kontobezeichnung wird nach Eingabe der Kontonummer automatisch eingetragen. Sie können aber auch die Kontobezeichnung zur Suche nutzen. 

Tooltip: 

Kontobezeichnung 

### Tabellenspalte Schlussbestand 
Hier wird der Betrag angegeben. 

Tooltip: 

Angabe des Betrags 

### Tabellenspalte S/H 
Hier wird das Soll- / Haben-Kennzeichen angegeben. 

Tooltip: 

Soll / Haben-Kennzeichnung 

### Tabellenspalte Eröffnungsbestand 
Hier wird der Eröffnungsbestand angegeben. 

Tooltip: 

Angabe des Eröffnungsbestands 

### Tabellenspalte S/H 
Hier wird das Soll- / Haben-Kennzeichen angegeben. 

Tooltip: 

Soll / Haben-Kennzeichnung 

### Tabellenspalte Zu buchen 
Hier wird der zu buchende Betrag angegeben. 

Tooltip: 

Zu buchender Betrag 

### Tabellenspalte S/H 
Hier wird das Soll- / Haben-Kennzeichen angegeben. 

Tooltip: 

Soll / Haben-Kennzeichnung 

### Button OK 

Mit diesem Button wird der Haken in der Spalte 'zu buchen' gesetzt. 

Tooltip: 

Selektierte Sätze für die Verbuchung markieren 

### Button Löschen 

Mit diesem Button wird der Haken in der Spalte ‚zu buchen‘ wieder entfernt. 

Tooltip: 

Markierung "Buchen" der selektierten Sätze aufheben.
