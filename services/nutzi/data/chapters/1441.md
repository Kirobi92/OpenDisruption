# Vermietungsposition 
Vermietungsposition 

Dieser Dialog öffnet sich über den Button Ändern in der Vermietung auf der Registerkarte Allgemein . Hier werden detaillierte Informationen zur Position angezeigt, für die auf der Registerkarte Allgemein kein Platz ist. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Vermietungsposition 
Tooltip: 

In diesem Dialog werden detaillierte Informationen zur Mietposition angezeigt. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung. 

## Menüeinträge 

### Menü Belegfolge 
Über dieses Menü öffnet sich die Belegfolge. Alle Vorgänge der Belegkette inkl. der Druckbelege werden auf einem Zeitstrahl angezeigt. 

Siehe Kapitel Belegfolge . 

### Menü Html Report 
Über dieses Menü öffnet sich der Dialog Html Report . 

Siehe Kapitel Html Report . 

### Menü Html Report Wahl 
Über dieses Menü öffnet sich der Dialog Html Report Auswahl . Dort stehen sämtliche, bereits angelegte Html Reports zur Auswahl. 

Siehe Kapitel Html Report Auswahl . 

### Menü Hilfe 
Über dieses Menü gelangen Sie in die Online-Hilfe und Sie erhalten Informationen zu dem Dialog, aus dem heraus Sie die Hilfe geöffnet haben. 

### Menü Customizing 
Über dieses Menü gelangen Sie ins Client Side Customizing. 

Siehe Kapitel Client Side Customizing . 

### Menü Zusatzfelder konfigurieren 
Über dieses Menü erreichen Sie den Dialog Zusatzfelder definieren . Die dynamischen Zusatzfelder dienen der Ablage zusätzlicher Informationen zum Vorgang, ohne dass eine neue Programmierung notwendig ist. 

Siehe Kapitel Zusatzfelder definieren .

### Button Neu 

Mit diesem Button leeren Sie die Felder. 

Tooltip: 

Erneuern 

### Button Schließen

### Checkbox Vermietet 
Diese Checkbox ist bereits vorbelegt aus dem Dialog Vermietung und bedeutet, dass der Artikel zurzeit vermietet ist. 

### Feld Artikel 
Hier wird die Artikelnummer der markierten Position (im Dialog Vermietung ) angezeigt. 

Tooltip: 

Angabe der Artikelnummer 

### Feld Bezeichnung 
Hier wird die Originalbezeichnung des Artikels (wie im Artikelstamm hinterlegt) angezeigt. 

Tooltip: 

Originalbezeichnung des Artikels laut Artikelstamm. 

### Feld Mietobjekt 
Die Nummer des Mietobjekts wird angezeigt. 

### Feld Bezeichnung 
Der Name des Mietobjekts wird angezeigt. 

Tooltip: 

Originalbezeichnung des Mietobjektes bei Mietartikeln, sonst leer. 

### Feld MP-Bezeichnung 
Hier wird der Name der Mietposition aus der Vermietung aus dem Mietobjekt / Artikelstamm vorbelegt. Der Name kann hier geändert werden. 

Tooltip: 

Bezeichnung der Mietposition in der Vermietung. Wird aus Mietobjekt/Artikelstamm vorbelegt und kann hier geändert werden. 

### Feld Menge 
Hier geben Sie die Menge des Artikels ein. 

### Feld Einheit 
Hier wird die Verkaufseinheit angegeben. 

Tooltip: 

Verkaufseinheit 

### Feld Abrechnung von 
Hier wird ein Startdatum der Abrechnung für die Selektion angegeben. 

Tooltip: 

Geben Sie das Startdatum der Abrechnung zur Selektion ein. 

### Feld Abrechnung bis 
Hier wird ein Bis-Datum angegeben. 

Tooltip: 

Geben Sie ein Bis-Datum der Abrechnung zur Selektion ein. 

### Checkbox Folgemietpauschale 
Wenn Sie diese Checkbox aktivieren, wird die Folgemiete pauschal berechnet. 

Tooltip: 

Pauschal Folgemiete 

### Checkbox Gebraucht 
Falls es sich um einen gebrauchten Artikel handelt, ist diese Checkbox bereits aus der Vermietung angehakt. 

### Combobox Lager 
Hier wird das Lager angegeben, von welchem das Mietobjekt/der Artikel bei der Vermietung entnommen wird. 

Tooltip: 

Lager, von welchem das Mietobjekt/der Artikel bei der Vermietung entnommen wird. 

### Combobox Ziellager 
Hier wird das Lager angegeben, auf welches das Mietobjekt/der Artikel bei der Retoure gebucht wird. 

Tooltip: 

Angabe der Ziellagernummer
