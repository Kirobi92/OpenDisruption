# Besuchskalender Registerkarte Terminübersicht 
Besuchskalender Registerkarte Terminliste 

Diese Registerkarte bietet eine Tabelle mit allen Terminen innerhalb des angegebenen Zeitraums. 

Beim Druck der Terminübersicht aus dem Besuchskalender heraus werden nur die Termine von den aktuell ausgewählten Mitarbeitern berücksichtigt. Damit ist die Terminliste sehr übersichtlich. 

Mit Doppelklick auf einen Eintrag öffnet sich der Dialog Besuchstermin . Hier können Sie Änderungen für die Terminplanung durchführen. 

Siehe Kapitel Besuchstermin-Verwaltung . 
### Button Bericht 

Mit diesem Button wird der Besuchsbericht aufgerufen. 

### Feld Datum von 
In diesem Feld wird das Datum angezeigt, das auf der Registerkarte Kalender den Beginn des gewählten Zeitraums darstellt. 

Beispiel: Sie markieren die KW 38, dann steht in diesem Feld der 15.09. (Montag) und im Feld Bis der 20.09. (Samstag). 

### Feld bis 
In diesem Feld wird das Datum angezeigt, das auf der Registerkarte Kalender das Ende des gewählten Zeitraums darstellt. 

Beispiel: Sie markieren die KW 38, dann steht in diesem Feld der 20.09. (Samstag) und im Feld Von der 15.09. (Montag).
