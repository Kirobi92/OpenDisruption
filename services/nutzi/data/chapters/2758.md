### Combobox Bezugsgröße 1 
Bezugsgrößen sind Teiler, mit denen der Saldo der Kostenstellen geteilt wird. So können z.B. Maschinenstunden, Prozesskosten, Kennzahlen wie Kosten/Mitarbeiter usw. berechnet werden. 

Die Bezugsgrößen werden im Menü Stamm >> Umlagen angelegt und gepflegt.

### Feld Zuständig 
Hier kann der für die Kostenstelle zuständige Abteilungsleiter o.ä. eingetragen werden. 

### Checkbox Vorkostenstelle 
Durch diese Checkbox wird eine Hilfskostenstelle als Vorkostenstelle gekennzeichnet. Dadurch wird die Verteilung dieser Kostenstelle auf andere Hilfskostenstellen möglich. Andere Hilfskostenstellen können sonst nur auf Hauptkostenstellen verteilt werden, es sei denn Sie haben dies in den Parametern ausdrücklich anders festgelegt. 

### Checkbox Nicht manuell bebuchbar 
Wird diese Checkbox gesetzt, steht diese Hilfskostenstelle in der Finanzbuchhaltung für eine manuelle Buchung nicht zur Verfügung. 

In der Kostenrechnung kann weiterhin manuell auf diese Hilfskostenstelle gebucht werden. 

Wird durch ein Setzen dieser Checkbox eine Kore-Schablone verändert, so erscheint eine Hinweismeldung.
