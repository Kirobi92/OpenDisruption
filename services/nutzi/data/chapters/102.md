# Parameter eGate Registerkarte Allgemein 
Parameter eGate Registerkarte Allgemein 

Auf dieser Registerkarte definieren Sie die Einstellungen für den Webshop. 
### Feld Shop ID 
Hier wird die Shop-ID angegeben. 

### Feld Name 
Hier wird die Bezeichnung des Shops angegeben. 

### Feld Shop URL 
Hier wird die URL für die Internetadresse angegeben. 

Tooltip: 

eGate Shop Url 

### Feld Service URL 
Hier wird die URL für die Internetadresse angegeben. (eGate OData Service Url) 

Tooltip: 

eGate OData Service Url 

### Feld Auth. Token 
Hier wird das Authentifizirungs-Token angegeben. 

Tooltip: 

Angabe des Authentifizierungs Token
