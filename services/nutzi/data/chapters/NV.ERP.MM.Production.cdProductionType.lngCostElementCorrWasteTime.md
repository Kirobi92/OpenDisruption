# Kapitel NV.ERP.MM.Production.cdProductionType.lngCostElementCorrWasteTime

Hier geben Sie die Kostenart für Schrott aus Fertigung ein. 

Damit kann Ausschuss-Material (Schrott) korrekt erfasst und verbucht werden. So kann die Berechnung der IST-Kosten bzw. der tatsächlichen Herstellkosten korrekt ausgeführt werden. Der Kostenträger wird entlastet. 

Tooltip: 

Kostenart für den Fertigungsanteil für den erstellten Schrott, mit dem der KTR entlastet werden soll.
