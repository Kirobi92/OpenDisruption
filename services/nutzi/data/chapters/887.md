# Provisionsauswertung 
Provisionsauswertung 

Dieser Dialog öffnet sich im Vertreter >> Statistik >> Provisionsauswertung . 

Eine (Vertreter-)Provisionszuordnung kann sowohl auf Kopf- als auch auf Positionsebene stattfinden. 

Die Berechnung erfolgt immer über einen anzugebenden Zeitraum und kann auf die bezahlten Rechnungen eingeschränkt werden. Dies bedeutet, dass nur für die Aufträge eine Provision bezahlt wird, welche durch die Finanzbuchhaltung als vollständig bezahlt markiert wurden (dies geschieht automatisch). 

Bei der Berechnung der einzelnen Auftragspositionen werden die Provisionssätze der Auftragsposition übernommen. Diesen sind die entsprechenden Werte anhand der Priorität eingetragen. 

In der Auswertung werden auch Aufträge angezeigt, für welche keine Provision bezahlt wird. So werden auch Vorgänge erfasst, die provisionsberechtigt sind, für welche aber eine 0 eingetragen ist. Diese Information kann als Argumentationshilfe bei Provisionsverhandlungen genutzt werden. 

Im Standard wird der Provisionssatz entweder nur auf Basis des Umsatzes oder nur auf Basis des Ertrags berechnet. Diese Berechnung kann durch eine Gewichtung der Faktoren erweitert werden. 

Dabei gilt für die Provisionsabrechnung: 

Positionsprovisionen haben Vorrang vor den Kopfprovisionen, eine in der untenstehenden Hierarchie speziellere Provision (kleinere Ordnungsnummer) übersteuert eine allgemeinere. 

Vergleichen Sie hierzu das Kapitel Provisionshierarchie .
