# Kapitel NV.ERP.Base.Common.cdSystemMM.decReturnsPercentStock

(= Abschlag für Retouren von Aufträgen mit Lagerware) 

Hier kann standardmäßig ein Prozentsatz für die Rücknahme einer Auftragsposition eingetragen werden. Um diesen Prozentsatz wird die Gutschrift bei Rücknahme der Ware gemindert. 

Die Einstellung im Kundenstamm auf der Registerkarte Warenwirtschaft hat Vorrang! 

Vergleichen Sie hierzu das Kapitel Retourenposition .
