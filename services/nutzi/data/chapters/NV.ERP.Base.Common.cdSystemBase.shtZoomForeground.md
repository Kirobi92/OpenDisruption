# Kapitel NV.ERP.Base.Common.cdSystemBase.shtZoomForeground

Wenn Sie diesen Parameter aktivieren, werden für den Benutzer beim Starten der Anwendung die Zoom-Elemente als erster Dialog angezeigt. 

Dies können vom Benutzer selbst angelegte oder vom Administrator eingerichtete Zoom-Elemente sein, die auf "Automatisch sichtbar" gesetzt sind. 

Wenn über die Favoriten Einträge für den Autostart eingerichtet sind, dann haben diese höhere Priorität und werden trotz dieser Einstellung über den Zoom-Dialog gelegt. 

Ist der Parameter nicht aktiv, wird der Start-Bildschirm von eNVenta als erster Dialog beim Start angezeigt. 

Tooltip: 

Beim Start der Anwendung wird der Zoom-Dialog im Vordergrund angezeigt, sofern es sichtbare Zoom-Elemente gibt.
