# Kreditlimit Ausnutzung Registerkarte Halbjahr 
Kreditlimit Ausnutzung Registerkarte Halbjahr 

Auf dieser Registerkarte werden die Daten zur Kreditlimitausnutzung pro Halbjahr angezeigt. 
### Tabelle Mit Halbjahresübersicht 

In der Tabelle geben Sie die Details ein, dies sind u.a. Informationen zum Kreditlimit in der Warenwirtschaft, Sicherheiten, Versicherungsbetrag und Saldo. 
### Tabellenspalte KL-Summe WaWi 
Hier wird die Summe des Kreditlimits aus der Warenwirtschaft angezeigt. 

Tooltip: 

Summe Kreditlimit Warenwirtschaft 

### Tabellenspalte KL-WaWi 
Hier wird das Kreditlimit aus der Warenwirtschaft angezeigt. 

Tooltip: 

Angabe des Kreditlimits der Warenwirtschaft 

### Tabellenspalte Sonder-KL WaWi 
Hier wird das Sonderkreditlimit aus der Warenwirtschaft angezeigt. 

Tooltip: 

Sonderkreditlimit Warenwirtschaft 

### Tabellenspalte Sicherheiten 
Hier werden die Sicherheiten angegeben. 

Tooltip: 

Angabe der Sicherheiten 

### Tabellenspalte Vers. Betrag 
Hier wird der Betrag aus der Kreditversicherung angegeben. 

Tooltip: 

Versicherungsbetrag 

### Tabellenspalte Vers.-Limit 
Hier wird das versicherte Limit eingetragen. 

Tooltip: 

Versicherungslimit 

### Tabellenspalte KL-Projekt 
Hier wird das Kreditlimit des Projekts angegeben. 

Tooltip: 

Angabe des Kreditlimits des Projektes 

### Tabellenspalte KL-Summe 
Hier wird die Summe des Kreditlimits angegeben. 

Tooltip: 

Sume Kreditlimit 

### Tabellenspalte Saldo 
Hier wird der Saldo angegeben. 

Tooltip: 

Angabe des Saldos 

### Tabellenspalte KL-Offen 
Hier wird das noch offene Kreditlimit angegeben. 

Tooltip: 

Nutzung Kreditlimit
