# Schlagworte 
Schlagworte 

In diesem Dialog können die verfügbaren Schlagworte eines Archivs betrachtet werden. Diese Schlagworte können nicht geändert werden. 

Falls das Archiv zulässige Dateitypen ermittelt hat, wird beim Anlegen einer neuen Archiv-Mappe die Datei-Auswahl entsprechend eingeschränkt. Dazu steht Ihnen das Feld Datei-Typen zur Verfügung. Die zulässigen Typen werden auch im Dialog Schlagworte angezeigt. 

Diese Dateitypen sind vom Archivsystem vorgegeben. 

Diese Schlagworte werden bei Easy-Archiv dynamisch ausgelesen und dienen der Suche im Archiv. 

Beim Schließen des Dialogs werden alle Daten aus dem Archiv neu eingelesen. 

Bei einem DB-File Archiv legen Sie hier die Schlagworte an, die in das Archiv eingetragen werden. 

Die Schlagwort-ID wird beim Einfügen automatisch vergeben. Daneben geben Sie eine physische Bezeichnung/einen Namen als Schlagwort an. 

In der Combobox wählen Sie aus, von welchem Datentyp das Schlagwort ist. Zur Auswahl stehen dabei 
- Text - Ganzzahl - Datum - Dezimalzahl 
Wählen Sie den passenden Datentyp aus. Falls Sie nicht sicher sind, benutzen Sie den Datentyp ‚Text‘.
