# Buchen mit Kostenrechnung (Verteilung) 
Verteilung 

Wird mit dem Modul Kostenrechnung gearbeitet, kann beim Buchen mit einem Sachkonto, welches im Sachkontenstamm als kostenrechnungsrelevant gekennzeichnet ist, nach Betätigen des Buttons der Dialog Verteilung aufgerufen. 

Nehmen Sie durch direkten Eintrag in den einzelnen Tabellenspalten die Verteilung der Kosten auf die gewünschten Kostenstellen und Kostenträger vor. 

Generell gilt: Beträge können nur auf Perioden innerhalb eines Geschäftsjahrs verteilt werden. Deshalb erfolgen an diversen Stellen Prüfungen auf Plausibilität. 

Bei Eingabe einer ungültigten Nummer (Beispiel -1) in der Spalte 'Kostenstelle' bzw. 'Kostenträger' öffnet sich der Dialog Kostenstellenauswahl bzw. Kostenträgerauswahl . 

Der Wert in der Spalte ‚Betrag‘ wird immer netto angezeigt. 

Analog zum Buchen -Dialog kann das Zeichen „+“ für die Buchung über den Ziffernblock verwendet werden. Wenn die Zeilen nicht 100% ergeben, wird bei Angabe des Zeichens „+“ eine weitere Zeile eingefügt und der Restbetrag und die verbleibenden % vorgetragen. Nach erfolgreicher Verteilung auf die Kostenstellen / Kostenträger kann durch die Eingabe von „+“ in der letzten Zeile die Buchung ausgelöst werden. 

#### Suchen der Kostenstellen und Kostenträger durch Texteingabe 
Im Dialog Verteilung in den Spalten ‚Kostenstelle‘ und ‚Kostenträger‘ können auch Buchstaben und Sonderzeichen eingegeben werden. So erfolgt die Suche nicht nur nach Nummern, sondern auch über die Bezeichnung der Kostenstellen bzw. Kostenträger. 

Wenn es keinen eindeutigen Treffer gibt, öffnet sich der Dialog Kostenstellenauswahl bzw. Kostenträgerauswahl und weist die Treffer aus.

### Combobox Schablone 
Mit Hilfe dieser Combobox haben Sie die Möglichkeit, eine Schablone, die im Menü Buchen >> Stamm >> Kore-Schablonen festgelegt wurde, auszuwählen und den Betrag gemäß der Verteilung der ausgewählten Schablone zuordnen zu lassen. 

Ist eine Schablone ausgewählt, wird sie zu diesem Vorgang aufgerufen. Ist keine ausgewählt, wird die aktuelle Verteilung übertragen und die Prozentwerte werden ermittelt. 

Siehe Kapitel Kore-Schablone . 

Der letzte Eintrag muss so gewählt werden, dass sich genau 100% ergeben. 

### Button Verzweigen 

Dieser Button öffnet den Dialog Kore-Schablone . 

### Field Panel #fpnDistributionData# 

In den Comboboxen Verteilung und Verteilungsart werden die Stammdaten aus Kostenarten bzw. Leistungen sichtbar und zugänglich. Bei Auswahl Monat in der Combobox Verteilung wird zusätzlich das Feld V erteil.perioden eingeblendet. 

Bei der Buchung in der Fibu wird nun hier abweichend von den Stammdaten der Kostenart / Leistung die Anzahl der Perioden angegeben, auf die in der Kore verteilt werden muss. Dadurch kann auf eine Vielzahl von wiederkehrenden Buchungen, verbunden mit aufwendigen Kontrollen und Abstimmungen in der Fibu verzichtet werden.

### Tabellenspalte Betrag 
In dieser Spalte wird der Betrag aus dem Dialog Buchen (Feld Betrag ) angezeigt, gekürzt um die Umsatzsteuer bzw. Vorsteuer. 

Soll der Betrag nur auf eine Kostenstelle, bzw. einen Kostenträger gebucht werden, bestätigen Sie Ihre Eingaben mit dem Button . 

Wollen Sie den Betrag auf mehrere Kostenstellen / Kostenträger verteilen, ändern Sie den Betrag nach Auswahl der Kostenstelle / des Kostenträgers auf den gewünschten Betrag ab. 

Dies wiederholen Sie solange, bis der gesamte Betrag auf den von Ihnen gewünschten Kostenstellen / Kostenträgern verteilt ist. 

Tooltip: 

Angabe des Betrags
