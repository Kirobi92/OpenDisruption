# Projekt-Leistungscontrolling Registerkarte Aufgaben 
Projekt-Leist.controlling Registerkarte Aufgaben 

Auf dieser Registerkarte werden die Projekte zur selektierten Kundennummer aufgelistet. In der unteren Tabelle erhalten Sie detaillierte Informationen zu einem Eintrag der oberen Tabelle. 
### Tabelle 

In dieser Tabelle erhalten Sie eine Übersicht zu Aufgaben, bezogen auf Kunden. 
### Tabellenspalte Kunde 
Hier wird die Kundennummer angegeben. 

Tooltip: 

Angabe der Kundennummer 

### Tabellenspalte Suchname 
Hier wird der Suchname angegeben. Außerdem kann hier eine Belegnummer zu einem Vorgang eingetragen werden. Wenn der Vorgang existiert, wird der Vorgang geladen. 

Tooltip: 

Angabe des Suchnamens 

### Tabellenspalte Projekt 
Hier wird eine eindeutige Projektnummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Projektnummer ein. 

### Tabellenspalte Projektname 
Hier wird der Projektname angegeben. 

Tooltip: 

Name des Projekts 

### Tabellenspalte Aufgabe 
Hier wird eine Aufgabennummer angegeben. 

Tooltip: 

Tragen Sie eine eindeutige Aufgabennummer ein.

Tooltip:
