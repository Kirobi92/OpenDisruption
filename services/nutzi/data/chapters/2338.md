# IDEA Export Registerkarte Spalten 
IDEA Export Registerkarte Spalten 

Über diese Registerkarte kann eingestellt werden, welche Spalten einer markierten Tabelle exportiert werden. Damit diese Registerkarte zugänglich wird, ist es notwendig vorher auf der Registerkarte Tabelle eine Auswahl zu treffen. 

Ebenfalls wird in dieser Registerkarte das Kennzeichen gesetzt, welches mit dem Prüfzeitraum abgeglichen wird, z.B. Buchungsdatum der Tabelle Journal . Wenn Sie auf der linken Seite des Dialogs eine Tabellenspalte markieren und die Checkbox PK aktivieren , wird diese Spalte als Primary Key markiert. Dieses Datenfeld wird dann in der XML-Datei ebenso markiert. Im Bereich Einschränkung beim Export kann deshalb auf Jahr bzw. Datum eingeschränkt werden. Diese Einschränkung bezieht sich auf den Eintrag in der Spalte ‚Typ‘. Wenn Sie Datum aktivieren, werden die Exportdaten auf das Datum, welches im Dialog GoBD Prüfplattform eingestellt ist, eingeschränkt. Wenn Sie Jahr aktivieren, werden die Exportdaten auf das Jahr (Felder Prüfperiode Von … Bis ), welches im Dialog GoBD Prüfplattform eingestellt ist, eingeschränkt.
