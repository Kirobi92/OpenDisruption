# Kapitel NV.ERP.Base.Storage.cdStorageLocation.shtArticleCheckup

Ist dieser Parameter gesetzt, prüft das Programm bei der Belegerfassung, ob der Artikel auf das Lager zugelassen ist und lehnt gegebenenfalls die Erfassung ab. Die Zuordnung von Artikeln zu einem oder mehreren Lagern kann im Artikelstamm vorgenommen werden. 

Die Prüfung, ob ein Artikel für einen Lagerort zugelassen ist, erfolgt nur für lagerführende Artikel. 

Siehe Kapitel Lagerbestand (Lager pro Artikel) . 

Tooltip: 

Wenn aktiviert, prüft das Programm bei der Belegerfassung, ob der Artikel auf das Lager zugelassen ist 
und lehnt gegebenenfalls die Erfassung ab.
