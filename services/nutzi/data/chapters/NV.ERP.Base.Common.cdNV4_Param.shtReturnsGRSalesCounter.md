# Kapitel NV.ERP.Base.Common.cdNV4_Param.shtReturnsGRSalesCounter

Wenn Sie diesen Parameter aktivieren, kann am Retouren-Bildschirm ein verkürzter Wareneingang durchgeführt werden. Hierbei wird ein Etikett an der Theke erzeugt, mit der die retournierte Ware wieder eingelagert werden kann. Die Retoure muss dann nicht über den Einlagerungsdialog durchgeführt werden.
