Lagerverwaltungssystem (LVS) 

# Lagerplatz für LVS-Lager generieren 
Lagerplatz generieren 

Dieser Dialog öffnet sich, wenn Sie im Dialog Lagerplatz ein LVS-Lager in der Combobox Lager ausgewählt haben und den Button drücken. 
### Lagerplatz mit Schema 
Wenn die in der unteren Tabelle angelegten Lagerplätze gespeichert werden, dann wird das Feld Schema automatisch gefüllt und kann anschließend beliebig häufig geladen und angewendet werden. 

Mit dem Button können Sie ein bereits angelegtes Schema auswählen. 
### Platzart 
Hier können Sie auswählen, von welcher Platzart die anzulegenden Plätze sein sollen. 

- Blocklagerplatz: Platz, der für mehr als nur einen Artikel verwendet werden kann. - Erstlagerplatz: Fixplatz, in den ein Artikel bevorzugt eingelagert wird. - Normal: Platz, der für mehr als nur einen Artikel verwendet werden kann. - Reserveplatz: Platz, an dem der Reservebestand eines Artikels eingelagert werden soll. - Systemplatz: Platz, der vom System erstellt wird. - Systemplatz für Auslagerung: Platz, der vom System für Auslagerungen erstellt wird. - Tourenlagerplatz: Platz für die Tourenlagerbuchung (Einlagerung von verpackter Ware in Packstücken) - Zweitlagerplatz: zweiter, Fixplatz. Dieser befindet sich direkt oberhalb oder unterhalb des Erstplatzes, zwecks automatischem Nachschub. 
### Lager 
Hier wählen Sie aus, in welchem Lager die Plätze generiert werden sollen. 
#### Zone 
Hier wählen Sie aus, in welcher Zone die Plätze generiert werden sollen. 
#### Ebene / Gang und von / bis 
Hier geben Sie an, von welcher Ebene und Gang bis zu welcher Ebene und Gang das Schema neue Plätze anlegen soll. 
#### Untere Tabelle 
Hier legen Sie über den Button das Haus, Y- und X-Koordinaten an. 

Mit dem Button können Sie die einzelnen Einträge aus der Tabelle wieder löschen. 

Die Lagerplätze sind zunächst inaktiv, da sie im Lager zuerst etikettiert werden müssen. Nach Anlegen der Plätze müssen diese daher manuell aktiv gesetzt werden.
