# Kapitel NV.ERP.FI.FinancialAcc.PaymentTransactions.cdCollectionParameter.shtPartPaymentConditionID

Für die Anlage eines OPs aus einer Teilzahlung können Sie eine Zahlungsbedingung dazu hinterlegen. Dafür steht diese Combobox zur Verfügung. Hier wird dann eine Zahlbedingung ohne Skonto ausgewählt. 

Ist hier eine Zahlungsbedingung hinterlegt, wird für die OP-Anlage aus der Teilzahlung diese Zahlungsbedingung genutzt. 

Ist hier keine Zahlungsbedingung hinterlegt, wird der Sachbearbeiter, wenn er eine Teilzahlung durchführt, mit einer Meldung darauf hingewiesen, dass dieses Feld nicht gefüllt ist. Wird diese Meldung ignoriert, wird für die OP-Anlage die Zahlungsbedingung aus dem Kundenstamm benutzt (wie bisher). 

Tooltip: 

Wählen Sie eine Zahlungsbedingung für die Anlage eines OPs aus einer Teilzahlung.
