# Sicherheitengruppen 
Sicherheitengruppen 

Dieser Dialog öffnet sich über den Button Gruppe im Dialog Sicherheiten . Hier können Sie Sicherheiten in Gruppen einteilen. 

Die Sicherheitengruppen ermöglichen eine Auswertung über die Sicherheitenliste, bzw. bieten eine grundsätzliche Gliederung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Sicherheitengruppen 
Tooltip: 

In diesem Dialog können Sie Sicherheiten in Gruppen einteilen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Tabelle Sicherheitengruppen 

In dieser Tabelle legen Sie die Sicherheitengruppen an bzw. erhalten eine Übersicht über bereits angelegte Sicherheitengruppen. 
### Tabellenspalte Gruppe 
In dieser Spalte geben Sie der Sicherheitengruppe eine Nummer. 

Tooltip: 

Angabe der Gruppennummer 

### Tabellenspalte Bezeichnung 
In dieser Spalte geben Sie der Sicherheitengruppe einen Namen. 

Tooltip: 

Angabe der Bezeichnung 

### Tabellenspalte Typ 
Diese Spalte bietet 2 Typen an: erhalten und erteilt . 

erhalten: 
Ihrem Unternehmen wurde eine Sicherheit angeboten, z.B. ein Wertpapier, eine Immobilie. 

erteilt: 
Sie haben eine Sicherheit gegeben, z.B. einen Wechsel an Ihre Bank. 

Tooltip: 

Angabe des Typs 

### Tabellenspalte Gesperrt 
Wenn Sie die Checkbox in dieser Spalte aktivieren, ist die Sicherheitengruppe gesperrt. 

Tooltip: 

Aktiviert: Setzt den Datensatz auf 'gesperrt'.
