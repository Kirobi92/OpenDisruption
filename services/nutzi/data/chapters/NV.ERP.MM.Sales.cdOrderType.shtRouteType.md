# Kapitel NV.ERP.MM.Sales.cdOrderType.shtRouteType

Wenn Sie aus dieser Combobox einen Transportweg auswählen, wird dieser Transportweg im Auftrag voreingestellt. 

Mögliche Werte sind: 

leer = Automatisch 
0 = Fuhrpark 
1 = Paketdienst (Feld Versender und Versand Code müssen gefüllt sein) 
2 = Overnight Dienst (Feld Versender und Versand Code müssen gefüllt sein) 
3 = Spedition
