# Sachkontenstamm Registerkarte Memo 
Sachkonto Registerkarte Memo 

Auf dieser Registerkarte können Sie allgemeine Informationen für den internen Gebrauch hinterlegen. 
### Feld Allgemein 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit.
