# Auswahl Wareneingang 
Wareneingangsauswahl 

Dieser Dialog öffnet sich im Wareneingang mit dem Button neben dem Feld Wareneingang . Hier können Sie über die Lieferantennummer, Lieferscheinnummer oder Bestellnummer nach Wareneingängen suchen. Über die Checkbox Nur offene können Sie steuern, wenn in der Tabelle nur offene Wareneingänge angezeigt werden sollen. Die Checkbox Belastung grenzt die Wareneingänge noch weiter ein. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Wareneingangsauswahl 
Tooltip: 

In diesem Dialog können Sie Wareneingänge suchen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Checkbox Wenn aktiviert, wird nach Belastungen selektiert. 
Wenn Sie diese Checkbox aktivieren, wird die Selektion auf Belastungen eingeschränkt. 

### Checkbox Nur offene 
Wenn Sie diese Checkbox aktivieren, wird die Selektion auf offene Wareneingänge eingeschränkt. 

Ein Wareneingang ist offen, solange keine Rechnung dazu erfasst wurde. Selbst wenn der Wareneingang schon "geprüft" gebucht ist, ist dieser solange offen bis eine Rechnung erfasst wird. 

Tooltip: 

Wenn aktiviert, wird die Selektion auf offene Wareneingänge eingeschränkt.

### Feld Anzahl max. 
Über dieses Feld ist in den Suchdialogen die Anzahl der einzulesenden Datensätze limitiert. Die einzelnen Datensätze werden im Dialog entsprechend gesucht und nach Erreichen der maximalen Anzahl Datensätze abgebrochen. Auf diese Weise kann eine Suche über alle Datensätze verhindert werden, welche die gesamte Performance beeinträchtigen könnte. 

Diese Einstellung nehmen Sie im Dialog Parameter Allgemein auf der Registerkarte Allgemein vor. 

Siehe Kapitel Parameter Allgemein Registerkarte Allgemein . 

Diese Einstellung lässt sich in den entsprechenden Suchdialogen (z.B. Artikelsuche, Kundenauswahl) ändern bzw. ausschalten (ausgenommen Stammdatenprogramme). 

Tooltip: 

Anzahl Sätze, die maximal eingelesen werden.
