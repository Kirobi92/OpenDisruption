# Stückliste kopieren 
Stückliste kopieren 

Dieser Dialog öffnet sich über den Button Kopieren im Dialog Stückliste . Die am Bildschirm angezeigte Stückliste kann so auf einen anderen Stücklistenartikel kopiert werden. 

Wenn die neue Stückliste bereits Komponenten hat, ist das Kopieren nicht erlaubt. Es erscheint eine Meldung, die auf die Existenz von Komponenten hinweist und darauf, dass nicht kopiert werden kann. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Stückliste kopieren 
Tooltip: 

In diesem Dialog kann die angezeigte Stückliste auf einen anderen Stücklistenartikel kopiert werden. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.
