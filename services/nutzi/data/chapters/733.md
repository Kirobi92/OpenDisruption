# Stücklistendruck 
Stücklistendruck 

Der Button im Dialog Stückliste öffnet diesen Dialog zum Drucken von Stücklisten. 

Sie haben die Möglichkeit verschiedene Stücklistendarstellungen zu drucken. Zur Auswahl stehen: 

- Strukturstückliste (normale, mehrstufige Stücklistendarstellung) - Mengenstückliste (keine Baugruppen, nur summierte Endteile) - Baukastenstückliste (auf eine Ebene verdichtet) - Variantenstückliste (Stückliste in bestimmter vorgegebener Variante) 
Je nach Wunsch wählen Sie einen der Radiobuttons im Bereich Stücklistenart aus. 

Wenn ein Kunde eine Stückliste bestellt, so wird dieser Stücklisten-Kopf auf den Lieferschein gedruckt und in der Kommissionierung angezeigt. 

Damit erhält der Kunde die Gewissheit, dass er nicht einzelne Komponenten gekauft hat, sondern die komplette Stückliste. 

Im Bereich Format entscheiden Sie, wie das Formular gedruckt werden soll; zur Verfügung stehen: 

- PDF - Rich Text - Html 4.0 - Word - Excel 
Im Bereich Modus können Sie wählen, ob der Beleg gedruckt, angezeigt oder als E-Mail versendet werden soll. 

Siehe Kapitel Drucken in eNVenta ERP .

### Checkbox Briefkopf 
Diese Checkbox wird aus dem Dialog Parameter Allgemein (Registerkarte Allgemein ) vorbelegt und kann hier im Einzelfall deaktiviert werden. 

Erläuterung: 

Im Rahmen von Geschäftsprozessen ist es immer wieder notwendig, Belege physisch oder digital als Ausdrucke bereitzustellen. Aus rechtlichen Gründen muss dies bei offiziellen Dokumenten mit bestimmten Informationen (Anschrift, BLZ, …) – in Kopf- und Fußzeile - geschehen. Diese Informationen werden im Folgenden als Briefkopf bezeichnet. 

Dies können Kunden nutzen, die ihre Firmeninformationen direkt mit dem Druck des Belegs auf ein Blanko-Papier drucken möchten. 

Die Checkbox Briefkopf dient in den Dialogen als Vorbelegung und im automatisierten Druck als Einstellung für den Druck. 

Im Einzelfall kann in den Druck-Dialogen der einzelnen Vorgänge die Checkbox Briefkopf deaktiviert werden, falls Sie ausnahmsweise auf ein Briefpapier drucken möchten. 

Tooltip: 

Wenn aktiv, wird der Briefkopf z.B. Logo, Absender auf den Beleg gedruckt.
