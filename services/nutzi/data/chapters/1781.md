# Lieferstatistik in der Bestellung Registerkarte Chart 
Lieferstatistik Registerkarte Chart 

Auf dieser Registerkarte werden die Umsätze grafisch dargestellt. 

### Button Säulendiagramm 

Wenn Sie diesen Button drücken, erfolgt die Anzeige als vertikales Spaltendiagramm. 

### Button Balkendiagramm 

Wenn Sie diesen Button drücken, erfolgt die Anzeige als horizontales Balkendiagramm. 

### Button Liniendiagramm 

Wenn Sie diesen Button drücken, erfolgt die Anzeige als Kurvendiagramm. 

### Button Kuchendiagramm 

Wenn Sie diesen Button drücken, erfolgt die Anzeige als Kuchendiagramm. 

### Button Diagrammeinstellungen 

Dieser Button öffnet die Diagramm Einstellungen . Dort finden Sie vielfältige Auswahlmöglichkeiten für die Darstellung der Werte in der Grafik. 

Siehe Kapitel Chart Einstellungen .
