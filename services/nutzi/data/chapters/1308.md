# Rabattpflege Registerkarte Artikel 
Registerkarte Artikel 

Hier werden Artikel eingetragen / angezeigt, für die ein Rabattsatz gepflegt werden soll. 
### Tabelle 

### Tabellenspalte Intern ID 
Hier wird eine interne, laufende Nummer angegeben. 

Tooltip: 

Interne laufende Nummer 

### Tabellenspalte Artikel 
Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte AG/RG 
Hier wird eine Artikel- oder Rabattgruppe angegeben. 

Tooltip: 

Artikel- oder Rabattgruppe 

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Tooltip: 

Artikelauswahl 

### Button Satz hinzufügen 

Mit diesem Button öffnet sich eine neue Zeile und ein neuer Artikel kann angelegt werden. 

Tooltip: 

Neue Zeile 

### Button Löschen 

Mit diesem Button entfernen Sie einen Eintrag. 

Tooltip: 

Zeilen löschen
