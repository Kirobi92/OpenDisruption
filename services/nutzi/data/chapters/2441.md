# OP-Verrechnung Registerkarte Salden-Verrech. Debitoren 
OP-Verrechnung Registerkarte Salden-Verrech. Debitoren 

Auf dieser Registerkarte werden die Belege eingelesen, die den Selektionskriterien im Kopf des Dialogs entsprechen. So kann eine Verrechnung auf Saldo-Ebene vorgenommen werden. 
### Button Löschen 

Über diesen Button können Sie einzelne, markierte Zeilen aus der Selektion löschen. 

### Tabelle 

### Tabellenspalte Konto
