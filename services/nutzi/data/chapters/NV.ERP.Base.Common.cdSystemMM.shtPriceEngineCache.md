# Kapitel NV.ERP.Base.Common.cdSystemMM.shtPriceEngineCache

Über diese Checkbox wird der Preisfindungscache aktiviert. Zusätzlich müssen auch die zugehörigen Preisgruppen hinterlegt werden. 

Ist der Preisfindungscache aktiviert, werden in ausgewählten Fällen bei der Preisfindung alle nötigen Daten von vorneherein gecached. Dies bedeutet eine bessere Laufzeit zu Lasten von höherem Arbeitsspeicheraufwand. 

Ist diese Checkbox nicht gesetzt, laufen die Datenbankzugriffe aus der Preisfindung zwar weiter über die Cache Komponente, es werden jedoch keine Ergebnisse zwischengespeichert oder Daten im Voraus geladen.
