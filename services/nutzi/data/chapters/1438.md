# Vermietung Registerkarte Memo 
Vermietung Registerkarte Memo 

Auf dieser Registerkarte können weitere Informationen zur Vermietung eingetragen werden. 
### Feld Memo 
Hier wird ein individueller Text zur internen Information angegeben. 

Ist ein Eintrag erfasst, wird auf der Registerkarte Memo mit einem Sternchen (*) gekennzeichnet, dass Text hinterlegt ist. So erkennen Sie sehr schnell, ob es eine interne Information zu einem Datensatz gibt. Dies gilt systemweit. 

Tooltip: 

Enthält interne Textinformationen. 

### Feld Bes. Vereinb. 
Hier können besondere Vereinbarungen angegeben werden. 

Tooltip: 

Besondere Vereinbarungen 

### Button Öffnen 

Mit diesem Button können Sie einen Textbaustein auswählen. 

Tooltip: 

Auswahl der Textbausteine öffnen 

### Button Html-Editor 

Dieser Button öffnet den HTML-Editor. 

Tooltip: 

Html-Editor öffnen, um die Texte zu formatieren. 

### Feld Versand 
Hier kann ein Text für den Versand angegeben werden. 

Dieses Memo lässt sich auf die Versandpapiere drucken. 

### Feld Starttext 
Hier kann ein individueller Text als Starttext im Vorgang eingetragen werden. Dieser kann auf den Formularen angedruckt werden. 

### Button Öffnen 

Mit diesem Button können Sie einen Textbaustein auswählen. 

Tooltip: 

Auswahl der Textbausteine öffnen 

### Button Html-Editor 

Dieser Button öffnet den HTML-Editor. 

Tooltip: 

Html-Editor öffnen, um die Texte zu formatieren. 

### Feld Endtext 
Hier kann ein individueller Text als Endetext im Vorgang eingetragen werden. Dieser kann auf den Formularen angedruckt werden. 

### Button Öffnen 

Mit diesem Button können Sie einen Textbaustein auswählen. 

Tooltip: 

Auswahl der Textbausteine öffnen 

### Button Html-Editor
