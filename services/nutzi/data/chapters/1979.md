# Material disponieren Registerkarte Produzieren 
Material disponieren Registerkarte Produzieren 

Auf dieser Registerkarte kann für die gewählte Materialposition ein untergeordneter Betriebsauftrag generiert werden (z.B. bei mangelnder Verfügbarkeit einer Baugruppe). Menge, Termin und Lieferant/Produzent können angepasst werden. 

Ein Unter-Betriebsauftrag kann nur für Materialpositionen vom Artikeltyp „Produktions-Stückliste“ erstellt werden. 

In der Tabelle sehen Sie eventuell bereits vorhandene Unter-Betriebsaufträge. 
### Button Gen. BA 

Dieser Button ist nur aktiv, wenn eine Produktionsstückliste markiert ist und generiert einen neuen Betriebsauftrag. Die Referenz wird nach erfolgreicher Generierung in der Tabelle unten auf der Registerkarte angezeigt. 

Tooltip: 

Generiert einen Betriebsauftrag. 

### Button Gen.BA M. 

Mit diesem Button können Sie Betriebsaufträge für markierte Positionen generieren. 

Tooltip: 

Generiert Betriebsaufträge für MARKIERTE Positionen. 

### Feld Menge 
Übernahme der Menge aus der Artikelposition im Betriebsauftrag. Sie kann manuell angepasst werden. 

Tooltip: 

Menge aus der Artikelposition 

### Feld Lieferdatum 
Hier wird der Liefertermin angegeben. 

Tooltip: 

Angabe des Liefertermins 

### Feld Lieferant 
Hier wird die Lieferantennummer angegeben. 

Tooltip: 

Angabe der Lieferantennummer 

### Button Suche 

Über diesen Button können Sie den Lieferanten suchen. 

Siehe Kapitel Lieferantenauswahl . 

### Feld Firma 1 
Die Firmenbezeichnung wird automatisch vom System eingetragen, nachdem die Lieferantennummer eingetragen wurde. 

Tooltip: 

Zeile 1 des Firmennamens.
