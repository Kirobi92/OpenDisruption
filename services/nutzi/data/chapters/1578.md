# Umlagerung Intern Registerkarte Vorschlag 
Umlagerung intern Registerkarte Vorschlag 

Auf dieser Registerkarte werden Vorschläge angezeigt, die den Selektionskriterien im Kopf des Dialogs entsprechen. 
### Button Alle buchen 

Mit diesem Button werden alle Vorschläge verbucht. 

### Button Einzeln buchen 

Mit diesem Button werden einzelne Vorschläge verbucht. Dazu muss die jeweilige Zeile markiert werden. 

### Button Zeile löschen 

Dieser Button löscht eine markierte Zeile der Tabelle. 

### Button Charge 

Dieser Button öffnet den Dialog Chargenzuordnung zu einer markierten Position. 

Siehe Kapitel Chargenzuordnung . 

### Button Lagerauftrag Erzeugen 

Dieser Button erstellt einen Lagerauftrag für die gewählten Umlagerungsvorschläge. 

### Tabelle 

### Tabellenspalte Ziellager 
Hier wird die Ziellagernummer angegeben. Das ist das Lager, welches aufgefüllt wird. 

Tooltip: 

Definiert das Lager, auf das zugelagert werden soll. 

### Tabellenspalte Ziel-Platz 
Hier wird der Ziel-Lagerplatz angegeben. 

Tooltip: 

Angabe des Lagerplatzes 

### Tabellenspalte Ziel-LHM 
Hier wird das Ziel-Ladehilfsmittel angegeben. 

Tooltip: 

Angabe des Ziel-Ladehilfsmittels 

### Tabellenspalte Quelllager 
Hier werden die Nummer und der Name des Lagers angegeben. 

Tooltip: 

Definiert das Lager 

### Tabellenspalte Quell-Platz 
Hier wird angegeben, auf welchem Lagerplatz die Packstücknummer zu finden ist. 

Tooltip: 

Angabe des Lagerplatzes

Hier wird die Artikelnummer angegeben. 

Tooltip: 

Angabe der Artikelnummer 

### Tabellenspalte Bezeichnung 
Die Bezeichnung des Artikels wird automatisch eingetragen, nachdem die Artikelnummer eingetragen wurde. 

Tooltip: 

Artikelbezeichnung 

### Tabellenspalte Charge 
Hier wird die Chargennummer angegeben. 

Die Chargenverwaltung steht nur zur Verfügung, wenn sie lizenziert ist. Dies erkennen Sie, wenn im Dialog Konfiguration die Checkbox aktiv gesetzt ist. 

Siehe Kapitel Konfiguration der Parameter . 

Tooltip: 

Angabe der Chargennummer

Tooltip: 

Angabe der Lagereinheit. Dazugehörige Pickliste heißt „Mengeneinheit“.
