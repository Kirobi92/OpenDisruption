# Budget generieren 
Budget generieren 

Dieser Dialog öffnet sich im Sachkontenstamm >> Optionen >> Budget generieren . Wie bereits in der Kostenrechnung ist es auch in der Finanzbuchhaltung möglich, für jedes neue Geschäftsjahr die voraussichtlichen Kosten zu planen. 

Diese neuen Budget-Daten stehen dann im Dialog Sachkonto auf der Registerkarte Budget zur Verfügung. 
## Steuerelemente des Formulars 

### Bezeichnungsfeld Budget generieren 
Tooltip: 

In diesem Dialog können Sie die voraussichtlichen Kosten für ein Geschäftsjahr planen. 
Für mehr Informationen drücken Sie F1. 

### Button Form-Menü 

Mit Klick auf diesen Button werden je nach Kontext und Berechtigung die unterschiedlichen Optionen des jeweiligen Dialogs hinter dem Button zur Auswahl angeboten. 

Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button starten Sie die Suche. 

Sie können ganz allgemein suchen, indem Sie den Button drücken, gleich nach Öffnen des Dialogs. Oder Sie geben im Kopf des Dialogs die Selektionskriterien vor und drücken dann diesen Button. 

### Button Neu 

Dieser Button leert die Daten in den Kopf-Feldern. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Dieser Button startet eine Abfrage, ob das Budget in das Jahr xxxx vorgetragen werden soll, mit dem Hinweis auf Überschreiben der bisherigen Daten. Außerdem werden die geänderten Werte auf den Registerkarten in die Datenbank zurückgeschrieben. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

### Button Löschen 

Mit diesem Button werden Zeilen temporär aus der Tabelle gelöscht. 

### Button Schließen 

Mit diesem Button schließen Sie den Dialog. Bitte denken Sie daran, Ihre Eingaben / Änderungen vorher zu speichern! 

### Field Panel Angabe der Selektionsrkriterien 

Mit einer Eingabe in die Felder Periode von … bis werden nicht selektierte Perioden in der Tabelle ausgeblendet. 

Die Felder von Kontonr und bis Kontonr dienen der Selektion der Konten. 
### Combobox Jahr 
Für die Generierung der Planwerte wählen Sie hier das Jahr der Basis (Ist-Werte) aus. 

### Combobox Periode von 
Die Periode, die Sie hier auswählen wird in der Tabelle ausgeblendet. 

### Combobox Periode bis 
Wenn Sie mehrere Perioden ausblenden möchten, wählen Sie hier die nächste Periode aus.

### Field Panel Budget modifizieren 

In diesem Bereich wird das Budget modifiziert. 

Ist im Feld Betrag ein Wert eingetragen, wird dieser zu jedem Periodenbudget addiert. 

Ist im Feld Prozent ein Wert erfasst, wird jedes Periodenbudget prozentual verändert. 
### Feld Betrag 
Der Betrag, den Sie hier eingeben, wird zu jedem Periodenbudget addiert. 

Tooltip: 

Angabe des Betrags 

### Feld Prozent 
Wenn Sie hier einen Wert eingeben, wird das Periodenbudget prozentual verändert.
