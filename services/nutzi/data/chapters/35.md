# Globale Suche

eNVenta ERP 

# Registerkarte Suche 
Registerkarte Suche 

Auf dieser Registerkarte können Sie nach Vorgängen suchen. Sie gelangen auf diese Registerkarte, wenn Sie auf das Symbol klicken. Dazu steht ein Eingabefeld für die konkrete Suche und Comboboxen für eine Filterung über Methode und Filter zur Verfügung. 

Bei der Suche nach einem Namen wird in der Ergebnisliste zwischen Kunde und Ansprechpartner als Objekt unterschieden. Auf diese Weise ist auf einen Blick erkennbar, ob es sich um einen Firmennamen oder eine natürliche Person (Ansprechpartner) handelt. 

Bei Doppelklick auf einen Treffer vom Typ „Ansprechpartner“ öffnet sich der entsprechende Dialog Ansprechpartner . 

Die globale Suche erlaubt es dass alle Vorgänge, für die eine Wiedervorlage erfasst werden kann, auch in der Suche berücksichtigt sind. Die Suche kann auch nach Vorgang gefiltert werden. Die Vorgänge, die in der Combobox enthalten sind, sind auch diejenigen, die indexiert werden und die bei der Suche berücksichtigt werden. 

Zu diesem Zweck ist die externe Datei-basierte Suche-Engine Lucene.Net an das System angebunden. Dabei handelt es sich um eine geschwindigkeitsoptimierte .Net API Indizierungs- und Recherchefunktionalität.

Es wird ein unabhängiger Index aufgebaut, der die Daten aller recherchierten Objekte enthält. Dies erlaubt eine objektübergreifende Suche. Es ist möglich die Suche auf eine bestimmte Objektart (z.B. nur Artikel) einzuschränken. 

Mehr zum Thema Lucene-Suche erfahren Sie im Kapitel Parameter Allgemein auf der Registerkarte Suche . 
## Steuerelemente des Formulars 

### Feld Lucene-Suche 
Hier geben Sie einen Begriff ein, den Sie mit Lucene suchen wollen. 

### Combobox Methode 
Hier wählen Sie eine Methode zur Filterung der Daten aus. 

### Combobox Filter 
Hier wählen Sie einen Filter zur Selektion aus. 

### Bezeichnungsfeld Aktuelle Information 
Hier werden die aktuelle Datenbank und der aktuelle Benutzer angegeben.
