Die Funktionen für das Client Side Customizing sowie die Online-Hilfe stehen hier ebenfalls zur Verfügung.

### Button Suche 

Mit diesem Button wird die ursprüngliche Einstellung wieder hergestellt. 

### Button Neu 

Dieser Button leert den Dialog für eine neue Eingabe. 

Ist in den Parametern für die Sachbearbeiter die Checkbox 2 Schritt Neu aktiviert, bleiben die Suchkriterien in den Stammdialogen erhalten. 

Siehe Kapitel Parameter Sachbearbeiter . 

### Button Speichern 

Mit diesem Button speichern Sie Ihre Eingaben / Änderungen. 

### Button Neuer Satz 

Mit diesem Button fügen Sie weitere Daten in den Dialog ein. 

In diesem Fall öffnet sich der Dialog Server Datei Auswahl , um einen Pfad auszuwählen. 

Siehe Kapitel Server Datei Auswahl . 

Tooltip: 

Datei(en) hinzufügen 

### Button Löschen 

Dieser Button löscht Ihre Eingaben. Genauer gesagt, die markierte Zeile wird mit dem versehen und erst endgültig gelöscht, wenn Sie den Button drücken. 

### Button Schließen 

Dieser Button schließt den Dialog. 

Vergessen Sie nicht Ihre Eingaben zuvor zu speichern! 

### Feld Referenz

Generierter Anzeigetext oder Referenztext. 

### Tabelle Dateien 

Hier werden verschiedene Dateien angeboten. 

### Combobox Link-Typ 
Aus dieser Combobox wählen Sie aus, mit was das Bild verknüpft werden soll. 

Beispiel: 

- Artikelbild - Zeugnis - Konzept - Datenblatt - Gefahrguthinweis 
Einem Artikel können nur Bilddateien zugeordnet werden. Hierbei unterstützt die Software folgende Bildformate: 

- BMP - GIF - PNG - JPG - JPEG - WMF - TIF 
Für ein Bild ist der Verwendungszweck ‚Bild‘ im Dialog Referenz-Link-Typen unbedingt zu setzen. Nur so bleiben die Daten bei der Speicherung korrekt sichtbar und wird entsprechend gespeichert. 

Siehe Kapitel Referenz-Link-Typen .
