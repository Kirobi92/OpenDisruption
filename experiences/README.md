# Experiences – Lebendiges Gedächtnis

**Zone:** Gemischt (WORKSPACE/FAMILY_PRIVATE) | **Verantwortlich:** Alle Agenten

## Zweck
Das `experiences/`-Verzeichnis ist Kirobis lebendiges Gedächtnis. Hier werden Erfahrungen, Projekte, Lernpunkte und Experimente dokumentiert. Im Gegensatz zu `canon/` (kanonische Wahrheit) enthält `experiences/` das "gelebte Wissen" – die konkreten Erlebnisse und Erkenntnisse aus dem Alltag.

## Struktur

| Verzeichnis | Beschreibung |
|-------------|-------------|
| `family/` | Familien-Erfahrungen (FAMILY_PRIVATE) |
| `projects/` | Projekt-Dokumentationen |
| `learnings/` | Lernpunkte und Best Practices |
| `experiments/` | Experimente und Evaluierungen |
| `knowledge/` | Tiefere Wissensartikel |
