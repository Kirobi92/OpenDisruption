# Emotionales Manometer – Familie Darusi

*Das Manometer ist ein Werkzeug um familiäre Stimmungen bewusst zu erfassen und Muster zu erkennen.*

## Skala

```
1 - Sehr belastet / Krisenhafte Situation
2 - Belastet / Herausfordernd
3 - Neutral / Ruhig
4 - Gut / Positiv
5 - Sehr gut / Fließend
```

## Format für Einträge

```
### YYYY-MM-DD [optional: Tageszeit]
Sven: [1-5] – [kurze Notiz optional]
Samira: [1-5] – [kurze Notiz optional]
Sineo: [1-5] – [kurze Notiz optional]
Familie: [1-5] – [Gesamtstimmung]
```

## Nutzungshinweis

Das Manometer ist freiwillig. Kein Familienmitglied muss teilnehmen. samira-heart-agent kann Muster analysieren und proaktiv berichten wenn Trends erkennbar sind.
